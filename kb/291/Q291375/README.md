## Q291375: Microsoft Exchange 2000 Server Resource Kit Comments And Correct

	Article: Q291375
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 13-MAR-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Exchange 2000 Server Resource Kit ISBN 0-7356-1017-7 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Microsoft Exchange 2000 Server Resource
	Kit, ISBN 0-7356-1017-7.
	
	The following topics are covered:
	
	- Page 577: In Table A.2, London-02 System Requirements Are Incorrect
	
	MORE INFORMATION
	================
	
	Page 577:  In Table A.2, London-02 System Requirements Are Incorrect
	--------------------------------------------------------------------
	
	On page 577, in Table A.2, The CPU and Memory fields for Server LONDON-02 do not
	meet the minimum system requirements for Windows 2000 Server. The requirements
	should read "133 MHz or higher Pentium-compatible" for CPU and "128 MB" for
	Memory.
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
