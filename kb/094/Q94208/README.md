## Q94208: Support for Canon LBP-200 Series Printers

	Article: Q94208
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	Canon has two lines of the LBP series printers: a consumer line and a commercial
	line. Identical printers may be called by different names if one is in the
	consumer line and the other is in the commercial line. The following chart shows
	the equivalencies:
	
	  Consumer           Commercial
	  -----------------------------
	
	  LBP-200SI   <==>   LBP-8I
	  LBP-200SII  <==>   LBP-8II
	
	  LBP-200SAI  <==>   LBP-8AI
	  LBP-200SAII <==>   LBP-8AII

	performance or reliability.
