## Q94031: Club 486/25 Machine Restarts or Hangs Intermittently

	Article: Q94031
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	According to Club American technical support, if you run Windows on some
	models of Club 486/25, the system may restart randomly or stop responding
	(hang) while printing.
	
	Club recommends selecting Everex Step/25 as the computer type when
	installing Windows 3.0, 3.0a, or 3.1. If the problem persists, contact Club
	American technical support to upgrade the machine.
	
	The Club 486/25 is not on the hardware compatibility list for Windows
	3.0, 3.0a or 3.1.
