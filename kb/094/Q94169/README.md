## Q94169: IBM 4226 Printer Emulation for Windows

{% raw %}

	Article: Q94169
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	Windows 3.1 and Windows for Workgroups 3.1 do not support the IBM 4226.
	
	According to IBM/Lexmark technical support, the following emulation modes and
	printer drivers should be selected for the 4226 (9-pin wide-carriage) printer:
	
	  IBM Mode:      IBM Proprinter XL III
	  Epson Mode:    Epson DFX 5000
	
	The printer itself should be set for the correct mode (IBM or Epson), depending
	on the emulation chosen.
	
	MORE INFORMATION
	================
	
	For more information about settings, see the "IBM 4226 Printer Manual" or call
	Lexmark technical support.
