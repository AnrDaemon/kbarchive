## Q94560: Sound Finder &amp; Quick Recorder Description &amp; Label Text Limits

{% raw %}

	Article: Q94560
	Product(s): Miscellaneous Windows Products
	Version(s): 1.00
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows Sound System, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Sound Finder and Quick Recorder applications included with Windows Sound
	System can attach descriptions and labels to .WAV files. However, there are
	limits to the number of characters that can be entered into a .WAV file's
	description and label.
	
	MORE INFORMATION
	================
	
	In Sound Finder, the maximum length possible for a description is 4095
	characters, entered into a maximum of 1365 lines.
	
	In Quick Recorder, the maximum length possible for a description is 255
	characters, entered into a maximum of 85 lines.
	
	
	The maximum length of an attached label is 255 characters. This limit applies to
	both Sound Finder and Quick Recorder.
	
	REFERENCES
	==========
	
	"Microsoft Windows Sound System User's Guide," pages 24, 105-106, 110
