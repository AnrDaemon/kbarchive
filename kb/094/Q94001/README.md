## Q94001: Houston Instruments Plotter Driver

{% raw %}

	Article: Q94001
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following plotters are supported by the Houston Instruments Plotter Driver
	for Microsoft Windows:
	
	- Image Maker
	
	- DMP 29
	
	- DMP 40/40/42
	
	- DMP 51/52 and Multipen 51/52
	
	- DMP 56A
	
	- DMP 61/62
	
	- DMP 161/162
	
	This driver can be obtained from Houston Instruments.
	
	MORE INFORMATION
	================
	
	The Houston Instruments product included here is manufactured by a vendor
	independent of Microsoft; we make no warranty, implied or otherwise, regarding
	this product's performance or reliability.
