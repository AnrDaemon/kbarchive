## Q94557: Write Restores Subscripts to Incorrect Font Size

{% raw %}

	Article: Q94557
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SYMPTOMS
	========
	
	When Microsoft Windows or Windows for Workgroups Write, the font point size may
	be set incorrectly when restoring from a subscript mode to the original state.
	
	WORKAROUND
	==========
	
	To work around this problem:
	
	1. Select the incorrectly sized text.
	
	2. From the Character menu in Write, choose Font to manually reset the font to
	  its proper size.
	
	MORE INFORMATION
	================
	
	This problem applies to all font types (TrueType, system, and so on) set at font
	point sizes 4-127, with the exception of point sizes 6, 8, 10, 12, 14, 24, 26,
	36, 72, and 127.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Write. We are researching this
	problem and will post new information here as it becomes available.
