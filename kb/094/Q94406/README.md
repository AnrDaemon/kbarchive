## Q94406: DCA's IRMA Board Requires Updated Drivers

	Article: Q94406
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	According to DCA, when using DCA's terminal emulation board with Microsoft
	Windows for Workgroups version 3.1, the updated DCA drivers, TOK_A0.DCA and
	TOK_380.DCA, must be replaced with version 2.02. These new drivers can be
	downloaded from DCA's BBS at (404) 740- 8428. The files to download are:
	
	  ITR_2021.EXE
	  ITR2022.EXE
	
	MORE INFORMATION
	================
	
	Contact DCA for more information.
	
	To obtain phone numbers for DCA, query on the following words in the Microsoft
	Knowledge Base:
	
	  PSS and DCA and phonebook

	Microsoft; we make no warranty, implied or otherwise, regarding these products'
	performance or reliability.
