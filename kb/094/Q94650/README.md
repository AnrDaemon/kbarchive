## Q94650: SoundBits: Media Browser Set Sounds Button Fails

	Article: Q94650
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft SoundBits (all collections), version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Set Sounds button in Media Browser (MBROWSER.EXE may fail if the Control
	Panel is open, and a Control Panel icon has been activated.
	
	NOTE: The Set Sounds button does not function in Windows 95.
	
	
	WORKAROUND
	==========
	
	Focus is set on the Control Panel Colors dialog box. To work around this issue,
	close any Control Panel applications prior to selecting the Set Sounds button in
	Media Browser.
