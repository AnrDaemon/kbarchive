## Q94818: The 44 TrueType Fonts Offered in TrueType Font Pack 2

	Article: Q94818
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft TrueType Font Pack for Windows 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following are the 44 TrueType fonts offered in the Microsoft TrueType Font
	Pack 2 for Windows:
	
	Headline Fonts
	--------------
	
	1. Braggadocio
	2. Britannic Bold
	3. Castellar
	4. Contemporary Brush
	5. Contemporary Brush Bold
	6. Desdemona
	7. Elephant
	8. Elephant Italic
	9. Gill Sans Ultra Bold
	10. Impact
	11. Onyx
	12. Playbill
	13. Ransom
	14. Ransom Bold
	15. Ransom Italic
	16. Ransom Bold Italic
	17. Stencil
	18. Stop
	19. Wide Latin
	
	Decorative Fonts
	----------------
	
	20. Augsburger Initials
	21. BriemScript
	22. Eckmann
	23. Edda
	24. Gradi
	25. Harrington
	26. Mistral
	27. Old English Text MT
	28. Parade
	29. Peignot Medium
	
	Text Fonts
	----------
	
	30. Baskerville Old Face
	31. Bell MT
	32. Bell MT Bold
	33. Bell MT Italic
	34. Centaur
	35. Eurostile
	36. Eurostile Bold
	37. Futura Medium
	38. Futura Bold
	39. Futura Oblique
	40. New Caledonia
	41. New Caledonia Bold
	42. New Caledonia Italic
	
	Symbol Fonts
	------------
	
	43. Wingdings2
	44. Wingdings3
	
	REFERENCES
	==========
	
	"Microsoft TrueType Font Pack 2 User's Guide," pages 83-102
