## Q94360: Sound Recorder Does Not Record in Stereo

	Article: Q94360
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 26-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	The Sound Recorder application included with Microsoft Windows operating
	system version 3.1 cannot record sounds in stereo. However, it is capable
	of playing stereo wave (.WAV) files. Sound Recorder records sounds in mono.
