## Q47209: New Features for Flight Simulator Version 4.00

{% raw %}

	Article: Q47209
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	Primary new features for Flight Simulator Version 4.00 are the following:
	
	  Aircraft Design
	  Glider Simulator
	  Improved Flight Characteristics
	  Dynamic Scenery (background air traffic, fuel trucks, and more)
	  Random Weather Generator
	  80386 Optimization
	
	There is no upgrade policy for Flight Simulator Version 4.00. To obtain Flight
	Simulator Version 4.00, call Microsoft Sales at (800) 426-9400.
