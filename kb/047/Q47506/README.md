## Q47506: Mac Wkst: Address Mail and Notifier Window Keyboard Shortcuts

	Article: Q47506
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:3.0,3.0a,3.0b,3.1,3.1a,3.1c
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, versions 3.0, 3.0a, 3.0b, 3.1, 3.1a, 3.1c 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following are keyboard shortcuts for the Microsoft Mail Address Mail and
	Notifier windows:
	
	Address Mail Window
	-------------------
	
	Description                         Command           Notes
	
	Remove Recipient Name               COMMAND+R
	Search for Characters               COMMAND+S         2.0
	Close Window                        COMMAND+W
	Blind Carbon Copy                   COMMAND+OPTION+B
	Carbon Copy                         COMMAND+OPTION+C
	To                                  COMMAND+OPTION+T
	Close Address window                COMMAND+W
	Find name(s)                        COMMAND+F         3.0 and later only
	Select nth directory icon*          COMMAND+n*        3.0 and later only
	Details                             COMMAND+D         3.0 and later only
	Get Help                            COMMAND+?
	
	* n is any number from 1 through 9.
	
	Notifier Window
	---------------
	
	Description                         Command
	
	Read Later                          COMMAND+L
	Close Window                        COMMAND+W
	Get Help                            COMMAND+?
	
	For more information, see Appendix A of the Microsoft Mail "User's Guide."
