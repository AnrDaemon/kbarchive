## Q47520: Mac Wkst: Summary Window Keyboard Shortcuts

	Article: Q47520
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:3.0,3.0a,3.0b,3.1,3.1a,3.1c
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, versions 3.0, 3.0a, 3.0b, 3.1, 3.1a, 3.1c 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following are keyboard shortcuts for the Microsoft Mail Summary window:
	
	Description                         Command           Notes
	
	Delete message                      COMMAND+D
	Forward message                     COMMAND+F
	Open Send Note window               COMMAND+N         2.0 only
	Print message                       COMMAND+P
	Close and sign out                  COMMAND+Q
	Reply to message*                   COMMAND+R         (See below)
	Save message                        COMMAND+S         2.0 only
	Close window                        COMMAND+W
	OpenSend nth message type**         COMMAND+n*        n is a number
	                                                     from 1 to 9
	Get help                            COMMAND+?
	Reply to all*                       COMMAND+OPTION+R  (See below)
	Open personal address book          COMMAND+B         3.0 and later only
	Go to Mailbox folder                COMMAND+G         3.0 and later only
	Move message into a mail folder     COMMAND+M         3.0 and later only
	Copy message into a mail folder     COMMAND+OPTION+M  3.0 and later only
	Create new folder                   COMMAND+N         3.0 and later only
	
	* If you turned on the Address Reply To All Original Recipients check box in the
	Preferences dialog box, the Reply button changes to Reply All. In this case,
	COMMAND+R replies to all recipients, and OPTION+COMMAND+R replies only to the
	sender.
	
	** n is any number from 1 through 9. For example, COMMAND+4 opens the Send window
	for the 4th message type (Inquiry). The key combinations COMMAND+5 through
	COMMAND+9 apply only if your network manager creates additional message types.
	
	For more information, see Appendix A of the Microsoft Mail "User's Guide."
