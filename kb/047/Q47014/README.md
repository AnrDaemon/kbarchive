## Q47014: Mac Srv: Mail Server and AppleShare Server Coexistence

	Article: Q47014
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:2.0,2.0a,2.0b,3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	A Microsoft Mail server and an AppleShare version 2.01 server can coexist on the
	same Macintosh.
	
	When installing the Mail server on the AppleShare server, the Installer puts the
	Microsoft Mail files in the folder containing the AppleShare files, rather than
	in the System Folder. This folder is normally named "Server Folder."
