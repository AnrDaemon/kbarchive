## Q47332: Proper Dip-Switch Settings for an IBM Proprinter II

	Article: Q47332
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:2.x,3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 2.03, 2.1, 2.11, 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	This article lists the recommended dip-switch settings for printing to an IBM
	Proprinter II with Windows applications.
	
	MORE INFORMATION
	================
	
	  Option                        Setting
	  ------                        -------
	
	  Form Feed Enable              On
	  Select Near Letter Quality    On
	  Auto Line Feed                Off
	  Form Length                   Off
	  Character Set                 Off
	  Auto Carriage Return          Off
	  Reserved                      On
	
	If the Auto Line Feed and Auto Carriage return switches are set to On, you may
	get extra line spaces when printing from Windows applications.
