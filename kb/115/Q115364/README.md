## Q115364: X400: Comma Before Final Term Omitted by Display Format

	Article: Q115364
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to X.400, version 3.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The -AR option in version 3.0 displays the address in the following format:
	
	  givenName initials surname, generationQualifier
	
	but the current address display format omits the comma before the final term:
	
	  givenName initials surname generationQualifier
	
	
	Additional query words: 3.20
