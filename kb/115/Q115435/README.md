## Q115435: DriveSpace Help Refers to MS-DOS 6 Backup and &quot;User's Guide&quot;

	Article: Q115435
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you compress a drive with DriveSpace, you are prompted to back up your
	system before continuing with the compression. The Help topic for this dialog
	box refers to MS-DOS 6 Backup and the MS-DOS 6 "User's Guide."
	
	You should use MS-DOS 6.22 Backup and refer to the MS-DOS 6.22 "User's Guide."
	
	Additional query words: screen DRVSPACE msbackup mwbackup doc err docerr
