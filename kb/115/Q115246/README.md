## Q115246: Mac MSPlus: Schedule+ Does Not Propagate Old Data

	Article: Q115246
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for AppleTalk Networks, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In Schedule+ for Microsoft Mail for AppleTalk Networks, old Schedule+ calendar
	data is not propagated to new Mail servers.
	
	When a new Microsoft Mail server appears on the network, existing Mail servers
	send data from the current date and continuing for four months if the correct
	preferences have been enabled by the Network Manager.
