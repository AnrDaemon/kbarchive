## Q115205: PC Tools Disk Formatter Err Msg: Can't Format Strange Disk

	Article: Q115205
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 6.22 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you try to format an MS-DOS DoubleSpace- or DriveSpace-compressed floppy disk
	with the Windows Disk Formatter from PC Tools version 8.0, you receive the
	following error message:
	
	  Can't format strange disk.

	================

	regarding these products' performance or reliability.
