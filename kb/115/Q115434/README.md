## Q115434: Setup Does Not Write Files to Automounted Uninstall Disk

{% raw %}

	Article: Q115434
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 6.22 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When a DoubleSpace-compressed floppy disk in drive A is mounted automatically
	before you run MS-DOS 6.22 Setup, no information is written to the Uninstall
	disk. As a result, you cannot uninstall MS-DOS 6.22.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in MS-DOS version 6.22 Setup. We
	are researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
	
	Additional query words: dblspace automounted automount auto mount mounted
