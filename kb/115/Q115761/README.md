## Q115761: SAMPLE: SPINDIB Demonstrates 90-degree DIB Rotation

	Article: Q115761

	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows Software Development Kit (SDK) 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	SPINDIB demonstrates how to perform 90-degree rotation on a device- independent
	bitmap (DIB). SPINDIB supports 1-bit-per-pixel (BPP), 4-BPP, and 8-BPP DIB
	rotation. This sample also demonstrates how to flip and/or mirror an image using
	StretchBlt/StretchDIBits.
