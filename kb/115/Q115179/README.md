## Q115179: Avoid Running Software Carousel Task Swapper and Setup/Defrag

{% raw %}

	Article: Q115179
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 03-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you are using Software Carousel version 6.0 as a task swapper, you may be
	able to run MS-DOS Setup, Defragmenter (Defrag), DoubleSpace, or DriveSpace
	successfully. However, there is a risk of data loss if your applications are
	loaded in the Software Carousel task swapper.
	
	To avoid potential data loss, ensure you are not running applications from the
	Software Carousel task swapper when you run MS-DOS Setup, Defrag, DoubleSpace,
	or DriveSpace.
