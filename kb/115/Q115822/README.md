## Q115822: CD Sampler: Mouse Required Because No Keyboard Equivalents

	Article: Q115822
	Product(s): Microsoft Home Multimedia Titles
	Version(s): Spring 1994
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Home CD Sampler Spring 1994 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	It is not possible to navigate through the Home CD Sampler using the keyboard.
	To choose demonstration buttons and/or exit the program, you must use the mouse.
	
	MORE INFORMATION
	================
	
	The list of system requirements includes "Microsoft Mouse or compatible pointing
	device."
