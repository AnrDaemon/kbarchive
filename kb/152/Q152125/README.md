## Q152125: Underscore Treated as Wildcard in Address Book Find Feature

{% raw %}

	Article: Q152125
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.2,1.3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-AUG-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.2, 1.3 
	-------------------------------------------------------------------------------
	
	The Find feature of the e-mail address book on MSN, The Microsoft Network,
	treats the underscore character as a wildcard character. As a result, the
	Find feature cannot locate member IDs that begin with an underscore.
	
	
	Additional query words: msn 1.20 1.30
