## Q152772: Mapiform.exe How to Send Mail Messages with MAPI

	Article: Q152772
	Product(s): Microsoft FoxPro
	Version(s): WINDOWS:3.0,3.0b,6.0

	SUMMARY
	=======
	
	Mapiform.exe is a sample that uses the MAPI OLE Controls to send mail messages
	to multiple recipients with the choice of attaching a file.
