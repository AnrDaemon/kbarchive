## Q152454: Error Saving Write Document to a Network Drive

	Article: Q152454
	Product(s): Microsoft Windows NT
	Version(s): 3.51
	Operating System(s): 
	Keyword(s): kbnetwork
	Last Modified: 20-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.51 
	- Microsoft Windows NT Server version 3.51 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	Microsoft Write may report the following error when trying to save a file to a
	network drive:
	
	  Cannot save file. Make sure file or disk is not damaged.
	
	This problem only occurs on a fast computer, such as with a Pentium 166 or higher
	processor.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 3.51. This
	problem was corrected in Windows NT version 4.0.
