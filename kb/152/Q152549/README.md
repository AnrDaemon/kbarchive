## Q152549: Mac Bookshelf '96 - '97: How to Determine Installed Version

{% raw %}

	Article: Q152549
	Product(s): Microsoft Home Multimedia Titles
	Version(s): MACINTOSH:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf 1996-97 for Macintosh 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following article tells how to determine what version of Bookshelf '96- '97
	is installed on your computer.
	
	The Microsoft Bookshelf 1996-97 compact disc contains an installation for the
	Power Macintosh (also referred to as the PPC or Native) version and the 68K
	(680x0) version of the software. The installer installs one of these versions,
	depending on the type of Macintosh you are using.
	
	MORE INFORMATION
	================
	
	To determine the version of Bookshelf '96-'97 installed on your computer, do the
	following:
	
	1. In the Finder of your hard drive, locate the Bookshelf program icon.
	
	2. Click the icon one time.
	
	3. On the File menu, click Get Info.
	
	4. If the application's size is 4.7 MB, then the 68K version is installed.
	
	  If the application's size is 5.3 MB, then the PPC version is installed.
