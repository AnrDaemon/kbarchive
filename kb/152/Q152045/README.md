## Q152045: &quot;Use Mappings in file&quot; Check Box Grayed Out

{% raw %}

	Article: Q152045
	Product(s): Microsoft Windows NT
	Version(s): winnt:3.51
	Operating System(s): 
	Keyword(s): kbnetwork
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 3.51 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	In the Migration Tool (Nwconv.exe), the Use Mappings in File check box, the
	Create button and Edit button are grayed out("disabled") on "User and Group
	Options."
	
	
	WORKAROUND
	==========
	
	- Click OK on "User and Group Options" and continue with migration process;
	  because the Use Mappings in File option is grayed out, it will not cause a
	  problem.
	
	  OR -
	
	- Go back to the Users and Group Options window and clear the Use Mappings in
	  File box and continue
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 3.51. A fix
	to this problem is available, but has not been fully regression-tested and
	should be applied only to systems experiencing this specific problem. Unless you
	are severely impacted by this specific problem, Microsoft recommends that you
	wait for the next Service Pack that contains this fix. Contact Microsoft Product
	Support Services for more information.
