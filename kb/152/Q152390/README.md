## Q152390: Gardening: Back Button Does Not Return to Original Category

	Article: Q152390
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Complete Gardening for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you change Plant Category using the Plant Selector in Microsoft Complete
	Gardening, when you click Back, you do not return to the previously selected
	Plant Category.
	
	Example
	-------
	
	If you select a Plant Category in the Plant Selector and view the photos for that
	category in the Photo Browser, when you return to the Plant Selector and change
	the category, the Back button does not return you to the original Plant Category
	viewed in the Photo Browser.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Complete Gardening
	version 1.0 for Windows. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
