## Q152581: Complete Gardening: Can't Add Just One Plant to Plant List

	Article: Q152581
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Complete Gardening for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	You cannot add a single plant name from the Plant Selector to a Plant List in
	List Maker. You must add an entire list.
	
	MORE INFORMATION
	================
	
	This behavior is by design. Plant Lists are designed for saving lists created by
	using queries in the Plant Selector.
	
	Additional query words: 1.0 multi media multimedia multi-media mmtitles kbmm gardens
