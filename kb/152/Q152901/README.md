## Q152901: XADM: MTA Reports Repeated Occurrence of Event Id 1209

	Article: Q152901
	Product(s): Microsoft Exchange
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 29-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you view the Windows NT Event Viewer application log, the following error
	might appear multiple times:
	
	  MSExchangeMTA Event ID 1209
	
	  Unable to recognize an internal message identifier (LPI) from entity <x.500
	  name >. The incorrect LPI is %1 - % 2 - %3. Control block index %4.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server
	version 4.0. This problem was corrected in Microsoft Exchange Server 4.0 U.S.
	Service Pack 2. For information on obtaining the service pack, query on the
	following word in the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
	
	
	Additional query words: event mta site connector
