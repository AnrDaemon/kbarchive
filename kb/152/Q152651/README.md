## Q152651: Audit32.log Displays &quot;Unable to write Status MIF&quot; Error

	Article: Q152651
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.0,1.1
	Operating System(s): 
	Keyword(s): kbnetwork kbDatabase smsdatabase
	Last Modified: 31-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server versions 1.0, 1.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	On clients that have been successfully audited, the Audit32.log file reports the
	following error:
	
	  Initializing.
	  Reading packages from input file.
	  Scanning drives for packages.
	  Writing found packages to MIF.
	  Copying MIF to server.
	  Terminating.
	  Unable to write Status MIF.
	
	MORE INFORMATION
	================
	
	This error message can be ignored. It has no bearing on the actual audit package
	job status.
