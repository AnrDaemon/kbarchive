## Q264428: XFOR: Finding Instructions for Exchange PROFS Connector Setup

	Article: Q264428
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5 SP3
	Operating System(s): 
	Keyword(s): exc55sp3
	Last Modified: 26-JUN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 SP3 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article provides the location of detailed instructions on how to set up the
	Microsoft Exchange Connector for IBM OfficeVision/VM (PROFS).
	
	MORE INFORMATION
	================
	
	The Readme.rtf file that is located in the Eng\Exchconn\Setup\I386\Ntsconn
	folder on the Exchange Server 5.5 Service Pack 3 CD-ROM includes detailed
	step-by-step instructions on how to set up the Exchange PROFS Connector.
