## Q264729: WD97: Deleting Bulleted Text May Cause Hebrew Word 97 to Stop

{% raw %}

	Article: Q264729
	Product(s): Word 97 for Windows
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdta word8 word97 kbnumbering
	Last Modified: 04-MAR-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	NOTE: The information in this article applies to the 
	Hebrew version(s) of Microsoft Word 97.
	
	SYMPTOMS
	========
	
	When you attempt to delete bulleted text, Microsoft Word may stop responding
	(hang) when the following conditions are true:
	
	- You are using the Hebrew version of Microsoft Word 97 for Windows.
	
	  -and-
	
	- The "Track changes while editing" option is turned on.
	
	
	CAUSE
	=====
	
	This problem occurs when the document was originally created in a version of
	Word that is not enabled for right-to-left (RTL) text [for example, the English
	(United States) version of Microsoft Word 97 for Windows].
	
	RESOLUTION
	==========
	
	This problem has been corrected in the Microsoft Office 97 Arabic Edition
	Software Update 2. To obtain this update, please contact Microsoft Israel. Or
	check the following Microsoft Web sites for downloads of this update:
	
	  http://www.microsoft.com/middleeast
	
	  http://www.microsoft.com/israel
	
	NOTE: This update is only for right-to-left language versions of Microsoft Office
	97 for Windows Arabic Edition.
	
	
	WORKAROUND
	==========
	
	To temporarily work around this problem, turn off the "Track changes while
	editing" option. To do this, follow these steps:
	
	1. On the Tools menu, point to Track Changes and then click Highlight Changes.
	
	2. In the Highlight Changes dialog box, click to clear the "Track changes while
	  editing" check box, and then click OK.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
