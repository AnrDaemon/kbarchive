## Q264046: WD97: Cannot Insert Special Character Using a Macro

	Article: Q264046
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta word8 word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you use a macro to show the Symbol dialog box, you cannot insert a
	character from the Special Characters tab. When you attempt to insert a special
	character, such as a copyright symbol, the character selected on the Symbols tab
	is inserted instead of the specified symbol.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
