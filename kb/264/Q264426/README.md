	SUMMARY
	=======
	
	This article provides a list of Microsoft Knowledge Base articles that may help
	you understand some basic X.400 connector concepts, and that may be useful for
	initial basic X.400 connector troubleshooting. This is not a complete list; it
	is simply a reference to some potentially useful KB articles about the X.400
	connectors that are used in Exchange Server versions 4.0, 5.0, and 5.5.
