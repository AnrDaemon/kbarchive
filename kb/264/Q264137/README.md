## Q264137: MTM2: Cannot Change Weather on the Graveyard Track

	Article: Q264137
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:2.0
	Operating System(s): 
	Keyword(s): kbimu msgame
	Last Modified: 12-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Monster Truck Madness 2, version 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you race on the Graveyard track in Microsoft Monster Truck Madness 2, you
	cannot change either the time of day or the weather conditions after the track
	is loaded.
	
	MORE INFORMATION
	================
	
	The Graveyard track is designed to be run at night and in the rain.
	
	Additional query words: 2.00 msgame mtm2 effects dark raining
