## Q98293: Err Msg: This Version of the Help File Not Supported

	Article: Q98293
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0,6.2,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.0, 6.2, 6.22 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You may receive the following error message when you try to start a Microsoft
	Windows help file installed by MS-DOS 6.0 or later:
	
	  This version of help file not supported.
	
	CAUSE
	=====
	
	This error occurs because some MS-DOS help files are not compatible with Windows
	3.0. The following table shows which Windows help files are compatible with
	Microsoft Windows 3.0:
	
	  3.0-Compatible      Not 3.0-Compatible
	  ---------------------------------------
	
	  MWAV.HLP            DBLWIN.HLP
	  MWBACKUP.HLP        SMARTMON.HLP
	  MWUNDEL.HLP
	
	Additional query words: 6.00 winhelp 6.20 3.10 3.11
