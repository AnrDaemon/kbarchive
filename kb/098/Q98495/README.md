## Q98495: Windows FTADMIN Generates Error 2140

{% raw %}

	Article: Q98495
	Product(s): Microsoft LAN Manager
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SYMPTOMS
	========
	
	Windows FTADMIN generates an error 2140 if used with Windows for Workgroups.
	
	CAUSE
	=====
	
	Windows for Workgroups is using an old NETAPI.DLL file.
	
	RESOLUTION
	==========
	
	Obtain the current NETAPI.DLL. Contact Microsoft Product Support Services.
	
	Additional query words: wfw wfwg 2.10 2.1 2.10a 2.1a 2.20 2.2
