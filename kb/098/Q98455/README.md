## Q98455: VSAFE Memory Usage

{% raw %}

	Article: Q98455
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you have conventional and extended memory available, VSAFE uses approximately
	23 kilobytes (K) of conventional memory and 23K of extended memory.
	
	If you have conventional and expanded memory available, VSAFE uses only 7K of
	conventional and 64K of expanded memory.
	
	VSAFE cannot use both extended and expanded memory.
