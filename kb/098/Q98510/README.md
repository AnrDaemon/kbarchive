## Q98510: How to Exit Defrag Without Losing Data

{% raw %}

	Article: Q98510
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0,6.2,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.0, 6.2, 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This information applies to both Microsoft DoubleSpace and Microsoft DriveSpace.
	For MS-DOS 6.22, use DRVSPACE in place of DBLSPACE for commands and filenames.
	
	Do not turn off the power or press the reset button when you are defragmenting
	any drive (including a DoubleSpace drive) using Microsoft Defragmenter (Defrag)
	or DBLSPACE /DEFRAG. If you want to exit Defrag or DBLSPACE /DEFRAG, press the
	ESC key to interrupt the program.
	
	MORE INFORMATION
	================
	
	If you believe Defrag or DBLSPACE /DEFRAG has stopped responding, press the ESC
	key to exit. If this does not work, you must restart your computer. If you have
	to restart your computer, it is preferable to press the CTRL+ALT+DELETE key
	combination; however, if this does not work, press the reset button or power off
	your computer.
	
	Additional query words: 6.00 hang hung frozen reboot 6.20
