## Q98340: PROFS: How to Check VM Host for REROUTE Statements

{% raw %}

	Article: Q98340
	Product(s): Microsoft Mail For PC Networks
	Version(s): 3.1,3.2,3.3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to IBM PROFS and OfficeVision, versions 3.1, 3.2, 3.3 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Use the following statement to check that the REROUTE statements for the PROFS
	gateway are correct and properly initialized.
	
	     SM ms_rscs_node_name Q SYS RE
	
	It shows all the REROUTE statements that were initialized the last time the given
	RSCS was rerouted.
	
	
	Additional query words: 3.10 3.20 3.30
