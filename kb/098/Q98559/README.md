## Q98559: Undelete or Anti-Virus Err Msg: NQ30.ZFD Not Found

{% raw %}

	Article: Q98559
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you are running Zenographics Corporation SuperPrint with MS-DOS 6.0 or later,
	you may receive the following error message when you run Microsoft Undelete for
	Windows or Anti-Virus for Windows:
	
	  NQ30.ZFD not found
	
	RESOLUTION
	==========
	
	For instructions on how to disable SuperPrint or work around this problem,
	contact Zenographics Corporation.
	
	For more information on SuperPrint, query on the following words in the Microsoft
	Knowledge Base:
	
	  " superprint and zenographics " (without the quotation marks)
	
	Additional query words: 6.22 6.00 3rd party third-party third msav mwav 6.20
