## Q98574: PC WSPlus: Cannot View Another User's Notes

	Article: Q98574
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:1.0,1.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, versions 1.0, 1.0a 
	- the operating system: Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use Microsoft Schedule+ for Windows to open another user's Appointment
	Book, it is not possible to view that user's Notes.
	
	Regardless of what level of access privileges you are given, you will not be able
	to view another user's Notes.
	
	STATUS
	======
	
	This is by product design.
