## Q98311: Firmware V135 Required for Gateway 2000 Handbook Computers

	Article: Q98311
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 23-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 6.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The following problems have been reported on Gateway 2000 Handbook 80286
	computers running MS-DOS 6.0 and DoubleSpace:
	
	- Data Error Reading Drive X: (Choosing the Retry button may correct this
	  error.)
	
	-and-
	
	- Random system hangs that cannot be corrected by rebooting. (Pressing
	  CTRL+ALT+DEL doesn't work; you must turn the computer off and then on again.
	
	WORKAROUND
	==========
	
	To avoid these problems, upgrade your Handbook firmware to version 135. For more
	information, contact Gateway 2000.
	
	
	Additional query words: 6.00 double space dblspace hardware
