## Q98537: PC DOS: Starting Monitor in Monochrome Mode

	Article: Q98537
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	It is possible to start MONITOR.EXE, which is included with version 3.0 of
	Microsoft Mail for PC Networks, in monochrome mode.
	
	To start Monitor in monochrome, add the -tmono parameter to the Monitor command
	line. The following is an example of a Monitor command line:
	
	  Monitor <mailbox> -ppassword -d<drive> -tmono
