## Q108025: Suffix String Not Available for Microsoft At Work PC Fax

{% raw %}

	Article: Q108025
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft At Work PC Fax, which is included with Windows for Workgroups version
	3.11, does not include a suffix string feature.
	
	MORE INFORMATION
	================
	
	To send a string after connecting with Microsoft At Work PC Fax, use commas for
	pauses, then add the suffix string to the number, as in the following example:
	
	  1-800-111-2222,,,333
	
	This would dial 1-800-111-2222, wait 6 seconds, and then dial the number 333. The
	wait time for a comma can be set in the Fax Setup dialog box in Control Panel.
	The default wait time is 2 seconds per comma.
