## Q108850: Soft Fonts Do Not Appear on Font List with Panasonic

	Article: Q108850
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	Soft fonts (downloadable printer fonts) installed through the Fonts dialog box
	of the Panasonic Laser Printer driver (PANLP121.DRV version 1.21 or PANLP122.DRV
	version 1.22) using the Hewlett-Packard Font Installer (FINSTALL.DLL) do not
	appear in the font list of any Windows-based applications.
	
	RESOLUTION
	==========
	
	This problem was corrected in PANLP123.DRV version 1.23, which is available from
	the Panasonic bulletin board service or from Panasonic Technical Support.

	================

	of Microsoft; we make no warranty, implied or otherwise, regarding these
	products' performance or reliability.
