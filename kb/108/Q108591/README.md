## Q108591: Joystick Stops Working After Situation Reset

	Article: Q108591
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	kbusage
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for MS-DOS, version 5.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The joystick works correctly until the situation is reset or another situation
	or aircraft is chosen.
	
	RESOLUTION
	==========
	
	To solve this problem, do the following:
	
	1. From the Options menu, click Preferences.
	
	2. Click Joystick. The Joystick Preferences dialog box opens.
	
	3. Clear the Load Sensitivities Saved With Situation option.
	
	4. Exit Flight Simulator, then type "fs5" (without the quotation marks) to
	  restart it for these changes to take effect.
	
	MORE INFORMATION
	================
	
	This option also appears in the General Preferences dialog box under Situation
	Options; however, it has no effect.
	
	Additional query words: 5.00 flightsim fltsim dos fs5 crash
