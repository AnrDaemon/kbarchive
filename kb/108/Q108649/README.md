## Q108649: Flight Simulator: Yoke for Flight Simulator

{% raw %}

	Article: Q108649
	Product(s): Microsoft Home Games
	Version(s): MACINTOSH:4.0; MS-DOS:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for the Macintosh, version 4.0 
	- Microsoft Flight Simulator for MS-DOS, version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Colorado Spectrum, an independent third-party vender, manufactures a flight
	stick called The Mouse Yoke, which is documented to be compatible with Microsoft
	Flight Simulator. For more information, please contact Colorado Spectrum at
	(800) 238-5983.
	
	MORE INFORMATION
	================
	
	The third-party products discussed in this article are manufactured by vendors
	independent of Microsoft; we make no warranty, implied or otherwise, regarding
	these products' performance or reliability.
