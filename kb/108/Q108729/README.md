## Q108729: OFF97: Color Shift Occurs When Printing in Windows 98

{% raw %}

	Article: Q108729
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta
	Last Modified: 01-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	- Microsoft Excel 97 for Windows 
	- Microsoft PowerPoint 97 for Windows 
	- the operating system: Microsoft Windows 98 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you print a document from an Office 97 program, the colors contained in the
	document may not print as expected. For example, the color red prints as orange.
	
	CAUSE
	=====
	
	This problem may occur if the following conditions are true:
	
	- You are using a printer driver that supports Image Color Matching (ICM).
	
	  -and-
	
	- You are printing in Microsoft Windows 98.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the products listed at the
	beginning of this article. This problem was corrected in Microsoft Office 97
	Service Release 2.
	
	For additional information about SR-2, please see the following article in the
	Microsoft Knowledge Base:
	
	  Q151261 OFF97: How to Obtain and Install MS Office 97 SR-2
