## Q83956: Write Does Not Double-Space Fonts That Are 22 Points or Larger

	Article: Q83956
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft Windows Write does not allow one-and-one-half or double-spacing of
	fonts that are 22 points or larger.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows versions 3.0 and 3.1. We
	are researching this problem and will post new information here as it becomes
	available.
	
	Additional query words: 3.10 3.11 space linespace
