## Q83584: Madison Avenue Requires DMABufferSize=64 Setting

	Article: Q83584
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you run Madison Avenue by Eclipse Technologies within Microsoft Windows
	version 3.0, you must use the DMABufferSize=64 setting in the [386Enh] section
	of the SYSTEM.INI file.
	
	MORE INFORMATION
	================
	
	Windows may generate an error message indicating that you should increase the
	DMABufferSize from 16 (the default setting) to 32; however, DMABufferSize
	actually needs to be increased to 64. Madison Avenue is a Multimedia MS-DOS
	application that can be run from within Windows.
	
	For more information on Madison Avenue, contact Eclipse Technologies technical
	support.
	
	Madison Avenue is manufactured by Eclipse Technologies, a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
	
	This information applies to Microsoft Windows versions 3.0 and 3.0a.
