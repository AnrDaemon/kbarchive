## Q83329: GP Faults when Printing over Banyan Vines 4.11

{% raw %}

	Article: Q83329
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When using Banyan vines 4.11, with Windows 3.1, you may receive general
	protection (GP) faults when printing to any port other than FILE:.
	
	To correct this problem, obtain Patch 5 from Banyan Vines.
