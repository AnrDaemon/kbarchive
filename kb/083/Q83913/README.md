## Q83913: NEC Pinwriter P6 Printer Driver Increases Top Margin

	Article: Q83913
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you print to an NEC Pinwriter P6 from Microsoft Windows version 3.1, the
	top margin may increase by approximately .75 inch per sheet after the first few
	pages have printed.
	
	WORKAROUND
	==========
	
	To work around this problem, try different settings in Control Panel for the
	User-Defined Page Size until you find a setting that consistently prints the
	correct top margin.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the NEC Pinwriter printer driver
	included with Windows version 3.1. We are researching this problem and will post
	new information here as it becomes available.
	
	MORE INFORMATION
	================
	
	The Pinwriter is manufactured by NEC, a vendor independent of Microsoft; we make
	no warranty, implied or otherwise, regarding this product's performance or
	reliability.
