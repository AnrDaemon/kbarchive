## Q83521: Inconsistent File Manager Tiling Behavior

	Article: Q83521
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	There are two ways to tile windows in File Manager. The first is to choose the
	Tile command from the Window menu. The second is to use the key combination
	SHIFT+F4.
	
	MORE INFORMATION
	================
	
	Choosing Tile from the Window menu tiles the windows horizontally (one above the
	other). Using the SHIFT+F4 key combination tiles the windows vertically, (side
	by side).
