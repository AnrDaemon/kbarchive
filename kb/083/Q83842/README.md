## Q83842: Windows Err Msg: Unable to Play Sound. Sound Blaster in Use...

	Article: Q83842
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you receive the following error message while trying to play sounds from the
	MS-DOS command prompt under Windows version 3.1, you need to get an updated
	device driver from Creative Labs.
	
	  Unable to play sound - Sound Blaster is in use by another application.
	
	MORE INFORMATION
	================
	
	Creative Labs has an updated version of VSBD.386 for use with Windows 3.1 and
	Creative Labs Sound Blaster cards, and is distributing the file with a new set
	of drivers via the Creative Labs Bulletin Board Service (BBS).
	
	For more information, please contact Creative Labs.
	
	The Creative Labs products included here are manufactured by vendors independent
	of Microsoft; we make no warranty, implied or otherwise, regarding these
	products' performance or reliability.
