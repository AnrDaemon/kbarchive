## Q83239: Epson Action Laser II Printer and Windows

{% raw %}

	Article: Q83239
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	Epson does not have a printer driver available for the Epson Action Laser II
	printer for Microsoft Windows 3.0.
	
	The Epson Action Laser II is equipped with three emulation modes of operation.
	
	MORE INFORMATION
	================
	
	The Epson Action Laser II printer has three emulation modes:
	
	- Hewlett-Packard (HP) LaserJet IIP
	
	- Epson FX
	
	- Epson LQ
	
	This printer is shipped with the emulation mode for the HP LaserJet IIP set.
	Epson technical support recommends using the HP LaserJet IIP emulation mode. To
	determine the current mode of operation, examine the front panel of the printer,
	the mode will be displayed here.
	
	Any questions concerning the functioning of this printer under Microsoft Windows
	should be directed to Epson Technical Support.
	
	The Epson product included here is manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
