## Q83925: 16.66 cpi Line Printer Font on HP LaserJet IIIsi

	Article: Q83925
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The 16.66 characters-per-inch (cpi) Line Printer font is available to
	Windows-based applications that use the Hewlett-Packard (HP) LaserJet IIIsi and
	related drivers. Some Windows-based applications may list the size of this font
	as 8 points instead of 8.5 points. Nevertheless, selecting the Line Printer
	font, regardless of the listed size, ensures that the driver uses the 16.66 cpi
	font.
	
	MORE INFORMATION
	================
	
	The 8.5 point font (16.66 cpi Line Printer font) is an HP-printer internal font,
	and is available in this one size only. Note that the Line Printer font is not
	available when using the HP LaserJet IIISi PostScript driver.
