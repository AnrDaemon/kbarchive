## Q83536: Mac Wkst Srv: Installing the MailMaker Software

{% raw %}

	Article: Q83536
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft Mail for the Macintosh version 3.0 users should use the following
	versions of MailMaker software:
	
	  MailSender          version 1.1.4
	  MailViewer          version 1.02
	  MM Form Installer   version 3.0
	
	MORE INFORMATION
	================
	
	Please use the versions as recommended. They are not interchangeable. When
	installing new MailSenders, make sure to remove old MailSenders from the System
	Folder. Since they are named differently, the old MailSender may not be deleted
	when the new MailSender is installed.
