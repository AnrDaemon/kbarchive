## Q83560: Lotus 1-2-3 for Windows 1.0 Does Not Display TrueType Fonts

	Article: Q83560
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	TrueType fonts are not displayed when running Lotus 1-2-3 for Windows
	version 1.0, under Microsoft Windows operating system version 3.1.
	
	According to Lotus technical support, version 1.1 of 1-2-3 for Windows
	corrects the problem.
	
	For additional information, please call Lotus Technical Support.
	
	The Lotus products included here are manufactured by vendors
	independent of Microsoft; we make no warranty, implied or otherwise,
	regarding these products' performance or reliability.
