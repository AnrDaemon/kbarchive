## Q83357: Windows PSCRIPT.DRV Does Not Support Lower Resolutions

	Article: Q83357
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The PSCRIPT.DRV PostScript driver included with the Microsoft Windows operating
	system versions 3.0 and 3.1 does not have an option for changing printer
	resolutions.
	
	MORE INFORMATION
	================
	
	PostScript printers have the option to print different resolutions. These
	resolutions range from 75 to 600 DPI and higher. The PostScript driver provided
	with Windows defaults to the highest DPI that the driver can use to print with
	the selected printer.
	
	There is no option for changing printer resolutions.
