## Q83824: SoundBlaster MCV Sound Card Not Supported by Windows 3.1

{% raw %}

	Article: Q83824
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 02-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The sound card drivers that ship with Windows 3.1 do not support the
	SoundBlaster MCV sound card. The card works with MS-DOS, but not with Windows
	3.1.
	
	MORE INFORMATION
	================
	
	The SoundBlaster MCV sound card was designed for use with the IBM Micro Channel
	architecture. The card functions properly with MS-DOS, but in Windows 3.1 after
	installing any of the Windows 3.1 sound drivers, all Setup options are
	unavailable (dimmed). Additionally, all abilities to play sound are unavailable,
	or the following message appears:
	
	  No recording or playback devices are present
	
	None of the drivers shipped with Windows 3.1 will work properly with the
	SoundBlaster MCV.
	
	Additional query words: 3.10 MCV w_mmwext
