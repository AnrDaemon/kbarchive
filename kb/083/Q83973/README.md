## Q83973: Large Titles and Icons on EVEREX/AGI 286

{% raw %}

	Article: Q83973
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When Microsoft Windows operating system version 3.1 is installed on an
	EVEREX/AGI 286 model 3602A laptop, the icons and icon titles appear to be twice
	the size of the same display under Windows 3.0. This happens with either the CGA
	or EGA drivers.
	
	There is no solution to this problem.
	
	MORE INFORMATION
	================
	
	This laptop has a gas plasma screen with the capacity of handling 640 x 400
	pixel resolution and dimensions of 211mm X 132mm (8.3 inches across by 5.2
	inches tall). Windows 3.1 otherwise runs fine except for the icons and words
	being oversized.
	
	The hardware product included here is manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding the product's
	performance or reliability.
