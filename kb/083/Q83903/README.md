## Q83903: Compaq Adapt Utility Shifts MS-DOS Colors with Windows 3.1

	Article: Q83903
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you use Compaq's ADAPT.COM screen utility with Microsoft Windows version 3.1,
	after you exit Windows, the screen colors are different from when you started
	Windows.
	
	This problem does not occur if you use the version of ADAPT.COM that comes on the
	Compaq User Program disk version 7.21.
	
	MORE INFORMATION
	================
	
	ADAPT.COM is located on the Compaq User Program disk. ADAPT.COM ships with the
	following computers: LTE, LTE/286, LTE 386s/20, Portable III, Portable 386,
	SLT/286, and SLT 386s/20.
