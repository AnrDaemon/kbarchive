## Q83840: Emulations for Seikosha Printers

	Article: Q83840
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	Currently, there are no Seikosha printer drivers for any version of Microsoft
	Windows. This article provides a list of recommended emulations.
