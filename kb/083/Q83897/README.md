## Q83897: Err Msg: Fatal Disk Error (with HP Advanced Link)

	Article: Q83897
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Hewlett-Packard's (HP) Advanced Link (any version) doesn't run in Microsoft
	Windows version 3.1 in 386 enhanced mode. When you attempt to run in 386
	enhanced mode, the following error message is displayed:
	
	  FATAL DISK ERROR
	
	This problem does not occur with Windows 3.0.
	
	STATUS
	======
	
	HP is aware of this problem and is currently researching it.
	
	MORE INFORMATION
	================
	
	The HP Advanced Link connectivity package allows a workstation to communicate
	over a standard communications port with a HP 2392 minicomputer.
	
	The HP product included here is manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
