## Q83231: The 44 TrueType Fonts Offered in Windows 3.1 Font Pack 1

	Article: Q83231
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following are the 44 TrueType fonts offered in the Microsoft TrueType Font
	Pack for Windows:
	
	Font Pack Fonts               Font Types
	---------------               ----------
	
	Arial Narrow                  Regular, Bold, Italic, Bold Italic
	Bookman Antiqua               Regular, Bold, Italic, Bold Italic
	Bookman Old Style             Regular, Bold, Italic, Bold Italic
	Century Gothic                Regular, Bold, Italic, Bold Italic
	Century Schoolbook            Regular, Bold, Italic, Bold Italic
	Lucida Blackletter
	Lucida Bright                 Regular, Demibold, Italic, Demibold
	                             Italic
	Lucida Bright Math Extension
	Lucida Bright Math Italic
	Lucida Bright Math Symbol
	Lucida Calligraphy Italic
	Lucida Fax                    Regular, Demibold, Italic, Demibold
	                             Italic
	Lucida Handwriting
	Lucida Sans                   Regular, Demibold, Italic, Demibold
	                             Italic
	Lucida Sans Typewriter        Regular, Bold, Oblique, Bold Oblique
	Monotype Corsiva
	Monotype Sorts
	
	Additional query words: 3.10 3.11 true type mono
