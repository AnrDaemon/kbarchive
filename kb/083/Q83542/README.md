## Q83542: Mac Fax: Fax Gateway Versions

	Article: Q83542
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:2.0,2.0a,2.0b,3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	- Microsoft Mail for AppleTalk Networks Gateway to Fax v1.1.0:
	
	  For use with Mail for AppleTalk Networks 2.0.
	
	- Microsoft Mail for AppleTalk Networks Gateway to Fax v1.1.2:
	
	  For use with Mail for AppleTalk Networks 2.0. Fixed some problems Fax Gateway
	  had when running in tight memory situations.
	
	- Microsoft Mail for AppleTalk Networks Gateway to Fax v3.0-ms:
	
	  For use with Mail for AppleTalk Networks 3.0. This version did not include
	  the version of the MS Mail GW required to run with System 7.0
	
	- Microsoft Mail for AppleTalk Networks Gateway to Fax v3.00a-ms:
	
	  For use with Mail for AppleTalk Networks 3.0. This version shipped with a
	  System 7.0 compatible version of the MS Mail GW, version 3.00a.
	
	Additional query words: 2.00 2.00a 2.00b 3.00 faxgate
