## Q83587: Error when Installing Fastback for Windows on Windows 3.1

	Article: Q83587
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When running the installation program for Fifth Generation Systems' Fastback for
	Windows, the installation program will hang during the process. Moreover,
	running the FB.EXE file directly (from File Manager), you will receive the
	following error:
	
	  Cannot run on anything except Windows V3.0
	
	MORE INFORMATION
	================
	
	Fifth Generation Systems' Fastback for Windows is designed specifically for
	Windows 3.0 or 3.0a. Apparently, supported Window's version numbers are written
	into Fastback, and thus will not run on Windows 3.1.
	
	
	Additional query words: 3.10 fastback hang error install
