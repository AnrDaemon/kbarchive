## Q83732: Wrong Tray Selection Causes Epson Printer Error in Windows 3.1

{% raw %}

	Article: Q83732
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	In Microsoft Windows version 3.1, if you print to an Epson dot-matrix printer
	with a cut-sheet feeder and the wrong paper feed source is selected, the Epson
	prints one line of text and then takes the printer offline.
	
	RESOLUTION
	==========
	
	To correct this problem, use the following steps:
	
	1. Run Control Panel, and choose the Printers icon.
	
	2. Select the Epson dot-matrix printer, and choose the Setup button.
	
	3. For the cut-sheet feeder, select Upper Tray in the Paper Source box.
