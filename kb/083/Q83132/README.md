	The information in this article applies to:
	
	- Microsoft Windows versions 2.03, 2.1, 2.11, 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Logitech Technical Support confirms that for Windows 2.03 and later, all
	Logitech pointing devices should use the Logitech mouse driver that is included
	in the Windows package, unless otherwise noted in the documentation that comes
	with the mouse. The serial Logitech Trackball, and the bus Logitech Trackball
	can be set up using the Serial Logitech Mouse driver and the Bus Logitech Mouse
	driver, respectively.
	
	NOTE: The Logitech Dexxa Mouse and the Logitech Cordless Mouse should be set up
	using the "Microsoft, or IBM PS/2 mouse" option.
	
	MORE INFORMATION
	================
	
	Logitech has a number of different pointing devices available for PCs; among
	them are the Trakman, which is a trackball mouse, and the Series 9 serial mouse.
