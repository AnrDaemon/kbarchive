## Q83879: Screen Saver Disabled When MS-DOS Application Is Active

	Article: Q83879
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The screen savers in the Microsoft Windows operating system version 3.1 do not
	run if the active application is an MS-DOS-based application, regardless of
	whether the MS-DOS-based application is running full-screen or in a window.
	
	Additional query words: 3.10 screensaver 3.11 DOS
