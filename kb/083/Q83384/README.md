## Q83384: Registration Cards Not Included with Windows 3.1 Upgrade

	Article: Q83384
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Customers that purchase the Microsoft Windows operating system version 3.1
	upgrade edition directly from Microsoft will not receive a registration card
	because they are already registered users of Windows.
	
	MORE INFORMATION
	================
	
	The version of Windows 3.1 available in retail stores includes a registration
	card. Customers unsure of their registration status should call Microsoft Sales
	Information Center (MSIC) at (800) 426-9400.
