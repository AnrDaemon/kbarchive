## Q166052: WV: Menus in Word Viewer Do Not Respond on First Click

{% raw %}

	Article: Q166052
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbui kbdta kbviewer word97
	Last Modified: 19-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word Viewer 97-2000 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Microsoft Word Viewer 97-2000, menus may not respond on the first click.
	
	CAUSE
	=====
	
	This problem occurs if Word Viewer is busy processing another command (for
	example, a command that changes a view setting).
	
	RESOLUTION
	==========
	
	The menu responds if you click it a second time.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
	
	REFERENCES
	==========
	
	For more information about Word Viewer 97-2000, please see the following
	Knowledge Base article:
