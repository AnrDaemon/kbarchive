## Q166553: XCON: Encapsulated Message Not Delivered Over X.400 Connector

{% raw %}

	Article: Q166553
	Product(s): Microsoft Exchange
	Version(s): 5.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 08-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	A message that is encapsulated by the Internet Mail Service (IMS) and destined
	for an X.400 address may result in a non-delivery report (NDR) from the X.400
	Connector. The following error message appears:
	
	  A syntax error was detected in the content of the message.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server
	version 5.0. This problem was corrected in the latest Microsoft Exchange Server
	5.0 U.S. Service Pack. For information on obtaining the service pack, query on
	the following word in the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
