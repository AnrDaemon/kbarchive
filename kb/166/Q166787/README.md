## Q166787: WD97: ODMA File Open Not Supported in Mail Merge

	Article: Q166787
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kb3rdparty
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you attempt to create a data source from the Mail Merge Helper, the Word
	Open dialog box is displayed instead of the Document Management System (DMS)
	custom Open dialog box.
	
	WORKAROUND
	==========
	
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
