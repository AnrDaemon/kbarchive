## Q166985: Earlier MSN Signup Window with Install MSN 2.0

	Article: Q166985
	Product(s): The Microsoft Network
	Version(s): WINDOWS:2.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbsetup
	Last Modified: 13-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you install MSN, The Microsoft Network, version 2.0, the MSN Sign-up
	window and the MSN icon on your desktop may be from an earlier version of MSN.
	
	CAUSE
	=====
	
	The MSN 2.0 installation did not overwrite the Guide.exe file.
	
	RESOLUTION
	==========
	
	To resolve this issue, reinstall MSN 2.0.
