## Q166434: Abnormal Size of Character Map Under Taiwan Locale.

	Article: Q166434
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kbnetwork
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation, Pan Chinese Edition, version 4 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you use 640 x 480 screen resolution in Pan Chinese Windows NT Workstation
	version 4.0 under Chinese (Taiwan) locale, the Character Map window will not fit
	in the screen. The problem persists if you change the resolution to 800 x 600 or
	1024 x 768.
	
	If you change locale to Chinese (PRC) or (Hong Kong), the Character Map window is
	normal size.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
