## Q166304: Flight Simulator 95: No Brakes Available on the Sopwith Camel

	Article: Q166304
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for Windows 95, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	There are no brakes on the Sopwith Camel in Flight Simulator for Windows 95.
	
	MORE INFORMATION
	================
	
	In reality, the Sopwith Camel had no brakes. The brake function was removed from
	Flight Simulator to improve realism.
	
	Additional query words: 1.00 6.00 fs6 fs95 sopwith camel brakes stopping
