## Q166208: Kids Plus!: How To Remove Plus! For Kids From Your Computer

	Article: Q166208
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Plus! for Kids, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following article provides steps to completely remove Microsoft Plus! For
	Kids from your computer. These steps are required because you cannot uninstall
	Protect It! from Add/Remove Programs for security reasons.
	
	MORE INFORMATION
	================
	
	To remove Microsoft Plus! For Kids, do the following:
	
	1. Select a new Windows Desktop Theme. Select a theme that is NOT a Microsoft
	  Plus! For Kids Desktop Theme. The best choice is Windows default.
	  a. Click the Start button, point to Settings, and then click Control Panel.
	
	  b. Double-click Desktop Themes.
	
	  c. From the Theme drop-down list, select Windows Default, or any theme that
	     is not included with Microsoft Plus! For Kids.
	
	2. Remove the program.
	  a. Click the Start button, point to Settings, and then click Control Panel.
	
	  b. Double-click Add/Remove Programs.
	
	  c. From the Install/Uninstall tab, select Microsoft Plus! For Kids, and then
	     click Add/Remove.
	
	  d. Follow the on screen instructions to remove the program.
	
	3. Remove Protect It!
	  a. Insert the Microsoft Plus! For Kids compact disc into your CD-ROM drive.
	
	  b. Click the Start button, and then click Run.
	
	  c. Type the following:
	
	  "rundll32.exe d:\protect!\piclean.dll,clean" (without the quotation marks)
	
	     (where D is the drive letter of your CD-ROM drive).
	
	  d. Follow the on screen instructions to remove Protect It!.
	
	4. Remove User Profiles (optional).
	
	  To bypass the User Profiles at log on, please see the following article(s) in
	  the Microsoft Knowledge Base:
	
	  Q164368 Kids Plus!: How to Disable Login Screen When Starting Windows
