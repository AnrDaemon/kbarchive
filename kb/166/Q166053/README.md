## Q166053: WV: Document Menu Unavailable in Word Viewer

	Article: Q166053
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbui kbviewer word97
	Last Modified: 19-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word Viewer 97-2000 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Microsoft Word Viewer 97-2000, if you maximize the document window, the
	document menu is unavailable and the document icon (the icon in the upper- right
	corner of the windows) changes from a Word icon to a Windows icon.
	
	RESOLUTION
	==========
	
	The options that should be available on the document menu when the document is
	maximized are Restore, Minimize, and Close.
	
	To work around this problem, use the appropriate method below.
	
	To Restore the Document
	-----------------------
	
	Click the Restore button.
	
	The Restore button is the center button in the upper-right corner of the document
	window. It is located above the vertical scrollbar and below the Maximize
	button.
	
	To Minimize the Document
	------------------------
	
	Click the Minimize button.
	
	The Minimize button is the left-most button in the upper-right corner of the
	document window.
	
	To Close the Document
	---------------------
	
	To close the document, do any of the following:
	
	- Click the Close button.
	
	  The Close button is the right-most button in the upper-right corner of the
	  document window. The Close button has an "X" on it.
	
	  -or-
	
	- On the File menu, click Close.
	
	  -or-
	
	- Press CTRL+W on the keyboard.
