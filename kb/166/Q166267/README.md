## Q166267: Office Shortcut Bar Fonts Appear as Non-Cyrillic on Russian NT

	Article: Q166267
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbuikbbuglist kbfixlist
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	After you install Microsoft Office 97 on the Russian version of Windows NT 4.0,
	the Microsoft Office Shortcut Bar fonts appear as non-Cyrillic. This may be
	reproduced if you use the right mouse button to click the shortcut and then
	click Customize. The text Show These Folders As Toolbars may be displayed using
	a non-Cyrillic font.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 4.0. This
	problem was corrected in the latest Microsoft Windows NT 4.0 U.S. Service Pack.
	For information on obtaining the service pack, query on the following word in
	the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
