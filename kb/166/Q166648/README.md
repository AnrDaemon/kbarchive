## Q166648: Word Cannot Retrieve the Business Address from Outlook

{% raw %}

	Article: Q166648
	Product(s): Word 97 for Windows
	Version(s): 
	Operating System(s): 
	Keyword(s): kbinterop kbusage kbdta winword word97
	Last Modified: 18-APR-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	- Microsoft Outlook 97 
	-------------------------------------------------------------------------------
	
	For a Microsoft Word 2002 version of this article, see Q290980.
	
	SYMPTOMS
	========
	
	In Microsoft Outlook, you can set the mailing address to a Business, Home, or
	Other address. If you set the Microsoft Outlook Home address with the This is
	the mailing address setting, you will be unable to use the Insert Address button
	to insert the Outlook Business address in Microsoft Word 97.
	
	WORKAROUND
	==========
	
	In Outlook, use the following steps to change the designation of the This is the
	mailing address to the Business address:
	
	1. Open Microsoft Outlook and click the Contacts button.
	
	2. Double-click a name and click the General tab.
	
	3. Click the drop-down menu for the type of address.
	
	  You should see the three choices of Business, Home, and Other. Click Business.
	
	4. Click the This is the mailing address check box to select it.
	
	5. Click Save And Close.
	
	6. Minimize or close Microsoft Outlook.
	
	7. In Microsoft Word 97, click the Insert Address button.
	
	8. In the Select Name dialog box, click the down arrow next to Show Names from
	  the, and click Contacts (under the Outlook Address Book item).
	
	9. Click the name that you want to use, click the Mailing button and click OK.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
	
	MORE INFORMATION
	================
	
	The Insert Address button can be accessed by doing any of the following:
	
	- On the Tools menu, click Envelopes And Labels. -or-
	
	- From the Letter Wizard, use the Recipient Information tab. -or-
	
	- Place the Insert Address command or button on a toolbar.
