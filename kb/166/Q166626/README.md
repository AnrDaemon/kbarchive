## Q166626: XADM: Thai Character Set Enabled in Exchange 5.0 SP1

{% raw %}

	Article: Q166626
	Product(s): Microsoft Exchange
	Version(s): winnt:5.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 08-JAN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	
	Microsoft Exchange Server 5.0 Service Pack 1 will support the Thai character set.
	
	STATUS
	======
	
	Microsoft feature was included in the latest Microsoft Exchange Server 5.0 U.S.
	Service Pack. For information on obtaining the service pack, query on the
	following word in the Microsoft Knowledge Base:
	
	  " SERVPACK " (without the quotation marks)
	
	MORE INFORMATION
	================
	
	Microsoft Exchange Server currently employs two font size tables: one for
	single-byte character sets (SBCS) and one for double-byte character sets. (DBCS)
	A third table has been added for Thai character support.
