## Q166576: XFOR: GPF in Store.exe When Doing Fixed Width Font Conversion

	Article: Q166576
	Product(s): Microsoft Exchange
	Version(s): 5.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 13-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you are running Microsoft Exchange Server with the Internet Mail Service
	(IMS) and you have the Use-Fixed Width Font option selected, you may experience
	a general protection fault (GPF) in the Microsoft Exchange information store
	service (Store.exe).
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server
	version 5.0. This problem was corrected in the latest Microsoft Exchange Server
	5.0 U.S. Service Pack. For information on obtaining the service pack, query on
	the following word in the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
