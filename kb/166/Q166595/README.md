## Q166595: XFOR: Japanese Web Client Messages Corrupted When Posted

	Article: Q166595
	Product(s): Microsoft Exchange
	Version(s): 5.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 15-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you run the Japanese version of Microsoft Exchange Server version 5.0
	Active Server Commponents against a mailbox on a computer running the US version
	of Microsoft Exchange Server, posted messages may be corrupt and unreadable.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server
	version 5.0. This problem was corrected in the latest Microsoft Exchange Server
	5.0 U.S. Service Pack. For information on obtaining the service pack, query on
	the following word in the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
	
	
	Additional query words: SP1 Kluane DBCS client IMAIL XWEB
