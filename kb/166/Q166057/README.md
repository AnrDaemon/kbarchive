## Q166057: WV: &quot;What's This?&quot; Doesn't Present Word Viewer Help

	Article: Q166057
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbusage kbdta kbviewer word97
	Last Modified: 19-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word Viewer 97-2000 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you are in Microsoft Word Viewer 97-2000 and you right-click a button or an
	item in a dialog box, a "What's This?" Help option appears. However, clicking
	this option does not bring up Help.
	
	CAUSE
	=====
	
	Microsoft Word Viewer 97-2000 does not have context-sensitive Help.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.

	================
	
	For more information about Word Viewer 97-2000, please see the following
	Knowledge Base article:
