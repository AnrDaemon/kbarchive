## Q149435: AST Bravo LC 4/100 Requires BIOS Version 1.02

	Article: Q149435
	Product(s): Microsoft Windows NT
	Version(s): 3.50 3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation versions 3.5, 3.51 
	- Microsoft Windows NT Server versions 3.5, 3.51 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you attempt to set up Microsoft Windows NT on an AST Bravo LC 4/100 computer,
	the system may operate erratically. Failures may occur, and the following stop
	message may appear:
	
	  STOP: 0x0000007b Inaccessible Boot Device
	
	CAUSE
	=====
	
	The AST Bravo LC 4/100 requires AST BIOS 1.02 in order to run Microsoft Windows
	NT. The AST Bravo was certified for the Microsoft Windows NT Hardware
	Compatibility List (HCL) with this specific BIOS version. Use of other BIOS
	versions may produce unpredictable operation, including the error above, which
	is due to unrecognized hard disk configuration
