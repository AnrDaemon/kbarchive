## Q149795: Explorapedia People: Installs Visual FoxPro

{% raw %}

	Article: Q149795
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Explorapedia series: World of People for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you insert the Explorapedia People compact disc in your CD-ROM drive and
	run Setup, Microsoft Visual FoxPro may be installed instead of Explorapedia.
	
	RESOLUTION
	==========
	
	Contact Microsoft Product Support Services at (425) 635-7140 and ask for a
	replacement compact disc.
	
	MORE INFORMATION
	================
	
	In the manufacturing process, there were some compact discs produced with the
	Explorapedia silkscreen on the disc, but the disc contains Microsoft Visual
	FoxPro.
