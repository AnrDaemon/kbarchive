## Q149103: MSN Interactive Teen: How to Send Suggestions

	Article: Q149103
	Product(s): Microsoft Home Kids Products
	Version(s): 1.0; WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MSN Interactive Teen, version 1.0 
	- the operating system: Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article discusses ways for you to give us feedback and suggestions on our
	MSN Interactive Teen site.
	
	MORE INFORMATION
	================
	
	If you would like to request an MSN Interactive Teen enhancement, or if you
	would like to submit feature suggestions for future releases of MSN Interactive
	Teen, or if you have read product content you believe is incorrect, we would
	like to hear from you.
	
	The Product Team can be reached by sending email to the following address:
	
	  iteen@microsoft.com
	
	We look forward to hearing from you.
