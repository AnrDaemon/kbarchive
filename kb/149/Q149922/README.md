## Q149922: Error Message: Unable to Parse Setup Parameters

	Article: Q149922
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.2,1.3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.2, 1.3 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to open a service on MSN, The Microsoft Network, you may
	receive the following error message:
	
	  Unable to Parse Setup Parameters.
	
	CAUSE
	=====
	
	You attempted to open a service that was created using features of MSN version
	1.3.
	
	RESOLUTION
	==========
	
	Install MSN version 1.3, and then open the service.
	
	Additional query words: msn
