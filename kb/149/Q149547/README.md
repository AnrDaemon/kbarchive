## Q149547: MSN Phonebook Updated Automatically

	Article: Q149547
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.3,2.0,2.5,2.51,2.52,2.6
	Operating System(s): 
	Keyword(s): kbenv kbmsn
	Last Modified: 08-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.3, 2.0, 2.5, 2.51, 2.52, 2.6 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you are using MSN, The Microsoft Network, version 1.3 or later, phonebook
	updates occur automatically.
	
	MORE INFORMATION
	================
	
	MSN automatically checks the version of the access number phonebook that is on
	your computer. If you do not have the most recent version of the phonebook, MSN
	downloads the new version. This process is invisible to you.
	
	The phonebook update that occurred in earlier versions of MSN when you clicked
	Access Numbers in the Connection Settings dialog box while you were connected to
	MSN has been removed.
	
	
	Additional query words: msn phone book
