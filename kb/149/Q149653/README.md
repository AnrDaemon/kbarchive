## Q149653: TN3270 Admin Command Line Options

	Article: Q149653
	Product(s): Microsoft SNA Server
	Version(s): WINDOWS:2.11,2.11 SP1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft SNA Server, versions 2.11, 2.11 SP1, on platform(s):
	   - the operating system: Microsoft Windows NT 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	TN3270 Server Admin supports the following two undocumented command line
	switches:
	
	     /c
	
	     /no3d.
	
	MORE INFORMATION
	================
	
	/c - This will invoke the Config option when TN 3270 Admin is started.
	
	/no3d - This causes the TN 3270 Admin to not use visual 3D effects
	
	       supported by the Windows NT Ctl3d32.dll module.
	
	Additional query words: prodsna 2.11 sp1 parameters
