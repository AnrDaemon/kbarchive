## Q149914: Cannot Use Special Graphics Character Set in VT100 Emulation

	Article: Q149914
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you change the character set of the VT100 emulation in HyperTerminal to
	Special Graphics, the change is not saved. Instead, the character set changes to
	United Kingdom.
	
	You can choose the ASCII and United Kingdom character sets without problems.
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
