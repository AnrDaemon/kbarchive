## Q149313: Goto Date Box Changes When a Number Is Entered

	Article: Q149313
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use Microsoft Schedule+, version 7.0, a hot key CTRL+G allows your to
	quickly go to a specific date. When using this hot key, a Go To dialog appears
	with the day of the week highlighted by default. Sun, Mon, Tue, etc... Even
	though the numeric date is not highlighted by default, a number can still be
	entered in and the date will change to the next occurrence with in the same
	month.
	
	CAUSE
	=====
	
	This is by design.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
