## Q149544: MPR DLL Files Not Installed From Windows NT 3.51 SP 3

	Article: Q149544
	Product(s): Microsoft Windows NT
	Version(s): 3.50 3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.51 
	- Microsoft Windows NT Server version 3.51 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you install MPR from the Windows NT 3.51 Service Pack 3 compact disc, new
	DLL files are not installed for MPR, such as ROUTETAB.DLL.
	
	RESOLUTION
	==========
	
	To correct this problem, upgrade to Windows NT 3.51 Service Pack 4. For more
	information, read RIPROUTE.WRI in the MPR directory on the Service Pack.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 3.51 Service
	Pack 3. This problem was corrected in the latest Windows NT 3.51 U.S. Service
	Pack. For information on obtaining the Service Pack, query on the following word
	in the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
	
	Additional query words: prodnt sp3
