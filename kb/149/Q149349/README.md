## Q149349: Using UMAX Scanners with Windows NT

{% raw %}

	Article: Q149349
	Product(s): Microsoft Windows NT
	Version(s): 3.5,3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 20-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation versions 3.5, 3.51 
	- Microsoft Windows NT Server versions 3.5, 3.51 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Although UMAX scanners are not supported directly by Microsoft, you can use UMAX
	scanner models UC630, UC840, UC1200SE, UC1260, PowerLook, Vista- S6, Vista-S8,
	Vista-T630, Mirage D-16L, PowerLook II, and Gemini D-16 with Windows NT.
	
	MORE INFORMATION
	================
	
	UMAX offers Windows NT drivers that support the adapters that are bundled with
	the scanner as well as other ASPI-compliant SCSI adapters. These drivers are not
	included with UMAX scanners but are available from UMAX directly.
	
	Currently, the driver needed to use a UMAX scanner on Windows NT is MagicScan
	version 2.2 for Windows NT. Version 2.3 is now available. The scanners supported
	are the UC630, UC840, UC1200SE, UC1260, PowerLook, Vista- S6, Vista-S8,
	Vista-T630, Mirage D-16L, PowerLook II, and Gemini D-16. The driver also
	supports the UDS-IS10, UDS-IS11, and ASPI-compatible SCSI adapters.
	
	For additional information, contact UMAX Technical Support at (510) 651- 8883, or
	visit the following UMAX Web site:
	
	  http://www.umax.com.
