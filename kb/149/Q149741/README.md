## Q149741: Using Uppercase Letters in Find Tab in Help Does Not Work

	Article: Q149741
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95; winnt:4.0
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Typing uppercase letters in the Find tab in Windows 95 Help causes only words
	that are capitalized exactly as you type them to be found.
	
	RESOLUTION
	==========
	
	Use lowercase letters when you type text in the Find tab.
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
	
	MORE INFORMATION
	================
	
	When you use lowercase letters in the Find tab, all matches are found regardless
	of case.
	
	Note that the Index tab in Help is not case sensitive; only the Find tab is case
	sensitive.
