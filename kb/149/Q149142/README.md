## Q149142: Dummy.txt File Left After Removing Windows 95 Service Pack 1

	Article: Q149142
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you uninstall Microsoft Windows 95 Service Pack 1, a Dummy.txt file may be
	left in the Temp folder.
	
	CAUSE
	=====
	
	This is a placeholder file from the uninstall process.
	
	RESOLUTION
	==========
	
	You can safely delete this file after the uninstall process is finished.
	
	
	Additional query words: remove
