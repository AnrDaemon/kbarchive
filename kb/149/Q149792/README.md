	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article provides the steps to use to find the solution for the GhostWriter
	mystery "Case of the Blue Makva."
	
	RESOLUTION
	==========
	
	1. Start Creative Writer.
	
	2. Open the tenth case of the GhostWriter set, "The Final Makva."
	
	3. Decode the file. To decode the file, click the Transformer tool, then click
	  the Secret Agent tool. In the Password box, type "Detective McZee to the
	  Rescue" (without the quotation marks) and hold down the CTRL+SHIFT keys when
	  you click OK to decode.
	
	4. Find the final clue section, highlight it, and use the Letter Tools to remove
	  the shimmer and turn the color of the letters from white to black.
	
	5. You have solved the case!
