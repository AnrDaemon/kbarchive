## Q34231: Keys Used to Replace SCROLL LOCK Key on Tandy 1000 Keyboards

	Article: Q34231
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	The following keys replace the SCROLL LOCK key on Tandy 1000 keyboards when
	using Microsoft Flight Simulator:
	
	Versions 2.12 and 2.13:
	
	  HOLD key
	
	Versions 3.00 and 4.00:
	
	  BREAK key
