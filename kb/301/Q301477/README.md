## Q301477: Broadband-Supported USB-Ethernet Adaptors

{% raw %}

	Article: Q301477
	Product(s): The Microsoft Network
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This is a list of USB to Ethernet adaptors which are supported by MSN Companion.
	Devices not on this list are not supported.
	
	MORE INFORMATION
	================
	
	ADMtek based dongles
	
	- Linksys USB10TX v3
	
	- Linksys USB100TX 100T
	
	- D-Link 10 USB Ethernet
	
	HPNA 2.0
	
	- Netgear PHONELINE 10X 10MB USB HOME NETWORK ADAPTER
	
	- Linksys USB200HA
	
	- D-Link DNH-120
