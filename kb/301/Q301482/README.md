## Q301482: Connection Error: e12

	Article: Q301482
	Product(s): The Microsoft Network
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You attempt unsuccessfully to log on to the Internet using the MSN Companion.
	During the attempted connection, the Helper advises you to "try again later" or
	"contact technical support". If you select "contact technical support", a page
	appears that advises you of error "e12".
	
	RESOLUTION
	==========
	
	To resolve the issue
	
	- Unplug the unit and plug it back in. This should resolve the problem.
	
	If this does not resolve the issue, contact Technical Support.
