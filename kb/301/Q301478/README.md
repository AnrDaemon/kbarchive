## Q301478: Broadband: OOBE with V2.0

{% raw %}

	Article: Q301478
	Product(s): The Microsoft Network
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You cannot complete Setup or "Out of Box Experience" (OOBE) with an MSN
	Companion Verson 2.0 device that is connected via broadband and uses a proxy
	server.
	
	CAUSE
	=====
	
	When you purchase a device with Version 2.0 already loaded on it and attempt to
	complete OOBE via a Broadband connection, the device will fail to
	connect--unless Auto-proxy is set up on the network.
	
	RESOLUTION
	==========
	
	Do one of the following:
	
	- Configure an Auto-proxy
	
	  Or
	
	- Complete OOBE via a dial-up connection.
