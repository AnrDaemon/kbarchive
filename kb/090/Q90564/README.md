## Q90564: MS-DOS Err Msg: DoubleSpace Cannot Defragment Because...

	Article: Q90564
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0,6.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.0, 6.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you are running a screen-saver program when you attempt to install
	DoubleSpace from MS-DOS 6 or 6.2 Upgrade, you may receive the following error
	message:
	
	  DoubleSpace cannot defragment C because of an unknown error
	
	WORKAROUND
	==========
	
	To work around this problem, remove the screen-saver from your CONFIG.SYS or
	AUTOEXEC.BAT file, restart your computer and install DoubleSpace. After
	DoubleSpace is successfully installed, you can add the screen-saver to your
	CONFIG.SYS or AUTOEXEC.BAT file.
	
	Additional query words: 6.00 screensaver blank black blanker tsr dblspace 6.20
