## Q90283: NDW 2.0 Network Disconnect Brings up WFWG Network Disconnect

{% raw %}

	Article: Q90283
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): 3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The network disconnect dialog box in Norton Desktop for Windows (NDW) version
	2.0 displays the Windows for Workgroups connect dialog box instead of the Norton
	Desktop for Windows network connections dialog box.
	
	STATUS
	======
	
	Symantec, which distributes Norton Desktop for Windows, is aware of this
	problem. Please contact Symantec for additional information.
