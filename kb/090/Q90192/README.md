## Q90192: DataProducts LZR 650 Printer Problem with Windows

{% raw %}

	Article: Q90192
	Product(s): Microsoft Windows Printing Issues
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	According to DataProducts, the DataProducts LZR 650 printers have printing
	problems when used with Microsoft Windows 3.1. The problem is in the circuit
	card, which can be upgraded by contacting DataProducts.
	
	This problem does not occur with Windows 3.0.
	
	MORE INFORMATION
	================
	
	Although the DataProducts LZR 650 can emulate the Hewlett-Packard (HP) LaserJet
	Series II, or the Epson FX-80, or the IBM Graphics Printers, these emulations
	are not workarounds for the printing problem. The only workaround is to upgrade
	the circuit card.
	
	The DataProducts LZR 650 is manufactured by a vendor independent of Microsoft; we
	make no warranty, implied or otherwise, regarding this product's performance or
	reliability.
