## Q90231: Central Point Backup Confidence Test Fails in WFWG

{% raw %}

	Article: Q90231
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 31-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you run the disk confidence check in the MS-DOS-based version of Central
	Point's Backup program under Windows for Workgroups, the test may fail.
	
	WORKAROUND
	==========
	
	Formatting the disk prior to the test may result in a successful confidence
	check. Using the Windows-based version may also work well.
	
	The Central Point product included here is manufactured by a vendor independent
	of Microsoft; we make no warranty, implied or otherwise, regarding this
	product's performance or reliability.
