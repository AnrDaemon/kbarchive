## Q90213: AT&amp;T Mouse Problems with Windows for Workgroups

	Article: Q90213
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you are using a three-button AT&T mouse with Windows for Workgroups and
	you have the Windows Logitech mouse driver installed, your NUM LOCK, CAPS LOCK,
	and SCROLL LOCK indicator lights may blink.
	
	WORKAROUND
	==========
	
	To correct this problem, run Windows for Workgroups Setup and change the mouse
	driver to "Microsoft or IBM PS/2."
