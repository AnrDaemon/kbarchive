## Q90526: Cannot Set Default Appointment Time in Schedule+

{% raw %}

	Article: Q90526
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:1.0,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	- Microsoft Schedule+ for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you create a new appointment in Windows for Workgroups Schedule+ or
	Schedule+ 1.0, the Start Time and End Time are determined by the currently
	selected time slot.
	
	There is no way to specify a default appointment time in Schedule+. This is by
	design.
