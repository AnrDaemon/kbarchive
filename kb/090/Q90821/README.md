## Q90821: Write: How to Enter a Soft Return

	Article: Q90821
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:2.x,3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 2.03, 2.1, 2.11, 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	Windows documentation does not explain how to enter a soft carriage return
	(one that starts a new line but does not insert a paragraph mark) into your
	Write document. If you press ENTER, Write inserts a paragraph mark. To
	insert a soft return, press CTRL+K.
