## Q90269: NetWare Printing in Windows for Workgroups

	Article: Q90269
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	In general, Microsoft Windows for Workgroups Print Manager extends the
	printing capabilities from those in Microsoft Windows version 3.1. In
	the case of NetWare printers, the capabilities are much the same as in
	Windows 3.1. Only the jobs submitted by the user are shown, and jobs
	may be paused, resumed, or deleted. These actions can be done from the
	document menu or the appropriate buttons on the toolbar.

	independent of Microsoft; we make no warranty, implied or otherwise,
	regarding this product's performance or reliability.
