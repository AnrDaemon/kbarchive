## Q90757: PC WFW: Admin Not Included in Directory Synchronization

	Article: Q90757
	Product(s): Microsoft Mail For PC Networks
	Version(s): 3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail and Schedule+ Extensions for Windows for Workgroups, version 3.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you use version 3.0 of Extensions for Windows for Workgroups to upgrade a
	Windows version 3.1 for Workgroups postoffice to a version 3.0 Microsoft Mail
	for PC Networks postoffice, the users with administrative privileges are not
	included in directory synchronization. However, the users with user privileges
	are included in directory synchronization.
	
	RESOLUTION
	==========
	
	To include users with administrative privileges in directory synchronization,
	the administrator must use the Mail Administrator program (ADMIN.EXE) to modify
	those users with administrative privileges to include them in directory
	synchronization.
	
	Additional query words: 3.00 dir-sync dirsync dirsynch
