## Q90479: PC DOS: Err Msg: No Users Defined in Personal Address List

	Article: Q90479
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, version 3.0, on platform(s):
	   - the operating system: MS-DOS 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to create personal groups using the version 3.0 MS-DOS client,
	you receive the following error message if there are no users defined in the
	Personal Address List:
	
	  Notice 52: No users defined in Personal Address List
	
	RESOLUTION
	==========
	
	Users to be included in the Personal Group must first be added to the Personal
	Address List before any groups can be created. To add users to your Personal
	Address List, select Address, Browse, and then make selections from the desired
	address lists.
