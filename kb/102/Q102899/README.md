## Q102899: Only 30 Characters of Local Group Name Shown in Properties

	Article: Q102899
	Product(s): Microsoft Windows NT
	Version(s): 3.1
	Operating System(s): 
	Keyword(s): kbnetwork
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Advanced Server, version 3.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you add a local group with a long name (30 or more characters), its name is
	not shown correctly in the Properties dialog box. Anything beyond the thirtieth
	character is not displayed.
	
	CAUSE
	=====
	
	This is caused by a problem in the Properties dialog box.
	
	RESOLUTION
	==========
	
	Create local group names with 30 or less characters.
