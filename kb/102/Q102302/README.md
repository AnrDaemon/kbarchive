## Q102302: Trap 0002 Exceptions Indicate Physical RAM Damage

	Article: Q102302
	Product(s): Microsoft LAN Manager
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	Trap 0002 exceptions like the one reported in this message:
	
	  The system has detected an interrupt that cannot be masked
	  Trap 0002
	
	  Internal processing error at location 0228:3785
	
	usually indicate physical RAM damage. Check your RAM to ascertain that it is
	okay.
	
	Additional query words: 2.00 2.0 2.10 2.1 2.10a 2.1a 2.20 2.2
