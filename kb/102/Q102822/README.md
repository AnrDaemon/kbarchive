## Q102822: &quot;Fatal Error on Drive 1&quot; Using Exabyte 4200 with IBM

	Article: Q102822
	Product(s): Microsoft Windows NT
	Version(s): WinNT:3.1,3.5,3.51,4.0
	Operating System(s): 
	Keyword(s): kbhw kbHardware
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 3.1 
	- Microsoft Windows NT Workstation version 3.1 
	- Microsoft Windows NT Advanced Server, version 3.1 
	- Microsoft Windows NT Workstation versions 3.5, 3.51, 4.0 
	- Microsoft Windows NT Server versions 3.5, 3.51, 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you use the Exabyte 4200 tape drive with the IBM SCSI adapter, you receive
	the message "Fatal error on drive 1" in NTBACKUP.
	
	CAUSE
	=====
	
	The Exabyte 4200 is not compatible with the IBM SCSI adapter.
	
	WORKAROUND
	==========
	
	Replace one of the devices. Use a hardware compatibility list to see what model
	to replace it with.
