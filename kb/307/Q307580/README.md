## Q307580: Train Simulator: Err Msg: Failed to Load Activity Manager

	Article: Q307580
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kberrmsg kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Train Simulator, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to start Microsoft Train Simulator, you may receive an error
	message similar to one of the following:
	
	  Failed to load Activity Manager.
	
	  -or-
	
	  Failed to initialize Activity Manager.
	
	CAUSE
	=====
	
	This behavior occurs if the driver for your Logitech mouse is incompatible with
	Train Simulator.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest driver for your mouse
	from the following Logitech Web site:
	
	  http://www.logitech.com
