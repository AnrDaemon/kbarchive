## Q272489: Plug-ins on MSN Companion

	Article: Q272489
	Product(s): The Microsoft Network
	Version(s): 1.0,2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, versions 1.0, 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Mariner software supports Internet Explorer-supported plug-ins, such as
	Windows Media Player or Macromedia Flash.
	
	It does not support Netscape plug-ins, such as RealPlayer or Shockwave.
