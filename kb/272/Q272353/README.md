## Q272353: ELL: Sound Problems or Script Error Message When You Run Encarta

	Article: Q272353
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 
	Operating System(s): 
	Keyword(s): kbsound kbimu
	Last Modified: 06-AUG-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Language Learning French 
	- Microsoft Encarta Language Learning French Deluxe 
	- Microsoft Encarta Language Learning Spanish 
	- Microsoft Encarta Language Learning Spanish Deluxe 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use any of the Microsoft Encarta Language Learning programs listed at
	the beginning of this article, you may experience the following symptoms:
	
	- When you start the program, the introductory music may not be played.
	
	- When you play a video clip, the sound track may not be played.
	
	- When you use the Virtual Challenge feature, the screen may be blank. If you
	  click Reset, you may receive a script error message.
	
	CAUSE
	=====
	
	These behaviors can occur if the Microsoft ActiveX components in Microsoft
	Internet Explorer are missing or damaged.
	
	RESOLUTION
	==========
	
	To resolve this issue, uninstall and then reinstall Internet Explorer.
	
	Uninstall Internet Explorer 5
	-----------------------------
	
	To uninstall internet Explorer 5:
	
	1. Click Start, point to Settings, and then click Control Panel.
	
	2. Double-click Add/Remove Programs.
	
	3. In the list of programs, click "Microsoft Internet Explorer 5 and Internet
	  Tools", and then click Add/Remove.
	
	4. Click Restore Previous Windows Configuration, and then click OK.
	
	5. Restart your computer.
	
	Install Internet Explorer 5 from the Encarta Language Learning CD-ROM:
	
	To install Internet Explorer 5:
	
	1. Insert the Encarta Language Learning CD-ROM into the CD-ROM drive.
	
	2. Click Start, and then click Run.
	
	3. In the Open box, type the following line
	
	  <drive>:\IE5\ie5setup.exe
	
	  where <drive> is the drive letter of the CD-ROM drive.
	
	NOTE: If you are prompted, click "Complete install" to install all of the
	components in Internet Explorer.
	
	MORE INFORMATION
	================
	
	To download and install the latest version of Internet Explorer, visit the
	following Microsoft Web site:
	
	  http://windowsupdate.microsoft.com
