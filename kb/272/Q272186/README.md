## Q272186: MapPoint 2001 Err Msg: This Application Is Unable to Continue...

	Article: Q272186
	Product(s): Microsoft Automap
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbenv kberrmsg kbimu
	Last Modified: 11-JUN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MapPoint 2002 
	- Microsoft MapPoint 2001 
	- Microsoft Streets & Trips 2002, version 1.0 
	- Microsoft Streets and Trips 2001 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start one of the Microsoft programs listed at the beginning of this
	article, you may receive the following error message:
	
	  This application is unable to continue because of problems accessing the
	  network.
	
	CAUSE
	=====
	
	This behavior can occur if either of the following conditions is true:
	
	- The CD-ROM or DVD-ROM drive letter has changed since you installed the
	  program.
	
	- The network location from which you installed the program is either
	  unavailable or has been changed.
	
	RESOLUTION
	==========
	
	To work around this issue, uninstall and reinstall the program:
	
	1. Click Start, point to Settings, and then click Control Panel.
	
	2. Double-click Add/Remove Programs.
	
	3. Click Microsoft Streets and Trips 2001 or Microsoft MapPoint 2001, and then
	  click Add/Remove.
	
	4. Follow the instructions on the screen to uninstall the program. When you are
	  prompted to restart the computer, click No.
	
	5. Insert the Microsoft Streets and Trips Setup CD-ROM or the MapPoint Setup
	  CD-ROM into the CD-ROM or DVD-ROM drive, and then follow the instructions on
	  the screen to install the program.
	
	NOTE: Microsoft recommends that you use the Full installation option if you use a
	removable drive that causes the CD-ROM drive letter to change every time the
	removable drive is attached or removed from the computer.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
