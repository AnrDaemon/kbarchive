## Q272037: Encarta 2001 Err Msg: ENC2001 Caused an Invalid Page Fault...

	Article: Q272037
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 
	Operating System(s): 
	Keyword(s): kberrmsg kbimu
	Last Modified: 07-JAN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Encyclopedia Standard 2001 for Windows 
	- Microsoft Encarta Encyclopedia Deluxe 2001 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click Register Online Now to attempt to register Microsoft Encarta
	after you install the program, you may receive the following error message:
	
	  ENC2001 caused an invalid page fault in module ENC2001.EXE.
	
	CAUSE
	=====
	
	This behavior can occur if both of the following conditions are true:
	
	- You click Register Online Now the first time that you start Encarta after you
	  install it.
	
	- The Encarta CD-ROM is not inserted in the CD-ROM drive.
	
	RESOLUTION
	==========
	
	To resolve this issue, insert the Encarta CD-ROM into the CD-ROM drive, and then
	follow the instructions on the screen to complete Online Registration.
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
