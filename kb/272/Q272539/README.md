## Q272539: Encarta 2001: Program Does Not Respond When You Click Web Links

	Article: Q272539
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 
	Operating System(s): 
	Keyword(s): kbenv kbnetwork kbimu
	Last Modified: 26-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Encyclopedia Deluxe 2001 for Windows 
	- Microsoft Encarta Encyclopedia Standard 2001 for Windows 
	- Microsoft Encarta Reference Suite 2001 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click a Web link on the Microsoft Encarta 2001 Encyclopedia Home Page,
	the program may not respond, and your computer may not attempt to connect to the
	Internet.
	
	CAUSE
	=====
	
	This behavior can occur if Encarta 2001 Encyclopedia is not configured to ask
	you for confirmation before you connect to the Internet.
	
	RESOLUTION
	==========
	
	To resolve this issue, configure Encarta 2001 Encyclopedia to ask you for
	confirmation before you connect to the Internet:
	
	1. Start Encarta Encyclopedia.
	
	2. On the Options menu, click Settings.
	
	3. Click to select the "Ask me for confirmation before connecting to the
	  Internet" check box, and then click OK.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
