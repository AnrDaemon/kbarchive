## Q176837: Bookshelf 98: Computer Stops Responding When Searching Online

	Article: Q176837
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kb3rdparty kbmm kbui kbimukbfaq
	Last Modified: 07-APR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf 98 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click any button on the Go Online tab in Microsoft Bookshelf 98, your
	computer may seem to stop responding (hang).
	
	CAUSE
	=====
	
	This behavior can occur if Netscape Communicator 4.0 is your default Web
	browser. Netscape Communicator may not open in the foreground when the program
	starts.
	
	RESOLUTION
	==========
	
	To resolve this issue, click Netscape Communicator on the taskbar to bring the
	Web browser window to the foreground.
	
	MORE INFORMATION
	================
	
	If Netscape Communicator does not appear on the taskbar, please see the
	following article in the Microsoft Knowledge Base:
	
	  Q176831 Multimedia: Computer Stops Responding When Accessing Web Links
