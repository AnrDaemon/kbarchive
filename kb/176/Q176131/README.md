## Q176131: XCON: Recipient Added from PAB Does Not Convert MSXCX500 DDA

	Article: Q176131
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 14-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	If you submit a message to a recipient located in your Personal Address Book
	(PAB) who resides on a Defense Messaging System (DMS) Exchange Server computer,
	the recipient's MSXCX500 DDA will not be converted to an X.500 Distinguished
	Name (DN).
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server
	version 5.0. This problem has been corrected in the latest U.S. Service Pack for
	Microsoft Exchange Server version 5.0. For information on obtaining the Service
	Pack, query on the following word in the Microsoft Knowledge Base (without the
	spaces):
	
	  S E R V P A C K
