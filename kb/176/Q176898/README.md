## Q176898: FTP &quot;Resume&quot; Functionality Not Supported

{% raw %}

	Article: Q176898
	Product(s): Internet Information Server
	Version(s): WINDOWS:1.0; winnt:1.0,2.0,3.0,4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 02-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Internet Information Server versions 1.0, 2.0, 3.0, 4.0 
	- Microsoft Peer Web Server version 1.0 for Windows NT 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Some FTP servers support the capability to "resume" an interrupted download.
	
	Microsoft Internet Information Server and Microsoft Peer Web Server do not
	support this capability.
