## Q176299: Bridge Too Far: Location of Product Identification Number (PID)

	Article: Q176299
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:2.0
	Operating System(s): 
	Keyword(s): kbusage kbtlckbfaq
	Last Modified: 12-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Close Combat: A Bridge Too Far, version 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how you can locate the Product Identification Number
	(PID) in Close Combat: A Bridge Too Far.
	
	MORE INFORMATION
	================
	
	You can find the Close Combat: A Bridge Too Far PID by following these steps:
	
	1. Start Close Combat: A Bridge Too Far.
	
	2. Press the F8 key to open the Close Combat Options dialog box.
	
	3. Click About Close Combat. The PID is listed at the bottom of the screen.
	
	Additional query words: 2.00 cc abtf product id pin
