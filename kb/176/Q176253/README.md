## Q176253: Plus! For Kids: &quot;Restrictions&quot; Err Msg When You Start Windows

	Article: Q176253
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:1.0,95
	Operating System(s): 
	Keyword(s): kbenv kberrmsg kbusage kbimu
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Plus! for Kids, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You may receive one or more instances of the following error message when you
	start Windows:
	
	  Restrictions
	  This operation has been cancelled due to restrictions in effect on this
	  computer. Please contact your system administrator.
	
	CAUSE
	=====
	
	You have restricted access to certain programs installed on your computer. These
	programs are in your Startup folder and automatically run every time you start
	Windows.
	
	RESOLUTION
	==========
	
	You can either remove these programs from the Windows Startup folder or add them
	to the list of Allowed Programs in Protect It!
	
	To remove a program from the Windows Startup folder, see the Windows Help topic
	"Removing a program from the Start or Programs menu. "
	
	For more information about adding Allowed Programs in Protect It!, see the
	Microsoft Plus! For Kids online Help.
	
	Additional query words: user log on in password logon login
