## Q176925: Multimedia Err Msg: Cannot Load Msreftl.dll

	Article: Q176925
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kberrmsg kbsetup kbimu
	Last Modified: 18-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf 98 for Windows 
	- Microsoft Encarta 98 Encyclopedia for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to install one of the programs listed at the top of this
	article, you may receive the following error message:
	
	  Cannot load Msreftl.dll.
	
	When you click OK, Setup terminates and you are returned to Windows.
	
	CAUSE
	=====
	
	This behavior can occur if the Msreftl.dll file is damaged.
	
	RESOLUTION
	==========
	
	To resolve this issue, delete the Msreftl.dll file. To do so, use the following
	steps:
	
	1. Click Start, point to Find, and then click Files Or Folders.
	
	2. In the Named box, type "msreftl.dll" (without the quotation marks), and then
	  click Find Now.
	
	3. In the list of found files, right-click the Msreftl.dll file, and then click
	  Delete.
	
	4. Close the Find: Files Named Msreftl.dll window.
	
	After you complete these steps, you should be able to install the program
	normally.
