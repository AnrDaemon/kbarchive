## Q124161: Space Required Value Based on Cluster Size

	Article: Q124161
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In the Add/Remove Programs utility in Control Panel, the size displayed in the
	Components list may differ from the size specified for the Space Required value.
	(To view the Components list, click the Windows Setup tab.) The size in the
	Components list is the total size of the actual files. The size specified for
	the Space Required value is calculated using the cluster size of the hard disk
	drive, which varies depending on the size of the hard disk and how it is
	formatted.
	
	Additional query words: space free install size
