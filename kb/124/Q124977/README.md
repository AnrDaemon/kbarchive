## Q124977: Protocol Services Change Connecting Over RAS

	Article: Q124977
	Product(s): Microsoft Windows NT
	Version(s): 3.50
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.5 
	- Microsoft Windows NT Server version 3.5 
	-------------------------------------------------------------------------------
	
	When you connect to a remote network site using Windows NT Remote Access
	Services (RAS), the protocol services may change. By default, RAS connects
	you to remote sites using their protocol. To change this:
	
	1. From Remote Access, choose the Edit button.
	
	2. Choose the Network button
	
	3. In the Network Protocol Settings dialog box, clear Use Default Gateway on
	  Remote Network.
