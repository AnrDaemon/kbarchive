## Q124731: AST Plug and Play BIOS Is Not Registered as Plug and Play

	Article: Q124731
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Your AST Plug and Play BIOS is not registered as Plug and Play capable in
	Microsoft Windows 95.
	
	
	CAUSE
	=====
	
	The AST Plug and Play BIOS contains incorrect information in its 16-bit
	protected-mode entry point. When Windows 95 detects this incorrect code in the
	AST BIOS it does not recognize the BIOS as Plug and Play capable.
	
	RESOLUTION
	==========
	
	You should contact AST about a possible update for your BIOS.
	
	Additional query words: plug n play 3rdparty winpnp
