	SUMMARY
	=======
	
	The following is the README.DOC file contained on disk 1 of the Paris Scenery
	add-on for Flight Simulator 5.0:
	
	              ***************************************
	              MICROSOFT FLIGHT SIMULATOR  Version 5.0
	                        PARIS SCENERY ADD-ON
	                     R E L E A S E    N O T E S
	              ***************************************
	
	Included with this scenery are 5 additional situations that demonstrate
	the Paris scenery - "PARIS - Art Flight", "PARIS - Fly down the Seine",
	"PARIS - La Defense", "PARIS - Night Flight"' and "PARIS - Sacre
	Coeur". To access these situations select them from the Situations
	dialog box by selecting the Options menu and then Situations.
	
	A utility is also shipped with this product- FSSYSTEM.EXE.
	The more expanded memory you have available, the better the Paris
	Scenery will perform.  FSSYSTEM.EXE will create a boot disk with the
	optimum memory configuration for your machine.  See the inside back
	cover of your Microsoft Flight Simulator Pilot's Handbook for more
	information on how to use FSSYSTEM.EXE.
