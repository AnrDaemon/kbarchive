## Q124644: ATI Phone Number Incorrect in README.TXT Files

	Article: Q124644
	Product(s): Microsoft Home Multimedia Titles
	Version(s): '94-'95 edition,1.0; WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Isaac Asimov's The Ultimate Robot for Windows, version 1.0 
	- Microsoft Complete Basketball, version '94-'95 edition 
	- Microsoft Multimedia Strauss for Windows, version 1.0 
	- Microsoft The Ultimate Frank Lloyd Wright, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The phone number for ATI Technical Support is incorrectly listed in the
	README.TXT file included with each of the programs listed above.
	
	MORE INFORMATION
	================
	
	The phone number listed in the README.TXT file is (416) 756-0711. This phone
	number is no longer in service. The correct contact information for ATI
	Technologies is:
	
	  Address:
	
	  33 Commerce Valley Drive East
	  Thornhill, Ontario
	  Canada L3T 7N6
	
	  Phone Numbers:
	
	     Main:            (905) 882-2600
	     User Support:    (905) 882-2626
	     FAX:             (905) 882-2620
	     BBS:             (905) 764-9404
	
	  CompuServe Forum:
	
	  GO ATITECH
