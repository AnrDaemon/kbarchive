## Q124418: Cannot Set Up CD Sampler Because It Is Already Running

{% raw %}

	Article: Q124418
	Product(s): Microsoft Home Multimedia Titles
	Version(s): Fall 1994 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Home CD Sampler Fall 1994 edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to run the Setup program (SETUP.EXE) on the Microsoft Home CD
	Sampler CD-ROM, you receive the following error message:
	
	  Setup for Microsoft CD Sampler cannot be run with the Sampler running. Please
	  exit from the Sampler and then run Setup. Setup cannot proceed. Please choose
	  OK to exit Setup.
	
	RESOLUTION
	==========
	
	To solve this problem, quit all other programs and then run Setup again.
	
	MORE INFORMATION
	================
	
	The error described above can occur if another program that requires
	VBRUN300.DLL is running at the same time you are trying to run Setup.
