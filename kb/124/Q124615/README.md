## Q124615: PC Gen: Post Microsoft Mail 3.2a File Updates

	Article: Q124615
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, version 3.2a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Because of constant improvements to our product, it is necessary for Microsoft
	Mail for PC Networks customers to upgrade individual files to assure them of
	product integrity and up-to-date functionality. Below is a concise listing of
	required files for customers to have the most current client/postoffice
	configurations.
	
	MORE INFORMATION
	================
	
	As of January 1995, version 3.2a is the most recent revision of our PC Mail
	product. Executable (*.EXE) files that shipped with version 3.2a are version
	3.2.9. .DLL files shipped in 3.2a are version 3.2.0.4074. Although version 3.2a
	is the latest full revision, we continually update individual files to include
	fixes and improve performance. The following is a list of updated files you
	should obtain in conjunction with version 3.2a to ensure that you are running
	the latest version.

	You can download any of these self-extracting files from the following services:
	
	- Microsoft's World Wide Web Site on the Internet
	
	  On the www.microsoft.com home page, click the Support icon.
	  Click Knowledge Base, and select the product.
	  Enter kbfile <FILENAME>.EXE, and click GO!
	  Open the article, and click the button to download the file.
	
	- Internet (anonymous FTP) ftp ftp.microsoft.com
	
	  Change to the Softlib/Mslfiles folder.
	  Get <FILENAME>.EXE
	
	- The Microsoft Network
	
	  On the Edit menu, click Go To, and then click Other Location. Type "mssupport"
	  (without the quotation marks).
	  Double-click the MS Software Library icon.
	  Find the appropriate product area.
	  Locate and Download <FILENAME>.EXE.
