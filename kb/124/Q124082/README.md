## Q124082: Remote Control Causes GP Fault on Windows 95 Client

	Article: Q124082
	Product(s): Microsoft Systems Management Server
	Version(s): 1.1
	Operating System(s): 
	Keyword(s): kbnetwork kbtshoot smsremtshoot kbRemoteProg
	Last Modified: 06-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.1 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	Remote Control causes a general protection fault (GPF) on a Windows 95 client
	that has the Windows 95 Plus Pack loaded and font smoothing turned on with more
	than 256 colors set.
	
	This problem occurs regardless of the size of the screen (640x480, 800x600,
	1024x768) or the theme that is loaded.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.1. This problem has been corrected in Systems Management Server
	version 1.2.
