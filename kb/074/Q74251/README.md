## Q74251: MS-DOS 5.0 and Windows Expand Utilities Not Interchangeable

{% raw %}

	Article: Q74251
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): MS-DOS:5.x,6.0,6.2,6.21,6.22; WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	- Microsoft MS-DOS operating system versions 5.0, 5.0a, 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	The EXPAND.EXE utilities that come with Microsoft Windows and the MS-DOS
	are not interchangeable. For example, if you try to expand compressed
	MS-DOS 6.x or Windows 3.1 file using the EXPAND command from MS-DOS 5.0 you
	receive the following error message:
	
	  Input file <filename> already in expanded format.
	
	If a problem occurs when expanding files from the Windows or MS-DOS disks,
	use the respective EXPAND.EXE file that came with the product.
