## Q74836: Specifying HIMEM Handlers for Acer Computers

	Article: Q74836
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.0,5.0a,6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 5.0, 5.0a, 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	During Acer's hardware compatibility test for Acer computers, certain machines
	were found to work optimally using the /Machine:7 (/M:7) HIMEM.SYS handler. The
	normal handler for Acer 1100 computers is /M:6, or /M:ACER1100.
	
	MORE INFORMATION
	================
	
	The following is a list of Acer machines that use the /M:7 handler:
	
	  Acer 1100/25E
	  Acer 1116/SX
	  Acer 1120/SX
	  Acer 1170
	  Acer 1200
	  Acer 915P
	
	Additional query words: 5.00 5.00a 6.00 6.20
