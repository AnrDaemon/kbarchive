## Q74332: INFO: Actions Prohibited in System Modal Dialog Boxes

	Article: Q74332
	Product(s): Microsoft Windows Software Development Kit
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): kb16bitonly kbDlg kbSDKPlatform
	Last Modified: 10-JUN-1999
	
	3.00 3.10
	WINDOWS
	kbprg
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows Software Development Kit (SDK) 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following actions are not supported in a system modal dialog box in the
	Microsoft Windows graphical environment:
	
	- Sending an intertask message
	
	- Changing the focus
	
	- Activating another application
