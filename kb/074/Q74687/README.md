## Q74687: Notepad Err Msg: File Is Empty and Will Be Deleted...

	Article: Q74687
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you delete all the contents of a Notepad file and then attempt to save it, a
	message similar to the following is displayed:
	
	  This file is empty and will be deleted.
	
	Additional query words: 3.00 3.0 3.0a 3.00a 3.10 3.11 blank clear
