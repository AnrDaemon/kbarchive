## Q74259: Windows Err Msgs: Compressed Windows Utilities

	Article: Q74259
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run a compressed Windows-based application without any other
	applications running, you receive one of the following error messages:
	
	  Insufficient Memory to Run Application; Close one or more Applications to
	  Increase Available Memory and try again.
	
	  -or-
	
	  This Application has Violated System Integrity and will be terminated. Close
	  all applications, exit Windows, and reboot your computer.
	
	The first error may occur when there is ample memory available and other
	applications run without error. The screen may blink once before returning the
	error message.
	
	An MS-DOS icon generally appears at the bottom of the screen, indicating that an
	MS-DOS-based application has been minimized.
	
	CAUSE
	=====
	
	This problem can occur when the application's .EXE file is corrupted.
	
	RESOLUTION
	==========
	
	To correct this problem, expand a new copy of the .EXE file from your original
	Windows or Windows for Workgroups disks.
	
	Additional query words: 3.00 3.00a 3.10 3.11 unexpanded uncompressed utility filename file name compressed expanded size calculator calc.exe calendar calendar.exe cardfile cardfile.exe clipboard clipbrd.exe clock clock.exe control panel control.exe notepad notepad.exe paintbrush pbrush.exe pif editor pifedit.exe print manager printman.exe recorder recorder.exe reversi reversi.exe solitaire sol.exe system sysedit.exe task taskman.exe terminal terminal.exe
