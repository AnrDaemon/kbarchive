## Q74587: Older Mouse Systems Drivers Incompatible with MS-DOS 5.0 Shell

{% raw %}

	Article: Q74587
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 03-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Mouse Systems mouse driver version 6.23 or earlier is incompatible with
	Microsoft MS-DOS version 5.0 Shell. Version 7.01 of the Mouse Systems mouse
	driver is compatible with MS-DOS 5.0. This driver is available from the Mouse
	Systems electronic bulletin board service (BBS). Mouse Systems does not charge
	for this service.
