## Q74182: ANSI.SYS /L Switch Not Available in MS-DOS 5.0 or Later

	Article: Q74182
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.x,6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 5.0, 5.0a, 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The ANSI.SYS /L switch is not available in MS-DOS 5.0 or later, as it was with
	MS-DOS 4.x. The function of this switch is to retain the number of screen rows
	for application programs that return the number of screen rows to the default
	value of 25.
	
	The /L switch can be added as a parameter to ANSI.SYS, but it is not functional
	in the same manner it was with MS-DOS 4.x.
