## Q74252: Using PC Fullback and Windows 3.0

{% raw %}

	Article: Q74252
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	PC Fullback technical support does not recommend running its PC Fullback
	software under Windows 3.0 in 386 enhanced mode. PC Fullback may not run
	reliably in standard or real modes, either.
	
	MORE INFORMATION
	================
	
	PC Fullback attempts to use a DMA channel that can be used by the floppy
	controller or the hard drive controller.
	
	NOTE: The manufacturers of PC Fullback are not planning to create a Windows
	version of PC Fullback.
	
	For more information, contact PC Fullback technical support.
	
	PC Fullback is manufactured by a vendor independent of Microsoft; we make no
	warranty, implied or otherwise, regarding this product's performance or
	reliability.
