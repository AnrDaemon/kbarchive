## Q74650: BUG: &quot;Insufficient Floppy Drive&quot; (DataView Snap 1+1)

	Article: Q74650
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.x
	Operating System(s): 
	Keyword(s): kb3rdparty kberrmsg kbhw kbsetup msdos kbHardware
	Last Modified: 24-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 5.0, 5.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you attempt to install Microsoft MS-DOS 5 Upgrade onto a DataView Snap 1+1
	8088, you receive the following error message:
	
	  insufficient floppy drive capacity
	
	RESOLUTION
	==========
	
	DataView has not tested MS-DOS version 5.0 and has no plans to support MS-DOS
	5.0 with DataView Snap 1+1. Only DataView's DOS versions 2.11 and 3.30 are
	supported.

	================
