## Q74534: Everex EFAX with MS-DOS 5.0

	Article: Q74534
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.x
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 5.0, 5.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Everex produces a fax/modem board that works with a memory resident software
	program called EFAX. For compatibility with MS-DOS version 5.0, you must have
	EFAX version 4.22 or later.
	
	MORE INFORMATION
	================
	
	To obtain the updated version of EFAX, contact Everex Systems.

	reliability.
