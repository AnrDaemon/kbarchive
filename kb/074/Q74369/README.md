## Q74369: Compuadd Mouse Driver for MS-DOS 5.0 Shell

{% raw %}

	Article: Q74369
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.x
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 5.0, 5.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	According to Compuadd Technical Support, you need the Compuadd Mouse driver,
	CMOUSE, to use the Compuadd Mouse with version 5.0 of the MS-DOS Shell.
	
	
	To obtain an updated driver, contact Compuadd Technical Support or download the
	driver from Compuadd's bulletin board service (BBS).
	
	MORE INFORMATION
	================
	
	For more information, contact Compuadd.
	
	The Compuadd products included here are manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding these products'
	performance or reliability.
