## Q261033: MCM2: Missing Textures on Bike &amp; Rider Tab After You Exit Event

	Article: Q261033
	Product(s): Microsoft Home Games
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): kbdisplay kbimu msgamekbfaq
	Last Modified: 12-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Motocross Madness 2, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you exit an event or a race in Microsoft Motocross Madness 2, the textures
	on the motorcycle and rider on the Bike & Rider tab are missing.
	
	CAUSE
	=====
	
	This behavior can occur if a Creative Labs 3D Blaster Savage 4 PCI video adapter
	is installed in your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest version of the video
	driver for your video adapter from the following Creative Labs Web site:
	
	  http://www.creative.com
