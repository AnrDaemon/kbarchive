## Q261036: MCM2: Textures of the Motorcycles and Riders Appear Distorted

	Article: Q261036
	Product(s): Microsoft Home Games
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbgraphic kbimu msgamekbfaq
	Last Modified: 12-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Motocross Madness 2, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Motocross Madness 2, textures of the motorcycles and
	riders may appear distorted.
	
	CAUSE
	=====
	
	This behavior can occur if a Creative Labs Graphics Blaster RIVA TNT video
	adapter is installed in your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest version of the RIVA TNT
	reference driver from the following Nvidia Web site:
	
	  http://www.nvidia.com
	
	NOTE: Some features of the Creative Labs video adapter may not function as
	expected after you install the RIVA TNT reference driver.
	
	MORE INFORMATION
	================
	
	To determine the brand and model of your video adapter:
	
	1. Click Start, point to Settings, and then click Control Panel.
	
	2. Double-click System.
	
	3. Click Device Manager.
	
	4. Double-click Display Adapters.
	
	In the branch under Display Adapters, the brand and model of your video adapter
	is displayed.
	
	5. Close Control Panel.
