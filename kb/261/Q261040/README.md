## Q261040: Motocross Madness 2: Black Screen When You Start the Program

	Article: Q261040
	Product(s): Microsoft Home Games
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Motocross Madness 2, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Motocross Madness 2, you may experience any of the
	following symptoms:
	
	- You may see a black screen when you start the program.
	
	- Textures on the motorcycles and riders may be distorted.
	
	CAUSE
	=====
	
	This behavior can occur if a Diamond Stealth III S540 video adapter is installed
	in your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest driver for the Diamond
	Stealth III S540 video adapter from the following S3 Web site:
	
	  http://www.s3.com/customer_care.htm
