## Q261177: XADM: Adminpak Needed for Installing Exchange System Manager

	Article: Q261177
	Product(s): Microsoft Exchange
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 31-JAN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange 2000 Server 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	To install Microsoft Exchange System Management Tools on a workstation running
	Microsoft Windows 2000 Professional, you must first install the Microsoft
	Windows 2000 Adminpak (Adminpak.msi) on the workstation.
	
	The Adminpak.msi file can be found on the Microsoft Windows 2000 Server CD in the
	I386 folder.
