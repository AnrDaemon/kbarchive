## Q261041: Motocross Madness 2: Game Quits Unexpectedly When You Start Race

	Article: Q261041
	Product(s): Microsoft Home Games
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 12-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Motocross Madness 2, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to start any race in Microsoft Motocross Madness 2, the game
	may quit unexpectedly.
	
	CAUSE
	=====
	
	This behavior can occur if the following conditions are true:
	
	- A Diamond Viper V770 video adapter is installed in your computer.
	
	- The video driver for your video adapter is outdated.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest video driver for the
	Diamond Viper V770 video adapter from the following S3 Web site:
	
	  http://www.s3.com
	
	
	MORE INFORMATION
	================
	
	This behavior has been reported with the version 4.10.1.401 of the video driver
	for the Diamond Viper V770 video adapter.
