## Q261028: MCM2: Game Hangs When the Introductory Video Clip Is Played

	Article: Q261028
	Product(s): Microsoft Home Games
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): kbdisplay kbimu
	Last Modified: 12-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Motocross Madness 2, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start Microsoft Motocross Madness 2, the game may stop responding
	(hang) when the introductory video clip is played.
	
	If you press the ESC key to bypass the introductory video clip, Motocross Madness
	2 may hang when you start the game.
	
	CAUSE
	=====
	
	This behavior can occur if an ATI 3D RagePro AGP video adapter is installed on
	your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest video driver for the ATI
	3D RagePro AGP video adapter from the following ATI Web site:
	
	  http://www.atitech.com
	
	NOTE: Although the latest driver for the ATI RagePro AGP video adapter resolves
	this issue, the game may run very slowly when you use an ATI 3D RagePro AGP
	video adapter.
	
	
	MORE INFORMATION
	================
	
	This issue has been reported with the 8 MB (megabyte) ATI 3D RagePro AGP video
	adapter.
