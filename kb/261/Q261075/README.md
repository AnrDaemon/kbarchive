## Q261075: Motocross Madness 2: Startup Screen Is Not Displayed Properly

	Article: Q261075
	Product(s): Microsoft Home Games
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgamekbfaq
	Last Modified: 12-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Motocross Madness 2, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start Microsoft Motocross Madness 2, the startup screen and the game
	options screens may not be displayed properly.
	
	CAUSE
	=====
	
	This behavior can occur if an Accell Graphicstar2 video adapter is installed in
	your computer.
	
	NOTE: The Accell Graphicstar2 video adapter is based on the Permedia2 chip set.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest reference driver for the
	Permedia2 chip set from the following 3Dlabs Web site:
	
	  http://www.3dlabs.com
	
	MORE INFORMATION
	================
	
	To determine the brand and model of your video adapter:
	
	1. Click Start, point to Settings, and then click Control Panel.
	
	2. Double-click System.
	
	3. Click Device Manager.
	
	4. Double-click Display Adapters.
	
	In the branch under Display Adapters, the brand and model of your video adapter
	is displayed.
	
	5. Close Control Panel.
