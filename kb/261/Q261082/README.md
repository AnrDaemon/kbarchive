## Q261082: MCM2: &quot;Ready&quot; Unavailable on Multiplayer Game Settings Screen

	Article: Q261082
	Product(s): Microsoft Home Games
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): kbui igz kbimu msgamekbfaq
	Last Modified: 12-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Motocross Madness 2, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to join a multiplayer game in Microsoft MotoCross Madness 2,
	the Ready button next to your name on the game settings screen may be
	unavailable.
	
	CAUSE
	=====
	
	This behavior can occur if the host starts a game based on a custom track that
	is not installed on your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, make sure that the track that the host is using is
	installed on your computer before you join the game.
	
	MORE INFORMATION
	================
	
	To use a custom track in a multiplayer game, all participants in the game must
	install the custom track on their computers.
