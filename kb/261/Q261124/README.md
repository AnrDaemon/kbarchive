## Q261124: FS2000: Vocal Instructions Are Not Available in Tutorial Flights

{% raw %}

	Article: Q261124
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbsound fsim kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator 2000 
	- Microsoft Flight Simulator 2000 Professional Edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you fly a Tutorial flight in Microsoft Flight Simulator 2000, vocal
	instuctions are not available.
	
	MORE INFORMATION
	================
	
	Tutorial flights do not feature vocal instructions. Tutorial flights are
	designed to be used in conjunction with the manual in a step-by-step fashion.
	
	NOTE: Vocal instructions are available in the Lessons and in some of the
	Adventures.
