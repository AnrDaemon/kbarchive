## Q261035: MCM2: Textures Become Distorted After Text Is Displayed

{% raw %}

	Article: Q261035
	Product(s): Microsoft Home Games
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgamekbfaq
	Last Modified: 12-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Motocross Madness 2, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Motocross Madness 2, textures in the game may become
	distorted after text is displayed on the screen.
	
	This symptom can occur after the user interface screens are displayed or while
	you are racing.
	
	CAUSE
	=====
	
	This behavior can occur if a Creative Labs 3D Blaster RIVA TNT2 Ultra video
	adapter is installed in your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest version of the video
	driver for your video adapter from the following Creative Labs Web site:
	
	  http://www.creativelabs.com/
