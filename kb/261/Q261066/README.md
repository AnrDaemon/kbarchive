## Q261066: Motocross Madness 2: Results for Previous Race Are Not Recorded

{% raw %}

	Article: Q261066
	Product(s): Microsoft Home Games
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Motocross Madness 2, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Motocross Madness 2, the results for the previous race
	that you completed may not be recorded.
	
	CAUSE
	=====
	
	This behavior can occur if both of the following conditions are true:
	
	- Motocross Madness 2 is installed on a Microsoft Windows 2000-based computer.
	
	- The computer stops responding during the podium sequence for an event type
	  (such as an Enduro Race) on the Pro-Circuit.
	
	RESOLUTION
	==========
	
	To resolve this issue, repeat the race to complete the event type.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Motocross Madness 2,
	version 2.0.
