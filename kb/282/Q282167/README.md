## Q282167: Microsoft Plus! Game Pack: Minimum Requirements

	Article: Q282167
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Plus! Game Pack: Cards and Puzzles, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the minimum system requirements that are needed to run
	Microsoft Plus Game Pack Cards & Puzzles.
	
	MORE INFORMATION
	================
	
	To run Microsoft Plus Game Pack Cards & Puzzles, you need:
	
	- Multimedia personal computer with a 166 megahertz (MHz) Pentium processor;
	  Pentium II 233 MHz processor recommended
	
	- Microsoft Windows 95, Microsoft Windows 98 or later, or Microsoft Windows
	  2000 operating system
	
	- 32 megabytes (MB) of random access memory (RAM)
	
	- 150 MB of available hard disk space
	
	- Quad-speed CD-ROM drive or faster
	
	- Audio board and speakers recommended
	
	- Microsoft Mouse or compatible pointing device
	
	- Super VGA video display with 2 MB of video RAM (VRAM); 4 MB VRAM recommended
	
	Requirements for Multiplayer Play:
	
	- A 28.8 kilobits per second (Kbps) modem (faster modem recommended), local
	  area network with TCP/IP or IPX protocol, or nullmodem cable required for
	  connection to other systems
	
	- Internet access required for Internet play. May require payment of a separate
	  fee to an Internet service provider (ISP). Local and long-distance charges
	  may apply.
	
	- To play on Zone.com, you must have Microsoft Internet Explorer 3.02, Netscape
	  Navigator 4.x, or later browser software.
