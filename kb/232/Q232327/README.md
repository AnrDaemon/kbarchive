## Q232327: Microsoft Office 2000 Resource Kit Comments and Corrections

{% raw %}

	Article: Q232327
	Product(s): Microsoft Press
	Version(s): WINDOWS:2000
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 16-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Office 2000 Resource Kit 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Microsoft Office 2000 Resource Kit, ISBN
	0-7356-0555-6.
	
	The following topics are covered:
	
	- Back Cover: Add System Requirements
	
	- Back Cover: Ownership is Misspelled
	
	MORE INFORMATION
	================
	
	Back Cover: Add Systems Requirements
	------------------------------------
	
	Back Cover:
	Change:
	"For System Requirements, see the information near the CD-ROM"
	
	To:
	"Microsoft Office 2000 must be installed in order to use the CD-ROM included with
	this book."
	
	
	Back Cover: Ownership is Misspelled
	-----------------------------------
	
	Back Cover, Paragraph 1, Last Line:
	
	Replace with:
	"...Ownership..."
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
