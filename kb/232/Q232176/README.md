## Q232176: Adding Multilingual Support Removes the Euro Fonts

{% raw %}

	Article: Q232176
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): kbenv win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you type the euro symbol in a document, you may see a square box instead of
	the euro symbol.
	
	CAUSE
	=====
	
	Windows 95 does not have built-in euro support. Installing the euro patch update
	the fonts and the Graphic Device Interface (GDI). When you install Multilingual
	support after installing the euro patch, you restore the non-euro fonts.
	
	RESOLUTION
	==========
	
	To resolve this issue, reinstall the euro patch. The euro patch is available on
	the Microsoft web site at:
	
	  http:\\www.microsoft.com\euro
