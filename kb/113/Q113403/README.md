## Q113403: Drive Fails with Conner CFP 1060S Fixed Disk &amp; Firmware 1823

{% raw %}

	Article: Q113403
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 3.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use a Conner CFP 1060S fixed disk with firmware version 1823, the disk
	drive fails. This problem can occur in both the MS-DOS and Windows environments.
	The operating system is still running, but no disk access is available.
	
	
	MORE INFORMATION
	================
	
	If you experience this problem, contact Conner Technical Support for assistance.
