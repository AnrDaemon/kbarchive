## Q113100: Multimedia Titles: Cannot Print Selected Areas of Text

	Article: Q113100
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 
	Operating System(s): 
	Keyword(s): kbbuglist kbfaq
	Last Modified: 25-OCT-2001
	
	WINDOWS
	kbprint kbbuglist kbmm kbprb kbfaq
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for the Macintosh, 1993, 1994, 1995 editions 
	- Microsoft Bookshelf 1996-97 for Windows 
	- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia 
	- Microsoft Cinemania for Windows, 1992, 1993, 1994, 1995, 1996, 1997 editions 
	- Microsoft Music Central for Windows, 1996, 1997 edition 
	- Microsoft Encarta World Atlas for Windows, 1997 World English Edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In the multimedia titles listed above, you are unable to print selected areas of
	text.
	
	RESOLUTION
	==========
	
	Copy the text into the Clipboard and then paste it into a word processor or
	Notepad:
	
	1. Select the desired text with your mouse cursor.
	
	2. Press CTRL+INSERT to copy the text.
	
	3. Open Notepad or a Word Processor.
	
	4. Press SHIFT+INSERT to paste the text.
	
	MORE INFORMATION
	================
	
	When you select a topic, highlight some of the text, then try printing the
	selection, the radio button is unavailable (dimmed) or there is no option for
	printing a selection.
	
	This is true whether you press and hold the CTRL key while you press the P key
	(CTRL+P), choose Print Article from the File menu, or choose the Print button to
	access the Print dialog box.
	
	In Bookshelf 96-97, when you print selected text, only one line of the text may
	print. Use the above resolution to work around this problem.
	
	Microsoft has confirmed this to be a problem with the applications listed above.
	This problem was corrected in Encarta Encyclopedia for Windows, 1996 edition and
	Bookshelf '95 for Windows 95.
