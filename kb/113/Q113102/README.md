## Q113102: Bookshelf: Pronunciations Don't Work under Windows NT

	Article: Q113102
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1994 edition,1995 edition; winnt:3.1,3.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for Windows versions 1994 edition, 1995 edition 
	- the operating system: Microsoft Windows NT 3.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you run Bookshelf under Windows NT, pronunciations and other audio clips may
	not be audible.
	
	MORE INFORMATION
	================
	
	The following information is from the README.TXT file included with Bookshelf
	1994:
	
	  When Bookshelf is installed on a machine running Windows NT 3.1, you may not
	  hear audio clips, such as pronunciations, in Bookshelf. Windows NT 3.1 cannot
	  play audio in the format that Bookshelf uses.
	
	
	Support for GSM 6.10, the audio compression format used by Bookshelf
	pronunciations, was added to Windows NT version 3.51.
	
	REFERENCES
	==========
	
	The README.TXT file included with Bookshelf 1994 for Windows.
