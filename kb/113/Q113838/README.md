## Q113838: Comprehensive List of MS-DOS 6.21 Documentation Errors

{% raw %}

	Article: Q113838
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.21
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 6.21 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains information on the documentation errors in MS-DOS Help.
	
	
	The following topics are covered:
	
	Incorrect References to EGA3.CPI, ISO.CPI, and COUNTRY.TXT
	----------------------------------------------------------
	
	MS-DOS 6.21 Help incorrectly references EGA3.CPI, ISO.CPI, and COUNTRY.TXT. These
	files are not included with MS-DOS 6.21.
