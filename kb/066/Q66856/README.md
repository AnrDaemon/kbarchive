## Q66856: File Date Equals Date of Entertainment Pack Installation

	Article: Q66856
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	After running the Microsoft Windows Entertainment Pack (WEP1) Setup program, the
	creation date of the files copied to the hard disk will be equal to the date set
	on the computer when the Entertainment Pack was installed.
	
	The files on the original disks will be the only indication of the original
	creation date of the Windows Entertainment Pack.
