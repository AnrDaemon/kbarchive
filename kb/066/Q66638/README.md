## Q66638: ASD: Coordinates for Microsoft Campus

	Article: Q66638
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	Microsoft headquarters in Redmond, WA can be found at the following coordinates
	using the Seattle scenery found on the Flight Simulator Aircraft & Scenery
	Designer version 4.00:
	
	  North: 21396.0000
	  East:   6662.8000
	
	  Altitude:  505 ft.
	  Heading:   180
	
	To jump to these coordinates, choose menu 5, option A to activate the Position
	Set option.
