	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s):

	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	The coordinates for the Statue of Liberty are as follows:
	
	North     17048.5423
	East      20958.8116
	Altitude  1999
	Heading   105.44
	
	For more information about setting coordinates, see page 76 of the "Information
