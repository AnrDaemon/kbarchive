## Q66419: Write: Italic Fonts Truncated in Windows

	Article: Q66419
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:2.x,3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 2.03, 2.1, 2.11, 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you use italic formatting in Microsoft Windows Write and the last italic
	character is followed by a non-italic character, the last italic character may
	appear to be truncated.
	
	CAUSE
	=====
	
	This behavior occurs because of the way Windows handles italic screen fonts.
	Such truncations do not occur if you use TrueType fonts, and do not affect
	printed output.
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here as it
	becomes available.
