## Q66843: ASD: 747 Panel Checklist Is Stuck on an Action

	Article: Q66843
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	The checklist on the 747 panel is not always able to correctly determine when an
	action has been completed.
	
	If a checklist appears stuck on an action that has already been performed, press
	SHIFT+Z to have the checklist processor scan the current conditions.
