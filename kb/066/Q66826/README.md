## Q66826: Flt Sim: Crop Duster Score Window Stays Active in Other Modes

{% raw %}

	Article: Q66826
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	The following procedure will cause the Crop Duster score window to appear in
	other modes:
	
	1. From the Mode menu, select option 5, Entertainment.
	
	2. Select option 3, Crop Duster.
	
	3. From the Mode menu, select option B and change to a different mode.
	
	4. The Crop Duster score window remains active, either blinking or stationary.
	
	To properly exit an entertainment mode, from the Mode menu, select Normal Flight.
