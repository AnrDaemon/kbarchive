## Q66985: Negative Score in Tetris Near 32,000

{% raw %}

	Article: Q66985
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	When you play Tetris, included in the Microsoft Windows Entertainment Pack
	(WEP1), the score changes to a negative value when it reaches 32,767.
	
	Microsoft is researching this problem and will post new information here as it
	becomes available.
