## Q66961: Starting Windows Applications in Full Screen Mode

{% raw %}

	Article: Q66961
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	There is no way to set the following Microsoft Windows applications to
	start in a full screen:
	
	Calculator      Calendar        Cardfile
	Clipboard       Clock           Control Panel
	File Manager    Notepad         Paintbrush
	Recorder        Reversi         Print Manager
	Solitaire       PIF Editor      Windows Setup
	Write
	
	Additional query words: calc 3.00 3.0 3.0a 3.00a 3.10 maximize 3.11 maximized
