## Q66071: Don't Run Windows 3.0 with Amdek DOS Version 3.21

{% raw %}

	Article: Q66071
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Neither Microsoft Windows version 3.0 nor the SMARTDrive program provided with
	Windows 3.0 should be run on a system that uses Amdek Enhanced DOS version 3.21.
	It has been reported that this combination of software can damage the File
	Allocation Table (FAT), which could cause your hard disk to crash.
	
	MORE INFORMATION
	================
	
	If you have Amdek Enhanced DOS version 3.21 and want to run Windows version 3.0
	on your machine, upgrade to Amdek Enhanced DOS 3.30. Amdek Enhanced DOS 3.30
	appears to run Windows without any problems.
