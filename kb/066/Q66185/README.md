## Q66185: Flight Simulator: Airport in the Sky in Tower and Track View

	Article: Q66185
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	If the altitude of the control tower is set below ground level, the airport may
	appear to be floating in the sky.
	
	To correct the problem, set the altitude of the control tower higher than the
	altitude of the airport. Do the following to change the altitude of the control
	tower:
	
	1. Choose option 5, Nav/Com.
	
	2. Choose option A, Position Set.
	
	3. Choose option 8, Control Tower Altitude, and enter an altitude above ground
	  level.
	
	MORE INFORMATION
	================
	
	Airport altitudes can be found in Appendix A of the "Information Manual and
	Flight Handbook."
