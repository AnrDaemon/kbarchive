## Q66639: ASD: Scenery on the Box Is Not Included

	Article: Q66639
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	The scenery pictured on the back of the Aircraft & Scenery Designer box is
	not included with the package. The pictures demonstrate what can be created with
	the package.
