## Q66574: ASD: 747-400 Separate Engine Control

{% raw %}

	Article: Q66574
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	Individual engines cannot be controlled on the Boeing 747-400 or the Beechcraft
	Starship. Individual engine control is not a feature of Aircraft & Scenery
	Designer version 4.00.
	
	MORE INFORMATION
	================
	
	When using the enhanced (747-400) instrument panel, there are four separate
	engine efficiency indicators. The four engines cannot be individually adjusted.
	The panel will always display the same value for all four engines.
