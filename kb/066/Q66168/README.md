## Q66168: No CGA Support in Windows Entertainment Pack

	Article: Q66168
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	The Microsoft Windows Entertainment Pack only supports a EGA, VGA, 8514/A,
	Hercules graphics card, or compatible video-graphics adapter and monitor. CGA is
	not supported in the Windows Entertainment Pack and is not recommended.
