## Q66357: Using MS-DOS Applications on HP Vectra in Standard Mode

	Article: Q66357
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If your Hewlett-Packard (HP) Vectra is running in standard mode in Windows 3.0,
	you cannot use a hot key to move to an MS-DOS application. If you attempt to do
	this, the screen becomes blank and your system stops responding (hangs).
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows version 3.0. We are
	researching this problem and will post new information here as it becomes
	available.
	
	Additional query words: 3.00 3.00a 3.0 3.0a KBHW blanks out
