## Q66142: System Requirements for Windows Entertainment Pack

{% raw %}

	Article: Q66142
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	The following are the system requirements for the Microsoft Windows
	Entertainment Pack:
	
	  80286 or above based machine
	  Hercules, EGA, VGA (CGA is not supported)
	  Microsoft Mouse or compatible
	  Hard drive
	  MS-DOS or PC-DOS version 3.10 or later
	  Microsoft Windows 3.00
	
	Additional query words: 3.00 3.0 3.0a 3.00a
