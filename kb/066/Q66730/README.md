## Q66730: Video Seven BIOS Versions Required for Use with Windows

{% raw %}

	Article: Q66730
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following is a table that contains the ROM BIOS revisions that are required
	for Video Seven to work correctly with Microsoft Windows:
	
	  Types of Video Seven Cards   Supported ROM Revisions
	  --------------------------   -----------------------
	
	  Vega                           1.47  1.48  1.78
	  VRAM                           1.1   1.18
	  FastWrite                      1.1   1.18
