## Q66585: Flight Simulator: Support for MCGA

	Article: Q66585
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	When using Flight Simulator 4.00 with an MCGA video adapter, select one of the
	following in the start-up menus (depending on the type of monitor used):
	
	Selection  Monitor
	---------  -------
	
	C          RGB monitor (CGA)
	F          EGA monochrome
	I          IBM PS/2 16 color 320 x 200
	J          IBM PS/2 256 color 320 x 200
	
	Option J provides the highest resolution.
