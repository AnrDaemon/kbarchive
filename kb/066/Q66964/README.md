## Q66964: Displaying Outline, Shadow Text in Windows Paintbrush

{% raw %}

	Article: Q66964
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	To display outline or shadow text in Microsoft Windows Paintbrush, do the
	following:
	
	1. Select the Text tool.
	
	2. From the Text menu, choose Outline or Shadow.
	
	3. Use the right mouse button to select a background color other than white.
	
	4. Use the left mouse button to select a foreground color that is different from
	  the background color.
	
	MORE INFORMATION
	================
	
	If you attempt to use white as a background color, the text is not displayed in
	an outline or shadow format.

	==========
	
	"Microsoft Windows User's Guide," version 3.1, pages 339-343
