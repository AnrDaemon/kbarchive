## Q66745: ASD: Support for Sublogic Instant Facilities Locator

	Article: Q66745
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	Sublogic's Instant Facilities Locator (IFL) is a Flight Simulator add-on
	product. IFL requires additional free memory, above the requirements of Flight
	Simulator and Aircraft & Scenery Designer.
	
	When IFL is installed, an option G for Facilities Locator appears on the Nav/Com
	menu between the ESC and SPACE options.
	
	Additional query words: asd A SD low limited memory warning
