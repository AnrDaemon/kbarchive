## Q66167: Setup Needs the Following Disk: Entertainment Pack Disk 1

{% raw %}

	Article: Q66167
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	If setting up the Microsoft Windows Entertainment Pack from a drive other than
	Drive A, you will receive the following message:
	
	  Setup needs the following disk: Entertainment Pack Disk 1. Please insert the
	  disk in the drive named below, or type the name of the drive where the disk
	  is located.
	
	This problem is resolved by typing the correct drive letter where the
	Entertainment Pack Disk 1 is located.
