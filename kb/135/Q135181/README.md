## Q135181: MS Works for MS-DOS File Has Garbled Text in Quick View

{% raw %}

	Article: Q135181
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use Quick View to view a document created in Microsoft Works version
	3.0 for MS-DOS, you see garbled characters at the top of the document.
	
	CAUSE
	=====
	
	Files created in Microsoft Works 3.0 for MS-DOS are not compatible with Quick
	View.
	
	RESOLUTION
	==========
	
	Open the file in Microsoft Works 3.0 for MS-DOS.
