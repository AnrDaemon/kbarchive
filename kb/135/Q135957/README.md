## Q135957: Incorrect Free Space Reported After Uncompressing Drive

{% raw %}

	Article: Q135957
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	- Microsoft Plus! for Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you uncompress a drive that was compressed with DriveSpace 3, the final
	dialog box lists less free space than actually exists.
	
	RESOLUTION
	==========
	
	To get an accurate report of the amount of free disk space, follow these steps:
	
	1. Close the dialog box, and allow the computer to restart if you are prompted
	  to do so.
	
	2. Double-click My Computer.
	
	3. Use the right mouse button to click the drive, and then click Properties on
	  the menu that appears.
