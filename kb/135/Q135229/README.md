## Q135229: Paradox Coaches Button is Not Displayed

	Article: Q135229
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run Paradox, you cannot gain access to the Coaches button in the
	lower-left corner of the screen.
	
	CAUSE
	=====
	
	The Windows 95 taskbar is covering the Coaches button.
	
	WORKAROUND
	==========
	
	To work around this behavior, move the taskbar to the left or right side, or to
	the top of the screen.
