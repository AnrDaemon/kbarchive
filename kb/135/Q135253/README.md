## Q135253: E-mail Lost When Disk Space Is Low

{% raw %}

	Article: Q135253
	Product(s): The Microsoft Network
	Version(s): 1.00 1.2 1.3 2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.0, 1.05, 1.2, 1.3, 2.0, 2.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to download an e-mail message, you may receive the following
	error message:
	
	  The mail could not be delivered. Action canceled.
	
	CAUSE
	=====
	
	Your computer's hard disk may be out of space.
	
	MORE INFORMATION
	================
	
	If you attempt to download an e-mail message when your hard disk is full, the
	e-mail message is deleted from the server without any notification to you.
