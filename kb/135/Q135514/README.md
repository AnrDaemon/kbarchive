## Q135514: FltSim 5.1: Setup Option to Save Disk Space and Time

	Article: Q135514
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:5.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for MS-DOS, version 5.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	During the installation of Microsoft Flight Simulator version 5.1, you can use
	one of two setup options:
	
	- Save Disk Space
	
	  -or-
	
	- Save Loading Time
	
	The Save Disk Space option prevents 3,578,030 bytes of data from being stored on
	the hard disk. These files are Texture and Scenery files. The Save Disk Space
	option stores the files on your hard disk in compressed mode. Compressed mode
	saves disk space, but takes longer to load or change scenery because the files
	have to be decompressed before they can be used.
	
	The Save Loading Time option decompresses and copies more the files to the hard
	drive. Decompressed files use more disk space, but take less time to load or
	change scenery.
