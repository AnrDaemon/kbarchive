## Q135608: Creative Labs SoundBlaster 32 and AWE 32 Support in Windows NT

	Article: Q135608
	Product(s): Microsoft Windows NT
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation versions 3.5, 3.51 
	- Microsoft Windows NT Server versions 3.5, 3.51 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Creative Labs now has Windows NT drivers available for the SoundBlaster 32 and
	SoundBlaster AWE 32 Wavetable sound cards. These drivers work under Windows NT
	3.5 and 3.51.
	
	MORE INFORMATION
	================
	
	You can download NTAWE32.EXE from the Creative Labs Bulletin Board Service (BBS)
	at (405) 742-6660, Creative Labs world wide web page at www.soundblaster.com, or
	from the Creative Labs ftp server at FTP.CREAF.COM. Run the self extracting
	executable file with the -d parameter.
	
	The archived file includes a README.TXT file describing the installation process
	for the driver. The SoundBlaster driver included with Windows NT needs to be
	installed first. This driver will provide the SoundBlaster 32 or SoundBlaster
	AWE 32 with basic functionality. The driver from Creative Labs provides support
	for the extended features on the SoundBlaster 32 and SoundBlaster AWE 32.
	
	The following features and utilities are not currently supported by Creative Labs
	for the SoundBlaster 32 or SoundBlaster AWE 32 under Windows NT:
	
	- Software volume control
	
	- The Text Assist utility
	
	- The Voice Assist utility
	
	For more information, contact Creative Labs Technical Support at (405) 742-6655.
