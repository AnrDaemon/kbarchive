## Q135089: Program Group Control Causes GP Fault on Windows 95 Clients

	Article: Q135089
	Product(s): Microsoft Systems Management Server
	Version(s): WINDOWS:95; winnt:1.1
	Operating System(s): 
	Keyword(s): kbnetwork kbPCM smspcm
	Last Modified: 30-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	When you run Program Group Control on a Windows 95 client, a General Protection
	(GP) Fault with an Illegal Instruction error appears.
	
	This symptom occurs sporadically when you run Program Group Control.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.1. We are researching this problem and will post new information in
	the Microsoft Knowledge Base as it becomes available.
