## Q135009: DCS EM320W Terminal Emulator Slows Down Programs and Printing

	Article: Q135009
	Product(s): Microsoft Windows NT
	Version(s): 3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.51 
	- Microsoft Windows NT Server version 3.51 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	Your programs and print jobs slow down significantly when Diversified Computer
	Systems (DCS) EM320W terminal emulator (a 16-bit Windows program) is running in
	the background.
	
	
	CAUSE
	=====
	
	Your DCS EM320W terminal emulator program prevents your other programs from
	getting CPU time.
	
	
	WORKAROUND
	==========
	
	To work around this problem, quit EM320W when you are using other programs.
	
	For additional information, call Diversified Computer Systems technical support
	at (303) 447-9251.
