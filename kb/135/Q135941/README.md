## Q135941: Write Protection Error Message with DriveSpace 3

	Article: Q135941
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	- Microsoft Plus! for Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	On a computer with a write-protected, mounted, compressed removable drive, you
	may receive a write error message on a blue screen from the compression driver.
	
	CAUSE
	=====
	
	This error occurs because the compression driver does not have write access to
	the host drive.
	
	RESOLUTION
	==========
	
	Remove the write protection from the removable drive and try the operation
	again.
