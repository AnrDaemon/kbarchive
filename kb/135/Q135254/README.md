## Q135254: How to Display the Date and Time a Message Was Sent in the Inbox

	Article: Q135254
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.2,1.3,2.0,2.5,95
	Operating System(s): 
	Keyword(s): kbui kbmsn
	Last Modified: 16-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.2, 1.3, 2.0, 2.5 
	- the operating system: Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	By default, the Microsoft Exchange Inbox displays the date and time you received
	a message, not the date and time the message was sent.
	
	MORE INFORMATION
	================
	
	If you want the date and time the message was sent to appear in the Inbox,
	follow these steps:
	
	1. In the Inbox, on the View menu, click Columns.
	
	2. In the Available Columns list, click Sent.
	
	3. Click Add, and then click OK.
	
	Additional query words: msn email e-mail
