## Q135152: Text Appears as Blocky Characters with S3 Video Card

	Article: Q135152
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are using an S3 911/924 video adapter, a line of text on the screen may
	appear as a series of blocky characters.
	
	CAUSE
	=====
	
	This problem can occur if you are using a color cursor.
	
	RESOLUTION
	==========
	
	To work around this problem, use either of the following methods:
	
	- Use only black and white cursors.
	
	- Change the graphics performance to Basic or None. To do so, follow these
	  steps:
	
	  1. In Control Panel, double click the System icon.
	
	  2. On the Performance tab, click Graphics.
	
	  3. Move the Hardware Acceleration slider all the way to the left (the None
	     position) or one notch to the right of None.
	
	  4. Click OK. Click Close. When you are prompted to restart your computer, do
	     so.
