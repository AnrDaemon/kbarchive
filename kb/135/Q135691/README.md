## Q135691: SMS: Invmac Hangs After Cold-Boot on Quadra 800 or Centra 650

	Article: Q135691
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.0
	Operating System(s): 
	Keyword(s): kbnetwork kbInventory smsinv
	Last Modified: 25-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	When you run SMS Inventory on a Apple Macintosh Quadra 800 or Centra 650, the
	inventory process stops responding (hangs) after the computer is restarted.
	Manually running INVMAC will succeed, but running it from the startup folder
	will fail while inventorying the networking components.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.0. This problem was corrected in Systems Management Server version
	1.1.
	
	Additional query words: prodsms sms cold boot
