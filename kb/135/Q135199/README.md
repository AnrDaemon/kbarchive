## Q135199: Cannot Paste Text into Paint Document

	Article: Q135199
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to copy text to a document in Paint, the Paste command may be
	unavailable.
	
	CAUSE
	=====
	
	You can paste text into a Paint document only in text insertion mode.
	
	RESOLUTION
	==========
	
	Enter text insertion mode before you try to paste the text. To do so, follow
	these steps:
	
	1. Click the Text tool in the toolbox.
	
	2. Click the document where you want to insert the text.
	
	3. Paste the text.
