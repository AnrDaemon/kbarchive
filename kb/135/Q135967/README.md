## Q135967: Matrox Display Drivers Can Cause General Protection Fault

{% raw %}

	Article: Q135967
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are using a Matrox video adapter in 1600 x 1200 resolution in high-
	color (16-bit) mode, you may receive a general protection (GP) fault in Mga.drv.
	
	RESOLUTION
	==========
	
	Change the color palette to 256 colors or less.
	
	STATUS
	======
	
	This situation is most likely caused by design changes in Windows 95. Microsoft
	has confirmed that it is not caused by a problem in Windows 95. For more
	information about resolving this issue, contact the video adapter manufacturer.
