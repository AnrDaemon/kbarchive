## Q135264: PPT7: Genigraphics Wizard Replaces Genigraphics Driver

{% raw %}

	Article: Q135264
	Product(s): Microsoft PowerPoint for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbprint kbPrinting
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft PowerPoint for Windows 95, version 7.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Genigraphics Driver and GraphicsLink programs that ship with PowerPoint
	2.0-4.0 do not ship with later versions of PowerPoint for Windows. The later
	versions of PowerPoint include a Genigraphics Wizard instead.
	
	To start the Genigraphics Wizard, click Send To Genigraphics on the File menu.
	
	For more information about the Genigraphics Wizard, click Contents And Index on
	the Help menu, click the Index tab in PowerPoint Help, type the following text
	
	  "genigraphics" (without the quotation marks)
	
	and then double-click the selected text to go to the "Prepare files for printing
	by Genigraphics or another service bureau" topic.
