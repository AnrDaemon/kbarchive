## Q135015: Explorapedia: Clearing/Removing List of User Names

	Article: Q135015
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:1.0,1.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 29-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Explorapedia series: World of Nature for Windows, versions 1.0, 1.2 
	- Microsoft Explorapedia series: World of People for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	You can easily clear the list of user names in Explorapedia by deleting a single
	file.
	
	MORE INFORMATION
	================
	
	To clear the list of user names in Explorapedia, use the following steps. These
	steps require searching for files and deleting files. For more information about
	how to accomplish these tasks in Windows, see your Windows printed documentation
	or online help.
	
	If You Are Running Explorapedia on Windows 3.x
	----------------------------------------------
	
	1. Exit Explorapedia.
	
	2. In the MSKids\Users folder, find the file named Userpref.bin.
	
	3. Delete Userpref.bin.
	
	4. Restart Explorapedia.
	
	After following these steps, there are no user names in Explorapedia.
	
	NOTE: All settings that you have made in Explorapedia, such as the color of jump
	words, are reset to their defaults by removing the Userpref.bin file.
	
	If You Are Running World of Nature version 1.2 on Windows 95
	------------------------------------------------------------
	
	NOTE: World of Nature version 1.2 has the "Designed for Microsoft Windows 95"
	logo on the front of the CD. World of Nature version 1.0 has the Microsoft
	Windows Compatible logo on the front of the CD.
	
	1. Exit Explorapedia.
	
	2. In the Program Files\Microsoft Kids\Common Files\Users folder, find the file
	  named Explora.usr.
	
	3. Delete Explora.usr.
	
	4. Restart Explorapedia.
	
	After following these steps, there are no user names in Explorapedia.
