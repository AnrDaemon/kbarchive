## Q135740: PPT7: Format Painter Doesn't Copy Custom Background in Sorter

{% raw %}

	Article: Q135740
	Product(s): Microsoft PowerPoint for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 15-APR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft PowerPoint for Windows 95, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to use Format Painter in Slide Sorter view to format the slide
	background of one slide to the format of another slide, nothing happens. This
	does work in PowerPoint for Windows, version 4.0.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
