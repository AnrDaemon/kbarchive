## Q135104: Sign-up Does Not Display the Complete Phone Number Dialed

	Article: Q135104
	Product(s): The Microsoft Network
	Version(s): 1.0,1.2,1.3,2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.0, 1.2, 1.3, 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Sign-up displays only the basic phone number, not the complete number actually
	dialed. That is, numbers to access an outside line and long-distance access
	codes, if required, are not displayed during sign-up.
	
	RESOLUTION
	==========
	
	To see the number dialed, click Settings. The number dialed is shown next to
	Calling From: Default Location.
