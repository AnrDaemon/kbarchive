## Q135125: German File Transfer Window on Clients Has English Text

	Article: Q135125
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.0
	Operating System(s): 
	Keyword(s): kbnetwork smsgeneral kbArtTypeINF
	Last Modified: 27-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	When you use the German version of Systems Management Server and use the Help
	Desk feature to transfer files to a client, a dialog box appears on the client
	that has English instead of German text.
	
	CAUSE
	=====
	
	The text in the dialog box appearing on the client is not localized to German.
	
	RESOLUTION
	==========
	
	Upgrade to Systems Management Server version 1.1.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.0. This problem has been corrected in Systems Management Server
	version 1.1.
