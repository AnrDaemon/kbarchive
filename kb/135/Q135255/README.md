## Q135255: Personal Address Book Displays At Sign (@) for Address

{% raw %}

	Article: Q135255
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.0,1.2,1.3,2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.0, 1.2, 1.3, 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you add an address of type "Internet over The Microsoft Network" to your
	personal address book, the address appears in the address book as an at sign
	(@).
	
	RESOLUTION
	==========
	
	Even though the address does not look correct, it will work correctly.
	
	Additional query words: 1.2 email e-mail
