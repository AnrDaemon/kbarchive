## Q135910: Flight Simulator: List of Sound Options

	Article: Q135910
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:5.0,5.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for MS-DOS, versions 5.0, 5.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following article contains a list of the sound options in Microsoft Flight
	Simulator.
	
	MORE INFORMATION
	================
	
	To change the current Sound option, do the following:
	
	1. Start Microsoft Flight Simulator.
	
	2. Click Options (or press the ALT+O key combination).
	
	3. Click Preferences.
	
	4. Click Sound.
	
	Flight Simulator 5.0
	--------------------
	
	PC Speaker
	Adlib/PC Speaker
	Adlib/Covox LPT1
	Adlib/Covox LPT2
	Sound Blaster
	Thunderboard
	ATI Stereo F/X
	Sound Blaster Pro
	Pro Audio Spectrum
	Microsoft Windows Sound System
	No Sound Device
	
	Flight Simulator 5.0a
	---------------------
	
	PC Speaker
	Adlib/PC Speaker
	Adlib/Covox LPT1
	Adlib/Covox LPT2
	Sound Blaster/Sound Blaster Pro
	Thunderboard
	ATI Stereo F/X
	Sound Blaster Universal
	Pro Audio Spectrum
	Microsoft Windows Sound System
	No Sound Device
	Gravis Ultrasound (SBOS)
	
	Flight Simulator 5.1

	Adlib/Covox LPT2
