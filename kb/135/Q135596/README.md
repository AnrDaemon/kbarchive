## Q135596: DOC: Controls Subdirectory Misspelled

	Article: Q135596
	Product(s): Microsoft FoxPro
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): kbdocfix
	Last Modified: 15-FEB-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual FoxPro for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The first sentence of the first paragraph on page 234 in the Visual FoxPro
	Developer's Guide states that "A sample form that demonstrates how to add items
	to a list box is available in the Visual FoxPro Samples directory as
	Contols\Lists\Ladd.scx." The subdirectory name is actually Controls.
	
	This documentation error has been fixed in Visual FoxPro 5.0 for Windows.
	
	Additional query words: 3.00 docerr VFoxWin 5.00
