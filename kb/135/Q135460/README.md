## Q135460: PC Mac: Not Able to Print Contents of Inbox

	Article: Q135460
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.x
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, version 3.x, on platform(s):
	   - the operating system: Mac OS (ALL) 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Macintosh workstation, included with version 3.2 of Microsoft Mail for PC
	Networks, can not print the contents of the Inbox to a printer or file.
	
	WORKAROUND
	==========
	
	You can do a screen dump, as explained in the following steps:
	
	1. Run the Macintosh client, and open the Inbox.
	
	2. Press the SHIFT+COMMAND+3 key combination.
	
	  You will hear the sound of a shutter click, and a file named Picture 1 will
	  appear on the root of the hard disk.
	
	3. Open this file with SimpleText or other editor. Then, paste it into a
	  document for printing.
	
	Additional query words: 3.00 3.20 client mac
