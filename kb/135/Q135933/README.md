## Q135933: Cannot Install MS Plus! on Shared or RPL System

{% raw %}

	Article: Q135933
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	- Microsoft Plus! for Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to install Microsoft Plus! for Windows 95, you may receive the
	following error message:
	
	  Setup could not open the file: <path>\<filename> <RETRY>
	  <CANCEL>
	
	CAUSE
	=====
	
	This problem can occur if Setup tries to write to a write-protected directory in
	a shared system, such as a shared Windows 95 installation or an RPL (remote
	program load) system.
	
	RESOLUTION
	==========
	
	Microsoft Plus! does not support installing to this type of configuration.
