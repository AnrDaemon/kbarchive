## Q135956: Check Box Updated Slowly When Upgrading with DriveSpace 3

{% raw %}

	Article: Q135956
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	- Microsoft Plus! for Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run DriveSpace 3 on slower computers, the Hide Host Drive check box in
	the Upgrade Options dialog box may take a couple of seconds to be updated.
	
	MORE INFORMATION
	================
	
	When you are using DriveSpace 3 to upgrade a DoubleSpace 1.0 or 2.0 compressed
	drive, if you select the Hide Host Drive check box in the Upgrade Options dialog
	box, and then click No in the Caution dialog box, it takes a couple seconds or
	longer on slower computers for the check mark to be cleared.
