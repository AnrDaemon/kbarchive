## Q135923: ES688 Sound Chip Driver Available for Toshiba Notebook Computers

	Article: Q135923
	Product(s): Microsoft Windows NT
	Version(s): 3.50 3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation versions 3.5, 3.51 
	- Microsoft Windows NT Server versions 3.5, 3.51 
	-------------------------------------------------------------------------------
	
	The following Toshiba notebook computers are manufactured with the ESS
	Technologies ES688 sound chip:
	
	  T2150CDS/CDT
	  T2400CS/CT
	  T2450CT
	  400
	  400CS/CDT
	  410
	  610
	  Protege 610CT
	  700
	
	Windows NT does not include drivers for the ES688 sound chip. To allow
	full use of sound capabilities on the Toshiba notebook computers with the
	ES688 sound chip, download 688NT35.ZIP from one of the following locations:
	
	- Toshiba Related file library (3) in the Toshiba Forum (GO TOSHIBA) on
	  CompuServe.
	
	- ESS Technologies Bulletin Board Service (BBS) at (510) 226-1937.
	
	- Toshiba Web Site at http://www.toshiba.com/tais/csd/support. Choose Software
	  and Informational Files, and then Windows NT Drivers.
	
	For more information, contact ESS Technologies Technical Support at
	(510) 226-1088.
	
	The third-party products discussed here are manufactured by ESS
	Technologies and Toshiba, vendors independent of Microsoft; we make no
	warranty, implied or otherwise, regarding these products' performance or
	reliability.
