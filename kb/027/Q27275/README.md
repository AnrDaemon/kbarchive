## Q27275: IBM Quietwriter Support in Windows

	Article: Q27275
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The IBM Quietwriter III printer is supported under Microsoft Windows. This
	printer is not supported under any earlier release of Windows.
	
	The IBM Quietwriter I and II printers are NOT supported under any version of
	Windows.
	
	The Quietwriter I, II, and III are manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding these products'
	performance or reliability.
