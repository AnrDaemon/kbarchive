## Q228775: Baseball 2000: Cannot Locate Season Database in General Manager

	Article: Q228775
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbtool kbimu msgame
	Last Modified: 19-APR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Baseball 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start General Manager (a utility included in Microsoft Baseball 2000),
	you may be unable to locate the season database.
	
	CAUSE
	=====
	
	This behavior can occur if you have not yet played Baseball 2000.
	
	RESOLUTION
	==========
	
	To prevent this issue from occurring, start and play Baseball 2000 before you
	attempt to use General Manager.
