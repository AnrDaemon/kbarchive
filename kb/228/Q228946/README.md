## Q228946: Midtown Madness: Textures Are Not Displayed When You Start Race

{% raw %}

	Article: Q228946
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgamekbfaq
	Last Modified: 20-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Midtown Madness, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start a race in Microsoft Midtown Madness, textures may not be
	displayed on the game screen.
	
	CAUSE
	=====
	
	This behavior can occur if an S3 Virge video adapter is installed in your
	computer.
	
	MORE INFORMATION
	================
	
	The S3 Virge video adapter does not support the 3D graphics required to display
	textures in Midtown Madness.
