## Q228978: MidMad: Game Unexpectedly Quits After You Start a Race

	Article: Q228978
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 22-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Midtown Madness, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you start a race in Microsoft Midtown Madness, the game may unexpectedly
	quit.
	
	CAUSE
	=====
	
	This behavior occurs if the following conditions are true:
	
	- A Matrox Millennium II video adapter is installed in the computer.
	
	- The screen resolution in the game is set to 1024 x 768 pixels.
	
	RESOLUTION
	==========
	
	To resolve this issue, lower the screen resolution in the game:
	
	1. On the Midtown Madness startup screen, click Options.
	
	2. Click Graphic Options.
	
	3. Under Resolution, click a lower screen resolution.
	
	4. Click Done.
