## Q228795: XADM: Posting to a Public Folder for Another User

	Article: Q228795
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-APR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	Posting to a public folder as another user displays "Send on Behalf of"
	information on the From line.
	
	MORE INFORMATION
	================
	
	
	This is by design to make it possible to identify who actually posts messages to
	a public folder.
