## Q225242: MDAC Folder Is Not Included in Windows NT Service Pack 5

{% raw %}

	Article: Q225242
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Server, Enterprise Edition version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft Data Access Components (MDAC) is not included in any Microsoft Windows
	NT 4.0 Service packs . Fixes for MDAC are included with Microsoft Internet
	Explorer, and can be downloaded from www.microsoft.com/data.
	
	MORE INFORMATION
	================
	
	MDAC is included on the root of the Windows NT 4.0 Service Pack 4 CD-ROM in the
	Mdac\Alpha folder or the Mdac\I386 folder.
