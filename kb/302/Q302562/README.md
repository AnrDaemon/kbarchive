## Q302562: Microsoft Natural USB Keyboard And The Compaq IA-1

{% raw %}

	Article: Q302562
	Product(s): The Microsoft Network
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You attempt to connect a Microsoft Natural USB Keyboard to the Compaq IA-1, but
	the keyboard does not function.
	
	CAUSE
	=====
	
	The Microsoft Natural USB Keyboard is not supported on the IA-1.
	
	RESOLUTION
	==========
	
	The MSN Companion product team is aware of this and is working to resolve this
	issue.

	================
