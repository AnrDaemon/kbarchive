## Q302569: Printer Stops Printing If Document Is Too Long

	Article: Q302569
	Product(s): The Microsoft Network
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SYMPTOMS
	========
	
	The printer stops printing after several pages, and returns the following
	error:
	
	"Please make sure your printer is switched on, that it is properly connected to
	your MSN Companion and that no warning lights appear on the printer. If your
	printer is in the process of warming up, try printing again."
	
	CAUSE
	=====
	
	The paper is not properly positioned in the printer's feed tray.
	
	RESOLUTION
	==========
	
	To resolve this issue
	
	1. Remove the paper from the feed tray in the printer.
	
	2. Make sure that the paper is aligned properly.
	
	3. Try to print again.
