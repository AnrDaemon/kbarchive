## Q302785: List Of Users Doesn't Display When Setting &quot;Parental Controls&quot; i

{% raw %}

	Article: Q302785
	Product(s): The Microsoft Network
	Version(s): 6.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 25-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network version 6.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Note: Parental Controls will be available in a future release of MSN Explorer.
	
	You sign in to MSN Explorer with the account manager's account (the account
	associated with a paying MSN Internet Access customer). You click Help and
	Settings, click Parental Controls, and then click Set Parental Controls for an
	Account. The list of users on the computer does not appear, even though there
	are multiple users set up on MSN Explorer on your computer.
	
	CAUSE
	=====
	
	This is caused by a temporary issue with the Parental Controls service.
	
	RESOLUTION
	==========
	
	This should be a short-term issue. Try setting the Parental Controls again
	later.
