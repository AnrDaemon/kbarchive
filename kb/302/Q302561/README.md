## Q302561: LinkSys USB 10T Ethernet Adaptor Status Light

	Article: Q302561
	Product(s): The Microsoft Network
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The status light on the LinkSys USB 10T Adaptor never comes on.
	
	MORE INFORMATION
	================
	
	MSN Companion will continue to function normally as long as the TRX/RCV light
	continues to work.
