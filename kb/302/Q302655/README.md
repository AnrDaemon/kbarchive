## Q302655: Weather On Start-Page Not Updating After Changing The Zip Code

	Article: Q302655
	Product(s): The Microsoft Network
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You change your zip code and the start page does not display the correct weather
	information for the updated zip code.
	
	CAUSE
	=====
	
	When you signed up for MSN at a retailer through Remote Account Creation (RAC),
	you specified a zip code. There is no way to change it.
	
	RESOLUTION
	==========
	
	This is a known issue and the product team is working to resolve it.
