## Q302974: &quot;Unable To Find Dial-up Credentials&quot; Error (MSN Explorer 6.1)

	Article: Q302974
	Product(s): The Microsoft Network
	Version(s): 6.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 27-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network version 6.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You attempt to sign in with MSN Explorer 6.1 and you get the error message "MSN
	Explorer is unable to find the proper MSN dial-up credentials on your computer".
	
	CAUSE
	=====
	
	Your dial-up credentials have been erased.
	
	RESOLUTION
	==========
	
	To reset the connection settings in MSN Explorer and use the default dialer.
	
	1. Start MSN Explorer.
	
	2. Click Connection Settings, click Change how you connect to the Internet, and
	  then click Continue.
	
	3. Select MSN Dial-up Internet Access, and then click Continue.
	
	4. Select the appropriate dialing options, and then click Continue.
	  MSN Explorer will now dial a 1-800 number. After you are connected, a window
	  appears stating "Please select the account you want to upgrade to use MSN
	  Internet Access". In the center of the screen is a section labeled This is
	  the e-mail account you will be upgrading. Your primary MSN Internet Access
	  account should be listed here.
	
	5. Type your password in the box, and then click Continue.
	
	6. Type your area code and telephone number in the boxes, and then click
	  Continue.
	
	7. Next to each local access number, select the Use box, and then clear the
	  boxes next to all numbers that are not local.
	
	8. Select the Area Code or Dial 1 box for each number (depending on the
	  requirements of your local phone service), and then click Continue.
	  MSN Explorer will now dial the first local access number chosen and sign you
	  in.
