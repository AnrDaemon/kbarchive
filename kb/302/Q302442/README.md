## Q302442: &quot;No Local Numbers&quot; Error When Switching Connection From LAN To M

{% raw %}

	Article: Q302442
	Product(s): The Microsoft Network
	Version(s): 6.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 20-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network version 6.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You install MSN Explorer Version 6.1,and you add a non-paying account over a
	Local Area Network (LAN) connection. Once this account is set up, add a paying
	MSN Internet Access account over the LAN connection, and click Yes when asked if
	you would also like to use MSN Internet Access. After this account is set up,
	you sign out and click Connection Settings on the sign-in screen. You click
	Change how to connect to the internet, and then click OK. You click MSN dial-up
	Internet Access, and then click OK. An error message appears stating "we didn't
	find any local numbers in your area. make sure the number you entered is
	correct", even though the number you are dialing from has not been entered.
	
	CAUSE
	=====
	
	This is a known issue.
	
	RESOLUTION
	==========
	
	To resolve this issue
	
	- When the error message appears, click OK.
	  You should now be able to enter the phone number being dialed. MSN Explorer
	  will dial a toll free number to search for local numbers in your area. You
	  can now complete Setup by following the directions.
