## Q302526: Removing A User

{% raw %}

	Article: Q302526
	Product(s): The Microsoft Network
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	You are a master user on your MSN Companion and you would like to remove a user
	from the device.
	
	MORE INFORMATION
	================
	
	1. Start MSN Companion.
	
	2. At the top of the screen, click settings.
	  The Top 10 Settings page appears.
	
	3. Select additional settings.
	  The change my MSN Companion settings page appears.
	
	4. Select remove users from MSN Companion.
	  The remove user page appears.
	
	5. Select the user you want to remove.
	  A box appears asking you to verify that you want to remove the selected user.
	
	6. Do one of the following:
	   - Click cancel to terminate the process.
	
	   - Click ok to remove the user.
