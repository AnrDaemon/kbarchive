## Q302728: &quot;View Message Source&quot; Has No Effect On Unprepared Message In MSN

	Article: Q302728
	Product(s): The Microsoft Network
	Version(s): 6.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 24-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network version 6.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You sign in to MSN Explorer 6.1, and then check the Read Mail Offline box. You
	select a message that has not been prepared for offline viewing. You receive a
	message stating "The message you want to open is not available offline." You
	click the More button, and then click View Message Source, but nothing happens.
	
	CAUSE
	=====
	
	This is a known issue.
	
	RESOLUTION
	==========
	
	If you want to view the message source, click the Send/Receive link in the body
	of the mail message to go online and download the contents of each message in
	the Inbox. This prepares the messages to be viewed offline. You can also go
	online by clicking the Go Online button in the My Stuff bar.

	================
