## Q302782: Installing MSN Explorer On Windows 95 May Disable LAN Access, In

{% raw %}

	Article: Q302782
	Product(s): The Microsoft Network
	Version(s): 6.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 25-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network version 6.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You install MSN Explorer on a Windows 95 (SR2) computer using a Local Area
	Network (LAN-ethernet) connection for your Internet connection. The LAN
	connection is no longer active, and you cannot set up MSN Explorer or use
	Internet Explorer through the LAN connection.
	
	CAUSE
	=====
	
	Installing MSN Explorer may cause conflicts with your computer's Network
	Interface Card.
	
	RESOLUTION
	==========
	
	Reinstall the drivers for the network interface card installed on the machine.
	
	Note: It is best to contact either the manufacturer of the computer or the
	manufacturer of the network card for assistance in re-installing drivers.
