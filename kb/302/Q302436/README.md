## Q302436: Japanese Version Of MSN Explorer Contains Unreadable Characters

	Article: Q302436
	Product(s): The Microsoft Network
	Version(s): 6.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 20-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network version 6.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You install the Japanese version of MSN Explorer on Windows 95 Gold with a
	version of Internet Explorer 4.0 or earlier. Some of the characters in the Setup
	interface may be unreadable.
	
	CAUSE
	=====
	
	The Japanese version of MSN Explorer uses a font that may not be available on
	computers that have Internet Explorer 4.0 installed.
	
	RESOLUTION
	==========
	
	To resolve this issue
	
	1. Download the latest Japanese version of Internet Explorer at
	  http://www.microsoft.com/Windows/IE/intl/default.asp.
	
	2. In the list, select Japanese, and then click Go.
