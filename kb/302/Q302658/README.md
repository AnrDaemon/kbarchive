## Q302658: Trouble Shooting Code - &quot;Mail&quot;

{% raw %}

	Article: Q302658
	Product(s): The Microsoft Network
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you type the code word "mail" in to the troubleshooting code word box, all
	accounts on MSN Companion will have the "mail fix" initiated.
	
	MORE INFORMATION
	================
	
	This will have no adverse effects on the accounts that are not experiencing the
	problems.
	
	Additional query words: kbimu; MSN Companion
