## Q302571: Printer Only Prints One Page

	Article: Q302571
	Product(s): The Microsoft Network
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You attempt to print an e-mail message that was forwarded to you.
	
	CAUSE
	=====
	
	The displayed frame is not captured in the print.
	
	RESOLUTION
	==========
	
	The issue will be resolved in a future MSN Companion update.
	
	Additional query words: kbimu; MSN Companion
