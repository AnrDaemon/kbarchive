## Q302565: Music.MSN.com Will Not Play On MSN Companion

	Article: Q302565
	Product(s): The Microsoft Network
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You set your browser to http://www.music.msn.com and logs in with your user name
	for that site. The site never loads a radio station.
	
	CAUSE
	=====
	
	The MSN Companion cannot is not designed to handle multiple windows and the
	music.msn.com site does not provide an error message when the Window cannot
	load.
	
	RESOLUTION
	==========
	
	This is a known issue; no resolution exists at this time.
