## Q302654: Account Settings: Adding A Child User

{% raw %}

	Article: Q302654
	Product(s): The Microsoft Network
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SYMPTOMS
	========
	
	You cannot connect through MSN Companion after setting up a child account
	through MSN Explorer.
	
	CAUSE
	=====
	
	The MSN Companion does not support Kids Passport.
	
	RESOLUTION
	==========
	
	To resolve this issue
	- Remove filtering on the Kids Passport account.
	  This is a known issue that the MSN Companion team is working to resolve.
