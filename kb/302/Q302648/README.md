## Q302648: Settings UserName

	Article: Q302648
	Product(s): The Microsoft Network
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You user tile disappears and when you position the mouse over the tile, the user
	tile box does not highlight.
	
	CAUSE
	=====
	
	You have changed your user name to include an open bracket (the character that
	appears when you press SHIFT+, or SHIFT+.).
	
	RESOLUTION
	==========
	
	Change your user name to remove the open bracket.
	
	Additional query words: kbimu; MSN Companion
