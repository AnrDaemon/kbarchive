## Q302580: Printing Epson-777 Word-95-Document

	Article: Q302580
	Product(s): The Microsoft Network
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When printing a Windows 95 Word Document with an embedded image and using an
	Epson 777 Stylus Color Printer, you receive an error message and cannot print.
	
	CAUSE
	=====
	
	This is a known issue.
	
	RESOLUTION
	==========
	
	There is no resolution at this time; the product team is working to resolve this
	issue.
