## Q302659: MSN Companion Start Page: Changing Local Information

{% raw %}

	Article: Q302659
	Product(s): The Microsoft Network
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	You would like to change the information that appears on the MSN Companion Start
	Page, to personalize or "localize" it for the area that you live in.
	
	MORE INFORMATION
	================
	
	To personalize your home page
	
	1. Set MSN Companion to http://www.msn.com.
	
	2. Click personalize your home page.
	
	3. Enter your zip code.
	  Information displayed on you MSN Companion Start Page will now be specific to
	  the area you live in.
