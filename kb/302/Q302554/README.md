## Q302554: General Information About Favorites

	Article: Q302554
	Product(s): The Microsoft Network
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	You would like to know how many Favorites MSN Companion allows per user, and
	also how many Thumbnails (a small screenshot of the Web page you selected as a
	Favorite) MSN Companion allows.
	
	MORE INFORMATION
	================
	
	Each MSN Companion can have up to nine users, and each user can have up to 50
	favorites.
	
	Each MSN Companion can have a total of 40 Thumbnails. The 40 are assigned the
	following way:
	
	- The first 40 favorites are represented by a corresponding Thumbnail,
	  regardless of which user selects the Favorites.
	
	- If the you delete a Favorite that has a Thumbnail, the next Favorite you
	  choose will get a Thumbnail.
