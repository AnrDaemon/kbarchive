## Q302791: Windows XP Beta 2 Users May Lose MSN Explorer Favorites When Usi

	Article: Q302791
	Product(s): The Microsoft Network
	Version(s): 6.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network version 6.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you use the Windows XP, Beta 2 version, and you use MSN Explorer on the
	Windows XP computer and on a computer with an earlier version of Windows
	installed, you may lose your Favorites in MSN Explorer.
	
	CAUSE
	=====
	
	This is a known issue.
	
	RESOLUTION
	==========
	
	This issue will be resolved in future versions of Windows XP.
	
	MORE INFORMATION
	================
	
	
	Additional query words: kbimu; MSN Explorer
