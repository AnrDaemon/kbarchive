## Q133210: Adding Note Information to Fax Cover Page

{% raw %}

	Article: Q133210
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	You can insert a note field on a fax cover page only by using the Compose New
	Fax Wizard. You cannot add note information to a fax cover page in Microsoft
	Exchange.
	
	MORE INFORMATION
	================
	
	If you need to add note information to a fax cover page, run the Compose New Fax
	Wizard, rather than composing the cover page in Microsoft Exchange. To do so,
	click the Start button, point to Programs, point to Accessories, point to Fax,
	and then click Compose New Fax.
