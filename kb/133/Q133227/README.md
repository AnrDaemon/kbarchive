## Q133227: PPT7: Presentation Conference Doesn't End for Audience Member

	Article: Q133227
	Product(s): Microsoft PowerPoint for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): kbnetwork kbualink97
	Last Modified: 15-APR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft PowerPoint for Windows 95, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you participate as an audience member in a Presentation Conference using
	PowerPoint for Windows 95, version 7.0, the presentation may not end for you
	when the presenter ends the conference. Instead, the last slide shown may remain
	on your screen.
	
	This can occur if the presenter added your computer first to the list of
	participants in the Presentation Conference Wizard.
	
	WORKAROUND
	==========
	
	You can end the presentation yourself by clicking the Popup menu button and then
	clicking End Show.
	
	Additional query words: 7.00 ppt95 moderator presentation conferencing add-in stuck remains fails
