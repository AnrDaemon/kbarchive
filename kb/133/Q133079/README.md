## Q133079: Blank Page Previewing One-Page Document

	Article: Q133079
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you preview a one-page fax in Fax Viewer, or a one-page document in
	WordPad, two pages appear. The second page is blank.
	
	CAUSE
	=====
	
	This behavior can occur if you previously previewed a multiple-page document in
	Fax Viewer or WordPad and chose to view two pages at a time. When you preview a
	one-page document with the preview mode set to Two Page, two pages appear. The
	second page is blank and the One Page button is unavailable.
	
	RESOLUTION
	==========
	
	To work around this problem, open a multiple-page document in Fax View or
	WordPad and preview the document. Click the One Page button, then close the
	preview. Open the one-page document and preview it.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Windows 95. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
