## Q133170: Encarta: Incorrect Font is Displayed

{% raw %}

	Article: Q133170
	Product(s): Microsoft Home Kids Products
	Version(s): 1995 edition,1996 edition; WINDOWS:
	Operating System(s): 
	Keyword(s): win31
	Last Modified: 26-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia for Macintosh 
	- Microsoft Encarta 96 Encyclopedia for Windows 
	- Microsoft Encarta 97 Encyclopedia for Windows 
	- Microsoft Encarta Encyclopedia 97 Deluxe for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run Microsoft Encarta, the Title Bar and other sections may display an
	incorrect font.
	
	RESOLUTION
	==========
	
	Run the Setup program for Encarta and click the Remove All option. Then run
	Setup again to reinstall Encarta. This updates missing fonts.
	
	MORE INFORMATION
	================
	
	During Setup, Encarta installs the following fonts to your system:
	
	  Arial             (TrueType)
	  MS Reference 1    (TrueType)
	  MS Reference 2    (TrueType)
	  Phonetic
	  Symbol            (TrueType)
	
	Encarta requires these fonts in addition to Times New Roman (TrueType), supplied
	by Microsoft Windows, to correctly display all Encarta type elements.
	
	If one or more of these fonts is missing, Encarta substitutes another available
	font.
	
	If this occurs, a section may display in an Italic, Bold, or Symbol type font.
	
	By removing Encarta before you reinstall, Setup is forced to check Encarta system
	files. Any missing system files located on the Encarta CD, including fonts, that
	are needed to run the program correctly will be reinstalled.
