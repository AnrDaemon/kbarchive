## Q133168: Mac Ancient Lands: Folder not Removed By Installer

	Article: Q133168
	Product(s): Microsoft Home Multimedia Titles
	Version(s): MACINTOSH:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Ancient Lands for the Macintosh, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the Remove option in the Ancient Lands Installer to remove the
	program from your computer, an empty Ancient Lands folder remains on your hard
	drive.
	
	RESOLUTION
	==========
	
	Drag the Ancient Lands folder to the Trash. Empty the Trash.
	
	MORE INFORMATION
	================
	
	To remove Ancient Lands using the Installer Remove function, do the following:
	
	1. Insert the Ancient Lands CD-ROM into your CD-ROM drive.
	
	2. Double-click the Installer.
	
	3. Click Customize.
	
	4. Highlight the items want to remove. To select multiple items, press and hold
	  down the SHIFT key while you click items.
	
	5. Press and hold down the OPTION key. The Install button changes to a Remove
	  button.
	
	6. Click Remove.
	
	7. Remove the empty Ancients Lands folder using the steps given in the
	  Resolution section of this article.
