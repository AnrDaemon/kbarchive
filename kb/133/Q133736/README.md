## Q133736: PPT7: Error Message: L.ppt - Sorry, the File Cannot Be Found

	Article: Q133736
	Product(s): Microsoft PowerPoint for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 27-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft PowerPoint for Windows 95, version 7.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you drag a shortcut icon for a presentation on top of an icon for the
	PowerPoint Viewer, whether it is a copy of the Viewer file or a shortcut to the
	Viewer, you will receive the following error:
	
	  L.ppt
	
	  Sorry, the file cannot be found.
	
	WORKAROUND
	==========
	
	Method 1
	--------
	
	Drag the icon for the presentation itself onto the icon for the Viewer.
	
	Method 2
	--------
	
	Install the 32-bit PowerPoint Viewer. This viewer runs with Windows 95 or Windows
	NT 3.51 and allows you to view PowerPoint presentations without having
	PowerPoint installed.
	
	To Obtain PPTVW32.EXE
	---------------------
	
	The following file is available for download from the Microsoft Download Center:
	
	  Pptvw32.exe
	  (http://download.microsoft.com/download/powerpoint95/viewer1/1/WIN98/EN-US/pptvw32.exe)
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
	
	Microsoft used the most current virus detection software available on the date of
	posting to scan this file for viruses. Once posted, the file is housed on secure
	servers that prevent any unauthorized changes to the file.
