## Q133728: Cannot Access System Icons on Taskbar Without Pointing Device

	Article: Q133728
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You are unable to access system icons on the taskbar without using a pointing
	device (such as a Microsoft Mouse).
	
	RESOLUTION
	==========
	
	You can use the MouseKeys option to move the mouse pointer, click, double-
	click, and drag items using keys on the numeric keypad. To enable the MouseKeys
	option, follow these steps:
	
	1. Click the Start button, then click Help.
	
	2. On the Index tab, type "mousekeys, turning on" (without the quotation marks).
	
	3. Click Display, and then follow the instructions in the Help topic.
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
	
	MORE INFORMATION
	================
	
	You may be able to access system icons on the taskbar using the keyboard by
	repeating the steps you used to start the program. When you try to restart the
	program, the version already running as a system icon should become active.
