## Q133091: Mac Bookshelf 1995: Cannot Find Demo

	Article: Q133091
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1995 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for the Macintosh 1995 edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Bookshelf CD-ROM jewel case booklet contains incorrect information about a
	program demo. The instructions recommend using the demo to familiarize yourself
	with Bookshelf, however, Bookshelf does not include a demo.
