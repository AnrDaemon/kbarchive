## Q133232: Dogs: Pressing Enter in Find List Doesn't Jump to Page

	Article: Q133232
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Dogs for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are in the Dogs Index, if press ENTER while an item in the Find drop
	down list is highlighted, Microsoft Dogs does not jump to the appropriate page.
	
	RESOLUTION
	==========
	
	Click OK to jump to the appropriate page.
	
	MORE INFORMATION
	================
	
	Pressing ENTER only works if the Find dialog box is the active area. After a
	selection has been selected, click one time within the Find dialog box, then
	press ENTER to jump to the desired selection.
