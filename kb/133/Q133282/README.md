## Q133282: Favorites Not Shared Between Internet Explorer and MS Network

	Article: Q133282
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.0,1.05,1.2,1.3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.0, 1.05, 1.2, 1.3 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you add a location to Favorites in Internet Explorer, the location does not
	show up in the Favorite Places folder in The Microsoft Network.
	
	CAUSE
	=====
	
	Favorite locations for Internet Explorer and The Microsoft Network are
	maintained separately. When you add a location to one, it does not appear in the
	other.
