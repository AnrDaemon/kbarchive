## Q133408: Upgrading Windows NT Resets Boot Menu Timer

{% raw %}

	Article: Q133408
	Product(s): Microsoft Windows NT
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server versions 3.5, 3.51 
	- Microsoft Windows NT Workstation versions 3.5, 3.51 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you upgrade your Windows NT installation to a later version of Windows NT,
	your customized boot menu time-out period is reset to the default of 30 seconds.
	
	WORKAROUND
	==========
	
	To work around this problem, run Control Panel, choose System, and change the
	Show List For <> Seconds setting.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT versions 3.5 and
	3.51. We are researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
