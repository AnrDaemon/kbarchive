## Q133222: Zypcom Z34-sx v.34 Modem Does Not Work as Class 1 Fax

	Article: Q133222
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Zypcom Z34-sx (ZC1101) v.34 modem fails to receive either class 1 or class 2
	faxes.
	
	CAUSE
	=====
	
	This problem occurs because the modem requires an EPROM upgrade.
	
	RESOLUTION
	==========
	
	To resolve this problem upgrade your modem's EPROM to v1.07 or later. For
	instructions on how to do this, please contact Zypcom.
