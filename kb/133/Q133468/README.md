## Q133468: Julia Child: Find Dialog Box Remains in Foreground

	Article: Q133468
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Julia Child: Home Cooking with Master Chefs for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the Index option Find feature to search on Topics in Julia Child,
	the Find dialog box remains in the foreground on all of the Topics except for
	Tips, Ingredients and Cookware. But when you close a Tip, Ingredient, or
	Cookware screen, the Find dialog box reappears.
	
	The find dialog box does not reappear with Chefs, Recipes, Lessons or Julia's
	Kitchen.
	
	RESOLUTION
	==========
	
	To return to the Find feature for Chefs, Recipes, Lessons or Julia's Kitchen,
	you must close the Find dialog box, then press and hold down the CTRL key while
	you press the F key.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Julia Child: Home Cooking with
	the Master Chef's version 1.0 for Windows. We are researching this problem and
	will post new information here in the Microsoft Knowledge Base as it becomes
	available.
