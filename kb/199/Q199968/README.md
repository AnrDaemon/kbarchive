## Q199968: XFOR: Remote Admin of Internet Mail Service Loses RAS Connection

	Article: Q199968
	Product(s): Microsoft Exchange
	Version(s): winnt:5.0
	Operating System(s): 
	Keyword(s): exc5
	Last Modified: 17-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	If you remotely administer a Microsoft Exchange Server version 5.0 Internet Mail
	Service, you may lose any Remote Access Service (RAS) connection information
	that you previously configured.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server
	version 5.0.
