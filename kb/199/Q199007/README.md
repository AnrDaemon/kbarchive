## Q199007: EVG99: Different Graph Scale Displayed When Statistic Map Opened

	Article: Q199007
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbtool kbui kbimu
	Last Modified: 25-JAN-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Virtual Globe 99, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you add a statistic map from the Statistics Center to the Favorites menu in
	Microsoft Encarta Virtual Globe 99, a different graph scale may be displayed
	when you open the map from the Favorites menu.
	
	CAUSE
	=====
	
	This behavior can occur because Encarta Virtual Globe 99 does not save the graph
	scale information with the statistic map when you add the map to the Favorites
	menu from the Statistics Center.
	
	RESOLUTION
	==========
	
	To work around this issue, click the icon next to Linear By Rank or Logarithmic
	By Rank to toggle between the available graph scales.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Encarta Virtual Globe
	99, version 1.0.
