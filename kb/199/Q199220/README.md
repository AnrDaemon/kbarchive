## Q199220: Err Msg: The System Cannot Find the Path Specified

{% raw %}

	Article: Q199220
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-AUG-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0, Terminal Server Edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to connect to a Terminal Server computer with the RDP client,
	the following error is displayed, and the client is disconnected:
	
	  The system can not find the path specified.
	  Failing component path
	
	
	RESOLUTION
	==========
	
	To resolve this problem, specify an existing path for the working directory of a
	Terminal Server Client Connection.
