## Q199857: Return of Arcade: Game Quits and You Are Returned to Desktop

	Article: Q199857
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbimu msgame
	Last Modified: 09-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Return of Arcade for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click any of the games included in Microsoft Return of Arcade, the game
	does not start and you are returned to the Windows desktop.
	
	NOTE: The following games are included in Return of Arcade:
	
	- Dig Dug
	- Galaxian
	- Pac Man
	- Pole Position
	
	CAUSE
	=====
	
	This behavior can occur if you are running Cybermedia's Guard Dog or Compaq's
	Watch Dog software on your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, quit Guard Dog or Watch Dog before you start any of the
	games included in Return of Arcade. To do this, follow these steps:
	
	1. Press CTRL+ALT+DELETE to open the Close Program dialog box.
	
	2. In the list of running programs, click Guard Dog or Watch Dog, and then click
	  End Task.
	
	3. When you are prompted to confirm that you want to quit Guard Dog or Watch
	  Dog, click Yes.
	
	NOTE: Both Guard Dog and Watch Dog are started again the next time you start
	Windows.
