## Q199093: XIMS: Performing WHOIS on Invisible User Returns No Information

	Article: Q199093
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-APR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When performing a WHOIS on a user who is invisible and connected to a Microsoft
	Exchange Chat Server, no information on that user is returned.
	
	STATUS
	======
	
	This is by product design.
