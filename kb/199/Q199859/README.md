## Q199859: Combat Flight Simulator: Computer Hangs After EULA Is Displayed

	Article: Q199859
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbenv kbsetup kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator: WWII Europe Series, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to install Microsoft Combat Flight Simulator, your computer may
	stop responding (hang) after the installation program displays the End User
	License Agreement (EULA).
	
	CAUSE
	=====
	
	This behavior can occur if the microprocessor (CPU) installed in your computer
	does not meet the minimum system requirements needed to play Combat Flight
	Simulator.
	
	MORE INFORMATION
	================
	
	To play Combat Flight Simulator, a Pentium 133 MHz or faster CPU must be
	installed in your computer.
	
	For a list of the minimum system requirements needed to play Combat Flight
	Simulator, please see the following article in the Microsoft Knowledge Base:
	
	Combat Flight Simulator: Minimum System Requirements
