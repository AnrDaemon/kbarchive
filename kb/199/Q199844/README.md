## Q199844: FS98/2000: No Lessons Are Displayed in the Lessons List

{% raw %}

	Article: Q199844
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbsetup kbui fs98 kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator 2000 
	- Microsoft Flight Simulator 2000 Professional Edition 
	- Microsoft Flight Simulator 98 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Microsoft Flight Simulator 98 or Microsoft Flight Simulator 2000, when you
	click Learn to Fly and then click Lessons, no lessons may be displayed in the
	Lessons list.
	
	CAUSE
	=====
	
	This behavior can occur if you selected the Custom installation option and did
	not choose to install the lessons.
	
	RESOLUTION
	==========
	
	To resolve this issue, remove and reinstall Flight Simulator 98 or Microsoft
	Flight Simulator 2000 using the Typical installation option. To do this:
	
	1. Click Start, point to Settings, and then click Control Panel.
	
	2. Double-click Add/Remove Programs.
	
	3. On the Install/Uninstall tab, click Microsoft Flight Simulator 98 or
	  Microsoft Flight Simulator 2000, and then click Add/Remove.
	
	4. When you are prompted to remove Microsoft Flight Simulator, click OK.
	
	5. Reinstall Flight Simulator 98 or Flight Simulator 2000. When you are prompted
	  to select an installation option, click Typical.
	
	6. Follow the instructions on the screen to install the program.
