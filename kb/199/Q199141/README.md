## Q199141: XCON: Creating Connector Disks from Select CD Subscription

	Article: Q199141
	Product(s): Microsoft Exchange
	Version(s): winnt:4.0,5.0,5.5
	Operating System(s): 
	Keyword(s): exc4 exc5 exc55
	Last Modified: 04-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 4.0, 5.0, 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Customers with Select CD subscriptions install the Exchange Server Standard
	version. They have the ability to create the connector disks, which allows them
	to install some or all connectors as needed for their organization.
	
	MORE INFORMATION
	================
	
	To create the connector disk:
	
	1. Insert the May 1996 Complete Release Burgundy CD from Server Pool PC Server 4
	  Intl English into your CD-ROM drive.
	
	2. Select the connector you need from the options for connectors which are:
	
	   - Exchange Connector (MIPS/INTEL/ALPHA)
	
	   - Exchange Internet Mail Connector (MIPS/INTEL/ALPHA)
	
	   - Exchange X.400 Connector (MIPS/INTEL/ALPHA)
	
	Each connector requires one 1.44 MB formatted floppy disk.
	
	NOTES: The Exchange Connector disk applies to Exchange Server versions 4.0 and
	5.0. The Exchange Internet Mail Connector disk applies only to Exchange Server
	version 4.0. The Exchange X.400 Connector disk applies only to Exchange Server
	version 5.5. None of these disks apply to Exchange Server Enterprise Edition for
	any version.
