## Q199125: XCON: Connectors Included with Microsoft Exchange Server 5.5

{% raw %}

	Article: Q199125
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following connectors are included in Exchange Server 5.5 Single Server
	Edition:
	
	- Connector for Lotus cc:Mail
	- Connector for Lotus Notes
	- Directory Replication Connector
	- Dynamic RAS Connector
	- Internet Mail Service
	- Internet News Service/NNTP
	- Message Transfer Agent
	- Microsoft Mail Connector (includes both PC Mail and AppleTalk Mail)
	- Site Connector
	
	In addition to the connectors listed above, the following connectors are included
	with Exchange Server 5.5 Enterprise Edition:
	
	- Connector for SNADS
	- Connector for IBM OfficeVision/VM
	- X.400 Connector
	
	MORE INFORMATION
	================
	
	For more information regarding configuring and troubleshooting these connectors
	and other Exchange Server components, please refer to Books Online included with
	Exchange Server, the additional documentation included on the Exchange Server
	CD-ROM, and the Knowledge Base included with TechNet and at our Support Online
	Web site located on the Internet at http://msdn.microsoft.com/support.
