## Q199438: WD97: Cannot Add Vertical Bar Tab from Ruler

	Article: Q199438
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): word97 kbformat
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	You cannot add vertical bar tabs from the ruler in Microsoft Word.
	
	WORKAROUND
	==========
	
	To add a vertical bar tab, follow these steps:
	
	1. On the Format menu, click Tabs.
	
	2. Type a position in the Tab stop position text box.
	
	3. Under Alignment, click Bar.
	
	4. Click OK.
