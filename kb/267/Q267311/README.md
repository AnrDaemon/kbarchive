## Q267311: MapPoint 2000: Minimum System Requirements

{% raw %}

	Article: Q267311
	Product(s): Microsoft Automap
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbref kbimu
	Last Modified: 14-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MapPoint 2000 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains a list of the minimum system requirements that are needed
	to run Microsoft MapPoint 2000.
	
	MORE INFORMATION
	================
	
	To run Microsoft MapPoint 2000, you need:
	
	- Multimedia PC with a Pentium 90 MHz or higher processor
	- Microsoft Windows 95 or later operating system, or Microsoft Windows NT
	  Workstation operating system version 4.0 or later with Service Pack 3 or
	  later
	- 20 megabytes (MB) of RAM for Windows 95 or Windows 98; 36 MB of RAM for
	  Windows NT Workstation version 4.0 or later
	- 135 megabytes of available hard disk space (your hard disk usage will vary
	  depending on configuration)
	- Quad-speed or faster CD-ROM drive.
	- Super VGA, 256-color monitor supporting 640 x 480 or higher resolution
	- Microsoft Mouse, Microsoft IntelliMouse, or compatible pointing
	  device
	
	Additional items or services required to use certain features:
	
	- 9600 baud modem; 14,400 or higher-baud modem recommended
	- Microsoft Mail, Microsoft Exchange, or other MAPI-compliant messaging
	  software required to use SendTo
	- Access to and use of the Internet require payment of a separate fee to an
	  Internet service provider and local telephone charges may apply.
	
	NOTE: Integrating data into Microsoft Office programs requires 97 or later
	versions of Microsoft Access, Excel, FrontPage, Outlook, PowerPoint, Publisher,
	or Word.
	
	Integrating data from Microsoft Office programs requires 97 or later versions of
	Microsoft Access, Excel, and Outlook; or SQL Server 6.5 or later.
