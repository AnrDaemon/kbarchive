## Q218999: WD97: Cannot Select an Object Hidden Behind Another Object

	Article: Q218999
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta word8 word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When a picture or object has been sent behind another object, you may be unable
	to click the picture or object to select it.
	
	CAUSE
	=====
	
	The picture or object may be smaller than the object that covers it and cannot
	be selected either by clicking it or by using the Select Objects button on the
	Drawing toolbar.
	
	RESOLUTION
	==========
	
	To select and move the hidden picture or object, use the following steps:
	
	1. On the View menu, point to Toolbars and click Drawing.
	
	2. Click the picture or object that is covering the object you want to move and
	  then press TAB or SHIFT+TAB until the picture or object you want to move is
	  selected.
	
	3. On the Drawing toolbar, click Draw, point to Order, and then click Bring
	  Forward.
	
	MORE INFORMATION
	================
	
	For more information about positioning graphics, click "Contents and Index" on
	the Help menu, click the Index tab in Word Help, type the following text
	
	  drawing objects, stacking
	
	and then double-click the selected text to go to the "Bring a drawing object to
	the front" topic. If you are unable to find the information you need, ask the
	Office Assistant.
