## Q239863: NFL Fever 2000: Cannot Select Resolution Higher Than 640 x 480

	Article: Q239863
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 27-AUG-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NFL Fever 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft NFL Fever 2000, you may be unable to select a screen
	resolution higher than 640 x 480 pixels in the game. In addition, the only
	available Texture Detail setting may be Low.
	
	CAUSE
	=====
	
	This behavior can occur if the following conditions are true:
	
	- An ATI Rage Pro AGP video adapter is installed in your computer.
	
	- You play NFL Fever 2000 on a Microsoft Windows 95-based computer.
