## Q239607: World Atlas: Error Message on Blue Screen When You Start Program

	Article: Q239607
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbenv kbimu
	Last Modified: 25-JUN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Interactive World Atlas 2000 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start Microsoft Encarta World Atlas 2000, you may receive an error
	message on a blue screen and the program may quit unexpectedly.
	
	CAUSE
	=====
	
	This behavior can occur if the following conditions are true:
	
	- An ATI Rage 3D video adapter is installed in your computer.
	
	- The video driver for the ATI Rage 3D video adapter is outdated.
	
	- The Microsoft Windows color palette setting is set to High Color or True
	  Color.
	
	RESOLUTION
	==========
	
	To work around this issue, reduce the Windows color palette setting to 256
	colors:
	
	1. Click Start, point to Settings, and then click Control Panel.
	
	2. Double-click Display.
	
	3. Click the Settings tab.
	
	4. In the Color Palette or Colors box, click 256 Colors.
	
	5. Click OK, and then click OK again.
	
	6. Close Control Panel.
