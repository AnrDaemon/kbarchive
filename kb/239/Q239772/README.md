## Q239772: NFL Fever 2000: Unable to Pause a Game or a Practice

{% raw %}

	Article: Q239772
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbui kbimu msgame kbFAQ
	Last Modified: 28-AUG-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NFL Fever 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft NFL Fever 2000, you may be unable to pause a game or a
	practice. As a result, you may be unable to gain access to the Paused Game menu.
	
	CAUSE
	=====
	
	This behavior can occur if the following conditions are true:
	
	- Your game pad does not have a Start button.
	
	- You are playing the game or the practice in two-player mode.
	
	RESOLUTION
	==========
	
	To work around this issue, play the game or the practice in one-player mode and
	press ESC to pause the game.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft NFL Fever 2000.
	
	MORE INFORMATION
	================
	
	NFL Fever 2000 is designed to be played using a game pad with at least six
	buttons. For optimum performance, an eight-button game pad is recommended.
