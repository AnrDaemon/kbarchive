## Q239123: Flight Simulator 5.1: How to Create and Use a Boot Disk

	Article: Q239123
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:5.1
	Operating System(s): 
	Keyword(s): kbenv kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for MS-DOS, version 5.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Flight Simulator for MS-DOS, version 5.1, you may
	experience any of the following symptoms:
	
	- You may receive insufficient memory errors.
	
	- The frame rate (the rate at which images or frames are drawn on the screen)
	  may be very slow.
	
	- Video playback may be jerky or uneven.
	
	CAUSE
	=====
	
	This behavior can occur if you do not have sufficient system memory available.
	
	RESOLUTION
	==========
	
	To resolve this issue, create a boot disk for use with Flight Simulator 5.1. To
	do this:
	
	1. Insert a floppy disk into the floppy disk drive in your computer.
	
	2. Click Start, point to Find, and then click "Files or Folders."
	
	3. In the Named box, type "Fssystem.exe" (without the quotation marks).
	
	4. In the "Look in" box, click My Computer, and then click Find Now.
	
	5. In the list of found files, double-click Fssystem.exe or Fssystem.
	
	6. Follow the instructions on the screen to create the boot disk.
	
	NOTE: You may need to install MS-DOS drivers for your CD-ROM drive and pointing
	device on the boot disk. For information about how to do this, contact your
	hardware manufacturer.
	
	MORE INFORMATION
	================
	
	To use the Flight Simulator 5.1 boot disk:
	
	1. Shut down and turn off the computer.
	
	2. Insert the boot disk into the floppy disk drive, and then turn the computer
	  on.
	
	3. When the menu appears, press 1 to run Flight Simulator 5.1.
