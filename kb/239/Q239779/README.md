## Q239779: NFL Fever 2000: Context-Sensitive Help Text Is Invisible

{% raw %}

	Article: Q239779
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbenv kbimukbfaq
	Last Modified: 29-AUG-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NFL Fever 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	On the Microsoft NFL Fever 2000 Main screen, all of the text displayed inside a
	context-sensitive Help window may be invisible, or the links to other Help
	topics may be invisible.
	
	CAUSE
	=====
	
	This behavior occurs if the Window color in the Display tool in Control Panel is
	set to light gray or dark green.
	
	RESOLUTION
	==========
	
	To resolve this issue:
	
	1. Quit NFL Fever 2000 if it is running.
	
	2. Click Start, point to Settings, and then click Control Panel.
	
	3. Double-click Display.
	
	4. Click the Appearance tab.
	
	5. In the Item box, click Window.
	
	6. In the Color box next to the Item box, select a color other than light gray
	  or dark green..
	
	7. Click OK.
	
	8. Close Control Panel.
	
	9. Start NFL Fever 2000.
