## Q239882: NFL Fever 2000: Graphics in the Game Do Not Appear Smooth

	Article: Q239882
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 19-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NFL Fever 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft NFL Fever 2000, the graphics in the game may not appear
	smooth.
	
	CAUSE
	=====
	
	This behavior can occur if the video adapter installed in your computer is based
	on the S3 Virge VX chip set.
	
	RESOLUTION
	==========
	
	To work around this issue, run the game in software mode:
	
	1. Start NFL Fever 2000.
	
	2. On the Main screen, click Options, and then click Display.
	
	3. Under Driver, click to select Software.
	
	4. Click OK.
