## Q239811: NBA/NFL 2000: 3D Rendering Is Unavailable

{% raw %}

	Article: Q239811
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame kbFAQ
	Last Modified: 28-AUG-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NBA Inside Drive 2000, version 1.0 
	- Microsoft NFL Fever 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft NBA Inside Drive 2000 or Microsoft NFL Fever 2000, 3D
	hardware rendering may be unavailable.
	
	CAUSE
	=====
	
	This behavior can occur if the following conditions are true:
	
	- A Diamond Viper V330 video adapter is installed in your computer.
	
	- The latest video driver for the Diamond Viper V330 video adapter is not
	  installed on your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest video driver for your
	video adapter from the following Diamond Multimedia Web site:
	
	  http://www.diamondmm.com
	
	MORE INFORMATION
	================
	
	If the issue continues to occur, download and install the latest Riva 128
	reference driver from the following nVidia Web site:
	
	  http://www.nvidia.com
