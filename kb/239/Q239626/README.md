## Q239626: World Atlas 2000: Quits When You Click a Media Viewer Thumbnail

	Article: Q239626
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kberrmsg kbui kbimu
	Last Modified: 25-JUN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Interactive World Atlas 2000 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click a Media Viewer thumbnail image in Microsoft Encarta World Atlas
	2000, you may experience one of the following symptoms:
	
	- Encarta World Atlas 2000 may quit unexpectedly.
	
	- You may receive an error message similar to the following:
	
	  Encarta Interactive World Atlas is unable to continue running. The problem
	  may be the result of low system resources such as memory or disk space. Try
	  closing other running applications or freeing up disk space before restarting
	  Encarta Interactive World Atlas.
	
	CAUSE
	=====
	
	This behavior can occur if the Statistics Center window is open when you click
	the Media Viewer thumbnail image.
	
	RESOLUTION
	==========
	
	To prevent this issue from occurring, use one of the following methods:
	
	- Close Media Viewer before you open Statistics Center.
	
	- Close Statistics Center before you open Media Viewer.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Encarta World Atlas
	2000.
