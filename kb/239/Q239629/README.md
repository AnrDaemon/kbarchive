## Q239629: World Atlas: Setup Quits Unexpectedly During CD-ROM Verification

	Article: Q239629
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kb3rdparty kbsetup kbimu
	Last Modified: 25-JUN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Interactive World Atlas 2000 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you install Microsoft Encarta World Atlas 2000, the installation program
	may quit unexpectedly immediately after you receive the following message:
	
	  Verification in progress. This could take a few minutes. Please wait.
	
	CAUSE
	=====
	
	This behavior can occur if Mindspring Internet software is installed on your
	computer.
	
	Mindspring Internet software includes the Armon32.exe program that can interfere
	with the Encarta World Atlas 2000 CD-ROM verification process.
	
	RESOLUTION
	==========
	
	To resolve this issue, quit Armon32.exe before you install Encarta World Atlas
	2000:
	
	1. Press CTRL+ALT+DELETE.
	
	2. In the Close Program dialog box, click Armon32, and then click End Task.
	
	  If you receive a message that the program is busy or not responding, click End
	  Task again.
	
	3. Install Encarta World Atlas 2000.
