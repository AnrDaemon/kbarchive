## Q239800: Attachmate ISCA Adapter Support

	SUMMARY
	=======
	
	Attachmate has discontinued support for the Attachmate ISCA adapter (also known
	as the DCA ISCA Adapter) and has replaced it with the Advanced ISCA Plug and
	Play Adapter for ISA and PCI bus machines.
	
	MORE INFORMATION
	================
	
	Microsoft recommends using the Attachmate Advanced ISCA Adapter driver included
	in SNA Server 4.0 with Service Pack 2 on Pentium Processor computers. For these
	reasons, Microsoft will provide support for the Attachmate ISCA adapter included
	in Microsoft SNA Server versions 2.11, 2.11 SP1, 2.11 SP2, 3.0, 3.0 SP1, 3.0
	SP2, 3.0 SP3, 4.0, 4.0 SP1, and 4.0 SP2.
