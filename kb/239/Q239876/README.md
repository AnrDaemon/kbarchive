## Q239876: NFL Fever 2000: Main Screen Is Not Centered on Your Monitor

	Article: Q239876
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 18-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NFL Fever 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start Microsoft NFL Fever 2000, the Main screen may not be centered on
	your monitor.
	
	CAUSE
	=====
	
	This behavior can occur if the following conditions are true:
	
	- A Creative Labs Graphics Blaster Exxtreme video adapter is installed in your
	  computer.
	
	- The video driver for the Creative Labs Graphics Blaster Exxtreme video
	  adapter is outdated.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest video driver for the
	Creative Labs Graphics Blaster Exxtreme video adapter from the following
	Creative Labs Web site:
	
	  http://support.soundblaster.com/files/download.asp
