## Q64823: Windows 3.0 and Award BIOS for 286/386 Computers

	Article: Q64823
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	According to Award, the following versions of the 286/386 BIOS work correctly
	with Microsoft Windows version 3.0:
	
	- 3.05 (may not work correctly in all situations)
	
	- 3.1x (where x refers to machine-specific versions)
	
	MORE INFORMATION
	================
	
	You can upgrade your BIOS by contacting Award Software or a regional Award
	distributor.

	we make no warranty, implied or otherwise, regarding these products' performance
	or reliability.
