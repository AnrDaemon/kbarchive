## Q64269: Windows Accessories Do Not Save Page Settings

	Article: Q64269
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): win31
	Last Modified: 17-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you choose Page Setup from the File menu in Notepad, Cardfile, Calendar, or
	Paintbrush, the settings are not saved when the file is saved.
	
	CAUSE
	=====
	
	All accessory applications use a flat-file format and the formatting information
	is not saved with the file; therefore, it is not shown with its last settings
	when the file is opened.
	
	MORE INFORMATION
	================
	
	Note that this information applies to the version of Notepad included with
	Microsoft Windows 95 also.
