## Q64264: No Mouse in Windows 3.0 with DEC Mouse

	Article: Q64264
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the Digital Equipment Corporation (DEC) mouse with Microsoft
	Windows version 3.0, no mouse pointer appears on the screen.
	
	RESOLUTION
	==========
	
	You need the new drivers supplied by DEC Technical Support. They currently have
	drivers for the following Digital Hardware:
	
	  DEC LK250 Keyboard
	  DEC Mouse
	  DEC Pathworks Network
	
	To obtain these drivers, contact DEC Technical Support.
