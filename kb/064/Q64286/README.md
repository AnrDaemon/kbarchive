## Q64286: Crash Indicator Bar Remaining in Hangar Buzz Mode

	Article: Q64286
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	When playing the Hangar Buzz mode in Flight Simulator version 4.00, the crash
	indicator bar may remain on the screen after the plane crashes and the mode is
	reset.
	
	Press the ESC key twice to remove the crash indicator bar and return the screen
	to normal.
