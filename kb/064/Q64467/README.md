## Q64467: TI Omnilaser 2115 Requires Firmware Version 2.613 or 2.635

{% raw %}

	Article: Q64467
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	You need Firmware version 2.613 or 2.635 to print to the Texas Instruments (TI)
	Omnilaser 2115 printer from Microsoft Windows version 3.0. If the printer has
	either of these Firmware versions, the ROM version does not matter.
	
	MORE INFORMATION
	================
	
	You can determine what Firmware version is being used by printing a printer
	status report. The Firmware version should appear at the top of the page. For
	further information, please refer to the Omnilaser's documentation for the exact
	method of determining the Firmware version.
	
	For more information, contact Texas Instruments.

	or reliability.
