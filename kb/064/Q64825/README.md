## Q64825: Windows 3.0 Support for the Tektronix Printers

	Article: Q64825
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Currently, there are Windows 3.0 drivers for four Tektronix printer models: the
	4693DX, the Phaser IIDX, the Phaser IISX, and the 4697.
	
	MORE INFORMATION
	================
	
	Printer drivers for the Tektronix printer models 4693DX, Phaser IIDX, and Phaser
	IISX are supplied by Zenographics. The driver for the 4697 is available from
	Tektronix. The part number is #063-0570-00.
	
	The driver for the Tektronix 4697 printer supports scalable fonts in all modes of
	Windows, provided the application being run supports them or you have a scalable
	font package such as Adobe Type Manager (ATM) or Bitstream FaceLift. This driver
	also makes use of available bitmap fonts.
	
	There is no Windows 3.0 driver for the Tektronix 4696 printer and Tektronix does
	not guarantee that the 4697 driver works with this printer.
	
	For more information, contact Zenographics and Tektronix.
	
	The products included here are manufactured by vendors independent of Microsoft;
	we make no warranty, implied or otherwise, regarding these products' performance
	or reliability.
