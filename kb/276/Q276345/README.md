## Q276345: Microsoft Casino: How to Improve Game Performance

	Article: Q276345
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbsound kbimu msgame kbgraphxlinkcritical
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Casino, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to improve game performance in Microsoft Casino.
	
	MORE INFORMATION
	================
	
	To improve game performance, turn off Game Sound Volume, Dealer Speech Volume,
	and Game Animation on the Game Options screen:
	
	1. On the Casino startup screen, click Options.
	
	  From any other game screen, press the F8 key.
	
	2. Move the Game Sound Volume slider all the way to the left (the zero setting).
	
	3. Move the Dealer Speech Volume slider all the way to the left (the zero
	  setting).
	
	4. Click to clear the Game Animation check box, and then click OK ([GRAPHIC:the
	  star and circle with the green check mark]).
	
	Additional query words: 1.00 msgame mscasino speed increase faster
