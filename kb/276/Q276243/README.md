## Q276243: Microsoft Classic Board Games: Minimum System Requirements

	Article: Q276243
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbref kbimu msgame
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Classic Board Games, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the minimum system requirements that are needed to play
	Microsoft Classic Board Games.
	
	MORE INFORMATION
	================
	
	To play Microsoft Classic Board Games, you need:
	
	- Multimedia PC with a Pentium 133 MHz or higher processor (Pentium II 266 MHz
	  processor recommended)
	
	- Microsoft Windows 95, Microsoft Windows 98, Microsoft Windows Millennium
	  Edition (Me), or Microsoft Windows 2000 operating system
	
	- Microsoft DirectX 7.0a (included on CD-ROM)
	
	- 16 megabytes (MB) of random access memory (RAM) for Windows 95 or Windows 98;
	  32 MB or RAM for Windows Me; 64 MB of RAM for Windows 2000
	
	- 40 MB of available hard disk space (an additional 15 MB of disk space is
	  required for DirectX 7.0a if it is not already installed on your computer
	
	- Video adapter with 2 MB or more of video memory (4 MB or more recommended)
	  capable of displaying a High Color (16 bit) color palette at a screen
	  resolution of 800 by 600 pixels
	
	- Super VGA monitor capable of displaying a screen resolution of 800 by 600
	  pixels
	
	- Double-speed CD-ROM drive (quad-speed or faster recommended)
	
	- Sound card with speakers or headphones for sound and music playback
	
	- Microsoft Mouse or compatible pointing device
