## Q276508: XADM: Inconsistent MsgID Formats Between Exchange 5.5 and 2000

{% raw %}

	Article: Q276508
	Product(s): Microsoft Exchange
	Version(s): 5.5
	Operating System(s): 
	Keyword(s): exc55
	Last Modified: 13-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you send a message with message tracking enabled from an Exchange Server
	5.5 computer to a Microsoft Exchange 2000 server through an Internet Mail
	Service connector, the format of the RemoteMsgID in the Exchange Server 5.5 log
	file is not consistent with the format of MsgID in the Exchange 2000 log file.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server
	version 5.5.
