## Q260903: StarLancer: Stops Responding at Title Screen or During Briefing

	Article: Q260903
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft StarLancer, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start Microsoft StarLancer, you may experience either of the following
	symptoms:
	
	- The computer may stop responding (hang) when the title screen is displayed.
	
	- When you start a new mission, the game may hang during the mission briefing.
	
	  If you press the ESC key, the game continues, and then the computer hangs
	  during the fighter launch sequence.
	
	CAUSE
	=====
	
	This behavior can occur if a Hercules Terminator Beast video adapter is
	installed in your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest version of the S3 Savage
	3D reference driver for your video adapter from the following S3 Web site:
	
	  http://www.s3.com
	
	To work around this issue, configure the game to use the software renderer:
	
	1. On the StarLancer startup screen, click Game Options.
	
	2. Click Video.
	
	3. Click one of the arrows next to 3D Render Mode to change the setting to
	  Software Renderer.
	
	4. Click Main Menu.
