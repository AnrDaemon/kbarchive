## Q260003: Asheron's Call: Unable to Restore Characters or Lost Items

{% raw %}

	Article: Q260003
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu msgame
	Last Modified: 16-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Asheron's Call, version 1.0 
	- Microsoft Asheron's Call: Dark Majesty, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	It is not possible to restore characters or items that are lost in the Microsoft
	Asheron's Call game world due to, but not limited to, the occurrence of any of
	the following events.
	
	- Death
	- Server errors
	- Emergency server maintenance
	- Lag
	- Server disconnection
	- Theft
	- Unrecoverable corpse
	- Server Rollback
	
	MORE INFORMATION
	================
	
	For additional information about how to find items after your character dies in
	Asheron's Call, click the article number below to view the article in the
	Microsoft Knowledge Base:
	
	  Q252960 Asheron's Call: How to Retrieve Items After Your Character Dies
	
	For additional information about how to find a missing character in Asheron's
	Call, click the article number below to view the article in the Microsoft
	Knowledge Base:
	
	  Q244736 Asheron's Call: Character Created at a Previous Time Is Missing
