## Q260859: StarLancer: Computer Hangs When You Start Instant Action Mission

{% raw %}

	Article: Q260859
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft StarLancer, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start an Instant Action mission in Microsoft StarLancer, your computer
	may stop responding (hang).
	
	CAUSE
	=====
	
	This behavior can occur if an ASUS 3D Explorer 128 video adapter is installed in
	your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest version of the video
	driver for your video adapter from the following ASUS Web site:
	
	  http://www.asus.com
