## Q260862: StarLancer: Hangs When You Click an Option on the Game Menu

	Article: Q260862
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft StarLancer, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you fly a mission in Microsoft StarLancer, your computer may stop
	responding (hang) if you press the ESC key and then click an option on the game
	menu.
	
	CAUSE
	=====
	
	This behavior can occur if a video adapter that is based on the ATI Rage II/II+
	PCI chipset is installed in your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest version of the video
	driver for your video adapter from the following ATI Web site:
	
	  http://www.atitech.com
