## Q260016: StarLancer Err Msg: DirectX 7.0a Is Required to Run StarLancer

	Article: Q260016
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbenv kberrmsg kbimu msgame
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft StarLancer, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft StarLancer, you may receive an error message similar to
	the following:
	
	  DirectX 7.0a is required to run StarLancer.
	
	CAUSE
	=====
	
	This behavior can occur if Microsoft DirectX 7.0a is not installed on the
	computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, install DirectX 7.0a from the StarLancer CD-1 CD-ROM:
	
	1. Insert the StarLancer CD-1 CD-ROM into the CD-ROM drive. Press and hold down
	  the SHIFT key as you insert the CD-ROM to prevent the game from starting
	  automatically.
	
	2. Double-click the My Computer icon on the desktop.
	
	3. Right-click the icon for the CD-ROM drive that contains the StarLancer CD-1
	  CD-ROM, and then click Explore.
	
	4. Double-click the DirectX folder.
	
	5. Double-click the Dxsetup.exe file.
	
	6. Follow the instructions on the screen to install DirectX 7.0a.
	
	7. When you are prompted to restart the computer, do so.
	
	Additional query words: 1.00 msgame star lancer direct-x dx7a
