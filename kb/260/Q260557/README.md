## Q260557: Microsoft Mail Password Recovery

	Article: Q260557
	Product(s): Microsoft Mail For PC Networks
	Version(s): 2.0,2.0a,2.1,2.1a,2.1b,2.1c,2.1d,2.1e,2.1x,2.x,3.0,3.0a,3.0b,3.1,3.1a,3.2
	Operating System(s): 
	Keyword(s): kbfaq
	Last Modified: 12-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, versions 2.0, 2.0a, 2.1, 2.1a, 2.1b, 2.1c, 2.1d, 2.1e, 2.1x, 2.x, 3.0, 3.0a, 3.0b, 3.1, 3.1a, 3.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft no longer recovers administrator passwords for Microsoft Mail or
	Workgroup Post Office (WGPO) mail systems. However, Microsoft will continue to
	assist administrators if they cannot start the Administrator program because of
	damaged files.
