## Q260863: StarLancer: Heads-Up Display and Radar Screen Appear Blurred

	Article: Q260863
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft StarLancer, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft StarLancer, the heads-up display and the radar screen
	may appear blurred or out of focus.
	
	CAUSE
	=====
	
	This behavior can occur if an STB Nitro DVD video adapter is installed in your
	computer.
	
	RESOLUTION
	==========
	
	To work around this issue, configure the game to use the software renderer:
	
	1. On the StarLancer startup screen, click Game Options.
	
	2. Click Video.
	
	3. Click one of the arrows next to 3D Render Mode to change the setting to
	  Software Renderer.
	
	4. Click Main Menu.
