## Q260734: StarLancer: Graphics in the Game Are Very Dark

{% raw %}

	Article: Q260734
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft StarLancer, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft StarLancer, the graphics in the game may be very dark,
	even when you increase the brightness setting on your monitor.
	
	CAUSE
	=====
	
	This behavior can occur if an ATI All-In-Wonder Pro PCI video adapter is
	installed in your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest version of the video
	driver for your video adapter from the following ATI Web site:
	
	  http://www.atitech.com
