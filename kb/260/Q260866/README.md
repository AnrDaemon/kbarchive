## Q260866: FS2000: Gauges Display Incorrect Data in Instant Replay Mode

{% raw %}

	Article: Q260866
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbdisplay kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator 2000 
	- Microsoft Flight Simulator 2000 Professional Edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you view a flight in Instant Replay or Flight Video mode, the gauges on the
	instrument panel may not function correctly.
	
	The gauges may display incorrect data, appear to move rapidly, or may not
	function at all.
	
	MORE INFORMATION
	================
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
