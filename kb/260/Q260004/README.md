## Q260004: StarLancer: Throttle Control on SideWinder Joystick Is Disabled

{% raw %}

	Article: Q260004
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbhw kbimu msgame kbHardware
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft StarLancer, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you use a Microsoft SideWinder joystick to play Microsoft StarLancer, the
	throttle control on the joystick may not function properly.
	
	CAUSE
	=====
	
	This behavior can occur if the joystick is connected to the Pass-Through port on
	a Microsoft SideWinder gamepad, and is not directly connected to the game port
	in your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, quit StarLancer, connect the joystick directly to the
	game port in your computer, and then restart the computer.
