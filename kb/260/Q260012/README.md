## Q260012: StarLancer: Cannot Change the Default Video Settings in the Game

{% raw %}

	Article: Q260012
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft StarLancer, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft StarLancer, you may experience one or more of the
	following symptoms:
	
	- You may be unable to change the default video settings in the game.
	
	- Textures in the game may appear jagged or blocky.
	
	CAUSE
	=====
	
	This behavior can occur if an ATI Rage 128 GL AGP video adapter is installed in
	your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest version of the video
	driver for your video adapter from the following ATI Web site:
	
	  http://www.atitech.com
