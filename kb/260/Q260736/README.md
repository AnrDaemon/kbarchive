## Q260736: StarLancer: How to Play the Game in Greyscale Mode

{% raw %}

	Article: Q260736
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbimu msgame
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft StarLancer, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	You may be able to improve graphics performance or resolve choppy video clip
	playback in Microsoft StarLancer by playing StarLancer in greyscale mode.
	
	This article describes how to play StarLancer in greyscale mode.
	
	MORE INFORMATION
	================
	
	To play StarLancer in greyscale mode:
	
	1. Right-click the StarLancer shortcut icon on the Windows desktop, and then
	  click Properties.
	
	2. Click the Target box on the Shortcut tab.
	
	3. Press the END key, press the SPACEBAR, and then type "-greyscale" (without
	  the quotation marks).
	
	4. Click OK.
	
	5. Double-click the StarLancer shortcut icon on the Windows desktop to play the
	  game in greyscale mode.
