## Q202046: SMS: Remote Control Supports Microsoft TCP/IP Stack Only

{% raw %}

	Article: Q202046
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.2,2.0
	Operating System(s): 
	Keyword(s): kbRemote kbsms kbsms200 kbsms100 kbsms110 kbsms120 kbRemoteProg
	Last Modified: 27-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server versions 1.2, 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Systems Management Server versions 1.2 and 2.0 currently only support
	Microsoft's TCP/IP stack when TCP/IP is the underlying protocol for the Remote
	Control Agent. If another vendor's TCP/IP stack is installed, Microsoft cannot
	guarantee the functionality of Remote Control in either version.
