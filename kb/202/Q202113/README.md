## Q202113: XCON: Connectors in Standard &amp; Enterprise Exchange Server 5.5

	Article: Q202113
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5
	Operating System(s): 
	Keyword(s): kbfaq
	Last Modified: 22-APR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article explains what connectors are available in the Standard and
	Enterprise versions of the Exchange Server 5.5 software package.
	
	The Standard edition of Exchange Server 5.5 comes with the following connectors:
	
	- Site Connector
	- Microsoft Mail Connector
	- cc:Mail Connector
	- Microsoft Notes Connector
	- Internet Mail Service
	- Internet News Connector
	
	The Enterprise edition of Exchange Server 5.5 includes a superset of the above
	listed connectors as well as:
	
	- X.400 Connector
	- SNADS Connector
	- OV/VM(PROFS) Connector
