## Q202011: XWEB: Meeting Requests in OWA Generate Unread Calendar Items

	Article: Q202011
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook Web Access, version 5.5 Service Pack 1 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you create a new meeting request and send the request using Outlook Web
	Access, you will have an unread calendar item if you log on to an Exchange
	Server or Outlook Client. This behavior only occurs in Outlook Web Access
	version 5.5 Service Pack 1.
	
	RESOLUTION
	==========
	
	To resolve this problem and clear the unread items, on the Edit menu, click Mark
	All As Read or open the meeting in calendar.
	
	Microsoft has confirmed this to be a problem in Exchange Server version 5.5. This
	problem has been corrected in the latest U.S. service pack for Exchange Server
	version 5.5. For information on obtaining the service pack, query on the
	following word in the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
