## Q224379: Characters Unreadable After Applying Pan-Chinese Windows NT SP3

	Article: Q224379
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbWinNT400sp4fix
	Last Modified: 18-APR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	This article applies to the Pan-Chinese version of Windows NT Workstation 4.0.
	
	If you select Chinese (Taiwan) as the default locale after you install Windows NT
	Service Pack 3 (SP3), the characters on the screen may not be readable.
	
	CAUSE
	=====
	
	This problem occurs because incorrect fonts are included in Pan-Chinese Windows
	NT Workstation version 4.0 SP3.
	
	For example:
	
	  Sserife.fon
	  Sseriff.fon
	
	RESOLUTION
	==========
	
	To resolve this problem, obtain the latest service pack for Windows NT
	Workstation 4.0. For additional information, please see the following article in
	the Microsoft Knowledge Base:
	
	  Q152734 How to Obtain the Latest Windows NT 4.0 Service Pack
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Pan-Chinese version of
	Windows NT Workstation 4.0. This problem was first corrected in the Pan-Chinese
	version of Windows NT Workstation 4.0 Service Pack 4.
