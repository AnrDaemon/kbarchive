## Q224274: Streets and Trips 2000: Hangs When You Print to HP LaserJet 4500

	Article: Q224274
	Product(s): Microsoft Automap
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kb3rdparty kbprint kbimu
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Expedia Streets and Trips 2000 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you attempt to print from Microsoft Expedia Streets and Trips 2000 to a
	Hewlett-Packard LaserJet Color 4500 printer, the program may stop responding
	(hang).
	
	CAUSE
	=====
	
	This behavior can occur if the Printer Control Language (PCL) driver for the
	LaserJet Color 4500 printer is installed on your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, install the PostScript (PS) printer driver for the
	LaserJet Color 4500 printer.
