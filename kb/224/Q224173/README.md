## Q224173: XCON: Cannot Enable Trace Mode for Cirel Winsock X.25 Connection

	Article: Q224173
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 28-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	If you configure an X.400 Connector to use a Cirel Winsock X.25 card installed
	in an Exchange Server computer, select the Services branch in the FPX
	Communication Manager, and then click the Enable Trace Mode check box to select
	it, the Cirel Winsock software may stop responding.
	
	WORKAROUND
	==========
	
	To work around this problem, click the Enable Trace Mode check box to clear it.
	For additional information about this problem, including possible additional
	workarounds, please contact Cirel.
