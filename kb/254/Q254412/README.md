## Q254412: Err Msg: Msvcp60.dll File Is Linked to Missing Export Msvcrt.dll

{% raw %}

	Article: Q254412
	Product(s): Microsoft Automap
	Version(s): 
	Operating System(s): 
	Keyword(s): kbenv kberrmsg kbsetup kbimu
	Last Modified: 07-JUN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MapPoint 2001 
	- Microsoft Streets and Trips 2001 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start one of the programs listed at the beginning of this article, you
	may receive the following error message:
	
	  The MSVCP60.DLL file is linked to missing export MSVCRT.DLL
	
	CAUSE
	=====
	
	This behavior can occur if either of the following conditions are true:
	
	- You install the program and then attempt to start the program before you
	  restart your computer.
	
	- The Windows Libraries Update is not installed on your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, restart your computer.
	
	If the issue continues to occur, download and install the Windows Libraries
	Update. To do this:
	
	1. Visit the following Microsoft Web site:
	
	  http://www.microsoft.com/downloads/search.asp?
	
	2. Next to "Search using", click Keyword Search.
	
	3. In the Keywords box, type "libraries update" (without the quotation marks).
	
	4. In the Operating System box, click the appropriate operating system for your
	  computer.
	
	5. Under "Show Results For", select "All Downloads", and then click "Find It!"
	
	6. Under Title, click the Microsoft Libraries Update Web link.
	
	7. Follow the instructions on the screen to download and install the Microsoft
	  Libraries Update.
	
	REFERENCES
	==========
	
	For additional information, click the article number below to view the article
	in the Microsoft Knowledge Base:
	
	  Q261157 Streets and Trips: Computer Stops Responding When You Start the
	  Program
