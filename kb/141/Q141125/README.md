## Q141125: Cannot Insert Package into BBS Message

	Article: Q141125
	Product(s): The Microsoft Network
	Version(s): 1.20 1.30
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.2, 1.3 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You cannot insert a Package object into a bulletin board (BBS) message.
	
	CAUSE
	=====
	
	This behavior is by design.
	
	MORE INFORMATION
	================
	
	Packages are not allowed to prevent people from posting items that might damage
	your computer when you open the package. For packages posted before version 1.2
	of The Microsoft Network, you receive a warning if you open a message with a
	package in it.
