## Q141741: Macintosh Dinosaurs 1.0: Files Installed During Installation

	Article: Q141741
	Product(s): Microsoft Home Multimedia Titles
	Version(s): MACINTOSH:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Dinosaurs for the Macintosh 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the files and folders installed by the Dinosaurs version 1.0
	installer.
	
	MORE INFORMATION
	================
	
	The following files and folders are installed to the hard disk using the
	installer. You can also manually copy these files from the Dinosaurs compact
	disc to the appropriate locations on the hard disk.
	
	Files and Folders Installed on the Hard Disk
	--------------------------------------------
	
	Microsoft Dinosaurs Folder
	
	- Microsoft Dinosaurs
	- Dinosaurs--ReadMe
	- TeachText
	
	System Folder
	
	  Extensions Folder
	
	- Quicktime (version 1.6)
	- 32-Bit QuickDraw (version 1.2)
	- Sound Manager (version 3.0)
	
	  Control Panels Folder
	
	- Sound (version B1-8.0.1)
