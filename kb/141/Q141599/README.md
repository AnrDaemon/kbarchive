## Q141599: Lens Screen Saver Hangs on Dell Omniplex 4100

	Article: Q141599
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	- Microsoft Plus! for Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When the Lens screen saver is activated on a Dell Omniplex 4100 computer, a
	square may be displayed around the lens and the computer may stop responding
	(hang).
	
	CAUSE
	=====
	
	This behavior can occur if the Omniplex 4100 computer has a Phoenix BIOS version
	A05.
	
	RESOLUTION
	==========
	
	Upgrade the computer's BIOS to Phoenix BIOS version A10 or later. For
	information about obtaining the current flash BIOS upgrade, please contact
	Dell's Technical Support or BBS.
