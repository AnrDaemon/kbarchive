## Q141387: Ball Hesitates with Flipper Movement Playing 3D Pinball

	Article: Q141387
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	- Microsoft Plus! for Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play 3D Pinball, the ball may hesitate each time you move a flipper.
	
	CAUSE
	=====
	
	This behavior can be caused by low resources or sound conflicts within 3D
	Pinball.
	
	RESOLUTION
	==========
	
	To resolve this situation, try the following methods:
	
	- Close any other running program while you play 3D Pinball.
	
	- Disable sounds in 3D Pinball by clicking Sounds or Music on the Options menu.
	
	- Adjust the sound by running the Wmconfig tool. For information about using
	  this tool, please see the following article in the Microsoft Knowledge Base:
	
	  Q140081 Purpose of the Wmconfig.exe and Wavemix.inf Files
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Plus! version 1.0 for
	Windows 95. We are researching this problem and will post new information here
	in the Microsoft Knowledge Base as it becomes available.
