## Q141002: Bookshelf: Can't Use Undo with Copy To Excel Command

	Article: Q141002
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:; :1995 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf 1996-97 for Windows 
	- Microsoft Bookshelf for Windows 1995 edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the Bookshelf Copy To Excel command, the Excel Undo feature cannot
	undo the information you pasted.
	
	WORKAROUND
	==========
	
	To work around this problem, use a regular copy and paste operation. To copy and
	paste text between Bookshelf and Excel, follow these steps:
	
	1. Start Bookshelf and select the text you want to copy.
	
	2. On the Edit menu, click Copy.
	
	3. Start Excel.
	
	4. Select a cell or the upper-left cell of the area where you want to paste the
	  text.
	
	5. On the Edit menu, click Paste.
	
	The Excel Undo feature can now to undo the paste.
	
	MORE INFORMATION
	================
	
	The Bookshelf Copy To Excel command works with Excel 7.0. To use this command,
	do one of the following:
	
	- On the Edit menu, click Copy to Excel.
	
	  -or-
	
	- Press and hold down the keyboard CTRL key while you press the E key (the
	  CTRL+E keyboard shortcut).
