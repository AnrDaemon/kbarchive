## Q141255: Help Menu Items Unavailable in Full Screen Mode

	Article: Q141255
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbbuglist
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Oceans for Windows, version 1.0 
	- Microsoft World of Flight for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you select some of the Help menu items in the multimedia programs listed at
	the top of this article, no information is displayed.
	
	CAUSE
	=====
	
	If you run the program in full screen mode using a screen resolution of 800 x
	600 or 1024 x 768, some of the Help menu items are unavailable.
	
	RESOLUTION
	==========
	
	To resolve this problem, do the following:
	
	1. In the multimedia product you are using, click the Options button, and then
	  click Audio/Video.
	
	2. Under Run Full Screen, click Off. Click OK.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Oceans for Windows,
	version 1.0 and Microsoft World of Flight for Windows, version 1.0. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
