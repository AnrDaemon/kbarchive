## Q141428: Bookshelf 1995: Search Not Restarted After Spell Check

	Article: Q141428
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1995 edition; WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for Windows 1995 edition 
	- Microsoft Bookshelf '95 for Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Bookshelf, when you search for a misspelled word in the Advanced Find window
	while the Automatic Spelling feature is turned on, a list of suggested words is
	provided.
	
	When you click the correct word, the word is entered into the text box but does
	not restart the search.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Bookshelf programs listed at
	the top of this article. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
	
	This problem does not occur in Microsoft Bookshelf '96-'97 for Windows.
	
	WORKAROUND
	==========
	
	Click Find! again to manually restart the search.
