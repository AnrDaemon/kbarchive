## Q141945: Hewlett-Packard Activity Center Decreases MSN Performance

{% raw %}

	Article: Q141945
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.2,1.3,2.0,2.5
	Operating System(s): 
	Keyword(s): kb3rdparty kbmsn
	Last Modified: 14-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.2, 1.3, 2.0, 2.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use a Hewlett-Packard computer with the Activity Center program
	installed, navigation on MSN, The Microsoft Network, is very slow.
	
	CAUSE
	=====
	
	Activity Center interferes with the modem as it is transferring data from MSN to
	your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, disable Activity Center.

	================
