## Q141894: Cinemania 1996 for the Macintosh System Requirements

	Article: Q141894
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1996 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Cinemania for the Macintosh 1996 edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the minimum system requirements to run the 1996 edition of
	Microsoft Cinemania for the Macintosh.
	
	MORE INFORMATION
	================
	
	The minimum system requirements to run Cinemania 1996 for the Macintosh are:
	
	- Any Apple Macintosh computer supporting a 256-color, 640x480 resolution
	  (13-inch), or higher resolution monitor.
	
	- System 7 or later
	
	- Minimum 8 MB RAM memory
	
	- Double-speed CD-ROM: The CD-ROM drive should be able to transfer data from
	  the CD-ROM at 300k per second while using less than 40% of the CPU bandwidth
	  and has an average seek time of less than 400 milliseconds.
	
	The minimum system requirements to use the Cinemania Monthly Updates are:
	
	- Hayes or Hayes-compatible modem
	
	- Access to the Internet
	
	- Minimum 6 MB of hard disk space to download all 12 monthly update files
