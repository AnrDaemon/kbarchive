## Q141742: Mac Encarta 95: Files Installed During Installation

	Article: Q141742
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1995 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 26-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia for Macintosh 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the files and folders installed by the Encarta 1995
	installer.
	
	MORE INFORMATION
	================
	
	The following files and folders are installed to the hard disk when you use the
	Encarta 1995 installer.
	
	For additional information, please see the following article in the Microsoft
	Knowledge Base:
	
	  Q131433 Macintosh Encarta 1995: Manual Installation Instructions
	
	Files and Folders Installed on the Hard Disk
	--------------------------------------------
	
	Microsoft Encarta 95 folder
	
	- Microsoft Encarta '95
	- Encarta '95 ReadMe
	- Encarta & Memory ReadMe
	- Animation Launcher
	- Encarta Projector
	- SimpleText
	- Animation (Projector Document)
	
	  Interactivities folder
	
	- Country Facts
	- Ellis Island
	- Exploring Fractals
	- Orbit Simulation
	- Personal Nutrition
	- Pop/Clim Charts
	- World Languages
	- World Music
	- MindMaze
	
	  Microsoft System Information folder
	
	- Microsoft System Information
	- MSInfo
	
	  Data Files folder
	
	- Encarta '95 Data
	- Performance File
	- See Also Index
	- Time1 Data
	- Time2 Data
	
	System Folder
	
	  Extensions folder
	
	- Apple Multimedia Tuner Extension (version 2.0.1)
	- FIF Component Extension
	- Sound Manager Extension (version 3.0)
	- QuickTime (version 2.0)
	- QuickTime PowerPlug (version 2.0)
	- QuickTime Musical Instruments (version 2.0)
	
	  Control Panels folder
	
	- Sound (version 8.0.1)
	
	  Preferences folder
	
	  Encarta '95 Preferences folder
	
	- Encarta '95 Folder Alias
	- Encarta '95 Preferences
	
	  Encarta 1994 Preferences folder (empty if Encarta 1994 is not installed)
	
	  NOTE: To preserve compatibility with previous versions of Microsoft Works for
	  the Macintosh, version 4.0, the Encarta 1994 Preferences Folder is required.
	
	  Fonts folder
	
	- Arial
	- MS Reference
	- Symbol
	- Times New Roman
