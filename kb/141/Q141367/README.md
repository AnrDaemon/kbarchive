## Q141367: Supported Network Adapters for Windows 95 RPL Boot

	Article: Q141367
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 20-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following RPL-capable network adapters are compatible with Windows 95 RPL
	Boot to Novell NetWare servers:
	
	3Com EtherLink II
	3Com EtherLink III
	Cabletron E2000
	Cabletron E2100
	Cabletron E3112x
	IBM TokenRing
	Intel EtherExpress 16
	Intel EtherExpress PRO
	NE2000 compatibles
	Novell/Anthem NE2000
	SMC
