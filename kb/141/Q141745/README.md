## Q141745: Macintosh Wine Guide 1.0a: Files Installed During Installation

	Article: Q141745
	Product(s): Microsoft Home Multimedia Titles
	Version(s): MACINTOSH:1.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Wine Guide for the Macintosh, version 1.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the files and folders installed by the Wine Guide installer.
	
	MORE INFORMATION
	================
	
	The following files and folders are installed to the hard disk using the
	installer, or by manually copying the files from the compact disc to the correct
	locations on the hard disk.
	
	Files and Folders Installed on the Hard Disk
	--------------------------------------------
	
	Microsoft Wine Guide folder
	
	- Microsoft Wine Guide
	- Microsoft Wine Guide ReadMe
	- SimpleText
	
	System Folder
	
	  Extensions folder
	
	- Apple Multimedia Tuner (version 2.0.1)
	- QuickTime (version 2.0)
	- QuickTime Musical Instruments (version 2.0)
	- QuickTime Power Plug (version 2.0)
	- Sound Manager (version 3.1)
	
	  Control Panels folder
	
	- Sound (version 8.0.1)
