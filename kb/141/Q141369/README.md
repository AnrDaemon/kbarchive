## Q141369: Cannot Change Service Type While Connected to MSN

{% raw %}

	Article: Q141369
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.2,1.3,2.0
	Operating System(s): 
	Keyword(s): kbenv kbmsn
	Last Modified: 13-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.2, 1.3, 2.0 
	-------------------------------------------------------------------------------
	
	If you change the service type you are using while you are connected
	to MSN, The Microsoft Network, the change is not effective until you
	exit MSN and sign in again. The different service types use different
	transport software to connect to MSN.
