## Q141731: Mac Musical Instruments: Files Installed During Installation

	Article: Q141731
	Product(s): Microsoft Home Multimedia Titles
	Version(s): MACINTOSH:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Musical Instruments for the Macintosh, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the files and folders installed to your hard disk when you
	install Microsoft Musical Instruments.
	
	MORE INFORMATION
	================
	
	The following files and folders are installed to the hard disk whether you use
	the installer, or if you manually copy the files from the Musical Instruments
	compact disc to the appropriate locations on the hard disk.
	
	Files and Folders Installed on the Hard Disk
	--------------------------------------------
	
	Musical Instruments folder
	
	- Microsoft Musical Instruments
	- Musical Instruments-Read Me
	- TeachText
	
	System Folder
	
	  Extensions folder
	
	- 32-Bit QuickDraw (version 1.2)
	- QuickTime (version B1-1.5)
	
	NOTE: The installer program will overwrite the existing version of QuickTime even
	if the version on your computer is newer.
