## Q244953: HOWTO: Implement SourceSafe Integration with FrontPage 2000

{% raw %}

	Article: Q244953
	Product(s): Microsoft SourceSafe
	Version(s): 5.0,6.0
	Operating System(s): 
	Keyword(s): kbFrontPage kbSSafe500 kbSSafe600 kbFAQ kbDSupport kbFPServX kbSsafe600FAQ
	Last Modified: 18-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual SourceSafe for Windows, versions 5.0, 6.0 
	- Microsoft FrontPage 2000 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article explains where to locate information about how to implement
	SourceSafe integration with FrontPage 2000.
	
	MORE INFORMATION
	================
	
	The information provided in the FrontPage 2000 user guide with respect to
	integrating FrontPage 2000 and Visual SourceSafe inadvertently applies to the
	previous version (FrontPage 98) and not FrontPage 2000.
	
	For help setting up this integration, see the Advanced Administrative Tasks topic
	in the FrontPage 2000 Resource Kit on the following Microsoft Web site at:
	
	  http://officeupdate.microsoft.com/frontpage/wpp/serk/adadv_6.htm
