## Q244859: WD97: Nothing Happens When You Click Web Page Preview

{% raw %}

	Article: Q244859
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta word8 word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you attempt to preview your Word document as a Web page (on the File menu,
	click Web Page Preview), nothing happens.
	
	CAUSE
	=====
	
	This problem occurs if Microsoft Photo Editor is open.
	
	WORKAROUND
	==========
	
	To view your Word document as a Web page, close Microsoft Photo Editor.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
