## Q244592: Unavailable Language Error Message After Reinstall of Windows 95

	Article: Q244592
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): osr1 osr2 win95
	Last Modified: 29-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 OEM Service Release, versions 2.0, 2.1, 2.5 
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you reinstall Windows 95 over a previous installation with Lotus Smart Suite
	96 installed, you may receive the following error message during the last time
	Windows restarts in Setup:
	
	  Unavailable langauge
	  Selected language is not available.
	
	RESOLUTION
	==========
	
	To resolve this issue, reinstall Lotus Smart Suite 96.
