## Q244370: Mastering eCommerce Dev.: Business to Consumer Corrections

{% raw %}

	Article: Q244370
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 20-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Mastering eCommerce Development: Business to Consumer ISBN 0-7356-0891-1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Mastering eCommerce Development: Business
	to Consumer, ISBN 0-7356-0891-1.
	
	The following topics are covered:
	
	- Correction To Running Headers
	
	- Page xv: Correction To Appendix C Title
	
	- Page 143: Lab 8 Title Correction
	
	MORE INFORMATION
	================
	
	Correction To Running Headers
	-----------------------------
	
	Change the running headers from:
	"Building E-Commerce Solutions: Business to Consumer"
	
	To:
	"eCommerce Development: Business to Consumer"
	
	
	Page xv: Correction To Appendix C Title
	---------------------------------------
	
	Change:
	"Appendix C: Microsoft Site Server 3.0 Commerce Edition Perfomrance Kit"
	
	To:
	"Appendix C: Microsoft Site Server 3.0 Commerce Edition Performance Kit"
	
	
	Page 143: Lab 8 Title Correction
	--------------------------------
	
	Change:
	"Lab 8: Tracking Shoppers Using Registration Table"
	
	To:
	"Lab 8: Tracking Shoppers Using Registration Tables"
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
	
	Additional query words: 0-7356-0891-1 DEVBOOK
