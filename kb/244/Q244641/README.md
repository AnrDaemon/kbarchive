## Q244641: Documentation Error in the Windows NT Server Networking Guide

	Article: Q244641
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	There is a documentation error on page 212, chapter 5 of The Windows NT Server
	Networking Guide. Under the "Configuring A DHCP Relay Agent" section, it
	incorrectly states that it is possible to use a Window NT Workstation-based
	computer as a DHCP Relay agent. This is the incorrect text:
	
	  The two subnets that do not contain computers running Windows NT Server DHCP
	  service have a computer running Windows NT Workstation or Windows NT Server
	  that acts as DHCP relay or bootP relay agent.
	
	CAUSE
	=====
	
	This problem occurs because of a documentation error in The Windows NT Server
	Networking Guide.
	
	MORE INFORMATION
	================
	
	The Windows NT Server Networking Guide is part of the Windows NT Server Resource
	Kit from Microsoft Press.
