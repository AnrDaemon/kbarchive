## Q233451: MSBP: Version Tab Missing from File Properties Dialog Box

	Article: Q233451
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbenv kbimu
	Last Modified: 02-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Business Planner 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to view the properties of certain Microsoft Business Planner
	files, the Version tab may be missing from the file properties dialog box.
	
	CAUSE
	=====
	
	This behavior occurs if the version number of the file is not included in the
	file properties.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Business Planner.
	
	MORE INFORMATION
	================
	
	This issue has been reported with the following Business Planner files:
	
	- Mhk3cee.exe
	- Msbp_stb.exe
