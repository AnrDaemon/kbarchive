## Q233440: MSBP: Complete Text of Article Not Displayed in Text Only Mode

	Article: Q233440
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbdisplay kbui kbimu
	Last Modified: 23-JUN-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Business Planner 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you attempt to view an article in Text Only Mode in Microsoft Business
	Planner, the complete text of the article may not fit inside the Text Only Mode
	window.
	
	RESOLUTION
	==========
	
	To resolve this issue, maximize the Text Only Mode window.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Business Planner.
	
	Additional query words: msbp truncated cut short text-only
