## Q233468: Business Planner: Unable to Open Certain Help Topics

	Article: Q233468
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbnetwork kbsetup kbimu
	Last Modified: 27-JUN-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Business Planner 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click any of the following Help topics in Microsoft Business Planner,
	the Help topic may not open.
	
	- About the Content in Business Planner
	- About Categories, Topics, and Articles
	- About Wizards
	- About the Resource Center
	
	CAUSE
	=====
	
	This behavior can occur if you install Business Planner using the "Run from
	Network" installation option.
	
	RESOLUTION
	==========
	
	To work around this issue, reinstall Business Planner to run from your hard
	disk.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Business Planner.
