## Q233433: MSBP: Text Only Mode Unavailable When You View Wizard Interview

	Article: Q233433
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbtool kbui kbimu
	Last Modified: 26-JUN-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Business Planner 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you view a wizard interview page in Microsoft Business Planner, Text Only
	Mode is unavailable.
	
	NOTE: Text Only Mode is available when you view any other page in a Business
	Planner wizard.
	
	STATUS
	======
	
	This behavior is by design.
	
	MORE INFORMATION
	================
	
	A wizard interview page is a page within a Business Planner wizard that contains
	fields into which you can type data.
	
	Additional query words: msbp box disappears text-only
