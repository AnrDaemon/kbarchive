## Q259156: Allegiance: Program Stops Responding

	Article: Q259156
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay igz kbimu msgame
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Allegiance, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Allegiance, the program may stop responding (hang).
	
	CAUSE
	=====
	
	This behavior can occur if the following conditions are true:
	
	- A Cyrix microprocessor is installed in your computer.
	
	- A video adapter based on the nVIDIA chip set is installed in your computer.
