## Q259952: StarLancer: 3D Accelerator Card Is Not Detected by the Game

{% raw %}

	Article: Q259952
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbimu msgame
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft StarLancer, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If a 3D accelerator card is installed in your computer in addition to a primary
	display adapter, the 3D accelerator card is not detected by Microsoft
	StarLancer.
	
	StarLancer only detects the primary video adapter that is installad in the
	computer.
	
	MORE INFORMATION
	================
	
	Because the 3D accelerator card that is installed in your computer is not
	detected by StarLancer, you can only configure 3D Render Mode to use your
	primary video adapter or the Software Renderer when you play the game.
