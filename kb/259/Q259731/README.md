## Q259731: Asheron's Call Err Msg: Subscription Coupon Number Is Not Valid

	Article: Q259731
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kberrmsg kbimu msgame
	Last Modified: 16-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Asheron's Call, version 1.0 
	- Microsoft Asheron's Call: Dark Majesty, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you type the Subscription Coupon Number on the Asheron`s Call Subscription
	Coupon page and then click Submit, you may receive the following error message:
	
	  The Subscription Coupon Number you entered is not valid.
	
	CAUSE
	=====
	
	This behavior can occur for one of the following reasons:
	
	- You did not type in the Subscription Coupon Number correctly on the Asheron`s
	  Call Subscription Coupon page.
	- You selected the wrong game to subscribe to.
	
	RESOLUTION
	==========
	
	Subscribe to the game you have purchased by clicking the Subscription link at
	the bottom of the following Zone Web site for your game:
	
	Asheron's Call (Coupon Number starts with CARAC)
	
	  http://zone.msn.com/asheronscall/minilaunch2.asp
	
	Asheron's Call Dark Majesty (Coupon Number starts with CARXP)
	
	  http://zone.msn.com/asheronscallxp/minilaunch2.asp
	
	
	Additional query words: 1.00 msgame igz msngz ashcall ac subscribe asherons invalid
