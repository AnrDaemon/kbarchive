## Q118674: Gtwy: Gateway Connectivity Information

	Article: Q118674
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:2.1,3.0,3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, versions 2.1, 3.0, 3.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following list identifies the Microsoft Mail gateways that can be used to
	integrate Mail in a heterogeneous mail environment. Additional gateways that use
	the Microsoft Mail File Format API (FFAPI) may also be available.
	
	NOTE: All gateways in the following list are Microsoft gateways unless otherwise
	noted.
