## Q118344: Setup Incorrectly Detects Multiple Floppy Drive Controllers

	Article: Q118344
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Windows 95 Setup procedure detects more than one floppy disk drive
	controller when you only have one.
	
	CAUSE
	=====
	
	Ultrastor Technical Support has confirmed that systems with the Ultrastor 24F
	EISA SCSI adapter demonstrate this behavior.
	
	
	WORKAROUND
	==========
	
	Please contact Ultrastor about a possible controller upgrade to work around this
	behavior.
