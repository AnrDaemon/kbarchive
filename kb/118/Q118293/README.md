## Q118293: MAC Encarta: Hang in MindMaze or Dictionary with CacheOut

	Article: Q118293
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1994 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 24-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 1994 The Complete Multimedia Encyclopedia 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Encarta may stop responding (hang) if you are using the CacheOut control panel.
	This problem occurs when trying to open, run, or close MindMaze, the dictionary,
	animations, and the orbit simulator.
	
	This problem is due to a conflict between CacheOut and Encarta. For more
	information, refer to the Encarta 1994 readme file.
	
	RESOLUTION
	==========
	
	To avoid this problem, disable CacheOut before using Encarta.
	
	MORE INFORMATION
	================
	
	CacheOut is a control panel for 68040 Macintoshes that lets you select specific
	applications that have the 68040 cache enabled. If this control panel is
	installed and enabled, and the Background option is selected, Encarta may hang.
