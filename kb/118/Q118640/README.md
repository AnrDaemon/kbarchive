## Q118640: Bookshelf 1994: Atlas Gallery List Is Blank

{% raw %}

	Article: Q118640
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1994 edition,1995 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for Windows versions 1994 edition, 1995 edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In Bookshelf 1994 Atlas Gallery, all of the Media options are unavailable
	(dimmed) and the Gallery list is empty.
	
	In Bookshelf 1995 Atlas Gallery, Audio, and Animation & Video are dimmed.
	Picture and All Media are the only available selections.
	
	MORE INFORMATION
	================
	
	The Bookshelf 1994 Atlas Help topic states:
	
	NOTE: The following information is in the Atlas Help file, under the heading The
	Atlas Gallery Tabbed List.
	
	  When you want to browse through all the multimedia items in the atlas, click
	  the Gallery Tab. In the Atlas Gallery, click the All Media button to see a
	  list of all the maps that contain the different multimedia items. Use the
	  scroll bar to browse through the list. Click a list item to open the article.
	
	However, all media gallery categories are unavailable in the Atlas. This is
	because the maps and sounds are included in each individual entry, available
	under Contents. There is no browse feature for maps or sounds.
