## Q118929: PC Win: Change Password Is Case Sensitive

	Article: Q118929
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0,3.0b,3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, versions 3.0, 3.0b, 3.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In the Microsoft Mail for PC Networks Windows client, the password dialog box is
	case sensitive. If you type text using a different case in the New Password and
	Verify New Password boxes, the OK button remains unavailable (grayed).
	
	Additional query words: 3.00 3.20 gray grey greyed
