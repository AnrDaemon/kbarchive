## Q118522: Numerical Recipes Disk Included with PowerStation

	Article: Q118522
	Product(s): Microsoft Fortran Compiler
	Version(s): 1.0,1.0a,4.0
	Operating System(s): 
	Keyword(s): kb3rdparty
	Last Modified: 06-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft FORTRAN PowerStation for MS-DOS, versions 1.0, 1.0a 
	- Microsoft Fortran Powerstation 32 for Windows NT, versions 1.0, 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	A disk containing numerical algorithms was included in some boxes of FORTRAN
	PowerStation. Support for this disk, including replacement disks, is provided by
	Numerical Recipes Software. You can also order copies of the "Numerical Recipes"
	book from Cambridge University Press, which contains the source code and
	explains the algorithms.
