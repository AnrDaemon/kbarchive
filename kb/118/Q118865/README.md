## Q118865: FAX: FAXVIEW.EXE Postscript Printing Fails under Win 3.0, 3.0a

	Article: Q118865
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to Fax, versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When printing to postscript printers under Windows 3.0 or 3.0a, the Microsoft
	Mail Windows client FAX driver PSCRPT.DVR prints only a top band of the
	document.
	
	CAUSE
	=====
	
	The cause has not been ascertained. Under Windows 3.0 and 3.0a, some
	applications print correctly and others do not.
	
	RESOLUTION
	==========
	
	Upgrade the workstation to Windows 3.1. Many postscript printing problems were
	resolved with Windows 3.1.
