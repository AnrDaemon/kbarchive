## Q118967: ATT: Removal of Gateway Fails to Update NETWORK.GLB

	Article: Q118967
	Product(s): Microsoft Mail For PC Networks
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 03-MAR-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to AT T, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The AT&T Gateway Access component does not complete a remove correctly from
	an AT&T Gateway postoffice.
	
	CAUSE
	=====
	
	The INSTALL -R command fails to update the NETWORK.GLB file.
	
	RESOLUTION
	==========
	
	Call Microsoft Product Support Services for assistance with correcting the
	NETWORK.GLB file.
