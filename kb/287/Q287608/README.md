## Q287608: FS2000 Err Msg: Your Computer Has Run Out of Available Memory

	Article: Q287608
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): kberrmsg kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator 2000 
	- Microsoft Flight Simulator 2000 Professional Edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you fly an aircraft in Microsoft Flight Simulator 2000 north of 67 degrees
	north latitude, you may receive the following error message:
	
	  Your computer has run out of available memory. Flight Simulator will now
	  Exit.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
	
	Additional query words: msgame fltsim flightsim fsim
