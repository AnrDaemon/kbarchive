## Q237526: Encarta Dictionary 2000: Office Integration Is Lost

{% raw %}

	Article: Q237526
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:; :
	Operating System(s): 
	Keyword(s): kbinterop kbsetup kbui kbimukbfaq
	Last Modified: 27-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta World English Dictionary 
	- Microsoft Encarta World English Dictionary 2001 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you install both Microsoft Bookshelf 2000 and Microsoft Encarta World English
	Dictionary 2000 on your computer, you may experience one of the following
	symptoms.
	
	- Bookshelf 2000 is integrated with Microsoft Office 2000, but Encarta World
	  English Dictionary 2000 is not:
	
	   - When you start Microsoft Word, the Bookshelf 2000 button appears on the
	     toolbar, but the Encarta World English Dictionary 2000 button does not.
	
	   - When you start Microsoft PowerPoint or Word, and then click Look Up
	     Reference on the Tools menu, Bookshelf 2000 is listed in the "Available
	     reference titles" box, but Encarta World English Dictionary 2000 is not.
	
	   - When you click the Encarta Dictionary QuickShelf button on the taskbar,
	     Encarta World English Dictionary 2000 does not start.
	
	- Encarta World English Dictionary 2000 is integrated with Office 2000, but
	  Bookshelf 2000 is not:
	
	   - When you start Word, both the Encarta World English Dictionary 2000 and
	     the Bookshelf 2000 buttons appear on the toolbar, but the Bookshelf 2000
	     button starts Encarta World English Dictionary 2000.
	
	   - When you start PowerPoint or Word, and then click Look Up Reference on the
	     Tools menu, Encarta World English Dictionary is listed in the "Available
	     reference titles" box, but Bookshelf 2000 is not.
	
	   - When you click the QuickShelf 2000 button on the taskbar, you are prompted
	     to insert the Bookshelf 2000 CD-ROM or to type the path to Bookshelf 2000
	     in the Drive box.
	
	CAUSE
	=====
	
	This behavior can occur if either of the following conditions are true:
	
	- The first symptom mentioned in the "Symptoms" section of this article can
	  occur if you install Encarta World English Dictionary 2000 before you install
	  Bookshelf 2000.
	
	- The second symptom mentioned in the "Symptoms" section of this article can
	  occur if you install Bookshelf 2000 before you install Encarta World English
	  Dictionary 2000.
	
	RESOLUTION
	==========
	
	To prevent this issue from occurring, install and use only one dictionary,
	preferably Encarta World English Dictionary 2000. Encarta World English
	Dictionary 2000 contains more features than Bookshelf 2000 and is more
	up-to-date.
	
	To install and use only one dictionary, use the appropriate method for the
	dictionary you want to use.
	
	Install Only Encarta World English Dictionary 2000
	--------------------------------------------------
	
	If Encarta World English Dictionary 2000 is integrated with Office 2000 but
	Bookshelf 2000 is not, remove Bookshelf 2000.
	
	If Bookshelf 2000 is integrated with Office 2000 but Encarta World English
	Dictionary 2000 is not:
	
	1. Remove Bookshelf 2000.
	
	2. Remove Encarta World English Dictionary 2000.
	
	3. Reinstall Encarta World English Dictionary 2000.
	
	Install Only Bookshelf 2000
	---------------------------
	
	If Bookshelf 2000 is integrated with Office 2000 but Encarta World English
	Dictionary 2000 is not, remove Encarta World English Dictionary 2000.
	
	If Encarta World English Dictionary 2000 is integrated with Office 2000 but
	Bookshelf 2000 is not:
	
	1. Remove Encarta World English Dictionary 2000.
	
	2. Remove Bookshelf 2000.
	
	3. Reinstall Bookshelf 2000.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
