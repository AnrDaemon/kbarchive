## Q237713: NFL Fever 2000: Graphics Displayed Using Green Color Palette

	Article: Q237713
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 23-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NFL Fever 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft NFL Fever 2000, the graphics in the game may be
	displayed using a green color palette.
	
	CAUSE
	=====
	
	This behavior can occur if the following conditions are true:
	
	- An Intergraph Intense 3D 1000 (Rendition Verite) video adapter is installed
	  in your computer.
	
	- The video driver for your video adapter is outdated.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest version of the video
	driver for your video adapter from the following Intergraph Computer Systems Web
	site:
	
	  http://www.intergraph.com/ics/voodoo/support.asp
