## Q237531: Encarta 2000: Macromedia Shockwave Director Documentation Error

	Article: Q237531
	Product(s): Microsoft Home Multimedia Titles
	Version(s): ; WINDOWS:
	Operating System(s): 
	Keyword(s): kbdocerr kbimu
	Last Modified: 06-AUG-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Encyclopedia 2000 
	- Microsoft Encarta Interactive World Atlas 2000 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes an error in the documentation that accompanies Macromedia
	Shockwave Director, which is included with the programs listed at the beginning
	of this article.
	
	MORE INFORMATION
	================
	
	The Macromedia Shockwave Director diagnostic movie page (Default.htm) mistakenly
	states that Microsoft Encarta Encyclopedia 2000 and Microsoft Encarta World
	Atlas 2000 shipped with version 7.0.0.204 of Macromedia Shockwave Director.
	
	The version of Macromedia Shockwave Director that shipped with Encarta
	Encyclopedia 2000 and Encarta World Atlas 2000 is 7.0.0.205.
	
	NOTE: The path to the Macromedia Shockwave Director diagnostic movie home page
	is
	
	  <drive>:/Support/Dirhtml/Default.htm
	
	where <drive> is the drive letter of your CD-ROM drive.
	
	This file is located on the Installation and Resources CD-ROM of both the Encarta
	Encyclopedia 2000 and the Encarta World Atlas 2000 CD-ROM set.
