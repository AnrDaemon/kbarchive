## Q237725: NFL Fever 2000: Textures Displayed Incorrectly or Game Hangs

	Article: Q237725
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbenv kbimu msgame
	Last Modified: 13-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NFL Fever 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to restore Microsoft NFL Fever 2000 by right-clicking the
	button on the taskbar or by pressing ALT+TAB, you may encounter either of the
	following symptoms:
	
	- The game textures may not be displayed correctly.
	
	- The program may stop responding (hang).
	
	CAUSE
	=====
	
	This behavior can occur if the following conditions are true:
	
	- A Diamond Viper V770 Ultra video adapter is installed in your computer.
	
	- The video driver for the Diamond Viper V770 Ultra video adapter is outdated.
	
	RESOLUTION
	==========
	
	To work around this issue, download and install the appropriate NVidia 2.08
	reference driver for your operating system from the following NVidia Web site:
	
	  http://www.nvidia.com
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft NFL Fever 2000 version
	1.0.
