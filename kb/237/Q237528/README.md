## Q237528: Encarta 2000: Unable to Play Song of Red-Winged Blackbird

	Article: Q237528
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 
	Operating System(s): 
	Keyword(s): kbsound kbimu
	Last Modified: 25-JUN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Encyclopedia 2000 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you attempt to play the song of the red-winged blackbird in the Bird
	Identification InterActivity in Microsoft Encarta Encyclopedia 2000, the song is
	not played.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Encarta Encyclopedia
	2000.
	
	Additional query words: multi multi-media media mm ee2k birdsong silent
