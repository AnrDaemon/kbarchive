## Q237533: Reference Suite 2000 Err Msg: Insufficient Hard Disk Space

{% raw %}

	Article: Q237533
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:; :
	Operating System(s): 
	Keyword(s): kberrmsg kbsetup kbimu
	Last Modified: 27-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Reference Suite 2000 
	- Microsoft Encarta Reference Suite 2001 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to install Microsoft Encarta Reference Suite 2000, you may
	receive an error message similar to the following even if sufficient hard disk
	space is available to install Encarta Reference Suite 2000:
	
	  There is insufficient hard disk space to install Microsoft Encarta Reference
	  Suite 2000.
	
	If you click Retry, Setup appears to finish successfully but you receive an error
	message stating that the program is not properly installed.
	
	CAUSE
	=====
	
	This behavior can occur if the following conditions are true:
	
	- Insufficient space is available on the hard disk on which Microsoft Windows
	  is installed.
	
	- You attempt to install Encarta Reference Suite 2000 on a hard disk other than
	  the hard disk on which Windows is installed.
	
	NOTE: You can also receive the initial error message if you attempt to install
	Encarta Reference Suite 2000 to the same hard disk on which Windows is
	installed.
	
	RESOLUTION
	==========
	
	To resolve this issue, make sure at least 48 megabytes (MB) of hard disk space
	is available on the hard disk on which Windows is installed.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Encarta Reference
	Suite 2000.
	
	MORE INFORMATION
	================
	
	Certain components of Encarta Reference Suite 2000 are installed on the same
	hard disk on which Windows is installed even if you install Encarta Reference
	Suite 2000 to a different hard disk.
