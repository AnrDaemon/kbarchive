## Q237510: Encarta 2000: Err Msg: An Error Occurred During Printing

{% raw %}

	Article: Q237510
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 
	Operating System(s): 
	Keyword(s): kberrmsg kbprint kbimu
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Encyclopedia 2000 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to print a state or provincial symbol or a map from Microsoft
	Encarta Encyclopedia 2000, you may receive the following error message:
	
	  An error occurred during printing.
	
	RESOLUTION
	==========
	
	To work around this issue:
	
	1. Click OK to dismiss the error message.
	
	2. On the Options menu, click Print.
	
	3. Click Print Setup.
	
	4. Click OK.
	
	5. Click Print.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Encarta Encyclopedia
	2000.
	
	MORE INFORMATION
	================
	
	This issue does not occur on Microsoft Windows NT 4.0-based computers.
