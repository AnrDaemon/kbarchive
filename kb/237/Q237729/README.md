## Q237729: NFL Fever: Only One of Two Installed Video Adapters Is Listed

	Article: Q237729
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbui kbimu msgame
	Last Modified: 14-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NFL Fever 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If two video adapters are installed in your computer, only one of the video
	adapters may appear in the list of drivers on the Display Options screen in
	Microsoft NFL Fever 2000.
	
	CAUSE
	=====
	
	This behavior can occur if two 3D video adapters are installed in your
	computer.
	
	This behavior does not occur if you use a 3D accelerator card to display 3D
	graphics.
	
	RESOLUTION
	==========
	
	To prevent this issue from occurring, disable one of the video adapters
	installed in your computer. To do this, consult the printed documentation
	provided with the video adapter that you want to disable.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft NFL Fever 2000,
	version 1.0.
