## Q237718: NFL Fever 2000: Weather Remains Clear When You Play the Game

	Article: Q237718
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 12-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NFL Fever 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you select either weather conditions of rain or snow for a game in
	Microsoft NFL Fever 2000, the weather may remain clear when you play the game.
	
	CAUSE
	=====
	
	This behavior can occur if a video adapter based on the 3Dfx Voodoo II chip set
	is installed in your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest video driver for your
	video adapter from the following 3Dfx Interactive Web site:
	
	  http://www.3dfx.com/view.asp?PAGE=nusdrivers
