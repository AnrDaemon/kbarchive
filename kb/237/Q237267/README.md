## Q237267: Encarta 2000: Cannot Start Program After Upgrade to Windows 2000

	Article: Q237267
	Product(s): Microsoft Home Multimedia Titles
	Version(s): ; WINDOWS:
	Operating System(s): 
	Keyword(s): kbenv kbimu
	Last Modified: 25-JUN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Encyclopedia 2000 
	- Microsoft Encarta Reference Suite 2000 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you upgrade to Microsoft Windows 2000, you may be unable to start any of
	the following programs:
	
	- Microsoft Encarta Encyclopedia 2000
	- Microsoft Encarta Reference Suite 2000
	- Microsoft Encarta Research Organizer
	
	RESOLUTION
	==========
	
	To work around this issue, remove and then reinstall the programs listed in the
	"Symptoms" section of this article. To do this:
	
	1. Click Start, point to Settings, and then click Control Panel.
	
	2. Double-click Add/Remove Programs.
	
	3. In the list of installed programs, click one of the programs listed in the
	  "Symptoms" section of this article.
	
	4. Click Add/Remove.
	
	5. Follow the instructions on the screen to remove the program.
	
	6. Repeat steps 3-5 for each program listed in the "Symptoms" section of this
	  article.
	
	7. Close Control Panel.
	
	8. Insert the Installation and Resources CD-ROM for the program you want to
	  reinstall, and then follow the instructions on the screen to reinstall the
	  program.
	
	9. Repeat step 8 for each program you want to reinstall.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
