## Q237270: Encarta 2000: How to Troubleshoot Issues Using Registry Keys

	SUMMARY
	=======
	
	Both the Encarta Encyclopedia 2000 and the Encarta Reference Suite 2000
	Installation and Resources CD-ROM contain a set of registry keys you can use to
	modify program behavior.
	
	The following table lists both the file names and the functions of these registry
	keys as well as their location on the Installation and Resources CD-ROM.
	
	CAUTION: Do not modify the contents of these registry keys unless you are
	explicitly instructed to do so by a Microsoft Support Professional.
	
	+------------------------------------------------------------------------------------------------------------------------------------------+
	| File name    | Function                                                                                          | Location              | 
	+------------------------------------------------------------------------------------------------------------------------------------------+
	| Admin.reg    | Network Administrator- Sets program to look for yearbook updates locally                          | \Support\Regfiles\Ee  | 
	+------------------------------------------------------------------------------------------------------------------------------------------+
	| Dl_off.reg   | Download Reminder Off                                                                             | \Support\Regfiles\Ee  | 
	+------------------------------------------------------------------------------------------------------------------------------------------+
	| Dl_on.reg    | Download Reminder On                                                                              | \Support\Regfiles\Ee  | 
	+------------------------------------------------------------------------------------------------------------------------------------------+
	| Eesndoff.reg | Encarta Encyclopedia Sound Off                                                                    | \Support\Regfiles\Ee  | 
	+------------------------------------------------------------------------------------------------------------------------------------------+
	| Eesndon.reg  | Encarta Encyclopedia Sound On                                                                     | \Support\Regfiles\Ee  | 
	+------------------------------------------------------------------------------------------------------------------------------------------+
	| Eevmoff.reg  | Encarta Encyclopedia View Off                                                                     | \Support\Regfiles\Ee  | 
	+------------------------------------------------------------------------------------------------------------------------------------------+
	| Eevmon.reg   | Encarta Encyclopedia View On                                                                      | \Support\Regfiles\Ee  | 
	+------------------------------------------------------------------------------------------------------------------------------------------+
	| Local.txt    | Network administrator - Sets program to look for yearbook updates locally in a specified location | \Support\Regfiles\Ee  | 
	+------------------------------------------------------------------------------------------------------------------------------------------+
	| Mm_off.reg   | Mindmaze Off                                                                                      | \Support\Regfiles\Ee  | 
	+------------------------------------------------------------------------------------------------------------------------------------------+
	| Mm_on.reg    | Mindmaze On                                                                                       | \Support\Regfiles\Ee  | 
	+------------------------------------------------------------------------------------------------------------------------------------------+
	| Server.txt   | Network Administrator - Sets program to look for yearbook updates on a specified network location | \Support\Regfiles\Ee  | 
	+------------------------------------------------------------------------------------------------------------------------------------------+
	| Vcerroff.reg | Voice Control Error Off                                                                           | \Support\Regfiles\Ee  | 
	+------------------------------------------------------------------------------------------------------------------------------------------+
	| Vcerron.reg  | Voice Control Error On                                                                            | \Support\Regfiles\Ee  | 
	+------------------------------------------------------------------------------------------------------------------------------------------+
	| Erovwoff.reg | Modifies view settings for Encarta Research Organizer 2000                                        | \Support\Regfiles\Ero | 
	+------------------------------------------------------------------------------------------------------------------------------------------+
	| Erovwon.reg  | Modifies view settings for Encarta Research Organizer 2000                                        | \Support\Regfiles\Ero |

	For additional information about the function of registry keys when
	troubleshooting issues in Encarta 2000, click the article number below to view
	the article in the Microsoft Knowledge Base:
	
	  Q238922 Encarta 2000: Detailed Description of Registry Key Functions
	
	MORE INFORMATION
	================
	
	To use these registry keys to troubleshoot issues in Encarta 2000:
	
	1. Insert the Encarta Encyclopedia 2000 or the Encarta Reference Suite 2000
	  Installation and Resources CD-ROM into your CD-ROM or DVD-ROM drive. Press
	  and hold down the SHIFT key as you insert the CD-ROM to prevent the program
	  from starting automatically.
	
	2. Click Start, and then click Run.
	
	3. In the Open box, type the following line and then click OK
	
	  "<cd-rom>:\Support\Regfiles" (without the quotation marks)
	
	  where <cd-rom> is the drive letter of your CD-ROM or DVD-ROM drive.
	
	4. Double-click the Ee or Ero folder.
	
	5. Double-click the registry key you want to use to modify program behavior, and
	  then follow the instructions on the screen if necessary.
	
	6. Start Encarta Encyclopedia 2000 and test the results.
