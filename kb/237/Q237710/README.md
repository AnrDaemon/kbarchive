## Q237710: NBA 2000: Pointer Disappears When Pointing at Help Menu Items

{% raw %}

	Article: Q237710
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbimu msgame
	Last Modified: 28-AUG-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NBA Inside Drive 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click the Help (?) button in NBA Inside Drive 2000, the pointer may
	flicker or disappear when you point to one of the options on the Help menu.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft NBA Inside Drive 2000,
	version 1.0.
