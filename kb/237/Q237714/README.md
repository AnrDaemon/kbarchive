## Q237714: NFL Fever 2000: Quits Unexpectedly When You Select a Play

{% raw %}

	Article: Q237714
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 12-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NFL Fever 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you select a play in Microsoft NFL Fever 2000, the program may quit
	unexpectedly.
	
	CAUSE
	=====
	
	This behavior can occur if the following conditions are true:
	
	- A Matrox Mystique video adapter is installed in your computer.
	
	- The video driver for the Matrox Mystique video adapter is outdated.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest video driver for the
	Matrox Mystique video adapter from the following Matrox Web site:
	
	  http://www.matrox.com/mgaweb/drivers/latest_drivers/home.htm
