## Q237716: NFL Fever 2000: User Interface Text Is Illegible

	Article: Q237716
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbui kbimu msgame
	Last Modified: 19-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NFL Fever 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start Microsoft NFL Fever 2000, the user interface text may be
	illegible.
	
	CAUSE
	=====
	
	This behavior can occur if the fonts used in the NFL Fever 2000 user interface
	are not installed properly during Setup.
	
	RESOLUTION
	==========
	
	To resolve this issue:
	
	1. Quit NFL Fever 2000.
	
	2. Click Start, point to Settings, and then click Control Panel.
	
	3. Double-click Fonts.
	
	4. Right-click Trebuchet MS, click Delete, and then click Yes.
	
	5. Repeat step 4 for each of the following fonts:
	
	   - Trebuchet MS Bold
	   - Trebuchet MS Bold Italic
	   - Trebuchet MS Italic
	
	6. Close the Fonts window, and then close Control Panel.
	
	7. Insert the NFL Fever 2000 CD-ROM into your CD-ROM drive. Press and hold down
	  the SHIFT key as you insert the CD-ROM to prevent NFL Fever 2000 from
	  starting automatically.
	
	8. Double-click the My Computer icon on the desktop.
	
	9. Right-click the Football (CD-ROM drive) icon, and then click Explore.
	
	10. Double-click the Game folder.
	
	11. In the right pane, click to select the Trebuc.ttf font file.
	
	12. Press and hold down the CTRL key as you click to select each of the
	  following font files:
	
	   - Trebucbd.ttf
	   - Trebucbi.ttf
	   - Trebucit.ttf
	
	13. On the Edit menu, click Copy.
	
	14. Close all open windows on the desktop.
	
	15. Click Start, point to Settings, and then click Control Panel.
	
	16. Double-click Fonts.
	
	17. On the Edit menu, click Paste.
	
	18. Close the Fonts window, and then close Control Panel.
	
	19. Restart the computer.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft NFL Fever 2000,
	version 1.0.
