## Q237727: NFL Fever 2000: Color of Taskbar Button Changes

	Article: Q237727
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 12-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NFL Fever 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you press ALT+TAB to switch from the Microsoft NFL Fever 2000 Players Guide
	to the Microsoft Windows desktop, the color of the Start button and the NFL
	Fever 2000 taskbar button may change.
	
	CAUSE
	=====
	
	This behavior can occur if either of the following conditions are true:
	
	- A Diamond Viper V770 Ultra video adapter is installed in your computer.
	
	- A video adapter based on the Voodoo II chip set is installed in your
	  computer.
