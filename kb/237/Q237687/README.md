## Q237687: Obtaining the ITSEC Evaluated Y2K Update Patch for WinNT 4.0 SP3

{% raw %}

	Article: Q237687
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 20-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Server, Enterprise Edition version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Target of Evaluation for ITSEC (http://www.itsec.gov.uk/) certified Windows
	NT 4.0 Service Pack 3 (UK ITSEC Certification Report No. P121) optionally
	includes a Y2K software update dated March 27, 1998. To obtain this version of
	the Y2K software update, contact Microsoft Product Support Services. For a
	complete list of Microsoft Product Support Services phone numbers and
	information on support costs, please go to the following address on the World
	Wide Web:
	
	  http://support.microsoft.com/default.aspx?scid=fh;EN-US;CNTACTMS
	
	
	MORE INFORMATION
	================
	
	The March 27, 1998 Y2K software update, optionally identified as part of the
	target of evaluation of ITSEC certified Windows NT 4.0 Service Pack 3 (SP3),
	contains the following updates:
	
	  Q175093 User Manager Does Not Recognize February 2000 As a Leap Year
	
	  Q180122 After Changing the Time, Windows NT May Skip a Day
	
	  Q183123 Find Files Displays Garbled Date if Year is 2000 or Greater
	
	  Q183125 Custom Dates in Office Display Date Incorrectly After 2000
	
	Users of the March 27, 1998 Y2K software update should be aware that additional
	Y2K-specific issues have been identified since the ITSEC evaluation of Windows
	NT 4.0 SP3. Microsoft has subsequently released a newer Y2K software update at
	the following Internet address:
	
	  ftp://ftp.microsoft.com/bussys/winnt/winnt-public/fixes/usa/nt40/hotfixes-postSP3/y2k2-fix/
	
	The additional issues addressed by this latest software update are listed at the
	following location:
	
	  Windows NT 4.0 SP3 Y2K Compliance Document
	  (http://microsoft.com/technet/year2k/product/user_view68532EN.htm)
	
	For more information on E3/F-C2 Evaluation, go to the following Microsoft Web
	site:
	
	  http://www.microsoft.com/security/default.asp
	
	
	Additional query words: IT Security Evaluation Criteria FC2-E3 year 2000
