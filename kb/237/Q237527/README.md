## Q237527: Encarta Dictionary 2000: Pointing Device Behaves Unexpectedly

	Article: Q237527
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:; :
	Operating System(s): 
	Keyword(s): kbhw kbinterop kbimu kbHardware
	Last Modified: 25-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta World English Dictionary 
	- Microsoft Encarta World English Dictionary 2001 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you use Microsoft Encarta World English Dictionary 2000 with version 2.2c
	or earlier of the Microsoft IntelliPoint software, your pointing device may
	behave unexpectedly.
	
	MORE INFORMATION
	================
	
	To resolve any issues you may experience between Encarta World English
	Dictionary 2000 and the IntelliPoint software, download and install the latest
	version of the IntelliPoint software. To do this, visit the following Microsoft
	Web site:
	
	  http://www.microsoft.com/products/hardware/mouse/driver/default.htm
