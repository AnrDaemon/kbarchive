## Q237719: NFL Fever 2000: Stops Responding When You Start Program

{% raw %}

	Article: Q237719
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 12-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NFL Fever 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start Microsoft NFL Fever 2000, the program may stop responding.
	
	CAUSE
	=====
	
	This behavior can occur if the following conditions are true:
	
	- A Hercules Terminator Beast video adapter is installed in your computer.
	
	- The video driver for the Hercules Terminator Beast video adapter is outdated.
	
	- The resolution is set to 1024x768 in the NFL Fever 2000 Display options
	  window.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest video driver for the
	Hercules Terminator Beast video adapter from the following Hercules Web site:
	
	  http://www.hercules.com/support/drivers/
