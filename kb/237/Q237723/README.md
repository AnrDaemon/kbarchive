## Q237723: NFL Fever 2000: Poor Graphics Performance with ATI Rage IIc AGP

{% raw %}

	Article: Q237723
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbhw kbimu msgame kbHardware
	Last Modified: 25-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NFL Fever 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft NFL Fever 2000, you may experience poor graphics
	performance and a slow video frame rate in the game.
	
	CAUSE
	=====
	
	This behavior can occur if an ATI Rage IIc AGP video adapter is installed in
	your computer.
	
	MORE INFORMATION
	================
	
	Microsoft recommends that you do not run NFL Fever 2000 if an ATI Rage IIc AGP
	video adapter is installed in your computer.

	independent of Microsoft; we make no warranty, implied or otherwise, regarding
	these products' performance or reliability.
