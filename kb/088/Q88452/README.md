## Q88452: Printing Score Card on LaserJet Printer with 512K

{% raw %}

	Article: Q88452
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	On some laser printers with only 512K, such as the Okidata OL-400, the entire
	score card does not print. To print the complete score card you need to set the
	printer driver's graphics resolution to 75 dpi.
