## Q88472: Golf: No Sound/Reward for a Hole-in-One

	Article: Q88472
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	If you make a hole-in-one there is no sound or reward. When you make an eagle,
	the commentator says "nice eagle." This will be addressed in a future version of
	Golf.
