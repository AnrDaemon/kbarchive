## Q88778: SMTP: Software &amp; Hardware Required for Downstream Postoffice

	Article: Q88778
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to SMTP, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The downstream postoffice requires only the Microsoft Mail Gateway Access to
	SMTP package, which includes one floppy disk labeled "The Microsoft Mail Gateway
	Access to SMTP" (the gateway access component).
