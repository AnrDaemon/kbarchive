## Q88515: Additional Golf Courses Available from Access Software

	Article: Q88515
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-AUG-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Golf for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	All of the Access Software LINKS championship courses are compatible with Golf
	for Windows. The following information is available in the New Game dialog box
	by selecting [Information on additional courses] from the Course list, or see
	page 2 of the "Player's Guide for Microsoft Golf:
	
	  Torrey Pines South Course -- is shipped with Golf for Windows.
	
	List of additional courses available as of 3/15/94....
	
	  Banff Springs - Banff, Alberta, Canada
	  Barton Creek Country Club - Austin, Texas
	  Bay Hill Club - Orlando, Florida
	  Bountiful Municipal Golf Club - Bountiful, Utah
	  Dorado Beach East Course (Hyatt) - Dorado, Puerto Rico
	  Firestone Country Club - Akron, Ohio
	  Innisbrook - Tarpon Springs, Florida
	  Mauna Kea - Island of Hawaii, Hawaii
	  Pebble Beach, Montery Peninsula, California
	  Pinehurst Resort & Country Club - North Carolina
	  The Belfry - England
	  Troon North - Scottsdale, Arizona

	359-2900 or fax (801) 359-2968. The Access Software address is:
