## Q88204: PageMaker 4.0 Cannot Print Duplex on HP IID

	Article: Q88204
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	Aldus PageMaker version 4.0 cannot duplex on a Hewlett-Packard (HP)
	LaserJet IID from Windows. This is a PageMaker limitation.
	
	Contact Aldus technical support to find out when a correction will be
	available.
	
	
	The Aldus PageMaker product included here is manufactured by a vendor
	independent of Microsoft; we make no warranty, implied or otherwise,
	regarding this product's performance or reliability.
