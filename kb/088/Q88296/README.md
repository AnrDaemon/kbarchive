## Q88296: LANtastic Server Causes Communications Slow Down

	Article: Q88296
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When running Microsoft Windows version 3.1 from a LANtastic version 4.1 server,
	serial data communication speeds may be degraded due to the network software.
	Bit rates of 9600 bits per second (BPS) or higher may not run reliably.
	
	Contact LANtastic technical support for possible solutions to this problem.
	
	WORKAROUND
	==========
	
	To work around this problem, run your communication software at speeds of 2400
	BPS.
	
	Additional query words: 3.10 3.11 LANTNET.DRV 3rdparty
