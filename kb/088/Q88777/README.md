## Q88777: SMTP: RFCs Supported by the Gateway

	Article: Q88777
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 31-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to SMTP, version 3.0 
	-------------------------------------------------------------------------------
	
	These requests for comments (RFCs) are supported by the Microsoft Mail
	Gateway to SMTP:
	
	
	- RFC-821 (SMTP) for message exchange
	
	- RFC-822 for message format
	
	- RFC-1154 for attachment support
	
	REFERENCES
	==========
	
	Titled attachments on page 80 (in Appendix C, "Gateway Features that Affect Mail
	Users") of the "Microsoft Mail Gateway to SMTP Administrator's Guide"
