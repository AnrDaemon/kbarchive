## Q88310: Configure Qumax's Mighty KT30S Keyboard as Genius Mouse

	Article: Q88310
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	According to Qumax technical support, the Qumax Mighty KT30S keyboard with
	built-in three-button trackball should be set up as a "Genius serial mouse on
	COM1" or "Genius serial mouse on COM2" in Microsoft Windows version 3.1.
	
	MORE INFORMATION
	================
	
	For steps on how to configure the keyboard/trackball for Genius mouse on COM1 or
	COM2, please contact Qumax technical support.
	
	The Qumax product included here is manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
