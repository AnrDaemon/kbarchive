## Q88460: Controlling Line Wrap in Terminal

{% raw %}

	Article: Q88460
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When receiving incoming text from a remote service in Terminal, you may not see
	the entire line of text being sent. To correct this problem, adjust the Line
	Wrap setting under Terminal Preferences.
	
	MORE INFORMATION
	================
	
	Steps to Change Line Wrap
	-------------------------
	
	1. From the Settings menu, choose Terminal Preferences.
	
	2. Select the Line Wrap box and then choose 80 or 132 in the Columns box.
	
	If you are still missing the last couple of characters, then you may need to
	disable Terminal's Scroll Bars by clearing the Show Scroll Bars check box.
