## Q88446: Top View Doesn't Show Total Yardage of Hole in Golf

	Article: Q88446
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	When you hold down the left mouse button in the Top View window in Microsoft
	Golf for Windows version 1.0, the Ball To Cursor and Cursor To Pin yardage show,
	but the total yardage of the hole isn't shown. This is due to the limited area
	available in the Top View window.
