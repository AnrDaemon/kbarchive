## Q88264: Dell 433DE Requires 1.00 A07 BIOS for More than 16 MB

	Article: Q88264
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Dell Computer Corporation's model 433DE computer requires a BIOS version of 1.00
	A07 or later for Microsoft Windows 3.1's HIMEM.SYS version 3.07 to recognize
	memory configurations greater than 16 megabytes (MB).
	
	For information about obtaining an updated BIOS, Contact Dell Computer Corp.
	
	MORE INFORMATION
	================
	
	The 433DE is a 486 EISA computer. Using HIMEM.SYS's /EISA switch was not needed
	for HIMEM.SYS to see the greater than 16 MB memory configuration. This
	information may apply to other Dell computers. Many Dell computers use "flash
	ROM" in which you can update your ROMs by using a floppy disk.
	
	Additional query words: 3rdparty winboot 3.10 3.11
