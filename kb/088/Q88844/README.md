## Q88844: QMS ColorScript 100 Has No Legal Size Option

{% raw %}

	Article: Q88844
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The QMS ColorScript 100 driver that ships with Windows 3.1 does not have the
	legal size (8.5 x 14) option.
	
	To use this size, contact QMS to obtain the QMS ColorScript 100 printer driver
	version 2.10.
	
	MORE INFORMATION
	================
	
	QMS has confirmed that the QMS ColorScript 100 printer driver version 2.10 for
	Windows 3.1 is now available. This driver gives the user the option of selecting
	the legal (8.5 x 14) paper size.
	
	The driver is available via modem, or you can call QMS to request the driver. If
	you have a modem, dial the QMS bulletin board service (BBS), and select message
	#13524 located in PostScript Fig.
	
	The QMS ColorScript driver included here is manufactured by a vendor independent
	of Microsoft; we make no warranty, implied or otherwise, regarding this
	product's performance or reliability.
