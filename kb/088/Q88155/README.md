## Q88155: Notepad Print Setup May Not Appear

	Article: Q88155
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-JAN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When using the Notepad and Calendar applications included with Windows 3.1, the
	Print Setup dialog box may not appear after changing the default printer in the
	Control Panel while these applications are running.
	
	WORKAROUND
	==========
	
	Save the current file, exit, and restart the application.
	
	Additional query words: 3.10 3.11 notepad calendar set up
