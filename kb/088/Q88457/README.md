## Q88457: Choosing Golf Help in Post Shot Selection with DTK BIOS Hangs

{% raw %}

	Article: Q88457
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	On computers with a DTK BIOS, the system freezes if you choose Microsoft Golf
	for Windows Help from the Post Shot Selections window. To regain control of your
	computer, turn the computer's Turbo feature off and back on again.
