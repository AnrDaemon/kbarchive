## Q88424: DOS Applications Running in a Window Can't Use a Mouse

	Article: Q88424
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Windows versions 3.0 and 3.0a do not support using a mouse in an MS-DOS-based
	application running in a window.
	
	WORKAROUND
	==========
	
	- Upgrade to Windows 3.1, which supports this functionality.
	
	- Run the MS-DOS-based application in full-screen mode to use the mouse.
	
	Additional query words: 3.00 3.00a
