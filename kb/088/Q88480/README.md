## Q88480: Some Video Seven Drivers Cause Display Problems

{% raw %}

	Article: Q88480
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	Some Video Seven video drivers cause display problems in Microsoft Golf. The
	graphics for both the main and top view windows will be painted under the window
	frames.
	
	Make sure you are using the Video Seven driver shipped with Windows 3.1 or the
	latest driver for your video card. Contact Headland Technology Inc. for updated
	drivers.
	
	Additional query words: 1.00 videoseven video-seven video 7
