## Q88474: Printing in Landscape Truncates Golf Score Card

{% raw %}

	Article: Q88474
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	When you print the Microsoft Golf for Windows score card in landscape mode the
	bottom of the card will be truncated. The score card is designed to print in
	portrait mode. Change the orientation in the Windows printer driver to portrait.
