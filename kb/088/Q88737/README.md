## Q88737: Text Does Not Appear with Picture Publisher 3.0

	Article: Q88737
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use Micrografx's Picture Publisher version 3.0 with Microsoft Windows
	3.1, text may not appear when it is typed. This problem only occurs when a
	256-color video driver is installed.
	
	RESOLUTION
	==========
	
	To correct this problem, either install a 16-color driver or upgrade to Picture
	Publisher version 3.1.
	
	MORE INFORMATION
	================
	
	For more information, contact Micrografx.

	performance or reliability.
