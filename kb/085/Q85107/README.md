## Q85107: Tektronix PostScript Printer Driver Does Not Support TrueType

	Article: Q85107
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	According to Tektronix, Tektronix PostScript printer driver version 1.3 does not
	support TrueType fonts in Microsoft Windows version 3.1.
	
	MORE INFORMATION
	================
	
	The disk that Tektronix sends to customers is labeled "Windows 3.0/3.1 Printer
	Driver," but this driver only works correctly with Windows 3.0. Tektronix is
	currently working on version 2.0 of the printer driver, which will be fully
	compatible with Windows 3.1.

	or reliability.
