## Q85255: AT&amp;T 570 Printer Not Supported Under Windows 3.1

{% raw %}

	Article: Q85255
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The AT&T 570 printer is not supported under the Microsoft Windows version
	3.1. However, AT&T recommends emulating the IBM Proprinter XL or Epson
	FX-86e printer as a workaround to print under Windows.
	
	
	MORE INFORMATION
	================
	
	For more information, contact AT&T technical support.
	
	
	The product included here, the AT&T 570 printer, is manufactured by vendors
	independent of Microsoft; we make no warranty, implied or otherwise, regarding
	this product's performance or reliability.
