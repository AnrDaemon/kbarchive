## Q85964: Windows 3.1 Paintbrush Loses Part of Display

	Article: Q85964
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 3.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the Paintbrush application included with Microsoft Windows version
	3.1, a portion of a display can be lost after you use the Flip Horizontal or
	Flip Vertical feature.
	
	WORKAROUND
	==========
	
	You should immediately copy the object to the Clipboard and paste it into a new
	picture, then cut and paste it back into the original.
	
	Additional query words: 3.10 paint brush
