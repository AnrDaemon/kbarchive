## Q85180: Windows 3.1 Screen Savers Do Not Run with Tutorial Active

	Article: Q85180
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): win31
	Last Modified: 07-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	Microsoft Windows screen savers do not function when full-screen online
	tutorials are active. This behavior is by design.
	
	Additional query words: excel winword word screensaver 3.10 3.11
