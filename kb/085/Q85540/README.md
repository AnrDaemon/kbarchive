## Q85540: Sending a Break Sequence in Terminal

	Article: Q85540
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	If you want to send a break in Microsoft Windows version 3.1's
	Terminal program, use the following:
	
	  CTRL+BREAK
	
	Other escape sequences can be found in the "Windows User's Guide" for
	version 3.1 in Appendix B.
	
	Additional query words: 3.1 3.11 3.10 3/10/92
