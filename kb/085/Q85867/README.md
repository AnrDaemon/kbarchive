## Q85867: Novell NWPOPUP Truncates Messages at 40 Characters

	Article: Q85867
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When the Novell NetWare NWPOPUP.EXE program that comes with Microsoft Windows
	versions 3.0 and 3.1 is used to receive incoming messages, the messages will be
	truncated at 40 characters.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in NWPOPUP.EXE version 3.0 and 3.1.
	We are researching this problem and will post new information here as it becomes
	available.
	
	Additional query words: 3.0 3.0a 3.00 3.10 3.1 3.11 limit
