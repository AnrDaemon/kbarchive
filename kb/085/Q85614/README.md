## Q85614: FAX: Primary Causes of Mail Failures to Fax Machines

{% raw %}

	Article: Q85614
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to Fax, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In most cases, mail fails to be sent to a fax machine for one of three reasons:
	
	- Wrong phone number
	
	- Incorrect file type
	
	- Phone busy or off the hook
	
	Each of these situations causes an error message to be displayed on the fax
	gateway screen.
