## Q85387: Limitations in Windows File Association Capabilities

{% raw %}

	Article: Q85387
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	When using Microsoft Windows File Manager File menu Association command you
	cannot:
	
	- Associate a file that does not have an extension.
	
	- Associate the same extension to two different programs and be able to choose
	  between them when a data file with that extension is selected in File
	  Manager.
	
	It is possible that a third-party application may have been written to
	provide this functionality.
	
	Additional query words: 3.10 3.1 3.11 associate regedit
