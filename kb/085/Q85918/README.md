## Q85918: TI Travelmate 3000 May Have Distorted Display During Setup

	Article: Q85918
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 3.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you install Microsoft Windows version 3.1 on a Texas Instruments (TI)
	Travelmate 3000, you notice a distorted display after Setup switches into
	Windows mode.
	
	WORKAROUND
	==========
	
	To avoid this problem, select the VGA (version 3.0) video driver.
	
	
	MORE INFORMATION
	================
	
	For more information, contact Texas Instruments Technical Support.
