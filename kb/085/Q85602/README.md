## Q85602: FAX: What Occurs in the Event of a Mail Failure

	Article: Q85602
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 04-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to Fax, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If the Microsoft Mail Gateway to Fax is unable to transmit an outgoing mail
	item, the gateway attempts to return the mail to the sender.
	
	If an incoming fax item cannot be delivered, the gateway copies the file from the
	fax board queue to the fax subdirectory in the Microsoft Mail database.

	==========
