## Q85653: IBM QuickWriter Missing Paper Sources

	Article: Q85653
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): win31
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 3.1 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	The IBM QuickWriter 5204 printer driver version 1.0 for Microsoft Windows
	version 3.1 only supports tractor feed for a paper source.
	
	There are cut sheet feeders available from IBM for this printer, but the driver
	IBM5204.DRV does not support these sources.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in IBM5204.DRV version 1.0. We are
	researching this problem and will post new information here as it becomes
	available.
