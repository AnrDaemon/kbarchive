## Q85162: Forcing the Installation of Only MS-DOS 5 or Windows 3.1

{% raw %}

	Article: Q85162
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 3.1 
	- Microsoft MS-DOS operating system version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Switches, parameters, and/or command line settings do not exist to force the
	Windows & MS-DOS 5 for IBM PS/2 Setup program to install only Windows 3.1 or
	MS-DOS 5. However, there are methods to do this.
	
	To Install MS-DOS 5 Only
	------------------------
	
	To install MS-DOS 5 only, cancel the Windows installation by pressing F3 twice
	when you are prompted for the Windows 3.1 directory.
	
	To Install Windows 3.1 Only
	---------------------------
	
	If you have already installed MS-DOS 5 and want to install or reinstall only
	Windows 3.1, run the Microsoft Windows and MS-DOS 5 for IBM PS/2 Setup program
	again. It detects that MS-DOS 5 is already installed and installs only Microsoft
	Windows 3.1.
