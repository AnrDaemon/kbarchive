## Q85603: FAX: Retry Mechanism the Gateway Responds To

	Article: Q85603
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to Fax, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Mail Gateway to Fax has a hard coded retry count of 3. The gateway
	tries three times to deliver a mail item. Once it has made three attempts, the
	gateway returns the mail item.
	
	REFERENCES
	==========
	
	"Microsoft Mail Gateway to Fax Administrator's Guide"
