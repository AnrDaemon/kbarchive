## Q85355: FAX: Parameters Required in the PROFILE.GLB File

{% raw %}

	Article: Q85355
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to Fax, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	These parameters in the Microsoft Mail Gateway to Fax PROFILE.GLB file should be
	modified.
	
	 Parameter            Description
	 ---------            -----------
	
	 FAX Server           Drive and path to data files
	 Network name         Name of network for gateway postoffice
	 Postoffice name      Name of gateway postoffice
	 Administrator        Mailbox ID of person to receive incoming faxes
	 FAX#                 Fax number
	 Logo file            Path to logo file
	 Printer path         Path to printer file, PPB.EXE
	 FAX board            Fax board
	 Printer port         Printer Port
	 V2.0 compatibility   Modify if connecting to version 2.0 postoffices
	
	A full description of these and other parameters is in the "Microsoft Mail
	Gateway to Fax Administrator's Guide."
