## Q85374: HP LaserJet Does Not Support International Cartridge

	Article: Q85374
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The Microsoft Windows operating system version 3.1 printer driver for the
	Hewlett-Packard (HP) LaserJet Series II does not include the option of choosing
	the HP International Collection cartridge.
	
	NOTE: This is not the same as the C cartridge International 1.
	
	MORE INFORMATION
	================
	
	To Use the International Cartridge
	----------------------------------
	
	1. Locate the disk that came with the cartridge. The file INTL.PCM is on this
	  disk.
	
	2. Run Control Panel.
	
	3. Choose the Printers icon and choose the Setup button.
	
	4. Choose the Fonts button and then the Add Fonts button.
	
	5. Insert the HP disk in the floppy disk drive to load the .PCM file.
	
	These steps also apply to any cartridge designed for use with Windows.
