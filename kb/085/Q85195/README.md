## Q85195: PC Win: System Requirements for Mail for Windows

	Article: Q85195
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following are the system requirements for Microsoft Mail for Windows,
	version 3.0:
	
	- 80286 or higher processor (80386 recommended)
	
	- 2 megabytes (MB) of memory (4 MB recommended)
	
	- Microsoft Windows, version 3.0a or later
	
	Note: If you are running Windows 3.0a on a 2 MB machine, it must be running in
	Enhanced 386 Mode to load Mail. If it is a 2 MB 80286 machine, Windows version
	3.1 must be used to run Mail.
