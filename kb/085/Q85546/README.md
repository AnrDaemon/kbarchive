## Q85546: Linotronic Printer Drivers Not Available for Windows 3.1

	Article: Q85546
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	Linotype Company has confirmed that there are currently no Microsoft Windows 3.1
	printer drivers available for the following Linotronic printers:
	
	- Linotronic 70/90
	
	- Linotronic 170/190
	
	- Linotronic 260
	
	- Linotronic 830/930
	
	- Laser Printer 60
	
	If possible, emulate a Linotronic printer that has a Windows 3.1 driver.
	
	MORE INFORMATION
	================
	
	Although there are no printer drivers available for the above Linotronic
	printers, according to Linotype Company, the following Windows 3.1 drivers can
	be used:
	
	- Linotronic 200 for the 70/170
	
	- Linotronic 230 for 90/190
	
	- Linotronic 330 for 260
	
	- Linotronic 630 for 830/930
	
	There is no compatible emulation for the Laser Printer 60.
	
	NOTE: If you use these Linotronic emulations, Linotype states that you should not
	use the advanced options for these drivers.
	
	Linotype Company is a hardware manufacturer of image setters and printers.
	Linotype Company does not make printer drivers.
	
	For information concerning Linotype printers or image setters, contact the
	Linotype Company.
	
	The products included here are manufactured by vendors independent of Microsoft;
	we make no warranty, implied or otherwise, regarding these products' performance
	or reliability.
