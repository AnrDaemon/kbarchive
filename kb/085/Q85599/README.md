## Q85599: No Duplex Printing Options on HP IID PostScript Driver

	Article: Q85599
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The driver for the Hewlett-Packard (HP) LaserJet IID PostScript printer version
	3.5, included with Microsoft Windows version 3.1, does not support duplex
	printing (that is, printing on both sides of the paper).
	
	Duplex printing is also unavailable with the HP LaserJet IID PostScript driver
	HPIID522.WPD version 52.2 from the Supplemental Driver Library (SDL) for Windows
	version 3.0.
