## Q85940: SYSADMIN.WRI Not Included with Windows &amp; MS-DOS 5 for IBM PS/2

{% raw %}

	Article: Q85940
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 3.1 
	- Microsoft MS-DOS operating system version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Page 98 of the "Getting Started" guide for Windows & MS-DOS 5 for IBM PS/2
	explains how to place the Windows files on a network server. At the end of the
	page, there is a note describing how to view the SYSADMIN.WRI file to obtain
	further information about installing Windows on a network server. However, there
	is no such file included with Windows & MS-DOS 5 for IBM PS/2.
	
	MORE INFORMATION
	================
	
	The information contained in the SYSADMIN.WRI file was incorporated into the
	"Windows Resource Kit" manual for version 3.1. For more detailed information
	about using Windows 3.1 in a network environment and for information about
	obtaining the Windows Resource Kit, contact the Microsoft Sales Information
	Center (MSIC) at (800) 426-9400.
