## Q85051: FFAPI: The Components of Gateway FFAPI

	Article: Q85051
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Software Development Kit: FFAPI for Gateways and Applications, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Here is a list of the components of the Gateway File Format Application
	Programming Interface (FFAPI):
	
	Gateway FFAPI
	-------------
	
	  Filename      Description
	  --------      -----------
	
	  GTWCFG.EXE    Creates FFAPI postoffice on MS Mail postoffice
	                that acts as message store for foreign mail system.
	
	  GTWPUT.EXE    Inserts mail items into MS Mail database.
	
	  GTWGET.EXE    Extracts mail destined for FFAPI postoffice from
	                MS Mail database.

	==========

	API"
