## Q85053: FFAPI: The FFAPI Postoffice and How It is Used

	Article: Q85053
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Software Development Kit: FFAPI for Gateways and Applications, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	A File Format Application Programming Interface (FFAPI) postoffice is a logical
	postoffice that is defined on an existing Microsoft Mail postoffice.
	
	More information on creating and configuring an FFAPI postoffice can be found in
	the "Microsoft Mail for PC Networks Software Development Kit Gateway File Format
	API."
