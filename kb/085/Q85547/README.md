## Q85547: Windows 3.1 Hangs when IBM BASIC is Run in Enhanced Mode

	Article: Q85547
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 29-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you run BASIC.COM or BASICA.COM on an IBM PS/2 under 386 enhanced mode of
	Windows version 3.1, the machine may stop (hang).
	
	WORKAROUND
	==========
	
	The only work around is to run BASIC.COM or BASICA.COM outside of Windows or
	from Windows in standard mode.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows version 3.1. We are
	researching this problem and will post new information here as it becomes
	available.
