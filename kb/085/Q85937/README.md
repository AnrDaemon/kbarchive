## Q85937: Windows: Can't Use Two Roland MPU401 Sound Cards at Same Time

	Article: Q85937
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Windows version 3.1 does not support using two Roland MPU401 sound cards at the
	same time.
	
	To use two Roland MPU401s, obtain a driver from the software package you are
	using to support them.
	
	
	Additional query words: 3.10 MPU 2 3rdparty
