## Q85318: Can't Change Paper Orientation with Bitstream FaceLift 2.0

	Article: Q85318
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	According to Bitstream Technical Support, when you use Bitstream FaceLift
	version 2.0 with Microsoft Windows 3.1 and certain printer drivers, you cannot
	change the paper orientation from inside an application using the Windows 3.1
	common dialog boxes (COMMDLG.DLL).
	
	RESOLUTION
	==========
	
	Bitstream has a patch available to FaceLift 2.0 owners that corrects this
	problem. To obtain this patch, call Bitstream.
	
	MORE INFORMATION
	================
	
	FaceLift is manufactured by Bitstream, a vendor independent of Microsoft; we
	make no warranty, implied or otherwise, regarding this product's performance or
	reliability.
