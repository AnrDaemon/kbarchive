## Q85352: FAX: How to Configure the Fax Gateway

	Article: Q85352
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to Fax, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Configure the Microsoft Mail Gateway to Fax by running the PROFILE.EXE program.
	This program customizes the PROFILE.GLB file.
	
	The PROFILE.EXE program is kept in the mail executables directory
	
	For more information on running the PROFILE.EXE program, see the "Microsoft Mail
	Gateway to Fax Administrator's Guide."
