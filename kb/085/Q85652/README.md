## Q85652: PC MAPI: Alert Displayed with Invalid User Name or Password

	Article: Q85652
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The following message is displayed when you try to log in to Mail:
	
	  STOP
	  The mailbox name or password is incorrect
	
	CAUSE
	=====
	
	This error occurs if an invalid value for either the name or password is passed
	to MAPILogon.
