## Q85541: HP ThinkJet Driver Missing Paper Sizes in Common Dialog Boxes

	Article: Q85541
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	You are unable to change the paper size for a Hewlett-Packard (HP) ThinkJet
	(2225 C-D). The Paper Size option may be unavailable (dimmed) for the HP
	ThinkJet in any application that uses the Windows 3.1 common dialog boxes
	(COMMDLG.DLL), such as Windows Write.
	
	WORKAROUND
	==========
	
	To work around this problem, change the paper size using the Printers icon in
	the Control Panel before printing the document.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in THINKJET.DRV version 1.0. We are
	researching this problem and will post new information here as it becomes
	available.
	
	Additional query words: 3.10 greyed grayed out grey gray 3.11
