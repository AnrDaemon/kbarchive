## Q85346: FAX: What Software Drivers Are Running on the Gateway PC

	Article: Q85346
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to Fax, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The only terminate-and-stay-resident (TSR) program that should be running on the
	Microsoft Mail Gateway to Fax postoffice is the driver for the fax board in the
	PC.
	
	Once this program is running, the only program that should run on the PC is the
	fax gateway program.
	
	REFERENCES
	==========
	
	The "Installing the Fax Board and Software" chapter of the "Microsoft Mail
	Gateway to Fax Administrator's Guide"
