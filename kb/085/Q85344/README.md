## Q85344: FAX: Questions That Need to Be Answered During Installation

	Article: Q85344
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to Fax, version 3.0 
	-------------------------------------------------------------------------------
	
	To install the Fax Gateway access component on a gateway postoffice, you
	must know the location of the Microsoft Mail data files and executables.
	If you are installing on a downstream postoffice, you must know the
	network and postoffice name of the gateway postoffice.
	
	REFERENCES
	==========
	
	The "Installing the Gateway" chapter of the "Microsoft Mail Gateway to Fax
	Administrator's Guide"
	
	For more information, query on the following words in the Microsoft Knowledge
	Base (depending on the topic of your interest):
	
	  pcmail and fax and gateway and postoffice
	
	  -or-
	
	  pcmail and fax and gateway and access and component
	
	
	Additional query words: 2.1a pcmail
