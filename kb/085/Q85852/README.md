## Q85852: Resizing Calendar May Not Repaint Up/Down Scroll Bar

	Article: Q85852
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When Microsoft Windows Calendar is shrunk to its minimum size possible and then
	expanded to approximately half its original size, the up/down scroll bar on the
	right side will not be repainted. Selecting the scroll bar will repaint it.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Calendar version 3.1. We are
	researching this problem and will post new information here as it becomes
	available.
	
	Additional query words: 3.10 3.1 blank missing
