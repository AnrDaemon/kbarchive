## Q60311: Notepad Does Not Save Word Wrap As Default

{% raw %}

	Article: Q60311
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	Microsoft Windows Notepad has a Word Wrap command available in the Edit
	menu. Notepad has no method of saving the Word Wrap setting as the default.
	If you want to use the Word Wrap command, you have to manually change this
	setting each time you start Notepad. Once this option is selected, it
	remains in effect for the current session, even if a new file is opened.
