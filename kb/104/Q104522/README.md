## Q104522: Incorrect Swap File Name in Windows Resource Kit for 3.1

	Article: Q104522
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	On page 244 of the "Microsoft Windows Resource Kit" for version 3.1,
	the name given for the Windows permanent swap file 386PART.PAR is
	incorrect.
	
	The correct name for the Windows permanent swap file is 386SPART.PAR.
	
	Additional query words: 3.10 swapfile
