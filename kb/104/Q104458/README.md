## Q104458: Dinosaurs: Pteranodon and Diplodocus Missing from Word List

	Article: Q104458
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbfaq
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Dinosaurs for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Pteranodon and Diplodocus are both missing from the Find word list. Other
	specific dinosaur names may also be missing. These dinosaurs are instead listed
	under broader category names. For example, Pteranodon is listed under Pterosaurs
	- Pteranodon, and Diplodocus is listed under Sauropods.
	
	
	Additional query words: multi media multimedia multi-media index listing locate name species race word list dinos C Usage
