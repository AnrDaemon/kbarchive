## Q104027: PC Ext: External Runs on 2.1 Postoffices

	Article: Q104027
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0,3.2,3.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, versions 3.0, 3.2, 3.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	All versions of the MS-DOS-based External Mail program (EXTERNAL.EXE) and the
	OS/2-based Multitasking MTA (MMTA) run against version 2.1 postoffices.
	
	
	Additional query words: 3.00 3.20 3.50
