## Q104850: Flight Simulator for Mac: Quits with Type 4 Error

	Article: Q104850
	Product(s): Microsoft Home Games
	Version(s): MACINTOSH:4.0,4.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	kbbuglist
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for the Macintosh, versions 4.0, 4.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When View1 is closed or hidden, Flight Simulator may quit with the following
	error message:
	
	  The Application "Flight Simulator" has unexpectedly quit, because an error of
	  type 4 occurred.
	
	This problem is independent of which aircraft or situation is currently in use.
	
	RESOLUTION
	==========
	
	There is currently no resolution to this problem.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Flight Simulator for the
	Macintosh, version 4.0 and 4.0a.
	
	MORE INFORMATION
	================
	
	Steps to reproduce the problem:
	
	1. Start Flight Simulator.
	
	2. Close View 1 using one of the following methods:
	
	  a. Click the close box in the upper-left corner of the view window.
	
	  b. From the View menu, choose Hide View 1.
	
	Flight Simulator will quit within a few seconds.
	
	Additional query words: 4.00 4.00a flightsim fltsim mac fs
