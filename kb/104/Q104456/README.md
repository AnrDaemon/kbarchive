## Q104456: Dinosaurs: GP Fault When Using ATI Card with Mach32 Chip Set

	Article: Q104456
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Dinosaurs for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You may receive the following error message if you run Dinosaurs with an ATI
	video card that uses the Mach32 chip set:
	
	  MSDINOS caused a General Protection Fault in module MACH32.DRV at 0006:2333
	
	This error is caused by a problem in the Mach32 driver.
	
	RESOLUTION
	==========
	
	To run Dinosaurs successfully with ATI video cards that use the Mach32 chip set,
	set the 256 Color Palette option to On. This option is in the Advanced Features
	section of the ATI Control Panel.
	
	For more information on ATI video cards or drivers, contact ATI Technologies Inc.
	at (905) 882-2626.
