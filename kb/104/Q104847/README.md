## Q104847: MemMaker Appears to Hang for One Minute or Longer

	Article: Q104847
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): msdos
	Last Modified: 21-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	MemMaker may pause for a minute or longer (appearing to stop responding [hang])
	before displaying the "Welcome to MemMaker" screen.
	
	CAUSE
	=====
	
	This problem occurs on computers that respond very slowly when MemMaker scans
	the A000-AFFF portion of video memory. For example, the NEC Ultralite Versa/33
	and NCR Safari 3180 Color Laptop respond this way.
