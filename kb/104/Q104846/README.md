## Q104846: MS-DOS Upgrade Does Not Include MS-DOS Shell

	Article: Q104846
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The MS-DOS 6.2 Upgrade and MS-DOS 6.21 Upgrade do not include MS-DOS Shell
	program files. To obtain Shell, you need to order the MS-DOS Supplemental Disk.
	To do so, use the coupon in the back of your printed MS-DOS documentation.
	
	MORE INFORMATION
	================
	
	Shell is not included in MS-DOS 6.2 because of decreased customer demand for
	Shell.
