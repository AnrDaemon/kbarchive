## Q104251: Dinosaurs: No Audio When After Dark Is Running

{% raw %}

	Article: Q104251
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Dinosaurs for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you run After Dark with a module that includes sound, the audio in Dinosaurs
	will not function until you restart Windows.
	
	RESOLUTION
	==========
	
	To eliminate this problem, turn off or remove the installation of After Dark and
	restart Windows. To avoid this problem in the future, either do not use After
	Dark, or select a screen saver module that does not use sound.
