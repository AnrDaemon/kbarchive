## Q104539: SideKick Casues Help and Editor to Stop Responding (Hang)

	Article: Q104539
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If SideKick version 2.0 is resident in memory and you run Help, MS-DOS Editor
	(EDIT.EXE), or QBasic, your machine eventually stops responding (hangs) as you
	scroll through text.
	
	RESOLUTION
	==========
	
	To work around this problem, remove SideKick from memory before running Help or
	MS-DOS Editor.
