## Q117924: Clipper Err Msg: ERROR: Invalid Extended Handle 65535...

{% raw %}

	Article: Q117924
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you close Computer Associates Clipper 5.01 editor (PE.EXE), you receive the
	following error:
	
	  ERROR: Invalid extended handle 65535 passed in
	
	
	NOTE: This error does not cause the system to stop responding (hang) nor does it
	prevent you from continuing.
	
	
	STATUS
	======
	
	This situation is most likely caused by design changes in Windows 95. Microsoft
	has confirmed that it is not caused by a problem in Windows 95. For more
	information about resolving this issue, contact Computer Associates
	International.
