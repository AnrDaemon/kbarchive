## Q117602: Information About Setting Up WFWG 3.11 and Personal NetWare

	Article: Q117602
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 04-JUL-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	According to Novell Technical Support, it is possible to set up Windows for
	Workgroups 3.11 and Novell Personal NetWare version 1.0. Customers should
	download document 3700 from the Novell bulletin board service (BBS) for more
	information.
	
	MORE INFORMATION
	================
	
	Personal NetWare 1.0 is manufactured by Novell, Inc., a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
