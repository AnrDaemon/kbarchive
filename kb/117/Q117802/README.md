## Q117802: PC Ext: Display Options in Verbose Mode

	Article: Q117802
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2,3.2a,3.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, versions 3.2, 3.2a, 3.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The External Mail program (EXTERNAL.EXE) does not display the options selected
	unless verbose mode is selected.
	
	The External Mail program will display the options selected for the operation of
	the program if /LogVerbose, -v, or -q1 is selected. Otherwise, the options will
	not be logged in the EXTERNAL.INI file on the screen or in the SESSION.LOG
	file.
