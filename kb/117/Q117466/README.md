## Q117466: Cannot Create Shortcuts with File Manager in Windows 95

	Article: Q117466
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95 appscomp kbAppCompatibility
	Last Modified: 28-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Dragging a file or folder from File Manager onto the desktop or into a group
	folder fails to create a shortcut.
	
	CAUSE
	=====
	
	File Manager is a Windows 3.1-based application and does not support the
	enhanced features of OLE drag and drop for creating shortcuts that are included
	with Windows 95.
	
	RESOLUTION
	==========
	
	To create a shortcut on the desktop:
	
	1. Find the program, folder, printer, or computer you want to create a shortcut
	  to in My Computer or Windows Explorer.
	
	2. Click the item, and then click Create Shortcut on the File menu.
	
	3. Drag the shortcut icon to the desktop.
