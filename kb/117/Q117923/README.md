## Q117923: HP ScanJet IIcx Runs Slowly Under Windows 95

	Article: Q117923
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Hewlett-Packard (HP) ScanJet IIcx, a 24-bit scanner, operates slowly under
	Windows 95.
	
	STATUS
	======
	
	This situation is most likely caused by design changes in Windows 95. Microsoft
	has confirmed that it is not caused by a problem in Windows 95. For more
	information about resolving this issue, contact Hewlett-Packard.
	
	
	MORE INFORMATION
	================
	
	The ScanJet IIcx is manufactured by Hewlett-Packard, a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
