## Q117800: Using the MMP16 Sound Card with Windows Sound System

	Article: Q117800
	Product(s): Miscellaneous Windows Products
	Version(s): 1.0,2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows Sound System, versions 1.0, 2.0 
	-------------------------------------------------------------------------------
	
	This article explains how to use the Packard Bell MMP16 sound card with
	Microsoft Windows Sound System based on information provided by Packard
	Bell Technical Support.
	
	
	From the C:\<sound subdirectory>\UTILITY directory, type the following
	command to set the mode on the sound card to Windows Sound System:
	
	  setmode /mss
	
	For more information, call Packard Bell Technical Support.
	
	The MMP16 sound card is manufactured by a vendor independent of Microsoft;
	we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
