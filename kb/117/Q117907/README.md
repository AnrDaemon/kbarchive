## Q117907: Encarta: Text in MindMaze Looks Cut Off

	Article: Q117907
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1994 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 24-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 1994 The Complete Multimedia Encyclopedia 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The text in Encarta's MindMaze game may appear to be cut off. This occurs if the
	Lucida font, which Encarta requires, is not installed on your system.
	
	For example, if you run MindMaze from the Encarta CD without first installing
	Encarta on your system, MindMaze will not be able to use the Lucida font, and
	text will display incorrectly.
	
	RESOLUTION
	==========
	
	To avoid this problem, use the Installer to install Encarta and its components
	on your system before you run MindMaze. The text should then display correctly.
