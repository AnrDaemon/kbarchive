## Q117746: How to Search for Special Characters

	Article: Q117746
	Product(s): Microsoft FoxPro
	Version(s): MACINTOSH:2.5b,2.5c; MS-DOS:2.5,2.5a,2.5b,2.6; WINDOWS:2.5,2.5a,2.5b,2.6,3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 03-AUG-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual FoxPro for Windows, version 3.0 
	- Microsoft FoxPro for Windows, versions 2.5, 2.5a, 2.5b, 2.6 
	- Microsoft FoxPro for MS-DOS, versions 2.5, 2.5a, 2.5b, 2.6 
	- Microsoft FoxPro for Macintosh, versions 2.5b, 2.5c 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	To search for special characters in the Find dialog box, use these character
	representations:
	
	  \r      ENTER (carriage return)
	  \t      Tab
	  \\      Backslash
	  \n      New line (linefeed)
	
	REFERENCES
	==========
	
	FoxPro FOXHELP.HLP online Help file, "Find Dialog" jump text, under the
	"Interface A-Z" topic
	
	FoxPro FOXHELP.DBF online Help file, "Find" topic, under "The FoxPro Interface"
	section
	
	Additional query words: VFoxWin FoxMac FoxDos FoxWin format codes newline line feed cr lf back slash backwards akz
