## Q258842: WD97: French or Spanish Word 97 Crashes When Saving to Full Disk

	Article: Q258842
	Product(s): Word 97 for Windows
	Version(s): Service Release 1 (SR-1),Service Release 2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows, versions Service Release 1 (SR-1), Service Release 2 
	-------------------------------------------------------------------------------
	
	NOTE: The information in this article applies to the French and Spanish version of Word 97.
	
	SYMPTOMS
	========
	
	French Word 97 and Spanish Word 97 stop responding or close suddenly without an
	error message when you try to save an existing document that is located on a
	full disk or a write-protected disk.
	
	On Microsoft Windows NT and Microsoft Windows 2000, Word closes immediately
	without an error message.
	
	On Microsoft Windows 95 or Microsoft Windows 98, you receive either a Dr. Watson
	error message similar to the following
	
	  Winword caused an invalid page fault in Kernel32.DLL
	
	or a blue screen with an Exception Error 0E.
	
	
	STATUS
	======
	
	Microsoft has confirmed that this is a problem in the Microsoft products that
	are listed at the beginning of this article.
