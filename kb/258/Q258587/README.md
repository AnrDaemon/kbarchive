## Q258587: Baseball 2001: Game Minimizes If You Use Context-Sensitive Help

	Article: Q258587
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbimu msgame
	Last Modified: 30-MAR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Baseball 2001 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you use context-sensitive Help in the Microsoft Baseball 2001 Player's
	Guide, the game is unexpectedly minimized.
	
	MORE INFORMATION
	================
	
	To restore the game, click Baseball 2001 on the taskbar.
	
	To use context-sensitive Help in the Baseball 2001 Player's Guide, right-click
	any item that is listed on the Contents tab, and then click What's This?
	
	Additional query words: 1.00 msgame bb2001 bball minimizes whats this online
