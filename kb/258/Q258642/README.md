## Q258642: Baseball 2001: Screen Turns Black When You Start Game

	Article: Q258642
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 31-MAR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Baseball 2001 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start Microsoft Baseball 2001, the screen may turn black.
	
	CAUSE
	=====
	
	This behavior can occur if the following conditions are true:
	
	- An STB Velocity 4400 video adapter is installed in your computer.
	
	- You are using the Detonator video driver for the STB Velocity 4400 video
	  adapter.
	
	RESOLUTION
	==========
	
	To resolve this issue, reinstall the original video driver that was provided
	with the STB Velocity 4400 video adapter.
