## Q258567: Baseball 2001: Problems When You Change Windows Settings

	Article: Q258567
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbtool kbimu
	Last Modified: 31-MAR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Baseball 2001 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	You may experience problems if you change any Microsoft Windows settings, such
	as display or screen resolution settings, while Microsoft Baseball 2001 is
	running.
	
	Microsoft recommends that you quit Baseball 2001 before you change any Windows
	settings, change the settings, and then restart Baseball 2001.
	
	Additional query words: 1.00 msgame bb2001 bball configuring
