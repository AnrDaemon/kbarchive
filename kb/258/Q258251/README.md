## Q258251: Allegiance: Pointing Device Does Not Function Properly

{% raw %}

	Article: Q258251
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbenv kbhw kbimu kbHardware
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Allegiance, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Allegiance on a Microsoft Windows 2000-based computer,
	your pointing device may not function properly.
	
	CAUSE
	=====
	
	This behavior occurs because Allegiance is not fully compatible with the
	DirectInput API in Windows 2000.
	
	STATUS
	======
	
	Microsoft has confirmed that this is a problem in Microsoft Allegiance, version
	1.0.
