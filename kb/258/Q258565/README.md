## Q258565: Baseball 2001: How to Resume Playing After Game Stops Responding

	Article: Q258565
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbimu msgame
	Last Modified: 30-MAR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Baseball 2001 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to reset Microsoft Baseball 2001 if the game stops
	responding (hangs) during game play.
	
	MORE INFORMATION
	================
	
	If Baseball 2001 stops responding (hangs) during game play, press CTRL+F3 to
	reset the game.
	
	NOTE: Although game play is resumed, a new batter is brought to the plate.
	
	Additional query words: 1.00 msgame bb2001 bball restore hangs freeze lock
