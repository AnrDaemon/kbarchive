## Q258641: Baseball 2001: Screen Turns Black When You Start the Program

{% raw %}

	Article: Q258641
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 06-APR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Baseball 2001 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to start Microsoft Baseball 2001, the screen may turn black.
	
	CAUSE
	=====
	
	This behavior can occur if the following conditions are true:
	
	- An STB Velocity 128 video adapter is installed in your computer.
	
	- You are using an nVIDIA video driver for the STB Velocity 128 video adapter.
	
	- You are using a Nokia monitor.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest video driver for your STB
	Velocity 128 video adapter from the following STB Web site:
	
	  http://www.stb.com/
