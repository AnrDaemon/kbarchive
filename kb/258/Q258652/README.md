## Q258652: Baseball 2001: Game Graphics Are Not Displayed Properly

	Article: Q258652
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 31-MAR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Baseball 2001 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click Play Ball to start a game in Microsoft Baseball 2001, the game
	graphics may not be displayed properly.
	
	CAUSE
	=====
	
	This behavior can occur if a Matrox Mystique 220 video adapter is installed in
	the computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, configure Baseball 2001 to run in Software mode:
	
	1. Start Baseball 2001.
	
	2. On the Home Plate screen, click Options.
	
	3. Click the Graphics tab.
	
	4. Under Mode, click the Software option, and then click OK.
