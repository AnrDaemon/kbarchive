## Q258650: Baseball 2001: Marker Is Not Displayed When You Hit a Fly Ball

	Article: Q258650
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 30-MAR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Baseball 2001 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Baseball 2001, the marker that appears when you hit a
	fly ball may not be displayed on the screen.
	
	CAUSE
	=====
	
	This behavior can occur if the following conditions are true:
	
	- A Hercules Thriller 3D video adapter is installed in your computer.
	
	- The video driver for the Hercules Thriller 3D video adapter is outdated.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest video driver for the
	Hercules Thriller 3D video adapter from the following Hercules Web site:
	
	  http://www.hercules.com/
	
	NOTE: Make sure that you also download and install the Thriller 3D BIOS when you
	download and install the updated video driver.
