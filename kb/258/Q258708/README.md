## Q258708: Allegiance: Cannot Connect to Server from Windows 98 SE Computer

	Article: Q258708
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbenv kbnetwork kbimu
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Allegiance, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You cannot establish a connection to a Microsoft Allegiance server from a
	Microsoft Windows 98 Second Edition-based computer that uses Internet Connection
	Sharing (ICS).
	
	CAUSE
	=====
	
	This behavior occurs because the networking API used by Allegiance is not
	compatible with the version of ICS in Windows 98 Second Edition.
	
	STATUS
	======
	
	Microsoft has confirmed that this is a problem in Microsoft Allegiance, version
	1.0.
	
	MORE INFORMATION
	================
	
	You may be able to resolve this issue by downloading and installing the latest
	version of Microsoft DirectX from the following Microsoft Web site:
	
	  http://www.microsoft.com/directx/homeuser/downloads/default.asp
	
	This issue does not occur with Microsoft Windows Millennium Edition.
