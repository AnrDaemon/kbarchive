## Q258624: Baseball 2001: Textures Are Displayed in the Wrong Location

{% raw %}

	Article: Q258624
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbdisplay kbui kbimu msgame
	Last Modified: 31-MAR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Baseball 2001 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you play Microsoft Baseball 2001, graphic textures may be displayed in the
	wrong location on the screen.
	
	MORE INFORMATION
	================
	
	To resolve this issue, turn off MIP mapping:
	
	1. Start Baseball 2001.
	
	2. On the Home Plate screen, click Options.
	
	3. Click the Graphics tab.
	
	4. Click to set the MIP Mapping option to Off, and then click OK.
	
	Additional query words: 1.00 msgame bb2001 bball graphics incorrect place
