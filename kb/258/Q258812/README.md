## Q258812: Small Business Solutions for Microsoft Windows 2000 Professional

{% raw %}

	Article: Q258812
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 08-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Small Business Solutions for Microsoft Windows 2000 Professional ISBN 0-7356-0856-3 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Small Business Solutions for Microsoft
	Windows 2000 Professional, ISBN 0-7356-0856-3.
	
	The following topics are covered:
	
	- Back Cover: Case Studies Not Included With Book
	
	MORE INFORMATION
	================
	
	Back Cover: Case Studies Not Included With Book
	-----------------------------------------------
	
	The back cover contains a bullet point that reads:
	
	"Review real-world case studies to see how companies like yours are already using
	Windows 2000 Professional to run -- and grow -- their business."
	
	These case studies are not included with the book. This bullet point should be
	disregarded.
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
	
	Additional query words: EUBOOK WIN2K PRO 0-7356-0856-3 ExecBK
