## Q275875: Combat Flight Simulator 2: Unable to Drop Bombs or Fire Rockets

	Article: Q275875
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbui kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to drop bombs or fire rockets in Microsoft Combat Flight
	Simulator 2, nothing may happen.
	
	CAUSE
	=====
	
	This behavior can occur if the ordnance is not loaded.
	
	RESOLUTION
	==========
	
	To arm ordnance, press the keyboard command to Cycle Ordnance.
	
	The following table lists the keyboard commands to arm and fire ordnance.
	
	+-----------------------------------------------------------------------------------------+
	| Action              | Combat Flight Simulator 2 command | Flight Simulator 2000 command | 
	+-----------------------------------------------------------------------------------------+
	| Cycle ordnance type | BACKSPACE                         | SHIFT+3                       | 
	+-----------------------------------------------------------------------------------------+
	| Fire ordnance       | ENTER                             | 3                             | 
	+-----------------------------------------------------------------------------------------+
	
	Additional query words: 1.00 msgame combatfs cfs2 ordinance weapons
