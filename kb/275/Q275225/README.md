## Q275225: Encarta Language Learning Online: Audio Dialogue Is Not Repeated

	Article: Q275225
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 
	Operating System(s): 
	Keyword(s): kb3rdparty kbsound kbimu
	Last Modified: 23-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Encarta Language Learning. Version 1.0 (online edition) 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use NullSoft WinAmp to play an audio dialogue in Microsoft Encarta
	Language Learning Online, the dialogue may not be repeated even when you enable
	the Repeat playback option.
	
	CAUSE
	=====
	
	This behavior can occur if the "Stop after current" playback option is enabled.
	
	RESOLUTION
	==========
	
	To resolve this issue, disable the "Stop after current" playback option.
