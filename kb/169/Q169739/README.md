## Q169739: WD97: Error Message on Clicking Help Button for Convert to Frame

{% raw %}

	Article: Q169739
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kberrmsg kbusage kbdta word97 kbframe
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click the double arrow button (>>) in the Convert To Frame Help
	topic, the following error message may appear:
	
	  The topic does not exist. Contact your application vendor for an updated Help
	  file. (129)
	
	When you click OK, you receive an unrelated Help topic.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows.
	
	This problem has been corrected in Word 2000.
	
	
	MORE INFORMATION
	================
	
	For more information about the difference between a frame and a text box, click
	the Office Assistant, type "what is the difference between a frame and a text
	box," click Search, and then click "The difference between a text box and a
	frame."
	
	NOTE: If the Assistant is hidden, click the Office Assistant button on the
	Standard toolbar. If Word Help is not installed on your computer, please see the
	following article in the Microsoft Knowledge Base:
	
	  Q120802 Office: How to Add/Remove a Single Office Program or Component
