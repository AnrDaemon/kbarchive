## Q169869: MDG: Gimport on the TechNet CD Before Nov 96 Created MEM

	Article: Q169869
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, version 3.5 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	GIMPORT on the TechNet CD prior to November 1996 will create a MEM file with
	TID+2. This has been correct in the November 1996 TechNet CD.
	
	MORE INFORMATION
	================
	
	If you are using the GIMPORT utility from the TechNet CD prior to November 1996,
	it will create a MEM with TID+2. You should use the one from the November 1996
	TechNet CD.
