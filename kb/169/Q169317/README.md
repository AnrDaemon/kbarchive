## Q169317: Close Combat Patch Is Available for Better Performance

	Article: Q169317
	Product(s): Microsoft Home Games
	Version(s): MACINTOSH:1.0; WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbfile kbusage
	Last Modified: 21-DEC-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Close Combat for Windows 1.0 
	- Microsoft Close Combat for the Macintosh 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you own the full version of Close Combat, you can download an update (patch)
	for faster performance and the ability to play head-to-head in Zone.com.
	
	NOTE: Only PC users are able to play on Zone.com.
	
	The file size for the Windows patch is approximately 1.3 MB in size. The file
	size for the Macintosh patch is approximately 8 KB in size.
	
	MORE INFORMATION
	================
	
	The following files are available for download from the Microsoft Download
	Center:
	
	Windows
	-------
	
	  Ccupdate.exe
	  (http://download.microsoft.com/download/ccombatwin/Patch/1/WIN98/EN-US/Ccupdate.exe)
	
	Macintosh
	---------
	
	  Mccupdat.hqx
	  (http://download.microsoft.com/download/ccombatmac/Update/1/MacOS/EN-US/Mccupdat.hqx)
	
	Release Date: Sep-09-1997
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
	
	Microsoft used the most current virus detection software available on the date of
	posting to scan this file for viruses. Once posted, the file is housed on secure
	servers that prevent any unauthorized changes to the file.
