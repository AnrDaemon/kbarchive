## Q169392: XFOR: LinkAge Connector Does Not Support Sharing Calendars

	Article: Q169392
	Product(s): Microsoft Exchange
	Version(s): winnt:5.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 23-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The LinkAge Notes Connector does not support sharing of calendars and schedules
	between Notes and Exchange users.
