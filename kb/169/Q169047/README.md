## Q169047: Full Court Press Trial Version Is Available

{% raw %}

	Article: Q169047
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 20-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NBA Full Court Press for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The trial version of Microsoft NBA Full Court Press, version 1.0 is now
	available for free download. The file is approximately 20.8 MB in size.
	
	TO OBTAIN THIS FILE
	-------------------
	
	You can find Fcptrial.exe, a self-extracting file, on the following services:
	
	- Microsoft World Wide Web site on the Internet
	
	  http://www.microsoft.com/sports/baskball
	
	  Click the Downloads icon. Click the NBA Full Court Press Trial Version link to
	  download the file.
	
	- Internet (anonymous FTP)
	
	  FTP://FTP.microsoft.com
	  Change to the Deskapps/Games/Public/FullCourtPress directory
	  Get FCPTRIAL.EXE
	
	Additional query words: 1.00 football basketball NBA full court press fcp tv kbimu homegame msgame
