## Q169430: Arcade: Updated Wavemix.ini File For Microsoft Arcade

	Article: Q169430
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbfile kbusagekbfaq
	Last Modified: 21-DEC-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Arcade for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	There is a patch available for the Wavemix.ini file that corrects some sound
	problems you may encounter when you run Microsoft Arcade version 1.0 on Windows
	95/98.
	
	To replace the Wavemix.ini file, download and install the Wavemix.exe file.
	Wavemix.exe is 111 KB in size.
	
	The following file is available for download from the Microsoft Download Center:
	
	  Wavemix.exe
	  (http://download.microsoft.com/download/returnarcade/Update/1/WIN98/EN-US/Wavemix.exe)
	
	Release Date: May-07-1998
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
	
	Microsoft used the most current virus detection software available on the date of
	posting to scan this file for viruses. Once posted, the file is housed on secure
	servers that prevent any unauthorized changes to the file.
	
	Additional query words: 1.00 w_arcade sound wave wav mix patch golf
