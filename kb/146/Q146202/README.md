## Q146202: AT&amp;T Globalyst TPC Modems Not Detected Properly

	Article: Q146202
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95,98
	Operating System(s): 
	Keyword(s): kbhw win95 win98 kbHardware
	Last Modified: 24-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	- Microsoft Windows 98 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	TPC modems in certain AT&T Globalyst computer models are not detected
	properly in Windows 95/98.
	
	RESOLUTION
	==========
	
	Contact AT&T Technical Support for updated drivers for the TPC modem.
	
	MORE INFORMATION
	================
	
	All AT&T Globalyst computers with model numbers in the "3348-23xx-xxxx"
	series require updated drivers to operate correctly with Windows 95/98.
