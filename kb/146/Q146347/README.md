## Q146347: XCLN: Digitally Signed Msgs. Marked as Read When Open Fail

{% raw %}

	Article: Q146347
	Product(s): Microsoft Exchange
	Version(s): 4.0 5.0
	Operating System(s): 
	Keyword(s): kbui
	Last Modified: 16-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Windows 95/98 client, versions 4.0, 5.0 
	- Microsoft Exchange Windows 3.x client, versions 4.0, 5.0 
	- Microsoft Exchange Windows NT client, versions 4.0, 5.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to open a message that is digitally signed and the open fails, and
	if you select the Cancel button on the password dialog box, the message is
	marked as read.
	
	CAUSE
	=====
	
	This is by product design.
