## Q146782: XFOR: Blank Space in Subject Line Not Retained Over IMC

{% raw %}

	Article: Q146782
	Product(s): Microsoft Exchange
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 19-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When a message that contains a blank space at the beginning of the subject line
	is sent out the Microsoft Exchange Internet Mail Connector, the blank space is
	not retained, as in the example below.
	
	  Original Subject:  This is a subject line
	  Received Subject: This is a subject line
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in versions 4.0, of Microsoft
	Exchange. We are researching this problem and will post new information here in
	the Microsoft Knowledge Base as it becomes available.
