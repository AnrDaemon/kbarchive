## Q146068: XCLN: Display Bug with OLE Objects Causes Size Change

{% raw %}

	Article: Q146068
	Product(s): Microsoft Exchange
	Version(s): 4.0,5.0
	Operating System(s): 
	Keyword(s): kbProgramming
	Last Modified: 20-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Windows 3.x client, versions 4.0, 5.0 
	- Microsoft Exchange Windows NT client, versions 4.0, 5.0 
	- Microsoft Exchange Windows 95/98 client, versions 4.0, 5.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In some cases, following a drag-copy of an OLE object, the original OLE object
	suddenly appears much smaller than original, while the new copy correctly
	remains the original object size.
	
	
	Additional query words: ole prog
