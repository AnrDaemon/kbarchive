## Q146001: AMD (NexGen) Nx586 Processor Identified as 80386

	Article: Q146001
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	- Microsoft Plus! for Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Windows 95 and other programs may identify the AMD (formerly NexGen) Nx586
	processor as an 80386 processor. You may experience the following symptoms with
	this processor:
	
	- You may receive the following error message when you run Microsoft Plus!
	  Setup:
	
	  MS Plus! requires a 486 processor or greater
	
	- The 3D Pinball game included with Microsoft Plus! may not run.
	
	- The Microsoft Hover! game included on the CD-ROM version of Windows 95 may
	  not run.
	
	- Microsoft Fury3 for Windows version 1.0 may not run.
	
	- You may not be able to install Microsoft Encarta World Atlas for Windows,
	  1996 edition. For additional information about this problem, please see the
	  following article in the Microsoft Knowledge Base:
	
	  Q142217 Encarta World Atlas 1996: Setup Fails with NexGen CPU
	
	CAUSE
	=====
	
	Programs developed before the NexGen Nx586 processor may mistakenly identify the
	processor as an 80386 processor.
	
	
	RESOLUTION
	==========
	
	To resolve this issue, contact Advanced Micro Devices (AMD) to inquire about a
	fix for this issue.
