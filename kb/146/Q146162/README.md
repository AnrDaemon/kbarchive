## Q146162: XCLN: Copy Across Store Does Not Change the Message Size

{% raw %}

	Article: Q146162
	Product(s): Microsoft Exchange
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kbenv
	Last Modified: 17-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Windows 95/98 client, version 4.0 
	- Microsoft Exchange Windows 3.x client, version 4.0 
	- Microsoft Exchange Windows NT client, version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you create a message, and before you send it, you can copy the message to
	another store or folder and the size of the message will remain the same.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange version 4.0
	Windows, Windows NT, and Windows 95 clients. We are researching this problem and
	will post new information here in the Microsoft Knowledge Base as it becomes
	available.
