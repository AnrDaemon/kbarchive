## Q146195: Only Text Can Be Pasted into a Meeting Request

	Article: Q146195
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows 3.x 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you send a meeting request, only the text can be pasted into the body of
	the message.
	
	CAUSE
	=====
	
	This is by design.
