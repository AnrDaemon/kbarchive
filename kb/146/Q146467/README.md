## Q146467: Recurring Meetings w/ Same Patterns May Fall on Different Days

{% raw %}

	Article: Q146467
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you create a recurring meeting, you can have two recurring meetings with
	the same pattern fall on different days because a recurring pattern is created
	based on the weeks start day.
	
	
	
	CAUSE
	=====
	
	This is by design.
