## Q146292: XADM: Create/Last Modified Unavailable After Clicking Apply

	Article: Q146292
	Product(s): Microsoft Exchange
	Version(s): 4.0 5.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 14-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you create a new Distribution List (or mailbox, or Directory replication
	connector, etc.) and click Apply, the Created and Last Modified Labels are
	unavailable (gray).
	
	This symptom only occurs when the object is first created.
	
	WORKAROUND
	==========
	
	After you click Apply, click OK and then view the properties for that user or
	mailbox, or Directory replication connector. All options are now available.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange version 4.0.
	We are researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
