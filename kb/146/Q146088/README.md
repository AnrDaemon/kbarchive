## Q146088: Internet Service Manager Does Not Sort By State

{% raw %}

	Article: Q146088
	Product(s): Internet Information Server
	Version(s): winnt:1.0
	Operating System(s): 
	Keyword(s): kbnetwork
	Last Modified: 07-JAN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Internet Information Server 1.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you select Sort by State from the View menu of Internet Service Manager,
	the Internet service states are not sorted.
	
	WORKAROUND
	==========
	
	To work around this problem, click the State column header.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Internet Information Server
	version 1.0. We are researching this problem and will post new information here
	in the Microsoft Knowledge Base as it becomes available.
