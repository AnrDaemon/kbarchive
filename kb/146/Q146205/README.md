## Q146205: Account Dates on MSN Are Not Displayed in Localized Format

{% raw %}

	Article: Q146205
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.2,1.3,2.0,2.5
	Operating System(s): 
	Keyword(s): kbui kbmsn
	Last Modified: 14-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.2, 1.3, 2.0, 2.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you view account information on MSN, The Microsoft Network, the dates are
	displayed in day/month/year format. This format does not change in localized
	versions of MSN to reflect formats, such as year/day/month, used outside the
	United States.
