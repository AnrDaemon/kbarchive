## Q146589: Group By With Multiple Entries Doesn't Repaint Bottom of Grid

{% raw %}

	Article: Q146589
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0; Win95:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the Group By function in Microsoft Schedule+ for Windows, the To Do
	list may not refresh the bottom field if all the fields are being used and the
	vertical scrollbar is active.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in version 7.0 of Microsoft
	Schedule+ for Windows. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
