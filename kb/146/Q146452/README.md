## Q146452: Go To: on Contact Tab Not Work Until Sort Contacts

{% raw %}

	Article: Q146452
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0; Win95:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you enter Contacts for the first time, and you try to use the Go to: field,
	Schedule+ will not search through the Contacts entered.
	
	CAUSE
	=====
	
	This is by design.
	
	RESOLUTION
	==========
	
	You need to sort the contacts prior to using the Go to: field (choose Sort from
	the View menu). Modify the Sort order, and click OK.
