## Q146363: XCLN: Using Space Bar in Name Field on Contact Page

	Article: Q146363
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows 95, version 7.0a 
	- Microsoft Schedule+ for Windows NT, version 7.0a 
	- Microsoft Schedule+ for Windows, version 7.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you are adding a contact in Microsoft Schedule+ for Windows version 7.0a
	and press the spacebar after you enter something in the First Name field, the
	cursor will jump to the Last Name field. This is by design.
