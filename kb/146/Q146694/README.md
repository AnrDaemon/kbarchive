## Q146694: XCLN: Remove All Leaves an Empty Program Group

	Article: Q146694
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:4.0,5.0
	Operating System(s): 
	Keyword(s): kbsetup
	Last Modified: 16-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Windows 3.x client, versions 4.0, 5.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run the Microsoft Exchange client for Windows 3.x Setup program and
	select Remove All, the program group containing the Microsoft Exchange icons
	will not be removed.
	
	MORE INFORMATION
	================
	
	This is by design. The program group can be deleted manually by selecting the
	empty program group and then clicking Delete on the File menu.
