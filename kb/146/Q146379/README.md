## Q146379: All Day Check Box Inconsistent in Recurring Appointments

	Article: Q146379
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you select All Day in the Appointment dialog box and then select Make
	Recurring results, the All Day is not being selected in the Appointment Series
	dialog box.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Schedule+ version 7.0.
	This problem was corrected in version 7.0a.
