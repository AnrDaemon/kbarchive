## Q146375: XCLN: Printing Weekly 5-day Prints from Monday through Friday

	Article: Q146375
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0a 
	- Microsoft Schedule+ for Windows NT 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you print the Weekly - 5 day layout in Microsoft Schedule+ version 7.0, the
	week printed will always start on Monday and end on Friday. It is not possible
	to print the work week starting on any other day.
	
	MORE INFORMATION
	================
	
	Even if you use the Week Starts On option, when you print the 5 day week layout,
	Monday through Friday will be printed.
	
	Microsoft has confirmed this to be a problem in Microsoft Schedule+ version 7.0a.
	We are researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
