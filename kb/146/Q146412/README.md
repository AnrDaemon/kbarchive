## Q146412: Cannot Paste Text from Japanese Newsgroup into Other Program

	Article: Q146412
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.2,1.3,2.0,2.5
	Operating System(s): 
	Keyword(s): kbui kbmsn
	Last Modified: 13-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.2, 1.3, 2.0, 2.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are using the Japanese version of MSN, The Microsoft Network, Japanese
	text you paste from a newsgroup into another program may appear in an unreadable
	MS Sans Serif font.
	
	RESOLUTION
	==========
	
	To work around this issue, use the Paste Special command on the Edit menu to
	paste the Japanese text into the other program.
