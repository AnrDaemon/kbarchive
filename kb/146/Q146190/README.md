## Q146190: Cut and Paste Available on Wisdom Tab

	Article: Q146190
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0,95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0, on platform(s):
	   - the operating system: Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the Covey Seven Habits Tools and change to the Wisdom Literature
	tab, you can select all titles listed in the sampler index by clicking the right
	mouse button in the sampler index, and then cut or paste the information. When
	you change to a different tab and back to the Wisdom tab, the original
	information will be listed.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Schedule+ version 7.0
	for Windows 95. We are researching this problem and will post new information
	here in the Microsoft Knowledge Base as it becomes available.
