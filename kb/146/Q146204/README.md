## Q146204: Sender Name Not Displayed in Messages in Sent Items Folder

	Article: Q146204
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.3,2.0,2.5
	Operating System(s): 
	Keyword(s): kbmsn
	Last Modified: 13-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.3, 2.0, 2.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	E-mail messages in the Sent Items folder have a blank From field.
	
	CAUSE
	=====
	
	This behavior is by design.
