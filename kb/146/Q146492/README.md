## Q146492: Setting a New Contact to Private Requires Data to Be Entered

	Article: Q146492
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you create a new contact using the grid in the Contacts tab, you must enter
	the data into the grid before you try to make the contact private by using the
	Private button on the toolbar. If you select the Private button, the system will
	beep at you.
	
	CAUSE
	=====
	
	This is by design.
