## Q146469: Printing Empty Schedule Pages Causes Separator Page to Print

	Article: Q146469
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you print to a printer that is configured to print a separator page between
	print jobs, a separator page might print when no Schedule+ data meets the
	printing options selected.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Schedule+ version 7.0.
	We are researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
