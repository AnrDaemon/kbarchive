## Q146052: &lt;1Ch&gt; Entries Can Be Removed From Static Mappings Dialog Box

{% raw %}

	Article: Q146052
	Product(s): Microsoft Windows NT
	Version(s): 3.5 3.51 4.0
	Operating System(s): 
	Keyword(s): kbnetwork
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server versions 3.5, 3.51, 4.0 
	-------------------------------------------------------------------------------
	
	When you add a static <1Ch> entry to the WINS database, the entry is added
	to the <1Ch> group with other dynamic and static <1Ch> entries for the
	specified domain. The group is then designated as a static group in the
	WINS database screen of WINS Manager.
	
	In the Static Mappings dialog box, you can remove any one of the dynamic or
	static <1Ch> entries for the domain. If you remove a <1Ch> static entry,
	the next entry in the group moves to the static mapping position. The <1Ch>
	entry is then designated as a static entry.
