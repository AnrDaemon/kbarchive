## Q146742: XCLN: Modified Column Widths are not Preserved

	Article: Q146742
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0; Win95:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you change the column widths in the Microsoft Schedule+ Edit Recurring
	Appointments or Task dialog boxes, the column widths will not be preserved after
	you close the dialog screen. Changes to columns apply only to the active screen.
	When the dialog box is closed, the columns will return to their original
	positions.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Schedule+ version 7.0.
	We are researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
