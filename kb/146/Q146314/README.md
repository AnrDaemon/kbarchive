## Q146314: Recur Task Continues if No End Date and No Auto-done Selected

	Article: Q146314
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you make a recurring task as a single instance without selecting Mark as Done
	After End Date and Until in the Task Series dialog box, the instance will be
	propagated as a recurring task.
	
	NOTE: If you select, Mark as Done After End Date, the instance completes and does
	not recur.
	
	
	CAUSE
	=====
	
	This is by design.
