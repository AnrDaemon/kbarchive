## Q127773: NetWare Execute-Only Attribute Not Supported

	Article: Q127773
	Product(s): Microsoft Windows NT
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.5 
	- Microsoft Windows NT Server version 3.5 
	-------------------------------------------------------------------------------
	
	Page 95 of the Windows NT Workstation version 3.5 Installation Guide states
	the following:
	
	  The Client Service does not support mapping for the following NetWare
	  file attributes: S (Shareable), T (Transactional), P (Purge),
	  Ra (Read audit), RW (Read/Write), Wa (Write audit), and
	  Ci (Copy inhibit).
	
	This statement is not complete. It does not include X (Execute-Only).
