## Q127930: Paint Shop Pro Screen Capture Does Not Work Correctly

	Article: Q127930
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to use the Capture Area or Capture Client Area feature from the
	Capture menu in Paint Shop Pro version 2.0, the feature does not function
	correctly and the mouse pointer does not become a crosshair as it should. The
	Full Screen and Window capture features do work correctly.
	
	STATUS
	======
	
	This behavior is a limitation of Paint Shop Pro version 2.0 running in Windows
	95. Upgrades are available from:
	
	JASC, Inc.
	10901 Red Circle Drive, Suite 340
	Minnetonka, MN 55343 USA
	
	
	MORE INFORMATION
	================
	
	Paint Shop Pro is manufactured by a vendor independent of Microsoft; we make no
	warranty, implied or otherwise, regarding this product's performance or
	reliability.
