## Q127749: MSB Solar: Sounds Echo with Mitsumi CD in Windows NT 3.5

	Article: Q127749
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Scholastic's Magic School Bus series: Explores the Solar System for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you run Microsoft Scholastic's Magic School Bus Explores the Solar System
	on Windows NT 3.5, if you are using a Mitsumi CD-ROM drive, you may experience
	sound "echoes" whenever sound is played. (This behavior may occur in Encarta
	1995 as well).

	Microsoft; we make no warranty, implied or otherwise, regarding these products'
	performance or reliability.
