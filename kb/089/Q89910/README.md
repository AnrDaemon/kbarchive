## Q89910: Error in Quick Recorder Help

{% raw %}

	Article: Q89910
	Product(s): Miscellaneous Windows Products
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows Sound System, versions 1.0, 1.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you view the Compression dialog box screen of Quick Recorder Help and
	select the "4 bits per sample" field, a second instance of the Compression
	dialog box appears with no explanation.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows Sound System Quick
	Recorder version 1.0. We are researching this problem and will post new
	information here as it becomes available.
