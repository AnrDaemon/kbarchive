## Q89368: PC Gen: 3.0 Working Model Runs Only on Windows 3.1

{% raw %}

	Article: Q89368
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Page 2 of the "Working Model" guide for version 3.0 of Microsoft Mail for PC
	Networks incorrectly states the following system requirements:
	
	- Microsoft Windows graphical environment version 3.0 or later
	
	- 80286 or higher processor (80386 recommended)
	
	- 2 MB of memory
	
	Version 3.0 of the Microsoft Mail Working Model, requires Microsoft Windows
	version 3.1 or later.
