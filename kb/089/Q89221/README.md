## Q89221: Minimized Clock Truncates Digits in Digital Mode

	Article: Q89221
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When the System, Fixedsys, or Courier fonts are used with the Microsoft Windows
	Clock version 3.1 in Digital mode, the display's last digit or two may not
	appear when Windows Clock is minimized.
	
	CAUSE
	=====
	
	The System, Fixedsys, and Courier fonts are fixed space fonts; they do not
	resize when Windows Clock is minimized.
	
	WORKAROUND
	==========
	
	To work around this problem, use the MS Sans Serif or a TrueType font with
	Windows Clock.
