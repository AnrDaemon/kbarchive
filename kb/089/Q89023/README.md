## Q89023: WRK Err: Vector Fonts Can Print on Dot-Matrix Printer

	Article: Q89023
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	Page 317 of the version 3.1 "Microsoft Windows Resource Kit" guide includes a
	chart incorrectly stating that the Windows vector fonts Modern, Roman, and
	Script cannot print on dot-matrix printers.
	
	In the third paragraph on page 322, the "Microsoft Windows Resource Kit" guide
	correctly states that vector fonts are printable on dot-matrix printers.
	
	
	Additional query words: 3.10 docerr win31 dot matrix wrk
