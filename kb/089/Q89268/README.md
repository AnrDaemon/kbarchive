## Q89268: Using the BROWSE Command in FoxBASE+

	Article: Q89268
	Product(s): Microsoft Fox Miscellaneous Products
	Version(s): MS-DOS:2.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 23-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft FoxBASE+ for MS-DOS, version 2.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following information applies to the BROWSE command in FoxBASE+:
	
	- The BROWSE command locks the entire database, which allows the user to edit
	  any field.
	
	- In the BROWSE FREEZE command, the user can edit only one field.
	
	- Because the BROWSE NOMODIFY command does not lock the database, the user
	  cannot edit any fields.
