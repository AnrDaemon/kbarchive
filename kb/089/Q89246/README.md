## Q89246: Windows Err Msg: Unable to Connect to a Network Drive

	Article: Q89246
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You may get the error message: "Unable to connect to a Network drive" when
	running Microsoft Windows on 3Com 3+ Open network. This error occurs in File
	Manager because File Manager is unable to connect to shared drives outside of
	the current user's domain.
	
	WORKAROUND
	==========
	
	- Connect to these drives before you start Windows.
	
	- Upgrade your 3Com network to LAN Manager 2.1.
	
	Additional query words: 3.10 3.00a 3.00 3.11 3 com 3rdparty
