## Q89019: Princeton SR 12 Does Not Work Under Windows

	Article: Q89019
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:2.03,2.1,2.11,3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 2.03, 2.1, 2.11, 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	The Princeton SR 12 monitor shipped with a propriety "scan doubler" video
	card that can be configured as either a Hercules graphics card or a CGA
	card. The SR 12 requires this card to operate properly. This card does not
	work with any version of Windows when set up as either CGA or Hercules
	graphics.
	
	Additional query words: 3.00 3.00a 3.10 3.11 2.03 2.10 double-scan 3rdparty
