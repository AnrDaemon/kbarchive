## Q89481: Serial Network Connections Not Supported in WFWG

{% raw %}

	Article: Q89481
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 02-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	Serial interfaces for network connections are not supported in
	Microsoft Windows for Workgroups. Using modems for network connections
	is also unsupported.
	
	Windows for Workgroups is designed for peer-to-peer networking.
	Compatible network cards must be used for network functionality.
	
	This feature may be implemented in future versions of Windows for
	Workgroups.
