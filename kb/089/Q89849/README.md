## Q89849: PC Ext: Slow Modem Communications with 9600 Baud Modems

{% raw %}

	Article: Q89849
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:2.1,3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, versions 2.1, 3.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Mail transfers slowly with 9600 baud modems.
	
	CAUSE
	=====
	
	This can be caused by outdated 3Com network drivers for Novell NetWare. All
	inbound communications will be very slow, but outbound communications will be at
	normal speed.
	
	RESOLUTION
	==========
	
	Contact Novell, Inc., for updated 3Com drivers.
