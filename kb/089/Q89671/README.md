## Q89671: Converting a FoxBASE File to FoxBASE+ Format

	Article: Q89671
	Product(s): Microsoft Fox Miscellaneous Products
	Version(s): MS-DOS:2.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 23-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft FoxBASE+ for MS-DOS, version 2.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	FoxBASE+ can read information from a FoxBASE file but cannot write data into
	one. Enter the following commands in the FoxBASE+ Command window to convert a
	FoxBASE file to FoxBASE+ format:
	
	- USE <filename>
	
	- COPY TO <newfile>
