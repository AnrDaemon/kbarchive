## Q89010: Print Command Unavailable When Using Windows 3.1 with Novell

	Article: Q89010
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): win31
	Last Modified: 30-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	While using Novell NetWare 3.11, the Print option in Windows-based applications
	(such as Write) may be unavailable (dimmed).
	
	WORKAROUND
	==========
	
	To correct this problem, make the printer connection permanent.
	
	NOTE: Before checking network connections, verify that a printer has been
	selected as the default printer.
	
	To Select a Permanent Printer Connection
	----------------------------------------
	
	1. From the Main group, run Control Panel and choose Printers.
	
	2. Choose the Network button.
	
	3. Select the print share and then select the Permanent check box.
