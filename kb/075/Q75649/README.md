## Q75649: Sysgen Durapak and MS-DOS 5.0

{% raw %}

	Article: Q75649
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Sysgen Durapak 15 MB removable hard disk system and its SYSGEN.SYS driver do
	not work with MS-DOS 5.0. According to Sysgen, the product has been discontinued
	and is no longer supported (no new drivers are available). Sysgen reports that
	the product works with MS-DOS versions 3.1 and earlier.
	
	For more information, contact Sysgen.

	or reliability.
