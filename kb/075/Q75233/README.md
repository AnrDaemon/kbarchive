## Q75233: DTK BIOS version 4.26 Pre-Revision 36 and IDE Drives

{% raw %}

	Article: Q75233
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.x,6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 5.0, 5.0a, 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following is a list of DTK BIOS 4.26 revision numbers. These revisions and
	all previous revisions may cause systems with IDE hard disks to hang when
	executing MS-DOS commands or running applications.
	
	  Machine            Revision
	  -------            --------
	
	  80286              25 and earlier
	  80386 SX           95 and earlier
	  80386 DX           35 and earlier
	  80486              45 and earlier
	
	MORE INFORMATION
	================
	
	DTK included corrections to many revisions for compatibility with IDE hard
	disks. To determine the BIOS version, start your machine and check the first two
	digits of the line starting with the number sign (#). For example, #34062890S
	contains 34 after the # sign, which refers to revision number 34.
	
	For more information about upgrading your BIOS, contact DTK technical support.
