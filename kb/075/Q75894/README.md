## Q75894: PaintJet Printer Fonts Included with Windows

{% raw %}

	Article: Q75894
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	There are three specific font files, .PJFs, for the HP PaintJet printer included
	with the Microsoft Windows version 3.0 and 3.0a Setup disks. The names of each
	of the files are listed below:
	
	- CR124PA.PJF
	
	- LG08RPA.PJF
	
	- LG12RPA.PJF
	
	MORE INFORMATION
	================
	
	The location of these files depends on the version of Windows (3.0 or 3.0a) and
	the size of the Setup disks being used. Use the following chart to pinpoint
	their location.
	
	Windows Version   Disk Size        Disk #
	-----------------------------------------------
	  3.00           [5.25\1.2MB]     Disk #5
	  3.00           [3.5\720K]       Disk #7
	  3.00           [5.25\360K]      Disk #12
	  3.00a          [5.25\1.2MB]     Disk #5
	  3.00a          [3.5\720K]       Disk #5
	  3.00a          [3.5\720K]       Disk #6
	
	CR12RPA.PJF, LG08RPA.PJF, LG12RPA.PJF
	-------------------------------------
	
	PJFs are PaintJet Fonts for the PaintJet driver that Zenographics wrote for
	Microsoft. They can be installed by using the Windows Control Panel using the
	font installer. The installation steps listed below will provide assistance in
	the configuration of these printer fonts.
	
	Steps to Install PaintJet Fonts
	-------------------------------
	
	1. Run the Control Panel.
	
	2. Select the Printers icon from the Control Panel.
	
	3. Double-click the PaintJet driver in the listing of Install Printers to make
	  it active or install the driver and then choose Active from the Status Box.
	
	4. Choose the Configure button on the right-hand side.
	
	5. Choose the Setup button from the Printers Configure dialog box.
	
	6. Choose the Fonts button from the HP PaintJet Setup dialog box.
	
	7. Choose the Add button and choose the desired fonts to be installed.
	
	8. Choose OK.
