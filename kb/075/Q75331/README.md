## Q75331: HP LaserJet III Driver Version 3.77 Does Not Save Changes

	Article: Q75331
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 31-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Changes made to the printer setup of the Hewlett-Packard (HP) LaserJet III
	driver version 3.77 configuration in the Control Panel are not saved when you
	restart Windows.
	
	When the Paper Source and Copies field settings are changed and Windows is closed
	and restarted, the changes revert back to the default settings.
	
	MORE INFORMATION
	================
	
	Steps to Reproduce Problem
	--------------------------
	
	1. From the Control Panel, select the Printers icon.
	
	2. From the main Printers dialog box choose the Add Printer button.
	
	3. Install the 3.77 driver available from HP.
	
	4. Choose the Configure and then Setup buttons.
	
	5. Change all default settings in the Setup window and choose OK all the way
	  out.
	
	6. Exit Windows and restart.
	
	7. From the Control Panel, select the Printer icon.
	
	8. View the changes you have made.
