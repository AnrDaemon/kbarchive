## Q75796: UAE When Running X-Vision 4.0

	Article: Q75796
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You may receive an Unrecoverable Application Error (UAE) when using X-Vision 4.0
	in Microsoft Windows version 3.0.
	
	CAUSE
	=====
	
	The XHOST.TXT file contains blank lines.
	
	WORKAROUND
	==========
	
	You may be able to work around the problem by deleting the XHOST.TXT file or
	filling the blank lines with text.
	
	MORE INFORMATION
	================
	
	For more information, contact Unipress Software Distributors technical support.

	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
