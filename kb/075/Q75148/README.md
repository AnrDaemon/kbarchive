## Q75148: System Requirements for IBM PC-DOS 5.0 Upgrade

{% raw %}

	Article: Q75148
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 03-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The system requirements for IBM PC-DOS 5.0 Upgrade are as follows:
	
	IBM Personal System/2 Model 25, 30, 25-286, 30-286, L40SX, 50, 50Z, 55SX, 60, 65,
	70, P70, P75, 80, 90, 95, or PS/1
	
	IBM Personal System/55 Model 5510, 5523, 5530-S/T, 5535-S, 5540-T, 5550-S/T/V,
	5560-W, 5570-T/V or 5580-Y
	
	IBM Personal Computer, Personal Computer XT, Personal Computer XT-286, Personal
	Computer AT, Portable Personal Computer, or PC Convertible
	
	IBM Display or equivalent
	
	512K of memory (minimum)
	
	One hard disk with a previous version of DOS (version 2.1 or later) already
	installed
	
	One 360K 5.25-inch floppy disk drive, or one 1.2 MB 5.25-inch floppy disk drive
	
	The DOS versions above refer to IBM PC-DOS. The IBM Upgrade will not install over
	any version of MS-DOS.
	
	The products included here (IBM) are manufactured by vendors independent of
	Microsoft; we make no warranty, implied or otherwise, regarding these products'
	performance or reliability.
