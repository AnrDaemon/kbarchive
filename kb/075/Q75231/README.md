## Q75231: MS-DOS and CSC Fastcache SCSI Controllers

	Article: Q75231
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.x,6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 5.0, 5.0a, 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you are using MS-DOS 5.0 or later with a CSC Fastcache SCSI Controller and
	experience problems writing to the floppy drives, you need to upgrade the ROM
	BIOS on your controller to version 2.01 or later.
	
	Additional query words: 6.22 5.00 5.00a 6.00 6.20
