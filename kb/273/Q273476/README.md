## Q273476: SMS: Buffer Overrun Problem in Several Network Monitor Parsers

	Article: Q273476
	Product(s): Microsoft Systems Management Server
	Version(s): 1.2,1.2 SP1,1.2 SP2,1.2 SP3,1.2 SP4,2.0,2.0 SP1,2.0 SP2
	Operating System(s): 
	Keyword(s): kbsms200 kbsms200bug kbNetMon kbsms200preSP3fix kbsms200SP3fix
	Last Modified: 07-FEB-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server versions 1.2, 1.2 SP1, 1.2 SP2, 1.2 SP3, 1.2 SP4, 2.0, 2.0 SP1, 2.0 SP2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Buffer overruns may cause security problems in the Network Monitor tool.
	
	RESOLUTION
	==========
	
	Systems Management Server 2.0
	-----------------------------

	Server version 2.0. For additional information, please see the following article

	  Q288239 SMS: How to Obtain the Latest Systems Management Server 2.0 Service
	  Pack
