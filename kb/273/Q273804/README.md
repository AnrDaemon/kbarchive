## Q273804: SMS Supports Windows Millennium Edition as a Client Platform

	Article: Q273804
	Product(s): Microsoft Systems Management Server
	Version(s): 2.0 SP2
	Operating System(s): 
	Keyword(s): kbsms200
	Last Modified: 22-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 2.0 SP2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft Systems Management Server (SMS) version 2.0 with Service Pack 2
	installed, supports and manages Microsoft Windows Millennium Edition (Me) as a
	client platform.
	
	MORE INFORMATION
	================
	
	Windows Me is not supported by any earlier versions of SMS.
	
	Additional query words: prodsms winme
