## Q273500: Encarta 2000: Program Installs/Runs Slowly on System w/ Sony DVD

	Article: Q273500
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0; :
	Operating System(s): 
	Keyword(s): kbsetup kbimu
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Africana 2000 
	- Microsoft Encarta Encyclopedia 2000 
	- Microsoft Encarta Reference Suite 2000 
	- Microsoft Encarta Interactive World Atlas 2000 
	- Microsoft Encarta World English Dictionary, version 1.0 
	- Microsoft Works Suite 2000 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use any of the Microsoft Encarta 2000 programs listed at the beginning
	of this article, you may experience the following symptoms:
	
	- The program may install more slowly than expected.
	
	- The program may run more slowly than expected and may appear to stop
	  responding (hang).
	
	- When you use Encarta World Atlas, the Map section may not build on the
	  screen.
	
	CAUSE
	=====
	
	This behavior can occur if you have a Sony DVD-ROM drive installed in your
	computer and either one of the following video adapters:
	
	- ATI Rage IIc
	
	- STB Sprint 32 PCI
	
	RESOLUTION
	==========
	
	To resolve this issue, either replace the video adapter or replace the DVD-ROM
	drive.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
