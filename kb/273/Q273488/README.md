## Q273488: Reader Err Msg: You Are Attempting to Connect to an Invalid...

	Article: Q273488
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1.0,1.5
	Operating System(s): 
	Keyword(s): kberrmsg kbsetup kbimu
	Last Modified: 22-JAN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Reader, version 1.5 
	- Microsoft Reader for the Pocket PC, version 1.0 
	- Microsoft Reader version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start one of the programs listed at the beginning of this article, you
	may receive the following error message even after you use the Reader Repair
	feature:
	
	  You are attempting to connect to an invalid Activation site. Please run
	  Microsoft Reader setup again to correct this problem. You can download a new
	  copy of the Microsoft Reader from http://www.microsoft.com/reader/.
	
	CAUSE
	=====
	
	This behavior can occur if there is a problem with the Bookstore registry
	entries.
	
	The Reader Repair feature that is included with Microsoft Reader does not restore
	the default Bookstore registry entries.
	
	RESOLUTION
	==========
	
	To resolve this issue do one of the following methods:
	
	Uninstall and Reinstall Microsoft Reader
	
	1. Click Start, point to Settings, and then click Control Panel.
	
	2. Double-click Add/Remove Programs.
	
	3. Click Microsoft Reader in the list of currently installed programs, and then
	  click Add/Remove.
	
	4. Follow the instructions on the screen to uninstall Microsoft Reader.
	
	5. Reinstall Microsoft Reader from the following Microsoft Web site:
	
	  http://www.microsoft.com/reader
	
	Upgrade to Microsoft Reader 2.0
	
	To upgrade to Microsoft Reader 2.0, download the latest version from the
	following website:
	
	  http://www.microsoft.com/reader
	
	If you can not upgrade Microsoft Reader on your handheld device, contact your
	hardware manufacturer for support.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
