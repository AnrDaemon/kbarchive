## Q21392: Math Coprocessors Not Supported by Flight Simulator

	Article: Q21392
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	Flight Simulator does not take advantage of the 8087, 80287, or 80387 math
	coprocessor chips.
