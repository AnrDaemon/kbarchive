## Q21355: Transponder Never Sends &quot;Squawk&quot; Code

	Article: Q21355
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	The transponder never sends a "squawk" code. Flight Simulator does not allow you
	to respond to the control tower and the transponder does not emit any messages.
	This feature has been implemented in Flight Simulator Version 4.00.
