## Q38133: Hewlett-Packard Font Cartridges &quot;A&quot; Through &quot;G&quot; Description

	Article: Q38133
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:1.x,2.x,3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 1.1, 2.03, 2.1, 2.11, 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Following is a description of the capabilities of the HP Font Cartridges "A"
	through "G".
	
	The font cartridges support a print resolution of 300 x 300 dots per inch and
	have an average lifetime of 500 insertions per cartridge. Separate fonts are
	required for portrait and landscape. Some fonts may not be available in both
	orientations on a specific cartridge.
	
	For further information relating to these font cartridges, please contact
	Hewlett-Packard.
