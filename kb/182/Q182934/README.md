## Q182934: WD97: Not Enough Memory or Disk Space Saving File (Windows NT)

{% raw %}

	Article: Q182934
	Product(s): Word 97 for Windows
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdta
	Last Modified: 19-APR-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows, used with:
	   - the operating system: Microsoft Windows NT 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you save a document in Word in a local folder or to a shared folder on an
	NT 4.0 server, you may receive the following error message:
	
	  There is not enough memory or disk space to complete the operation.
	
	CAUSE
	=====
	
	This problem can occur when IBM AntiVirus for Windows NT is installed after the
	installation of the Microsoft Office 97 SR-1 Enterprise Update.
	
	RESOLUTION
	==========
	
	To resolve this problem, install IBM AntiVirus for Windows NT before installing
	Microsoft Office 97 SR-1 Enterprise Update on the workstation.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
