## Q234163: CheckPoint FireWall-1 Version 3.x and 4.0 Do Not Support PPTP Th

{% raw %}

	Article: Q234163
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kb3rdparty kberrmsg kbnetwork win95 win98
	Last Modified: 12-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows 98 
	- Microsoft Windows 95 
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Routing and Remote Access Service Update for Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to use Point-to-Point Tunneling Protocol (PPTP) through a
	CheckPoint FireWall-1 version 3.x or 4.0-based firewall that is using Network
	Address Translation (NAT), you may receive the following error message on your
	client computer:
	
	  Event ID 721 "PPP remote peer not responding" error 721.
	
	CAUSE
	=====
	
	This issue occurs because CheckPoint FireWall-1 version 3.x or 4.0 does not
	support PPTP.
	
	RESOLUTION
	==========
	
	To resolve this issue, contact CheckPoint to inquire about the availability of a
	fix for this issue.
