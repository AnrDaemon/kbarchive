## Q234589: Running Microsoft Office 2000 Premium Comments and Corrections

{% raw %}

	Article: Q234589
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 08-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Running Microsoft Office 2000 Premium ISBN 1-57231-945-3 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Running Microsoft Office 2000 Premium, ISBN
	1-57231-945-3.
	
	The following topics are covered:
	
	- Page xxvii: Listing Is Incorrect
	
	- Page 126: Zoom Drop-down List Has Changed
	
	- Page 278: Horizontal Line Dialog Box Has Changed
	
	- Page 964: Text Correction
	
	- Page 975: Correction For Reopening A Folder
	
	- Page 975: Text Correction
	
	- Page 1005: Replace The Word Office With Outlook
	
	- Page 1139: Screen Shot Doesn't Match Procedure
	
	- Page 1068: Heading Is Incorrect
	
	- Page 1194: First Bullet Item Is Incorrect
	
	- Page 1250: The Header Is Incorrect
	
	MORE INFORMATION
	================
	
	Page xxvii: Listing Is Incorrect
	--------------------------------
	
	Change the Listing:
	"Building a Financial Database: The Import Wizard"
	
	To:
	"Building a Financial Database: The Database Wizard"
	
	
	Page 126: Zoom Drop-down List Has Changed
	-----------------------------------------
	
	Several commands have been added to the bottom of the Word Zoom drop-down list.
	The screen at the bottom of page 126 does not reflect this.
	
	
	Page 278: Horizontal Line Dialog Box Has Changed
	------------------------------------------------
	
	Two commands have been added to this dialog box, and the icon on the Home button
	has been changed. The screen at the bottom of the page does not reflect this.
	
	
	Page 964: Text Correction
	-------------------------
	
	On page 964, in the first paragraph change:
	"However, the Corporate Or Workgroup installation lacks some of the Internet
	e-mail and Internet collaboration features provided by the Internet Only
	installation - for example, support for IMAP e-mail, group scheduling on the
	Internet using the iCalendar protocol, and the ability to verify e-mail
	addresses using directory services."
	
	To:
	"However, the Corporate Or Workgroup installation lacks some of the Internet
	e-mail and Internet collaboration features provided by the Internet Only
	installation - for example, support for IMAP e-mail and group scheduling on the
	Internet using the iCalendar protocol."
	
	
	Page 975: Correction For Reopening A Folder
	-------------------------------------------
	
	On page 975, in the first bulleted paragraph, change the first sentence from:
	"You can reopen the folder you previously had open by pointing to Go To on the
	View menu and choosing it, or by clicking the Previous Folder button on the
	Advanced toolbar."
	
	To:
	"You can reopen the folder you previously had open by clicking the Previous
	Folder button on the Advanced toolbar."
	
	
	Page 975: Text Correction
	-------------------------
	
	On page 975, in the second bulleted paragraph change:
	"After you've gone back one or more times, you can go forward again by choosing
	Next Folder from the Go menu or by clicking the Next Folder button on the
	Advanced toolbar."
	
	To:
	"After you've gone back one or more times, you can go forward again by clicking
	the Next Folder button on the Advanced toolbar."
	
	
	Page 1005: Replace The Word Office With Outlook
	-----------------------------------------------
	
	On page 1005, in the second paragraph, change the second sentence from:
	"... (which runs after the Office 2000 Startup Wizard)..."
	
	To:
	"... (which runs after the Outlook 2000 Startup Wizard)..."
	
	
	Page 1068: Heading Is Incorrect
	-------------------------------
	
	The heading on this page is incorrect.
	
	Change:
	Starting with Blank Presentations
	
	To:
	Starting with Blank Publications
	
	
	Page 1139: Screen Shot Doesn't Match Procedure
	----------------------------------------------
	
	The tags
	
	<body> and </body>
	
	in the upper-left corner of the document window on this screen do not appear
	unless you choose the View-Reveal Tags menu option.
	
	
	Page 1194: First Bullet Item Is Incorrect
	-----------------------------------------
	
	The first bullet item on this page is incorrect. In the first sentence, change
	"Table toolbar" to "Tables toolbar".
	
	
	Page 1250: The Header Is Incorrect
	----------------------------------
	
	Page 1250 discusses the New Database Wizard rather than The Import Wizard.
	
	Change the header:
	"Building a Financial Database: The Import Wizard"
	
	To:
	"Building a Financial Database: The Database Wizard"
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
