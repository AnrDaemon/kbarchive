## Q234706: Microsoft Pocket Guide to Microsoft PowerPoint 2000 Comments and

	Article: Q234706
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 15-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Pocket Guide to Microsoft PowerPoint 2000 ISBN 1-57231-975-5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Microsoft Pocket Guide to Microsoft
	PowerPoint 2000, ISBN 1-57231-975-5.
	
	The following topics are covered:
	
	- Page 60: Text Corrections
	
	- Page 161: Grammar Error
	
	MORE INFORMATION
	================
	
	Page 60: Text Corrections
	-------------------------
	
	Additional information is needed in the beginning of the first paragraph. It
	should read:
	
	"Click the Show toolbar button. The pane on the left displays three tabs..."
	
	In the second to last sentence of the first paragraph, change "it" to "its". It
	should read:
	
	"On the Contents tab, click the plus sign next to a book to display its topics."
	
	
	Page 161: Grammar Error
	-----------------------
	
	There is a typo in the last entry on the page under "About Microsoft".
	
	Change:
	"Displays the copyright notice and the software PowerPoint version number."
	
	To:
	"Displays the copyright notice and the software version number."
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
	
	Additional query words: 1-57231-975-5 EUBOOK PPT
