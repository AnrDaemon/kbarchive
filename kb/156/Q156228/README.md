## Q156228: Incorrect Help for NE1000 Network Adapter Setup

	Article: Q156228
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kbsetup kbui
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you click Help while installing a Novell NE1000 network adapter, the Help
	dialog boxes that are displayed indicate the NE2000 network adapter.
	
	RESOLUTION
	==========
	
	This symptom does not present a problem. The NE1000 adapter and the NE2000
	adapter are functionally similar and the Help files are generic.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Windows NT version
	4.0. We are researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
