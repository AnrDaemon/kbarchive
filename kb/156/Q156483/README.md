## Q156483: Support for Diamond Stealth64 Graphics 2001 Series Video Card

	Article: Q156483
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kbhw kbHardware
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Diamond Stealth64 Graphics 2001 series video card is not on the Windows NT
	4.0 Hardware Compatibility List (HCL). According to Diamond Multimedia's Web
	site, the Stealth64 Graphics 2001 series video card is not supported under
	Windows NT 4.0. The following Web page explains that a special trade- in price
	can be made for the Stealth64 Video 2001 accelerator card:
	
	  http://www.diamondmm.com/product-support/drivers/stealth64g-2001s.html
	
	MORE INFORMATION
	================
	
	The Diamond Stealth64 Graphics 2001 video card uses the ARK2000 chip. Hercules
	Computer Technology Stingray video cards also uses the ARK2000 chip. Hercules
	has created a Windows NT 4.0 beta driver (Spnt40.exe). It can be downloaded from
	the following FTP site:
	
	  ftp://ftp.hercules.com/pub/stingray/SPNT40.EXE
	
	The Hercules Stingray driver has been known to work with the Diamond Stealth64
	Graphics 2001 video card.
