## Q156529: Deleting Favorite Place May Cause Page Fault

{% raw %}

	Article: Q156529
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.2,1.3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.2, 1.3 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you delete a service from Favorite Places on MSN, The Microsoft Network,
	you may receive the following error message:
	
	  - EXPLORER caused an invalid page fault in module MCM.DLL.
	
	CAUSE
	=====
	
	This page fault is known to occur if you delete a service with a long name from
	Favorite Places.
