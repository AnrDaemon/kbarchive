## Q156979: Natural Keyboard Context Sensitive Key Not Working in Wordpad

	Article: Q156979
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kbnetwork
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Microsoft Natural Keyboard has a key that simulates the right mouse button
	and brings up the context sensitive menu. The context sensitive key on the
	Natural Keyboard does not work with Wordpad in Windows NT 4.0.
	
	WORKAROUND
	==========
	
	Use the right mouse button in Wordpad.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 4.0. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
