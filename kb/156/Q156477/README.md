## Q156477: SMS: Remote Control Agent Service Stops on a Windows NT Client

	Article: Q156477
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.2
	Operating System(s): 
	Keyword(s): kbnetwork kbtshoot kbsmsAdmin smsadmin smsremtshoot kbRemoteProg
	Last Modified: 31-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.2 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	If you select another Personal Computer Property, the next time Remote Control
	of a client computer running Windows NT is attempted, the Systems Management
	Server Remote Control Agent service stops.
	
	WORKAROUND
	==========
	
	To work around this problem, switch to another Personal Computer Property after
	stopping the Remote Control session. You can do this by closing the Quick Viewer
	window.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.2. We are researching this problem and will post new information here
	in the Microsoft Knowledge Base as it becomes available.
