## Q156523: SMS: Remote Control Redraw Problems with Diamond Video Card

	Article: Q156523
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.0,1.1,1.2
	Operating System(s): 
	Keyword(s): kbnetwork kbtshoot kbsmsAdmin smsadmin smsremtshoot kbRemoteProg
	Last Modified: 31-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server versions 1.0, 1.1, 1.2 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	Remote control sessions get established without problem, but almost immediately
	afterwards, redraw problems begin in the QuickViewer. After you click buttons or
	icons, QuickViewer fails to refresh correctly.
	
	These symptoms are not exhibited on the client's screen.
	
	This problem has been found when using the Diamond Stealth64 S3-968 version
	4.02.325 display driver for Windows 95.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	versions 1.0, 1.1, and 1.2. This problem was corrected in the latest Microsoft
	Systems Management Server 1.2 U.S. Service Pack. For information on obtaining
	the service pack, query on the following word in the Microsoft Knowledge Base
	(without the spaces):
	
	  S E R V P A C K
	
	Additional query words: diagnostics helpdesk usertsr win95
