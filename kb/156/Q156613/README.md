## Q156613: Writer 2: &quot;The Printer Could Not Be Found&quot; Error Message

{% raw %}

	Article: Q156613
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:2.0
	Operating System(s): 
	Keyword(s): kbfaq
	Last Modified: 12-JUN-2001
	
	2.00
	WINDOWS
	kbprint kbfaq
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Creative Writer for Windows, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you try to print from within Creative Writer 2, you may receive the following
	error:
	
	  The printer could not be found.
	
	CAUSE
	=====
	
	This behavior may occur if you change and remove the current printer while
	Creative Writer 2 is open.
	
	Creative Writer defaults to the printer selected the first time you print from
	within the program. If you try to print to another printer within Creative
	Writer 2 after removing or renaming the printer driver, you may also get this
	error message.
	
	RESOLUTION
	==========
	
	Use one of the following solutions.
	
	Solution A
	----------
	
	Open a new document:
	
	1. Click the File folder icon, then click New Project.
	
	2. Confirm to save changes if you desire.
	
	Solution B
	----------
	
	Close and reopen Creative Writer:
	
	1. Click the File folder icon, then click Quit.
	
	2. Confirm to save changes if you desire.
