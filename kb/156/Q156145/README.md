## Q156145: Full Court Press: Setup Notes

	Article: Q156145
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NBA Full Court Press for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains information that will help you install Microsoft Full
	Court Press.
	
	MORE INFORMATION
	================
	
	If you are using a 2x CD-ROM drive, you should do a Typical or Custom
	installation for better performance.
	
	If you are using 16-bit CD-ROM drivers for your CD-ROM, Autorun may not work
	properly. To manually run Setup:
	
	1. Make sure your Full Court Press compact disc is inserted in your CD-ROM drive
	  and run Explorer.
	
	2. Double-click your CD-ROM drive (usually drive D.)
	
	3. Double-click the Setup icon to start installation.
