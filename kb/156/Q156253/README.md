## Q156253: Bunyan: 16-Bit Setup Creates Unused Directories

	Article: Q156253
	Product(s): Microsoft Home Kids Products
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft P.J.'s Reading Adventures: Microsoft Paul Bunyan, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The 16-bit Setup for Paul Bunyan creates several empty directories on your hard
	drive.
	
	MORE INFORMATION
	================
	
	After installing the 16-bit version of Paul Bunyan, you will have the following
	directory structure on your hard drive, where C: is the drive on which you
	installed the program:
	
	  c:\mskids\bunyan
	  c:\mskids\shared
	  c:\mskids\shared\stuff
	  c:\mskids\shared\users
	  c:\mskids\users
	
	Of these directories, the following are empty:
	
	  c:\mskids\shared
	  c:\mskids\shared\stuff
	  c:\mskids\shared\users
	  c:\mskids\users
	
	Additional query words: kids mskids kbmm multimedia multi-media multi media 1.00 pj bunny 16 rabbit ears
