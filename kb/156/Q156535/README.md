## Q156535: Cannot Open Phone Book File After Using Internet Setup Wizard

{% raw %}

	Article: Q156535
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.2,1.3,2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.2, 1.3, 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you remove the Dial-Up adapter using the Add/Remove Programs tool in Control
	Panel and run the Internet Setup Wizard by double-clicking the Internet Explorer
	icon, you may receive the following error message when you attempt to connect to
	MSN, The Microsoft Network:
	
	  Unable to set the Internet phone number because of the following error:
	  Cannot open the phone book file.
	
	You may also receive this message:
	
	  Cannot open the phone book file.
	
	CAUSE
	=====
	
	The changes made by the Internet Setup Wizard are not effective until you
	restart the computer.
	
	RESOLUTION
	==========
	
	Restart the computer.
	
	MORE INFORMATION
	================
	
	If you run the Internet Setup Wizard by double-clicking the The Microsoft
	Network icon instead of double-clicking the Internet Explorer icon, you are
	prompted to restart your computer and the system is set up correctly.
