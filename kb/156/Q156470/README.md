## Q156470: Kids: Summary of Audio Used in Kids Titles

	Article: Q156470
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:1.0,1.1,1.1a,1.1b,1.1c,2.0; :1.0
	Operating System(s): 
	Keyword(s): kbsound
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft 3D Movie Maker for Windows, version 1.0 
	- Microsoft Creative Writer for Windows, versions 1.1, 1.1a, 1.1b, 1.1c, 2.0 
	- Microsoft Explorapedia series: World of People for Windows, version 1.0 
	- Microsoft Explorapedia series: World of Nature for Windows, version 1.0 
	- Microsoft Fine Artist for Windows, versions 1.1, 1.1a, 1.1b, 1.1c 
	- Microsoft P.J.'s Reading Adventures: Microsoft Koi and the Kola Nuts, version 1.0 
	- Microsoft P.J.'s Reading Adventures: Microsoft Paul Bunyan, version 1.0 
	- Microsoft How the Leopard Got His Spots, version 1.0 
	- Scholastic's Magic School Bus series: Explores the Human Body for Windows, version 1.0 
	- Scholastic's Magic School Bus series: Explores the Ocean for Windows, version 1.0 
	- Scholastic's Magic School Bus series: Explores the Rainforest for Windows, version 1.0 
	- Scholastic's Magic School Bus series: Explores the Solar System for Windows, version 1.0 
	- Scholastic's Magic School Bus series: Explores Inside the Earth for Windows, version 1.0 
	- Scholastic's Magic School Bus series: Explores in the Age of Dinosaurs for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following is a listing of the type of audio used in the existing Kids
	products:
	
	Products                 Type of Audio
	--------                 -------------
	
	Magic School Bus:
	MSB Human Body          WAV, compressed WAV, MIDI
	MSB Solar               WAV only
	MSB Earth               WAV, compressed WAV, CD Audio
	MSB Ocean               WAV, compressed WAV, CD Audio
	MSB Dinosaurs           WAV, compressed WAV only
	MSB Rainforest          WAV, compressed WAV only
	
	3D Movie Maker           WAV, compressed WAV, MIDI
	Creative Writer/ 
	Fine Artist 1.x          Compressed WAV
	Creative Writer 2        WAV (uses SoundMan), MIDI
	
	Explorapedia             WAV, compressed WAV
	
	P.J.'s Reading
	Adventures               WAV, compressed WAV
