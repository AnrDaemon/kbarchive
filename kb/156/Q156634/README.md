## Q156634: Writer 2: Drop Caps Disappear When Posted

{% raw %}

	Article: Q156634
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Creative Writer for Windows, version 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you are using the Web Page tool in Creative Writer 2, features that are
	available in the Creative Writer 2 word processor may not translate to HTML
	correctly. For example, although drop caps are available in the Web Page tool,
	they do not translate to a posted HTML page.
	
	MORE INFORMATION
	================
	
	Drop caps disappear on an HTML web page. Similarly, drop caps do not hold a link
	color like other characters do.
