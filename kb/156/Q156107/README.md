## Q156107: Encarta 1996: Unable to Enter Yearbook Builder

	Article: Q156107
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 96 Encyclopedia for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Yearbook Builder does not open. At the Encarta Home screen, when you click
	Stay Current, the Yearbook Builder doesn't open. The mouse cursor changes
	briefly to an hourglass, then changes back to a pointer.
	
	CAUSE
	=====
	
	A bad copy of the August 1996 Yearbook file was downloaded.
	
	RESOLUTION
	==========
	
	Download the Aug96a.eyb file from the Internet again, if the update is still
	available. If the August 96 update is no longer available, delete the Yb896a.ybk
	file from the Yearbook folder.
	
	MORE INFORMATION
	================
	
	A corrupted version of the August 96 Yearbook Builder file was posted to the
	Internet and The Microsoft Network (MSN). The bad version of this file will have
	a file date earlier than 8/14/96.
	
	To delete the corrupted August Yearbook Builder file, do the following:
	
	Windows 95/98
	-------------
	
	1. Click the Start button, point to Find, and click Files or Folders.
	
	2. Type the following in the Named area and click Find Now:
	
	  "Yb896a.ybk" (without the quotation marks)
	
	3. When the file is found, press your keyboard DELETE key.
	
	4. Restart Encarta, and make sure the Yearbook Builder is operational.
	
	5. Download the August 96 Yearbook Builder file again.
	
	Windows 3.x
	-----------
	
	1. In the Main group program group, Open File Manager.
	
	2. On the File menu, click Search.
	
	3. Type the following in the Search For area:
	
	  "Yb896a.ybk" (without the quotation marks)
	
	4. Type the following in the Start From area and click OK:
	
	  "C:\" (without the quotation marks)
	
	5. When the file is found, press your keyboard DELETE key.
	
	6. Restart Encarta and make sure the Yearbook Builder is operational.
	
	7. Download the August 96 Yearbook Builder file again.
