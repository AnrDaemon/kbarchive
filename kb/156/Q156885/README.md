## Q156885: XGEN: Errors When Using the Header.exe Tool

	Article: Q156885
	Product(s): Microsoft Exchange
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 05-APR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you use the Header.exe tool from the Microsoft Exchange Resource Kit, you
	might receive the following error:
	
	  Export Error...
	
	  Following error occurred: Could not bind to the Microsoft Exchange Directory
	  server XXXXX. The Microsoft Exchange Server computer does not respond.
	
	This error will be generated if the Microsoft Exchange Directory Service is not
	running on the computer specified.
	
	MORE INFORMATION
	================
	
	This tool must be run on a computer that the Microsoft Exchange Server has been
	installed on.
