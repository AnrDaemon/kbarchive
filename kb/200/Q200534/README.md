## Q200534: Pointing Device Affects Encarta When Running in the Background

	Article: Q200534
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbenv kbtool kbimukbbuglist
	Last Modified: 25-JUN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Encyclopedia 99 
	- Microsoft Encarta Reference Suite 99 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you use your pointing device to click an item in an active program, you may
	also perform an unexpected pointing device action in Encarta 99 if an
	InterActivities window is open in the background.
	
	
	RESOLUTION
	==========
	
	To work around this issue, minimize Encarta before you switch to another
	program.
	
	Additional query words: kbimu multi multi-media media mm enc99
