## Q200843: Unable to Install Windows 95/98 Year 2000 Updates

	Article: Q200843
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 1,2,2.1,2.5
	Operating System(s): 
	Keyword(s): kbsetup win98
	Last Modified: 11-JAN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 98 Second Edition 
	- Microsoft Windows 98 
	- Microsoft Windows 95 OEM Service Release, versions 1, 2, 2.1, 2.5 
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you try to install the Windows 95 Year 2000 Update (W95y2k.exe), the
	Windows 98 Year 2000 Update (Y2k.exe or Y2kw98.exe), or the Windows 98 Year 2000
	Update 2 (Y2kw98_2.exe), you may receive the following error message:
	
	  This update is not designed for your version of Windows. Press OK to exit.
	
	CAUSE
	=====
	
	This issue can occur for either of the following reasons:
	
	- You try to install the Windows 95 Year 2000 Update (W95y2k.exe) on a computer
	  running Windows 98, Windows 98 Second Edition, or Windows NT. This update is
	  designed for use only on Windows 95-based computers.
	
	- You try to install the Windows 98 Year 2000 Update (Y2k.exe or Y2kw98.exe) or
	  the Windows 98 Year 2000 Update 2 on a computer running Windows 95, Windows
	  98 Second Edition, or Windows NT. These updates are designed for use only on
	  Windows 98-based computers.
	
	NOTE: Windows 98 Second Edition includes all of the Year 2000 fixes included in
	both the Windows 98 Year 2000 Update and the Windows 98 Year 2000 Update 2.
	
	STATUS
	======
	
	This behavior is by design.
	
	MORE INFORMATION
	================
	
	For additional information about the Windows 95 or Windows 98 Year 2000 Updates,
	please see the following articles in the Microsoft Knowledge Base:
	
	  Q167075 Availability of the Windows 98 Year 2000 Update
	
	  Q225050 Availability of the Windows 95 Year 2000 Update
