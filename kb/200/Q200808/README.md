## Q200808: SMS: Tabs in MMC Help Are Not Accessible with the TAB Key

	Article: Q200808
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:2.0
	Operating System(s): 
	Keyword(s): kbMMC kbsms200
	Last Modified: 21-MAY-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Systems Management Server version 2.0, the tabs in the Systems Management
	Server Help are not accessible by using the TAB or SHIFT+TAB keys.
	
	WORKAROUND
	==========
	
	The only way to access the Contents, Index, and Search tabs of Help by using a
	keyboard is to use the ALT+C or ALT+S keystroke combination.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 2.0.
