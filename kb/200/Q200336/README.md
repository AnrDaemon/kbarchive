## Q200336: SMS: PCM Application Pauses Using a Novell Distribution Server

{% raw %}

	Article: Q200336
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.2
	Operating System(s): 
	Keyword(s): kbsms120 kbOSNovell kbPCM kbSoftwareDist
	Last Modified: 07-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When using the Novell IntranetWare client for Windows 95 workstations version
	2.5 through 3.02, the package command manager appears to hang for 15 to 45
	seconds and will not respond to user interaction.
	
	WORKAROUND
	==========
	
	This problem does not occur when using earlier versions of the Novell client.
	
	MORE INFORMATION
	================
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.2. We are researching this problem and will post new information here
	in the Microsoft Knowledge Base as it becomes available.
