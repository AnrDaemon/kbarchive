## Q246796: STOP 0x01e Error Occurs with pcANYWHERE and Compaq Surveyor

	Article: Q246796
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use Symantec pcANYWHERE and Compaq Surveyor on a computer running
	Microsoft Windows NT Server 4.0 or Windows NT Workstation 4.0, the following
	STOP error message may appear periodically:
	
	  STOP 0x0000001e (0xc0000005,0x801922ab,0x00000000,0x00000004)
	
	CAUSE
	=====
	
	This behavior can occur because of a problem either with pcANYWHERE or with the
	interaction between pcANYWHERE and Surveyor.
	
	RESOLUTION
	==========
	
	To work around this behavior, disable or uninstall pcANYWHERE.
