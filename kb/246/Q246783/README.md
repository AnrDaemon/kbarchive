## Q246783: Server Running NetShield NT and QuotaAdvisor Stops Responding

	Article: Q246783
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-JUN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use McAfee NetShield NT 4.02 and WQuinn's QuotaAdvisor on a computer
	running Microsoft Windows NT Server 4.0 Service Pack 3, the computer may stop
	responding and Event Viewer may record Event ID 2022.
	
	CAUSE
	=====
	
	The cause is unknown.
