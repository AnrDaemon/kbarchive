## Q246121: XCLN: Contacts in Public Folder Are Not Shown in Bold

{% raw %}

	Article: Q246121
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:2000
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook 2000 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you use Outlook 2000 to view Contacts by category in a Public Folder, these
	items do not appear bold even if the items have not been read.
	
	MORE INFORMATION
	================
	
	Outlook 2000 uses bold to indicate a distribution list in the Contacts folder.
	Other contacts are shown in regular type.
