## Q246648: XCLN: How to Install the Routing Wizard Sample Application

	Article: Q246648
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:2000,98
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook 2000 
	- Microsoft Outlook 98 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Routing Wizard is a sample application that allows users to create workflow
	applications using routing objects on a computer running Microsoft Exchange
	Server 5.5 Service Pack 1 (SP1) or later. This allows administrators to create
	e-mail-based workflows to track approvals, rejections, and comments for a
	particular project.
	
	MORE INFORMATION
	================
	
	To install the Routing Wizard, you must have Outlook 98 or Outlook 2000
	installed on the computer on which you want to use the Routing Wizard.
	
	To install on a computer running Microsoft Windows 95 or Microsoft Windows 98:
	
	  On the SP1 CD, locate the Eng\Server\Support\Collab\Sampler\Routing\Win95
	  folder, and run RWSetup.exe.
	
	To install on a computer running Microsoft Windows NT:
	
	  On the SP1 CD, locate the
	  Eng\Server\Support\Collab\Sampler\Routing\Winnt\I386 folder, and run
	  RWSetup.exe.
	
	The application creates an application icon on the Start menu, under Programs,
	labeled Microsoft Exchange Routing Wizard. To start the Routing Wizard, click
	this icon.
