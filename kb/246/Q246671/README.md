## Q246671: MapPoint: Helpful Web Sites for Creating Maps

	Article: Q246671
	Product(s): Microsoft Automap
	Version(s): 
	Operating System(s): 
	Keyword(s): kburl kbimu
	Last Modified: 14-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MapPoint 2000 
	- Microsoft MapPoint 2001 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following Internet Web sites may be helpful when you create a map in
	Microsoft MapPoint.
	
	List of Counties
	----------------
	
	You can view a list of all of the counties in the United States at the following
	government census Web site:
	
	  http://www.census.gov/datamap/fipslist/AllSt.txt
	
	List of Abbreviations for States
	--------------------------------
	
	You can view a list of official abbreviations for each state and territory at the
	following United States Post Office (USPS) Web site:
	
	  http://www.usps.gov/ncsc/lookups/usps_abbreviations.htm
	
	Search by Zip Code
	------------------
	
	You can search for the city or state within which a certain zip code is located
	at the following USPS Web site:
	
	  http://www.usps.gov/ncsc/lookups/lookup_ctystzip.html
