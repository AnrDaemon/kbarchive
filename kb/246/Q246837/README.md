## Q246837: XWEB: OWA Can't Display Previous Month's Free and Busy Info

	Article: Q246837
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook Web Access, version 5.5 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you use Outlook Web Access (OWA) to view a meeting attendee's free and busy
	information, the free and busy information is unavailable for the previous
	month.
	
	CAUSE
	=====
	
	OWA is unable to view free and busy information for months previous to the
	current one.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Outlook Web Access
	5.5.
