## Q195058: Pinball Arcade: Distorted Graphics or Horizontal Lines in Game

	Article: Q195058
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbenv kbimu msgamekbfaq
	Last Modified: 15-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Pinball Arcade, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Pinball Arcade, you may experience the following symptoms:
	
	- If you press ALT+TAB to switch to a different program, and then press ALT+TAB
	  to switch back to Pinball Arcade, the graphics in the game may be severely
	  distorted.
	
	- When you play the game, horizontal lines may appear on the screen and
	  eventually hide the game graphics.
	
	CAUSE
	=====
	
	This behavior can occur if both of the following conditions are true:
	
	- An ATI 3D Rage Pro display adapter is installed in your computer.
	
	- You are using an outdated video driver for the ATI 3D Rage Pro display
	  adapter.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest video driver for the ATI
	3D Rage Pro display adapter from the following ATI Web site:
	
	  http://www.atitech.com
