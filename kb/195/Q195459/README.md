## Q195459: Tutor Err Msg: Insufficient Available Space to Install Program

{% raw %}

	Article: Q195459
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbenv kberrmsg kbsetup mskids kbimu
	Last Modified: 30-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft My Personal Tutor: Sam's Hide and Seek Adventure, version 1.0 
	- Microsoft My Personal Tutor: Sky's Space Station Voyage, version 1.0 
	- Microsoft My Personal Tutor: Turru's Daring Sea Quest, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to install a 1st & 2nd grade My Personal Tutor program, you
	may receive an error message stating that you do not have sufficient available
	space on your hard disk to install the program.
	
	NOTE: You may receive this message even if sufficient space is available on your
	hard disk.
	
	CAUSE
	=====
	
	This behavior can occur if you previously installed a My Personal Tutor program
	on a hard disk that does not have sufficient available space to install an
	additional My Personal Tutor program.
	
	RESOLUTION
	==========
	
	To prevent this issue from occurring, install all My Personal Tutor 1st &
	2nd grade programs on the same hard disk.
	
	NOTE: This hard disk must have sufficient available space for all of the My
	Personal Tutor programs you want to install.
	
	
	MORE INFORMATION
	================
	
	For more information about the amount of hard disk space required to install the
	My Personal Tutor 1st & 2nd grade programs, please see the following article
	in the Microsoft Knowledge Base:
	
	  Q192834 My Personal Tutor 1st & 2nd Grade: Minimum System Requirements
