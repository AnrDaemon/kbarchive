## Q195415: XADM: Key Management (KM) Server in Performance Monitor

	Article: Q195415
	Product(s): Microsoft Exchange
	Version(s): winnt:5.0,5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 26-APR-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 5.0, 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Key Management (KM) Server database and process is added to the Exchange
	Server computer when you install Key Management Server. As a result, the service
	can now be monitored in the Windows NT Performance Monitor.
	
	MORE INFORMATION
	================
	
	Through Task Manager, the process for KM Server can be monitored (click the
	Processes tab in Task Manager); it is listed as Kmserver.exe.
	
	Additionally, in Performance Monitor under the Process and Database objects, the
	Key Management Server can be monitored.
	
	Additional query words: KMS
