## Q195521: XGEN: Exchange &quot;How-To&quot; Documents Available on the WWW

	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Windows 3.x client, versions 4.0, 5.0 
	- Microsoft Exchange Windows 95/98 client, versions 4.0, 5.0 
	- Microsoft Exchange Windows NT client, versions 4.0, 5.0 
	- Microsoft Exchange Server, versions 4.0, 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	"How-To" documents are designed to prepare you for the successful completion of
	specific tasks. This is accomplished by clarifying prerequisite knowledge or
	information from a third party, other system(s), or a customer's environment.
	How-To documents consist of step-by-step instructions for commonly requested
	system configurations. These documents include graphics and screen shots to help
	illustrate the proper steps.
	
	The Exchange Server How-To documents are available on the World Wide Web site:
	
	  http://support.microsoft.com/support/exchange/content/Howtos/Howtos.asp
	
	
	MORE INFORMATION
	================
	
	The How-To documents for Exchange Server may change from time to time. The
	following lists the documents currently available at the above Web site. For
	assistance in finding a specific document on the Web, search on a Document
	Identifier supplied below.
	
	  Identifier  Title
	  ----------   -----
	  XGIMSHOST   Dial-up Internet Mail Service Host Configuration
	  XGRASTCP    How to Configure the Dynamic RAS Connector Over TCP/IP
	  XGCCMAIL    Microsoft Exchange Connector for Lotus cc:Mail
	  XGIMSREP    How to Replicate Exchange Server Directories Using the
	               Internet Mail Service
	  XGMSMAIL    How to Configure the Microsoft Mail Connector
	  XGOVM       Microsoft Exchange Connector for OfficeVision/VM Load
