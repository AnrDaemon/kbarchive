## Q195632: Err Msg: Encarta Virtual Globe Requires a Pentium 66...

	Article: Q195632
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbenv kberrmsg kbhw kbimu kbHardwarekbfaq
	Last Modified: 25-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Virtual Globe 99, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to start Encarta Virtual Globe 99, you may receive the
	following error message:
	
	  Encarta Virtual Globe requires a Pentium 66 or faster processor. You will not
	  be able to run Encarta Virtual Globe on this computer.
	
	CAUSE
	=====
	
	This behavior can occur if a non-Intel processor is installed in your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, click Cancel to bypass the error message. Encarta Virtual
	Globe should start and run without any further problems.
	
	MORE INFORMATION
	================
	
	Some processors do not identify themselves as a Pentium or Pentium-class
	processor to the operating system. Encarta Virtual Globe checks to ensure that
	the computer meets the minimum system requirements when the program starts.
