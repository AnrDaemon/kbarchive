## Q195729: Virtual Globe 99: Cannot Press ALT+SPACEBAR to Restore Program

{% raw %}

	Article: Q195729
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbenv kbimu
	Last Modified: 19-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Virtual Globe 99, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you minimize Microsoft Encarta Virtual Globe 99, you cannot press
	ALT+SPACEBAR to restore the program from the System menu. The Restore command on
	the System menu is unavailable.
	
	RESOLUTION
	==========
	
	To resolve this issue, click Encarta Virtual Globe on the taskbar to restore the
	program.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Encarta Virtual Globe
	99, version 1.0.
