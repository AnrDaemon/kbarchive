## Q195692: WD97: Collapsed Outline Appears Expanded in Print Preview

	Article: Q195692
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you view and print a collapsed outline from Print Preview, all levels are
	displayed and printed.
	
	WORKAROUND
	==========
	
	To view a collapsed outline, click the appropriate Show Heading button on the
	Outline toolbar. To print a collapsed outline, print your document from outline
	view.
	
	MORE INFORMATION
	================
	
	In outline view, you control which levels of your document you want to be
	displayed or printed using the Show Heading buttons on the Outline toolbar.
