## Q195814: Golf 1999 Edition: Minimum System Requirements

	Article: Q195814
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbref kbimu msgamekbfaq
	Last Modified: 09-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Golf 1999 Edition, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the minimum system requirements needed to play Microsoft Golf
	1999 edition.
	
	MORE INFORMATION
	================
	
	To play Golf 1999 edition, you need:
	
	- A Multimedia PC with a Pentium 90 MHz or faster processor.
	
	- Microsoft Windows 95 operating system or later, or Windows NT Workstation
	  operating system version 4.0 with Service Pack 3 or later.
	
	- 16 megabytes (MB) of random-access memory (RAM) for Windows 95 or later, or
	  24 MB of RAM for Windows NT with a 60-MB swap file.
	
	- 70 MB of available hard-disk space.
	
	- A DirectDraw compatible, 16-bit color PCI or AGP video adapter and monitor
	  that supports 640 x 480 or higher resolution (Direct3D accelerated video
	  adapter recommended).
	
	- A quad-speed or faster CD-ROM drive.
	
	- A Microsoft Mouse or compatible pointing device.
	
	- A DirectSound compatible sound card with speakers or headphones.
	
	- 28,800 bits per second (bps) or faster modem for Internet or head-to-head
	  play.
	
	- An Internet service provider (ISP) and access to the Internet for Internet
	  play.
	
	- A Web browser that supports ActiveX and frames such as Microsoft Internet
	  Explorer 3.02 or later for Zone.com play.
	
	  NOTE: Internet access may require payment of a separate fee to an Internet
	  service provider. Connect time charges may apply.
