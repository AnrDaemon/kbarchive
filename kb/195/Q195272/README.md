## Q195272: Running Microsoft Outlook 98 Comments and Corrections

	Article: Q195272
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocerr
	Last Modified: 16-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Running Microsoft Outlook 98 ISBN 1-57231-840-6 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Running Microsoft Outlook 98, ISBN
	1-57231-840-6.
	
	The following topics are covered:
	
	- Page 134: Text Incorrect
	
	MORE INFORMATION
	================
	
	Page 134: Text Incorrect
	------------------------
	
	Page 134, step 1 for resending an email:
	Change: "1. Choose Resend This Message from the Tools menu."
	To: "1. Choose Resend This Message from the Actions menu." (without the quotation
	marks)
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
