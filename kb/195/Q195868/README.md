## Q195868: Virtual Globe 99: Boxes Replace Letters in Some Find List Items

	Article: Q195868
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbimu
	Last Modified: 19-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Virtual Globe 99, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the Pinpointer in Microsoft Encarta Virtual Globe 99, some items in
	the Find list may be displayed with boxes replacing letters.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Encarta Virtual Globe
	99, version 1.0.
	
	Additional query words: multi multi-media media mm world atlas evg
