## Q195266: Microsoft Windows 98 Starts Here Comments and Corrections

	Article: Q195266
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocerr
	Last Modified: 08-JAN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Windows 98 Starts Here ISBN 1-57231-698-5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press title Microsoft Windows 98 Starts Here, ISBN
	1-57231-698-5.
	
	The following topics are covered:
	
	- Wordpad Document Option Not Available
	
	MORE INFORMATION
	================
	
	WordPad Document Option Not Available
	-------------------------------------
	
	Introducing Windows 98, Working with Files, Creating and Renaming Folders and
	Files, Part 2 of 2, Step 2 of 5 instructs you to point to New from the File menu
	and click WordPad document.
	
	If WordPad was not installed when Microsoft Windows 98 was installed, the WordPad
	Document option will not be there.
	
	To install WordPad, follow these steps:
	
	1. Click Start, point to Settings, and then click Control Panel.
	
	2. Double-click Add/Remove Programs.
	
	3. In Add/Remove Programs, select the Windows Setup tab.
	
	4. From the Components list click Accessories, and then click Details.
	
	5. Scroll to the bottom of the Components list and click WordPad, and click OK.
	
	6. Click OK.
	
	7. Depending on how you originally installed Microsoft Windows 98, you may be
	  asked to insert the Windows 98 compact disc into your CD-ROM drive.
	
	8. Once WordPad has been installed, close the Control Panel.
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
