## Q195344: My Personal Tutor: Sound Is Distorted When You Run Program

{% raw %}

	Article: Q195344
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbenv kbsound mskids kbimu
	Last Modified: 04-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft My Personal Tutor: Sam's Hide and Seek Adventure, version 1.0 
	- Microsoft My Personal Tutor: Sky's Space Station Voyage, version 1.0 
	- Microsoft My Personal Tutor: Turru's Daring Sea Quest, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run one of the My Personal Tutor programs listed earlier in this
	article, the sound may be distorted. The sound quality is especially poor when
	you sign in to the program.
	
	CAUSE
	=====
	
	This behavior can occur if both of the following conditions are true:
	
	- You are running a My Personal Tutor 1st & 2nd grade title on a Microsoft
	  Windows 95-based computer.
	
	- An ES1788 PnP sound card is installed in the computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest sound driver for the
	ES1788 PnP sound card from the following ESS Technology Web site:
	
	  http://www.esstech.com
