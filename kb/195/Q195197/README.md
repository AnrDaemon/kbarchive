## Q195197: WD97: &quot;Feed From Envelope&quot; Option Reset Under Windows NT 3.51

	Article: Q195197
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta kbenvelope word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	You cannot permanently set the Feed From option in the Envelopes And Labels
	feature on the Tools menu under Windows NT 3.51.
	
	
	CAUSE
	=====
	
	The Feed From option is restored to the default each time you quit Word.
	
	RESOLUTION
	==========
	
	To work around this problem, after you restart Word, change the Feed From option
	to the setting you want before you print envelopes using the Envelopes And
	Labels feature.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows.
	This problem was corrected in Microsoft Word 97 SR-1.
	
	For additional information about SR-1, click the article number below to view the
	article in the Microsoft Knowledge Base:
	
	  Q172475 OFF97: How to Obtain and Install MS Office 97 SR-1
