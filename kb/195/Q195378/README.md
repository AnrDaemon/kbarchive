## Q195378: DOC: Release Notes Incorrectly State that IStorage Is Broken

	Article: Q195378
	Product(s): Microsoft C Compiler
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbdocerr kbOAK kbVC500 kbOSWinCE210
	Last Modified: 17-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows CE 2.10 Enhancement Pack for Windows CE Embedded Toolkit for Visual C++ 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Release Notes for the product described above incorrectly state that the
	IStorage and OLE Storage API for Windows CE version 2.1 are broken.
	
	In fact, these functions are available in Windows CE version 2.1.
	
	REFERENCES
	==========
	
	Release Notes: Istorage and OLE Storage APIs
