## Q195134: WD97: Can't Remove Footnote Separator from Master Document

	Article: Q195134
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you delete the footnote separator from a master document that contains
	footnotes, the separator reappears the next time you open the master document.
	
	WORKAROUND
	==========
	
	Remove the footnote separator each time you open the master document, or record
	a macro to automate this process.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the products listed at the
	beginning of this article.
