## Q195282: Windows 98 Step by Step Comments and Corrections

	Article: Q195282
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocerr
	Last Modified: 14-JAN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Windows 98 Step by Step ISBN 1-57231-683-7 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book, "Microsoft Windows 98 Step by Step", ISBN
	1-57231-683-7.
	
	The following topics are covered:
	
	- Page xviii: Incorrectly Refers To Second Video Clip
	
	- Page 34: Incorrect Reference To My Documents
	
	- Page 40: Active Desktop Gallery Has Changed
	
	- Page 70: Incorrect Folder Name
	
	MORE INFORMATION
	================
	
	Page xviii: Incorrectly Refers To Second Video Clip
	---------------------------------------------------
	
	Change Step 4 from:
	
	"In the Registered File Types section, click the second Video Clip option."
	
	To:
	"In the Registered File Types section, click the first Video Clip option."
	
	
	Page 34: Incorrect Reference To My Documents
	--------------------------------------------
	
	The second line on page 34 contains an incorrect reference to the My Documents
	folder.
	
	Change:
	"you open My Documents using the single-click method."
	
	To:
	"you open My Computer using the single-click method."
	
	
	Page 40: Active Desktop Gallery Has Changed
	-------------------------------------------
	
	Page 40-41: "Add a new active content item to the desktop", steps 6 and 7: The
	instructions to locate MSNBC Weather Map in the gallery index and click the
	MSNBC Weather Map link can not be carried out because the Web site has changed.
	
	However, other Active Desktop Items are available. For further information on how
	to add an Active Desktop item to the desktop, please see the following article
	in the Microsoft Knowledge Base:
	
	  Q171437 How to Configure the Active Desktop
	
	
	
	Page 70: Incorrect Folder Name
	------------------------------
	
	On page 70, steps 4 and 5 refer to the "Rogue Cellars Logos" folder. It should
	refer to the "Rogue Cellars" folder to read as follows.
	
	4. In the Market Plans windows, click the Up One Level button, click the Client
	Logos folder, and then click the Rogue Cellars folder.
	The Rogue Cellars folder opens.
	
	5. On the Standard toolbar, click the Paste button.
	A copy of the Duffy Vineyards Plan file is placed in the Rogue Cellars folder.
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
