## Q195285: MS SNA Server Training Kit Comments &amp; Corrections

	Article: Q195285
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocerr
	Last Modified: 07-JAN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft SNA Server Training Kit 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book "Microsoft SNA Server Training Kit," ISBN
	1-57231-932-1.
	
	Contents:
	
	- Q & A Appendix Missing From Ebook
	
	MORE INFORMATION
	================
	
	Q & A Appendix Missing From Ebook
	---------------------------------
	
	The Questions and Answers Appendix for Microsoft SNA Server Training Kit was not
	included in the Ebook. The Questions and Answers Appendix is available in the
	print book, starting on page 771.
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
