## Q195061: Pinball Arcade: Cannot Start a Game on Any Table

{% raw %}

	Article: Q195061
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbenv kbimu msgamekbfaq
	Last Modified: 10-AUG-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Pinball Arcade, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to play Pinball Arcade, you may be unable to start a game on
	any pinball table.
	
	CAUSE
	=====
	
	This behavior can occur if all of the following conditions are true:
	
	- Your color palette is set to 256 colors.
	
	- Your screen resolution is set to 1024 x 768 pixels.
	
	- A Matrox Millennium II display adapter is installed in your computer.
	
	- You are using an outdated video driver for the Matrox Millennium II display
	  adapter.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest video driver for the
	Matrox Millennium II video adapter from the following Matrox Web site:
	
	  http://www.matrox.com
