## Q195687: WD97: Bookmark at Beginning of Line Is Lost When Copying

{% raw %}

	Article: Q195687
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you paste a selection that contains a bookmark as the first item in the
	selection into a new document, the bookmark may be lost.
	
	WORKAROUND
	==========
	
	To work around this behavior, make sure the selection contains some information
	(for example, a space) before the bookmark. When you paste the selection, the
	bookmark will be retained.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
	
	MORE INFORMATION
	================
	
	A bookmark is a location or selection of text that you name for reference
	purposes. Word marks the location with the name you specify. Bookmarks are more
	than placeholders; for example, you can use them to create and number
	cross-references.
