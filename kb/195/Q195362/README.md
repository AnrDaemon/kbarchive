## Q195362: DOC: Calendar Control Contains Proprietary Information

	Article: Q195362
	Product(s): Microsoft Visual Basic for Windows
	Version(s): WINDOWS:5.0
	Operating System(s): 
	Keyword(s): kbdocfix kbActiveX kbVBp kbVBp500 kbGrpDSVB
	Last Modified: 11-JAN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual Basic Professional Edition for Windows, version 5.0 
	- Microsoft Visual Basic Enterprise Edition for Windows, version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following appears near the top of the Code Window of the Calendar control,
	Calendar.ctl, that is part of the sample in the TOOLS\Unsupprt\Calendar
	directory of the Visual Basic 5.0 CD-ROM:
	
	  ' Information Contained Herein is Proprietary and Confidential
	
	This statement is wrong. You are free to use and distribute the code in this
	sample.
