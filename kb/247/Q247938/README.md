## Q247938: SMS 2.0 and SMS Service Pack 1 Are Not Supported on Windows 2000

	Article: Q247938
	Product(s): Microsoft Systems Management Server
	Version(s): WINDOWS:2000; winnt:2.0,2.0 SP1
	Operating System(s): 
	Keyword(s): kbOSWin2000 kbsms200 kbGrpDSW2KISV
	Last Modified: 18-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server versions 2.0, 2.0 SP1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Systems Management Server (SMS) 2.0 and SMS 2.0 Service Pack 1 (SP1) are not
	supported on Window 2000.
	
	MORE INFORMATION
	================
	
	Windows 2000 support for SMS will be released in SMS 2.0 Service Pack 2 (SP2).
