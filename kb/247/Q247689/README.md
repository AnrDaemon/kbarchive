## Q247689: XCLN: Readme in Exchange Server 5.5 SP3 Outlook Version Wrong

{% raw %}

	Article: Q247689
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5 SP3
	Operating System(s): 
	Keyword(s): exc55sp3
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 SP3 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The Readme file on the Exchange Server version 5.5 Service Pack 3 (SP3) CD
	states the following:
	
	  Before You Begin
	
	  Service Pack 3 for Microsoft Exchange Server version 5.5 is part of
	  Microsoft's commitment to provide important product refinements on a regular
	  basis. This service pack includes significant new enhancements, including
	  Microsoft OutlookTM 98, Microsoft's premier messaging and collaboration
	  client. In addition, Service Pack 3 includes many important new server
	  features, which are described in the following sections.
	
	The Outlook version that this section of the Readme file references is incorrect.
	The version is actually Outlook 2000. Outlook 98 is an older product.
