## Q97072: PC WRmt: Alphanumeric Characters Only in AT&amp;T Driver Addresses

	Article: Q97072
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Remote for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In Microsoft Mail Remote version 3.2 for Windows, if you use the AT&T
	driver, do not enter nonalphanumeric characters (for example, !, :, ~) into the
	address template for an AT&T user, a mailing list, or a shared folder.
	
	The AT&T Mail driver automatically adds nonalphanumeric characters as
	required. If you enter them into the address template, Mail creates invalid
	addresses and does not correctly deliver your messages.
