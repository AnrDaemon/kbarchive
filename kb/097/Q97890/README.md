## Q97890: PC WRmt: Cannot Download More Than 50 Messages at One Time

	Article: Q97890
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Remote for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	While downloading a large number of messages, you may receive a message in the
	Connection dialog box similar to the following:
	
	  Message transfer limit reached. Connect again to continue.
	
	CAUSE
	=====
	
	You cannot download more than 50 messages at one time with version 3.2 of
	Microsoft Mail Remote for Windows. This is due to a 50 messages per session
	limitation in EXTERNAL.EXE from version 3.0 or 3.2 of Microsoft Mail for PC
	Networks and not of Mail Remote for Windows.
	
	RESOLUTION
	==========
	
	You must reconnect until you retrieve all messages waiting to be downloaded.
