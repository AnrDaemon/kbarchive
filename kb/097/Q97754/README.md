## Q97754: RAS 1.1/VCOMMIOD.EXE Internal Modem Fix

	Article: Q97754
	Product(s): Microsoft LAN Manager
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	With RAS 1.1 running under Windows or Windows for Workgroups, Rasphone will hang
	when initially dialing on the Telepath and several other internal modems.
	
	To correct this problem, use the version of VCOMMIOD.EXE dated 3/4/93. It is
	available on CompuServe in the MSNETWORKS forum or by contacting Microsoft
	Product Support Services.
	
	Additional query words: wfw wfwg 2.20
