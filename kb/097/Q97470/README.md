## Q97470: PC WRmt: No Migration for Messages from MS-DOS Remote

{% raw %}

	Article: Q97470
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Remote for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After upgrading to version 3.2 of Microsoft Mail Remote for Windows from version
	3.0 of Microsoft Mail Remote for MS-DOS, there is no way to convert mail
	messages from Mail Remote for MS-DOS to Mail Remote for Windows.
	
	CAUSE
	=====
	
	This problem occurs due to possible security issues--a process or utility was
	not included.
	
	RESOLUTION
	==========
	
	One workaround is to forward the mail messages to yourself, using the Mail
	Remote for MS-DOS client, and then use the Mail Remote for Windows client to
	retrieve the messages.
