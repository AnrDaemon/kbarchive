## Q97621: Hyperdisk Version 4.6 or Higher Required With MS-DOS 6.0

	Article: Q97621
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 6.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	According to Hyperware technical support, Hyperdisk version 4.6 or higher is
	required when using Microsoft MS-DOS version 6.0.
	
	
	MORE INFORMATION
	================
	
	Hyperdisk is a disk-caching utility that loads in either the CONFIG.SYS or
	AUTOEXEC.BAT files.
	
	For upgrade information on Hyperdisk, contact Hyperware customer support.
	
	Hyperdisk is manufactured by Hyperware, a vendor independent of Microsoft; we
	make no warranty, implied or otherwise, regarding this product's performance or
	reliability.
