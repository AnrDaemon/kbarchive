## Q97751: PC WRmt: Seeing Which Driver Is Installed

{% raw %}

	Article: Q97751
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Remote for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	There is an easy way to see which mail driver is loaded without having to quit
	version 3.2 of Microsoft Mail Remote for Windows. Simply look on the Help menu
	for the options titled "AT&T Help" or "Remote Help."
	
	Usually, you would exit Mail Remote for Windows and run the utility Mail System
	Selector to change drivers for Mail Remote for Windows.
	
	Running this Selector application will also display the current driver you are
	using.
	
	Without running this Selector application, with Mail Remote for Windows running
	you can simply click on the Help menu and see what driver is loaded by the
	specific reference to its Help file.
