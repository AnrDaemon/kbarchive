## Q97493: Minimum LAN Manager MS-DOS Files and RAM Consumption

	Article: Q97493
	Product(s): Microsoft LAN Manager
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	Here are the minimum files needed for a LAN Manager MS-DOS client and their
	APPROXIMATE RAM consumption figures:
	
	  PROTMAN 6.1K
	  MAC driver (size varies) 11-14K
	  Minses 1.8K
	  Netwksta 92.5K (Basic redirector is 37.8K)
	  NetBEUI 36.2K
	
	  EMM386 8.2K
	  HIMEM 1.2K
	
	
	Additional query words: 2.00 2.10 2.10a 2.20
