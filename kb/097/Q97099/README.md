## Q97099: Windows NT Does Not Support Windows Printing System 1.0

	Article: Q97099
	Product(s): Microsoft Windows NT
	Version(s): 1.0,3.1,3.5,3.51,4.0
	Operating System(s): 
	Keyword(s): kbhw kbHardware
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server versions 3.1, 3.5, 3.51, 4.0 
	- Microsoft Windows NT Workstation versions 3.1, 3.5, 3.51, 4.0 
	- Microsoft Windows NT Advanced Server, version 3.1 
	- Microsoft Windows Printing System, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Windows NT does not support the Windows Printing System. This is true for the
	final product as well as the March release.
