## Q97803: MSBackup Stops on Disk Two And Continues to Count User Time

	Article: Q97803
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you back up your files, Microsoft Backup may not recognize the second disk
	and continue counting the User time.
	
	CAUSE
	=====
	
	This problem occurs when your computer passed the compatibility test but should
	not have.
	
	WORKAROUND
	==========
	
	To work around this problem, back up your files to the MS-DOS path instead of a
	specific disk size in drive A or B.
	
	Additional query words: 6.22 6.00 6.20 backup back up msbackup mwbackup
