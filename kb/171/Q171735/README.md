## Q171735: Err Msg: Detection Manager Error: Failed to Hook Ring 0 Code

{% raw %}

	Article: Q171735
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): kberrmsg kbtool win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to install a new device using the Add New Hardware tool in
	Control Panel, you may receive the following error message:
	
	  Detection Manager error: Failed to hook ring 0 code.
	
	CAUSE
	=====
	
	This error message can occur if a program designed to change your video
	resolution is currently running.
	
	
	RESOLUTION
	==========
	
	Quit any programs used to configure your display adapter before attempting to
	install a new device using the Add New Hardware tool in Control Panel.
