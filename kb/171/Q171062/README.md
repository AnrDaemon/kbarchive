## Q171062: XCON: Exchange RAS Does Not Support Windows NT Multilink

	Article: Q171062
	Product(s): Microsoft Exchange
	Version(s): winnt:4.0,5.0,5.5
	Operating System(s): 
	Keyword(s): exc4 exc5 exc55
	Last Modified: 19-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 5.5, 4.0, 5.0 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	Microsoft Windows NT Server version 4.0 provides a multilink feature that allows
	multiple modem links to operate as one link. This maximizes bandwidth using
	multiple phone lines on a single connection.
	
	The Microsoft Exchange MTA Dynamic RAS Connector does not support the multilink
	feature of Windows NT version 4.0.
