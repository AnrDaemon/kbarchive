## Q171690: WD97: Text and Images in CorelDRAW Graphic Appear Overlaid

{% raw %}

	Article: Q171690
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): winword word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you insert a CorelDRAW graphic (*.cdr) into a Microsoft Word 97 document,
	text and image objects contained in the CorelDRAW graphic may overlap.
	
	WORKAROUND
	==========
	
	To work around this problem, copy and paste the CorelDRAW graphic to Microsoft
	Word using the following steps:
	
	1. From CorelDRAW, open the CorelDRAW .cdr graphic file.
	
	2. On the Edit menu, click Select All.
	
	3. On the Edit menu, click Copy.
	
	4. Switch to Word and open the Word document.
	
	5. On the Edit menu, click Paste.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
	
	MORE INFORMATION
	================
	
	This problem may also occur when you preview the CorelDRAW graphic. To preview a
	graphic:
	
	1. On the Insert menu, point to Picture, and then click From File.
	
	2. Click to select the desired graphic file.
	
	3. On the Insert Picture toolbar, click the Preview button.
