## Q171864: SNA Manager Does Not Stay Maximized

	Article: Q171864
	Product(s): Microsoft SNA Server
	Version(s): WINDOWS:3.0,3.0 SP1
	Operating System(s): 
	Keyword(s): kbnetwork
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft SNA Server, versions 3.0, 3.0 SP1, on platform(s):
	   - the operating system: Microsoft Windows NT 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	After you maximize SNA Manager and then close it, when you reopen the window, it
	expands to the maximum size and then shrinks to the previous window size.
	
	CAUSE
	=====
	
	SNA Manager does not save the modification to the window size.
	
	RESOLUTION
	==========
	
	An update to SNA Server 3.0 (post SP1) is available to support this feature.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in SNA Server 3.0 and 3.0 Service
	Pack 1 (SP1). This problem was corrected in the latest SNA Server version 3.0
	U.S. Service Pack. For information on obtaining this Service Pack, query on the
	following word in the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
