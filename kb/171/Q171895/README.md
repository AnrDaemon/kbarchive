## Q171895: Mac Encarta 97: Orbit Interactivity Stops Right After Starting

	Article: Q171895
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1997 edition
	Operating System(s): 
	Keyword(s): kbdisplay kbusage
	Last Modified: 24-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 97 Encyclopedia for Macintosh 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In the Orbits Interactivity, the Orbit stops right after it starts.
	
	This occurs if clear the "Leave dot trail" and or "Show orbit type" options.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Encarta 97
	Encyclopedia. Microsoft is researching this problem and will post new
	information here as it becomes available.
	
	MORE INFORMATION
	================
	
	This problem does not occur in Microsoft Encarta 96 Encyclopedia.
	
	Additional query words: macmm multi multimedia media orbits ia ias rotate rotates
