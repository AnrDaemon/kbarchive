## Q171151: MS Press Support Policy for Windows NT 4.0 Evaluation Edition

	Article: Q171151
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbother
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following policy applies to the Microsoft Windows NT 4.0 Server and
	Workstation evaluation editions included with the Microsoft Press books listed
	in this article. Please note that not all of these books include the Microsoft
	Windows NT 4.0 Workstation Evaluation Edition.
	
	MORE INFORMATION
	================
	
	As a general rule, Microsoft and Microsoft Press do not support evaluation
	edition software. For the Windows NT 4.0 evaluation editions, Microsoft Press
	offers limited, commercially reasonable support in the following cases:
	
	- Initial setup of the evaluation software.
	
	- Possible discrepancies or errors in the printed material.
	
	If you have a question regarding one of these two areas, please send e-mail to
	TKINPUT@MICROSOFT.COM, or a fax to (425) 936-7329, or a letter to:
	
	  Microsoft Press Technical Support
	  One Microsoft Way
	  Redmond, WA 98052
	
	For more support information on Microsoft Windows NT, please check the Microsoft
	Technical Support Web site at:
	
	  http://www.microsoft.com/support/
