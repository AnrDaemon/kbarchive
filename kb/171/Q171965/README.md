## Q171965: Mac Bookshelf 98: Animations and Movies Slightly Out Of Frame

	Article: Q171965
	Product(s): Microsoft Home Multimedia Titles
	Version(s): MACINTOSH:
	Operating System(s): 
	Keyword(s): kbdisplay kbenv kbmmkbfaq
	Last Modified: 05-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf 98 for Macintosh 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play an animation or movie in Microsoft Bookshelf 98, the picture may
	be displayed slightly out of the frame.
	
	This occurs if you use an extension (such as Aaron, Kaleidoscope, or Mac OS 8
	Appearance Manager) that modifies the appearance of the operating system.
	
	RESOLUTION
	==========
	
	To resolve this problem, use Extensions Manager to disable the extension which
	modifies the appearance of the operating systemIf you are using Appearance
	Manager, see the Appearance Manager section after these steps.
	
	Extensions Manager
	------------------
	
	1. On the Apple menu, click Control Panels.
	
	2. Double-click Extensions Manager.
	
	3. On the list of extensions, locate the extension that modifies the appearance
	  of the operating system. Click the check box to the left of this extension to
	  clear it.
	
	4. Close Extensions Manager.
	
	5. On the Special menu, click Restart.
	
	Appearance Manager
	------------------
	
	You can only change the Appearance Manager properties in the Appearance Control
	Panel. To do so, follow these steps:
	
	1. On the Apple menu, click Control Panels.
	
	2. Double-click Appearance, and then click Options.
	
	3. Click the System-wide Platinum Appearance check box to clear it.
