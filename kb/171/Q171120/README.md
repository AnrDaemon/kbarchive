## Q171120: Useful Links to HCL Storage Device Configuration Information

	Article: Q171120
	Product(s): Microsoft Windows NT
	Version(s): winnt:3.5,3.51,4.0
	Operating System(s): 
	Keyword(s): kbhw kbsetup kbHardware
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation versions 3.5, 3.51, 4.0 
	- Microsoft Windows NT Server versions 3.5, 3.51, 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following manufacturers contain products that are found on the Hardware
	Compatibility List (HCL) for Windows NT 4.0, under Storage Devices and Hard
	Drives. You can use the following links to assist in configuring these devices
	to work properly in Windows NT.
	
	This information is helpful when you do not have jumper settings or other
	configuration information about your hard drive(s).
	
	Storage Configuration Information Locations
	-------------------------------------------
	
	Fujitsu:
	
	  http://www.fcpa.com/supmenuhd.html
	
	Seagate and Conner:
	
	  http://www.seagate.com/support/disc/specs/specfind.shtml
	
	Hewlett Packard:
	
	  http://www.hp.com/cposupport/nonjsnav/informatio10165.html
	
	I.B.M:
	
	  http://www.storage.ibm.com/storage/techsup/hddtech/hddtech.htm
	
	Maxtor:
	
	  http://www.maxtor.com/technology/index.html
	
	Quantum:
	
	  http://www.quantum.com/support/support.htm
	
	Western Digital:
	
	  http://www.wdc.com/acrobat/welcome.html
	
	Additional query words: jumper settings termination scsi ID
