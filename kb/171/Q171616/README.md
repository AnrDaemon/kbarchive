## Q171616: Update.exe Should Support DBCS Characters

{% raw %}

	Article: Q171616
	Product(s): Microsoft SNA Server
	Version(s): WINDOWS:3.0 SP1 and above
	Operating System(s): 
	Keyword(s): kbbuglist
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft SNA Server, versions 3.0 SP1, 3.0 SP2, 3.0 SP3, 3.0 SP4 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	The service pack Update.exe program does not support double-byte character set
	(DBCS) characters which are included in directory names and .ini files. This
	deficiency affects the SNA Server 3.0 Japanese SP1 update program.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in SNA Server, versions 3.0 and 3.0
	SP1. This problem was corrected in the latest SNA Server version 3.0 U.S.
	Service Pack. For information on obtaining this Service Pack, query on the
	following word in the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
	
	
	Additional query words: update DBCS ini path
