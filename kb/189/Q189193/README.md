## Q189193: XADM: Problems Occur When Incorrect Property Is Set on Message

	Article: Q189193
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.5,8.03,8.5
	Operating System(s): 
	Keyword(s): exc55sp2fix
	Last Modified: 09-JAN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	- Microsoft Outlook 97, version 8.03, used with:
	   - the operating system: Microsoft Windows 98 
	   - the operating system: Microsoft Windows 95 
	- Microsoft Outlook 98, version 8.5, used with:
	   - the operating system: Microsoft Windows 98 
	   - the operating system: Microsoft Windows 95 
	- Microsoft Outlook 97, version 8.03, used with:
	   - the operating system: Microsoft Windows NT 
	- Microsoft Outlook 98, version 8.5, used with:
	   - the operating system: Microsoft Windows NT 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You may experience any of the following symptoms:
	
	- You are unable to send a message through a Microsoft Exchange Server
	  computer.
	
	- You are unable to move a message or folder in Microsoft Outlook.
	
	- When you attempt to move a mailbox from one server to another, Event 1151
	  appears in the event log and the mailbox is not moved successfully.
	
	CAUSE
	=====
	
	These problems can occur when an incorrect property is set on a message or
	folder.
	
	
	RESOLUTION
	==========
	
	To resolve this problem, obtain the latest service pack for Exchange Server
	version 5.5. For more information, please see the following article in the
	Microsoft Knowledge Base:
	
	  Q191014 XGEN: How to Obtain the Latest Exchange Server 5.5 Service Pack
	
	
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server
	version 5.5.
