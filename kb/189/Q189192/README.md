## Q189192: Accessibility Options Do Not Work Properly on Compaq Computers

{% raw %}

	Article: Q189192
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): kb3rdparty kbenable win95 appscomp kbAppCompatibility
	Last Modified: 28-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run Windows 95 on a Compaq computer, you may experience one or more of
	the following symptoms:
	
	- Your Computer may stop responding (hang) when it attempts to resume from
	  Standby mode.
	
	- Your accessibility option may not function correctly.
	
	CAUSE
	=====
	
	This behavior can occur if you are running the Compaq Easy Access software and
	you have enabled Windows 95 accessibility options. Accessibility options are not
	compatible with the Compaq Easy Access software.
	
	RESOLUTION
	==========
	
	To resolve this issue, contact Compaq or visit the following Compaq Web site:
	
	  http://www.compaq.com
	
	
	MORE INFORMATION
	================
	
	This behavior is known to occur on the following Compaq computers:
	
	Presario 1000 Series
	--------------------
	
	Models 1010, 1020, 1030, 1060, 1060ES, 1065, 1070, 1080
	
	Presario 3000 Series
	--------------------
	
	Models 3020, 3060
	
	Presario 4000 Series
	--------------------
	
	Models 4104ES, 4406ES, 4104ES, 4170ES, 4460ES
	Models 4102, 4112, 4120, 4122, 4126, 4164, 4170, 4172, 4180
	Models 4402, 4410
	Models 4704, 4706, 4710, 4712, 4714, 4716, 4760, 4764, 4770, 4772
	Models 4774, 4776, 4780, 4782, 4784, 4786
	
	Presario 6000 Series
	--------------------
	
	Models 6704, 6708
	
	Presario 8000 Series
	--------------------
	
	Models 8702, 8706, 8710, 8712, 8770, 8772
	
	The Compaq Easy Access software controls such features as speakerphone, volume,
	audio CD player, SmartQ online help, and Standby mode.
