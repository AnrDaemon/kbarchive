## Q189203: Arcadia.exe Missing From BackOffice Resource Kit 2nd Ed.

	Article: Q189203
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocerr
	Last Modified: 08-JAN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft BackOffice Resource Kit, Second Edition ISBN 1-57231-632-2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Microsoft BackOffice Resource Kit, Second Edition should include a tool
	called Arcadia.exe in the \SQL\Arcadia folder on the CD-ROM. However, the
	\SQL\Arcadia folder is empty.
	
	RESOLUTION
	==========
	
	Download the Arcadia tool from the Microsoft BackOffice Resource Kit FTP site.
	The FTP site is available at:
	
	  ftp://ftp.microsoft.com/bussys/backoffice/reskit/arcadia/
	
	STATUS
	======
	
	Microsoft Press has confirmed this to be a problem in Microsoft BackOffice
	Resource Kit, Second Edition.
