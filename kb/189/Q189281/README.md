## Q189281: HOWTO: View Documents w/ FoxPro OLE Controls w/o MS Office

{% raw %}

	Article: Q189281
	Product(s): Microsoft FoxPro
	Version(s): WINDOWS:5.0,5.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-AUG-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual FoxPro for Windows, versions 5.0, 5.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Some FoxPro applications may need to display information taken from Microsoft
	Office products when the end user does not own the Office Product. Microsoft
	provides a number of downloadable viewer products for this purpose.
	
	MORE INFORMATION
	================
	
	Please use the following list of articles in the Microsoft Knowledge Base for
	information on how to obtain these free programs:
	
	
	  Q163410 ODE97: Creating Run-Time Apps for MS Excel, Word, or PowerPoint
	
	
	  Q165908 WD97: Versions of Word Viewer Available for Download
	
	  Q166636 WD97: Frequently Asked Questions About Word Viewer 97
	
	
	  Q169549 OFF97: Information About The Microsoft Office ValuPack
	
	
	  Q170329 PPT97: Contents of PowerPoint 97 Viewer Readme.txt File
	
	Additional file names that may be of interest are available from the Office
	Resource CD-ROM. There are other viewers available that may not listed in this
	article.
