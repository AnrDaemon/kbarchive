## Q189111: INFO: What Additional Components Are Installed with VB 6.0?

	Article: Q189111
	Product(s): Microsoft Visual Basic for Windows
	Version(s): WINDOWS:6.0
	Operating System(s): 
	Keyword(s): kbsetup kbVBp600 kbGrpDSVB
	Last Modified: 11-JAN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual Basic Professional Edition for Windows, version 6.0 
	- Microsoft Visual Basic Enterprise Edition for Windows, version 6.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When Visual Basic 6.0 is installed upon a development machine, several other
	components are installed as well. This articles lists the additional components
	installed along with Visual Basic 6.0.
	
	MORE INFORMATION
	================
	
	Visual Basic 6.0 requires a few additional system components in order to work on
	a development machine. The components installed include:
	
	  Microsoft Internet Explorer 4.1 SP1
	  Updated Java Virtual Machine
	  DCOM98
	  Visual Basic 6.0 Core Files
	  Additional Visual Basic 6.0 Features Selected (optional)
	  Microsoft Developer's Network (optional)
	  Server Components (optional)
	
	OPTIONAL COMPONENTS
	-------------------
	
	Additional Visual Basic 6.0 Features
	------------------------------------
	
	These are extra components selected to be installed along side the Core Files.
	You can select what you would like installed with a "Custom" setup. A Typical
	setup installs all the optional features except Graphics, ISAM drivers, and some
	ODBC drivers.
	
	Microsoft Developer's Network (MSDN)
	------------------------------------
	
	If you wish to access Online Help and documentation, you must install MSDN.
	Visual Basic 6.0's documentation is now a part of the Microsoft Developer's
	Network.
	
	Server Components
	-----------------
	
	There are numerous tools that ship with the Professional and Enterprise Edition
	of Visual Basic 6.0. Server Setup lets you choose additional tools to install.
