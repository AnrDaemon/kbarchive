## Q173898: XWEB: Message Sent w/ Incorrect Priority When You Press ENTER

{% raw %}

	Article: Q173898
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	When you use Outlook Web Access to compose a new mail message and you send it by
	pressing the ENTER key, the message is sent with normal priority even if you
	selected high or low priority. This happens if you press enter in the To, CC, or
	Subject fields.
	
	WORKAROUND
	==========
	
	To work around this problem:
	
	- Send the message using the Send button.
