## Q173433: Is MSN Supported on Macintosh Computers?

{% raw %}

	Article: Q173433
	Product(s): The Microsoft Network
	Version(s): 1.2,1.3,2.0,2.5
	Operating System(s): 
	Keyword(s): kbenv kbmsn
	Last Modified: 09-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.2, 1.3, 2.0, 2.5 
	-------------------------------------------------------------------------------
	
	Q. Can I connect to MSN, The Microsoft Network, on a Macintosh computer
	  using a PC Card or Windows emulation software?
	
	A. These configurations are not supported by Microsoft. MSN has not been
	  tested on a Macintosh using either a PC Card or emulation software such
	  as Virtual PC by Connectix.
