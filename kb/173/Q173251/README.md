## Q173251: Running Outlook 97  CD Will not Load with IE 4.0 Beta

	Article: Q173251
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 20-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Running Microsoft Outlook 97 ISBN 1-57231-608-X 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you attempt to insert the CD-ROM from the Microsoft Press book "Running
	Microsoft Outlook 97" into a system running Microsoft Internet Explorer 4.0 beta
	1 or beta 2, you may receive the following error message:
	
	  Unable to run your Web browser and find its main window.
	
	When you click OK, the desktop is restored.
	
	CAUSE
	=====
	
	"Running Microsoft Outlook 97" uses a third-party product called WebCD 1.6,
	manufactured by Marketscape, Inc. WebCD 1.6 is not compatible with the beta
	version of Microsoft Internet Explorer 4.0.
	
	
	RESOLUTION
	==========
	
	MarketScape Inc., the creator of WebCD 1.6, has created a patch for the tool
	that will allow it to run under the Microsoft Internet Explorer 4.0 beta.
	
	To download this tool, please point your browser to the following Web site:
	
	  http://www.marketscape.com/support/
	
	Please note that this site is not under the control of Microsoft and may change
	without notice. For Microsoft's official statement regarding other servers,
	please see the following World Wide Web site:
	
	  http://www.microsoft.com/misc/nonms.htm
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
	
	MORE INFORMATION
	================
	
	The third-party products discussed here are manufactured by vendors independent
