## Q173489: WD97: Cannot Open Hyperlinked Document Using Shortcut Menu

	Article: Q173489
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbnetwork winword word97 kbwdinternet
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Microsoft Word 97, when you right-click a hyperlink to a document, point to
	Hyperlink on the shortcut menu, and then click Open or Open in New Window, the
	document may not open.
	
	This problem only occurs when you right-click a hyperlink to a document saved on
	your computer (or network location). This problem does not occur when you
	right-click a hyperlink to the World Wide Web.
	
	WORKAROUND
	==========
	
	To open the file, click the hyperlink instead of right-clicking the hyperlink
	and using the shortcut menu.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows.
	For additional information about hyperlinks, please see the following article in
	the Microsoft Knowledge Base:
	
	  Q159940 WD97: General Information About Hyperlinks
