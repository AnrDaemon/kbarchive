## Q173832: Microsoft Team Manager 97 Step by Step Comments and Corrections

	Article: Q173832
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-JAN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Team Manager 97 Step by Step 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains information on known errors, corrections, and comments
	relating to the Microsoft Press book "Microsoft Team Manager 97 Step by Step."
	
	The following topics are covered:
	
	- Book Assumes Reader Is Using Microsoft Exchange
	
	- Page xvii: Incorrect Area Code
	
	- Pages 17-58: Miscellaneous Corrections
	
	- Page 252: Locate and open TMFILE11 not TMFILE12
	
	MORE INFORMATION
	================
	
	Book Assumes Reader Is Using Microsoft Exchange
	-----------------------------------------------
	
	Microsoft Team Manager 97 can be used with any of the following e-mail programs:
	
	- Microsoft Exchange Client or Microsoft Outlook
	
	- Microsoft Mail for Windows NT version 3.51 or later
	
	- Lotus Notes Release 4.1 or later
	
	- Lotus cc:Mail Release 7.0
	
	- Lotus cc:Mail Release 2.2 or 6.0
	
	- Lotus Notes Release 3.3 family
	
	Microsoft Team Manager 97 also allows users to exchange messages using a shared
	folder system.
	
	However, "Microsoft Team Manager 97 Step by Step" assumes that the reader is
	using Microsoft Exchange. The installation instructions and some of the lessons
	will not work correctly in other environments.
	
	Microsoft Press is currently investigating this issue and will post further
	information when it becomes available.
	
	
	Page xvii: Incorrect Area Code
	------------------------------
	
	Page xvii, last paragraph under Corrections, Comments, and Help:
	Change: "(206) 635-7065"
	To: "(425) 635-7065"
	
	
	Pages 17-58: Miscellaneous Corrections
	--------------------------------------
	
	Page 17, Step 6:
	Change Save Team File button icon to proper icon.
	
	Page 23, Section "Finish the Lesson," Step 3:
	Change: "...and then click the Team Manager SBS Practice\Lesson 1 folder."
	To: "...and then click the Team Manager SBS Practice folder."
	
	Page 49, Step 1:
	Change: "In Team Manager..."
	To: "In Team Manager for team members..."
	
	Page 58, section "Integrate Outlook or Schedule+ Tasks", Step 1:
	Change: "In Team Manager..."
	To: "In Team Manager for team members..."
	
	
	Page 252: Locate and open TMFILE11 not TMFILE12
	-----------------------------------------------
	
	Page 252: Step 2 lists wrong file name.
	
	Change:
	"Locate and open TMFILE12."
	To:
	"Locate and open TMFILE11."
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
