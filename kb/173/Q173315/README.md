## Q173315: PPT7: White Text Prints Black w/ HP Color LaserJet Driver

	SYMPTOMS
	========
	
	When you print an Organization Chart or Microsoft Graph object using the
	Hewlett-Packard (HP) Color LaserJet (MS) printer driver, white text prints
	black.
	
	CAUSE
	=====
	
	This problem occurs if the Organization Chart or Microsoft graph object has been
	ungrouped and converted to a PowerPoint drawing object.
	
	WORKAROUND
	==========
	
	To work around this problem, set the printer driver to download True Type fonts
	as bitmap soft fonts, following these steps:
	
	1. On the File menu, click Print.
	
	2. In the Printer Name box, select the HP LaserJet (MS) printer driver.
	
	3. Click Properties.
	
	4. Click the Fonts tab.
	
	5. Click the "Download True Type fonts as bitmap soft fonts," and then click OK.
	
	6. Click OK.
