## Q173354: XADM: Disk Space Required For Directory Upgrade From Exchange 4.

	Article: Q173354
	Product(s): Microsoft Exchange
	Version(s): winnt:5.0,5.5
	Operating System(s): 
	Keyword(s): exc5 exc55
	Last Modified: 16-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 5.0, 5.5 
	-------------------------------------------------------------------------------
	
	In order to successfully upgrade from Microsoft Exchange Server version 4.0
	to Exchange Server 5.0, the server must have enough free disk space (equal
	to or greater than the size of the Dir.edb file). The free disk space must
	be on the drive that contains the Dir.edb file.
	
	Additional query words: dir.edb upgrade
