## Q173948: XWEB: Text in Create New Folder Box Not Left-Justified

{% raw %}

	Article: Q173948
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.5
	Operating System(s): 
	Keyword(s): kbbuglist
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook Web Access, version 5.5 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	When you use Outlook Web Access (OWA) to access a Microsoft Exchange Server
	mailbox and create a new folder, you may notice misalignment of the following
	text:
	
	  Enter the name of the folder you want to create.
	
	The text appears with an extra space on the left side of the first line, rather
	than being left-justified. This occurs regardless of the screen resolution or
	browser used.
	
	STATUS
	======
	
	Microsoft has confirmed this to be problem in Microsoft Exchange Server version
	5.5. We are researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
