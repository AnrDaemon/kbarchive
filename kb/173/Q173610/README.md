## Q173610: Fonts Appear Too Large in MSN Program Viewer

	Article: Q173610
	Product(s): The Microsoft Network
	Version(s): WINDOWS:2.0,2.5
	Operating System(s): 
	Keyword(s): kbenv kbui kbmsn
	Last Modified: 13-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 2.0, 2.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are connected to MSN, The Microsoft Network, on a computer using an S3
	Trio64V+ PCI video card, the fonts in Program Viewer may appear too large and
	some content may not be displayed properly.
	
	RESOLUTION
	==========
	
	To work around this issue, scale the fonts down to a reduced size. To do so, use
	the following steps:
	
	1. Click Start, point to Settings, and then click Control Panel.
	
	2. Double-click Display.
	
	3. Click the Settings tab.
	
	4. In the Font Size box, click Custom.
	
	5. Drag the Sample ruler to the left until the value in the "Scale fonts to be
	  <n>% of normal size" box is 70%.
	
	6. Click OK until you return to Windows.
	
	7. When you are prompted to restart the computer, click Yes.
