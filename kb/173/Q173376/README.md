## Q173376: &quot;Stop 0x00000050&quot; Starting Windows NT 4.0

	Article: Q173376
	Product(s): Microsoft Windows NT
	Version(s): WINDOWS NT:4.0
	Operating System(s): 
	Keyword(s): kberrmsg
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start Windows NT 4.0, you may receive the following error message:
	
	  Stop 0x00000050 PAGE_FAULT_IN_NONPAGED_AREA
	
	CAUSE
	=====
	
	This error message can occur if the Matrox Millennium driver version 2.3 has
	been installed.
	
	RESOLUTION
	==========
	
	Contact Matrox for information concerning an updated version of this driver.
	
	Additional query words: 0x50
