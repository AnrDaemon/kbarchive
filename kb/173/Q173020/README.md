## Q173020: MASTERING: .Frm Files Missing From Democode Folder

{% raw %}

	Article: Q173020
	Product(s): Microsoft Mastering Series
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Mastering Microsoft Visual Basic 5 Fundamentals 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The .frm files referred to in Mastering Visual Basic 5 Fundamentals are missing
	from the Democode folder on the CD-ROM.
	
	MORE INFORMATION
	================
	
	To access the demonstration projects, you must download an update file from:
	
	  http://msdn.microsoft.com/training/technical/mvbfdemo.zip
	
	This file can be unzipped into the C:\Mvbf folder and the previously referenced
	Democode folder can be ignored.
	
	Additional query words: demo exist form
