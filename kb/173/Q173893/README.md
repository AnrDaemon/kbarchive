## Q173893: XCLN: Pronto 97 IMAP Client can't use Mailbox Name Over 23 Chars

	Article: Q173893
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5
	Operating System(s): 
	Keyword(s): kb3rdparty
	Last Modified: 16-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	Users of the Pronto 97 IMAP/POP mail client will not be able to download mail
	from the Microsoft Exchange Server if their account name is longer than 23
	characters. This is specified in the Online/Mailbox settings/Host Login field of
	the Pronto 97 client.
