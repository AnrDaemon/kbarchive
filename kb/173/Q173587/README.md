## Q173587: How to Connect to the Internet Using MSN

	Article: Q173587
	Product(s): The Microsoft Network
	Version(s): 1.0,5.4
	Operating System(s): 
	Keyword(s): kbmsn
	Last Modified: 02-JUL-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN version 5.4 
	- MSN version 5.6 
	- MSN Explorer, version 1.0 
	- MSN version 7.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to connect to the Internet using MSN, The Microsoft
	Network, as your Internet service provider (ISP).
	
	MORE INFORMATION
	================
	
	To connect to the Internet using MSN, use one of the following methods,
	depending on the version of MSN installed on your computer.
	
	Versions of MSN software prior to 5.4 are no longer supported.
	
	Note: MSN Explorer, version 1.0, is better known as MSN Explorer, version 6.1.
	MSN version 7.0 includes MSN Explorer version 7.0. This information is included
	in this article to avoid confusion over the naming of these products.
	
	Connect to MSN using the MSN Internet Access Icon (MSN 5.4 or 5.6)
	------------------------------------------------------------------
	
	1. Double-click the MSN Internet Access icon on the desktop.
	
	2. In the MSN Sign-In screen, type your MSN member identification in the Member
	  ID box, type your password in the Password box, and then click Connect.
	
	Connect to MSN using MSN Explorer (Versions 6.1 and 7.0)
	--------------------------------------------------------
	
	1. Click Start, point to Programs, and then click MSN Explorer.
	
	2. Click the appropriate user icon and type in the password for the account in
	  the Enter your password field provided below the user name.
	
	3. Click the Sign-in button and MSN Explorer will connect you to the Internet.
	
	Additional query words: msnet msnetwork microsoft-net m.s.n.
