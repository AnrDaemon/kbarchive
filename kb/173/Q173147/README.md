## Q173147: WD97: Removing Headings When Running Resume Wizard Causes Errors

	Article: Q173147
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta word8 word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run the Resume Wizard, you may receive the following error message when
	you try to remove a heading that has been moved down to the bottom of the list
	of headings:
	
	  Nothing selected from list to remove.
	
	When you click OK and then click Finish, you will get the following error:
	
	  Invalid use of Null.
	
	When you click OK, the Resume Wizard closes, and no document is created.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.<A0>This problem was corrected in Microsoft
	Word 2000.

	Knowledge Base:

	  Q157580 WD97: Some Wizards Don't Show Finish On Progress Indicator
	
	  Q156064 WD97: Saved File Name Does Not Appear on Document Title Bar
	
	  Q139868 WD: How to Install Missing Templates and Wizards
	
	For more information about using the resume wizard, click "Contents and Index" on
	the Help menu, click the Index tab in Microsoft Word Help, type the following
	text
	
	  resume wizard
	
	and then double-click the selected text to go to the "Create a resume" topic. If
	you are unable to find the information you need, ask the Office Assistant.
