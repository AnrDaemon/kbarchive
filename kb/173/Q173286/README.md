## Q173286: WD97: Clipart Images Display Incorrect Format Picture Options

	Article: Q173286
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbui word97 kbwdinternet
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Only the Position tab is present in the Format Picture dialog box after you
	insert a ClipArt Image into a Web page and then click Format Picture.
	
	CAUSE
	=====
	
	
	RESOLUTION
	==========
	
	Save the document as HTML, close, and then re-open the document. The ClipArt
	image will now have been converted to a GIF image and will display more option
	tabs in the Format Picture dialog box.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article
