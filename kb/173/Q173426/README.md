## Q173426: Err Msg: MSNVIEWR Caused an Invalid Page Fault in Module...

	Article: Q173426
	Product(s): The Microsoft Network
	Version(s): WINDOWS:2.0,2.5
	Operating System(s): 
	Keyword(s): kberrmsg kbmsn
	Last Modified: 12-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 2.0, 2.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you connect to MSN, The Microsoft Network, you may receive the following
	error message:
	
	  MSNVIEWR caused an invalid page fault in module _MSRWS32.DLL.
	
	CAUSE
	=====
	
	This problem can occur if you are connecting to MSN using the Microsoft Remote
	Winsock Proxy client.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in The Microsoft Network versions
	2.0 and 2.5.
