## Q173450: XWEB: OWA Returns Error When Creating First Meeting Request

	Article: Q173450
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5
	Operating System(s): 
	Keyword(s): kberrmsg
	Last Modified: 07-JAN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	
	When you access a Microsoft Exchange Server using Outlook Web Access and create a
	new Meeting Request for the first time, you may receive the following error when
	you select the Meeting Planner view tab:
	
	  Information not available for this recipient.
	
	MORE INFORMATION
	================
	
	Outlook Web Access users do not generate Free/Busy information until after
	adding an appointment to their Calendar. This behavior is by product design.
