## Q173581: WD97: Inserted Video Missing When HTML Document Is Reopened

	Article: Q173581
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbgraphic winword word97 kbwdinternet
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you insert a video into a Hypertext Markup Language (HTML) document, the
	video may be missing the next time you open the document.
	
	CAUSE
	=====
	
	This problem occurs if a picture that has either a right or a left wrapping
	attribute is selected when you insert the video.
	
	WORKAROUND
	==========
	
	To work around this problem, cancel selection of the picture by placing the
	
	insertion point in another part of the document before you insert the video.
	
	If you want to replace a picture with the video, delete the picture before you
	insert the video.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products
	
	listed at the beginning of this article.
