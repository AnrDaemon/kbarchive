## Q173200: MTG: Novell Discontinues Support for All MHS Products

	Article: Q173200
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, version 3.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Novell announced April 29, 1997 that it will discontinue support for MHS
	products as of July 31, 1997. The discontinued MHS product line includes the
	following:
	
	  NetWare Global MHS
	  NetWare MHS v1.5 (NetWare, Multi-Network, and Personal Edition)
	  NetWare Remote MHS
	  SNADS for NetWare Global MHS
	  SMTP Protocol Module for NetWare Global MHS
	  X.400 for NetWare Global MHS
	  NetWare Global MHS to PROFS Module
	
	These MHS products are not supported on intraNetWare. In Addition, technical
	support for these MHS products on current versions of NetWare (version 4.11 is
	not supported) will continue until July 31, 1997. After this date, all remaining
	technical support for MHS products will be discontinued.
	
	The NetWare Global MHS and NetWare MHS v1.5 (NetWare, Multi-Network, and Personal
	Edition) are required for MS Mail to MHS gateway to function.
