## Q173380: XWEB: Pressing Enter when Focus is on Header Field will Send Msg

	Article: Q173380
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5
	Operating System(s): 
	Keyword(s): kbBug kbISS
	Last Modified: 27-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	When you access a Microsoft Exchange Server using the Outlook Web Access
	component and press the Enter key when the To:, CC:, BCC:, or Subject: line has
	the focus, the message will be sent.
	
	NOTE: This behavior is exhibited only in the Windows NT and Windows 95 clients
	and does not occur on the Windows 3.x Client.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Outlook Web
	Access, version 5.5. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
