## Q173637: XWEB: Can't View Free/Busy Info on a DL

	Article: Q173637
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 19-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you are using Outlook Web Access and you use the Calendar to send a meeting
	request to a Distribution List (DL), you cannot view Free/Busy time for that DL.
	Instead you will see "Information not available for this recipient."
	
	CAUSE
	=====
	
	This behavior is by product design.
