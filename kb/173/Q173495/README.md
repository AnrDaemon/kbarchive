## Q173495: XADM: IMS Advanced Tab Removed in Administrator Program

	Article: Q173495
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-JAN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	You might notice that the Advanced tab was removed from the Microsoft Exchange
	Internet Mail Service (IMS) property page in the Microsoft Exchange
	Administrator program.
	
	MORE INFORMATION
	================
	
	Research and feedback showed that the features and functions provided on the
	Advanced tab were not being used. Therefore, it was removed.
