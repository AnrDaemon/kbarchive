## Q173454: XADM: System Messages Are Not Modifiable

	Article: Q173454
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-MAR-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	
	It is not possible to modify the system messages that are sent to users in
	Microsoft Exchange Server, with an exception of the Certificate Notification in
	Key Management Server. These messages are hard coded.
