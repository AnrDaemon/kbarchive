## Q173382: XWEB: Outlook Web Access Screen Hard to Read in 16-Color Mode

{% raw %}

	Article: Q173382
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.5
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook Web Access, version 5.5 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you access Microsoft Exchange Server with Outlook Web Access (OWA), it may
	be difficult to read certain text in 16-color mode.
	
	CAUSE
	=====
	
	When you run a VGA monitor in 16-color mode and use Outlook Web Access, certain
	text is difficult to read because the background is gray scale.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server
	version 5.5. We are researching this problem and will post new information here
	in the Microsoft Knowledge Base as it becomes available.
