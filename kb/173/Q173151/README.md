## Q173151: MDG: MS Mail to NGW Conversion Utility Written by Novell

{% raw %}

	Article: Q173151
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Novell has written a conversion utility to copy mail from a Microsoft Mail .mmf
	file to Novell GroupWise. This utility uses the Novell GroupWise API gateway to
	process a text file created from the .mmf file. Additional options are added to
	the File menu of the Microsoft Mail Windows client that creates this text file
	and copies the text file to the appropriate subdirectory of the NGW API gateway.
	The text file is converted to messages and are transmitted from the NGW API post
	office to the users mailbox on GroupWise.
	
	MORE INFORMATION
	================
	
	This utility is written and supported by Novell. The utility can be obtained
	from the Novell Web site at:
	
	  http://www.support.novell.com/
	
	Search for "MCU" (without the quotation marks) and "MS Mail" (without the
	quotation marks). The article discussing this utility is titled "GroupWise 4.1
	Conversion Utility for MS Mail."
	
	Additional query words: MCU Conversion Utility
