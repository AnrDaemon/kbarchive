## Q173342: How to Automatically Install TCP/IP in Windows 95

{% raw %}

	Article: Q173342
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): kbtool msnets win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	There is a sample Device Driver Kit (DDK) tool available from Microsoft that you
	can use to determine whether TCP/IP is installed on a computer. If TCP/IP is not
	installed, the tool installs TCP/IP for all network adapters installed in the
	computer.
	
	MORE INFORMATION
	================
	
	This tools adds TCP/IP, but does not customize it. However, the source code is
	included with the sample tool and can be modified for customization.
	
	To obtain the DDK sample, please see the following article in the Microsoft
	Knowledge Base:
	
	  Q185783 SAMPLE: IPInst.exe Determines Whether TCP/IP Is Installed
