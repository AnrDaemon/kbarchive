## Q173900: XWEB: Automatically Accepted Meetings Do Not Show in OWA

{% raw %}

	Article: Q173900
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook Web Access, version 5.5 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	If an account is configured in Schedule+ to automatically accept all meeting
	requests, then any meeting requests sent to that user through Schedule+ or
	Outlook Web Access will not available on the calendar when the user is logged on
	through Outlook Web Access.
