## Q173672: TCP/IP Training Kit Does Not Include CD-Key

{% raw %}

	Article: Q173672
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft TCP/IP Training 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you install the Microsoft Windows NT Server version 4.0 Evaluation Edition,
	the Setup program asks you for a CD-key and explains that this key may be found
	on the back of your CD-ROM jewel case. There is no CD-key on the back of the
	disc sleeve included in this kit.
	
	RESOLUTION
	==========
	
	When prompted, enter the following CD-key:
	
	  " 040-0048126 " (without the quotation marks)
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the "Microsoft TCP/IP Training"
	kit.
