## Q173831: MS Word 97 Advanced Topics Step by Step Comments and Corrections

{% raw %}

	Article: Q173831
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocerr
	Last Modified: 15-JUN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Word 97 Step by Step, Advanced Topics ISBN 1-57231-563-6 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains information on known errors, corrections, and comments
	relating to the Microsoft Press book "Microsoft Word 97 Step by Step, Advanced
	Topics," ISBN 1-57231-563-6.
	
	The following topic is covered:
	
	- Page 68: Instructions To Modify Chart Are Incorrect
	
	- Page 81: Instruction 6 Typo
	
	- Page 132: Incorrect Citation
	
	- Page 150: Topic Left Out Of Lesson Summary
	
	- Page 200: Text Error
	
	- Page 256: Text Error
	
	- Page 266: Text Error
	
	MORE INFORMATION
	================
	
	Page 68: Instructions To Modify Chart Are Incorrect
	---------------------------------------------------
	
	On page 68, under Modify the Chart, Step 1 reads: "Double-click the chart." This
	step produces the Format Data Series menu instead of the Chart menu.
	
	Change:
	"Double-click the chart."
	
	To:
	"Right-click the chart, and then click Chart Type."
	
	
	Page 81: Instruction 6 Typo
	---------------------------
	
	On page 81, in instruction 6 change:
	"In the Field Codes box, select the quotation mark, at sign, backslash, space
	and..."
	
	To:
	"In the Field Codes box, select the quotation mark, backslash, at sign, space
	and..."
	
	
	Page 132: Incorrect Citation
	----------------------------
	
	On page 132, in step 5, change:
	"Spoken as thoughts by Paul Attrades."
	
	To:
	"Spoken as thoughts by Paul Atreides."
	
	NOTE: Screen shot on page 131 must be changed to match correction.
	
	
	Page 150: Topic Left Out Of Lesson Summary
	------------------------------------------
	
	On page 150, hyperlinks were covered in Lesson 8, however they were overlooked in
	the Lesson Summary.
	
	
	Page 200: Text Error
	--------------------
	
	On page 200, in instruction 2 change:
	"On the file menu, point to Send To and then click Word Document."
	
	To:
	"On the file menu, point to Send To and then click Microsoft Word"
	
	
	Page 256: Text Error
	--------------------
	
	On page 256, at the bottom of the page, in the last sentence, "opening" should be
	"open".
	
	Change:
	"...button on the Master Document toolbar to opening the subdocuments inside the
	master document."
	
	To:
	"...button on the Master Document toolbar to open the subdocuments inside the
	master document."
	
	
	Page 266: Text Error
	--------------------
	
	On page 266, in Step 13, change:
	"...type a name File Name box in the Save As dialog box, and then click Save."
	
	To:
	"...type a name in the File Name box in the Save As dialog box, and then click
	Save."
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
