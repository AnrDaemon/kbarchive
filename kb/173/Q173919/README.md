## Q173919: How to Access Internet Sites with MSN Program Viewer

	Article: Q173919
	Product(s): The Microsoft Network
	Version(s): WINDOWS:2.0,2.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 2.0, 2.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to view Web sites with MSN Program Viewer.
	
	MORE INFORMATION
	================
	
	To view Web sites, use the following steps:
	
	1. In the upper-left corner of MSN Program Viewer, click the Click To Show
	  Internet Toolbar button. Note that the Click To Show Internet Toolbar button
	  looks like a small white triangle with a bar above it.
	
	2. In the Address box, type the Web address that you want to view, and then
	  press ENTER.
