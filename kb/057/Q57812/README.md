## Q57812: DOSEDIT Does Not Work in Windows 3.0 Enhanced Mode

{% raw %}

	Article: Q57812
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You use the public domain keyboard enhancement program DOSEDIT in Microsoft
	Windows 386 enhanced mode, and your system stops responding (hangs) or locks up.
	These problems usually occur when you run MS-DOS applications or an MS-DOS
	Prompt.
	
	CAUSE
	=====
	
	DOSEDIT doesn't work in 386 enhanced mode Microsoft Windows.
	
	RESOLUTION
	==========
	
	To correct the problem, remove DOSEDIT.
	
	Additional query words: 3.0 win30 3.00a 3.00 dos edit 3.10 3.11 Win31
