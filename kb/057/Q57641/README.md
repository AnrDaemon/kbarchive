## Q57641: Olivetti Keyboard Mouse Not Detected by Setup

	Article: Q57641
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 26-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	The AT&T/Olivetti Keyboard Mouse is not detected by Windows Setup. The
	mouse choice presented in this configuration is "No mouse or other
	pointing device." If you have this type of mouse, override this choice
	and choose "AT&T/Olivetti Keyboard Mouse" to install the proper mouse
	driver for your configuration.
	
	Additional query words: 3.00 3.00a
