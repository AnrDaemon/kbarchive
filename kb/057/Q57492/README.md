## Q57492: FltSim: Location of Nondirectional Radio Beacons

{% raw %}

	Article: Q57492
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	Nondirectional radio beacons (NDBs) are not shown on the charts provided with
	Flight Simulator Version 3.00. Their positions and frequencies are only listed
	in the README.DOC file, which is on the Program disk.
	
	In Flight Simulator Version 4.00, the positions and frequencies of NDBs are noted
	on the charts.
