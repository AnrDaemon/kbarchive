## Q57968: Flt Sim: Mouse Click Does Not Always Change MAG Indicator

	Article: Q57968
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	It may take several mouse clicks before the indicator changes if you are using
	the mouse to click on the MAG indicator in the instrument panel. This problem
	usually occurs when the OFF and ON settings are displayed on the MAG indicator.
	
	Microsoft is researching this problem and will post new information here as it
	becomes available.
