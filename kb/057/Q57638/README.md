## Q57638: Capturing 320 x 200 MS-DOS Apps in Low Memory

{% raw %}

	Article: Q57638
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	When you attempt to capture the screen of an MS-DOS application running
	in 320 x 200 4-color mode (modes 4 or 5) under Windows version 3.0 386
	enhanced mode in a low memory situation, it may be necessary to
	perform the capture operation twice.
	
	If you attempt to capture a 320 x 200 screen and find that the Clipboard is
	blank, repeat the operation and it should work.
	
	This problem occurs only in low memory situations under 386 enhanced
	mode.
