## Q57595: Amdek CD-ROM Drive and Windows 3.0

	Article: Q57595
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the Amdek CD-ROM drive with Microsoft Windows version 3.0 in
	standard or enhanced modes, the drive does not function correctly under Windows
	unless it has been previously accessed at the MS-DOS level.
	
	WORKAROUND
	==========
	
	Perform a directory (DIR) of the drive or other access to the drive in the
	AUTOEXEC.BAT to enable Windows to access the drive.
