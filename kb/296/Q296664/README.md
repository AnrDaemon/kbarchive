## Q296664: Microsoft Visual InterDev 6.0 Web Technologies Reference Comment

	Article: Q296664
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 25-APR-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Visual InterDev 6.0 Web Technologies Reference ISBN 1-57231-871-6 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Microsoft Visual InterDev 6.0 Web
	Technologies Reference, ISBN 1-57231-871-6.
	
	The following topics are covered:
	
	- Pages 134-135: NAME Parameter Incorrectly Defined
	
	MORE INFORMATION
	================
	
	Pages 134-135: Name Parameter Incorrectly Defined
	-------------------------------------------------
	
	On pages 134 and 135 the NAME parameter and the description of the NAME parameter
	are incorrect.
	
	Change:
	"NAME=window_name|_blank|_parent|_self|_top"
	
	To:
	"NAME=frame_name"
	
	On page 135 the description for the NAME parameter should also be changed.
	
	Change the description to:
	"Sets or retrieves the window or frame name. The name property identifies which
	frame displays the content of a linked document. To access a window's name
	property, use the "window" keyword."
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
