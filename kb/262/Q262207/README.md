## Q262207: Allegiance: Squad Member Is Unable to Join a Squad Game

	Article: Q262207
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kberrmsg kbimu msgame
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Allegiance, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Microsoft Allegiance, a player who has been accepted into a Squad game may
	continue to receive a message stating that they are not a member of a Squad. As
	a result, the player is unable to join the Squad game.
	
	CAUSE
	=====
	
	This behavior can occur because Squad membership data is not stored on the
	Allegiance game server.
	
	RESOLUTION
	==========
	
	To resolve this issue, quit Allegiance, restart Allegiance, and then join the
	Squad game.
	
	MORE INFORMATION
	================
	
	When you quit and then restart Allegiance, the game checks with the Squad server
	to determine who is eligible to play in the Squad game.
