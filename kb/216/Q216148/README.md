## Q216148: Help on Index Server Not Available from MMC

	Article: Q216148
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0,4.0a,4.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT version 4.0 Option Pack 
	- Microsoft BackOffice Small Business Server versions 4.0a, 4.5 
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The "Help on Index Server" menu item under the MMC Help menu does not start Help
	for Index Server. The documentation for Index Server has been moved to the HTML
	documentation for the Windows NT Option pack.
	
	MORE INFORMATION
	================
	
	For step-by-step instructions on using Index Server, see the HTML
	documentation:
	
	1. Click Start, click Programs, click Windows NT 4.0 Option Pack, and then click
	  Product Documentation.
	
	2. In Contents, click Microsoft Index Server.
