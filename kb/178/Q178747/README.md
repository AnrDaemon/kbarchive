## Q178747: How to Obtain a Foreign Language Version of DirectX

{% raw %}

	Article: Q178747
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): kbtool win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to obtain a foreign language (localized) version of
	DirectX.
	
	MORE INFORMATION
	================
	
	To obtain a localized version of DirectX, follow these steps:
	
	1. Go to the following Microsoft Web site:
	
	  http://www.microsoft.com/directx/
	
	2. Click Downloads, and then click DirectX 5.0 For End-users.
	
	3. Follow the instructions to install the localized version of DirectX you want.
	
	For more information about DirectX, see the following Microsoft Web site:
	
	  http://www.microsoft.com/directx/
