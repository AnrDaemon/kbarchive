## Q178203: FS98: Nothing Happens When You Click a Menu Command

	Article: Q178203
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbdisplay kbenv kbhw kbimu kbHardware
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator 98 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click a command on a Flight Simulator 98 menu, the command may not be
	carried out and the cockpit may reappear.
	
	This behavior can occur with any command, but it most frequently occurs with the
	Time & Season command on the World menu and the Select Aircraft command on
	the Aircraft menu.
	
	CAUSE
	=====
	
	This issue can occur if you have an ATI All In Wonder video adapter and the
	Television check box is selected in the Display tool in Control Panel.
	
	RESOLUTION
	==========
	
	To resolve this issue, clear the Television check box by following these steps:
	
	1. Click Start, point to Settings, and then click Control Panel.
	
	2. Double-click Display.
	
	3. Click the Settings tab.
	
	4. Click the Television check box to clear it.
	
	5. Click OK.
	
	6. If you are prompted to restart the computer, click Yes.
	
	  NOTE: If you are not prompted to restart the computer and you still experience
	  the behavior described in the "Symptoms" section of this article, restart the
	  computer.
	
	
	MORE INFORMATION
	================
	
	When the Television check box is selected, the ATI All In Wonder video adapter
	can use a television set to display screen images.
