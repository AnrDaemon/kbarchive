## Q178696: Age of Empires: List of Civilizations and Wonders

	Article: Q178696
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbui aoe kbimu msgamekbfaq
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Age of Empires, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes the Wonders that the different civilizations can build.
	
	MORE INFORMATION
	================
	
	Age of Empires includes twelve different civilizations. However, it includes
	only four different Wonders that the civilizations can build.
	
	The following table lists the twelve civilizations in Age of Empires and the
	Wonders they can build:
	
	  Civilization   Wonder
	  ---------------------------------------
	
	  Babylonian     Babylonian style pyramid
	  Hittite        Babylonian style pyramid
	  Persian        Babylonian style pyramid
	
	  Assyrian       Egyptian style pyramid
	  Egyptian       Egyptian style pyramid
	  Sumerian       Egyptian style pyramid
	
	  Greek          Greek style statue
	  Minoan         Greek style statue
	  Phoenician     Greek style statue
	
	  Choson         Oriental style shrine
	  Shang          Oriental style shrine
	  Yamato         Oriental style shrine
