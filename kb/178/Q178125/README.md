## Q178125: XWEB: OWA Users Unable to Access Organization Forms Library

{% raw %}

	Article: Q178125
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.5
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook Web Access, version 5.5 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When users' permissions are set to Read Only on the Organization Forms Library,
	users are unable to access the folder using Outlook Web Access. They may receive
	the error message MAPI_E_NOT_FOUND.
	
	WORKAROUND
	==========
	
	To work around this problem, grant Read and Create permissions to the users.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server
	version 5.5. We are researching this problem and well post new information here
	in the Microsoft Knowledge Base as it becomes available.
