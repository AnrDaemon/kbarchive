## Q192928: MCM: Motorcycle Does Not Move When You Attempt to Accelerate

	Article: Q192928
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbhw kbui kbimu msgame kbHardware
	Last Modified: 25-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Motocross Madness, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Motocross Madness, your motorcycle may not move when you
	attempt to accelerate.
	
	CAUSE
	=====
	
	This behavior can occur if you assign the acceleration function to a
	proportional device such as a joystick handle or throttle control.
	
	RESOLUTION
	==========
	
	To resolve this issue, assign the acceleration function to a button on the game
	controller or mouse, or to a key on the keyboard.
	
	STATUS
	======
	
	This behavior is by design.
