	SUMMARY
	=======
	
	Rs2dbf.exe is a sample file that shows how to convert an ActiveX Data Objects
	(ADO) recordset to a Visual FoxPro cursor. This self-extracting executable
	contains a Visual FoxPro program called Rs2dbf.prg and the necessary ADO include
	files. Running the program in Visual FoxPro creates an ADO recordset, converts
	it to a Visual FoxPro cursor, and then opens the cursor in a Browse window.
