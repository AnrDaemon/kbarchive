## Q192335: Err Msg: Information Is in an Invalid Format

	Article: Q192335
	Product(s): The Microsoft Network
	Version(s): 5.0,5.1,5.2,5.3
	Operating System(s): 
	Keyword(s): kbenv kberrmsg kbmsn
	Last Modified: 27-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 5.0, 5.1, 5.2, 5.3 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you connect to MSN, The Microsoft Network, you may receive the following
	error message:
	
	  Information is in an invalid format.
	
	CAUSE
	=====
	
	This behavior can occur if the Dial-Up Networking connections for MSN are
	damaged.
	
	RESOLUTION
	==========
	
	To resolve this issue, create new Dial-Up Networking connections for MSN. To do
	this, follow these steps:
	
	1. Double-click the My Computer icon on the desktop, and then double-click
	  Dial-Up Networking.
	
	2. Right-click each of the following Dial-Up Networking connections, and then
	  click Delete:
	
	  - MSN
	  - MSN (Backup for advanced use only)
	  - The Microsoft Network
	  - The Microsoft Network Backup
	
	3. Close the Dial-Up Networking window, and then restart the computer.
	
	4. On the desktop, right-click the MSN icon, and then click Connection Settings.
	
	5. On the Dialing tab, next to the Phone Number box, click Phone Book.
	
	6. Click the appropriate access number for your area, and then click OK.
	
	7. On the Dialing tab, next to the Backup Number box, click Phone Book.
	
	8. Click the appropriate backup access number for your area, click OK, and then
	  click OK again.
	
	9. Double-click the MSN icon on the desktop, type your Member ID and password in
	  the appropriate boxes, and then click Connect.
