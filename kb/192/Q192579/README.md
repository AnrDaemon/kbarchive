## Q192579: MCM: &quot;No 3D Accelerator&quot; Error Message, or Game Hangs or Quits

	Article: Q192579
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kberrmsg kbimu msgamekbfaq
	Last Modified: 16-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Motocross Madness, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to start Motocross Madness, you may experience any of the
	following symptoms:
	
	- You receive an error message stating that you do not have a 3-D accelerator.
	
	- Motocross Madness stops responding (hangs).
	
	- When you click OK after the initial video test, Motocross Madness quits
	  unexepectedly.
	
	CAUSE
	=====
	
	This behavior can occur if the both of the following conditions are true:
	
	- You are using a display adapter based on the Rendition chipset.
	
	- Your video driver does not fully support Direct3D.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the updated Rendition V1000
	Reference Driver from the following Rendition Web site:
	
	  http://www.rendition.com/download.html
