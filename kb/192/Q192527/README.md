## Q192527: Hot Swappable Drive Does Not Come Online on Alpha

	Article: Q192527
	Product(s): Microsoft Windows NT
	Version(s): WinNT:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you replace a hard drive on an Alpha system and have a hot swappable drive
	installed, the hot swappable drive will not come online without disabling
	Clusdisk.sys.
	
	NOTE: This works correctly on x86 platforms.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 4.0. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
