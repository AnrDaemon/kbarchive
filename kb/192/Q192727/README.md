## Q192727: XCLN: OWA Next/Prev Buttons Skip Messages Intermittently

	Article: Q192727
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.0,5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Active Server Components, version 5.0 
	- Microsoft Outlook Web Access, version 5.5 
	- Microsoft Outlook Web Access, version 5.5 Service Pack 1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	When you use the Outlook Web Access client to browse your Inbox, messages may be
	intermittently skipped if you open the last message and use the Previous (PREV)
	button to browse. Rather than the previous message being opened, one or more
	messages may be skipped in the Inbox contents.
	
	WORKAROUND
	==========
	
	To work around this problem, close the message you are viewing and select the
	previous message from the Inbox listing.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
