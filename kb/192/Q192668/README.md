## Q192668: SMS: Network Monitor Error &quot;The NAL Is Not Remotable&quot;

	Article: Q192668
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.2
	Operating System(s): 
	Keyword(s): kbsmsUtil smsnetmon smsutil
	Last Modified: 31-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When using Network Monitor to connect to a remote agent, the following error is
	displayed:
	
	  The NAL is not remotable.
	
	You will be unable to connect to a remote agent until you quit and restart
	Network Monitor.
	
	WORKAROUND
	==========
	
	To work around this problem, close Network Monitor and reopen it.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.2.
