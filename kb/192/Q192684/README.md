## Q192684: XCLN: Spelling Check Before Sending Ignored Sending from Office

{% raw %}

	Article: Q192684
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 24-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Windows NT client, version 5.0 
	- Microsoft Exchange Windows 95/98 client, version 5.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	When sending a message from one of the Microsoft Office applications using Send
	To on the File menu, you may notice that the "Always check spelling before
	sending" setting is ignored and the message is sent without the spelling checker
	being used.
	
	WORKAROUND
	==========
	
	To send a message with an Office attachment and have the message's spelling
	checked before it is sent, you need to compose the message and insert the
	attachment using an Exchange client interface.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
