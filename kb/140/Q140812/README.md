## Q140812: File Date Stamps in AUDIT.RUL Do Not Match Actual Date Stamps

{% raw %}

	Article: Q140812
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.1
	Operating System(s): 
	Keyword(s): kbnetwork kbAudit smsaudit
	Last Modified: 31-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.1 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	The date stamps of the Microsoft software products listed in the Systems
	Management Server AUDIT.RUL file do not match the date stamps on the files from
	the actual software products.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.1. This problem has been corrected in Systems Management Server
	version 1.2.
