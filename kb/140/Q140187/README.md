## Q140187: Setting Recurring Monthly Appointments to Day 31

	Article: Q140187
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 28-JUL-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you set recurring monthly appointments to day 31, you will find appointments
	scheduled only for months that have 31 days.
	
	CAUSE
	=====
	
	This is by product design.
	
	MORE INFORMATION
	================
	
	If you want to set a recurring monthly appointment for the last day of each
	month, follow the steps below:
	
	1. Open Schedule+, version 7.0.
	
	2. From the Insert menu, select Recurring Appointment.
	
	3. Select the When tab, then click Monthly. In the interval and day lists,
	  choose Last and Day (of every 1 month, which is the default).
	
	
	4. When you focus on the Planner view, the last day of every month will have the
	  appointment scheduled.
