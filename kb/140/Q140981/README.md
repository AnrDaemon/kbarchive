## Q140981: PPT7: Components That Cannot be Added or Removed

	Article: Q140981
	Product(s): Microsoft PowerPoint for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): kbsetup
	Last Modified: 16-APR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft PowerPoint for Windows 95, version 7.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The PowerPoint add-ins that come with PowerPoint for Windows 95 (listed below)
	cannot be added or removed using the PowerPoint or Office Setup program. These
	features are:
	
	- Style Checker
	
	- AutoContent Wizard
	
	- Interactive Settings
	
	- Presentation Conferencing
	
	- AutoClipart
	
	- Pack and Go
	
	- Genigraphics Wizard
	
	- Slide Navigator
	
	- Rehearsal
	
	- Slide Meter
	
	- Meeting Minder
	
	- Write-Up
	
	MORE INFORMATION
	================
	
	These features are accessed through two PowerPoint add-in files, Ppttools.ppa
	and Pptools1.ppa. These files are installed as part of the PowerPoint option in
	Setup and cannot be added or removed separately from PowerPoint.
