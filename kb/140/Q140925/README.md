## Q140925: SMS Setup Doesn't Run on Windows NT v. 3.5 w/ Service Pack 3

	Article: Q140925
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.1
	Operating System(s): 
	Keyword(s): kbnetwork kbsetup smssetup
	Last Modified: 27-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.1 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you run Systems Management Server Setup on your computer running Windows NT
	version 3.51 with Service Pack 3, the following error message appears:
	
	  SMS version 1.1 requires Windows NT version 3.5 with Service Pack 2 or
	  Windows NT version 3.51. Please upgrade the server operating system or
	  install the service pack.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.1. This problem has been corrected in Systems Management Server
	version 1.2.
