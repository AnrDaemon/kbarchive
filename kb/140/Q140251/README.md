## Q140251: ErrMsg When Using Windows NT: Cannot find USER.DLL

	Article: Q140251
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0; winnt:3.5,3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft 500 Nations, version 1.0 
	- Microsoft Dogs for Windows, version 1.0 
	- Microsoft Oceans for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you are using the multimedia programs listed at the beginning of this
	article, and you are running Microsoft Windows NT operating system version 3.5
	or 3.51, you may receive the following error message:
	
	  Cannot Find USER.DLL
	
	CAUSE
	=====
	
	The problem occurs because one of these multimedia programs removes the file
	User.dll from the system.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft multimedia
	products listed at the beginning of this article when you use Windows NT version
	3.5 and 3.51. We are researching this problem and will post new information here
	in the Microsoft Knowledge Base as it becomes available.
