## Q140801: German Office Pro 4.3c PDF and STF Files are Missing

	Article: Q140801
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.1
	Operating System(s): 
	Keyword(s): kbnetwork smsgeneral kbArtTypeINF
	Last Modified: 27-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.1 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	The following scripts (PDFs/STFs) exist for the U.S. Systems Management Server
	version 1.10, but are missing from the German version:
	
	  Windows NT 3.51
	  Office 4.2C
	  Office 4.3A
	  Office 4.3C
	  Access 2.0 Service Pack
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.1. This problem has been corrected in Systems Management Server
	version 1.2.
