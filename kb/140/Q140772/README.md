## Q140772: Some Desktop Themes Interfere with Online Viewer Text

	Article: Q140772
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.2,95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network version 1.2 
	- Microsoft Plus! for Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you have installed the Travel (High Color) or Science (256 Color) desktop
	theme from Microsoft Plus!, the text in Online Viewer's initial window is
	truncated. When you use these themes, the "P" and "t" in the message "Preparing
	title...Please wait" are cut off.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in The Microsoft Network version
	1.2.
