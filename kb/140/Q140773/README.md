## Q140773: Cannot Drag BBS Message to Create Shortcut

{% raw %}

	Article: Q140773
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.0,1.05,1.2,1.3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.0, 1.05, 1.2, 1.3 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You cannot drag a BBS message to another location to create a shortcut.
	
	RESOLUTION
	==========
	
	Use the Create Shortcut command on the File menu to create a shortcut.
