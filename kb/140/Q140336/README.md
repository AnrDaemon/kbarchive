## Q140336: Remote Mail and Inbox Display Different &quot;From&quot; Name

	Article: Q140336
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In Microsoft Exchange, the From field in the remote preview window may display a
	different name than the From field in the Inbox after down- loading the message.
	
	MORE INFORMATION
	================
	
	The From field in the remote preview window displays the address from the mail
	header. The From field in the Inbox displays the "friendly" name for the
	address.
	
	For example, the From field in the remote preview window may display "johnd"
	while the From field in the Inbox displays "John Doe."
