## Q140766: MSB Ocean: Video Distortion in Games When Using S3 Card

	Article: Q140766
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:1.0,3.x
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Scholastic's Magic School Bus series: Explores the Ocean for Windows, version 1.0 
	- the operating system: Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play games in Magic School Bus Explores the Ocean (MSB Ocean), you
	experience video distortion. This occurs if you are using version 2.1 of the S3
	video driver in Windows 3.x.
	
	RESOLUTION
	==========
	
	Upgrading to a newer version of the S3 driver for your video card resolves this
	problem.
	
	Find out the what video driver and version is installed on your computer. If you
	are using version 2.1, you need to use a more recent version.
	
	Checking Your Driver Version
	----------------------------
	
	1. In the Main program group, run Windows Setup.
	
	2. The information on the Display line describes which video driver and version
	  you are currently using. For example, if you are using an S3 video driver you
	  may see information similar to the following:
	
	  S3 2.1 640x480 256 Colors
	
	3. If your video driver is an S3 and the version is 2.1, you need to upgrade to
	  the newest version of the S3 driver.
	
	For more information about obtaining and installing the latest S3 driver, please
	contact your hardware manufacturer.
	
	MORE INFORMATION
	================
	
	Version 2.11 of the S3 video driver is known to resolve the video distortion in
	the games.
