## Q140710: Incorrect Help Topic: Cannot Select Service Type During Signup

	Article: Q140710
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.0,1.05,1.2,1.3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.0, 1.05, 1.2, 1.3 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are signing up for MSN, The Microsoft Network, you cannot select a
	service type. The Help topic indicating that you can select a service type is
	incorrect.
	
	MORE INFORMATION
	================
	
	The available service types are:
	
	- The Microsoft Network
	
	- Internet and The Microsoft Network
	
	You cannot select a service type because sign-up occurs only on MSN phone lines.
	
	Additional query words: msn sign-up sign up
