## Q140924: SMS Senders Stops if Estimated Bandwidth is Set to a High Value

{% raw %}

	Article: Q140924
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.1
	Operating System(s): 
	Keyword(s): kbnetwork smssenders kbSMSSender
	Last Modified: 31-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.1 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	If you set the Estimated Bandwidth in Systems Management Server to a high value
	(higher than the actual bandwidth), the sending thread hangs and the Senders
	stops.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.1. This problem has been corrected in Systems Management Server
	version 1.2.
