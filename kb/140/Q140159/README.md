## Q140159: Streets 4.0: Unable to Print Highways with HP Printer

	Article: Q140159
	Product(s): Microsoft Automap
	Version(s): WINDOWS:4.0,95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Automap Streets for Windows, version 4.0 
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you print a map to a Hewlett-Packard (HP) Deskjet 500C printer, the highway
	lines do not print.
	
	
	RESOLUTION
	==========
	
	Set the Road Drawing Style to Regular or Enhanced.
	
	Setting the Road Drawing Style
	------------------------------
	
	1. On the Map menu, choose Options.
	
	2. Click the Drawing tab, and then select Regular or Enhanced.
	
	3. Click OK.
	
	
	MORE INFORMATION
	================
	
	Although the highway lines fail to print, highway symbols print as expected.
