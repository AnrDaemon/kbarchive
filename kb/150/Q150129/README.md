## Q150129: FTM Reports Incorrect Connect Speed for ISDN Modems

{% raw %}

	Article: Q150129
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.2,1.3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.2, 1.3 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are connected to MSN, the Microsoft Network, and you download a file
	with an ISDN modem, the lower-right corner of the File Transfer Manager (FTM)
	reports the connect speed (BPS) as 1.
	
	CAUSE
	=====
	
	The File Transfer Manager is designed to operate with regular, not ISDN, modems.
