## Q150690: Err Msg: Unable to Complete This Task

	Article: Q150690
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.3,2.0,2.5,2.51,2.52,2.6
	Operating System(s): 
	Keyword(s): kbmsn
	Last Modified: 24-AUG-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.3, 2.0, 2.5, 2.51, 2.52, 2.6 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to run the sign-up program for MSN, The Microsoft Network, you
	may receive the following error message:
	
	  Unable to Complete This Task.
	
	CAUSE
	=====
	
	You may have insufficient available space on your hard disk.
	
	RESOLUTION
	==========
	
	Free up additional hard disk space, and then run the sign-up program again.
