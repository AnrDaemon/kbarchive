## Q150875: No Internet Tab Available in Compose Note Properties

	Article: Q150875
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	- Microsoft Plus! for Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click Properties on the File menu in Microsoft Exchange to change the
	encoding format type while you are composing a new Internet message, the
	Internet tab may not be available.
	
	CAUSE
	=====
	
	This behavior can occur if Microsoft Word is enabled as your e-mail editor.
	
	RESOLUTION
	==========
	
	To disable Word as your e-mail editor, follow these steps:
	
	1. In Microsoft Exchange, click WordMail Options on the Compose menu.
	
	2. Click the Enable Word As Email Editor check box to clear it, and then click
	  Close.
	
	Additional query words: mime uuencode
