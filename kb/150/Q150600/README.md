## Q150600: Full Screen Option Doesn't Change in Full Screen View

{% raw %}

	Article: Q150600
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbusagekbfaq
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Arcade for Windows, version 1.0 
	- Microsoft Return of Arcade for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you switch from window to full-screen view or vice-versa, the Options menu
	always displays in full-screen view. It does not change to reflect the window
	size.
	
	MORE INFORMATION
	================
	
	By default all games are displayed in a window. To switch to full-screen mode,
	click Options and then click Full Screen. To return to window view, click
	Options and click Full Screen. Note that the selection does not change.
