## Q150171: Baseball 1994: Minimum System Requirements

{% raw %}

	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1994 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Complete Baseball Guide for Windows 1994 edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following article contains a copy of the minimum system requirements for
	Microsoft Complete Baseball Guide for Windows, 1994 edition, as listed on the
	product box.
	
	MORE INFORMATION
	================
	
	The minimum system requirements to run Complete Baseball are:
	
	- A Multimedia PC or compatible with a 386SX or higher microprocessor.
	
	- MS-DOS operating system 3.1 or later
	
	- Microsoft Windows operating system version 3.1 or later.
	
	- CD-ROM drive.
	
	- 4 MB of RAM.
	
	- 2 MB available hard disk space.
	
	- VGA+ or higher resolution display (for 256 color support).
	
	- Audio board.
	
	- Speakers or headphones.
	
	- Microsoft Mouse or compatible pointing device.
	
	- Baseball Daily requires a 2400-baud or higher Hayes-compatible modem and an
	  additional 500k of hard disk space.
