## Q150722: Changing Country During Signup May Download Incorrect Info.

	Article: Q150722
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.0,1.2,1.3,2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.0, 1.2, 1.3, 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are running signup for MSN, The Microsoft Network, if you click
	Settings on the Calling screen and change your country, the information
	downloaded to your computer and the suggested access numbers may not be correct
	for the country you selected.
	
	CAUSE
	=====
	
	Signup uses the country that is specified in Dialing properties before you begin
	signup to determine which information to download and which access numbers to
	suggest.
	
	RESOLUTION
	==========
	
	Do not change the country using Settings on the Calling screen during signup.
	Make sure that your country is correctly set in Dialing properties before you
	run signup.
