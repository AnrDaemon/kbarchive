## Q150030: Complete Gardening: Unable To Set Zone On USDA Map

	Article: Q150030
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbfaq
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Complete Gardening for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In the Plant Selector, when you click the USDA Zone Map to set the USDA plant
	hardiness zone for a specific location, nothing happens. The dialog message that
	is supposed to confirm that your zone has been set does not appear, and the USDA
	zone you selected is not recorded.
	
	RESOLUTION
	==========
	
	Change the display color depth to 256 colors.
	
	MORE INFORMATION
	================
	
	This behavior has been reported with certain S3 video cards configured for High
	Color (16-bit).
	
	To change display color depth in Windows, do the following:
	
	1. Click Start, point to Settings, and then click Control Panel.
	
	2. Double-click Display.
	
	3. Click the Settings tab.
	
	4. In the Color Palette or Colors box, click 256 Colors.
	
	5. Click OK.
	
	6. Restart Windows if you are prompted.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Complete Gardening for
	Windows, version 1.0. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
