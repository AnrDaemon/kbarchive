## Q150618: Pregnancy: Program Window Does Not Reopen To Previous Size

{% raw %}

	Article: Q150618
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Pregnancy and Child Care, Preview on MSN 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you restart Microsoft Pregnancy and Child Care after resizing the program
	window, Pregnancy and Child Care returns to its original size. The program
	window does not open in its previous size.
	
	MORE INFORMATION
	================
	
	This behavior is by design.
