## Q150535: Cannot Create Backup Copies of Windows 95 Disks

{% raw %}

	Article: Q150535
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95kbfaq
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	You cannot make backup copies of your original Windows 95 disks.
	
	MORE INFORMATION
	================
	
	Windows 95 disks are Distribution Media Format (DMF) disks and are not
	compatible with the MS-DOS COPY and DISKCOPY commands. You cannot create backup
	copies of the Windows 95 disks.
