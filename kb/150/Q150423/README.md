## Q150423: INFO: Visual Basic 4.0a Fixes for Windows as of April 23, 1996

	Article: Q150423

	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kbList kbVBp kbVBp400 kbGrpDSVB kbDSupportkbfixlist
	Last Modified: 19-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual Basic Standard Edition, 32-bit, for Windows, version 4.0 
	- Microsoft Visual Basic Professional Edition, 16-bit, for Windows, version 4.0 
	- Microsoft Visual Basic Professional Edition, 32-bit, for Windows, version 4.0 
	- Microsoft Visual Basic Enterprise Edition, 16-bit, for Windows, version 4.0 
	- Microsoft Visual Basic Enterprise Edition, 32-bit, for Windows, version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Visual Basic 4.0a is a synchronization release of Visual Basic 4.0. that
	synchronizes the components shared between Microsoft Access For Windows 95 and
	Visual Basic 4.0. These components include the language engine (Visual Basic For
	Applications), the Jet database engine, and the Data Access Objects (DAO). This
	release also upgrades the following components: the data aware grid for both the
	16-bit and 32-bit editions, the Remote Data Objects and Remote Data Control for
	the 32-bit only edition, and the Windows 95 controls.
	
	To obtain this release, please call 1-800-360-7561. Although this release is free
	to all registered users, there is a $5 charge to cover shipping and handling.
