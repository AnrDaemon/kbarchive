## Q150267: Must Have Optional Duplexing Unit to Print Duplex to HP Printers

{% raw %}

	Article: Q150267
	Product(s): Microsoft Windows NT
	Version(s): 3.50 3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation versions 3.5, 3.51 
	- Microsoft Windows NT Server versions 3.5, 3.51 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you install a Hewlett-Packard (HP) printer driver that supports duplex
	printing, then choose the duplexing option, the printer does not print duplex.
	
	RESOLUTION
	==========
	
	Most HP printers require an optional duplexing unit to print duplex. For more
	information about your specific printer model, contact your Hewlett- Packard
	reseller.
