## Q150968: XADM: Initials Missing From Template Mapping Tables

	Article: Q150968
	Product(s): Microsoft Exchange
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 26-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Exchange template to match tables for directory synchronization
	(dirsync) with Microsoft Mail is missing the "Initials" attribute.
	
	MORE INFORMATION
	================
	
	How to view the attributes:
	
	1. Open the Exchange Administrator program.
	
	2. Go to the Server object that hosts the Directory Synchronization service
	  (DXA).
	
	3. View the Properties for the DXA.
	
	4. Select Outgoing or Incoming templates.
	
	5. Choose 'New'.
	
	You will now be able to select template information that you wish to map to the
	incoming or outgoing dirsync attributes. Notice that the Initials attribute is
	missing.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server,
	version 4.0. We are researching this problem and will post new information here
	in the Microsoft Knowledge Base as it becomes available.
