## Q150725: Cannot Run MSN Without Modem if You Do Not Sign Up on Computer

	Article: Q150725
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.2,1.3
	Operating System(s): 
	Keyword(s): kbui kbmsn
	Last Modified: 13-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.2, 1.3 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you are not using a modem to connect to MSN, The Microsoft Network, and you
	did not run the MSN sign-up program on your computer, clicking the MSN icon on
	the desktop always brings up the sign-up program.
	
	If you are already a member of The Microsoft Network and you want to use your MSN
	account on a new computer, at the sign-in screen, click: "Click here if you are
	already a member of The Microsoft Network."
	
	Additional query words: msn signin signup
