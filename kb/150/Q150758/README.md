## Q150758: Installing MSN 1.3 Downgrades Access Number Phone Book

	Article: Q150758
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network version 1.3 
	-------------------------------------------------------------------------------
	
	When you install version 1.3 of MSN, The Microsoft Network, an old version
	of the access number phone book is installed. This old phone book may not
	contain the access number you are currently using.
	
	The next time you connect to MSN, your phone book is automatically
	upgraded if MSN is idle for at least 45 seconds.
	
	Additional query words: msn
