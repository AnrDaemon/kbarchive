## Q150170: Telecommander Audio Driver Fails When You Open MS-DOS Prompt

	Article: Q150170
	Product(s): Microsoft Mail For PC Networks
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): win95 msphone1
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Phone, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you open an MS-DOS prompt while you are using audio features of the Diamond
	Telecommander, the audio appears to stop functioning.
	
	CAUSE
	=====
	
	The version of the Diamond Telecommander driver you are using is an older
	version.
	
	
	RESOLUTION
	==========
	
	For more information about obtaining an updated driver, contact Diamond
	Multimedia.
