## Q150626: Mac Bookshelf 1996-97: Items Installed by an Easy Install

{% raw %}

	Article: Q150626
	Product(s): Microsoft Home Multimedia Titles
	Version(s): MACINTOSH:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf 1996-97 for Macintosh 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the files and folders installed by the Installer when you
	perform an Easy Install.
	
	MORE INFORMATION
	================
	
	Files or Folders Installed            Version
	--------------------------            -------
	
	Bookshelf 1996-97 folder
	  Bookshelf 1996-97 Read Me
	  Bookshelf Guide                      1.0
	  Drop Updates Here alias folder
	  Internet Config                      1.2
	  Microsoft Bookshelf                  1996-97
	  Microsoft Multimedia Catalog alias
	  SimpleText                           1.3.1
	  Internet Explorer folder
	     Internet Explorer                 2.0b3
	     Internet Explorer License
	     Internet Explorer Read Me
	     Help folder
	        release.htm
	        topics.htm
	     Plug-ins folder
	        Default Plugin                 1.0b1
	
	System Folder
	  Control Panels folder
	     QuickShelf                        '96
	     Sound                             8.0.5
	  Extensions folder
	     AppleScriptLib                    1.1
	     7.0/7.1 Apple Guide Enabler
	     (Apple Guide Enabler is installed if the System is 7.1)
	     Apple Guide                       2.0.2
	     AppleScript                       1.1
	     Internet Config Extension         1.2
	     QuickTime                         2.1
	     QuickTime Musical Instruments     2.1
	     QuickTime PowerPlug               2.1
	     Sound Manager                     3.1
	     (Sound Manager is installed if the System is less than 7.5.3)
	  Fonts folder
	     Arial                             1.02
	     Bookshelf Fonts                   2.0
	     Symbol                            1.03
	     Times New Roman                   1.02
	  Preferences folder
	     Internet Preferences
