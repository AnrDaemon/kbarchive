## Q150857: No Sound Playing Audio CDs with Philips CD-ROM Drive

	Article: Q150857
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to play audio CDs using a Philips CM 205 Multisession, CM 206, or
	CM 207 CD-ROM drive, no music can be heard.
	
	CAUSE
	=====
	
	This problem is caused by an ATAPI firmware incompatibility between the Philips
	CD-ROM drive and the Windows 95 protected-mode driver.
	
	RESOLUTION
	==========
	
	To resolve this problem, please contact Philips to inquire about obtaining
	updated drivers that allow audio CDs to be played.
