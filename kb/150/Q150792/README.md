## Q150792: Support for HP 20XT Optical Jukebox

{% raw %}

	Operating System(s):

	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation versions 3.5, 3.51, 4.0 
	- Microsoft Windows NT Server versions 3.5, 3.51, 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article provides you with information on where you can purchase
	software/drivers that allow the Hewlett-Packard 20XT Optical Jukebox to work
	with Windows NT.
	
	MORE INFORMATION
	================
	
	Hewlett-Packard and Microsoft do not have any drivers that support the
	Hewlett-Packard 20XT Optical Jukebox with Windows NT. Below is a short list of
	companies that provide the drivers necessary for Optical Jukebox systems to work
	with Windows NT as well as many other OS platforms.
	
	The companies listed below provide drivers necessary for the following OS
	platforms:
	
	SunOS, Solaris, VAX VMS, OSF-3, Windows NT, NFS, Netware, Digital VMS, Ultrix,
	OSF, HPUX, IBM AIX
