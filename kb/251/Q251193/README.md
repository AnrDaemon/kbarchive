## Q251193: FS2000: How to Print a Navigation Log

	Article: Q251193
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbprint kbui fsim kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator 2000 
	- Microsoft Flight Simulator 2000 Professional Edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to print the navigation log for a flight in Microsoft
	Flight Simulator 2000.
	
	MORE INFORMATION
	================
	
	How to Print a Navigation Log
	-----------------------------
	
	To print a navigation log:
	
	1. On the Flights menu in Flight Simulator 2000, click Flight Planner.
	
	2. Under "Departure airport", click Select.
	
	3. Under "Search results", click to select your departure airport.
	
	4. Click the green check mark to save your selection.
	
	5. Under "Destination airport", click Select.
	
	6. Under "Search results", click to select your destination airport.
	
	7. Click the green check mark to confirm your selection.
	
	8. Under "Auto routing", click "Find route".
	
	9. Click NavLog.
	
	10. Click Print.
	
	NOTE: If the Print button is not present near the bottom of the NavLog dialog
	box, the Flight Simulator 2000 Update may not be installed.
	
	Download and Install the Flight Simulator 2000 Update
	-----------------------------------------------------
	
	To print the navigation log for a flight in Microsoft Flight Simulator 2000, you
	must first install the appropriate Flight Simulator 2000 Update for your version
	of Flight Simulator 2000.
	
	To download and install the Flight Simulator 2000 Update for your version of
	Flight Simulator 2000, visit the following Microsoft Web site:
	
	  http://www.microsoft.com/games/fs2000/downloads_fs2000update.asp
