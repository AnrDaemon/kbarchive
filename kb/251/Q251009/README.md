## Q251009: INFO: Where to Find the MMC Snap-In Designer for Visual Basic

	Article: Q251009
	Product(s): Microsoft Visual Basic for Windows
	Version(s): 6.0
	Operating System(s): 
	Keyword(s): kbtophit kbMMC kbVBp600 kbGrpDSPlatform kbDSupport kbSnapIn
	Last Modified: 25-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual Basic Professional Edition for Windows, version 6.0 
	- Microsoft Visual Basic Enterprise Edition for Windows, version 6.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Management Console (MMC) Snap-In Designer for Visual Basic is
	included with Microsoft Platform Software Development Kit (SDK), starting with
	the January 2000 edition.
	
	MORE INFORMATION
	================
	
	The Platform SDK can be downloaded from the following Web site:
	
	  http://www.microsoft.com/msdownload/platformsdk/setuplauncher.asp>
	
	Once you have installed the Platform SDK, you can find information about the
	Designer by browsing to the following location:
	
	  \Program Files\Microsoft Platform SDK\Samples\SysMgmt\MMC\Visual
	  Basic\Template\
	
	To install the MMC Snap-in Designer for Visual Basic from the platform SDK, you
	need to install the Microsoft Management Console Samples under the Sample
	Code\Management Samples section. This will install the core design-time and
	runtime files necessary for the Designer.
	
	The following table lists the core Designer files with version and size
	information:
	
	+-----------------------------------+
	| FILE         | VERSION  | SIZE    | 
	+-----------------------------------+
	| mmcproxy.dll | 5.0.2150 | 1162 KB | 
	+-----------------------------------+
	| mssnapr.dll  | 5.0.2150 | 1715 KB | 
	+-----------------------------------+
	| mssnapd.ocx  | 5.0.2150 | 1266 KB | 
	+-----------------------------------+
