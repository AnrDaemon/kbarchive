## Q223047: XCLN: AutoArchive Settings Do Not Work For Additional Mailboxes

	Article: Q223047
	Product(s): Microsoft Exchange
	Version(s): ; WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook 97 
	- Microsoft Outlook 98 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The AutoArchive settings that you configure on the Tools menu, by pointing to
	Options, and clicking AutoArchive are for the mailbox found by clicking on the
	Tools menu, pointing to Services, clicking Microsoft Exchange Server, and
	clicking Properties. These settings are not for additional mailboxes specified
	on the Advanced tab.
	
	MORE INFORMATION
	================
	
	To archive additional mailboxes that you may have added to your profile, follow
	these steps:
	
	1. select the mailbox or folder you want to archive, and on the File menu, click
	  Archive.
	
	2. Click the "Archive this folder and all subfolders" radio button and not the
	  "Archive all folders according to their AutoArchive settings" button.
	
	3. Choose the archive path and file for this information to be stored to.
