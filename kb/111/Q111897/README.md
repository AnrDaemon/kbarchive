## Q111897: PC WSPlus: Files Installed During Schedule+ 1.0a Setup

	Article: Q111897
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:1.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 1.0a 
	-------------------------------------------------------------------------------
	
	This article lists the files installed during setup of version 1.0a of
	Schedule+ for Windows.
	
	Files added to the Schedule+ 1.0a directory:
	
	  CBTLIB3.DLL    Computer-based training
	  DEMOFX.DEX     Demo file
	  MSREMIND.EXE   Reminder
	  MSREMIND.WAV   Reminder sound
	  MSSCHED.DLL    Back end, handles reading and writing to files
	  PLAY.EXE       Runs the demo
	  README.TXT     Read me
	  SCHDPLUS.EXE   Schedule+ program file
	  SCHDPLUS.HLP   Schedule+ help file
	  SCHDPLUS.LES   Computer-based training lessions
	  WAVES.DLL      Sounds
	  WIZARD.DLL     Sharp Wizard import/export
	
	Files added to the Windows SYSTEM directory:
	
	  COMMDLG.DLL    Common dialog boxes
	  DEMILAYR.DLL   Memory
	  FRAMEWRK.DLL   User interface
	  MAILMGR.DLL    Authorization
	  MAILSPL.EXE    Spooler
	  MSSFS.DLL      PC Mail transport
	  NVCONN.DLL     For Novell attachments
	  PABNSP.DLL     Personal address book name service provider
	  SHELL.DLL      Schedule+ for Windows shell
	  STORE.DLL      Message store
	  TRNSCHED.DLL   Transport how to find a .CAL file for a user
	
	Files added to the Windows directory (usually WINDOWS):
	
	  SCHDPLUS.INI   Schedule+ initalization file
	
	Additional query words: schedule plus 1.00a
