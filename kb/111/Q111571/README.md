## Q111571: Video Corruption with Diamond Stealth and WFWG VGA Driver

	Article: Q111571
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to use the Microsoft Windows for Workgroups version 3.11
	standard VGA video driver with a Diamond Stealth VRAM 3.33 video adapter, you
	encounter video corruption when you return to Windows from an MS-DOS- based
	application or the MS-DOS Prompt.
	
	WORKAROUND
	==========
	
	To work around this problem use a video driver supplied by Diamond.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows for Workgroups version
	3.11. We are researching this problem and will post new information here as it
	becomes available.
