## Q111261: Flight Simulator: Logbook Displays Hours Incorrectly

{% raw %}

	Article: Q111261
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	kbusage
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for MS-DOS, version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Total Column and Totals Row in the Logbook dialog will not completely
	display numbers larger than 99.9.
	
	MORE INFORMATION
	================
	
	Hours in excess of 99.9 will display only the last two digits and first decimal
	place of the total. For example, if the pilot has flown a total of 120.1 hours
	of day flight, the Totals Row for Day will display 20.1.
