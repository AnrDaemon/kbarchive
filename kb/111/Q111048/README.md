## Q111048: White Lines Appear in Printed Output with Toshiba P1351

{% raw %}

	Article: Q111048
	Product(s): Microsoft Windows Printing Issues
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you print to a Toshiba P1351 printer with the printer driver that ships
	with Microsoft Windows or Windows for Workgroups 3.1 or 3.11, you may receive
	white lines in the printed output.
	
	CAUSE
	=====
	
	Toshiba Technical Support has confirmed that this problem is caused by the
	stepper motor on the printer when the printer is in graphics mode.
	
	MORE INFORMATION
	================
	
	For more information on this issue, please contact Toshiba Technical Support.

	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
