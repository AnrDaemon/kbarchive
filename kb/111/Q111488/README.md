## Q111488: Pipe Dream: Scoring Error in Help File

	Article: Q111488
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-JUN-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows Entertainment Pack, volume 2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Pipe Dream, a game included with volume 2 of the Microsoft Windows Entertainment
	Pack, has an incorrect entry in the Help file.
	
	MORE INFORMATION
	================
	
	1. From the Help menu, click How To Play.
	
	2. Click Scoring. The seventh scoring topic states:
	
	  10,000 points if the goo crosses itself in 5 crosspieces.
	
	The score you will actually receive is 5000 points.
	
	Additional query words: 1.00 wep pipe pipedream docerr
