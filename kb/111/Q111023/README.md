	SUMMARY
	=======
	
	The Microsoft PSS Knowledge Base group regularly extracts the most current
	articles from the Windows database and creates Windows Help files that are then
	posted to the appropriate CIS forum and the Microsoft Download Center on CIS.
	These files are also updated as new information becomes available.
	
	The files listed below are available to download from the Microsoft Download
	Center. For information about downloading these files, please see the
	"Instructions for Downloading" section at the end of this article.
	
	The following file is available for download from the Microsoft Download Center:
	
	  Download Wp695.exe now
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
	
	Microsoft used the most current virus detection software available on the date of
	posting to scan this file for viruses. Once posted, the file is housed on secure
	servers that prevent any unauthorized changes to the file.
