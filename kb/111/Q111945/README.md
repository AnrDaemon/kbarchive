## Q111945: X400: Err Msg: Incoming Invalid SPDU on Connection X:

{% raw %}

	Article: Q111945
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to X.400, version 3.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Version 3.2 of Microsoft Mail Gateway to X.400 may give the following session
	error:
	
	  Incoming invalid SPDU on connection x:
	
	RESOLUTION
	==========
	
	The displayed invalid session Protocol Data Unit(PDU) was received. Check remote
	MTA for the correct Basic Activity Subset(BAS) configuration.
	
	
	Additional query words: 3.20 Errmsg
