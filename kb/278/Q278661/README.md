## Q278661: Possible Set-Up Errors With Web Companion

{% raw %}

	Article: Q278661
	Product(s): The Microsoft Network
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You may see one of these messages when connecting to MSN, the Microsoft Network
	using MSN Companion:
	
	  
	
	- A problem has been detected with the configuration of your e-mail
	
	- You were disconnected while setting up MSN Internet Access
	
	CAUSE
	=====
	
	After configuring the Web Companion for internet access the e-mail may not be
	configured properly.
	
	RESOLUTION
	==========
	
	Call MSN Companion Technical Support at 888-386-6497 for instructions.
