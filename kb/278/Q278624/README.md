## Q278624: How to determine if the MSN Companion unit is connected

	Article: Q278624
	Product(s): The Microsoft Network
	Version(s): 1.0,2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, versions 1.0, 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	On the MSN Companion unit is a Connected Indicator light.
	
	- If the indicator is on, the unit is utilizing the phone line.
	
	- If the indicator blinks, this indicates the modem is downloading information.
	
	- If the indicator is off, the unit is not connected.
