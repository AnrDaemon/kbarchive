## Q278654: MSN Companion Access Numbers

	Article: Q278654
	Product(s): The Microsoft Network
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you first connect to MSN, The Microsoft Network, using the Web Companion,
	MSN Web Companion attempts to chose a primary and backup access number for the
	customer.
	
	MORE INFORMATION
	================
	
	To set up your access numbers 
	-----------------------------
	
	1. Select the state you are in from drop down list.
	
	2. Select the access numbers that correspond to the area where you are located.
	
	3. Repeat this process to select the second access number.
	
	4. Verify that the numbers are local toll free calls. If they are not, click Let
	  me choose other numbers and select numbers that are local.
	
	5. Once the access numbers have been verified as local numbers click continue.
	
	Note: it is the customer's responsibility to verify that calls made are local and
	toll free. MSN, the Microsoft Network and Microsoft Corporation are not
	responsible for and will not be held liable for any telephone fees or charges
	that may accrue from the customer's use of MSN Companion.
	
	To change your access numbers 
	-----------------------------
	
	For instructions on changing access numbers it is recommended that you contact
	MSN Companion Technical Support at (888) 386-6497.
	
	Instructions for changing your access number can also be found in KB article
	Q278481 - MSN Companion FIXIT Pages.
