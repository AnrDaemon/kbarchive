## Q278912: XWEB: Exchange 5.5 OWA Unable to Open an Exchange 2000 Mailbox

	Article: Q278912
	Product(s): Microsoft Exchange
	Version(s): 5.5
	Operating System(s): 
	Keyword(s): kbExchange550sp4Fix
	Last Modified: 13-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you restart an Exchange Server 5.5 computer which has Microsoft Outlook Web
	Access (OWA) installed on it, users may not be able to use that Exchange Server
	computer to log on to Microsoft Exchange 2000 mailboxes unless someone first
	uses that computer to log on to an Exchange Server 5.5 mailbox.
	
	After you log on to an Exchange Server 5.5 mailbox, users can log on to Exchange
	2000 mailboxes.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server
	version 5.5. This problem was first corrected in Exchange Server 5.5 Service
	Pack 4.
