## Q278649: Verify Phone Number in MSN Web Companion

	Article: Q278649
	Product(s): The Microsoft Network
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	MSN Web Companion uses your home phone number in order to configure the settings
	properly and it can be changed at any time.
	
	MORE INFORMATION
	================
	
	Phone Number Set-up
	-------------------
	
	When you first connect to MSN, The Microsoft Network, using the Web Companion,
	MSN Web Companion asks for your home phone number in order to configure the
	settings properly. When asked, type your area code and phone number in the boxes
	provided.
	
	Phone Number Change
	-------------------
	
	To change the phone number on your MSN Companion
	
	1. On the Start page click settings.
	
	2. Click my account and on the my account page verify your phone number.
	
	3. If you need to change your phone number, click change information.
	
	4. In area code box and/or the phone number box delete the old information, type
	  the correct information, and then click done.
