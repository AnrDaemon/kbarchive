## Q278003: CFS2: Slow Frame Rate and Poor Graphics Performance

	Article: Q278003
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Combat Flight Simulator 2, you may experience a slow
	video frame rate and poor graphics performance in the game.
	
	CAUSE
	=====
	
	This behavior can occur if both of the following conditions are true:
	
	- An AMD processor is installed in the computer.
	
	- An Advanced Graphics Port (AGP) video adapter is installed in the computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the AGP miniport driver from the
	following AMD Web site:
	
	  http://www.amd.com/products/cpg/bin
