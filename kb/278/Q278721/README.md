## Q278721: How to change the &quot;helper&quot; on MSN Companion

	Article: Q278721
	Product(s): The Microsoft Network
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The "helper" is an animated character that is tied to the MSN Web Companion help
	features. You can choose either a dog or cat character.
	
	MORE INFORMATION
	================
	
	To change the helper:
	
	1. On the start page click settings.
	
	2. Click helper.
	
	3. Select the character you prefer.
	
	   - Rocky the Dog
	
	   - Links the Cat
	
	4. Click done.
