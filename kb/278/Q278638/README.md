## Q278638: MSN Companion Web page display and toolbar

	Article: Q278638
	Product(s): The Microsoft Network
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The MSN Companion Web page display provides access to the Internet and a toolbar
	that gives the user quick access to the most used browsing functions and
	information about the Web page being viewed.
	
	MORE INFORMATION
	================
	
	When you connect to the Internet using the MSN Companion, you are first taken to
	the Start page.
	
	What you see on the display is the Start page with a toolbar along the bottom.
	
	When you navigate to other Web pages, the page area updates to show the page
	you're browsing (just like in a PC browser) and the toolbar is always visible.
	
	MSN Companion toolbar options:
	------------------------------
	
	- back <A0><A0>Navigate back thru the list of pages the user has viewed.
	
	- start <A0><A0>Takes the user to the Start page.
	
	- mail <A0><A0>Takes the user to email inbox.
	
	- page options <A0><A0>Opens the page options panel.
	
	- Web address and search box
	
	- activity indicators:
	
	   - loading <A0><A0><A0>Appears when a page is being downloaded.
	
	   - security <A0><A0>Appears when the current page is loaded via a secure link.
	
	   - printing <A0><A0><A0>Appears when a page is being printed.
	
	- day and time <A0><A0><A0>Appears when no activity indicator is needed.
