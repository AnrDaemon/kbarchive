## Q164651: Tab Function Does Not Work on 3270 Printer LU Page

	Article: Q164651
	Product(s): Microsoft SNA Server
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft SNA Server, version 3.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	The tab button does not cycle through button options (OK, Cancel, Help) on the
	SNA Manager 3270 Printer LU page.
	
	CAUSE
	=====
	
	This is a problem in the SNA Server 3.0 Manager tool.
	
	RESOLUTION
	==========
	
	An update to SNA Server 3.0 Manager fixes this problem. The updated module is
	<snaroot>\system\snaexp.exe.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in SNA Server, version 3.0. This
	problem was corrected in the latest SNA Server version 3.0 U.S. Service Pack.
	For information on obtaining this Service Pack, query on the following word in
	the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
