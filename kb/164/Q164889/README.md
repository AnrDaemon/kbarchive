## Q164889: SMS: INVWIN32 Won't Check for Upgrade Until Next Inventory

{% raw %}

	Article: Q164889
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.0,1.1,1.2
	Operating System(s): 
	Keyword(s): kbusage kbInventory smsinvkbfixlist
	Last Modified: 25-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server versions 1.0, 1.1, 1.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	At this time, INVWIN32 does not check for the need to upgrade until the next
	inventory cycle set in the Systems Management Server Administrator program user
	interface, under Hardware and Software Inventory Frequency in the Inventory (on
	the Site menu, click Properties and then click Inventory).
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	versions 1.0, 1.1, and 1.2. This problem was corrected in the latest Microsoft
	Systems Management Server version 1.2 U.S. Service Pack. For information on
	obtaining the service pack, query on the following word in the Microsoft
	Knowledge Base (without the spaces):
	
	  S E R V P A C K
	
	Additional query words: prodsms
