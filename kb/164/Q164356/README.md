## Q164356: Mac Encarta 97: Computer Does Not Wake Up From Sleep Mode

{% raw %}

	Article: Q164356
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1997 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 24-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 97 Encyclopedia for Macintosh 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If try to use the Sleep feature on a computer running Mac OS when Encarta is the
	active application, you may not be able to wake the computer back up without
	restarting it.
	
	
	WORKAROUND
	==========
	
	To work around this problems, switch to the finder before you try to use to
	Sleep mode.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Encarta 97 Encyclopedia. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
	
	MORE INFORMATION
	================
	
	Not all computers that run the Mac OS support Sleep mode. If you do not have the
	option, Sleep, under the Special menu, your computer probably does not support
	the Sleep mode.
