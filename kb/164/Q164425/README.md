## Q164425: Cannot Use Keyboard to Set Connection Time in CompuServe Mail

{% raw %}

	Article: Q164425
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You cannot use the keyboard to select the connection time when you are setting
	up the CompuServe Mail service for Microsoft Exchange.
	
	RESOLUTION
	==========
	
	Use a mouse or other pointing device to change the connection times.
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
