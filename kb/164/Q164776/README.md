## Q164776: BUG: Credit Letter.Doc not on Access 97 Step By Step disk

	Article: Q164776
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbother
	Last Modified: 19-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Access 97 Step by Step ISBN 1-57231-316-1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Under "Establish the hyperlink" on page 120, step 2 asks you to select the
	Credit Letter.doc file. This file is not included on the Microsoft Access 97
	Step by Step disk.
	
	The self-extracting archive file, A97sbslt.exe, distributed by Microsoft Press,
	contains Credit Letter.doc and instructions for installing this file to your
	practice directory.
	
	The following file is available for download from the Microsoft Download Center:
	
	  A97SBSLT.EXE
	  (http://download.microsoft.com/download/access97/mspress/2.0/WIN98Me/EN-US/A97SBSLT.EXE)
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
	
	Microsoft used the most current virus detection software available on the date of
	posting to scan this file for viruses. Once posted, the file is housed on secure
	servers that prevent any unauthorized changes to the file.
	
	
	MORE INFORMATION
	================
	
	The following installation instructions are copied from the README.TXT file
	included in A97SBSLT.EXE
	
	***** BEGIN README.TXT EXCERPT *****
	
	INSTALLATION INSTRUCTIONS
	-------------------------
	1. Download A97SBSLT.EXE. 
	
	2. Copy or move A97SBSLT.EXE into the directory where the
	  Microsoft Access 97 Step by Step practice files are located.
	  By default, this directory is C:\Access SBS Practice.
	
	3. Double-click (or run) A97SBSLT.EXE in the Microsoft Access
	  97 Step by Step practice directory to expand README.TXT
	  (this file) and the Microsoft Word file.
	
	4. Delete A97SBSLT.EXE.
	
	If you encounter difficulties with these files, please send
	e-mail to Microsoft Press at:
	
	  MSPINPUT@MICROSOFT.COM
	
	Please include a problem description, the steps you have tried to
	correct the problem, and any error messages that you have encountered.
	
	Sincerely,
	Microsoft Press Technical Support
