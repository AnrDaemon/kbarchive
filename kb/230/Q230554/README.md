## Q230554: Event ID 3113 When Using Novell IntraNetWare 4.1a Client

{% raw %}

	Article: Q230554
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0,4.0 SP1,4.0 SP2,4.0 SP3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-JUN-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation versions 4.0, 4.0 SP1, 4.0 SP2, 4.0 SP3 
	- Microsoft Windows NT Server versions 4.0, 4.0 SP1, 4.0 SP2, 4.0 SP3 
	- Microsoft Windows NT Server, Enterprise Edition version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you install Novell IntraNetWare 4.1a client software on a computer running
	Windows NT 4.0 Service Pack 3, the Workstation service may not start and the
	following error message may be displayed in Event Viewer:
	
	  Initialization failed because the requested service could not be started.
	
	RESOLUTION
	==========
	
	To work around this issue, install the most recent version of the Novell
	IntraNetWare client software.
