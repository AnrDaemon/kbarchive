## Q230211: Tdpipe.sys Symbols Are Missing in Terminal Server Service Pack 4

	Article: Q230211
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0,4.0 SP4
	Operating System(s): 
	Keyword(s): kbProgramming
	Last Modified: 11-DEC-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server versions 4.0, 4.0 SP4, Terminal Server Edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Service Pack 4 version of the following drivers are missing from the symbols
	tree of Microsoft Windows NT Server 4.0, Terminal Edition Service Pack 4:
	
	- Tdasync.dbg
	
	- Tdipx.dbg
	
	- Tdnetb.dbg
	
	- Tdpipe.dbg
	
	- Tdspx.dbg
	
	- Tdtcp.dbg
	
	MORE INFORMATION
	================
	
	These symbols will be included in Windows NT Server 4.0, Terminal Server Edition
	Service Pack 5.
