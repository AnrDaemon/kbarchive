## Q230701: Security Certificates Appear Enlarged and Unusable with AVS

{% raw %}

	Article: Q230701
	Product(s): Microsoft Windows NT
	Version(s): WINDOWS:4.01,4.01a; winnt:4.0 SP4
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 SP4 
	- Microsoft Windows NT Workstation version 4.0 SP4 
	- Microsoft Windows NT Server, Enterprise Edition version 4.0 SP4 
	- Microsoft Internet Explorer Administration Kit versions 4.01, 4.01a 
	-------------------------------------------------------------------------------
	
	NOTE: The information in this article applies to the 
	Finnish version(s) of Windows NT 4.0 Service Pack 4.
	
	SYMPTOMS
	========
	
	When you run the Automatic Version Synchronization (AVS) feature in the
	Microsoft Internet Explorer Administration Kit (IEAK) Wizard installed on a
	computer running Windows NT 4.0 Service Pack 4, the security certificates
	associated with the Internet Explorer cab files appear enlarged and unusable.
	
	RESOLUTION
	==========
	
	A supported fix is now available from Microsoft, but it is only intended to
	correct the problem described in this article and should be applied only to
	systems experiencing this specific problem. This fix may receive additional
	testing at a later time, to further ensure product quality. Therefore, if you
	are not severely affected by this problem, Microsoft recommends that you wait
	for the next Windows NT 4.0 service pack that contains this fix.
	
	To resolve this problem immediately, contact Microsoft Product Support Services
	to obtain the fix. For a complete list of Microsoft Product Support Services
	phone numbers and information about support costs, please go to the following
	address on the World Wide Web:
	
	  http://support.microsoft.com/default.aspx?scid=fh;EN-US;CNTACTMS
	
	NOTE: In special cases, charges that are normally incurred for support calls may
	be canceled, if a Microsoft Support Professional determines that a specific
	update will resolve your problem. Normal support costs will apply to additional
	support questions and issues that do not qualify for the specific update in
	question.
	
	The Finnish version of this fix should have the following file attributes or
	later:
	
	  Date      Time     Size      File name     Platform
	  ----------------------------------------------------
	  04/20/99  05:03p   472,336   Cryptui.dll   x86
	
	This hotfix has been posted to the following Internet location as Ieakfxi.exe
	(x86):
	
	  ftp://ftp.microsoft.com/bussys/winnt/winnt-public/fixes/fin/nt40/hotfixes-postsp4/ieak-fix/
	
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Finnish version of Windows
	NT 4.0.
