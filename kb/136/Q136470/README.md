## Q136470: HP Vectra 486/50 Computer Hangs After Choosing to Restart

	Article: Q136470
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 19-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are using a Hewlett-Packard Vectra 486/50 computer, the computer may
	stop responding (hang) when you choose the Restart The Computer option in the
	Shut Down Windows dialog box.
	
	CAUSE
	=====
	
	This problem occurs because of a hardware cache problem. An address is being
	overwritten.
	
	RESOLUTION
	==========
	
	To resolve this problem, disable video shadowing in the computer's CMOS
	settings. For information about how to do so, please consult your computer
	documentation or manufacturer.
