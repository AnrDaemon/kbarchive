## Q136179: PPT7: You Can Add Only One Textured Background Format at a Time

	Article: Q136179
	Product(s): Microsoft PowerPoint for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 15-APR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft PowerPoint for Windows 95, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Microsoft PowerPoint 97, you can add only one textured background format at a
	time to a slide. (To do this, click Background on the Format menu and then click
	Fill Effects under Background Fill.) If you select one new texture, and then
	immediately try to add another one, the new one replaces the first one.
	
	CAUSE
	=====
	
	Although it may be unexpected, this behavior is by design.
