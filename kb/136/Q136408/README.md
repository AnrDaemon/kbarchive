## Q136408: MSN: BBS May Hang Opening Several Messages Simultaneously

	Article: Q136408
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.2,1.3
	Operating System(s): 
	Keyword(s): kbmsn
	Last Modified: 14-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.2, 1.3 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you select several messages in a bulletin board (BBS) and then click Open,
	the BBS may stop responding (hang) while opening the messages.
	
	RESOLUTION
	==========
	
	Press CTRL+ALT+DELETE, and then click End Task. When you are prompted to
	disconnect from MSN, The Microsoft Network, click No.
