## Q136585: PRB: Import Wizard Doesn't Automatically Browse Imported Table

	Article: Q136585
	Product(s): Microsoft FoxPro
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-FEB-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual FoxPro for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Import Wizard imports data into Visual FoxPro tables from other file
	formats. By design, it does not leave the newly created table open.
	
	RESOLUTION
	==========
	
	To browse the table, you need to open it as you would any other table.
	
	STATUS
	======
	
	This behavior is by design.
