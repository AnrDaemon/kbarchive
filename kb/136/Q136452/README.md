## Q136452: Context-Sensitive Help Appears Only Briefly After Pressing F1

	Article: Q136452
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95 appscomp kbAppCompatibility
	Last Modified: 28-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you press the F1 key in a Paint or WordPad dialog box to receive
	context-sensitive Help, the context-sensitive Help may appear only briefly
	before being replaced by the Windows 95 online Help for Paint or WordPad.
	
	RESOLUTION
	==========
	
	You may be able to work around this problem by clicking the question mark (?)
	button at the top of the dialog box, and then clicking the item you want to see
	Help about.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Windows 95. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
