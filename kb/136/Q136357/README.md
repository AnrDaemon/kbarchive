## Q136357: Fury3: Help Text Hard to Read in Certain Color Schemes

{% raw %}

	Article: Q136357
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	1.00
	WINDOWS
	kbenv kbprb
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Fury 3 for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Fury Help text may not display well under some Windows 3.x color schemes.
	
	For example, if you change the Windows color scheme so that Windows Text is
	displayed in white, this white text does not show up well against the yellow
	default background color of Help.
	
	MORE INFORMATION
	================
	
	Some of the Windows color schemes that display Windows text as white are:
	
	  Hot Dog Stand
	  LCD reverse Dark
	  Plasma Power Saver
