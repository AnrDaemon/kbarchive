## Q136430: Total Disk Space on Banyan Vines Server Incorrectly Reported

	Article: Q136430
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0; WINDOWS:3.11,95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 6.0 
	- Microsoft Windows for Workgroups version 3.11 
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	MS-DOS programs running on a Banyan Vines Client report the wrong total
	free disk space on Banyan file servers with large disks when free space on
	those disks is small. The reported amount is often less than is actually
	available.
	
	More information can be found in Banyan Technotes Bulletin number 288. This
	technote may be downloaded from the BANYAN forum on CompuServe or contact
	Banyan Vines technical support at (800) 222-6926.
