## Q136007: Error Message: Host Disconnected Due to Idle Timeout

{% raw %}

	Article: Q136007
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are using the CompuServe provider for Microsoft Exchange, your
	connection may be terminated approximately 15 minutes after you start the
	session. When this occurs, you receive the following error message:
	
	  Host disconnected due to idle timeout
	
	CAUSE
	=====
	
	CompuServe disconnects you after 15 minutes of inactivity.
	
	RESOLUTION
	==========
	
	To avoid being disconnected, do not allow 15 minutes to pass with no activity.
