## Q136685: Golf 2.0a Registry Entries made by Setup

	Article: Q136685
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-JUN-2001
	
	2.0a
	Windows 95
	KBENV
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Golf for Windows, version 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following is a list of the registry entries made by setup.
	
	HKEY_LOCAL_MACHINE
	SOFTWARE
	Microsoft
	MS Setup (ACME)
	Table Files
	MSGOLF.EXE@v2.0.0.0000  "C:\Program Files\Microsoft Sports\Golf
	2.0\Setup\setup.stf"
	
	HKEY_LOCAL_MACHINE
	SOFTWARE
	Microsoft
	Windows
	CurrentVersion
	Uninstall
	MSGOLF
	DisplayName "Microsoft Golf 2.0"
	UninstallString  "C:\Program Files\Microsoft Sports\Golf
	2.0\Setup\setup.exe /m"
	
	For this article, the default directories for Golf are used.
	
	Additional query words: 2.00 kbhowto Golf Registry Entries Windows95
