## Q136945: Location of Foxtools.fll Information

	Article: Q136945
	Product(s): Microsoft FoxPro
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-FEB-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual FoxPro for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Information about the FOXTOOLS API library is included in the Foxtools.hlp Help
	file located in the Vfp\ToolS directory.
	
	NOTE: The FOXTOOLS functions are not supported by Microsoft Technical Support
	either electronically or by telephone. Many of these functions are included for
	backward compatibility with FoxPro version 2.6. Some functions behave
	differently depending on the platform for which they are used.
	
	Additional query words: VFoxWin foxtools.fll foxtools.dll on-line documentation
