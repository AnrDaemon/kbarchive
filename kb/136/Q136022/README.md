## Q136022: Mac Bookshelf 95: Israel Anthem Suddenly Stops

	Article: Q136022
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1995 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for the Macintosh 1995 edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Israel Anthem in Bookshelf stops when the anthem nears the end of the audio
	clip.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the 1995 edition of Bookshelf
	Macintosh. We are researching this problem and will post new information here in
	the Microsoft Knowledge Base as it becomes available.
	
	MORE INFORMATION
	================
	
	There is no resolution to this problem. Although the sudden stop occurs, the
	problem does not cause any problem with the system.
