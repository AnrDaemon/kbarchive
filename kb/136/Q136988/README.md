## Q136988: Bookshelf 1995: Some Animations Do Not Have Audio

	Article: Q136988
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1995 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for Windows 1995 edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Some of the animations in the Encyclopedia's Gallery Animation & Video
	section are missing audio.
	
	The files missing audio are:
	
	- Chromosome(first animation)
	
	- Meiosis
	
	- Solar system
	
	
	NOTE: The Meiosis animation is the first animation on the Chromosome screen.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the 1995 edition of Bookshelf
	for Windows. This problem has been resolved in Microsoft Bookshelf '96-'97 for
	Windows.
	
	MORE INFORMATION
	================
	
	Animations that contain audio have a sound icon (a picture of a speaker). The
	above animations contain the speaker icon, but no audio is heard during
	playback.
	
	Additional query words: 1995 multi media multimedia multi-media mmtitles quiet missing nothing silence avi .avi
