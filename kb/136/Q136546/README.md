## Q136546: Help Desk Options Incorrectly Displays Error Message

	Article: Q136546
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.0,1.1
	Operating System(s): 
	Keyword(s): kbnetwork kbtshoot smsremtshoot kbRemoteProg
	Last Modified: 31-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server versions 1.0, 1.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	
	If you run Help Desk Options on a computer that does not have USERTSR or USERIPX
	loaded, the following incomplete error appears:
	
	  Current setting not available. Remote control agent
	
	The error should display:
	
	  Current setting not available. Remote control agent not loaded.
	
	Microsoft has confirmed this to be a problem in Systems Management Server version
	1.0 and 1.1. This problem was corrected in the latest US Service Pack for
	Systems Management Server. For information on obtaining this update, query on
	the following word in the Microsoft Knowledge Base:
	
	  " SERVPACK " (without the quotation marks)
	
	Additional query words: prodsms sms help desk remote
