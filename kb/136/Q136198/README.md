## Q136198: Problems Connecting NT to UNIX Based Portable NetWare Server

	Article: Q136198
	Product(s): Microsoft Windows NT
	Version(s): 3.5 3.51 4.0
	Operating System(s): 
	Keyword(s): kbnetwork
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation versions 3.5, 3.51, 4.0 
	- Microsoft Windows NT Server versions 3.5, 3.51, 4.0 
	-------------------------------------------------------------------------------
	
	Windows NT cannot connect to a Portable NetWare Server.
	
	There is no support for Windows NT connectivity to Portable NetWare servers
	because they are not "native" NetWare servers. Portable NetWare is NetWare
	emulation on a UNIX machine.
