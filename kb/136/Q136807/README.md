## Q136807: XADM: MTA Error: Too Many ECBs Processed In One Thread

	Article: Q136807
	Product(s): Microsoft Exchange
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 13-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If multiple Microsoft Exchange Internet Mail Connectors (IMCs) are configured in
	an organization, the following error message may appear in the Windows NT
	Application Event Log:
	
	  MSExchangeMTA event id 248
	  Too many ECBs processed in one thread.
	
	In addition, mail may stop processing and may be returned in Non-Delivery
	reports.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server
	version 4.0. This problem was corrected in Microsoft Exchange Server 4.0 U.S.
	Service Pack 2. For information on obtaining the service pack, query on the
	following word in the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
