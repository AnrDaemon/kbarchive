## Q136923: Purpose of Edit User Defined Properties and Methods Dialog Box

	Article: Q136923
	Product(s): Microsoft FoxPro
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-FEB-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual FoxPro for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes the purpose of the "Edit user-defined properties/ method"
	dialog box in the Form Designer.
	
	MORE INFORMATION
	================
	
	You can add properties and methods to a form in the Form Designer by clicking
	New Property or New Method on the Form menu. The Edit Property/Method dialog box
	allows you to remove these properties and methods at design time. This dialog
	box also lets users modify the description of the property or method that
	appears at the bottom of the property sheet in the Designer.
