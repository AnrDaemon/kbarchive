## Q136353: Fury3: Flashing screen during AVI sequence with Boca SuperX

{% raw %}

	Article: Q136353
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	1.00
	WINDOWS
	kbdisplay
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Fury 3 for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you are using the Boca SuperX PCI video card, you may notice flashing or
	other screen corruption with some of the AVIs that contain text.
	
	MORE INFORMATION
	================
	
	Microsoft has confirmed this to be a problem in Fury3 version 1.00.
