## Q136644: Icon Spacing Not Updated Using Large Icons in Microsoft Plus!

	Article: Q136644
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	- Microsoft Plus! for Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you select large icons on the Plus! tab in Display properties, the spacing
	of the icons on the desktop may not change, causing the icons to overlap.
	
	CAUSE
	=====
	
	The icons are not automatically rearranged unless the Auto Arrange option is
	selected for the desktop.
	
	RESOLUTION
	==========
	
	To arrange the icons automatically, use the right mouse button to click an empty
	area on the desktop, point to Arrange Icons on the menu that appears, and then
	click Auto Arrange.
