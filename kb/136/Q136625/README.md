## Q136625: PC Mac: Creating a Personal User List and Personal Group

{% raw %}

	Article: Q136625
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.x
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, version 3.x, on platform(s):
	   - the operating system: Mac OS (ALL) 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	You can create personal groups in the Macintosh Mail client for PC Networks.
	
	MORE INFORMATION
	================
	
	You must have personal user lists before you can create personal groups. To
	create a personal user list, from the Special menu, choose the Personal Users
	option.
	
	NOTE: When you view or edit the contents of personal groups, it can be awkward.
	As a result, you may want to create a personal list that includes the names you
	want in a personal group, and then create the group.
