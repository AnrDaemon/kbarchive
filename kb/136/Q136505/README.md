## Q136505: Fury3: Mute doesn't function during AVI sequences

	Article: Q136505
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbbuglist
	Last Modified: 08-NOV-2001
	
	1.00
	WINDOWS
	kbsound kbbuglist kbprb
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Fury 3 for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play a video (.avi) sequence in Fury3, sound plays even when you select
	the muting option.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Fury3 version 1.0 for Windows.
	
	Additional query words: fury 3 fury-3 1.00 multimedia multi-media multi media loud continuous blaring muted mute
