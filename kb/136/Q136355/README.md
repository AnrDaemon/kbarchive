## Q136355: Fury3: Mission 1 of Fury Shows 0 Ground Targets Destroyed

	Article: Q136355
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbbuglist
	Last Modified: 08-NOV-2001
	
	1.00
	WINDOWS
	kbother kbprb kbbuglist
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Fury 3 for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The mission debriefing screen in mission 1 of the Fury world always displays
	that zero percent of the ground targets have been destroyed.
	
	CAUSE
	=====
	
	The only ground targets on mission 1 of Fury are hovering gunships. Although
	these ships are truly airborne targets, they are displayed as green ground
	targets on the maps.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Fury3 for Windows,
	version 1.0.
