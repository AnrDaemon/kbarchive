## Q172015: FS95: Color and Markings (Aircraft Texture) Missing on Aircraft

{% raw %}

	Article: Q172015
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbusage kbhowto
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for Windows 95, version 1.0 
	- Microsoft Expansion Pack for Microsoft Flight Simulator for Windows 95, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The colors and markings (Aircraft Texture) on the default Aircraft in Flight
	Simulator for Windows 95 (FS95) can be turned off or on, as desired.
	
	To turn the markings off or on, do the following:
	
	1. From the Options menu, click Preferences.
	
	2. Click the Display And Sound tab.
	
	3. Click Display Options...
	
	4. From the Aircraft tab, you can turn the Aircraft Texture on or off as
	  desired.
	
	MORE INFORMATION
	================
	
	3rd Party aircraft may require additional files to activate aircraft markings.
	These Aircraft may need to be run through the FS95 Aircraft Converter. The
	converter is available on the World Wide Web at
	
	  http://www.microsoft.con/games/fsim.
