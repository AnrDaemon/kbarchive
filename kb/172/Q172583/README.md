## Q172583: WordArt Preview Is Blank After Installing Service Pack 5

	Article: Q172583
	Product(s): Microsoft Windows NT
	Version(s): winnt:3.51
	Operating System(s): 
	Keyword(s): kbtool
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 3.51 
	- Microsoft Windows NT Workstation version 3.51 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you insert a WordArt object into a document, the WordArt preview may appear
	blank.
	
	CAUSE
	=====
	
	This behavior occurs if you have installed Service Pack 5 in Windows NT 3.51.
