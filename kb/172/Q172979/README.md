## Q172979: Modem Status Icon Does Not Appear on Taskbar

{% raw %}

	Article: Q172979
	Product(s): The Microsoft Network
	Version(s): WINDOWS:2.5
	Operating System(s): 
	Keyword(s): kbenv kbmsn
	Last Modified: 09-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network version 2.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you connect to MSN, The Microsoft Network, the modem status icon may not
	appear on the taskbar.
	
	CAUSE
	=====
	
	The MSN Dial-Up Networking connection may not be configured to display the modem
	status.
	
	RESOLUTION
	==========
	
	To resolve this issue, use the following steps:
	
	1. Double-click My Computer.
	
	2. Double-click Dial-Up Networking.
	
	3. Right-click the MSN connection, and then click Properties.
	
	4. Click Configure.
	
	5. On the Options tab, under Status Control, click the Display Modem Status
	  check box to select it.
	
	6. Click OK, and then click OK again.
	
	Additional query words: msnet msnetwork systray system tray
