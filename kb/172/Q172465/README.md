## Q172465: Bookshelf 98: &quot;Licensed To&quot; Field in About Box Is Empty

{% raw %}

	Article: Q172465
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbui kbimukbfaq
	Last Modified: 08-APR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf 98 for Windows 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In Microsoft Bookshelf 98, if you click About on the Help menu, the field under
	the text "This product is licensed to:" may be empty.
	
	MORE INFORMATION
	================
	
	Bookshelf 98 does not prompt you to enter your user name or company name during
	installation. Instead, this information is obtained from a key in the Registry.
	If this Registry key is empty or does not exist, the field under the text "This
	product is licensed to:" is left empty.
