## Q172642: Media Player Hangs When Window Is Resized

{% raw %}

	Article: Q172642
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): Win2000:95
	Operating System(s): 
	Keyword(s): kberrmsg kbtool win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are using Media Player to play an .mpg file, if you resize the Media
	Player window when the file begins to be rewound automatically, Media Player may
	seem to stop responding (hang), or you may receive the following error message:
	
	  MCI Initialization Error
	
	RESOLUTION
	==========
	
	Use the Close Program dialog box to quit Media Player. To do so, follow these
	steps:
	
	1. Press CTRL+ALT+DELETE to open the Close Program dialog box.
	
	2. In the list of programs, click Media Player.
	
	3. Click End Task.
	
	If the computer hangs after you click End Task, turn the computer off and back on
	to restart Windows 95.
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
	
	Additional query words: mpeg repeat wmptop
