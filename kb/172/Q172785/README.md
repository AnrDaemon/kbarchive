## Q172785: Internet Icon Missing from Desktop After Installing MSN

	Article: Q172785
	Product(s): The Microsoft Network
	Version(s): WINDOWS:2.0
	Operating System(s): 
	Keyword(s): kbui
	Last Modified: 12-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you install MSN, The Microsoft Network, the The Internet icon may no
	longer appear on your desktop.
	
	CAUSE
	=====
	
	This behavior can occur if the registry is damaged.
	
	RESOLUTION
	==========
	
	To resolve this issue, reinstall the latest version of Microsoft Internet
	Explorer.
