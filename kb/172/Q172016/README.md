## Q172016: WD97: Some Text Attributes Lost Inserting Corel Graphic (.cdr)

	Article: Q172016
	Product(s): Word 97 for Windows
	Version(s): 
	Operating System(s): 
	Keyword(s): kb3rdparty
	Last Modified: 08-FEB-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you insert a CDR Graphic (Corel Draw, version 5 or 6) into a Microsoft Word
	97 document, some font attributes such as double underline and double overline
	are lost. In some cases superscript or subscript text may also be displayed
	incorrectly.
	
	WORKAROUND
	==========
	
	To preserve the font attributes in the document, follow these steps:
	
	1. In Corel Draw, use the Copy command to copy the graphic.
	
	2. In Microsoft Word, on the Edit menu, click Paste.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
