## Q172429: Mac Bookshelf 98: System Hangs When Adjusting Slider Bar

	Article: Q172429
	Product(s): Microsoft Home Multimedia Titles
	Version(s): MACINTOSH:
	Operating System(s): 
	Keyword(s): kbmm kbsoundkbfaq
	Last Modified: 05-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf 98 for Macintosh 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play an audio file from the Dictionary of Microsoft Bookshelf 98, your
	computer may stop responding (hang), or you may receive an error type 2 or error
	type 11.
	
	This occurs if you move the slider bar after the audio file begins playing.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Bookshelf 98 for the
	Macintosh.
	
	Additional query words: multi media multi-media mm sounds wav
