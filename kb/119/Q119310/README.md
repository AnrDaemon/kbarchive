## Q119310: FAX: Cover Sheet Size Requirements in Inches

{% raw %}

	Article: Q119310
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to Fax, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Page 47 of the "Microsoft Mail Gateway to FAX Administrator's Guide," version
	3.0, lists the maximum coversheet size as 1712 pels wide by 800 pels high. This
	equals 17.83 inches wide by 8.33 inches high.
