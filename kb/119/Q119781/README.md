## Q119781: Err Msg: There Is a Problem Loading Unicode Tables

	Article: Q119781
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start Windows or Windows for Workgroups while using Novell NetWare
	version 4.x and VLMS, you receive one of the following error messages:
	
	  NWDRV-3.00-30: there is a problem loading unicode tables.
	
	  -or-
	
	  We cannot access our mapped Novell Drives or our Novell Print queues.
	
	CAUSE
	=====
	
	If the Novell unicode tables are not found or are not accessible, the Directory
	Services cannot load. The Directory Services is a database that supplies drive
	mapping and printer queue information to the NETWARE.DRV file.
	
	RESOLUTION
	==========
	
	For information about correcting this problem, contact Novell or an authorized
	Novell reseller.
	
	Additional query words: 3.10 3.1 3.11 uni-code
