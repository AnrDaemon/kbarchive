	SUMMARY
	=======
	
	Fw1040.exe is a file that contains the Application Note entitled "Database
	Design (Relational vs. Flat-File)," which describes the four major areas of data
	processing that you should take into consideration when designing a relational
	database management system (RDBMS).

	INTRODUCTION TO FW1040
	
	Before creating a relational database management system (RDBMS) application using
	FoxPro, you should consider some logical design points. These design
	considerations involve four major areas of data processing:
	
	- Purpose
	
	- Input or data entry
	
	- Queries
	
	- Output or reports
