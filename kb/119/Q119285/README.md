## Q119285: APPICON.PCT Does Not Exist

{% raw %}

	Article: Q119285
	Product(s): Microsoft FoxPro
	Version(s): 2.5b,2.5c
	Operating System(s): 
	Keyword(s): 
	Last Modified: 04-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft FoxPro Distribution Kit for Macintosh, versions 2.5b, 2.5c 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Page 14 of the "User's Guide" for the FoxPro for Macintosh Distribution Kit
	incorrectly refers to a file called APPICON.PCT. This file does not exist.
	
	Additional query words: FoxMac 2.50b 2.50c 2.60a PICT dk
