## Q119558: PC Win: Hyphens in Columns 1 and 78 Replaced by Space

	Article: Q119558
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Using version 3.2 of the Microsoft Mail for Windows client, if you compose and
	send a message that contains hyphens in the body of the text, the hyphens in
	columns 1 and 78 will each be replaced by a space when the message is received.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in version 3.2 of Microsoft Mail
	for Windows. We are researching this problem and will post new information here
	in the Microsoft Knowledge Base as it becomes available.
