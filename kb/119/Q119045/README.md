## Q119045: PC Mac: Err Msg: This Message is Corrupt and Cannot Be...

	Article: Q119045
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, version 3.0, on platform(s):
	   - the operating system: Mac OS (ALL) 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When using version 3.0 of Microsoft Mail for PC Networks, Macintosh workstation
	and attempting to read a mail message sent from Microsoft Mail for Windows NT
	(32-bit Mail client), the following error may occur:
	
	  This message is corrupt and cannot be accessed. Contact your System
	  Administrator.
	
	NOTE: The Microsoft Mail for PC Networks, MS-DOS workstation does not have a
	problem viewing the message.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in version 3.0 and later of
	Microsoft Mail for PC Networks, Macintosh workstation. We are researching this
	problem and will post new information here in the Microsoft Knowledge Base as it
	becomes available.
