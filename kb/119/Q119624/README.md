## Q119624: FAX: Unable to Disable Cover Page and Distribution Box

	Article: Q119624
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to Fax, versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When faxing documents or items through version 3.0 and 3.0a of the Microsoft
	Mail Gateway to FAX, you cannot disable the cover page and distribution box that
	follows. This is by design: the gateway provides no direct way to disable them.
