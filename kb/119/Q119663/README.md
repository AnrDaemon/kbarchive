## Q119663: Mac Bookshelf 94, 95, 96-97: Fragmented Audio in Videos

	Article: Q119663
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1994, 1995, 96-97 editions
	Operating System(s): 
	Keyword(s): kbbuglist
	Last Modified: 11-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for the Macintosh 1994, 1995, 96-97 editions 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The following animations contain mild audio fragmentation:
	
	- Phase
	
	- Plate Tectonics
	
	- Solar System
	
	- Continental Drift
	
	- Jet Propulsion
	
	- Nuclear Reactor
	
	NOTE: On Apple Power Macintosh computers, the audio is greatly improved by using
	QuickTime 2.0 and higher.
	
	MORE INFORMATION
	================
	
	Microsoft has confirmed this to be a problem in Bookshelf for the Macintosh,
	1994, 1995, 1996-1997 editions. We are researching this problem and will post
	new information here in the Microsoft Knowledge Base as it becomes available.
