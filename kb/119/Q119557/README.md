## Q119557: PC Win: Err Msg: Bad Windows Class, Ignored

	Article: Q119557
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you choose Index from the Help menu in version 3.2 of Microsoft Mail for
	Windows, the following message may be displayed:
	
	  Bad Window Class, ignored
	
	CAUSE
	=====
	
	This error occurs when the MSMAIL.HLP file that is provided with Windows for
	Workgroups mail is used with version 3.2 of Microsoft Mail for Windows.
	
	RESOLUTION
	==========
	
	Copy the MSMAIL.HLP file that is provided with version 3.2 of Microsoft Mail for
	Windows to your MSMAIL executables directory.
