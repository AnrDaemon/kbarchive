## Q119527: PC Win: Err Msg: Undeliverable Mail

	Article: Q119527
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 31-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you send mail from the Windows NT Mail client to another Microsoft Mail
	postoffice using a custom address with lowercase characters in either the
	network or postoffice name, you receive the following error message:
	
	  Undeliverable Mail
	
	This applies to build 528 and earlier versions of Windows NT.
	
	RESOLUTION
	==========
	
	Type the network name and postoffice name in uppercase characters.
