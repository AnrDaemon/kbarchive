## Q95167: XGraphic Windows Accelerator Doesn't Work with SUPERVGA.DRV

{% raw %}

	Article: Q95167
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	- Microsoft Windows for Workgroups version 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	IIT's technical support has confirmed that IIT's XGraphic video card does not
	work with the Super VGA display driver that ships with Microsoft Windows. The
	XGraphic card does work with the Windows VGA driver. For the card to function at
	higher resolutions, you must use drivers provided by IIT.
	
	MORE INFORMATION
	================
	
	For additional information, contact Integrated Information Technology, Inc.
	(IIT) technical support.
	
	XGraphic is manufactured by a vendor independent of Microsoft; we make no
	warranty, implied or otherwise, regarding this product's performance or
	reliability.
