## Q95297: Orchid Fahrenheit Drivers Cause GP Faults

{% raw %}

	Article: Q95297
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you use the Orchid Fahrenheit 1280 VGA card with Orchid's video drivers dated
	8/10/92, the following problems may occur:
	
	- Random general protection (GP) faults
	
	- Garbled output when exiting Windows
	
	- System stops (hangs) when entering Windows
	
	RESOLUTION
	==========
	
	Orchid has confirmed that these drivers may cause problems with all versions of
	Windows. They recommend that you upgrade to the latest versions of their BIOS
	and driver, which are available from Orchid's bulletin board service (BBS).
	
	The new BIOS is version 3.0 and the corresponding driver is version 7.0.
