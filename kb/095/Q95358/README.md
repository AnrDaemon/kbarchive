## Q95358: Font Pack 2 Incorrectly States WIN.INI Backup Filename

	Article: Q95358
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	Page 3 of the "Microsoft TrueType Font Pack 2 User's Guide"
	incorrectly states the name of the backed up WIN.INI file.
	
	The following statement is incorrect:
	
	  Before automatically updating your WIN.INI file, the Setup program saves a
	  copy of your WIN.INI file and renames that copy WIN.OLD.
	
	The statement should read:
	
	  Before automatically updating your WIN.INI file, the Setup program saves a
	  copy of your WIN.INI file and renames that copy WIN.FP2
	
	Additional query words: 3.10 true type tt ttf documentation error fontpack doc err 3.11
