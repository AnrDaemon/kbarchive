## Q95721: Fox Mouse Support Numbers

{% raw %}

	Article: Q95721
	Product(s): Microsoft FoxPro
	Version(s): MS-DOS:2.0,2.5,2.5a; WINDOWS:2.5,2.5a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 03-FEB-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, 2.5a 
	- Microsoft FoxPro for Windows, versions 2.5, 2.5a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Fox Software marketed two different types of mice. One was a Dexxa mouse that
	did not have a Fox logo on it. The other mouse is a Logitech mouse; this mouse
	has a Fox logo on it.
	
	Hardware support issues for these mice are to be directed to Logitech Technical
	Support at (510) 795-0427. (The Dexxa mouse is manufactured and supported by
	Logitech.)
	
	The mice included here are manufactured by a vendor independent of Microsoft; we
	make no warranty, implied or otherwise, regarding these products' performance or
	reliability.
