## Q95029: CSS Laboratories 386-20 May Require BIOS Upgrade

{% raw %}

	Article: Q95029
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 25-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Microsoft Windows stops responding (hangs) in 386 enhanced mode on your CSS
	Laboratories 386/20 system.
	
	CAUSE
	=====
	
	According to CSS Laboratories, if you have a CSS Laboratories 386-20 with an
	Award BIOS version earlier than 3.11 or a keyboard BIOS earlier than version
	3.5, Windows may hang in enhanced mode.
	
	RESOLUTION
	==========
	
	To solve the problem, you need to obtain a BIOS upgrade from CSS Laboratories.
	
	MORE INFORMATION
	================
	
	For additional information, contact CSS Laboratories.
	
	The CSS Laboratories product included here is manufactured by a vendor
	independent of Microsoft; we make no warranty, implied or otherwise, regarding
	this product's performance or reliability.
