## Q95349: Cannot Open More Than One File at a Time with EXPAND

{% raw %}

	Article: Q95349
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	MS-DOS Help contains a phrase that may be unclear. The help file states that the
	filename parameter:
	
	  Specifies the location and name of a compressed file or files
	
	The EXPAND command does not support wildcards (such as * and ?) and cannot expand
	more than one file at a time.
	
	Additional query words: 6.22 6.00 6.20
