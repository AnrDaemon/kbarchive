## Q168338: WD97: Mouse Causes Equation Editor Toolbar to Disappear

	Article: Q168338
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:1.0,1.1,2.0,97
	Operating System(s): 
	Keyword(s): kbui word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	- Microsoft IntelliPoint software, versions 1.0, 1.1, 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you move the insertion point over the title bar of the Equation Editor
	toolbar, the toolbar may disappear.
	
	NOTE: The Equation Editor toolbar must be floating (not docked) for this problem
	to occur.
	
	CAUSE
	=====
	
	The Equation Editor toolbar will disappear when the Microsoft IntelliMouse
	pointer is set to "Select a window just by pointing at the title bar."
	
	WORKAROUND
	==========
	
	Method 1: Turn Off the IntelliMouse StepSaver Focus Option
	----------------------------------------------------------
	
	1. Click the Start button, point to Settings, and then click Control Panel.
	
	2. Double-click the Mouse icon in Control Panel.
	
	3. In the Mouse Properties dialog box, click the StepSavers tab.
	
	4. Click to clear the "Select a window just by pointing at the title bar" check
	  box, and then click OK.
	
	Method 2: Redisplay the Toolbar
	-------------------------------
	
	On the View menu, click Toolbar to show the toolbar again.
	
	NOTE: This is not a permanent change. The Equation Editor toolbar will close
	anytime that you move the Microsoft IntelliMouse pointer over the title bar of
	the Equation Editor toolbar.
	
	Method 3: Minimize Microsoft Word
	---------------------------------
	
	1. Click the Minimize button in the upper-right corner of Word to minimize the
	  window.
	
	2. Click Microsoft Word on the taskbar.
	
	The Equation Editor toolbar will be displayed again.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article. This problem has been corrected in Microsoft
	IntelliPoint version 2.2.
	
	MORE INFORMATION
	================
	
	For additional information about changes made to the IntelliMouse settings,
	please see the following article in the Microsoft Knowledge Base:
	
	  Q162400 WD97: IntelliMouse Changes Not Reflected in Current Session
