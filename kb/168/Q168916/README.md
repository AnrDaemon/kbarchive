## Q168916: Volume Control Icon No Longer Visible on the Taskbar

	Article: Q168916
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 4.0 95
	Operating System(s): 
	Keyword(s): kbsound kbusage win95
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Volume Control icon may no longer be visible on the taskbar.
	
	CAUSE
	=====
	
	Another user may have disabled the Volume Control icon.
	
	RESOLUTION
	==========
	
	To resolve this issue, follow these steps:
	
	1. In Control Panel, double-click Multimedia.
	
	2. On the Audio tab, click the "Show volume control on the taskbar" check box to
	  select it.
	
	3. Click OK.
	
	Additional query words: systray.exe system tray winnt
