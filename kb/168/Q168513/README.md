## Q168513: Windows NT Installable File System Kit (IFS) Description

{% raw %}

	Article: Q168513
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbotherkbfaq
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft(R) Windows NT(R) Installable File System (IFS) Kit is a
	developer's kit for the kernel mode file system and file system filter driver
	models.
	
	MORE INFORMATION
	================
	
	For the most current information about the Windows NT Installable File System
	Kit (IFS), see the following Microsoft Web site:
	
	  http://www.microsoft.com/hwdev/ntifskit/default.htm
