## Q168318: XADM: Guidgen.exe Available Only for Intel Platforms

	Article: Q168318
	Product(s): Microsoft Exchange
	Version(s): 4.0,5.0,5.5
	Operating System(s): 
	Keyword(s): kbusage exc4 exc5 exc55
	Last Modified: 07-MAR-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 4.0, 5.0, 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Guidgen.exe utility used in the rebuilding of Site Folders is available for
	the Intel platform only. However, this utility can be run on any Intel computer
	to generate a new globally unique identifier (GUID) for an Exchange Server
	running on Intel or non-Intel platform.
	
	MORE INFORMATION
	================
	
	The Guidgen.exe utility is used to randomly generate a GUID that will be used to
	rebuild the site folders. This utility is available in the Microsoft Developers
	Network under the product samples.
	
	For additional information about how to rebuild site folder or to run Guidgen,
	click the article number below to view the article in the Microsoft Knowledge
	Base:
	
	  Q152960 XADM: Reassigning Site Roles after Removing the First Server in an
	  Exchange Site
