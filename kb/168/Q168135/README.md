## Q168135: HP Font Attributes Display and Print Incorrectly

	Article: Q168135
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbprint
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Some of the true type fonts supplied with the Hewlett-Packard (HP) LaserJet 5P
	driver disk version 1.0.0 may display and print incorrectly. For example, when
	viewing the Albertus Medium font (click the Start button, point to Settings,
	click Control Panel, and then click Fonts) the font has the attributes of Arial
	instead.
	
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 4.0. This
	problem was corrected in the latest Microsoft Windows NT 4.0 U.S. Service Pack.
	For information on obtaining the service pack, query on the following word in
	the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K

	or reliability of these products.
