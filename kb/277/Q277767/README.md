## Q277767: XADM: User Without Proper Permissions Can Open Public Folders

{% raw %}

	Article: Q277767
	Product(s): Microsoft Exchange
	Version(s): 5.5
	Operating System(s): 
	Keyword(s): exc55
	Last Modified: 13-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you create a public folder on an Exchange Server 5.5 computer, users who do
	not have the correct permissions can see messages in your public folders.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server
	version 5.5.
	
	
	MORE INFORMATION
	================
	
	Users should be able to see the subfolders but not the messages within the
	folders or subfolders if they do not have specific permissions to do so.
