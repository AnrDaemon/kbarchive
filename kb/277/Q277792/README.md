## Q277792: Reader: &quot;Open With&quot; Dialog Box Appears When Opening eBook File

{% raw %}

	Article: Q277792
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1.5
	Operating System(s): 
	Keyword(s): kbfile kbimu
	Last Modified: 31-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Reader, version 1.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to download or open an eBook (.lit) file in Microsoft Reader,
	you may experience one of the following symptoms:
	
	- An Open With dialog box appears on the screen, prompting you to choose the
	  program that you want to use to open the file.
	
	- A Save As dialog box appears on the screen, prompting you to choose a
	  location in which to save the file.
	
	CAUSE
	=====
	
	This behavior can occur if Microsoft Internet Explorer 5.5 is installed on your
	computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, update the eBook (.lit) file association on your
	computer.
	
	The following file is available for download from the Microsoft Download Center:
	
	  Rdrmime.exe
	  (http://download.microsoft.com/download/msreader/Patch/1.0/W9X2KMe/EN-US/Rdrmime.exe)
	
	Release Date: Oct-24-2000

	posting to scan this file for viruses. Once posted, the file is housed on secure
