## Q277659: CFS2: Configuration File Is Incorrectly Named in Readme.rtf File

{% raw %}

	Article: Q277659
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbdocerr kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In the Readme.rtf file for Microsoft Combat Flight Simulator 2, the game
	configuration file is incorrectly named Combatfs.cfg in the following sections:
	
	- How can I change the Shift Z coordinate displays
	- How can I disable the opening movie
	- Multiplayer
	
	The correct name of the game configuration file for Combat Flight Simulator 2 is
	Cfs2.cfg.
	
	MORE INFORMATION
	================
	
	To edit the Cfs2.cfg file, follow these steps:
	
	1. Quit Combat Flight Simulator 2 if it is running.
	
	2. Click Start, point to Find or to Search, and then click "Files or Folders" or
	  "For Files or Folders".
	
	3. In the Named box, type "cfs2.cfg" (without the quotation marks).
	
	4. In the "Look in" box, click My Computer.
	
	5. Click Find Now or Search Now.
	
	6. Double-click the Cfs2.cfg file.
	
	7. In the "Choose the program you want to use" box, click Notepad.
	
	8. Click to clear the "Always use this program to open this file" check box, and
	  then click OK.
	
	9. When you are finished making the necessary changes to the file, click Exit on
	  the File menu.
	
	10. When you are prompted to save the changes, click Yes.
	
	11. Restart Combat Flight Simulator 2.
