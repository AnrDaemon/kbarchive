## Q87867: PC Win: Err Msg: The Application Could Not Be Executed

{% raw %}

	Article: Q87867
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Attempting to view a WordPerfect version 5.1 for Windows document as an
	attachment in version 3.0 of Microsoft Mail for Windows may generate the
	following error:
	
	  The application could not be executed.
	
	CAUSE
	=====
	
	WordPerfect for Windows allows only one session of the application to be
	running. Close the current session of WordPerfect and then view the attachment.
