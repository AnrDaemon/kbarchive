## Q87871: Bookshelf 1992 Animations Stop When Screen Saver Started

{% raw %}

	Article: Q87871
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1992 edition,1993 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for Windows versions 1992 edition, 1993 edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If a screensaver is started when Microsoft Bookshelf 1992 Edition is playing an
	animation, the animation will stop. If the animation is a narrated animation,
	the animation controller will be reset to ensure proper timing and
	synchronization. This is by design.
	
	Additional query words: 1.00 screensaver viewer 1.0 usage
