## Q87214: PROFS: Why Is the Gateway Program Called HostDisp?

{% raw %}

	Article: Q87214
	Product(s): Microsoft Mail For PC Networks
	Version(s): 3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to IBM PROFS and OfficeVision, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The gateway program HOSTDISP.EXE (also referred to as HostDisp) is short for
	Host Dispatcher. In version 3.0 of the Microsoft Mail Gateway to IBM PROFS and
	OfficeVision, the gateway program name was changed to VMGATE.EXE.
	
	REFERENCES
	==========
	
	"Microsoft Mail Gateway to IBM PROFS and OfficeVision LAN Administrator's Guide"
