## Q87049: MCI: Main Software Components in an MCI Gateway

	Article: Q87049
	Product(s): Microsoft Mail For PC Networks
	Version(s): 3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 04-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to MCI, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	There are two software components which run on the MCI gateway PC:
	
	- The X.PC driver, which must be memory resident when running the gateway
	  program
	
	- The gateway program itself
	
	If either of these programs is not configured correctly, the gateway cannot
	function.
	
	For more information on system requirements for the Microsoft Mail Gateway to
	MCI, see the "Microsoft Mail Gateway to MCI Administrator's Guide."
