## Q87259: Mac Wkst: Must Modify Address Book Online

	Article: Q87259
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, version 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In version 3.1 of Microsoft Mail for AppleTalk Networks, Personal Groups and
	Address Book information is stored on the Mail Server and can only be modified
	while working online. While using Mail offline, the Address Book and Personal
	Groups are available but cannot be modified.
	
	MORE INFORMATION
	================
	
	If the preference to add users to the Address Book is on, then the Address Book
	is updated when the message is sent from the Outbox to the server. The offline
	copy of the Address Book will be updated the next time the client downloads the
	resources from the server.
