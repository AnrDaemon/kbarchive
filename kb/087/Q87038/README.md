## Q87038: SMTP: Dir-Sync Effect on SMTP Addresses

{% raw %}

	Article: Q87038
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to SMTP, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Dir-Sync can affect SMTP addresses if it is enabled. The address lists that can
	be affected that have SMTP addresses are the global address list (GAL) and the
	SMTP address list.
	
	For more information on the use of DirSync, see the "Global Directory
	Synchronization" section of the "Microsoft Mail Administrator's Guide."
