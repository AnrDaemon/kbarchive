## Q87297: Additional Icons Available from WEP 1.0 and Windows 3.1

{% raw %}

	Article: Q87297
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 29-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Idlewild screen saver program included with Windows Entertainment Pack (WEP)
	version 1.0 contains several icons that may be used for applications in Windows.
	
	MORE INFORMATION
	================
	
	To access these icons, do the following:
	
	1. Highlight the icon you want to change.
	
	2. From the Program Manager File menu, choose Properties. Choose the Change Icon
	  button.
	
	3. In the File Name box, type the path to the WEP directory and to the file
	  IB.IW, which contains the icons (for example,C:\WEP\IB.IW) and choose OK. The
	  icons will now be shown in the Current Icon box to choose from.
	
	Windows 3.1 also comes with the file MORICONS.DLL, which includes many other
	icons to use. To use these icons as in the above example, replace the filename
	IB.IW with MORICONS.DLL.
