## Q87042: SMTP: SMTP Command-Line Parameters

	Article: Q87042
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to SMTP, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	These command-line parameters can be used with the Microsoft Mail Gateway to
	SMTP:
	
	  -BD          -BT          -BR          -C[newi]
	  -D           -G[acnsy]    -H           -I
	  -L[acnsy]    -MR          -MS          -Q
	  -P           -RE          -RR          -T
	  -V20         -W           -X           -MD
	
	For a full description of the usage of these parameters, see Chapter 4, "Running
	the Gateway," in the "Microsoft Mail Gateway to SMTP Administrator's Guide."
	
	
	Additional query words: pcmail
