## Q87392: PC DB: Which Programs Access the GLOBAL.GLB File

	Article: Q87392
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:2.1e,3.0,3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, versions 2.1e, 3.0, 3.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Mail Administrator program (ADMIN.EXE) maintains and updates the
	contents of the GLOBAL.GLB file.
	
	The Microsoft Mail for PC Networks, MS-DOS workstation reads this file to
	determine if it has the ability to exit to MS-DOS.
