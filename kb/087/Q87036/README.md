## Q87036: SMTP: Address Lists That Display SMTP Address

	Article: Q87036
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to SMTP, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Simple Mail Transfer Protocol (SMTP) addresses can be added to these address
	lists:
	
	- Personal address lists (PAL)
	
	  Users can modify their own PAL.
	
	- SMTP address list
	
	  Only the Mail administrator can add to this list.
	
	- Postoffice address list (POL)
	
	  Only the Mail administrator can add to the POL.
	
	- Global address list (GAL)
	
	  The Mail administrator must enable this list from the Admin program. Enabling
	  this feature will automatically port all existing addresses in the POL and
	  any "gateway" lists to the GAL. This list must be maintained. For information
	  on this list, see the "Microsoft Mail Administrator's Guide."
