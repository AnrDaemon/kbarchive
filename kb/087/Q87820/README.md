## Q87820: CORE MCK PROMs for Microsoft OS/2 1.3

	Article: Q87820
	Product(s): Microsoft LAN Manager
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	CORE has just released a new PROM for their MCK Controller, which is a popular
	subsystem for the IBM PS/2 series. This PROM corrects a problem that was
	encountered (especially with the IBM PS/2 Model 80) during testing of Microsoft
	OS/2 version 1.3.
