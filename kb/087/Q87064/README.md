## Q87064: MCI: How to Check to See If Mail Is Queued for the Gateway

{% raw %}

	Article: Q87064
	Product(s): Microsoft Mail For PC Networks
	Version(s): 3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to MCI, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	To check whether mail is queued for the gateway, choose Queue from the Admin
	program, then type MCI at the Network Name prompt, or select it from the list of
	available networks.
	
	A list of messages queued for the MCI gateway will be displayed.
	
	REFERENCES
	==========
	
	Microsoft Mail Gateway to MCI Administrator's Guide
