## Q87989: CAPS LOCK Key Problem on Packard Bell Legend 416 SX / 720 SX

{% raw %}

	Article: Q87989
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 26-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	To change between uppercase and lowercase characters, you need to press the CAPS
	LOCK key twice on your Packard Bell Legend 416 SX or Legend 720 (386SX, 20 MHz)
	computer.
	
	CAUSE
	=====
	
	Packard Bell technical support has confirmed that there is a problem with the
	keyboard BIOS on the Legend 416 SX and Legend 720 (386sx, 20 MHz) models.
	
	There is no correction available at this time.
	
	MORE INFORMATION
	================
	
	For further information, contact Packard Bell technical support.
	
	The Legend 416 SX and Legend 720 are manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
