## Q87053: MCI: Steps That Must Be Completed to Install the Gateway

	Article: Q87053
	Product(s): Microsoft Mail For PC Networks
	Version(s): 3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to MCI, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Complete instructions for installing the Microsoft Mail Gateway to MCI can be
	found in "Microsoft Mail Gateway to MCI Administrator's Guide."
	
	Before installation, ensure:
	
	- The drive and path of the directory containing the Microsoft Mail data files
	  is known.
	
	- The drive and path of the directory containing the Microsoft Mail executables
	  is known.
	
	- That if you are installing on the gateway postoffice, the gateway component
	  is installed first.
	
	REFERENCES
	==========
	
	Microsoft Mail Gateway to MCI Administrator's Guide
