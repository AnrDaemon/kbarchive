## Q87032: SMTP: Admin Options to Be Configured

{% raw %}

	Article: Q87032
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to SMTP, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following Admin options need to be configured for the Microsoft Mail Gateway
	to SMTP postoffice:
	
	- Address-Map
	
	- List
	
	- Setup
	
	- Postoffices
	
	For more information on what these options are and how and when to configure
	them, see Chapter 3 of the "Microsoft Mail Gateway to SMTP Administrator's
	Guide."
