## Q87198: PROFS: Information Required by PROFS Gateway Install Program

{% raw %}

	Article: Q87198
	Product(s): Microsoft Mail For PC Networks
	Version(s): 3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to IBM PROFS and OfficeVision, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The information prompted for by the Install program for the Microsoft Mail
	Gateway to IBM PROFS and OfficeVision is documented in the "Microsoft Mail
	Gateway to IBM PROFS and OfficeVision LAN Administrator's Guide."
	
	Before starting the Install program, you should know the following:
	
	- Location of MS Mail executables
	
	- Location of MS Mail data files
	
	- Name of the gateway
	
	- Type of 3270 emulator used on the gateway PC
	
	If you are installing only the Gateway Access component, you must know the
	location of the data files and the gateway name.
	
	REFERENCES
	==========
	
	"Microsoft Mail Gateway to IBM PROFS and OfficeVision LAN Administrator's
	Guide"
