## Q87222: PROFS: DOS Errorlevel Messages

	Article: Q87222
	Product(s): Microsoft Mail For PC Networks
	Version(s): 3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to IBM PROFS and OfficeVision, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	DOS Errorlevel return codes from HostDisp (or VMGATE) are listed in the
	"Microsoft Mail Gateway to IBM PROFS and OfficeVision, LAN Administrator's
	Guide" for version 3.0.
	
	Return codes for earlier versions of the gateway are not documented. Please call
	Microsoft Product Support Services for a definition of the DOS Errorlevel
	messages for these versions.
	
	
	Additional query words: 3.00 pcmail profslan DOS Errorlevel return codes
