## Q87487: X400: Setting the X.400 Log File Debug Option -LD

{% raw %}

	Article: Q87487
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to X.400, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	To set the debug option in the Microsoft Mail Gateway to X.400, use the -LD
	option (case insensitive) at the command line of the gateway program. You should
	disable message timeout (-rt), because it generates a number of debug messages.
	
	Use the -LD option in conjunction with, not in place of, the -LV logging option.
