## Q87405: PC DB: Structure of the HLP Directory

{% raw %}

	Operating System(s):

	Last Modified: 12-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, versions 2.1e, 3.0, 3.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The HLP files are encrypted text files that contain information describing
	functions of the Microsoft Mail programs. The information can be called up by
	pressing the F1 function key. There should be one help file per program, and
	they are normally flagged as read only during installation.
