## Q87869: Bookshelf: Cannot Access Estonia/Latvia/Lithuania with Locate

	Article: Q87869
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1992 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for Windows 1992 edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you select Estonia/Latvia/Lithuania with the Locate button in the Hammond
	Atlas in Microsoft Bookshelf for Windows 1992 Edition, the Balkans (Yugoslavia,
	Bulgaria, Romania, Albania, and Greece) are displayed.
	
	MORE INFORMATION
	================
	
	Workaround
	----------
	
	Estonia/Latvia/Lithuania can be accessed as follows:
	
	1. Select Poland.
	
	2. Select Lithuania OR select one of the following from the Index menu:
	  a. Estonia
	
	  b. Latvia
	
	  c. Lithuania
	
	  d. Baltics
	
	  e. Any related city
	
	  -or-
	
	1. Select one of the three flags.
	
	2. Select the country name.
	
	Steps to Reproduce Problem
	--------------------------
	
	1. Start the Hammond Atlas.
	
	2. Click Locate.
	
	3. Select Europe.
	
	4. Select Estonia/Latvia/Lithuania. The Balkans display.
	
	Additional query words: 1.00 win usage viewer 1.0
