## Q87256: CONN: Errors Running Connection Name Utility

	Article: Q87256
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): 1.0; WINDOWS:3.0,3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Connection Gateway, version 1.0 
	- Microsoft Mail for AppleTalk Networks, versions 3.0, 3.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you are using Microsoft Mail for AppleTalk Networks and you want to use the
	Connection Name utility, the Mail Connection Gateway must be running on the Mail
	server under MultiFinder with System 6.x or System 7.0.
	
	NOTE: Running the utility under Finder may cause problems.
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
