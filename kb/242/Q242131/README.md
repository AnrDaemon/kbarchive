## Q242131: How to: Display a List of Processes That Have Files Open

	Article: Q242131
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 27-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes Systems Internals's Process Explorer shareware utility.
	
	MORE INFORMATION
	================
	
	The Process Explorer utility lists the files, registry keys, and other objects
	that open processes are using or the dynamic-link libraries (DLLs) that the
	processes loaded. To obtain the Process Explorer utility, visit the following
	Systems Internals Web site:
	
	  http://www.sysinternals.com/ntw2k/freeware/procexp.shtml
