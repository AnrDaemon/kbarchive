## Q242813: FS2000: Plane of Another Player May Disappear Unexpectedly

	Article: Q242813
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbdisplay kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator 2000 
	- Microsoft Flight Simulator 2000 Professional Edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play a multiplayer game in Microsoft Flight Simulator 2000, the plane
	of another player may disappear unexpectedly, yet the player ID next to that
	plane may still be displayed on the screen.
	
	
	CAUSE
	=====
	
	This behavior can occur if you fly within .1 nautical miles of the plane that
	disappears.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
