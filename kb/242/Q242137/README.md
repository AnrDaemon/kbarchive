## Q242137: Cannot restore files that were deleted off server from a client.

	Article: Q242137
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to restore files that were deleted remotely from a computer running
	Windows NT Server using a computer that is running either Microsoft Windows NT
	Workstation or Microsoft Windows 95, you cannot restore the files using the
	Recycle Bin on the client or server computer.
	
	RESOLUTION
	==========
	
	This behavior is by design.
	
	Additional query words: recycle
