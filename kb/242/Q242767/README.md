## Q242767: INFO: Deploying WebClasses with PDW

	Article: Q242767
	Product(s): Microsoft Visual Basic for Windows
	Version(s): 4.0,5.0,6.0
	Operating System(s): 
	Keyword(s): kbList kbDeployment kbVBp600 kbWebClasses kbGrpDSASP kbDSupport kbiis400 kbiis500
	Last Modified: 25-APR-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual Basic Learning Edition for Windows, version 6.0 
	- Microsoft Visual Basic Professional Edition for Windows, version 6.0 
	- Microsoft Visual Basic Enterprise Edition for Windows, version 6.0 
	- Microsoft Internet Information Server versions 4.0, 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists resources and links to other Knowledge Base articles that are
	related to deploying Visual Basic 6.0 WebClasses (Internet Information Server
	Applications) with the Package and Deployment Wizard.
	
	MORE INFORMATION
	================
	
	The following is a link to a comprehensive article in the September issue of
	MIND magazine about deploying WebClass applications:
	
	  Using the Package and Deployment Wizard
	  (http://www.microsoft.com/Mind/0999/faq/faq0999.htm)
	
	For additional information about deploying WebClass applications, click the
	article numbers below to view the articles in the Microsoft Knowledge Base:
	
	  Q199017 HOWTO: Deploy Visual Basic IIS Applications with PDW
	
	  Q194296 HOWTO: Debug PDW Internet Deployment Issues
	
	  Q192116 HOWTO: Configure Posting Acceptor to Work With the PDW
	
	  Q192639 HOWTO: Use PDW to Deploy Using the FTP Web Publishing Method
	
	  Q242550 INFO: System Files Needed to Run WebClasses
	
	For additional information on resolving possible errors that you may receive
	while deploying your WebClass applications, click the article numbers below to
	view the articles in the Microsoft Knowledge Base:
	
	  Q234974 PRB: Unexpected Error C0042116 With PDW: Web Server Either Is Not
	  Responding or Returned an Unrecognized Response
	
	  
	
	  Unexpected error number C0042116 has occurred: The Web server either is not
	  responding or returned an unrecognized response.
	
	  Q234972 PRB: Unexpected Error 40042119 With PDW: There Is No PostInfo File on
	  the Server
	
	  
	
	  Unexpected error number 40042119 has occurred: There is no PostInfo file on
	  the server.
	
	  This error may occur if you have typed an incorrect URL or if the URL does not
	  refer to a correctly configured Web server. Ensure that the URL is correct
	  and the Web server is properly configured. For proper configuration, you may
	  need to install Microsoft Visual Studio's server-side setup on the Web
	  server.
	
	  Q234975 PRB: Unexpected Error C0042116 With PDW: You Do Not Have Write Access
	  to Destination URL
	
	  
	
	  Unexpected error number C0042116 has occurred: The Web server returned the
	  following error: You do not have write access to destination URL
	  http://<servername>/<directory>.
	
	  Q234976 PRB: Unexpected Error C0042116 When Using PDW: Access Denied
	
	  
	
	  Unexpected error number C0042116 has occurred: The Web server returned the
	  following error: Access Denied.
	
	  Q234977 PRB: Unexpected Error C0042116 With PDW: Cannot Move Filename to Its
	  Final Destination; Processing Has Stopped
	
	  
	
	  Unexpected error number C0042116 has occurred: The Web server returned the
	  following error: Cannot move filename "<filename>" to its final
	  destination. Further processing has stopped.
	
	  Q234978 PRB: Unexpected Error C0042116 With PDW: The Parameter Is Incorrect
	
	  
	
	  Unexpected error number C0042116 has occurred: The parameter is incorrect.
	
	  Q234973 PRB: Unexpected Error 80042114 With PDW: The Web Server You Selected
	  Does Not Indicate Support For the Service Provider You Selected
	
	  
	
	  Unexpected error number 80042114 has occurred: The Web server you selected
	  does not indicate support for the service provider you selected.
	
	  Q225120 PRB: Unexpected Error C0042116 With PDW: The Signature On Cabinet
	  File XX.CAB Could Not Be Verified
	
	  
	
	  Unexpected error number C0042116 has occurred: The Web server returned the
	  following error: The signature on cabinet file XX.CAB could not be verified.
	
	  Q223499 PRB: Unexpected Error C0042116 With PDW: Files Specified in Section
	  RInstallApplicationFiles of filename.INF File Are Busy
	
	  
	
	  Unexpected error number C0042116 has occurred: The Web server returned the
	  following error: Files specified in section RInstallApplicationFiles of INF
	  file jallen8.INF are busy. Can not install them unless allowed to do a
	  REBOOT.
