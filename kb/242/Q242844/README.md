## Q242844: FS2000: New Flight Starts as Minimized Task

	Article: Q242844
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbui kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator 2000 
	- Microsoft Flight Simulator 2000 Professional Edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start Microsoft Flight Simulator 2000, you may experience the following
	symptoms:
	
	- The Flight Simulator 2000 may be minimized when it plays the Microsoft logo
	  animation. However, the animation sounds are still played.
	
	- When you click Fly Now on the Flight Simulator 2000 startup screen, the
	  program may be minimized again.
	
	NOTE: If the Auto Hide feature for the Windows taskbar is enabled, you may not
	notice that Flight Simulator 2000 is minimized. Move your pointing device to the
	bottom of the screen to reveal the taskbar.
	
	
	CAUSE
	=====
	
	This behavior can occur if Flight Simulator 2000 was running in Full Screen mode
	the last time you quit the game.
	
	RESOLUTION
	==========
	
	To resolve this issue, restore Microsoft Flight Simulator 2000. To do this,
	click Microsoft Flight Simulator 2000 on the Windows taskbar. If the program is
	minimized again, click Microsoft Flight Simulator 2000 on the taskbar again.
	
	To prevent this issue from occurring, press ALT+ENTER to switch to Window mode
	before you quit Flight Simulator 2000, and press ALT+ENTER to switch to Full
	Screen mode when you start Flight Simulator 2000.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
