## Q101935: PC Win: How to Recover Missing Columns in Message Center Viewer

	Article: Q101935
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0,3.0b,3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, versions 3.0, 3.0b, 3.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Message Center Viewer (MCV) may become corrupted and the user can lose his
	Subject and/or Date columns. Rebuilding the Mail message file (MMF) may not
	restore the columns properly.
	
	RESOLUTION
	==========
	
	1. Close the MCV that does not have the columns.
	
	2. Select New Window.
	
	3. Exit Mail (this step is important because this saves the configuration).
	
	4. Restart Mail.
	
	Additional query words: 3.00 3.00b 3.20
