## Q101614: PC Win: Err Msg: File Error

	Article: Q101614
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0,3.0b,3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, versions 3.0, 3.0b, 3.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you select Demos from the Help menu, a "File Error" error message is
	generated.
	
	RESOLUTION
	==========
	
	Change the working directory for Microsoft Mail to point to the directory where
	MSMAIL.EXE is located.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in versions 3.0, 3.0b, and 3.2 of
	Microsoft Mail for Windows. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
	
	MORE INFORMATION
	================
	
	Here is the procedure for changing the working directory:
	
	1. In the Windows Program Manager, select the Microsoft Mail icon.
	
	2. From the File menu of Program Manager, choose Properties. This brings up a
	  window displaying the properties associated with that icon.
	
	3. Press ALT+W. This moves the cursor to the working directory. Type in the
	  complete path to the MSMAIL.EXE file. For example: c:\msmail.
	
	4. Choose the OK button.
	
	Additional query words: 3.00 3.0b 3.20 err msg errmsg alert
