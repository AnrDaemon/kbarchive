## Q101846: WFWG: Where Workgroup Mail Finds the Default Account Name

	Article: Q101846
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	When you start Workgroup Mail for the first time and choose to set up
	your own account, the default account name is the same name you used
	when installing Windows for Workgroups version 3.1.
	
	Workgroup Mail in Windows for Workgroups version 3.1 reads the
	customer name information from the USER.EXE file located in the
	WINDOWS\SYSTEM directory for the default account name.
	
	The account name can be changed during the setup process of
	Workgroup Mail.
	
	Additional query words: 3.10 3.1 3.11
