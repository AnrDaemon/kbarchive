## Q101172: MemMaker May Hang with the Bus Logic SCSI Controller Driver

{% raw %}

	Article: Q101172
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 23-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run MS-DOS MemMaker, the system may stop responding (hang) if the Bus
	Logic SCSI controller driver, BTDOSM.SYS, is detected.
	
	RESOLUTION
	==========
	
	To correct this problem, upgrade the SCSI controller driver to version 2.11 or
	later. For more information, call Bus Logic Technical Support.
	
	The product included here is manufactured by Bus Logic, a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
