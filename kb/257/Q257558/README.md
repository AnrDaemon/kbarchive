## Q257558: Microsoft Help Desk for Microsoft Office 2000 Comments and Corre

	Article: Q257558
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 09-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Help Desk for Microsoft Office 2000 ISBN 0-7356-0850-4 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Help Desk for Microsoft Office 2000, ISBN
	0-7356-0850-4.
	
	The following topics are covered:
	
	- Page vii: Roman Numerals Do Not Match Actual Page
	
	MORE INFORMATION
	================
	
	Page vii: Roman Numerals Do Not Match Actual Page
	-------------------------------------------------
	
	On page vii, the Roman numeral page numbers in the table of contents do not match
	the actual page numbers. In each case, the "xl" should be replaced with "xxx" to
	read as follows:
	
	Acknowledgements........................................xxxiii
	Introduction..............................................xxxv
	What the Help Desk Does...................................xxxv
	How to Use the Help Desk..................................xxxv
	If You Can't Find a Problem or Solution...................xxxv
	A Few Important Notes About the Registry.................xxxvi
	Command-Line Switches in Outlook........................xxxvii
	Programming Examples....................................xxxvii
	HTML in FrontPage......................................xxxviii
	The Location of Windows on Your Computer...............xxxviii
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
