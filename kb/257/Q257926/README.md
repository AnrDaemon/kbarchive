## Q257926: Encarta Dictionary: Essays in Hardbound Edition Are Not Included

	Article: Q257926
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0; :
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta World English Dictionary, version 1.0 
	- Microsoft Encarta World English Dictionary 2001 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Three introductory essays that are included in the Microsoft Encarta World
	English Dictionary Hardbound Edition are not included in the CD-ROM version.
	
	MORE INFORMATION
	================
	
	The hardbound edition of the Encarta World English Dictionary includes
	additional content that is not included in the dictionary's CD-ROM version.
	
	The CD-ROM version of Encarta World English Dictionary is both sold separately
	and as part of Microsoft Encarta Reference Suite 2000.
	
	The hardbound edition includes the following additional essays:
	
	- "History of the English Language" by Professor Christian J. Kay and Professor
	  Lee Pederson, with additional contribution by Anne H. Soukhanov.
	
	- "A Brief History of Dictionaries and Dictionary Makers" by Jonathon Green.
	
	- "World English" by Tom McArthur.
