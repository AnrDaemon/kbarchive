## Q257359: World Atlas 2000: World Atlas Has Encountered an Unknown Error

	Article: Q257359
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 
	Operating System(s): 
	Keyword(s): kb3rdparty kberrmsg kbhw kbimu kbHardware
	Last Modified: 09-OCT-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Reference Suite 2000 
	- Microsoft Encarta Interactive World Atlas 2000 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to install the Microsoft Encarta Reference Suite 2000 DVD-ROM,
	you may receive the following error message:
	
	  Invalid page fault in module Setup.exe.
	
	In addition, when you attempt to start Microsoft Encarta World Atlas 2000, you
	may receive the following error message:
	
	  World Atlas has encountered an unknown error.
	
	CAUSE
	=====
	
	This behavior can occur if the following conditions are true:
	
	- A Pioneer DVD-114 DVD-ROM drive is installed in your computer.
	
	- The firmware for the DVD-ROM drive is outdated.
	
	- A CD-ROM writer is connected to your computer.
	
	- The Adaptec EZ CD Creator software is installed on your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install version 1.24 of the firmware upgrade
	for the Pioneer DVD-114 DVD-ROM drive from the following Pioneer Web site:
	
	  http://www.pioneerelectronics.com/Pioneer/CDA/ServiceAndSupport/SelfHelpForIndFirmware/0,1465,60~6020~60201100,00.html
	
	NOTE: Please contact your hardware manufacturer before you update the firmware to
	make sure that you are installing the correct files.
