## Q162127: Error: Incorrect Version on File D:&#92;Data&#92;Data&#92;Base&#92;Weapons

	Article: Q162127
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Close Combat for Windows 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When starting Close Combat, you may receive the following error:
	
	  Incorrect Version on file D:\Data\Data\Base\Weapons
	
	RESOLUTION
	==========
	
	One of the following troubleshooting methods may resolve this issue.
	
	Method 1
	--------
	
	You may be able to resolve this issue by changing your CD-ROM cache size to 470
	KB. To do this, follow these steps:
	
	1. Click Start, point to Settings, and then click Control Panel.
	
	2. Double-click System.
	
	3. Click the Performance tab.
	
	4. Click File System.
	
	5. Click the CD-ROM tab.
	
	6. Move the Supplemental Cache Size slider to the middle position (so Windows is
	  using 470 KB for optimization).
	
	7. Click OK, and then click Close. If you are prompted to restart the computer,
	  click Yes.
	
	Method 2
	--------
	
	You may also be able to resolve this issue by removing Close Combat and all trial
	versions of Close Combat, deleting the Close Combat folder(s), and then
	reinstalling Close Combat into a different folder.
	
	MORE INFORMATION
	================
	
	Microsoft has confirmed this to be a problem in Close Combat version 1.0.
