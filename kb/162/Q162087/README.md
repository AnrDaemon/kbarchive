## Q162087: XCLN: Schedule + Meeting Wizard Won't Auto Pick Locations

{% raw %}

	Article: Q162087
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0; Win95:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you select multiple meeting locations when you create an appointment using
	the Schedule+ Meeting Wizard (click Make Meeting on the Tools menu), the Meeting
	Wizard examines the schedule for the first location but does not examine the
	schedules of the additional locations you selected.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Schedule+, version
	7.0. We are researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
	
	WORKAROUND
	==========
	
	To work around this problem, use the following steps to select the meeting
	location and to submit your meeting request:
	
	1. Start Schedule+.
	
	2. Click the Planner tab.
	
	3. Click the Invite button.
	
	4. In the Select Attendees dialog box, select the individuals you want to attend
	  from the address list box, and then click the Required (or Optional) button.
	
	5. In the Select Attendees dialog box, select the meeting locations from the
	  address list box, and click the Resource button.
	
	6. Click OK.
	
	7. On the Planner tab, select the location and time you want to schedule.
	
	8. Click Request Meeting.
	
	9. In the Available Resources dialog box, click the resource you want to use,
	  and then click Invite Selected.
