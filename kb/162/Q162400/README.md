## Q162400: WD97: IntelliMouse Changes Not Reflected in Current Session

	Article: Q162400
	Product(s): Word 97 for Windows
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 02-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you make a change to the Microsoft IntelliMouse, the change is not apparent
	in Word.
	
	WORKAROUND
	==========
	
	Quit the current Word session and restart Word. The changes made to the
	IntelliMouse will now be available in Word.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the products listed at the
	beginning of this article.
