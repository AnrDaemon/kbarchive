## Q162598: WD97: ActiveX Controls Lose Sizing After Previewing in Browser

	Article: Q162598
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:3.01,97
	Operating System(s): 
	Keyword(s): kbwdinternet
	Last Modified: 05-JAN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	- Microsoft Internet Explorer version 3.01 for Windows 95 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you view an HTML document in print preview that contains a sized ActiveX
	control in Microsoft Internet Explorer 3.01, the size information is lost and is
	reset to its original size.
	
	
	CAUSE
	=====
	
	This functionality occurs when you save the document as HTML in Word 97 and view
	the HTML document in Internet Explorer 3.01.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the products listed at the
	beginning of this article.
