## Q162112: &quot;Unable to Load the Unicode Font&quot; Opening a File with Notepad

	Article: Q162112
	Product(s): Microsoft Windows NT
	Version(s): WINNT:3.51
	Operating System(s): 
	Keyword(s): kbnetwork
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.51 
	- Microsoft Windows NT Server version 3.51 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you use Notepad to open a Unicode file and the font currently set in Notepad
	is not a Unicode font, you will receive the following error:
	
	  Unable to load the Unicode Font. Will not be able to display certain
	  characters.
	
	If you click OK, the Unicode file is loaded and displayed.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 3.51. This
	problem was corrected in Windows NT version 4.0.
