## Q162827: WD97: Problems Printing Document with Many Drawing Objects

	Article: Q162827
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbualink97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	A Microsoft Word 97 document that contains many drawing objects will take an
	unexpectedly long period of time to print or may hang the system.
	
	WORKAROUND
	==========
	
	Separate the document into different print jobs. This can be accomplished either
	by breaking the document into smaller documents, or by selecting a part of the
	document and printing the selection.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the products listed at the
	beginning of this article.
