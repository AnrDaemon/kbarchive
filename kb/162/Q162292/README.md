## Q162292: WD97: Menu Item Moved When Template Converted to Word 97

	Article: Q162292
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	A menu item stored in a Word 6.0 or 7.0 template doesn't retain the menu
	position when the template is converted to Word 97.
	
	WORKAROUND
	==========
	
	Use the following steps to reposition the menu item:
	
	1. In Microsoft Word 97 for Windows, open the affected template.
	
	2. On the Tools menu, click Customize.
	
	3. On the affected menu, click and drag the menu item to the correct position.
	
	4. Click Close to exit the Customize dialog box.
	
	5. Save and close the template.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the products listed at the
	beginning of this article.
