## Q299732: Train Simulator: Route Editor Hangs w/Desktop Resolution Change

	Article: Q299732
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Train Simulator, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you change the resolution of your display while you are using the Microsoft
	Train Simulator Route Editor, the Route Editor may stop responding (hang).
	
	RESOLUTION
	==========
	
	To work around this issue, do not change your display resolution while you are
	using the Route Editor or other Train Simulator tools.
