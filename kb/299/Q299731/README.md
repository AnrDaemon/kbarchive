## Q299731: Train Simulator: Clock in Cab Always Displays Time as 1:10

	Article: Q299731
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Train Simulator, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the Locomotive 2000 feature in Microsoft Train Simulator for any
	given activity, the clock inside the cab always displays the time as 1:10.
	
	CAUSE
	=====
	
	This behavior is by design. The clock is designed as static art and is not a
	working feature of the train.
