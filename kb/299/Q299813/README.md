## Q299813: Train Simulator: How to Find the Product ID

	Article: Q299813
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Train Simulator, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article discusses how to find the product ID (PID) for your installation of
	Microsoft Train Simulator.
	
	MORE INFORMATION
	================
	
	To find the PID, follow these steps:
	
	1. Click Start, point to Programs, point to Microsoft Games, point to Train
	  Simulator, and then click Train Simulator Troubleshooter.
	
	2. Click Troubleshooting.
	
	3. Click OK to close this dialog box.
