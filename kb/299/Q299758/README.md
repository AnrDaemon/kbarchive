## Q299758: Train Simulator: Can't Add Coal/Water to Goelsdorf 380 Tender

	Article: Q299758
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Train Simulator, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Microsoft Train Simulator, when you use the Goelsdorf 380 train to refuel the
	tender wagon with water or coal, the refueling operation does not take place as
	expected.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
	
	MORE INFORMATION
	================
	
	You can successfully refuel if you use the Flying Scotsman train.
