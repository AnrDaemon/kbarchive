## Q299425: Document Location for Passport Implementation on Unix Systems

	Product(s): Microsoft Developer Network
	Version(s): 1.3,1.4

	- Microsoft Passport, versions 1.3, 1.4 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists Web sites that contain Passport documentation.
	
	MORE INFORMATION
	================
	
	For documentation on typical Passport installation and implementation issues on
	Unix systems, see the following Microsoft Passport Web site:
	
	  http://www.rtr.com/passport/
	
	For other Passport documentation, see the following Passport Web site:
	
	  http://www.passport.com
