## Q299733: Train Simulator: Question Mark Help Buttons Don't Work

	Article: Q299733
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Train Simulator, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the Route Editor in Microsoft Train Simulator, the question mark
	(?) buttons in the Texture Selector windows do not work as expected.
	
	RESOLUTION
	==========
	
	To work around this issue, use the Help button in the main editor window.
