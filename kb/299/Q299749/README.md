## Q299749: Train Simulator May Not Start as Expected

	Article: Q299749
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Train Simulator, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are starting Microsoft Train Simulator, the program may appear to
	start, and then the main screen is displayed again.
	
	CAUSE
	=====
	
	This behavior can occur if you have installed Train Simulator to a custom path
	that is more than 75 characters long.
	
	RESOLUTION
	==========
	
	To resolve this issue, remove Train Simulator, and then reinstall Train
	Simulator with a path fewer than 75 characters long.
