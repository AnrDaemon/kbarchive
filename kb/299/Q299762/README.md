## Q299762: Train Simulator: Minimum System Requirements

{% raw %}

	Article: Q299762
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Train Simulator, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article summarizes the minimum system requirements to run Microsoft Train
	Simulator.
	
	MORE INFORMATION
	================
	
	To run Train Simulator, you need the following system resources:
	
	   - A multimedia computer with a 266-megahertz (MHz) processor or higher
	
	- A Microsoft Windows 95, Windows 98, Windows 2000, or Windows Millennium
	  Edition (Me) operating system
	
	- 32 megabytes (MB) of random access memory (RAM) for Windows 95 operating
	  systems or later, or 64 MB of RAM for Windows 2000
	
	- A quad speed CD-ROM or faster
	
	- A Super VGA (800 x 600 pixels) 16-bit color monitor
	
	- DirectX 8.0a (included on the Train Simulator CD-ROM)
	
	- 500 MB of hard-disk space (additional 50 MB of space required for DirectX 8
	  installation and swap file)
	
	- A Microsoft Mouse or compatible pointing device
	
	- A Microsoft DirectSound 7 API-compatible sound card, with speakers or
	  headphones for audio monitoring
	
	- A 3-D graphics accelerator card with a minimum of 4 MB memory, compatible
	  with Microsoft Direct 3D version 7.0 API
	
	- DirectX 7.0 or later compatible 4MB video card capable of 800X600 resolution
	  in 32K colors.
	
	NOTE: This product needs Internet Explorer version 4.1 or higher.
