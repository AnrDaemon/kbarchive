## Q299743: TS Err Msg: &quot;Setup encountered a disk error while writing ...&quot;

	Article: Q299743
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kberrmsg kbsetup kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Train Simulator, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are trying to install Microsoft Train Simulator, you may receive the
	following error message:
	
	  Train Simulator Setup encountered a disk error while writing to the file
	  C:\Program Files\Microsoft Games\Train Simulator\dialog.dll. Make sure your
	  hard disk is not full and the file is not in use.
	
	CAUSE
	=====
	
	This behavior can occur if a previous installation was interrupted and the
	original Train Simulator folder is still present on the system.
	
	
	RESOLUTION
	==========
	
	To resolve this issue, remove the existing Train Simulator folder and start the
	installation process again. To do this, follow these steps:
	
	1. Right-click Start, and then click Explore.
	
	2. In the left frame, double-click Program Files.
	
	3. Double-click Microsoft Games.
	
	4. Click the Train Simulator folder to highlight it, and then click Delete.
	
	5. Close Windows Explorer.
	
	Additional query words: msgame trainsim
