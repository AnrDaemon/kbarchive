## Q299818: Train Simulator: E, SHIFT+E Work in Hisatsu w/ Simple Controls

	Article: Q299818
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Train Simulator, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you are running the Hisatsu Line route, and you have selected Simple Controls
	in the Options dialog box, you can still use the E or SHIFT+E keys to move the
	acceleration up or down.
	
	MORE INFORMATION
	================
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
