## Q299822: MapPoint 2002: Default Speed for Streets Is Changed from 31 MPH

	Article: Q299822
	Product(s): Microsoft Automap
	Version(s): 
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 14-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MapPoint 2002 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Microsoft MapPoint, the default speed for streets under Route Options is 31
	miles per hour (mph). However, when you click Reset, the default speed is
	changed to 20 mph.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
