## Q299811: Train Simulator: Shortcuts Not Removed When You Cancel Reinstall

	Article: Q299811
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbhw kbimu kbHardware
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Train Simulator, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the reinstall option in Microsoft Train Simulator Setup, and then
	you cancel the reinstall operation in progress, the program files are removed.
	However, the desktop and Start menu shortcuts are not removed as expected.
	
	RESOLUTION
	==========
	
	To work around this issue, manually remove these shortcuts. To do this,
	right-click a given shortcut, and then click Delete.
	
	MORE INFORMATION
	================
	
	These shortcuts are no longer functional.
