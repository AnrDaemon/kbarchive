## Q227204: Baseball 2000: Graphics Are Distorted When You Switch to Game

	Article: Q227204
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbenv kbimu msgame
	Last Modified: 15-APR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Baseball 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you switch to another program running on your computer while you are playing
	Microsoft Baseball 2000, the game graphics may not be displayed correctly when
	you switch back to Baseball 2000.
	
	CAUSE
	=====
	
	The behavior can occur if you use ALT+TAB to switch between Baseball 2000 and
	the other program.
	
	RESOLUTION
	==========
	
	To resolve this issue, press CTRL+F5 to refresh the game graphics in Baseball
	2000.
	
	To prevent this issue from occurring, save your game and quit Baseball 2000
	before you switch to the other program.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Baseball 2000, version
	1.0.
