## Q71737: Aldus PageMaker version 4.0 and Bitstream Facelift

{% raw %}

	Article: Q71737
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Bitstream Technical Support has confirmed that if you use Bitstream Facelift
	versions earlier than 1.2 with Aldus PageMaker version 4.0, an Unrecoverable
	Application Error (UAE) will be generated.
	
	Registered users of Bitstream Facelift versions earlier than 1.2 should call
	Bitstream technical support to obtain an update to version 1.2.
	
	MORE INFORMATION
	================
	
	For more information, contact Bitstream Technical Support.
	
	The product included here is manufactured by a vendor independent of Microsoft;
	we make no warranty, implied or otherwise, regarding this product's performance
	or reliability.
