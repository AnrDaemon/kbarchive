## Q71563: Map Analysis Doesn't Run in the Background

	Article: Q71563
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Map Analysis, a program featuring demographics analyses written by National
	Planning Data Corp., does not run in the background under Microsoft Windows.
	
	MORE INFORMATION
	================
	
	The current version of Map Analysis is version 3.2. It requires 512K of
	conventional memory to run from MS-DOS. There are no plans to upgrade this
	product to run with Windows.
	
	Contact National Planning Data Corporation for more information.
	
	The product included here is manufactured by a vendor independent of Microsoft;
	we make no warranty, implied or otherwise, regarding this product's performance
	or reliability.
