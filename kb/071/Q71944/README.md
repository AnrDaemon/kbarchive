## Q71944: COMPAQ LTE 386/20 Reverse Video

	Article: Q71944
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When using Microsoft Windows on a COMPAQ LTE 386/20, the appearance of the video
	screen may look better if it is in reverse video mode (that is, light characters
	on a dark background).
	
	To put the screen into reverse video mode, press CTRL+ALT+[ (left square
	bracket).

	==========
