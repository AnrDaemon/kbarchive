## Q71785: Qualitas 386-to-the-Max Does Not Support Standard Mode

	Article: Q71785
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	According to Qualitas technical support, the memory manager product
	386-to-the-Max only supports Windows 3.0 in real and enhanced modes of
	operation.
	
	MORE INFORMATION
	================
	
	The latest version of 386-to-the-Max is 5.10.
	
	386-to-the-Max includes the Microsoft extended memory specification (XMS) as part
	of its interface. HIMEM.SYS should not be used with 386MAX.SYS because the
	HIMEM.SYS file's functionality is included in 386MAX.SYS.

	or reliability.
