## Q71878: Resource Kit Installation Leaves PPTMP Directory in Windows

	Article: Q71878
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you install the Microsoft Windows Resource Kit and choose to install the
	Microsoft Windows Productivity Pack for Windows 3.0, the Install program leaves
	a PPTMP directory with the compressed files in it.
	
	MORE INFORMATION
	================
	
	After the installation process is completed, this directory is no longer needed
	and can be deleted through Windows by using File Manager or through MS-DOS by
	using the appropriate commands.
	
	REFERENCES
	==========
	
	Microsoft Windows Resource Kit, page 180.
