## Q184448: HTML In Action Back Cover Lists Incorrect Contents for CD-ROM

	Article: Q184448
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocerr
	Last Modified: 25-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS HTML in Action ISBN 1-55615-948-X 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The back cover of "HTML in Action" notes that the following items are included
	on the companion CD-ROM:
	
	- TrueType fonts
	
	- Microsoft Internet Explorer
	
	- Internet Control Pack
	
	- Starter set of ActiveX controls
	
	However, only Internet Explorer is included on the CD-ROM.
	
	RESOLUTION
	==========
	
	There is no resolution at this time.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in "HTML in Action".
	
	MORE INFORMATION
	================
	
	The companion CD-ROM contains the following items:
	
	- Sample HTML files from chapters 1-5 and chapter 11.
	
	- A beta version of Microsoft Internet Explorer 3.0
	
	- The Internet Assistants for Microsoft Word 95, Excel 95, PowerPoint 95,
	  Access 95, and Schedule+ 2.0.
