## Q184012: FP97: Page Doesn't Preview in Window Size Specified

{% raw %}

	Article: Q184012
	Product(s): Word Front Page
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdta
	Last Modified: 26-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft FrontPage 97 for Windows 
	-------------------------------------------------------------------------------
	
	For a Microsoft FrontPage 98 version of this article, see Q193949.
	
	SYMPTOMS
	========
	
	Your Web page is not displayed in the window size you specified when you preview
	your page in a Web browser.
	
	CAUSE
	=====
	
	This behavior occurs if the Web browser is not running when you attempt to
	preview the page. In contrast, the window is sized correctly if the browser is
	running when you attempt to preview the page.
	
	RESOLUTION
	==========
	
	To preview the page using the window size you want, follow these steps:
	
	1. Start your Web browser.
	
	2. Open your page in FrontPage Editor.
	
	3. On the File menu, click Preview In Browser.
	
	4. Click the Window Size option you want and then click Preview.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
