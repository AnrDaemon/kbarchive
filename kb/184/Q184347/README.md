## Q184347: Program Is Still Listed After It Is Uninstalled

{% raw %}

	Article: Q184347
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): kbenv kbsetup win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you uninstall a program, it may still appear in the list of installed
	programs in the Add/Remove Programs tool in Control Panel.
	
	CAUSE
	=====
	
	This behavior can occur if the program's uninstall utility has not removed all
	of the registry entries associated with the program, or the uninstall was not
	successfully completed.
	
	RESOLUTION
	==========
	
	To resolve this issue, reinstall the program and then uninstall the program. If
	this does not resolve the issue, contact the program vendor for additional
	assistance.
