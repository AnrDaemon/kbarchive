## Q184404: XGEN: How to Obtain Exchange Server 5.5 Evaluation Edition

{% raw %}

	Article: Q184404
	Product(s): Microsoft Exchange
	Version(s): 5.5
	Operating System(s): 
	Keyword(s): exc55
	Last Modified: 26-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Exchange Server 5.5 Evaluation Edition is available for a 120 day
	trial. This program can be downloaded from the World Wide Web at the following
	URL:
	
	  http://www.microsoft.com/exchange/evaluation/overview/howeval.asp
	
	You can also order the product on CD-ROM for a nominal charge.
	
	MORE INFORMATION
	================
	
	To order the Exchange Server 5.5 Evaluation Edition on CD-ROM, please see the
	above URL. The CD-ROM contains the complete Microsoft BackOffice Trial Kit,
	which includes the following:
	
	- Microsoft Windows NT[ASCII 174] Server 4.0 with the Windows NT Option Pack
	  (120 days)
	
	- Microsoft SQL Server[ASCII 153] 6.5(120 days)
	
	- Microsoft Exchange Server 5.5 (120 days)
	
	- Microsoft Site Server Commerce Edition 3.0 Beta (through June 30, 1998)
	
	- Microsoft Proxy Server 2.0 (30 days)
	
	- Microsoft SNA Server 4.0 (120 days)
	
	- Microsoft SMS Server 1.2 (120 days)
	
	There is an Enterprise version of the CD-ROM, which includes the following:
	
	- Microsoft Windows NT Server 4.0, Enterprise Edition with the Option Pack (120
	  days)
	
	- Microsoft SQL Server 6.5, Enterprise Edition (120 days)
	
	- Microsoft Exchange Server 5.5, Enterprise Edition (120 days)
	
	If you decide to download the Exchange Server 5.5 Evaluation Edition from the
	Internet, it is recommended that you use a connection faster than 28.8 KBps.
	
	The following components are available for download:
	
	- Microsoft Exchange Server 5.5 for Windows NT Server (Intel) - 23 MB
	
	- Microsoft Exchange Server 5.5 for Windows NT Server (Alpha) - 34 MB
	
	- Microsoft Exchange Server 5.5 Connectors (Intel) - 3 MB
	
	- Microsoft Exchange Server 5.5 Connectors (Alpha) - 5 MB
	
	- Microsoft Exchange Server 5.5 Documentation (Intel) - 10 MB
	
	- Microsoft Exchange Server 5.5 Documentation (Alpha) - 10 MB
	
	- Microsoft Exchange Server 5.5 Support files (Intel). Includes useful
	  utilities, collaboration samples and debugging symbols - 35 MB
	
	- Microsoft Exchange Server 5.5 Support files (Alpha). Includes useful
	  utilities, collaboration samples and debugging symbols - 40 MB
	
	- Microsoft Outlook97 8.03 (Windows NT Intel) - 23 MB
	
	- Microsoft Outlook97 8.03 (Windows 95) - 23 MB
