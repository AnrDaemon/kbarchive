## Q184493: INFO: Programmer's Guide to Using Index Server Provider in VC++

	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual C++, 32-bit Enterprise Edition, versions 5.0, 6.0 
	- Microsoft Visual C++, 32-bit Professional Edition, versions 5.0, 6.0 
	- Microsoft Visual C++, 32-bit Learning Edition, version 6.0

	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	
	This article presents information that can be useful to anyone wanting to take
	advantage of the OLE DB Provider for Index Server (MSIDXS) in a Visual C++ 5.0
	ADO application to query Index Server 2.0. The following topics are discussed:
	
	- How to Obtain the OLE DB Provider for Index Server (MSIDXS)
	
	- MSIDXS and Index Server Resources/Points of Information
	
	- How to Find Knowledge Base Articles on MSIDXS Provider
	
	
	
	MORE INFORMATION
	================
	
	How to Obtain the OLE DB Provider for Index Server (MSIDXS)
	-----------------------------------------------------------
	
	Index Server 2.0 and the OLE DB Provider for Index Server (MSIDXS) ship with
	Windows NT 4.0 Option Pack, available free for download at:
	
	  http://www.microsoft.com/ntserver
	
	MSIDXS and Index Server Resources/Points of Information
	-------------------------------------------------------
	
	For more information on Index Server, refer to the Index Server documentation in
	Windows NT Option Pack documentation. The chapter "SQL Access to Index Server
	Data" is the best reference for SQL query syntax for Index Server.
	
	How to Find Knowledge Base Articles on MSIDXS Provider
	------------------------------------------------------
	
	To find articles on MSIDXS Provider in the Microsoft Knowledge Base (KB) on the
	Web, connect to
	
	  http://msdn.microsoft.com/support/
	
	On this site, select "OLE DB" as the product. To search for other Index Server
	articles, select "Index Server" as the product.
	
	For the MSDN or other non-Web-based Knowledge Base search mechanisms, you can use
	the keyword oledbProvIndex, which is included in all articles on MSIDXS
	Provider.
