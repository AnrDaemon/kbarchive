## Q184295: Compaq System Management Driver 2.05B in Service Pack 4

{% raw %}

	Article: Q184295
	Product(s): Microsoft Windows NT
	Version(s): WinNT:4.0
	Operating System(s): 
	Keyword(s): kbFEA kbWinNT400sp4fea
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 SP4 
	- Microsoft Windows NT Server version 4.0 SP4 
	- Microsoft Windows NT Server, Enterprise Edition version 4.0 SP4 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Windows NT 4.0 Service Pack 4 includes the Compaq System Management Driver
	(Sysmgmt.sys) from the Compaq Support Software for Microsoft Windows NT 4.0
	Support Software Diskette (SSD), version 2.05, revision B.
	
	MORE INFORMATION
	================
	
	When you apply Windows NT 4.0 Service Pack 4, any previous version of this
	driver is replaced. For additional information, please refer to the
	documentation for the Compaq SSD for Microsoft Windows NT 4.0 available at the
	following Compaq web site:
	
	  http://www.compaq.com/
