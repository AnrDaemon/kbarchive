## Q184437: XADM: Information Store Stops with Large Number of NNTP Users

	Article: Q184437
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-APR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	If a large number of Network News Transfer Protocol (NNTP) users are connected
	to a Microsoft Exchange Server computer, the information store may stop
	unexpectedly, and an error message similar to the following may occur:
	
	  The instruction at 0x77f0515d referenced memory at 0x00050026. The
	  memory could not be written.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server
	version 5.5. This problem has been corrected in the latest U.S. Service Pack for
	Microsoft Exchange Server version 5.5. For information about obtaining the
	Service Pack, query on the following word in the Microsoft Knowledge Base
	(without the spaces):
	
	  S E R V P A C K
