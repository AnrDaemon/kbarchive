## Q184764: XADM: Disk Space Required For Upgrade From Version 4.0 To 5.0

	Article: Q184764
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:4.0,5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 20-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 4.0, 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	To successfully upgrade from Microsoft Exchange Server version 4.0 to version
	5.0, the server must have enough free disk space equal to or greater than the
	size of the Dir.edb file.
	
	This free disk space must be on the same drive as the Dir.edb file.
