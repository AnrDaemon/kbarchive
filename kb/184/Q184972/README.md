## Q184972: Golf 98/99: Sounds Are Not Played in Multiplayer Game

	Article: Q184972
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbsound kbtlc kbimu msgame golf98kbfaq
	Last Modified: 09-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Golf 1998 Edition, version 1.0 
	- Microsoft Golf 1999 Edition, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are playing a multiplayer game of Microsoft Golf over the Microsoft
	Zone.com, you may not hear any sounds in the game.
	
	RESOLUTION
	==========
	
	To resolve this issue, follow these steps:
	
	1. Click Menu, point to Options, and then click Game Preferences.
	
	2. If the value inside the box beside Game Sounds is No, click the box to change
	  the value to Yes.
	
	  If the value inside the box beside Game Sounds is Yes, click the box to change
	  the value to No, and then click the box to change the value back to Yes.
	
	3. Click OK.
	
	Additional query words: msgame golf98 zone msngz
