## Q184710: XADM: Exchange Admin Program Will Not Run On Windows 95/98

	Article: Q184710
	Product(s): Microsoft Exchange
	Version(s): winnt:4.0,5.0,5.5
	Operating System(s): 
	Keyword(s): exc4 exc5 exc55
	Last Modified: 03-JUN-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 4.0, 5.0, 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Exchange Server Administrator program will not install or run on
	Windows 95/98.
	
	MORE INFORMATION
	================
	
	The Exchange Server Administrator program requires Windows NT workstation or
	Windows NT Server.
	
	Additional query words: Win95 OSR2
