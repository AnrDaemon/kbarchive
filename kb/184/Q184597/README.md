## Q184597: Visual Basic Script Error Message When You Connect to MSN

	Article: Q184597
	Product(s): The Microsoft Network
	Version(s): WINDOWS:2.51,2.52,2.6,4.0,4.01,4.01 Service Pack 1,4.01 Service Pack 2,5,95,98,98 Secon
	Operating System(s): 
	Keyword(s): kbenv kberrmsg kbinterop kbmsn
	Last Modified: 18-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 2.51, 2.52, 2.6 
	- Microsoft Internet Explorer versions 4.0, 4.01, 4.01 Service Pack 1, 4.01 Service Pack 2, 5 for Windows 95 
	- the operating system: Microsoft Windows 95 
	- the operating system: Microsoft Windows 98 
	- the operating system: Microsoft Windows 98 Second Edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you connect to MSN, The Microsoft Network, you may receive a Visual Basic
	script error message and you may not be able to access MSN On Stage or other
	sites on MSN.
	
	CAUSE
	=====
	
	This behavior can occur if the security level of the zone in which MSN is
	located is set too high.
	
	RESOLUTION
	==========
	
	To resolve this issue, follow these steps:
	
	1. Click the MSN Quick View icon on the taskbar, point to MSN Options, and then
	  click Internet Properties.
	
	2. Click the Security tab.
	
	3. In the Zone box, click each of the selections, and then click Reset for each
	  one individually.
	
	4. Click OK.
	
	Additional query words: msnet msnetwork microsoft-net m.s.n. 2.51 2.60
