## Q184509: How to Obtain SNA Server 4.0 Printed Documentation

	Article: Q184509
	Product(s): Microsoft SNA Server
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-MAR-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft SNA Server, version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft SNA Server 4.0 product CD no longer includes the product
	documentation in Word format, though it continues to include the documentation
	in online Help file format. The printed manuals can be ordered from Microsoft by
	referring to the SNA Server 4.0 Documentation Kit No. 211-00144. Also, the
	online Help files can be downloaded from ftp.microsoft.com.
	
	Additionally, Microsoft no longer produces printed manuals for the SNA Server
	Software Developer Kit programmer guides. The SDK manuals are included in online
	format with the product CD and can be optionally installed during installation
	of SNA Server 4.0.
	
	MORE INFORMATION
	================
	
	Ordering Printed SNA Server Documentation
	-----------------------------------------
	
	The SNA Server 4.0 Documentation Kit (Kit No. 211-00144) includes the following
	manuals:
	
	- Microsoft SNA Server Getting Started
	
	- Microsoft SNA Server Administration Guide
	
	- Microsoft SNA Server Planning Guide
	
	- Microsoft SNA Server Reference
	
	Copies of the documentation kit can be obtained using one of the following
	methods:
	
	If you currently own SNA Server or BackOffice:
	
	The documentation kit is available for U.S. $29.95/CDN $39.95. To order, complete
	the coupon at the back of the Microsoft SNA Server Getting Started manual
	included with the product and mail the coupon and payment to one of the
	following addresses:
	
	In the United States:
	
	Microsoft SNA Server Documentation Offer
	P.O. Box 1096
	Buffalo, NY 14240-1096
	
	In Canada:
	
	Microsoft SNA Server Documentation Offer
	Microsoft Canada Order Centre
	P.O. Box 643
	Fort Erie, Ontario L2A 6M1
	
	Please allow 2 to 4 weeks for delivery upon receipt of the order coupon by
	Microsoft. Offer good while supplies last. Offer good in the 50 United States,
	the District of Columbia, and Canada only. Offer not valid in U.S. Territories,
	including Puerto Rico, U.S. Virgin Islands, and Guam. To check the status of
	your order, or if you have any questions about this offer, call the Microsoft
	Supplemental Order Center at (800) 360-7561 (no phone orders). Customers who are
	deaf or hard-of-hearing can reach Microsoft text telephone (TTY/TDD) services by
	calling (800) 718-1599 (in the United States) or (905) 568-9641 (in Canada).
	
	If you are currently evaluating SNA Server, or located outside the 50 United
	States, the District of Columbia, or Canada: Contact your local Microsoft
	subsidiary or reseller for availability and pricing. Quote Kit No. 211-00144
	when ordering.
	
	The following self-extracting file contains the SNA Server 4.0 online version of
	documentation, stored in Windows Help file format:
	
	  ftp://ftp.microsoft.com/bussys/winnt/sna-docs/sna40doc.exe
	
	The Help files can be printed using the standard Windows Help engine, though
	there is no table of contents or index.
	
	The SNA Server 4.0 SDK online documentation, "infoview" viewing tool, and SDK
	tools can be downloaded separately. The SDK online documentation is stored in
	"infoview" format (the format used by other Microsoft SDK products):
	
	  ftp://ftp.microsoft.com/bussys/winnt/sna-docs/sna4sdk.exe
