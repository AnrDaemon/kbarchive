## Q121884: Golf Err Msg: Not Enough Memory to Zoom Top View

	Article: Q121884
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-JUN-2001
	
	kbother kbprb kberrmsg
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Golf for Windows, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The error message
	
	  Not Enough Memory To Zoom Top View
	
	occurs when you do the following:
	
	1. Either zoom in or zoom out, at least once, using the magnifying buttons on
	  the Top View box.
	
	2. Resize the Top View box, vertically, to the point where only a few pixels of
	  the Top View are showing in the box.
	
	The error message occurs and then causes the computer to stop responding (hang).
	
	MORE INFORMATION
	================
	
	In some cases you may receive other error message after the "Not Enough Memory
	To Zoom Top View" error message. Invariably, the computer will stop responding.
	
	Additional query words: 2.00 lock hang magnify reboot crash freeze lockup up
