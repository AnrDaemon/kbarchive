## Q121689: TCP/IP Help File States Incorrect Definition for LPQ

	Article: Q121689
	Product(s): Microsoft Windows NT
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	3.50
	
	WINDOWS
	
	kbnetwork kbdocerr
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.5 
	- Microsoft Windows NT Server version 3.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft TCP/IP Help file, TCPIP.HLP, that comes with Windows NT
	Workstation and Windows NT Server version 3.5 displays an incorrect definition
	for LPQ. After choosing the Printing and TCP/IP topic button, select the
	Overview of TCP/IP Printing subtopic. When you select LPQ, it incorrectly
	displays the definition for LPR.
