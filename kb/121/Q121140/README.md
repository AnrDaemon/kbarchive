## Q121140: Space: Some Scenery Objects Are Identical

	Article: Q121140
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-JUN-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Space Simulator for MS-DOS, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Some scenery objects in Space Simulator are identical in shape and surface
	texture.
	
	MORE INFORMATION
	================
	
	Stars in Space Simulator are all identical in shape and texture, but vary in
	size and color. Comets, asteroids, and some moons in Space Simulator are
	identical in shape, texture, and color and differ only in size. This is done to
	maximize Space Simulator's performance and reduce necessary disk space.
