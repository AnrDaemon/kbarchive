## Q121031: Bookshelf 1994: Tables Do Not Print Properly

{% raw %}

	Article: Q121031
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1994 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for Windows 1994 edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If the table you want to print has more columns than will print on one page, the
	columns that will not fit on the first page will not print.
	
	MORE INFORMATION
	================
	
	Steps to Reproduce Problem
	--------------------------
	
	1. Go the Almanac.
	
	2. Select the Find tab, and search for Nutrition.
	
	3. Print the Nutritive Value of Food.
	
	EXPECTED RESULT: The columns that do not fit on the first page should print on a
	different page.
	
	ACTUAL RESULT: The columns that do not fit on the first page do not print.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Bookshelf 1994. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
