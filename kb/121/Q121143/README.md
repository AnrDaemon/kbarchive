## Q121143: Flight Simulator Available Media

	Article: Q121143
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:5.0,5.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	kbusage
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for MS-DOS, versions 5.0, 5.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Flight Simulator version 5.0 is available on the following media:
	
	- Flight Simulator 5.0 3.5 LD
	
	- Flight Simulator 5.0a 3.5 HD
	
	- Flight Simulator 5.0 5.25 HD
	
	- Flight Simulator 5.0a 5.25 HD
	
	- Flight Simulator 5.0 San Francisco Scenery 3.5 HD
	
	- Flight Simulator 5.0 San Francisco Scenery 5.25 HD
	
	- Flight Simulator 5.0 Paris Scenery 3.5 HD
	
	- Flight Simulator 5.0 Paris Scenery 5.25 HD
	
	- Flight Simulator 5.0 New York 3.5 HD
	
	- Flight Simulator 5.0 New York 5.25 HD
	
	- Flight Simulator 5.0a Update 3.5 HD
	
	- Flight Simulator 5.0a Update 5.25 HD
	
	Additional query words: 5.00 5.00a flightsim fltsim fs5 dos disk
