## Q121086: Obtaining the NetShare Utility for Workgroup Add-On for MS-DOS

	Article: Q121086
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:3.11
	Operating System(s): 
	Keyword(s): kbfile msdos kbgraphxlinkcritical
	Last Modified: 21-DEC-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Workgroup Add-On for MS-DOS, version 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	NetShare is a utility you can use to easily share directories and printers in
	Microsoft Workgroup Add-On for MS-DOS, version 3.11. NetShare replaces the
	command-line sharing function with a user-friendly interface.
