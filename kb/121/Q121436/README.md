## Q121436: Doc Err: Entry for Port of LPD Service Not Available

	Article: Q121436
	Product(s): Microsoft Windows NT
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	3.50
	
	WINDOWS
	
	kbnetwork kbdocerr
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.5 
	- Microsoft Windows NT Server version 3.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Step 5 on page 135 of the Windows NT Workstation Installation Guide shows the
	Add LPR Compatible Printer dialog box with three entry lines. The "Port of LPD
	service on that machine" entry is not available in Windows NT version 3.5.
