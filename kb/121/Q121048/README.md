## Q121048: Determining the Version Number for HP DeskJet Drivers 4.x, 5.x

	Article: Q121048
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.1 
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to determine the Hewlett-Packard (HP) DeskJet printer
	driver version for the 4.x and 5.x series of drivers.
	
	MORE INFORMATION
	================
	
	In Windows Program Manager, use the following procedure to determine the version
	number:
	
	1. In Windows Control Panel, choose the Printers icon.
	
	2. From the Installed Printers list, select the HP DeskJet printer.
	
	3. Choose the Setup button.
	
	4. From the Control menu in the HP DeskJet dialog box, choose About. The printer
	  driver version number is displayed in the About dialog box. (To open the
	  Control menu, press ALT+SPACEBAR or click the Control-menu box in the
	  upper-left corner of the dialog box.)
	
	In most Windows-based applications, you can also use the following procedure to
	determine the version number:
	
	1. Open the Printer Setup dialog box. To do this, use the appropriate step
	  below.
	
	  a. In Microsoft Word 6.0 and Microsoft PowerPoint 4.0, choose Print from the
	     File menu and then choose the Printer button.
	
	  b. In Microsoft Excel 5.0, choose Print from the File menu and then choose
	     the Printer Setup button.
	
	  c. In Microsoft Access 2.0, choose Printer Setup from the File menu.
	
	  d. In Microsoft Mail 3.2, choose Print Setup from the File menu.
	
	2. From the Installed Printers list, select the HP DeskJet printer.
	
	3. Choose the Setup button.
	
	4. From the Control menu in the HP DeskJet dialog box, choose About. The printer
	  driver version number is displayed in the About dialog box. (To open the
	  Control menu, press ALT+SPACEBAR or click the Control-menu box in the
	  upper-left corner of the dialog box.)
