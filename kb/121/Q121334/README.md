## Q121334: Encarta 1994: Search Not Sorted By Word Frequency

	Article: Q121334
	Product(s): Microsoft Home Kids Products
	Version(s): 1994 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 26-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 1994 The Complete Multimedia Encyclopedia 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The list of topics produced by the Find command are not sorted in order of
	occurrence.
	
	MORE INFORMATION
	================
	
	The Encarta Help Topic incorrectly states regarding the Find command:
	
	  The topics are listed by title, with the topic containing the most
	  occurrences of the word or words listed first.
	
	NOTE: To find the above information in the Encarta Help file, choose Topics from
	the Help menu and select:
	
	1. The Button Bar: What Does It Do?,
	
	2. Finding Topics That Contain Certain Words.
	
	3. Doing a Simple Search Using Find
	
	The topics are neither sorted by frequency nor alphabetical order.
