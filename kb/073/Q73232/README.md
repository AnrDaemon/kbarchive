## Q73232: Cannot Delete Files in MS-DOS Executive with DELETE Key

	Article: Q73232
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	When using the MS-DOS Executive, you cannot delete files with the DELETE
	key. The files can only be deleted by using the Delete option in the File
	Menu.
