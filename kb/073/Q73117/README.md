## Q73117: MSU Introduces MS-DOS 5 Video Course for Support Engineers

	Article: Q73117
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft University's MS-DOS version 5.0 video course titled "Microsoft MS-DOS
	5 Installation and Optimization Video Training for Support Engineers and
	Advanced Users" has been produced and is available through Microsoft University
	(MSU). This video course consolidates information for support engineers who need
	to successfully install, optimize, customize, and support MS-DOS 5.0 on a group
	of PCs.
	
	For orders, support or further information, please call (425) 828-1507, dept.
	3005, or write to the following address for more information:
	
	  Microsoft University
	  MS-DOS 5 Video Course
	  One Microsoft Way
	  Redmond, WA 98052-6399.
