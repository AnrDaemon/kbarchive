## Q73411: Commenting Lines in DOSSHELL.INI

	Article: Q73411
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.x,6.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 5.0, 5.0a, 6.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	A semicolon (;) in the DOSSHELL.INI file signifies the beginning of a comment
	and causes all text to the right of the semicolon to be ignored when MS-DOS
	Shell is started. Thus, you can disable a line in the DOSSHELL.INI file by
	prefixing the line with a semicolon.
	
	Additional query words: 5.00 5.00a 6.00 6.20 dosshell dos shell
