## Q73225: Everex EGA Card Hangs When Running MS-DOS Apps

	Article: Q73225
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Your system may stop responding (hang) while running MS-DOS applications inside
	or outside of Microsoft Windows version 3.0 if you have an Everex EGA card model
	EV659.
	
	CAUSE
	=====
	
	Everex EGA card models EV659 with ROM versions earlier than 1.56 cause this
	problem.
	
	RESOLUTION
	==========
	
	Everex offers upgrade ROMS for this card. The new ROMs are available from Everex
	Technical Support.
	
	MORE INFORMATION
	================
	
	Contact Everex for more information.

	of Microsoft; we make no warranty, implied or otherwise, regarding these
	products' performance or reliability.
