## Q73415: Can't Backup/Restore from Peachtree Complete 3 with MS-DOS 5.0

	Article: Q73415
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 23-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 5.0 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	When you run the Peachtree Complete 3 Accounting Package with MS-DOS version
	5.0, you may not be able to backup/restore to drive A from the Utilities menu.
	The drive lights stay on and the system appears to hang. A patch is available
	from Peachtree to correct the backup/restore problem. This patch does not allow
	backup/restore options for the Peachtree Bank Account Manager.
	
	Customers should contact Peachtree Support for the patch.
