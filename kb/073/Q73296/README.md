## Q73296: MS-DOS 5.0 Technical Reference Availability

	Article: Q73296
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 26-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The technical reference "The MS-DOS Programmer's Reference" (product #
	1-55615-329-5), which includes information on MS-DOS version 5.0, is available
	through Microsoft Press, Penguin Books, and other retail outlets. It was
	officially released on June 13, 1991.
	
	MORE INFORMATION
	================
	
	U.S. Orders
	-----------
	
	To order the reference within the U.S., call Microsoft Press.
	
	International Orders
	--------------------
	
	To order the reference from outside of the U.S., call Penguin Books at
	0-11-081-071-938-2200.
