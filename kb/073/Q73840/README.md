## Q73840: Emulation Modes of DataProducts SPG8051 Printer

{% raw %}

	Article: Q73840
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	According to DataProducts technical support, the DataProducts SPG8051 printer
	can emulate the IBM Color Printer only.
	
	NOTE: It has been reported that you may experience difficulties when switching
	from NLQ mode to draft mode.
	
	MORE INFORMATION
	================
	
	For further information, contact DataProducts.

	we make no warranty, implied or otherwise, regarding this product's performance
	or reliability.
