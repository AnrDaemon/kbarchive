## Q73453: WordPerfect 5.1 and EMM386.EXE Conflict

	Article: Q73453
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.x,6.0,6.2,6.21,6.22; WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 5.0, 5.0a, 6.0, 6.2, 6.21, 6.22 
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Any WordPerfect 5.1 program dated earlier than 5/31/91 may not work correctly
	with MS-DOS EMM386.EXE dated 4/9/91 or later. WordPerfect users should obtain
	new Install Learn Utility Disks from WordPerfect.

	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
