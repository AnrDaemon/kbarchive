## Q73630: Cannot Swap Out of WordPerfect 5.1 in 132 Column Video Mode

	Article: Q73630
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When attempting to swap out of WordPerfect 5.1 with MS-DOS Shell's Task Swapper,
	the computer beeps and you remain in WordPerfect. This problem occurs when
	WordPerfect is set in 132 column video mode. Switching out of WordPerfect's 132
	column mode corrects the problem.

	warranty, implied or otherwise, regarding this product's performance or
	reliability.
