## Q73385: Notebook 386SX Screen Saver Reboots in Enhanced Mode

	Article: Q73385
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Notebook 386SX computer by Associates Mega Subsystems has a built-in screen
	saver. If Windows 3.0 is running in enhanced mode, the system reboots 4 minutes
	after the screen saver is activated. Standard and real modes are not affected.
	
	MORE INFORMATION
	================
	
	For more information, contact Associates Mega Subsystems.
	
	The product discussed here is manufactured by a vendor independent of Microsoft;
	we make no warranty, implied or otherwise, regarding this product's performance
	or reliability.
