## Q73604: Keytronics 122-Key Keyboard Compatible with Windows

	Article: Q73604
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Keytronics 122-key keyboard is designed for 3270 emulation. It is compatible
	with Microsoft Windows 3.0 or 3.1 if it is installed as an "Enhanced 101 or 102
	key US and Non US keyboards". However, you cannot use the additional keys when
	running Windows.
	
	MORE INFORMATION
	================
	
	For more information, call Keytronics Technical Support.

	or reliability.
