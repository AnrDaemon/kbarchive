## Q201301: FP98: Office Binder Documents Are Not Recognized

{% raw %}

	Article: Q201301
	Product(s): Word Front Page
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbusage kbInternet kbInetDevkbbuglist
	Last Modified: 28-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft FrontPage 98 for Windows 
	- Microsoft Excel 97 for Windows 
	- Microsoft Office Binder 97 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you add a Microsoft Office Binder document to your web that has hyperlinks
	in one of its components, FrontPage will not be able to display or track those
	hyperlinks.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
	
	MORE INFORMATION
	================
	
	
	Additional query words: Office Binder Document internal hyperlinks
