## Q201506: XFOR: Alarm Time Wrong After French Notes Migration

{% raw %}

	Article: Q201506
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-APR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When using the French version of the Microsoft Exchange Notes migration tool,
	the alarm settings for calendar entries will migrate incorrectly.
	
	For example, an alarm setting of 20 hours before the scheduled event would be 740
	hours after the migration.
	
	CAUSE
	=====
	
	This occurs because of problems with the French version of Microsoft Outlook 98.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server,
	version 5.5.
