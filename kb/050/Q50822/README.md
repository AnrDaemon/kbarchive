## Q50822: Second Print Job Fails after Deleting a Printer Driver

{% raw %}

	Article: Q50822
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	The Microsoft Windows version 3.0 Control Panel allows you to delete a
	printer while an application is printing to that printer.
	
	The print job completes normally because it can still access the
	correct driver; however if you try to start another print job later
	with the same driver, it fails.
	
	Additional query words: 3.00 3.00a 3.10 3.11
