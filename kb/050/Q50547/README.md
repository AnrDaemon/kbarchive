## Q50547: White Text Prints in Black

	Article: Q50547
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Text formatted as the color white prints black on black-and-white printers.
	Black-and-white drivers print all colored text on the screen in black. White, in
	this case, is considered a "color." This behavior is normal operation.
