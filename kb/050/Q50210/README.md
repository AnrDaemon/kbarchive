## Q50210: Flt Sim: Plane Position Shifts 2 Degrees per Minute

	Article: Q50210
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	After performing a bank and then centering the rudder and ailerons, the position
	of the plane shifts to the left at about 2 degrees per minute in Flight
	Simulator version 4.00.
	
	Microsoft has confirmed this to be a problem in Flight Simulator version 4.00.
	This problem was corrected in Flight Simulator version 4.00b.
