## Q50825: Terminal Uses Only 24 Lines

{% raw %}

	Article: Q50825
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	The Windows Terminal application that ships with Microsoft Windows
	and with Microsoft Windows for Workgroups uses only 24 lines of the
	display, even if a smaller font is selected.
	
	This behavior is intentional. Terminal was designed to scroll at the
	24-line position no matter what font size is used.
