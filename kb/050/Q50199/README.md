## Q50199: Black Scenery with Everex EGA in Flight Simulator

{% raw %}

	Article: Q50199
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	The Everex EGA (EV-659) card reportedly produces black water and land scenery in
	Flight Simulator Version 4.00. Everex has been notified of this problem and is
	researching it. The latest version of the card is 1.54.09.
	
	The Everex EGA (EV-659) card was not part of the configuration testing for Flight
	Simulator Version 4.00. Microsoft cannot guarantee proper results with this
	card.
	
	Everex can be reached at (415) 498-1115 for further details.
