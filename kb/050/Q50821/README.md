## Q50821: Windows Write 3.0 Cannot Search for &quot;?&quot;

	Article: Q50821
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 3.0,3.0a
	Operating System(s): 
	Keyword(s): kbtool
	Last Modified: 12-JUN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft Windows Write included with the Windows Version 3.0 package cannot
	search for the "?" (question mark) character, because this is used as a
	"wildcard" character.
