## Q186301: XWEB: Outlook Web Access Does Not Include Outlook Today

	Article: Q186301
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook Web Access, version 5.5 
	- Microsoft Outlook Web Access, version 5.5 Service Pack 1 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Microsoft Outlook Web Access does not include Outlook Today. Outlook Today
	allows the user to see, at a glance, the number of new mail messages, tasks for
	today, and Calendar items for today.
	
	CAUSE
	=====
	
	This functionality is not supported in the Outlook Web Access client.
	
	RESOLUTION
	==========
	
	Outlook Today can only be accessed by using the Outlook 98 client.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Outlook Web Access versions 5.5
	and 5.5 SP1.
