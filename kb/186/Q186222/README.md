## Q186222: BUG: Pasting Metafiles into Office Loses Lines and Box

	Article: Q186222
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Under Windows NT 4.0, if you insert Visio 5.0 enhanced metafile (EMF) graphics
	into a PowerPoint 97 presentation, save the presentation to PowerPoint 95
	format, close it, and then open it in PowerPoint 97 or PowerPoint 95, lines and
	boxes are sometimes lost.
	
	The problem does not occur on Windows 95, and it does not occur with all EMF
	graphics.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a bug in the Microsoft products listed at the
	beginning of this article. We are researching this bug and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
