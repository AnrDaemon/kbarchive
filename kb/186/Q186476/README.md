## Q186476: Changing Icon Names Changes the Name for All Users

	Article: Q186476
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-DEC-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0, Terminal Server Edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If an icon name is changed at the Terminal Server, that icon is renamed for all
	users who are currently logged on, rather than just for the users who are logged
	on at the server console. This behavior is scheduled to be changed in Terminal
	Server 5.0.
