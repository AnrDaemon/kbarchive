## Q186460: Setup Displays Completed Message But DirectX Is Not Installed

{% raw %}

	Article: Q186460
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0,2.0
	Operating System(s): 
	Keyword(s): kbenv kbsetup aoe ccabtf fs98 kbimu KbDirectX golf98 bball3d outwars mtm2kbfaq
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Age of Empires, version 1.0 
	- Microsoft Baseball 3D, version 1.0 
	- Microsoft CART Precision Racing, version 1.0 
	- Microsoft Close Combat: A Bridge Too Far, version 2.0 
	- Microsoft Flight Simulator 98 
	- Microsoft Golf 1998 Edition, version 1.0 
	- Microsoft Golf 1999 Edition, version 1.0 
	- Microsoft Monster Truck Madness 2, version 2.0 
	- Microsoft Outwars, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to reinstall DirectX, you may immediately receive the following
	message:
	
	  DirectX setup has completed successfully.
	
	CAUSE
	=====
	
	This behavior can occur if you attempt to install a version of DirectX that is
	older than the version of DirectX installed on your computer.
	
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest version of DirectX from
	the following Microsoft Web site:
	
	http://www.microsoft.com/directx/
	
	For information about how to download and install DirectX, please see the
	following article in the Microsoft Knowledge Base:
	
	  Q179113 How to Download and Install DirectX
