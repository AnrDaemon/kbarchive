## Q186846: WD97: New Information in Help About for New English Thesaurus

	Article: Q186846
	Product(s): Word 97 for Windows
	Version(s): Service Release 2
	Operating System(s): 
	Keyword(s): kbdta word97
	Last Modified: 23-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows, version Service Release 2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you review the information in the About Microsoft Word dialog box (click
	About Microsoft Word on the Help menu), the following information will appear
	for the Thesaurus:
	
	  English Thesaurus content developed for Microsoft by
	  Bloomsbury Publishing Plc.
	
	Earlier versions of Word will have the following information:
	
	  Thesaurus (c)1984-1996 Soft-Art, Inc.(R) All rights reserved.
	
	MORE INFORMATION
	================
	
	This information was changed when you installed Microsoft Word Service Release 2
	(SR-2). The Soft-Art Thesaurus was removed from all versions of Word except for
	the Spanish and German versions and is being replaced with a new English
	Thesaurus developed by Bloomsbury Publishing Plc. For each version of Word that
	ships with the new English Thesaurus, the Microsoft Word Help About dialog box
	will reflect the new information as shown in the Summary of this article.
