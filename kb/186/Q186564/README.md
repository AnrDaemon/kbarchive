## Q186564: Visible Servers in Terminal Server Administration

	Article: Q186564
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-DEC-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0, Terminal Server Edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Terminal Server Administration tool can be used to view and administer
	Terminal Servers in the same domain. Terminal Servers in trusted domains are
	also visible.
	
	For example, when an administrator uses Terminal Server Administrator on a
	Terminal Server computer that is a member of domain A, other Terminal Servers in
	domain A will be visible. If domain A trusts domain B, domain B's Terminal
	Server computers will also be visible. However, Terminal Servers in domains that
	trust domain A will not be visible.
