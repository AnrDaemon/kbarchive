## Q186911: XWEB: Cancelled Meeting Request Is Sent With High Importance

	Article: Q186911
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook Web Access, version 5.5 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	
	When you cancel a meeting using the Outlook Web Access client, the cancellation
	notification is sent with high importance to all other meeting attendees.
	
	MORE INFORMATION
	================
	
	This is the same behavior as the Outlook client and it is by design.
