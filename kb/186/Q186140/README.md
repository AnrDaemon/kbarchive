## Q186140: WD97: Clicking Hyperlink to View Web Page Closes Document

{% raw %}

	Article: Q186140
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta kbhtml word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click a hyperlink to view a Web page in a browser, the document in Word
	closes.
	
	NOTE: To return to the document containing the hyperlink, click the Back button
	in your browser.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
	
	This problem has been corrected in Word 2000.
	
	MORE INFORMATION
	================
	
	This behavior occurs when you click a hyperlink to a Web page in a saved Word
	document. In contrast, if you click a hyperlink to a Web page in an unsaved Word
	document, Word does not close the document.
	
	For additional information about hyperlinks, please see the following article in
	the Microsoft Knowledge Base:
	
	  Q159940 WD97: General Information About Hyperlinks
