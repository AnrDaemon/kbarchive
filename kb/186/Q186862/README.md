## Q186862: XWEB: OWA Logon Help TOC Label is Always &quot;Welcome&quot;

	Article: Q186862
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook Web Access, version 5.5 Service Pack 1 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	When you use the Help link on the Logon.asp page in the Outlook Web Access Client
	by selecting "For more information about this Outlook product, CLICK HERE," all
	content links display "Welcome" instead of the subject heading.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Outlook Web Access,
	version 5.5 Service Pack 1.
