## Q158021: GEX: System Requirements

	Article: Q158021
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbfaq
	Last Modified: 08-NOV-2001
	
	1.0
	WINDOWS
	kbusage kbreadme kbfaq kbfasttip
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft GEX, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	To run GEX, you will need:
	
	- Pentium 75 with local SVGA bus video, 256-color display
	- Microsoft Windows 95
	- 2X CD-ROM drive
	- 8 MB of memory (RAM)
	- 10 MB of available hard disk space
	- DirectX 2.0
	
	Recommended features:
	
	- Pentium 100 with local SVGA bus video, high-color (16-bit) display
	- 4X CD-ROM drive
	- 16+ MB of memory (RAM)
	- 2 MB of video memory
	- Joystick or four-button game pad
	- Windows 95-compatible sound card
