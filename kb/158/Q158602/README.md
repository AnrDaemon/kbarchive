## Q158602: WD97: Cannot Extend Selection Across Linked Text Boxes

	Article: Q158602
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you select text in one text box, you cannot extend the selection to include
	the text in the next linked text box.
	
	WORKAROUND
	==========
	
	To work around this problem, use either of the following methods:
	
	Method 1: Copy the text from each text box separately and paste it in the desired
	location.
	
	Method 2: Resize the first text box to include all the text, and then copy and
	paste the text to the new location.
