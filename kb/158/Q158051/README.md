## Q158051: Solution.exe Extends the Visual FoxPro Solutions Sample

	Article: Q158051
	Product(s): Microsoft FoxPro
	Version(s): WINDOWS:5.0,6.0
	Operating System(s): 
	Keyword(s): kbfile kbsample kbvfp500 kbvfp600
	Last Modified: 24-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual FoxPro for Windows, versions 5.0, 6.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Solution.exe is a sample that contains a reference to a sample form,
	Filtrlst.scx, that you can use to add new components and filters to the sample.
	For best results, place Filtrlst.scx and Filtrlst.sct in the
	Vfp\Samples\Solution directory in Visual FoxPro 5.0 and in the MSDN directory
	under Samples\vfp98\Solution directory in Visual FoxPro 6.0. The form itself
	contains instructions about its use.
	
	MORE INFORMATION
	================
	
	The following files are available for download from the Microsoft Download
	Center:
	
	  Solution.exe
	  (http://download.microsoft.com/download/vfox60/sample7/1/W9X/EN-US/Solution.exe)
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
	
	Microsoft used the most current virus detection software available on the date of
	posting to scan this file for viruses. Once posted, the file is housed on secure
	servers that prevent any unauthorized changes to the file.
	
	You can easily extend Solutions.exe by adding new components.
	
	For additional information about how to add a new form, query, view, or report to
	the Solutions sample, please see the following article in the Microsoft
	Knowledge Base:
	
	  Q157723 HOWTO: Extend the Visual FoxPro Solutions Sample
