## Q158949: Lexmark Monitor Reports Only Printer Error

{% raw %}

	Article: Q158949
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbprint
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Lexmark print monitor does not report back any more information than that
	there is a printer error (similar to Lprmon). Additional information in the
	status string is missing.
	
	MORE INFORMATION
	================
	
	This is a known limitation in the current version of this tool. Contact Lexmark
	for information regarding an updated version of the print monitor.
