## Q158860: Xircom CE2ps Network Card Is Not Detected On Insertion

	Article: Q158860
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 2,2.1
	Operating System(s): 
	Keyword(s): osr2 win95
	Last Modified: 31-JAN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 OEM Service Release, versions 2, 2.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are using a 3COM Elnk3 PCMCIA network card and you insert a Xircom
	CE2ps PCMCIA network card in another PCMCIA slot on a DEC HiNote Ultra CT475
	computer, the Xircom card is not recognized. It does not appear in Device
	Manager or the PCMCIA tool in Control Panel.
	
	RESOLUTION
	==========
	
	If you need to use these two cards together in a DEC HiNote Ultra CT475
	computer, insert the Xircom CE2ps card first
