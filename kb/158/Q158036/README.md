## Q158036: MSN Kids: Where Is MSN Kids?

	Article: Q158036
	Product(s): Microsoft Home Kids Products
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbfaq
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MSN Kids, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft MSN Kids, the web site devoted to Kids, has been discontinued and
	replaced by a new site. The new site is accessible by MSN members only, and it
	is called MSN Interactive Teen.
	
	MORE INFORMATION
	================
	
	To access the MSN Interactive Teen site, MSN users may connect to MSN Channel 6,
	and select Interactive Teen.
	
	
	For more information about obtaining an MSN account, double- click The Microsoft
	Network icon on your Windows desktop, and follow the instructions.
