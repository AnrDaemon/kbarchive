## Q158960: Files Installed by DirectX 2.0a

	Article: Q158960
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0,1.0a,3.0
	Operating System(s): 
	Keyword(s): kbsetup kbref kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Close Combat for Windows 1.0a 
	- Microsoft Deadly Tide for Windows 1.0 
	- Microsoft Flight Simulator for Windows 95, version 1.0 
	- Microsoft GEX, version 1.0 
	- Microsoft Hellbender for Windows, version 1.0 
	- Microsoft Monster Truck Madness, version 1.0 
	- Microsoft NBA Full Court Press for Windows, version 1.0 
	- Microsoft Soccer, version 1.0 
	- Microsoft Golf for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following is a list of the DirectX system files installed to your hard disk
	by DirectX 2.0a.
	
	NOTE: DirectX 2.0 includes newer versions of DirectX 1.0 files plus the
	introduction of Direct3D files.
	
	MORE INFORMATION
	================
	
	All files are marked as version 4.03.00.1096.
	
	Direct3D
	--------
	
	- D3dhalf.dll
	- D3dim.dll
	- D3drampf.dll
	- D3drg16f.dll
	- D3drg24f.dll
	- D3drg32f.dll
	- D3drg8f.dll
	- D3drgbf.dll
	- D3drm.dll
	- D3drm16f.dll
	- D3drm24f.dll
	- D3drm32f.dll
	- D3drm8f.dll
	- D3dxof.dll
	
	DirectDraw
	----------
	
	- Ddhelp.exe
	- Ddraw.dll
	- Ddraw16.dll
	
	DirectPlay
	----------
	
	- Dplay.dll
	- Dpserial.dll
	- Dpwsock.dll
	
	DirectSound
	-----------
	
	- Dsound.dll
	- Dsound.vxd
	
	Additional query words: 1.00 1.00a 6.00 direct-x dx dxsetup.exe ts troubleshoot
