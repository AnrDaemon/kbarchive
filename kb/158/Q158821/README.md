## Q158821: Cannot Install Gravis UltraSound Pro Drivers Correctly

	Article: Q158821
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 2,2.1
	Operating System(s): 
	Keyword(s): osr2 win95
	Last Modified: 31-JAN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 OEM Service Release, versions 2, 2.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After installing a Gravis UltraSound Pro Plug and Play sound card, you are
	prompted for the device manufacturer's floppy disk. However, you may receive an
	error message stating that the drivers could not be found.
	
	CAUSE
	=====
	
	There may be errors in the installation files on the manufacturer's disk.
	
	RESOLUTION
	==========
	
	Contact the hardware manufacturer for information about updated drivers.
