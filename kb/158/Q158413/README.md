## Q158413: AutoCorrect Does Not Automatically Scroll Correction List

	Article: Q158413
	Product(s): Microsoft PowerPoint for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): kbui kbusage
	Last Modified: 16-APR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft PowerPoint for Windows 95, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you type in the Replace Box in the PowerPoint AutoCorrect dialog box,
	PowerPoint does not automatically scroll to the appropriate part of the "Replace
	Text as You Type" list. In Microsoft Word for Windows 95, version 7.0, on the
	other hand, this list does scroll as you type in the Replace box.
	
	
	WORKAROUND
	==========
	
	To view the list of available corrections in the AutoCorrect dialog box you must
	manually scroll down the list.
