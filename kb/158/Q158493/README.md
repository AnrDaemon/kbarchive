## Q158493: WD97: Wizard Vanishes When Run from Banyan Vines Server

{% raw %}

	Article: Q158493
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run one of the wizards in Word 97, the wizard vanishes. Most of the
	wizards do not display an error message; however, the Fax Wizard will display an
	error indicating that it cannot locate a file.
	
	CAUSE
	=====
	
	This problem occurs when the wizards and templates are installed to a Banyan
	Vines server instead of to your local computer.
	
	WORKAROUND
	==========
	
	To work around this problem, copy the wizards and the templates to your local
	Template directory.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
	
	
	This problem was corrected in Microsoft Word 2000.
