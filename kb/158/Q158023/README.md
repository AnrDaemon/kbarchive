## Q158023: GEX: Gex.lnk Shortcut Error Message

	Article: Q158023
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbfaq
	Last Modified: 08-NOV-2001
	
	1.0
	WINDOWS
	kberrmsg kbreadme kbfaq kbfasttip
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft GEX, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You may receive the following error message:
	
	  The drive or network connection that the shortcut 'Gex.lnk' refers to is
	  unavailable. Make sure that the disk is properly inserted or the network
	  resource is available, and then try again.
	
	RESOLUTION
	==========
	
	If you receive this message, insert the GEX compact disc in your CD-ROM drive
	and then try running GEX again.
