## Q158945: Display Changes to Low Resolution After Rebooting

{% raw %}

	Article: Q158945
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 2,2.1
	Operating System(s): 
	Keyword(s): osr2 win95
	Last Modified: 31-JAN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 OEM Service Release, versions 2, 2.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you restart your computer, the display settings may revert to a 640 x 480
	resolution with 16 colors.
	
	RESOLUTION
	==========
	
	Contact the computer's manufacturer to inquire about the availability of a BIOS
	upgrade.
	
	
	MORE INFORMATION
	================
	
	After you change the display settings back to their previous state and restart
	the computer, the display settings remain correct. The next time the computer is
	restarted, the problem occurs again.
