## Q158597: Mac Encarta 97: Readme.txt: Minimum System Requirements

	Article: Q158597
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1997 edition
	Operating System(s): 
	Keyword(s): kbfaq
	Last Modified: 24-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 97 Encyclopedia for Macintosh 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following article contains the minimum system requirements needed to run
	Encarta Encyclopedia for the Macintosh, as stated in the Readme.txt file located
	on the Microsoft Encarta 97 Encyclopedia CD-ROM.
	
	MORE INFORMATION
	================
	
	To install and run Encarta Encyclopedia you need:
	
	  Any Apple Power Macintosh or Macintosh with a 25 MHz 68030
	  processor or faster (includes Macintosh IIci, LCIII, Performa 450
	  and later models)
	
	  Monitor supporting at least 256 colors and 640x480 or higher
	  resolution
	
	  System 7.1 or later
	
	  8 MB of random access memory (RAM)
	
	  16 MB of available hard-disk space
	
	  Double-speed or faster CD-ROM drive
	
	To use Encarta Yearbook, Web Links, and monthly online updates, you need:
	
	  Modem (9600 bps or faster)
	
	  Access to the Internet
	
	  4 MB of additional hard-disk space, plus 1 MB for each monthly
	  installment
