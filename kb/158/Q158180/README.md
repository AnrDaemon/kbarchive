## Q158180: GEX: System Hangs

	Article: Q158180
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbfaq
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft GEX, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	GEX may stop responding (hang) if you do not have your system configured
	correctly.
	
	RESOLUTION
	==========
	
	If GEX hangs, make sure your computer is at least a Pentium 75 with a minimum of
	8 MB of RAM. Running GEX on a computer with a lower configuration may result in
	hardware and performance difficulties.
	
	For more information about GEX hanging, see the following articles in the
	Microsoft Knowledge Base:
	
	  Q158093 GEX: Game Crashes if Compact Disc Is Removed
	
	  Q158094 GEX: Do Not Run System Agent While Playing Gex
	
	  Q158096 GEX: Running GEX on NexGen Chips and Processors
	
	  Q158101 GEX: Cannot Run GEX on Windows NT 3.51
	
	If after trying these solutions, GEX still hangs, disable the Sound Effects
	option:
	
	1. On the Options menu in GEX, click Sound.
	
	2. Clear the Sound Effects check box.
	
	Additional query words: 1.00 crash hang freezes up freeze-up boom stops ends black screen
