## Q158818: FP97: Help Topic Does Not Exist in Groups Tab of Web Permissions

{% raw %}

	Article: Q158818
	Product(s): Word Front Page
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdta
	Last Modified: 26-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft FrontPage 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you click Help on the Groups tab of the Permissions dialog box (Tools menu)
	while you are running Internet Information Server (IIS), a dialog box appears
	stating that the Help topic does not exist.
	
	If you press F1, the "Help on the Users Tab" topic is displayed.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article. This problem was corrected in Microsoft
	FrontPage 98 for Windows.
	
	MORE INFORMATION
	================
	
	If the topic for the Groups tab is available, it will show the same information
	that is available in the Users Tab topic.
	
	The Groups tab affects access to the IIS server under Windows NT Group accounts
	only; it does not apply to Windows NT User accounts.
