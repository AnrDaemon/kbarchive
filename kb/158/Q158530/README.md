## Q158530: XGEN: Build Numbers for Exchange

{% raw %}

	Article: Q158530
	Product(s): Microsoft Exchange
	Version(s): 4.0,5.0,5.5
	Operating System(s): 
	Keyword(s): exc4 exc5 exc55
	Last Modified: 12-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 4.0, 5.0, 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Each version of Microsoft Exchange Server and subsequent service packs have
	different build numbers. Listed below are the build numbers for each release:
	
	Microsoft Exchange Server 4.0   Build 837.10
	SP1                             Build 838.15
	SP2                             Build 993.4
	SP3                             Build 994.64
	SP4                             Build 995.54
	SP5                             Build 996
	
	Microsoft Exchange Server 5.0   Build 1457.10
	SP1                             Build 1458.50
	SP2                             Build 1460.13
	
	Microsoft Exchange Server 5.5   Build 1960
	SP1                             Build 2232
	SP2                             Build 2448
	SP3                             Build 2650
	SP4                             Build 2653
	
	You can determine the build number by viewing properties of the server.
