## Q235298: Microsoft Word 2000 Step by Step

	Article: Q235298
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 25-JUN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Word 2000 Step by Step ISBN 1-57231-970-4 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Microsoft Word 2000 Step by Step, ISBN
	1-57231-970-4.
	
	The following topics are covered:
	
	- Page 4: Callout Is Incorrect
	
	- Page 5: The Text In The Important Note Is Incorrect
	
	- Page 25: Quick Reference Entry Incorrect
	
	- Page 27: Quick Reference Entry Is Missing Step
	
	- Page 56: Remove Erroneous Sentence
	
	- Page 113: Step 3 Should Refer To The 5-inch Mark
	
	- Page 115: Correction For Step 3 Under Enter Text Into A Table
	
	- Page 122: Additional Step Required
	
	- Page 171: Typographical Error In Step 3
	
	- Page 192: Incorrect Steps in "Insert fields for a name and address" Section
	
	- Page 206: Change Quick Reference Entry
	
	- Page 268: File 12A Should Be 11A
	
	MORE INFORMATION
	================
	
	Page 4: Callout Is Incorrect
	----------------------------
	
	The callout on the graphic should be "I-beam pointer" instead of "insertion
	point."
	
	
	Page 5: The Text In The Important Note Is Incorrect
	---------------------------------------------------
	
	The Text in the "important" note should be edited to read as follows:
	"If the Office 2000 Registration Wizard appears, you can click Yes to register
	the product, or you can click No."
	
	
	Page 25: Quick Reference Entry Incorrect
	----------------------------------------
	
	The text in "Customize the Office Assistant" needs to be modified slightly.
	Please change:
	"...click Choose Office Assistant"
	
	To:
	"...Click Choose Assistant"
	
	
	Page 27: Quick Reference Entry Is Missing Step
	----------------------------------------------
	
	The text for "Ask the Answer Wizard a question" is missing a step. Please read
	the last sentence as follows:
	"In the What Would You Like To Do? box, type your question, and click Search."
	
	
	Page 56: Remove Erroneous Sentence
	----------------------------------
	
	Under the Using Additional Text Effects heading at the bottom of the page, please
	delete the following sentence:
	"You might find these effects particularly useful when designing Web pages, for
	example."
	
	
	Page 113: Step 3 Should Refer To The 5-inch Mark
	------------------------------------------------
	
	On page 113, step 3 should instruct you to drag the tab at the 5-inch mark on the
	ruler rather than at the 4-inch mark. The tab was set at the 5-inch mark on the
	previous page.
	
	Change:
	"Drag the tab at the 4-mark on the ruler into the document."
	
	To:
	"Drag the tab at the 5-mark on the ruler into the document."
	
	
	Page 115: Correction For Step 3 Under Enter Text Into A Table
	-------------------------------------------------------------
	
	On page 115, the information that you are entering into the table in step 3 does
	not match the illustrations on the following pages. The word Monday is spelled
	out in the illustrations and not abbreviated.
	
	Change:
	"Type 10-10 Tues - Sun, closed Mon and then press Tab."
	
	To:
	"Type 10-10 Tues - Sun, closed Monday and then press Tab."
	
	
	Page 122: Additional Step Required
	----------------------------------
	
	On page 122 an additional step needs to be added between steps 6 and 7. You will
	need to highlight the text before you can center it.
	
	Add the following step between steps 6 and 7:
	"Highlight the words Pacific Books."
	
	
	Page 171: Typographical Error In Step 3
	---------------------------------------
	
	In step 3 at the top of page 171 there is a typographical error.
	
	Change:
	"In the Front Edge area,"
	
	To:
	"In the From Edge area,"
	
	
	Page 192: Incorrect Steps in "Insert fields for a name and address" Section
	---------------------------------------------------------------------------
	
	The steps on page 192, under the section titled "Insert fields for a name and
	address" are out of order.
	
	Change:
	2) On the Forms toolbar, click the Text Form Field button.
	The Text Form Field Options dialog box appears.
	
	3) Double-click the text field.
	
	To:
	2) On the Forms toolbar, click the Text Form Field button.
	
	3) Double-click the text field.
	The Text Form Field Options dialog box appears.
	
	
	Page 206: Change Quick Reference Entry
	--------------------------------------
	
	The Quick Reference entry for Insert a calculation field is incorrect. Please
	change the last sentence to read as follows:
	"Enter a formula in the Expression box, and specify a number format, and click
	OK."
	
	
	Page 268: File 12A Should Be 11A
	--------------------------------
	
	There is an error in step 1 at the top of page 268.
	
	Change:
	"In the File Name box, the 12A file now has..."
	
	To:
	"In the File Name box, the 11A file now has..."
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
