## Q235130: Quick Course in Microsoft Office 2000 Comments and Corrections

	Article: Q235130
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 07-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Quick Course in Microsoft Office 2000 ISBN 1-57231-981-X 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Quick Course in Microsoft Office 2000, ISBN
	1-57231-981-X.
	
	The following topics are covered:
	
	- LOC Page: Name And Address of Online Press Have Changed
	
	- Page 58: Incorrect Graphic
	
	- Page 66: Errors In Graphic And Procedure
	
	- Page 67: Procedure Is Incorrect
	
	- Page 108: Graphic Is Inconsistent With Procedure
	
	- Page 128: Default Option Has Changed
	
	- Page 149: Procedure Is Incorrect
	
	- Page 196: Step 6 Is Incorrect
	
	- Page 216: Procedure Is Incorrect
	
	- Page 339: File Name Is Wrong
	
	- Page 359: Graphic Is Incorrect
	
	- Page 360: Graphic Is Incorrect
	
	- Page 372: Incorrect Information Due To Software Changes
	
	- Page 387: Dialog Box Has Changed
	
	MORE INFORMATION
	================
	
	LOC Page: Name And Address of Online Press Have Changed
	-------------------------------------------------------
	
	Online Press has merged with another company and now has a new name and address.
	Please replace the text in green on the LOC page with the following text:
	
	"A Quick Course Education/Training Edition for this title is published by Online
	Training Solutions, Inc. (OTSI). For information about supplementary workbooks,
	contact OTSI at 15422 Bel-Red Road, Redmond, WA 98052, USA. 1-800-854-3344.
	E-mail: quickcourse@otsiweb.com."
	
	
	Page 58: Incorrect Graphic
	--------------------------
	
	The Insert File dialog box on this page may not match what you see on the screen
	due to a late change in the software.
	
	
	Page 66: Errors In Graphic And Procedure
	----------------------------------------
	
	The graphic on page 66 no longer matches the interface.
	
	Please change the first sentence of step 3 to read:
	
	"In the Section Start area, select Continuous."
	
	Please change the second sentence to read
	
	"...select the Different First Page check box. Then click OK."
	
	
	Page 67: Procedure Is Incorrect
	-------------------------------
	
	Please change the first sentence to read the following:
	
	"Press Ctrl+Home and click the Print Preview button."
	
	
	Page 108: Graphic Is Inconsistent With Procedure
	------------------------------------------------
	
	Only cell A1 should be selected in the graphic on page 108, according to the
	procedure. In addition, the sum in the status bar is incorrect.
	
	
	Page 128: Default Option Has Changed
	------------------------------------
	
	The default option in the PowerPoint dialog box has changed.
	
	
	Page 149: Procedure Is Incorrect
	--------------------------------
	
	The procedure on this page is incorrect. Please read steps 1-2 as follows:
	
	"1. Type recycling in the Search For Clips box and press Enter to see results
	something like these:
	Any graphic that relates to recycling is displayed. (Don't worry if you see
	different or fewer graphics.)
	
	2. Click a graphic. (We selected the third graphic in the second row of our
	dialog box.) This palette drops down:"
	
	
	Page 196: Step 6 Is Incorrect
	-----------------------------
	
	Step 6 on this page is incorrect. Please read as follows:
	
	"Close the Customers form, saving the form design."
	
	
	Page 216: Procedure Is Incorrect
	--------------------------------
	
	Please change the first two steps in the procedure to read as follows:
	
	"1. First close the Data Access Page window, saving the page as Customers. Then
	close Access, clicking Yes when prompted to empty the Clipboard.
	
	2. Double-click the Internet Explorer icon on your desktop and choose Work
	Offline from the File menu."
	
	
	Page 339: File Name Is Wrong
	----------------------------
	
	The last sentence in step 1 in the second procedure on the page should have
	new_page_3 as the file name.
	
	
	Page 359: Graphic Is Incorrect
	------------------------------
	
	The graphic on this page no longer matches the interface.
	
	
	Page 360: Graphic Is Incorrect
	------------------------------
	
	The graphic on this page no longer matches the interface.
	
	
	Page 372: Incorrect Information Due To Software Changes
	-------------------------------------------------------
	
	Software changes after the printing of this book have caused some of the
	information to change. Please read the first paragraph after the bulleted list
	as follows:
	
	"Your toolbar may also display an Access, Excel, FrontPage, Outlook, PowerPoint,
	Publisher, or Word button. You can click this button to start the corresponding
	application and a new document."
	
	
	Page 387: Dialog Box Has Changed
	--------------------------------
	
	The dialog box on this page has changed, due to a late change in the software.
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
