## Q235297: Microsoft FrontPage 2000 At a Glance Comments and Corrections

{% raw %}

	Article: Q235297
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 13-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft FrontPage 2000 At a Glance ISBN 1-57231-951-8 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Microsoft FrontPage 2000 At a Glance, ISBN
	1-57231-951-8.
	
	The following topics are covered:
	
	- Page 20: The Second Tip Contains Inaccurate Information
	
	- Page 77: Step 2 Is Incorrect
	
	- Page 89: Step 6 Is Incorrect
	
	- Page 145: Changes to Steps 2, 5, and 6
	
	- Page 172: Forms Properties Dialog Box Incorrect
	
	MORE INFORMATION
	================
	
	Page 20: The Second Tip Contains Inaccurate Information
	-------------------------------------------------------
	
	The second tip contains inaccurate information.
	
	Change:
	"?choose Import from the File menu, FrontPage starts the Import Web Wizard,
	which?"
	
	To:
	"?choose Import from the File menu, FrontPage displays the New dialog box and
	selects the Import Web Wizard, which?"
	
	
	Page 77: Step 2 Is Incorrect
	----------------------------
	
	In Step 2 you should choose Picture Properties, not Image Properties.
	
	Change:
	"Choose Image Properties from the shortcut menu."
	
	To:
	"Choose Picture Properties from the shortcut menu."
	
	
	Page 89: Step 6 Is Incorrect
	----------------------------
	
	In step 6 you should click Text, not Styles.
	
	Change:
	"Click Styles, and use the Item and Font list boxes to modify the font for body
	text and headings."
	
	To:
	"Click Text, and use the Item and Font list boxes to modify the font for body
	text and headings."
	
	
	Page 145: Changes to Steps 2, 5, and 6
	--------------------------------------
	
	In Steps 2, 5, and 6, change the word "pages" to "files".
	
	Step 2 should read:
	"Place the cursor where you want to add the list of files."
	
	Step 5 should read:
	"Select an entry from the Sort Files By drop-down list box to specify how you
	want the files sorted."
	
	Step 6 should read:
	"Specify any additional information that should be included with the files."
	
	
	Page 172: Forms Properties Dialog Box Incorrect
	-----------------------------------------------
	
	On page 172, the Options button in the Form Properties dialog box is in the wrong
	location on the screen pictured on this page. The computer screen in this
	diagram should show the Options button located in the lower left hand corner of
	the dialog box.
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
