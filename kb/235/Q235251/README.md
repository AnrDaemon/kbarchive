## Q235251: Programming Microsoft Visual InterDev 6.0 Comments and Correctio

{% raw %}

	Article: Q235251
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 07-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Programming Microsoft Visual InterDev 6.0 ISBN 1-57231-814-7 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Programming Microsoft Visual InterDev 6.0,
	ISBN 1-57231-814-7.
	
	The following topics are covered:
	
	- Data on Companion CD-ROM Incompatible with SQL Server 7.0
	
	- Page 62: Remove Reference to Appendix B
	
	- Page 243: Remove Reference to Appendix E
	
	MORE INFORMATION
	================
	
	Data on Companion CD-ROM Incompatible with SQL Server 7.0
	---------------------------------------------------------
	
	If you are running SQL Server 7.0, it is incompatible with the backup device on
	the CD (created for SQL Server 6.5). Note that you can use the generic bulk
	insert from text files method with both SQL Server 6.5 and SQL Server 7.0.
	
	
	Page 62: Remove Reference to Appendix B
	---------------------------------------
	
	On page 62, in paragraph 4, remove the last sentence:
	
	"Appendix B lists all the winInet.dll error messages."
	
	There is no Appendix B
	
	
	Page 243: Remove Reference to Appendix E
	----------------------------------------
	
	On page 62, in paragraph 1, remove the following statement:
	
	"The Response object has several properties and methods that are fully detailed
	in Appendix E."
	
	There is no Appendix E.
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
