## Q235190: SMS Advertisements Are Immediately Locked and Cannot Be Modified

	Article: Q235190
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:2.0
	Operating System(s): 
	Keyword(s): kbsms200 kbsms200bug kbsms200fix kbAdvertisement kbSoftwareDist
	Last Modified: 11-JUL-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When advertisements are created using the Distribute Software Wizard at a child
	site using a package from a parent site, the advertisement is immediately locked
	and cannot be modified.
	
	NOTE: This problem does not occur if the Distribute Software Wizard is not used.
	
	WORKAROUND
	==========
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 2.0. This problem has been corrected in the latest U.S. service pack for
	Systems Management Server version 2.0. For information on obtaining the service
	pack, query on the following word in the Microsoft Knowledge Base (without the
	spaces):
	
	  S E R V P A C K
	
	Additional query words: prodsms sms advertisement locked
