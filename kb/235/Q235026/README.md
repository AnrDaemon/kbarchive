## Q235026: Quick Course in Microsoft Access 2000 Comments and Corrections

	Article: Q235026
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 07-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Quick Course in Microsoft Access 2000 ISBN 1-57231-979-8 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Quick Course in Microsoft Access 2000, ISBN
	1-57231-979-8.
	
	The following topics are covered:
	
	- LOC Page: Address And Name For Online Press Has Changed
	
	- Page 9: Inconsistencies Due To Changes In Software
	
	- Page 36: Graphic Is Incorrect
	
	- Page 37: Step 2 And Graphic Are Incorrect
	
	- Page 40: Incorrect Information
	
	- Page 108: Graphic Has Changed
	
	- Page 135: Procedure And Graphics Are Incorrect
	
	- Page 137: Screen Incorrect
	
	- Page 138: Graphic And Procedure Are Incorrect
	
	- Page 140: Graphic And Procedure Are Incorrect
	
	- Page 153: Graphic Incorrect
	
	MORE INFORMATION
	================
	
	LOC Page: Address And Name For Online Press Has Changed
	-------------------------------------------------------
	
	Online Press has merged with another company and now has a new name and address.
	Please replace the text in green on the LOC page with the following text:
	
	"A Quick Course Education/Training Edition for this title is published by Online
	Training Solutions, Inc. (OTSI). For information about supplementary workbooks,
	contact OTSI at 15422 Bel-Red Road, Redmond, WA 98052, USA. 1-800-854-3344.
	E-mail: quickcourse@otsiweb.com."
	
	
	Page 9: Inconsistencies Due To Changes In Software
	--------------------------------------------------
	
	Due to changes in Access after the first printing of this book, there are several
	problems with the information on this page.
	
	In step 2, change the second sentence to read:
	
	  "If it is available, double-click Nwind.mdb;"
	
	Change the first sentence in the second paragraph to read:
	
	  "The objects bar on the left contains icons..."
	
	Change the last sentence in the second paragraph to read:
	
	  "The Tables icon is active, and the database's tables are listed on the
	  right."
	
	
	Page 36: Graphic Is Incorrect
	-----------------------------
	
	The graphic on page 36 is incorrect. The Field size limit should be 20 instead of
	10.
	
	
	Page 37: Step 2 And Graphic Are Incorrect
	-----------------------------------------
	
	Step 2 is incorrect. Change it to read:
	
	  "Access will now accept only five digits in this field, even though the field
	  size is larger."
	
	In addition, the graphic is incorrect. The Field Size should be 30, and the
	Unicode Compression should be set to No
	
	
	Page 40: Incorrect Information
	------------------------------
	
	Due to a late-breaking change in software, some information on this page is
	incorrect. Change the last sentence of the first full paragraph to read the
	following:
	
	  "Nor will you need to set Unicode Compression, which decreases the total
	  storage size of the database."
	
	
	Page 108: Graphic Has Changed
	-----------------------------
	
	Due to late-breaking changes in the software, the graphic on this page is no
	longer correct, and will not match your computer screen.
	
	
	Page 135: Procedure And Graphics Are Incorrect
	----------------------------------------------
	
	Due to late-breaking changes in the software after the first printing of this
	book, information on this page is no longer correct.
	
	Change step 3 to read the following:
	
	  "Resize the window to look like this:"
	
	In addition, the graphic on this page is no longer correct, and will not match
	your computer screen.
	
	
	Page 137: Graphic Has Changed
	-----------------------------
	
	The graphic on this page is no longer correct, and will not match your computer
	screen.
	
	
	Page 138: Procedure And Graphic Are Incorrect
	---------------------------------------------
	
	The last sentence (within parentheses) in step 1 should be changed as follows:
	
	  "(Now the labels and controls are visible only in design view.)"
	
	In addition, the graphic on this page is no longer correct, and will not match
	your computer screen.
	
	
	Page 140: Graphic And Procedure Are Incorrect
	---------------------------------------------
	
	In step 4 on this page, change the first sentence to read:
	
	  "Click OK and then close the Properties dialog box."
	
	In addition, the graphic on this page is no longer correct, and will not match
	your computer screen.
	
	
	Page 153: Graphic Incorrect
	---------------------------
	
	The graphic on this page is no longer correct, and will not match your computer
	screen.
