## Q235045: Quick Course in Microsoft Word 2000 Comments and Corrections

	Article: Q235045
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 07-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Quick Course in Microsoft Word 2000 ISBN 1-57231-986-0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Quick Course in Microsoft Word 2000, ISBN
	1-57231-986-0.
	
	The following topics are covered:
	
	- LOC PAGE: Name And Address Of Online Press Have Changed
	
	- Page 74: Graphic Is Incorrect
	
	- Page 77: Graphic And Procedure Are Incorrect
	
	- Page 119: Filling In Forms Procedure Correction
	
	- Pages 120-121: Text In Graphics Do Not Match Procedure
	
	- Page 165: Graphic No Longer Matches Dialog Box
	
	MORE INFORMATION
	================
	
	LOC PAGE: Name And Address Of Online Press Have Changed
	-------------------------------------------------------
	
	Online Press has merged with another company and now has a new name and address.
	Please replace the text in green on the LOC page with the following text:
	
	"A Quick Course Education/Training Edition for this title is published by Online
	Training Solutions, Inc. (OTSI). For information about supplementary workbooks,
	contact OTSI at 15422 Bel-Red Road, Redmond, WA 98052, USA. 1-800-854-3344.
	E-mail: quickcourse@otsiweb.com."
	
	
	Page 74: Graphic Is Incorrect
	-----------------------------
	
	The Page Setup dialog box may not match the graphic that you see on page 74. This
	graphic will be updated in future printings.
	
	
	Page 77: Graphic and procedure are incorrect
	--------------------------------------------
	
	The Paragraph dialog box may not match the graphic that you see on page 77. This
	graphic will be updated in future printings.
	
	Step 3 is no longer correct due to changes in the software. Please change the
	second sentence to read as follows:
	
	"Word enters the default first-line indent in the By edit box and shows in the
	preview box below how your text will look with this setting."
	
	
	Page 119: Filling In Forms Procedure Correction
	-----------------------------------------------
	
	Page 119, Filling In Forms procedure, Step 3, 1st sentence:
	Change:
	"Click the Print button to print the form."
	
	To:
	"Choose Print from the File menu to print the form."
	
	
	Pages 120-121: Text In Graphics Do Not Match Procedure
	------------------------------------------------------
	
	The "Business Travel" text in these graphics should say "Marketing Project" to
	match the accompanying procedures.
	
	
	Page 165: Graphic No Longer Matches Dialog Box
	----------------------------------------------
	
	Due to changes in the software, the graphic on this page may not match the dialog
	box that you will see. The graphic will be updated for future printings.
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
