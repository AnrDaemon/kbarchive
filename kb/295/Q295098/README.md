## Q295098: No Automatic Virus Scanning In MSN Explorer E-mail

{% raw %}

	Article: Q295098
	Product(s): The Microsoft Network
	Version(s): 6.0,6.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 02-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 6.0, 6.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	You may have used an anti-virus program (such as Norton's Anti-Virus) in
	conjunction with Outlook or Outlook Express to scan incoming e-mail for viruses;
	you may want to use MSN Explorer in the same way.
	
	MORE INFORMATION
	================
	
	Currently, MSN Explorer does not allow for automatic virus scanning of e-mail.
	This feature will be added in a future version.
