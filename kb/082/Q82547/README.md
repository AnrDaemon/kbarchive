## Q82547: Gupta's Quest GP Faults When Importing dBASE/CSV

	Article: Q82547
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When using Gupta's Quest with Microsoft Windows operating system version 3.1 and
	attempting to import a dBASE or CSV file, a General Protection (GP) Fault may
	occur.
	
	To correct this problem, obtain an undated version of the software.
	
	
	Additional query words: 3.10 3.11 network
