## Q82574: GP Fault and Hang with Aristosoft's Wired for Sound

	Article: Q82574
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Aristosoft's Wired for Sound causes a general protection (GP) fault and causes
	the system to stop responding (hang) when run under Microsoft Windows version
	3.1.
	
	NOTE: This problem does not occur in Wired for Sound Pro.
	
	WORKAROUND
	==========
	
	To resolve the problem, you can either disable Wired for Sound by removing
	WFSOUND.EXE from the LOAD= line in the WIN.INI file, or you can obtain an
	updated driver from Aristosoft.
	
	MORE INFORMATION
	================
	
	According to Aristosoft technical support, an updated driver is available by
	contacting their Customer Service department.
