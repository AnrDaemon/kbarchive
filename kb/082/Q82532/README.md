## Q82532: GenSoft's BitCapture Causes GP Fault

{% raw %}

	Article: Q82532
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you are running Microsoft Windows operating system version 3.1, GenSoft's
	BitCapture program produces a general protection (GP) fault when you attempt to
	capture the screen. This problem also occurs under Windows 3.0. You need to try
	to perform a large screen capture.
	
	MORE INFORMATION
	================
	
	The GP fault occurs inside the application under 3.0 and 3.1 when it accesses a
	NULL pointer.
