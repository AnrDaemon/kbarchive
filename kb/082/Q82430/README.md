## Q82430: STAR Micronix Printers Run in Loops Printing to File

{% raw %}

	Article: Q82430
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	If you print from Microsoft Write to a printer that uses the STAR Micronix
	STARM.24.DRV driver from the Supplemental Driver Library (SDL), the printing
	doesn't stop. It cycles through the page number at the left and then starts over
	in a looping manner until you choose Cancel.
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here as it
	becomes available.

	or reliability.
