## Q82502: 8514 System Font Becomes Corrupted by MS-DOS Window

	Article: Q82502
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 29-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The 8514 system font used in an MS-DOS Prompt window can cause a few of the
	extended characters to become garbled. Applications that are localized for other
	languages, such as German, rely heavily on extended characters.
	
	
	Additional query words: 3.10 3.11 system font 8514 box 3.1
