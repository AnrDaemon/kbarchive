## Q82424: Seaview 1.0 GP Faults When Using a Cabinet

	Article: Q82424
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	SEA Labs Seaview version 1.0 causes a general protection (GP) fault when run in
	Microsoft Windows. The problem occurs when you try to use a cabinet in Seaview.
	(A cabinet is the basic unit of work in Seaview.)
	
	CAUSE
	=====
	
	This version of Seaview is incompatible with Windows.

	================
