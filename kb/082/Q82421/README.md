## Q82421: Lenel's Multimedia Organizer with TrueType Only Enabled

	Article: Q82421
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Multimedia Organizer made by Lenel Systems Intl, Inc. version 0.2R displays
	the error message:
	
	  MS Sans Serif is not loaded.
	
	If you choose the OK button, the error repeats itself three times. The
	application then runs normally without the MS Sans Serif font.
	
	CAUSE
	=====
	
	This problem occurs if Show Only True Type (TT) Fonts In Applications is enabled
	from the dialog box that appears when select the Fonts icon from the Control
	Panel. Default system settings are for TT Only to be disabled.
