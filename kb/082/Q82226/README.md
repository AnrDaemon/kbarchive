## Q82226: Minor Distortions Using TrueType Fonts

	Article: Q82226
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	You may encounter minor distortions when you use TrueType fonts at certain point
	sizes. For example, when you use Courier New Bold in 19 or 20 point, the
	horizontal bar of the "e" is only one pixel thick, but at smaller point sizes,
	it's thicker.
	
	WORKAROUND
	==========
	
	There is no workaround available.
