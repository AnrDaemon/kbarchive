## Q82426: KEAterm 1.1.0 Does Not Print TrueType Fonts to an HP III

	Article: Q82426
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	KEAterm version 1.1.0, a terminal emulation software package, prints garbage
	characters when you try to print TrueType fonts to a Hewlett- Packard (HP)
	LaserJet III printer using version 3.89 of the HPPCL5A.DRV driver. TrueType
	fonts must be turned off to print with this application to an HP III printer.
	
	
	STATUS
	======
	
	This problem does not occur with later version of KEAterm.
	
	
	MORE INFORMATION
	================
	
	KEAterm is manufactured by KEA Systems, a subsidiary of Attachmate Corporation,
	which is a vendor independent of Microsoft; we make no warranty, implied or
	otherwise, regarding this product's performance or reliability.
