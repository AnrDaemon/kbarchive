## Q82850: ToolBook Version 1.5 May Cause a GP Fault Under Windows 3.1

{% raw %}

	Article: Q82850
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If a picture is copied from Paintbrush and pasted into ToolBook version 1.5
	under the Microsoft Windows 3.1x a general protection (GP) fault occurs. Under
	Windows 3.0, you will receive and Unrecoverable Application Error (UAE).
	
	For additional information, please contact Asymetrix.
