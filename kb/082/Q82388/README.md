## Q82388: Mavis Beacon Teaches Typing Under Windows

{% raw %}

	Article: Q82388
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11,95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When run under Microsoft Windows versions 3.1 and 3.11, Mavis Beacon Teaches
	Typing may not function as expected. During the typing lessons, when it says the
	following in Windows 3.0,
	
	  let's go to the races
	
	you see a car driving down a road with bugs hitting your windshield as you make
	mistakes in typing. In Windows 3.1, you see no car and no bugs.
	
	CAUSE
	=====
	
	This problem is caused by Mavis Beacon Teaches Typing, not Windows 3.1.
	
	MORE INFORMATION
	================
	
	For additional information, please contact Mavis Beacon Teaches Typing technical
	support.

	we make no warranty, implied or otherwise, regarding this product's performance
	or reliability.
