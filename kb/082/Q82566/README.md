## Q82566: Norton Desktop's Batch Builder Help Error

{% raw %}

	Article: Q82566
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Trying to run Help, Commands or Help, Procedures from Symantec's Norton Desktop
	for Windows' Batch Builder under Microsoft Windows operating system version 3.1
	results in the following error message:
	
	  Invalid Keyword
	
	This is ONLY a problem in earlier versions of Norton Desktop for Windows dated
	before 5/25/91.

	we make no warranty, implied or otherwise, regarding this product's performance
	or reliability.
