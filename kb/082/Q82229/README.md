## Q82229: Windows Setup Updates &quot;DOS Prompt&quot; to &quot;MS-DOS Prompt&quot;

	Article: Q82229
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	When you install Microsoft Windows version 3.1 over Microsoft Windows
	version 3.0, the Windows 3.1 Setup program renames the "DOS Prompt" icon
	"MS-DOS Prompt." If the DOS Prompt icon has been renamed or moved to
	another group, Setup adds an "MS-DOS Prompt" icon to the Main group.
	
	Additional query words: 3.10 3.11
