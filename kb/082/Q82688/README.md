## Q82688: DCA WindowLink for Irma and Windows 3.1

	Article: Q82688
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	DCA's WindowLink for Irma is not compatible with the Microsoft Windows operating
	system version 3.1. The application was written in Windows version 2.x format
	and cannot be run in Windows 3.1. Windows 3.1 displays an error message if you
	try to run the program with Windows 3.1. The message indicates that DCA has a
	new application, Irma Workstation for Windows, which works with Windows 3.1.
	This new application has more options and an improved user interface.
	
	The products included here are manufactured by DCA Incorporated, vendor
	independent of Microsoft; we make no warranty, implied or otherwise, regarding
	these products' performance or reliability.
