## Q82295: PC Win: Search Results for GAL has a Maximum of 500 Names

{% raw %}

	Article: Q82295
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft Mail for Windows version 3.0 returns a maximum of 500 user names when
	using the Name Finder command from the Address Book.
	
	The Name Finder is used to find a particular user's name. It is run by selecting
	Address Book from the Mail menu and then clicking the magnifying glass icon.
