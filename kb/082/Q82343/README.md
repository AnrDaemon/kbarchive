## Q82343: Superbase 4 for Windows Causes System Resources Drain

{% raw %}

	Article: Q82343
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 31-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you use a database developed with Superbase for Windows version 1.3, Windows
	version 3.1 system resources drop as much as 1 to 3 percent for each action
	(such as editing or saving a record).
	
	If you perform heavy editing, system resources drop from 80 to below 20 percent
	in a few minutes. When you exit Superbase for Windows, system resources are not
	returned to Windows.
	
	WORKAROUND
	==========
	
	To regain resources, exit and restart Windows. The problem is within Superbase.
	
	For additional information, contact Superbase technical support.

	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
