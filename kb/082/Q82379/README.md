## Q82379: MicroCourier Displays Vertical Lines

	Article: Q82379
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Symantec's MicroCourier displays vertical lines in its status bar and in the
	terminal window when run from Microsoft Windows.
	
	CAUSE
	=====
	
	This problem occurs due to a problem with one of MicroCourier's built-in fonts.

	================
