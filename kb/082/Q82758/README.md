## Q82758: What Disk to Install for HP DeskJet 500C

	Article: Q82758
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	
	In Control Panel's list of supported printers, the "HP DeskJet 500C" option gives
	a dialog box with the message:
	
	  Insert unlisted, updated, or vendor-provided printer driver disk in:
	
	This message is asking you to insert a disk provided by Hewlett-Packard (HP). The
	HP DeskJet 500C is a listing for the HP Color DeskJet, for which Windows 3.1
	does not provide a printer driver. A printer driver is provided by
	Hewlett-Packard. Customers must obtain the disk from Hewlett-Packard.
	
	This driver can be obtained from the Hewlett-Packard Fulfillment Center.
