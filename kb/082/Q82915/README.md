## Q82915: PC Win: Err Msg: Stack Fault in MSSPELL.DLL

	Article: Q82915
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0,3.0b,3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, versions 3.0, 3.0b, 3.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Microsoft Mail for Windows sometimes displays the following error message when
	you are spell checking a very long string of characters containing no blank
	spaces:
	
	  Stack Fault in MSSPELL.DLL
	
	RESOLUTION
	==========
	
	To spell check your mail message without causing this error, highlight a shorter
	section of the message before you run the spell checker.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in versions 3.0, 3.0b, and 3.2 of
	Microsoft Mail for Windows. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
