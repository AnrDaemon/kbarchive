## Q82622: DoDot Uses More System Resources with Windows 3.1

	Article: Q82622
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When the Halcyon Software DoDot program (a graphics image converter) is run in
	Microsoft Windows version 3.1, system resources in the Help/About dialog box
	drop significantly as compared to Windows 3.0.
	
	When you quite the application the memory is recovered.
	
	CAUSE
	=====
	
	This problem is caused by the DoDot program. Halcyon Software is aware of the
	problem and is working on an upgrade.
	
	RESOLUTION
	==========
	
	Contact Halcyon Software for upgrade availability information.
	
	MORE INFORMATION
	================
	
	Contact Halcyon Software for more information.

	or reliability.
