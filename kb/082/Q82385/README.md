## Q82385: Windows: Turbo C++ Causes GP Fault After Numerous Searches

{% raw %}

	Article: Q82385
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you use Borland's Turbo C++ edit mode under Microsoft Windows, multiple
	searches on multiple source code files may cause Turbo C++ to cause a general
	protection (GP) fault or an Unrecoverable Application Error (UAE). In some
	situations, Turbo C++ may even cause the machine to reboot.
	
	MORE INFORMATION
	================
	
	For additional information, please contact Borland.

	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
