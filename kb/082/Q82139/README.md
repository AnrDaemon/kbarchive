## Q82139: IBM Personal Printer 2390/2391 Drivers Available for Windows

{% raw %}

	Article: Q82139
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The IBM Personal Printer Series II 2390 and 2391 are letter-quality, 24-wire
	dot-matrix printers. The printer driver currently being shipped by Lexmark, an
	IBM alliance company, is compatible with Microsoft Windows.
	
	NOTE: Microsoft Windows 3.0 does not supply a printer driver for the IBM 2390 and
	2391.
	
	MORE INFORMATION
	================
	
	The OPENMEFIRST disk is a drivers disk provided with IBM printers. If the
	OPENMEFIRST disk did not come with your IBM 2390 or 2391 printer or did not
	contain the Windows drivers, Lexmark will supply them.
	
	To obtain the drivers, contact Lexmark Technical Support or call the Lexmark
	bulletin board service (BBS).
