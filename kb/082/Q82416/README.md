## Q82416: Characters Stretched on UPC/EAN Cartridge

	Article: Q82416
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	In the Microsoft Windows version 3.1, the ASCII characters 196 and 252 print
	stretched out when printing with the UPC/EAN cartridge on a NEC P2200 dot matrix
	printer.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the NEC24PIN driver. We are
	researching this problem and will post new information here as it becomes
	available.
