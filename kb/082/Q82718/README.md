## Q82718: Norton Desktop for Windows Has a Repaint Problem

{% raw %}

	Article: Q82718
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Norton Desktop for Windows (NDW) may have screen refresh problems browsing for a
	file to print when you are running MS-DOS Prompt.
	
	
	WORKAROUND
	==========
	
	To work around this problem, close both the print option and MS-DOS Prompt. This
	problem does not occur when MS-DOS Prompt is not being run.
	
	It may be necessary to exit and restart Windows to prevent this problem from
	reoccurring, even if MS-DOS Prompt has been closed.
