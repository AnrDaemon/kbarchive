## Q82461: IBM Quiet Writer III Envelope Paper Source Not Functional

{% raw %}

	Article: Q82461
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	The envelope tray Paper Source selection provided by the IBM Quiet Writer III
	printer driver included with Microsoft Windows version 3.1 is not functional.
	
	WORKAROUND
	==========
	
	You must load the envelope manually to print the address.
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here as it
	becomes available.
