## Q82352: WindowBase 1.0 Screen Does Not Update Properly

	Article: Q82352
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the WindowBase version 1.0 Form Access utility with Microsoft
	Windows, the current form may look correct when started, but any process
	requiring the screen to be updated will leave the screen without cells inverted
	or "outlined."
	
	Inversion or outlining is used to indicate the current cell and current record
	for each particular query.
