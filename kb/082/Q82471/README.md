## Q82471: YourWay 2.0 Pushpin Window Not Visible

	Article: Q82471
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you run Prisma Software's YourWay version 2.0 personal information manager
	application under Microsoft Windows, red pushpins are not visible in the
	Cardfile selection.
	
	WORKAROUND
	==========
	
	You can work around this problem by using the Notes button under the menu bar.

	================

	or reliability.
