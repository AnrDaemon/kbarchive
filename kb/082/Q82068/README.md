## Q82068: PC Win: Err Msg: A Message Could Not Be Opened

{% raw %}

	Article: Q82068
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Microsoft Mail for Windows version 3.0, the user might receive the following
	error message:
	
	  A Message Could not be Opened.
	
	CAUSE
	=====
	
	This error can occur when the user is trying to open a message that is in his or
	her outbox. The message could not be opened because the mail transport was in
	the process of sending the message to the postoffice.
	
	MORE INFORMATION
	================
	
	The primary goal of the outbox is to allow users to see what messages are left
	to be sent and not to cancel, open, or move messages that are about to be sent.
