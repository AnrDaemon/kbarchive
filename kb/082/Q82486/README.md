## Q82486: Picture Publisher Stack Fault Changing Scanner Option

	Article: Q82486
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Picture Publisher version 3.0 displays the following error message when you use
	the mouse to try to change scanner options under Microsoft Windows:
	
	  PP caused a Stack Fault in module PP.EXE at 0048:034C
	
	WORKAROUND
	==========
	
	Use the keyboard to change the scanner options, not the mouse.
