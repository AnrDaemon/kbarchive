## Q82380: GP Fault Exiting HP NewWave 4.0

{% raw %}

	Article: Q82380
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you use the Hewlett-Packard (HP) NewWave file manager with Microsoft Windows,
	the HP NewWave file manager causes an Unrecoverable Application Error (UAE) or
	general protection (GP) fault if you have more than one application minimized.
	
	For example, the following error message may be displayed when you exit NewWave:
	
	  HPENCAPS General Protection Fault in KRNL386.exe 0001:4E00

	================
	
	For additional information, please contact Hewlett-Packard.

	or reliability.
