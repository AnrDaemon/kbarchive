## Q82519: SuperPrint Under Excel Slows Landscape Printing

	Article: Q82519
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	SuperPrint by Zenographics causes landscape printing under Excel 3.0 to print
	slowly when run under the Microsoft Windows operating system version 3.1. This
	is because SuperPrint will rotate every font character before it allows it to be
	printed.
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here as it
	becomes available.
