## Q82691: Novell NetWare Incompatible with XTreeNet 2.0 in VM

	Article: Q82691
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Windows operating system version 3.1 displays the following error
	message if you run Xtree's XTreeNet version 2.0 from MS-DOS-Prompt under 386
	enhanced mode Windows:
	
	  This application has violated system integrity and will be terminated
	
	There is currently no way to work around this problem. For more information,
	contact XTree.
