## Q82354: On Target 1.0 Does Not Install on Windows System

	Article: Q82354
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you have the Microsoft Windows on your system, On Target version 1.0 does not
	install. After running On Target's installation program, your system is caught
	in an infinite loop. This usually occurs after about 10 percent of the
	installation is complete. An error message similar to the following is
	displayed:
	
	  File error, can't find WIN310
	
	You are then prompted to press a button, but the message continues to be
	displayed.
	
	CAUSE
	=====
	
	This is due to a problem in On Target version 1.0. Later versions do not have
	this problem.
	
	RESOLUTION
	==========
	
	You need to obtain a later version of On Target
