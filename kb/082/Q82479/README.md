## Q82479: PostScript Italics Simulation Doesn't Print Italics

	Article: Q82479
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 3.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Printing simulated italics in the Symbol or ZapfChancery font causes the font to
	print out without being italicized on PostScript printers.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft Windows 3.1
	PostScript printer driver. We are researching this problem and will post new
	information here as it becomes available.
	
	
	Additional query words: 3.10 PSCRIP.EXE windrvr
