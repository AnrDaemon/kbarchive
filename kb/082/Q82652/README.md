## Q82652: Da Vinci eMAIL GPF in DSWIN.DLL

	Article: Q82652
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Da Vinci eMAIL may cause a general protection (GP) fault in DSWIN.DLL at
	0009:0016 when you attach a file to a message and try to send the message to
	multiple aliases. Under Windows 3.0, Da Vinci eMAIL will cause an Unrecoverable
	Application Error (UAE).
