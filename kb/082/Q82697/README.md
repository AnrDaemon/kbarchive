## Q82697: Novell Cannot Rename Directories with File Manager

	Article: Q82697
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When using Novell NetWare with the Microsoft Windows operating system version
	3.1, File Manager cannot rename directories and subdirectories on NetWare
	drives.
	
	
	Additional query words: 3.10 net ware
