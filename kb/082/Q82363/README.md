## Q82363: SMARTDrive 4.0 and RCD.EXE 4.x

	Article: Q82363
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	With SMARTDrive (SMARTDRV.EXE) 4.0 loaded, RCD.EXE version 4.x may issue an
	error message stating that RCD.SYS has not been loaded. This error message is
	displayed even if RCD.SYS has been loaded.
	
	RESOLUTION
	==========
	
	To solve this problem, either obtain RCD.EXE version 7.0 (this is a free update
	from Bernoulli), or load SMARTDRV.EXE after running RCD.EXE.
	
	MORE INFORMATION
	================
	
	The Bernoulli product included here is manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
