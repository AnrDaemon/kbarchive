## Q82362: Large TrueType Fonts Print with Fine Lines on LaserJet II

	Article: Q82362
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	Printing TrueType fonts larger than 80 points on some Hewlett-Packard (HP)
	LaserJet Series II printers sometimes adds or subtracts fine, pixel-wide lines.
	
	CAUSE
	=====
	
	This is a known firmware problem with HP LaserJet Series II printers and is not
	a problem with the TrueType rasterizer.
	
	
	WORKAROUND
	==========
	
	To work around this problem, print TrueType fonts as graphics, as follows:
	
	1. Run Control Panel.
	
	2. Choose the Printers icon.
	
	3. From the list of printers, select the HP LaserJet Series II printer.
	
	4. Choose the Setup button.
	
	5. Choose the Options button.
	
	6. Select the Print TrueType As Graphics check box.
	
	The HP products included here are manufactured by vendors independent of
	Microsoft; we make no warranty, implied or otherwise, regarding these products'
	performance or reliability.
