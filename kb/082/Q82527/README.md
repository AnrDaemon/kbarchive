## Q82527: GP Fault While Selecting Font in Zenographics Pixie

	Article: Q82527
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you select different fonts under low memory conditions in Pixie, it may give
	a General Protection (GP) Fault such as the following:
	
	  GP Fault in USER at B:97
	
	Zenographics Pixie is a Windows charting and drawing program.
	
	
	Avoid the low memory situation by closing other applications before using Pixie.
