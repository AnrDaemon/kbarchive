## Q82768: Using Windows 3.1 with Toshiba T2200SX and Trackball Mouse

	Article: Q82768
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	According to Toshiba technical support, if you use the Microsoft Windows
	operating system version 3.1 on a Toshiba T2200SX with a trackball mouse
	connected to the PS/2 port and BIOS versions earlier than 1.20, you will
	experience some problems with the mouse.
	
	To correct these problems, use one of the following solutions:
	
	- Attach the mouse to a COM port.
	
	  -or-
	
	- Obtain an updated BIOS version.
	
	Additional query words: 3.10 3.11 track ball
