## Q82414: CA Compete Dialog Text Is Not Correctly Truncated

	Article: Q82414
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	With Microsoft Windows 3.1 and 3.11, CA Compete's Curve Builder may have labels
	that are longer than the field width of the dialog boxes.
	
	With Windows 3.0, these same labels are correctly truncated to fit in the
	available space of the dialog boxes.
