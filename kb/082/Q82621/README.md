## Q82621: CoolSwitch Window Loses Document Title Switching to Excel

{% raw %}

	Article: Q82621
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you press ALT+TAB to switch between applications running under Microsoft
	Windows operating system version 3.1, and you switch to Excel 3.0, the current
	open document is not displayed in the CoolSwitch window.
	
	The CoolSwitch window says, "Microsoft Excel" and should say "Microsoft Excel
	C:\DATA\FILENAME.EXT."
