## Q82698: Novell NetWare &amp; NETBIOS.EXE Should Not Run in a VM

	Article: Q82698
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	All versions of Novell's NETBIOS.EXE should not be run in a Microsoft Windows
	operating system version 3.1 MS-DOS virtual machine (VM). Doing so may hang the
	workstation. NETBIOS.EXE is only supported when run globally, and it must be
	started before you start Windows.
	
	NETBIOS.EXE is manufactured by Novell, vendor independent of Microsoft; we make
	no warranty, implied or otherwise, regarding this product's performance or
	reliability.
