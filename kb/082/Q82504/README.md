## Q82504: Epson 7500: Second Page Printed Incorrectly

	Article: Q82504
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	Under the Microsoft Windows version 3.1, with the Epson 7500 printer driver
	installed, printing may fail when the second page of a document is printed to an
	alternate printing tray. The same document, printed from a single printing tray
	does not exhibit this problem.
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here as it
	becomes available.
