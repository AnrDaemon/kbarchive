## Q82337: Canon LBP Series Saves Cartridge Selections

	Article: Q82337
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The Microsoft Windows version 3.1 Canon LBP 4 and LBP III printer drivers save
	the same cartridge settings.
	
	The Canon LBP driver saves settings on a per driver basis. The LBP-4, LBP III,
	and LBP-8IIR are both supported in this version with the same driver. All other
	cartridges selected in the first driver are selected in the second Canon LBP
	driver when installed.
	
	MORE INFORMATION
	================
	
	To remove a cartridge, do the following:
	
	1. Run Control Panel.
	
	2. Select Printers.
	
	3. Choose the Configure button
	
	4. Choose the Setup button.
	
	5. Remove the cartridge, and choose OK.
	
	The correct fonts will be available in each application for printing.
