## Q82851: Extra Extended for MS-DOS version 2.3 requried for Windows 3.1

	Article: Q82851
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Attachmate EXTRA! Extended for MS-DOS, versions earlier than 2.3, do not operate
	under Microsoft Windows 3.1 in standard mode.
	
	If you are running an Attachmate EXTRA! version earlier than 2.3 and are using
	Windows 3.1, you need to update you Attachmate software.
	
	For more information, please contact Attachmate.
	
	The products included here are manufactured by vendors independent of Microsoft;
	we make no warranty, implied or otherwise, regarding these products' performance
	or reliability.
