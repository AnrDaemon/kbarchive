## Q82533: PackRat 3.0 and Windows 3.1

{% raw %}

	Article: Q82533
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Problems may be experienced when using PackRat version 3.0 and the Microsoft
	Windows operating system version 3.1. Problems might include lock ups,
	incorrectly painted dialog boxes and unexpected errors, which may be apparent
	especially when resizing PackRat dialog boxes.
	
	To correct these problems, upgrade to version 4.0 of PackRat.
	
	These problems do not occur in Windows version 3.0.
	
	Additional query words: 3.10 3.11
