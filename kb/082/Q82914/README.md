## Q82914: PC OS2: Icons Stack Over Each Other in Mail for OS/2

	Article: Q82914
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, version 3.0, on platform(s):
	   - the operating system: OS/2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If there are more icons than can adequately fit across the screen in one row,
	the extra icons are stacked on top of each other on the icon that is farthest to
	the right.
	
	STATUS
	======
	
	This is a design limitation of version 1.0 of the Windows Libraries for OS/2
	(WLO) dynamic-link libraries (DLLs).
	
	Additional query words: 3.00 dll
