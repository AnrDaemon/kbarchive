## Q82864: Poor Print Quality in MacInTax for Windows

	Article: Q82864
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Under Microsoft Windows operating system version 3.1, the printing quality for
	MacInTax for Windows is very poor if the files are printed at a high
	resolution.
	
	Changing the printer resolution to the lowest possible setting may correct this
	problem.
	
	MORE INFORMATION
	================
	
	When printing at a high resolution from MacInTax for Windows, the printing
	quality will be very poor. To correct this problem, change the printer
	resolution to the lowest possible setting.
	
	If you are printing with a Hewlett-Packard (HP) LaserJet series printer, be sure
	to install the HP soft fonts when installing MacInTax. Then add the soft fonts
	to Windows by running the Control Panel and choosing the Printers icon.
	
	For HP LaserJet printers, the lowest resolution is 75 dots-per-inch (dpi).
