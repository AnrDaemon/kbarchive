## Q82209: Printer Driver for the HP LaserJet IIP Plus

	Article: Q82209
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft Windows3.1 includes a printer driver for the Hewlett-Packard (HP)
	LaserJet IIP.
	
	There is currently no printer driver available for the HP LaserJet IIP Plus
	printer for Microsoft Windows version 3.0. According to Hewlett-Packard
	Technical Support, the Windows 3.0 HP LaserJet IIP driver should be used when
	printing to the HP LaserJet IIP Plus under the Windows 3.0 environment.
	
	MORE INFORMATION
	================
	
	To install the HP LaserJet IIP Plus driver (HPPCL.DRV version 1.1) with Windows
	3.1, start Control Panel, choose the Printers icon, choose the Add button, and
	select the HP LaserJet IIP Plus from the List Of Printers list.
	
	For additional information, contact Hewlett-Packard Technical Support.
	
	The LaserJet IIP and LaserJet IIP Plus are manufactured by a vendor independent
	of Microsoft; we make no warranty, implied or otherwise, regarding these
	products' performance or reliability.
