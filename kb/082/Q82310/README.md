## Q82310: BizWiz Context-Sensitive Help Doesn't Work with Windows

	Article: Q82310
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): 3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The context-sensitive help feature of BizWiz does not work with Microsoft
	Windows.
	
	
	MORE INFORMATION
	================
	
	BizWiz is manufactured by a vendor independent of Microsoft; we make no
	warranty, implied or otherwise, regarding this product's performance or
	reliability.
	
	Additional query words: 3.10 3.1 Biz Wiz 3.11
