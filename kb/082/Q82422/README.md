## Q82422: No Vector Fonts for HP DeskJet Plus at 300 Dpi

{% raw %}

	Article: Q82422
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When the Microsoft Windows version 3.1 Hewlett-Packard (HP) DeskJet Plus driver
	is used at 300 dots-per-inch (dpi) resolution, there are no vector fonts
	(Modern, Roman, Script) listed for applications that use printer fonts.
	
	WORKAROUND
	==========
	
	Change to a lower printer resolution allows the vector fonts to be listed.
