## Q82308: Using SMARTDrive 4.0 with QuarterDecks DesqView

	Article: Q82308
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SMARTDrive 4.0, included with Microsoft Windows version 3.1, is not fully
	compatible with DesqView.
	
	If DesqView is going to be run with SMARTDrive 4.0, it is best to turn
	off write-behind caching, or turn off SMARTDrive 4.0, when running a
	Windows session under DesqView.
	
	Additional query words: 3.10 3.11
