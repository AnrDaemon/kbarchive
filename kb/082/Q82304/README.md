## Q82304: PC OS2: Incorrect Minimize/Maximize Buttons

	Article: Q82304
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, version 3.0, on platform(s):
	   - the operating system: OS/2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In version 3.0 of Microsoft Mail for OS/2, when you maximize the Message Center
	Viewer (MCV), the minimize and maximize buttons will be positioned right beside
	the Help menu rather than to the right edge of the Mail window as it is under
	Microsoft Windows.
	
	STATUS
	======
	
	This is a design limitation of version 1.0 of the Windows Libraries for OS/2
	(WLO) dynamic-link libraries (DLLs).
	
	Additional query words: 3.00 dll
