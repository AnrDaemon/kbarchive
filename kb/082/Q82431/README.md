## Q82431: GP Fault in Charisma After Insufficient Memory

	Article: Q82431
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	A general protection (GP) fault or unrecoverable application error (UAE) occurs
	with Micrografx Charisma version 2.0 running under Microsoft Windows. The error
	occurs immediately after an insufficient memory message under low memory
	conditions.
