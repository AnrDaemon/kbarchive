## Q82405: Windows 3.1 and LAN Manager 2.1 NetBind Hangs

{% raw %}

	Article: Q82405
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 26-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you use Microsoft LAN Manager version 2.1 and NetBind is not started before
	Microsoft Windows, your computer may stop responding (hang or lock up).
	
	NOTE: This problem may occur under a variety of situations and not just Windows.
	
	WORKAROUND
	==========
	
	To work around this problem, run NetBind before you start Windows.
	
	MORE INFORMATION
	================
	
	NetBind is a utility that comes with LAN Manager version 2.1.
	
	Additional query words: 3.10 3.11 LANMAN net bind
