## Q82599: Mouse Disappears During Hollywood's Install with Idlewild

	Article: Q82599
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	The following process causes the mouse to be invisible during the installation
	of Hollywood:
	
	1. Run Windows Entertainment Pack's (WEP's) Idlewild program.
	
	2. From Program Manager or File Manager, choose Run from the File menu.
	
	3. With Hollywood Disk 1 in drive A, type the following and press ENTER:
	
	  a:install
	
	MORE INFORMATION
	================
	
	This behavior occurs because the Blank Pointer option in Idlewild is set.
	
	
	If Blank Pointer is set, Idlewild automatically hides the mouse cursor after a
	few seconds (that is, before the screen saver starts).
	
	Additional query words: 3rdparty idle wild 1.0
