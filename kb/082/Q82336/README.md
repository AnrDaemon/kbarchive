## Q82336: Windows 3.1: Okidata OL-400/800 Font Cards Not Supported

	Article: Q82336
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	Currently in Microsoft Windows version 3.1, there is no font cartridge support
	for the Okidata OL-400 and OL-800 laser printers (HP LaserJet II compatible).
	
	Also, there is no support for the built-in fonts on the OL-400 and OL-800. These
	include the font equivalent to HP font cartridges A, B, C, F, and L.
