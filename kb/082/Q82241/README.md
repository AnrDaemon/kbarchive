## Q82241: README.WRI Section 11.4 No Longer Valid (Everex, EMM386.EXE)

	Article: Q82241
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): 3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following section (11.4) of the Windows 3.1 README.WRI file no longer
	applies to Windows 3.1:
	
	- 11.4 Everex 386/25 with EMM386.EXE
	
	If you are using an Everex 386/25 with EMM386.EXE, include the following options
	on the device=emm386.exe command line in your CONFIG.SYS file:
	
	     device=EMM386.EXE X=C600-C7FF
	
	Additional query words: 3.1 3.10
