## Q82394: Freelance Arrange Align Graphic Display Is Missing

{% raw %}

	Article: Q82394
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When Lotus Freelance version 1.0's sample box demonstrating how to do alignment
	is displayed in Windows 3.1 or 3.11, the graphic is missing (everything else is
	there). In Windows 3.0, a small graphic schematic showing alignment is
	displayed.
	
	This is a cosmetic problem only; no functionality is lost.
	
	CAUSE
	=====
	
	This is caused by a problem in Lotus Freelance.
