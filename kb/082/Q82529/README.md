## Q82529: Using Norton Utilities with Erase Protect with Windows

	Article: Q82529
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Setting up a temporary swap file with Microsoft Windows operating system version
	3.1 may fail if Norton Utilities's EP.EXE is running. This problem occurs only
	with Norton Utilities version 5.0 or earlier. EP.EXE is Norton's Erase Protect
	program.
	
	This problem does not occur in Erase Protect from Norton Utilities version 6.0 or
	later.
	
	The Norton Untilities product included here is manufactured by a vendor
	independent of Microsoft; we make no warranty, implied otherwise, regarding this
	performance or reliability.
