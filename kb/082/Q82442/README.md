## Q82442: PowerPoint and Micrografx Picture Publisher

	Article: Q82442
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If Microsoft PowerPoint is running, Micrografx Picture Publisher cannot start.
	
	CAUSE
	=====
	
	Microsoft PowerPoint and Micrografx Picture Publisher do not run reliably at the
	same time under the Microsoft Windows. This is because the module name for
	PowerPoint is PP, and the module name and executable filename for Picture
	Publisher are PP and PP.EXE, respectively.
	
	MORE INFORMATION
	================
	
	The Micrografx products included here is manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
