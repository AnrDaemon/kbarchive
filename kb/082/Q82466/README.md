## Q82466: Cognose Powerplay's Files Box Is Not Refreshed Correctly

	Article: Q82466
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In the application Cognos Powerplay, the Files box is incorrectly refreshed when
	a new directory is selected in the Directories box.
	
	When you display a new directory, if it has fewer qualifying files than the first
	directory, some of the files from the first directory are still displayed.
	
	If there are fewer qualifying files in the second directory, then the remaining
	files from the first directory are still displayed along with the new files.
