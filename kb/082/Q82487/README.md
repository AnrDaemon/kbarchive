## Q82487: Dark Printing TIFF Images to Panasonic KX-P1180

	Article: Q82487
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Printing a monochrome TIFF image in Microsoft Publisher to a Panasonic KX-P1180
	printer results in overly dark output that may print over other images.
	
	RESOLUTION
	==========
	
	Either of the following set of printer settings should be used to correct this
	problem:
	
	  Resolution 120 x 144
	  Dithering=Fine
	  Intensity Control=Lightest
	
	  Resolution 244 x 144
	  Dithering=Coarse
	  Intensity Control=Lightest
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here as it
	becomes available.
