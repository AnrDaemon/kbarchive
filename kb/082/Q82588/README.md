## Q82588: Icon DoIt 3.0 Requires Update to Run with Windows 3.1

{% raw %}

	Article: Q82588
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): win31
	Last Modified: 06-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The animated icon feature in Icon DoIt does not work with Windows 3.1. Icon DoIt
	is an icon and cursor replacement/animation program manufactured by Moon Valley
	Software. This program also cannot read the existing icons in Program Manager,
	in Windows 3.1. (Icon DoIt could read the icons in Windows 3.0a.)
	
	These problems have been corrected in Icon DoIt version 3.1.
	
	For more information, contact Moon Valley Software.
	
	
	Icon DoIt is manufactured by Moon Valley Software, a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
