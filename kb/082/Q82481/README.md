## Q82481: Error: Unable to Retain Printer Codes

	Article: Q82481
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you upgrade to Microsoft Windows 3.1 or 3.11 over Windows 3.0, the
	following error message may be displayed:
	
	  Unable to retain printer codes.
	  Use default printer codes.
	
	CAUSE
	=====
	
	This error message is displayed if the Generic Text printer driver is being used
	in Windows 3.0. The Windows 3.1 Setup program does not preserve printer codes
	for this driver.
	
	WORKAROUND
	==========
	
	If this error message is displayed, you may need to reinstall the Windows 3.0
	printer driver.
