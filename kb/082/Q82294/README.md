## Q82294: PC Win: Client Version Control Not Available

	Article: Q82294
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft Mail for Windows version 3.0 does not provide client version control.
	
	Client version control is a feature that notifies the user if there is a more
	current version of the Windows client on the file server and asks them to
	upgrade.
