## Q82492: Canon LBP III: Time-Out Printing Mixed Orientations

{% raw %}

	Article: Q82492
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	Printing time-outs may occur if a document containing both portrait and
	landscape orientations is printed to a Canon LBP III printer under Microsoft
	Windows oversion 3.1. (This problem may occur on other Canon laser printers as
	well.)
	
	WORKAROUND
	==========
	
	To work around this problem, increase the time-out value to 120 or more. To
	increase the time-out value, do the following:
	
	1. Choose the Printers icon from the Control Panel window. The Printers dialog
	  box appears.
	
	2. Choose the Connect button.
	
	3. Specify the time-out setting you want.
	
	4. Choose the OK button.
