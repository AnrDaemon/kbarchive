## Q82516: DEC Pathworks System Error with Windows 3.1

	Article: Q82516
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you are using Microsoft Windows operating system version 3.1 on workstation
	A, and workstation B boots up to the Digital Equipment Corporation (DEC)
	Pathworks network with the same node name, a system error occurs on workstation
	A.
	
	The error dialog box allows only the Cancel button to be used. When Cancel is
	chosen, a general protection (GP) fault occurs in the USER.EXE file.
	
	Workstation B does not connect to the network.
	
	DEC Pathworks is manufactured by a vendor independent of Microsoft; we make no
	warranty, implied or otherwise, regarding this product's performance or
	reliability.
	
	Additional query words: 3.10 3.11 3.1
