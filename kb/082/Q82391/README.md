## Q82391: Windows Hangs with Kurta Mouse Driver

	Article: Q82391
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If the Kurta mouse driver is installed in the SYSTEM.INI file, Microsoft Windows
	may stop responding (hang) on startup.
	
	This problem did not occur in Windows 3.0.
	
	CAUSE
	=====
	
	This is a problem with the Kurta mouse driver.
	
	
	RESOLUTION
	==========
	
	Contact Kurta for information regarding an updated mouse driver.
