	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Microsoft SharePoint Portal Server 2001
	Resource Kit, ISBN 0-7356-1562-4.
	
	The following topics are covered:
	
	- Page 531: Correction To RTF IFilter Description
	
	MORE INFORMATION
	================
	
	Page 531: Correction To RTF IFilter Description
	-----------------------------------------------
	
	On page 531, under the section "RTF IFilter (filters\RTF)," change:
	
	"This tool reports the time it takes to access a document stored in a SharePoint
	Portal Server workspace. Use this tool to determine how load is effecting
	performance of the server. Use this tool as a simple test to verify that the
	server is functioning normally. The output of the access attempt is reported to
	help you to diagnose any failures that occur.
	
	This tool is located under \Tools\server_latency on the resource kit CD. The
	following files are included in this directory:"
	
	To:
	
	"This tool enables Microsoft SharePoint Portal Server 2001 to crawl documents
	that have an .rtf extension. Before use, the administrator must register the
	file on the server.
	
	This tool is located under \Tools\filters\RTF on the resource kit CD. The
	following files are included in this directory:"
	
	
	Microsoft Press is committed to providing informative and accurate books. All

	already contain most or all of the above corrections.
