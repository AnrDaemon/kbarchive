## Q326779: Supported IIS Configurations for Use with Sysprep

{% raw %}

	Article: Q326779
	Product(s): Internet Information Server
	Version(s): 4.0,5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 23-JUL-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Internet Information Server version 4.0 
	- Microsoft Internet Information Services version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes IIS configurations that are supported for use with
	Sysprep.
	
	MORE INFORMATION
	================
	
	A systems that is cloned with Sysprep is supported for Internet Information
	Server (IIS) 4.0 and Internet Information Services (IIS) 5.0 when the system
	uses the following:
	
	Microsoft Windows NT 4.0 with IIS 4.0:
	
	- Windows NT 4.0 with Service Pack 6a or later.
	
	- IIS 4.0 installed from the Windows NT Option Pack (NTOP).
	
	- Microsoft Internet Explorer 5.01 or later.
	
	Microsoft Windows 2000 with IIS 5.0:
	
	- Windows 2000 or later.
	
	- IIS 5.0 installed from Windows 2000.
	
	- Internet Explorer 5.01 or later.
	
	Microsoft does not support IIS when it is installed on earlier versions of
	Windows NT or earlier versions of Internet Explorer and the system is cloned
	with Sysprep.
	
	The information in this article supersedes the information in the following
	Microsoft Knowledge Base article:
	
	  Q248739 IIS Fails to Run on a Sysprep Cloned System
	
	The issue that is referenced in this article applies to earlier versions of
	Internet Explorer and Windows NT.
