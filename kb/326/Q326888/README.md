## Q326888: Network+ Certification Training Kit, Second Edition Comments and

	Article: Q326888
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 30-JUL-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Network+ Certification Training Kit Second Edition, ISBN 0-7356-1456-3 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Network+ Certification Training Kit, Second
	Edition, ISBN 0-7356-1456-3.
	
	The following topics are covered:
	
	- Page 240: Address Should Be Number
	
	- Page 241: 192.168.2.2 Should Be 1
	
	- Page 241: Correction To Second To Last Bullet
	
	- Page 370: Novell Product Name Is Incorrect
	
	- Page 470: Incorrect URL
	
	- Page 666: SPA Should Be STA
	
	MORE INFORMATION
	================
	
	Page 240: Address Should Be Number
	----------------------------------
	
	In the last bullet on page 240,
	
	Change:
	"...specifies the address of the network interface adapter..."
	
	To:
	"...specifies the number of the network interface adapter..."
	
	
	Page 241: 192.168.2.2 Should Be 1
	---------------------------------
	
	On page 241, in the command line in the middle of the page,
	
	Change:
	"ROUTE ADD 192.168.5.0 MASK 255.255.255.0 192.168.2.7. IF 192.168.2.2 METRIC 1"
	
	To:
	"ROUTE ADD 192.168.5.0 MASK 255.255.255.0 192.168.2.7. IF 1 METRIC 1"
	
	
	Page 241: Correction To Second To Last Bullet
	---------------------------------------------
	
	On page 241, in the second to last bullet,
	
	Change:
	"IF 192.168.2.2<A0><A0><A0><A0><A0>The address of the network interface adapter..."
	
	To:
	"IF 1<A0><A0><A0><A0><A0>The number of the network interface adapter..."
	
	
	Page 370: Novell Product Name Is Incorrect
	------------------------------------------
	
	In the last sentence on page 370,
	
	Change:
	"Novell with NetWare Cluster Services for NetWare 5.1"
	
	To:
	"Novell NetWare Cluster Services for NetWare 5.1"
	
	
	Page 470: Incorrect URL
	-----------------------
	
	The last bullet on page 470 (Technical Support Databases) lists an incorrect
	URL.
	
	Change:
	"http://www.microsoft.com/mspress/support/search.asp"
	
	To:
	"http://support.microsoft.com"
	
	
	Page 666: SPA Should Be STA
	---------------------------
	
	In three locations on the page 666 glossary, the acronym for the spanning tree
	algorithm is incorrect:
	
	Change:
	"SPA<A0><A0><A0><A0>See spanning tree algorithm (SPA)"
	
	To:
	"STA<A0><A0><A0><A0>See spanning tree algorithm (STA)"
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
