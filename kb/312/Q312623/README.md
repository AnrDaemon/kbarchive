## Q312623: Adding New Secondary User: After Choosing User Name, MSN Explore

{% raw %}

	Article: Q312623
	Product(s): The Microsoft Network
	Version(s): 6.0,6.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 6.0, 6.1 
	- The Microsoft Network Version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	- You have MSN Internet Access, and you click Add New User. You enter a user
	  name, click OK, and then no message returns to confirm for you that the user
	  name you chose was generated.
	
	- You have MSN Internet Access, and you click Add New User. You choose a user
	  name that is already taken, but no message returns to tell you that the user
	  ID you chose is already taken.
	
	CAUSE
	=====
	
	MSN Explorer was able to verify the user name has been taken, but no message
	returns to tell you that the user ID is already taken.
	
	RESOLUTION
	==========
	
	1. Start MSN Explorer, and then click Add New User.
	
	2. Go through the sign-up process for adding a new user.
	
	3. When prompted to choose a user name, pick one that is different from the
	  names you previously entered.
	
	4. If MSN Explorer does not return any message and just stops responding, do one
	  of the following:
	   - Type another user name, and then click OK.
	
	   - Click the Back button, and then type another user name.
