## Q312086: Indexing Service Office Filter Cannot Process Trademark Symbol

	Article: Q312086
	Product(s): Microsoft Windows NT
	Version(s): 4.0,SP2
	Operating System(s): 
	Keyword(s): kbenv kbtool kbCOMIS kbWin2000sp3fix kbOfficeXPsp2fix
	Last Modified: 16-AUG-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- the operating system: Microsoft Windows NT 4.0 
	- the operating system: Microsoft Windows 2000 SP2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Words that are followed by the "registered trademark" symbol may not be indexed
	correctly by the Indexing service.
	
	CAUSE
	=====
	
	This problem may be caused by a logic error in the Office filter.
	
	RESOLUTION
	==========
	
	To resolve this problem, obtain the latest service pack for Microsoft Office XP.
	For additional information, click the following article number to view the
	article in the Microsoft Knowledge Base:
	
	  Q307841 OFFXP: How to Obtain the Latest Office XP Service Pack
