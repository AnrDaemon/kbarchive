## Q198383: WD97: Text Fails to Appear in Envelopes/Labels Delivery Box

	Article: Q198383
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbenvelope word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	Address text at the insertion point does not appear in the Delivery Address or
	the Address box of the Envelopes And Labels dialog box.
	
	CAUSE
	=====
	
	Word does not recognize the text as a valid address.
	
	WORKAROUND
	==========
	
	If any line of the address begins with a tab, the address does not appear when
	you display the Envelopes and Labels dialog box.
	
	To work around this problem, do either of the following:
	
	- Select the address prior to choosing to display the Envelopes And Labels
	  dialog box (on the Tools menu).
	
	  -or-
	
	- Remove the tabs from the beginning of the address lines in the document.
	
	MORE INFORMATION
	================
	
	For additional information, please see the following article in the Microsoft
	Knowledge Base:
	
	  Q142343 WD: Selection with Tab Not Inserted in Delivery Address Box
