## Q198588: Spaces in Contact Web Page URL are Changed to %20

	Article: Q198588
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook Web Access, version 5.5 
	- Microsoft Outlook Web Access, version 5.5 Service Pack 1 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you use Outlook Web Access to view Contacts, spaces in a Contact's Web page
	address are converted to the string "%20" after you attempt to view the page.
	
	CAUSE
	=====
	
	The URL is encoded to fix a Netscape incompatability with URLs that have spaces
	in them.
