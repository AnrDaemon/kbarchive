## Q198103: Combat Flight Simulator: Textures in Cockpit Flicker

	Article: Q198103
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator: WWII Europe Series, version 1.0 
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Combat Flight Simulator or Microsoft Combat Flight
	Simulator 2, textures in the cockpit may flicker.
	
	CAUSE
	=====
	
	This behavior can occur if a display adapter based on the NVidia Riva TNT chip
	set is installed in your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, obtain and install the latest video driver for the
	display adapter installed in your computer. To do this, contact the manufacturer
	of your display adapter.
	
	For information about how to contact your display adapter manufacturer, query in
	the Knowledge Base for one of the following articles:
	
	  Q65416 Hardware and Software Third-Party Vendor Contact List, A-K
	
	  Q60781 Hardware and Software Third-Party Vendor Contact List, L-P
	
	  Q60782 Hardware and Software Third-Party Vendor Contact List, Q-Z
	
	
	If the issue continues to occur, download and install the generic Riva TNT video
	driver from the following NVidia Web site:
	
	  http://www.nvidia.com
