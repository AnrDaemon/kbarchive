## Q198323: WD: Cannot Move Indent Marker Without Moving Tab Marker

	Article: Q198323
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you try to move the indent marker (on the horizontal ruler) by positioning
	your mouse pointer over the marker and there is a tab marker set at the same or
	near the same location as the indent marker, the tab marker moves instead.
	
	WORKAROUND
	==========
	
	Move the tab marker, and then move the indent marker.
