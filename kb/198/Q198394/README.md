## Q198394: Combat Flight Simulator: Aircraft Lighting Is Not Available

	Article: Q198394
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbui kbdocerr kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator: WWII Europe Series, version 1.0 
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Aircraft lighting is not available for Combat Flight Simulator or Combat Flight
	Simulator 2 aircraft.
	
	The tables on pages 28 and 185 of the Pilot's Manual list keyboard commands to
	toggle All Lights (L), Landing Lights (CTRL+L), and Panel Lights (SHIFT+L).
	
	These keyboard commands are not available for Combat Flight Simulator aircraft,
	which have no lights.
	
	The table on page 274 of the Pilot's Manual No. 2 list keyboard commands to
	toggle All Lights (L), and Landing Lights (CTRL+L).
	
	These keyboard commands are not available for Combat Flight Simulator 2 aircraft,
	which have no lights.
	
	MORE INFORMATION
	================
	
	On aircraft you import from Microsoft Flight Simulator 98 or Microsoft Flight
	Simulator 2000, press L to toggle exterior lights, and CTRL+L to toggle landing
	lights.
	
	NOTE: Combat Flight Simulator does not support Panel Lights on aircraft you
	import from Flight Simulator 98 or Microsoft Flight Simulator 2000.
