## Q198363: WD97: Deleted Text Not Displayed When Hidden Text Option Set

	Article: Q198363
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you use the Track Changes While Editing option, text marked as Deleted Text
	disappears, even when the option to view all non-printing or hidden characters
	is turned on.
	
	CAUSE
	=====
	
	The Deleted Text Mark option has been set as Hidden.
	
	WORKAROUND
	==========
	
	Change the Deleted Text Mark option to Strikethrough.
	
	To change the Deleted Text Mark option to Strikethrough, follow these steps:
	
	1. On the Tools menu, point to Track Changes, and click Highlight Changes.
	
	2. Click Options.
	
	3. In the Delete text list, select Strikethrough.
	
	4. Click OK to accept and return to the Highlight Changes dialog box.
	
	5. Click OK to return to the document.
	
	Additional query words: redline strikeout dissappear missing
