## Q198576: Combat FS: Polygons Are Visible When You Fly Through Fog

	Article: Q198576
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbenv kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator: WWII Europe Series, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Combat Flight Simulator, you may see the edges or
	outlines of polygons when you fly through fog.
	
	CAUSE
	=====
	
	This behavior can occur if a display adapter based on the PowerVR PCX2 chipset
	is installed in your computer.
	
	NOTE: For technical information about the PowerVR PCX2 chipset, please visit the
	following PowerVR Web site:
	
	  http://www.powervr.com
	
	RESOLUTION
	==========
	
	To work around this issue, disable hardware rendering in Combat Flight
	Simulator. To do this, follow these steps:
	
	1. On the Combat Flight Simulator startup screen, click Settings.
	
	2. Click to clear the Enable Hardware Acceleration check box, and then click OK.
	
	MORE INFORMATION
	================
	
	This behavior has been reported with the following display adapter:
	
	- Grafixstar Apocalypse 5D
