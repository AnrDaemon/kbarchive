## Q198367: WD97: Text Box Position Not Retained When Copying and Pasting

{% raw %}

	Article: Q198367
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): word97 kblayout
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you copy and paste a text box to another page, the exact position may not
	be retained.
	
	For example, if you set horizontal and vertical positions to 2 inches for a text
	box and then copy and paste the text box to another page, the horizontal and
	vertical positions change to 2.17 inches.
	
	WORKAROUND
	==========
	
	Reset the position of the text box to the value you want.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
