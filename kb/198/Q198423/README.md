## Q198423: Combat Flight Simulator: How to Change the Simulation Rate

{% raw %}

	Article: Q198423
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbui kbdocerr kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator: WWII Europe Series, version 1.0 
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to change the simulation rate used in Microsoft
	Combat Flight Simulator.
	
	MORE INFORMATION
	================
	
	To change the simulation rate used in Combat Flight Simulator, use one of the
	following methods:
	
	- On the Options menu, point to Simulation Rate, and then click the simulation
	  rate you want to use.
	
	Combat Flight Simulator
	-----------------------
	
	- Press and hold down the R key, and then press the HYPHEN (-) key to reduce
	  the simulation rate by one setting, or the EQUAL SIGN (=) key to increase the
	  simulation rate by one setting.
	
	Combat Flight Simulator 2
	-------------------------
	
	- Press ALT+O, and then press ALT+M.
	
	The following simulation rates are available in Combat Flight Simulator:
	
	- Slowest
	- Half Speed
	- Normal
	- 2x Speed
	- 4x Speed
	
	The default simulation rate is Normal.
	
	NOTE: The Game Commands topic in Combat Flight Simulator online Help incorrectly
	states that you can change the simulation rate (select time compression) by
	pressing the R key alone.
