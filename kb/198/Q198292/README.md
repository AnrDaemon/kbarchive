## Q198292: WD97: Default AutoCaption Label for Equation Uses Figure

{% raw %}

	Article: Q198292
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you use the AutoCaption feature to insert a caption for a Microsoft
	Equation 3.0 object (on the Insert Menu, click Caption, and then click
	AutoCaption), the Use Label default is Figure instead of Equation.
	
	WORKAROUND
	==========
	
	To work around this problem, click Equation in the Use Label list.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
