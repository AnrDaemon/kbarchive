## Q198560: XFOR: Update Meeting Requests from Schedule+ to Profs

	Article: Q198560
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5 SP2,5.5 SP3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-APR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 5.5 SP2, 5.5 SP3 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When an update to a meeting request is sent from Schedule+ to a PROFS user, the
	update is received on the PROFS side as a new meeting request with no indication
	that it is an update.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server
	version 5.5.
