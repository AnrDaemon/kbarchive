## Q198077: WD97: Tab Leaders Unavailable in Format Tabs Dialog Box

	Article: Q198077
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you select Bar in the Alignment section of the Tabs dialog box (click Tabs
	on the Format menu) to set a custom tab stop, and then click Clear All, the
	alignment is set to Left automatically and the Leader options are not available
	(dimmed).
	
	WORKAROUND
	==========
	
	To work around this problem and make the Leader options available to select,
	click any alignment option other than Bar.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
