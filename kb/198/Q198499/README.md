## Q198499: XWEB: Post Reply to Custom HTML Form Posts in Web Forms Folder

	Article: Q198499
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.5,5.5SP1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook Web Access, version 5.5 
	- Microsoft Outlook Web Access, version 5.5 Service Pack 1 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you receive a custom form using a POP3 client, you get a URL to the Outlook
	Web Access (OWA) Server where the HTML form is installed. After you start OWA
	and access the form, if you post a reply, the form posts to the Web Forms
	folder.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
