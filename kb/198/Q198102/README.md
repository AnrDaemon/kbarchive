## Q198102: Combat Flight Simulator: Terrain and 3D Objects Flicker

{% raw %}

	Article: Q198102
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator: WWII Europe Series, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Combat Flight Simulator, terrain and other 3D objects on
	the screen may flicker. This effect is most noticeable over mountains.
	
	CAUSE
	=====
	
	This behavior can occur if an ATI Rage Pro display adapter or a display adapter
	based on the ATI Rage Pro chipset is installed in your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest video driver for your
	display adapter from the following ATI Web site:
	
	  http://www.atitech.com
