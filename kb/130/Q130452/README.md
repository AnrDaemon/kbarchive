## Q130452: Cannot Run Franz Allegro CL

	Article: Q130452
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You cannot run Franz Allegro CL, a development environment for the common LISP
	language by Franz, Inc., in Windows 95. The program's main window appears
	temporarily, but then vanishes and the program does not start.
	
	CAUSE
	=====
	
	This program is not compatible with Windows 95.
	
	RESOLUTION
	==========
	
	To run Franz Allegro CL in Windows 95, you must obtain a patch. To obtain the
	patch, please contact Franz, Inc.
