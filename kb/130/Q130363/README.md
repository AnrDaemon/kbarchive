## Q130363: Stack Overflow Playing Movie Clips in THE ANIMALS!

	Article: Q130363
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to play, or modify the volume level for, certain movie clips in THE
	ANIMALS!, you receive the following error message:
	
	  Stack Overflow
	
	CAUSE
	=====
	
	You are trying to play, or modify the volume level for, a movie clip that does
	not have sound associated with it from Animals Alphabetical. This is a known
	problem with THE ANIMALS! that causes a stack overflow to occur.
	
	RESOLUTION
	==========
	
	To resolve this problem, use either of the following methods:
	
	- Play, or modify the volume level for, the movie clip from Movie Library
	  instead of from Animals Alphabetical.
	
	- Upgrade to version 2.0 of THE ANIMALS!
