## Q130480: GSNW Share Names Displayed as DELETED with RAS Server Running

	Article: Q130480
	Product(s): Microsoft Windows NT
	Version(s): 3.50
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 3.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If the Gateway Service for NetWare (GSNW) and the Remote Access Service (RAS)
	server is running simultaneously, the Share Name in the Configure Gateway dialog
	box for GSNW displays the following:
	
	  *DELETED*
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 3.5. This
	problem was corrected in the latest US Service Pack for Windows NT. For
	information on obtaining this update, query on the following word in the
	Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
	
	Additional query words: prodnt nwc cpl novell
