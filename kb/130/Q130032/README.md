## Q130032: XCLN: Duplicate Address Types When Creating Entry in PAB

	Article: Q130032
	Product(s): Microsoft Exchange
	Version(s): 4.0 5.0
	Operating System(s): 
	Keyword(s): kbother
	Last Modified: 10-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Windows 3.x client, versions 4.0, 5.0 
	- Microsoft Exchange Windows NT client, versions 4.0, 5.0 
	- Microsoft Exchange Windows 95/98 client, versions 4.0, 5.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you add an entry to the personal address book, you may see duplicate
	address types.
	
	CAUSE
	=====
	
	This problem occurs when you add the Microsoft Mail transport (by choosing
	Services on the Tools menu, clicking the Add button, then selecting Microsoft
	Mail.
	
	MORE INFORMATION
	================
	
	The first entry listed is for sending mail through Microsoft Exchange. The
	second entry sends mail through the Microsoft Mail 3.2 transport.
