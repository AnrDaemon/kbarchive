## Q130403: Cannot Install Unlisted Driver During Setup

{% raw %}

	Article: Q130403
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You cannot install an unlisted or updated driver when you are installing Windows
	95.
	
	CAUSE
	=====
	
	The Windows 95 Setup program does not have a button or command to allow you to
	install an unlisted or updated driver.
	
	RESOLUTION
	==========
	
	Install the unlisted or updated driver after Setup is finished.
