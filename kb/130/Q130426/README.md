## Q130426: Schedule+ Tasks: Duration and Estimated Effort

	Article: Q130426
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0; Win95:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	The Duration field on the To Do tab and the Estimated Effort field on the
	Status tab of the Task dialog box can appear to present the same
	information.
	
	The Duration field on the To Do tab represents the time between when the
	task starts and when it is finished. The Estimated Effort field on the
	Status tab of the Task dialog box represents how much work will actually be
	performed on the task.
