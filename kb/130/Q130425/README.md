## Q130425: Weekly - 5 Day Print Layout Always Begins on Monday

	Article: Q130425
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0; Win95:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You can specify your week to start on any day of the week (for example,
	Wednesday) by choosing Schedule Settings on the Tools menu and setting the Week
	Starts On option to Wednesday. However, when you print your schedule and you
	select the Weekly - 5 Day print layout, the week always begins on Monday.
	
	WORKAROUND
	==========
	
	To print a schedule that reflects the Week Starts On setting, select either the
	Weekly - 7 Day or Monthly print layout.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Schedule+ version 7.0.
	We are researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
