## Q130714: ScanDisk Marks Numerous Bad Sectors with XtraDrive

	Article: Q130714
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	ScanDisk for Windows marks numerous bad sectors on an XtraDrive 3.0 compressed
	drive during a thorough test (surface scan).
	
	CAUSE
	=====
	
	ScanDisk's thorough test is incompatible with XtraDrive 3.0.
	
	RESOLUTION
	==========
	
	XtraDrive should repair the compressed volume file automatically after you
	restart your computer.
	
	MORE INFORMATION
	================
	
	You can use ScanDisk's standard test to check for file allocation table (FAT)
	errors. Please contact Integrated Information Technology for information about
	surface scan utilities that are compatible with XtraDrive.
	
	XtraDrive is manufactured by Integrated Information Technology, a vendor
	independent of Microsoft; we make no warranty, implied or otherwise, regarding
	this product's performance or reliability.
