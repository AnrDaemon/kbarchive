## Q130756: PC Adm: Err Msg: Notice 28 Mailbox Recovered

	Article: Q130756
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0a,3.2,3.2a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, versions 3.0a, 3.2, 3.2a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to recover a lost password with ADMIN.EXE, the following
	message shows on the screen at the end of the procedure:
	
	  Notice 28
	  Mailbox recovered.
	  Global password used.
	  Press Enter to continue
	
	WORKAROUND
	==========
	
	This is not an error message. It is a confirmation that the password has been
	recovered to the global password.
	
	MORE INFORMATION
	================
	
	For more information on password recovery, please refer to the Microsoft Mail
	for PC Networks "Administrator's Guide" section "Recovering a Lost Password"
	pages 104-105 for version 3.2 & 3.2a, page 99 for version 3.0b.
	
	Additional query words: 3.00a 3.20 3.20a
