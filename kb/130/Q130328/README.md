## Q130328: Text After Carriage Return Not Pasted in MS Exchange Subject

	Article: Q130328
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you paste text that contains a carriage return into the Subject line in a
	Microsoft Exchange message, any text after the carriage return is not pasted.
	
	CAUSE
	=====
	
	The Subject line in a Microsoft Exchange message is a single-line field. Text
	after a carriage return is automatically cut off.
	
	RESOLUTION
	==========
	
	Remove all the carriage returns from any text you want to copy and paste into
	the Subject line before you copy the text.
