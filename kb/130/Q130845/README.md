## Q130845: FAX: Intel SatisFAXtion 200 No Longer Sold by Intel

	Article: Q130845
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to Fax, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Intel SatisFAXtion 200 recommended in the Microsoft Mail Gateway to Fax
	"Administrator's Guide" is no longer sold by Intel. The board, now called PDI
	200, is now made by Pure Data. To order the PDI 200, call Pure Data at (905)
	731-6444.
	
	Additional query words: 3.00 Satisfaction Satisfaxion
