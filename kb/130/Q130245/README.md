## Q130245: MAC Dangerous Creatures: Boxes Appear When Typing

{% raw %}

	Article: Q130245
	Product(s): Microsoft Home Multimedia Titles
	Version(s): MACINTOSH:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Dangerous Creatures for the Macintosh, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you enter information in the Find or Gallery Lists, boxes may appear when
	any of following keyboard keys are pressed:
	
	  INS/HELP
	  HOME
	  PAGE UP
	  DEL
	  END
	  PAGE DOWN
	
	MORE INFORMATION
	================
	
	These six keys are available only on the Macintosh Extended Keyboard, directly
	above the between the numeric keypad and standard keyboard keys.
	
	Microsoft has confirmed this to be a problem in Dangerous Creatures version 1.0.
	We are researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
