## Q130852: Text Assist Requires Updated File to Function in Windows 95

	Article: Q130852
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	None of the programs in Text Assist, an application suite published by Creative
	Labs, functions in Windows 95.
	
	CAUSE
	=====
	
	The Text Assist application suite requires an updated file (CTSW.EXE) to work
	correctly in Windows 95.
	
	RESOLUTION
	==========
	
	To obtain the updated CTSW.EXE file, please contact Creative Labs.
