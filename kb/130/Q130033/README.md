	Microsoft Exchange Server Host Migration Tools Support and Services
	
	-------------------------------------------------------------------
	
	Microsoft Exchange Server Host Migration tools assist in the process of
	migrating from the following host based messaging systems to Microsoft
	Exchange Server:
	
	- IBM PROFS/OfficeVision
	- Digital ALL-IN-1
	- Verimation Memo
	
	These comprehensive tools provide host administrators with a means of
	migrating users accounts, private e-mail messages, shared information,
	attachments, documents, calendar data and personal address data to
	Microsoft Exchange Server.
	
	To receive support and customization services around these tools, host
	administrators have the following options:
	
	1. Microsoft Consulting Services. Microsoft Consulting Services is
	  providing services to assist in migrating to Exchange Server.
	
	2. Independent Consultants and Software Vendors. A number of Independent
	  Software Vendors and Consultants provide fee for service and products
	  to assist you in your migration planning, implementation and
	  customization.
	
	See information below for contact information and description of migration
	services and products. Also see the Microsoft Exchange Server home page
	http://www.microsoft.com/exchange for a up-to-date listing of Microsoft
	Exchange Server compatible products and services.
	
	3. Self support/customization. Included with host migration is tools and
	  documentation that will enable you to manage your migration
	  (documentation is located in the folder \migrate\docs). The source code
	  for each tool is provided which enables your system programmers,
	  solution providers or system integrators to support and tailor the
	  tools to your specific need.
	
	NOTE: The PROFS/OV, ALL-IN-1 and Memo host source extractors are not
	supported through standard Microsoft Product Support Services such as the
	Premier Support Plan.
