## Q130026: Best of Windows Entertainment Pack: Listing of Games

	Article: Q130026
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	kbother
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Best of Windows Entertainment Pack, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Best of Windows Entertainment Pack comprises 13 games from previously
	released Windows Entertainment Packs.
	
	The following games are included:
	
	- From Windows Entertainment Pack 1: Golf, Taipei, and Tetris.
	
	- From Windows Entertainment Pack 2: Free Cell, Pipe Dream, Rodent's Revenge,
	  and Tut's Tomb.
	
	- From Windows Entertainment Pack 3: SkiFree, TetraVex, TriPeaks.
	
	- From Windows Entertainment Pack 4: Chip's Challenge, Dr. Black Jack, and Jezz
	  Ball.
