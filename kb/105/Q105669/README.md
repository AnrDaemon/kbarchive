## Q105669: Scenes 1.0: Screen Saver Does Not Activate When Menu Is Active

{% raw %}

	Article: Q105669
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Scenes for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	If an option or command from the menu bar is dropped down or
	highlighted, the Microsoft Scenes screen saver does not automatically
	activate.
