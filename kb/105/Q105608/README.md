## Q105608: MS-DOS 6.2 MEM Supports /A Switch

	Article: Q105608
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The MS-DOS 6.2 MEM command supports the new /A switch. This switch displays how
	much memory is available in the high memory area (HMA). For example:
	
	  Available space in High Memory Area    16K  (16,880 bytes)
	  MS-DOS is resident in the high memory area.
	
	This switch was added to the product too late to be included in the
	documentation.
