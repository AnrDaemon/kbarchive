## Q105566: Gtwy: Exit Codes for Gateway MTA Versions 3.0 and Later

	Article: Q105566
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateways 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following is a list of the exit codes for versions 3.0 and later of the
	Microsoft Mail gateways:
	
	  BREAK_AB_TIME            60
	  BREAK_AB_DATE            61
	  BREAK_AB_TIMEDATE        62
	  BREAK_TIME_RELATIVE      63
	  CYCLE_COUNT_REACHED      64
	  ERROR_EXIT               65
	  TERMINATED_BY_USER       66
	
	Exit codes are displayed when you exit the Gateway program to MS-DOS or another
	program such as DISPATCH.EXE.
	
	
	Additional query words: 3.00
