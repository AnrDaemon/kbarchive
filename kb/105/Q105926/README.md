## Q105926: MSD Version 2.1 Does Not Detect Gateway2000 ColorBook

{% raw %}

	Article: Q105926
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 03-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.11 
	-------------------------------------------------------------------------------
	
	Microsoft Diagnostics (MSD) version 2.1 detects the Gateway2000 ColorBook
	as an unknown computer. Since the Gateway2000 ColorBook BIOS does not
	contain the word Gateway, MSD cannot identify the computer type.
	
	Additional query words: 3.11 gateway 2000 color book
