## Q105930: PC WRmt: Mail Remote Displays &quot;SENDING&quot; When It Isn't Sending

	Article: Q105930
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Remote for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	With version 3.2 of Microsoft Mail Remote for Windows, if you choose not to send
	any messages, or if you select Get Headers from the headers dialog box, the
	string "[xx:xx] SENDING..." may still be shown in the Communication dialog
	display, even though you are not sending any messages.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in version 3.2 of Microsoft Mail
	Remote for Windows. We are researching this problem and will post new
	information here in the Knowledge Base as it becomes available.
