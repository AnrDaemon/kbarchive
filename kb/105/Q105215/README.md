## Q105215: SMTP: Inbox Date Does Not Match Message Body Date

{% raw %}

	Article: Q105215
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:2.1,3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to SMTP, versions 2.1, 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft Mail Gateway to SMTP sometimes produces two dates for messages.
	
	The SMTP Gateway generates a date field for incoming messages presented in the
	inbox. Once the message is opened, another date field (generated by the sender's
	message system when the message was sent) appears among the header fields.
	
	Additional query words: 2.10 3.00 date time
