## Q105929: MS-DOS 6.2 Setup Truncates Long Lines in SYSTEM.INI

{% raw %}

	Article: Q105929
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If your Windows SYSTEM.INI file contains very long lines and you install the
	Windows-based version of Microsoft Backup, Microsoft Anti-Virus, or Microsoft
	Undelete, the long lines in your SYSTEM.INI file are truncated to 158
	characters.
	
	Additional query words: msbackup 6.22 6.20
