## Q105276: Differences Between LAN Manager and WFWG WinPopup

	Article: Q105276
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes the differences between Microsoft LAN Manager versions
	2.1 and 2.2 WinPopup and Windows for Workgroups version 3.11 WinPopup.
	
	MORE INFORMATION
	================
	
	LAN Manager 2.1 and 2.2
	-----------------------
	
	LAN Manager WinPopup can store five messages at a time. When more than five
	messages are in the queue, the oldest message is stored in a MESSAGES.LOG file.
	
	Windows for Workgroups 3.11
	---------------------------
	
	Windows for Workgroups WinPopup can store up to 15 messages and does not use a
	message log. Windows for Workgroups WinPopup also has scrolling ability to allow
	you to view previous messages or jump ahead in the message list.
