## Q105632: Scenes 1.0: Will Not Activate With MS-DOS Prompt

	Article: Q105632
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Scenes (all collections), version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft Scenes does not activate when the MS-DOS Prompt or another MS-DOS
	based application is the active window. The screen saver does not activate
	automatically, nor does it activate when you press the hot key combination
	CTRL+SHIFT+S.
	
	Scenes does activate if the MS-DOS Prompt or another MS-DOS-based application is
	minimized (even if it is the active icon).
	
	Additional query words: msscenes screen saver screensaver
