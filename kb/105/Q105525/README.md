## Q105525: FFAPI: Cannot Mail to Group Members on External Post Offices

	Article: Q105525
	Product(s): Microsoft Mail For PC Networks
	Version(s): 2.1,3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Software Development Kit (FFAPI), versions 2.1, 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Mail File Format API (FFAPI) program cannot deliver mail to group
	members on external postoffices; group members receive only mail sent to the
	FFAPI Gateway postoffice. These FFAPI programs are affected:
	
	GTWPUT.EXE
	LGTWPUT.EXE
	OGTWPUT.EXE
	APPPUT.EXE
	LAPPPUT.EXE
	OAPPPUT.EXE
	RMTPUT.EXE
	LRMTPUT.EXE
	
	Additional query words: 2.10 2.1 3.00 3.0 FFAPI groups group.glb
