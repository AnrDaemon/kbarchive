## Q114414: Writer: ErrMsg: Invalid Print Data. Aborting Print Job

	Article: Q114414
	Product(s): Microsoft Home Kids Products
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Creative Writer for Windows, versions 1.0, 1.1, 1.1a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When printing to a Hewlett-Packard (HP) DeskJet 520 or 560 printer, you may
	receive the following error message:
	
	  Invalid Data. Aborting Print Job.
	
	MORE INFORMATION
	================
	
	This error occurs when no paper selection is listed in the HP DeskJet 4.0
	Windows driver. Often the HP driver installs with no paper setting.
	
	To correct this problem, follow these steps:
	
	1. Run Control Panel from the Main group and choose the Printers option.
	
	2. Choose Setup.
	
	3. Set the paper size to the desired setting.
	
	4. Return to Creative Writer.
	
	The Creative Writer document should now print properly.
