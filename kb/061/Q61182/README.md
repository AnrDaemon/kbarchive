## Q61182: Mac Wkst: Possible Cause of &quot;Looking for Server&quot; Message

	Article: Q61182
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:2.0,2.0a,2.0b,3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 24-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The message "Looking for the Microsoft Mail Server ... Give Up" may appear when
	SuperClock version 3.8 is being used on the same machine as a Microsoft Mail
	workstation. The user may also notice slow access to his or her mailbox. The
	only solution is to remove SuperClock from the System Folder.
	
	SuperClock version 3.8 has also been known to affect other network applications,
	such as TOPS and AppleShare.
	
	Additional query words: 2.00 2.00a 2.00b 3.00
