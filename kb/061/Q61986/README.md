## Q61986: Sample Communications Server Program Using TRANSFER_MS_DATA

	Article: Q61986
	Product(s): Microsoft LAN Manager
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbfile kbnetwork
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- DCA/Microsoft Communications Server, version 1.0 
	- DCA/Microsoft Communications Workstation, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	A sample program named NMVT that demonstrates the use of the TRANSFER_MS_DATA
	common service verb (CSV) API function is available. The TRANSFER_MS_DATA verb
	is available with Communications Server and Communications Workstation version
	1.00. It supports the ability to send network management data to an IBM host
	that supports the NetView application.
	
	You can find NMVT.EXE,
	(http://support.microsoft.com/download/support/mslfiles/NMVT.EXE,) a
	self-extracting file, on the following services:
	
	- Microsoft's World Wide Web Site on the Internet
	  On the www.microsoft.com home page, click the Support icon.
	  Click Knowledge Base, and select the product.
	  Enter kbfile NMVT.EXE, and click GO!
	  Open the article, and click the button to download the file.
	
	- Internet (anonymous FTP)
	  ftp ftp.microsoft.com
	  Change to the Softlib/Mslfiles folder.
	  Get NMVT.EXE
	
	- Microsoft Download Service (MSDL)
	  Dial (425) 936-6735 to connect to MSDL
	  Download NMVT.EXE
	
	
	For additional information about downloading, please see the following article in
	the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
