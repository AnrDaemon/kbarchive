## Q61707: Epson EPL-6000 Laser Prints Incorrect Font

	Article: Q61707
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:2.x,3.0,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 2.03, 2.1, 2.11, 3.0, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you use downloadable fonts or font cartridges with an Epson EPL-6000 Laser
	Printer, built-in fonts -- not downloaded fonts or fonts from the cartridge --
	print on the documents.
	
	CAUSE
	=====
	
	This is caused by a problem in the printer ROM.
	
	RESOLUTION
	==========
	
	Obtain ROM version 9200 or later to correct the problem.
	
	MORE INFORMATION
	================
	
	For an updated ROM, contact your local Epson dealer.
	
	To find out what version of ROM your printer has, run the self-test from the
	printer's front panel. The ROM version appears on the top of the self-test
	printout.

	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
