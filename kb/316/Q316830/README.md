## Q316830: Offline E-mail Features in MSN Explorer--For MSN Internet Access

{% raw %}

	Article: Q316830
	Product(s): The Microsoft Network
	Version(s): 6.0,6.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-JAN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network Version 7.0 
	- The Microsoft Network versions 6.0, 6.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes the offline e-mail features that are available for MSN
	Internet Access subscribers only. It describes e-mail features that can be used
	offline and how to use these features.
	
	MORE INFORMATION
	================
	
	Mail Features
	-------------
	
	While offline, you can access any e-mail that you have read already in any mail
	folder(s). You must be online to access any unread e-mail messages in any
	folder(s), or to read new e-mail messages. You can compose e-mails and save them
	as draft messages, but you will need to be online to actually send out any
	e-mails you compose.
	
	Address Book 
	------------
	
	You can access your Address Book offline, but to add or edit any names in your
	Address Book, you must be online.
	
	To get to Offline Mode in MSN Explorer
	
	1. Start MSN Explorer and click your user tile.
	
	2. Select the Read mail offline box, and then click Sign In.
	
	3. You will be signed on in offline mode.
	  Note: The e-mail and Address Book that downloaded the last time you were
	  signed will be available to you in offline mode.
	
	To go Online from Offline mode
	- Click the MSN butterfly on the Home page and select Go Online.
