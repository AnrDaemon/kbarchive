## Q316268: INFO: Product Key For Products from Subscriber Downloads Site

	Article: Q316268
	Product(s): Microsoft Developer Network
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Developer Network Subscriptions 
	- Microsoft Developer Network (MSDN) 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The MSDN Subscriber Downloads sites provide the Product Keys for products that
	are downloaded from those sites.
	
	If you cannot find a Product Key that you have already used, or if you have lost
	your record of the Product Key, you can recover it by browsing back to the
	Downloads site where you first obtained the product.
	
	MORE INFORMATION
	================
	
	If you cannot browse to the original site, contact the MSDN Fulfillment Center
	by telephone at 1-800-759-5474, or send an e-mail message to msdn@microsoft.com
	and request your replacement Product Key.

	  http://msdn.microsoft.com/subscriptions/resources/support.asp
