## Q84467: Err Msg: Badly Formed Setup Script Command Installing WRK

{% raw %}

	Article: Q84467
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 02-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following error may occur when installing the Microsoft Windows Resource Kit
	(WRK) under Microsoft Windows operating system version 3.1:
	
	  Badly Formed Setup Script Command;
	  Add Section Files to Copy List Files Size, Files A:\
	
	If this problem occurs, remove the PATH statement and reinstall the Windows
	Resource Kit. To remove the PATH statement from memory:
	
	1. Exit Windows if it is running.
	
	2. From the MS-DOS command prompt, issue the following command to remove the
	  PATH environmental variable from memory:
	
	  set path=
	
	The Microsoft Windows Resource Kit is available for $19.95 (plus applicable sales
	tax) and $2.50 shipping and handling. To order, call (800) 642-7676.
