## Q84156: Novell Printer Dialog Not Fully Repainted on EGA

{% raw %}

	Article: Q84156
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	When you Windows version 3.1 with Novell NetWare on an EGA display, some of
	the network specific dialog boxes are not repainted properly.
	
	There is no solution to this problem.
	
	Additional query words: 3.10 3.11 Novell network
