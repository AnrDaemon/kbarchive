## Q84047: Text of DEC Pathworks SETHOST.EXE Is Invisible

	Article: Q84047
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you use the Digital Equipment Corporation (DEC) Pathworks SETHOST.EXE
	MS-DOS utility with Microsoft Windows operating system version 3.1, the text
	will not be visible on the screen when the application is run in full- screen
	mode; however, when run in a window it is visible. The application can be placed
	into a window by pressing ALT+ENTER.
	
	WORKAROUND
	==========
	
	To work around this problem:
	
	1. Run Windows Maintenance Setup icon in the Main group.
	
	2. Choose Options and Change System Settings.
	
	3. Change Display to VGA (Version 3.0).

	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
