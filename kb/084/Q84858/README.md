## Q84858: Masque Blackjack for Windows Screen Redraw Problem

	Article: Q84858
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run version 1.0 of Masque Blackjack for Windows in Microsoft Windows
	version 3.1, some of the cards are not drawn on the screen properly or displayed
	as black boxes.
	
	CAUSE
	=====
	
	This problem occurs when MPCARDS.DLL is damaged.
	
	RESOLUTION
	==========
	
	To correct this problem:
	
	1. Delete the MPCARDS.DLL from the WINDOWS\SYSTEM directory.
	
	2. Reinstall Masque Blackjack.
