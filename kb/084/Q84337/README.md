## Q84337: PageMaker Version 4.0 F6, F7, F8 Keys Do Not Function

	Article: Q84337
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When running Aldus PageMaker version 4.0 under Microsoft Windows operating
	system version 3.1, the F6, F7, and F8 function keys do not work as they did
	under Windows 3.0.
	
	WORKAROUND
	==========
	
	To work around this problem you may need to use the following equivalent
	combination keys:
	
	  Function Key    Combination Keys        Function
	  ------------    -----------------       --------
	
	  F6              CTRL+SHIFT+B            Bold
	  F7              CTRL+SHIFT+I            Italic
	  F8              CTRL+SHIFT+U            Underline
	
	For example, to type in bold, press CTRL+SHIFT+B.
	
	MORE INFORMATION
	================
	
	The F6, F7, and F8 keys are internal accelerator keys for PageMaker version 4.0.
	They allow you to press a function key and then start typing in bold, italic, or
	underline.
