## Q84967: Buffalo Print Sharing Box Requires Fast Print Disabled

	Article: Q84967
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	According to Buffalo, Inc., makers of printer peripherals, the Buffalo, Inc.,
	print sharing boxes work with Microsoft Windows version 3.1 if Fast Printing
	Direct To Port is disabled.
	
	For more information about print sharing boxes or devices, query on the following
	words in the Microsoft Knowledge Base:
	
	  "windows" (without the quotation marks) and "3.1" (without the quotation
	  marks) and "tshoot" (without the quotation marks) and "print" (without the
	  quotation marks) and "sharing" (without the quotation marks)
	
	MORE INFORMATION
	================
	
	Steps to Disable Fast Printing
	------------------------------
	
	1. Run Control Panel and choose the Printers icon.
	
	2. Choose the Connect button.
	
	3. Clear the Fast Printing Direct To Port box.
	
	The Buffalo, Inc., product included here is manufactured by a vendor independent
	of Microsoft; we make no warranty, implied or otherwise, regarding this
	product's performance or reliability.
