## Q84708: Icon Background Colors in File Manager Don't Update

{% raw %}

	Article: Q84708
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Windows 3.0 File Manager will not update background colors on its document
	icons if changes are made to the color settings through the Control Panel while
	File Manager is running. The icon background colors will update automatically if
	File Manager is closed and reopened.
	
	This problem does not occur in laver versions of Microsoft Windows File Manager.
	
	MORE INFORMATION
	================
	
	This occurs whether the File Manager is running in a window or running
	minimized. Document icons for subdirectories and files will show the background
	color previously assigned to their Window until File Manager is closed and
	reopened.
