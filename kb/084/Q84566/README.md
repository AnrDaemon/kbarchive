## Q84566: Panasonic KX-P1624 Has Top and Bottom Margin Shift

	Article: Q84566
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you print to a Panasonic KX-P1624 printer with Microsoft Windows version
	3.1 using driver version 1.0, the top and bottom margins may roll. This causes
	the top margin to increase and the bottom margin to decrease when you print
	multiple pages.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Windows version 3.1 printer
	driver. We are researching this problem and will post new information here as it
	becomes available.
	
	Additional query words: 3.10 3.11 KX-P 1624
