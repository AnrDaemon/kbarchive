## Q84921: FTP VxD Causes Network Communications Problems

	Article: Q84921
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	According to FTP Software Inc.'s technical support, an earlier version of FTP
	Software's virtual device driver (VxD), VPCTCP.386, may cause problems with some
	network communication packages running under Microsoft Windows version 3.1.
	
	MORE INFORMATION
	================
	
	If you are using Pericom's Team Talk 340 version 1.8d, you may receive
	application errors and lost or dropped connections if an earlier version of this
	file is being used. The version of VPCTCP.386 dated 5/1/92 that corrects this
	problem is available from FTP Software.
	
	FTP support has stated that the VPCTCP.386 dated 5/1/92 may cause problems with
	Visionware's X-Vision 4.0, and that Visionware is aware of this.
