## Q84572: Red Vertical Lines Through Icons Caused by Sound Drivers

	Article: Q84572
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 02-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Some versions of the Pro AudioSpectrum drivers can cause red vertical lines
	through icons and program control buttons under Microsoft Windows version 3.1.
	
	If red vertical lines are visible in your icons and you are using a Pro
	AudioSpectrum 16 sound card, contact Media Vision for updated drivers.
	
	The Pro AudioSpectrum 16 sound card is manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
	
	Additional query words: 3.10 3.11 proaudio spectrum 3rdparty
