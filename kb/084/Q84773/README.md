## Q84773: Default Program and Group Icons Added During Windows 3.1 Setup

	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following is a list of the default program and group icons added to Program
	Manager during installation of Microsoft Windows operating system versions 3.0,
	3.0a and 3.1. These groups can also be created by using the following steps:
	
	1. Run Program Manager.
	
	2. From the File menu, choose Run.
	
	3. In the Command Line dialog box, enter "Setup /p" (without the quotation
	  marks).
	
	4. Choose OK.
	
	MORE INFORMATION
	================
	
	The settings for Windows 3.0 and 3.0a are identical and listed first; the
	settings for Windows 3.1 are listed second.
