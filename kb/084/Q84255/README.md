## Q84255: File Manager Reports Incorrect Disk Size on Novell Network

	Article: Q84255
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Microsoft Windows File Manager may report an incorrect value for total disk
	space on a Novell network drive. The values reported by File Manager for total
	disk space may be lower than the actual total disk space. The value for free
	disk space should be correct.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows. We are researching this
	problem and will post new information here as it becomes available.
