## Q84989: Access for Windows 3270 Does Not Support TrueType

{% raw %}

	Article: Q84989
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	According to Access technical support, Access for Windows 3270
	version 3.12 cannot display or print TrueType fonts. TrueType fonts
	are not listed in Access's font dialog box.
	
	
	Additional query words: 3.10 tt ttf true type
