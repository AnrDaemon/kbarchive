## Q84636: Terminal Changes Filename to Have &quot;TRM&quot; Extension

{% raw %}

	Article: Q84636
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 02-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	Microsoft Windows Terminal (TERMINAL.EXE) will only save configuration
	files with the .TRM extension. For example, if you create a configuration
	file and attempt to save the file as COMPSERV.PHO, Terminal will change the
	name to COMPSERV.TRM and will not display a message concerning the change.
	
	This is a limitation of Terminal. Other accessory applications, such
	as Cardfile and Calendar, do not have this limitation.
