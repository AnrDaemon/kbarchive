## Q84518: Cannot Switch from WordPerfect for MS-DOS Version 5.1

{% raw %}

	Article: Q84518
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	You may not be able to switch from WordPerfect for MS-DOS version 5.1 while
	running Windows in standard mode if one of the following is selected as the Text
	Screen Type from the Setup Display menu in WordPerfect:
	
	- Paradise VGA Plus
	
	- 132 x 25 16 Color
	
	- 132 x 43 16 Color
	
	Running in enhanced mode or changing the Text Screen Type to
	
	  80 x 25 IBM VGA Compatible
	
	allows you to switch from WordPerfect version 5.1.
	
	MORE INFORMATION
	================
	
	To change the Text Screen Type option from the Setup Display menu in
	WordPerfect:
	
	1. Choose File, Setup, Display from the WordPerfect version 5.1 menu.
	
	2. Choose option 3, Text Screen Type.
	
	3. Select 80 x 25 IBM VGA Compatible.
	
	This problem does not occur when running Windows 3.1 in enhanced mode.
