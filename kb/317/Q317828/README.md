## Q317828: FS2002: How to Import Aircraft from Combat Flight Simulator

	Article: Q317828
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator 2002 Professional Edition 
	- Microsoft Flight Simulator 2002 
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to import an aircraft from Combat Flight Simulator 2
	to Flight Simulator 2002.
	
	MORE INFORMATION
	================
	
	To import an aircraft and aircraft gauges from Combat Flight Simulator 2 to
	Flight Simulator 2002, follow these steps.
	
	NOTE: CFS2 contains gauges for planes in Flight Simulator 2000 and may overwrite
	gauges for planes in Flight Simulator 2002.
	
	1. Start Windows Explorer.
	
	2. Browse to C:\Program Files\Microsoft Games\CSF2\Aircraft.
	
	3. Click and drag the appropriate aircraft files into C:\Program Files\Microsoft
	  Games\FS2002\Aircraft. When you receive the confirmation message, click Yes.
	
	4. Browse to C:\Program Files\Microsoft Games\CFS2\Gauges.
	
	5. Click and drag the appropriate gauge files to C:\Program Files\Microsoft
	  Games\FS2002\Gauges. When you receive the confirmation message, click Yes.
	
	6. Quit Windows Explorer, and then start Flight Simulator.
