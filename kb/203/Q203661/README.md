## Q203661: Error Message: Windows Has Encountered a Problem Accessing Mmsys

	Article: Q203661
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): kberrmsg kbtool win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you double-click Sounds in Control Panel, you may receive the following
	error message:
	
	  Windows has encountered a problem accessing Mmsys.cpl
	
	CAUSE
	=====
	
	This issue can occur if the Mmsys.cpl file is missing or damaged.
	
	RESOLUTION
	==========
	
	To resolve this issue, extract the Mmsys.cpl file from your original Windows
	media to the Windows\System folder. For additional information about how to do
	so, click the article number below to view the article in the Microsoft
	Knowledge Base:
	
	  Q129605 How to Extract Original Compressed Windows Files
