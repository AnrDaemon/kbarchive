## Q203260: Bookshelf 98: Using Keyboard to Copy Image Copies Entire Article

	Article: Q203260
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 22-JAN-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf 98 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you use the TAB key to select a hot spot or an image from an article in
	Microsoft Bookshelf 98, and then copy the selected item, the entire article is
	pasted when you attempt to paste the item you selected.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Bookshelf 98 for
	Windows. This problem has been corrected in Microsoft Bookshelf 99.
