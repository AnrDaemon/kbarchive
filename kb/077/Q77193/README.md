## Q77193: Xerox 4020 Color Inkjet Printer and Windows

	Article: Q77193
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The Xerox 4020 is not a supported printer under Microsoft Windows 3.0, 3.0a,
	3.1, or 3.11.
	
	MORE INFORMATION
	================
	
	Xerox does not currently supply drivers for this printer. To use this printer,
	applications must provide specific drivers that support the capabilities of this
	printer.
	
	If you want to print with this printer, try the Generic/Text Only printer driver;
	however, the output quality will be limited.
	
	The Xerox 4020 printer is manufactured by Xerox, a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
