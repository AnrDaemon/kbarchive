## Q77878: IBM PC 3270 Emulation Ver. 1.22 Incompatible with MS-DOS 5.0

	Article: Q77878
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.x
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 5.0, 5.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	IBM has confirmed that the IBM PC 3270 emulation program version 1.22 is
	incompatible with both MS-DOS 5.0 and IBM PC-DOS 5.0.
	
	Running the IBM PC 3270 emulator with MS-DOS 5.0 may cause intermittent lock-ups,
	or lowercase letters may automatically change to uppercase letters.
	
	MORE INFORMATION
	================
	
	IBM technical support recommends replacing the IBM PC 3270 emulation with a new
	release called the Entry Level Emulation Program version 2.0. The Entry Level
	Emulation Program is available from IBM and is compatible with both MS-DOS 5.0
	and IBM PC-DOS 5.0.
