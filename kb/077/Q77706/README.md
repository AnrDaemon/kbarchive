## Q77706: Himem /M:12 on Vendex VTI with Award 3.03 BIOS

	Article: Q77706
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	On Vendex VTI 286 systems with an Award 3.03 BIOS and an AST 6 Pack board, it
	may be necessary to load HIMEM.SYS with the /MACHINE:12 parameter.
	
	It may be possible to load with A20 handler number 1 unless SMARTDRV.SYS is
	loaded.
