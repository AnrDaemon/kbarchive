## Q77723: Paintbrush Loses Copied Images in Windows 3.0

	Article: Q77723
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When images are copied three times or more using the SHIFT-drag method in
	Paintbrush and the file is then saved or Paintbrush is minimized, some of the
	images are lost.
	
	STATUS
	======
	
	This problem does not occur in later versions of Windows.
	
	Additional query words: 3.00 3.0 3.0a 3.00a win30 Paint brush
