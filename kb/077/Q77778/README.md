## Q77778: BTRIEVE Versions 5.1 and Later Compatible with MS-DOS 5.x/6.x

	Article: Q77778
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Novell has confirmed that BTRIEVE.EXE versions earlier than 5.1 are not
	compatible with MS-DOS 5.x. 6.x, or 6.2. Incompatible versions of BTRIEVE.EXE
	may cause an MS-DOS 5.0 system to stop responding (hang or lock up) when you try
	to run the utility. Incompatible versions of BTRIEVE.EXE may cause an MS-DOS 6.x
	system to stop responding upon exiting BTRIEVE.EXE.
	
	MORE INFORMATION
	================
	
	The Novell utility BTRIEVE.EXE is a self-contained executable program that
	performs as a record management system. For more information on the Novell
	BTRIEVE utility, query on the following in the Microsoft Knowledge Base or
	contact Novell technical support:
	
	  BTRIEVE and MS-DOS
