## Q77978: Packard Bell 9500 Laser Printer Emulation

{% raw %}

	Article: Q77978
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Currently there are no Microsoft Windows 3.0 printer drivers for the Packard
	Bell 9500 laser printer. There are two emulation modes available:
	
	  Primary Emulation       Secondary Emulation
	  -----------------       -------------------
	
	  HP LaserJet II          IBM Proprinter X24
	
	MORE INFORMATION
	================
	
	For more information, contact Packard Bell.
	
	The Packard Bell product included here is manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
