## Q77783: Modal Dialog Child of Modeless Dialog Box Sample Code

	Article: Q77783
	Product(s): Microsoft Windows Software Development Kit

	The information in this article applies to:
	
	- Microsoft Windows Software Development Kit (SDK) versions 3.0, 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	MdlMdls.exe is a sample in the Microsoft Download Center that demonstrates how
	to create a modal child of a modeless dialog box. The sample uses two modeless
	dialog boxes. However, when the second dialog is created, the first dialog and
	the main application window are disabled.
