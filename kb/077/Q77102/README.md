## Q77102: PC Rmt: COM3 and COM4 Not Available on MCA Machines

	Article: Q77102
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:2.1,3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Remote for MS-DOS, versions 2.1, 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Mail Administrator program and Remote client software do not work with COM3
	or COM4 on IBM computers using Micro Channel Architecture (MCA).
	
	IBM MCA machines use different serial addresses for COM3 and COM4 than Industry
	Standard Architecture (ISA) or Enhanced Industry Standard Architecture (EISA).
