## Q123822: PC Win: First Tab Prints As a Space

{% raw %}

	Article: Q123822
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0,3.0b,3.2,3.2a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, versions 3.0, 3.0b, 3.2, 3.2a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you print from Microsoft Mail for Windows, the first tab character on a
	line is printed as a space.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in versions 3.0, 3.0b, 3.2, and
	3.2a of Microsoft Mail for Windows. We are researching this problem and will
	post new information here in the Microsoft Knowledge Base as it becomes
	available.
