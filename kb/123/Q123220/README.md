## Q123220: Fatal Error on Drive 1 with the Exabyte EXB-8505

{% raw %}

	Article: Q123220
	Product(s): Microsoft Windows NT
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	3.50
	
	WINDOWS
	
	kb3rdparty
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.5 
	- Microsoft Windows NT Server version 3.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the Windows NT Backup program with an Exabyte EXB-8505 tape backup
	device, the backup process may stop and display either one of the following
	error messages:
	
	  Fatal error on drive 1.
	  Did not respond in timeout period.
	
	  -OR-
	
	  Fatal error on drive 1. The device detected a controller error on \\tape0.
	
	CAUSE
	=====
	
	This problem occurs with earlier versions of the Exabyte EXB-8505 firmware.
	
	RESOLUTION
	==========
	
	To correct this problem, upgrade the firmware on the Exabyte EXB-8505 to
	revision 8SC-0730 or later. For more information, contact Exabyte Technical
	Support at (800) 445-7736.
