## Q123818: Windows NT 3.5 Does Not Support TI MicroLaser Pro 600 Printer

	Article: Q123818
	Product(s): Microsoft Windows NT
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.5 
	- Microsoft Windows NT Server version 3.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Windows NT does not provide a printer driver for a TI MicroLaser Pro 600
	printer. This printer is not supported by Windows NT at this time.
	
	SYMPTOMS
	========
	
	When you print on a TI MicroLaser Pro 600 printer using the TI MicroLaser 600
	printer driver, the first two lines print correctly, but the remainder of the
	print job is unreadable.
	
	
	WORKAROUND
	==========
	
	To work around this problem, use the Apple LaserWriter II series printer driver.
	For additional information, contact Texas Instruments Technical Support at (800)
	848-3927.
	
	
	The MicroLaser Pro 600 printer discussed here is manufactured by Texas
	Instruments, a vendor independent of Microsoft; we make no warranty, implied or
	otherwise, regarding this product's performance or reliability.
