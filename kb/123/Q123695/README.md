## Q123695: Cirrus Logic Card w/ 5402 Chip Set Does Not Work w/ 256 Colors

{% raw %}

	Article: Q123695
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	Cirrus Logic has a video card with a 5402 chip set. This card is no longer
	manufactured or supported by Cirrus Logic, nor are they not providing new
	drivers for it.
	
	The Windows 95 standard VGA driver can be used with this card; however, the
	driver does not work with 256 colors (or more).
	
	Additional query words: 3rdparty cyrus cirus
