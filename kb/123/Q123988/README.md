## Q123988: Cannot Clear the Screen in HyperTerminal

{% raw %}

	Article: Q123988
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	HyperTerminal does not provide the ability to delete characters that are already
	on your screen. Selecting text and pressing the Delete key does not clear the
	screen. Other than exiting Hyperterminal and restarting it, no workaround is
	available.
	
	Additional query words: 4.00 del Hyperterm buffer
