## Q123167: Support for the Compaq Ethernet 16TP Network Adapter

	Article: Q123167
	Product(s): Microsoft Windows NT
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	3.50
	
	WINDOWS
	
	kb3rdparty
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.5 
	- Microsoft Windows NT Server version 3.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you install Windows NT version 3.5 with a COMPAQ Ethernet 16TP network
	adapter, use the Intel EtherExpress 16TP network adapter driver. The COMPAQ
	Software Support Disk version 1.12a does not include a specific network adapter
	driver for the COMPAQ Ethernet 16TP network adapter. For more information, call
	COMPAQ Technical Support at (800) 652-6672.
	
	
	The Compaq products discussed here are manufactured by Compaq Computer Corp., a
	vendor independent of Microsoft; we make no warranty, implied or otherwise,
	regarding the products' performance or reliability.
