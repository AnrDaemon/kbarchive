## Q123339: Stability Problems on NEC Duo MIPS Computers

	Article: Q123339
	Product(s): Microsoft Windows NT
	Version(s): 3.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.5 
	- Microsoft Windows NT Server version 3.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you experience general system stability problems on NEC Duo MIPS computers,
	including STOP messages, you may need to upgrade to a new motherboard, CPU, and
	I/O card firmware version. The current recommended revisions are:
	
	  Firmware      Revision
	  ----------------------
	
	  Motherboard   D7B
	  I/O Card      B5A
	  CPU Card      B5A
	
	Firmware upgrades can be obtained from NEC Technical Support at (800) 325- 5500,
	access code 700.
	
	The products included here are manufactured by vendors independent of Microsoft;
	we make no warranty, implied or otherwise, regarding these products' performance
	or reliability.
