## Q86837: POINTER.EXE 8.1 May Slow Windows Applications

	Article: Q86837
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 29-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Using Microsoft Mouse driver version 8.1 POINTER.EXE may cause some Windows
	applications to slow down or stop when attempting to write to the screen. To
	correct the problem, use POINTER.EXE supplied with Microsoft Mouse driver
	version 8.20a.
	
	
	Additional query words: SimCity Procomm 8.10 8.20 3rdparty
