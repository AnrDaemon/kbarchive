## Q86936: 3COM: Main Software Components in a 3Com Gateway

	Article: Q86936
	Product(s): Microsoft Mail For PC Networks
	Version(s): 3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to 3Com 3+Mail, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Mail Gateway to 3Com 3+Mail consists of one executable program,
	GATEWAY.EXE. Apart from the necessary network drivers, no other software needs
	to run on the gateway PC. The PC must be logged into the 3+Mail service.
	
	REFERENCES
	==========
	
	"Microsoft Mail Gateway to 3Com 3+Mail Administrator's Guide"
