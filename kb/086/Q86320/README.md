## Q86320: PC Win: Message Finders Closing Automatically

	Article: Q86320
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Message Finder is closed automatically when you exit Mail for Windows if the
	search criteria are blank.
	
	Mail works under the assumption that if there is no search criteria, there is no
	reason to bring up a blank Message Finder when Mail for Windows is restarted.
