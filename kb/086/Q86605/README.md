## Q86605: PC WSPlus: Clicking Schedule+ Icon Gives Beep

	Article: Q86605
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If Microsoft Schedule+ for Windows is set to open minimized and the sign-in box
	is ignored or hidden behind other applications, double-clicking the Schedule+
	icon will only give a beep. If this icon does not maximize, you are not logged
	in yet.
	
	RESOLUTION
	==========
	
	Either close all open windows to reveal the sign-in window or switch to Sign In
	in the Windows Task List.
