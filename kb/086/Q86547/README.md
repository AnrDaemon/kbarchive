## Q86547: MHS: Software Required to Install the Gateway Postoffice

	Article: Q86547
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to MHS, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Mail Gateway to MHS package contains two floppy disks, which are
	labeled as follows:
	
	- The Microsoft Mail Gateway to MHS (the gateway component)
	
	- The Microsoft Mail Gateway Access to MHS (the gateway access component)
