## Q86843: PC WSPlus: Appending to Existing Archive File

	Article: Q86843
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	Version 1.0 of Microsoft Schedule+ for Windows lets you archive old notes
	and appointments to a file. This reduces the size of the calendar file both
	locally and on the server. To append later archives onto the existing file,
	choose Create Archive from the File menu and save the new archive with the
	same name as the existing file and in the same directory.
