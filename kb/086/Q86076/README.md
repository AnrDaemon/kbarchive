## Q86076: Shortcut Keys Not Available in EGA Setup

{% raw %}

	Article: Q86076
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When installing Microsoft Windows version 3.1 on a computer with an EGA video
	card, the shortcut keys available in the Exit Windows dialog box to restart your
	computer or return you to MS-DOS are not available. The shortcut keys are
	available on VGA and higher resolutions.
	
	WORKAROUND
	==========
	
	Use the TAB key to switch between the two options, instead of holding down the
	ALT key while pressing the hot key.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem. We are researching this problem
	and will post new information here as it becomes available.
