## Q86997: Drivers Available for COMPAQ Q-Vision Video

	Article: Q86997
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 3.1,3.11
	Operating System(s): 
	Keyword(s): win31
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	COMPAQ has released video drivers for using its Q-Vision video standard with
	Microsoft Windows version 3.1.
	
	The following resolutions are available:
	
	- 1024 x 768 with 256/16 colors
	
	- 800 x 600 with 256/16 colors
	
	- 640 x 480 with 256/16 colors
	
	NOTE: Some newer COMPAQ models have Q-Vision video on the motherboard that may
	only have 512K video RAM and may not be capable of using all the
	resolution/color options of these drivers.
	
	MORE INFORMATION
	================
	
	These drivers are available on the COMPAQ User Programs disk version 7.28 or
	later, and on CompuServe in the COMPAQ forum as SPO298.ZIP.
	
	Steps to Download SPO298.ZIP
	----------------------------
	
	1. Log on to CompuServe and type "go compaq" (without the quotation marks) to go
	  to the COMPAQ software library.
	
	2. Choose 4 from the menu to download a file. (Check your CompuServe
	  documentation for more details on downloading a file.)
	
	3. Type "SPO298.ZIP" (without the quotation marks) for the filename to be
	  downloaded, and select the appropriate transfer protocol from the Access
	  Protocol menu.
	
	4. After downloading the desired file, you will need to use the unarchiving
	  utility PKUNZIP.EXE.
	
	If you do not have PKUNZIP.EXE, you can download it as follows:
	
	1. Type "go ibmnew" (without the quotation marks) to go to the IBM New Users
	  Forum.
	
	2. Type "down" (without the quotation marks) to request that a file be
	  downloaded.
	
	3. Type "pkz110.exe" (without the quotation marks) to request that the program
	  file be downloaded.
	
	4. Select the appropriate transfer protocol.
	
	The COMPAQ and PKUNZIP.EXE products included here are manufactured by vendors
	independent of Microsoft; we make no warranty, implied or otherwise, regarding
	these products' performance or reliability.
