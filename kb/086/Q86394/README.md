## Q86394: PC OS2: MCV Date Format Incorrectly Set in US-Style

	Article: Q86394
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0,3.0b,3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, versions 3.0, 3.0b, 3.2, on platform(s):
	   - the operating system: OS/2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Microsoft Mail for Presentation Manager, version 3.0, 3.0b, or 3.2, the date
	column in the Message Center Viewer (MCV) is always in United States format
	(that is, MM/DD/YY), even if the CONFIG.SYS country setting is for another
	country.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Mail for Presentation
	Manager versions 3.0, 3.0b, and 3.2. We are researching this problem and will
	post new information here in the Microsoft Knowledge Base as it becomes
	available.
