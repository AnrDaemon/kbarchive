## Q86606: PC WSPlus: Projects Sorted Alphabetically Only

	Article: Q86606
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Projects in the Task view of Microsoft Schedule+ for Windows, version 1.0, are
	sorted alphabetically by project name. Currently, there is no way to change this
	sort order to priority or due dates.
	
	STATUS
	======
	
	This feature is under review and will be considered for inclusion in a future
	release.
