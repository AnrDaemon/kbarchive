## Q86571: Earlier Hitachi CD-ROMs May Not Work with Media Player

	Article: Q86571
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	Earlier Hitachi CD-ROM drives, such as the Amdek Laserdrive I, may not
	be able to play audio CDs through the Media Player included with Windows
	version 3.1.
	
	Driver version 2.2 is required to play audio CDs through the Windows
	3.1 Media Player. HITACHIA.SYS is required for non-MCA machines, and
	HITACHIB.SYS is required for MCA machines.
	
	Additional query words: 3.10 CD media player older
