## Q86177: Citizen GSX-130 Tractor Paper Misfeeds

	Article: Q86177
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When a document is printed to a Citizen GSX-130 from Microsoft Windows 3.1, if
	the top margin is set to less than .5 inch, the printer will misadvance the
	tractor paper.
	
	CAUSE
	=====
	
	According to Citizen Technical Support, this is a limitation of the printer. The
	printer does not support a top margin less than .5 inch.
