## Q86442: Remote Access Service Problems Connecting Mouse Cable

	Article: Q86442
	Product(s): Microsoft LAN Manager
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SYMPTOMS
	========
	
	When using a 25-pin/9-pin cable with a 9-pin/25-pin coverter that comes with the
	mice packs, the following problem occurs: the modem answers and the connections
	suddenly are disconnected after the protocol was negotiated but before the
	authentication dialog box.
	
	CAUSE
	=====
	
	Users should not use cables or adapter converters provided with mice packs.
	
	Additional query words: 2.10 mouse paks RAS
