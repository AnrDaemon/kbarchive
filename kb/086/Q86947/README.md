## Q86947: 3COM: What Occurs in the Event of a Mail Failure

	Article: Q86947
	Product(s): Microsoft Mail For PC Networks
	Version(s): 3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to 3Com 3+Mail, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If mail fails to be delivered to a 3Com 3+Mail user from a Microsoft Mail user,
	a non-delivery notice is generated and sent back to the originator. The same
	applies to mail being sent from 3+Mail to Microsoft Mail.
	
	REFERENCES
	==========
	
	"Microsoft Mail Gateway to 3Com 3+ Mail Administrator's Guide"
