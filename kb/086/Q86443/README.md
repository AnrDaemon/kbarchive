## Q86443: LANtastic for Windows Updated LANTNET.DRV Available

	Article: Q86443
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run Windows version 3.1 on a LANtastic for Windows network, LPT1 will
	be listed as a local printer even though the port is being redirected across the
	network.
	
	RESOLUTION
	==========
	
	To correct this and other problems, Artisoft has released updated LANTNET.DRV
	version 4.1 and Windows Net Manager version 4.1.
	
	MORE INFORMATION
	================
	
	LANtastic versions 3.x and 4.x are supported by Windows 3.1. LANtastic for
	Windows is a separate product and has not been tested by Microsoft.
	
	The LANtastic products included here are manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding these products'
	performance or reliability.
