## Q86004: PC Win: Setup Does Not Remove 2.1 Windows Client

{% raw %}

	Article: Q86004
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Setup program for version 3.0 of Microsoft Mail for Windows does not remove
	the Microsoft Mail version 2.1 Windows Client.
	
	RESOLUTION
	==========
	
	To remove the Mail version 2.1 Windows client, delete the following files from
	the local Windows directory:
	
	  WINMAIL.EXE
	  WINMAIL.NLS
	  WINMON.BIN
	  MAILFONT.FON
