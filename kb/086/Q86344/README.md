## Q86344: DELL PVGA600.DRV Video Driver Causes Corrupted Printing

	Article: Q86344
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	Dell technical support has confirmed that using Microsoft Windows version 3.1
	with DELL's 800 x 600 256-color display driver PVGA600.DRV may cause corrupted
	printing.
	
	MORE INFORMATION
	================
	
	PVGA600.DRV is a Western Digital/Paradise display driver that was released with
	some earlier Dell desktop computers.
	
	Workarounds
	-----------
	
	1. Replace the driver with the Super VGA (800 x 600, 16 colors) driver that
	  comes with Windows 3.1.
	
	2. Obtain the updated PVGAA04.DRV driver by contacting Dell technical support or
	  download PVGAA04.ZIP from the Dell bulletin board.

	performance or reliability.
