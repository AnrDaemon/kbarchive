## Q86188: MR BIOS Version 1.2 or Later Needed for Windows 3.1

	Article: Q86188
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	According to Microid Research, MR BIOS version 1.2 or later should be used with
	Microsoft Windows version 3.1.
	
	MORE INFORMATION
	================
	
	Steps to Find the BIOS Version on a Machine with MR BIOS
	--------------------------------------------------------
	
	1. After the system starts, press CTRL+ALT+ESC.
	
	2. In the upper-right corner of the CMOS screen, the BIOS version will be listed
	  to the right of VER:.
	
	3. Press F10 to exit the CMOS screen.
	
	The MR BIOS product included here is manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
