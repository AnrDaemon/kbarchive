## Q86323: PC Win: How to Move Folder to Top Level

{% raw %}

	Article: Q86323
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In version 3.0 of Microsoft Mail for Windows, there are two ways to move or copy
	a folder to the top level.
	
	1. Select the folder, then choose Move or Copy from the File menu. When the
	  dialog box comes up, hold down the HOME key while clicking OK.
	
	2. Select the folder, then choose Folder Properties from the File menu. Select
	  Top Level and click OK.
