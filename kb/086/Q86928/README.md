## Q86928: 3COM: The Role of the 3Com Gateway

	Article: Q86928
	Product(s): Microsoft Mail For PC Networks
	Version(s): 3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to 3Com 3+Mail, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Mail Gateway to 3Com allows 3+Mail users to exchange electronic
	mail and files with Microsoft Mail users.
	
	The gateway runs on a dedicated workstation that has access to the Microsoft Mail
	data files and is also logged into the 3+Mail service as an administrative
	user.
	
	For more information on how the gateway works, see the "Microsoft Mail Gateway to
	3+Mail Administrator's Guide."
