## Q86549: MHS: Steps That Must Be Completed Prior to Installation

{% raw %}

	Article: Q86549
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to MHS, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Before installing the Microsoft Mail Gateway Access to MHS, MHS should already
	be installed and functioning, a gateway name should be decided on, and the
	gateway's usage should be formulated.
	
	Before you install the gateway, please refer to the "Microsoft Mail
	Administrator's Guide" for a good understanding of the product and its
	functionality.
	
	Before installing MHS and the MS Mail gateway, please review the compatibility
	issues among different gateways. There are many differences among the versions
	of MHS available, and only some may fit the needs for a given environment.
