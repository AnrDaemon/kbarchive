## Q86876: TopDesk Window May Not Retain Its Size or Position

	Article: Q86876
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The TopDesk utility, which is included with the Windows Resource Kit (WRK) for
	Windows 3.1, will not retain resized or repositioned window information unless
	you close TopDesk before you exit Windows.
	
	WORKAROUND
	==========
	
	To work around this problem:
	
	1. Resize and/or reposition TopDesk as desired.
	
	2. From the Options menu, choose Preferences.
	
	3. Choose the OK button.
	
	Your changes will be saved, and you won't have to quit and restart TopDesk for
	the window information to be saved.
