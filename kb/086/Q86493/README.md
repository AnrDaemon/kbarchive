## Q86493: Printing with the Xerox 3700 and Windows 3.1

{% raw %}

	Article: Q86493
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The Xerox 3700 printer is not supported under Microsoft Windows version 3.1.
	
	To print from Windows using the Xerox 3700 printer, Xerox Technical Support
	suggests emulating the Xerox 4045 printer.
	
	MORE INFORMATION
	================
	
	The Xerox 4045 is also an unsupported printer; however, you can obtain a printer
	driver for the 4045 by calling Xerox Technical Support.
	
	To obtain phone numbers for Xerox, query on the following words in the Microsoft
	Knowledge Base:
	
	  "Xerox" (without the quotation marks) and "phonebook" (without the quotation
	  marks)
	
	
	The Xerox 3700 printer is manufactured by a vendor independent of Microsoft; we
	make no warranty, implied or otherwise, regarding this product's performance or
	reliability.
