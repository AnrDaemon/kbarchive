## Q86348: Hyundai 386/D CAPS LOCK May Not Work in Windows

	Article: Q86348
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you run Microsoft Windows version 3.1 on a Hyundai 386/D system, you must
	press the CAPS LOCK key twice before the function takes effect. This behavior
	occurs when you run Windows 3.1 in both standard and enhanced modes.
	
	STATUS
	======
	
	Hyundai Corporation is aware of this problem with the 386/D computer. They are
	researching this problem and will provide more information as it becomes
	available.
	
	MORE INFORMATION
	================
	
	For more information please contact Hyundai technical support.
	
	
	The Hyundai product included here is manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
