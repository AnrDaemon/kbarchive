## Q86832: Orchid Video Driver May Cause Incorrect Output on HP LaserJet

{% raw %}

	Article: Q86832
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	Orchid has corrected these problems and provides an upgraded BIOS and driver.
	The new BIOS is version 3.0 and the corresponding driver is 7.0
	
	To obtain the phone numbers for Orchid, query on the following words in the
	Microsoft Knowledge Base:
	
	  "Orchid" (without the quotation marks) and "phonebook" (without the quotation
	  marks) and "pss" (without the quotation marks)
	
	SYMPTOMS
	========
	
	Version 2.1 of the Orchid Fahrenheit 1280 video driver for Microsoft Windows
	version 3.1 may cause incorrect output fine lines are printed from any
	application using the Hewlett-Packard (HP) LaserJet II, IIP, and LaserJet III
	printer driver.
	
	When fine lines (less than 1/4 point) are printed from any application, lines may
	be randomly printed across the page. Lines thicker than 1/4 point print
	properly.
	
	STATUS
	======
	
	Orchid has confirmed this to be a problem with its current Windows 3.1 video
	driver (version 2.1) and plans to update this driver. For more information on
	this update, contact Orchid Technology.
	
	MORE INFORMATION
	================
	
	
	For more information on this problem, query on the following words in the
	Microsoft Knowledge Base:
	
	  "Fahrenheit" (without the quotation marks) and "1280" (without the quotation
	  marks) and "interlaced" (without the quotation marks) and "VGA" (without the
	  quotation marks)
	
	
	The Fahrenheit 1280 video driver is manufactured by Orchid Technology, a vendor
	independent of Microsoft; we make no warranty, implied or otherwise, regarding
	this product's performance or reliability.
	
	
	Additional query words: 3.10 3.1 technologies WIN31 3.11 farenheit
