## Q86548: MHS: Software &amp; Hardware Required for Downstream Postoffice

	Article: Q86548
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to MHS, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The downstream postoffices require only the Microsoft Mail Gateway Access to MHS
	package, which includes one floppy disk labeled The Microsoft Mail Gateway
	Access to MHS (the gateway access component).
	
	
	Additional query words: pcmail
