## Q86231: Creating a Page Break in Windows Write

	Article: Q86231
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	To create a page break in the Microsoft Windows Write:
	
	1. Place the cursor at the beginning of the line of text just below the desired
	  insertion point for the page break.
	
	2. Press CTRL+ENTER. The page break will appear above this line of text.
	
	To delete the page break:
	
	1. Place the cursor at the beginning of the page break.
	
	2. Press the DEL key. The page break will disappear.
	
	Additional query words: 3.00 3.00a 3.10 3.11 pagebreak
