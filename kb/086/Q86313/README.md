## Q86313: PC OS2: PM Client About Box Reads &quot;Mail for Windows&quot;

	Article: Q86313
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0,3.0b,3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, versions 3.0, 3.0b, 3.2, on platform(s):
	   - the operating system: OS/2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The startup logo and About Box read "Microsoft Mail for Windows" instead of
	"Microsoft Mail for Presentation Manager."
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Mail for Presentation
	Manager versions 3.0, 3.0b, and 3.2. We are researching this problem and will
	post new information here in the Microsoft Knowledge Base as it becomes
	available.
	
	Additional query words: 3.00 3.00b 3.20
