## Q86946: 3COM: Valid Options for the GATEWAY.INI File

{% raw %}

	Article: Q86946
	Product(s): Microsoft Mail For PC Networks
	Version(s): 3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 20-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to 3Com 3+Mail, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The GATEWAY.INI file contains all the configuration options for the Microsoft
	Mail Gateway to 3Com 3+Mail. A complete description of all the options is in the
	"Microsoft Mail Gateway to 3Com 3+Mail Administrator's Guide."
