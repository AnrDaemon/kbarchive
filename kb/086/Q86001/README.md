## Q86001: PC Win: Horizontal Scroll Bar Blocks Bottom Line

	Article: Q86001
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When the horizontal scroll bar is displayed in version 3.0 of Microsoft Mail for
	Windows and you open a note, the bottom line of the message cannot be viewed.
	
	RESOLUTION
	==========
	
	To remove the scroll bars from the Mail desktop, all windows and icons need to
	be viewable in their entirety. To do this, choose Arrange Icons from the Window
	menu of Mail for Windows.
