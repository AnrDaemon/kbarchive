## Q86172: Hercules VGA May Cause Yellow Screen in Windows

	Article: Q86172
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Hercules Graphics Card GB500 may display a yellow screen when you run
	Microsoft Windows.
	
	RESOLUTION
	==========
	
	Obtaining a replacement card from Hercules will solve this problem.
	
	MORE INFORMATION
	================
	
	According to Hercules technical support, cards with "V9" as the first two digits
	of the 6- to 10-digit serial number (for example, "V9_ _ _ _"), have this
	problem. It is related to the date the card was manufactured.
	
	The Hercules products included here are manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding these products'
	performance or reliability.
