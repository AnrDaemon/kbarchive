## Q86656: TrueType Symbol Fonts Display in Italics

	Article: Q86656
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 29-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The TrueType Symbol font displays certain point sizes in italics on the screen,
	but prints correctly.
	
	CAUSE
	=====
	
	The italicized text will vary with different video drivers, and only lowercase
	characters are affected.
	
	The following Microsoft drivers have been tested, and display italicized in the
	following point sizes:
	
	  Driver                 Point Size
	  ------                 ----------
	
	  EGA                    12, 14, 18, 24
	  VGA (Version 3.0)      8, 10, 12, 18, 24
	  VGA                    8, 10, 12, 18, 24
	  Super VGA (800 x 600)  8, 10, 12, 18, 24
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows version 3.1. We are
	researching this problem and will post new information here as it becomes
	available.
