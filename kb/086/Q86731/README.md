## Q86731: Tandy LP-990 Prints Solid Black When Using TrueType

{% raw %}

	Article: Q86731
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	The Tandy LP-990 laser printer prints a solid black sheet of paper if TrueType
	fonts are used.
	
	WORKAROUND
	==========
	
	To work around this problem, use resident printer fonts or screen fonts. All
	users who are having this problem should contact Tandy Technical Support.
	
	MORE INFORMATION
	================
	
	Tandy Corporation/Radio Shack has confirmed this to be a problem with the ROM
	BIOS of the printer. Tandy is currently working on a ROM upgrade for this
	printer and will release it to users who are having this problem.

	or reliability.
