## Q86786: LAYOUTS.WRI Lists Wrong Size for UNIDRV.DLL

	Article: Q86786
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The LAYOUTS.WRI file included with the Microsoft Windows Resource Kit (WRK) for
	Windows 3.1 incorrectly lists the expanded file size for UNIDRV.DLL. The listed
	size is 119008 bytes.
	
	The correct expanded size of this file is 119296 bytes with a date of 3/10/92.
	
	MORE INFORMATION
	================
	
	There are currently two other valid date and size combinations for UNIDRV.DLLL:
	
	- UNIDRV.DLL 3/26/92 119344 bytes
	
	- UNIDRV.DLL 6/29/92 127152 bytes
