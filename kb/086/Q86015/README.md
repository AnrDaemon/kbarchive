## Q86015: Star Gemini 10x Printer and Windows

{% raw %}

	Article: Q86015
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The Star Gemini 10x printer is not supported under Microsoft Windows. Star
	technical support is aware that there are no drivers available within Windows
	for this discontinued model. Star has no plans to produce a Windows driver for
	this printer.
	
	This also applies to the following earlier printer models:
	
	- Star Radix 10x
	
	- Star Delta 10x
	
	Star states that there are no known emulations for these printers.
