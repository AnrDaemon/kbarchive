## Q221758: XADM: Exchange 4.0/5.0 Administrator Help File Is Incorrect

	Article: Q221758
	Product(s): Microsoft Exchange
	Version(s): winnt:4.0,5.0
	Operating System(s): 
	Keyword(s): exc4 exc5
	Last Modified: 18-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 4.0, 5.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	The Help file in the Exchange Server 4.0 and 5.0 Administrator program
	incorrectly explains that the "No. of Contacts" column in the Public Folder
	Resources refers to the number of times folder has been used. It should read:
	
	  The number of users who are designated as folder contacts for this folder.
	  This is included in the default column view.
	
	RESOLUTION
	==========
	
	This error was corrected in the Exchange Server 5.5 Administrator Help file to
	read:
	
	  The number of users who are designated as folder contacts for this folder.
	  This is included in the default column view.
