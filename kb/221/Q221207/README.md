## Q221207: INFO: Getting Visual Basic 4.0 or 5.0 After Purchasing VB 6.0

	Article: Q221207
	Product(s): Microsoft Visual Basic for Windows
	Version(s): WINDOWS:6.0
	Operating System(s): 
	Keyword(s): kbVBp400 kbVBp500 kbVBp600 kbGrpDSVB
	Last Modified: 20-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual Basic Learning Edition for Windows, version 6.0 
	- Microsoft Visual Basic Professional Edition for Windows, version 6.0 
	- Microsoft Visual Basic Enterprise Edition for Windows, version 6.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you require a previous version of Microsoft Visual Basic, you may purchase
	version 6.0 and then downgrade to a prior version.
	
	In order to acquire the correct license:
	
	1. Purchase Microsoft Visual Basic 6.0 from an authorized reseller.
	
	2. If you have access to existing licensed software you may use it, or you can
	  request the software media only for a small fee (around $20US/$25CDN) plus
	  shipping and handling from the Microsoft Supplemental and Replacement Parts
	  Center at 800-360-7561. Outside the United States and Canada, please contact
	  your local Microsoft subsidiary.
	
	For additional information please see:
	
	Visual Basic Downgrade Information
	
	MORE INFORMATION
	================
	
	The following editions are available as a downgrade to Visual Basic 6.0:
	
	- Visual Basic 5.0
	
	   - Enterprise Edition
	   - Professional Edition
	   - Learning Edition
	
	- Visual Basic 4.0
	
	   - Enterprise Edition
	   - Professional Edition
	   - Standard Edition
	
	For more information on your support options for this product, please refer to
	Support Options by Product online at:
	
	  http://support.microsoft.com/default.aspx?scid=fh;EN-US;CNTACTMS"
