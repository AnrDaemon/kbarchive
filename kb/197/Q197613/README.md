## Q197613: Internet Directory: Updates have been discontinued

	Article: Q197613
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocerr
	Last Modified: 01-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Official Microsoft Bookshelf Internet Directory, 1998 Edition ISBN 1-57231-617-9 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Page 1136 contains the "About the Compact Disc" section and is located at the
	end of the book between the license agreement and the registration card. This
	page contains the following text:
	
	  The compact disc will also take you to a Web site to help you stay current
	  with new Internet listings. This resource, updated often, lists addresses and
	  links that are too new to have been included on your CD. To view links to new
	  sites, open the Help menu and then click New Internet Directory Listings
	  Online."
	
	However, there is no New Internet Directory Listings Online option in the Help
	menu.
	
	CAUSE
	=====
	
	Updates for this directory have been discontinued.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Official Microsoft Bookshelf
	Internet Directory.
