## Q197603: Learn Visual Basic Scripting Now Corrections and Comments

	Article: Q197603
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocerr
	Last Modified: 20-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Learn Microsoft Visual Basic Scripting Edition Now ISBN 1-57231-347-1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Learn Microsoft Visual Basic Scripting
	Edition Now, ISBN 1-57231-347-1.
	
	The following topics are covered:
	
	- Page 62: Inaccurate description of Appendix A
	
	- Page 240: Incorrect Page Reference
	
	MORE INFORMATION
	================
	
	Page 62: Inaccurate description of Appendix A
	---------------------------------------------
	
	Page 62:
	Note at the bottom of the page, the last sentence, includes an inaccurate
	description of Appendix A.
	
	Delete the sentence:
	"Appendix A explains how to use Microsoft Visual Basic version 5 to write your
	VBScript code."
	
	
	Page 240: Incorrect Page Reference
	----------------------------------
	
	Page 240, last paragraph:
	Change:
	"Figure 8-1 on page 233"
	To:
	"Figure 8-1 on page 231"
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
