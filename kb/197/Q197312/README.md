## Q197312: WD97: Thesaurus Suggests Incorrect Synonym

{% raw %}

	Article: Q197312
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Microsoft Word 97 thesaurus may suggest an incorrect synonym.
	
	CAUSE
	=====
	
	If the thesaurus can't find the selected word or phrase, it searches through a
	list of words to suggest. This list is an alphabetically ordered list, and it
	has no logical correlation to the word, sentence, or phrase.
	
	Additional query words: incorrect wrong suggestion coincidence strange
