## Q197386: Random STOPs Using Cirrus Logic Display Adapters &amp; pcANYWHERE

{% raw %}

	Article: Q197386
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	On systems with some Cirrus-compatible display adapters and pcANYWHERE 8.02,
	Windows NT may randomly and infrequently stop responding (hang) and report the
	following blue screen error messages:
	
	  STOP 0x00000050 error
	  PAGE_FAULT_IN_NONPAGED_AREA.
	
	-or-
	
	  STOP 0x0000001e error
	  KMODE_EXCEPTION_NOT_HANDLED
	
	RESOLUTION
	==========
	
	To resolve this issue, contact Symantec Technical Support, or download the
	pcANYWHERE 8.02B update from:
	
	  http://www.symantec.com/
	
	For additional information, please see the following article in the Microsoft
	Knowledge Base:
	
	  Q188303 Random Stop 0x50 Errors on Cirrus Video Adapters

	products' performance or reliability.
