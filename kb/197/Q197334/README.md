## Q197334: XFOR: Setting All Chat TCP Ports to the Same Number Stops Linkin

	Article: Q197334
	Product(s): Microsoft Exchange
	Version(s): WinNT:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Microsoft Exchange Chat Server will not link to other Microsoft Exchange
	Chat Servers through a portal if both the chat clients and servers are
	configured to use the same TCP port number. Internet Relay Chat clients will not
	be able to connect to any Microsoft Exchange Chat Server in this circumstance.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft Exchange Server
	version 5.5 Chat Server program.
