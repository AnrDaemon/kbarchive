## Q197763: WD97: &quot;Styles in Use&quot; List Displays Styles Not Applied in Doc

	Article: Q197763
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	In the Format Style dialog box, when you select Styles In Use in the List box,
	Word displays some styles that have not been applied in your document.
	
	CAUSE
	=====
	
	The Styles In Use setting lists the built-in styles that you modified or applied
	in your template as well as any styles you created in the active document. Word
	lists the latter regardless of whether you have applied these styles in the
	document.
