## Q197772: PRB: Error Renaming LFN After Creating at CMD Prompt or Notepad

{% raw %}

	Article: Q197772
	Product(s): Microsoft Windows NT
	Version(s): WinNT:4.0
	Operating System(s): 
	Keyword(s): kbnetwork
	Last Modified: 10-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Windows NT Workstation 4.0 with SP3 or SP4 running Client Services for NetWare
	may receive an error when attempting to rename long file names (LFN) after
	creating them at a command prompt using COPY CON into a directory on a NetWare
	3.x or 4.x server. The error also occurs if the file is created in Notepad.
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
