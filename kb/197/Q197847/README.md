## Q197847: Application Cannot Be Started After Installing Service Pack 4

	Article: Q197847
	Product(s): Microsoft Windows NT
	Version(s): WinNT:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After installing Microsoft Windows NT Service Pack 4, RichWin 97 for NT cannot
	start, install, or uninstall.
	
	RESOLUTION
	==========
	
	To resolve this issue, please contact RichWin to obtain a patch for this issue.
	
	
	The third-party product discussed in this article is manufactured by a vendor
	independent of Microsoft; we make no warranty, implied or other- wise, regarding
	this product's performance or reliability.
	
	
	MORE INFORMATION
	================
	
	RichWin 97 For NT is a add-on program that can let user type DBCS characters on
	applications that run on Windows NT
