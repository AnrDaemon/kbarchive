## Q197664: Empty Browser Window Opens When You View Multimedia Catalog

	Article: Q197664
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:; :
	Operating System(s): 
	Keyword(s): kbtool kbimu
	Last Modified: 27-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Encyclopedia 99 
	- Microsoft Encarta Encyclopedia 2000 
	- Microsoft Encarta Encyclopedia Deluxe 2001 for Windows 
	- Microsoft Encarta Encyclopedia Standard 2001 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to view the Microsoft Multimedia Catalog included with Encarta
	Encyclopedia, your Web browser may start, but the browser window may be empty.
	
	CAUSE
	=====
	
	This behavior can occur if you are using an older version of Microsoft Internet
	Explorer.
	
	RESOLUTION
	==========
	
	To resolve this issue, obtain and install Internet Explorer version 3.02 or
	later.
	
	The latest version of Internet Explorer is available as a free download from the
	following Microsoft Web site:
	
	  http://www.microsoft.com/windows/ie/download/default.htm
