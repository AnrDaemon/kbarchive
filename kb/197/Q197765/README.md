## Q197765: WD97: List of Search Criteria for Next Citation

{% raw %}

	Article: Q197765
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbfield word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The following is a list of items that Next Citation searches for when marking
	citations for a table of authorities. Word finds any whole word or partial word
	instance of these items.
	
	  in re
	  v.
	  Id.
	  , Supra
	  Ibid
	  , Infra
	  Cong. (for Congress)
	  Sess. (for Session)
	  18xx) (to find dates in parentheses; x is any number)
	  19xx) (to find dates in parentheses; x is any number)
	  <section symbol>
	  Any previously defined short form of an authority that is unmarked
	
	MORE INFORMATION
	================
	
	To find Next Citation, click Index And Tables on the Insert menu, select the
	Table Of Authorities tab, click Mark Citation, and then click Next Citation.
	
	Additional query words: 8.0 8.00
