## Q197371: MCM: Quantum3D Obsidian2 X-24 3D Accelerator Card Not Listed

	Article: Q197371
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbui kbimu msgame
	Last Modified: 21-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Motocross Madness, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you install Motocross Madness on a computer with a Quantum3D Obsidian2 X-24
	3D accelerator card installed, the card is detected but is not listed in the
	Choose Your 3D Accelerator dialog box.
	
	RESOLUTION
	==========
	
	To use the Obsidian2 X-24 3D accelerator card to render graphics in Motocross
	Madness, click the blank line beneath your primary display adapter in the Choose
	Your 3D Accelerator dialog box.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Motocross Madness,
	version 1.0.
