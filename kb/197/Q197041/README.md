## Q197041: Encarta Encyclopedia 99: Video Clips Are Not Played Smoothly

	Article: Q197041
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu
	Last Modified: 15-AUG-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Encyclopedia 99 
	- Microsoft Encarta Reference Suite 99 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play a video clip in Microsoft Encarta Encyclopedia 99, video playback
	may be choppy or start and stop at random intervals.
	
	CAUSE
	=====
	
	This behavior can occur if either of the following conditions are true:
	
	- The clock speed of the microprocessor installed in your computer is 333 MHz
	  or faster and you are using an Intel Indeo video codec earlier than version
	  5.06.
	
	- You installed DirectX 6.0 and are using the Intel Indeo video codec version
	  5.06.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest Intel Indeo video codec
	from the following Intel Web site:
	
	  http://developer.intel.com/ial/indeo/video/driver.htm
