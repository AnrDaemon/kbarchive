## Q197673: WD97: Wrong Description for AutoScroll

	Article: Q197673
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you select the AutoScroll command (on the Tools menu, click Customize and
	then select All Commands) and then click Description, Word shows the following
	incorrect description:
	
	  More Colors
	
	  Opens the Colors dialog box, where you can pick a different color or
	  mix your own custom color.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
	
	MORE INFORMATION
	================
	
	The AutoScroll command is used to automatically start scrolling the active
	document. Word shows an incorrect description of the AutoScroll command as
	explained in the Symptoms section of this article. However, the AutoScroll
	command works correctly after you place the command on a menu or toolbar.
	
	For additional information about the AutoScroll command, please see the following
	article in the Microsoft Knowledge Base:
	
	  Q197674 WD97: How to Automatically Scroll While Reading Document
	
	Additional query words: auto scroll
