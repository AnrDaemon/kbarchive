## Q58259: Windows 2.x Should Not be Run from Windows 3.0

	Article: Q58259
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	Windows/286 and Windows/386 versions 2.x should not be run from Windows 3.0
	or 3.0a. Numerous problems can result if one version encounters the other's
	files. Also, mouse operation is lost when returning to Windows 3.0 or 3.0a
	from Windows 2.x.
	
	You can work around this limitation by running Windows 3.0 or 3.0a in real
	mode under Windows 3.0 or 3.0a standard or enhanced modes. Windows 3.0 and
	3.0a real mode is compatible with Windows 2.x applications. If Windows 2.x
	must be run, it should be run outside of Windows 3.0 and 3.0a.
	
	Additional query words: 3.0 3.00 3.0a 3.00a
