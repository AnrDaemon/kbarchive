## Q58260: Mouse Lost Under IBM DisplayWrite IV

	Article: Q58260
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you run IBM DisplayWrite IV under Microsoft Windows 3.0, your mouse may not
	work when you return to Windows.
	
	CAUSE
	=====
	
	This is a known problem and is due to DisplayWrite IV reprogramming the mouse.
	
	WORKAROUND
	==========
	
	No workaround is available at this time.

	================

	Microsoft; we make no warranty, implied or otherwise, regarding theis product's
