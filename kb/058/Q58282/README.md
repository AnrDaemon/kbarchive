## Q58282: Some Mice Not Supported in Windows 3.0, 3.1

{% raw %}

	Article: Q58282
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	The following pointing devices are supported in Microsoft Windows versions
	2.x but are not supported in Windows 3.0 or later:
	
	  HP Touch Screen
	  HP Graphics Tablet
	  Olivetti Bus Mouse
	
	Please contact your pointing-device manufacturer for information on the
	availability of a compatible pointing device driver.
