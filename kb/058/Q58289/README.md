## Q58289: Graphics Pasted Too Small into Write

{% raw %}

	Article: Q58289
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	Graphics are pasted into Windows Write at the printer resolution. The
	size of the pasted bitmap depends on the graphics resolution setting
	of the current printer driver. If the graphic you paste into Write is
	too small or too large, vary the printer resolution to obtain the
	desired size.
	
	This information applies to Microsoft Windows versions 3.0 and
	3.0a. This does not apply to later versions of Windows.
