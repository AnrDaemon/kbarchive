## Q58313: Visual Edge Not Supported in Windows 3.0 or 3.1 HPPCL Driver

	Article: Q58313
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The Intel Visual Edge printer enhancement card is supported in the Windows
	Versions 2.x HPPCL driver. It is no longer supported in the Windows Version 3.0
	or 3.1 HPPCL driver.
	
	
	Additional query words: 3.00 3.0 3.0a 3.00a WIN286 WIN386 3.10
