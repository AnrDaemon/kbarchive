## Q58283: Setting Default Printer in Control Panel

	Article: Q58283
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	To make a printer the default printer, double-click on the desired driver.
	It is not sufficient to select the desired printer in the Control Panel
	Printer configuration and choose OK.
	
	Using the keyboard, select the desired driver and press ALT+D to make it
	the default driver.
	
	Additional query words: 3.0 3.0a 3.00 3.00a 3.10 3.11
