## Q58272: Ericsson Displays Not Supported

	Article: Q58272
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	The Ericsson Color Display 9116 and the Ericsson Monochrome Display
	3111, 3712, or 3715 are supported under Windows/286 versions 2.x. They
	are no longer supported in Windows 3.0, 3.0a, 3.1, or 3.11.
	
	There is no workaround available at this time.
