## Q58386: Okidata 292 Printer Unsupported

	Article: Q58386
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The Okidata Microline 292 printer is not supported under Microsoft Windows,
	although most other Okidata dot-matrix printers are supported. The Okidata 292
	does not work properly with any of the other Okidata dot-matrix drivers, nor
	does it work correctly with the Epson 9 pin, IBM Graphics, or Generic/Text Only
	driver.
