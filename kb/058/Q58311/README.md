## Q58311: Adding Same Screen Font Twice Causes Problems

	Article: Q58311
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you add the same screen font twice through the Control Panel screen font
	installer, unpredictable problems may result, including the inability to add or
	delete fonts.
	
	WORKAROUND
	==========
	
	To work around this problem, manually edit the WIN.INI file and remove the
	duplicate font entries.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows versions 3.0 and 3.1. We
	are researching this problem and will post information here as it becomes
	available.
	
	Additional query words: 3.00 3.0 3.0a 3.00a win30 3.10 3.11 win31
