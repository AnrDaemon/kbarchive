## Q58939: Mac Form: Scrolling in FatBits When Making a Custom Form Icon

{% raw %}

	Article: Q58939
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:2.0,2.0a,2.0b,3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	It is possible to scroll in FatBits mode while designing an icon for a custom
	form in Microsoft Mail.
	
	Holding down the OPTION key activates the Browse Tool, allowing free scrolling.
