## Q58375: Erratic Output on IBM Proprinter X24 and XL24

	Article: Q58375
	Product(s): Microsoft Windows Printing Issues
	Version(s): 3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	You may experience erratic output while printing graphics from Microsoft Windows
	to the IBM Proprinter X24 or XL24.
	
	RESOLUTION
	==========
	
	To resolve this problem, disable Print Manager and turn off the Fast Printing
	Directly To Port option.
	
	To Turn Off Print Manager
	-------------------------
	
	1. Run Control Panel.
	
	2. Choose the Printers icon.
	
	3. Clear the Select Use Print Manager check box.
	
	4. Choose the OK button.
	
	5. Close the Printers dialog box.
	
	To Turn Off Fast Printing Direct To Port
	----------------------------------------
	
	1. Run Control Panel.
	
	2. Choose the Printers icon.
	
	3. Choose the Connect button.
	
	4. Clear the Fast Printing Direct To Port check box.
	
	5. Choose the OK button.
	
	6. Close the Printers dialog box.
	
	This problem should not occur on the IBM Proprinter X24e or the IBM Proprinter
	XL24e.
