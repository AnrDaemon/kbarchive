## Q58031: Flt Sim: ATIS and CT Transmissions Scroll After Pausing

	Article: Q58031
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	Radio transmissions from an ATIS (Air Traffic Information Service) or CT
	(Control Tower) will not stop scrolling when the PAUSE key is pressed. To replay
	the message, press "C" to activate the COM radio; the message will be repeated.
