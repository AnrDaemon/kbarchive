## Q58304: Varityper 11 x 17 Paper Not Supported

{% raw %}

	Article: Q58304
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The Windows PostScript driver (PSCRIPT.DRV) does not include an 11-inch x
	17-inch paper option for the Varityper VT-600 printer. The VT-600 printer
	supported by the driver is the old VT-600, which did not support 11 x 17
	printing.
