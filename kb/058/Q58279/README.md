## Q58279: ETAP Displays Not Supported in Windows

	Article: Q58279
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	The ETAP NEFTIS and ATRIS displays are supported in Microsoft Windows
	version 2.x but are not supported in Windows 3.0 or later. Please contact
	your display manufacturer for information on the availability of a Windows
	3.0 or 3.1 compatible display driver.
