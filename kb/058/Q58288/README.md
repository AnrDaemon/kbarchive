## Q58288: National Micro Systems Flash 386/20 and Windows

{% raw %}

	Article: Q58288
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Some National Micro Systems Flash 386/20 systems (with Micronix WT-386
	motherboards with 32K cache) do not run Microsoft Windows in 386 enhanced mode.
	The system stops responding (hangs) and displays the following message:
	
	  Windows/386 protection error. Please reboot your computer.
	
	CAUSE
	=====
	
	The Flash 386/20 is not a supported system under Windows 3.0 or 3.1.
