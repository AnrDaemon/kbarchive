## Q196185: XADM: Reply All to Moved Messages Sends Copy to Sender

	Article: Q196185
	Product(s): Microsoft Exchange
	Version(s): WINNT:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When a user sends a "reply to all" using a message that was received before a
	server move, the user receives a copy of the reply. This applies only to users
	whose Microsoft Exchange Server computer was moved using the Move Server Wizard.
	Users will not receive copies when they send "reply to all" messages using mail
	received after the move.
	
	MORE INFORMATION
	================
	
	During testing scenarios, this issue does not appear to be client specific.
