## Q196747: Org Chart Fails to Open When Called from Office Application

	Article: Q196747
	Product(s): Microsoft Windows NT
	Version(s): WINDOWS:97; winnt:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-DEC-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- the operating system: Microsoft Windows NT 4.0 
	- Microsoft Office 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to open an Organization Chart from within Microsoft PowerPoint,
	Microsoft Word, or Microsoft Excel, you may receive the following error message
	and you will be unable to continue with the application:
	
	  Unable to load MS Organizational Chart.
	  Press F1 for Help.
	
	The Terminal Server client session may become unresponsive at this point.
	
	RESOLUTION
	==========
	
	To resolve this issue, see the following article in the Microsoft Knowledge
	Base:
	
	  Q215358 GetWindowsDirectory May Return Incorrect Data
