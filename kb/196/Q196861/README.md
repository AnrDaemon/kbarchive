## Q196861: WD97:Numbering Displays in Outline View with Show Formatting Off

	Article: Q196861
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta word97 kbnumbering
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	In outline view in Microsoft Word, if you have numbering, heading numbering, or
	bullets applied to your outline, and then turn Show Formatting off, heading
	numbering is still displayed, although other formatting is suppressed.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
