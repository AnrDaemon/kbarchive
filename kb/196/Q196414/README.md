## Q196414: XADM: Opening Move Server Documentation Crashes JPN Netscape

	Article: Q196414
	Product(s): Microsoft Exchange
	Version(s): WinNT:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	Opening the United States (U.S.) version of the Microsoft Exchange Server Move
	Server Wizard documentation in a Japanese version of Netscape Communicator
	causes Netscape to crash.
	
	WORKAROUND
	==========
	
	The Japanese Move Server Wizard documentation works fine with the Japanese
	version of Netscape Communicator. Microsoft only supports the Japanese Move
	Server Wizard on a Japanese operating system and the U.S. Move Server Wizard on
	a U.S. operation system.
