## Q196540: WD97: Misspelled Words in Main Dictionary

{% raw %}

	Article: Q196540
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta kbproof word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	The spelling checker does not indicate that the following words are misspelled:
	
	- histyory
	
	- continuos
	
	  (Continuos is the plural of continuo, which is a musical term for a bass
	  accompaniment.)
	
	WORKAROUND
	==========
	
	Create an exclude dictionary that contains the words that you want to be checked
	during a spelling check.
	
	For additional information about how to create an exclude dictionary, please see
	the following article in the Microsoft Knowledge Base:
	
	  Q142100 How to Create an Exclude Dictionary in Word
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the products listed at the
	beginning of this article.
