## Q196418: XADM: Move Server Pauses if not Enough Space in Application Log

{% raw %}

	Article: Q196418
	Product(s): Microsoft Exchange
	Version(s): WINNT:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	Prior to executing the Move Server operation, you need to make sure that the
	application event log has enough room for the Move Server Wizard events. If the
	application log is full, the move will be paused until you change settings in
	the event log.
