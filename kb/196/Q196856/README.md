## Q196856: WD97: Part of Table Missing; Table Prints over Text

	Article: Q196856
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta winword word97 kbtable
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	You cannot see all of your table in page layout view or print preview. However,
	when you print the document, the entire table is printed. It may be printed on
	top of other text.
	
	CAUSE
	=====
	
	The document or section has been formatted with columns, and the table is too
	wide fit within the column.
	
	WORKAROUND
	==========
	
	Method 1: Modify the Column Widths of the Table
	-----------------------------------------------
	
	Modify the column widths of the table such that the table fits within the
	columns.
	
	Method 2: Insert a Section Break
	--------------------------------
	
	Insert a section break above the table and below it, and format this new section
	(with the table) for one column.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
	
	This problem has been corrected in Word 2000.
