## Q196660: WD97: Insertion Point Positioned Left of Inserted Fields

	Article: Q196660
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta kbfield word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you insert one of the following fields into a Microsoft Word document, the
	insertion point is incorrectly positioned to the left of the field:
	
	  =
	  ASK
	  AUTOTEXT
	  COMPARE
	  DATABASE
	  IF
	  INCLUDETEXT
	  INFO
	  LINK
	  MERGEFIELD
	  NOTEREF
	  PAGEREF
	  REF
	  SEQ
	  SET
	  STYLEREF
	  TOA
	
	WORKAROUND
	==========
	
	Move the insertion point to the right of the field using either the mouse or the
	arrow keys.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
