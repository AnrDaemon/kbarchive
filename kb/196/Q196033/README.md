## Q196033: XCLN: Cannot Delete Birthday and Anniversary in OWA

	Article: Q196033
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook Web Access, version 5.5 
	- Microsoft Outlook Web Access, version 5.5 Service Pack 1 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	You are unable to delete birthday and anniversary information from a contact
	using the Microsoft Outlook Web Access client.
	
	WORKAROUND
	==========
	
	Delete these items using the standard Outlook client.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Outlook Web Access.
