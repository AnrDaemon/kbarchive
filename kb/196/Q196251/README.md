## Q196251: XFOR: 1007 Character Limit for &quot;Message of the Day&quot; in Web Admin

{% raw %}

	Article: Q196251
	Product(s): Microsoft Exchange
	Version(s): WINNT:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you attempt to create a "Message of the Day" with 1007 characters using the
	Microsoft Exchange Chat Service Web Administrator, the message will not be
	created.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Chat Server
	version 5.5. This problem was corrected in the latest Microsoft Exchange 5.5
	U.S. Service Pack. For information on obtaining the service pack, query on the
	following word in the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
