## Q196841: WD97: Footnote Not Found When You Click &quot;Footnote Mark&quot; in Find

{% raw %}

	Article: Q196841
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you are searching for a footnote, and choose Footnote Mark from the Special
	button in the Find dialog box, Word only finds the footnote references in the
	document and not the actual footnotes.
	
	CAUSE
	=====
	
	The Search option is set to Down or Up, rather than All.
	
	RESOLUTION
	==========
	
	Change the Search option to All, and search again. Word now searches the
	footnote page of the document, rather than just the body of the document.
	
	Additional query words: searching misses missing missed skipped skips
