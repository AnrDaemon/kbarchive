## Q196437: WD97: Cannot Cancel Find Record Search

	Article: Q196437
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta winword word97 kbmerge
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you click the Find Record button to search for a particular record in a
	mail merge document, pressing ESC to end the search may have no effect.
	
	NOTE: The status bar shows the message "Press ESC to cancel". However, pressing
	ESC may not cancel the search.
	
	WORKAROUND
	==========
	
	There is no workaround for this problem. You must wait until Word is finished
	searching the data records.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
