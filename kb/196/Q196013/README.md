## Q196013: Virtual Globe 99: Video Continues to Play After You Press CTRL+S

	Article: Q196013
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbui kbimu
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Virtual Globe 99, version 1.0 
	- the operating system: Microsoft Windows 98 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you move the pointer over the Stop button when you play a video clip in
	Encarta Virtual Globe 99, the following tip may be displayed:
	
	  Stop (Ctrl+S)
	
	However, when you press CTRL+S, the video clip continues to play, and the World
	Statistics dialog box opens on the screen on top of the video clip.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Encarta Virtual Globe
	99, version 1.0.
	
	MORE INFORMATION
	================
	
	If you want to stop the video clip before it is finished playing, click the Stop
	button below the video window.
	
	NOTE: This issue does not occur on Microsoft Windows 95-based computers.
