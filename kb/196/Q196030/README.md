## Q196030: XCLN: The Attachment Is a Type That Is Not Yet Supported

	Article: Q196030
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook Web Access, version 5.5 Service Packs 1, 2, 3 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	After you attempt to open an embedded attachment in a message sent by either an
	Outlook Express client or a Netscape Navigator client, the following error
	message may be displayed:
	
	  The attachment is a type that is not yet supported
	
	CAUSE
	=====
	
	Currently, embedded message attachments (MHTLM) are not supported in the Outlook
	Web Access client.
	
	WORKAROUND
	==========
	
	To work around this problem, open the embedded message from a POP3 client, for
	example, Outlook Express.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
	
	Additional query words: attachment, embedded, message
