## Q79349: Windows: Paper Feed Option on Printer Setup for HP Plotter

	Article: Q79349
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you set up a Hewlett-Packard (HP) Plotter using the HPPLOT.DRV driver in
	Microsoft Windows, the 7550A, 7586B, and the Draftmaster II models allow you to
	choose the Automatic paper feed option.
	
	When you set up Windows for any other HP Plotter model, the Automatic option is
	unavailable (dimmed). All plotter models allow Manual and Preloaded to be
	chosen.
