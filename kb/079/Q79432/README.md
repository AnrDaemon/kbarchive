## Q79432: PIF Editor Help Screen for Each Option

	Article: Q79432
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 31-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	When you are using the PIF Editor and you want to find additional
	information about the PIF Editor and its options, press the F1 key to
	access Help.
	
	If you place the blinking cursor in the text box of the area you have
	questions about and press F1, the PIF Editor accesses the Help screen
	that pertains to that particular selection. This enables you to get
	specific help on the section more efficiently than if you choose Index
	(Windows 3.0) or Contents (Windows 3.1) from the Help menu.
	
	Additional query words: 3.0 3.00 3.00a 3.0a 3.1 3.10 3.11
