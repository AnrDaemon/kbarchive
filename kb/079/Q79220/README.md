## Q79220: Archive Fastape Version 5.0 with MS-DOS 5.0 and Later

	Article: Q79220
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.x,6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 5.0, 5.0a, 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Maynard Electronics recommends using version 5.0 of the Archive Fastape backup
	utility with Microsoft MS-DOS 5.0 or later.
	
	Earlier versions of Fastape may not load or complete the backup/restore procedure
	successfully with MS-DOS 5.0 or later, thus causing system lockups.
	
	MORE INFORMATION
	================
	
	Archive Fastape software is designed to backup files to a Fastape drive (for
	example, FT60 or FT180) for a stand-alone PC.

	we make no warranty, implied or otherwise, regarding these products' performance
	or reliability.
