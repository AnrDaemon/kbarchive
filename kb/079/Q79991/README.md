## Q79991: Sound in Tetris Game

{% raw %}

	Article: Q79991
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	The Tetris program included with the Windows Entertainment Pack ONLY makes
	sounds when a row is cleared from the board. The Option menu's Sound command
	turns this sound off and on.
	
	Additional reference word(s): sound tetris wep I 1.0
