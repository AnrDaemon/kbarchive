## Q79915: TriPeaks Shows No Icon When Minimized

	Article: Q79915
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	If TriPeaks, an application in the Windows Entertainment Pack volume three, has
	been maximized and then minimized, it will show no icon.
	
	Steps to Reproduce Problem
	--------------------------
	
	1. Run TriPeaks from the Entertainment Pack Three group.
	
	2. Maximize TriPeaks by clicking the up-pointing arrow in the upper-right
	  corner.
	
	3. Minimize the application. Note that the name of the application shows at the
	  bottom of the screen, but no icon appears. Also, all icons and icon titles
	  that were to the right of TriPeaks will not appear.
	
	WORKAROUND
	==========
	
	When the above problem occurs, double click on the desktop or press CTRL+ESC to
	bring up the TASK LIST. Then press the Arrange Icons button. This will refresh
	the icons and all icons will be display correctly.
	
	Microsoft is researching this problem and will post new information here as it
	becomes available.
