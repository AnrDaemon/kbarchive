## Q79992: Saving High Scores in Tut's Tomb

	Article: Q79992
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	To save High Scores in Tut's Tomb when you are finished playing, choose Exit
	from the Game menu. This will save your high scores.
	
	If you double-click the Control menu (-) in the upper-left corner or press ALT+F4
	to close the game, High Scores will not be saved.
	
	Additional query words: 1.00 1.0 wep
