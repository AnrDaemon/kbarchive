## Q181452: XADM: How to Make Exchange Extensions Available in User Manager

	Article: Q181452
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:4.0,5.0,5.5
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 01-APR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 4.0, 5.0, 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In order to make the Microsoft Exchange extensions available in the Microsoft
	Windows NT User Manager for Domains, the Microsoft Exchange Administrator
	program must be installed on the computer running User Manager for Domains.
	
	MORE INFORMATION
	================
	
	1. To install the Microsoft Exchange Administrator program, run the Microsoft
	  Exchange Server Setup program.
	
	2. Select the Complete/Custom option.
	
	3. Select only the Microsoft Exchange Administrator option, and complete the
	  Setup process.
	
	4. After the Administrator program is installed, the next time User Manager for
	  Domains is run, the Exchange extensions will be available.
