## Q181661: Files Included with the USB Supplement in OSR2.1 and OSR2.5

	Article: Q181661
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 2.1,2.5
	Operating System(s): 
	Keyword(s): osr2 win95
	Last Modified: 31-JAN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 OEM Service Release, versions 2.1, 2.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the files included with the Universal Serial Bus (USB)
	Supplement on the Windows 95 OEM Service Release 2.1 and 2.5 CD-ROM.
	
	For additional information about USB support in Windows 95, click the article
	number below to view the article in the Microsoft Knowledge Base:
	
	  Q253756 Availability of Universal Serial Bus Support in Windows 95
