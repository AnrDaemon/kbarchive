## Q181681: Microsoft Project 98 Step by Step Corrections and Comments

{% raw %}

	Article: Q181681
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 01-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Project 98 Step by Step ISBN 1-57231-605-5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains information on known errors, corrections, and comments
	relating to the Microsoft Press book Microsoft Project 98 Step by Step, ISBN
	1-57231-605-5.
	
	Contents:
	
	- Microsoft Project 98 Has No MOUS Exam
	
	- Page 39: Step 3 Should Be Click Left Arrow
	
	- Page 88: Text Error
	
	- Page 96: Text error In Note On Left Side Of Page
	
	- Page 124: Text Error
	
	- Page 159: Incorrect Text
	
	- Page 168: Step 4 Should Be Deleted
	
	- Page 169-170: Project File Is Dated 1999, Instructions 1998
	
	- Page 196: Incorrect Icon
	
	MORE INFORMATION
	================
	
	Microsoft Project 98 Has No MOUS Exam
	-------------------------------------
	
	There is no Microsoft Office User Specialist (MOUS) exam for Microsoft Project
	98.
	
	Please disregard the reference to a Microsoft Project 98 MOUS exam that is
	included in the chart on pages xvii - xviii, as well as the MOUS icon that is on
	both the front and back covers.
	
	
	Page 39: Step 3 Should Be Click Left Arrow
	------------------------------------------
	
	Page 39, "Enter project information", Step 3:
	
	Clicking the right arrow was only applicable before August 1998.
	
	Change Step 3:
	"Click the right arrow until August 1998 is displayed on the calendar."
	
	To:
	"Click the left arrow until August 1998 is displayed on the calendar." (without
	the quotation marks)
	
	
	Page 88: Text Error
	-------------------
	
	Page 88, text note:
	
	Change:
	"...are discussed further in Appendix D."
	
	To:
	"...are discussed further in Appendix C." (without the quotation marks)
	
	
	Page 96: Text error In Note On Left Side Of Page
	------------------------------------------------
	
	Page 96, note on the left side of the page:
	
	Change:
	"You can also double-click a task name..."
	
	To:
	"You can also double-click a resource name..." (without the quotation marks)
	
	
	Page 124: Text Error
	--------------------
	
	Page 124, margin text:
	
	Change:
	"ALT+HOME scrolls the Gantt chart to the taskbar for the selected task."
	
	To:
	"ALT+HOME scrolls the Gantt chart timescale to the beginning of the project."
	(without the quotation marks)
	
	
	Page 159: Incorrect Text
	------------------------
	
	Page 159, Table heading:
	
	Change:
	"Is flexible for"
	
	To:
	"Is inflexible for" (without the quotation marks)
	
	
	Page 168: Step 4 Should Be Deleted
	----------------------------------
	
	On page 168, if you complete step 4 at the bottom of the page, which instructs
	you to level the project, you will not be able to complete the remaining steps
	under the "Use Automatic Leveling" section.
	
	Delete:
	"4. Level the project. (Hint: Be sure the bottom pane of the Resource Allocation
	view is selected.)
	
	
	Page 169-170: Project File Is Dated 1999, Instructions 1998
	-----------------------------------------------------------
	
	In step 1 we open the project file Review3. This file refers to a project taking
	place in 1999. The steps on pages 169 and 170 indicate changes being made in
	that file to reflect the year 1998.
	
	Change all dates within this particular exercise to reflect the year 1999.
	
	Dates will be changed in:
	
	Step 2, #5 and #6
	Step 4, #3 and #4
	
	
	Page 196: Incorrect Icon
	------------------------
	
	Page 196, left margin, icon:
	
	Change:
	The Gantt Chart Wizard icon (Standard toolbar)
	
	To:
	"The Gantt Chart icon (View Bar)" (without the quotation marks)
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
