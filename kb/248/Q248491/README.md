## Q248491: Fighter Ace II: Game Hangs After You Change Screen Resolution

	Article: Q248491
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbenv igz kbimu msgame
	Last Modified: 09-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Fighter Ace II 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you change the screen resolution in Microsoft Fighter Ace II, the game may
	stop responding (hang) when you click Fly Now.
	
	CAUSE
	=====
	
	This behavior can occur if a Real Starfighter 3D video adapter is installed in
	your computer.
	
	RESOLUTION
	==========
	
	To work around this issue, configure Fighter Ace to use software rendering:
	
	1. Start Fighter Ace II.
	
	2. On the startup screen, click the Settings tab.
	
	3. Under Display, click Software.
