## Q248311: Fighter Ace II: Game Quits When Detecting 3D Accelerators

	Article: Q248311
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): kb3rdparty kbenv kbsetup igz kbimu msgame
	Last Modified: 09-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Fighter Ace II 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When the Microsoft Fighter Ace profiler attempts to detect 3D accelerators, the
	game may suddenly quit.
	
	CAUSE
	=====
	
	This behavior can occur if the following conditions are true:
	
	- A Diamond Stealth 3D 2000 video adapter is installed in the computer.
	
	- The video driver for the Diamond Stealth 3D 2000 video adapter is outdated.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest driver for the Diamond
	Stealth 3D 2000 video adapter from the following Diamond Multimedia Web site:
	
	  http://www.diamondlegacy.com/
