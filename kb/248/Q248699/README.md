## Q248699: Iomega Tools Causes Blank File Name When Renaming Files

	Article: Q248699
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you install Iomega Tools, when you try to rename a file in Explorer, the
	file name may go blank. Although you can still rename files and folders, you
	cannot see what you are typing.
	
	CAUSE
	=====
	
	This behavior can occur when you use a version of Iomega Tools provided with
	older Zip and Jaz drives.
	
	RESOLUTION
	==========
	
	To resolve this behavior, obtain the latest version of the Iomega software for
	your Zip or Jaz drive.
