## Q288974: Registration Intitiates Before Each MSN Companion Session

	Article: Q288974
	Product(s): The Microsoft Network
	Version(s): 1.0,2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, versions 1.0, 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Before every MSN companion session, you have to log on via the 1-800
	registration number before you can log on and browse the Internet. Common user
	comments are as follows:
	
	- Why do I have to go through signup each time I start the device?
	
	- Why do I keep getting "Welcome" messages?
	
	CAUSE
	=====
	
	One of the files on the device has become corrupt.
	
	RESOLUTION
	==========
	
	Conduct a "FAILSAFE" procedure.
	See KB article Q274256 MSN Companion Troubleshooting Tool FAILSAFE.
	(http://support.microsoft.com/support/kb/articles/Q274/2/56.ASP)

	================
