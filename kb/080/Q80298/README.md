## Q80298: AST Premium Exec Notebook May Need Reverse Video

	Article: Q80298
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you use the AST Premium Exec Notebook 386/25 and Microsoft Windows, the LCD
	display is easier to read if it is set to reverse video.
	
	To set reverse video on the AST Premium Exec Notebook, hold down the FUNCTION key
	on the lower-left side of the keyboard and press the R key.
