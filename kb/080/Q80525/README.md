## Q80525: FltSim: Using System Time Causes Night Flight

	Article: Q80525
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	During the months of October, November, and December 21-31, using the system
	time with Microsoft Flight Simulator for the Macintosh version 4.0 will cause
	Flight Simulator to always fly at night.
	
	MORE INFORMATION
	================
	
	To correct this problem, do the following:
	
	1. From the World menu, choose Set Time and Season.
	
	2. Note that the Season option is blank. Choose a season from the list of
	  Seasons and click OK.
	
	Flight Simulator will now operate correctly.
	
	Microsoft has confirmed this to be a problem in Flight Simulator for the
	Macintosh version 4.0. This problem was corrected in version 4.0a.
	
	Additional query words: 4.00 season October November December
