## Q80652: PC Win: Scroll Bars Are Not Available on Desktop

{% raw %}

	Article: Q80652
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Microsoft Mail for Windows version 3.0, when a message viewer, document, or
	icon is off or outside the desktop, scroll bars are not available.
	
	CAUSE
	=====
	
	Mail for Windows follows the design of Microsoft Word for Windows, Microsoft
	Excel for Windows, Microsoft Project for Windows, and Microsoft PowerPoint for
	Windows.
	
	RESOLUTION
	==========
	
	To get all the icons on the desktop, choose Arrange Icons from the Window menu.
	To get all the windows on the desktop, choose Cascade or Tile from the Window
	menu.
