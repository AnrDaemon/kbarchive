## Q153972: Err Msg: Error 1:There Is an Internal Error on the IRDA Device

{% raw %}

	Article: Q153972
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you double-click the Infrared icon in Control Panel, you may receive the
	following error message:
	
	  ERROR 1: There is an internal error on the IRDA device...
	
	CAUSE
	=====
	
	This error can occur if the infrared port is disabled in the computer's CMOS
	settings.
	
	RESOLUTION
	==========
	
	Enable the Infrared port in the computer's CMOS settings. For information about
	changing your computer's CMOS settings, please consult the computer's
	documentation or manufacturer.
