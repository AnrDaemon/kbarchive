## Q153796: Close Combat: Hang at Splash Screen

	Article: Q153796
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbenv kbmmkbfaq
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Close Combat for Windows 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to run Close Combat for Windows, your computer may hang (stop
	responding) at the animated Microsoft logo splash screen.
	
	CAUSE
	=====
	
	You may experience this behavior with certain video cards if you have configured
	Windows 95 to play video in full-screen mode.
	
	RESOLUTION
	==========
	
	The following steps describe how to resolve this problem:
	
	1. Click the Start button, point to Settings, and then click Control Panel.
	
	2. Double-click Multimedia, click the Video tab, and then click the Window
	  button. In the windows list, click Original size.
	
	3. Click OK.
	
	MORE INFORMATION
	================
	
	This problem appears to occur just as the animated Microsoft logo splash screen
	(Mslogo.avi) begins playing. It occurs as the Mslogo.avi switches from playing
	in a window to playing in a full screen.
	
	
	Additional query words: 1.00 lock crash hanging hung freeze freezes stop responding frozen
