## Q153778: Close Combat: Unit Status Messages Out of Order

	Article: Q153778
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbui cc kbimu msgame
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Close Combat for Windows 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Close Combat, you may receive a message that a unit or vehicle has
	been destroyed before you receive a message that it has been hit.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Close Combat, version
	1.0.
	
	Additional query words: 1.00 cc radio msgame
