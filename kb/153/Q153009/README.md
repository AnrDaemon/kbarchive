## Q153009: XCLN: Installing Microsoft Mail Service on Windows 3.x Client

	Article: Q153009
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:4.0,5.0
	Operating System(s): 
	Keyword(s): kbsetup
	Last Modified: 31-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Windows 3.x client, versions 4.0, 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you install the Microsoft Exchange client for Windows 3.x, the Microsoft
	Mail service will not appear as a service to be added to a Microsoft Exchange
	profile even if the custom setup option is selected.
	
	MORE INFORMATION
	================
	
	To add the Microsoft Mail service to an already installed Microsoft Exchange
	client, follow these steps:
	
	1. Run the Setup program off of the Microsoft Exchange Client Software CD.
	
	2. Click the Add/Remove button.
	
	3. Click the Change Option button.
	
	4. Select Information Services.
	
	5. Click the Change Option button.
	
	6. Check the Microsoft Mail checkbox.
	
	7. Click OK.
	
	8. Click OK.
	
	9. Click Continue.
	
	10. Click OK.
