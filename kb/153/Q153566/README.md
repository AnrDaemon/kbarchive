## Q153566: SMS: Remote Control of Schedule+ shows incorrect colors

	Article: Q153566
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.0,1.1
	Operating System(s): 
	Keyword(s): kbnetwork kbtshoot smsremtshoot smsgeneral kbArtTypeINF kbRemoteProg
	Last Modified: 31-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server versions 1.0, 1.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	When you remote control a Windows for Workgroups client computer running
	Schedule+ from the Systems Management Server Administrator program, you are
	unable to view the different colors in the display options for Appointment
	Book/Planner backgrounds. All of the colors can be viewed correctly from the
	client computer.
	
	NOTE: The colors that cannot be viewed correctly are Schedule+ pastel Background
	colors. They are displayed in gray colors.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	versions 1.0 and 1.1. This problem has been corrected in Systems Management
	Server version 1.2.
	
	Additional query words: prodsms Schedule+ Remote Control grey
