## Q153933: SMS: INVWIN32 Does Not Collect IRQ Information

{% raw %}

	Article: Q153933
	Product(s): Microsoft Systems Management Server
	Version(s): MS-DOS:1.0,1.1,1.2
	Operating System(s): 
	Keyword(s): kbnetwork kbInventory smsinv
	Last Modified: 25-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Basic Professional Development System for MS-DOS, versions 1.0, 1.1, 1.2 
	-------------------------------------------------------------------------------
	
	The Windows NT hardware inventory (INVWIN32) does not collect IRQ
	information. Therefore, an icon for the IRQ table in the Windows NT PC
	Properties window of Systems Management Server does not exist.
	
	IRQ information is available in Windows NT Diagnostics in PC Properties.
	
	Note: You cannot query against the information in Windows NT Diagnostics.
	
	Additional query words: prodsms IRQ NT inventory
