## Q153459: Kids: Troubleshooting Help File Included on Compact Disc

	Article: Q153459
	Product(s): Microsoft Home Kids Products
	Version(s): 1.0; WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbmm kbusagekbfaq
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft 3D Movie Maker 
	- Microsoft Nickelodeon 3D Movie Maker for Windows, version 1.0 
	- Scholastic's Magic School Bus series: Explores the Ocean for Windows, version 1.0 
	- Scholastic's Magic School Bus series: Explores the Human Body for Windows, version 1.0 
	- Scholastic's Magic School Bus series: Explores Inside the Earth for Windows, version 1.0 
	- Scholastic's Magic School Bus series: Explores the Solar System for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Magic School Bus series and Microsoft 3D Movie Maker compact discs
	include a troubleshooting Help file to help you make changes to your computer
	configuration that may be causing problems for your Microsoft Kids application.
	
	Common areas of interest, such as video, audio, and CD-ROM configuration are
	addressed in each Help file.
	
	MORE INFORMATION
	================
	
	To find a Microsoft Kids Help file:
	
	Windows 95
	----------
	
	Click the Start button, point to Programs, point to Microsoft Kids, then click
	the Help file appropriate to your kids program.
	
	Windows 3.x:
	------------
	
	1. In Program Manager, go to the MS Kids program group.
	
	2. The Help files are listed in the MS Kids program group by their program name.
	  Double-click the appropriate Help file icon.
