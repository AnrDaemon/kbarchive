## Q153708: Cinemania 1996: Button Images in Find Menu Misplaced

	Article: Q153708
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1996 edition; WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Cinemania for Windows 1996 edition 
	- the operating system: Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you are using a Matrox MGA Millennium video card, the button images in the
	Cinemania Find menu look like they overlap the category labels.
	
	CAUSE
	=====
	
	Device Bitmap Caching for the Matrox MGA Millennium video card is turned on
	(enabled).
	
	RESOLUTION
	==========
	
	To resolve this problem, do the following:
	
	1. Click the Start button, point to Settings, and then click Control Panel.
	
	2. Double-click Display.
	
	3. Click the MGA Settings tab, and then click Advanced button.
	
	4. Clear the check box next to the Device Bitmap Caching option.
	
	5. Click OK.
	
	6. Restart the computer when you are prompted.
	
	MORE INFORMATION
	================
	
	These symptoms have not been observed in version 3.0 of the Matrox MGA
	Millennium video driver.
