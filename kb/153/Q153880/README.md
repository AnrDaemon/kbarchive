## Q153880: INFO: Visigenic Oracle Driver and Visual C++ Enterprise Edition

{% raw %}

	Article: Q153880
	Product(s): Microsoft C Compiler
	Version(s): 4.2
	Operating System(s): 
	Keyword(s): kb3rdparty kbother kbEEdition kbVC
	Last Modified: 31-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual C++, 32-bit Enterprise Edition, version 4.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains the latest information on the Visigenic Oracle ODBC driver
	that is to be used to provide Enterprise Edition support for Oracle data
	sources. This article will be updated as the status of the driver changes.
	
	MORE INFORMATION
	================
	
	The Visigenic Oracle ODBC driver currently shipping with Visual C++ Enterprise
	Edition, version 2.00.03M2, is a beta release that was available at the time the
	product was shipped. Microsoft is planning on providing the released version of
	the driver when it becomes available. Details on how to obtain the updated
	driver will be published in this Knowledge Base article when it is updated.
