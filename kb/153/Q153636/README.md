## Q153636: SMS Client Group Remains on the Windows 95 Task Bar

	Article: Q153636
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.1
	Operating System(s): 
	Keyword(s): kbsetup smssetup
	Last Modified: 27-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	The SMS Client group remains on the Windows 95 task bar after the SMS client
	installation has been completed.
	
	CAUSE
	=====
	
	This problem occurs if you enable the option "Display the full MS-DOS path in
	the title bar."
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.1. This problem has been corrected in Systems Management Server
	version 1.2
	
	Additional query words: prodsms win95 group
