## Q153853: XADM: MSDN July CD has Hidden Setup.exe Files

	Article: Q153853
	Product(s): Microsoft Exchange
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbsetup
	Last Modified: 31-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 4.0 
	-------------------------------------------------------------------------------
	
	On the July 1996 Microsoft Developers Network (MSDN) compact disc set, the
	Setup.exe file for Exchange is marked as Hidden. To install the server you
	must first set File Manager to view Hidden files.
