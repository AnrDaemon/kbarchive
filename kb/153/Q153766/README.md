## Q153766: XCLN: Substitute IP Address for Server Name

	Article: Q153766
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:4.0,5.0
	Operating System(s): 
	Keyword(s): kbenv
	Last Modified: 02-APR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Windows 3.x client, versions 4.0, 5.0 
	- Microsoft Exchange Windows NT client, versions 4.0, 5.0 
	- Microsoft Exchange Windows 95/98 client, versions 4.0, 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If the ability of a network to resolve NetBIOS names is in question, or you want
	to target name resolution as the network fault, the following technique may be
	useful.
	
	You can substitute the IP address (xxx.xxx.xxx.xxx) for the Microsoft Exchange
	Server Name in the Microsoft Exchange Setup Wizard or when adding a profile in
	the Windows 3.1, Windows 95 and Windows NT clients.
	
	The Exchange MS-DOS client will not allow this substitution.
