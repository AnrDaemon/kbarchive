## Q153156: Sched.exe OLE Automation Adds Schedule+ Appointments

{% raw %}

	Article: Q153156
	Product(s): Microsoft FoxPro
	Version(s): WINDOWS:3.0,3.0b
	Operating System(s): 
	Keyword(s): kbfile kbsample kbvfp300 kbDSupport
	Last Modified: 11-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual FoxPro for Windows, versions 3.0, 3.0b 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Sched.exe is a sample that demonstrates how to use OLE Automation to add
	appointments to your Microsoft Schedule+ file.
	
	MORE INFORMATION
	================
	
	The following files are available for download from the Microsoft Download
	Center:
	
	  Sched.exe
	  (http://download.microsoft.com/download/fox30pro/demo/1/WIN98/EN-US/Sched.exe)
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
	
	Microsoft used the most current virus detection software available on the date of
	posting to scan this file for viruses. Once posted, the file is housed on secure
	servers that prevent any unauthorized changes to the file.
	
	The Readme.txt file contained in the Sched.exe self-extracting file provides the
	location of the key elements of the process (within the form and class library)
	of adding Schedule+ appointments.
