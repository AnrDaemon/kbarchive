## Q153712: Windows 95 Hangs with Always IN2000 SCSI Controller

	Article: Q153712
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are using a computer with an Always IN2000 SCSI controller installed,
	Windows 95 may seem to stop responding (hang).
	
	CAUSE
	=====
	
	This behavior can occur if the IN2000 SCSI controller uses firmware revision 2.7
	or earlier.
	
	RESOLUTION
	==========
	
	Upgrade the firmware on the IN2000 SCSI controller. The revision can be obtained
	from Bason Computer.
	
	MORE INFORMATION
	================
	
	Always is no longer in business. Bason Computer, Inc., provides limited support
	for the Always IN2000 card.
