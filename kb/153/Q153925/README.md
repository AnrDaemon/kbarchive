## Q153925: FAQ: How Can I Find Out if My VSS Database Has Been Upgraded?

	Article: Q153925
	Product(s): Microsoft SourceSafe
	Version(s): 
	Operating System(s): 
	Keyword(s): kbSSafe600 kbAnalyze kbSSafeAnalyzekbfaq
	Last Modified: 31-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual SourceSafe for Windows, version 6.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Q. How can I find out if my Visual SourceSafe database has been upgraded to the
	new format?
	
	A. To determine if your Visual SourceSafe 6.0 database has been upgraded to the
	new database format, check your Visual SourceSafe data directory for a file
	named Crcs.dat and a subdirectory named Labels.
