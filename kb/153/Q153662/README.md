## Q153662: SMS: Internet Explorer GP Faults When Being Remote Controlled

	Article: Q153662
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.0,1.1
	Operating System(s): 
	Keyword(s): kbnetwork kbtshoot smsremtshoot kbRemoteProg
	Last Modified: 31-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server versions 1.0, 1.1 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you run the Microsoft Internet Explorer to view a web site within a remote
	control session, the following error message appears on the client computer
	using a Cirrus video driver:
	
	  IExplore caused a General Protection Fault in module IDIS_LM.DLL at
	  <address>
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.1. This problem has been corrected in Systems Management Server
	version 1.2.
	
	Additional query words: prodsms explorer internet remote control
