## Q153968: Unattended Installation of Service Pack 4 After NT Installed

	Article: Q153968
	Product(s): Microsoft Windows NT
	Version(s): 3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.51 
	- Microsoft Windows NT Server version 3.51 
	-------------------------------------------------------------------------------
	
	Windows NT 3.51 Service Pack 4 can be installed unattended on an
	existing installation of Windows NT, by means of the /U parameter.
	
	From a booted copy of Windows NT, choose File Run in Program Manager.
	Type the path to the Service Pack directory for your computer type,
	followed by Update.exe. Add the /U parameter to the end of the line.
	
	Example:
	
	  e:\i386\update.exe /u
