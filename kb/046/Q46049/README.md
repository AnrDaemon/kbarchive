## Q46049: Paper Sizes and Names

{% raw %}

	Article: Q46049
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	This article lists paper sizes and their common names.
	
	MORE INFORMATION
	================
	
	American
	--------
	
	  Name       Size (in Inches)
	  ----       ----------------
	  Mini        5.5  x  8.5
	  Executive   7.25 x 10.5
	  Letter      8.5  x 11.0
	  Foolscap    8.5  x 13.0
	  Legal       8.5  x 14.0
	  Ledger     11.0  x 17.0
	
	European
	--------
	
	          Size
	  Name    (in Millimeters)  Size (in Inches)
	  ----    ----------------  ----------------
	  A0      841 x 1189        33.1  x 46.8
	  A1      594 x  841        23.4  x 33.1
	  A2      420 x  594        16.5  x 23.4
	  A3      297 x  420        11.7  x 16.5
	  A4      210 x  297         8.27 x 11.7
	  A5      148 x  210         5.83 x  8.27
	  B4      257 x  364        10.1  x 14.3
	  B5      182 x  257         7.16 x 10.1
