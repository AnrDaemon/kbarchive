## Q253446: Allegiance Err Msg: Invalid Page Fault in Module &lt;Unknown&gt;

	Article: Q253446
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kberrmsg kbsetup igz kbimu msgamekbfaq
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Allegiance, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After Microsoft Allegiance automatically installs a software update, Allegiance
	may quit unexpectedly, and you may receive the following error message:
	
	  Allegiance caused an invalid page fault in module <unknown>.
	
	CAUSE
	=====
	
	This behavior can occur if Allegiance does not restart properly after it
	installs the update.
	
	RESOLUTION
	==========
	
	To resolve this issue, restart your computer, and then start Allegiance again.
