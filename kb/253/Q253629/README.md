## Q253629: FS2000: No Control Tower Clearance for Takeoffs or Landings

{% raw %}

	Article: Q253629
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbui fsim kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator 2000 
	- Microsoft Flight Simulator 2000 Professional Edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In previous versions of Microsoft Flight Simulator, you can request clearance
	for takeoffs and landings. These basic control tower clearance requests are not
	available in Flight Simulator 2000, except in certain lessons and adventures.
