## Q253328: Flight Simulator Troubleshooters Available

	Article: Q253328
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator: WWII Europe Series, version 1.0 
	- Microsoft Flight Simulator 98 
	- Microsoft Flight Simulator 2000 
	- Microsoft Flight Simulator 2000 Professional Edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article discusses the availability of Microsoft Flight Simulator
	troubleshooters.
	
	The Flight Simulator troubleshooters provide step-by-step instructions to help
	you resolve common issues with the Microsoft programs listed at the beginning of
	this article.
	
	MORE INFORMATION
	================
	
	The Flight Simulator troubleshooters are available at the following Microsoft
	Web site:
	
	  http://support.microsoft.com/support/games/fsim/fstshoot/default.asp
	
	Additional query words: 1.00 msgame combatfs cfs flightsim fsim fs2000 fs2k fs98 ts t-shoot
