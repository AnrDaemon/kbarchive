## Q253930: Expedia Streets and Trips 2000: How to Clear the Find List

	Article: Q253930
	Product(s): Microsoft Automap
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbui kbimu
	Last Modified: 27-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Expedia Streets and Trips 2000 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	To remove previous locations from the Find list in Microsoft Expedia Streets and
	Trips 2000, download and install the Mruclean.exe file from the Microsoft
	Download Center.
	
	NOTE: You must quit Streets and Trips 2000 before you install the Mruclean.exe
	file.
	
	MORE INFORMATION
	================
	
	The following file is available for download from the Microsoft Download
	Center:
	
	  Mruclean.exe
	  (http://download.microsoft.com/download/expediast2000/Utility/1.0/WIN98/EN-US/mruclean.exe)
	
	Release Date: Feb-15-2000
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
	
	Microsoft used the most current virus detection software available on the date of
	posting to scan this file for viruses. Once posted, the file is housed on secure
	servers that prevent any unauthorized changes to the file.
