## Q63770: Bitstream Font Versions Compatible with Windows 3.0

{% raw %}

	Article: Q63770
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	Bitstream Fontware versions 2.00 and later are compatible with Microsoft Windows
	version 3.0. Although Bitstream has released a 3.00 version of Fontware for
	Windows, Bitstream has confirmed that Fontware versions 2.00 and later are
	compatible with Windows 3.0.
	
	MORE INFORMATION
	================
	
	For additional information, contact Bitstream.
	
	The product included here is manufactured by a vendor independent of Microsoft;
	we make no warranty, implied or otherwise, regarding this product's performance
	or reliability.
