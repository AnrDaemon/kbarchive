## Q63776: Flt Simulator: Incorrect ILS Glideslope at San Francisco Intl

{% raw %}

	Article: Q63776
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	When you are approaching runways 19L (108.9) and 28R (111.7) on an ILS approach
	at San Francisco International Airport, the glideslope reportedly incorrectly
	leads to the far end of the runway.
	
	This problem has been reported with Flight Simulator version 4.00. Microsoft is
	researching this problem and will post new information here, as it becomes
	available.
