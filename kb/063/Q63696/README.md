## Q63696: Hardware Options Offered When Installing Windows

	Article: Q63696
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the hardware options offered during the installation of
	Microsoft Windows version 3.0 or 3.0a. If the Setup program offered any other
	choices for hardware, it is not Microsoft Windows but an OEM (original equipment
	manufacturer) version of Windows; additional support must be obtained from the
	manufacturer.
	
	MORE INFORMATION
	================
	
	The following choices are listed during the Setup process.
	
	Computer
	--------
	
	  All 80386- and 80486-based AST machines
	  All 80386-based Zenith machines
	  All Hewlett-Packard machines
	  Everex Step 386/25
	  MS-DOS or PC-DOS system
	  NCR PC386SX
	  NCR PC 925
	  NEC PowerMate SX Plus
	  NEC ProSpeed 386
	  Toshiba 1600
	  Toshiba 5200
	  Other (Requires disk provided by a hardware manufacturer)

	  Video Seven VGA with 512K

	Mouse
