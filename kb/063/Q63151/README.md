## Q63151: Flt Sim 4.00: Screen Options for Vectra CS-120 Portable

{% raw %}

	Article: Q63151
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	Flight Simulator version 4.00 will not display in the EGA display mode on the
	Hewlett-Packard Vectra CS-120 (the 8086 laptop model) equipped with an EGA LCD
	screen.
	
	To use the Hewlett-Packard Vectra CS-120 with Flight Simulator, select the
	display option of color composite CGA or black-and-white monitor (CGA).
	
	Microsoft is researching this problem and will post new information here as it
	becomes available.
