## Q63594: Incorrect Fonts Displayed with Canon LBP-8II Driver

{% raw %}

	Article: Q63594
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Canon LBP-8II printer driver for Microsoft Windows version 3.0 incorrectly
	lists the names for the Swiss and Humanist font cartridges. If you choose the
	Swiss 721p cartridge, the font available in Windows is Humanist. If the Humanist
	cartridge is selected, the font available in Windows is Helvetica (Swiss). This
	information applies to both the Portrait and the Landscape cartridges. The Dutch
	and Century cartridges are not reversed in this manner.
