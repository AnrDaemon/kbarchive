## Q63249: Mac AL: AppleLink GW Recipients Are Listed in Simple User List

	Article: Q63249
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:2.0,2.0a,2.0b,3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	AppleLink gateway recipients that are defined on an AppleLink gateway server
	will propagate to all AppleLink gateway access servers created from this gateway
	server.
	
	AppleLink gateway recipients can be accessed through the Address Window or
	through the Simpler User List.
	
	For more information on creating AppleLink gateway recipients, see the "Gateway
	for AppleLink User's Guide."
