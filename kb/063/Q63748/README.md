## Q63748: Using Colorgraphics Dual VGA Card with Windows 3.0

	Article: Q63748
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Colorgraphic Communications Corporation's Dual VGA Plus Card is a video card
	that supports resolutions of up to 800 x 600 on two or more monitors.
	Colorgraphic Communications has a driver for this card for Microsoft Windows 3.0
	that allows two VGA monitors to be used side-by-side. This doubles the Windows
	workspace area and acts as one large monitor. Information can be displayed in
	either the left monitor, the right monitor, or across the monitors.
	
	MORE INFORMATION
	================
	
	This card comes in both an AT-compatible and an MCA-compatible (PS/2-
	compatible) configuration. For more information, contact Colorgraphic
	Communications Corporation.
