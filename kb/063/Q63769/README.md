## Q63769: No AppleTalk Support in Windows

{% raw %}

	Article: Q63769
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 02-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The AppleTalk Network is not supported in Microsoft Windows. APPLETLK.DLL
	included with Windows 2.11 (only) was written by an outside vendor and was not
	designed to run under later versions of Windows.
	
	NOTE: It has been reported that the APPLETLK.DLL file works with Windows 3.0 if
	Windows 3.0 is running in real mode.
	
	MORE INFORMATION
	================
	
	The AppleTalk network is a network that is included with the Apple Macintosh
	line of personal computers.
	
	The Apple products included here are manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding these products'
	performance or reliability.
