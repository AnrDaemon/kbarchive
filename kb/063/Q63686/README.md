## Q63686: Disk Space Required for Windows 3.0 Installation

	Article: Q63686
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 3.0 
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following is a list of the disk-space requirements for the installation of
	Microsoft Windows versions 3.0 and 3.00a. Note that if disk space is not
	available, some files may be deleted after installation. See chapter 13 of the
	"Microsoft Windows User's Guide" version 3.0 manual for more information on
	which files may be removed.
	
	MORE INFORMATION
	================
	
	Standalone Installations
	------------------------
	
	Machine with 80386 processor: Approximately 6.3 million bytes (6.1 MB).
	(Add 2K with 3.00a update.)
	
	Machine with 80286 processor: Approximately 4.4 million bytes (4.4 MB).
	(Add 1K with 3.00a update.)
	
	Network Installations
	---------------------
	
	Network Workstation: Approximately 200,000 bytes (195K).
	
	Network Server: Approximately 9,663,160 bytes (9.4 MB).
	
	Optional ToolBook: Approximately 1,002,703 bytes (979K)
	
	Additional query words: 3.00 3.00a
