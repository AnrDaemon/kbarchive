## Q63668: Superbase 4 Version 1.2 or Later Required for Windows 3.0

	Article: Q63668
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You are running Superbase 4 with Microsoft Windows 3.0, and the 16-color
	pictures display as distorted grayscale pictures.
	
	CAUSE
	=====
	
	According to the manufacturer, you need Superbase 4 version 1.2 or later to
	properly run Superbase with Microsoft Windows version 3.0. Superbase 4 version
	1.11 runs in real mode of Windows 3.0, but it uses the Windows 2.x help files
	and displays pictures as in Windows 2.x.
	
	RESOLUTION
	==========
	
	Contact Software Publishing Corp. for an upgrade.

	independent of Microsoft; we make no warranty, implied or otherwise, regarding
	this product's performance or reliability.
