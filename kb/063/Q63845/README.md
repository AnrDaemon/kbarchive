## Q63845: Using the HP QuietJet Printer with Windows Version 3.0

	Article: Q63845
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The Hewlett-Packard (HP) QuietJet printer is not directly supported in Microsoft
	Windows version 3.0. To use the QuietJet printer with Windows 3.0, put the
	printer in alternate mode and use the Epson FX100 or Epson FX100+ printer
	driver.
	
	
	Additional query words: 3.00 3.0 3.0a 3.00a 3rdparty third party quiet jet
