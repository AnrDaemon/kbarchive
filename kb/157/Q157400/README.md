## Q157400: Soccer: Using the IntelliPoint Mouse with Soccer

	Article: Q157400
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbfaq
	Last Modified: 08-NOV-2001
	
	1.0
	WINDOWS
	kbusage kbfaq
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Soccer, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you've installed IntelliPoint Mouse software, it may interfere with the
	functionality of Soccer. Specifically, if the ClickLock option on the
	Productivity tab is on, and you play Soccer, two clicks are required to do what
	one click ordinarily does.
	
	Additional query words: 1.00 football soccer intelli-point conflict double click double-click
