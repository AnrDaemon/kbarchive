## Q157094: WD97: Using StartConversionWizard to Convert Multiple Documents

	Article: Q157094
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbfaq
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to use the StartConversionWizard to convert multiple
	documents to and from Word 97. The StartConversionWizard runs when you open the
	Convert8.wiz file, located in the Office\Macros subfolder.
	
	NOTES: The Convert8.wiz file is not installed when you choose a typical setup. To
	install this wizard, rerun Setup in maintenance mode, choose Add/Remove, and
	install the Macro Templates.
	
	Before you run the StartConversionWizard, check to ensure that the text
	converters and graphics filters you need are installed. If the appropriate
	converters and filters are not installed, rerun Setup in maintenance mode,
	choose Add/Remove, and install the converters and filters you want.
	
	MORE INFORMATION
	================
	
	To use the StartConversionWizard, do the following:
	
	1. On the File menu, click Open.
	
	2. Locate and open the Convert8.wiz file. Typically this file is located in the
	  C:\Program Files\Microsoft Office\Office\Macros subfolder.
	
	3. If you get a Warning dialog box, click Enable Macros. The Wizard starts
	  automatically. If you do not get this warning, skip to step 4.
	
	4. Click the button next to A Batch Conversion of Files.
	
	5. Click the Next button.
	
	6. On the To/From panel, select one of the following options:
	
	   - Convert From Another Format To Word.
	
	   - Convert From Word To Another Format.
	
	7. (Optional) If you want to change conversion options, click the Options
	  button, make your changes, and then click OK.
	
	8. Click Next.
	
	9. On the Folder Selection panel, click the Browse button next to Source Folder,
	  select the folder that contains the files you want to convert, and click OK.
	
	10. Click the Browse button next to Destination Folder, select the folder where
	  you want to copy the converted documents, and click OK.
	
	  Note: If you are converting a large number of documents, do not select the
	  root directory, C:\.
	
	11. Click Next.
	
	12. On the File Selection panel, double-click the file names of the documents
	  you want to convert, or click Select All.
	
	13. Click Next.
	
	14. Click Finish.
	
	15. When the wizard finishes converting your documents, click Yes to do another
	  conversion or click No to close the wizard.
	
	16. Close the Convert8.wiz file.
	
	For information about how to do this in earlier versions of Word, please see the
	following article in the Microsoft Knowledge Base:
	
	  Q107439 Word for Windows Batch Conversion Macro
