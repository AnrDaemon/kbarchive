## Q157434: PPT: &quot;There are Too Many Windows Open&quot; Error Message

	Article: Q157434
	Product(s): Microsoft PowerPoint for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): kberrmsg
	Last Modified: 16-APR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft PowerPoint for Windows 95, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you try to open a presentation or if you click New Window on the Window menu,
	you may receive this error message:
	
	  There are too many windows open. Please close one.
	
	This error occurs if you already have fifty PowerPoint windows open.
	
	CAUSE
	=====
	
	PowerPoint 7.0 is designed to support up to fifty windows open simultaneously.
	
	Additional query words: 7.00 views multiple slideshows
