## Q157241: Reboot Prompt After Installing Compaq Business Audio Driver

{% raw %}

	Article: Q157241
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kbhw kbHardware
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you install the Compaq Business Audio driver, a dialog box prompts you to
	restart the computer before the driver will work. However, the driver begins
	working immediately.
	
	STATUS
	======
	
	This behavior is by design while Plug and Play features are being integrated
	into Windows NT.
