## Q157294: WD97: Some Commands Not Available When Control Object Selected

{% raw %}

	Article: Q157294
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the commands that are unavailable if you have a control
	object selected. A control object is a check box, list box, or command button
	you create using the ActiveX tools on the Control Toolbox toolbar. You use
	Design Mode to place the control objects on the form. To access Design mode, do
	the following:
	
	1. On the View menu, click Toolbars.
	
	2. Click Control Toolbox on the Toolbars submenu.
	
	3. Click the Design Mode button on the Control Toolbox toolbar.
	
	MORE INFORMATION
	================
	
	The following commands are not available when you have a control object
	selected:
	
	File menu:
	
	  Page Setup
	
	Edit menu:
	
	  Paste As Hyperlink
	  Links
	
	View menu:
	
	  Footnotes (will be available if document contains footnotes)
	  Comments (will be available if document contains comments)
	
	Insert menu:
	
	  Break
	  Date and Time
	  Field
	  Symbol
	  Cross-reference
	  Index and Tables
	  File
	  Object
	
	Format menu:
	
	  Font
	  Paragraph
	  Bullets and Numbering
	  Columns
	  Tabs
	  Drop Cap
	  Text Direction
	  Change Case
	  Style
	
	Tools menu:
	
	  Look Up Reference
	  Protect Document
	
	Table menu:
	
	  All commands, except Hide/Show Gridlines
	
	Window menu:
	
	  All commands are available
	
	Help menu:
	
	  All commands are available
	
	The following commands are not available when you are in Design Mode even when
	the control object is not selected:
	
	Edit menu:
	
	  Paste As Hyperlink
	  Links
	
	Tools menu:
	
	  Protect Document
	
	Format menu:
	
	  Text Direction
