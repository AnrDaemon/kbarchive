## Q157141: PPT7: Pack and Go Never Prompts to Insert Disk into Drive A:

	Article: Q157141
	Product(s): Microsoft PowerPoint for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): kberrmsg kbui kbusage
	Last Modified: 16-APR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft PowerPoint for Windows 95, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	While attempting to run Pack And Go you may receive the following error
	message:
	
	  A:\ is not accessible.
	
	CAUSE
	=====
	
	When running Pack And Go and selecting the A drive as the drive to which to save
	the file, you are never prompted to insert a disk into Drive A. If there is no
	disk in Drive A, the error appears.
	
	RESOLUTION
	==========
	
	Insert a disk into Drive A and click Retry. It should now work as expected.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
