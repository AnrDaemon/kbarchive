## Q157849: WD97: Text Color in Text Box Not Changed to White with Dark Fill

	Article: Q157849
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you apply a black fill to a text box, the text color is not changed to
	white.
	
	NOTE: This functionality is different from Word 6.x and 7.x, where the text color
	was changed automatically to contrast the dark background.
	
	WORKAROUND
	==========
	
	To work around this problem, select the text and format it as white.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows.
