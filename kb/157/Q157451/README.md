## Q157451: No Driver for Cirrus Logic PBXGB-AA Adapter for DEC Alpha

	Article: Q157451
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kbdisplay
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Display tool in Control Panel supports only 16 colors for the Cirrus Logic
	PBXGB-AA video adapter.
	
	CAUSE
	=====
	
	This card is based on the Cirrus Logic 542x chip set; the driver supports only
	16 colors.
	
	RESOLUTION
	==========
	
	Contact Cirrus Logic for information regarding an updated driver for this video
	adapter.
