## Q157640: WD97: Insertion Point Disappears After Inserting Scrolling Text

	Article: Q157640
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbwdinternet
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	After you insert scrolling text, the insertion point disappears and, if you
	begin typing, the text is not displayed on the screen.
	
	CAUSE
	=====
	
	The scrolling text control has the focus after you close the Scrolling Text
	dialog box.
	
	WORKAROUND
	==========
	
	To work around this problem, click anywhere in the document to take the focus
	off the scrolling text control. You will now see the insertion point and the
	text you type appears on screen.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
