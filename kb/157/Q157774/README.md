## Q157774: WD97: Can't Open HTML Document with Nested Tables

	Article: Q157774
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): word97 kbwdinternet
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to open an HTML document that contains nested tables, the
	following message appears, and the document does not open:
	
	  The document contains nested tables that are too complex to represent.
	
	CAUSE
	=====
	
	Microsoft Word 97 does not support nested tables.
	
	WORKAROUND
	==========
	
	To work around this problem, open the document in another HTML editor, such as
	the Microsoft FrontPage Editor.
	
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows.
	
	This problem has been corrected in Word 2000.
