## Q157199: WD97: Memo Wizard Quits If Memo Title Is Over 255 Characters

{% raw %}

	Article: Q157199
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta kbtemplate
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you type a title that is longer than 255 characters in the title box of the
	Memo Wizard and then click Finish, the following message appears:
	
	  String is longer that 255 characters.
	
	When you click OK in the message dialog box, the wizard quits.
	
	WORKAROUND
	==========
	
	To work around this problem, either leave the title box blank or type a shorter
	title. When the wizard is finished, type the title in the document that the
	wizard generates.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
