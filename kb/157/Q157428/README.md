## Q157428: XCLN: Err Msg: Cannot Find the Exchng.hlp File

	Article: Q157428
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:4.0
	Operating System(s): 
	Keyword(s): kberrmsg kbusage
	Last Modified: 08-APR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Windows 3.x client, version 4.0 
	- Microsoft Exchange Windows 95/98 client, version 4.0 
	- Microsoft Exchange Windows NT client, version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to invoke Help from the Microsoft Exchange client, the following
	error message may be displayed:
	
	  Cannot open Help file
	
	CAUSE
	=====
	
	This error occurs either because a minimum install of the Microsoft Exchange
	client was done (this does not install help files), or because the Exchng.hlp
	file has been deleted from the system.
	
	RESOLUTION
	==========
	
	To get the help files, re-install the Microsoft Exchange client and select to
	install the help files.
	
	MORE INFORMATION
	================
	
	This behavior is by product design.
