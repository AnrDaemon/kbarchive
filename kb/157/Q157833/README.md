## Q157833: WD97: &quot;PAGE &#92;#...&quot; Displays in ScreenTip for Comments

	Article: Q157833
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbualink97 kbdta
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When View Field Codes is turned on, the ScreenTip for comments displays the
	following field code information in addition to the author's name and comment
	text:
	
	PAGE \# "Page: '#'"
	
	ScreenTips for footnotes, endnotes, and track changes do not display this field
	information.
	
	WORKAROUND
	==========
	
	To work around this problem and hide the field code information, turn off View
	Field Codes.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
	
	
	MORE INFORMATION
	================
	
	This field code is inserted automatically by Word before each comment in the
	comment pane. If you choose to print comments when you print the document, the
	page number is printed with the comment to identify its location in the
	document.
