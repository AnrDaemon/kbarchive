## Q157055: INF: ODBC Access Driver Does Not Support Create or Drop View

	Article: Q157055
	Product(s): Open Database Connectivity (ODBC)
	Version(s): WINDOWS:3.4
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-JAN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft ODBC Driver for Access, version 3.4 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The ODBC Access driver does not support Create View or Drop View.
	
	MORE INFORMATION
	================
	
	The Access driver uses the Jet engine. Jet does not support Create View or Drop
	View.
