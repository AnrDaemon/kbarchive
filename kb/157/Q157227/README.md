## Q157227: WD97: Page Border Not Visible Inserted in Normal, Outline Views

	Article: Q157227
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbualink97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you are in normal or outline view and you insert a page border, you do not
	see the border and Word does not switch to page layout view to display the
	border.
	
	WORKAROUND
	==========
	
	To view the page border, switch to either print preview or page layout view.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows.
