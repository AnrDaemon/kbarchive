## Q91028: TBMI2.COM Removed from AUTOEXEC.BAT in WFWG

	Article: Q91028
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	Whenever you use the Network setup option of Microsoft Windows Setup
	to add or remove Novell NetWare connectivity from Windows for
	Workgroups, TBMI2.COM is removed from the AUTOEXEC.BAT file. TBMI2.COM
	does not cause any problems with Windows for Workgroups and can be
	added back to the AUTOEXEC.BAT file.
	
	TBMI2.COM acts as an interface between NetWare and the standard mode
	Windows task switcher.
	
	Additional query words: 3.11 3.10 net ware
