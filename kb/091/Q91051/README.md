## Q91051: WFWG Logo Screen Has An Unexpected Blue Cursor

	Article: Q91051
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 04-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start Windows for Workgroups, you may see a blue cursor (mouse pointer)
	drawn in the left portion of the new Windows for Workgroups logo screen.
	
	CAUSE
	=====
	
	Qualitas' 386 to the Max memory manager causes this behavior in Windows for
	Workgroups. There are no other problems related to this behavior.

	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
