## Q91071: PC WFW: Users Don't Appear in the GAL After Upgrade

{% raw %}

	Article: Q91071
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 04-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, version 3.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you use Extensions for Windows for Workgroups to upgrade a Windows version
	3.1 for Workgroups postoffice to a version 3.0 Microsoft Mail for PC Networks
	postoffice, those mail accounts created before the Extend program was run are
	not displayed in the Global Address List (GAL).
	
	RESOLUTION
	==========
	
	To add those users to the Global Address List, manually export the user list
	from the Mail Administrator program (ADMIN.EXE) by selecting Config, Dir-Sync,
	Requestor, Export.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in version 3.0 of Extensions for
	Windows for Workgroups. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
