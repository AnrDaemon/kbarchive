## Q91096: Font Corruption in Ventura Publisher 4.0

{% raw %}

	Article: Q91096
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 03-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Ventura Publisher version 4.0 under Microsoft Windows or Microsoft Windows
	for Workgroups, you may notice some font corruption in a few Ventura Publisher
	fonts. Text may display in one or two columns only. This may occur when using
	these fonts: CG Times, Univers, and Univers CD. If you change to a TrueType font
	included with Windows for Workgroups, the problem no longer occurs.

	Microsoft Knowledge Base as it becomes available.
