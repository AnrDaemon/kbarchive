## Q91115: Windows Drivers for HP LaserJet IV Printer

	Article: Q91115
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	Neither Microsoft Windows nor Microsoft Windows for Workgroups ships with
	Hewlett-Packard (HP) LaserJet IV printer drivers. To obtain Windows drivers for
	the LaserJet IV printer, you must contact Hewlett-Packard.
	
	NOTE: Although initial shipments of LaserJet IV printers did not include Windows
	drivers, HP has since started including the drivers with LaserJet IV shipments.
	
	MORE INFORMATION
	================
	
	For more information, please call the Hewlett-Packard Customer Information
	Center.
	
	The LaserJet IV and the LaserJet IV printer drivers are manufactured by
	Hewlett-Packard, a vendor independent of Microsoft; we make no warranty, implied
	or otherwise, regarding these products' performance or reliability.
