## Q300970: Train Simulator: Joystick Does Not Work as Expected

	Article: Q300970
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Train Simulator, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you attempt to use a joystick in Microsoft Train Simulator, the joystick does
	not work as expected.
	
	CAUSE
	=====
	
	This behavior occurs because Train Simulator is designed to work only with mouse
	and keyboard input.
