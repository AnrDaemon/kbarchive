## Q300507: Passport Encryption Key Sent Through Express Mail

	Article: Q300507
	Product(s): Microsoft Developer Network
	Version(s): 1.3,1.4
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Passport, versions 1.3, 1.4 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Due to security concerns, the Passport encryption key for all production sites
	is sent by express mail.
	
	MORE INFORMATION
	================
	
	Passport encryption keys for the 2 test environments, Preproduction (PREP) and
	Beta Preproduction (Beta PREP), are sent by e-mail.
	
	For more information on Passport key issues or to make changes to existing sites,
	send e-mail to the following addresses:
	
	For PREP or Beta PREP sites:
	
	  patest@microsoft.com
	
	For production sites:
	
	  palive@microsoft.com
