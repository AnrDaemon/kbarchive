## Q297351: Last Number Dialed Appears Incorrectly In MSN Explorer 6.1

	Article: Q297351
	Product(s): The Microsoft Network
	Version(s): 6.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network version 6.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	An MSN Internet Access customer signs in to MSN Explorer (MSN Version 6.1) via a
	dial-up connection. The customer goes to Connection Settings and sees the
	section that reads "modem dialing xxxxxxx." The number shown may not be the
	actual number dialed. The number shown is always the first number in the list of
	numbers they selected to use.
	
	CAUSE
	=====
	
	This is a known issue with MSN Explorer (MSN Version 6.1).
	
	RESOLUTION
	==========
	
	This issue does not affect the numbers dialed or other functionality of MSN
	Explorer in any way. This issue will be resolved in a future update to MSN
	Explorer.

	================
