## Q297047: Microsoft Office 2000 Expert Companion Comments And Corrections

	Article: Q297047
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 21-DEC-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Office 2000 Expert Companion ISBN 0-7356-0527-0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Microsoft Office 2000 Expert Companion,
	ISBN 0-7356-0527-0.
	
	The following topics are covered:
	
	- CD-ROM: Missing Sample Files
	
	MORE INFORMATION
	================
	
	CD-ROM: Missing Sample Files
	----------------------------
	
	The \Expert\CH10 and \Expert\CH14 folders on the companion CD are missing some
	sample files referenced in Chapters 10 and 14 of the book. The missing files are
	available in the self-extracting executable file Samples.exe.
	
	Please note that when opening the Price.doc and Main.doc files, you need to
	manually specify the data source file (Toys.mdb) in order to open the files
	successfully.
	
	The following file is available for download from the Microsoft Download Center:
	
	  Samples.exe
	  (http://download.microsoft.com/download/MSPressPub/supp/1.0/NT5/EN-US/Samples.exe)
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
	
	Microsoft used the most current virus detection software available on the date of
	posting to scan this file for viruses. Once posted, the file is housed on secure
	servers that prevent any unauthorized changes to the file.
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
