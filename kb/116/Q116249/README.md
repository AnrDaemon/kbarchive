## Q116249: Object Packager Font Is Unreadable in Windows 95

	Article: Q116249
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you upgrade Windows 3.1, or Windows for Workgroups 3.1 or 3.11, to Windows
	95, the fonts in Object Packager may be unreadable.
	
	CAUSE
	=====
	
	Object Packager is not available in the floppy disk version of Windows 95. If
	you install the floppy disk version of Windows 95, Object Packager is retained
	after a successful upgrade. However, when you run this tool, the font for the
	following headings may be unreadable:
	
	- Appearance
	
	- Content
	
	- Description
	
	- Insert Icon
	
	- Picture
	
	- View
	
	RESOLUTION
	==========
	
	To resolve this problem, obtain the Windows 95 version of Object Packager.
	
	NOTE: Object Packager is included in the CD-ROM version of Windows 95, but not in
	the floppy disk version. If you are using the floppy disk version of Windows 95
	and you want to use Object Packager, please see the following article in the
	Microsoft Knowledge Base for information about the availability of Object
	Packager:
	
	  Q135315 CD-ROM Extras for Microsoft Windows 95 Upgrade
