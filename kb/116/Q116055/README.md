## Q116055: Baseball: Help Stays Open After Baseball Closes

	Article: Q116055
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1994 edition,1995-1996 edition; WINDOWS:
	Operating System(s): 
	Keyword(s): kbbuglist
	Last Modified: 09-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Complete Baseball for Windows, version 1994 edition 
	- Microsoft Complete Baseball Guide for Windows, 1995 edition 
	- Microsoft Complete NBA Basketball for Windows 1995-1996 edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you launch the application's Help tool and then quit the application, the
	Help tool will remain open after the application has closed. You must close the
	Help application separately.
	
	Microsoft has confirmed this to be a problem in products listed at the top of
	this article. We are researching this problem and will post new information here
	in the Microsoft Knowledge Base as it becomes available.
