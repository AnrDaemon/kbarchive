## Q116165: Baseball: List and Filenames of Video Clips in Baseball

{% raw %}

	Article: Q116165
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1994 edition
	Operating System(s):

	The information in this article applies to:
	
	- Microsoft Complete Baseball for Windows, version 1994 edition

	SUMMARY
	=======
	
	The following are the .avi (video) clips that are included with Complete
	Baseball:
	
	  Year  Description                          Filenames
	  -------------------------------------------------------
	
	  1993  Joe Carter's home run                T162361A.AVI
	  1991  Nolan Ryan's seventh no hitter       T162218A.AVI
	  1988  Kirk Gibson's home run               T161761A.AVI
	  1985  Pete Rose's 4192nd hit               T161668A.AVI
	  1983  George Brett's pine tar incident     T161927A.AVI
	  1977  Reggie Jackson's third home run      T162289A.AVI
	  1975  Carlton Fisk's home run              T161525A.AVI
	  1974  Henry Aaron's 715th home run         T162273A.AVI
	  1960  Pirates win the pennant              T162104A.AVI
	  1956  Jackie Robinson                      T162052A.AVI
	  1951  Bobby Thompson's "Giant's Win"       T162034A.AVI
	  1945  Pete Gray, one-armed ballplayer      T162020A.AVI
	
	All of these videos are located in the VIDEO directory on the Complete Baseball
	CD. You can view them in the Almanac in the Season Summary for the respective
	year.
