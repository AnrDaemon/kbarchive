## Q116400: Comprehensive List of Video for Windows Documentation Errors

	Article: Q116400
	Product(s): Microsoft PowerPoint for Windows
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Video for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains information about documentation errors in the Microsoft
	Video for Windows version 1.0 "User's Guide." The following topics are covered:
	
	- Valid File Formats List for BitEdit Is Misleading
	
	MORE INFORMATION
	================
	
	Valid File Formats List for BitEdit Is Misleading
	-------------------------------------------------
	
	Page 2 in Chapter 5 of the "User's Guide" states that BitEdit imports the
	following graphic file formats:
	
	  PLT AutoCad Import
	  CGM Computer Graphics Metafile
	  WPG DrawPerfect
	  EPS Encapsulated PostScript
	  HGL HP Graphic Language
	  DRW Micrografx Designer Plus/Draw
	  WMF Microsoft Windows Metafile
	  TIF Tagged Image File Format
	
	These file formats (and others) are available only if an import graphic filter
	has been added to Windows by another application.
