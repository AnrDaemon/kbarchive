## Q116379: Resource Creator &amp; List of File Types for FoxPro for Mac

	Article: Q116379
	Product(s): Microsoft FoxPro
	Version(s): MACINTOSH:2.5b,2.5c
	Operating System(s): 
	Keyword(s): 
	Last Modified: 03-AUG-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft FoxPro for Macintosh, versions 2.5b, 2.5c 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The information below lists the file types and the associated extensions of all
	FoxPro for Macintosh files. All of these files have the creator FOXX. This
	information is also found in the "Creator File Types" topic in the online Help
	file.
	
	MORE INFORMATION
	================
	
	Creator and File Types
	----------------------
	
	The following list includes the file types used by FoxPro for Macintosh. All
	FoxPro for Macintosh files have the creator FOXX.
	
	  File Type  FoxPro Files                      File Extension
	  -----------------------------------------------------------
	
	  F+DB       Table                                 .DBF
	  F+DT       FoxBASE+ and FoxPro memo files        .DBT,.FPT
	  F+FM       Format file                           .FMT
	  F+FR       Screen file                           .SCX
	  F+IX       Single-entry index file               .IDX
	  F+LB       Label file                            .LBX
	  F+ME       Memory variable save file             .MEM
	  F+PR       FoxPro program                        .PRG
	  F+RP       Report file                           .FRX
	  F+VU       View file                             .VUE
	  FACT       FoxDoc action diagram                 .ACT
	  FAPP       Generated application                 .APP
	  FBAK       File backup                           .BAK
	  FCDX       Compound index file                   .CDX
	  FDOC       FoxDoc reports                        .DOC
	  FERR       Compilation error file                .ERR
	  FEXE       Executable program                    .EXE
	  FFKY       Macro files                           .FKY
	  FFRT       Report memo                           .FRT
	  FFXP       Compiled FoxPro program               .FXP
	  FINT       Code page & collation sequence file   .INT
	  FLBT       Label memo                            .LBT
	  FMNT       Menu memo                             .MNT
	  FMNX       Menu file                             .MNX
	  FMPR       Generated menu program                .MPR
	  FMPX       Compiled menu program                 .MPX
	  FPJT       Project memo                          .PJT
	  FPJX       Project                               .PJX
	  FPRX       Compiled format                       .PRX
	  FQPR       Generated query program               .QPR
	  FQPX       Compiled query program                .QPX
	  FSCT       Screen memo                           .SCT
	  FSPR       Generated screen program              .SPR
	  FSPX       Compiled screen program               .SPX
	  FTBK       Memo file backup                      .TBK
	  FTMP       Temporary file                        .TMP
	  FWIN       Window save file                      .WIN
	  HELP       Graphical Help file                   .HLP
	  LIBR       Macintosh library                     .MLB
	  TEXT       Text file                             .TXT
	
	Additional query words: FoxMac foxhelp.dbf foxhelp.hlp on- line akz
