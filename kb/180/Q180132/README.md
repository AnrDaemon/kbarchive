## Q180132: Kids Plus!: You Cannot Remove a Family Member in Protect It!

	Article: Q180132
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbenv kbtool
	Last Modified: 04-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Plus! for Kids, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You cannot remove a family member using Protect It!.
	
	CAUSE
	=====
	
	The family members (users) that you add in Protect It! are added as user
	profiles in Microsoft Windows.
	
	RESOLUTION
	==========
	
	To remove a family member if you are using Microsoft Windows 95, install
	Microsoft Internet Explorer version 4.0. When you install Internet Explorer, the
	Users tool is added to Control Panel. You can use this tool to remove a family
	member's user profile in Windows 95, which also removes the family member from
	Protect It!.
	
	NOTE: If you are using Microsoft Windows 98, Internet Explorer 4.0 is already
	installed.
	
	Internet Explorer version 4.0 is available free of charge from the following
	Microsoft Web site:
	
	  http://www.microsoft.com/ie
	
	After you install Internet Explorer, you can remove a family member by following
	these steps:
	
	1. Click Start, point to Settings, and then click Control Panel.
	
	2. Double-click Users.
	
	3. On the User List tab, click the name of the family member you want to remove,
	  and then click Delete.
	
	4. When you are prompted to confirm the deletion of the user, click Yes.
	
	5. Click Close, and then close Control Panel.
	
	MORE INFORMATION
	================
	
	For more information about how to delete user profiles in Windows, please see
	the following article in the Microsoft Knowledge Base:
	
	  Q156826 How to Disable and Delete User Profiles
