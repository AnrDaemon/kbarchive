## Q180460: Rumba v5.1 May Cause Stop 0x0000007f

{% raw %}

	Article: Q180460
	Product(s): Microsoft Windows NT
	Version(s): winnt:3.51,4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation versions 3.51, 4.0 
	- Microsoft Windows NT Server versions 3.51, 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Using Rumba v5.1 by Wall Data may cause the following STOP error message:
	
	  STOP 0x0000007f (0x00000008, 0x00000000, 0x00000000, 0x00000000)
	  UNEXPECTED_KERNEL_MODE_TRAP
	
	NOTE: The parameters should be as listed for this particular issue.
	
	RESOLUTION
	==========
	
	Contact Wall Data for an updated patch for this product.
	
	
	NOTE: This update requires the English version of RUMBA.

	of Microsoft; we make no warranty, implied or otherwise, regarding these
	products' performance or reliability.
