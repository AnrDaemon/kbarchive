## Q180523: Ifilttst.exe &amp; Ifilttst.ini Files Missing from IFilter Document

	Article: Q180523
	Product(s): Internet Information Server
	Version(s): WINNT:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 31-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Internet Information Server 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The January 1998 MSDN Library and Platform Software Development Kit (SDK)
	contains documentation on programming with the IFilter interface. The
	documentation mentions a tool that will verify an IFilter implementation. The
	tool consists of two files: Ifilttst.exe and Ifilttst.ini. These files were not
	included with the documentation.
	
	RESOLUTION
	==========
	
	You can get the missing files by downloading the IFilter SDK from the following
	location:
	
	  http://msdn.microsoft.com/library/psdk/indexsrv/ixufilt_9279.htm
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft Internet
	Information Server version 4.0.
	
	MORE INFORMATION
	================
	
	Documentation on the IFilter interface test tool is located in the Platform SDK
	documentation at the following location:
	
	  
	
	  Internet/Internet/Extranet Services
	   Indexing Service
	       Indexing Service Filter Implementation
	           Filter Examples
	               IFilter Test Suite
