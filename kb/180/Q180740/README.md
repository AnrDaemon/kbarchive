## Q180740: PPT7: Embossed Graphics in Converted File Appear as Solid Color

	Article: Q180740
	Product(s): Microsoft PowerPoint for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): kbgraphic kbinterop kbdta kbconversion
	Last Modified: 16-APR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft PowerPoint for Windows 95, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you open a Microsoft PowerPoint 4.0 presentation in PowerPoint for Windows
	95 version 7.0, a graphic that has an embossed shadow is displayed as a colored
	box.
	
	CAUSE
	=====
	
	The functionality of the embossed shadow format is different in the versions of
	PowerPoint listed at the beginning of this article.
	
	WORKAROUND
	==========
	
	To work around this behavior, change or remove the embossed shadow from the
	picture by using one of the following methods.
	
	Method 1: Remove the Embossed Shadow in the Destination File
	------------------------------------------------------------
	
	1. Open the PowerPoint 4.0 presentation.
	
	2. Select the picture that has the embossed shadow.
	
	3. On the Format menu, click Shadow.
	
	4. Select No Shadow to remove the embossed shadow.
	
	Method 2: Remove the Embossed Shadow from the Source
	----------------------------------------------------
	
	1. Open the PowerPoint 4.0 presentation.
	
	2. Click the picture that has the embossed shadow.
	
	3. On the Format menu, click Shadow.
	
	4. In the Color list, select an option other than Embossed, or click No Shadow
	  to remove the embossed shadow.
