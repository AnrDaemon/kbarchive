## Q174532: 16-bit Application Cannot Select a Network Drive

	Article: Q174532
	Product(s): Microsoft Windows NT
	Version(s): WinNT:4.0
	Operating System(s): 
	Keyword(s): kbnetwork
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click Open on the File menu of a 16-bit application, the available
	drives listed may not include network drives that are mapped to a drive letter.
	
	CAUSE
	=====
	
	This problem is a timing issue caused by two threads trying to access the same
	function.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 4.0. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
