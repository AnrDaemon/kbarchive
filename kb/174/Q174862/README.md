## Q174862: XCLN: Duplicate Message Created with Clean Mailbox Utility

	Article: Q174862
	Product(s): Microsoft Exchange
	Version(s): WinNT:5.5
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 08-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Duplicate deleted messages will be created if you send a message to yourself,
	receive the message, and then run the Clean Mailbox utility to delete all
	messages in your mailbox.
	
	CAUSE
	=====
	
	The messages in the Inbox and in the Sent Items folder are moved to the Deleted
	Items folder. This is by product design.
