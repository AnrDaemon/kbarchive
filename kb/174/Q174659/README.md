## Q174659: Start Button, Taskbar, and Desktop Icons Missing After Logon

	Article: Q174659
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kbenv kbui
	Last Modified: 30-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Internet Explorer version 4.0 for Windows NT 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you log on to Windows NT, the Start button, taskbar, and desktop icons may
	not be displayed. The only visible object on the screen may be the mouse
	pointer.
	
	CAUSE
	=====
	
	This behavior can occur if you restore the System and Software registry keys
	from an Emergency Repair Disk (ERD) that was created before you installed
	Internet Explorer 4.0 with the Windows Desktop Update component.
