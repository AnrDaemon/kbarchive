## Q174747: XWEB: OWA Calendar Displays Only 2 Months in Date Picker Field

{% raw %}

	Article: Q174747
	Product(s): Microsoft Exchange
	Version(s): 5.5
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook Web Access, version 5.5 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Calendar view of Microsoft Outlook Web Access can only display two months in
	the Date Picker field. This is different from the Microsoft Outlook client,
	which allows you to customize the Date Picker field to display one or more
	months.
	
	MORE INFORMATION
	================
	
	This behavior is by design.
