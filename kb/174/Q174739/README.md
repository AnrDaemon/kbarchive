## Q174739: XWEB: Empty Rectangles Appear in Outlook Web Access

	Article: Q174739
	Product(s): Microsoft Exchange
	Version(s): WinNT:5.5
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook Web Access, version 5.5 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are accessing an Exchange Server computer by using Outlook Web Access
	and Netscape Navigator version 4.0, empty rectangles appear in various areas of
	the screen.
	
	STATUS
	======
	
	This problem occurs when you use Microsoft Outlook Web Access version 5.5 and
	Netscape Navigator 4.0. This problem will not occur if you use Netscape
	Navigator 4.01 or later.
	
	MORE INFORMATION
	================
	
	These rectangles appear in various areas of the screen but have no detrimental
	effect on the Outlook Web Access client.
