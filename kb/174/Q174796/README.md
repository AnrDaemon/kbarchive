## Q174796: ZAK: Internet Explorer Launches Taskbar

	Article: Q174796
	Product(s): Microsoft Windows NT
	Version(s): WinNT:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 20-JAN-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Zero Administration Kit for Windows NT Workstation 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use Microsoft Zero Administration Kit (ZAK) for Windows NT Workstation
	4.0, Internet Explorer (IE) launches a taskbar when selecting Open History
	Folder or pressing Ctrl+H. IE will also launch a taskbar when you select Read
	Mail or Read News.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Zero Administration
	Kit for Windows NT Workstation 4.0. We are researching this problem and will
	post new information here in the Microsoft Knowledge Base as it becomes
	available.
