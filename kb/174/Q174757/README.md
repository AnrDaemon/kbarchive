## Q174757: XADM: INS Service Is Not Removed After Removing All Newsfeeds

	Article: Q174757
	Product(s): Microsoft Exchange
	Version(s): WinNT:5.0,5.5
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 08-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 5.0, 5.5 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	The Internet News Service (INS) is not removed when you remove all the newsfeeds
	from a Microsoft Exchange Server computer.
	
	CAUSE
	=====
	
	This is by design.
	
	RESOLUTION
	==========
	
	To remove the INS, run the Microsoft Exchange Server Setup program and select
	the Remove All option.
