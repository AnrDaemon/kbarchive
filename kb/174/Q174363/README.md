## Q174363: FS98: CH Force FX Does Not Work with Flight Simulator

	Article: Q174363
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kb3rdparty kbhw kbtlc kbimu kbHardware
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator 98 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The CH Force FX joystick does not provide force feedback in games, such as
	Flight Simulator 98, that are designed for Windows 95/98 and use DirectX version
	5.0.
	
	CAUSE
	=====
	
	The updated DirectX version 5.0 drivers for the CH Force FX have not been
	installed on the computer.
	
	RESOLUTION
	==========
	
	Download and install the latest DirectX version 5.0 drivers that are available
	for the CH Force FX joystick, or contact CH Products for more information.
	
	You can download updated beta drivers for the CH Force FX joystick and DirectX
	version 5.0 from the following Web site: http://www.force-feedback.com
	
	MORE INFORMATION
	================
	
	CH Force FX is manufactured by CH Products, a vendor independent of Microsoft;
	we make no warranty, implied or otherwise, regarding this product's performance
	or reliability.
