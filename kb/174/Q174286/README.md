## Q174286: Cannot Close Regional Settings Properties Dialog Box

{% raw %}

	Article: Q174286
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): kbtool kbui win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start the Regional Settings Properties dialog box from Windows 95 Help,
	you may not be able to close the Regional Settings Properties dialog box if you
	make any changes. In addition, the dialog box may not be refreshed correctly.
	
	RESOLUTION
	==========
	
	To close the Regional Settings Properties dialog box, follow these steps:
	
	1. Press CTRL+ALT+DELETE to open the Close Program dialog box.
	
	2. Click Regional Settings Properties, and then click End Task.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
