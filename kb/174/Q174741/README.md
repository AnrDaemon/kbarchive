## Q174741: XFOR: Exchange Dirsync Server Allows Empty Admin Mailbox

{% raw %}

	Article: Q174741
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5
	Operating System(s): 
	Keyword(s): kbusage exc5 exc55
	Last Modified: 18-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you are configuring an Exchange Server computer as a dirsync server, it is
	possible to select the Copy Administrator on Outgoing Messages option and
	Forward Incoming Dirsync Messages to Administrator option, even though there is
	no dirsync administrator selected.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Exchange Server version 5.5. We
	are researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
	
	
	MORE INFORMATION
	================
	
	Dirsync will continue to function as normal although there is no administrator
	selected.
	
	Additional query words: DXA dir sync
