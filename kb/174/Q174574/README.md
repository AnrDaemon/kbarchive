## Q174574: SMS: Inventory Reports Incorrect Macintosh Processor

	Article: Q174574
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.2
	Operating System(s): 
	Keyword(s): kbenv kbInventory smsinv
	Last Modified: 25-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	A Macintosh Systems Management Server client may report incorrect processor
	information in its hardware inventory. For example, a Macintosh client with a
	PowerPC 604 processor may instead report the processor name in hardware
	inventory as PowerPC 601. In addition, certain Macintosh client systems may not
	have the processor name reported at all.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.2. This problem was corrected in the latest Microsoft Systems
	Management Server 1.2 U.S. Service Pack. For information on obtaining the
	service pack, query on the following word in the Microsoft Knowledge Base
	(without the spaces): S E R V P A C K The third-party products discussed here
	are manufactured by vendors independent of Microsoft; we make no warranty,
	implied or otherwise, regarding these products' performance or reliability.
