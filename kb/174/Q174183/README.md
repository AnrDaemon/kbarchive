## Q174183: XADM: Cannot Get Member List of DLs in Offline Address Book

	Article: Q174183
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 17-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	When you are using the Offline Address Book to get details of a Distribution List
	while working off-line, the DL members are not listed. This happens even if the
	OAB is downloaded with full details.
	
	CAUSE
	=====
	
	This is by design.
