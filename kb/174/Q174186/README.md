## Q174186: Files the Setup Wizard Installs

	Article: Q174186
	Product(s): Microsoft FoxPro
	Version(s): WINDOWS:5.0,5.0a
	Operating System(s): 
	Keyword(s): kbfile kbsetup kbnokeyword kbvfp500 kbDSupport
	Last Modified: 11-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual FoxPro for Windows, versions 5.0, 5.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The files for this article provide information on the components installed by
	the Visual FoxPro 5.0 and 5.0a Setup Wizard under various options chosen by the
	user.
	
	MORE INFORMATION
	================
	
	The following files are available for download from the Microsoft Download
	Center:
	
	  SWFiles.exe
	  (http://download.microsoft.com/download/vfox50/list/1/W9X/EN-US/SWFiles.exe)
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
	
	Microsoft used the most current virus detection software available on the date of
	posting to scan this file for viruses. Once posted, the file is housed on secure
	servers that prevent any unauthorized changes to the file.
	
	Setupwiz.xls and Setwiz5a.xls are Microsoft Excel 97 and 5.0/95 workbooks that
	contain a list of files installed by the Visual FoxPro Setup Wizard. This is an
	accurate list of files for the most current version of Visual FoxPro. However,
	if a newer version of the Setup Wizard for Visual FoxPro 5.0 becomes available,
	these files may change. The columns listed across the top the spreadsheet
	indicate the various options you might choose. The specific files are listed
	along the right-hand side of the spreadsheet. The asterisks indicate the
	installation of a file under that particular option.
	
	Additional query words: SWFiles Setup wizard
