## Q69335: LogiTech's CATCHWORD 1.10 Intelligent Character Recognition

{% raw %}

	Article: Q69335
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	CATCHWORD version 1.10 is an Intelligent Character Recognition MS-DOS
	application by Logitech, Inc. According to Logitech, CATCHWORD runs only with
	the Logitech ScanMan Scanner hardware. Logitech recommends that you do not run
	this application from Microsoft Windows.
	
	MORE INFORMATION
	================
	
	For more information, contact Logitech technical support.
	
	The product included here is manufactured by a vendor independent of Microsoft;
	we make no warranty, implied or otherwise, regarding this product's performance
	or reliability.
