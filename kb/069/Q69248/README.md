## Q69248: Using the MSC Technologies Serial and Bus Mouse Under Windows

{% raw %}

	Article: Q69248
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 02-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Use the following information below to configure the MSC Serial or Bus Mouse
	properly for Microsoft Windows 3.0.
	
	MORE INFORMATION
	================
	
	To operate the MSC Bus Mouse with Microsoft Windows 3.0, install the MCS mouse
	driver from Windows Setup.
	
	To Install the MSC Bus Mouse Driver
	-----------------------------------
	
	1. From the MS-DOS prompt, change to the WINDOWS directory.
	
	2. Type "SETUP" (without the quotation marks).
	
	3. Select Mouse and then select Other.
	
	4. Follow the instructions for installing the mouse driver.
	
	NOTE: You may have to contact MSC Technologies to order a Microsoft Windows 3.0
	compatible bus mouse driver if your mouse did not come with one.
	
	Using MSC Technologies Serial Mouse
	-----------------------------------
	
	To configure the MSC Serial Mouse properly for Windows 3.0, you need to select
	one of the following choices from the System Settings dialog box, depending on
	your configuration:
	
	- Mouse Systems (or VisiOn) connected to COM1
	
	- Mouse Systems (or VisiOn) connected to COM2
