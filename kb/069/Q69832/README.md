## Q69832: Emulation Modes of Facit Printers with Windows

	Article: Q69832
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	According to Facit, the following list outlines the available Facit printers and
	their emulation modes. Consult the printer documentation to configure the
	printer for a particular emulation mode.
	
	If the printer exhibits problems in the particular emulation mode given, the
	output should be tested on the actual Epson, IBM, or Hewlett-Packard (HP)
	printer listed. If the actual Epson, IBM, or HP printer processes the output
	correctly, the printer may not be emulating correctly and you should contact
	Facit.
	
	MORE INFORMATION
	================
	
	Printer           Primary Emulation       Optional Emulation
	-------           -----------------       -------------------
	
	B1200             IBM Proprinter II
	                 Epson FX-80
	
	B2100, B2150      IBM Proprinter
	                 Epson FX-800
	
	B2400, B2450      IBM Proprinter X24
	                 Epson LQ-850
	
	B3100, B3150      IBM Proprinter
	                 Epson FX/JX
	
	B3450             IBM Proprinter XL       IBM Proprinter
	                 Epson LQ-2500
	                 Diablo 630
	
	B3550             IBM Proprinter
	                 Epson FX/JX
	
	DOCUMATE          Facit 4528              IBM Proprinter
	
	P6060             HP LaserJet Series II   IBM Proprinter

	implied or otherwise, regarding this product's performance or reliability.
