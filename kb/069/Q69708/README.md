## Q69708: Debug Doesn't Accept SAL Mnemonic

	Article: Q69708
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:2.x,3.x,4.x,5.x,6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 2.11, 3.1, 3.2, 3.21, 3.3, 3.3a, 4.0, 4.01, 5.0, 5.0a, 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The SAL mnemonic, which is equivalent to the SHL mnemonic in the 80x86
	instruction set, is not accepted by the Debug editor in MS-DOS. Using the
	mnemonic results in Debug reporting an error. The SHL mnemonic must be used
	instead.
	
	Additional query words: 6.22 2.11 3.00 3.20 3.30 3.30a 4.00 4.01 4.01a 5.00 5.00a 6.00 6.20
