## Q69379: COMPAQ CDROM Adapters and Windows 3.00

	Article: Q69379
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you use COMPAQ SLT 386s/20 and LTE 386s/20 CDROM adapters with Windows 3.0
	in 386 enhanced mode, the adapters require the COMPAQ CDROM driver version 1.01
	or later.
	
	If you are using version 1.00 of the driver, contact your COMPAQ dealer to obtain
	the latest version.
	
	MORE INFORMATION
	================
	
	The COMPAQ products included here are manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding these products'
	performance or reliability.
