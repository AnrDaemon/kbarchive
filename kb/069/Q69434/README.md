## Q69434: Pagemaker Err Msg: Memory Manager Failure

{% raw %}

	Article: Q69434
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Aldus Pagemaker 3.0 may display the following error message if Microsoft Windows
	3.0 is running in standard or enhanced mode:
	
	  MEMORY MANAGER FAILURE
	
	The system then stops responding (hangs, or locks up).
	
	CAUSE
	=====
	
	Aldus Pagemaker version 3.0 or earlier was designed to run with Windows versions
	2.x and does not operate correctly with Windows 3.0 running in standard or
	enhanced mode.
	
	WORKAROUND
	==========
	
	You should run this version of Pagemaker in real mode only.
	
	MORE INFORMATION
	================
	
	Aldus reports that Pagemaker version 3.01 operates correctly with Windows 3.0
	running in standard or enhanced mode.
	
	For additional information, contact Aldus technical support.

	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
