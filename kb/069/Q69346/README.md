## Q69346: BUG: MASM 5.10 Installation Asks for Disk &quot;OS/2 Program&quot;

{% raw %}

	Article: Q69346
	Product(s): Microsoft Macro Assembler
	Version(s): 5.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 04-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Macro Assembler (MASM), version 5.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Microsoft Macro Assembler (MASM) version 5.1 installation program RUNME.EXE
	may prompt you to insert a disk named "OS/2 Program."
	
	There is no disk labeled "OS/2 Program" in the MASM 5.1 distribution disks
	package; the actual disk the installation program is looking for is labeled "MS
	OS/2 Tools."
	
	RESOLUTION
	==========
	
	When the installation program asks for the disk named "OS/2 Program" install the
	disk labeled "MS OS/2 Tools."
	
	Additional query words: 5.10
