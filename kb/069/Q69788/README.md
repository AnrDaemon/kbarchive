## Q69788: ASD: 747 Panel Cannot Be Moved or Sized

{% raw %}

	Article: Q69788
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	Due to the complexity of the 747 enhanced panel, it cannot be moved or sized;
	however, the original Flight Simulator control panel (that is, the Cessna panel)
	can be moved and sized.
	
	Lack of this feature is by design in Aircraft & Scenery Designer version
	4.00.
