## Q69244: Calculator Err Msg: List of Error Messages

	Article: Q69244
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Below is a list of the possible error messages you may receive from the
	Calculator application (CALC.EXE) included with Microsoft Windows version 3.00.
	
	MORE INFORMATION
	================
	
	+-----------------------------------------------------------------------------------------------------------+
	| 1 | Cannot divide by zero                                                                                 | 
	+-----------------------------------------------------------------------------------------------------------+
	| 2 | Cannot open Clipboard.                                                                                | 
	+-----------------------------------------------------------------------------------------------------------+
	| 3 | Insufficient memory for data; close one or more Windows
	  Applications to increase available memory. | 
	+-----------------------------------------------------------------------------------------------------------+
	| 4 | Invalid input for function                                                                            | 
	+-----------------------------------------------------------------------------------------------------------+
	| 5 | Result is too large for display                                                                       | 
	+-----------------------------------------------------------------------------------------------------------+
	| 6 | Result is too small for display                                                                       | 
	+-----------------------------------------------------------------------------------------------------------+
	| 7 | Result of function is undefined                                                                       | 
	+-----------------------------------------------------------------------------------------------------------+
	
	Additional query words: 3.00 3.0 3.0a 3.00a err msg errmsp
