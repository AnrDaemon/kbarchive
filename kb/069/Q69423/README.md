## Q69423: Printing from Windows Notepad Using the Word Wrap Option

	Article: Q69423
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	Microsoft Windows Notepad supports Word Wrap. When enabled, the Word Wrap
	feature wraps text at the right window border and not at the right-margin
	page setting. When you print from Notepad, the output will be dependent on
	the left- and right-margin page settings, not the Word Wrap window borders.
	Text will be aligned between the left and right margin settings, which are
	set from the Page Setup under the File menu.
	
	To print your Notepad document as it appears on screen between the window
	borders, press ENTER at the end of each line before it scrolls past the
	right border.
	
	Another alternative is to use Windows Write.
