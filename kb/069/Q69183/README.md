## Q69183: Okidata Laser 400 Emulates HP Series II Printer

	Article: Q69183
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	
	The Okidata Laser 400 printer is not supported under Microsoft Windows operating
	system version 3.0. However, this printer can use the Hewlett-Packard (HP)
	LaserJet Series II printer driver as an emulation.
	
	To obtain phone numbers for Okidata, query on the following words in the
	Microsoft Knowledge Base:
	
	  "Okidata" (without the quotation marks) and "phonebook" (without the
	  quotation marks)
	
	
	Additional query words: 3rdparty WIN30 3.0 3.00a 3.0a OL400 OL 400 LED page pageprinter
