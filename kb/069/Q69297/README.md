## Q69297: Windows Err Msgs: Paintbrush Message List

{% raw %}

	Version(s): WINDOWS:3.0,3.0a

	Last Modified: 08-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Below is a list of the possible error messages you may receive from the
	Paintbrush application (PBRUSH.EXE) included with Microsoft Windows version
	3.00.
	
	NOTE: The "(null)" and "0%" symbols in a message indicate a variable string used
	by the application.
