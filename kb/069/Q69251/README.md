## Q69251: Citizen Printer Emulation Modes for Microsoft Windows 3.0

{% raw %}

	Article: Q69251
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The following is a list of Citizen America printers and the emulation modes of
	each printer. Emulation information was obtained from Citizen America.
	
	Printer                 Driver Emulation
	------------------------------------------------------------------------
	
	HSP 500/550             Epson FX/EX or IBM ProPrinter XL
	                       Color EX 800
	
	200 GX                  Epson FX 80/850 or IBM ProPrinter III
	                       Color=EX 800
	
	140/145 GSX             Epson LQ series or IBM ProPrinter X24
	                       Color = Epson LQ 2500
	
	MSP Series (10 - 55)    Epson EX or IBM ProPrinter
	                       Color = EX
	
	TRIBUTE 124/224         Toshiba P351, Qume Sprint, Diablo 630
	                       Color = Toshiba 351 - C (124 only)
	
	PREMIERE 35             Diablo 630, Qume Sprint, NEC 3550
	
	120 D/180-D             Epson FX, IBM Graphics Printer, TTY Type
	                       printer
	
	MORE INFORMATION
	================
	
	Printer        Primary Emulation     Secondary Emulation
	-----------------------------------------------------------
	
	GSX SERIES     Epson LQ2500          IBM Graphics
	GS200          Epson FX850,EX800     IBM ProPrinter
	120D           Epson FX80            IBM ProPrinter
	140            Epson FX80            IBM ProPrinter
	180D           Epson FX80            IBM ProPrinter
	MSP            Epson FX80            IBM ProPrinter
	5200           Epson FX80            IBM ProPrinter
	5800           Epson FX80            IBM ProPrinter
	PREMIER35      NEC 3550, QUMES II,   none
	              Diablo 6300
	Tribute        Toshiba P35,          none
	              Diablo 6300
	
	Printer mode is set with internal dip switches or key combinations on the front
	of the printer.
	
	The products included here, Citizen printers, are manufactured by vendors
	independent of Microsoft; we make no warranty, implied or otherwise, regarding
	these product's performance or reliability.
