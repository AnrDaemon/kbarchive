## Q69092: Paintbrush Prints Blank Sheet on HP Printers

	Article: Q69092
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	When you print an image from Windows Paintbrush on a Hewlett-Packard
	(HP) LaserJet series printer, be sure the Quality section is set to
	"Proof" in the File menu's Print dialog box.
	
	If "Draft" is selected, the printer will print a blank sheet of
	paper. This is also true when using the Deskjet family of printers.
