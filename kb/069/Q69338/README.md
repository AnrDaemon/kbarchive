## Q69338: Mouse Pointer Disappears in Quattro Pro Graph Annotations

	Article: Q69338
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The mouse pointer may disappear when you annotate a graph in Quattro Pro. To
	bring the mouse back:
	
	1. Switch back to program manager by pressing ALT+TAB.
	
	2. Switch back to the Quattro Pro graph by pressing ALT+TAB.
	
	The pointer should appear again.
	
	MORE INFORMATION
	================
	
	This problem only seems to occur when running Microsoft Windows 3.0 in 386
	enhanced mode. Adjusting the Quattro Pro PIF settings does not affect the
	mouse.
