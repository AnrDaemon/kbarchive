## Q159658: Mac Encarta 97 Err Msg: Required Files Are Missing...

	Article: Q159658
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 97 Encyclopedia Deluxe for Macintosh 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start Microsoft Encarta 97 for Macintosh, you may receive the following
	error message:
	
	  Required Files Are Missing. Please reinstall from Encarta 97 Disc 1.
	
	CAUSE
	=====
	
	This error message can occur for two reasons:
	
	- The Updates folder has been deleted or removed from within the Encarta 97
	  Encyclopedia folder.
	
	  -or-
	
	- A file with a .ewl extension (for example, catlnk.ewl) has been deleted or
	  removed from the Updates folder that resides inside the Encarta 97
	  Encyclopedia folder.
	
	  NOTE: This file could also be corrupted.
	
	RESOLUTION
	==========
	
	To resolve this problem, use one of the following steps that is appropriate for
	your problem:
	
	- Put the Updates folder back into Encarta 97 Encyclopedia folder.
	
	- Put the file, catlnk.ewl, back into the Updates folder inside the Encarta 97
	  Encyclopedia folder.
	
	- Reinstall Microsoft Encarta Encyclopedia for Macintosh 1997.
	
	These steps should return Encarta 97 Encyclopedia to its normal operation.
	
	MORE INFORMATION
	================
	
	After receiving this error message, you should be able to utilize Microsoft
	Encarta 1997 for the Macintosh. However, not all features may be available. You
	will continue to receive the error message while downloading updates or when
	starting the program. You should still be able to download the update files from
	the Encarta web site, but the yearbook file may not integrate successfully.
