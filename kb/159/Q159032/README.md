## Q159032: Err Msg: Data in Clipboard Is in an Unknown Format

	Article: Q159032
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 2,2.1
	Operating System(s): 
	Keyword(s): osr2 win95
	Last Modified: 31-JAN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 OEM Service Release, versions 2, 2.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are copying an item to Clipboard Viewer, you may receive the following
	error message:
	
	  Data in clipboard is in an unknown format
	
	RESOLUTION
	==========
	
	Minimize or close Clipboard Viewer when you are copying an item.
