## Q159993: Unable to Administer Macintosh Volumes from Windows NT Explorer

	Article: Q159993
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	There is no option to Create, View, or Modify Macintosh volumes in Windows
	Explorer in Windows NT 4.0.
	
	CAUSE
	=====
	
	Windows Explorer in Windows NT does not have the ability to administer Macintosh
	volumes. If you use Windows Explorer to change the permissions of a share, it
	will not affect the Macintosh volume permissions.
	
	RESOLUTION
	==========
	
	In Windows NT 4.0, use File Manager or Server Manager to administer Macintosh
	volumes.
