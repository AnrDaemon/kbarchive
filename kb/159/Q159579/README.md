## Q159579: Slovenian Keyboard Layout Shows Dot Instead of Comma

{% raw %}

	Article: Q159579
	Product(s): Microsoft Windows NT
	Version(s): WinNT:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.00 
	- Microsoft Windows NT Workstation version 4.00 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Slovene keyboard layout types a period (.) when you press the comma (,) key
	on the numeric keypad.
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
