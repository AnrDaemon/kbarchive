## Q159158: Cannot Answer Incoming Call in Microsoft Phone

{% raw %}

	Article: Q159158
	Product(s): Microsoft Mail For PC Networks
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Phone, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to answer an incoming call in Microsoft Phone version 1.0, a
	connection may be established only briefly before you are disconnected.
	
	CAUSE
	=====
	
	This problem can occur when a telephone is connected to the voice modem
	installed in your computer and the telephone is not properly communicating with
	the modem.
	
	RESOLUTION
	==========
	
	To determine if the problem occurs because the telephone is not properly
	communicating with the voice modem, disconnect the telephone from the modem. If
	doing so corrects the problem, the telephone is not properly communicating with
	the modem. To resolve this problem, use a different telephone.
