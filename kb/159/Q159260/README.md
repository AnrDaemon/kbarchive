## Q159260: Spike's World: What Is Shareware?

	Article: Q159260
	Product(s): Microsoft Home Kids Products
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbfaq
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Spike's World, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Spike's World is a place on MSN where you can download shareware games.
	Shareware is a method for distributing low-cost, high-quality software.
	
	Shareware software is generally a smaller version of the full-featured product.
	The author of the program hopes you will like the small version enough to pay
	money for the full "registered" version. The full version usually adds extra
	features or gameplay.
	
	MORE INFORMATION
	================
	
	All the shareware games in Spike's World tell you how to register to get the
	full version of the game.
