## Q159917: PPT7: PowerPoint Launches In Background From Windows 3.x Program

{% raw %}

	Article: Q159917
	Product(s): Microsoft PowerPoint for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): kbenv kbinterop
	Last Modified: 16-APR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft PowerPoint for Windows 95, version 7.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you start Microsoft PowerPoint for Windows 95, version 7.0, from Program
	Manager, File Manager, or from any Windows 3.x application, such as Microsoft
	Mail 3.2 by either double-clicking the PowerPoint icon or the file, the
	PowerPoint window appears behind the other program.
	
	PowerPoint functions correctly; you can press "ALT+TAB" (without the quotation
	marks) to switch to it.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
