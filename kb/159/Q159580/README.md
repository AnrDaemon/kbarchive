## Q159580: WordPad Does Not Show Printer Fonts

{% raw %}

	Article: Q159580
	Product(s): Microsoft Windows NT
	Version(s): WinNT:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	WordPad may not show printer fonts in Font list.
	
	CAUSE
	=====
	
	Printer fonts may not be shown in WordPad before the printer dialog is called
	and the printers are initialized. After initialization, all printer fonts are
	available.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 4.0. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
