## Q159853: Display Toolbar Checkbox Not Functional on MSDOS Shortcut

	Article: Q159853
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbnetwork
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	A toolbar is not displayed after you enable the Display Toolbar option for an
	MS-DOS-based application's shortcut.
	
	CAUSE
	=====
	
	Windows NT does not provide a toolbar for console windows.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 4.0. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
	
	
	MORE INFORMATION
	================
	
	If you perform the following steps on an MS-DOS-based application's shortcut, a
	toolbar is not displayed:
	
	1. Right-click the MS-DOS-based application's shortcut icon.
	
	2. Click the Screen tab and then click Display Toolbar so that it is selected,
	  and then click OK.
	
	3. Double-click the MS-DOS-based application's shortcut icon to launch it. No
	  toolbar appears.
