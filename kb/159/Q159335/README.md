## Q159335: XFOR: IMC Perfmon Counters Available Only When IMC is Running

	Article: Q159335
	Product(s): Microsoft Exchange
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 10-APR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Internet Mail Connector (IMC) performance counters are available in
	Performance Monitor only when the IMC service is started. This is different from
	the MTA and Information Store counters; these counters are available in
	Performance Monitor even when the service is not started.
	
	MORE INFORMATION
	================
	
	This behavior is by product design.
