## Q159820: XGEN: How to Obtain Exchange Server 5.0 U.S. Service Pack 1

	Article: Q159820
	Product(s): Microsoft Exchange
	Version(s): winnt:5.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 10-APR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	** THIS U.S. SERVICE PACK 1 IS FOR MICROSOFT EXCHANGE SERVER 5.0 ONLY!! **
	
	The Microsoft Exchange Server version 5.0 U.S. Service Pack 1 is a cumulative
	build of all fixes to Microsoft Exchange Server version 5.0.
	
	To find out which bugs are fixed by this update and how to install this Service
	Pack, see the Readme.txt file that accompanies the Service Pack or query on the
	following word in the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
	
	MORE INFORMATION
	================
	
	Available Formats for Microsoft Exchange Server 5.0 Service Pack 1
	------------------------------------------------------------------
	
	Service Pack 1 is available as single self-extracting archive file for Intel
	(x86), ALPHA, and Macintosh-based computers.
	
	Ordering Microsoft Exchange Server 5.0 Service Pack 1
	-----------------------------------------------------
	
	If you do not have access to the electronic services listed below, you can order
	Service Pack 1 (CD-ROM or Disk Set) from the Microsoft Sales Information Center
	(MSIC) at (800) 426-9400.
	
	Instructions for Downloading
	----------------------------
	
	Because of their large size, the self-extracting files for Microsoft Exchange
	version 5.0 Service Pack 1 are only available on the Internet. To obtain these
	self-extracting files using anonymous FTP on the Internet, go to:
	
	  ftp://ftp.microsoft.com
	
	-or-
	
	  ftp://207.46.133.140
	
	and then go to:
	
	  /bussys/exchange/exchange-public/fixes/eng/exchg5.0/sp1
	
	-or-
	
	  /Softlib/MSLFILES
	
	The files will soon be available on Microsoft's World Wide Web site on the
	Internet. To find them:
	
	1. On http://www.microsoft.com, click the Support button.
	
	2. Select Exchange and click GO.
	
	3. Click the Service Packs link.
	
	4. Click the Microsoft Exchange Service Packs hyperlink.
	
	5. Click the name of the file you want to download.
	
	
	For additional information about downloading, please see the following article in
	the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
