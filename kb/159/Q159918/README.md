## Q159918: Err Msg: &quot;Urlmon.dll Is Missing&quot;

{% raw %}

	Article: Q159918
	Product(s): The Microsoft Network
	Version(s): 1.3,2.0,2.5
	Operating System(s): 
	Keyword(s): kberrmsg kbsetup kbmsn
	Last Modified: 10-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.3, 2.0, 2.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to connect to MSN, The Microsoft Network, you may receive the
	following error message:
	
	  Urlmon.dll is missing.
	
	CAUSE
	=====
	
	This behavior can occur if Microsoft Internet Explorer was not installed
	properly during MSN Setup, or if you deleted the Urlmon.dll file when you
	uninstalled Netscape Navigator 2.0.
	
	RESOLUTION
	==========
	
	Please go to: Q184691 - MSN Err Msg: Urlmon.dll Is Missing
