## Q129590: Multimedia Strauss: BETA Misprinted Under Copyright Text

	Article: Q129590
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Multimedia Strauss for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The word "BETA" (without the quotes) is printed on the retail version of the
	Microsoft Multimedia Strauss CD-ROM disc. The word is located underneath the
	copyright information where the Disk Assembly and Part Number should be.
	
	MORE INFORMATION
	================
	
	The word "BETA" is a production error. The product is a retail version.
	
	Microsoft has confirmed this to be a problem with Multimedia Strauss, Three Tone
	Poems. We are researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
