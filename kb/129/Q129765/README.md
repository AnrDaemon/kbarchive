## Q129765: ProShare 1.8 Does Not Work with Windows 95 TCP/IP

	Article: Q129765
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Intel ProShare version 1.8 cannot find the TCP/IP network protocol when you scan
	for it.
	
	CAUSE
	=====
	
	ProShare does not recognize and does not work with the Windows 95 TCP/IP network
	protocol.
	
	RESOLUTION
	==========
	
	To work around this behavior, use either of the following methods:
	
	- Contact Intel about a possible upgrade to ProShare.
	
	- Use a different network protocol.
	
	MORE INFORMATION
	================
	
	Intel ProShare uses a digital camera board and an audio/ISDN board to provide
	video conferencing.
	
	ProShare is manufactured by Intel Corporation, a vendor independent of Microsoft;
	we make no warranty, implied or otherwise, regarding this product's performance
	or reliability.
