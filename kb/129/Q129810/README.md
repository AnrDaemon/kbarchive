## Q129810: MS Paint Free-Form Select Tool Appears to Select Rectangle

	Article: Q129810
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you select an area in Microsoft Paint using the Free-Form Select tool, it
	appears as if you have selected a rectangular area. However, the free-form area
	is actually selected.
	
	STATUS
	======
	
	This behavior is by design.
	
	Additional query words: irregular
