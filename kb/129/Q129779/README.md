## Q129779: PC Mac: Err Msg: File Pathname Exceeds Max. Length of 67 Char.

{% raw %}

	Article: Q129779
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0,3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, versions 3.0, 3.2, on platform(s):
	   - the operating system: Mac OS (ALL) 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In versions 3.0 and 3.2 of Microsoft Mail for PC Networks, Macintosh
	workstation, if you try to store the private folders to local storage to a
	folder that exceeds 67 characters, the following error will be displayed:
	
	  File Pathname exceeds maximum length of 67 characters
	
	WORKAROUND
	==========
	
	To store the private folders to local storage, choose a path that is less than
	67 characters. At this time, there are no other known resolutions.
