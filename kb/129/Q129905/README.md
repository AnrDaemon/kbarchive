## Q129905: BUG: &quot;Hide All Windows&quot; Does Not Hide the Command Window

{% raw %}

	Article: Q129905
	Product(s): Microsoft FoxPro
	Version(s): WINDOWS:3.0,3.0b,5.0,5.0a,6.0
	Operating System(s): 
	Keyword(s): kbui kbvfp300bBUG kbvfp500aBUG kbvfp500bugkbbuglist
	Last Modified: 10-FEB-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual FoxPro for Windows, versions 3.0, 3.0b, 5.0, 5.0a, 6.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Holding down the SHIFT key as you choose Hide All from the Window menu does not
	hide the Command window or the Standard toolbar. However, holding down the SHIFT
	key as you choose Close All from the File menu does close all windows including
	the Command window.
	
	RESOLUTION
	==========
	
	To hide the Command window, press the CTRL+ALT+SHIFT keys, which hides all
	system windows. You can also choose Close form the File menu to close the
	selected window.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
