## Q129083: Microsoft Visual Basic Toolbox Does Not Minimize

	Article: Q129083
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are running Microsoft Visual Basic Professional Edition version 3.0 for
	Windows in Windows 95, you cannot minimize the Visual Basic toolbox.
	
	CAUSE
	=====
	
	When you click the taskbar with the right mouse button and then click Minimize
	All Windows on the menu that appears, all windows except the Visual Basic
	toolbox are minimized. The Visual Basic toolbox does not have a Minimize option.
	
	RESOLUTION
	==========
	
	To minimize the Visual Basic toolbox, use the Visual Basic window's Minimize
	button or Control menu.
