## Q129915: Cannot Print Selected Text in WordPad or Paint

	Article: Q129915
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you select a section of text in Microsoft WordPad or Microsoft Paint so
	that you can print only the selected text, you find that the Selection option
	button in the Print dialog box is unavailable and you cannot print only the
	selected text.
	
	RESOLUTION
	==========
	
	To work around this behavior, copy the selected text and paste it into a new
	document. Print the text from the new document.
