## Q129223: Export to Timex Data Link Watch Overwrites Old Information

	Article: Q129223
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0; Win95:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you export information to a Timex Data Link watch, the information
	currently held in the watch is completely overwritten by the new information.
	
	CAUSE
	=====
	
	The Timex Data Link watch cannot append to the information it already contains.
	
	RESOLUTION
	==========
	
	Make sure that the information you send to the watch contains all of the
	information you want to be on the watch, rather than just updates.
	
	MORE INFORMATION
	================
	
	The Timex Data Link watch is manufactured by Timex, a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
