## Q129295: Closing Thread with Open Message Confuses Navigation in BBS

	Article: Q129295
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.0,1.05,1.2,1.3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.0, 1.05, 1.2, 1.3 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You expand a thread (conversation) in a bulletin board, open a message in the
	thread, and then collapse the thread while the message is still open. If you
	then click View Previous Message, the last message in the bulletin board is
	displayed. If you click View Next Message, the first message in the bulletin
	board is displayed.
	
	WORKAROUND
	==========
	
	Do not collapse a thread while a message in the thread is still open.
