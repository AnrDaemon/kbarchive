## Q129101: 500 Nations: Videos Have Psychedelic Background Pattern

{% raw %}

	Article: Q129101
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft 500 Nations, version 1.0, on platform(s):
	   - the operating system: Microsoft Windows NT, on platform(s):
	      - the operating system: Microsoft Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Microsoft 500 Nations, when you play videos on a computer using a Diamond
	Viper Pro video card with the Wieteck P9100 chipset, the screen background may
	change to a multicolored or psychedelic pattern.
	
	MORE INFORMATION
	================
	
	Even though the background pattern is incorrect, the video continues to play
	normally, without discoloration.
	
	Microsoft has confirmed this to be a problem in Microsoft 500 Nations. We are
	researching this problem and will post new information here in the Knowledge
	Base as it becomes available.

	of Microsoft; we make no warranty, implied or otherwise, regarding these
	products' performance or reliability.
