## Q310999: Can't Use the Spell Check Function Of Hotmail

	Article: Q310999
	Product(s): The Microsoft Network
	Version(s): 1.0,2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, versions 1.0, 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You cannot use the spelling checker function of Hotmail while you are using MSN
	Companion.
	
	CAUSE
	=====
	
	A recent change in the Hotmail interface has caused this issue.
	
	RESOLUTION
	==========
	
	This is a known issue and may be resolved in a future version of MSN Companion
