## Q310630: MechWarrior 4 Black Knight: Known Video Issues

	Article: Q310630
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 16-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MechWarrior 4: Black Knight 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article discusses known video issues with Microsoft MechWarrior 4: Black
	Knight.
	
	MORE INFORMATION
	================
	
	The following video issues are known to occur with MechWarrior 4: Black Knight.
	Most video issues can be resolved by updating to the latest video card drivers.
	
	Number Nine SR-9 Pro
	--------------------
	
	  You might experience periodic instability with the game, or the game may stop
	  responding (hang). If you experience problems with a Number Nine card, it is
	  recommended that you upgrade your card.
	
	nVidia
	------
	
	  GeForce 256 card
	
	  When you attempt to launch certain missions while in 1600 x 1200 x 32
	  resolution, your computer might restart. If you have this problem, turn off
	  1600 x 1200 x 32 resolution bit depth.
	
	  TNT and GeForce Cards
	
	  If you run MechWarrior 4: Black Knight on a system with a GeForce or TNT card
	  and an ASUS motherboard, the game may periodically stop responding or quit.
	  If you have this problem, contact your manufacturer for driver updates.
	
	Via Chipsets
	------------
	
	  Stability issues may occur with these motherboard chipsets when you do not
	  use the latest drivers. To fix this, download and install the latest chipset
	  drivers for your motherboard at the following Web site:
	
	  http://www.viatech.com (http://www.viatech.com)
	
	AMD Athlon/Duron Processors
	---------------------------
	
	  If you experience instability when running MechWarrior 4: Black Knight,
	  download and install the latest AMD processor utilities and drivers for your
	  operating system. You can find these at one of the following Web sites:
	
	  http://www.amd.com/support/software.html
	  (http://www.amd.com/support/software.html)
	
	  -or-
	
	  http://www.amd.com/products/cpg/athlon-duron/amd_win2k_patch.html
	  (http://www.amd.com/products/cpg/athlon-duron/amd_win2k_patch.html)
	
	3DFX Voodoo 5 5500
	------------------
	
	  Some operating systems have problems displaying the mouse cursor and other
	  pictures. If this occurs, download the latest drivers at the following Web
	  site:
	
	  http://www.3dfx.com/downloads.htm (http://www.3dfx.com/downloads.htm)
