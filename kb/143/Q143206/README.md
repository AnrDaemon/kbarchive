## Q143206: Bookshelf 1995 for the Macintosh Minimum System Requirements

	Article: Q143206
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1995 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for the Macintosh 1995 edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following article lists the minimum system requirements for the 1995 edition
	of Microsoft Bookshelf for the Macintosh.
	
	MORE INFORMATION
	================
	
	The minimum system requirements to run Bookshelf 1995 edition are:
	
	- Color Macintosh II or higher, Color Macintosh Classic, or PowerBook
	
	- System 7 or later
	
	- Minimum 4 MB RAM memory
	
	- Minimum 2 MB of available hard-disk space
	
	- Double-speed CD-ROM: The CD-ROM drive can transfer data from the CD-ROM at
	  300k per second while using less than 40 percent of the CPU bandwidth and has
	  an average seek time of less than 400 milliseconds.
