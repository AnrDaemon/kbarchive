## Q143267: Scenes 2.0: List of Known Content Errors

	Article: Q143267
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Scenes (all collections), version 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains a list of known content errors for Microsoft Scenes
	version 2.0.
	
	MORE INFORMATION
	================
	
	Flight Collection:
	
	Image 6 is labeled as a "Suchoi SU-35 Super Tanker." The correct spelling is
	"Sukhoi" (without the quotation marks). Sukhoi is spelled correctly on image 24,
	Sukhoi SU-25 Frogfoot.
	
	
	Sierra Club Nature Collection:
	
	Image 21 is labeled as "Organ Pipe Cactus National Monument, Arizona." While the
	label is correct, the cactus shown in the picture is actually a Saguaro cactus
	that can be found near Tucson, Arizona.
