## Q143272: How to Restart Windows Without Restarting the Computer

	Article: Q143272
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	NOTE: This article contains information about using Windows in a
	configuration that has not been tested and is not supported by Microsoft.
	If the steps described in this article do not function properly, do not
	use this method.
	
	SUMMARY
	=======
	
	This article describes how to restart Windows without restarting your computer.
	
	MORE INFORMATION
	================
	
	To restart Windows without restarting your computer, follow these steps:
	
	1. Click Start, and then click Shut Down.
	
	2. Press and hold down the SHIFT key while you click Restart The Computer, and
	  then click Yes.
	
	3. As soon as the screen clears, release the SHIFT key.
	
	NOTE: If you hold the SHIFT key too long, Windows restarts in Safe mode.
