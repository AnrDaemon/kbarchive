## Q143055: Voice Command Does Not Deactivate Screen Saver

{% raw %}

	Article: Q143055
	Product(s): Microsoft Mail For PC Networks
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): win95 msphone1
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Phone, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Using a voice command while a screen saver is active may not deactivate the
	screen saver. For example, the command "What can I say?" may be recognized and
	run, but the screen saver may remain in the foreground.
	
	RESOLUTION
	==========
	
	Create a command to deactivate the screen saver. This command should contain a
	keystroke such as ESC as the only action. The screen saver will be deactivated
	when you issue this command.
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
