## Q143191: Real Audio Is Supported on Windows NT Internet Explorer

	Article: Q143191
	Product(s): Internet Information Server
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 08-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Internet Information Server 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Windows NT Internet Explorer for Intel-processor-based workstations are capable
	of playing "Real Audio". This feature will be available on other Microsoft
	Windows NT Internet Explorer versions in future releases.
	
	This product is available in the CLIENTS\WINNT\I386 directory of the Internet
	Information Server compact disc.
