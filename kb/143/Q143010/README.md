## Q143010: Slide Colors Change When Editing Graph or Organization Chart

{% raw %}

	Article: Q143010
	Product(s): Microsoft PowerPoint for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): kbole kbgraph kborg
	Last Modified: 16-APR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft PowerPoint for Windows 95, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In a PowerPoint slide show running under Windows NT 3.51, when you edit an
	object created in Organization Chart or Microsoft Graph, the colors in the
	slides may be different from what you expect when you exit and return to the
	slide show.
	
	RESOLUTION
	==========
	
	Instead of using Animation Settings to edit the object, you can set the object's
	Interactive Settings to Edit when you click the object.
	
	Steps to set Interactive Settings
	---------------------------------
	
	1. On the Tools Menu, click Interactive Settings.
	
	2. Click Object Action and click Edit on the drop down menu.
	
	3. Click OK.
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
