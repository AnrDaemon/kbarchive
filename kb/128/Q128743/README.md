## Q128743: PC Win WRmt: Asterisk Appears in From Field When View Message

{% raw %}

	Article: Q128743
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0b,3.2,3.2a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, versions 3.0b, 3.2, 3.2a 
	- Microsoft Mail Remote for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you view a message, an asterisk (*) may appear left of the From field in
	the message header.
	
	MORE INFORMATION
	================
	
	This indicates that the message body was changed since the message was
	originally received. Once the message has been changed, there is no way to
	remove the asterisk.
