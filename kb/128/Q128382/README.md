## Q128382: UT2: Access Upsizing Tools Information Available

	Article: Q128382
	Product(s): Microsoft Access Distribution Kit
	Version(s): WINDOWS:2.0
	Operating System(s): 
	Keyword(s): kbfile kbgraphxlinkcritical
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Access 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Upsize.exe file contains a press release and summary information about the
	Microsoft Access Upsizing Tools.
	
	The following file is available for download from the Microsoft Download Center:
	
	  Download Upsize.exe now
	  (http://download.microsoft.com/download/access20/whitep40/1/WIN98/EN-US/Upsize.exe)
	
	The Upsize.exe file contains the following files:
	
	  Uptoolpr.doc     The Microsoft Access Upsizing Tools announcement.
	
	  Uptoolpr.wri     A text file of the Microsoft Access Upsizing Tools
	                   announcement.
	
	  Ut1summ.doc      A summary document describing the Microsoft Access
	                   Upsizing Tools.
	
	  Ut1summ.wri      A summary text file describing the Microsoft Access
	                   Upsizing Tools.
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
	
	Microsoft used the most current virus detection software available on the date of
	posting to scan this file for viruses. Once posted, the file is housed on secure
	servers that prevent any unauthorized changes to the file.
	
	Additional query words: Upsize exe
