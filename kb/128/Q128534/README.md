## Q128534: Microsoft Systems Management Server Help File, November 1995

	Article: Q128534
	Product(s): Microsoft Systems Management Server
	Version(s): 1.0,1.1
	Operating System(s): 
	Keyword(s): smsgeneral kbArtTypeINF kbgraphxlinkcritical
	Last Modified: 02-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server versions 1.0, 1.1 
	-------------------------------------------------------------------------------
	
	The Microsoft Product Support Services (PSS) Knowledge Base group regularly
	compiles the articles about Microsoft Systems Management Server and creates
	a Windows Help file that is available in the Microsoft Download Center.
	
	The Help file for Microsoft Systems Management Server is in a self-
	extracting file named SMSKB.EXE and contains technical articles written by
	PSS support professionals and Knowledge Base writers published prior to December 5, 1995. These articles contain:
	
	- References to files available in the Microsoft Download Center
	
	- Bug lists and fix lists for Systems Management Server
	
	- Additional references
	
	- Questions and answers
	
	- Recent product information not addressed in the user documentation
	
	- Helpful tips
	
	NOTE: This Help file can only be run from Windows 95 or Windows NT version
	3.51. If you have Windows NT 3.5 or earlier, you must upgrade to version
	3.51 to view this Help file.
	
	SMSKB.EXE contains the following files:
	
	  SMSKB.HLP   The KB Help file
	  SMSKB.CNT   The Contents file
	
	
	FILES AVAILABLE FROM ONLINE SERVICES
	------------------------------------
	
	The following file is available for download from the Microsoft Download Center:
	
	  DownloadDownload Smskb.exe now
	
	For additional information about how to download Microsoft Support
	files, click the article number below 
	to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services

	on the date of posting to scan this file for viruses. Once posted,

	changes to the file.
