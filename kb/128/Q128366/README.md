## Q128366: Mac Wkst: Resizing Graphic Images

{% raw %}

	Article: Q128366
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, version 3.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you view a graphic in the image form, if the graphic is too large, it will
	be truncated to fit in the window.
	
	WORKAROUND
	==========
	
	It is possible to view the entire graphic by double-clicking anywhere on the
	graphic image. The form will resize the graphic to fit in the window.
