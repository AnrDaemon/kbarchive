## Q128383: ADT2: Updated ADT License Agreement Available

	Article: Q128383
	Product(s): Microsoft Access Distribution Kit
	Version(s): WINDOWS:2.0
	Operating System(s): 
	Keyword(s): kbfile kbgraphxlinkcritical
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Access Developer's Toolkit, version 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Adtlic.exe file contains the updated license agreement for the Microsoft
	Access Developer's Toolkit (ADT) version 2.0 and an explanation of the
	distribution rights for the Data Outline control.
	
	The following file is available for download from the Microsoft Download Center:
	
	  Download Adtlic.exe now
	  (http://download.microsoft.com/download/access20/update6/1/WIN98/EN-US/adtlic.exe)
	
	The Adtlic.exe file contains the following files:
	
	  Adtla.wri       A text file containing the ADT License Agreement.
	  Adtlett.wri     A text file explaining the changed distribution rights
	                  for the Data Outline control.
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
	
	Microsoft used the most current virus detection software available on the date of
	posting to scan this file for viruses. Once posted, the file is housed on secure
	servers that prevent any unauthorized changes to the file.
