## Q128371: Error 5733 with InfoExpress Real-Mode Drivers

	Article: Q128371
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 25-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you load the real-mode drivers supplied with EFA Corporation's InfoExpress
	S16 network card, you receive the following error message:
	
	  Error 5733: The protocol manager has reported an incomplete binding.
	
	STATUS
	======
	
	EFA Corporation has confirmed this to be a problem in the InfoExpress S16
	network card driver version 1.2 for Windows for Workgroups version 3.11.
	
	WORKAROUND
	==========
	
	EFA recommends using the Windows for Workgroups 3.11 NE2000-compatible network
	card driver as a workaround for this problem. For additional information about
	this problem, please contact EFA Corporation.
