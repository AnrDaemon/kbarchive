## Q194030: XCLN: MS Exchange Server 5.5 Service Pack 1 Incorrect CD Key

{% raw %}

	Article: Q194030
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5 SP1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-JUN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 SP1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The CD key on the compact disc (CD) provided with Microsoft Exchange Server
	version 5.5 Service Pack 1 (SP1) does not work correctly during Setup of
	Microsoft Outlook 98.
	
	WORKAROUND
	==========
	
	Please call a Microsoft Customer Service Representative for a new CD key. The
	telephone number to call is 800-936-5900. Wait for the automated answering
	system to complete its offerings without pressing any numbers on your phone, and
	a representative will answer your call.
	
	MORE INFORMATION
	================
	
	Microsoft Exchange Server 5.5 Service Pack 1 Package includes:
	
	Microsoft Exchange Server 5.5 Service Pack 1
	Microsoft Outlook CD (Part Number X03-68288)
