## Q194350: Encarta Virtual Globe 99: Incorrect URL in Bibliography

	Article: Q194350
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbref kbimu
	Last Modified: 12-AUG-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Virtual Globe 99, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Bibliography in Encarta Virtual Globe may contain incorrect or outdated
	Universal Resource Locator (URL) links.
	
	This article lists the URLs that are known to have changed.
	
	MORE INFORMATION
	================
	
	The following list contains the changed URLs in Virtual Globe.
	
	- United States National Aeronautics and Space Administration. Remote Sensing
	  in History.
	
	  Existing URL:
	
	  http://observe.ivv.nasa.gov/observe/exhibit/rem_sen/history /history_2.html
	
	  Changed URL (as of 9/4/98):
	
	  http://observe.ivv.nasa.gov/nasa/exhibits/history/history_0.html
	
	If you find a Web site location that has changed but is not in the list, please
	contact Microsoft using one of the following contact points:
