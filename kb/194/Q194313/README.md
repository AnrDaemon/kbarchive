## Q194313: XCLN: Outlook for Windows 3.x Remove All Does Not Delete Recall

	Article: Q194313
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:8.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook, Exchange Server Edition, version 8.0, used with:
	   - the operating system: Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you select the Remove All option in the Setup program for Microsoft Outlook
	for Windows 3.x, the file Recall.dll is not removed. This is a non-shared
	component installed in the Windows\System folder that would normally be removed
	when uninstalling.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Outlook for Windows
	3.x, Exchange Server Edition, version 8.0.
