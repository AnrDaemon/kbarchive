## Q194107: Cannot Connect to Newsgroups on Msnnews.msn.com News Server

	Article: Q194107
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.3,2.0,2.5
	Operating System(s): 
	Keyword(s): kbenv kbui kbmsn
	Last Modified: 10-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.3, 2.0, 2.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to read Internet newsgroups on msnnews.msn.com, you may not see
	any content or you may receive an error message.
	
	CAUSE
	=====
	
	This behavior can occur if Microsoft Internet News is not properly configured to
	read Internet newsgroups on msnnews.msn.com.
	
	RESOLUTION
	==========
	
	To configure Internet News to read Internet newsgroups on msnnews.msn.com,
	follow these steps:
	
	1. Start Internet News.
	
	2. On the News menu, click Options.
	
	3. Click the Server tab.
	
	4. Type your name and e-mail address in the appropriate boxes.
	
	5. In the News Server(s) box, click Add.
	
	6. In the News Server Name box, type "msnnews.msn.com" (without the quotation
	  marks).
	
	7. Click the Connection tab.
	
	8. Click "I use a modem to access my newsgroups."
	
	9. In the "Use the following Dial-Up Networking connection" box, click MSN or
	  The Microsoft Network, and then click OK.
	
	10. Click OK until you return to Windows.
	
	Additional query words: msn
