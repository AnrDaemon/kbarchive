## Q194995: Pinball Arcade: Game Pauses or Game Performance Is Slow

	Article: Q194995
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbenv kbhw kbimu msgame kbHardwarekbfaq
	Last Modified: 25-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Pinball Arcade, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Pinball Arcade, you may experience either of the
	following symptoms:
	
	- The game pauses intermittently.
	
	- Game performance is extremely slow.
	
	CAUSE
	=====
	
	This behavior can occur for either of the following reasons:
	
	- You are running the Microsoft Sidewinder Game Controller software on your
	  computer, and you disconnect the SideWinder game controller while the
	  computer is running.
	
	- You are using a Microsoft Sidewinder 3D Pro joystick on a Microsoft Windows
	  NT 4.0-based computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, use the appropriate method listed below.
	
	Disconnected SideWinder Game Controller
	---------------------------------------
	
	To resolve this issue, restart the computer, or refresh the game controller
	status in Microsoft Windows.
	
	To refresh the game controller status in Windows, follow these steps:
	
	1. Click Start, point to Settings, and then click Control Panel.
	
	2. Double-click Game Controllers.
	
	3. Verify that "Not Connected" is displayed in the Status column next to your
	  SideWinder game controller, and then click OK.
	
	4. Close Control Panel.
	
	SideWinder 3D Pro on a Windows NT 4.0-based Computer
	----------------------------------------------------
	
	To resolve this issue, download and install the latest Microsoft Windows NT 4.0
	Service Pack from the following Microsoft Web site:
	
	  http://support.microsoft.com/support/downloads
