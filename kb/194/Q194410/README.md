## Q194410: XWEB: No Response Buttons in Meeting Request in OWA

	Article: Q194410
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook Web Access, version 5.5 
	- Microsoft Outlook Web Access, version 5.5 Service Pack 1 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	When you use the Outlook Web Access client to open a meeting request on behalf of
	the owner of a mailbox, you cannot respond; there are no options for Accept,
	Decline, or Tentative.
	
	CAUSE
	=====
	
	Full support for Delegate access is not included in the current release.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
