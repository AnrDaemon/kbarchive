## Q194162: Encarta Virtual Globe 99: Error Message if You Remove CD-ROM

{% raw %}

	Article: Q194162
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbenv kberrmsg kbimu
	Last Modified: 06-AUG-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Virtual Globe 99, version 1.0 
	- Microsoft Encarta Reference Suite 99 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you remove the Microsoft Encarta Virtual Globe 99 CD-ROM from your CD-ROM
	drive, you may receive the following error message on a Windows 95/98-based
	computer
	
	  Error Reading CD-ROM in Drive drive:
	
	  Please Insert CD-ROM with Serial Number xxxx-xxxx in drive drive:
	
	  If the CD-ROM is still in the drive, it may require cleaning.
	
	  Press ENTER for OK or ESC to Cancel:
	
	where drive is the letter of your CD-ROM drive.
	
	If you are running a Windows NT-based computer, you may receive either of the
	following error messages:
	
	   - Microsoft Encarta Virtual Globe 99 contains two CDs. Please insert the
	  'Encarta Virtual Globe 99' disc into your CD-ROM drive to run the program.
	
	   - Microsoft Encarta Virtual Globe 99 contains two CDs. Please insert the
	  'Installations and Flights' disc into your CD-ROM drive to run the program.
	
	CAUSE
	=====
	
	This behavior can occur if you remove the Encarta Virtual Globe 99 CD-ROM from
	the CD-ROM drive when you are using the world map or the Virtual Flights
	feature.
	
	RESOLUTION
	==========
	
	To resolve this issue, insert the correct CD-ROM into the CD-ROM drive, and then
	press ENTER.
	
	If you are using the world map or viewing articles, insert the Encarta Virtual
	Globe 99 CD-ROM or Disc 1 CD-ROM.
	
	If you are using the Virtual Flights feature, insert the Disc 2 (Installation and
	Resources) CD-ROM.
