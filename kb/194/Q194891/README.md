## Q194891: Virtual Globe 99: Very Small Image of Photograph Is Displayed

{% raw %}

	Article: Q194891
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbimu
	Last Modified: 12-AUG-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Virtual Globe 99, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you double-click a slidetray photograph in Encarta Virtual Globe 99, a very
	small image of the photograph may be displayed on the screen.
	
	RESOLUTION
	==========
	
	To prevent this issue from occurring, click the slidetray photograph only once
	to display an enlarged image of the photograph on the screen.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Encarta Virtual Globe
	99, version 1.0.
