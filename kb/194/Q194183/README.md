## Q194183: Virtual Globe: Photo Credit Is Truncated When You Copy Picture

	Article: Q194183
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbimu
	Last Modified: 19-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Virtual Globe 99, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you copy a picture from Encarta Virtual Globe 99, the photo credit may be
	cut off at the margin of the picture.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Encarta Virtual Globe
	99, version 1.0.
	
	MORE INFORMATION
	================
	
	This behavior has been reported with the following pictures:
	
	- Italy: Commercial Fishing
	
	- South Africa: Casting Gold In Johannesburg
	
	- Tajikistan: Village Musician
