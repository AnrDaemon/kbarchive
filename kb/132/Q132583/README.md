## Q132583: You're Hired! Ver. 1.02 May Not Run Again After Local Reboot

{% raw %}

	Article: Q132583
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You're Hired! version 1.02 for Windows may not run after being terminated with a
	local reboot until you restart Windows 95.
	
	CAUSE
	=====
	
	Some 16-bit Windows-based programs require that the computer be restarted before
	running again after being terminated with a local reboot. This occurs because a
	local reboot may not perform all the functions that are performed when you exit
	the program normally.
	
	RESOLUTION
	==========
	
	You must restart your computer to run You're Hired! again.
