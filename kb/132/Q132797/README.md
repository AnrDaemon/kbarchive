## Q132797: Minimize Button in MS PowerPoint Viewer Closes PowerPoint

{% raw %}

	Article: Q132797
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you minimize Microsoft PowerPoint version 4.0 Viewer, Viewer is closed
	rather than minimized.
	
	CAUSE
	=====
	
	In Windows 95, the Close button within PowerPoint 4.0 changes to the Minimize
	button, but retains the function of closing Viewer.
	
	RESOLUTION
	==========
	
	To minimize PowerPoint 4.0 Viewer, using the right mouse button, click
	PowerPoint Viewer on the taskbar, and then click Close on the menu that appears.
	
	STATUS
	======
	
	This behavior no longer occurs in Microsoft Office 95.
