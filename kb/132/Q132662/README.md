## Q132662: Cannot Copy Mail with Long Title to Desktop

	Article: Q132662
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): kbtool kbui win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you try to copy a mail message to the desktop by dragging it to the desktop,
	the copy operation may not succeed. When this occurs, you do not receive any
	error message.
	
	CAUSE
	=====
	
	This behavior occurs if the combined path and name for the message exceeds the
	maximum allowable length of 260 characters.
	
	RESOLUTION
	==========
	
	Shorten the mail message's title, then try to copy it again.
