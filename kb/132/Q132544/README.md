## Q132544: 500 Nations: README.TXT Incorrect Location of Video Drivers

	Article: Q132544
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft 500 Nations, version 1.0, on platform(s):
	   - the operating system: Microsoft Windows NT, on platform(s):
	      - the operating system: Microsoft Windows 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The 500 Nations README.TXT contains the wrong location for the program's video
	drivers.
	
	MORE INFORMATION
	================
	
	The correct video driver location on the CD-ROM is the \Nations\Zdrivers folder.
