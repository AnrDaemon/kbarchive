## Q132777: Twain32 (for NT) Scanner Driver Does Not Work in Windows 95

{% raw %}

	Article: Q132777
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	In PhotoShop version 3.0, when you click Acquire on the File menu, you may
	receive the following error messages:
	
	  Sample Source Error
	  General Failure in response to DAT_SETUPMEMXFER
	  not supported
	
	Then, you fall back to the program. The scanner is never accessed.
	
	
	CAUSE
	=====
	
	PhotoShop 3.0 does not support Twain in Windows 95.
	
	RESOLUTION
	==========
	
	PhotoShop version 3.0.5 does support Twain in Windows 95.
	
	Contact Adobe for more information concerning PhotoShop, updated drivers, and
	their installation.
