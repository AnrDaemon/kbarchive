## Q132849: GPF or Poor Performance with Interleaf Running in Windows 95

	Article: Q132849
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run any version of Interleaf prior to version 6.1.1, you may receive a
	general protection fault, or the program may perform poorly.
	
	CAUSE
	=====
	
	Interleaf version 6.1.0 is a beta version and does not run in Windows 95.
	
	RESOLUTION
	==========
	
	Contact Interleaf and request Interleaf version 6.1.1.
	
	MORE INFORMATION
	================
	
	Interleaf is manufactured by Interleaf, Inc., a vendor independent of Microsoft;
	we make no warranty, implied or otherwise, regarding this product's performance
	or reliability.
