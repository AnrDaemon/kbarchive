## Q132651: Cannot Shut Down with Microsoft Visual C++ 2.1 Running

	Article: Q132651
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you try to shut down Windows 95 while Microsoft Visual C++ version 2.1 is
	running, your computer may stop responding (hang).
	
	RESOLUTION
	==========
	
	To avoid this problem, exit Microsoft Visual C++ 2.1 before shutting down
	Windows 95.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Windows 95. We are
	researching this problem and will post additional information here in the
	Microsoft Knowledge Base as it becomes available.
