## Q132834: Problems with Media Blitz 3.0 Setup

	Article: Q132834
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you install Media Blitz version 3.0, when you try to start ClipMaker or
	ScoreMaker you receive an Open With dialog box.
	
	CAUSE
	=====
	
	Media Blitz 3.0 Setup adds associations for these files to the Win.ini file
	rather than to the registry.
	
	RESOLUTION
	==========
	
	Restart your computer to resolve the problem.
