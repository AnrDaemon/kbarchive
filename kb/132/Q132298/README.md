## Q132298: Bookshelf 1995 for the Macintosh: Movie Skips As It Plays

	Article: Q132298
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1995 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for the Macintosh 1995 edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When a movie is played in Microsoft Bookshelf, the narration skips if the mouse
	pointer is moved on top of the article list.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Bookshelf 1995. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
	
	RESOLUTION
	==========
	
	Do not move the mouse pointer while movies play.
