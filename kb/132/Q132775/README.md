## Q132775: Alacrity Equip+ Does Not Function Properly in Windows 95

{% raw %}

	Article: Q132775
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to install Alacrity Equip+ on a computer running Windows 95, you
	find that you are unable to install the program. If you are able to install the
	program, you find that many of the program's features may not function properly.
	In particular, the Tooltips, OCR, and Fax features may not function properly or
	may not function at all.
	
	CAUSE
	=====
	
	Alacrity Equip+ versions 6.0 and earlier are not completely compatible with
	Windows 95.
	
	RESOLUTION
	==========
	
	For information about a possible software upgrade that corrects these problems,
	contact your software manufacturer.
	
	MORE INFORMATION
	================
	
	Equip+ is manufactured by Alacrity, a vendor independent of Microsoft; we make
	no warranty, implied or otherwise, regarding this product's performance or
	reliability.
