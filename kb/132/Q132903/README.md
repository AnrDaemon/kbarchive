## Q132903: Err Msg Using NetBIOS over TCP/IP (NETBT.SYS): STOP 0x0000000A

{% raw %}

	Article: Q132903
	Product(s): Microsoft Windows NT
	Version(s): 3.5,3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation versions 3.5, 3.51 
	- Microsoft Windows NT Server versions 3.5, 3.51 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the NetBIOS over TCP/IP transport (NETBT.SYS) in Windows NT 3.5 or
	3.51, the following error message appears:
	
	  STOP: 0x0000000A
	
	
	RESOLUTION
	----------
	
	To correct this problem, upgrade to Windows NT 3.51 (if you have not already done
	so) and install the latest U.S. Service Pack for Windows NT version 3.51.
	
	STATUS
	------
	
	Microsoft has confirmed this to be a problem in Windows NT version 3.5 and 3.51.
	This problem has been corrected in the latest U.S. Service Pack for Windows NT
	version 3.51. For information on obtaining the Service Pack, query on the
	following word in the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
	
	Additional query words: prodnt 3.50 3.51 trap 0x0a 0xa blue screen
