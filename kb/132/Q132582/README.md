## Q132582: Backup File Damaged Saving Multiple Volumes to a Network Drive

	Article: Q132582
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you perform a backup that spans several logical drives and you place the
	.qic file on a network drive, the resultant file may be damaged. You may also
	receive an error message stating that the backup set is damaged.
	
	
	CAUSE
	=====
	
	This problem has been seen on a Compaq 386/20 computer that was upgraded with a
	Cyrix 486/25 chip. The problem is due to a hardware incompatibility.
	
	
	RESOLUTION
	==========
	
	To work around this problem, back up only one logical drive at a time.
