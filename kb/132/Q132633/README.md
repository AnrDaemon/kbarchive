## Q132633: Installing Lotus Notes Over a Network

{% raw %}

	Article: Q132633
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	To install Lotus Notes from another computer on a network, you must first assign
	(map) a drive letter to the computer that contains the Lotus Notes Setup
	program.
	
	MORE INFORMATION
	================
	
	To map a drive letter to the computer that contains the Lotus Notes Setup
	program, follow these steps:
	
	1. Click the Start button, point to Programs, and then click Windows Explorer on
	  the menu that appears.
	
	2. On the Tools menu in Windows Explorer, click Map Network Drive.
	
	3. In the Path box, type the path to the computer that contains the Lotus Notes
	  Setup program, and then click the OK button.
	
	4. Click the drive you mapped, find the Lotus Notes Setup program, and then
	  install Lotus Notes.
