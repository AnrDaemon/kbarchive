## Q132844: Talking Clock Causes GP Fault Shutting Down Windows 95

{% raw %}

	Article: Q132844
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If the Talking Clock program is loaded when you shut down Windows 95, you
	receive a general protection (GP) fault error message.
	
	CAUSE
	=====
	
	This program makes an invalid system call.
	
	
	RESOLUTION
	==========
	
	Close Talking Clock before shutting down Windows 95.

	================
