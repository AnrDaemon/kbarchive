## Q132106: Mac Encarta 95: Jewel Box Text Has Wrong Close Button Location

	Article: Q132106
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1995 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia for Macintosh 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Page 5 of the Encarta CD-ROM jewel case cover book contains an error about the
	location of the Close button. The Close button is actually located in the
	upper-left corner of the Encarta title bar.
	
	MORE INFORMATION
	================
	
	The following is the text of the error:
	
	  To quit Encarta, click the Close button (square symbol) in the upper- right
	  corner of the Main screen, or choose Quit from the Encarta File menu.
	
	The Close button symbol is located on the upper left hand side of the Microsoft
	Encarta application title bar. The Window Sizing button is located in the upper
	right hand corner.
