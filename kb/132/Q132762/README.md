## Q132762: Format Type Options Are Unavailable

	Article: Q132762
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to format a floppy disk, the only available option in the Format
	Type group may be the Full option. The Quick and the Copy System Files Only
	options may be unavailable.
	
	CAUSE
	=====
	
	This problem can occur if a formatting error occurred after you chose the Copy
	System Files Only option and clicked Start.
	
	RESOLUTION
	==========
	
	To correct this problem, follow these steps:
	
	1. Close the Format dialog box and then open it again.
	
	2. Remove the floppy disk from the drive, click the Full option, and then click
	  Start. When you receive an error message, click Cancel.
	
	3. Insert the floppy disk in the drive and format it using the options you want.
