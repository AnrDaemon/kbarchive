## Q132813: Battery Meter Shows Unknown Battery Status On Toshiba Laptop

{% raw %}

	Article: Q132813
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you suspend a Toshiba laptop, change the battery, and then resume Windows 95,
	the battery meter may show an unknown battery status.
	
	CAUSE
	=====
	
	After you change the battery, Toshiba laptops may not return information about
	the new battery to the battery meter. As a result, the battery meter shows an
	unknown battery status.
	
	RESOLUTION
	==========
	
	Shut down the computer and restart Windows so that the Toshiba laptop can report
	the correct information to the battery meter.

	================

	independent of Microsoft; we make no warranty, implied or otherwise, regarding
	these products' performance or reliability.
