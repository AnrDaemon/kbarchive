## Q132628: Resolution Change Does Not Realign Microsoft Office Toolbar

	Article: Q132628
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you dynamically change to a lower display resolution, the Microsoft Office
	version 4.x toolbar may be left on the screen.
	
	CAUSE
	=====
	
	The Microsoft Office toolbar prevents itself from being moved even when the
	operating system tries to move it.
	
	RESOLUTION
	==========
	
	Close the Microsoft Office toolbar before changing the resolution and then
	restart it.
	
	MORE INFORMATION
	================
	
	This behavior does not occur with the Microsoft Office 95 toolbar.
