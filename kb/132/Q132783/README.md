## Q132783: Norton Navigator Causes Page Fault During Hardware Detection

	Article: Q132783
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	During the hardware detection portion of Windows 95 Setup, you receive a page
	fault (at 0028:05880016) error message on a blue screen when the hardware
	detection is about 50 percent complete.
	
	CAUSE
	=====
	
	This problem can be caused by a Beta version of Norton Navigator for Windows 95.
	
	RESOLUTION
	==========
	
	To prevent this problem, disable the Undo feature in Navigator Control Center.
