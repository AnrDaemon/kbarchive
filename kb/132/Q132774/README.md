## Q132774: U.S. Atlas Version 4.0 Does Not Play Video Clips in Windows 95

	Article: Q132774
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The video clips in U.S. Atlas version 4.0 do not play in Windows 95. The viewing
	area in U.S. Atlas is black and the video control buttons are unavailable.
	
	CAUSE
	=====
	
	U.S. Atlas version 4.0 is designed to run with Video For Windows version 1.0
	only.
	
	RESOLUTION
	==========
	
	Contact the manufacturer for an upgrade. U.S. Atlas version 5.0 operates
	properly in Windows 95.
	
	MORE INFORMATION
	================
	
	U.S. Atlas is manufactured by Software Toolworks, a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
