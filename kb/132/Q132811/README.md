## Q132811: Unable to Run Kurzweil Voice Version 1.0

	Article: Q132811
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run the Kurzweil Voice program on Windows 95, you may receive the
	following error message:
	
	  Wrong version of Windows: Version 4.0. VOICE requires Windows version 4.0 or
	  higher.
	
	CAUSE
	=====
	
	The Kurzweil Voice program is version bound to Windows version 3.1 and uses a
	part of the Win32s thunking mechanism that is not compatible with Windows 95.
	
	RESOLUTION
	==========
	
	This version of Kurzweil Voice does not run on Windows 95. Please contact
	Kurwzeil Applied Intelligence for more information about this problem and a
	possible solution.
