## Q132624: Protocols Can Be Installed When Network Is Not Installed

	Article: Q132624
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You can install third-party network protocols for SunSoft PC-NFS, Banyan VINES,
	and Digital (DEC) PATHWORKS networks even though the appropriate network is not
	installed on the computer.
	
	CAUSE
	=====
	
	Setup does not disable the ability to install third-party network protocols
	unless specific networks are detected on the computer.
	
	RESOLUTION
	==========
	
	During Setup, do not install third-party network protocols unless the
	appropriate network is installed and running on the computer.
