## Q132416: PRB: Visual FoxPro Does Not Run Under OS/2

	Article: Q132416
	Product(s): Microsoft FoxPro
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): kbenv
	Last Modified: 15-FEB-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual FoxPro for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Attempting to run Visual FoxPro under OS/2 version 2.1 or OS/2 Warp version 3.0
	generates the following Win32s errorS:
	
	  Missing Numeric CO-Processor Virtual Device Driver (VMCPD)
	
	  Unexpected DOS Error:20
	
	MORE INFORMATION
	================
	
	Visual FoxPro version 3.0 is designed to run under Win32s, Windows 95, and
	Windows NT.
	
	NOTE: Visual FoxPro for Windows requires version 1.25.142.0 or later of Win32s.
	
	OS/2 is manufactured by IBM, a vendor independent of Microsoft; we make no
	warranty, implied or otherwise, regarding this product's performance or
	reliability.
