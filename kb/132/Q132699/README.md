## Q132699: Pressing ALT+SPACEBAR Does Not Open System Menu

	Article: Q132699
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbenv kbui kbimu
	Last Modified: 21-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft 500 Nations, version 1.0, on platform(s):
	   - the operating system: Microsoft Windows NT, on platform(s):
	      - the operating system: Microsoft Windows 
	- Microsoft Dogs for Windows, version 1.0 
	- Microsoft Julia Child: Home Cooking with Master Chefs for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The System menu does not open when you press ALT+SPACEBAR in the programs listed
	in the applies to section of this article.
	
	This problem occurs in Microsoft Windows 3.x and Microsoft Windows 95/98.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the programs listed in the
	applies to section of this article.
	
	RESOLUTION
	==========
	
	To open the program's System menu with the keyboard, press and release the ALT
	key, and then press the SPACEBAR.
	
	MORE INFORMATION
	================
	
	The System menu contains the following commands:
	
	- Restore
	
	- Move
	
	- Size
	
	- Minimize
	
	- Maximize
	
	- Close
