## Q132804: WinBackup Does Not Identify Files with Long Filenames

{% raw %}

	Article: Q132804
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you restore files that you backed up with WinBackup, any file with a long
	filename is not identified correctly.
	
	CAUSE
	=====
	
	Older backup programs, such as WinBackup, do not work correctly with long
	filenames.
	
	RESOLUTION
	==========
	
	Contact the manufacturer of your backup software to see if an upgrade is
	available.
