## Q132821: Toshiba T4900 May Not Switch from LCD to External Monitor

{% raw %}

	Article: Q132821
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you place a Toshiba T4900 computer into its docking station while Windows 95
	is running (warm dock), the display may not switch from the LCD screen to the
	external monitor.
	
	CAUSE
	=====
	
	Toshiba's Plug and Play (PNP) BIOS does not switch the display from the LCD
	screen to the external monitor.
	
	
	RESOLUTION
	==========
	
	For a short-term workaround, press the F5 key to manually toggle the display
	between the LCD screen and the external monitor. Then, contact Toshiba for a PNP
	BIOS upgrade.
	
	MORE INFORMATION
	================
	
	The Toshiba T4900 computer is manufactured by Kabushiki Kaisha Toshiba, a vendor
	independent of Microsoft; we make no warranty, implied or otherwise, regarding
	this product's performance or reliability.
