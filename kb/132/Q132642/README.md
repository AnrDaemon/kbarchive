## Q132642: Saving Clipboard Files May Lose Some Formats

{% raw %}

	Article: Q132642
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you save a .clp file in Clipboard Viewer, some data formats may be lost.
	
	CAUSE
	=====
	
	Some programs, and OLE on behalf of programs, put items on the clipboard in
	formats for which they supply no actual data. These formats cannot be saved to a
	file.
	
	You should use caution when using .clp files to save data.
