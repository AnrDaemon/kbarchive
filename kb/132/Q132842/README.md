## Q132842: Act 2.04 Prints to Color Printer in Black and White

	Article: Q132842
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you print from Act version 2.04 to a color printer, the output may be in
	black and white.
	
	CAUSE
	=====
	
	The file that you are printing was saved while a black and white printer was
	selected as the default printer.
	
	RESOLUTION
	==========
	
	To resolve this problem, use either of the following methods:
	
	- Before you print the file, follow these steps:
	
	  1. On the File menu, click Printer Setup.
	
	  2. Click Options.
	
	  3. On the Graphics tab, click Color, then click "Print in color with image
	     color matching the host."
	
	  4. Click OK and Apply.
	
	  5. Print the file.
	
	  Note that you must follow these steps each time you print the document.
	
	- Retype the document and save it while a color printer is selected as the
	  default printer.
