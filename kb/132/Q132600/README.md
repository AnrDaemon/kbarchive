## Q132600: Animations Flicker in Lion King 1.0

	Article: Q132600
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are running The Lion King version 1.0, the animations do not play
	correctly. Parts of the characters are not drawn in some frames, making portions
	of the characters seem to disappear briefly.
	
	CAUSE
	=====
	
	Windows 95 handles the timing of animation playback differently than Windows
	3.x.
	
	RESOLUTION
	==========
	
	To correct this problem, upgrade to The Lion King version 1.1.
