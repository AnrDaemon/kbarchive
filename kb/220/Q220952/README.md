## Q220952: XWEB: &quot;There Were Problems Sending The Item&quot; Saving Contact in O

	Article: Q220952
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook Web Access, version 5.5 Service Packs 1, 2, 3 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the Outlook Web Access (OWA) client to access your Contact folder
	and attempt to save a new contact, the following error message may occur:
	
	  There were problems sending the item. Your server may have been temporarily
	  unavailable or you may not have permissions to send or change the item. Try
	  again or cancel.
	
	CAUSE
	=====
	
	Selecting the "Use Microsoft Schedule Plus as my Primary Calendar" option will
	cause this behavior to occur.
	
	WORKAROUND
	==========
	
	Access the Contacts folder using a standard Outlook client, or use the Outlook
	Calendar rather than Schedule+ as your primary calendar.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
