## Q220968: Microsoft Windows Language Versions Supported by DirectX 6.1

	SUMMARY
	=======
	
	This article contains a list of the Microsoft Windows language versions
	supported by DirectX 6.1.
	
	NOTE: If you install DirectX 6.1 on a computer running a language version of
	Windows that does not appear on this list, the language in the dialog boxes used
	by DirectPlay to connect to other computers changes to English. In addition, you
	may be unable to connect to a multiplayer game.
	
	MORE INFORMATION
	================
	
	DirectX 6.1 supports the following language versions of Microsoft Windows:
	
	- Chinese (Simplified)
	- Chinese (Traditional)
	- Czech
	- Dutch
	- English
	- French
	- German
	- Italian
	- Japanese
	- Korean
	- Polish
	- Portuguese
	- Russian
	- Spanish
	- Swedish
	
	Additional query words: msgame direct-x dx61 dx6.1 foreign international localized
