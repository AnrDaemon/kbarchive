## Q220704: W3C Logging Defaults to Extended Properties

	Article: Q220704
	Product(s): Internet Information Server
	Version(s): winnt:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 04-FEB-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Internet Information Services version 5.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use Internet Explorer 3.x to administer your Web site with the HTML
	Administration pages (Internet Services Manager HTML), you may encounter
	problems with the logging properties on your Web site(s).
	
	When you view the logging properties, the extended logging properties are
	launched instead of the general properties.
	
	CAUSE
	=====
	
	This behavior is due to a limitation in Internet Explorer 3.x.
	
	RESOLUTION
	==========
	
	To resolve this problem, upgrade to Internet Explorer 4.x.
