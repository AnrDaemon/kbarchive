## Q252678: INFO: No API to Programmatically Manipulate Disk Partitions

	Article: Q252678
	Product(s): Microsoft Windows NT
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbOSWin2000 kbDSupport
	Last Modified: 24-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 2000 Advanced Server 
	- Microsoft Windows 2000 Server 
	- Microsoft Windows 2000 Professional 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Developers may encounter application designs that aim to use program code to
	create or delete partitions on dynamic disks under Windows 2000.
	
	Currently, there is no API available to perform these operations using program
	code.
