## Q252901: SMS: Status Filter Rules Restored to Defaults After a Site Reset

	Article: Q252901
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:2.0
	Operating System(s): 
	Keyword(s): kbtool kbConfig kbMMC kbsms200 kbsms200bug kbsmsAdmin
	Last Modified: 01-JUN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you perform a site reset, Status Filter Rules that had been previously
	modified are restored to their default configurations, and all of your previous
	modifications are not preserved.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 2.0.
