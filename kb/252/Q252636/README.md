## Q252636: Allegiance: Sphere Around Object Is Opaque

	Article: Q252636
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbhw igz kbimu msgame kbHardware
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Allegiance, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you build an object in Microsoft Allegiance, the sphere that appears around
	the object may be opaque instead of translucent.
	
	CAUSE
	=====
	
	This behavior can occur if the video adapter installed in your computer is based
	on the Voodoo2 chip set.
