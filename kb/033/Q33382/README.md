## Q33382: DOCERR: Programmer's Guide FREE ST(num) Reference Is Incorrect

	Article: Q33382
	Product(s): Microsoft Macro Assembler
	Version(s): 5.0,5.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Macro Assembler (MASM), versions 5.0, 5.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	On Page 406 of the "Microsoft Macro Assembler 5.1 Programmer's Guide," the FFREE
	instruction is incorrectly given as FREE. The instruction appears correctly in
	the "Microsoft Macro Assembler 5.1 Reference".
	
	Additional query words: 5.00 5.10
