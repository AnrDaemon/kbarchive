## Q112266: PC DOS: Updating Monitor Hotkey Status

	Article: Q112266
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0,3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, versions 3.0, 3.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you change the hotkey when using the Monitor program with versions 3.0x and
	3.2 of Microsoft Mail for PC Networks, MS-DOS workstation, the change is not
	updated in the hotkey status box when you exit Monitor.
	
	You must unload and restart the Monitor program to refresh the contents of the
	status box.
