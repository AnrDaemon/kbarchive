## Q112990: Zoom F9624 Fax Modem Requires ROM Revision 1.15 or Later

	Article: Q112990
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.11 
	-------------------------------------------------------------------------------
	
	The Zoom model F9624 fax modem requires ROM revision of 1.15 or later to
	work correctly with Microsoft At Work PC Fax.
	
	In Microsoft Windows for Workgroups Terminal, you can determine the ROM
	revision of your modem by typing "ATI3" (without the quotation marks). This
	command should echo the revision number.
	
	Additional query words: 3.11 send/receive 96/24
