## Q112921: PC Win: Wrong Error Creating Shared Folder with Network Down

	Article: Q112921
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you try to create a shared folder using version 3.2 of Microsoft Mail for
	Windows when you do not have network access, you will receive the following
	error:
	
	  Not enough memory or system resources to complete transaction
	
	This error is an invalid error message. The error message should be about not
	being able to connect to network.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in version 3.2 of Microsoft Mail
	for Windows. We are researching this problem and will post new information here
	in the Microsoft Knowledge Base as it becomes available.
