## Q112920: PC WSPlus: Users Disappear from Set Access Privileges List

	Article: Q112920
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:1.0,1.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, versions 1.0, 1.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Schedule+ for Windows discards users who have been added with access privileges
	that are of equal or lesser value than default access privileges.
	
	CAUSE
	=====
	
	There is no need to keep track of such privileges; therefore, Schedule+ for
	Windows discards users who have been added with such privileges.
	
	
	Additional query words: schedule plus 1.00a delete remove gone go away
