## Q112236: FaxDrv: WinWord Graphics Elongated by FAX Printer Driver

	Article: Q112236
	Product(s): Microsoft Mail For PC Networks
	Version(s): 3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail FAX Printer Driver, version 3.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Microsoft Mail Gateway to FAX, FAX Printer Driver (FPD) may elongate
	graphics created with Microsoft Word for Windows (WinWord). The degree of
	elongation depends on the format of the graphic image embedded in the document
	and Windows screen driver being used.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft Mail Gateway to
	FAX, FAX Printer Driver, version 3.0a. We are researching this problem and will
	post new information here in the Microsoft Knowledge Base as it becomes
	available.
