## Q112231: FAX: FAX Boards Supported in Version 3.0a

	Article: Q112231
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to Fax, version 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Page 2 of the "Microsoft Mail Gateway to FAX Administrator's Guide" improperly
	states that the GammaLink GammaFAX XP board is supported: it is not supported.
	
	These FAX boards are supported with the Microsoft Mail FAX Gateway version 3.0a:
	
	  Intel SatisFAXtion
	  Intel SatisFAXtion CoProcessor
	  Intel SatisFAXtion Modem 200
	  GammaLink GammaFAX CP
	  GammaLink GammaFAX CP-DFX
	  Kortex Fax Modem
	  Ferrari Ferrari Modem
	
	The error is corrected in the "Microsoft Mail Gateway to FAX Release Notes"
	included with version 3.0a.
