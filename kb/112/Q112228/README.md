## Q112228: FAX: FAX Boards Supported with the FAX Gateway Version 3.0

	Article: Q112228
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to Fax, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Page 2 of the "Microsoft Mail Gateway to FAX Administrator's Guide" improperly
	states that the GammaLink GammaFAX XP board is supported. It is not supported.
	
	These FAX boards are supported with the Microsoft Mail FAX Gateway version 3.0:
	
	  Intel SatisFAXtion
	  Intel SatisFAXtion CoProcessor
	  Intel SatisFAXtion Modem 200
	  GammaLink GammaFAX CP
	  GammaLink GammaFAX CP-DFX
	
	The "Microsoft Mail Gateway to FAX Release Notes" included with version 3.0a
	corrects the Administrator's Guide.
	
	Additional query words: 3.00 Intel GammaLink DOCERR
