## Q112922: PC Win: Shortcut Key F1 in Export Dialog Invokes Import Help

	Article: Q112922
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0b,3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, versions 3.0b, 3.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In versions 3.0b and 3.2 of Microsoft Mail for Windows, if you press the
	shortcut key F1 for Help after selecting Export Folder from the File menu, the
	Help option is invoked for Import Folder Dialog Box instead of the Help option
	for Export Folder Dialog Box.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in versions 3.0b and 3.2 of
	Microsoft Mail for Windows. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
