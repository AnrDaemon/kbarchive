## Q112811: Print Notification Message has Incorrect Time and Date

	Article: Q112811
	Product(s): Microsoft Windows NT
	Version(s): 3.1
	Operating System(s): 
	Keyword(s): kbprint kbPrinting
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 3.1 
	- Microsoft Windows NT Workstation version 3.1 
	- Microsoft Windows NT Advanced Server, version 3.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When your print job has finished printing on a remote Windows NT or Windows NT
	Advanced Server print server, the printing notification dialog displays the
	incorrect time and date. This problem happens on all client operating systems.
	
	CAUSE
	=====
	
	The Windows NT print spooler is sending an incorrect date and time.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT and Windows NT
	Advanced Server version 3.1. This problem has been corrected in Windows NT
	Workstation and Windows NT Server version 3.5.
