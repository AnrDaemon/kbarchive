## Q112910: PC WRmt: Selector Hangs if Fax Printer Driver Missing

	Article: Q112910
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Remote for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Microsoft Mail System Selector is a utility that is installed with version
	3.2 of Microsoft Mail Remote for Windows. This utility stops responding (hangs)
	if the Fax printer driver is corrupt or missing. As a result, Mail Remote for
	Windows crashes.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in version 3.2 of Microsoft Mail
	Remote for Windows. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
