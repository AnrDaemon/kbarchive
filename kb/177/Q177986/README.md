## Q177986: FS98: Auto-Coordination Not Available for Bell 206B JetRanger

	Article: Q177986
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator 98 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Flight Simulator 98 Pilot's Help incorrectly indicates that the
	Auto-Coordination feature is available for the Bell 206B JetRanger III
	helicopter.
	
	MORE INFORMATION
	================
	
	
	When you select the Bell 206B JetRanger helicopter, the Auto-Coordination check
	box is unavailable.
