## Q177933: How to Change Menu Font Size in MSN Program Viewer

	Article: Q177933
	Product(s): The Microsoft Network
	Version(s): WINDOWS:2.0,2.5
	Operating System(s): 
	Keyword(s): kbmsn
	Last Modified: 14-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 2.0, 2.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to change the menu font size in MSN, The Microsoft
	Network, Program Viewer.
	
	MORE INFORMATION
	================
	
	To change the menu font size in MSN Program Viewer, use the appropriate steps
	for your version of MSN:
	
	MSN 2.0
	-------
	
	1. Double-click the MSN icon on the desktop. If you are prompted to sign in to
	  MSN, click Connect.
	
	2. Click the MSN icon in the upper-left corner of Program Viewer.
	
	3. Click MSN Settings.
	
	4. If you are prompted to select a country, click OK, and then click your
	  country in the Country box under Review Settings.
	
	5. Click Menu Font Size.
	
	6. Click either Medium or Large, and then click OK.
	
	NOTE: These changes take effect the next time you start Program Viewer.
	
	MSN 2.5
	-------
	
	1. Click the MSN Quick View icon on the taskbar, point to MSN Options, and then
	  click Customize MSN.
	
	2. If you are prompted to sign in to MSN, click Connect.
	
	3. Click Menu Font Size.
	
	4. Click either Medium or Large, and then click Apply.
	
	NOTE: These changes take effect immediately.
