## Q177280: FS98: Global On Cockpit Instruments Not Always Visible

	Article: Q177280
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbdisplay kbenv
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator 98 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you select the instrument panel's Global On setting from a view other than
	Cockpit view, the Global On setting is turned off and the instrument panel
	disappears when you switch between the different views.
	
	RESOLUTION
	==========
	
	To work around this issue, follow these steps:
	
	1. On the Views menu, click Undock View.
	
	2. Close the undocked view window.
	
	3. On the Views menu, point to New View, and then click a view.
