## Q177701: Macintosh Version of Internet Explorer Not Included on Disc

	Article: Q177701
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocerr
	Last Modified: 20-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Official Microsoft Internet Explorer 4 Book ISBN 1-57231-576-8 
	-------------------------------------------------------------------------------
	
	MORE INFORMATION
	================
	
	Download the current version of Microsoft Internet Explorer for the Macintosh
	from the Microsoft Internet Explorer Web site:
	
	  http://www.microsoft.com/ie/
	
	NOTE: Because the Microsoft Web site is constantly updated, the site address may
	change without notice. If this occurs, link to the Microsoft home page at the
	following address:
	
	  http://www.microsoft.com/
	
	
	SUMMARY
	=======
	
	The back cover of the "Official Microsoft Internet Explorer 4 Book" notes that
	the current version of Microsoft Internet Explorer for the Macintosh is included
	on the companion CD-ROM. However, the Macintosh version of Internet Explorer was
	not included with this book.
