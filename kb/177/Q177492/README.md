## Q177492: XADM: Key Management Server Fails to Reissue Key

	Article: Q177492
	Product(s): Microsoft Exchange
	Version(s): winnt:4.0,5.0
	Operating System(s): 
	Keyword(s): exc4 exc5
	Last Modified: 17-FEB-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 4.0, 5.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	The Microsoft Exchange Server Key Management Server (KMS) version 4.0 and 5.0
	may fail to reissue security keys. This problem will continue to occur even
	after a new token has been issued.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server,
	versions 4.0 and 5.0. This problem has been corrected in Microsoft Exchange
	Server, version 5.5.
