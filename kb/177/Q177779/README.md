## Q177779: XADM: Error Message in Store.exe on Messages Sent from PROFS

{% raw %}

	Article: Q177779
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 19-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	Messages sent from PROFS version 1.3B may cause Store.exe to fail and generate
	the following Dr. Watson error message:
	
	  exited. Exception: Access violation (0xc0000005), Address:
	  0x0043af00
	
	This problem is primarily present only when you install Microsoft Exchange Server
	5.0 using the Exchange Server 5.0 CD on a clean computer.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server
	version 5.0. This problem has been corrected in the latest U.S. Service Pack for
	Microsoft Exchange Server version 5.0. For information on obtaining the Service
	Pack, query on the following word in the Microsoft Knowledge Base (without the
	spaces):
	
	  S E R V P A C K
