## Q177787: Age of Empires: WordPad Cannot Read Stratsmp.doc or Strattyp.doc

	Article: Q177787
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbenv kbtool aoe
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Age of Empires, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you open the Stratsmp.doc or Strattyp.doc file in WordPad, the file may
	seem to contain random or unintelligible characters.
	
	These files are located in the Docs folder on the Age of Empires CD-ROM.
	
	CAUSE
	=====
	
	The Stratsmp.doc and Strattyp.doc files are formatted for Microsoft Word 97.
	
	RESOLUTION
	==========
	
	To view the contents of the Stratsmp.doc and Strattyp.doc files, please see the
	following articles in the Microsoft Knowledge Base:
	
	  Q177874 Age of Empires: Contents of the Strattyp.doc File
	
	  Q177861 Age of Empires: Contents of the Stratsmp.doc File
	
	You can also view these files by opening them in Word 97.
	
	NOTE: The version of WordPad included with Microsoft Windows 98 can open Word 97
	files.
	
	MORE INFORMATION
	================
	
	The Stratsmp.doc file contains a sample strategy file. The Strattyp.doc file
	contains a list of strategy files you can use in Age of Empires.
	
	Both of these files are approximately 50 KB in size and can be copied to a floppy
	disk. Insert the floppy disk containing the files into a floppy disk drive on a
	computer that has Word 97 installed. After you open the files in Word 97, print
	the files for future reference.
	
	You can open all other files in the Docs folder with WordPad.
