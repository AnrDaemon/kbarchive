## Q177232: FP/VID VSS Control of OOP Application Causes Random Errors

	Article: Q177232
	Product(s): Internet Information Server
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-JUN-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Internet Information Server 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use Microsoft FrontPage (FP) or Microsoft Visual InterDev (VID) to
	author a site under Microsoft Visual SourceSafe (VSS) to manage an Out-
	of-Process (OOP) application, you may get random failures or random error
	messages.
	
	STATUS
	======
	
	Microsoft has confirmed this to be problem in Microsoft Internet Information
	Server version 4.0.
