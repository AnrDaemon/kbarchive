## Q256914: XIMS: Error Message: Your Configuration May Not Be Complete

	Article: Q256914
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5
	Operating System(s): 
	Keyword(s): exc55
	Last Modified: 24-MAR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you close the Internet Mail Service properties, the following error
	message may be displayed:
	
	  Your configuration may not be complete. You will not receive Internet
	  messages unless you configure other SMTP hosts to deliver to this Internet
	  Mail Service.
	
	  You can use Domain Name System (DNS) or Host tables to deliver mail to this
	  Internet Mail Service. For more information, see Microsoft Exchange Server
	  Operations Guide or Getting Started Guide.
	
	CAUSE
	=====
	
	This informational "error message" is only displayed once. It is displayed when
	the Internet Mail Service properties are opened and closed for the first time.
	This error message informs new administrators of general Internet Mail Service
	procedures and resources.
