## Q256245: XADM: Mailbox Manager Deletes All Recurring Calendar Items

	Article: Q256245
	Product(s): Microsoft Exchange
	Version(s): 5.5,5.5 SP1,5.5 SP2,5.5 SP3
	Operating System(s): 
	Keyword(s): exc55 exc55sp1 exc55sp2 exc55sp3 kbExchange550preSP4fix kbExchange550sp4Fix
	Last Modified: 25-JAN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 5.5, 5.5 SP1, 5.5 SP2, 5.5 SP3 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you use Mailbox Manager to delete recurring Calendar items that are older
	than N days, all instances of a recurring appointment are deleted.
	
	RESOLUTION
	==========
	
	To resolve this problem, obtain the latest service pack for Exchange Server 5.5.
	For additional information, click the following article number to view the
	article in the Microsoft Knowledge Base:
	
	  Q191014 XGEN: How to Obtain the latest Exchange Server 5.5 Service Pack
	
	
	STATUS
	======
	
	Microsoft has confirmed that this is a problem in the Microsoft products that
	are listed at the beginning of this article. This problem was first corrected in
	Exchange Server 5.5 Service Pack 4 Support Utilities. This Service Pack must be
	installed separately from the normal Service Pack 4 Setup.
	
	The path to run this update from the Exchange 5.5 Service Pack 4 CD is:
	
	  \ENG\SERVER\SUPPORT\MBMNGR\SETUP\I386\SETUP.EXE.
