## Q147844: Dial-Up Networking Server Not Supported with ISDN Adapter

	Article: Q147844
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): dun win95 kbDialUp
	Last Modified: 28-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Your ISDN adapter may not appear as an available device in Dial-Up Networking
	Server configuration dialog boxes.
	
	MORE INFORMATION
	================
	
	The current version of the ISDN Accelerator Pack for Microsoft Internet Explorer
	2.0 does not support using your ISDN adapter with the Dial-Up Networking Server
	tool included with Microsoft Plus! for Windows 95.
