## Q147419: Macintosh Frank Lloyd Wright: Minimum System Requirements

	Article: Q147419
	Product(s): Microsoft Home Multimedia Titles
	Version(s): MACINTOSH:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft The Ultimate Frank Lloyd Wright for the Macintosh, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following article contains a copy of the minimum system requirements
	information printed on the Microsoft The Ultimate Frank Lloyd Wright for the
	Macintosh program package.
	
	MORE INFORMATION
	================
	
	The minimum system requirements to run The Ultimate Frank Lloyd Wright for the
	Macintosh are:
	
	- Any Apple Macintosh computer supporting a 256-color, 640x480 resolution
	  (13-inch), or higher resolution monitor.
	
	- System 7 or later.
	
	- Minimum 8 MB RAM memory.
	
	- Minimum 2 MB available hard-disk space.
	
	- Double-speed CD-ROM: The CD-ROM drive should be able to transfer data from
	  the CD-ROM drive at 300 KB per second while using less than 40 percent of the
	  CPU bandwidth and has an average seek time of less than 400 milliseconds.
	
	- Speakers or headphones recommended.
