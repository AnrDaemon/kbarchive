## Q147382: Number of Message Recipients Limited to 64

	Article: Q147382
	Product(s): The Microsoft Network
	Version(s): 2.6,5.0,5.1,5.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 2.6, 5.0, 5.1, 5.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	On MSN, The Microsoft Network, you can address an e-mail message to a maximum of
	64 addresses. This number is the total from the To, Cc, and Bcc boxes.
	
	This limit includes the contents of distribution lists. Therefore, if a
	distribution list contains 64 entries, you can send a message to that
	distribution list only.
