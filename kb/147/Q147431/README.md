## Q147431: Cannot Hide Host Drive for Compressed Volume on Removable Disk

	Article: Q147431
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you try to hide the host drive for a compressed volume created on a
	removable drive, the Hide Host Drive check box is unavailable.
	
	MORE INFORMATION
	================
	
	This behavior is by design. You cannot hide the host drive for compressed
	volumes created on removable drives. This includes floppy disk drives, Zip
	drives, Bernoulli drives, and external hard disk drives.
	
	Additional query words: grayed out greyed dimmed
