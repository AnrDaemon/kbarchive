## Q147345: Error Message: The Search Could Not Be Started

	Article: Q147345
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you try to use the Microsoft Exchange Find command on a public or shared
	folder on a Microsoft Mail 3.x post office or a Workgroups post office, you
	receive the following error message:
	
	  The search could not be started [80040102]
	
	MORE INFORMATION
	================
	
	These post offices do not support the Microsoft Exchange Find command on a
	shared or public folder.
