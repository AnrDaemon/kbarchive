## Q147325: Julia Child: Minimum System Requirements

	Article: Q147325
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1.00
	Operating System(s): 
	Keyword(s): kbfaq
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Julia Child: Home Cooking with Master Chefs for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following article lists the minimum system requirements to run Microsoft
	Julia Child: Home Cooking with Master Chefs version 1.0 for Windows.
	
	MORE INFORMATION
	================
	
	The minimum system requirements to run Julia Child are:
	
	- A Multimedia PC or compatible with a 386SX or higher microprocessor.
	
	- MS-DOS operating system 3.1 or later.
	
	- Microsoft Windows operating system version 3.1 or later, including Windows
	  95.
	
	- Double-speed (2x) or better CD-ROM drive.
	
	- 4 MB of RAM.
	
	- 6 MB of available hard disk space.
	
	- VGA or higher-resolution monitor, minimum 256-color support.
	
	- Windows-compatible sound card.
	
	- Speakers or headphones.
	
	- Microsoft Mouse or compatible pointing device.
