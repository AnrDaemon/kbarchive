## Q147290: Changing Focus From Role Box Loses Newly Typed Data

	Article: Q147290
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0; Win95:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you choose Seven Habits Tools from the Tools menu, type a role in the Role
	box and change the focus to another object, such as the Seven Habits Tools
	window, the data typed is lost when you choose Add.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in version 7.0 of Microsoft
	Schedule+ for Windows. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
