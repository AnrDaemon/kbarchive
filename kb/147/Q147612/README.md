## Q147612: Microsoft Internet Jump Start CD for Education

	Article: Q147612
	Product(s): Windows for Workgroups and Windows NT Networking Issues
	Version(s): 1.0,1.0z,2.0z,3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Internet Jump Start CD for Education 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft(R) Internet Jump Start CD for Education is a set of Microsoft
	software and tools for exploring and publishing on the Internet. This CD-ROM is
	available for no charge to K-12 educational institutions in the U.S. It provides
	all the software a school needs for Web page creation and publishing, including
	the network operating system and the Microsoft Internet Information Server to
	host a Web site that's your very own. Or, create your school's home page and let
	us host it free at the Global Schoolhouse site on the World Wide Web - all
	possible thanks to a collaboration between Microsoft, MCI and the Global
	SchoolNet Foundation.
	
	MORE INFORMATION
	================
	
	The Internet Jump Start CD for Education includes:
	
	- Microsoft Windows NT (TM) 3.51 Server This is a special version of Windows NT
	  Server limited to 2 File and Print connections at one time.
	
	- Microsoft Internet Information Server (IIS)
	
	- Microsoft Word Internet Assistant for Windows(R) (1.0z and 2.0z)
	
	- Microsoft Word Internet Assistant for the Macintosh(R) (Beta)
	
	- Microsoft Internet Explorer This includes versions for Windows(R) 3.1,
	  Windows(R) 95, Windows NT, and Macintosh(R).
	
	- WebWhacker by the ForeFront Group, Inc. This tool downloads Web pages and
	  creates local versions of them.
	
	Information and links to Web sites for K-12 educators.
	
	- The contents of the Education Web site. http://www.microsoft.com/education/
	
	- The contents of the Global Schoolhouse Web site. http://www.gsh.org/
	
	If you need technical support for the Internet Jump Start CD for Education, see
	the electronic support options available on the Microsoft Internet web site at:
	
	  http://www.microsoft.com/support/
	
	This site includes a link to the Microsoft Knowledge Base, which allows You to
	search the database of over 70,000 answers to the technical support questions.
	
	Support for WebWhacker is provided by the ForeFront Group, Inc.
	
	For Support via email from Microsoft for your Internet Jump Start CD for
	Education, email JumpStart@microsoft.com
	
	There is no free phone support included with the Internet Jump Start CD For
	Education. However the paid support options are available through Microsoft
	Technical Support services. For more information contact Microsoft Support Sales
	at 1(800)936-3500.
