## Q122929: DEC Alpha STOP Message: System File S3 is Bad or Missing

	Article: Q122929
	Product(s): Microsoft Windows NT
	Version(s): 3.50
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	3.50
	
	WINDOWS
	
	kbhw
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.5 
	- Microsoft Windows NT Server version 3.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The following STOP message may appear when you install Windows NT version 3.5 on
	a DEC Alpha computer:
	
	  Stop C0000143, system file S3 is bad or missing.
	
	CAUSE
	=====
	
	This problem is caused by using an incorrect firmware version.
	
	RESOLUTION
	==========
	
	To determine which firmware version you should be using, please see the DEC
	README.TXT file, which can be downloaded from the DEC4WNT forum on CompuServe,
	or call DEC technical support at (800) DIGITAL (344-4825).
