## Q122526: IntelliPoint Mouse Software Not Supported Under Windows NT

	Article: Q122526
	Product(s): Microsoft Windows NT
	Version(s): ; WINDOWS:2000; winnt:3.5,3.51,4.0
	Operating System(s): 
	Keyword(s): kbinterop kbsetup
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 3.1 
	- Microsoft Windows NT Workstation version 3.1 
	- Microsoft Windows NT Advanced Server 
	- Microsoft Windows NT Workstation versions 3.5, 3.51, 4.0 
	- Microsoft Windows NT Server versions 3.5, 3.51, 4.0 
	- Microsoft Windows 2000 Professional 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you attempt to install the Microsoft IntelliPoint 1.0, 1.1, or 3.0 mouse
	software, the following error message appears:
	
	  Setup cannot continue. This software runs with the Microsoft Windows 3.x
	  operating system. Contact Microsoft Customer Service for an upgrade to run
	  with later versions of Microsoft Windows and Microsoft Windows NT operating
	  systems.
	
	NOTE: This error message no longer appears when you install IntelliPoint 3.0 or
	later on Windows NT 4.0 Service Pack 3 or later. Also, both the mouse pointer
	and keyboard may stop working if you are using the Universal Serial Bus (USB)
	version of the Microsoft IntelliMouse and it is plugged into the keyboard.
	
	CAUSE
	=====
	
	These IntelliPoint versions (3.0 and earlier) are not compatible with any
	version of Windows NT or Windows 2000.
	
	RESOLUTION
	==========
	
	Download the IntelliPoint software version 3.1a for any of the following
	operating systems: Microsoft Windows 95; Microsoft Windows 98; Microsoft Windows
	98 Second Edition; Windows NT 4.0 with Service Pack 3, 4, or 5; or Windows 2000
	from the following Microsoft Web site:
	
	  http://www.microsoft.com/products/hardware/mouse/driver/drivers_pc31a.htm
