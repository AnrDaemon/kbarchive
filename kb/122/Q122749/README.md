## Q122749: Arcade for Macintosh: Minimum System Requirements

{% raw %}

	Article: Q122749
	Product(s): Microsoft Home Games
	Version(s): MACINTOSH:1.0
	Operating System(s): 
	Keyword(s): kbsetup kbimu msgamekbfaq
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Arcade for the Macintosh, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes the minimum system requirements to run Microsoft Arcade
	for the Macintosh, version 1.0.
	
	MORE INFORMATION
	================
	
	The minimum system requirements to run Microsoft Arcade for the Macintosh are:
	
	- A Macintosh II-family or faster computer, or a PowerBook(TM) 160 or faster
	  computer.
	- A 68020 or faster processor.
	- Apple (R) System 7 operating system or higher.
	- 3 megabytes (MB) of random access memory (RAM).
	- 3 MB of available hard-disk space.
	- A 16 color or 256 color video adapter and monitor.
