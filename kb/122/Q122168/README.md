## Q122168: WFWG Hangs with Intel EtherExpress ISA and Adaptec 2940 Cards

	Article: Q122168
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	On a computer with both an Intel EtherExpress ISA network card and an Adaptec
	AHA 2940 PCI SCSI controller, your computer stops responding (hangs) or a
	general protection (GP) fault occurs when you start Windows for Workgroups.
	
	CAUSE
	=====
	
	According to Adaptec, the problem is caused by the Intel EtherExpress ISA
	network card.
	
	STATUS
	======
	
	No workaround is available at this time.
	
	Additional query words: 3.10 3.11 hang hung crash crashed locks locked frozen freezes crashing quit quits stopped crashes bomb bombs bombed 3rdparty gpf
