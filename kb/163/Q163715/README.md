## Q163715: ODE97: ODE Help Workshop - Cancel and Close Buttons Fail

{% raw %}

	Article: Q163715
	Product(s): Microsoft Access Distribution Kit
	Version(s): 
	Operating System(s): 
	Keyword(s): kbui
	Last Modified: 25-JUN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Office 97 Developer Edition 
	-------------------------------------------------------------------------------
	
	
	Novice: Requires knowledge of the user interface on single-user computers.
	
	SYMPTOMS
	========
	
	When you click the New button in Microsoft Help Workshop, the New dialog box is
	opened. In the New dialog box, if you click the Cancel button or the Close
	button, the dialog box remains open.
	
	RESOLUTION
	==========
	
	To close the New dialog box, you must first click OK to open a new Help Project,
	and then click Cancel in the Project File Name box.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Office 97 Developer
	Edition Tools.
	
	MORE INFORMATION
	================
	
	Steps to Reproduce Problem
	--------------------------
	
	1. On the Start menu, point to Programs, point to Microsoft ODE Tools, and then
	  click Microsoft Help Workshop.
	
	2. On the File menu, click New.
	
	3. In the New dialog box, click Cancel or click the Close button. Note that the
	  New dialog box remains on the screen.
