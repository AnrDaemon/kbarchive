## Q163201: XCLN: Illegal Operation Err Msg When Form Minimized

	Article: Q163201
	Product(s): Microsoft Exchange
	Version(s): Win95:4.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 22-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Client for Windows 95, version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	When you quit Microsoft Windows 95 while a minimized Exchange Forms Designer Form
	for Microsoft Exchange is running, the following exception may occur:
	
	  <Form executable name>
	  This program has performed an illegal operation and will be shut down.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem with Microsoft Exchange version
	4.0. We are researching the problem and will post new information here in the
	Knowledge Base as it becomes available.
