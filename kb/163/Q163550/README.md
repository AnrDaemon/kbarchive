## Q163550: WD97: Letter Wizard Subject Line Not Selected

	Article: Q163550
	Product(s): Word 97 for Windows
	Version(s): 
	Operating System(s): 
	Keyword(s): kbualink97 kbtemplate
	Last Modified: 06-MAR-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you select the Subject check box on the Other Elements tab in the Letter
	Wizard, and you create a letter, the Subject check box may not be selected when
	you click Letter Wizard on the Tools menu again.
	
	WORKAROUND
	==========
	
	Do not use the Letter Wizard on the Tools menu to start your letter. Start your
	letter using the Letter Wizard template. To do this, use the following steps:
	
	1. On the File menu, click New.
	
	2. Click the Letter & Faxes tab.
	
	3. Click the "Letter Wizard.wiz" icon, and then click OK.
	
	4. From the Letter Wizard, click the Other Elements tab. Under Include, be sure
	  to click Subject. Then finish the wizard.
	
	Once the letter is created, you can use the Letter Wizard on the Tools menu and
	Subject will be correctly selected on the Other Elements tab.
