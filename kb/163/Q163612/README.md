	SUMMARY
	=======
	
	This article contains a sample Microsoft Visual Basic for Applications macro
	(Sub procedure) that applies bold formatting to the current selection.

	Sample Visual Basic Procedure
	-----------------------------
	
	NOTE: This code will generate a run-time error if nothing is selected or if the
	current selection is invalid.
	
	     Sub SetFontBold()
	
	        ' Used to store a font object reference.
	        Dim oFont As font
	
	        ' Get a reference to the font property.
	        Set oFont = ActiveWindow.Selection.TextRange.font
	
	        ' See whether the selected text is bold.
	        If oFont.Bold <> msoTrue Then
	           oFont.Bold = msoTrue
	        Else
	           ' The font selection was already bold.
	           MsgBox "The selected text is already bold."
	        End If
	
	     End Sub
