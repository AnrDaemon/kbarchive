## Q163476: MSB Ocean: Files Installed on the Macintosh

	Article: Q163476
	Product(s): Microsoft Home Kids Products
	Version(s): MACINTOSH:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Scholastic's Magic School Bus series: Explores the Ocean for the Macintosh, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following article lists the files and folders installed by The Magic School
	Bus Explores the Ocean for the Macintosh.
	
	MORE INFORMATION
	================
	
	The following files and folders are installed to the hard disk when you use the
	Magic School Bus installation program:
	
	Files and Folders Installed on the Hard Disk
	--------------------------------------------
	
	  MSKids (folder)
	     Magic School Bus - Ocean (folder)
	        Magic School Bus - The Ocean (application)
	        Data (folder) This folder contains numerous data files.
	
	  System Folder
	     Extensions
	        Sound Manager Extension (version 3.1)
	        QuickTime (version 2.1)
	        QuickTime PowerPlug (version 2.1))
	        QuickTime Musical Instruments (version 2.1)
	        Indeo Video (version 3.22.24.09)
	        Intel Raw Video (version 1.10.20.02)
	
	     Control Panels
	        Sound (version 8.0.5)
