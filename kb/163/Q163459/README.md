## Q163459: WD97: Footnote/Endnote Deleted by Grammar Checker

	Article: Q163459
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbualink97 kbproof kbfield
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the Grammar Checker, footnotes or endnotes may be deleted from your
	document.
	
	CAUSE
	=====
	
	This problem occurs when the following conditions are true:
	
	- A custom reference mark of @ (an "at" sign) was used to mark a footnote or
	  endnote.
	
	  -and-
	
	- The custom reference mark follows a period.
	
	  -and-
	
	- You click the Change button in the Spelling & Grammar dialog box while
	  the Grammar Checker is running.
	
	
	WORKAROUND
	==========
	
	Use a different custom footnote or endnote reference mark.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
	
	This problem has been corrected in Word 2000.
