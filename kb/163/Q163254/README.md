## Q163254: WD97: Valupk8.hlp Topic for Macintosh Converter Incorrect

	Article: Q163254
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The topic for the Microsoft Word 97 converter found in the Valupk8.hlp states
	that the Microsoft Word 97 converter converts documents from Microsoft Word 97
	format to the Macintosh format. This information is incorrect.
	
	WORKAROUND
	==========
	
	In Word 97, save the document you want to open on the Macintosh to a format
	supported by your Macintosh program. For example:
	
	To save a Word 97 document to Microsoft Word 5.1 for the Macintosh document
	format:
	
	1. In Word 97, on the File menu, click Save As.
	
	2. In the "Save As Type" box, click a format supported by your Macintosh
	  program.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the products listed at the
	beginning of this article. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
	
	MORE INFORMATION
	================
	
	The Microsoft Word 97 converter will not install on the Macintosh and will not
	convert documents from Microsoft Word 97 format to the Macintosh.
