## Q163377: XADM: Messages Forwarded from All-In-One Have Wrong Sent Date

	Article: Q163377
	Product(s): Microsoft Exchange
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 16-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When a Dec All-In-One user sends a message to a Microsoft Exchange Server user
	through XMR and Mailbus 400, the message arrives at Microsoft Exchange Server
	with the date in the sent date field of the XMR forwarding, not the actual date
	of the original message.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server 4.0.
	This problem was corrected in the latest Microsoft Exchange 4.0 U.S. Service
	Pack. For information on obtaining the service pack, query on the following word
	in the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
