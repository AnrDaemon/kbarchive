## Q163228: Printing from Windows NT 4.0 to NetWare 3.12 Seems Slow

	Article: Q163228
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kbnetwork 3rdpartynet kb3rdPartyNetClient
	Last Modified: 10-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Connectivity between Client Services for NetWare (CSNW) and Novell NetWare 3.12
	print queues may be slow in Windows NT Workstation 4.0. Programs that use
	printers connected to NetWare 3.12 print queues may seem to start slowly.
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information in the
	Microsoft Knowledge Base as it becomes available.
