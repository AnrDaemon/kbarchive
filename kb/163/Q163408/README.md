## Q163408: ODE97: MSDN Sampler: License Help Topic Incorrect

	Article: Q163408
	Product(s): Microsoft Access Distribution Kit
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): 
	Last Modified: 25-JUN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Access 97 
	- Microsoft Office 97 Developer Edition 
	-------------------------------------------------------------------------------
	
	Novice: Requires knowledge of the user interface on single-user computers.
	
	SYMPTOMS
	========
	
	When you open the License.hlp file that comes with the Microsoft Office
	Developer Sampler, you will see the following instruction:
	
	  When you are finish reading the License Agreement, click Accept to
	  return to Setup.
	
	This information is incorrect because there is no Accept button to click.
	
	CAUSE
	=====
	
	The License.hlp file is designed to be opened during the Setup process.
	
	RESOLUTION
	==========
	
	To close the License.hlp file, click Done.
	
	MORE INFORMATION
	================
	
	When you run Setup for the Microsoft Office Developer Sampler, the License.hlp
	file is displayed automatically in an Office Developer Edition Sampler Setup
	dialog box. This dialog box contains the Accept button you will need to click to
	Accept the License agreement.
