## Q163462: Kids Plus!: Fonts Installed with Microsoft Plus! For Kids

	Article: Q163462
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Plus! for Kids, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft Plus! For Kids installs 33 new fonts on your computer. The following
	is a list of those fonts:
	
	Alger, Arial, Arial Bold, Arial Italics, Arial Narrow, Arial Narrow Bold Italics,
	Arial Narrow Italics, Comic Sans, Comic Sans Bold, Curlz, Gothic, Gothic Bold,
	Gothic Bold Italics, Gothic Italics, Haettenschweiler, Lucida Handwriting
	Italics, Lucida Sans, Matisse, Mercurius Script Bold, OCR A Extended, Ransom,
	Ransom Bold, Ransom Bold Italics, Ransom Italics, Snap ITC, Stop, Tempus Sans
	ITC, Viner Hand ITC, Vladimir, Westminster, Western, Wingdings 2, Wingdings 3.
	
	MORE INFORMATION
	================
	
	It may not appear that Microsoft Plus! For Kids installed 33 new fonts, because
	some of the fonts may have already been installed on your computer.
