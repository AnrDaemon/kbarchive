## Q163399: Matrox Millennium PCI Video Card Causes STOP 0X0000000A

	Article: Q163399
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbhw kbHardware
	Last Modified: 10-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Computers using the Matrox Millennium PCI video card may experience an
	intermittent STOP 0X0000000A error.
	
	
	RESOLUTION
	==========
	
	To resolve this problem, do either of the following:
	
	- Change the driver to VGA Compatible Display Adapter at 640 x 480 resolution.
	
	  -or-
	
	- Obtain new Windows NT driver files and card BIOS from Matrox. The Matrox Web
	  site is:
	
	  http://www.matrox.com/
