## Q163096: WD97: Error Installing Alki Language Proofing Tools

	Article: Q163096
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbinterop kbsetup kbproof word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to install the Alki(R) Language Proofing Tools for Word 7.0 for
	Windows on a computer running Microsoft Word 97 for Windows, you will receive
	the following error message:
	
	  Setup is unable to locate Word 7.0. Make sure Word 7.0 is installed, then run
	  Proofing Tools Setup again.
	
	When you click OK, the following error message is displayed:
	
	  Microsoft Word Proofing Tools Setup was interrupted before your new software
	  could be fully installed. You can run the Microsoft Word Proofing Tools Setup
	  again at a later time to complete the installation.
	
	CAUSE
	=====
	
	The Alki Language Proofing Tools for Word 7.0 for Windows software was designed
	to work with Microsoft Word for Windows 95, versions 7.0 and 7.0a.
