## Q163294: Exchange MTA Build Numbers

{% raw %}

	Article: Q163294
	Product(s): Microsoft Exchange
	Version(s): 4.0,5.0,5.5
	Operating System(s): 
	Keyword(s): exc4 exc5 exc55
	Last Modified: 12-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 4.0, 5.0, 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The build numbers for the Microsoft Exchange Server message transfer agent (MTA)
	are as follows:
	
	  Version 4.0 RTM    Build 837.9
	  Service Pack 1     Build 838.10
	  Service Pack 2     Build 993.6
	  Version 4.0a       Build 993.6
	  Service Pack 3     Build 994.54
	  Service Pack 4     Build 995.54
	  Service Pack 5     Build 996.72
	  Version 5.0 RTM    Build 1457.8
	  Service Pack 1     Build 1458.56
	  Service Pack 2     Build 1460.13
	  Version 5.5 RTM    Build 1960.3
	  Service Pack 1     Build 2232.18
	  Service Pack 2     Build 2448.1
	  Service Pack 3     Build 2650.24
	  Service Pack 4     Build 2653.23
	
	
	MORE INFORMATION
	================
	
	You can verify the MTA build on a Microsoft Exchange Server by opening
	MSExchangeMTA X.400 Service events 9298 or 9299 in the Windows NT Event Viewer.
	These events are logged at MTA startup and shutdown respectively.
