## Q163226: PaperPort Driver Setup Button May Be Unavailable

	Article: Q163226
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): kbsetup win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you install the PaperPort driver that is included in Microsoft Windows 95
	Service Pack 1, you are required to enter your name and the serial number of
	your PaperPort product. The Continue button is unavailable until this
	information is entered.
	
	MORE INFORMATION
	================
	
	The PaperPort driver included in the Drivers\Misc\Pportdrv folder on the Windows
	95 Service Pack 1 CD-ROM is made exclusively for the Visioneer PaperPort
	scanner. Please contact Visioneer if you need assistance in finding the serial
	number for your product.
	
	For additional information about Service Pack 1, please see the following article
	in the Microsoft Knowledge Base:
	
	  Q142794 Availability of Microsoft Windows 95 Service Pack 1
	
	For information about contacting Visioneer, please see the following article in
	the Microsoft Knowledge Base:
	
	  Q60782 Hardware and Software Third-Party Vendor Contact List, Q-Z
