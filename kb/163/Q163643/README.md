## Q163643: XFOR: Notifications not Delivered to Public Folder

{% raw %}

	Article: Q163643
	Product(s): Microsoft Exchange
	Version(s): winnt:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	If you configure a Public Folder to be the administrator of the Microsoft
	Exchange Server Internet Mail Service (IMS), notifications will not be
	delivered.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server,
	version 5.0. We are currently researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
