## Q163649: WD97: Mailing Label Wizard Prints Only in Column 1

	Article: Q163649
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbprint kbualink97 kbtemplate
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When use the Mailing Label Wizard to print a single label, Word prints the label
	in column 1 (not in the row and column of the label sheet you specified).
	
	WORKAROUND
	==========
	
	Do not use the Mailing Label Wizard when you want to print just one label.
	Instead, use the following procedure to print a single label:
	
	1. On the Tools menu, click Envelopes And Labels.
	
	2. Click the Labels tab.
	
	3. Under Address, type the label address.
	
	4. Click Options and select the label type you are printing to, and then click
	  OK.
	
	5. On the Labels tab, under Print, click Single Label. Set the Row and Column
	  settings to the correct label on your label sheet you want to print.
	
	6. Click Print.
