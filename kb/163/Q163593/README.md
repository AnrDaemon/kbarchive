## Q163593: FP97: Uninstall Doesn't Remove Fpwpp.dll

	Article: Q163593
	Product(s): Word Front Page
	Version(s): 
	Operating System(s): 
	Keyword(s): kbsetup kbdta
	Last Modified: 26-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft FrontPage 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you uninstall FrontPage, the Fpwpp.dll is not removed.
	
	CAUSE
	=====
	
	The Fpwpp.dll file is a system-level component for the WebPost driver. This
	driver is used by the Microsoft Web Publishing Wizard to communicate with a
	FrontPage enabled Web server, even if FrontPage is not installed on your
	computer.
