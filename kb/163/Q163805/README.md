## Q163805: Comparison of Bookshelf 1996-97 with Bookshelf Basics

	Article: Q163805
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 25-OCT-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf 1996-97 for Windows 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft Bookshelf Basics is a component of Microsoft Office 97, Professional
	Edition. This component is a smaller edition of Microsoft Bookshelf 1996-97.
	This article discusses the main feature differences between the two editions.
	
	You can access the programs directly from Microsoft Word, Microsoft Excel, and
	Microsoft PowerPoint, or you can run them as a stand-alone program.
	
	MORE INFORMATION
	================
	
	Bookshelf Basics
	----------------
	
	Bookshelf Basics includes the following online references:
	
	- The Original Roget's Thesaurus
	
	- The American Heritage Dictionary, Third Edition
	
	- The Columbia Dictionary of Quotations
	
	Instead of Bookshelf's Daily feature, Bookshelf Basics has Preview Of The Day.
	Each time you start Bookshelf Basics, Preview Of The Day shows you how one of
	the books works.
	
	
	Bookshelf 1996-97
	-----------------
	
	Bookshelf 1996-97 includes the following online references in addition to the
	books included with Bookshelf Basics:
	
	- The World Almanac and Book of Facts 1996
	
	- Concise Encarta 96 World Atlas
	
	- The People's Chronology
	
	- The Concise Columbia Encyclopedia, Third Edition
	
	- Bookshelf Internet Directory 96
	
	- National Five-Digit ZIP Code and Post Office Directory (use this feature to
	  locate cities, states, and ZIP Codes for most U.S. addresses)
	
	Bookshelf Daily appears each time you start Bookshelf. This feature shows a word
	definition and a quote chosen at random.
