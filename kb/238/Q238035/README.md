## Q238035: Encarta 2000: Only the First Line of a Quick Fact Is Printed

	Article: Q238035
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 
	Operating System(s): 
	Keyword(s): kbprint kbimu
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Encyclopedia 2000 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to print a Quick Fact from Microsoft Encarta Encyclopedia 2000,
	only the first line of the Quick Fact may be printed.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Encarta Encyclopedia
	2000.
	
	Additional query words: multi multi-media media mm ee2k
