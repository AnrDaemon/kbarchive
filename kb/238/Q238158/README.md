## Q238158: Reference Suite 2000: Locations of Network Administrator's Guide

	Article: Q238158
	Product(s): Microsoft Home Multimedia Titles
	Version(s): ; WINDOWS:
	Operating System(s): 
	Keyword(s): kbnetwork kbref kbimu
	Last Modified: 25-JUN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Encyclopedia 2000 
	- Microsoft Encarta Reference Suite 2000 
	- Microsoft Encarta Interactive World Atlas 2000 
	- Microsoft Encarta World English Dictionary 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the name of the Network Administrator's Guide file as well as
	the folder in which it is located for each of the programs listed at the
	beginning of this article.
	
	MORE INFORMATION
	================
	
	The following table lists the file name of the Network Administrator's Guide
	file and the folder in which it is located on the program CD-ROM or DVD-ROM for
	each of the programs listed at the beginning of this article.
	
	+------------------------------------------------------------+
	| Program                              | File name | Folder  | 
	+------------------------------------------------------------+
	| Encarta Encyclopedia 2000            | Nag.doc   | Support | 
	+------------------------------------------------------------+
	| Encarta Encylopedia 2000 DVD-ROM     | Nag.doc   | Support | 
	+------------------------------------------------------------+
	| Encarta Reference Suite 2000         | Nag.doc   | Support | 
	+------------------------------------------------------------+
	| Encarta Reference Suite 2000 DVD-ROM | Nag.doc   | Support | 
	+------------------------------------------------------------+
	| Encarta World English Dictionary     | Nag.doc   | Support | 
	+------------------------------------------------------------+
	
	NOTE: The file name and the contents of the Network Administrator's Guide file
	are both different for Microsoft Bookshelf 2000.
	
	REFERENCES
	==========
	
	For additional information about the Network Administrator's Guide for Bookshelf
	2000, click the article number below to view the article in the Microsoft
	Knowledge Base:
	
	  Q225211 Location of the Network Administrator's Guide
