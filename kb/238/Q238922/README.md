## Q238922: Encarta 2000: Detailed Description of Registry Key Functions

{% raw %}

	Article: Q238922
	Product(s): Microsoft Home Multimedia Titles
	Version(s): ; WINDOWS:
	Operating System(s): 
	Keyword(s): kbenv kbimu
	Last Modified: 25-JUN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Encyclopedia 2000 
	- Microsoft Encarta Reference Suite 2000 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article descibes the functions of the following registry keys in the
	programs listed at the beginning of this article.
	
	MORE INFORMATION
	================
	
	Admin.reg:
	
	Sets the program to search for yearbook updates on the local hard disk. Refer to
	the Network Administrators Guide (Nag.hlp) for more information.
	
	Dl_off.reg:
	
	On the tenth day of each month, or upon the first use of the program after that
	date, users will not be reminded that updates are available for download.
	
	Dl_on.reg:
	
	On the tenth day of each month, or upon the first use of the program after that
	date, users will be reminded that updates are available for download.
	
	Local.txt:
	
	Same function as Admin.reg, but this file requires that you modify the UpdatesDAT
	and UpdatesPath values in the registry to reflect the location of the yearbook
	update files on the local hard disk.
	
	Refer to the Network Administrators Guide (Nag.hlp) for more information.
	
	Server.txt:
	
	Sets the program to search for yearbook updates on a network server.
	
	This file requires that you modify the UpdatesDAT and UpdatesPath values in the
	registry to reflect the location of the yearbook update files on the server.
	
	Refer to the Network Administrators Guide (Nag.hlp) for more information.
	
	NOTE: The Network Administrator's Guide also contains information about how to
	update a client workstation registry using a Registry Editor batch file.
	
	Eesndoff.reg and Eesndon.reg:
	
	Turns on or off the following options (as shown in the Settings dialog box):
	
	- Play menu and button sounds
	- Display the Home screen when Encarta Encyclopedia starts
	- Turns on or off the music and sound in MindMaze
	
	Eevwoff.reg and Eevwon.reg:
	
	Turns on or off the following options (as shown in the Settings dialog box):
	
	- Display the Home screen when Encarta Encyclopedia starts
	- Skip Contents Page
	- Display Tool Tips
	- Allow searching with Microsoft Natural Language (if installed)
	- Closed Captions
	- Disable JPEG images from being displayed in certain areas of the program
	- Disable the Media bar from scrolling at the top of the Contents pages
	
	Vcerroff.reg and Vcerron.reg:
	
	Sets Voice Control settings to default or troubleshooting mode. These settings
	adjust the error allowance, maximum number of errors, and other speech-related
	parameters.
	
	For additional information about errors you may encounter when you use Voice
	Control, click the article number below to view the article in the Microsoft
	Knowledge Base:
	
	  Q223172 Encarta: Help Topic Appears When You Enable Voice Control
	
	Mm_off.reg and Mm_on.reg:
	
	Disables or enables the ability to run MindMaze.
	
	Erovwoff.reg and Erovwon.reg:
	
	Turns on or off Cue Cards, Toolbar, Tooltips, and sets Last Window Position and
	Mouse Over behavior.
