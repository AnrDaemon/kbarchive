## Q238979: WD97: Two Prompts Inserting AutoText Entry with ASK or FILLIN

{% raw %}

	Article: Q238979
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta word8 kbfield word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you insert an AutoText entry that contains an ASK field or a FILLIN field,
	you are prompted to fill in the field when you select it and after you click
	Insert on the AutoText tab in the AutoCorrect dialog box. If you answer the
	first prompt but not the second prompt, the field does not contain a result
	after it is inserted in the document.
	
	
	WORKAROUND
	==========
	
	Click OK when you receive the prompt after selecting the AutoText entry, and
	respond to the prompt you receive after clicking Insert.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
