## Q238807: Err Msg: The Files Necessary to Run Virtual Tours Are Missing...

	Article: Q238807
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kberrmsg kbsetup kbimu
	Last Modified: 06-AUG-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 98 Encyclopedia for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start Microsoft Encarta 98 Encyclopedia for Windows, you may receive
	the following error message when you attempt to use the Virtual Tours feature:
	
	  The files necessary to run Virtual Tours are missing or have become
	  corrupted. Please reinstall Encarta by running setup again.
	
	CAUSE
	=====
	
	This behavior can occur if you remove QuickTime for Windows (32-bit).
	
	RESOLUTION
	==========
	
	To resolve this issue, reinstall QuickTime for Windows:
	
	1. Restart the computer.
	
	2. Insert the Encarta 98 Deluxe Edition Disc 1 CD-ROM into the CD-ROM drive.
	  Press and hold down the SHIFT key as you insert the CD-ROM to prevent Encarta
	  98 Encyclopedia from starting automatically.
	
	3. Click Start, and then click Run.
	
	4. In the Open box, type the following line, and then click OK
	
	  <cdrom>:\Aamsstp\Qtinst32.exe
	
	  where <cdrom> is the drive letter of the CD-ROM drive.
	
	5. Follow the instructions on the screen to install QuickTime for Windows.
	
	Additional query words: multi multi-media media mm qtw quick time uninstall
