	SUMMARY
	=======
	
	This article explains how to install the Microsoft Technical Support Media Test
	Tool, which you can use to diagnose, test, and solve problems related to audio,
	video, and other media types.
	
	MORE INFORMATION
	================
	
	To install and run the Microsoft Technical Support Media Test Tool:
	
	1. Insert the program CD-ROM or the Installation and Resources CD-ROM into your
	  CD-ROM drive. Press and hold down the SHIFT key when you insert the CD-ROM to
	  prevent the program from starting automatically.
	
	2. Click Start, and then click Run.
	
	3. In the Open box, type the following line, and then click OK
	
	  "<cdrom>:\support\mtstool.exe" (without the quotation marks)
	
	  where <cdrom> is the drive letter of your CD-ROM drive.
	
	4. Follow the instructions on the screen to complete the installation.
	
	To use the Media Test tool:
	
	1. Click Start, point to Find, and then click "Files or Folders."
	
	2. In the Named box, type "mtstool.exe" (without the quotation marks).
	
	3. In the "Look in" box, click My Computer, and then click Find Now.
	
	4. In the list of found files, double-click Mtstool.exe.
	
	5. Test each media type by double-clicking on the item. If you receive an error
	  message when you play a specific media item, note the exact error message or
	  symptom. This information can assist you in solving the problem.
