## Q238039: Encarta Dictionary 2000: Define Button Missing from Word Toolbar

	Article: Q238039
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:2000; :
	Operating System(s): 
	Keyword(s): kbinterop kbui kbimu
	Last Modified: 08-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta World English Dictionary 
	- Microsoft Encarta World English Dictionary 2001 
	- Microsoft Word 2000 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start Microsoft Word 2000, the Define button that integrates Word with
	Microsoft Encarta World English Dictionary 2000 may be missing from the standard
	toolbar.
	
	CAUSE
	=====
	
	This behavior can occur if you remove any of the following programs:
	
	- Microsoft Bookshelf 98 for Windows
	- Microsoft Bookshelf 99
	- Microsoft Bookshelf 2000
	
	RESOLUTION
	==========
	
	To work around this issue, remove the version of Bookshelf installed on your
	computer, and then remove and reinstall Encarta World English Dictionary 2000.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
	
	MORE INFORMATION
	================
	
	If you installed Encarta World English Dictionary 2000 from Microsoft Encarta
	Reference Suite 2000, you must reinstall Encarta World English Dictonary by
	running the Setup.exe file located in the Ewed folder on the Encarta Reference
	Suite 2000 Installation and Resources CD-ROM.
