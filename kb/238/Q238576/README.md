## Q238576: Midtown Madness: Objects Appear Suddenly on Screen

	Article: Q238576
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbui kbimu
	Last Modified: 05-AUG-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Midtown Madness, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Midtown Madness, objects such as buildings may appear
	suddenly on the screen rather than appearing gradually as you approach them.
	
	CAUSE
	=====
	
	This behavior can occur if the Far Clip Distance graphics option is set too low.
	
	RESOLUTION
	==========
	
	To resolve this issue, increase the Far Clip Distance setting:
	
	1. Start Midtown Madness.
	
	2. On the Midtown Madness startup screen, click Options.
	
	3. Click Graphics Options.
	
	4. Click the PLUS SIGN (+) under Far Clip Distance to increase the setting.
	
	5. Click Done.
	
	If the issue continues to occur, repeat these steps to further increase the Far
	Clip Distance setting.
	
	Additional query words: 1.00 midmad instantly appears
