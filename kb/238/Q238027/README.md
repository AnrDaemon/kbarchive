## Q238027: Winning Chess Strategies Comments and Corrections

	Article: Q238027
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 12-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Winning Chess Strategies ISBN 0-7356-0604-8 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Winning Chess Strategies, ISBN
	0-7356-0604-8.
	
	The following topics are covered:
	
	- Page 7: Point Value of the Knight is Incorrect
	
	- Page 11: "Kd4-c6" Should Be "Nd4-c6"
	
	- Page 44: Analysis of Move 12 is Incorrect
	
	- Page 212: Algebraic Notation Error
	
	MORE INFORMATION
	================
	
	Page 7: Point Value of the Knight is Incorrect
	----------------------------------------------
	
	Page 7, Middle of the Second line from the bottom of the page:
	Change:
	"...pawn: 1 point; Knight: 2 points; Bishop:..."
	
	To:
	"...pawn: 1 point; Knight: 3 points; Bishop:..."
	
	
	Page 11: "Kd4-c6" Should Be "Nd4-c6"
	------------------------------------
	
	On page 11 there is an error in the 4th line.
	
	Change:
	"Note that 23...Qd7 stops Kd4-c6 but..."
	
	To:
	"Note that 23...Qd7 stops Nd4-c6 but..."
	
	
	Page 44: Analysis of Move 12 is Incorrect
	-----------------------------------------
	
	Page 44, second line after the bold move 12:
	Change
	" The best move was 12...d6, which restrains..."
	
	To:
	"The best move was 12...b6, which restrains..."
	
	
	Page 212: Algebraic Notation Error
	----------------------------------
	
	Change:
	"15. Rc1 Bd7"
	
	To:
	"15. Rac1 Bd7"
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
