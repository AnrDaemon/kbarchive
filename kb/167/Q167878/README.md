## Q167878: WD97: Progress Meter Remains Visible in Status Bar After Error

	Article: Q167878
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kberrmsg kbtool kbdta kbtemplate word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you run the Conversion Wizard, you may receive the following error
	message:
	
	  Word cannot complete the conversion of this file <filename>. Would you
	  like to continue with the batch conversion?
	
	where <filename> is the name of the file that caused the conversion process
	to halt. If you click No, the progress meter, which appears in the status bar,
	remains visible.
	
	RESOLUTION
	==========
	
	To resolve this problem and refresh the status bar, quit and restart Word.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
