## Q167360: SMS: Network Monitor - Print Advanced Should Use Radio Buttons

	Article: Q167360
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.0,1.1
	Operating System(s): 
	Keyword(s): kbenv smsui
	Last Modified: 17-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server versions 1.0, 1.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In the Advanced Printing options dialog box of Network Monitor, the Print Same
	As View and Expand All Levels options are mutually exclusive; these options
	should use radio buttons instead of check boxes. To locate these options in
	Network Monitor, click Print of the File menu, and then click Advanced.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	versions 1.0 and 1.1. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
