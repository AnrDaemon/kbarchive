## Q167156: BUG: OLEISAPI2 Missing from Visual Basic 5.0 CD-ROM

	Article: Q167156
	Product(s): Microsoft Visual Basic for Windows
	Version(s): 
	Operating System(s): 
	Keyword(s): kbVBp500bug
	Last Modified: 11-JAN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual Basic Professional Edition for Windows, version 5.0 
	- Microsoft Visual Basic Enterprise Edition for Windows, version 5.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You cannot locate OLEISAPI2.dll on the Visual Basic 5.0 CD-ROM.
	
	CAUSE
	=====
	
	Visual Basic 5.0 Help describes a sample called OLEISAPI2.dll and states that it
	is located in the Tools directory of the Visual Basic CD-ROM. This information
	is incorrect. The OLEISAPI2.dll sample is missing from the CD-ROM.
	
	RESOLUTION
	==========
	
	The OLEISAPI2.DLL sample can be found on the following Microsoft Web Site:
	
	  http://msdn.microsoft.com/visualc/downloads/download.asp?ID=008
	
	The file contains both source code and compiled DLLs.
	
	Users of this sample should be aware that it is for demonstration purposes only
	and is not supported by Microsoft Technical Support (MTS).
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
