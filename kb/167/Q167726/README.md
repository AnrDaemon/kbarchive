## Q167726: XFOR: Err Msg: The Message Could Not Be Delivered

	Article: Q167726
	Product(s): Microsoft Exchange
	Version(s): WinNT:5.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 30-APR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you receive mail with the Windows messaging client included with Windows 95
	or Windows NT 4.0, and you are running the Internet Mail and Personal Folder
	services only, you may receive the following error message:
	
	  The message could not be delivered because a network problem occurred
	  while transferring the message. If the problem persists, contact your
	  Internet Service Provider.
	
	MORE INFORMATION
	================
	
	This error occurs when you by use the Windows messaging client with only the
	Internet Mail Service configured to connect to an Exchange Server 5.0 computer
	running the POP3 client service.
	
	This problem does not occur in Exchange Client 4.0 or 5.0.
