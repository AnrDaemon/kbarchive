## Q167671: WD97: Can't Check Spelling in Text Box Located in Header/Footer

	Article: Q167671
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbusage kbproof word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you check spelling in a text box in the header or footer of a document, you
	may experience the following behavior:
	
	  Microsoft Word for Windows 97:
	
	  The Header/Footer toolbar appears on top of the Spelling dialog box, and
	  you are unable to continue.
	
	WORKAROUND
	==========
	
	If the Header And Footer toolbar appears on top of the Spelling dialog box, use
	the following steps to continue to check spelling in your document:
	
	1. Click the Header And Footer toolbar and move it out of the way of the
	  Spelling dialog box.
	
	2. Use the Spelling dialog box to continue checking spelling in the document.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
