## Q167027: WinNT Workstation Resource Kit Refers to File Not on CD

{% raw %}

	Article: Q167027
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The "Microsoft Windows NT 4.0 Workstation Resource Kit," Chapter 7, Page 204
	states the following for the Specifying Virtual Printer Memory subheading:
	
	  To determine the right value, copy the Testps.txt file (supplied with the
	  Windows NT Resource Kit) to the printer, and use the recommended virtual
	  memory value printed on the resulting page.
	
	CAUSE
	=====
	
	The file is not located on the CD.
	
	RESOLUTION
	==========
	
	Testps.txt can be obtained from the Windows 95 CD or the Windows NT 3.51 CD in
	the Clients\Wfw\Netsetup directory.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Windows NT Workstation
	version 4.0. We are researching this problem and will post new information here
	in the Microsoft Knowledge Base as it becomes available.
