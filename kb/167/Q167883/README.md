## Q167883: WD97: Computer Hangs While Editing WordMail Template

	Article: Q167883
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97; :4.0
	Operating System(s): 
	Keyword(s): kbusage kbtemplate winword word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	- Microsoft Exchange, version 4.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you try to edit a WordMail template from Microsoft Exchange, your computer
	may appear to stop responding (hang) and display an hourglass.
	
	CAUSE
	=====
	
	The Word Macro Virus Protection Warning dialog box is in the background and
	isn't visible for you to respond to.
	
	WORKAROUND
	==========
	
	To work around this problem, use any of the following methods.
	
	Method 1: Switch to Word
	------------------------
	
	Press ALT+TAB to switch to Word. The Macro Virus Protection Warning dialog box
	should be in the foreground.
	
	Method 2: Turn Off Macro Virus Protection
	-----------------------------------------
	
	NOTE: Microsoft Technical Support does not recommend that you disable the Macro
	Virus Protection feature. If you do turn this feature off, we suggest you turn
	the feature back on when you have finished editing your e-mail templates.
	
	1. On the Tools menu, click Options.
	
	2. Click the General tab.
	
	3. Under General Options, click to clear the Macro Virus Protection check box.
	
	4. Click OK to close the Options dialog box.
	
	You should now be able to edit your WordMail template from Microsoft Exchange.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
	
	MORE INFORMATION
	================
	
	To Edit a Template from Exchange
	--------------------------------
	
	1. Start Microsoft Exchange.
	
	2. On the Compose menu, click WordMail Options.
	
	3. Click the template you want and then click Edit.
	
	NOTE: You can also use the "Enable Word as e-mail editor" option in the WordMail
	Options dialog box.
