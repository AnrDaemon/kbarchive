## Q70888: Can't Use Some Attributes on Redirected Drive

{% raw %}

	Article: Q70888
	Product(s): Microsoft LAN Manager
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	BUG: 1237
	
	SYMPTOMS
	========
	
	Directory-located file attributes beyond those supported by MS-DOS, such as
	fdateCreation and fdateLastAccess, are not available on redirected HPFS/HPFS386
	drives.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in DOS LAN Manager version 2.0b. We
	are researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
	
	Additional query words: 2.00b attributes
