## Q70422: MS-DOS Editor &amp; QBasic (MCGA w/Mono. Monitor)

{% raw %}

	Article: Q70422
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.x,6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 5.0, 5.0a, 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run MS-DOS version 5.0 or later QBasic or Editor on a system with an
	MCGA display adapter and a monochrome monitor, the actual position of the mouse
	cursor does not match the position of the visible cursor on the screen.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem with QBasic and Editor in MS-DOS
	versions listed above. We are researching this problem and will post new
	information here as it becomes available.
