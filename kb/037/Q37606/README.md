## Q37606: Entering Airplane Position Using Keyboard

	Article: Q37606
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	In the NAV/COM menu, you may set the airplane's current position using the
	Position Set command (item A). To enter the coordinates, you must use the number
	keys on the main keyboard. The numbers on the keypad (on the far right side of
	the PC keyboard) will not work.
