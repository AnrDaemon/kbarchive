## Q37986: &quot;Cannot create PIF File&quot; Message when No Directory Exists

{% raw %}

	Article: Q37986
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	If you try to save a PIF file and get the error message 
	
	  Cannot create PIF file
	
	check the pathname entry in the Program Name field to be
	sure that the directory specified already exists. The Windows PIF
	Editor (PIFEDIT.EXE) requires that the directory specified in this
	field be created before the PIF file can be saved.
	
	Additional query words: 3.00 3.00a 3.10 win30 win31 .pif
