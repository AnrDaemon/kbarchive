## Q137830: Slide Numbers in Title Not Printed in Outline View

	Article: Q137830
	Product(s): Microsoft PowerPoint for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): kbdisplay kbprint kbPrinting
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft PowerPoint for Windows 95, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If a slide number placeholder is placed in the title or body text of a slide,
	the "<#>" character is printed instead of the slide number if you print
	the outline view.
	
	
	WORKAROUND
	==========
	
	Print from any other view. When you print from another view, the slide number is
	printed correctly.
	
	STATUS
	======
	
	This behavior is by design of Microsoft PowerPoint. When you print from outline
	view, the slide number is automatically displayed in the left margin.
