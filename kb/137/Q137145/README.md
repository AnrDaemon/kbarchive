## Q137145: Help Buttons Disappear After Printing Network Services

	Article: Q137145
	Product(s): Microsoft Windows NT
	Version(s): 3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.51 
	- Microsoft Windows NT Server version 3.51 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you choose one of the Additional Network Services bitmap buttons for
	Windows NT Server or Workstation in Windows NT Help (WINNT.HLP), choose Print,
	and choose Back after the printing completes, all buttons in Windows NT Help
	disappear.
