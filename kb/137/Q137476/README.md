## Q137476: NTVDM Error with Matrox Video Card

	Article: Q137476
	Product(s): Microsoft Windows NT
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation versions 3.5, 3.51 
	- Microsoft Windows NT Server versions 3.5, 3.51 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The following error message appears on your Windows NT computer using a Matrox
	Impression VL-bus or Matrox Impression Plus video card:
	
	  An application error has occurred.
	  An application error log is being generated.
	  NTVDM.EXE exception: access violation (0x0000005), 0x01f23a4f
	
	RESOLUTION
	==========
	
	Enable video shadowing in your computer's CMOS or update the driver to version
	2.12 by calling Matrox technical support at 514-685-0270 or their BBS number
	514-685-2630 and download the file named NT212_24.ZIP. For more information
	contact Matrox technical support.
	
	The Matrox Impression video cards are manufactured by Matrox, a vendor
	independent of Microsoft; we make no warranty, implied or otherwise, regarding
	this product's performance or reliability.
