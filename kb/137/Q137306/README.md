## Q137306: Fonts Incorrectly Displayed During Windows 95 Remote Control

{% raw %}

	Article: Q137306
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.1
	Operating System(s): 
	Keyword(s): kbnetwork kbtshoot kbui smsremtshoot kbRemoteProg
	Last Modified: 31-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	Fonts are incorrectly displayed in the Quick Windows Viewer during a Windows 95
	remote control session if the Windows 95 Plus Pack is installed on the client.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.1. This problem was corrected in the latest US Service Pack for
	Systems Management Server. For information on obtaining this update, query on
	the following word in the Microsoft Knowledge Base:
	
	  " SERVPACK " (without the quotation marks)
	
	Additional query words: prodsms sms win95 helpdesk
