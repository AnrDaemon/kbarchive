## Q137204: Attachments Sent to Schedule+ 1.0 User Cannot Be Saved

	Article: Q137204
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:1.0,7.0; Win95:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, versions 1.0, 7.0 
	-------------------------------------------------------------------------------
	
	When a Microsoft Schedule+ version 7.0 user sends a meeting request with an
	attachment to a Schedule+ 1.0 user, the recipient will be able to open the
	message but will be unable to view or save the attachment.
	
	This is by product design.
