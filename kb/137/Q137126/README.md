## Q137126: DOC: README.HLP Contains Incorrect Language Reference Info

	Article: Q137126
	Product(s): Microsoft FoxPro
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): kbdocfix
	Last Modified: 12-FEB-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual FoxPro for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Visual FoxPro 3.0 Readme.hlp file, which installs with the Professional
	Edition, contains corrections to printed documentation. There are two Language
	Reference corrections identified in the Readme.hlp file that do not exist in the
	Language Reference:
	
	  Language Reference, READ Command
	  Language Reference, UIEnable Event
	
	This documentation error has been fixed in Visual FoxPro 5.0 for Windows.
	
	Additional query words: docerr VFoxWin 5.00
