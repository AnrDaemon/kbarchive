## Q137960: Wine Guide 1.0: Incorrect BACK Functionality in Encyclopedia

	Article: Q137960
	Product(s): Microsoft Home Multimedia Titles
	Version(s): MACINTOSH:1.0; WINDOWS:1.0,1.0a,2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Wine Guide for Windows, versions 1.0, 1.0a, 2.0 
	- Microsoft Wine Guide for the Macintosh, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you jump to different pages in the Wine Encyclopedia and select Back to the
	Encyclopedia, the Back feature returns you to the Encyclopedia List page.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the programs listed at the top
	of this article. We are researching this problem and will post new information
	here in the Microsoft Knowledge Base as it becomes available.
