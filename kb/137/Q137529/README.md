## Q137529: Cannot Use DriveSpace to Compress a Substituted Drive

	Article: Q137529
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you try to use DriveSpace to compress a drive substituted with the SUBST
	command, you receive the following error message:
	
	  Windows cannot work with drive <X> because an application is currently
	  using that drive.
	
	MORE INFORMATION
	================
	
	You cannot use DriveSpace to compress a substituted drive.
