## Q137580: Compaq Computers That May Require Updates for Windows 95

	Article: Q137580
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): Win2000:95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains information provided by Compaq Computer Corporation
	regarding Compaq computers that may require additional Compaq drivers or
	supplements for Windows 95. These additional drivers can be obtained from
	Compaq's Softpaq library. For additional information about Compaq supplemental
	updates, contact Compaq Computer Corporation at 1-800-952- 7689. For a list of
	current Softpaqs, call the Compaq Fax Back Service at 1- 800-345-1518 and select
	the appropriate PAQFAX menu options to order PAQFAX Document #7, which contains
	a catalog of Windows 95 information.
	
	MORE INFORMATION
	================
	
	The following information has been provided by Compaq Computer Corporation and
	is current as of 9/27/95:
	
	Computers that may require additional Compaq software drivers, updates, or ROM
	upgrades for Windows 95:

	     Contura Aero (all models)
	     Contura 4/25
	     Contura 400/410

	     LTE Lite 4/25, 4/33
	     LTE Elite
