## Q137913: Problems Printing to PCPI Laser Image 1030 Printer

{% raw %}

	Article: Q137913
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): kbprint win95 kbPrinting
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the Windows PostScript printer driver to print to an older PCPI
	Laser Image series 1030 printer, the printout may be garbled.
	
	CAUSE
	=====
	
	This printer is not fully compatible with Windows 95.
	
	MORE INFORMATION
	================
	
	The printer manufacturer has stated that the driver for this printer will not be
	updated. For additional information about PCPI products, please contact the
	manufacturer.
