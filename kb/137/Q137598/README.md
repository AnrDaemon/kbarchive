## Q137598: PPT7: PowerPoint Presentation Conferencing Unavailable

{% raw %}

	Article: Q137598
	Product(s): Microsoft PowerPoint for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbnetwork kbdtakbbuglist
	Last Modified: 21-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft PowerPoint for Windows 95, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click the Tools menu in PowerPoint, the Presentation Conferencing
	command may be unavailable (dimmed).
	
	CAUSE
	=====
	
	This problem may occur if the following conditions are true:
	
	- TCP/IP is not installed on your computer.
	
	  -or-
	
	- You are running a shared version of Windows 95.
	
	  -or-
	
	- TCP/IP is installed or configured incorrectly on your computer.
	
	
	RESOLUTION
	==========
	
	To correct this problem, use any of the following methods:
	
	- Install TCP/IP if it is not installed.
	
	- Install a local copy of Windows 95.
	
	- Reconfigure your TCP/IP configuration.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
	
	MORE INFORMATION
	================
	
	For additional information, please see the following articles in the Microsoft
	Knowledge Base:
	
	  Q135325 PPT: Presentation Conference Troubleshooting Guide
	
	  Q137055 PPT: Presentation Conference Fails With Errors
