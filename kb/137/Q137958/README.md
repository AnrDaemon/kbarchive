## Q137958: Bookshelf '95 editions: Daily Dialog Settings are Saved on ESC

	Article: Q137958
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1995 edition; WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for Windows 1995 edition 
	- Microsoft Bookshelf '95 for Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you press the ESC key to close Bookshelf Daily, the Bookshelf Daily dialog
	setting changes are saved as you last had them, rather than in the default
	settings.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Bookshelf editions listed at
	the top of this article. This problem has been resolved in Microsoft Bookshelf
	'96-'97 for Windows.
	
	WORKAROUND
	==========
	
	Undo any unwanted changes before you press the ESC key to exit Bookshelf Daily.
