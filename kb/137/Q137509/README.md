## Q137509: XCLN: Notification Alerts When Exchange Is the Active App

	Article: Q137509
	Product(s): Microsoft Exchange
	Version(s): 4.0 5.0
	Operating System(s): 
	Keyword(s): kbusage win95
	Last Modified: 20-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange MS-DOS client, versions 4.0, 5.0 
	- Microsoft Exchange Windows 3.x client, versions 4.0, 5.0 
	- Microsoft Exchange Windows NT client, versions 4.0, 5.0 
	- Microsoft Exchange Windows 95/98 client, versions 4.0, 5.0 
	- the operating system: Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you choose Options on the Tools menu and then activate notification options
	on the General tab, you will be notified of the arrival of new mail even when
	Microsoft Exchange is the active foreground application.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft Exchange version
	4.0 MS-DOS, Windows 3.x, Windows 95, and Windows NT clients. We are researching
	this problem and will post new information here in the Microsoft Knowledge Base
	as it becomes available.
