## Q137300: Setup Graphic Not Displayed Setting up MSN Using Remote Control

	Article: Q137300
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.1
	Operating System(s): 
	Keyword(s): kbnetwork kbtshoot kbui smsremtshoot kbRemoteProg
	Last Modified: 31-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	When you use Remote Control to setup Microsoft Network on a Windows 95 client,
	the graphic in the Setup window does not display in Quick Windows Viewer on the
	controlling computer.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem Systems Management Server version
	1.1. This problem was corrected in the latest US Service Pack for Systems
	Management Server. For information on obtaining this update, query on the
	following word in the Microsoft Knowledge Base (without the spaces):
	
	  " SERVPACK " (without the quotation marks)
	
	Additional query words: prodsms sms rc helpdesk msn
