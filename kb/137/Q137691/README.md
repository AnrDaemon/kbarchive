## Q137691: Bookshelf 1995: Cannot Alt+Tab from Quickshelf Keys

	Article: Q137691
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1995 edition; WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for Windows 1995 edition 
	- Microsoft Bookshelf 1996-97 for Windows 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you run Bookshelf with the QuickShelf Keys dialog box open, the Alt+Tab key
	combination used for task switching to other open applications is
	non-functional.
	
	MORE INFORMATION
	================
	
	This behavior is by design and not a problem with Bookshelf.
