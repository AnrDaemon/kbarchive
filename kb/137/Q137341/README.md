## Q137341: Appointments Print on a Weekly Basis Only

	Article: Q137341
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you send multiple appointments to the printer using the All Text print
	format, the job will be printed on a page per week basis. The result will be
	pages filled with text or pages occupied mostly by white space. This per week
	page printing applies regardless of the start day declared on the General tab
	under Tools, Options.
	
	CAUSE
	=====
	
	This is by product design.
