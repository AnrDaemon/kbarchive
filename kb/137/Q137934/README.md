## Q137934: TN3270 Refresh Cycle Time Description

{% raw %}

	Article: Q137934
	Product(s): Microsoft SNA Server
	Version(s): WINDOWS:2.11,3.0
	Operating System(s): 
	Keyword(s): kbinterop kbnetwork
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft SNA Server, versions 2.11, 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The "Refresh Cycle Time" option under TN3270 Admin's Option Menu is used to
	modify the refresh rate of the information displayed in the Session Status
	dialog box. The valid range is 0-60 seconds; 0 means no refresh. If the refresh
	option is not set, you have to manually refresh the Session Status dialog box.
