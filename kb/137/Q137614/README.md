## Q137614: PC WRmt: Files That Contain User Detail Information

{% raw %}

	Article: Q137614
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Remote for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	You can resolve user detail information from more than one location. These
	details are from the sources listed below.
	
	MORE INFORMATION
	================
	
	When you view user details, the following default information will be
	
	  ALIAS: <UserEmailName>
	  NAME: <UserFullName>
	  MAILBOX:
	  Postoffice: <Postoffice Name>
	  Network: <Network Name>
	
	The above information is resolved from the following files:
	
	  ALIAS: <GAL.NME>
	  NAME: <XXXXXXXX.USR>
	  MAILBOX: <GALNETPO.GLB>
	  Postoffice: <GALNETPO.GLB>
	  Network: <GALNETPO.GLB>
