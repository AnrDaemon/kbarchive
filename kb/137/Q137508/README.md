## Q137508: Overlapping Appointments Cannot Be Modified

	Article: Q137508
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0; Win95:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft Schedule+ users can disallow overlapping appointments. However, if you
	try to modify overlapping appointments after enabling this restriction, the
	following error is generated:
	
	  The item attribute cannot be changed. Only one appointment can be scheduled
	  for a single time period on this schedule when the Disallow Overlapping
	  Appointments option is in effect.
	
	CAUSE
	=====
	
	This is by product design.
