## Q137242: After Dark 3.0 May GP Fault in Windows NT

	Article: Q137242
	Product(s): Microsoft Windows NT
	Version(s): 3.5,3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation versions 3.5, 3.51 
	- Microsoft Windows NT Server versions 3.5, 3.51 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you install Berkeley Systems After Dark 3.0 and restart Windows NT, a
	general protection (GP) fault occurs in module KRNL386.EXE.
	
	WORKAROUND
	==========
	
	For more information about resolving this issue, contact Berkeley Systems at
	(510) 549-2300.
