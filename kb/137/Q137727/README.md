## Q137727: Timex Data Link Watch Rounds Appointment Times

	Article: Q137727
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0; Win95:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When appointments are transferred from Microsoft Schedule+ to the Timex Data
	Link watch, the appointments are rounded to the nearest quarter hour. For
	example, an appointment starting at 12:07 would be rounded to 12:00.
	
	CAUSE
	=====
	
	This is a limitation of the Timex Data Link watch firmware.
	
	
	MORE INFORMATION
	================
	
	The Timex Data Link watch is manufactured by Timex, a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
