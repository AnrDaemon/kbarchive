## Q137990: Error Printing Command Reference Topic in Windows NT Help

	Article: Q137990
	Product(s): Microsoft Windows NT
	Version(s): 3.51
	Operating System(s): 
	Keyword(s): kbprint kbPrinting
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.51 
	- Microsoft Windows NT Server version 3.51 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you print the Windows NT Command Reference topic in Windows NT Help
	(WINNT.HLP), the following error message appears one or more times:
	
	  An error exists in this Help file. Contact your application vendor for an
	  updated Help file. (1049)
	
	After you choose OK in each error message dialog box, the Help topic prints.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 3.51. This
	problem was corrected in Windows NT version 4.0.
