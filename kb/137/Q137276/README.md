## Q137276: Cannot Enter Page Breaks in WordPad

	Article: Q137276
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	WordPad does not support inserting page breaks in documents by pressing
	CTRL+ENTER. This feature is supported in Write, a tool included with Windows
	3.x, but is not supported in WordPad.
	
	MORE INFORMATION
	================
	
	You cannot insert a page break in WordPad. To begin a new page, press ENTER
	until you reach the beginning of a new page.
	
	To view the page layout of a document in WordPad, click Print Preview on the File
	menu.
