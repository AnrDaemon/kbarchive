## Q240112: NBA 2000: Pointer Is Frozen When Startup Screen Is Displayed

	Article: Q240112
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbhw kbimu msgame kbHardware
	Last Modified: 25-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NBA Inside Drive 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you click a button on your pointing device before the Microsoft NBA Inside
	Drive 2000 startup screen is displayed, the pointer is frozen when the startup
	screen is displayed.
	
	RESOLUTION
	==========
	
	To work around this issue, do not click a button on your pointing device until
	the NBA Inside Drive 2000 startup screen is displayed.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft NBA Inside Drive,
	version 1.0.
