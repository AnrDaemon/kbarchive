## Q240639: World Atlas 2000 Err Msg: Unable to Locate Favorites Folder...

	Article: Q240639
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kberrmsg kbui kbimu
	Last Modified: 25-JUN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Interactive World Atlas 2000 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click the Favorites menu in Microsoft Encarta World Atlas 2000, you may
	receive the following error message:
	
	  Unable to locate Favorites Folder. Creating new one.
	
	If you click OK, the Favorites dialog box opens.
	
	If you click Cancel, Encarta World Atlas 2000 quits.
	
	You may also experience this symptom when you attempt to add a favorite to the
	Favorites menu.
	
	CAUSE
	=====
	
	This behavior can occur if either of the following conditions is true:
	
	- You delete all of the favorites that are located in the Favorites folder
	  before you start Encarta World Atlas 2000.
	
	- You delete the Favorites folder.
	
	  NOTE: Although the error message specified in the "Symptoms" section of this
	  article states that Encarta World Atlas 2000 is creating a new Favorites
	  folder, a Favorites folder is not created, and you remain unable to add a
	  favorite.
	
	RESOLUTION
	==========
	
	To work around this issue, manually create a Favorites folder:
	
	1. Double-click the My Computer icon on the desktop.
	
	2. Double-click the hard disk on which Microsoft Windows is installed.
	
	3. Double-click the Windows folder.
	
	4. Double-click the Application Data folder.
	
	5. Double-click the Microsoft folder.
	
	6. Double-click the Encarta Interactive World Atlas folder.
	
	7. On the File menu, point to New, and then click Folder.
	
	8. Right-click the New Folder folder, and then click Rename.
	
	9. Type "Favorites" (without the quotation marks), and then press ENTER.
	
	10. Close the Encarta Interactive World Atlas window.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Encarta World Atlas
	2000.
