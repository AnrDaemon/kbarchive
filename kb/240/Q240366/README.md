## Q240366: NFL Fever: Unexpectedly Minimized When You Open Players Guide

	Article: Q240366
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbui kbimu msgame
	Last Modified: 18-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NFL Fever 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click Players Guide on the Help menu in Microsoft NFL Fever 2000, NFL
	Fever 2000 is unexpectedly minimized.
	
	RESOLUTION
	==========
	
	To resolve this issue, click the NFL Fever 2000 taskbar button to restore NFL
	Fever 2000 and display the Players Guide on the desktop.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft NFL Fever 2000,
	version 1.0.
