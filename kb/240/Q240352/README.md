## Q240352: NBA Inside Drive 2000: Screen Resolutions Supported by the Game

	Article: Q240352
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbimu msgame
	Last Modified: 04-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NBA Inside Drive 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the screen resolutions that are supported by Microsoft NBA
	Inside Drive 2000.
	
	MORE INFORMATION
	================
	
	NBA Inside Drive 2000 supports the following screen resolutions:
	
	- 640 x 480 pixels
	- 800 x 600 pixels
	- 1024 x 768 pixels
	
	NOTE: In order to use a screen resolution higher than 640 x 480 pixels, the video
	adapter installed in your computer must support the higher resolution.
	
	If you configure the game to use a screen resolution higher than 640 x 480 pixels
	and your video adapter does not support the higher resolution, the game is
	displayed at a screen resolution of 640 x 480 pixels.
