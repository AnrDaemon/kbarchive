## Q240675: SMS: IBM Client Access Software May Cause SMS Client to Hang

	Article: Q240675
	Product(s): Microsoft Systems Management Server
	Version(s): 2.0,2.0 SP1,2.0 SP2
	Operating System(s): 
	Keyword(s): kbsms200 kbsms200bug
	Last Modified: 05-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server versions 2.0, 2.0 SP1, 2.0 SP2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Computers with both Systems Management Server client software and IBM AS/400
	Client Access software installed may respond very slowly. In addition, you may
	see Microsoft Windows NT logon windows where the domain name is not displayed.
	
	This problem does not always occur on these systems, is difficult to reproduce,
	and may never occurr on systems with both IBM Client Access and Systems
	Management Server client.
	
	
	STATUS
	======
	
	
	
	MORE INFORMATION
	================
	
	For the problem to occur, the following components must be installed and
	running:
	
	- IBM Client Access.
	
	- SMS 2.0 client - any version.
