## Q240256: Understanding COM+ Comments and Corrections

	Article: Q240256
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 12-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Understanding COM+ ISBN 0-7356-0666-8 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Understanding COM+, ISBN 0-7356-0666-8.
	
	The following topics are covered:
	
	- Page 82: Figure 3-7, Misspelled Word
	
	- Page 126: Figure 4-8, CoGetObject Function Call Incorrect
	
	MORE INFORMATION
	================
	
	Page 82, Figure 3-7: Misspelled Word
	------------------------------------
	
	On page 82, in Figure 3-7, change Item 5 from:
	"The MoveMoney object receieves..."
	
	To:
	"The MoveMoney object receives..."
	
	
	Page 126: Figure 4-8, CoGetObject Function Call Incorrect
	---------------------------------------------------------
	
	On page 126, in Figure 4-8, change capitalization in Item 1 from:
	"Client calls the COGetObject function..."
	
	To:
	"Client calls the CoGetObject function..."
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
	
	Additional query words: 0-7356-0666-8 DEVBOOK
