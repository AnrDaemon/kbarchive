## Q240985: Pandora's Box: Graphics Not Displayed Properly with 3D Rage Pro

{% raw %}

	Article: Q240985
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbimu msgamekbfaq
	Last Modified: 10-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Pandora's Box, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Pandora's Box, the graphics in the game may not be
	displayed properly.
	
	CAUSE
	=====
	
	This behavior can occur if an ATI 3D Rage Pro video adapter is installed in your
	computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest version of the video
	driver for your ATI 3D Rage Pro video adapter from the following ATI
	Technologies Web site:
	
	  http://www.atitech.com
	
	NOTE: You may need to remove the current ATI video driver before you install the
	updated video driver. To remove the current video driver:
	
	1. Click Start, point to Settings, and then click Control Panel.
	
	2. On the Install/Uninstall tab, double-click Add/Remove Programs.
	
	3. Click ATI Mach 64 Display Driver, and then click Remove.
	
	For additional information about how to install an updated video driver, please
	click the article number below to view the article in the Microsoft Knowledge
	Base:
	
	  Q131806 Windows 95/98: How to Install or Change a Display Driver
