## Q240259: Microsoft Computer Dictionary 4th Ed. Comments and Corrections

	Article: Q240259
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 04-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Computer Dictionary, Fourth Edition ISBN 0-7356-0615-3 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Microsoft Computer Dictionary, Fourth
	Edition, ISBN 0-7356-0615-3.
	
	The following topics are covered:
	
	- Page 62: Term British Naval Connector Is Misspelled
	
	- Page 210: Full Name Of The ITU Spelled Incorrectly
	
	- Page 217: Incorrect Microhenries Symbol
	
	- Page 418: Spacebar Entry Incorrectly Capitalized
	
	MORE INFORMATION
	================
	
	Page 62: Term British Naval Connector Is Misspelled
	---------------------------------------------------
	
	Change:
	"British Navel Connector"
	
	To:
	"British Naval Connector"
	
	
	Page 210: Full Name Of The ITU Spelled Incorrectly
	--------------------------------------------------
	
	The definitions for H.320, H.323, and H.324 each have the full name of the ITU
	spelled incorrectly.
	
	Change:
	"International Telecommunications Union"
	
	To:
	"International Telecommunication Union"
	
	
	Page 217: Incorrect Microhenries Symbol
	---------------------------------------
	
	On page 217, the microhenris symbol in the definition of "henry" contains a
	typo.
	
	Change:
	<MU>H=
	
	To:
	<B5>H=
	
	
	Page 418: Spacebar Entry Incorrectly Capitalized
	------------------------------------------------
	
	Change:
	"Spacebar"
	
	To:
	"spacebar"
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
	
	Additional query words: 0-7356-0615-3 EUBOOK
