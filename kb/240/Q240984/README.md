## Q240984: Pandora's Box: Graphics Not Displayed Properly with STB Velocity

	Article: Q240984
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbenv kbimu msgamekbfaq
	Last Modified: 10-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Pandora's Box, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Pandora's Box, the graphics on the screen may not be
	displayed properly.
	
	CAUSE
	=====
	
	This behavior can occur if an STB Velocity 128 video adapter is installed in
	your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, use the appropriate method for your video adapter.
	
	STB Velocity 128 (Retail)
	-------------------------
	
	Download and install the latest version of the video driver for your STB Velocity
	128 video adapter from the following STB Web site:
	
	  http://www.stb.com
	
	Dell STB Velocity 128
	---------------------
	
	Download and install the latest version of the video driver for your Dell STB
	Velocity 128 video adapter from the following Dell Web site:
	
	  http://www.dell.com
	
	NOTE: If your Dell computer shipped with a pre-installed Velocity 128 video
	adapter, use only the version of the video driver available from Dell. Retail
	versions of the Velocity 128 video driver may cause screen distortion and may
	cause the computer to become unstable.
