## Q240636: World Atlas: Find Box Is Invisible in High Contrast Color Scheme

{% raw %}

	Article: Q240636
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbdisplay kbenv kbimu
	Last Modified: 27-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Interactive World Atlas 2000 
	- Microsoft Encarta Interactive World Atlas 2001 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start Microsoft Encarta World Atlas 2000, the Find box in the
	Pinpointer may not be visible.
	
	CAUSE
	=====
	
	This behavior can occur if you select a high contrast color scheme that sets the
	window background color to black.
	
	RESOLUTION
	==========
	
	To resolve this issue, select a different color scheme:
	
	1. Click Start, point to Settings, and then click Control Panel.
	
	2. Click the Appearance tab.
	
	3. In the Scheme box, click a color scheme that is not a high contrast color
	  scheme that sets the window background color to black.
	
	4. Click OK.
	
	5. Close Control Panel.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Encarta World Atlas
	2000.
