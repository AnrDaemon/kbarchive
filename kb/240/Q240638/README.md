## Q240638: World Atlas 2000: Character in Favorite Name Appears As a Dash

	Article: Q240638
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbdisplay kbui kbimu
	Last Modified: 27-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Interactive World Atlas 2000 
	- Microsoft Encarta Interactive World Atlas 2001 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you use any of the following characters in a favorite name when you add a
	favorite in Microsoft Encarta World Atlas 2000, the character appears as a dash
	in the list of favorites on the Favorites menu:
	
	- $
	- &
	- %
	- \
	- /
	- _
	- @
	- {
	- }
	- ~
	- '
	- !
	- #
	- ^
	- ;
	- ?
	- *
	- |
	- :
	- "
	- <
	- >
	
	NOTE: You do not receive a warning message stating that you cannot use one of
	these characters in a favorite name.
	
	CAUSE
	=====
	
	The characters listed in the "Symptoms" section of this article are not valid
	characters for a favorite name.
	
	RESOLUTION
	==========
	
	Do not use any of the characters listed in the "Symptoms" section of this
	article when you add a favorite in Encarta World Atlas 2000.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Encarta World Atlas
	2000.
	
	Additional query words: multi multi-media media mm ewa2k hyphen minus
