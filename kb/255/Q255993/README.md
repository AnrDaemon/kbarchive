## Q255993: Allegiance: Game Hangs Before the Introductory Movie Is Played

	Article: Q255993
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbdisplay igz kbimu msgame
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Allegiance, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start Microsoft Allegiance, the game may stop responding (hang) before
	the introductory movie is played.
	
	RESOLUTION
	==========
	
	To work around this issue, start Microsoft Allegiance with the movies disabled.
	
	To do this, click Start, point to Programs, point to Microsoft Games, point to
	Allegiance, point to Trouble Shooting, and then click Run Allegiance With No
	Movies.
	
	Additional query words: 1.00 msgame igz msngz msn gaming zone lock crash freeze
