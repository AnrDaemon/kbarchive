## Q255960: Allegiance: Colors of the Aiming Reticule Described Incorrectly

	Article: Q255960
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbdocerr kbimu msgame
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Allegiance, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you play Training Mission 3 in Microsoft Allegiance, the colors of the
	aiming reticule are described incorrectly.
	
	MORE INFORMATION
	================
	
	The aiming reticule is colored gray when the enemy is within range. When the
	aiming reticule is on target, it changes color to green.
	
	Additional query words: 1.00 msgame igz msngz crosshairs scope
