## Q255682: XFOR: Connector for Lotus Notes Does Not Support Encryption

	Article: Q255682
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5,5.5 SP1,5.5 SP2,5.5 SP3
	Operating System(s): 
	Keyword(s): exc55 exc55sp1 exc55sp2 exc55sp3
	Last Modified: 09-MAR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 5.5, 5.5 SP1, 5.5 SP2, 5.5 SP3 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Exchange Connector for Lotus Notes does not support Lotus Notes
	encryption.
	
	MORE INFORMATION
	================
	
	This behavior is by design. Both messaging systems support Secure/Multipurpose
	Internet Mail Extension (S/MIME) as an alternative solution for encrypted mail
	flow.
