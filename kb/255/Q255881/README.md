## Q255881: Allegiance: Game Hangs When You Attempt to Launch a Ship

	Article: Q255881
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Allegiance, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Allegiance, the game may stop responding (hang) when you
	attempt to launch a ship.
	
	CAUSE
	=====
	
	This behavior can occur if the following conditions are true:
	
	- A video adapter based on the Permedia3 chip set is installed in the computer.
	
	- Allegiance is configured to use hardware rendering.
	
	RESOLUTION
	==========
	
	To resolve this issue, configure Allegiance to run in software mode.
	
	To do this, click Start, point to Programs, point to Microsoft Games, point to
	Allegiance, point to Trouble Shooting, and then click Run Allegiance In Software
	Mode.
