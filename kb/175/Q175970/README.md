	SUMMARY
	=======
	
	This white paper is an excerpt from the book "Dan Appleman's Developing ActiveX
	Components with Visual Basic 5.0: A Guide to the Perplexed," published by
	Ziff-Davis Press. The white paper discusses developing advanced ActiveX controls
	using Visual Basic.
