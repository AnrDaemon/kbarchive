## Q175874: Flight Simulator 95: How To Check Product ID (PID)

	Article: Q175874
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbsetup
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for Windows 95, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Product ID (PID) number for Flight Simulator for Windows 95 is available
	after a successful installation of the program. To access the PID number, follow
	these steps:
	
	1. Double-click My Computer.
	
	2. Right-click your CD-ROM drive, and then click Open.
	
	3. Double click FSInfo.
	
	MORE INFORMATION
	================
	
	Use the Product ID (PID) number when contacting Microsoft Technical Support.
