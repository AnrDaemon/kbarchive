## Q175140: Bookshelf/Dictionary: Application Key Does Not Function

	Article: Q175140
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbhw kbinterop kbimu kbHardwarekbfaq
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf 98 for Windows 
	- Microsoft Bookshelf 99 
	- Microsoft Bookshelf 2000 
	- Microsoft Encarta World English Dictionary 
	- Microsoft Natural Keyboard version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you press the Application key on the Microsoft Natural Keyboard while using
	Microsoft Bookshelf, the program may not respond to the keystroke.
	
	RESOLUTION
	==========
	
	To work around this problem, right-click instead of using the Application key.
	You may also select a word from left to right or double-click a word in an
	article.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
	
	MORE INFORMATION
	================
	
	This behavior can occur if you select a word from the article panel by dragging
	the mouse pointer over the word from the right to the left or by placing the
	text cursor to the right of the word, pressing and holding down the SHIFT key,
	and then pressing the LEFT ARROW key.
