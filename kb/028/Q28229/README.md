## Q28229: DOCERR: Result of Operation Stored in DESTINATION, Not SOURCE

	Article: Q28229
	Product(s): Microsoft Macro Assembler
	Version(s): 5.0,5.1,5.1a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Macro Assembler (MASM), versions 5.0, 5.1, 5.1a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	On Page 308 of the "Microsoft Macro Assembler 5.0 Programmer's Guide" and on
	page 308 of the "Microsoft Macro Assembler 5.1 Programmer's Guide" the first
	sentence incorrectly states "The result of the operation is stored in the
	source..."
	
	The sentence should read "The result of the operation is stored in the
	destination..."
	
	Additional query words: 5.00 5.10 5.10a
