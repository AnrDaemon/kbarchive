## Q28747: Proper Installation for NEC PowerMate LCD Monitor

	Article: Q28747
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:2.x,3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 2.03, 2.1, 2.11, 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you install Microsoft Windows with a NEC PowerMate LCD monitor, configure
	the monitor for EGA mode (consult the documentation included with the hardware),
	and select the "IBM EGA with High-Resolution Monochrome Display" driver during
	Windows Setup.

	================

	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
