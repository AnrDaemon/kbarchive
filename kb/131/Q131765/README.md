## Q131765: Clear Image Command Unavailable in Paint

	Article: Q131765
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When the Text tool is selected in Paint, the Clear Image command on the Image
	menu is unavailable.
	
	CAUSE
	=====
	
	The Clear Image command is unavailable only when the Text tool is the active
	tool.
	
	RESOLUTION
	==========
	
	Click a different tool in the tool box.
