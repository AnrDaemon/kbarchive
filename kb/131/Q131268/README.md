## Q131268: Baseball 95: Season Schedule Does Not Print With HP560C Driver

	Article: Q131268
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:3.x
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Complete Baseball Guide for Windows, 1995 edition 
	- the operating system: Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to print a Season Schedule in Windows 3.x while using one of the
	following printer drivers, the print dialog box fails to appear and the Season
	Schedule does not print:
	
	- Hewlett-Packard (HP) DeskJet 500C version 4.0, 4.1, 5.0, or 6.1d
	
	- Hewlett-Packard (HP) DeskJet 560C version 4.0, 4.1, 5.0, or 6.1d
	
	- Hewlett-Packard (HP) DeskJet 660C version 6.1d
	
	WORKAROUND
	==========
	
	To print the schedule, use one of the methods below:
	
	- Install the HP DeskJet 500 series printer driver included with Microsoft
	  Windows. For more information about how to perform this task in Windows, see
	  your Windows printed documentation or online Help.
	
	  -or-
	
	- Copy the schedule and paste it into a word processor. Print the schedule from
	  the word processor.
	  1. In Complete Baseball, click Options.
	
	  2. Click Copy.
	
	  3. Click Season Schedule. Click OK.
	
	  4. Open your word processor. Create a new document. Paste the schedule from
	     the Clipboard into your document using the appropriate steps for your word
	     processor.
	
	     For example, in Microsoft Word you would use the following steps:
	     a. On the Edit menu, click Paste.
	
	     b. The schedule is pasted and can be printed like a normal word processor
	        document.
	
	MORE INFORMATION
	================
	
	The Season Schedule prints correctly in Windows 95 using the DeskJet 500 Series
	printer driver version 6.5.5 provided with Windows 95.
