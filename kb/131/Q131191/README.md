## Q131191: Cannot Run The Microsoft Network from Server

	Article: Q131191
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.2,1.3,2.0,2.5
	Operating System(s): 
	Keyword(s): kbnetwork kbmsn
	Last Modified: 14-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.2, 1.3, 2.0, 2.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you connect to another computer, you cannot connect to MSN, The Microsoft
	Network, from that computer.
	
	CAUSE
	=====
	
	You must run The Microsoft Network locally. MSN components cannot be shared over
	a local area network.
	
	RESOLUTION
	==========
	
	To resolve this issue, install The Microsoft Network software on your computer.
