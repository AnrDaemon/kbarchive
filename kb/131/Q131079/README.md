## Q131079: Mac MSPlus: Schedule Information Only Distributed for 3 Months

	Article: Q131079
	Product(s): Microsoft Schedule+ for Windows
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for AppleTalk Networks 
	-------------------------------------------------------------------------------
	
	Schedule+ for AppleTalk Networks will only distribute schedule information
	to other servers for three months, not counting the current month. For
	example, if it is March, the other servers will only receive schedule
	information for April, May, and June.
	
	This is a built-in time frame and cannot be changed.
	
	Additional query words: schedule plus 1.00 mac
