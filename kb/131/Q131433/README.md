## Q131433: Macintosh Encarta 1995: Manual Installation Instructions

	Article: Q131433
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1995 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 26-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia for Macintosh 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the steps to manually install Encarta 1995 for the Macintosh.
	
	MORE INFORMATION
	================
	
	To perform a manual installation of Encarta 1995 for the Macintosh, do the
	following:
	
	1. Insert the Encarta compact disc into the CD-ROM drive.
	
	2. Create a new folder on your hard disk where you want to install Encarta. Name
	  the folder Encarta 1995.
	
	3. Double-click the Encarta CD-ROM icon.
	
	4. Locate and open the Installer Files folder.
	
	5. Drag the following items to the Encarta 1995 folder on your hard drive:
	
	  Animations Launcher
	  Encarta '95
	  Encarta Projector
	  Animation
	
	6. Locate and open the System Files folder on the CD-ROM.
	
	7. Drag all the files in the System Files folder on the disc to the System
	  Folder icon on your hard disk. When you are prompted to place items into the
	  appropriate locations, click OK.
	
	8. Close the System Files folder.
	
	9. Close the Installer Files folder.
	
	10. Open the Encarta 1995 folder on the hard disk.
	
	11. Create two new folders in the Encarta 1995 folder. Name the folders:
	
	  InterActivities
	  Data Files
	
	12. Locate and open the Encarta Files folder on the CD-ROM.
	
	13. Copy the following files to the Data Files folder you created on your hard
	  drive:
	
	  Encarta '95 Data
	  Performance File
	  See Also Index
	  Time1 Data
	  Time2 Data
	
	14. Locate and open the Programs folder located in the Encarta Files folder on
	  the CD-ROM.
	
	15. Make an alias for the following nine items:
	
	  Country Facts
	  Ellis Island
	  Exploring Fractals
	  Mind Maze
	  Orbit Simulation
	  Personal Nutrition
	  Pop/Clim Charts
	  World Languages
	  World Music
	
	  NOTE: For more information about creating an alias, see your printed
	  Macintosh documentation or online help.
	
	16. Drag the aliases into the InterActivites folder in the Encarta 1995 folder
	  on the hard disk.
	
	17. Close the Encarta 1995 folder on the hard disk.
	
	18. Make an alias of the Encarta 1995 folder.
	
	19. Drag the Encarta 1995 alias folder to the desktop.
	
	20. Create a new folder named Encarta '95 Preferences.
	
	21. Drag the Encarta '95 Preferences folder to the desktop.
	
	22. Close all windows.
	
	23. Drag the Encarta 1995 alias folder into the Encarta '95 Preferences folder.
	
	24. Drag the Encarta '95 Preferences folder into the Preferences folder in your
	  System Folder.
	
	25. If you have Microsoft Works 4.0x installed on this machine, create a folder
	  named Encarta 1994 Preferences. Place this folder in the Preferences folder
	  in your System Folder.
	
	26. Close all the windows and restart your Macintosh.
