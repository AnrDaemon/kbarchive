## Q131987: MS PROJECT 4 FOR WIN SBS Corrections and Comments

{% raw %}

	Article: Q131987
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 25-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Project 4 for Windows Step by Step ISBN 1-55615-595-6 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains information on known errors, corrections, and comments
	relating to the Microsoft Press book "Microsoft Project 4 for Windows Step by
	Step."
	
	The following topic is covered:
	
	- Page 31: Step 6 References Unseen Start Date Column
	
	MORE INFORMATION
	================
	
	In addition to a description of the book's problems, this document also includes
	a section labeled "Correction." Please note that the "Correction" section is
	worded for correcting the book and does not necessarily address the problem
	introduced by the book error.
	
	Page 31: Step 6 References Unseen Start Date Column
	---------------------------------------------------
	
	Although step 6 discusses the start date, the illustration on page 31 does not
	show the start date column.
	
	Correction:
	
	Add the following margin note below step 6:
	
	  " You may need to move the vertical divider to the right in order to see the
	  start date column. " (without the quotation marks)
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
