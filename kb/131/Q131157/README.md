## Q131157: Err Msg Installing LaserWriter Pro 810: Could Not Create...

	Article: Q131157
	Product(s): Microsoft Windows NT
	Version(s): 3.50
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.5 
	- Microsoft Windows NT Server version 3.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to install a printer driver for the Apple LaserWriter Pro 810,
	the files appear to be copied to your computer and then the following error
	appears:
	
	  Could not create printer: Printer driver is unknown.
	
	This problem occurs when a postscript printer was not previously installed.
	Further attempts to install a printer continue to produce the error.
	
	WORKAROUND
	==========
	
	To work around this problem, install another postscript printer and then install
	the Apple LaserWriter Pro 810.
	
	NOTE: You must reboot the computer before installing another postscript printer
	driver.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 3.5. This
	problem was corrected in Windows NT version 3.51.
