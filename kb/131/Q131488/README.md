## Q131488: Today's Date Is the Default Date in Go To Dialog Box

	Article: Q131488
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0; Win95:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	When you choose Go To on the Edit menu or press CTRL+Q to bring up the Go
	To dialog box, the default date and week in the Go To dialog box will be
	the current date and week. This functionality is similar to the Today
	button provided for mouse users and makes it convenient for you to return
	to today's date or some other specific date if you frequently use the
	keyboard.
	
	Additional query words: schedule plus 7.00
