## Q131343: &quot;Invalid Page Fault&quot; Running Office 95 Help Wizard

	Article: Q131343
	Product(s): Microsoft Windows NT
	Version(s): 3.51
	Operating System(s): 
	Keyword(s): kbtool
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.51 
	- Microsoft Windows NT Server version 3.51 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you install Office 95 under Windows NT 3.51, start help from one of the
	Office applications, start the application, and then choose an "Answer Wizard"
	from the Help menu, the help menu may disappear or appear as an incomplete
	dialog box. If the Find tab button is blank and you attempt to use this
	function, the following error appears:
	
	  WINHLP32 caused an invalid page fault in module WINHLP32.EXE at
	  0137:00401CF5.
	
	RESOLUTION
	==========
	
	To correct this problem, install the latest U.S. Service Pack for Windows NT
	version 3.51.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 3.51. This
	problem has been corrected in the latest U.S. Service Pack for Windows NT
	version 3.51. For information on obtaining the Service Pack, query on the
	following word in the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
	
	Additional query words: prodnt
