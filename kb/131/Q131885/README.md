## Q131885: AMI BIOS Upgrade Needed for Adaptec 2940 PCI SCSI Controller

	Article: Q131885
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Windows 95 cannot detect or enumerate an Adaptec 2940 SCSI controller installed
	on a PCI bus on computers with an AMI ROM BIOS version 1.00.07.AF2.
	
	RESOLUTION
	==========
	
	To resolve this problem, upgrade the AMI ROM BIOS to version 1.00.09.AF2. For
	information about obtaining this BIOS upgrade, please contact the computer's
	manufacturer.
	
	If upgrading the ROM BIOS does not immediately resolve the problem, turn your
	computer off and back on, and then run Windows 95 Setup again.
