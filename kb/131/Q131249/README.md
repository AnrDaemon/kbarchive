## Q131249: Cannot Post BBS Message with Copied Attachment

	Article: Q131249
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.0,1.05,1.2,1.3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.0, 1.05, 1.2, 1.3 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you copy an attachment from an existing bulletin board (BBS) message, paste
	that attachment into a new message, and then post the new message, you receive
	one of the following error messages:
	
	- 
	
	  EXPLORER caused an invalid page fault in module CRTDLL.DLL at
	  0137:7fd5c8e6.
	
	- 
	
	  This program has performed an illegal operation and will be shut down.
	
	RESOLUTION
	==========
	
	Insert the attachment into the message from your computer's hard disk.
