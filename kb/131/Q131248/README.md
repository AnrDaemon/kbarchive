## Q131248: In-Progress BBS Message Not Saved When Disconnect Occurs

	Article: Q131248
	Product(s): The Microsoft Network
	Version(s): 1.00 1.05 1.2 1.3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.0, 1.05, 1.2, 1.3 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	While you are composing a bulletin board (BBS) message, The Microsoft Network
	disconnects and does not save your message.
	
	RESOLUTION
	==========
	
	If The Microsoft Network presents you with a Disconnect dialog box, before you
	click OK, manually switch back to the BBS Compose window and click Save As on
	the File menu. Otherwise, you must start over after you reconnect to The
	Microsoft Network.
