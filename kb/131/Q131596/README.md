## Q131596: MAC ANCIENT LANDS: Movies Look Hazy and Washed Out

	Article: Q131596
	Product(s): Microsoft Home Multimedia Titles
	Version(s): MACINTOSH:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Ancient Lands for the Macintosh, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you view a movie, colors appear to fade out, and then the movie looks hazy.
	
	CAUSE
	=====
	
	Your computer monitor is probably set for 256 colors. The movies appear sharper
	if you set your monitor for greater than 256 colors.
	
	RESOLUTION
	==========
	
	To change you monitor's color settings, do the following:
	
	1. From the Apple menu, choose Control Panels.
	
	2. Click Monitors.
	
	3. Select a setting greater than 256 colors.
	
	4. Close the Control Panel.
	
	MORE INFORMATION
	================
	
	Section 4 of the Ancient Lands ReadMe file states:
	
	Ancient Lands will run on any 256-color, 640x480 (13-inch) or higher resolution
	Macintosh compatible monitor, although the movies look and perform best in
	"Thousands of colors."
	
	For more information about monitor settings, see the Ancient Lands ReadMe file.
