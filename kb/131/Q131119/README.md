## Q131119: Compilers Supported by Visual FoxPro LCK

	Article: Q131119
	Product(s): Microsoft FoxPro
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-FEB-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual FoxPro for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Visual FoxPro Library Construction Kit (LCK) provides an application
	programming interface (API) to extend the FoxPro Xbase language. Using libraries
	of C or assembly language functions, a programmer can create code to provide
	additional flexibility and functionality to Visual FoxPro.
	
	The Visual FoxPro LCK supports the Microsoft C/C++ Compiler (CL.EXE) that ships
	with Microsoft Visual C++, 32-bit Edition, version 2.0 or later.
