## Q131682: PC Win: Err Msg:...Not Have Proper Access to the Postoffice...

	Article: Q131682
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you store your mail message files (MMFs) locally on another network share,
	and you rename that share or lose access to it, you will not be able to log into
	Mail. You will receive the following error message:
	
	  You do not have the proper access to the postoffice. Please contact your
	  system administrator.
	
	RESOLUTION
	==========
	
	Create a new share with the same share name as where the MMFs used to be
	located. Now you can log on with no errors and point to the new location of the
	MMFs.
