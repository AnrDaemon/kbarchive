## Q131075: Mac Wkst: Default User Preferences

	Article: Q131075
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, version 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	New users added to an AppleTalk Mail Server will inherit the Network Manager's
	preferences at the time they are created.
	
	If the Network Manager changes preferences, the existing user's preferences will
	not be affected.
