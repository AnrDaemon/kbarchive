## Q131250: Clicking Top or Bottom of BBS Toolbar Causes Attachment View

	Article: Q131250
	Product(s): The Microsoft Network
	Version(s): 1.00 1.05 1.2 1.3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.0, 1.05, 1.2, 1.3 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The message list in the bulletin board (BBS) window appears in Attachment view
	when you did not select this view. The window may be empty, if no messages have
	attachments.
	
	CAUSE
	=====
	
	Clicking the top or bottom of the toolbar forces the BBS window to appear in
	Attachment view.
	
	RESOLUTION
	==========
	
	On the View menu, click the view you want.
