## Q131592: Mac Encarta: Cannot Check Spelling

	Article: Q131592
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1995 edition; MACINTOSH:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia for Macintosh 
	- Microsoft Encarta 96 Encyclopedia for Macintosh 
	- Microsoft Encarta 97 Encyclopedia for Macintosh 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Encarta for the Macintosh does not include a spelling checker.
	
	MORE INFORMATION
	================
	
	This feature is under review and will be considered for inclusion in a future
	release.
