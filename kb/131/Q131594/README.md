## Q131594: MAC ANCIENT LANDS: Movies Look Hazy and Washed Out

	Article: Q131594
	Product(s): Microsoft Home Multimedia Titles
	Version(s): MACINTOSH:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Ancient Lands for the Macintosh, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	While watching a movie, some of the colors may appear to fade out and then the
	movie will begin to look hazy.
	
	CAUSE
	=====
	
	The computer monitor is probably set for 256 colors. The movies will appear
	clearer after choosing a color setting greater than 256 colors.
	
	RESOLUTION
	==========
	
	To change you monitor's color settings:
	
	1. From the Apple menu choose Control Panels.
	
	2. Select Monitors.
	
	3. Choose a setting greater than 256 colors.
	
	4. Close the Monitors control panel.
	
	MORE INFORMATION
	================
	
	Section 4 of the Ancient Lands Read Me file states:
	
	Ancient Lands will run on any 256-color, 640x480 (13-inch) or higher resolution
	Macintosh compatible monitor, although the movies look and perform best in
	"Thousands of colors."
