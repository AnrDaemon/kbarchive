## Q319028: How To Import Contacts From Netscape Address Book To Microsoft E

	Article: Q319028
	Product(s): The Microsoft Network
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 28-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Internet Access for Macintosh, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to import contacts from Netscape Mail to Microsoft
	Entourage.
	
	The information in this article applies to:
	- Microsoft Entourage
	
	MORE INFORMATION
	================
	
	To import contacts from Netscape Mail to Microsoft Entourage
	
	1. Start Microsoft Entourage.
	
	2. Click File, and then click Import.
	
	3. In the Import window, select Import information from one of these programs,
	  and then click Netscape Communicator.
	
	4. Click the right arrow to continue.
	
	5. Select Addresses as the data you want to import.
	
	6. Click the right arrow to continue, and then click Finish.
	  Your imported Addresses will now be located in the Address Book folder.
