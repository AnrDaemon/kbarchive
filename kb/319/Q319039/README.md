## Q319039: How To Manage Identities In Outlook Express 5.0.4 For Macintosh

	SUMMARY
	=======
	
	This article describes how to manage identities in Outlook Express 5.0.4.
	
	The information in this article applies to:
	- Macintosh Outlook Express 5.0.4
	
	MORE INFORMATION
	================
	
	To create an Identity in Outlook Express 5.0.4
	
	1. Start Outlook Express.
	
	2. On the File menu, click Switch Identity.
	
	3. When prompted "Are you sure you want to close all connections and windows for
	  the current identity?", click Switch.
	
	4. In the Select an identity window, click New.
	  If you prefer to be prompted to select an identity each time you start Outlook
	  Express, select Show this list at startup.
	
	5. After Identity Name:, type the label you wish to associate with the identity.
	
	6. Confirm that under Outlook Express Defaults, Base initial settings on is
	  selected.
	
	7. Click OK.
	  An alternate identity has been created. You must now configure separate e-mail
	  account settings for this identity.
	
	To switch between Identities in Outlook Express 5.0.4
	
	1. On the File menu, click Switch Identity.
	
	2. When prompted "Are you sure you want to close all connections and windows for
	  the current identity?", click Switch.
	
	3. Select the Identity you want to use.
	
	4. Click OK.
