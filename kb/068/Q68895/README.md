## Q68895: Adobe Type Manager 1.0 Does Not Print in Color

{% raw %}

	Article: Q68895
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Adobe Type Manager 1.0 for Windows 3.0 does not print in color to the
	Hewlett-Packard (HP) PaintJet family of printers. When this utility is
	installed, text that normally would print in color prints in black, regardless
	of the text color.
