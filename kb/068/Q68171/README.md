## Q68171: Archive's Qicstream Tape Backup Hangs Windows

	Article: Q68171
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 31-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You may be unable to run Microsoft Windows version 3.0 after doing a tape backup
	using Archive's Qicstream tape backup software.
	
	WORKAROUND
	==========
	
	To work around the problem, do one of the following:
	
	- Restore the Windows directory from the tape backup.
	
	  -or-
	
	- Reinstall Microsoft Windows.
	
	MORE INFORMATION
	================
	
	Maynard Electronics is currently researching a solution to this problem. No
	upgrade is currently available.
	
	For more information, contact Maynard Electronics Technical Support.
	
	The Qicstream product included here is manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
