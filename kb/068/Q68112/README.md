## Q68112: ASD: Coordinate Systems Available

	Article: Q68112
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	By default, Microsoft Aircraft & Scenery Designer uses the Flight Simulator
	coordinate system. To change the coordinate system, do the following:
	
	1. From the Mode menu, choose option J, Scenery Design.
	
	2. From the Scenery Designer menu, choose option 1, Static Scenery Designer.
	
	3. Choose option L, Design Preferences.
	
	4. Press 0 (zero) to select the Location System that you want.
	
	Note that Location System contains the following three options:
	
	- FS Coordinate System
	
	- North Latitude/West Longitude
	
	- North Latitude/East Longitude
