## Q68758: Problems with Canon LBP-8II Driver in Windows 3.0

	Article: Q68758
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The Canon LBP 8II printer driver shipped with Microsoft Windows version 3.0 does
	not support scalable fonts. Canon has a new driver that does support scalable
	fonts.
	
	To obtain the new driver, contact Canon. Use the Courier font from the current
	driver as a workaround to this problem until the new driver arrives.

	we make no warranty, implied or otherwise, regarding this product's performance
	or reliability.
