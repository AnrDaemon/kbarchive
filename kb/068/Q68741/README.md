## Q68741: ASD: 747 Checklist Doesn't Reset After Mode Reset

{% raw %}

	Article: Q68741
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	Features that are unique to the 747 are not reset after a crash or after mode
	reset occurs. This is because the new features are not stored in the mode file.
	Options that are saved with a mode have remained unchanged from Flight Simulator
	4.00.
	
	The following features will not reset automatically:
	
	- Parking brakes
	
	- Speed brakes
	
	- Thrust reversers
	
	- EICAS checklists
