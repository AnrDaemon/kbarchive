## Q68904: Windows Applications that Work in the Windows Working Model

	Article: Q68904
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	Only the following Windows applications are functional in the Microsoft
	Windows Working Model:
	
	- File Manager
	
	- Solitaire
	
	- Control Panel
	
	- Clock
	
	- Write
	
	All other Windows applications in the Working Model are disabled. When
	you attempt to start a disabled application, Windows Working Model
	returns the following warning message
	
	  Available in the full version of Windows:...
	
	and a brief description about what the application does.
