## Q68683: ASD: Unable to Rotate Objects in the Scenery Editor

{% raw %}

	Article: Q68683
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	To rotate objects in the Scenery Editor, you must select the Revise option. Note
	that some objects, such as mountains, cannot be revised. When the Revise option
	is not listed in the Edit menu of the Static Scenery Designer, you cannot change
	the heading of the object.
