## Q68133: Primage Daisy-Wheel Printers and Windows 3.00

{% raw %}

	Article: Q68133
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	Primage Printers are daisy-wheel printers and require the Generic Text Only
	printer driver (supplied with Microsoft Windows version 3.0) to operate under
	Windows 3.0.
	
	The Primage product included here is manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
