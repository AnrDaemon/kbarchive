## Q68802: INFO: Obtaining Support for Adobe Type Manager (ATM)

	Article: Q68802
	Product(s): Microsoft Windows Software Development Kit
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): kb16bitonly kbSDKWin16
	Last Modified: 26-JUN-1999
	
	3.00 3.10
	WINDOWS
	kbprg
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows Software Development Kit (SDK) 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft does not address any programming or user questions regarding the Adobe
	Type Manager (ATM). Registered owners of ATM can obtain support by calling (408)
	986-6530. The Adobe technicians are prepared to respond to programming and user
	questions.
