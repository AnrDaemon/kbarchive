## Q68131: Windows Err Msg: You Cannot Format the Current Drive

	Article: Q68131
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	In File Manager, a "Format Diskette Error" dialog box appears if you
	attempt to format the current active floppy disk drive. The following
	error message displays:
	
	  You cannot format the current drive
	
	You cannot format a disk from a floppy disk drive that you are
	currently logged onto. Make another drive active such as the hard disk
	or another floppy drive before trying to format from the desired
	floppy drive.
	
	Additional query words: 3.0 3.00 3.0a 3.00a
