## Q68896: Setting a Startup Directory in WordPerfect 5.1

	Article: Q68896
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	WordPerfect 5.1 has an internal setting for the default startup directory that
	will override the default startup directory specified in the WordPerfect PIF
	file. To use the startup directory specified in the WordPerfect PIF file, you
	must clear WordPerfect's "Location of Files" setting.
	
	MORE INFORMATION
	================
	
	To change the Location of Files setting to an empty field:
	
	1. Start WordPerfect.
	
	2. Press SHIFT+F1 to take you to a setting menu.
	
	3. Choose Location of Files.
	
	4. Delete all the characters in this field, including blanks.
	
	5. Exit the configuration menu.
	
	6. Exit WordPerfect.
	
	7. When you restart WordPerfect, the startup directory specified in the
	  WordPerfect PIF file will be effective.
