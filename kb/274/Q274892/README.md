## Q274892: CFS2: HUD Displays Throttle Control For Number One Engine Only

	Article: Q274892
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you fly an aircraft with multiple engines in Microsoft Combat Flight
	Simulator 2, the Heads-Up Display (HUD) view displays the throttle control for
	the number one engine only.
	
	MORE INFORMATION
	================
	
	Microsoft has confirmed this to be a problem in Microsoft Combat Flight
	Simulator 2: WWII Pacific Theater, version 1.0.
	
	Additional query words: 1.00 msgame combatfs cfs2 hud missing indicator two
