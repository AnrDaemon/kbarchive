## Q274606: CFS2: Collision Detection Does Not Occur in Carrier Mission

	Article: Q274606
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you fly the "Carrier Approach or Landing" mission in Microsoft Combat
	Flight Simulator 2, collision detection does not occur when you land your
	aircraft on an aircraft carrier.
	
	As a result, you can fly your aircraft through the island and other structures.
	
	MORE INFORMATION
	================
	
	Microsoft has confirmed this to be a problem in Microsoft Combat Flight
	Simulator 2: WWII Pacific Theater, version 1.0.
	
	Additional query words: 1.00 msgame combatfs cfs2 landing collides detects
