## Q274616: CFS2: Some Buildings Are Displayed with a Lower Level of Detail

	Article: Q274616
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Combat Flight Simulator 2, some buildings may be
	displayed with a lower level of detail and texturing.
	
	CAUSE
	=====
	
	This behavior can occur if you use custom buildings that are made for Microsoft
	Combat Flight Simulator: WWII Europe Series or Microsoft Flight Simulator 2000.
	
	RESOLUTION
	==========
	
	To work around this issue, turn off the Auto-performance adjustment feature in
	Combat Flight Simulator 2:
	
	1. Start Combat Flight Simulator 2.
	
	2. On the startup Screen, click Settings.
	
	3. Under Display, click to clear the Auto-performance adjustment (affects
	  graphics) check box, and then click OK.
	
	NOTE: Turning off Auto-performance adjustment can adversely affect the
	performance of Combat Flight Simulator 2.
