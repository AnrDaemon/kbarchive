## Q274911: Combat Flight Simulator 2 Quits Unexpectedly

{% raw %}

	Article: Q274911
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbenv kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start a flight or select an aircraft in Microsoft Combat Flight
	Simulator 2: WWII Pacific Theater, the game may quit unexpectedly.
	
	CAUSE
	=====
	
	This behavior can occur if both of the following conditions are true:
	
	- You have copied aircraft from either Microsoft Combat Flight Simulator: WWII
	  Europe Series or Microsoft Flight Simulator 98 into the Combat Flight
	  Simulator 2 Aircraft folder.
	
	- You have not restarted the game.
	
	RESOLUTION
	==========
	
	To resolve this issue, restart the game.
