## Q274864: Simulators: Host Is the Only Player in a Multiplayer Game

{% raw %}

	Article: Q274864
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbui kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	- Microsoft Flight Simulator 2002 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you host a multiplayer game of Microsoft Combat Flight Simulator 2 or
	Microsoft Flight Simulator 2002, you may be the only player in the game.
	
	CAUSE
	=====
	
	This behavior can occur if either of the following conditions are true:
	
	- You click the Ready or OK button before other players have joined the game.
	
	NOTE: You can also eject all other players from a multiplayer game and host a new
	game without any other players.
	
	- You connecting through a proxy server or firewall.
	
	RESOLUTION
	==========
	
	To resolve this issue, use the following methods in the order in which they are
	presented. If the issue continues to occur, proceed to the next method.
	
	Verify At Least Two Players Are Displayed
	
	  Combat Flight Simulator 2:
	
	  To prevent this issue from occurring, make sure that at least two players are
	  displayed in the Status/Player/Aircraft box on the Multiplayer startup screen
	  before you click the Ready button.
	
	  Flight Simulator 2002:
	
	  To prevent this issue from occurring, make sure that at least two players are
	  displayed in the Players box on the Settings - Multiplayer Host Server screen
	  before you click the OK button.
	
	If the issue continues to occur, proceed to the next step:
	
	Configure the Proxy Server or Firewall
	
	For additional information about how to configure a proxy server or firewall for
	use with Combat Flight Simulator 2 and Flight Simulator 2002, click the article
	number below to view the article in the Microsoft Knowledge Base:
	
	  Q240429 DirectX: Ports Required to Play on a Network
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Combat Flight
	Simulator 2: WWII Pacific Theater, version 1.0.
