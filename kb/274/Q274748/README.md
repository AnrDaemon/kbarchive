## Q274748: CFS2: Tarmac and Textures Are Not Displayed Correctly

{% raw %}

	Article: Q274748
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbgraphic kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator: WWII Europe Series, version 1.0 
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you attempt to add the Abacus Pacific Theater Add-On for Microsoft Combat
	Flight Simulator to Microsoft Combat Flight Simulator 2, you may experience some
	or all of the following symptoms:
	
	- Some tarmacs may not be displayed or may be displayed incorrectly.
	
	- Some textures may not be displayed or may be displayed incorrectly.
	
	- Some textures may appear to be in motion.
	
	CAUSE
	=====
	
	This behavior occurs because of performance-related changes and scenery design
	changes in Combat Flight Simulator 2.
	
	MORE INFORMATION
	================
	
	Microsoft does not support third-party scenery or aircraft in Combat Flight
	Simulator 2.
	
	Other third-party scenery and aircraft programs that were created for Flight
	Simulator 98 or Combat Flight Simulator may exhibit behavior similar to the
	symptoms mentioned in the "Symptoms" section of this article.
	
	
	The third-party products discussed in this article are manufactured by vendors
	independent of Microsoft; we make no warranty, implied or otherwise, regarding
	these products' performance or reliability.
