## Q274602: CFS2: Cannot Upgrade from Combat Flight Simulator

	Article: Q274602
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbsetup kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator: WWII Europe Series, version 1.0 
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	You cannot upgrade from Microsoft Combat Flight Simulator to Microsoft Combat
	Flight Simulator 2.
	
	MORE INFORMATION
	================
	
	Combat Flight Simulator and Combat Flight Simulator 2 are different programs.
	
	NOTE: You can import missions from Combat Flight Simulator to Combat Flight
	Simulator 2.
	
	If you want to install both Combat Flight Simulator and Combat Flight Simulator
	2, you must install each program in a different folder.
	
	If you installed both programs in the same folder, uninstall both programs,
	delete the folder in which you installed the programs, and then reinstall each
	programs in a different folder.
	
	Additional query words: 1.00 msgame combatfs cfs2 destination setup
