## Q274611: CFS2: Green Lines Appear on Screen or Buttons Are Misplaced

	Article: Q274611
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you adjust the sound volume in Microsoft Windows, you may experience the
	following symptoms in Microsoft Combat Flight Simulator 2:
	
	- Green lines may appear on the screen.
	
	- Buttons on the screen may be misplaced.
	
	CAUSE
	=====
	
	This behavior can occur if both of the following conditions are true:
	
	- You adjust the volume when you calibrate your game controller in Combat
	  Flight Simulator 2.
	
	- You click Cancel in the Calibration dialog box when you calibrate the game
	  controller.
	
	RESOLUTION
	==========
	
	To resolve this issue, minimize and then maximize Combat Flight Simulator 2.
	
	If the issue continues to occur, quit and then restart Combat Flight Simulator 2.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Combat Flight
	Simulator 2: WWII Pacific Theater, version 1.0.
