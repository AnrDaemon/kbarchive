## Q274398: MSN Companion Entries Missing in Address Book

	Article: Q274398
	Product(s): The Microsoft Network
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Symptoms include the following:
	
	- When the user goes to the Address Book, addresses that were previously
	  inputted are missing.
	
	- When the user tries to re-type some of the missing addresses, an error
	  message appears stating that the address already exists.
	
	CAUSE
	=====
	
	This issue is caused by the MSN Companion address caching system.
	
	RESOLUTION
	==========
	
	1. Have the user unplug the device, and then plug it back in.
	
	2. After the device starts up again, the problem should be resolved.
	
	Note: You can also have the user wait until the next day and try again, but
	unplugging and restarting is preferred.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in MSN Companion Version#1.0.
