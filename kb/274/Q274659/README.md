## Q274659: Combat Flight Simulator 2: Screen Textures Are Distorted

	Article: Q274659
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbgraphic kbhw kbimu kbHardware
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Combat Flight Simulator 2, the textures on the screen
	may be distorted.
	
	CAUSE
	=====
	
	This behavior can occur if a video adapter that is based on the Intel 810 or 815
	graphic chip set is installed in your computer.
	
	RESOLUTION
	==========
	
	You may be able to resolve this issue by installing the latest video driver for
	the Intel chip set from the following Intel Web site:
	
	  http://support.intel.com
	
	If you install the latest video driver and the issue continues to occur, turn off
	Mip Mapping within the game. To do this, follow these steps:
	
	1. Start Combat Flight Simulator 2.
	
	2. On the startup screen, click Settings.
	
	3. Under Display, click Advanced Settings.
	
	4. Click to clear the "MIP mapping" check box, and then click OK twice to return
	  to the startup screen.
