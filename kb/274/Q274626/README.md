## Q274626: Combat Flight Simulator 2: IME Toolbar May Not Be Displayed

{% raw %}

	Article: Q274626
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbenv kbui kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Combat Flight Simulator 2, the Input Method Editor (IME)
	toolbar may not be displayed.
	
	CAUSE
	=====
	
	This behavior can occur if you are using a Japanese-language version of
	Microsoft Windows.
	
	RESOLUTION
	==========
	
	To resolve this issue, click an area on the screen to take the focus away from
	the Edit box, and then click inside the Edit box to display the IME toolbar.
	
	MORE INFORMATION
	================
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
