## Q274177: Visual C++ 6.0 Deluxe Learning Edition Comments and Corrections

	Article: Q274177
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 16-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Visual C++ 6.0 Deluxe Learning Edition ISBN 0-7356-0636-6 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Microsoft Visual C++ 6.0 Deluxe Learning
	Edition, ISBN 0-7356-0636-6.
	
	The following topics are covered:
	
	- Visual C++ Programmer's Guide: Missing CD-ROM
	
	- Learn Visual C++ 6.0 Now
	  Page xxix: Incorrect Instructions For Installing Sample Files
	
	MORE INFORMATION
	================
	
	Visual C++ Programmer's Guide: Missing CD-ROM
	---------------------------------------------
	
	Page xxii of the Visual C++ 6.0 Programmer's Guide that is included in the Visual
	C++ Deluxe Learning Edition refers to the companion CD attached to the back of
	the book. This CD was not included in this package.
	
	The companion CD for this book includes project files for the sample programs
	that are covered in the book. It also includes an index program that supplements
	the book index. Microsoft Press has made these files available in the self
	extracting files listed below.
	
	The following files are available for download from the Microsoft Download
	Center:
	
	  VCPGCode.exe
	  (http://download.microsoft.com/download/vc60pro/mspress/1.0/WIN98Me/EN-US/VCPGCode.exe)
	
	  VCPGIndex.exe
	  (http://download.microsoft.com/download/vc60pro/mspress/1.0/WIN98/EN-US/VCPGIndex.exe)
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
	
	Microsoft used the most current virus detection software available on the date of
	posting to scan this file for viruses. Once posted, the file is housed on secure
	servers that prevent any unauthorized changes to the file.
	
	
	Learn Visual C++ 6.0 Now
	
	Page xxix: Incorrect Instructions For Installing Sample Files
	--------------------------------------------------------------------------------------
	
	Page xxix of the Learn Microsoft Visual C++ 6.0 Now book, included in the Visual
	C++ Deluxe Learning Edition, contains an incorrect setup.exe directory and
	incorrect CD information under the section on "Installing the Samples on Your
	Computer".
	
	Change Step 2:
	"With the Learn Microsoft Visual C++ 6.0 Now compact disk...."
	
	To:
	"With the Microsoft Visual C++ 6.0 Starts Here compact disk..."
	
	Change Step 3:
	"In the Open box, type D:\Examples\setup.exe."
	
	To:
	"In the Open box, type D:\setup.exe."
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
