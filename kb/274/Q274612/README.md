## Q274612: CFS2: Game Minimizes When You Attempt to Calibrate the Joystick

	Article: Q274612
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbenv kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click Calibrate Joystick, Microsoft Combat Flight Simulator 2 may
	minimize to the taskbar.
	
	MORE INFORMATION
	================
	
	Some video adapter chip sets or video drivers may not be able to display Combat
	Flight Simulator 2 and the Calibrate Joystick dialog box at the same time.
	
	NOTE: You may also experience this symptom when you attempt to switch from the
	Calibrate Joystick dialog box to Combat Flight Simulator 2.
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
