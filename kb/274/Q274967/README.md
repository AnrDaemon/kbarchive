## Q274967: CFS2: First Player to Exit Team Game Receives Credit for Kills

	Article: Q274967
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The first player that exits a multiplayer team game of Microsoft Combat Flight
	Simulator 2 receives credit for all of the kills and deaths that were recorded
	by that player's team.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Combat Flight
	Simulator 2: WWII Pacific Theater, version 1.0.
	
	Additional query words: 1.00 msgame combatfs cfs2
