## Q274946: Combat Flight Simulator 2: Virtual Cockpit View Is Too Dark

{% raw %}

	Article: Q274946
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Combat Flight Simulator 2, the Virtual Cockpit view may
	appear too dark.
	
	CAUSE
	=====
	
	This behavior can occur if the lights in the virtual cockpit window are turned
	off.
	
	RESOLUTION
	==========
	
	To resolve this issue, press the L key on the keyboard to turn on the lights in
	the virtual cockpit window.
