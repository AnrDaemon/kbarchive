## Q44767: Cannot Create Color Text in Write

	Article: Q44767
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:2.x,3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 2.03, 2.1, 2.11, 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	Microsoft Windows Write does not have the capability to create editable
	text in different colors. You can use the Clipboard to import color text
	and graphics into Write; however, the text and graphics are not editable
	beyond changing the size.
