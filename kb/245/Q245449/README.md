## Q245449: Err Msg: The Server Has Found an Unexplained Error...

{% raw %}

	Article: Q245449
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kberrmsg kbimu msgamekbfaq
	Last Modified: 16-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Asheron's Call, version 1.0 
	- Microsoft Asheron's Call: Dark Majesty, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you create a new character in Microsoft Asheron's Call, you may receive the
	following error message when you click Done:
	
	  The server has found an unexplained error with this new character. The data
	  may be corrupt or out of date.
	
	CAUSE
	=====
	
	This behavior can occur if the Asheron's Call program is no longer synchronized
	with the Asheron's Call server.
	
	RESOLUTION
	==========
	
	To work around this issue, quit and restart Asheron's Call, and then re-create
	the character.
