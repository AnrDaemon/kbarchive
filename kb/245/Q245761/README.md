## Q245761: PRB: False Report of Trojan Horse Virus in Colchng.Exe

	Article: Q245761
	Product(s): Microsoft Developer Network
	Version(s): WINDOWS:October 99
	Operating System(s): 
	Keyword(s): kbHTMLHelp kbMSDN kbDSupport kbGrpDSTools
	Last Modified: 05-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Developer Network (MSDN) October 99 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Customers running Norton AntiVirus 2000 Version 6.0 for Windows 95 and Windows
	98 are receiving a false report of a Trojan horse virus in the file Colchng.exe
	when trying to install the MSDN Library - October 1999.
	
	CAUSE
	=====
	
	Symantec Corp., the publisher of Norton AntiVirus, has confirmed that this is a
	false report.
	
	RESOLUTION
	==========
	
	An update to the Norton AntiVirus definition set that fixes this problem is
	available to customers on the publisher's Web site.
