## Q245521: French Accented Characters Are Not Displayed in Uppercase

	Article: Q245521
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbhw kbui kbHardware
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you add the French Multilingual or French Canadian keyboard and you enable
	the CAPS LOCK key, some accented characters are not displayed in uppercase.
	
	RESOLUTION
	==========
	
	To work around this problem, press SHIFT to create uppercase accented
	characters.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
