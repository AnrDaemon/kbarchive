## Q134893: 3D OpenGL Screen Saver Restores Windows NT 3.51 Help

	Article: Q134893
	Product(s): Microsoft Windows NT
	Version(s): 
	Operating System(s): 
	Keyword(s): kbui
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.51 
	- Microsoft Windows NT Server version 3.51 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you return to your desktop from any of Windows NT 3D OpenGL screen savers,
	any minimized Windows NT 3.51 Help files that use the Windows 95 Help engine are
	restored to full size.
	
	WORKAROUND
	==========
	
	To work around this problem, use a non-OpenGL screen saver.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 3.51. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
