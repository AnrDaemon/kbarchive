## Q134354: Mustek TwainScan CG-8400 Hand Scanner Does Not Work

{% raw %}

	Article: Q134354
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you upgrade to Windows 95, your Mustek TwainScan CG-8400 hand scanner does
	not work.
	
	CAUSE
	=====
	
	There is no DMA channel available for the hand scanner's card to use.
	
	RESOLUTION
	==========
	
	Mustek has a new scanner card available that does not require a DMA channel. To
	resolve the problem described in this article, obtain the new scanner card from
	Mustek.
