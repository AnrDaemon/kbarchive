## Q134526: Compaq Deskpro Drive D Unavailable After Installing Windows 95

	Article: Q134526
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	On a Compaq Deskpro, after you install Windows 95 and start the computer the
	first time, drive D (attached to a controller in an EISA slot) is inaccessible.
	The second time you start the computer and thereafter, drive D works correctly.
	
	RESOLUTION
	==========
	
	If after you install Windows 95, drive D is inaccessible, shut down the computer
	and restart Windows 95. The second time you start Windows 95, Windows 95
	recognizes and configures drive D correctly.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Windows 95. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
