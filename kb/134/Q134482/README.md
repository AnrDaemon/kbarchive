## Q134482: After Running Setup Uninstall, Directories Not Deleted

{% raw %}

	Article: Q134482
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you run the Uninstall option in Setup to go back to a previous Windows
	version, you find that Uninstall did not deleted some files and directories.
	
	CAUSE
	=====
	
	Uninstall may leave behind some directories and files that you may have created.
	
	RESOLUTION
	==========
	
	Manually delete these files and directories.
	
	Additional query words: remove Win31
