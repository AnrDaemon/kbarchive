## Q134480: &quot;Wizardry - Crusaders of the Dark Savant&quot; Game Terminated

	Article: Q134480
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run the game "Wizardry - Crusaders of the Dark Savant," by Sir-Tech on
	a Pentium system, you may receive the following error message:
	
	  The program terminated unexpectedly
	
	If you run the game in MS-DOS mode, the system may spontaneously restart.
	
	CAUSE
	=====
	
	The Intel Pentium processor is not compatible with the game's built-in copy-
	protection code.
	
	
	RESOLUTION
	==========
	
	Contact the vendor to see if an updated version of the game is available. The
	vendor's techical support number as of 1992 (the release date of the program) is
	(315) 393-6644, M - F 9:00 A.M. - 9:00 P.M. Eastern time.
