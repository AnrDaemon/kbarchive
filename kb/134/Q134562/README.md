## Q134562: HyperTerminal: File Transfers Using Kermit Fails

{% raw %}

	Article: Q134562
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95 appscomp kbAppCompatibility
	Last Modified: 28-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you upload a file in HyperTerminal using the Kermit file transfer protocol,
	the Kermit File Receive dialog box displays the following error message in the
	Last Error field:
	
	  No response (timeout)
	
	CAUSE
	=====
	
	HyperTerminal does not support the Kermit file transfer protocol on CompuServe.
	
	RESOLUTION
	==========
	
	Transfer the file by using a different protocol within HyperTerminal, such as
	the Z-Modem file transfer protocol.
