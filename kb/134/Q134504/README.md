## Q134504: Zeos Pantera CMOS Settings Corrupted During Windows 95 Setup

	Article: Q134504
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run Windows 95 Setup on a Zeos Pantera, the computer does not restart.
	You may not be successful when you try to enter the BIOS Setup.
	
	CAUSE
	=====
	
	The CMOS or NVRAM settings in the BIOS have become corrupted.
	
	RESOLUTION
	==========
	
	You may have to clear the CMOS settings. Contact Zeos Technical Support for
	instructions about the location and use of the jumper for this purpose.
