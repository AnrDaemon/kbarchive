## Q134598: Corel Draw 3.0 TrueType Fonts Printed Incorrectly

{% raw %}

	Article: Q134598
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you print Corel Draw 3.0 TrueType fonts to a PostScript printer, incorrect
	characters may be printed. For example, when you use the Penguin font, a "T" may
	be printed as a "D."
	
	CAUSE
	=====
	
	Some of the TrueType fonts included with Corel Draw 3.0, including Penguin,
	Bodnoff, Aardvark, Bangkok, and Chalk, print incorrectly.
	
	RESOLUTION
	==========
	
	Please contact Corel Corporation for further information about this issue.
