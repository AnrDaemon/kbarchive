## Q134490: GP Fault with PowerBuilder 4.0 Sample Applications

{% raw %}

	Article: Q134490
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you open the sample file Org.pbl in PowerBuilder 4.0 for Windows NT, you
	may receive a general protection (GP) fault when you click a chart in
	PowerBuilder.
	
	CAUSE
	=====
	
	Some of the sample applications shipped with PowerBuilder 4.0 for Windows NT
	contain multiline edit controls that are not compatible with Windows 95.
