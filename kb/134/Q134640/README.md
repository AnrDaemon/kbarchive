## Q134640: How to Force Uppercase Input by Using the Form Designer

{% raw %}

	Article: Q134640
	Product(s): Microsoft FoxPro
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-FEB-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual FoxPro for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	You can force text in an edit box or text box to be converted to uppercase as it
	is typed in a form.
	
	MORE INFORMATION
	================
	
	The Format property for a text or edit box is roughly equivalent to the FoxPro
	version 2.x function codes on an @ GET statement. If you set this property equal
	to an exclamation point (!), text will automatically be converted to uppercase
	as it is typed.
