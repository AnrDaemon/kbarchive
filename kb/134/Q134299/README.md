## Q134299: Sound Blaster WaveStudio 1.1 Out of Memory in Windows 95

	Article: Q134299
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you record sounds using Sound Blaster WaveStudio version 1.1, you may
	receive the following error message:
	
	  Not enough memory available for recording.
	
	CAUSE
	=====
	
	WaveStudio version 1.1 is not compatible with Windows 95.
	
	RESOLUTION
	==========
	
	Please contact Create Labs for information about upgrading to a newer version of
	WaveStudio.

	================
