## Q134519: Adjusting Color Intensity in Picture Publisher Has no Effect

	Article: Q134519
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): kb3rdparty kbprint kbPrinting
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you print from Micrografx Picture Publisher 4.0, adjusting the color
	intensity seems to have no effect.
	
	CAUSE
	=====
	
	Picture Publisher 4.0 dithers the image to 4 bits per pixel before passing the
	image to the printer.
	
	RESOLUTION
	==========
	
	Contact the program's manufacturer to inquire about obtaining an updated version
	of the program.
