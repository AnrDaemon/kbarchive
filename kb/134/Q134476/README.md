## Q134476: MS Paint Color Eraser Tool Does Not Erase Bitmap

	Article: Q134476
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to use the Color Eraser tool in Microsoft Paint, you find that you
	cannot make changes to a bitmap.
	
	CAUSE
	=====
	
	You are trying to edit a bitmap that originated as a 24-bit image.
	
	RESOLUTION
	==========
	
	After you make a backup copy of the image in its current format, save the bitmap
	in 256 (or fewer) color format, and then open it again.
