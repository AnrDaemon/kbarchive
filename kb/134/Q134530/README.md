## Q134530: Error Message: VGASWAP VDD Control Hook Recursion

	Article: Q134530
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You may receive the following error message in Windows 95:
	
	  386MAX ERROR: VGASWAP VDD Control Hook Recursion
	
	CAUSE
	=====
	
	The Qualitas 386Max Memory Manager VGASWAP feature is not completely compatible
	with Windows 95.
	
	RESOLUTION
	==========
	
	Disable the VGASWAP feature. For information about doing so, please consult the
	product documentation or call Qualitas Technical Support.
