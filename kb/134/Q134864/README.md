## Q134864: PPT7: Levels of Undo Greater Than Specified in PowerPoint

	Article: Q134864
	Product(s): Microsoft PowerPoint for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): kbui kbusage
	Last Modified: 15-APR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft PowerPoint for Windows 95, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the multiple Undo button, you can undo more previous edits on a
	PowerPoint presentation than specified in the Options dialog box.
	
	MORE INFORMATION
	================
	
	You can set the maximum number of undos by following these steps:
	
	1. On the Tool menu, click Options.
	
	2. Click the Edit tool.
	
	3. In the Maximum Number Of Undos box, enter a value between 3 and 150 or click
	  the arrows to select a new number.
