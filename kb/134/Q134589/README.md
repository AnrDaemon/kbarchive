## Q134589: Some Programs May Not Print in Windows 95

	Article: Q134589
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Some Windows 3.x-based programs may not print correctly in Windows 95.
	
	CAUSE
	=====
	
	These programs are not compatible with the stricter printing APIs in Windows 95.
	
	RESOLUTION
	==========
	
	Contact the program's manufacturer to inquire about obtaining an upgraded
	version of the program that solves this problem.
	
	STATUS
	======
	
	This situation is most likely caused by design changes in Windows 95. Microsoft
	has confirmed that it is not caused by a problem in Windows 95.
