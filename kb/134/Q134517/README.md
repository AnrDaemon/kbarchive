## Q134517: No Sound Playing Turtle Tools CD-ROM from SoundBank

{% raw %}

	Article: Q134517
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	SoundBank's Turtle Tools does not play sounds on a Sony CDU-541 CD-ROM drive.
	
	CAUSE
	=====
	
	The Turtle Tools CD-ROM contains an invalid table of contents that cannot be
	read using the Windows 95 protected-mode drivers.
	
	RESOLUTION
	==========
	
	To play sounds from this CD-ROM in Windows 95, you must use real-mode CD-ROM
	drivers.
