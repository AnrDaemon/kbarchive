## Q134801: Help Not Available for WINS Manager Import Static Mappings

{% raw %}

	Article: Q134801
	Product(s): Microsoft Windows NT
	Version(s): 3.5 3.51
	Operating System(s): 
	Keyword(s): kbtool
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server versions 3.5, 3.51 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you choose Help in the "Select Static Mapping File" dialog box to import
	static mappings in Windows Internet Name Service (WINS) Manager (WINSADMN.EXE),
	the following error messages appear in the respective versions of Windows NT:
	
	Windows NT 3.5
	--------------
	
	  HELP is not available for this request.
	
	Windows NT 3.51
	---------------
	
	  The topic does not exist. Contact your application vendor for an updated Help
	  file. (129)
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT versions 3.5 and
	3.51. This problem was corrected in Windows NT Workstation or Server version
	4.0.
	
	Additional query words: dhcp winsadmn.hlp entries
