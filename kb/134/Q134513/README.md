## Q134513: ClarisWorks Setup Error Message: Cannot Find Win32s

{% raw %}

	Article: Q134513
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to run the Setup program for ClarisWorks version 1.0 for Windows
	3.0, you receive the following error message:
	
	  Cannot Find Win32s
	
	CAUSE
	=====
	
	The Setup program checks for the existence of several components that are not
	actually required in Windows 95.
	
	RESOLUTION
	==========
	
	To work around this problem, use the alternate Setup program (Setupcw.exe)
	located on disk 2.
	
	STATUS
	======
	
	This problem no longer occurs in ClarisWorks version 2.0 for Windows 3.0.
