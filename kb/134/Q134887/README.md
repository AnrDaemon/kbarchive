## Q134887: Encarta '95: No Scroll Bars in Some Interactivities

	Article: Q134887
	Product(s): Microsoft Home Kids Products
	Version(s): 1995 edition
	Operating System(s): 
	Keyword(s): kbgraphic kbmm kbui kbimukbbuglist
	Last Modified: 26-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia for Macintosh 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When run Encarta 95 on a computer running Microsoft Windows 95 or Microsoft
	Windows 98, you may not see scroll bars when you view the Personal Nutrition
	Interactivity or the Exploring Fractals Interactivity.
	
	CAUSE
	=====
	
	This behavior can occur because Encarta 95 Interactivities uses non- standard
	scroll bars that do not function properly in Windows 95/98.
	
	STATUS
	======
	
	This problem has been resolved in Microsoft Encarta Encyclopedia for Windows,
	1996 edition.
	
	MORE INFORMATION
	================
	
	If you install Microsoft IntelliPoint version 2.2 and enable the Universal
	Scrolling feature, you can use the IntelliMouse wheel to scroll the
	interactivity windows in Encarta 95.
