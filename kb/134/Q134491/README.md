## Q134491: Stuttering Sound in Lion King 1.1 with ESS488 Sound Card

	Article: Q134491
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run certain programs (such as Lion King 1.1) with an ESS488 sound card,
	the sound may seem to stutter at times.
	
	CAUSE
	=====
	
	The ESS488 sound card has performance problems with some CD-ROM drives.
	
	RESOLUTION
	==========
	
	Try using the CD-ROM drive's real-mode drivers instead of the Windows 95
	protected-mode drivers.
