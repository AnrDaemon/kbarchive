## Q134494: Drop-Down Menu Not Repainted Properly in WinFax Pro 4.0

	Article: Q134494
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click an option in a drop-down menu while you are editing a fax in
	WinFax Pro 4.0, all the items in the list become shaded as if they are being
	selected.
	
	CAUSE
	=====
	
	WinFax Pro 4.0 is not handling the menu tracking features for Windows 95
	properly.
	
	RESOLUTION
	==========
	
	This issue is cosmetic only; there are no adverse effects.
