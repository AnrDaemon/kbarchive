## Q134229: Attachmate Zip Host Manager Requires Patch

	Article: Q134229
	Product(s): Microsoft SNA Server
	Version(s): WINDOWS:2.0,2.1,2.11,3.0,4.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbnetwork sna4
	Last Modified: 12-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft SNA Server, versions 2.0, 2.1, 2.11, 3.0, 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use Attachmate ZIP! Host Print Manager in conjunction with the an SNA
	Server Windows 3.x client, the following errors may occur:
	
	- A null RU can be sent to the mainframe causing printing to fail.
	
	- Chaining is not performed correctly.
	
	- The selected font size does not take affect until the second page of a
	  printout.
	
	- UNBIND is not recognized as the end of a print job. (End Bracket is
	  recognized).
	
	RESOLUTION
	==========
	
	To correct the problem, contact Attachmate Technical Support at (425) 957-
	4607.
