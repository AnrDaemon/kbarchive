## Q134499: Unreadable Font in Info Window in Map Info

	Article: Q134499
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click Info in the toolbox in Map Info, the text is in an unreadable
	font. There is no option to change fonts.
	
	CAUSE
	=====
	
	This is caused by a problem in Map Info versions earlier than 3.00.05.
	
	
	RESOLUTION
	==========
	
	Upgrade to Map Info version 3.00.05 or later.
