## Q134574: Cannot Format SRAM Card

	Article: Q134574
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You cannot format an SRAM card using the graphical user interface (GUI) because
	the Full and Quick format options are not available.
	
	CAUSE
	=====
	
	There is a problem with the device driver for the SRAM card.
	
	RESOLUTION
	==========
	
	Contact the SRAM card manufacturer to inquire about obtaining an updated driver
	that solves this problem.
	
	MORE INFORMATION
	================
	
	The SRAM device driver is returning the device parameters for a 128K SRAM card,
	no matter what size card is actually inserted in the PCMCIA slot.
