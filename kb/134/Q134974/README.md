## Q134974: Microsoft Fax Transmits at Lower Connection Speed

{% raw %}

	Article: Q134974
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are faxing a document with Microsoft Fax, the default connection speed
	may be set lower than the highest possible connection speed.
	
	CAUSE
	=====
	
	This behavior occurs if you are using one of the following modems. Although you
	can use a higher transmission speed with these modems, transmissions may be
	unreliable.
	
	- US Robotics modem
	
	- Intel 14.4
	
	- Gateway Telepath
	
	RESOLUTION
	==========
	
	To use a higher transmission speed, follow these steps:
	
	1. Double-click the Mail And Fax tool in Control Panel.
	
	2. Click Microsoft Fax, and then click Properties.
	
	3. On the Modem tab, click your modem, and then click Properties.
	
	4. Click the Advanced button.
	
	5. Click the Disable High Speed Transmissions check box to clear it.
	
	6. Click OK.
