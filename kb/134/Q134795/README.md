## Q134795: Streets 4.0: &quot;No Place Near&quot; Message When Clicking Map Edge

{% raw %}

	Article: Q134795
	Product(s): Microsoft Automap
	Version(s): 4.00
	Operating System(s): 
	Keyword(s): kbbuglist
	Last Modified: 25-AUG-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Automap Streets for Windows, version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you click a street that is very close to the edge of a map window the
	following message appears on the Status Line, located at the lower left corner
	of the screen:
	
	  No place near
	
	RESOLUTION
	==========
	
	Scroll the map window to move the desired street away from the edge of the
	window, then click the street. The makes the street name display properly.
	
	MORE INFORMATION
	================
	
	When you click a street, a blue box appears that contains the street name; the
	Status Line contains the location address.
	
	Microsoft has confirmed this to be a problem in Microsoft Automap Streets version
	4.0 for Windows. We are researching this problem and will post new information
	here in the Microsoft Knowledge Base as it becomes available.
