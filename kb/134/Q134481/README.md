## Q134481: MS Paint Text Tool Does Not Work with Large Size Zoom Option

	Article: Q134481
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are using the Large Size option on the Zoom menu in Microsoft Paint,
	the Text tool is disabled.
	
	CAUSE
	=====
	
	The Text tool types the exact text image directly into the bitmap, without any
	provision for scaling.
	
	RESOLUTION
	==========
	
	Do not use the Text tool in Microsoft Paint when you are using the Large Size
	option on the Zoom menu.
