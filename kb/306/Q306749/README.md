## Q306749: ENC2002: ShockWave Script Error When You Double-Click Back

	Article: Q306749
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Encyclopedia Deluxe 2002, version 1.0 
	- Microsoft Encarta Encyclopedia Standard 2002, version 1.0 
	- Microsoft Encarta Reference Library 2002, version 1.0 
	- Microsoft Encarta Reference Library 2002 - DVD Edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you visit the following Interactivities
	
	  - City Climate
	  - Population chart
	
	and then double-click the Back button, you may receive the following message:
	
	  Shockwave
	  Script error. Continue?
	  Yes No
	
	If you do not click either Yes or No, you receive the following error message:
	
	  One or more core application components are not functioning properly. Some
	  features of Encarta Encyclopedia will not work. Click OK to continue without
	  fixing the problem. Click Cancel to exit the encyclopedia.
	
	If you click Cancel, you receive the following Watson error message:
	
	  Encarta 2002 has encountered a problem and needs to close. We are sorry for
	  the inconvenience.
	
	  AppName: enc2002.exe
	  AppVer: 11.0.1013.809
	  ModName: unknown
	  ModVer: 0.0.0.0
	  Offset: 6906ee8f
	
	CAUSE
	=====
	
	This behavior occurs when you click the Back button twice.
	
	RESOLUTION
	==========
	
	To resolve this issue when you use the Back button, click the button only once.
	Wait for a response before you click Back a second time.
	
	To work around this issue, click Yes or OK when you receive the initial script
	error message.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
