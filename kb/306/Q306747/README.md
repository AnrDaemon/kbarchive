## Q306747: ENC: How to Obtain Free Membership to Encarta Online Deluxe

{% raw %}

	Article: Q306747
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 28-JUN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Encyclopedia Deluxe 2003 
	- Microsoft Encarta Reference Library 2003, version 1.0 
	- Microsoft Encarta Reference Library 2003 - DVD Edition 
	- Microsoft Encarta Encyclopedia Deluxe 2002, version 1.0 
	- Microsoft Encarta Reference Library 2002 - DVD Edition 
	- Microsoft Encarta Reference Library 2002, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to obtain and use a free membership to Microsoft
	Encarta Online Deluxe Encyclopedia if you have a North American edition of
	Encarta Encyclopedia Deluxe 2002 or 2003 or Encarta Reference Library 2002 or
	2003 on CD-ROM or DVD-ROM.
	
	MORE INFORMATION
	================
	
	Encarta Online Deluxe Encyclopedia is a premium Internet multimedia encyclopedia
	with 42,000 articles, an atlas, a dictionary, Web links, and more.
	
	When you join Club Encarta (by registering online), you are offered a free
	subscription to Encarta Online Deluxe Encyclopedia. This is good for one year
	from the date of registration, not from the date of purchase.
	
	NOTE: This offer expires Oct 31, 2003.
	
	For example, if you join Club Encarta on Oct 31, 2003, your membership to Encarta
	Online Deluxe is good until Oct 31, 2004. If you register on November 1st, 2003
	or later, membership to Encarta Online Deluxe is not offered.
	
	You can use the Online Deluxe Encyclopedia from any computer with Internet
	access. The Encarta CD-ROM or DVD-ROM discs are not required, except for initial
	membership.
	
	Join Club Encarta
	-----------------
	
	To sign up for your free membership subscription to Encarta Online Deluxe
	Encyclopedia, join Club Encarta:
	
	1. Connect to your Internet service provider (ISP).
	
	2. Start Encarta Reference Library.
	
	3. On the Help menu, click Join Club Encarta.
	
	4. Follow the instructions on the screen to register Encarta Reference Library
	  online and to start your free Encarta Online Deluxe Encyclopedia membership.
	
	Sign In
	-------
	
	To sign in and use the Encarta Online Deluxe Encyclopedia, follow these steps:
	
	1. Browse to the following MSN Web site:
	
	  http://encarta.msn.com (http://encarta.msn.com)
	
	2. Click Deluxe Sign In.
	
	3. If you are prompted, sign in to your Passport account.
	
	NOTE: If your Passport account is configured to sign you in automatically, you
	are not prompted to sign in.
	
	Please use the same Passport account that you used when you joined Club Encarta.
	If you use a different Passport, you receive a message that states there is no
	subscription associated with your Passport. If this occurs, click Passport Sign
	Out, and then sign in with your original Passport (the one that you used when
	joining Club Encarta).
	
	If you have forgotten your password, click the Help link above the "Sign-in name"
	box.
	
	Support for Encarta Online Deluxe Encyclopedia
	----------------------------------------------
	
	To request support for Encarta Online Deluxe Encyclopedia, use the form on the
	Web site. To do this, follow these steps:
	
	1. Browse to the following Web site:
	
	  http://encarta.msn.com (http://encarta.msn.com)
	
	2. Click Help.
	
	3. Click Contact Us.
	
	
	So what happens after the one year access expires?
	--------------------------------------------------
	
	Is there a cost to continue? No, when the free access to Online Deluxe runs out,
	there is no cost; access just expires. Users may purchase next year's Library to
	get more access. For more information on the latest Reference Library product,
	see the following:
	
	  http://encarta.msn.com/products
