## Q306000: Answer to Anniversary Secret Question Date Format Not Validated

{% raw %}

	Article: Q306000
	Product(s): Microsoft Developer Network
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Passport, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Passport accepts input in any format for the answer to the "Anniversary" secret
	question. No error is returned if the answer does not comply with the mm/dd/yy
	format.
	
	MORE INFORMATION
	================
	
	When a user signs up for a Passport, the Secret Answer field can contain any
	string, even when the secret question is "Anniversary (mm/dd/yy)?"
	
	Pasport may enforce the mm/dd/yy format in future releases.
