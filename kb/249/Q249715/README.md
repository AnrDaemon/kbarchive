## Q249715: Soccer 2000: Minimum System Requirements

	Article: Q249715
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbimu msgame
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft International Soccer 2000, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the minimum system requirements needed to run Microsoft
	International Soccer 2000.
	
	MORE INFORMATION
	================
	
	To run International Soccer 2000, you need:
	
	- A Multimedia PC with a Pentium 133 MHz or faster processor (Pentium 200 MHz
	  or faster processor recommended)
	
	- Microsoft Windows 95/98 operating system
	
	- 32 megabytes (MB) of RAM (64 MB of RAM recommended)
	
	- 100 MB of available hard disk space for minimum installation or 500 MB of
	  available hard disk space for maximum installation (50 MB of additional hard
	  disk space required for the DirectX 6.1 installation and Windows swap file)
	
	- A quad-speed or faster CD-ROM drive
	
	- A PCI video adapter with 2 MB or more of video memory (DirectX 6.1-compatible
	  3D video adapter with 4 MB of video RAM or greater recommended)
	
	- A DirectX 6.1-compatible sound card
	
	- Speakers or headphones required for audio
	
	- A Microsoft mouse or compatible pointing device
	
	- Optional: A compatible game pad device (Microsoft Sidewinder game pad
	  recommended)
