## Q193357: City Symbol Used for Country in &quot;Name That Place&quot; Game

	Article: Q193357
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbtool kbui kbimu
	Last Modified: 19-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Virtual Globe 99, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you play the "Name That Place" game in Encarta Virtual Globe 99, the city
	symbol (bullseye) may be used when the available answers are countries.
	
	MORE INFORMATION
	================
	
	This behavior occurs if the country being displayed is too small to be easily
	identified with the yellow highlighted area used to display countries in the
	game.
	
	For information about how to play the "Name That Place" game, please see the Quiz
	topic in Encarta Virtual Globe 99 online Help.
