## Q193001: Encarta Virtual Globe 99: Cannot Disable Sounds in Game

{% raw %}

	Article: Q193001
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbsound kbui kbimu
	Last Modified: 19-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Virtual Globe 99, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click to clear the Play Background Music And Sounds check box on the
	Encarta Virtual Globe 99 Settings screen, sounds continue to be played in the
	Name That Place game.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Encarta Virtual Globe
	99, version 1.0.
	
	MORE INFORMATION
	================
	
	To display the Encarta Virtual Globe 99 Settings screen, click Settings on the
	Options menu.
