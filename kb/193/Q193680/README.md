## Q193680: DOCERR: AGETFILEVERSION() Function Incorrectly Documented

	Article: Q193680
	Product(s): Microsoft FoxPro
	Version(s): WINDOWS:6.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual FoxPro for Windows, version 6.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Visual FoxPro 6.0 Language Reference and help topic for the
	AGETFILEVERSION() function describes the second argument, cFileName, as
	follows:
	
	  Specifies the name file for which information is placed in the array. The
	  class library name you specify is displayed in the File name text box. An
	  error is generated if the class library you specify doesn't exist.
	
	This text is incorrect.
	
	It should say:
	
	  Specifies the name of the file for which information is placed in the array.
