## Q193445: Virtual Globe 99: Cannot Copy or Print Frame from Video Clip

{% raw %}

	Article: Q193445
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbprint kbimu
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Virtual Globe 99, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to copy or print a frame from a video clip in Microsoft Encarta
	Virtual Globe 99, only the caption and copyright information are copied or
	printed.
	
	RESOLUTION
	==========
	
	To work around this issue, use the Print Screen function of Microsoft Windows to
	capture the frame from the video clip.
	
	For information about how to use the Print Screen function of Windows to capture
	the contents of a window or the screen, click Start, click Help, click the Index
	tab, type "print screen" (without the quotation marks), and then click Display.
	
	STATUS
	======
	
	This behavior is by design.
