## Q193289: Encarta 99 Err Msg: Enc99 Caused a Stack Fault in Module...

	Article: Q193289
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kberrmsg kbimukbfaq
	Last Modified: 26-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Encyclopedia 99 
	- Microsoft Encarta Reference Suite 99 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to use the World Music Interactivity in Encarta Encyclopedia
	99, you may receive one of the following error messages:
	
	   - Enc99 caused a stack fault in module Iml32.dll.
	
	   - Enc99 caused a stack fault in module Kernel32.dll.
	
	
	CAUSE
	=====
	
	This behavior can occur if the World Music interactivity attempts to open a file
	that does not exist on the Encarta Encyclopedia CD-ROM or Encarta Reference
	Suite DVD-ROM.
	
	RESOLUTION
	==========
	
	Replacement discs that resolve this problem are available for Encarta
	Encyclopedia 99 and Encarta Reference Suite 99.
	
	To obtain replacement discs for Encarta Encyclopedia 99 or Encarta Reference
	Suite 99, call the Microsoft Order Desk at (800) 360-7561 [and request the
	appropriate DVD-ROM or CD-ROM set for your copy of Encarta]. If you are outside
	the United States, contact the Microsoft subsidiary for your area. To locate
	your subsidiary, see the Microsoft World Wide Offices Web page at:
	
	  http://www.microsoft.com/worldwide/default.htm
	
	Additional query words: multi multi-media media mm worldmusic shockwave
