## Q193541: INFO: Visual Basic 5.0 Product Documentation Print Version

	Article: Q193541
	Product(s): Microsoft Visual Basic for Windows
	Version(s): WINDOWS:5.0
	Operating System(s): 
	Keyword(s): kbGrpDSVB
	Last Modified: 11-JAN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual Basic Learning Edition for Windows, version 5.0 
	- Microsoft Visual Basic Professional Edition for Windows, version 5.0 
	- Microsoft Visual Basic Enterprise Edition for Windows, version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article provides information for ordering the printed version of the Visual
	Basic 5.0 product documentation in the event that the documentation coupon
	included with the product is lost.
	
	MORE INFORMATION
	================
	
	The printed version of the Visual Basic 5.0 product documentation is published
	by Microsoft Press and is available from your favorite bookseller or directly
	from Microsoft Press.
	
	For more information about ordering from Microsoft Press, go to:
	
	http://mspress.microsoft.com/
	
	The following are the printed versions of the Visual Basic 5.0 product
	documentation published by Microsoft Press:
	
	- Microsoft Visual Basic 5.0 ActiveX Controls Reference
	  ISBN# 1-57231-508-3
	  For more information, see:
	  http://mspress.microsoft.com/prod/books/1064.htm
	
	- Microsoft Visual Basic 5.0 Language Reference
	  ISBN# 1-57231-507-5
	  For more information, see:
	  http://mspress.microsoft.com/prod/books/1063.htm
	
	- Microsoft Visual Basic 5.0 Programmer's Guide
	  ISBN# 1-57231-604-7
	  For more information, see:
	  http://mspress.microsoft.com/prod/books/1370.htm
