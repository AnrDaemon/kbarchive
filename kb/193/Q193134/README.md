## Q193134: XCLN: Contact Folders Use Mail Folder Icon in OWA

	Article: Q193134
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 20-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	A public folder created in Outlook that holds contact items has a mail folder
	icon when viewed using the Outlook Web Access (OWA) client. When you attempt to
	access the folder, the following error message appears, and the contacts are not
	available:
	
	  This version of Outlook Web Access only supports your primary contact
	  folder.
	
	STATUS
	======
	
	This behavior is by design.
