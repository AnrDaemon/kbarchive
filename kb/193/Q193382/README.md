## Q193382: Virtual Globe 99: Lunar Map Links to Earth-Related Content

{% raw %}

	Article: Q193382
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbui kbimu
	Last Modified: 12-AUG-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Virtual Globe 99, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you view the lunar surface in Encarta Virtual Globe 99 and click on or near
	the Moon's North or South pole, the program displays a menu that contains links
	to content describing the Earth's poles.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Encarta Virtual Globe
	99, version 1.0.
	
	MORE INFORMATION
	================
	
	Encarta Virtual Globe 99 does not contain associated content specific to the
	Moon. The map of the lunar surface displayed in Encarta Virtual Globe does not
	link to any articles or other content.
