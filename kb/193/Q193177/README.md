## Q193177: WD97: AutoNum Field Not Updated Correctly Inside Table Cell

	Article: Q193177
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you use the AUTONUM field in Word to number several paragraphs inside a
	single cell in a table, each paragraph displays the same number.
	
	CAUSE
	=====
	
	The AUTONUM field is used to number paragraphs, incrementing each new paragraph
	in sequence automatically. Word treats each row in a table as a separate
	paragraph.
	
	RESOLUTION
	==========
	
	To number several paragraphs inside a single cell in a table, apply paragraph
	numbering.
	
	
	Additional query words: para numbering reverts
