## Q222970: List of Bugs Fixed in Terminal Server Edition

	Article: Q222970
	Product(s): Microsoft Windows NT
	Version(s): 4.0 SP4
	Operating System(s): 
	Keyword(s): kbOSWinNT400sp1kbfixlist
	Last Modified: 08-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 SP4, Terminal Server Edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article is a list of the article numbers for bugs that were fixed in the
	latest Windows NT Server 4.0, Terminal Server Edition Service Pack. Service
	packs are cumulative. This means that the bugs fixed in a service pack are also
	fixed in subsequent service packs.
	
	For information on obtaining the latest service pack for Windows NT 4.0 and
	Terminal Server Edition, please see the following article in the Microsoft
	Knowledge Base:
	
	  Q152734 How to Obtain the Latest Windows NT 4.0 Service Pack
	
	
	MORE INFORMATION
	================
	
	Service Pack 4
	--------------
	
	Service Pack 4 for Windows NT Server 4.0, Terminal Server Edition includes the
	bugs fixed in Windows NT 4.0 Service Pack 4. For additional information, please
	see the following article(s) in the Microsoft Knowledge Base:
	
	  Q150734 List of Bugs Fixed in Windows NT 4.0 and Terminal Server Edition
	  Service Pack4 (Part 1)
	
	  Q194834 List of Bugs Fixed in Windows NT 4.0 and Terminal Server Edition
	  Service Pack 4 (Part 2)
	
	  Q224793 List of Bugs Fixed in Windows NT 4.0 and Terminal Server Edition
	  Service Pack 4 (Part 3)
