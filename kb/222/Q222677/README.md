## Q222677: SMS: SMS 2.0 from Feb. Select Kit Prompts for Product ID Number

	Article: Q222677
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:2.0
	Operating System(s): 
	Keyword(s): kbsms200
	Last Modified: 19-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Select February 1999 Update Kit includes a version of Systems Management
	Server 2.0 that prompts you for a product identification number (PID) during
	installation.
	
	WORKAROUND
	==========
	
	To work around this problem, use the following CD key code to complete the
	installation of Systems Management Server 2.0:
	
	  32345-270-99
	
	
	Additional query words: prodsms cant cannot install set up setup
