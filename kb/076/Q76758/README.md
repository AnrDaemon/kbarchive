## Q76758: Free Disk Space Required for Windows Resource Kit

	Article: Q76758
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	To install the entire set of utilities that comes with the Windows Resource Kit,
	you need the original disks (two 5.25-inch disks or three 3.5-inch disks) and
	7.5 MB of free space on your hard disk.
	
	MORE INFORMATION
	================
	
	After installation, you can recover 1.2 megabytes of disk space by deleting the
	temporary directory, but you need the full 7.5 MB to successfully complete
	installation.
	
	NOTE: If you do not have enough disk space, and you still want to install some of
	the utilities, please refer to the section titled "Installing Individual
	Utilities" in the "Windows Resource Kit" manual.
