## Q76815: No Multimedia Features in Real Mode with Multimedia Windows

{% raw %}

	Article: Q76815
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 23-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows with Multimedia Extensions, version 1.0 
	-------------------------------------------------------------------------------
	
	Multimedia features only work in standard or 386 enhanced mode. If
	Windows with Multimedia Extensions is run in real mode, animations,
	audio, and MIDI (musical instrument digital interface) features do not
	operate.
	
	This information does not apply to later versions of Windows.
	
	Additional query words: MMWIN 1.00 3.00
