## Q76441: Flight Simulator: Keyboard Input Is Not Accepted During Demo

	Article: Q76441
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	During Demo Playback, Flight Simulator for the Macintosh version 4.0 does not
	accept any keyboard input other than COMMAND+PERIOD, which cancels Demo
	Playback.
	
	You cannot change views during Demo Playback using the keyboard. To change views
	during Demo Playback, choose View Tools from the View menu, and choose a
	different view.
	
	Additional query words: 4.00
