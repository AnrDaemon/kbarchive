## Q76750: MS-DOS 5.0 on Zeos Computer with Mylex BIOS Earlier than 6.04

	Article: Q76750
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Zeos computers with Mylex BIOS versions earlier than 6.04 may experience
	problems running Microsoft MS-DOS 5.0. A BIOS upgrade is available free from
	Zeos for customers experiencing problems running MS-DOS 5.0.
