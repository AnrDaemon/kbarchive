## Q76760: Productivity Pack: &#91;COLORS&#93; DimmedText= Does Not Work

	Article: Q76760
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Productivity Pack incorrectly states that you can change the color of dimmed
	text by entering the line
	
	     DimmedText=red green blue
	
	in the [COLORS] section of the WIN.INI file.
	
	MORE INFORMATION
	================
	
	To locate the instructions in the Productivity Pack, select:
	
	1. Working Smarter
	
	2. A Personal Touch
	
	3. Adding color to buttons and dimmed text
	
	The suggested line "DimmedText=red green blue" is not valid.
	
	However, the instruction
	
	  GrayText=red green blue
	
	will produce the desired change in color.
	
	Additional query words: wpp 1.00 Windows Productivity Pack 1.0 3.00 3.00a
