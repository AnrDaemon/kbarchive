## Q76094: CONN: Methods to Toggle the Quit Dialog Box in MSGATE.EXE

	Article: Q76094
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Connection Gateway, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	There are three different methods to force the Microsoft Mail Connection to
	display the Quit dialog box:
	
	- Press the ESC key.
	
	-or-
	
	- Press the CTRL+] keys.
	
	-or-
	
	- Press the CTRL+ALT+] keys.
	
	Any one of these methods allows you to exit the Mail Connection.
