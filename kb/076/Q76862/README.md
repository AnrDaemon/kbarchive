## Q76862: JetWare SuperSet+ 425-in-1 Causes UAEs when Printing

{% raw %}

	Article: Q76862
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Printing with version 1.02 of the JetWare SuperSet+ 425-in-1 printer drivers
	causes Unrecoverable Application Errors (UAEs) with Windows-based software.
	Computer Peripherals, Inc. (CPI) technical support has confirmed this to be a
	problem in the driver.
	
	MORE INFORMATION
	================
	
	For the latest driver to correct printing problems from Windows 3.0, contact CPI
	technical support.
	
	JetWare SuperSet+ 425-in-1 printer drivers are manufactured by a vendor
	independent of Microsoft; we make no warranty, implied or otherwise, regarding
	these product's performance or reliability.
