## Q76065: Summa Graphics Pointing Devices and MS-DOS 5.0

{% raw %}

	Article: Q76065
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 03-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Summa Graphics recommends that version 7.02 of their utilities disk be used with
	Microsoft MS-DOS 5.0.
	
	MORE INFORMATION
	================
	
	Attempting to use earlier versions of the Summa Graphics utilities or drivers
	may result in system lockups or hangs while loading in the MS-DOS 5.0
	environment.
	
	The driver names for the Digitizing Tablets are TABLET.COM and TABLET.SYS, and
	are found in the TABLETBL.DVR directory of the utilities disk. The Summa Mouse
	is no longer manufactured and there is no updated driver available.
	
	To receive an upgrade to these utilities and drivers for the Summa Graphics
	Digitizing Tablet, contact Summa Graphics. Ask for the latest version of the
	utilities disk (version 7.02, part no. 64-5010-010).
	
	The product included here is manufactured by a vendor independent of Microsoft;
	we make no warranty, implied or otherwise, regarding this product's performance
	or reliability.
