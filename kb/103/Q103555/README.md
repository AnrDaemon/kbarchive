## Q103555: Defrag Treats Lost Clusters as Unmovable Blocks

{% raw %}

	Article: Q103555
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 03-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft Defragmenter (Defrag) does not detect lost clusters during the
	defragmentation process. Defrag marks lost clusters as unmovable blocks ("X" on
	the drive map screen display) and then proceeds to move the data around them.
	
	Additional query words: 6.22 6.00 6.20 allocation unit non-movable nonmovable
