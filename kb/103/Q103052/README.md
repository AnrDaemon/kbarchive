## Q103052: Dysfunctional Browsing on Wide Area Network (WAN)

{% raw %}

	Article: Q103052
	Product(s): Microsoft Windows NT
	Version(s): 3.1
	Operating System(s): 
	Keyword(s): kbnetwork
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Advanced Server, version 3.1 
	-------------------------------------------------------------------------------
	
	
	Wide are network (WAN) browsing may fail when there have been two
	domain controllers present for some amount of time. This usually
	occurs when promoting a server to a domain controller; there is
	usually a short amount of time when there are two domain controllers
	in the domain.
