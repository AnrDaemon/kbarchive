## Q103904: PC WRmt: Resolving Mail to External Groups

	Article: Q103904
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0,3.0a,3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, versions 3.0, 3.0a, 3.2 
	- Microsoft Mail Remote for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you send a mail message from version 3.2 of Microsoft Mail Remote for Windows
	to a version 3.0 or 3.2 Microsoft Mail for PC Networks administrator-defined
	postoffice group and that message includes an external user, Mail Remote fails
	to deliver the message to the external user.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in versions 3.0, 3.0a, and 3.2 of
	Microsoft Mail for PC Networks. We are researching this problem and will post
	new information here in the Microsoft Knowledge Base as it becomes available.
