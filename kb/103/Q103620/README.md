## Q103620: Changes to Windows NT Driver Library: Printers

	Article: Q103620
	Product(s): Microsoft Windows NT
	Version(s): WinNT:3.5
	Operating System(s): 
	Keyword(s): kb3rdparty kbhw kbHardware
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.5 
	- Microsoft Windows NT Server version 3.5 
	-------------------------------------------------------------------------------
	
	- THESE DRIVERS ARE TESTED AND CERTIFIED WITH WINDOWS NT VERSION 3.5 ONLY -
	
	SUMMARY
	=======
	
	Below is a summary of the changes to the printer drivers in the Microsoft
	Windows NT Driver Library (WNTDL). The WNTDL offers support for several
	printers, displays, and network adapter cards not provided with Windows NT.
	
	For additional information about WNTDL, please click the article number below to
	view the article in the Microsoft Knowledge Base:
	
	  Q100209 Windows NT 3.5 Driver Library
	
	NOTE: Wntdl.txt contains a complete listing of the WNTDL's contents and
	instructions for obtaining the WNTDL.
	
	MORE INFORMATION
	================
	
	
	Printer: HP 4V/4MV 1.0 ALPHA (HP4V_AXP.EXE) S15014
	--------------------------------------------------
	
	11/29/94 - HP4V_AXP.EXE has been added to the WNTDL. This driver supports the
	Hewlett-Packard 4V and 4MV LaserJet printers on Alpha-based computers.
	
	Printer: HP 4V/4MV 1.0 MIPS (HP4VMIPS.EXE) S15015
	-------------------------------------------------
	
	11/29/94 - HP4VMIPS.EXE has been added to the WNTDL. This driver supports the
	Hewlett-Packard (HP) 4V and 4MV LaserJet printers on MIPS-based computers.
	
	Printer: HP 4V/4MV 1.0 X86 (HP4VI386.EXE) S15016
	------------------------------------------------
	
	11/29/94 - HP4VI386.EXE has been added to the WNTDL. This driver supports the
	Hewlett-Packard 4V and 4MV LaserJet printers on X86-based computers.
