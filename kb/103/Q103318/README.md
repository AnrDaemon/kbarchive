## Q103318: MoveLst.exe Moves Item in List Box Using Drag and Drop

	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	MoveLst.exe demonstrates how to reorder the items in a list box using drag and
	drop. The user clicks an item, drags it to the new position, and drops the item.
	The list box will scroll if necessary.
	
	MoveLst.exe also shows how to implement a dialog box as a main window using a
	private dialog class.
