## Q126242: PC Win: New Mail Interval Does Not Accept White Space

{% raw %}

	Article: Q126242
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	From the Mail menu, if you choose Options, the Check for New Mail option will be
	displayed. If you select OK, the following message may be displayed even if the
	number visible in the Check for New Mail option is between 1 and 9999:
	
	  Stop The new mail interval requires a number between 1 and 9999.
	
	CAUSE
	=====
	
	The Check For New Mail entry cannot accept white space.
	
	RESOLUTION
	==========
	
	Clear and re-enter only the number, without white space.
	
	
	Additional query words: 3.20 whitespace
