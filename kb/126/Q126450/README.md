## Q126450: Lemmings 3: Error 55.53: Unable to Calibrate Timer

{% raw %}

	Article: Q126450
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): win95 appscomp kbAppCompatibility
	Last Modified: 28-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to start the MS-DOS-based game Lemmings 3, you receive the
	following error message:
	
	  error 55.53: unable to calibrate timer
	
	When you try to run the L3.BAT file in an MS-DOS window, your computer stops
	responding (hangs) and you must restart the computer.
	
	CAUSE
	=====
	
	Lemmings 3 cannot start or run in a windowed MS-DOS environment. Lemmings 3
	requires a full-screen MS-DOS environment.
	
	RESOLUTION
	==========
	
	To correct this problem, set the Screen Usage settings for Lemmings 3 as
	follows:
	
	1. Use the right mouse button to click the Lemmings 3 icon and then click
	  Properties on the menu that appears.
	
	2. Click the Screen tab.
	
	3. Under Usage, click the Full-Screen option button.
	
	4. Click the Apply button and then click the OK button.
