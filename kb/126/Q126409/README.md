## Q126409: Automap Streets: No County Boundaries in Streets

	Article: Q126409
	Product(s): Microsoft Automap
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-AUG-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Automap Streets for Windows, version 1.01 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When Microsoft Automap Streets version 1.01 is installed as a standalone
	application, or installed with Automap Road Atlas versions 3.0, 3.02, 3.04, and
	3.05 for Windows on the same computer system, the County Boundaries overlay will
	not appear.
	
	The Overlay option on the toolbar will not work and the Data drop down menu
	Overlay selection will not be available.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Automap Streets version 1.01. We
	are researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
