	SYMPTOMS
	========
	
	When you try to play a compressed wave audio file, the Roland Rap-10 sound card
	may cause the following error message:
	
	  PROGMAN.EXE caused a General Protection Fault in MSACM.DRV
	
	RESOLUTION
	==========
	
	The problem occurs if the Roland Toolworks software version 3.5c is installed.
	To correct this problem, contact Roland to obtain Toolworks version 3.7 or
	later.
