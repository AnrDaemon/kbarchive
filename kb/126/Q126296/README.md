## Q126296: OEM ATI Drivers Available for Windows NT 3.5

	Article: Q126296
	Product(s): Microsoft Windows NT
	Version(s): 3.50
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.5 
	- Microsoft Windows NT Server version 3.5 
	-------------------------------------------------------------------------------
	
	The Windows NT 3.5 video drivers, released by ATI Technologies for their
	Mach8, Mach32, and Mach64 version 1.41 video cards, can be found in the
	following locations:
	
	- ATI BBS: (905) 764-9404
	
	- CompuServe: GO ATITECH
	
	- Internet: www.atitech.ca
	
	The driver name is 64NT35.ZIP. The ATI driver runs on the following
	products:
	
	- GRAPHICS ULTRA PRO (1MB, 2MB)(Mach32)
	
	- GRAPHICS ULTRA PLUS (1MB, 2MB)(Mach32)
	
	- GRAPHICS ULTRA (1MB)(Mach32)
	
	- 8514 ULTRA (1MB)(Mach8)
	
	- GRAPHICS VANTAGE (1MB)(Mach8) and other OEM products identifying the Mach8
	  (ATI38800) or Mach32 (ATI68800) graphics accelerators.
	
	- GRAPHICS PRO TURBO (2MB, 4MB) (PCI, VLB, ISA) (Mach64)
	
	- GRAPHICS XPRESSION (2MB) (PCI, VLB, ISA) (Mach64)
	
	You can ask questions of, and reports problems to, ATI Driver Development
	using the ATI forum on CompuServe.
	
	The third-party products discussed here are manufactured by vendors
	independent of Microsoft; we make no warranty, implied or otherwise,
	regarding these products' performance or reliability.
