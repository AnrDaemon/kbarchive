## Q126059: Automap Err Msg: Cannot Find VER.DLL

{% raw %}

	Article: Q126059
	Product(s): Microsoft Automap
	Version(s): 1.06,1.1,3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 25-OCT-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Automap Road Atlas for Windows, versions 1.06, 1.1, 3.0 
	- Microsoft Windows 3.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you open Automap Road Atlas for Windows in Microsoft Windows version 3.0,
	you may receive the error message:
	
	  Cannot Find VER.DLL
	
	RESOLUTION
	==========
	
	Use one of the following methods to correct the problem.
	
	Method 1
	--------
	
	Check the hard drive for the file VER.DLL. If the file is present, move the file
	to the SYSTEM subdirectory of your WINDOWS directory and restart Windows.
	
	For more information about how to perform this task in Windows, see your Windows
	printed documentation or online Help.
	
	Method 2
	--------
	
	Upgrade to Road Atlas 3.04 for Windows.
	
	Method 3
	--------
	
	Upgrade to a later Microsoft Windows operating system.
	
	MORE INFORMATION
	================
	
	The file VER.DLL is shipped with Microsoft Windows version 3.1.
	
	To obtain new or updated Microsoft products, call Microsoft Information and
	Customer Service at (800) 426-9400.
