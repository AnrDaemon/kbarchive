## Q126299: Erratic Keyboard Behavior with Comptex 386SX-25

	Article: Q126299
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run Microsoft Windows version 3.1 or 3.11 in 386 enhanced mode on a
	Comptex 386SX-25 computer, the keyboard behaves erratically, producing incorrect
	characters.
	
	CAUSE
	=====
	
	This problem can occur on Comptex 386SX-25 computers with an AMI BIOS dated
	04/25/91 and a SCAT revision 3.0 chip set.
	
	WORKAROUND
	==========
	
	To work around this problem, disable the Turbo Switch function in the computer's
	advanced CMOS Setup. For more information about the problem, contact the
	computer's manufacturer.
	
	Additional query words: 3.10 3.11 erratic
