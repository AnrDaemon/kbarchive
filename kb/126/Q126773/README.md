## Q126773: Writer/Artist: Using Manual Feed on a Panasonic KX-P2123

	Article: Q126773
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:1.0,1.1,1.1a,3.x
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Creative Writer for Windows, versions 1.0, 1.1, 1.1a 
	- Microsoft Fine Artist for Windows, versions 1.0, 1.1, 1.1a 
	- the operating system: Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you print from Creative Writer or Fine Artist to a Panasonic KX-P2123 or
	similar printer, using Manual Feed, the computer does not alert you when it is
	time to insert paper into the printer.
	
	RESOLUTION
	==========
	
	To resolve this problem, upgrade to Windows 95, or if you're using Windows 3.x,
	use the following steps:
	
	1. When you are running Creative Writer, click the Print Toaster, then click the
	  left-hand Print Toaster, then click Next.
	
	2. Press "CTRL+ESC" (without the quotation marks) on the keyboard, which brings
	  up your Windows Task List.
	
	3. Click Print Manager, then click Switch To.
	
	4. Feed the paper into the printer and click the OK button.
	
	After following these steps, your document should print successfully.
	
	MORE INFORMATION
	================
	
	Microsoft has confirmed this to be a problem in Creative Writer and Fine Artist
	for Windows, versions 1.0, 1.1, and 1.1a. We are researching this problem and
	will post new information here in the Microsoft Knowledge Base as it becomes
	available.
