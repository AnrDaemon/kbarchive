## Q126823: Writer/Artist: Setup on Novell Personal NetWare

	Article: Q126823
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:1.1,1.1a,3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Creative Writer for Windows, versions 1.1, 1.1a 
	- Microsoft Fine Artist for Windows, versions 1.1, 1.1a 
	- the operating system: Microsoft Windows 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you set up Creative Writer or Fine Artist on a Novell Personal NetWare
	network, the share name for the CD-ROM drive must match the volume name of the
	CD.
	
	  For Creative Writer version 1.1, the CD-ROM share name must be MSCR110_01.
	
	  For Fine Artist version 1.1, the CD-ROM share name must be MSFA110_01.
	
	  For Creative Writer/Fine Artist version 1.1a, the CD-ROM share name must be
	  CWFA110_01.
	
	MORE INFORMATION
	================
	
	This may apply to other CD-ROM programs used on Novell Personal NetWare, if they
	search for a volume name on the CD-ROM. This may also apply to other networks in
	addition to Novell NetWare. Windows for Workgroups does not display this
	behavior.
