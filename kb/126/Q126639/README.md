## Q126639: Terminate Viewer Option in Remote Control Agent Does Not Work

	Article: Q126639
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.0
	Operating System(s): 
	Keyword(s): kbnetwork smsgeneral kbArtTypeINF
	Last Modified: 27-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you take remote control of a client computer from Help Desk and minimize
	the Quick Windows Viewer in Systems Management Server version 1.0, the Terminate
	Viewer option in the Remote Control Agent on the client computer does not work.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.0. This problem was corrected in Systems Management Server version
	1.1.
