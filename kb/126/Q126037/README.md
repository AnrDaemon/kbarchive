## Q126037: Problems with Pivot Functions on Radius Pivot Monitors

	Article: Q126037
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The pivot capabilities of the Radius Full Page Pivot monitor do not function
	correctly in Microsoft Windows 95.
	
	MORE INFORMATION
	================
	
	The Radius drivers for the Full Page Pivot monitor are for Windows version 3.x
	and are not fully compatible with Windows 95. Radius currently has no plans to
	update these drivers.
	
	The Radius Full Page Pivot monitor is manufactured by Radius Inc., a vendor
	independent of Microsoft; we make no warranty, implied or otherwise, regarding
	this product's performance or reliability.
