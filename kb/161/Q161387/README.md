## Q161387: Golf 3.0: Installing Links LS Courses

	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Golf for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When your are installing a Access Software Links LS course, Microsoft Golf 3.0
	is not found.
	
	RESOLUTION
	==========
	
	To install the Links LS course(s) for use with Golf 3.0, create a text file
	named Msgolf3.exe in the Msgolf folder.
	
	1. Create a new file in Notepad or Wordpad.
	
	  NOTE: This file does need to contain any information.
	
	2. Rename the file you created in Step 1 to Msgolf3.exe.
	
	3. Copy or Move file to the MSGOLF directory.
	
	  NOTE: For more instructions about how copy, move, or rename a file, see your
	  Windows printed documentation or online Help.
	
	4. Install the Links LS course.
