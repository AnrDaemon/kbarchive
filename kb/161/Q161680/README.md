## Q161680: Kids Plus!: The Background Displays Different Color

{% raw %}

	Article: Q161680
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Plus! for Kids, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Paint It! background color indicator may display a different color than the
	color of the background.
	
	MORE INFORMATION
	================
	
	In the Color Box there are two squares that are larger than the other squares in
	this Color Box. The large square that is on top of the other large square
	displays the color of the foreground. The large square on the bottom displays
	the color of the background. The background square may display a different color
	than the actual color of the background of the drawing.
	
	Steps to Reproduce Problem
	--------------------------
	
	1. Open the Paint It! program.
	
	2. Click View and make sure there is a check next to the Color Box option. If
	  there is not a check, click Color Box.
	
	3. With the right mouse button, click any of the smaller squares in the Color
	  Box to set the background color.
	
	4. On the File menu, click New.
	
	You will notice that the background color square and the actual background color
	are different.
	
	Additional query words: strange same odd palette
