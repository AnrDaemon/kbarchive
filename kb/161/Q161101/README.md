## Q161101: Flight Simulator 95 Error: The File Dciman32.dll Cannot Start

{% raw %}

	Article: Q161101
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for Windows 95, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to run Flight Simulator for Windows 95, you may receive the
	following error message:
	
	  The DCIMAN32.DLL file cannot start. Check the file to determine the problem.
	
	RESOLUTION
	==========
	
	This error can occur if the Dciman.dll file is damaged or missing from the
	Windows\System folder. To resolve this issue, replace the file on the hard drive
	with the original file extracted from the Windows 95/98 compact disc or floppy
	disks.
	
	NOTE: Remove or rename any existing copies of Dciman.dll from the Windows\System
	folder. For more information about how to accomplish this task in Windows, see
	your Windows printed documentation or online Help.
	
	For information about how to extract the Dciman.dll file from your Windows CD-ROM
	or disk into the \Windows\System folder, please see the following article in the
	Microsoft Knowledge Base:
	
	  Q129605 How to Extract Original Compressed Windows Files
	
	MORE INFORMATION
	================
	
	If the error still occurs, follow the steps above to extract Dciman32.dll. If
	you still receive the error message, you can correct the problem by reinstalling
	Windows. However, this should be used as a last resort. Reinstalling Windows
	resets many of your custom settings to the Windows default settings.
