## Q161081: Windows NT Imaging Scanning Options Unavailable

	Article: Q161081
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Windows NT Imaging, the "Scan New," "Select Scanner," and "Scan Preferences"
	options are unavailable.
	
	MORE INFORMATION
	================
	
	The Windows NT Imaging help file states that the scan options will work with a
	Twain-compatible driver installed. This is a documentation error; even with a
	Twain-compatible driver installed, the scan options above will not be available
	in the Windows NT Imaging application.
	
	RESOLUTION
	==========
	
	Scanning is not supported by the Windows NT Imaging application at this time.
