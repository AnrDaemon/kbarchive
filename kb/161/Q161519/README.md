## Q161519: WD97: Table Column Width Changes When Opened As HTML

	Article: Q161519
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Table column widths may resize based on the amount of text within the columns
	when the document containing a table is opened as HTML.
	
	
	CAUSE
	=====
	
	The table size information was incomplete when the document was saved as HTML.
	
	WORKAROUND
	==========
	
	Define the column widths in the table using the following procedure:
	
	1. Select a column in the table.
	
	2. On the Table menu, click Cell Properties.
	
	3. From the Cell Properties dialog, change Width to the desired setting for the
	  selected table column.
	
	4. Repeat steps 1-3 for each table column.
	
	5. Save the document as HTML.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the products listed at the
	beginning of this article.
