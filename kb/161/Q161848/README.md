## Q161848: FP: Custom Color Palette Reset After Editor Session

{% raw %}

	Article: Q161848
	Product(s): Word Front Page
	Version(s): windows:1.1,97
	Operating System(s): 
	Keyword(s): kbgraphic kbusage kbdta
	Last Modified: 04-OCT-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft FrontPage 97 for Windows with Bonus Pack 
	- Microsoft FrontPage for Windows, versions 1.0a, 1.1 
	-------------------------------------------------------------------------------
	
	For a Microsoft FrontPage 98 version of this article, see Q194267.
	
	SYMPTOMS
	========
	
	When you add a custom color in the Color dialog box in FrontPage Editor, the
	color is not available the next time you open a page in FrontPage Editor. (To
	locate the Color dialog box, click the Text Color button on the Format toolbar.)
	
	CAUSE
	=====
	
	The custom color is not retained from one FrontPage Editor session to the next.
	
	RESOLUTION
	==========
	
	You can apply the color within the same FrontPage Editor session by following
	these steps:
	
	1. In FrontPage Editor, open a page that uses the custom color.
	
	2. Select the item on the page that uses the custom color.
	
	3. Click the Text Color button on the Format toolbar.
	
	4. In the Color dialog box, click Define Custom Colors.
	
	5. Click Add to Custom Colors.
	
	6. Click OK.
	
	To use the custom color, follow these steps:
	
	1. Select the text you want to change.
	
	2. Click the Text Color button on the Format toolbar.
	
	3. In the Color dialog box, under Custom Colors, click the color you want to
	  use, and then click OK.
