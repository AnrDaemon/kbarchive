## Q161267: Kids Plus!: Screen Savers Settings Require System Agent

	Article: Q161267
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Plus! for Kids, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You may receive the following error message when trying to change the settings
	of a Microsoft Plus! For Kids screen saver:
	
	  Your must have the System Agent loaded in order for the mouse hot corners to
	  be active.
	
	MORE INFORMATION
	================
	
	You may receive this message if you try to change some of the settings in the
	Baseball, Bugs, Jungle, RE-man, Snowboarding, Space, and Underwater themes. This
	message will be displayed if the System Agent that ships with the Microsoft
	Plus! Pack is not installed. The System Agent will need to be installed to set
	the custom features for these screen saver settings.
