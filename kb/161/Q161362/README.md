## Q161362: Windows NT 4.0 Driver for SideWinder Now Available

	Article: Q161362
	Product(s): Microsoft Windows NT
	Version(s): 1.0,4.0 SP2
	Operating System(s): 
	Keyword(s): kbhw kbsetup kbHardware
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 SP2 
	- Microsoft Windows NT Server version 4.0 SP2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The CD-ROM version of Windows NT 4.0 Service Pack 2 contains an updated driver
	for the Microsoft SideWinder 3D Pro joystick. This driver is available in the
	Drvlib\Joystick\Sidewind folder on the CD-ROM and is an update to the driver
	included with Windows NT 4.0.
	
	Note that this driver is not for use with other SideWinder joysticks such as the
	Force Feedback.
	
	MORE INFORMATION
	================
	
	Perform the following steps to install the SideWinder driver:
	
	1. Log on as Administrator.
	
	2. Click the Start button, point to Settings, click Control Panel, and then
	  double-click Multimedia.
	
	3. Click the Devices tab and then click Add.
	
	4. Click Unlisted or Updated Driver and then click OK.
	
	5. Insert the path to install the SideWinder driver. For example, for
	  Intel-based computers, the path is:
	
	  <Drive Letter>:\Drvlib\Joystick\Sidewind\I386
	
	6. Click OK, click Microsoft SideWinder 3D Pro Joystick, and then click OK.
	
	7. Click OK to close the Microsoft SideWinder Joystick Configuration dialog box.
	
	8. Restart your computer.
	
	Additional query words: prodnt gamepad alpha i386 ppc
