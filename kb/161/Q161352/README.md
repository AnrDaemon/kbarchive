## Q161352: Problems Using the SideWinder Joystick with Games on NT 4.0

	Article: Q161352
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0; winnt:4.0
	Operating System(s): 
	Keyword(s): kbenv kbhw kbimu msgame kbHardware
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Deadly Tide for Windows 1.0 
	- Microsoft NBA Full Court Press for Windows, version 1.0 
	- Microsoft Hellbender for Windows, version 1.0 
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft GEX, version 1.0 
	- Microsoft Soccer, version 1.0 
	- Microsoft Monster Truck Madness, version 1.0 
	- Microsoft Flight Simulator for Windows 95, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to play one of the games listed at the beginning of this
	article with a Microsoft SideWinder 3D Pro joystick on a Microsoft Windows NT
	4.0-based computer, the joystick may behave erratically.
	
	RESOLUTION
	==========
	
	To resolve this issue, obtain and install the latest version of the Windows NT
	4.0 driver for the SideWinder 3D Pro. To do this, download and install Windows
	NT 4.0 Service Pack 5 or later from the following Microsoft Web site:
	
	  http://www.microsoft.com/ntworkstation/downloads/
	
	Additional query words: 1.00 1.00b 4.00 NT SW3D sp5 sp4 sp3 sp2 sp flightsim mtm erratic sidewndr.sys direction joy stick
