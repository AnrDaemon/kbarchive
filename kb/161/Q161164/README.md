## Q161164: Duplex Setting May Not Change When Target Printer Changes

	Article: Q161164
	Product(s): Microsoft Windows NT
	Version(s): WinNT:4.0
	Operating System(s): 
	Keyword(s): kbprint
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The duplexing option is not set correctly when two printers are printing to the
	same print device with different duplex options.
	
	
	WORKAROUND
	==========
	
	Create only one printer for computer1. Before printing, if you want to select
	single-sided printing, you can click Properties and change the print setting to
	simplex from within the application.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 4.0. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
