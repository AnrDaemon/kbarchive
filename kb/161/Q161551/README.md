## Q161551: WD97: AutoSummarize Doesn't Find Some Text to Summarize

	Article: Q161551
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbualink97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Word does not summarize all the text in your document or you receive the
	following error message:
	
	  Word cannot find any text to summarize.
	
	CAUSE
	=====
	
	The AutoSummarize feature does not summarize text if the text is:
	
	- formatted in a language other than English.
	
	- formatted as 'no proofing'.
	
	- inside a text box.
	
	- inside a frame.
	
	- inside a table.
	
	For more information about creating summaries automatically, click the Office
	Assistant, type "AutoSummarize" (without the quotation marks), click Search, and
	then click to view "Automatically summarize a document."
	
	NOTE: If the Assistant is hidden, click the Office Assistant button on the
	Standard toolbar. If Microsoft Help is not installed on your computer, please
	see the following article in the Microsoft Knowledge Base:
	
	  Q120802 Office: How to Add/Remove a Single Office Program or Component
