## Q161168: Encarta 97: Removing Program Without Losing Yearbook Updates

	Article: Q161168
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 97 Encyclopedia for Windows 
	- Microsoft Encarta Encyclopedia 97 Deluxe for Windows 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you remove or uninstall Encarta Encyclopedia from your hard drive, you can
	keep the Yearbook and Web Link Updates by clicking Yes to the following message
	during the Uninstall program:
	
	  Do you want to keep Yearbook and Web Link monthly update files? All Encarta
	  97 Encyclopedia Yearbook and World Wide Web links that you have downloaded
	  will be removed if you click No. If you do not remove these files, they will
	  not harm your system, and you can reuse them if you reinstall Encarta 97
	  Encyclopedia later. If you are not sure what to do, clicking Yes is
	  recommended.
	
	MORE INFORMATION
	================
	
	If you click Yes, you will not have to download any previously downloaded
	Encarta Yearbook Updates again. The Updates will be available for you when you
	reinstall Encarta.
