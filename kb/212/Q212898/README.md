## Q212898: SMS: Advertisement Start Time Is 1 Hour Earlier w/ IntraNetware

	Article: Q212898
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:2.0,4.0 SP3,4.0 SP4
	Operating System(s): 
	Keyword(s): kbOSWinNT400sp3bug kbOSWinNT400sp4bug kbsms200 kbsms200bug
	Last Modified: 18-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	On Systems Management Server site server computers that are running the Japanese
	language version of Windows NT 4.0 with Service Pack 3 or Service Pack 4 and
	Novell IntranetWare installed, advertisement start time is set one hour earlier
	in a daylight saving time (DST) time zone with DST enabled.
	
	
	STATUS
	======
	
	This problem only occurs with Systems Management Server running on the Japanese
	version of Windows NT version 4.0 Service Pack 3 or Service Pack 4 with
	IntranetWare client installed and a daylight saving time zone with DST enabled.
