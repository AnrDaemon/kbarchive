## Q212790: SMS Installer Does Not Calculate Required Disk Space

{% raw %}

	Article: Q212790
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:2.0
	Operating System(s): 
	Keyword(s): kbsms200 kbsms200bug
	Last Modified: 22-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you install program components with the Systems Management Server
	Installer, the Disk Space Required value does not accurately reflect the size of
	the total required disk space.
	
	CAUSE
	=====
	
	The Disk Space Required value does not include the required space for
	non-optional files.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in System Management Server version
	2.0.
