## Q120530: How to Obtain Printer Drivers from Hewlett-Packard

{% raw %}

	Article: Q120530
	Product(s): Microsoft Windows Printing Issues
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	This article lists the methods Hewlett-Packard (HP) uses to distribute printer
	drivers.
	
	MORE INFORMATION
	================
	
	Hewlett-Packard distributes printer drivers by the following means:
	
	Telephone
	---------
	
	Call (970) 339-7009. Be sure you know what driver and disk size you want.
	There may be a charge for some drivers.
	
	CompuServe
	----------
	
	- Type "go hp" (without the quotation marks) to enter the HP forum. Drivers can
	  be found in the Peripherals library.
	
	-or-
	
	- Type "go!hpper" (without the quotation marks) to enter the HP Peripherals
	  forum directly.
	
	Electronic Bulletin Board System (BBS)
	--------------------------------------
	
	(208) 344-1691
	
	9600 baud, no parity, 8 data bits, 1 stop bit
	
	Internet FTP
	------------
	
	IP Address: 192.6.71.2
	Logon with User ID "anonymous" (without the quotation marks) and your email
	address as the password.
	
	
	Additional query words: winword macword win31 laserjet deskjet drv hppcl5e deskjetc deskjetp hplj hpdj
