## Q120211: Cannot Specify Save Directory for Flight Plans

	Article: Q120211
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-JUN-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Space Simulator for MS-DOS, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When saving Flight Plans in Microsoft Space Simulator 1.0, you cannot specify
	the directory to which you save the Flight Plans. All Flight Plans must be saved
	to the directory in which Space Simulator was installed.
	
	MORE INFORMATION
	================
	
	When saving videos, screen shots, and situations, you do have the capability to
	specify a directory other than the default directory.
	
	Additional query words: 1.00 spacesim ssim dos ss1
