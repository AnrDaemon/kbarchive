## Q120742: Qualcomm Eudora v1.4 Displays Only 1 Mail Message in Window

{% raw %}

	Article: Q120742
	Product(s): Microsoft Windows NT
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	3.50
	
	WINDOWS
	
	kb3rdparty
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.5 
	- Microsoft Windows NT Server version 3.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The shareware version of Qualcomm's Eudora v1.4 has a bug that causes it to
	display only one mail message in a window.
	
	MORE INFORMATION
	================
	
	Eudora v1.4.2 contains the fix for this problem. It can be found at
	ftp.qualcom.com in the quest\windows\eudora\1.4\beta directory.
	
	Eudora is manufactured by Qualcomm, a vendor independent of Microsoft; we make no
	warranty, implied or otherwise, regarding this product's performance or
	reliability.
