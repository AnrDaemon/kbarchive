## Q120815: Modified Windows Sound System Setup with Adaptec 1570 &amp; 1572

{% raw %}

	Article: Q120815
	Product(s): Miscellaneous Windows Products
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows Sound System, version 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Windows Sound System software found on Adaptec 1570 and 1572 audio
	cards has been modified by Adaptec.
	
	The Adaptec SCSI 1570 and 1572 integrated audio cards use the Windows Sound
	System software version 2.0. According to Adaptec, the Setup process has been
	modified for these particular cards. Therefore, Setup issues on these cards are
	handled by Adaptec. Usability issues are supported by Microsoft.
