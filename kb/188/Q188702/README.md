## Q188702: Epson Stylus 800 Not Compatible with Epson Stylus COLOR 800

	Article: Q188702
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0,4.0a
	Operating System(s): 
	Keyword(s): sbs
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server, Enterprise Edition version 4.0 
	- Microsoft BackOffice Small Business Server versions 4.0, 4.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Print jobs sent to an Epson Stylus Color 800 printer using an Epson Stylus 800
	ESC/P 2 printer driver on a Windows NT machine result in incorrect output.
	
	CAUSE
	=====
	
	The Epson Stylus 800 ESC/P 2 printer driver is not compatible with the Epson
	Stylus Color 800 printer. The Stylus 800 ESC/P 2 uses InkJet technology where
	the Stylus Color 800 uses Color InkJet technology.
