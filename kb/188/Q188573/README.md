## Q188573: INFO: Install Visual Basic 6.0 Complete from Visual Studio

	Article: Q188573
	Product(s): Microsoft Visual Basic for Windows
	Version(s): WINDOWS:6.0
	Operating System(s): 
	Keyword(s): kbGrpDSVB
	Last Modified: 11-JAN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual Basic Professional Edition for Windows, version 6.0 
	- Microsoft Visual Basic Enterprise Edition for Windows, version 6.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Visual Basic 6.0 ships separately and as a part of Visual Studio 6.0. This
	article describes how to achieve a complete installation of Visual Basic 6.0
	using the Visual Studio installation program.
	
	MORE INFORMATION
	================
	
	Each component listed below will need to be installed with the Visual Studio 6.0
	installation program to ensure the same development environment that Visual
	Basic 6.0 standalone offers.
	
	Microsoft Visual Basic 6.0
	
	- Core Files
	
	- Package And Deployment Wizard
	
	- Wizards and Templates
	
	- Webclass Designer
	
	Microsoft Visual SourceSafe 6.0 ActiveX (available from the main Custom setup
	dialog)
	
	Data Access
	
	- ADO, RDS, and OLE DB Providers
	
	- Microsoft ODBC Drivers
	
	- Jet IISAM Drivers*
	
	- Remote Data Objects and Controls
	
	- Data Environment
	
	Enterprise Features And Tools**
	
	- Application Performance Explorer
	
	- Visual Basic Enterprise Components
	
	- Visual Component Manager
	
	- Repository
	
	- Microsoft Visual Modeler
	
	Graphics*
	
	- Metafiles
	
	- Bitmaps
	
	- Cursors
	
	- Icon Library
	
	- Video
	
	Tools
	
	- Ms Info
	
	- API Text Viewer
	
	*Not installed with a "Typical" installation of Visual Basic 6.0. A "Custom"
	installation is needed to add these components.
	
	**Only available with the Enterprise Edition of Visual Basic 6.0.
