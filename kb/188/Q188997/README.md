## Q188997: Microsoft NetBIOS Computer Naming Conventions

	Article: Q188997
	Product(s): Windows for Workgroups and Windows NT Networking Issues
	Version(s): 3.11,3.51,4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 31-JAN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 3.11 
	- Microsoft Windows 95 
	- Microsoft Windows 98 
	- Microsoft Windows NT Server versions 3.51, 4.0 
	- Microsoft Windows NT Workstation versions 3.51, 4.0 
	- Microsoft Windows NT Server, Enterprise Edition version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article is intended to clarify what are valid NetBIOS computer names.
	
	MORE INFORMATION
	================
	
	A computer name can be up to 15 alphanumeric characters with no blank spaces.
	The name must be unique on the network and can contain the following special
	characters:
	
	  ! @ # $ % ^ & ( ) - _ ' { } . ~
	
	The Following characters are not allowed:
	
	  \ * + = | : ; " ? < > ,
