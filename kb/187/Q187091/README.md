## Q187091: Cannot Close a Shadow Session

	Article: Q187091
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-DEC-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0, Terminal Server Edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run a shadow session, you may not be able to close it by pressing
	CTRL+ALT+ESC.
	
	RESOLUTION
	==========
	
	Press CTRL+* (from the numeric keypad) to terminate a shadow session.
	
	Additional query words: quit terminate endtask
