## Q187448: Upgrading Does Not Copy Riched20.dll Correctly

	Article: Q187448
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-DEC-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0, Terminal Server Edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	During an upgrade installation of computers on which Microsoft Office 97 is
	installed, you may receive the following error:
	
	  The file riched20.dll was not copied correctly.
	
	RESOLUTION
	==========
	
	You may see this message during upgrade installations on computers running
	Microsoft Office 97. It is safe to ignore this warning, and to proceed with the
	installation. Riched20.dll is shipped with Windows NT to support WordPad.
