## Q187848: XWEB: Comic Sans Font in Meeting Requests

{% raw %}

	Article: Q187848
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook Web Access, version 5.5 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	When you use the Microsoft Outlook Web Access client, it may be difficult to tell
	which tab is active in a meeting request if the browser font is set to Comic
	Sans.
	
	WORKAROUND
	==========
	
	To work around this problem, use a different font other than Comic Sans.
