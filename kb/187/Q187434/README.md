## Q187434: Admin Process List Does Not Remove Deleted Processes

{% raw %}

	Article: Q187434
	Product(s): Microsoft Windows NT
	Version(s): WinNT:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-DEC-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0, Terminal Server Edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run logon and logoff tests and Terminal Server Admin is used to display
	the active processes, the processes that are no longer active remain in the list
	of processes.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a bug in Windows NT Server 4.0 Terminal
	Server Edition.
