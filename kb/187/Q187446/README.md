## Q187446: Uninstall Routines May Not Complete Successfully for Users

	Article: Q187446
	Product(s): Microsoft Windows NT
	Version(s): WinNT:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-DEC-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0, Terminal Server Edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	
	
	SYMPTOMS
	========
	
	Uninstall routines that require a restart of the computer will fail for users
	without administrator rights. Ordinary users cannot shutdown the Terminal
	Server.
	
	CAUSE
	=====
	
	This behavior is by design.
