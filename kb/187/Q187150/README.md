## Q187150: Icons Appear Too Large on System Desktop

{% raw %}

	Article: Q187150
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-DEC-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0, Terminal Server Edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you make a network connection from the host computer, the icons on the
	Start menu may grow to twice their original size.
	
	CAUSE
	=====
	
	When the Administrator enables the large icon option in the Display tool in
	Control Panel, this setting applies to all users (even if done remotely from a
	client).
	
	
	STATUS
	======
	
	This behavior is by design.
