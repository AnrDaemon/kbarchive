## Q187408: Shutdown of Terminal Server Takes Longer Than Windows NT 4.0

{% raw %}

	Article: Q187408
	Product(s): Microsoft Windows NT
	Version(s): WinNT:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-DEC-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0, Terminal Server Edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Shutdown takes longer on Windows NT Server version 4.0, Terminal Server Edition
	than on a Windows NT Server. Writing unsaved data to disk takes about 45-75
	seconds.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 4.0, Terminal
	Server Edition.
