## Q165789: DOC: Tab Text is too Small to Read

	Article: Q165789
	Product(s): Microsoft C Compiler
	Version(s): winnt:5.0,6.0
	Operating System(s): 
	Keyword(s): kbdocerr kbVC500 kbVC600 kbGrpDSTools
	Last Modified: 31-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual C++, 32-bit Enterprise Edition, versions 5.0, 6.0 
	- Microsoft Visual C++, 32-bit Professional Edition, versions 5.0, 6.0 
	- Microsoft Visual C++, 32-bit Learning Edition, version 6.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The text in the tabs for debug windows, such as the Output and Locals windows,
	is too small to read.
	
	MORE INFORMATION
	================
	
	The tab size is controlled by the desktop's scrollbar size. To change the
	scrollbar size, right-click the desktop and choose properties, or from the
	Taskbar click Start, Settings, Control Panel, and double-click the Display icon.
	Choose the Appearance tab and then change the Item selection to Scrollbar.
	
	Additional query words: unreadable
