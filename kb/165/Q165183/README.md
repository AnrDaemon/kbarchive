## Q165183: SMS: &quot;Could Not Spawn Initmain.exe&quot; Error Message

{% raw %}

	Article: Q165183
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.2
	Operating System(s): 
	Keyword(s): kberrmsg kbnetwork kbsetup kbInventory smssetup smsinv
	Last Modified: 27-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.2 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	Systems Management Server clients may encounter the following error message
	during the execution of the Smsls.bat file:
	
	  Could Not Spawn Initmain.exe
	
	This error message may occur when inventorying a computer by means of a logon
	script, and the computer is low on executable memory (possibly because Emm386 is
	not loaded).
	
	
	Additional query words: prodsms initmain
