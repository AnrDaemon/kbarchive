## Q165530: WD97: ODMA TeamLinks: Cannot Open Reserved File

{% raw %}

	Article: Q165530
	Product(s): Word 97 for Windows
	Version(s): 97
	Operating System(s): 
	Keyword(s): kb3rdparty kberrmsg kbusage winword word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you try to open a TeamLinks document that is already open by another user,
	you will receive the following message:
	
	  "<filename>". The object is already reserved by <username>. You
	  may open the file, but you cannot save changes back to the original.
	
	When you click OK, the document doesn't open.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
	
	MORE INFORMATION
	================
	
	Microsoft Word won't open a TeamLinks document as read-only if the document is
	reserved by another user.
	
	For additional information, please see the following article in the Microsoft
	Knowledge Base:
	
	  Q159963 Word 97 Supports ODMA
