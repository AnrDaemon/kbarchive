## Q165087: &quot;Change&quot; Keystroke on Server Configuration Dialog Box Is Missing

{% raw %}

	Article: Q165087
	Product(s): Microsoft SNA Server
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): kbnetwork
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft SNA Server, version 3.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you click the Server Configuration tab when you view server properties from
	within the SNA Server 3.0 Manager, you will see that there is no accelerator key
	for the Change button.
	
	RESOLUTION
	==========
	
	To resolve this problem, obtain the hotfix mentioned below. With the hotfix, the
	dialog box has now been modified so that you can select the Change button by
	pressing "C" on the keyboard instead of clicking the button with the mouse.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in SNA Server version 3.0. This
	problem was corrected in the latest Microsoft SNA Server 3.0 U.S. Service Pack.
	For information on obtaining the service pack, query on the following word in
	the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
	
	
	MORE INFORMATION
	================
	
	An accelerator key allows you to select an option without having to press ENTER
	or click on the option.
