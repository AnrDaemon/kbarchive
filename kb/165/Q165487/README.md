## Q165487: Incorrect or Garbled Text on Tab Labels in Properties

{% raw %}

	Article: Q165487
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): kbfile win95 kbgraphxlinkcriticalkbfixlist
	Last Modified: 03-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	- Microsoft Windows 95 OEM Service Release, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You may see incorrect or garbled text on tab labels when you are looking at an
	item's properties.
	
	
	STATUS
	======
	
	This issue is resolved by the following updated file for Windows 95:
	
	  Comctl32.dll version 4.70 (dated 5/19/97) and later
	
	For information about about obtaining the latest version of the Comctl32.dll
	file, please click the article number below to view the article in the Microsoft
	Knowledge Base:
	
	  Q186176 INFO: Redistribution of Comctl32.dll
