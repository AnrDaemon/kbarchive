## Q165477: How to Add Modem Tracking to System Monitor

{% raw %}

	Article: Q165477
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): kbtool win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to add modem tracking to System Monitor in Windows
	95.
	
	MORE INFORMATION
	================
	
	To add modem tracking to System Monitor, follow these steps:
	
	1. In Control Panel, double-click Modems. Click Properties, and then click the
	  Connection tab. Click Advanced, and then click the Record A Log File check
	  box to select it. Click OK or Close until you return to Control Panel.
	
	2. Connect to a remote service or server with HyperTerminal or Dial-Up
	  Networking.
	
	3. While you are connected, run System Monitor. To start System Monitor, click
	  the Start button, point to Programs, point to Accessories, point to System
	  Tools, and then click System Monitor.
	
	4. On the Edit menu in System Monitor, click Add Item. If the modem is
	  connected, the modem driver appears in the list. Click Bytes Received and
	  Bytes Sent and click OK.
	
	NOTE: System Monitor monitors modem usage only when you are using a 32-bit
	communications program.
