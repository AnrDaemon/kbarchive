## Q165660: 3270 Wizard Dialog Does Not Work Properly When Adding Users

	Article: Q165660
	Product(s): Microsoft SNA Server
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): kbnetwork
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft SNA Server, version 3.0, on platform(s):
	   - the operating system: Microsoft Windows NT 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	The 3270 Wizard dialog does not function properly when adding users. The TAB key
	does not select the first available user in the list and the Add and Add All
	buttons are not in tab order.
	
	RESOLUTION
	==========
	
	SNA Server version 3.0 has been updated to correct this problem. To resolve this
	problem, obtain the hotfix mentioned below. The updated file is:
	
	  Snawiz.dll
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in SNA Server version 3.0. This
	problem was corrected in the latest Microsoft SNA Server 3.0 U.S. Service Pack.
	For information on obtaining the service pack, query on the following word in
	the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
