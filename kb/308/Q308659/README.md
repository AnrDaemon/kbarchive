## Q308659: SMS: CTRL+N Closes SMS Administrator Console

	Article: Q308659
	Product(s): Microsoft Systems Management Server
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): kbenv kbtool kbsms200
	Last Modified: 10-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you press CTRL+N while the Systems Management Server (SMS) Administration
	console is running, the SMS console may quit and be replaced by the standard
	Microsoft Management Console (MMC) console root.
	
	CAUSE
	=====
	
	This issue can occur if you are running MMC version 1.2 on a Microsoft Windows
	2000-based computer.
	
	MORE INFORMATION
	================
	
	To use the SMS Administration console, you must restart it.
