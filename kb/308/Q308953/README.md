## Q308953: FP: Hover Buttons Appear Dimmed on Web Pages

	Article: Q308953
	Product(s): Word Front Page
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdta
	Last Modified: 29-JAN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft FrontPage 2000 
	- Microsoft FrontPage 2002 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Microsoft FrontPage, when you preview a Web page that contains hover buttons,
	one or more of the hover buttons may appear dimmed and may not link as
	expected.
	
	NOTE:These symptoms also occur if you view the Web page in a browser.
	
	CAUSE
	=====
	
	This behavior can occur if the Fphover.class file points to an invalid
	directory.
	
	RESOLUTION
	==========
	
	To resolve this issue, change the HTML code to reflect the correct Fphover.class
	path.
