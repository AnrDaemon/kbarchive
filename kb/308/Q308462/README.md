## Q308462: MCSE Readiness Review Exam 70-228 Microsoft SQL Server 2000 Admi

	Article: Q308462
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 11-JUL-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS MCSE Readiness Review Exam 70-228 Microsoft SQL Server 2000 Administration ISBN 0-7356-1250-1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book MCSE Readiness Review Exam 70-228 Microsoft
	SQL Server 2000 Administration, ISBN 0-7356-1250-1.
	
	The following topics are covered:
	
	- Page 8: Correction To Install SQL Server Instance
	
	- Pages 113 And 115: Correction To Question 70-228.03.05.001
	
	- Page 151: Correction To Question 70-228.04.04.003
	
	- Page 164: Distribution Database Should Be Distributor Servers
	
	- Page 165: Incorrect Answer For Question 70-228.04.06.001
	
	- Page 178: Correction To Question 70-225-.05.01.001
	
	MORE INFORMATION
	================
	
	Page 8: Correction To Install SQL Server Instance
	-------------------------------------------------
	
	On page 8, in the second paragraph, remove:
	
	"SQL Server does not limit the number of instances you can install on a single
	server, but your server's hardware limitations will."
	
	
	Pages 113 And 115: Correction To Question 70-228.03.05.001
	----------------------------------------------------------
	
	On page 113, in question 70-228.03.05.001,
	
	Change:
	"You want to run an integrity check on the tables in the database."
	
	To:
	"You want to run an integrity check on the tables in the database with repair
	options"
	
	On page 115, make the same change in the explanations for this question .
	
	
	Page 151: Correction To Question 70-228.04.04.003
	-------------------------------------------------
	
	On page 151, in question 70-228.04.04.003,
	
	Change:
	"...required to change the collation type for databases..."
	
	To:
	"...required to change the collation type for servers..."
	
	
	Page 164: Distribution Database Should Be Distributor Servers
	-------------------------------------------------------------
	
	On page 164, in question 70-228.04.06.005,
	
	Change:
	"B. Memory in the distribution database"
	
	To:
	"B. Memory in the Distributor servers."
	
	
	Page 178: Correction To Question 70-225-.05.01.001
	--------------------------------------------------
	
	On page 178, in question 70-228.05.01.001, the correct answer should be A, not A
	and B.
	
	Change the text to be:
	
	"Correct Answers: A
	...
	B. Incorrect: sp_serverproperty is not a stored procedure.
	..."
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
	
	Additional query words: TKBOOK 0-7356-1250-1 CHAUDHRY BARTHOLOMEW
