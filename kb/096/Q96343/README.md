## Q96343: Usage of PAGENO() Function in FoxBASE+/Mac

	Article: Q96343
	Product(s): Microsoft Fox Miscellaneous Products
	Version(s): MACINTOSH:2.01
	Operating System(s): 
	Keyword(s): 
	Last Modified: 23-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft FoxBASE+ for Macintosh, version 2.01 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The PAGENO() function can be used in FoxBASE+/Mac to specify page numbers in a
	report.
	
	MORE INFORMATION
	================
	
	The PAGENO() function is not listed in the FoxBASE+/Mac "Commands &
	Functions" manual, nor is it contained in the FoxBASE+/Mac Help file. The only
	reference to the PAGENO() function is in the FoxBASE+/Mac Demonstration Pack,
	which is no longer being distributed. The PAGENO() function can be found on the
	Math menu in the Expression Builder.
