## Q96719: PC WRmt: Demos Not Included Nor Can They Run in Mail Remote

	Article: Q96719
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Remote for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	To avoid redundant information, the version 3.2 of Microsoft Mail Remote for
	Windows does not include nor can it run the demos from version 3.2 of Microsoft
	Mail for Windows.
	
	The demos were replaced by a comprehensive Help file that aids in the
	installation and use of the remote client.
	
	Additional query words: 3.20
