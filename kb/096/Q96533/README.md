## Q96533: PC WRmt: Installing Mail Remote from a Network

	Article: Q96533
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Remote for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Version 3.2 of Microsoft Mail Remote for Windows can be loaded on a network to
	allow users across the network to install the program.
	
	Copy the contents of Disk 1 to a subdirectory called DISK1, Disk 2 to DISK2 and
	Disk 3 to DISK3 on a network share, then run SETUP.EXE, located in the DISK1
	directory of that share.
