## Q96785: Quitting Uninstall Causes Computer to Reboot

{% raw %}

	Article: Q96785
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you run the MS-DOS Uninstall program after starting (booting) your computer
	from drive C and then press the letter E to exit, your computer reboots instead
	of returning to the MS-DOS command prompt (C:).
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
