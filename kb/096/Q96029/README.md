## Q96029: SPEEDCACHE PLUS Version 4.3 Not MS-DOS 6.x-Compatible

{% raw %}

	Article: Q96029
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Iomega's SPEEDCACHE PLUS version 4.3 is not compatible with Microsoft MS-DOS 6.0
	and later.
	
	MORE INFORMATION
	================
	
	SPEEDCACHE PLUS is used to decrease the access time on Bernoulli drives. Iomega
	is testing SpeedCache with MS-DOS 6 and 6.2 and will make an updated version
	available soon.
	
	For more information, contact Iomega.
	
	The product included here is manufactured by Iomega, a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
