## Q96692: PC WRmt: Some View Menu Items Do Not Have Short-Cut Keys

{% raw %}

	Article: Q96692
	Product(s): Microsoft Mail For PC Networks
	Version(s): 3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 20-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, Windows Client, version 3.2 
	-------------------------------------------------------------------------------
	
	In Microsoft Mail Remote version 3.2 for Windows, the View menu
	contains several items that do not correspond to a short-cut key.
	
	Some common items, such as Connect Now and View Headers, cannot be
	assigned to a short-cut key. These menu items are defined in the
	Custom Commands section of the SHARED.INI file. Custom commands do not
	support short-cut keys.
