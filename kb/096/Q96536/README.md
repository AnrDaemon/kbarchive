## Q96536: PC WRmt: Exclusive During Active Modem Session

{% raw %}

	Article: Q96536
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Remote for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Version 3.2 of Microsoft Mail Remote for Windows takes exclusive control of
	Microsoft Windows while transmitting and receiving mail. Thus, you are in a
	single task environment and cannot switch away from Mail.
	
	Mail Remote for Windows does not support any other tasks while the modem session
	is active, either sending or receiving mail.
