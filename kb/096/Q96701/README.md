## Q96701: DoubleSpace Err Msg: Unknown Error Cause by Watchdog PC

	Article: Q96701
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0,6.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 23-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.0, 6.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	You may receive the following error message when you install DoubleSpace:
	
	  Unknown Error
	
	CAUSE
	=====
	
	This error is caused by a conflict between DoubleSpace and Watchdog PC Data
	Security software.
	
	WORKAROUND
	==========
	
	To work around this problem, disable your Watchdog PC Data Security software
	before you install DoubleSpace.
	
	The Watchdog product included here is manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
