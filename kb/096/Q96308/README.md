## Q96308: Mac Srv: Force Quit of Server as Application Doesn't Work

	Article: Q96308
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, version 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Using the COMMAND+OPTION+ESC key combination to stop a version 3.1 of Microsoft
	Mail for AppleTalk Networks server running as an application is not supported.
	The server must have time to write current activities to disk and shutdown
	normally. Restarting or shutting down the server's File menu are the only safe
	ways to force the server to stop.
