## Q96626: Uninstall Does Not Remove DEFAULT.&#42;, DOSBACK.&#42; and .INI Files

{% raw %}

	Article: Q96626
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 6.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you uninstall MS-DOS 6 Upgrade, the DEFAULT.* and DOSBACK.* files are not
	removed from the DOS directory.
	
	All files installed by Setup are removed by Uninstall. DEFAULT.*, DOSBACK.*, and
	any .INI files created by optional utilities are not removed.
