## Q96532: PC WRmt: Installing Over Another Copy of Windows Mail Client

{% raw %}

	Article: Q96532
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Remote for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When version 3.2 of Microsoft Mail Remote for Windows is installed over an
	existing installation of Microsoft Mail for Windows, the current mail driver
	does not change.
	
	The Mail Remote driver can be changed with the Microsoft Mail System Selector.
