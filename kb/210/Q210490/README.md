## Q210490: Err Msg: Msgolf99 Caused an Invalid Page Fault in Module Unknown

	Article: Q210490
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kberrmsg kbimu msgame
	Last Modified: 09-FEB-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Golf 1999 Edition, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start Microsoft Golf 1999 Edition, minimize and then restore the
	program, and then click Quick Start on the Golf 99 startup screen, you may
	receive the following error message:
	
	  Msgolf99 caused an invalid page fault in module Unknown.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Golf 1999 Edition,
	version 1.0.
	
	Additional query words: 1.00 msgame golf99 quickstart ipf
