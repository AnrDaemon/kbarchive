## Q210528: Golf 99: Incorrect Par Rating Stated in Flyby Commentary Script

	Article: Q210528
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbui kbimu msgame
	Last Modified: 09-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Golf 1999 Edition, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play the sixth hole in the Bay Harbor Golf Club course in Microsoft
	Golf 1999 Edition, the par rating for the hole displayed on the screen and
	listed on the scorecard does not match the par rating for the hole stated in the
	flyby commentary script for the hole.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Golf 1999 Edition,
	version 1.0.
	
	MORE INFORMATION
	================
	
	A par rating of four is displayed on the screen for the sixth hole, matching the
	par rating for the hole that is listed on the scorecard. However, the flyby
	commentary script for the sixth hole states that the par rating for the hole is
	five.
	
	Additional query words: 1.00 msgame golf99 number value shots
