## Q210961: Files Added to Favorites Folder May Not Appear in Favorites Menu

{% raw %}

	Article: Q210961
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:2000,4.01 Service Pack 2,4.x,5,5.01,5.5; Win2000:95,98,98 Second Edition; Win98
	Operating System(s): 
	Keyword(s): kbenv
	Last Modified: 13-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	- Microsoft Windows 98 
	- Microsoft Windows 98 Second Edition 
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows 2000 Advanced Server 
	- Microsoft Windows 2000 Datacenter Server 
	- Microsoft Windows 2000 Server 
	- Microsoft Windows 2000 Professional 
	- Microsoft Internet Explorer version 5.5 for Windows 2000 
	- Microsoft Internet Explorer versions 4.0, 4.01, 4.01 Service Pack 1, 4.01 Service Pack 2, 5, 5.01, 5.5 for Windows NT 4.0 
	- Microsoft Internet Explorer versions 5.01, 5.5 for Windows 98 Second Edition 
	- Microsoft Internet Explorer versions 4.01 Service Pack 2, 5, 5.01, 5.5 for Windows 98 
	- Microsoft Internet Explorer versions 4.0, 4.01, 4.01 SP1, 4.01 SP2, 4.5, 5, 5.01, 5.5 for Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you add files to the Favorites folder, they may not appear on the Favorites
	menu in Internet Explorer or on the Favorites menu on the Start menu.
	
	CAUSE
	=====
	
	This behavior occurs because these files are not shortcuts (.lnk or .url files).
	
	RESOLUTION
	==========
	
	To resolve this issue, do not place files in the Favorites folder. Create
	shortcuts to the files and place the shortcuts in the Favorites folder instead.
	
	To work around this issue, use one of the following methods:
	
	- If the files do not appear on the Favorites menu in Internet Explorer, click
	  the Favorites toolbar button and gain access to the files.
	
	- If the files do not appear on the Favorites menu on the Start menu, use
	  Windows Explorer to gain access to the files.
