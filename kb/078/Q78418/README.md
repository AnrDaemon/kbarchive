## Q78418: Marstek Mouse Driver 6.01b Required with MS-DOS 5.0

	Article: Q78418
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Marstek, Inc. has confirmed that the Marstek mouse requires version 6.01b or
	later of the Marstek mouse driver to work properly with MS-DOS Shell.
	
	MORE INFORMATION
	================
	
	Attempting to use an earlier version of the Marstek mouse driver with MS-DOS
	Shell may result in an invisible mouse pointer, a distorted mouse pointer, or no
	mouse pointer at all. The name of the Marstek mouse driver is MAMOUSE.COM.
