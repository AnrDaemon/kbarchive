## Q78947: Flight Sim: Screen Flashes on Stormy ILS Approach to Oakland

	Article: Q78947
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	If you select EFIS Navigation Challenges by choosing Entertainment from the
	Options menu, and you also select the option for Stormy ILS Approach to Oakland,
	View 1 periodically flashes. The flash varies between 10 and 30 seconds per
	flash.
	
	The flashing represents lightning from the thunderstorm in the clouds below the
	plane. This flashing or pulsing occurs anytime a flying situation involves
	lightning.
	
	Additional query words: 4.00
