## Q78946: Flight Sim: Turning System Sound Off Doesn't Turn Off FS Sound

{% raw %}

	Article: Q78946
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for the Macintosh, version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In Microsoft Flight Simulator for the Macintosh, you cannot turn the sound off
	by setting the volume control in the Sound control panel to zero.
	
	To completely turn off the sound in Flight Simulator, use one of the following
	methods:
	
	- Press the Q key.
	
	  -or-
	
	- Press COMMAND+U.
	
	  -or-
	
	- From the Options menu, clear the Sound option.
	
	Alternatively, choose Preferences from the Edit menu, and set Sounds to an option
	other than All Sounds (for example, select Cockpit).
	
	In Flight Simulator versions 1.0 and 1.2, you can turn off the sound by choosing
	Sound from the Sim menu.
