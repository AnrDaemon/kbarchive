## Q78461: Flashing Mouse Cursor in Windows 3.0 on AST Premium Exec

	Article: Q78461
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Not loading a special video driver supplied with the AST Premium Exec may
	produce a flashing Mouse cursor in Microsoft Windows 3.0.
	
	This information applies to all serial versions of the Microsoft Mouse for MS-DOS
	based systems.
	
	MORE INFORMATION
	================
	
	When using a Microsoft serial Mouse connected to the serial port of an AST
	Premium Exec, you may encounter a flashing Mouse cursor in Microsoft Windows
	3.0.
	
	The video display for the AST Premium Exec requires the installation of a video
	driver for Windows 3.0 called VGA640.DRV. This driver is included on the AST
	Utilities disk.

	we make no warranty, implied or otherwise, regarding this products' performance
	or reliability.
