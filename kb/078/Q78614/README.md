## Q78614: Houston Instruments Plotter Emulations

	Article: Q78614
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	According to Houston Instruments (HI) technical support, the following Houston
	Instrument Plotters can emulate Hewlett-Packard (HP) plotters:
	
	  HI Plotter              HP Plotter
	  ----------              ----------
	  DMP 60                  HP 7580
	  DMP 160                 HP 7580
	  HI 1117                 HP 7475A, HP 7550
	
	The products included here are manufactured by vendors independent of Microsoft;
	we make no warranty, implied or otherwise, regarding these products' performance
	or reliability.
