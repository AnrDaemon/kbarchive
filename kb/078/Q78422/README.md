## Q78422: Incorrect DIR Listing with Personics Ultravision

{% raw %}

	Article: Q78422
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:5.x,6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 5.0, 5.0a, 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	According to Personics, Ultravision customers must obtain a new directory
	expander for Ultravision to be compatible with MS-DOS 5.0 and later.
	
	MORE INFORMATION
	================
	
	All versions of Personics Ultravision may cause problems when using the MS-DOS
	DIR command. Ultravision is a terminate-and-stay-resident (TSR) utility that
	changes the colors of filenames in a directory listing. The problems may
	include, but are not limited to, the following:
	
	- Two or more files listed on the first line of the default listing and a blank
	  second line.
	
	- A blank second line appearing during a wide listing.
	
	- Incorrect use of the Ultravision color setup.
