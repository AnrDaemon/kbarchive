## Q32465: Switching from Header or Footer Dialog Box and Text in Write

	Article: Q32465
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:2.x,3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 2.03, 2.1, 2.11, 3.0, 3.0a, 3.1, 3.11 
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	When you create a header or footer in Microsoft Windows Write without using
	a mouse, you must use ALT+F6 to go back and forth between the dialog box
	and the header or footer text. The ALT+(underlined character in the dialog
	box) key code does not work until the dialog box is active.
	
	Additional query words: 3.00 3.00a 3.10 3.11
