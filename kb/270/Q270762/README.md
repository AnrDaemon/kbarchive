## Q270762: ELL FR &amp; SP: Setup Takes Longer Than You Expect

	Article: Q270762
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 
	Operating System(s): 
	Keyword(s): kb3rdparty kbsetup kbimu
	Last Modified: 23-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Language Learning French 
	- Microsoft Encarta Language Learning French Deluxe 
	- Microsoft Encarta Language Learning Spanish 
	- Microsoft Encarta Language Learning Spanish Deluxe 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you install any of the programs listed at the beginning of this article,
	you may wait for up to three minutes for the Setup wizard to be displayed on the
	screen.
	
	CAUSE
	=====
	
	This behavior can occur if one of the following CD-ROM drives is installed in
	your computer:
	
	- Fujitsu BIBLO NE/36L
	
	- Matsushita UDJB110 (PCMCIA)
	
	- NEC PC98-NX ValueStar VS30
	
	- Sony PCGA-CD5 (PCMCIA)
	
	The third-party products discussed in this article are manufactured by vendors
	independent of Microsoft; we make no warranty, implied or otherwise, regarding
	these products' performance or reliability.
