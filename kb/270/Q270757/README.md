## Q270757: ELL: DirectX Is Not Installed on Windows NT 4.0-Based Computers

{% raw %}

	Article: Q270757
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 
	Operating System(s): 
	Keyword(s): kbsetup kbimu
	Last Modified: 28-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Language Learning French 
	- Microsoft Encarta Language Learning French Deluxe 
	- Microsoft Encarta Language Learning Spanish 
	- Microsoft Encarta Language Learning Spanish Deluxe 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you attempt to install any of the programs listed at the beginning of this
	article on a Microsoft Windows NT 4.0-based computer, Microsoft DirectX 5.0 is
	not installed.
	
	MORE INFORMATION
	================
	
	Windows NT 4.0 includes a version of DirectX that meets the requirements of the
	Encarta Language Learning programs.
