## Q40797: Mac DOS: Connect/Access AppleTalk Networks at Remote Sites

	Article: Q40797
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): 3.0,3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, MS-DOS workstation, versions 3.0, 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Many companies make products that enable a Macintosh to operate as a remote
	AppleTalk node by accessing an AppleTalk network from a distant location. There
	are also products available for connecting remote AppleTalk networks. These
	products enable the remote sending and receiving of electronic mail, such as
	Microsoft Mail.
	
	NOTE: The products included here are manufactured by vendors independent of
	Microsoft; we make no warranty, implied or otherwise, regarding these products'
	performance or reliability.
