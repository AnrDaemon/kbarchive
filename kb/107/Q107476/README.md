## Q107476: Video Seven VRAM II Video Adapter Hangs WFWG 3.11 Setup

	Article: Q107476
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 31-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you use the Video Seven VRAM II video adapter and the VGA, Super VGA, or
	Super VGA (256 color) video driver with Microsoft Windows for Workgroups, your
	computer stop responding (hang) during Setup.
	
	CAUSE
	=====
	
	The Video Seven VRAM II video adapter is incompatible with these drivers in
	Windows for Workgroups 3.11.
	
	RESOLUTION
	==========
	
	Run Setup and select Custom Installation. Select the appropriate Video Seven
	display driver to install.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Windows for Workgroups
	version 3.11. We are researching this problem and will post new information here
	as it becomes available.
	
	Additional query words: 3.11 video seven vram 3rdparty
