## Q107055: CD Sampler: No Support for XGA

{% raw %}

	Article: Q107055
	Product(s): Microsoft Home Multimedia Titles
	Version(s): Fall 1993 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Home CD Sampler Fall 1993 edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft CD Sampler does not support displays configured for XGA.
	
	MORE INFORMATION
	================
	
	The IBM Extended Graphics Array (XGA) adapter is a memory-mapped graphics
	coprocessor for machines equipped with a Micro Channel Architecture (MCA) bus.
	The XGA adapter offers built-in VGA backward compatibility. The XGA adapter is
	shipped with 512K display memory (with an optional expansion to a total of 1
	MB). The IBM XGA is shipped as the standard display adapter for PS/2 models 90
	and 95.
	
	The system requirements for the Microsoft CD Sampler are:
	
	- A personal computer with a 386SX or higher microprocessor
	
	- 2 MB of available hard disk space
	
	- 4 MB RAM
	
	- MS-DOS 3.1 or later
	
	- Microsoft Windows 3.1 or later
	
	- Microsoft Mouse or compatible pointing device
	
	- 256-color VGA
	
	- MPC compatible CD-ROM
	
	- Audio board
	
	NOTE: Headphones or speakers are recommended.
