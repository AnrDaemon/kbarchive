## Q107475: Total Number of Workgroups That Can Be Displayed

{% raw %}

	Article: Q107475
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 03-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.11 
	-------------------------------------------------------------------------------
	
	Microsoft Windows for Workgroups version 3.11 is limited to displaying
	1,400 different workgroups simultaneously.
	
	
	Additional query words: gpf 3.11 limit maximum number total greater than less at one time
