## Q107823: Incorrect Sort Order in File Manager with Some Languages

	Article: Q107823
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): kbbuglist
	Last Modified: 01-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	- Microsoft Windows for Workgroups version 3.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you have the Swedish, Norway, Finnish, or Icelandic keyboard drivers
	installed for Windows, you may see an incorrect sort order in File Manager.
	
	CAUSE
	=====
	
	The incorrect sort order is caused by a problem with the LANGSCA.DLL file in
	Windows and Windows for Workgroups. (The LANGSCA.DLL file is the Finnish,
	Icelandic, Norwegian, and Swedish language driver used in Windows and Windows
	for Workgroups.)
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows versions 3.1 and 3.11.
	This problem was corrected in Windows for Workgroups version 3.11.
