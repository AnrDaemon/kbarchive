## Q107054: CD Sampler: Err Msg: Unable to Find Movie Named...

	Article: Q107054
	Product(s): Microsoft Home Multimedia Titles
	Version(s): Fall 1993 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Home CD Sampler Fall 1993 edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When attempting to run the Microsoft Home CD Sampler in a low memory situation,
	the following error message may occur:
	
	  Unable to find movie named A:\MSHOME.MMM.
	
	Choosing OK will usually allow the CD Sampler to terminate normally and return to
	the Program Manager.
	
	MORE INFORMATION
	================
	
	The system requirements for the Microsoft CD Sampler are:
	
	- A personal computer with a 386sx or higher microprocessor
	
	- 2 MB of available hard disk space
	
	- 4MB RAM
	
	- MS-DOS 3.1 or later
	
	- Microsoft Windows 3.1 or later
	
	- Microsoft Mouse or compatible pointing device
	
	- 256-color VGA
	
	- MPC compatible CD-ROM
	
	- Audio board
	
	NOTE: Headphones or speakers are recommended.
