## Q107375: WFWG 3.11: Hardware Compatibility List - Network Cards

	Article: Q107375
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 24-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists network interface cards (NICs) that have been tested by
	Microsoft and certified to be compatible with Windows for Workgroups 3.11. The
	list is alphabetical by manufacturer. Microsoft will add additional information
	to this article as it becomes available.
