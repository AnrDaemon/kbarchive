## Q107620: MAC Dinosaurs: Unable to Cancel Introductory Audio

	Article: Q107620
	Product(s): Microsoft Home Multimedia Titles
	Version(s): MACINTOSH:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Dinosaurs for the Macintosh 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When Microsoft Dinosaurs for the Macintosh is started, the introductory audio
	cannot be canceled by clicking the mouse on the active window.
	
	The only way to cancel the introductory audio is by pressing the COMMAND and
	PERIOD keys simultaneously.
	
	MORE INFORMATION
	================
	
	Microsoft Dinosaurs for the Macintosh is based on the original Windows version
	of Dinosaurs. Unlike the Macintosh version, when the Windows version of
	Dinosaurs is started, the introductory audio can be canceled by clicking the
	mouse on the active window.
