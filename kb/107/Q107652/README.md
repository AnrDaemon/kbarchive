## Q107652: Writer: Necessary Lantastic Network User Rights

{% raw %}

	Article: Q107652
	Product(s): Microsoft Home Kids Products
	Version(s): 1.00 1.10 1.10a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 04-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Creative Writer for Windows, versions 1.0, 1.1, 1.1a 
	- Microsoft Fine Artist for Windows, versions 1.0, 1.1, 1.1a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	As a network administrator, when you install Creative Writer or Fine Artist to a
	Lantastic network file server, you will need to give users the following
	rights:
	
	R - Read
	L - File Lookup
	E - Execute
	W - Write
	D - Delete
	
	To create new user installations, you will require the following rights:
	
	C - Create
	M - Make Directory
	
	NOTE: By default, Creative Writer or Fine Artist searches the root for the MSKIDS
	directory.
