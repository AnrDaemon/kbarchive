## Q107389: Cannot Print Directly to Imagewriter Printer

	Article: Q107389
	Product(s): Microsoft Home Multimedia Titles
	Version(s): MACINTOSH:1.0
	Operating System(s): 
	Keyword(s): kbbuglist
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Dinosaurs for the Macintosh 1.0 
	- Microsoft Musical Instruments for the Macintosh, version 1.0 
	- Microsoft Art Gallery for the Macintosh, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Attempting to print to an Imagewriter printer will result in no output, a single
	line of text (copywright line) or just a form feed.
	
	RESOLUTION
	==========
	
	To work around this problem, copy the image to the clipboard and paste it in
	another application such as a word processor.
	
	MORE INFORMATION
	================
	
	Microsoft has confirmed this to be a problem in the applications listed above.
	We are researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
