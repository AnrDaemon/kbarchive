## Q107095: PROFS: Incorrect Time and Date Stamp on Gateway Mail

{% raw %}

	Article: Q107095
	Product(s): Microsoft Mail For PC Networks
	Version(s): 3.1,3.2,3.3,3.4
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to IBM PROFS and OfficeVision, versions 3.1, 3.2, 3.3, 3.4 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	With Microsoft Mail Gateway to IBM PROFS, incoming mail sometimes bears the
	incorrect time or date stamp.
	
	CAUSE
	=====
	
	The Gateway PC time-stamps messages as they move from the VM environment to the
	PC LAN. If the Gateway PC clock is incorrect, the mail is stamped incorrectly.
	
	RESOLUTION
	==========
	
	Correct the time and date settings on the Gateway PC clock.
