## Q107177: Flight Simulator: No Icon Available in Windows

	Article: Q107177
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for MS-DOS, version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you set up Flight Simulator 5.0 for use under Microsoft Windows, the standard
	MS-DOS icon is displayed and not the icon the Microsoft Flight Simulator Pilot's
	Handbook pictured on page 194.
	
	MORE INFORMATION
	================
	
	Pages 193-194 of the manual provide a brief explanation of how to run Flight
	Simulator 5.0 under Microsoft Windows, and page 193 shows a picture of a Flight
	Simulator icon. However the icon shown does not ship with Flight Simulator 5.0
	and is not available on the system.
	
	Microsoft has confirmed this to be a problem in Flight Simulator for MS-DOS,
	version 5.0.
