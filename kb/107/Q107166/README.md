## Q107166: Using the TITLE Option in Viewer

	Article: Q107166
	Product(s): Miscellaneous Software Development Kits
	Version(s): 2.0,2.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Multimedia Viewer Publishing Toolkit, versions 2.0, 2.0a 
	-------------------------------------------------------------------------------
	
	The TITLE option, which can be specified in the [OPTIONS] section of
	the .MVP file, is used only if the "main" window has not been given a
	caption. A caption for the main window can be defined in its window
	definition in the [WINDOWS] section of the .MVP file. The Project
	Editor automatically gives the main window the caption "Untitled".
