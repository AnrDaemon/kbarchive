## Q107129: How to Disable 32-Bit File Access or 32-Bit Disk Access

	Article: Q107129
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.11 
	-------------------------------------------------------------------------------
	
	To disable 32-bit file access and start Windows for Workgroups version
	3.11, type the following at the MS-DOS prompt and then press ENTER:
	
	  win /d:c
	
	To disable 32-bit disk access and start Windows for Workgroups version
	3.11, type the following at the MS-DOS prompt and then press ENTER:
	
	  win /d:f
	
	Additional query words: 32-bit 3.11 WIN.COM
