## Q107104: X400: How to Add Route Table Expression

	Article: Q107104
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 29-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to X.400, version 3.2 
	-------------------------------------------------------------------------------
	
	The routing section of the Microsoft Mail Gateway to X.400 administration
	utility sometimes returns this message:
	
	  In the Route Table, indicate where to add the new route expression.
	
	
	To do this, highlight the preceding route expression, type in the route
	expression you want to add, and choose the Add button.
	
	Additional query words: 3.20 errmsg
