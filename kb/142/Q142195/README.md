## Q142195: CONN: Menu Options Grayed Out in the CNU

{% raw %}

	Article: Q142195
	Product(s): Microsoft Mail For PC Networks
	Version(s): 3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Connection for PC and AppleTalk Networks, version 3.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Some menu options in the Connection Name Utility may not be available and will
	appear gray in color. In order for these menu options to be available, you must
	be signed in to Microsoft Mail for AppleTalk Networks as the Network Manager.
	
	MORE INFORMATION
	================
	
	The following menu options will be grayed out if you have not signed in to
	Microsoft Mail as the Network Manager:
	
	File Menu
	---------
	
	  Save To File
	  Export Directory
	  Import Directory
	  Resync Cycle
	  Remove Mac Names from PC
	  Remove PC Names from Mac
	
	Configure Menu
	--------------
	
	  Select Gateway
	  Application
	  Address Filtering
	
	NOTE: The Application and Address Filtering commands will be available, but you
	will not be able to make any changes in these dialog boxes.
	
	For further information on setting up and running the CNU, see Chapter 5 in the
	version 3.2 Microsoft Mail Connection for PC and AppleTalk Networks
	"Administrator's Guide."
