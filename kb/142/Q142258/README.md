## Q142258: XADM: Exception Error on MTA Service Shutdown

	Article: Q142258
	Product(s): Microsoft Exchange
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kbother
	Last Modified: 19-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you stop the Microsoft Exchange Message Transfer Agent (MTA) service, you
	might see the following Dr. Watson error:
	
	  An application error has occurred and an application error log is being
	  generated
	
	  EMSMTA.EXE
	  Exception [0xe005085f] at address [0x77f3cbcd]
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server
	version 4.0. This problem was corrected in Microsoft Exchange Server 4.0 U.S.
	Service Pack 2. For information on obtaining the service pack, query on the
	following word in the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
