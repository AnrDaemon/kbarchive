	SUMMARY
	=======
	
	The information in this article is a copy of the information in the Readme.txt
	file included with the Windows 95 updated modem .inf files.
	
	MORE INFORMATION
	================
	
	Windows 95(R) Updated Modem INF Files
	
	INTRODUCTION
	------------
	
	This package installs .inf files for these specific modems:
	
	  mdmdsi.inf    Creative Labs Phone Blaster
	  mdmrock2.inf  Diamond Multimedia TeleCommander 2500
	  mdmrock3.inf  Aztech Systems
	  mdmzoom2.inf  Zoom Comstar
	
	The next time you go to install a modem, Windows 95 will rebuild the modem
	manufacturer database automatically. If you need additional assistance with
	installing your modem under Windows 95, search on the word "modem" in Windows 95
	help.
	
	Older versions of these files have been backed up to the windows inf\ backup
	directory.
	
	VOICEVIEW
	---------
	
	If you have Windows 95 on disk and you have a VoiceView modem, please download
	the Microsoft File Transfer package. That package contains files necessary for a
	complete installation of some modems in the above- mentioned INF files.
	
	The following file is available for download from the Microsoft Download Center:
	
	  Download Vvsetup.exe now
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
	
	Microsoft used the most current virus detection software available on the date of
	posting to scan this file for viruses. Once posted, the file is housed on secure
	servers that prevent any unauthorized changes to the file.
