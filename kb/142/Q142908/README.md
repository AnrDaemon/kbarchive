## Q142908: Intel SatisFAXtion 400i Driver Support for Windows NT 3.51

{% raw %}

	Article: Q142908
	Product(s): Microsoft Windows NT
	Version(s): 3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.51 
	- Microsoft Windows NT Server version 3.51 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The drivers (Intel SatisFAXtion 100 and Intel SatisFAXtion 400e) provided in
	Windows NT 3.51 does not support the Intel SatisFAXtion 400i internal modem. The
	Intel SatisFAXtion 400i is not in the Windows NT 3.51 Hardware Compatibility
	List (HCL).
	
	To obtain the drivers for the Intel SatisFAXtion 400i modem, contact Pure Data
	Technical Support at (905) 731-4533 or download NT351.EXE (self- extracting
	file) from one of the following electronic services:
	
	 -CompuServe
	     GO PCVENF
	     Search for NT351.EXE
	     Display results and download
	 
	 -Pure Data Bulletin Board Service (BBS)
	     Dial (905) 731-4679 to connect to the BBS
	     Download NT351.EXE

	of Microsoft; we make no warranty, implied or otherwise, regarding these
	products' performance or reliability.
