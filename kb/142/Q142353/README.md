## Q142353: Scenes 2.0: ErrMsg: Illegal Instruction In Module MSSCMN.DLL

	Article: Q142353
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Scenes (all collections), version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to run Scenes 2.0 under Windows NT 3.51 running on a Alpha, MIPS,
	or PowerPC computer, you receive the following error message:
	
	  Application Error
	
	  SCENES caused an illegal instruction in module MSSCMN.DLL at 0005:0487.
	
	The error occurs immediately after you double-click on the Microsoft Scenes icon
	or program (.exe) file.
	
	MORE INFORMATION
	================
	
	Scenes 2.0 is compatible only with the Intel version of Windows NT.
