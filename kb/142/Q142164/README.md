## Q142164: Admin UI Does Not Display Computers in Domains

	Article: Q142164
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.1
	Operating System(s): 
	Keyword(s): kbnetwork kbui kbsmsUtil smsutil
	Last Modified: 31-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.1 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	After you upgrade to Systems Management Server 1.1, the Admin user interface
	does not display the computer inventory information when you select a domain.
	However, if you refresh the display, the computer inventory information appears.
	This problem occurs when you select any domain in the Admin user interface.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.1. This problem has been corrected in Systems Management Server
	version 1.2.
