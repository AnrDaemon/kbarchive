## Q142535: MSN Icon Missing from Desktop

	Article: Q142535
	Product(s): The Microsoft Network
	Version(s): 2.5,2.6,5.0,5.1,5.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 2.5, 2.6, 5.0, 5.1, 5.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You may not see an icon for MSN, The Microsoft Network, on the desktop, even
	though you have installed MSN.
	
	CAUSE
	=====
	
	The icon may be hidden behind another icon or it may have been deleted.
	
	RESOLUTION
	==========
	
	To resolve this issue, you must first determine if the icon is just hidden. To
	do this, right-click on an empty area on the desktop, and then click Auto
	Arrange on the menu that appears. If the icon does not appear, then use the
	method for the version of MSN that you are using.
	
	MSN 2.5x and 2.6
	----------------
	
	1. Hold down the shift key and insert the MSN CD into the CD ROM drive.
	
	2. Click Start, point to Find, and then click "Files or Folders".
	
	3. In the Named box, type coreui.exe, click the CD-ROM drive in the Look In box,
	  and then click Find Now.
	
	4. In the list of found files, double-click coreui.exe.
	
	MSN 5.x
	-------
	
	1. Click Start, select Find, and then click Files or Folders.
	
	2. In the Named box type MSNCMCFG.EXE and then click on Find now.
	
	3. Find and double-click MSNCMCFG.EXE.
	
	4. When asked if you want to install MSN click Yes.
	
	5. When the Connection Window pops up, click Cancel.
	
	6. Close all programs and go back to the desktop.
