## Q142569: Mac Encarta Err Msg: Encarta Cannot Play the Animation

	Article: Q142569
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1995 edition,1996 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 24-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 96 Encyclopedia for Macintosh 
	- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia for Macintosh 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to view a Media Gallery animation or an animation within an
	article, you may receive the following message:
	
	  Encarta cannot play the animation. Close Encarta and start the Animation
	  Launcher program to view the animation.
	
	CAUSE
	=====
	
	Insufficient memory is available to display the animation when Encarta is
	running.
	
	RESOLUTION
	==========
	
	To solve this problem, use one of the following methods:
	
	Method 1
	--------
	
	The following method describes how to run the Animation Launcher outside Encarta
	'95.
	
	To run the animation launcher do the following:
	
	1. Quit Encarta '95.
	
	2. Open the Encarta '95 folder.
	
	3. Open the Animation Launcher program.
	
	4. Select an animation from the list.
	
	Method 2
	--------
	
	The following method describes how to check to see if other programs are running
	in the background and if so, close them:
	
	1. Go to the Programs menu, on the right end of the menu bar, and see if any
	  programs are listed. If so, click the program name; this will bring the open
	  program to the active window.
	
	2. Click Quit on the File Menu to close the program.
	
	Method 3
	--------
	
	The following method describes how to determine if Virtual Memory is turned off,
	and the steps needed to turn it on:
	
	1. On the Apple menu, click Control Panels and then click Memory.
	
	2. In the Virtual Memory section, click ON.
	
	3. To set the amount of disk space your Macintosh uses for RAM, and the total
	  amount of RAM available to your computer, click the up or down arrows to the
	  right of the box showing the amount of RAM available After restart.
	
	4. Close the Memory control panel.
	
	5. On the Special menu, click Restart.
	
	MORE INFORMATION
	================
	
	For additional information on how to optimize system memory, refer to the
	Encarta & Memory ReadMe file located on the Encarta '95 CD-ROM.
