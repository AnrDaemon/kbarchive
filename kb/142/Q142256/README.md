## Q142256: PSI Hard Disk Controllers Incompatible with Windows 95

	Article: Q142256
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Windows 95 may stop responding (hang) during startup, even in Safe mode, if you
	are loading the graphical user interface (GUI). However, you can boot Windows 95
	to a command prompt.
	
	CAUSE
	=====
	
	This problem can occur if your computer contains one of the following hard disk
	controllers by Perceptive Solutions, Inc. (PSI):
	
	- Easy Cache
	
	- Easy Cache Pro
	
	- Hyperstore
	
	- Hyperstore Easy Cache Pro
	
	According to PSI Technical Support, these hard disk controllers are incompatible
	with Windows 95.
	
	
	RESOLUTION
	==========
	
	Contact PSI for information about upgrading your hard disk controller.
