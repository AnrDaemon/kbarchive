## Q142426: MSN Icon Not Removed from Desktop When MSN Is Removed

	Article: Q142426
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.2,1.3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.2, 1.3 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you remove MSN, The Microsoft Network, from your computer using the
	Add/Remove Programs tool in Control Panel, the icon for MSN is not removed from
	the desktop.
	
	RESOLUTION
	==========
	
	The icon is removed when you restart your computer.
	
	Additional query words: msn
