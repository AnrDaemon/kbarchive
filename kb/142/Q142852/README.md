## Q142852: Unable to Install Digiboard PC 8e (8k) Adapter

	Article: Q142852
	Product(s): Microsoft Windows NT
	Version(s): 3.51
	Operating System(s): 
	Keyword(s): kb3rdparty
	Last Modified: 07-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.51 
	- Microsoft Windows NT Server version 3.51 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you install the Digiboard PC/8e (8K) adapter driver in Windows NT 3.51 on a
	Compaq Proliant computer, the following error message appears:
	
	  Please insert the disk: "Windows NT Workstation CD-ROM" into <CD-ROM
	  drive>".
	
	After you enter the correct path or replace the following files, the error
	message continues to appear:
	
	  - OEMNADDS.INF
	  - UTILITY.INF
	  - SUBROUTN.INF
	
	RESOLUTION
	==========
	
	To correct this problem, obtain the updated driver (version 1.2.1) from one of
	the following Digiboard electronic services or contact Digiboard International
	at (800) 344-4273 for more information:
	
	  -World Wide Web
	  ftp://ftp.dgi1i.com/support/drivers/winnt/released/async/
	  Download 821P.EXE
	
	Additional query words: prodnt
