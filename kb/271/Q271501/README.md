## Q271501: Golf 2001: Cannot Use Wheel on Pointing Device to Scroll

	Article: Q271501
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): kbhw kbimu msgame kbHardware
	Last Modified: 25-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Golf 2001 Edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Golf 2001 Edition, you cannot use the wheel on your
	pointing device to scroll within dialog boxes in the game.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Golf 2001 Edition.
	
	Additional query words: msgame golf01 intelli point mouse scrolls unable
