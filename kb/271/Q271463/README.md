## Q271463: Golf 2001 Edition: Minimum System Requirements

	Article: Q271463
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): kbref kbimu msgame
	Last Modified: 09-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Golf 2001 Edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the minimum system requirements that are needed to play
	Microsoft Golf 2001 edition.
	
	MORE INFORMATION
	================
	
	To play Microsoft Golf 2001 edition, you need:
	
	- Multimedia PC with a Pentium 166 MHz or faster processor with MMX (Pentium
	  233 MHz processor with MMX recommended)
	
	- Microsoft Windows 95 or Windows 98 operating system, Windows NT Workstation
	  operating system version 4.0 with Service Pack 3 or later, or Microsoft
	  Windows 2000 operating system
	
	- 32 megabytes (MB) of RAM for Windows 95 or Windows 98 (64 MB recommended); 64
	  MB of RAM for Windows NT 4.0 or Windows 2000
	
	- 260 MB of available hard disk space (490 MB recommended)
	
	- Quad-speed or faster CD-ROM drive (8X or faster CD-ROM drive recommended)
	
	- Local bus video adapter with at least 2 MB of memory capable of displaying
	  32,000 colors at a screen resolution of 800 x 600 pixels
	
	- Sound card with speakers or headphones for audio
	
	- Microsoft Mouse or compatible pointing device and a keyboard
	
	- Internet service provider (ISP) and access to the Internet for Internet play
	
	- Web browser that supports ActiveX and frames (such as Microsoft Internet
	  Explorer 3.02 or later) for Zone.com play
	
	NOTE: Internet access may require payment of a separate fee to an Internet
	service provider. Connect time charges may apply.
