## Q271254: Microsoft Reader: How to Listen to an Audiobook

	Article: Q271254
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1.5
	Operating System(s): 
	Keyword(s): kb3rdparty kbui kbimu
	Last Modified: 31-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Reader, version 1.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to download and install the software that you need to
	listen to an audiobook using the Microsoft Reader software for desktop and
	laptop computers, and also describes how to listen to an audiobook after you
	install the software.
	
	MORE INFORMATION
	================
	
	Microsoft Reader is AudibleReady? and can play any of the over 12,000 audio
	titles available at the following Audible Web site:
	
	  http://www.audible.com
	
	After you purchase an Audible title, you need to download and install the
	AudibleManager<AE> software. The AudibleManager<AE> software is used to download your
	purchases and enables Microsoft Reader to automatically display them in the
	Microsoft Reader Library.
	
	To download the AudibleManager<AE> software, visit the Audible Web site and click
	"Download Software to Listen" under Audible.com Basics on the left side of the
	screen. To listen to an audiobook after you have installed the AudibleManager<AE>
	software:
	
	1. Click the book in the Microsoft Reader Library to display its cover page.
	
	2. Click Play in the control area at the bottom of the page to listen to the
	  book.
	
	If you need help or have questions about Audible titles, contact Audible.com
	Customer Support at (888) 283-5051 or send an e-mail message to
	customer_service@audible.com.
