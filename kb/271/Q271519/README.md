## Q271519: ELL French &amp; Spanish: Minimum System Requirements

	Article: Q271519
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 28-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Language Learning French 
	- Microsoft Encarta Language Learning Spanish 
	- Microsoft Encarta Language Learning French Deluxe 
	- Microsoft Encarta Language Learning Spanish Deluxe 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the minimum system requirements that you need to run the
	Encarta Language Learning programs that are listed at the beginning of this
	article.
	
	MORE INFORMATION
	================
	
	The minimum system requirements to run the Encarta Language Learning programs
	are:
	
	- Multimedia PC with Pentium 166 MHz or higher processor
	
	- Microsoft Windows 95 or later with Microsoft Internet Explorer 3.02 or later,
	  or Microsoft Windows NT 4.0 with Service Pack 3 or later.
	
	- 32 megabytes (MB) of memory (RAM) for Windows 95, Microsoft Windows 98, and
	  Windows NT 4.0. 64 MB of memory (RAM) for Microsoft Windows 2000
	
	- 160 MB of available hard disk space for the installation
	
	- 110 MB of available hard disk space to run the software
	
	- 12X or faster CD-ROM drive
	
	- Super VGA, 16-bit color or higher monitor that supports 800 X 600 or higher
	  screen resolution
	
	- Microsoft mouse or compatible pointing device
	
	- 16-bit sound card with speakers or headphones
	
	- Microphone with stereo output plug (included with the Encarta Language
	  Learning programs)
	
	- Microsoft Internet Explorer 5.01 (included with the Encarta Language Learning
	  programs) or later
	
	To access Encarta Language Learning Online, you need:
	
	- 14.4-baud modem (28.8-baud or faster recommended) and an account with an
	  Internet service provider (ISP) or a network connection to the Internet
	
	- Web browsing software
	
	NOTE: To gain access to and to use the Internet may require payment of a separate
	fee to an ISP. Local and long distance telephone charges may apply.
