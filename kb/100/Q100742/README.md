## Q100742: Problems Running MSAV While MIDI File Is Playing

	Article: Q100742
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 23-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you play a MIDI file while you scan for viruses with Microsoft Anti-Virus for
	Windows, your system may stop responding (hang) or you may receive a general
	protection (GP) fault in KRNL386.EXE or your sound driver.
	
	WORKAROUND
	==========
	
	To avoid this problem, do not play MIDI files when you scan for viruses with
	Microsoft Anti-Virus for Windows.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in MS-DOS 6.0, 6.2, 6.21, and 6.22
	Anti-Virus for Windows. We are researching this problem and will post new
	information here as it becomes available.
