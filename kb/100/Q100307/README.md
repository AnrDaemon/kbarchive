## Q100307: No Default Guest Account in Windows NT Advanced Server

{% raw %}

	Article: Q100307
	Product(s): Microsoft Windows NT
	Version(s): 3.1
	Operating System(s): 
	Keyword(s): kbsetup
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Advanced Server, version 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	By default, the Windows NT Advanced Server Setup program does not enable a guest
	account. If desired, use the User Manager to create a guest account.
