## Q100404: How to Upgrade a Micro House DrivePro or EZ-Drive System

	Article: Q100404
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you try to upgrade a computer using Micro House DrivePro or EZ-Drive,
	MS-DOS Setup refers you to the README.TXT file. According to Micro House, the
	instructions in the README.TXT file may not work correctly.
	
	If your hard disk drive was set up using DrivePro or EZ-Drive, contact Micro
	House for assistance installing MS-DOS 6.0 or 6.2.
	
	Additional query words: 6.22 6.00 6.20 docerr doc err section 1.16 microhouse
