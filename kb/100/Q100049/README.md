## Q100049: HP OmniBook: Default Printer Drivers

{% raw %}

	Article: Q100049
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Hewlett-Packard (HP) OmniBook 300 comes with several Microsoft Windows
	version 3.1 printer drivers preinstalled. The complete set of Windows 3.1
	printer drivers is not included.
	
	To obtain Windows 3.1 printer drivers not included with the HP OmniBook, contact
	Hewlett-Packard Technical Support.
	
	MORE INFORMATION
	================
	
	Below is a listing of the drivers installed by default to the Hewlett- Packard
	OmniBook 300:
	
	  Epson FX-80 on LPT1
	  Epson FX-80 on LPT2
	  Epson LQ-510 on LPT1
	  Epson LQ-510 on LPT2
	  HP DeskJet 500 on LPT1
	  HP DeskJet 500 on LPT2
	  HP LaserJet on LPT1
	  HP LaserJet on LPT2
	
	The OmniBook 300 is manufactured by Hewlett-Packard, a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
