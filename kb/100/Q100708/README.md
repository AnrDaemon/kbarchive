## Q100708: SMTP: Err Msg: ERROR write: (119) Network Error

	Article: Q100708
	Product(s): Microsoft Mail For PC Networks
	Version(s): MS-DOS:2.1,3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to SMTP, versions 2.1, 3.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Versions 2.1 and 3.0 of Microsoft Mail Gateway to SMTP may display this error
	message:
	
	  ERROR write: (119) network error
	
	CAUSE
	=====
	
	This indicates an error at the TCP or IP layer.
	
	RESOLUTION
	==========
	
	Send the message again. If this error occurs frequently, check the network with
	a protocol analyzer to determine the exact problem.
	
	Additional query words: 2.10 3.00 errmsg alert
