## Q100092: MS-DOS 6 and 6.2 Boot-Sector Identification Tag Is &quot;MSDOS5.0&quot;

	Article: Q100092
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 24-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The MS-DOS 6.0 and 6.2 original equipment manufacturer (OEM) boot-sector
	identification tag is "MSDOS5.0," the same as for MS-DOS 5.0.
	
	MS-DOS 6.0 and 6.2 use the same boot-sector identification tag as MS-DOS 5.0 to
	maintain compatibility with some third-party programs.
	
	
	When you use the VER command or interrupt 21h function 3306h (Get MS-DOS Version)
	to check the version, MS-DOS reports the correct verison number.
	
	
	Additional query words: 6.22 5.00 5.00a 6.00 6.20 OEM ID
