## Q100117: Expert Developer CD Disk Information

	Article: Q100117
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Expert Developer CD, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Expert Developer CD is provided to participating developers, and
	contains only strategic Microsoft products and a disk imaging utility.
	
	MORE INFORMATION
	================
	
	These products are included on the Microsoft Expert Developer CD:
	
	- C Compiler
	
	- C 7.00 Patch
	
	- FORTRAN
	
	- QuickC for MS-DOS
	
	- QuickC for Windows
	
	- Macro Assembler
	
	- Visual Basic for MS-DOS
	
	- Visual Basic for Windows
	
	- Test for Windows
	
	- FoxPro for MS-DOS
	
	- FoxPro for Windows
	
	- Access
	
	Additional query words: 1.00 multi media multi-media multimedia
