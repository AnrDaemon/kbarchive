## Q100353: True Type Fonts Supported by Windows NT

	Article: Q100353
	Product(s): Microsoft Windows NT
	Version(s): winnt:3.1,3.5,3.51; :
	Operating System(s): 
	Keyword(s): kbother
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server versions 3.1, 3.5, 3.51 
	- Microsoft Windows NT Workstation versions 3.1, 3.5, 3.51 
	- Microsoft Windows NT Advanced Server 
	-------------------------------------------------------------------------------
	
	True Type fonts designed for Microsoft Windows operating system
	version 3.1 are supported by Windows NT. Any installed True Type fonts
	will be available to both 16-bit and 32-bit Windows-based
	applications. They will not be available in the character-based
	subsystems, such as MS-DOS, OS/2, and POSIX.
	
	Replacement font managers designed for Windows 3.1, such as Adobe Type
	Manager and Bitstream FaceLift, are not supported and will not work
	under Windows NT. These products require updates to work under Windows
	NT. Contact your font manager vendor for more information.
