## Q100723: HP PaintJet XL 300 Driver Limited to Eight Colors

{% raw %}

	Article: Q100723
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The printer driver for the Hewlett-Packard PaintJet XL 300 provided with
	Microsoft Windows version 3.1 is limited to eight colors. This color limitation
	is determined by code that cannot be changed in UNIDRV.DLL and DMCOLOR.DLL.
	
	To work around this limitation, use the printer driver supplied by
	Hewlett-Packard.
	
	MORE INFORMATION
	================
	
	To obtain phone numbers for Hewlett-Packard, query on the following words in the
	Microsoft Knowledge Base:
	
	  "Hewlett-Packard" (without the quotation marks) and "phonebook" (without the
	  quotation marks)
