## Q100091: MSAV Display Problems with Older 8514 Display Adapters

	Article: Q100091
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Microsoft Anti-Virus has display problems or causes your machine to stop
	responding (hang).
	
	CAUSE
	=====
	
	You have an older 8514 display adapter.
	
	NOTE: This problem affects very few 8514 display adapters.
	
	WORKAROUND
	==========
	
	To correct this problem, upgrade your video BIOS.
	
	Additional query words: 6.22 6.00 6.20 msav.exe 8514/a 8514a
