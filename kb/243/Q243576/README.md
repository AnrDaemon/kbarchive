## Q243576: Encarta Encyclopedia 99: No Audio Output When You Play Certain S

{% raw %}

	Article: Q243576
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbsound kbimu
	Last Modified: 21-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Encyclopedia 99 
	- Microsoft Encarta Encyclopedia Deluxe 99 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to play the following sounds from Microsoft Encarta
	Encyclopedia, you<A0>do not hear any audio output.
	
	Encarta 99 Standard
	-------------------
	
	- Bach's Goldberg Variations
	- Charles De Gaulle
	- Common Hippopotamus
	- Elephant with Calf
	
	Encarta 99 Deluxe
	-----------------
	
	- Common Hippopotamus
	- Elephant with Calf
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
	
	Additional query words: multi multi-media media mm enc99 silent quiet
