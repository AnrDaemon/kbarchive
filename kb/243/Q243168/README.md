## Q243168: Asheron's Call Requires a Pentium 166 or Higher...

{% raw %}

	Article: Q243168
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbenv kberrmsg kbimu msgame
	Last Modified: 16-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Asheron's Call, version 1.0 
	- Microsoft Asheron's Call: Dark Majesty, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to install Microsoft Asheron's Call, you may receive the
	following error message:
	
	  Microsoft Asheron's Call requires a Pentium 166 or higher processor. Setup
	  has not detected the required minimum processor; however, you may continue
	  the installation.
	
	When you attempt to install Microsoft Asheron's Call: Dark Majesty, you may
	receive the following error message:
	
	  Requires a Pentium 333 or Higher Processor.
	
	CAUSE
	=====
	
	This behavior can occur if any of the following conditions are true:
	
	- You attempt to install Asheron's Call on a computer that does notmeet the
	  minimum system requirements needed to run Asheron's Call.
	
	- A Cyrix microprocessor is installed in the computer.
	
	- You attempt to install Asheron's Call on a Sony laptop computer.
	
	RESOLUTION
	==========
	
	If your computer does not meet the minimum system requirements needed to run
	Asheron's Call, Microsoft recommends that you do not continue the installation.
	
	If a Cyrix microprocessor running at a clock speed of 166 Mhz or higher is
	installed in your computer, you may continue the installation.
	
	If a Sony laptop computer is based on a processor running at a clock speed of 166
	Mhz or higher, you may continue the installation.
