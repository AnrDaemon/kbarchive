## Q243613: Unable to access AOL with MSN

	Article: Q243613
	Product(s): The Microsoft Network
	Version(s): WINDOWS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-APR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network version 5.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You have both AOL and MSN installed, but after the MSN installation, you are
	unable to access MSN.
	
	CAUSE
	=====
	
	Your connection settings may still be set for use with AOL.
	
	RESOLUTION
	==========
	
	To resolve this issue, you need to configure AOL to use TCP/IP access:
	
	1. At the AOL login screen, click Setup.
	
	2. Select Edit Location.
	
	3. Choose TCP/IP as your network type.
	
	Once this has been resolved, you should be able to use both AOL and MSN by
	connecting to the Internet using your MSN account and then accessing AOL.
