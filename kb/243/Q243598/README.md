## Q243598: Err Msg: Setup Is Unable to Continue Due to an Error on the...

{% raw %}

	Article: Q243598
	Product(s): The Microsoft Network
	Version(s): WINDOWS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network version 5.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to install the MSN 5.0 CD-ROM you may receive this error
	message:
	
	  Setup is unable to continue due to an
	  error on the MSN Internet Access CD-ROM.
	
	  Please click OK and then reinsert the MSN
	  Internet Access CD. If you continue to see
	  this message, please order a new MSN
	  Internet Access CD at the following Web
	  site: http://free.msn.com.
	
	CAUSE
	=====
	
	The CD-ROM you are trying to install may be damaged.
	
	RESOLUTION
	==========
	
	To resolve this issue, please go to http://free.msn.com and order a new CD
	online or call MSN Member Support at 1-800-386-5550.
