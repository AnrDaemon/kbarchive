## Q243824: XFOR: How to Install the Application Converter for Lotus Notes

{% raw %}

	Article: Q243824
	Product(s): Microsoft Exchange
	Version(s): 5.5
	Operating System(s): 
	Keyword(s): exc55
	Last Modified: 26-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Exchange Application Converter for Lotus Notes is available as a
	Web release at the following Microsoft Web site:
	
	  http://www.microsoft.com/exchange/downloads/55/NotesAppServices.asp
	
	The download file name is Mxsetup.exe. This article explains how to install the
	Application Converter for Lotus Notes.
	
	MORE INFORMATION
	================
	
	To install the Application Converter for Lotus Notes:
	
	1. Download the latest version of the Application Converter for Lotus Notes from
	  the Microsoft Web site listed in the "Summary" section to your computer. Note
	  the folder that it was saved to.
	
	2. Click Start, and then click Run.
	
	3. In the Run dialog box, type "c:\temp\mxsetup.exe" (without the quotation
	  marks), and then click OK. If you saved the file in a folder other than
	  C:\Temp, substitute the correct folder name, noted in step 1 into the
	  preceding command.
