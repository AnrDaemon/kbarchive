## Q243947: FTP Service Not Installed by Default in IIS 5.0

	Article: Q243947
	Product(s): Internet Information Server
	Version(s): winnt:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 04-FEB-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Internet Information Services version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft Internet Information Services (IIS) version 5.0 does not install the
	FTP service by default. This is by design.
	
	MORE INFORMATION
	================
	
	To install the FTP service, use the following steps:
	
	1. In Control Panel, click Add/Remove Programs.
	
	2. Select Add/Remove Windows Components.
	
	3. Select Internet Information Services.
	
	4. Click to check the File Transfer Protocol check box.
	
	5. Click OK.
	
	6. Click Finish.
