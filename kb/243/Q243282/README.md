## Q243282: Find Tools Locates Files with a Time Stamp of a Future Date

	Article: Q243282
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): kbtool kbui win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you configure the Find tool to find all files created or modified during
	the previous 1 day, your search results may include files with a time stamp of a
	future date.
	
	CAUSE
	=====
	
	This behavior can occur if your computer includes files that have been time
	stamped with a future date. Note that some software companies include files in
	their programs that have been time stamped with a future date.
