## Q243228: Asheron's Call: Action Does Not Occur When Using Key Combination

	Article: Q243228
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbhw kbimu msgame kbHardware
	Last Modified: 16-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Asheron's Call, version 1.0 
	- Microsoft Asheron's Call: Dark Majesty, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you press the SHIFT key to modify the action of another key, you may be
	unable to press an additional key to cause an action in the game.
	
	For example, if you press SHIFT+W to move your character at an altered pace, you
	may be unable to press the SPACEBAR to cause your character to jump.
	
	CAUSE
	=====
	
	This behavior can occur if you are using a Gateway keyboard.
	
	This problem has been observed with the following Gateway keyboard model:
	
	- 2194013-xx-xxx
