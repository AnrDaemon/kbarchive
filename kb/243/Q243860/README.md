## Q243860: Flight Simulator 2000: Minimum System Requirements

	Article: Q243860
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbsetup kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator 2000 
	- Microsoft Flight Simulator 2000 Professional Edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the minimum system requirements needed to run Microsoft
	Flight Simulator 2000 or Microsoft Flight Simulator 2000 Professional.
	
	MORE INFORMATION
	================
	
	To run Microsoft Flight Simulator 2000 or Microsoft Flight Simulator 2000
	Professional, you need:
	
	- A multimedia PC with a Pentium 166 MHz or faster processor (SSE or 3DNow
	  instruction set processor recommended).
	
	- Microsoft Windows 95 operating system or later, or Microsoft Windows NT 4.0
	  with Service Pack 3 operating system or later.
	
	- 32 megabytes (MB) of RAM for Windows 95/98 (64 MB recommended), 64 MB of RAM
	  for Windows NT 4.0.
	
	- 350 MB of available hard disk space for Flight Simulator 2000 (430 MB
	  recommended), 420 MB of available hard disk space for Microsoft Flight
	  Simulator 2000 Professional (500 MB recommended).
	
	- 50 MB of available hard disk space for the swap file.
	
	- A quad-speed or faster CD-ROM drive.
	
	- A Super-VGA video adapter and monitor capable of displaying 16-bit color
	  (Microsoft Direct3D 7.0-compatible 3D graphics accelerator card or video
	  adapter with 4 MB or more of video memory recommended).
	
	- A Microsoft DirectSound 7.0-compatible sound card with speakers or headphones
	  for sound playback.
	
	- A Microsoft mouse or compatible pointing device.
	
	- Optional: A joystick or flight yoke game controller (Microsoft DirectInput
	  7.0-compatible force feedback game controller recommended).
	
	- A 28,800 bits-per-second (bps) or faster modem, a local area network (LAN)
	  with TCP/IP or IPX protocol, or a null-modem cable to play multiplayer games.
	
	- Microsoft Internet Explorer 3.02 or later or Netscape Communicator 4.0 or
	  later to play multiplayer games on Zone.com.
	
	  NOTE: Internet features require Internet access. Connect time charges may
	  apply.
