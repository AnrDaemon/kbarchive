## Q191297: SMS: SMSEXEC May Dr Watson If a Distribution Server Is Inactive

{% raw %}

	Article: Q191297
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 04-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When performing a distribution, if a target distribution server is offline or
	inactive, the Systems Management Server Executive service may generate a Dr.
	Watson error.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in System Management Server version
	1.2. This problem has been corrected in the latest U.S. service pack for Systems
	Management Server version 1.2. For information on obtaining the service pack,
	query on the following word in the Microsoft Knowledge Base (without the
	spaces):
	
	  S E R V P A C K
	
	Additional query words: prodsms sp sp4 SMSExec exec
