## Q191037: INFO: VB6 Learning, Professional, and Enterprise Features

	Article: Q191037
	Product(s): Microsoft Visual Basic for Windows
	Version(s): 6.0
	Operating System(s): 
	Keyword(s): kbsetup kbVBp kbVBp600 kbGrpDSVB kbDSupport
	Last Modified: 31-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual Basic Learning Edition for Windows, version 6.0 
	- Microsoft Visual Basic Professional Edition for Windows, version 6.0 
	- Microsoft Visual Basic Enterprise Edition for Windows, version 6.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes the primary features supported in the Visual Basic
	version 6.0 Learning, Professional, and Enterprise Editions.
	
	MORE INFORMATION

	The table below lists the key features of Microsoft Visual Basic 6.0 and which
	versions these features ship with (Ent = Enterprise, Pro = Professional, Lrn =
	Learning). An "X" indicates that the feature is included with the appropriate
	edition of Visual Basic.
