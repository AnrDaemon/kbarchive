## Q191376: WD97: Tabs Button Doesn't Work with Envelopes and Labels

	Article: Q191376
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta winword word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In the Envelopes And Labels dialog box (on the Tools menu, click Envelopes And
	Labels), when you click the Tabs button on the Indents and Spacing tab
	(right-click in the Address box on the Labels tab, and then click Paragraph on
	the shortcut menu), Word returns to the Labels tab and the Tabs dialog box is
	not displayed.
	
	NOTE: The Tabs dialog box is displayed after you click Print, New Document, or
	Cancel in the Envelopes And Labels dialog box.
	
	WORKAROUND
	==========
	
	Click Cancel to close the Tabs dialog box.
	
	NOTE: If you set a tab in the Tabs dialog box, the tab will be set at the
	insertion point of the active document.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
	
	This problem has been corrected in Word 2000.
	
	MORE INFORMATION
	================
	
	In Microsoft Word 97 for Windows, when you are using paragraph formatting in the
	Address box on the Labels tab, the Tabs button on the Indents and Spacing tab
	should be unavailable as in earlier versions of Microsoft Word for Windows.
