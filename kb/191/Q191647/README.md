## Q191647: Motocross Madness: Minimum System Requirements

	Article: Q191647
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbref kbimu msgamekbfaq
	Last Modified: 21-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Motocross Madness, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the minimum system requirements needed to play Microsoft
	Motocross Madness.
	
	MORE INFORMATION
	================
	
	To play Motocross Madness, you need:
	
	- A Multimedia PC with a Pentium 133-MHz processor and a 4-megabyte (MB) 3-D
	  accelerator, or a Multimedia PC with a Pentium 166-MHz processor and a 2-MB
	  3-D accelerator.
	
	- Microsoft Windows 95 or Microsoft Windows 98 operating system.
	
	- A Microsoft DirectX 5.0-compatible 3-D accelerator.
	
	- 16 MB of RAM.
	
	- 60 MB of available hard disk space for minimum installation or 180 MB of
	  available hard disk space for typical installation (with 50 MB available hard
	  disk space for the swap file).
	
	- A quad-speed or faster CD-ROM drive.
	
	- A Microsoft Mouse or compatible pointing device.
	
	- A DirectX 5.0-compatible sound card.
	
	- Speakers or headphones are recommended.
	
	- A DirectX 5.0-compatible joystick or game pad is recommended.
	
	- A 28.8K modem is required for head-to-head play.
	
	- Internet access is required for Internet play (connect time charges may
	  apply).
	
	NOTE: Game performance may vary depending on your 3-D hardware.
