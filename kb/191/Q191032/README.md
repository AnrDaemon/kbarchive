## Q191032: WD97: HTML Images Greater Than 6&quot; Are Scaled on Import

	Article: Q191032
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta kbhtml
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you import an HTML document that contains an images with a width greater
	than 6 inches or 572 pixels, the width of the image is scaled to 6 inches.
	
	CAUSE
	=====
	
	This is a limitation of the HTML converter.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
	
	This problem was corrected in Microsoft Word 2000.
