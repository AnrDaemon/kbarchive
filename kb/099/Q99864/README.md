## Q99864: Buttons Seem Taller on 12-Inch Screen

{% raw %}

	Article: Q99864
	Product(s): Microsoft Home Multimedia Titles
	Version(s): MACINTOSH:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Musical Instruments for the Macintosh, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The buttons in Musical Instruments appear proportionately taller on 12-inch
	screens than they do on 13-inch screens. The buttons are actually the same
	number of pixels high, but they look taller on a 12-inch screen because the
	pixel density is lower.
