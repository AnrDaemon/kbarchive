## Q99058: PC Forms: Telephone Note Accelerator Keys

{% raw %}

	Article: Q99058
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:1.0,3.0,3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 19-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Electronic Forms Designer, version 1.0, used with:
	   - Microsoft Mail for PC Networks, versions 3.0, 3.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	On the Telephone Note E-form included with version of 1.0 of Microsoft
	Electronic Forms Designer, there is only one accelerator key in the group of 3-D
	check boxes above the Comments field. The option that accepts an accelerator key
	is the Telephoned option.
	
	CAUSE
	=====
	
	This is by product design.
	
	RESOLUTION
	==========
	
	You can assign accelerator keys to the other 3-D check boxes available, and then
	recompile the form.
