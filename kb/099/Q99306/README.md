## Q99306: PC Forms: Err Msg: Another Application Refused the Request

	Article: Q99306
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:1.0,3.0,3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Electronic Forms Designer, version 1.0 
	- Microsoft Mail for PC Networks, versions 3.0, 3.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to Exit and Sign Out from version 3.0 or 3.2 of Microsoft Mail for
	Windows or from version 1.0 of Microsoft Schedule+ for Windows with a form
	created using version 1.0 of Microsoft Electronic Forms Designer, the following
	error message appears:
	
	  Another Application Refused the Request.
	
	CAUSE
	=====
	
	This error is generated by Microsoft Mail.
	
	RESOLUTION
	==========
	
	Close all open electronic forms before you Exit and Sign Out of Mail.
