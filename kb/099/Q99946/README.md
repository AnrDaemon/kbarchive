## Q99946: PC WRmt: Available Modem Scripts

	Article: Q99946
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 28-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Remote for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Script files developed for version 3.0 of Microsoft Mail Remote for MS-DOS
	should work unmodified with version 3.2 of Microsoft Mail Remote for Windows. If
	a script does not work correctly, there may be a timing problem in the script.
	If your modem does not appear on the list below, try using the "US-Generic
	modem" script (for example, this can be used for Compaq 9600 internal modems).
	
	To edit or create a custom script, contact your mail administrator. Modem script
	source files are in text form and found in files with the type .MDM. Once you
	edit the source script, you must compile it before it can be used by Mail Remote
	for Windows. Your mail administrator has the script compiler (SCRCOMP.EXE) you
	must use.
	
	The following is a list of the script files included with Mail Remote for
	Windows:
	
	General Purpose Modem Script Files
	----------------------------------
	
	- Null modem NULMODM.SCR
	
	- Generic 1200 and 2400 bps modems DEFAULT.SCR

	----

	script (for example, it can be used for Compaq 9600 internal modems).
	
	The Hayes V-series Smart 9600 script also supports the Compaq internal 9600
	modem.
