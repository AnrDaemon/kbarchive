## Q99009: Use Asynchronous Negotiation Only with Toshiba 3401

	Article: Q99009
	Product(s): Microsoft LAN Manager
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	The Toshiba 3401 CD-ROM drive has a bug in synchronous negotiation. If you want
	to use it as a CD controller, use asynchronous negotiation.
	
	Additional query words: 2.00 2.0 2.10 2.1 2.10a 2.1a 2.20 2.2
