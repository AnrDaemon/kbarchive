## Q99011: Encarta Err Msg: This List Is Corrupt!

	Article: Q99011
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1993 editions,1995 editions,2.0; WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 26-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia 
	- Microsoft Encarta 96 Encyclopedia for Windows 
	- Microsoft Multimedia Viewer, version 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you start Encarta, the following error message may occur:
	
	  This list is corrupt!
	
	This error message is caused by a corruption in the ENCARTA.ANN file. The
	ENCARTA.ANN file contains the annotation information for Encarta.
	
	NOTE: In Encarta 1995, the file is named ENCART95.ANN.
	
	In Encarta 1996, the file is named ENCART96.ANN.
	
	RESOLUTION
	==========
	
	To eliminate this problem, follow these steps:
	
	1. In Encarta 1993, delete the ENCARTA.ANN file from the WINDOWS directory.
	
	  In Encarta 1995, delete the ENCART95.ANN file from the WINDOWS directory.
	
	  In Encarta 1996, delete the ENCART96.ANN file from the Encarta directory.
	
	2. Restart Encarta.
	
	When Encarta restarts, Encarta 1993 rebuilds the ENCARTA.ANN file, Encarta 1995
	rebuilds ENCART95.ANN, and Encarta 1996 rebuilds ENCART96.ANN.
