## Q99096: ERR: Variable &quot;XXXXXXXXXXX&quot; Not Found

	Article: Q99096
	Product(s): Microsoft Fox Miscellaneous Products
	Version(s): MACINTOSH:2.01
	Operating System(s): 
	Keyword(s): 
	Last Modified: 23-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft FoxBASE+ for Macintosh, version 2.01 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	FoxBASE+/Mac will not accept a variable longer than 11 characters. If this
	maximum length limit is exceeded, the following error message will be
	displayed:
	
	  Variable 'XXXXXXXXXXX' Not Found.
	
	NOTE: The error message will display only 11 characters of the variable name.
	
	Additional query words: 2.01 errmsg err msg
