## Q99581: Mac Adm: Incorrect Country Code May Be Displayed

	Article: Q99581
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 24-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, version 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Mail Network Administrator utility for version 3.1 of Microsoft Mail for
	AppleTalk Networks may display the incorrect Country Code on the Global Server
	Report regarding servers of other languages. This only affects multi-national
	networks.
	
	This problem has been corrected in version 3.1a of Microsoft Mail for AppleTalk
	Networks.
	
	Additional query words: 3.10
