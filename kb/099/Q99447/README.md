## Q99447: Password Protection Option Is Not Saved in MWBACKUP SET File

	Article: Q99447
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.0,6.2,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 26-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.0, 6.2, 6.22 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you save your configuration options in Microsoft Backup for Windows, the
	Password Protection option is not saved.
	
	You must reselect it each time you start Microsoft Backup for Windows.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Backup for Windows. We
	are researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
	
	
	Additional query words: MWBackup 6.00 msbackup back up 6.20
