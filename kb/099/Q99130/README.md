## Q99130: DocErr: IsDomainMasterBrowser Not Valid Key in Windows NT 3.5

	Article: Q99130
	Product(s): Windows for Workgroups and Windows NT Networking Issues
	Version(s): 3.5,3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Windows NT Resource Kit, versions 3.5, 3.51 
	-------------------------------------------------------------------------------
	
	On page 491, the Windows NT version 3.5x Resource Guide incorrectly
	documents one of the Browser Service Entries as IsDomainMasterBrowser.
	This same error appears on page 133 and 134 of the Windows NT 4.0
	Networking Guide. This is the correct key for Windows NT 3.1. The
	correct key for Windows NT 3.5x and 4.0 is IsDomainMaster.
