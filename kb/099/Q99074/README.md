## Q99074: Only IBM/National Semiconductor PCMCIA Cards Supported

	Article: Q99074
	Product(s): Microsoft LAN Manager
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	The IBM and National Semiconductor cards are the only PCMCIA Ethernet cards that
	have been certified by our MCL labs and that Microsoft has drivers for. Other
	cards MAY work, if the manufacturer provides an NDIS driver, but the
	configuration will be UNSUPPORTED.
