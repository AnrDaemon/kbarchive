## Q99257: PC WRmt: Multiple MSMAIL.INI files

	Article: Q99257
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Remote for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Do not store the MSMAIL.INI file in the same directory as the MSMAIL.EXE file
	unless you use the Windows directory for both.
	
	If you store these two files in the same directory (a directory other than the
	Windows directory), version 3.2 of Microsoft Mail Remote for Windows will not
	find the correct .INI settings.
	
	Additional query words: 3.20
