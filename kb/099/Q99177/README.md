## Q99177: Bookshelf: QuicKeys Dialog Box Does Not Include OK and Cancel

	Article: Q99177
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1993 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for Windows 1993 edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The QuicKeys dialog box contains only a Close button, instead of OK and Cancel
	buttons.
	
	This is because the QuicKeys dialog box is a direct interface with the driver,
	and the driver doesn't offer the option of canceling. Choosing Close is the same
	as choosing OK.
	
	Additional query words: multimedia multi media multi-media quickkeys quick keys missing buttons
