## Q190007: Location of Instructions on How to Debug ISAPI Extensions

	Article: Q190007
	Product(s): Internet Information Server
	Version(s): WINNT:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 04-AUG-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Internet Information Server 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Documentation on how to debug ISAPI extensions is provided in the Windows NT 4.0
	Option Pack Documentation, in the "Establishing a Debugging Environment" topic.
	
	To navigate to this topic, open the following in the documentation's table of
	contents:
	
	   - Microsoft Internet Information Server
	      - Advanced Web Application Development
	         - Debugging Your Application
	            - Establishing a Debugging Environment
	
	MORE INFORMATION
	================
	
	If the Internet Information Server (IIS) Software Developer's Kit (SDK)
	documentation subcomponent was not selected during installation, the
	"Establishing a Debugging Environment" topic will not be available. Although the
	topic is listed in the table of contents, attempting to view it displays the
	text "File Not Found" in the documentation's Subject pane.
	
	For additional information, see the following article in the Microsoft Knowledge
	Base:
	
	  Q184540 : "File Not Found" Error Displayed in Product Documentation
