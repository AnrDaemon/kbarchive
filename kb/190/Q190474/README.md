## Q190474: WD97: International, Localized Versions of Word for Windows

	Article: Q190474
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------

	  French                   Windows 95/98   97        Available
	  German                   Windows 95/98   97        Available
	  Greek                    Windows 95/98   97        Available
	  Hebrew                   Windows 95/98   97        Available
	  Hungarian                Windows 95/98   97        Available
	  Italian                  Windows 95/98   97        Available
	  Japanese                 Windows 95/98   97        Available
	  Korean (Hanguel)         Windows 95/98   97        Available
	  Norwegian                Windows 95/98   97        Available
	  Polish                   Windows 95/98   97        Available
	  Portuguese               Windows 95/98   97        Available
