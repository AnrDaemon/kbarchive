## Q190795: DOCERR: cAlias Property Missing in _login Class

{% raw %}

	Article: Q190795
	Product(s): Microsoft FoxPro
	Version(s): WINDOWS:6.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual FoxPro for Windows, version 6.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	According to the Visual FoxPro 6.0 help file, the cAlias Property in the
	Password Dialog Box can be used to specify the alias of the password table.
	However, when the _login (Password Dialog Box) class in the _dialogs.vcx class
	library is open in the class designer, there is no reference to the cAlias
	Property in Properties page of the class.
	
	This is a documentation error and the reference of the cAlias in the help file is
	incorrectly documented.
