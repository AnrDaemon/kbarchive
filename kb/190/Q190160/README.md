## Q190160: Select &amp; International Versions of Terminal Server Deployment

{% raw %}

	Article: Q190160
	Product(s): Microsoft Windows NT
	Version(s): WinNT:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-DEC-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0, Terminal Server Edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The select and international English versions of Terminal Server include
	incorrect licensing information. The licensing information states that the
	products are "only for deployment in North America." This is incorrect. The
	Select and international English versions can be deployed outside of North
	America.
