## Q190707: XADM: Where to Find Exchange Server 4.0 Service Pack 4

	Article: Q190707
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 02-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 4.0 
	-------------------------------------------------------------------------------
	
	On the TechNet Exchange Server 4.0 Service Pack disc (the most recent
	version) in the Exch4sp4 folder is a Server folder. In that folder is a
	Readme.txt file that says the following:
	
	  All of Microsoft Exchange 4.0 service pack 4 server files have been
	  superceded by those in service pack 5. You can find the updated server
	  files in the Exch4sp5 folder of this CD.
	
	Exchange Server 4.0 Service Pack 4 can be found in either of the following
	two places:
	
	On the TechNet Exchange Server Service Packs disc ( June 97).
	
	  -or-
	
	On ftp.microsoft.com/bussys/exchange/exchange-
	public/fixes/eng/exchg4.0/sp4/
