## Q281327: Casino: Black Screen When You Press ALT+TAB to Return to Casino

	Article: Q281327
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Casino 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you press ALT+TAB to return to the Microsoft Casino game screen, the screen
	may turn black.
	
	CAUSE
	=====
	
	This behavior can occur if a Diamond Viper II video adapter is installed in your
	computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, press ALT+TAB again to minimize Casino, right-click the
	Casino button on the taskbar, click Close, and then restart Casino.
	
	To prevent this issue from occurring, download and install the latest video
	driver for your video adapter from the following S3 Web site:
	
	  http://www.s3.com
