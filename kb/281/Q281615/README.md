## Q281615: Combat Flight Simulator 2: Frame Rate Drops with Tool Tips

{% raw %}

	Article: Q281615
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbhw kbimu kbHardware
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use tool tips in Combat Flight Simulator 2, you may experience slow
	performance or a drop in the frame rate in the game.
	
	CAUSE
	=====
	
	This behavior can occur if a 3dfx Voodoo 5 video adapter is installed in your
	computer.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Combat Flight
	Simulator 2.
