## Q281381: Error Message &quot;STOP 0xA&quot; While Using Novell Client Version 4.7

	Article: Q281381
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbnetwork
	Last Modified: 08-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0, Terminal Server Edition 
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use a Windows NT-based computer that is running Novell Netware Client
	version 4.7, you may receive the following error message:
	
	  STOP 0x0000000A (0x00000063, 0x0000001c, 0x00000000, 0x8011B868)
	
	NOTE: The last parameter might vary depending on the version of Windows NT that
	you are running.
	
	CAUSE
	=====
	
	This error message can be caused by version 4.7 of the Novell Netware Client
	software.
	
	RESOLUTION
	==========
	
	To resolve this issue, either contact Novell for an updated version of Novell
	Netware Client or remove Novell Netware Client from the computer.
