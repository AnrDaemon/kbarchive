## Q281667: Links 2001: Distance from Aiming Marker to Pin Shown Incorrectly

	Article: Q281667
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 14-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Links 2001 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you move the aiming marker to one side of the hole or the other, it may
	show an incorrect distance between the marker and the hole.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
	
	MORE INFORMATION
	================
	
	This problem can occur when you are putting or chipping.
