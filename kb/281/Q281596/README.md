## Q281596: Fighter Ace II: No Training Demos Available

	Article: Q281596
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Fighter Ace II, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	No Training Demos are available on the Training tab.
	
	CAUSE
	=====
	
	Microsoft removed the Training Demos feature from Microsoft Fighter Ace II prior
	to release; however, the text for this feature remains in the user interface.
