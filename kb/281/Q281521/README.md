## Q281521: How to change passwords in MSN Companion

{% raw %}

	Article: Q281521
	Product(s): The Microsoft Network
	Version(s): 1.0,2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, versions 1.0, 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The user is not satisfied with a chosen password and wants to change it.
	
	CAUSE
	=====
	
	The user simply prefers to have a different password.
	
	RESOLUTION
	==========
	
	1. From the Start page, click the settings link.
	
	2. Click the password link.
	
	3. Enter the current password to verify, and then click the continue button.
	
	4. On the next page, enter the new password, and then enter it again to confirm.
	
	5. Click the done button.
