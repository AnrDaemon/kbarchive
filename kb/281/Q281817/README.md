## Q281817: Expedia Streets and Trips 2000: How to Submit Map Feedback

{% raw %}

	Article: Q281817
	Product(s): Microsoft Automap
	Version(s): 
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 27-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Expedia Streets and Trips 2000 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to submit feedback to Microsoft about the map data in
	Microsoft Expedia Streets and Trips 2000.
	
	MORE INFORMATION
	================
	
	If you find any errors in the map or map data that is included with Expedia
	Streets and Trips 2000, use the following steps to submit feedback to
	Microsoft:
	
	1. On the File menu, click Help.
	
	2. Click "Feedback on the Map".
	
	3. Follow the instructions in the Feedback Wizard to submit the data to
	  Microsoft.
	
	If you have a suggestion for a feature that you would like to see in future
	editions of Streets and Trips, or if you have a comment about the current
	version of Streets and Trips, please send them to
	
	  Microsoft Streets & Trips Program Manager
	  Microsoft Corporation
	  One Microsoft Way
	  Redmond, WA 98052-6399
	
	or visit the following Microsoft Web site:
	
	  http://www.microsoft.com/mswish
