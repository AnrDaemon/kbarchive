## Q93564: PC Ext: Err Msg: Error on Log File Open

{% raw %}

	Article: Q93564
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, version 3.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In version 3.0 of Microsoft Mail for PC Networks, when you redirect the output
	from the External Mail program (EXTERNAL.EXE) log file to a floppy disk drive,
	you may see the following two error messages:
	
	  Critical Error has occurred.
	  Error on log file open.
	
	CAUSE
	=====
	
	The disk in the floppy disk drive is write-protected or there is no disk in the
	drive.
	
	RESOLUTION
	==========
	
	Remove write-protection from the disk or load a disk that is not write-
	protected in the floppy disk drive, and then restart EXTERNAL.EXE.
