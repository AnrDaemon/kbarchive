## Q93266: Identifying Third-Party Mouse by FCC ID Number

	Article: Q93266
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): MS-DOS:3.x,4.x,5.x,6.0,6.2,6.21,6.22; WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	- Microsoft MS-DOS operating system versions 3.1, 3.2, 3.21, 3.3, 3.3a, 4.0, 4.01, 5.0, 5.0a, 6.0, 6.2, 6.21, 6.22 
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following is a list of FCC ID (first three digits) numbers and the mice that
	use those FCC ID numbers. This can help in determining which Windows or MS-DOS
	mouse driver to use.
	
	The FCC ID number can usually be found on the bottom of the mouse.
	
	MORE INFORMATION
	================
	
	FCC ID #               Mouse
	-------------------------------
	
	ANO                    IBM
	BCG                    Apple
	BKL                    Kraft
	B94                    Hewlett-Packard
	C3K                    Microsoft
	CWE                    Mouse Systems
	DZL                    Logitech
	E5Y                    Diamond Flower
	E6Q                    Znix
	EMJ                    Cutie

	F4Z                    AMBRA

	X31                    Z-NIX Inc.
