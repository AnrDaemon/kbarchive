## Q93239: PC DB: Mail Message (.MAI) File Format

	Article: Q93239
	Product(s): Microsoft Mail For PC Networks
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	The mail message (.MAI) file format in Microsoft Mail versions 2.1 and 3.0 is
	based on the Federal Information Processing Systems (FIPS) message format
	described in FIPS PUB 98. This format is a subset of the CCITT X.400 standard
	for interpersonal messaging.
	
	FIPS refers to the format used to store the raw data files. Vendor-defined field
	refers to an individual element of the message. For example, TO:, FROM:, and
	ATTACHMENT:, are all vendor-defined fields.
	
	Additional query words: 2.10 3.00
