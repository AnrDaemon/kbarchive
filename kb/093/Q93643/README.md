## Q93643: Sound System: Proofreader and Lotus 1-2-3 Compatibility

	Article: Q93643
	Product(s): Miscellaneous Windows Products
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows Sound System, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Proofreader application included with version 1.0 of the Microsoft Windows
	Sound System may not function correctly with Lotus 1-2-3, version 1.1.
	Proofreader works with Lotus 1-2-3 version 1.10.01.
	
	After you install Windows Sound System, the following message may appear when you
	load Lotus 1-2-3 version 1.1, stating you should upgrade to a later version of
	Lotus 1-2-3:
	
	  123 for Windows Proofreader does not work with this version (1.10.00) of
	  Lotus 123 for Windows. An update is available from Lotus.
	
	MORE INFORMATION
	================
	
	Lotus Corp. is aware of this problem and will upgrade its users to Lotus 1-2-3,
	version 1.10.01, which works correctly with the Proofreader.
	
	To upgrade, contact Lotus Corp. Customer Service at (800) 343-5414 and ask for
	the Lotus "1.1 Update."
	
	To find out what version of Lotus you have, type the following in any shell of an
	open Lotus spreadsheet:
	
	  @info("release")
	
	For more information about the Windows Sound System, query on the following words
	in the Microsoft Knowledge Base:
	
	  windows and sound and system
	
	Additional query words: 1.00 123
