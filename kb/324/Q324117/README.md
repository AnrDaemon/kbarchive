## Q324117: Microsoft Computer Dictionary, Fifth Edition Comments and Correc

	Article: Q324117
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 06-JUN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Computer Dictionary, Fifth Edition, ISBN 0-7356-1495-4 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Microsoft Computer Dictionary, Fifth
	Edition, ISBN 0-7356-1495-4.
	
	The following topics are covered:
	
	- Page 631: Error In Numeric Equivalents Table
	
	MORE INFORMATION
	================
	
	Page 631: Error In Numeric Equivalents Table
	--------------------------------------------
	
	There is an error in the Numeric Equivalents table on page 631.
	
	Change:
	
	9  06  11  00001001
	
	To:
	
	9  09  11  00001001
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
