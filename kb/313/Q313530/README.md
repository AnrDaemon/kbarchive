	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft FrontPage 2000 Step by Step Courseware Expert Skills Color Class Pack ISBN 0-7356-0711-7 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Microsoft FrontPage 2000 Step by Step
	Courseware Expert Skills Color Class Pack, ISBN 0-7356-0711-7.
	
	The following topics are covered:
	
	- Page 1.11: Extras Folder Is Not Included On CD-ROM
	
	MORE INFORMATION
	================
	
	Page 1.11: Extras Folder Is Not Included On CD-ROM
	--------------------------------------------------
	
	Page 1.11 makes reference to a Climbing.htm file in the FrontPage Expert
	Practice\Extras folder. The Extras folder does not exist. Please disregard this
	reference.
