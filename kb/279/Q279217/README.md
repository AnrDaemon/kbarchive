## Q279217: MSN Companion unit shows &quot;MSN&quot; splash screen and never boots

	Article: Q279217
	Product(s): The Microsoft Network
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you turn on the MSN Companion unit, it only shows the white "MSN" splash
	screen and hangs there permanently.
	
	CAUSE
	=====
	
	There is some low-level problem with the device or the Operating System.
	
	RESOLUTION
	==========
	
	This is a serious problem. Contact MSN Companion Technical Support at (888)
	386-6497.
	
	With the assistance of Technical Support failsafe the device. Failsafe will use
	the phone line for approximately 1 hour. For instuction to failsafe the unit see
	<A0><A0><A0><A0>Q 274256 - MSN Companion Troubleshooting Tool FAILSAFE
	
	If failsafe doesn't fix the problem, then there is a problem with the hardware
	and the device needs to be replaced. Contact the manufacturer for a replacement.
