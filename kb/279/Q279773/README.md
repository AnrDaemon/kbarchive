## Q279773: Combat Flight Simulator 2: Cannot Set Precipitation Type to Snow

	Article: Q279773
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbdocerr kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Microsoft Combat Flight Simulator 2 online Help incorrectly states that you can
	set the Precipitation Type to Snow.
	
	Snow effects are not available in Combat Flight Simulator 2.
	
	Additional query words: 1.00 msgame combatfs cfs2
