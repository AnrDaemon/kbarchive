## Q279351: Connectivity: MSN Companion only uses dial-up modem

	Article: Q279351
	Product(s): The Microsoft Network
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Can MSN Companion be used to access the Internet over a LAN, DSL, Ethernet.
	
	MORE INFORMATION
	================
	
	The only connection available in MSN Companion Version 1.0 is the built-in 56k
	modem.
	
	The MSN Companion team is working on high-speed connections for future versions.
	Some Compaq devices may have Ethernet ports on the back, but there is no
	software onboard to support these ports.
