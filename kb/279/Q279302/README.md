## Q279302: CFS2: Pilot's Manual Incorrectly States You Can Drop Torpedoes

{% raw %}

	Article: Q279302
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbdocerr kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Combat Flight Simulator 2 Pilot's Manual incorrectly states that
	you can drop torpedoes.
	
	MORE INFORMATION
	================
	
	Only computer-controlled aircraft and boats can drop torpedoes in Combat Flight
	Simulator 2.
	
	The following aircraft and boats can drop torpedoes in Combat Flight Simulator
	2:
	
	- Douglas TBD-1 Devastator
	
	- Grumman TBF Avenger
	
	- Nakajima B5N2 (Kate)
	
	- Tidori Class Torpedo Boat
