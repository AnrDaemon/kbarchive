## Q279294: Combat Flight Simulator 2: How to Start Your Aircraft Engine

	Article: Q279294
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbui kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to start your aircraft engine in Microsoft Combat
	Flight Simulator 2.
	
	MORE INFORMATION
	================
	
	To start your aircraft engine in Combat Flight Simulator 2, use one of the
	following methods:
	
	Automatically Start the Engine
	------------------------------
	
	To automatically start the engine, press and hold down the E key on your keyboard
	until the engine starts.
	
	The Propeller Control knob, the Mixture Control lever, the Throttle Control
	lever, the Master Battery Switch (MSTR) control, the Engine Ignition &
	Magnetos Switch control, and the Engine Start Switch control automatically move
	to their proper positions.
	
	After the engine starts, the Engine Start Switch control automatically moves to
	the Off position.
	
	Manually Start the Engine
	-------------------------
	
	To manually start the engine, follow these steps:
	
	1. Press the F5 key to open the engine controls window.
	
	2. Press SHIFT+EQUAL SIGN to push the Propeller Control knob to the "Max.
	  r.p.m." (100%) position.
	
	3. Press SHIFT+4 to move the Mixture Control lever to the Rich (100%) position.
	
	4. Press the EQUAL SIGN to move the Throttle Control lever two notches above the
	  Close position (the 1% position).
	
	  NOTE: To move the Throttle Control lever down a notch, press the MINUS SIGN.
	
	5. Click to move the Master Battery Switch (MSTR) control to the On position.
	
	6. Press the F5 key to close the engine controls window.
	
	7. If your aircraft has a Engine Ignition & Magnetos Switch control, click
	  the Engine Ignition & Magnetos Switch control until the control is set to
	  the Both (Both Magnetos On) position.
	
	8. To engage the starter, click the Engine Start Switch control.
	
	  Hold the pointing device button down until the engine starts, and then release
	  the button.
	
	NOTE: You cannot use the Engine Start Switch control to turn off the engine.
	
	Manually Start a Multi-Engine Aircraft
	--------------------------------------
	
	In a multi-engine aircraft such as the P-38, you must select each engine to start
	it manually:
	
	1. Press CTRL+S to enter Select Engine mode.
	
	2. To select engine 1, press SHIFT+1.
	
	  To select engine 2, press SHIFT+2.
	
	3. To increase the throttle on the selected engine, press the EQUAL SIGN.
	
	  To decrease the throttle on the selected engine, press the MINUS SIGN.
	
	4. To exit Select Engine mode, press CTRL+S.
