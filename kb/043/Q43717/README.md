## Q43717: Flight Sim Err Msg: FOREIGN.FS3 File Not Found

	Article: Q43717
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	If Flight Simulator Version 3.00 or 4.00 is started from a directory other than
	the Flight Simulator directory (i.e., the one containing the Flight Simulator
	program files), the following message is displayed, and the computer may hang:
	
	  FOREIGN.FS3 file not found.
	
	To correct this problem, start the program when the Flight Simulator directory is
	the current directory.
