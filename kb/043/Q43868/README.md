## Q43868: Dip Switch Settings for Epson 286e

	Article: Q43868
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:2.x,3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 2.03, 2.1, 2.11, 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following dip switch settings have been tested and work correctly on the
	Epson 286e when printing from Windows-based applications. These settings are
	also for use with Microsoft Windows, Microsoft Excel, and Microsoft Pageview.
	
	The settings are as follows:
	
	  Switch    Setting
	
	  SW1:      1 off (dip switch down is off)
	            2 off
	            3 off
	            4 off
	            5 off
	            6 on  (dip switch up)
	            7 on
	            8 on
	
	  SW2:      1 off
	            2 off
	            3 off
	            4 off
	
	MORE INFORMATION
	================
	
	These dip switches are located on the back of the Epson 286e. These settings
	conform to the settings listed when the Info menu option of the Epson 9-pin
	printer driver is selected from the Control Panel's Setup/Printer menu. The
	listed settings are:
	
	  Skip-Over-Perf OFF
	  Auto LF OFF
