## Q139749: Program Starts During Microsoft Plus! Setup

{% raw %}

	Article: Q139749
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	- Microsoft Plus! for Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	While you are installing Microsoft Plus!, a program may seem to start
	automatically.
	
	CAUSE
	=====
	
	This behavior can occur if you press the ENTER key while the group converter is
	converting a program. If this occurs, the program that is being converted may
	start.
	
	RESOLUTION
	==========
	
	Do not press ENTER while Microsoft Plus! Setup is running, except when you are
	prompted to.
	
	Additional query words: application grpconv.exe
