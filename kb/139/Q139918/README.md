## Q139918: Bookshelf 95: Ctl3dv2.dll Conflicts with Corel Gallery II

	Article: Q139918
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1995 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for Windows 1995 edition 
	- Microsoft Bookshelf '95 for Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you install Bookshelf for Windows or Bookshelf '95 for Windows 95, Corel
	Corporation's Corel Gallery II generates error messages.
	
	CAUSE
	=====
	
	Bookshelf installs a version of Ctl3dv2.dll that is incompatible with Corel
	Gallery II.
	
	RESOLUTION
	==========
	
	To resolve this problem, try the following:
	
	- Replace Ctl3dv2.dll located in the C:\Windows\System folder with the copy of
	  this file supplied with Corel Gallery II.
	
	For more information about how to perform this task in Windows, see your Windows
	printed documentation or online Help.
	
	MORE INFORMATION
	================
	
	The version of Ctl3dv2.dll that ships with Corel Gallery II has a uncompressed
	file size of 21,648 bytes and a date of 8/31/93.
	
	The version of Ctl3dv2.dll that ships with Bookshelf has a file size of 26,832
	bytes and a date of 11/30/94.
