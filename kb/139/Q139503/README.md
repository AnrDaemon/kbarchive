## Q139503: Alarm Will Not Activate on an Archive File with Reminders

{% raw %}

	Article: Q139503
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-MAY-2001
	
	7.00
	WINDOWS
	kbusage
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you open an archived Microsoft Schedule+ .SCD file and create an appointment,
	the archive file will allow you to set a reminder with alarms. Although the
	alarm icon shows an appointment, the alarm will not activate.
	
	This is by design.
