## Q139670: Setup Does Not Detect Creative Labs Wave Blaster

	Article: Q139670
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you install Windows 95, your Creative Labs Wave Blaster (a MIDI
	daughterboard device) may not function properly.
	
	CAUSE
	=====
	
	Windows 95 Setup does not automatically detect and configure this device.
	
	RESOLUTION
	==========
	
	To configure Windows 95 to recognize the Wave Blaster, follow these steps:
	
	1. In Control Panel, double-click the Multimedia icon.
	
	2. On the MIDI tab, click the Single Instrument option.
	
	3. In the list of instruments, click MIDI For External MIDI Port.
	
	4. Click OK.
