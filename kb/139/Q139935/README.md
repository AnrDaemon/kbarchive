## Q139935: Unable to Print to Shared HP DeskJet 1200C Printer

	Article: Q139935
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to print from Microsoft Excel version 5.0a to a Hewlett- Packard
	(HP) DeskJet 1200C or DeskJet 1200C PostScript printer, you may receive the
	following error message:
	
	  Error printing to HP1200
	
	CAUSE
	=====
	
	This problem can occur if the following conditions exist:
	
	- You are using the printer drivers for the HP DeskJet 1200C printers that are
	  included with Windows 95.
	
	  The following drivers for the HP DeskJet 1200C printers are included with
	  Windows 95:
	
	  - HP DeskJet 1200C (MS)
	  - HP DeskJet 1200C/PS
	
	- The HP DeskJet 1200C printer you are using is a network printer connected to
	  a computer running Windows for Workgroups version 3.1x.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Windows 95. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
