## Q139742: Cannot Print Hard Copy of Billing Statement

	Article: Q139742
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.0,1.05,1.2,1.3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.0, 1.05, 1.2, 1.3 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you view your billing statement on MSN, The Microsoft Network, it is not
	possible to print a hard copy of the statement.
	
	RESOLUTION
	==========
	
	To print a hard copy of your statement, use the following workaround:
	
	1. Open your billing statement.
	
	2. Press ALT+PRINT SCREEN to copy the screen.
	
	3. Open Microsoft Paint (or any other graphics program).
	
	4. On the Edit menu, click Paste.
	
	5. Print the copy of the statement from Paint.
