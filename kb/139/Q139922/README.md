## Q139922: Encarta 1995: Word Processor Starts Only Once

{% raw %}

	Article: Q139922
	Product(s): Microsoft Home Kids Products
	Version(s): 1995 edition
	Operating System(s): 
	Keyword(s): kbbuglist
	Last Modified: 26-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia for Macintosh 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The word processor option in Microsoft Encarta 1995 only starts one time each
	session. If you started the option during the current session of Encarta and
	then closed it, the word processor will not restart.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem Microsoft Encarta for Windows, 1995
	edition. We are researching this problem and will post new information here in
	the Microsoft Knowledge Base as it becomes available.
	
	This problem does not occur in Encarta Encyclopedia for Windows, 1996 edition.
	
	WORKAROUND
	==========
	
	To run your word processing program a second time, you must exit Microsoft
	Encarta 95 and then reopen it.
