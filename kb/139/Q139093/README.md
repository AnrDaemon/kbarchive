## Q139093: Encarta 1996: Computer Hangs When Playing Video Clips

	SYMPTOMS
	========
	
	When you play an Video clip (.avi file), the computer stops responding (hangs)
	and must be restarted.
	
	This occurs on computers using the Advance Logic 2301/1301 local bus video card.
	
	CAUSE
	=====
	
	The Advance Logic 2301/1301 local bus video card installs an Advance Logic Video
	icon in its own program group. The program has an option called Video Wizard.
	The problem occurs when this option uses the full- motion video feature.
	
	RESOLUTION
	==========
	
	Turn off the Video Wizard option using the steps below.
	
	Windows 3.x
	-----------
	
	1. Exit the Microsoft multimedia program.
	
	2. Double-click the Advance Logic Video icon.
	
	3. Clear the Video Wizard check box.
	
	4. Click close.
