## Q139497: Bitmaps and Objects Removed from Messages in No-Attachment BBS

	Article: Q139497
	Product(s): The Microsoft Network
	Version(s): 1.0,1.05,1.2,1.3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 20-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.0, 1.05, 1.2, 1.3 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Bulletin boards that do not permit attachments will let you insert bitmaps and
	objects into messages. However, the bitmaps and objects will be removed from the
	message before it is posted.
