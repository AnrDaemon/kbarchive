## Q139455: Obtaining the Designed For Microsoft Windows 95 Catalog

{% raw %}

	Article: Q139455
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Designed For Microsoft Windows 95 Product Catalog is currently available on
	the Microsoft World Wide Web site located at:
	
	  http://www.microsoft.com
	
	This catalog contains a list of products that have received the Designed For
	Windows 95 logo. It also contains information about the requirements that must
	be met for a product to receive the logo, and about how vendors can apply for
	the logo to appear on their products.
	
	MORE INFORMATION
	================
	
	The Designed For Microsoft Windows 95 Product Catalog contains a list of
	hardware and software products that have received the Designed For Windows 95
	logo. This logo indicates that the product is completely compatible with Windows
	95, and is designed to take advantage of the features and advanced technology
	built into Windows 95.
	
	To receive the Designed For Microsoft Windows 95 logo, software products must be
	32-bit Windows-based programs. They must also feature the enhanced user
	interface of Windows 95, support for long file names, automated installation,
	and uninstall capability. Hardware products that receive the Designed For
	Microsoft Windows 95 logo must support Plug and Play in Windows 95.
	
	For information about receiving the Designed For Windows 95 logo for your
	product, see the following Microsoft Web page:
	
	  http://www.microsoft.com/windows95/info/hardwareshow.htm
	
	To view a list of Designed For Windows 95 products, see the following article in
	the Microsoft Knowledge Base to access the Windows 95 Software Compatibility
	List:
	
	  Q136660 Windows 95 Software Compatibility List
