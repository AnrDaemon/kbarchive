## Q139345: Problems with Spanish Keyboard and Learn to Speak Spanish

{% raw %}

	Article: Q139345
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95 appscomp kbAppCompatibility
	Last Modified: 28-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the Spanish keyboard layout with Learn to Speak Spanish version
	5.0, the characters you type may not be displayed properly. Incorrect characters
	may be displayed, or extra characters may be added to the characters you type.
	
	CAUSE
	=====
	
	Learn to Speak Spanish version 5.0 is not completely compatible with Windows 95.
	
	RESOLUTION
	==========
	
	To resolve this problem, contact the program's manufacturer to inquire about
	obtaining a newer version of the program that corrects the problem described in
	this article.
	
	To work around this problem with Learn to Speak Spanish version 5.0, use a
	keyboard layout other than the Spanish keyboard layout.
	
	
	MORE INFORMATION
	================
	
	Learn to Speak Spanish is manufactured by HyperGlot Software, a vendor
	independent of Microsoft; we make no warranty, implied or otherwise, regarding
	this product's performance or reliability.
