## Q139066: DOC: Incorrect Reference to TabWidth Property in Help

	Article: Q139066
	Product(s): Microsoft FoxPro
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): kbdocfix
	Last Modified: 11-FEB-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual FoxPro for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	On page 805 of Visual FoxPro 3.0 Language Reference under the Tabs Property,
	there is an incorrect reference to the TabWidth property in the "See Also"
	section and the "Remark" paragraph.
	
	In the online Help, you can also see a incorrect reference to the TabWidth
	property in the "Remark" paragraph of the Tabs Property topic.
	
	This documentation error has been fixed in Visual FoxPro 5.0 for Windows.
	
	MORE INFORMATION
	================
	
	The TabWidth property does not exist in the current product.
	
	Additional query words: docerr VFoxWin 5.00
