## Q139178: Installing Video and SCSI Drivers on Toshiba T2450CT

	Article: Q139178
	Product(s): Microsoft Windows NT
	Version(s): 3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.51 
	- Microsoft Windows NT Server version 3.51 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article discusses installation of video and the built-in SCSI-2 adapter in
	the Toshiba T2450CT notebook computer.
	
	MORE INFORMATION
	================
	
	T2450CT users should use the "Western Digital Graphics Adapter" video driver.
	
	If NT does not automatically recognize the built-in SCSI-2 port, then the user
	should specify the Adaptec 151x/152x/AIC 6260/6360 driver.
