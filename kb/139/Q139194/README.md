## Q139194: Encarta 1994: Problems Printing to the Color StyleWriter 2400

	Article: Q139194
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1994 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 1994 The Complete Multimedia Encyclopedia 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to print from Encarta to a Color StyleWriter 2400 you receive the
	following error message:
	
	  There is a problem with your printer setup. Please use the Chooser to make
	  sure a printer has been selected.
	
	After you use Chooser to select the Color StyleWriter 2400 printer driver and the
	correct printer port, if you again try to print from Encarta you may receive the
	following message,
	
	  Error Type 11
	
	or, the system may stop responding (hang).
	
	This occurs if you are using version 2.1.1 of the Color SW 2400 printer driver.
	
	RESOLUTION
	==========
	
	Change the Color StyleWriter 2400 printer driver to version 1.6.1. This older
	driver should resolve the problem.
	
	MORE INFORMATION
	================
	
	The symptoms occur with printer driver version 2.1.1. The problem does not occur
	with Microsoft Encarta for the Macintosh 1995 or 1996.
	
	Version 1.6.1 of the Color StyleWriter 2400 printer driver can be obtained
	directly from Apple Technical Support.
	
	Apple Technical Support: 1(800) SOS-APPL, or 1(800)767-2775.
