## Q139198: How to Run the Tour Outside of Mastering Visual FoxPro Program

	Article: Q139198
	Product(s): Microsoft Mastering Series
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbmm
	Last Modified: 22-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Mastering Microsoft Visual FoxPro 3.0, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you are having trouble running the Tour inside the Mastering Microsoft Visual
	FoxPro program, you can still view the Tour by running the Tour's Video for
	Windows file directly from the compact disc.
	
	MORE INFORMATION
	================
	
	The Tour Video for Windows file is located in the \Mvfp\Tour directory of the
	compact disc. To view this file, start Media Player (or any other Video for
	Windows view), and open the Tour.avi file in the \Mvfp\Tour directory.
