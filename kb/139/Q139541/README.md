## Q139541: How WINS Static Entries Replicate to Partners

	Article: Q139541
	Product(s): Microsoft Windows NT
	Version(s): 3.5,3.51,4.0
	Operating System(s): 
	Keyword(s): kbnetwork
	Last Modified: 20-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server versions 3.5, 3.51, 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Static entries from a WINS Server do not show up in the Mappings\Static Mappings
	of its replication partners.
	
	Static entries in a WINS database are entered within Mappings\Static Mappings.
	After the entry is added to the local database of the owner, it replicates to
	its partners as a database entry. A WINS Static entry that replicates to a
	partner does not show up in the Mapping\Static Mappings box of its partners.
	
	Additional query words: prodnt static mapping does not replicate do
