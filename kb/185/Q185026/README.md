## Q185026: XADM: Local Message Delivery Not Subject To Size Restriction

{% raw %}

	Article: Q185026
	Product(s): Microsoft Exchange
	Version(s): winnt:4.0,5.0,5.5
	Operating System(s): 
	Keyword(s): exc4 exc5 exc55
	Last Modified: 12-FEB-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 4.0, 5.0, 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Administrators of Microsoft Exchange Server will find that they cannot place
	size restrictions on messages that are delivered locally.
	
	MORE INFORMATION
	================
	
	Messages sent between users homed on the same Exchange Server computer do not
	use the Message Transfer Agent (MTA) for delivery. Therefore, they are not
	subject to the options available for external message control.
