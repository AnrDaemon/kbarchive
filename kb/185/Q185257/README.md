## Q185257: How to Disable the Delete Confirmation Dialog Box

	Article: Q185257
	Product(s): Windows for Workgroups and Windows NT Networking Issues
	Version(s): WINNT:4.0
	Operating System(s): 
	Keyword(s): kbui win95 win98
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows 98 
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you delete a file in Windows, you may receive a message prompting you to
	confirm deleting the file. This article describes how to disable the Delete
	Confirmation dialog box.
	
	MORE INFORMATION
	================
	
	To disable the Delete Confirmation dialog box, use the following steps:
	
	1. Right-click Recycle Bin, and then click Properties.
	
	2. Click the Display Delete Confirmation Dialog Box check box to clear it, and
	  then click OK.
