## Q185984: WD97: Outlook Message Icons Remain Unopened with Routed Document

	Article: Q185984
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbinterop kbdta word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you route a document (on the File menu, point to Send To and click Routing
	Recipient) the icon for the mail message does not change from an unread message
	to a read message, after the recipient opens the message in Microsoft Outlook.
	Nor does it change even if the recipient edits the attached document.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
	
	MORE INFORMATION
	================
	
	For more information about routing documents, click Contents And Index on the
	Help menu, click the Index tab in Word Help, type the following text
	
	  routing documents
	
	and then double-click the selected text to go to the "Route a document" topic. If
	you are unable to find the information you need, ask the Office Assistant.
