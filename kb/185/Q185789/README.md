## Q185789: INFO: QLogic QLA1040 SCSI Miniport driver added to SP4

	Article: Q185789
	Product(s): Microsoft Windows NT
	Version(s): WinNT:4.0
	Operating System(s): 
	Keyword(s): kbFEA kbWinNT400sp4fea
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 SP4 
	- Microsoft Windows NT Server, Enterprise Edition version 4.0 SP4 
	- Microsoft Windows NT Workstation version 4.0 SP4 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Support for the QLA1040 Wide UltraSCSI Host Bus Adapter by QLogic Corporation,
	was not included in Service Pack 3. The driver for this adapter, Ql10wnt.sys,
	has been added to Windows NT Service Pack 4.
