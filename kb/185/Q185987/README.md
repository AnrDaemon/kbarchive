## Q185987: Virtual Globe 98: World Flights: Flight Does Not Stop

{% raw %}

	Article: Q185987
	Product(s): Microsoft Automap
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbui kbimu
	Last Modified: 26-AUG-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Virtual Globe, 1998 edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click Take A World Flight in Microsoft Encarta Virtual Globe 98, and
	then click one of the seven regions to start a new flight, if you click Stop
	(the red rectangle), forward motion stops, but sideways motion continues.
	
	RESOLUTION
	==========
	
	To work around this issue, click the viewing screen to stop the sideways motion
	after you first click Stop in a new world flight. You must repeat this process
	each time you start a new world flight.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Encarta Virtual Globe,
	1998 Edition.
