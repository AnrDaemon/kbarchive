## Q185530: WD97: Unexpected Behavior Paste Linking Ami Pro Document Object

{% raw %}

	Article: Q185530
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	
	SYMPTOMS
	========
	
	When you paste link Lotus Ami Pro 3.0 or 3.1 document objects from the Windows
	clipboard, you will receive the following error message:
	
	  Word cannot obtain the data for the AmiPro Document link.
	
	CAUSE
	=====
	
	Lotus Ami Pro 3.0 and 3.1 does not fully support OLE 2.0
	
	MORE INFORMATION
	================
	
	You will be able to paste link a Lotus Ami Pro object without an error message
	if you upgrade from Lotus Ami Pro 3.0 and 3.1 to Lotus Word Pro 96; however,
	Word does not directly support the import and export of Lotus Word Pro 96. For
	additional information about this supportability, please see the following
	article in the Microsoft Knowledge Base:
	
	  Q139792 WD: How to Convert Between Lotus Word Pro 96 and Word
	
	For information about how to contact Lotus Development Corporation, query in the
	Knowledge Base for one of the following article:
	
	  Q60781 Hardware and Software Third-Party Vendor Contact List, L-P
	
	
	Ami Pro is manufactured by Lotus Development Corp., a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
