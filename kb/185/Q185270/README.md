## Q185270: WD97: Paragraph Fails to Indent When Indent Applied with Ruler

	Article: Q185270
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you use the ruler to set paragraph indents in Word 97 for Windows, some
	paragraphs may not be indented on the first attempt. You can, however, indent
	the paragraphs on a second attempt.
	
	WORKAROUND
	==========
	
	To indent the paragraphs, use either of the following methods:
	
	- Apply the indent again using the ruler.
	
	  -or-
	
	- Click Paragraph on the Format menu, and apply the indent using the Paragraph
	  dialog box.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
