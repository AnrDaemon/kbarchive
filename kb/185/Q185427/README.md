## Q185427: WD97: Top of Text Truncated with Contemporary Report Template

	Article: Q185427
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you base your document on the Contemporary Report Template, the first line
	of text on the second and subsequent pages appears to be truncated; however, the
	text prints correctly.
	
	CAUSE
	=====
	
	This problem occurs when any of the following conditions is true:
	
	- Your document is based on the Contemporary Report Template.
	
	- You are in page layout view.
	
	- The text is on the first line of a page other than page 1.
	
	- The text is formatted with the Chapter Title style.
	
	WORKAROUND
	==========
	
	To view the text correctly, use one of the following methods.
	
	Method 1: Change the Line Spacing
	---------------------------------
	
	1. Place the insertion point in the affected text, and then click Paragraph on
	  the Format menu.
	
	2. Click the Indents and Spacing tab.
	
	3. In the spacing section, do one of the following:
	   - From the Line Spacing list, select At Least. In the At box, select 27 and
	     type a new font size, such as 16.
	
	  -or-
	
	   - From the Line Spacing list, select Single.
	
	4. Click OK.
	
	Method 2: Switch to Normal View
	-------------------------------
	
	To change to normal view, click Normal on the View menu.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
	
	This problem was corrected in Microsoft Word 2000.
