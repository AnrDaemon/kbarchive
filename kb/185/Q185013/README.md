## Q185013: XADM: Using # and ^ Symbols in Display Names

	Article: Q185013
	Product(s): Microsoft Exchange
	Version(s): 4.0,5.0,5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-MAR-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 4.0, 5.0, 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Display Names of objects within the Microsoft Exchange Server Administrator
	program may include up to 256 characters, including the number (#) and caret (^)
	symbols.
	
	When viewing the Directory Name, or the results of an LDAP query, of objects
	which include a # or ^ symbol, the symbols may be replaced with question marks
	(?).
	
	Additional query words: caret pound sign
