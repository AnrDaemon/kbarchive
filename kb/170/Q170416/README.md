## Q170416: ZAK Users Cannot Shut Down the Workstation

	Article: Q170416
	Product(s): Microsoft Windows NT
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbenv kbui
	Last Modified: 21-FEB-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Zero Administration Kit for Windows NT Workstation 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Zero Administration Kit (ZAK) Appstation and Taskstation users may be unable to
	shut down the Windows NT workstation.
	
	CAUSE
	=====
	
	This behavior can occur if the system administrator has used a system policy to
	remove the Everyone and Users groups from the list of those who can shut down
	the workstation.
	
	RESOLUTION
	==========
	
	Contact your system administrator.
