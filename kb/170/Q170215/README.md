## Q170215: XCON: Message Text Word-Wrap Broken in MS Exchange 5.0

{% raw %}

	Article: Q170215
	Product(s): Microsoft Exchange
	Version(s): 5.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 16-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	The "Message Text Word-Wrap" feature in the X.400 Connector does not work
	correctly in Microsoft Exchange Server 5.0.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server
	version 5.0. This problem was corrected in the latest Microsoft Exchange Server
	5.0 U.S. Service Pack. For information on obtaining the service pack, query on
	the following word in the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
	
	MORE INFORMATION
	================
	
	The problem only affects the first release of Microsoft Exchange Server 5.0 and
	does not apply to version 4.0.
