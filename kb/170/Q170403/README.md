## Q170403: XCON: Err Msg: Unable to Interrogate Database...

{% raw %}

	Article: Q170403
	Product(s): Microsoft Exchange
	Version(s): winnt:4.0,5.0,5.5
	Operating System(s): 
	Keyword(s): kbtshoot exc4 exc5 exc55
	Last Modified: 19-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 5.5, 4.0, 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you attempt to run MTACHECK /v /f from a command prompt, you might receive
	the following error:
	
	  Unable to interrogate the service control database.
	
	This error is reported when you are attempting to run MTACHECK remotely. MTACHECK
	must be run from the computer running the Microsoft Exchange Server.
	
	MORE INFORMATION
	================
	
	For more details on the MTACHECK command line utility, please see the following
	article in the Microsoft Knowledge Base:
	
	  Q148284 XCON: When and How to use the Mtacheck Utility
