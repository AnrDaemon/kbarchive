## Q170233: MDG:Purpose of the POBOX in Remote Mail

	Article: Q170233
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Remote for Windows, version 3.2 
	-------------------------------------------------------------------------------
	
	
	INFORMATION
	===========
	
	The Remote Mail Client for Windows contains a key/mbg pair of mailbag files
	called POBOX. This mailbag's function is to store and view only message headers
	when the "send all and retreive selected headers" option is enabled in the Mail,
	Session Setup option. When View, Headers is selected, the header window shows
	the contents of the Pobox.mbg file.
