## Q170877: WD97: Equation Editor Error: Elements That Are Not Available

	Article: Q170877
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbtool word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Word 97, when you double-click an equation that contains a horizontal brace
	character, the following error message appears:
	
	  This equation contains elements that are not available in Equation Editor.
	
	WORKAROUND
	==========
	
	To work around this problem, follow these steps to insert the Equation Editor
	object as an Enhanced Metafile:
	
	1. Open the document and click the Equation Editor Object.
	
	2. On the Edit menu, click Copy.
	
	3. On the Edit menu, click Paste Special, and then click Picture (Enhanced
	  Metafile).
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows.
	This problem was corrected in Microsoft Word 97 for Windows, Service Release 1.
	
	MORE INFORMATION
	================
	
	For more information about the Equation Editor symbols and templates, do the
	following:
	
	1. Start the Equation Editor.
	
	2. On the Help menu, click Equation Editor Help Topics.
	
	3. On the Contents tab, double-click Reference Information.
	
	4. Click "Inserting mathematical templates in an equation" and then click
	  "Inserting fence templates in an equation."
	
	NOTE: If Equation Editor Help is not installed on your computer, please see the
	following article in the Microsoft Knowledge Base:
	
	  Q120802 Office: How to Add/Remove a Single Office Program or Component
