## Q170211: WD97: Bullets in Multilevel Numbering Convert to Boxes

	Article: Q170211
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbinterop kbusage word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you open a Rich Text Format (RTF) document in Word 97, symbol characters in
	a multilevel numbering scheme are changed to square boxes.
	
	
	CAUSE
	=====
	
	This problem occurs when you save a document in Rich Text Format (RTF) in
	Microsoft Word for Windows 95, version 7.0, and then import the document into
	Word 97. The ANSI symbol character is incorrectly converted to Unicode.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows.
	This problem was corrected in Microsoft Word 97 SR-1.
	
	For additional information about SR-1, please see the following article in the
	Microsoft Knowledge Base:
	
	  Q172475 OFF97: How to Obtain and Install MS Office 97 SR-1
