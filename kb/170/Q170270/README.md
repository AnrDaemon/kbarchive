## Q170270: Availability of ZAK for Windows NT Workstation 4.0

{% raw %}

	Article: Q170270
	Product(s): Microsoft Windows NT
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbtool
	Last Modified: 20-FEB-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Zero Administration Kit for Windows NT Workstation 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes the availability of Microsoft Zero Administration Kit
	(ZAK) for Windows NT Workstation 4.0.
	
	MORE INFORMATION
	================
	
	Zero Administration Kit for Windows NT Workstation 4.0 is available for download
	from the following Microsoft Web site:
	
	  http://www.microsoft.com/windows/zak/default.htm
	
	For a description of Microsoft Zero Administration Kit for Windows NT Workstation
	4.0, see the following article in the Microsoft Knowledge Base:
	
	  Q170271 Description of ZAK for Windows NT Workstation 4.0
