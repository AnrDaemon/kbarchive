## Q170467: Poor Color Quality with HP DeskJet 800-Series Printers

	Article: Q170467
	Product(s): Microsoft Windows NT
	Version(s): 3.51 4.0
	Operating System(s): 
	Keyword(s): kbprint
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server versions 3.51, 4.0 
	- Microsoft Windows NT Workstation versions 3.51, 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you print to a Hewlett-Packard (HP) DeskJet 800-series printer, the color
	quality of printed output from Windows NT 4.0 may not be as high as the color
	quality of printed output to the same printer from Microsoft Windows 95 or
	Windows 3.1.
	
	CAUSE
	=====
	
	The Windows NT 4.0 printer drivers for HP DeskJet printers do not support Color
	Resolution Enhancement Technology (CRET) or the following HP features:
	
	- Color Matching
	
	- ColorSmart Technologies
	
	- Photo Pen
	
	- Photo Paper
	
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
