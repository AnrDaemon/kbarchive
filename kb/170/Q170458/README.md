## Q170458: Flashing Area Appears on the Desktop on a ZAK Workstation

	Article: Q170458
	Product(s): Microsoft Windows NT
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbui
	Last Modified: 21-FEB-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Zero Administration Kit for Windows NT Workstation 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you double-click the desktop on a Windows NT Zero Administration Kit (ZAK)
	client, you may see a rectangular area flicker on and off.
	
	CAUSE
	=====
	
	This behavior is caused by the way that Windows NT Explorer hides items on the
	desktop on a ZAK client.
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
	
	MORE INFORMATION
	================
	
	You can ignore the flickering areas. They do not affect the ZAK client or the
	programs being run.
