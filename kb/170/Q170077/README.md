## Q170077: XFOR: IMC Hangs When VRFY is Enabled and VRFY Command.

{% raw %}

	Article: Q170077
	Product(s): Microsoft Exchange
	Version(s): WinNT:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	Administrators of Microsoft Exchange Server 5.0 will notice that when VRFY is
	enabled and the VRFY command is issued against a valid account, the Internet
	Mail Connector (IMC) halts and produces a Dr. Watson log.
	
	STATUS
	======
	
	This problem been fixed and is now available in Microsoft Exchange Service Pack
	1.
	
	
	Additional query words: IMS VRFY
