## Q170573: MDG: Remote Mail is BlackListed

	Article: Q170573
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Remote for Windows, version 3.2a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Remote Mail using Megahertz XJack series 288 PCMCIA Modems connecting to
	External with Hayes Accura or Optima 288 and highsp11 script results in Remote
	Mail error in the Session.log:
	
	  In:[32]
	  Response Code:32
	  BLACKLISTED
	
	  [networkname:poname]
	  disconnected
	
	RESOLUTION
	==========
	
	This error is not a Microsoft Mail Remote error message.
	
	This is a problem with the Megahertz XJack series. Megahertz recommends an update
	to the modem firmware and PCMCIA drivers be applied to the remote workstation.
