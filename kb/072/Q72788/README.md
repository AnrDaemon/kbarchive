	SUMMARY
	=======
	
	At the Windows World exposition in Atlanta, Georgia in May, 1991, Microsoft
	distributed a document titled "Multimedia Personal Computing: The Microsoft
	View." This document describes multimedia, its application to business and home
	computing and the Microsoft commitment to help fulfill the vision of
	"Information at Your Fingertips"(TM) through multimedia technology.
	
	The text of the document is available in two different formats in the
	Software/Data Library: MMBACKWW.EXE has a copy of the file in Word for Windows
	format and MMBACK.EXE has a copy of the file as plain text.
	
	The following files are available for download from the Microsoft Download
	Center:
	
	  Download Mmback.exe now
	
	  Download Mmbackww.exe now
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
