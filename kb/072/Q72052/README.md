## Q72052: Duplicate Dates in Month View When Converting Calendar Files

	Article: Q72052
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Reading Calendar files created in Microsoft Windows versions earlier than 3.0
	may show duplicate dates in month view. In particular, some of the dates that
	were marked in the old version appear with the wrong date in Windows 3.0.
	
	WORKAROUND
	==========
	
	To prevent this from happening, remove the mark from the date in earlier
	versions before saving the file.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Calendar version 3.0.
