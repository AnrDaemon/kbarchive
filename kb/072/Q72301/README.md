## Q72301: Creating Wallpaper with Windows 3.0 Paintbrush

	Article: Q72301
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	To create Desktop Wallpaper the size of the display screen in Windows 3.0, you
	must draw the image to the pixel resolution (also known as pels resolution) of
	the display driver being used. The chart below lists the common video standards
	with their widths and heights:
	
	  Video Display      Width     Height
	  -------------      -----     ------
	
	  Standard VGA       640       480
	  Super VGA          800       600
	  8514/a             1024      768
	  EGA                640       350
	  CGA                640       200
	  Hercules           720       384
	
	MORE INFORMATION
	================
	
	To set up Paintbrush for standard VGA display, do the following:
	
	1. From the Paintbrush Options menu, choose Image Attributes.
	
	2. From the Units box, select the Pels button.
	
	3. In the Width box, type 640. In the Height box, type 480.
	
	4. Choose the OK button.
	
	5. From the File menu, choose New. Draw the image.
	
	You can use the same steps to set up for other display resolutions.
