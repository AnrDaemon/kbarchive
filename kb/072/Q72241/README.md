## Q72241: HP IIID PostScript Driver Available from HP

	Article: Q72241
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 26-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Hewlett-Packard (HP) LaserJet IIID PostScript Cartridge printer driver for
	Microsoft Windows version 3.00 is available from HP's fulfillment center.
	
	MORE INFORMATION
	================
	
	The HP LaserJet III PostScript Cartridge driver ".WPD" file for Windows 3.0 is
	available in the Microsoft Windows 3.0 SDL but does not support the HP IIID with
	the PostScript option.
	
	The HP LaserJet IIID PostScript Cartridge printer driver does not provide the
	duplexing feature (printing on both sides).
	
	The filename is HP_3D522.WPD version 52.2.
	
	For information on installing unlisted printer drivers, please refer to the
	"Installing a Printer" section on page 158 of the "Microsoft Windows User's
	Guide."
	
	For information about installing the PostScript cartridge, query on the following
	words:
	
	  iiid and postscript
	
	Additional query words: 3.00 win30 3rdparty 3.00A
