## Q72519: NCR 286 Requires BIOS 4.42 or Later to Run in Standard Mode

{% raw %}

	Article: Q72519
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	NCR 286 machines require BIOS version 4.42 or later to run Microsoft
	Windows version 3.0 in standard mode. To obtain the updated BIOS,
	contact a local NCR dealer and request kit #3299-K040.
