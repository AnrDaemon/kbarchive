	- Microsoft Windows Software Development Kit (SDK) versions 3.0, 3.1

	SUMMARY
	=======
	
	ToolHelp.exe is a sample program that demonstrates some of the capabilities of
	ToolHelp.dll.
	
	The ToolHelp.exe contains the following files:
	
	  Filename     Description
	  --------     -----------
	
	  Test.*       Sample code that demonstrates some of ToolHelp's
	               capabilities.
	
	  ToolHelp.*   The DLL, LIB, and so on, needed to use this version of
	               the ToolHelp DLL.
