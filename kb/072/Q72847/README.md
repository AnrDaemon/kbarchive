## Q72847: PC Win: Win Client Creates Shared Folders Without Privileges

{% raw %}

	Article: Q72847
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:2.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 2.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In version 2.1 of Microsoft Mail for Windows, you can create a shared folder
	even if you have view-only privileges, because selecting child and then peer
	from the New Folder window gives you access to the shared option.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in version 2.1 of Microsoft Mail
	for Windows. This problem was corrected in version 3.0.
