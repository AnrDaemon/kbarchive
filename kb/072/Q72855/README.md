## Q72855: Automated Test Facilities and Microsoft Windows

{% raw %}

	Article: Q72855
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s):

	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Automated Test Facilities is a program developed by Softbridge, Inc., that
	allows you to edit mouse strokes as well as keystrokes in the Recorder program
	packaged with Windows or with Softbridge's Bridge Toolkit.
	
	Automated Test Facilities has not been tested by Microsoft. However, Softbridge
	states that the product works with Microsoft Windows versions 3.0 and later.
	
	MORE INFORMATION
	================
	
	Users of Automated Test Facilities should call Softbridge for additional
	information.
	
	The product included here is manufactured by a vendor independent of Microsoft;
	we make no warranty, implied or otherwise, regarding this product's performance
	or reliability.
