## Q231535: MapPoint 2000: Drawing Tools Do Not Work in World View

	Article: Q231535
	Product(s): Microsoft Automap
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbtool kbui kbimu
	Last Modified: 14-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MapPoint 2000 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you zoom out to World view in Microsoft MapPoint 2000 and attempt to use
	drawing tools in the area of space surrounding the earth, the tools do not work.
	
	STATUS
	======
	
	This behavior is by design.
