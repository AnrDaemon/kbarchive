## Q231455: Internet Explorer 5 At a Glance Corrections and Comments

{% raw %}

	Article: Q231455
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 11-JAN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Internet Explorer 5 At a Glance ISBN 1-57231-964-X 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book "Internet Explorer 5 At A Glance," ISBN
	1-57231-964-X.
	
	The following topics are covered:
	
	- Page 33: Button name in Step 5 is incorrect
	
	- Page 133: Junk E-Mail procedure should be deleted
	
	- Page 234: Web URL incorrect
	
	MORE INFORMATION
	================
	
	Page 33: Button name in Step 5 is incorrect
	-------------------------------------------
	
	In step 5, the button name should be "click OK" instead of "click Update".
	
	
	Page 133: Junk E-Mail Procedure should Be Deleted
	-------------------------------------------------
	
	The top procedure, "Set Rules for Detecting Junk E-mail," will not work because
	this feature was removed from the final version of Microsoft Internet Explorer
	5. This procedure should be deleted.
	
	
	Page 234: Web URL Incorrect
	---------------------------
	
	On page 234, step 1, change the URL.
	
	Change:
	http://www.microsoft.com/sbnmember/levels/getielogo.asp
	
	To:
	http://msdn.microsoft.com/osig/ie/usage.asp
