## Q231778: Migrated NetWare Users Do Not Appear in User Manager

	Article: Q231778
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the migration utility to migrate users from NetWare to Windows NT,
	the users do not appear in User Manager when you view NetWare-compatible users.
	
	CAUSE
	=====
	
	This problem occurs because the migration utility does not correctly set an
	option in the user's properties.
	
	STATUS
	======
