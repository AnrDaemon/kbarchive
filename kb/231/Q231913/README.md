## Q231913: Midtown Madness: Game Hangs When You Change Screen Resolution

	Article: Q231913
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbimu msgamekbfaq
	Last Modified: 21-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Midtown Madness, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you change your screen resolution in Microsoft Windows, Microsoft Midtown
	Madness may stop responding (hang).
	
	CAUSE
	=====
	
	This behavior can occur if you change your screen resolution in Windows while
	Midtown Madness is running or is minimized on the taskbar.
	
	RESOLUTION
	==========
	
	To prevent this issue from occurring, quit Midtown Madness before you change
	your screen resolution in Windows.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Midtown Madness,
	version 1.0.
