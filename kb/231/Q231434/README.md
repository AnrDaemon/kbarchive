## Q231434: Encarta Encyclopedia: Problems When Running on Windows 2000

	Article: Q231434
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbenv kbimu
	Last Modified: 09-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 98 Encyclopedia for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run older versions of Microsoft Encarta Encyclopedia on a Microsoft
	Windows 2000-based computer, Encarta may not function properly or may behave
	unexpectedly.
	
	This article describes some of the symptoms you may encounter and explains how to
	resolve this behavior for each version of Encarta Encyclopedia.
	
	RESOLUTION
	==========
	
	To resolve this issue, upgrade to the current version of Microsoft Encarta
	Encyclopedia.
