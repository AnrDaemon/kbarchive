## Q231907: Midtown Madness: Game Is Minimized Unexpectedly

{% raw %}

	Article: Q231907
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbimu msgamekbfaq
	Last Modified: 20-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Midtown Madness, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to switch to the Windows desktop or another running program by
	pressing ALT+TAB, Microsoft Midtown Madness may be minimized unexpectedly.
	
	RESOLUTION
	==========
	
	To work around this issue, click the Midtown Madness button on the taskbar to
	restore Midtown Madness.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Midtown Madness,
	version 1.0.
