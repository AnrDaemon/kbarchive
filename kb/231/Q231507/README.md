## Q231507: MapPoint 2000: Found Addresses Are Not Marked with Yellow Circle

	Article: Q231507
	Product(s): Microsoft Automap
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbdisplay kbimu
	Last Modified: 14-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MapPoint 2000 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In Microsoft MapPoint 2000, a pushpin associated with a found address is not
	marked with a solid yellow circle, but is surrounded by a transparent square
	box.
	
	MORE INFORMATION
	================
	
	This behavior is by design.
	
	Additional query words: mp2000 cannot search find colored
