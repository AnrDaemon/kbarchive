## Q231627: XFOR: &quot;No Active Recipients&quot; Sending from GroupWise to Exchange

	Article: Q231627
	Product(s): Microsoft Exchange
	Version(s): winnt:5.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-JUN-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you send mail from Novell GroupWise to Microsoft Exchange Server, the API
	gateway or the GroupWise Internet agent may log the following error message:
	
	  No active recipients.
	
	Furthermore, when you reply to a message sent from Exchange Server to GroupWise,
	the To box may be blank, which may be the cause of the "No active recipients"
	error.
	
	MORE INFORMATION
	================
	
	This issue has been addressed and documented by Novell. You can obtain
	documentation on this issue from the following Novell Web site:
	
	  http://www.support.novell.com
	
	The article discussing this issue is titled "API Gateway reports 'no active
	recipients.'"
