## Q231542: MapPoint 2000: &quot;Send Behind Roads&quot; Command Behaves Unexpectedly

{% raw %}

	Article: Q231542
	Product(s): Microsoft Automap
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbgraphic kbimu
	Last Modified: 14-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MapPoint 2000 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use a drawing tool to draw on a map in Microsoft MapPoint 2000, and
	then click Send Behind Roads on the Draw menu, features other than roads may be
	displayed in front of the drawing.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft MapPoint 2000.
	
	MORE INFORMATION
	================
	
	The following features may be displayed in front of the drawing:
	
	- International and state borders
	- Boundaries between time zones
	- Rivers and streams
	- Mountains and volcanoes
	- Place names and symbols
	
	NOTE: Boundaries between differently shaded areas, such as national parks and
	Indian reservations, are always displayed behind the drawing.
	
	Additional query words: mp2000 layers above below over under
