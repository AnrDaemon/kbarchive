## Q231859: Midtown Madness: Game Is Not Recognized on Zone.com

	Article: Q231859
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbenv kbimu msgamekbfaq
	Last Modified: 09-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Midtown Madness, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to start a multiplayer game of Microsoft Midtown Madness on
	Zone.com, the Zone may not recognize that Midtown Madness is installed on your
	computer.
	
	CAUSE
	=====
	
	This behavior can occur if the current version of Microsoft DirectX is not
	installed on your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the Microsoft DirectPlay 6.1a
	Upgrade from the following Microsoft Web site:
	
	  http://www.microsoft.com/games/midtown/patch.htm
	
	MORE INFORMATION
	================
	
	Midtown Madness is a copy-protected product. Copy-protected products require the
	updated version of DirectPlay included with DirectX 6.1a to be compatible with
	multiplayer gameplay on Zone.com.
