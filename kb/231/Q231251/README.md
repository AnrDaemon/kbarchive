## Q231251: SMS: IBM Networks Coordinated Logon Client Hangs Installation

{% raw %}

	Article: Q231251
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:2.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbsms200
	Last Modified: 27-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The IBM Networks Coordinated Logon Client for Windows NT client software may
	cause the Systems Management Server (SMS) client installation process to stop
	responding (hang). When this occurs, the last log entry in the Wn_manual or
	Wn_logon.log file is "Waiting for 1 minute for CliSvc to start CCIM." The client
	installation process does not proceed, and the client computer may also hang.
	
	RESOLUTION
	==========
	
	To resolve this issue, please contact IBM Technical Support to obtain version
	4.2.4 or later of the IBM Networks Coordinated Logon Client for Windows NT
	client software.
