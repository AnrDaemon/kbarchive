## Q231595: Midtown Madness: How to Improve Game Performance

	Article: Q231595
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbdisplay kbui kbimu msgamekbfaq
	Last Modified: 20-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Midtown Madness, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes several methods you can use to improve game performance
	in Microsoft Midtown Madness.
	
	MORE INFORMATION
	================
	
	To improve game performance in Midtown Madness, use the following methods:
	
	Quit Unnecessary Programs
	-------------------------
	
	To increase the amount of available system resources, quit all programs that are
	running before you start Midtown Madness.
	
	Select a Lower Display Resolution
	---------------------------------
	
	To select a lower display resolution in Midtown Madness, follow these steps:
	
	1. On the Midtown Madness startup screen, click Options.
	
	2. Click Graphics Options.
	
	3. In the Resolution box, click a lower display resolution, and then click Done.
	
	4. Click the Back (<<) button to return to the startup screen.
	
	Disable Sky Textures
	--------------------
	
	To disable Sky Textures, follow these steps:
	
	1. On the Midtown Madness startup screen, click Options.
	
	2. Click Graphics Options.
	
	3. Click to clear the Textured Sky check box, and then click Done.
	
	4. Click the Back (<<) button to return to the startup screen.
	
	Reduce the Far Clip Distance
	----------------------------
	
	To reduce the Far Clip Distance, follow these steps:
	
	1. On the Midtown Madness startup screen, click Options.
	
	2. Click Graphics Options.
	
	3. Click the left arrow (<) under Far Clip Distance to reduce the Far Clip
	  Distance setting, and then click Done.
	
	4. Click the Back (<<) button to return to the startup screen.
	
	Hide Pedestrians
	----------------
	
	To hide pedestrians, follow these steps:
	
	1. On the Midtown Madness startup screen, click Options.
	
	2. Click Graphics Options.
	
	3. Click to select the Hide Pedestrians check box, and then click Done.
	
	4. Click the Back (<<) button to return to the startup screen.
	
	Disable Cloud Shadows and Vehicle Reflections
	---------------------------------------------
	
	To disable Cloud Shadows and Vehicle Reflections, follow these steps:
	
	1. On the Midtown Madness startup screen, click Options.
	
	2. Click Graphics Options.
	
	3. Click to clear the Cloud Shadows check box.
	
	4. Click to clear the Vehicle Reflections check box.
	
	5. Click Done, and then click the Back (<<) button to return to the
	  startup screen.
	
	Change Game Views
	-----------------
	
	To change your game view during a race, press the appropriate key on your
	keyboard or button on your game controller. The following table lists the
	available game views and their corresponding keys or buttons:
	
	  Keyboard   Joystick   Game View
	  ----------------------------------------------------------------------
	
	  C          Button 4   Toggles between the Chase Near, Chase Far, and
	                        Cockpit game views.
	
	  W          N/A        Toggles between the Normal and Widescreen game
	                        views.
