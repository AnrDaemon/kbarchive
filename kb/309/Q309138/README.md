## Q309138: Audio synchronization problems in French Interactive Training

{% raw %}

	Article: Q309138
	Product(s): Microsoft Press
	Version(s): 3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-MAR-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft OfficeXP Professional Step By Step Interactive, version 3.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Audio is not properly synchronized with the events on screen in the lesson
	"Creation et utilisation de contacts." All topics in this lesson except the
	"Introduction" and "Resume de la lecon" are affected.
	
	CAUSE
	=====
	
	Issues with audio/event timing.
	
	WORKAROUND
	==========
	
	This is a known issue and there is currently no fix available. The user is
	advised to proceed to the next lesson in the training, or to any other lesson
	using the Table of Contents navigation.
