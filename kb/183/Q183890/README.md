## Q183890: WD97: Can Save Changes to Read-Only Document on Novell Server

{% raw %}

	Article: Q183890
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kb3rdparty kbnetwork
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you use the Microsoft client for NetWare, you can save changes to read-only
	documents stored on Novell 3.12 and 4.x servers. After you save changes, the
	read-only attribute is removed from the document.
	
	WORKAROUND
	==========
	
	Use the Novell 32-bit client for NetWare with Windows 95.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
