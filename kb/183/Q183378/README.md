## Q183378: HTML Help Authoring Kit: vcSampleDoc Files Missing from CD-ROM

	Article: Q183378
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbHTMLHelp
	Last Modified: 19-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Official Microsoft HTML Help Authoring Kit ISBN 1-57231-603-9 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to compile the vcSample application, you may receive
	compilation errors.
	
	CAUSE
	=====
	
	Three files needed to compile the vcSample application were not included with
	the Official HTML Help Authoring Kit companion CD-ROM. These three files are:
	
	  vcSample\vcSampleDoc.cpp
	  vcSample\vcSampleDoc.h
	  vcSample\res\vcSampleDoc.ico
	
	RESOLUTION
	==========
	
	Microsoft Press has prepared a patch file, called Htmlhak1.exe, that includes
	these missing files.
	
	The following file is available for download from the Microsoft Download Center:
	
	  Htmlhak1.exe
	  (http://download.microsoft.com/download/MSPressPub/mspress/2.0/WIN98/EN-US/Htmlhak1.exe)
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
	
	Microsoft used the most current virus detection software available on the date of
	posting to scan this file for viruses. Once posted, the file is housed on secure
	servers that prevent any unauthorized changes to the file.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the "Official Microsoft HTML
	Help Authoring Kit."
