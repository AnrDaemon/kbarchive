## Q183778: XCLN: ALT+TAB Shows Two ODK Icons

	Article: Q183778
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:98
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook 98 Deployment Kit 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run the Outlook Deployment Kit (ODK) Wizard and use the ALT+TAB key
	combination to switch to another program, two ODK Wizard icons are visible.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Outlook 98.
	
	MORE INFORMATION
	================
	
	The Outlook Deployment Kit is used to create custom install packages of Outlook
	and related components for distribution to your users.
