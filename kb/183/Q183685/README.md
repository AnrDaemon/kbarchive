## Q183685: WD97: Format Painter Cancelled When Using Next Page Scroll Bar

	Article: Q183685
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbdta word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you select formatted text and then click the Format Painter icon, if you
	scroll through the document using the next page or previous page (double arrow)
	icons, the format painter is disabled.
	
	RESOLUTION
	==========
	
	Scroll through the document by clicking the scroll bar or by clicking the single
	arrow that scrolls the document line by line.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
