## Q183079: WD97: Graphics Not Displayed in Binder

	Article: Q183079
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbgraphic kbinterop kbdta kbtemplate word97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you add a file (on the Section menu, click Add) to a Microsoft Office
	Binder, some graphics in the template may not be displayed.
	
	NOTE: This problem has been noticed to occur with the Contemporary Memo.dot and
	Contemporary Report.dot templates.
	
	WORKAROUND
	==========
	
	Use "Add from File" to add a copy of the file to the binder, using the following
	steps:
	
	1. On the Section menu, click Add from File.
	
	2. In the Add from File dialog box, change the Files of type box to "All Files
	  (*.*)", locate and select the template you want, and then click Add.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
