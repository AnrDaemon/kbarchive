## Q183779: XCLN: Cannot Clear Combo Boxes in Policy Editor

	Article: Q183779
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:98
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Outlook 98 Deployment Kit 
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In the Outlook Deployment Kit (ODK), you cannot clear combo boxes after making a
	selection in Customize User Settings (under Stage 5, User Settings
	Customization) of the Policy Editor.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Outlook 98 Deployment
	Kit.
	
	MORE INFORMATION
	================
	
	The Outlook Deployment Kit is used to create custom install packages of Outlook
	and related components for distribution to your users.
