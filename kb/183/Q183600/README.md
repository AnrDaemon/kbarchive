## Q183600: AOE Err Msg: DirectX 5.0 Is Required to Run This Game

	Article: Q183600
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kberrmsg kbsetup kbtool aoe kbimu msgame KbDirectX
	Last Modified: 05-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Age of Empires, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start Microsoft Age of Empires, you may receive the following error
	message:
	
	  DirectX 5.0 is required to run this game.
	
	CAUSE
	=====
	
	This behavior can occur if the installation of DirectX 5.0 did not complete
	successfully.
	
	RESOLUTION
	==========
	
	Install Latest Version of DirectX
	---------------------------------
	
	Download and install the latest version of DirectX from the following Microsoft
	Web site:
	
	  http://www.microsoft.com/directx/homeuser/downloads/default.asp
	
	For additional information about how to install the latest version of DirectX,
	click the article number below to view the article in the Microsoft Knowledge
	Base:
	
	  Q179113 How to Download and Install DirectX
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Age of Empires,
	version 1.0.
