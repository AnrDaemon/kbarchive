## Q183002: WD97: Third Automatic Bullet or Number Is Left-Aligned

{% raw %}

	Article: Q183002
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:7.0a
	Operating System(s): 
	Keyword(s): kbdta word97 kbnumbering
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	- Microsoft Word for Windows 95, version 7.0a 
	-------------------------------------------------------------------------------
	
	MORE INFORMATION
	================
	
	To locate the Automatic Bulleted Lists or Automatic Numbered Lists check boxes,
	click AutoCorrect on the Tools menu, and then click the AutoFormat As You Type
	tab.
	
	For additional information about help for WordPerfect users, please see the
	following articles in the Microsoft Knowledge Base:
	
	  Q107584 WD: Unusual Behavior with "WordPerfect Navigation Keys" On
	
	  Q111717 NEWS: MS Offers Word Support for Former WordPerfect Users
	
	For additional information about AutoFormat features, please see the following
	articles in the Microsoft Knowledge Base:
	
	  Q139564 WD97: How to Disable Word AutoFormat Features
	
	  Q159941 WD97: General Information about "Auto" Features in Word 97
	
	SYMPTOMS
	========
	
	In Microsoft Word, if you use Automatic Bulleted Lists or Automatic Numbered
	Lists, your third automatic bullet or number may be left-aligned, even if the
	bullets or numbers above it are indented.
	
	CAUSE
	=====
	
	This problem occurs when Help For WordPerfect Users is turned on.
	
	WORKAROUND
	==========
	
	To allow Word to correctly use the Bullets And Numbering feature, turn off Help
	For WordPerfect Users before you use the AutoFormat As You Type options. To turn
	off Help For WordPerfect Users, follow these steps:
	
	1. On the Tools menu, click Options.
	
	2. On the General tab, click to clear the Help For WordPerfect Users check box,
	  and then click OK.
	
	Automatic Bulleted Lists or Automatic Numbered Lists will now function properly.
