## Q183193: Mac Encarta 98: Minimum System Requirements

{% raw %}

	Article: Q183193
	Product(s): Microsoft Home Multimedia Titles
	Version(s): MACINTOSH:
	Operating System(s): 
	Keyword(s): kbref kbimu
	Last Modified: 19-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 98 Encyclopedia for Macintosh 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the minimum system requirements to run Encarta 98
	Encyclopedia for Macintosh.
	
	MORE INFORMATION
	================
	
	The minimum system requirements to run Encarta 98 Encyclopedia for Macintosh
	are:
	
	- A Power Macintosh, or a Macintosh with a 25 MHz, 68030 or faster processor
	  (includes Macintosh IIci, LCIII, Performa 450, and later models).
	
	- System 7.1 or later.
	
	- 8 megabytes (MB) of random access memory (RAM) for System 7.1; 16 MB of RAM
	  for system 7.5 or later.
	
	- 43 MB of available hard disk space.
	
	- 8 MB of available hard disk space for Microsoft Encarta Research Organizer.
	
	- An additional 3 MB of RAM if you run Encarta Research Organizer at the same
	  time as Encarta 98 Encyclopedia.
	
	- A double-speed or faster CD-ROM drive.
	
	- A monitor that supports 256 colors at 640 X 480 or higher resolution.
	
	To use Encarta Yearbook, Web Links, and monthly online updates, you need:
	
	- A modem (9600 bits per second or faster).
	
	- Access to the Internet.
	
	- 4 MB of additional hard disk space, plus 1 MB for each monthly update.
