## Q183385: WD97: No Insertion Point Deleting Text Box from Header/Footer

	Article: Q183385
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you delete a text box that has been placed in a header or footer, the
	insertion point is lost.
	
	WORKAROUND
	==========
	
	Click in the Header/Footer window for the insertion point to reappear.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
	
	This problem has been corrected in Word 2000.
