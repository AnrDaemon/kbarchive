## Q138040: Full Text Search Help Files Not Removed During Uninstall

	Article: Q138040
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	1.00
	WINDOWS
	kbsetup kbusage kbprb
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Fury 3 for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use the uninstall process to remove Fury 3, the Fury 3 full-text search
	help files, Fury3.fts and Fury3.gid, are not removed from the hard drive.
	
	This occurs when you answer No to the following question during the uninstall
	process:
	
	  Do you want to remove all your saved games?
	
	RESOLUTION
	==========
	
	Manually delete the Fury3.fts and Fury3.gid files from the hard drive.
	
	NOTE: The Fury3.fts is only created if you choose Find while using Help.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Fury 3 version 1.0 for Windows.
