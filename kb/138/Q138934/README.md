## Q138934: Chips and Technologies CT-65545 Does Not Run 256 Color Video

	Article: Q138934
	Product(s): Microsoft Windows NT
	Version(s): 3.5 3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation versions 3.5, 3.51 
	- Microsoft Windows NT Server versions 3.5, 3.51 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Laptop computers using the Chips & Technologies Inc. CT-65545 video
	controller do not run in 256 color mode. Some of the affected laptops are:
	
	  IBM ThinkPad 701C
	  Toshiba T4900CT
	  NEC Versa M
	  Digital Equipment Corp HiNote and HiNote Ultra
	
	
	RESOLUTION
	==========
	
	The drivers required to run 256 color video can be purchased from the Chips
	& Technologies Inc. authorized distributor:
	
	  Added Value Electronics Distributor
	  Shelby Miles
	  714-573-5000
	  price: $50.00 (US)
