## Q138031: Bookshelf 1995: Some Long Addresses Not Recognized

	Article: Q138031
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1995 edition; WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for Windows 1995 edition 
	- Microsoft Bookshelf '95 for Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you type in a long address, Address Builder may fail to recognize it.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
