## Q138228: Accessibility High Contrast Setting Reverts to Custom

	Article: Q138228
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): kbenable kbEnableSight kbEnableLearn
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you choose the Black On White option in the Settings For High Contrast dialog
	box in Accessibility properties, close the dialog box, and then open it again,
	the Black On White option is changed to Custom. If you then click OK or Apply,
	the custom scheme is not applied and there is no change in the high contrast
	color scheme.
	
	RESOLUTION
	==========
	
	To use the Custom option, you must use a different scheme.
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
