## Q138982: 3D Movie Maker: Minimum System Requirements

{% raw %}

	Article: Q138982
	Product(s): Microsoft Home Kids Products
	Version(s): 1.00
	Operating System(s): 
	Keyword(s): kbfaq
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft 3D Movie Maker for Windows, version 1.0 
	- Microsoft Nickelodeon 3D Movie Maker for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the minimum system requirements for Microsoft 3D Movie Maker.
	
	MORE INFORMATION
	================
	
	The minimum system requirements to run 3D Movie Maker are:
	
	- Multimedia PC with a 486/50 MHz or higher processor
	
	- Microsoft Windows 95 operating system, or Microsoft Windows NT version 3.51
	  or later
	
	- Double-speed (2x) or better CD-ROM drive
	
	- 8 MB of memory (RAM)
	
	- 12 MB of available hard disk space
	
	- Super VGA 256-color monitor
	
	- Local bus or PCI bus video
	
	- 16-bit sound card and headphones or speakers (microphone also recommended)
	
	- Microsoft Mouse or compatible pointing device
