## Q138041: 1ToMany.exe Form for One-to-Many Order Entry Application

	Article: Q138041
	Product(s): Microsoft FoxPro
	Version(s): WINDOWS:3.0,5.0,6.0
	Operating System(s): 
	Keyword(s): kbfile kbsample kbnokeyword kbvfp300 kbvfp500 kbvfp600
	Last Modified: 24-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual FoxPro for Windows, versions 3.0, 5.0, 6.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	1ToMany.exe is a sample that shows the minimum key components needed to create a
	customer lookup screen in Visual FoxPro. It is designed to help you understand
	components, properties, and event code.
	
	MORE INFORMATION
	================
	
	The following file is available for download from the Microsoft Download
	Center:
	
	  1ToMany.exe
	  (http://download.microsoft.com/download/fox30std/sample/1/WIN98/EN-US/1ToMany.exe)
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
	
	Microsoft used the most current virus detection software available on the date of
	posting to scan this file for viruses. Once posted, the file is housed on secure
	servers that prevent any unauthorized changes to the file.
	
	
	There is a Readme.txt file included in the 1ToMany.exe sample that gives detailed
	information.
	
	All the required files have been placed in a project file. You can use the sample
	project and form as a reference, or you can modify them and incorporate them
	into your own application.
	
	When you open the project in Visual FoxPro 6.0, a prompt appears that asks you to
	convert the project to Visual FoxPro 6.0 format. To convert the files to Visual
	FoxPro 6.0 format, click Continue.
