## Q138926: Print Jobs Not Clearing for Compaq Pagemarq Printers

{% raw %}

	Article: Q138926
	Product(s): Microsoft Windows NT
	Version(s): 3.51 4.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbprint kbPrinting
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation versions 3.51, 4.0 
	- Microsoft Windows NT Server versions 3.51, 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you print to a Windows NT 3.51 server connected to a Compaq Pagemarq 15 or
	Pagemarq 20 network laser printer, print jobs are not deleted from the print
	queue.
	
	RESOLUTION
	==========
	
	Check the firmware revision of your printer. Your Pagemarq printer should have a
	firmware revision of 1.13 or later. To obtain the latest firmware contact Compaq
	Technical Support at (800) 652-6672.
