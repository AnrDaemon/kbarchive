## Q138074: Word 95 Step By Step Book: Corrections and Comments

	Article: Q138074
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-JAN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Word for Windows 95 Step by Step ISBN 1-55615-828-9 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains information on known errors, corrections, and comments
	relating to the Microsoft Press book "Microsoft Word for Windows 95 Step by
	Step."
	
	The following topics are covered:
	
	- Page 24, 30: Favorites Folder Missing Winword SBS Practice
	
	MORE INFORMATION
	================
	
	In addition to a description of the book's problems, this document also includes
	sections labeled "Correction" and "Comments." Please note that the "Correction"
	section is worded for correcting the book and does not necessarily address the
	problem introduced by the book error. The "Comments" section contains specific
	information for working around problems.
	
	Page 24, 30: Favorites Folder Missing Winword SBS Practice
	----------------------------------------------------------
	
	Pages 24 and 30 provide instructions for accessing the practice files for this
	book. However, the Winword SBS Practice folder is not located in the Favorites
	folder, as described in the book.
	
	Comments:
	
	Please make a note of the following corrections in this book:
	
	Replace page 24, step 2 with:
	
	If the Winword SBS Practice folder is not currently displayed in the Save In box,
	click in the Save In box and then from the list that appears, click the C: hard
	disk. Next double-click the Winword SBS Practice folder icon.
	
	Replace page 30, step 3 with:
	
	If a folder name other than Winword SBS Practice appears in the Look In box,
	click in the Look In box and then from the list that appears, click the C: hard
	disk.
	
	This information applies to the first printing of Microsoft Word for Windows 95
	Step by Step.
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
