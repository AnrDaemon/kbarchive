## Q138208: How to Create a Flight Simulator Shortcut in Windows 95/98

	Article: Q138208
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:5.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for MS-DOS, version 5.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following is a description of how to manually create a Windows 95/98
	shortcut for Flight Simulator 5.1 without having to start Flight Simulator in
	MS-DOS mode.
	
	How to Create a Windows 95/98 Shortcut
	--------------------------------------
	
	1. Double-click My Computer, and then open the Fltsim5 folder.
	
	2. Right-click the Fs.bat file, and then click Create Shortcut.
	
	  NOTE: A file called Shortcut To Fs is created.
	
	3. Right-click the new Shortcut To Fs file, and then click Properties.
	
	4. Click the Program tab.
	
	5. Click the Close On Exit check box to select it.
	
	6. Click Advanced.
	
	7. Click the "Prevent MS-DOS-based programs from detecting Windows" check box to
	  select it.
	
	8. Click OK, and then click OK.
	
	9. Double-click the shortcut to run Flight Simulator.
	
	After the shortcut has been created, you can rename it. To rename the shortcut,
	follow these steps:
	
	1. Right-click the shortcut.
	
	2. Click Rename.
	
	3. Type "Flight Simulator" (without the quotation marks), and then press ENTER.
	
	You may also want to move the shortcut to a more convenient location, such as the
	Start Menu.
	
	To move the icon to the Start Menu, drag the shortcut icon onto the Start button.
	
	MORE INFORMATION
	================
	
	To achieve optimal system performance, the shortcut works best when no MS-DOS
	drivers or minimal MS-DOS drivers are executed from the Autoexec.bat and
	Config.sys files.
	
	For information about running Flight Simulator 5.1 in Windows protected mode,
	please see the following article(s) in the Microsoft Knowledge Base:
	
	  Q140600 DF1269: Optimizing Windows 95 for Flight Simulator
