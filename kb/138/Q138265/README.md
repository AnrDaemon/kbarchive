## Q138265: Creator Field Empty When Creating Recurring Tasks

	Article: Q138265
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0; Win95:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you create recurring meetings, the creator field in the To Do (Active) list
	is empty.
	
	CAUSE
	=====
	
	This is by product design.
	
	The contents of the creator field will only be displayed when a user given the
	modify role modifies or creates recurring tasks in another person's schedule.
