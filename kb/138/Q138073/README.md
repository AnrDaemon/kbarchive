## Q138073: Saved Fury3 Game Does Not Save the Window Size

{% raw %}

	Article: Q138073
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	1.00
	WINDOWS
	kbui kbusage
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Fury 3 for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Fury3 does not save the window size when you save a game. Fury3 also does not
	save the window size when you exit Fury3. When you resume a saved game or start
	a new game, Fury3 starts in a maximized window.
