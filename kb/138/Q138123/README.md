## Q138123: Bookshelf 1995: No Keyboard Access to Audio, Video

	Article: Q138123
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1995 edition; WINDOWS:
	Operating System(s): 
	Keyword(s): msnetwork
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for Windows 1995 edition 
	- Microsoft Bookshelf '95 for Windows 95 
	- Microsoft Bookshelf for Windows, 1995 Intro Edition 
	- Microsoft Bookshelf 1996-97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You are unable to use keyboard keys to control audio and video functionality in
	the programs listed at the top of this article.
	
	NOTE: Pronunciations, National Anthems and Animations play using keyboard
	controls. Press the tab key until the pronunciation speaker icon is selected,
	the animation picture is selected or the National Anthem flag is selected and
	then press the ENTER key to play.
	
	STATUS
	======
	
	The behavior is by design and not a programming problem with Bookshelf.
	
	The mouse must be used to control audio and video.
	
	Additional query words: 1995 multi media multimedia multi-media pronounciation mmtitles kbmm avi wav disable
