## Q138513: Cannot Change Font Used in Printed Online Help Topic

	Article: Q138513
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you print from Windows 95 online Help, the font that is used is 8-point MS
	Sans Serif. The font and font size cannot be changed.
	
	MORE INFORMATION
	================
	
	You can change the size of the font that is displayed on the screen in online
	Help, but you cannot change the font that is printed. To change the size of the
	font on the screen when you are viewing a Help topic, click Options, point to
	Font, and then click Small, Normal, or Large.
	
	To change the font that is printed, you can copy the Help topic text and then
	paste it into a word processor. Then, change the font in the word processor and
	print the document. To copy Help topic text, click Options, and then click Copy.
