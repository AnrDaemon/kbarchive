## Q138210: Julia Child Err Msg: &quot;Unable to Find File&quot;, &quot;Error Opening...&quot;

	Article: Q138210
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Julia Child: Home Cooking with Master Chefs for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to run Julia Child, you receive the following error message:
	
	  Unable to find file. Please reinsert the CD and/or compare your MSJULIA.INI
	  setting with those listed in the README.WRI file.
	
	When you click OK, you receive another message:
	
	  There was an error opening or saving files. Home Cooking with Master Chefs is
	  unable to function properly. Please compare your MSJULIA.INI setting with
	  those listed in the README.WRI file.
	
	When you click OK, you receive another message:
	
	  Home Cooking with Master Chefs is unable to function properly. Your computer
	  may be low on memory or hard- disk space. Close other application or delete
	  unnecessary files.
	
	When you click OK, the program closes.
	
	CAUSE
	=====
	
	These messages occur if no compact disc is in the CD-ROM drive, or if a disc
	other than Julia Child is in the drive when you run the program.
	
	RESOLUTION
	==========
	
	Method 1
	--------
	
	Insert the Julia Child disc before you attempt to run the program.
	
	-or-
	
	Method 2
	--------
	
	Insert the Julia Child disc after you receive any of the above messages, and then
	click OK.
