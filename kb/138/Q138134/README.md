## Q138134: RPLDISK.SYS Error on 3Com 3Station Remoteboot Workstation

{% raw %}

	Article: Q138134
	Product(s): Microsoft Windows NT
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server versions 3.5, 3.51 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you configure a 3Com 3Station remoteboot workstation to a setting other than
	DOS6.22 (Default) using Remoteboot Manager in a Windows NT environment, the
	following error message appears:
	
	  NOT USER LEVEL SEC!
	  RPLDISK.SYS
	  Error in server '\\SERVERNAME'.
	  System files not found (Non-System disk).
	
	RESOLUTION
	==========
	
	To correct this problem, configure the 3Com 3Station to DOS6.22(Default).
