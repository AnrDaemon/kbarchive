## Q138836: Flying Fonts Screen Saver Leaves Afterimages on Screen

{% raw %}

	Article: Q138836
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	- Microsoft Plus! for Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Flying Font screen saver included with the Microsoft TrueType Font Pack
	version 2.0 may leave "ghost" afterimages of the font on the screen.
	
	CAUSE
	=====
	
	Microsoft Plus! includes a feature called Font Smoothing that causes the
	TrueType fonts included with the TrueType Font Pack 2.0 to leave afterimages
	when you use the Flying Fonts screen saver.
	
	RESOLUTION
	==========
	
	To work around this problem, disable the Font Smoothing feature in Microsoft
	Plus!. To do so, follow these steps:
	
	1. In Control Panel, double-click the Display icon.
	
	2. On the Plus! tab, click the Smooth Edges Of Screen Fonts check box to clear
	  it.
	
	3. Click OK.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Plus! for Windows 95.
	We are researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
