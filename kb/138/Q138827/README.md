## Q138827: ALT+F4 Hangs Paint When in View Bitmap Mode

	Article: Q138827
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Microsoft Paint may stop responding (hang) if you press ALT+F4 when Paint is in
	View Bitmap mode.
	
	CAUSE
	=====
	
	Paint does not handle the ALT+F4 key combination correctly when it is in View
	Bitmap mode.
	
	RESOLUTION
	==========
	
	To close Paint when this problem occurs, press CTRL+ALT+DELETE, click Paint, and
	then click End Task.
	
	To avoid this problem, cancel View Bitmap mode by clicking the bitmap or pressing
	any key before you press ALT+F4 to quit Paint.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Paint. We are researching this
	problem and will post new information here in the Microsoft Knowledge Base as it
	becomes available.
