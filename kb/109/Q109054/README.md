## Q109054: Games Group Installed Even If Games Are Not Selected

	Article: Q109054
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you install Windows for Workgroups 3.11 by selecting the Custom Setup
	option and choosing not to install any of the games, Program Manager still
	contains an empty Games group even though you directed Setup not to install any
	games.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows for Workgroups version
	3.11. We are researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
	
	Additional query words: 3.11 progman.exe blank empty nothing uninstall don't install do not add/remove windows components
