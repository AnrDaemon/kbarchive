## Q109858: No Dynamic Scenery with Paris and New York Scenery Disks

	Article: Q109858
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for MS-DOS, version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Configuring dynamic scenery in the Paris or New York scenery area has no effect.
	
	MORE INFORMATION
	================
	
	The Paris and New York Scenery add-on disks for Flight Simulator 5.0 do not add
	dynamic scenery.
	
	Additional query words: 5.00 flightsim fltsim dos NY France FS5
