## Q109860: Flight Simulator: Engine Sound Audible After Engine Turned Off

{% raw %}

	Article: Q109860
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for MS-DOS, version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Engine sounds are still audible when the engines are turned off if you are using
	synthesized sound.
	
	MORE INFORMATION
	================
	
	This occurs with all powered planes and all sound cards when the engine sounds
	are set to synthesized.
	
	Microsoft has confirmed this to be a problem in Flight Simulator for MS-DOS,
	version 5.0.
