## Q110616: Flight Simulator: Diamond Video Card Settings

{% raw %}

	Article: Q110616
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for MS-DOS, version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Diamond Computer Systems technical support has confirmed that the following
	devices operate with Flight Simulator 5.0 in these configurations:
	
	Card Name                 Flight Simulator SVGA Setting
	---------                 -----------------------------
	
	Diamond Viper
	
	    1 MB                 VGA (Cannot support SVGA in Flight Simulator
	                         5.0.)
	    2 MB                 VESA 1.2 (Requires VPRMODE version 2.04 or
	                                   later. Available from Diamond)
	
	Diamond Stealth
	
	    VRAM                S3 86C911
	    24                  S3 86C911
	    PRO                 S3 86C911
	    32 (PCI)            TSENG 4000
	    64 (PCI)            S3 86C911
	
	VGA mode
	
	Diamond Speed Star
	
	    VGA                 TSENG 4000
	    PLUS                TSENG 4000
	    24                  TSENG 4000
	    24X                 Paradise SVGA
	    PRO                 Cirrus Logic 542X
	    VGA Hicolor         TSENG 4000
	
	Diamond Turbo
	
	    VGA                 TSENG 4000
