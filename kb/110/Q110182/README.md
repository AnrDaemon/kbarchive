## Q110182: Err Msg: ATIOEM Accelerator Not Found

	Article: Q110182
	Product(s): Microsoft Windows NT
	Version(s): 3.1
	Operating System(s): 
	Keyword(s): kbhw kbsetup kbHardware
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 3.1 
	- Microsoft Windows NT Workstation version 3.1 
	- Microsoft Windows NT Advanced Server, version 3.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Windows NT defaults to standard VGA and the following Event Viewer error message
	appears when you use an ATI display driver on an AST Premmia series computer:
	
	  ATIOEM accelerator not found.
	
	CAUSE
	=====
	
	Windows NT does not properly recognize the ATI display adapter on these systems.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT and Windows NT
	Advanced Server version 3.1. This problem was corrected in Windows NT version
	3.5.
	
	For more information about this problem, see the README.WRI file in Windows NT.
