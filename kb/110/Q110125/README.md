## Q110125: &quot;Volume&quot; and &quot;Drive&quot; Synonymous in Dialog Boxes &amp; Manuals

{% raw %}

	Article: Q110125
	Product(s): Microsoft FoxPro
	Version(s): MACINTOSH:2.5b,3.0b
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-FEB-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft FoxPro for Macintosh, version 2.5b 
	- Microsoft Visual FoxPro for Macintosh, version 3.0b 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The standard Macintosh term for referring to any mounted mass-storage device is
	"volume." However, in the FoxPro for Macintosh dialog boxes and documentation,
	these devices are referred to as "drives." These terms are synonymous.
