## Q110251: COMMAND Key in FoxPro for Mac Is Treated Same as CTRL Key

{% raw %}

	Article: Q110251
	Product(s): Microsoft FoxPro
	Version(s): MACINTOSH:2.5b,3.0b
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-FEB-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft FoxPro for Macintosh, version 2.5b 
	- Microsoft Visual FoxPro for Macintosh, version 3.0b 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In all areas of FoxPro for Macintosh, the COMMAND key is treated the same as the
	CTRL key in order to ensure cross-platform compatibility.
