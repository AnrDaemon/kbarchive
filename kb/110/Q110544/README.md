## Q110544: Crop Duster Summary Appears in Other Situations

	Article: Q110544
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	kbusage
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for MS-DOS, version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Crop Duster summary window that contains information for Score, Smoke
	expended, Covered area, and Time may appear in other situations. This occurs
	when a different situation has been selected after playing the Crop Duster game.
	
	MORE INFORMATION
	================
	
	The Crop Duster game loads special scenery elements and windows that are
	designed for use with the Crop Duster situation only. To switch to a different
	situation after playing the Crop Duster game, first click Normal Flight from the
	Options menu. After Normal Flight has been chosen, all other situations will
	work normally.
	
	Additional query words: 5.00 5.00a flightsim fltsim dos scorecard FS5
