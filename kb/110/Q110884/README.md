## Q110884: Arcade for Windows: Minimum System Requirements

	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Arcade for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes the minimum system requirements to run Microsoft Arcade
	for Windows, version 1.0.
	
	MORE INFORMATION
	================
	
	The minimum system requirements for Microsoft Arcade for Windows are:
	
	- MS-DOS operating system version 3.2 or later.
	
	- Microsoft Windows version 3.1 or later.
	
	- A 386-SX/20 or faster processor.
	
	- 2 megabytes (MB) of random access memory (RAM).
	
	- A high-density floppy disk drive.
	
	- A VGA or higher resolution video adapter and monitor.
	
	- A Microsoft mouse or compatible pointing device.
	
	- A Sound card plus speakers or headphones required to hear sounds
	  (recommended).
