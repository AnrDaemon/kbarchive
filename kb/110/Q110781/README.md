## Q110781: BUG: Window Controls Are Blank When Using Multiple Monitors

{% raw %}

	Article: Q110781
	Product(s): Microsoft FoxPro
	Version(s): MACINTOSH:2.5b,3.0b
	Operating System(s): 
	Keyword(s): kbbuglist
	Last Modified: 05-FEB-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft FoxPro for Macintosh, version 2.5b 
	- Microsoft Visual FoxPro for Macintosh, version 3.0b 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are using FoxPro for Macintosh with multiple monitors, a window that is
	dragged from a color monitor to a black-and-white monitor will have blank window
	controls. The blank windows controls include the close box, zoom box, and grow
	box. Although these window controls are blank, they are still functional.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
	
	MORE INFORMATION
	================
	
	Steps to Reproduce Problem
	--------------------------
	
	1. Start the Macintosh and FoxPro for Macintosh from the color monitor.
	
	2. Create a screen with the Screen Builder.
	
	3. Drag the screen to the second monitor.
	
	The close box, zoom box, and grow box are blank but are functional.
