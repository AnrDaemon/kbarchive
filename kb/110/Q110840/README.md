## Q110840: Flight Simulator: Altitude Hold Must Be Within 35,000 Feet

{% raw %}

	Article: Q110840
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	kbusage
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for MS-DOS, version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you use the Altitude Hold function (available by choosing Autopilot from
	the Nav/Com menu), the hold elevation must be within 35,000 feet of the current
	elevation.
	
	If a hold elevation is specified more than 35,000 feet above or below the current
	elevation, the autopilot will place the aircraft in a shallow dive. It will not
	ascend or descend to the hold altitude.
