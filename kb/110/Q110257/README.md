## Q110257: Flight Simulator: Landing Light Does Not Turn on Automatically

{% raw %}

	Article: Q110257
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for MS-DOS, version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Flight Simulator 5.0 may not automatically turn on the landing light when Set
	Time Of Day is set to Dusk, Night, or Dawn.
	
	MORE INFORMATION
	================
	
	When Set Time Of Day is set to Dusk, Night, or Dawn, Flight Simulator should
	automatically turn on the instrument lights and all exterior lights; however,
	sometimes it will not turn on the landing light. To force Flight Simulator to
	turn on the landing light, press the L key twice. This will turn the lights off
	and on again and will activate the landing light.
	
	Steps to Reproduce Problem
	--------------------------
	
	1. Start Flight Simulator 5.0.
	
	2. From the World menu, click Set Time And Season.
	
	3. Change the time to Night.
	
	NOTE: Instrument panel lights are on but the landing light is not.
