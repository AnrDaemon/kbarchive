## Q110190: MoreFonts Not Supported with Windows for Workgroups

	Article: Q110190
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 31-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you use MoreFonts with Windows for Workgroups, the mouse cursor may disappear
	but not lose its functionality. However, the opposite may also occur: the mouse
	cursor may remain visible but not function. The keyboard may also lose partial
	or full functionality.
	
	MORE INFORMATION
	================
	
	According to MicroLogic Technical Support, MoreFonts is not supported under any
	version of Windows for Workgroups. This applies to both stand-alone and network
	versions of MoreFonts.
	
	RESOLUTION
	==========
	
	MicroLogic recommends using its MoreType font conversion utility to convert the
	MoreFonts fonts to TrueType fonts. Once the conversion is completed, MoreFonts
	can be completely removed from the system. Contact MicroLogic Technical Support
	for information about the MoreType utility.
	
	
	MoreFonts is manufactured by MicroLogic, a vendor independent of Microsoft; we
	make no warranty, implied or otherwise, regarding this product's performance or
	reliability.
