## Q110545: No Runway, Grid, or Summary in Crop Duster

{% raw %}

	Article: Q110545
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	kbusage
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for MS-DOS, version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	After selecting the Crop Duster situation (CROPDUST.STN), you will not see the
	runway, crop dusting grid, or Summary window, which contains the information for
	score, smoke expended, crops covered, and time.
	
	MORE INFORMATION
	================
	
	The Crop Duster situation (CROPDUST.STN) is only one component of the Crop
	Duster game. It does not load the runway, crop dusting grid, or Summary window.
	
	To load all components of the Crop Duster game, click Entertainment from the
	Options menu and click Crop Duster.
