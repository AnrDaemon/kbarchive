## Q110262: Flight Simulator: Error Code 06 When Editing a Situation

	Article: Q110262
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	kbusage kberrmsg
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for MS-DOS, version 5.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You may receive the following error message and your machine may stop responding
	(hang) if you try to change the title of an existing situation that has no text
	in the Description field:
	
	  Disk Error
	  Error Reading file
	  Filename: FS0001.FSO
	  Error Code: (06)
	
	RESOLUTION
	==========
	
	Add text to the Description field.
	
	1. From the Options menu, click Situations.
	
	2. Select the situation and click Change Information.
	
	3. Select the Description field and type a description for the situation.
	
	MORE INFORMATION
	================
	
	Steps to Reproduce Problem
	--------------------------
	
	1. From the Options menu, click Situations.
	
	2. Select a situation that has no information in the Description field.
	
	3. Click Change Information.
	
	4. Select the Title field.
	
	The error message will appear after selecting the title.
