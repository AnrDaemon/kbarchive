## Q110258: Changing Dynamic Scenery Options Has No Effect

	Article: Q110258
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	kbusage
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for MS-DOS, version 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Changing dynamic scenery options from the Scenery menu may not affect the active
	scenery area.
	
	MORE INFORMATION
	================
	
	Not all scenery areas in Flight Simulator 5.0 have dynamic scenery elements.
	Changing dynamic scenery preferences will not affect those areas that do not
	include dynamic scenery. By default, Flight Simulator 5.0 ships with dynamic
	scenery elements for the following areas:
	
	  USA Chicago, Meigs Field
	  USA Chicago, O'Hare International
	  USA San Francisco, San Francisco International
	  USA San Francisco, Oakland International
