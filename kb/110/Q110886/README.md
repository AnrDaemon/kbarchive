## Q110886: System Requirements for Golf 1.0

	Article: Q110886
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 11-JUN-1999
	
	kbusage
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Golf for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	The following are the system requirements for Microsoft Golf version 1.0:
	
	- MS-DOS operating system version 3.1 or later
	
	- Microsoft Windows version 3.0 or later
	
	- 80286 or higher processor
	
	- 2 MB RAM
	
	- High-density disk drive or low-density (720K) disk drive, and a hard drive
	
	- VGA or higher resolution monitor required
	
	- Microsoft Mouse or compatible pointing device
	
	- Sound board recommended
