## Q160481: Encarta 97: Setup Status Never Reaches 100 Percent

{% raw %}

	Article: Q160481
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1997
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 97 Encyclopedia for Windows 
	- Microsoft Encarta Encyclopedia 97 Deluxe for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to install Encarta Encyclopedia, you may be notice that the
	installation status bar does not go all the way to 100 percent.
	
	MORE INFORMATION
	================
	
	Before the installation status bar for Encarta Encyclopedia reaches 100 percent,
	the Setup has advanced to the final portion of the Setup program which includes
	profiling the display and creating program icons.
	
	You should consider Encarta completely and correctly setup once the icons are
	created and you receive the Run Encarta option.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Encarta Encyclopedia for Windows
	1997 edition. We are researching this problem and will post new information here
	in the Microsoft Knowledge Base as it becomes available.
