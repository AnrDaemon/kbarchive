## Q160285: STOP 0x0000000a in Ntoskrnl.exe with InocuLAN Virus Software

	Article: Q160285
	Product(s): Microsoft Windows NT
	Version(s): WinNT:3.51
	Operating System(s): 
	Keyword(s): kb3rdparty
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 3.51 
	- Microsoft Windows NT Server version 3.51 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You may receive the following blue screen STOP error message while using
	InocuLAN for Windows NT version 1.01 build 50:
	
	  STOP 0x0000000A (0x00000001, 0x00000002, 0x00000001, 0x8010D8C5)
	  IRQL_NOT_LESS_OR_EQUAL Address 8010D8C5 has base at 80100000
	  NTOSKRNL.EXE
	
	NOTE: The parameters in parenthesis are specific to your system configuration,
	and may differ from the above information.
	
	RESOLUTION
	==========
	
	To resolve this problem, please contact Cheyenne technical support for
	information about an updated version of InocuLAN for Windows NT.
