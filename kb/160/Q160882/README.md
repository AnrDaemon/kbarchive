## Q160882: Additional 4mm dat Tape Support in Windows NT 4.0 SP2

	Article: Q160882
	Product(s): Microsoft Windows NT
	Version(s): WinNT:4.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbhw kbHardware
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Windows NT 4.0 Service Pack 2 and later includes support for the following tape
	drives:
	
	  DEC_DLTTAPE_TAPE = "DEC", "TZ89" which is the DEC Model TZ89.
	  4MMDAT_TAPE = "DEC", "TLZ9"      which is the DEC Model TLZ9.
	  4MMDAT_TAPE = "Sony", "SDT-9000" which is the Sony Model SDT-9000.
	
	For information on obtaining the service pack, query on the following word in the
	Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
	
	Additional query words: sp2
