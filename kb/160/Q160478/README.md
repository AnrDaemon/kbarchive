## Q160478: Encarta 97: Copying Climate Charts Cuts Off Decimal Amounts

{% raw %}

	Article: Q160478
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 97 Encyclopedia for Windows 
	- Microsoft Encarta Encyclopedia 97 Deluxe for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you Copy and Paste Climate Chart information from the Climate Interactivity
	in Encarta Encyclopedia, the decimal portion of the value becomes cut off or
	truncated.
	
	MORE INFORMATION
	================
	
	
	Microsoft has confirmed this to be a problem in Encarta Encyclopedia for Windows
	1997. We are researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
