## Q160167: NBA Full Court: Incorrect 256 Color Error Message During Setup

	Article: Q160167
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbdisplay
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft NBA Full Court Press for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	During the Setup procedure of NBA Full Court Press, the following error will
	occur if a 16-color palette is used:
	
	  Setup has detected that your computer does not support 256 colors, which is
	  required to run Soccer.
	
	The phrase "NBA Full Court Press" should have been used in place of the word
	"Soccer"
	
	CAUSE
	=====
	
	The computer must display 256 colors to install and run NBA Full Court Press.
	
	RESOLUTION
	==========
	
	To display 256 colors in Windows:
	
	1. Click Start, point to Settings, and then click Control Panel.
	
	2. Double-click the Display icon.
	
	3. Click the Settings tab.
	
	4. In the Color Palette or Colors box, click 256 Color.
	
	  NOTE: If the computer doesn't have the necessary 256 color driver, or if the
	  user isn't sure that the video card supports 256 colors, contact the video
	  card manufacturer.
	
	5. Click OK.
	
	6. Restart Windows.
	
	Additional query words: 1.00 setup 16 color
