## Q160835: FP: Renamed Executable Folder No Longer Executable

	Article: Q160835
	Product(s): Word Front Page
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdta
	Last Modified: 10-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft FrontPage 97 for Windows with Bonus Pack 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you rename an executable folder in the FrontPage Explorer, the folder loses
	its executable property.
	
	RESOLUTION
	==========
	
	After you rename the folder, right-click the folder, and then click Properties.
	On the General tab, click to select the "Allow scripts or programs to be run"
	check box, and then click OK.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article. This problem was corrected in Microsoft FrontPage
	98 for Windows.
