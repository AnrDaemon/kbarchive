## Q160241: WD97: Page Break Deleted After Drawing Table

	Article: Q160241
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbualink97
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you draw a table immediately after a manual page break, the page break is
	deleted.
	
	WORKAROUND
	==========
	
	To work around this problem, use the following steps:
	
	1. Press ENTER to insert a paragraph immediately following the page break.
	
	2. Draw the table below the inserted paragraph.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows.
