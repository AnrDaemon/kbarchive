## Q160795: Incorrect Application Jumps to Foreground

	Article: Q160795
	Product(s): Microsoft Windows NT
	Version(s): 4.0,4.2c
	Operating System(s): 
	Keyword(s): kbui
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When opening, minimizing, and maximizing multiple Microsoft Word and Microsoft
	Excel files from Microsoft Office 4.2c, the incorrect application jumps to the
	foreground.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 4.0. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
	
	Additional query words: maximize foreground application word excel winnt
