## Q160851: MSN Setup Does Not Connect If Third-Party Connection Selected

	Article: Q160851
	Product(s): The Microsoft Network
	Version(s): 2.00
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	At the completion of Setup for MSN, The Microsoft Network, if you select "Signup
	for MSN but use my existing Internet Connection," you receive the following
	message when Setup attempts to connect:
	
	  Cannot find site Http://signup.msn.com/
	
	CAUSE
	=====
	
	Setup is designed to connect to MSN signup using MSN.
	
	RESOLUTION
	==========
	
	Connect to your Internet service provider, and then go to:
	
	  http://signup.msn.com/
