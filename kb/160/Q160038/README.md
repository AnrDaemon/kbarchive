## Q160038: Socket EA Type II PC Card Hangs HP 800CT

	Article: Q160038
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 2,2.1
	Operating System(s): 
	Keyword(s): osr2 win95
	Last Modified: 31-JAN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 OEM Service Release, versions 2, 2.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you insert a Socket EA Type II PC Card into a CardBus socket in a
	Hewlett-Packard (HP) 800CT computer, the computer may stop responding (hang)
	when the card is initialized and the drivers are loaded.
	
	CAUSE
	=====
	
	This may be an I/O problem with hardware speed transfer rates when the Socket EA
	Type II PC Card is used in combination with the HP 800CT laptop TI 1130 CardBus
	controller.
	
	RESOLUTION
	==========
	
	Use a Socket EA Ethernet Type I PC Card, or another Ethernet PC Card.
