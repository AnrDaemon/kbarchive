## Q160249: Duplicate Chats Appear in Favorite Chats List

	Article: Q160249
	Product(s): The Microsoft Network
	Version(s): WINDOWS:2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-AUG-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Your Favorite Chats list may contain the same chat more than once.
	
	CAUSE
	=====
	
	The Favorite Chats list is set up to contain five chats. If you select fewer
	than five items for the list, chats are randomly added to the list. A randomly
	added chat may be the same as one you selected for the list.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
