## Q160335: RealAudio Plays at 14,400 bps

{% raw %}

	Article: Q160335
	Product(s): The Microsoft Network
	Version(s): WINDOWS:2.0,2.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 2.0, 2.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	RealAudio plays at 14,400 bps on MSN, The Microsoft Network, even if you are
	using a higher-speed modem.
	
	MORE INFORMATION
	================
	
	RealAudio lets you listen to audio files in real-time over a dial-up modem or
	direct network Internet connection. Currently, MSN Setup sets the speed for
	RealAudio at 14,400 bps and does not offer you the option to change the speed.
