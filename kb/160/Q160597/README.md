## Q160597: Mac Multimedia: Page Setup Option Doesn't Function Properly

	Article: Q160597
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1997 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 22-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Cinemania for the Macintosh 1997 edition 
	- Microsoft Music Central for the Macintosh 1997 edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you select Page Setup on the File menu in Cinemania 97 or Music Central 97,
	the Page Setup window does not appear. The Page Setup menu option appears to be
	non functional.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem the programs listed at the top of
	this article. We are researching this problem and will post new information here
	in the Microsoft Knowledge Base as it becomes available.
	
	MORE INFORMATION
	================
	
	You may notice that when you pull down the File menu while running Cinemania 97
	or Music Central 97, the Page Setup menu option is unavailable (grayed out).
	Only after you have pulled down the Find menu is the Page Setup menu option
	available (although it still does not function).
