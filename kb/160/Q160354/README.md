## Q160354: Mouse and Keyboard Can Disappear when Replacing Drivers

	Article: Q160354
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kbhw kbsetup kbHardware
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The mouse and keyboard can stop functioning when replacing drivers from the
	Control Panel.
	
	Installing the Microsoft Natural Intellipoint Keyboard drivers can cause loss of
	functionality or possible stop screens.
	
	Upgrading a previous version of Windows NT mouse driver can cause the mouse to
	disappear.
	
	WORKAROUND
	==========
	
	The problems exhibited by the keyboard and mouse have been fixed in Windows NT
	4.0 Service Pack 2 and later.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 4.00. This
	problem has been corrected in the latest U.S. Service Pack for Windows NT
	version 4.00. For information on obtaining the Service Pack, query on the
	following word in the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
	
	Additional query words: sermouse prodnt
