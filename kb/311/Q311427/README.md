## Q311427: MSN Explorer Background Appears White in Microsoft Windows XP

	Article: Q311427
	Product(s): The Microsoft Network
	Version(s): 6.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network Version 7.0 
	- The Microsoft Network version 6.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you upgraded your operating system to Microsoft Windows XP, your MSN
	Explorer color scheme turned white.
	
	CAUSE
	=====
	
	Your Microsoft Windows XP "theme" setting is causing this issue.
	
	RESOLUTION
	==========
	
	Change the theme from XP to Classic, and then back to XP. This should restore
	MSN Explorer background colors.
	To change the theme from XP to Classic, and then back to XP
	
	1. Click Start, and then click Control Panel.
	
	2. Under Pick a Category, click Appearance and Themes.
	
	3. Under Pick a task... click Change the computer's theme.
	
	4. Click the Theme dropdown and select Windows Classic.
	
	5. Click Apply, click the Theme dropdown, and then select Windows XP.
	
	6. Click Apply, and then cick OK.
	
	7. Start MSN Explorer and check the colors.
	  This should restore MSN Explorer background colors.

	================
