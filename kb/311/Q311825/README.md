## Q311825: SMS: IME May Not Be Viewable in a Remote Control Session

{% raw %}

	Article: Q311825
	Product(s): Microsoft Systems Management Server
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): kbenv kbtool kbui kbsms200 kbsms200bug
	Last Modified: 06-DEC-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use Systems Management Server (SMS) to remotely control Microsoft
	Windows XP-based computers, you may not be able to locate the Input Method
	Editor (IME), or the Language bar, in the SMS Remote Control session.
	
	WORKAROUND
	==========
	
	To work around this problem, attach the IME bar to the task bar before you start
	the SMS Remote Control session.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 2.0.
