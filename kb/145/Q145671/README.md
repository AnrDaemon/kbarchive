## Q145671: How to Uninstall Microsoft Pregnancy and Child Care Program

	Article: Q145671
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Pregnancy and Child Care, Preview on MSN 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article tells how to remove the Microsoft Pregnancy and Child Care program
	files from your computer. This makes available 4 MB of hard disk space.
	
	MORE INFORMATION
	================
	
	To uninstall Microsoft Pregnancy and Child Care from your hard drive, do the
	following:
	
	1. Click the Start button, point to Settings, and then click Control Panel.
	
	2. Double-click Add/Remove Programs.
	
	3. On the Install/Uninstall tab, click Microsoft Pregnancy and Child Care, and
	  then click Add/Remove.
	
	4. Click OK.
	
	5. When the following message appears, click OK.
	
	  Some files and/or directories could not be deleted
	
	6. When you are told the program has been removed, click OK.
	
	Manually Removing the Program
	-----------------------------
	
	If you have problems using the Add/Remove Programs feature, manually remove the
	program from your hard drive:
	
	1. Open the following folder:
	
	  C:\Program Files\The Microsoft Network\Custom Applications
	
	  Where C is the letter of your hard drive where you installed Windows.
	
	2. Delete the Pregnancy And Child Care folder and its contents.

	questions about using Microsoft applications. To access the Knowledge Base,
	return to the container where you found this document and click the Microsoft
	Knowledge Base icon.
