## Q145903: XCLN: Buttons/Menus Unavailable When Opening New Mail

	Article: Q145903
	Product(s): Microsoft Exchange
	Version(s): 4.0 5.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 16-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Windows 3.x client, versions 4.0, 5.0 
	- Microsoft Exchange Windows NT client, versions 4.0, 5.0 
	- Microsoft Exchange Windows 95/98 client, versions 4.0, 5.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In the Microsoft Exchange Windows 3.x, Windows 95, and Windows NT clients, when
	you open a message by clicking the Yes button in the New Mail Notification
	dialog box, the Next and Previous buttons and the menus are unavailable.
	
	RESOLUTION
	==========
	
	Close the message and reopen the message from the message viewer. All the
	functions will now be available.
	
	Additional query words: new message
