## Q145933: XGEN: X.400 &quot;OR Attribute Name&quot; Length Values

	Article: Q145933
	Product(s): Microsoft Exchange
	Version(s): 4.0 5.0
	Operating System(s): 
	Keyword(s): kbnetwork
	Last Modified: 13-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft Exchange is compliant with the X.400 1988 and 1984 EN/ENV
	41.201/41.202 profiles for addressing schemes for OR Names. The following is the
	standard table that Microsoft Exchange is compliant with and the length values
	that it supports for OR Name attributes:
	
	  Attribute                    Length
	  -------------------------------------------------
	  Country Name                   3 characters
	  ADMD                          16 characters
	  PRMD                          16 characters
	  OrgName                       64 characters
	  FreeFormName (Common Names)   64 characters
	  OrgUnits 1-4                  32 characters
	  DDA Type                       8 characters each
	  DDA value                    128 characters each
	  Surname                       40 characters
	  Given Name                    16 characters
	  Initials                       5 characters
	  GQ                             3 characters
