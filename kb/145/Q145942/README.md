## Q145942: Two Scroll Bars Appear in Internet Service Manager

	Article: Q145942
	Product(s): Internet Information Server
	Version(s): winnt:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-JAN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Internet Information Server 1.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you reduce the size of the Internet Service Manager window vertically to a
	list of at least four servers or services or the minimum size horizontally, two
	scroll bars appear.
	
	WORKAROUND
	==========
	
	Do not reduce the size of the Internet Service Manager window. The problem
	mentioned above does not affect the performance of the services.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Internet Information Server
	version 1.0. We are researching this problem and will post new information here
	in the Microsoft Knowledge Base as it becomes available.
