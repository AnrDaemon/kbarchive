## Q145866: Scenes 1.0: Genigraphics Scanning Special Offer Expired

	Article: Q145866
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Scenes (all collections), version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Included in the Scenes package is a Genigraphics Corporation order form. This
	order form mentions a special offer. The special offer expired on Jan. 1, 1994.
	
	MORE INFORMATION
	================
	
	The order form includes a special offer for up to five images scanned free with
	the purchase of a specified number of images.
	
	Printed on the order form is the statement, "Offer subject to change or
	cancellation without notice after January 1, 1994."
	
	This order form can still be used to order scanned images. Please call
	Genigraphics Corporation for current pricing.
	
	- Genigraphics Corporation: (800) 638-7348

	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
