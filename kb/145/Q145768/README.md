## Q145768: Cannot View Member Properties in a Newsgroup

	Article: Q145768
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.0,1.05,1.2,1.3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.0, 1.05, 1.2, 1.3 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you are reading a newsgroup message posted by a member of MSN, The Microsoft
	Network, and you double-click the sender's name, you receive the following
	message:
	
	  Cannot display member properties. Sender is not a member of The Microsoft
	  Network
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
	
	Additional query words: msn news group
