## Q145804: Pregnancy: What Do I Do with Problems Not Addressed in FAQ?

{% raw %}

	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Pregnancy and Child Care, Preview on MSN 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you have problems when using Microsoft Pregnancy and Child Care and the
	Frequently Asked Questions bulletin board does not provide a resolution, you can
	post your support questions to the Microsoft Pregnancy and Child Care Support
	Desktop bulletin board.
	
	MORE INFORMATION
	================
	
	The Microsoft Pregnancy and Child Care Support Desktop bulletin board is located
	in the Microsoft Support Desktop folder.
	
	To access the Support Desktop bulletin board, see the following shortcut to exit
	the Frequently Asked Questions bulletin board and switch to the Microsoft
	Pregnancy and Support Desktop bulletin board.
	
	  Microsoft Pregnancy and Child Care Support Desktop
	
	  Categories\Computers & Software\Software\All Software
	  Companies\Microsoft\Microsoft Support Desktop\Microsoft Product Support
	  Desktop\MS Home Support Desktop\Microsoft Pregnancy and Child Care
