## Q145722: Where Can I Go to Discuss Pregnancy and Child Care Issues?

	Article: Q145722
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): 
	Last Modified: 01-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Pregnancy and Child Care, Preview on MSN 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Pregnancy and Child Care host application offers discussion on
	these subjects. These discussions are located on the following bulletin boards
	on MSN, The Microsoft Network online service:
	
	Pregnancy:
	
	  Categories\Health & Fitness\Microsoft Pregnancy and Child Care\Bulletin
	  Boards\Pregnancy
	
	Child Care:
	
	  Categories\Health & Fitness\Microsoft Pregnancy and Child Care\Bulletin
	  Boards\Child Care
	
	MORE INFORMATION
	================
	
	The Microsoft Pregnancy and Child Care folder located on MSN, The Microsoft
	Network online service contains a bulletin board (BBS) folder that guides you to
	the discussion bulletin boards. Use either of the above shortcuts to exit the
	Frequently Asked Questions bulletin board and switch to the respective bulletin
	board.
