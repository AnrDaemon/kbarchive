## Q322033: How To Import E-mail From Netscape Mail to Microsoft Entourage

	Article: Q322033
	Product(s): The Microsoft Network
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-APR-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Internet Access for Macintosh, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to import e-mail from Netscape Mail to Microsoft
	Entourage.
	
	MORE INFORMATION
	================
	
	To import e-mail from Netscape Mail to Microsoft Entourage
	
	1. Start Microsoft Entourage.
	
	2. Click File, and then click Import.
	
	3. In the Import window, select Import information from one of these programs,
	  and then click Netscape Communicator.
	
	4. Click the right arrow to continue.
	
	5. Select Messages as the data you want to import.
	
	6. Click the right arrow to continue, and then click Finish.
	  Your imported e-mail messages will be located in the Communicator Import
	  folder.
