## Q322360: List of UNIX Platforms on Which Passport Is Supported

	Article: Q322360
	Product(s): Microsoft Developer Network
	Version(s): 1.1,1.4
	Operating System(s): 
	Keyword(s): kbDSupport
	Last Modified: 21-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Passport, versions 1.1, 1.4 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Passport Software Development Kit (SDK) has been ported to run on
	several UNIX platforms. This article lists the UNIX platforms on which Passport
	is supported.
	
	MORE INFORMATION
	================
	
	Passport 1.1
	------------
	
	Passport version 1.1. for UNIX is supported on:
	
	- Sun Solaris 2.6
	- Sun Solaris 2.7
	- RedHat Linux 6.1
	- RedHat Linux 6.2
	- IBM AIX 4.3, 4.3.3
	- HPUX 11.0
	- FreeBSD 3.x
	
	Passport 1.4
	------------
	
	Passport version 1.4 for UNIX is supported on:
	
	- Sun Solaris 2.7
	- Sun Solaris 2.8
	- RedHat Linux 6.1
	- RedHat Linux 6.2
	
	Note that Passport may run succesfully on other versions of these operating
	systems. However, only those versions that are listed in this article are
	supported. There are no plans at this time to certify other versions.
	
	For additional information, visit the following Microsoft Web site:
	
	  http://www.microsoft.com/myservices/passport/overview.asp
