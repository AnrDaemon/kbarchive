## Q214584: Close Combat III: Game Does Not Start at High Resolutions

	Article: Q214584
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbenv kbimu msgamekbfaq
	Last Modified: 12-APR-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Close Combat III: The Russian Front 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You may be unable to start Microsoft Close Combat III: The Russian Front on a
	Windows 95-based computer if you use a game resolution of 1280 x 1024 pixels.
	
	CAUSE
	=====
	
	This behavior can occur if both of the following conditions are true:
	
	- A StarFighter AGP video adapter is installed in your computer.
	
	- You are using version 4.10.1.259 or earlier of the video driver for the
	  Starfighter AGP video adapter.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest video driver for the
	StarFighter AGP video adapter from the following Real 3D Web site:
	
	  http://www.real3d.com/
	
	MORE INFORMATION
	================
	
	To work around this issue, use a game resolution of 1024 x 768 pixels or 800 x
	600 pixels.
