## Q214553: Close Combat III: Game Screen Is Too Large to Fit on Monitor

	Article: Q214553
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgamekbfaq
	Last Modified: 12-APR-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Close Combat III: The Russian Front 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Close Combat III: The Russian Front, the game screen may
	be too large to fit on your monitor and may be cut off.
	
	CAUSE
	=====
	
	This behavior can occur if all of the following conditions are true:
	
	- An STB Nitro 3D video adapter is installed in your computer.
	
	- You are using the STB Vision 95 resolution-switching utility.
	
	- The Close Combat III game resolution is set to 1152 x 864 pixels or higher.
	
	RESOLUTION
	==========
	
	To resolve this issue, install the latest video driver available for your STB
	video adapter. You can obtain the latest video driver for the STB Nitro 3D video
	adapter from the following STB Web site:
	
	  http://www.stb.com/
	
	You can also work around this issue by reducing the game resolution. To do this,
	follow these steps:
	
	1. Start Close Combat III: The Russian Front.
	
	2. On the startup screen, click Options.
	
	3. Click the General tab.
	
	4. Click 1024 x 768 @ 16bpp or 800 x 600 @ 16bpp in the Screen Resolution box
	  under Performance Options, and then click OK.
