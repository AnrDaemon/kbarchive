## Q214587: Close Combat III: Game Quits When You Move Pointer Over Unit

{% raw %}

	Article: Q214587
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbnetwork kbimu msgamekbfaq
	Last Modified: 12-APR-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Close Combat III: The Russian Front 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Close Combat III: The Russian Front with another player,
	the game may quit without warning when you move the pointer over a unit.
	
	NOTE: This behavior occurs very rarely.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Close Combat III: The
	Russian Front, version 1.0.
	
	
	Additional query words: 1.00 msgame closecombat russianfront cc3 cciii stop crash
