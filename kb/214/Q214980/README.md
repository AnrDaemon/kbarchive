## Q214980: SMS: Mouse Not Reported in Hardware Inventory on 16-Bit Clients

	Article: Q214980
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:2.0
	Operating System(s): 
	Keyword(s): kbenv kbhw kbsms200 kbHardware
	Last Modified: 25-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you check the hardware inventory on a 16-bit client computer using Systems
	Management Server 2.0, a mouse driver may not be displayed or you may see the
	following message in the Pointing Devices inventory group:
	
	  No Mouse Installed
	
	Microsoft has confirmed this to be a problem in Systems Management Server version
	2.0.
	
	Additional query words: prodsms hwinv16
