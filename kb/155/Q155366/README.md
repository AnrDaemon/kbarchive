## Q155366: Error Message Trying to Run OSR2 Compression Agent

{% raw %}

	Article: Q155366
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	- Microsoft Plus! for Windows 95 
	- Microsoft Windows 95 OEM Service Release, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to run Compression Agent in Windows 95 OEM Service Release 2
	(OSR2), you may receive the following error message:
	
	  Microsoft Plus! is not installed.
	
	  The Compression Agent supports additional compression types, which deliver up
	  to 50% more compression than standard compression. In order to use the
	  Compression Agent you must install Microsoft Plus!.
	
	CAUSE
	=====
	
	Compression Agent (Cmpagent.exe) does not work in OSR2 unless Microsoft Plus! is
	installed.
	
	RESOLUTION
	==========
	
	To use Compression Agent, install Microsoft Plus! for Windows 95.
