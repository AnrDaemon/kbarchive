## Q155166: Fonts That Do Not Change After Reapplying Slide Master

	Article: Q155166
	Product(s): Microsoft PowerPoint for Windows
	Version(s): WINDOWS:4.0,7.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 16-APR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft PowerPoint for Windows 95, version 7.0 
	- Microsoft PowerPoint for Windows, version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you reapply the Slide Master in your PowerPoint presentation, some or all
	of your fonts may not be changed.
	
	CAUSE
	=====
	
	Some special fonts are not affected by reapplying the Slide Master and may not
	change. The following is a list of fonts not affected by reapplying the Slide
	Master in PowerPoint:
	
	- Wingdings
	
	- Symbol
	
	- Monotype Sorts
	
	- MS Line Draw
	
	- Dingbats
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft PowerPoint for
	Windows, version 4.0 and PowerPoint for Windows 95, version 7.0. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
