## Q155740: .avi File Hangs When Played with MIDI on Alpha Computer

	Article: Q155740
	Product(s): Microsoft Windows NT
	Version(s): winnt:3.1,3.51,4.0
	Operating System(s): 
	Keyword(s): kbsound
	Last Modified: 10-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server versions 3.1, 3.51, 4.0 
	- Microsoft Windows NT Workstation versions 3.1, 3.51, 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play MIDI and .avi files simultaneously on an Alpha-based computer with
	a Media Vision ProAudio Spectrum 16 or Media Vision Pro Audio Studio sound card,
	the .avi file seems to stop playing every few seconds and plays without sound.
	This problem also affects .wav files.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT 4.0. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
