## Q155644: Available Resources for Windows NT 4.0 Deployment

	Article: Q155644
	Product(s): Microsoft Windows NT
	Version(s): 
	Operating System(s): 
	Keyword(s): kbsetupkbfaq
	Last Modified: 10-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server 
	- Microsoft Windows NT Workstation 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	This article lists some available information and resources designed to aid in
	the deployment of Microsoft Windows NT.
	
	MORE INFORMATION
	================
	
	The Windows NT Deployment Guide, the Automating Windows NT Setup Deployment
	Guide, and other resources designed to aid in the deployment of Microsoft
	Windows NT are available for free download from the following Microsoft Web
	site:
	
	  http://www.microsoft.com/ntworkstation/deployment/deployment/default.asp
	
	In addition, the product documentation for Windows NT Server, Windows NT
	Workstation, and the Windows NT 4.0 Resource Kit provide much of the information
	required concerning deployment of Microsoft Windows NT 4.0.
	
	The following resources offer fee-based support for the deployment of Microsoft
	products:
	
	- Microsoft Consulting Line (800) 936-5200
	
	- Microsoft Consulting Services (MCS) (800) 426-9400
	  http://www.microsoft.com/msconsult/
	
	- Microsoft Gold Certified Support Partners for Support Services
	  http://www.microsoft.com/support/partner/
