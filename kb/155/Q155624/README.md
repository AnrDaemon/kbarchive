## Q155624: Cannot Print Gray Scale to LexMark Optrac PS Printer

	Article: Q155624
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbprint kbPrinting
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you print a color bitmap and select the gray scale option in the Print
	dialog box, the picture may be printed in color instead of gray scale.
	
	CAUSE
	=====
	
	The gray scale option has no effect on bitmap images.
	
	STATUS
	======
	
	This behavior is by design.
