## Q155362: Problems Viewing Driver Information in OSR2 Device Manager

	Article: Q155362
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	- Microsoft Windows 95 OEM Service Release, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you view the Drivers tab for devices listed in Device Manager in Windows 95
	OEM Service Release 2 (OSR2), you may see the following status:
	
	  No driver files are required or have been loaded for this device
	
	In addition, no information may be available in the Provider and Version
	listings.
	
	RESOLUTION
	==========
	
	Use the updated Hardware Diagnostics tool (Hwdiag.exe) located on the OSR2
	CD-ROM in the Other\Misc\Hwtrack folder.
	
	This tool includes the same functionality as Device Manager as well as several
	other options for viewing information. To view a report about all the devices in
	your system, run Hwdiag.exe and click All Devices on the View menu.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Windows 95 OSR2. We
	are researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
