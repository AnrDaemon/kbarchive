## Q155679: Cannot Use Defragment Now Button in Drive Properties

{% raw %}

	Article: Q155679
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbtool
	Last Modified: 10-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Defragment Now button on the Tools tab in the properties for a drive may be
	unavailable.
	
	CAUSE
	=====
	
	This behavior occurs if there is no disk defragmenter tool installed. Windows NT
	4.0 does not include a disk defragmenter tool.
	
	RESOLUTION
	==========
	
	To use the Defragment Now button, you must first install a third-party disk
	defragmenter tool.
