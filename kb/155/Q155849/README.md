## Q155849: Tape Drive Error Message When Cataloging Tape

	Article: Q155849
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbhw kbHardware
	Last Modified: 10-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are cataloging a tape, you may receive the following hardware error
	message in Microsoft Windows NT Backup using a Sony 7000 4-milli- meter digital
	audio tape (DAT) device:
	
	  Tape Drive Error
	
	CAUSE
	=====
	
	This behavior can occur if the Sony 7000 4-millimeter DAT device has older
	firmware.
	
	RESOLUTION
	==========
	
	Contact Sony to update the device with the latest firmware revision.
