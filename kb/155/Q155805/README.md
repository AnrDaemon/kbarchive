## Q155805: XCLN: Macintosh: Accessing Client Software on 4.0 CD

	Article: Q155805
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:4.0,5.0
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 04-APR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Macintosh client, versions 4.0, 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Macintosh client software on the Microsoft Exchange client compact disc or
	Microsoft Exchange 4.0 Service Pack 2 compact disc is only accessible from a
	Macintosh CD-ROM.
	
	MORE INFORMATION
	================
	
	The Macintosh client software on the Microsoft Exchange client compact disc or
	the Microsoft Exchange 4.0 Service Pack 2 compact disc cannot be seen from a
	Windows based CD-ROM. It must be read using a Macintosh compatible CD-ROM. If
	necessary, a Macintosh can be used to copy these files to a Macintosh share on a
	Windows NT server for network access.
