## Q155725: Description Window in Devices Tool Is Too Small

	Article: Q155725
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbtool kbui
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In the Devices tool in Control Panel, the window for the description of devices
	is too small for some descriptions to be read fully, and there is no horizontal
	scroll bar to let you see the rest of the description.
	
	STATUS
	======
	
	This is a known problem with Windows NT 4.0.
