## Q155693: Calling Card Does Not Work During MSN Signup

	Article: Q155693
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.3,2.0,2.5,2.51,2.52,2.6
	Operating System(s): 
	Keyword(s): kberrmsg kbmsn
	Last Modified: 14-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.3, 2.0, 2.5, 2.51, 2.52, 2.6 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use a telephone calling card during the sign-up process for MSN, The
	Microsoft Network, you receive the following error message:
	
	  There was a problem completing this task, would you like to try again?
	
	CAUSE
	=====
	
	You cannot use a telephone calling card during the MSN sign-up process.
	
	Additional query words: msn 2.00 2.50 2.60
