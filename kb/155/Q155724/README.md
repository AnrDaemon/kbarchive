## Q155724: Energy Star Compliance Feature Unavailable in Windows NT 4.0

{% raw %}

	Article: Q155724
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbhw kbHardware
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The power-saving features of your Energy Star-compliant monitor do not work in
	Windows NT 4.0.
	
	CAUSE
	=====
	
	Windows NT 4.0 does not support the power-saving features of Energy Star-
	compliant monitors.
	
	STATUS
	======
	
	This behavior is by design. This feature may be added in a future release of
	Windows NT.
