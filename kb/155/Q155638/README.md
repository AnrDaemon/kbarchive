## Q155638: Monster Truck Madness: Minimum System Requirements

	Article: Q155638
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0

	Keyword(s): kbmm kbrefkbfaq

	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Monster Truck Madness, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains the minimum system requirements for Microsoft Monster
	Truck Madness.
	
	MORE INFORMATION
	================
	
	The following are the minimum system requirements to run Monster Truck Madness:
	
	- Pentium processor
	
	- Windows 95 or later operating system
	
	- 8 MB Memory (12 MB recommended)
	
	- 20 MB available hard disk space for compact install (100 MB for typical
	  install)
	
	- 256 Color 640 x 480 display
	
	- 2X CD-ROM (not required for trial version)
	
	- Audio board plus speakers or headphones (required to hear audio)
	
	- Hayes-compatible modem --14.4Kbps for head-to-head play, (9600 kbps or higher
	  recommended for connection to the Monster Truck Web Stop site)
	
	Recommended:
	
	- A joystick or compatible device (optional, race car controller also
	  supported)
	
	- Microsoft mouse or compatible pointing device
	
	- For best play, 12 MB (or more) RAM
