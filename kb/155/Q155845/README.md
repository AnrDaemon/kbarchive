## Q155845: Mouse Distorts Display on MIPS NEC RISCstation 2200 with TGA

	Article: Q155845
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbdisplay
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you move the mouse pointer across controls, dialog boxes, and so on, the
	display may become distorted directly above the pointer. The distortion looks
	like a piece of the mouse pointer.
	
	CAUSE
	=====
	
	This problem occurs on the MIPS NEC RISCStation 2200 with a built-in Digital TGA
	display adapter.
	
	RESOLUTION
	==========
	
	This is a hardware issue. Please contact your hardware vendor for assistance
	with this problem.
	
	MORE INFORMATION
	================
	
	The following information is displayed in the Display Type dialog box if your
	computer has the affected TGA display adapter:
	
	  Chip Type: Digital 21030
	  DAC Type: Brooktree Bt485
	  Memory Size: 2MB
	  Adapter String: Digital ZLXp-E1
	
	Additional query words: prodnt corrupt
