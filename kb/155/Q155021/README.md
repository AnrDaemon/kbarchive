## Q155021: Cannot Cancel Location Information

	Article: Q155021
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kbui dun kbDialUp
	Last Modified: 13-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	When you start the Telephony tool in Control Panel, you cannot cancel the
	Location Information dialog box that appears.
	
	MORE INFORMATION
	================
	
	You must enter an area code to close this dialog box. If you are unsure what
	number to enter, enter any number and then correct the number later when you
	know what number to use.
	
	Once you have entered an area code and closed the dialog box, you can start the
	Telephony tool and close it with a blank area code field.
