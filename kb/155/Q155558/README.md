## Q155558: Deleting Clipbook Page Causes Overwrite of Clipboard Contents

{% raw %}

	Article: Q155558
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbtool
	Last Modified: 10-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you delete a clipbook page in Clipbook Viewer, the data in the local
	Clipboard is overwritten.
	
	STATUS
	======
	
	This is a known problem with the Windows NT 4.0 Clipbook Viewer. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
