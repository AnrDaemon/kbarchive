## Q155395: Multiple Instances of Control Panel Tool Started

{% raw %}

	Article: Q155395
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbtool
	Last Modified: 10-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you double-click an icon in Control Panel twice quickly (for a total of
	four clicks), two instances of the Control Panel tool are started.
	
	This issue affects the following Control Panel tools:
	
	- Add/Remove Programs
	
	- Devices
	
	- Display
	
	- Modems
	
	- Monitoring Agent
	
	- Mouse
	
	- Multimedia
	
	- Network
	
	- PC Card
	
	- Regional Settings
	
	- Server
	
	- Services
	
	- Sounds
	
	- System
	
	- Tape Devices
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Windows NT version
	4.0. We are researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
	
	MORE INFORMATION
	================
	
	If you double-click one of the tools listed above, pause, then double- click the
	tool again, a second instance of the tool is not started. Instead, focus is
	returned to the first instance of the tool.
