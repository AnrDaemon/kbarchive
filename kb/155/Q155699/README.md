## Q155699: Display Type Dialog Box Does Not Detect DAC Type

	Article: Q155699
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbdisplay
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The DAC type information is not provided in the Display Type dialog box for
	Number Nine display adapters with the S3-928 chip set.
	
	CAUSE
	=====
	
	Windows NT 4.0 does not detect the DAC (digital-to-analog converter) on Number
	Nine display adapters with the S3-928 chip set.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 4.0. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
