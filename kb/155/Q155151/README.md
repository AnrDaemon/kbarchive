## Q155151: Writer 2: Help Is Limited in Certain Areas

	Article: Q155151
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Creative Writer for Windows, version 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In most of the areas in the program, there is both audio and visual help
	available when you hold the mouse over a button or tool. However, the following
	areas do not have one or both types of help:
	
	- Picture Picker
	
	- Border Picker
	
	- Texture Picker
	
	- Splot Machine
	
	- Story Starter
	
	- WordArt (Press F1 to access tool tips for each item in this dialog box)
	
	MORE INFORMATION
	================
	
	Use the Help tool (the green question mark in the upper-right corner of the
	program) to find out more about how to use the various other tools in the
	program.
