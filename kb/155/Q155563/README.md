## Q155563: Toshiba T4900CT Laptop Computer Hangs at NTDETECT Screen

	Article: Q155563
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbhw kbHardwarekbfaq
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	The Toshiba T4900CT laptop computer may stop responding (hang) at the NTDETECT
	screen.
	
	RESOLUTION
	==========
	
	Upgrade the computer's BIOS to version 1.50 or later.
