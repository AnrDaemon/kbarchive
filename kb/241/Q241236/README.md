## Q241236: Running Microsoft Office 2000 Indexes Contain Errors

{% raw %}

	Article: Q241236
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbfile
	Last Modified: 08-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Running Microsoft Office 2000 Premium ISBN 1-57231-945-3 
	- MSPRESS Running Microsoft Office 2000 Small Business ISBN 1-57231-958-5 
	- MSPRESS Running Microsoft Office 2000 Professional ISBN 1-57231-936-4 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The indexes for all three of the Running Microsoft Office 2000 books contain a
	number of errors. These indexes have been replaced, and are available for
	download in Microsoft Word format. The indexes can be opened using Wordpad or
	Microsoft Word 95 or later.
	
	MORE INFORMATION
	================
	
	The following file is available for download from the Microsoft Download
	Center:
	
	  Ro2kindx.exe
	  (http://download.microsoft.com/download/office2000/mspress/2.0/WIN98Me/EN-US/Ro2kindx.exe)
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
	
	Microsoft used the most current virus detection software available on the date of
	posting to scan this file for viruses. Once posted, the file is housed on secure
	servers that prevent any unauthorized changes to the file.
	
	The contents of the Ro2kindx self-extracting file are:
	
	Readme.txt    - Summary and installation instructions
	1-936-4.doc   - Index for Running Microsoft Office 2000 Professional
	1-945-3.doc   - Index for Running Microsoft Office 2000 Premium
	1-958-5.doc   - Index for Running Microsoft Office 2000 Small
	                Business
	
	To install this file, follow these instructions:
	
	1. Download Ro2kindx.exe into a temporary directory. If your software is not
	  capable of downloading the file into a particular directory, download the
	  file and copy or move it to a temporary directory.
	
	2. Double-click (or run) Ro2kindx.exe to extract the necessary files.
	
	3. If you are asked to confirm the replacement of files, please press Y to
	  confirm the file replacement.
	
	4. Delete any index files that you do not need. For example, if you own only the
	  Running Microsoft Office 2000 Professional book, you can delete 1-945-3.doc
	  and 1-958-5.doc, which apply to other Microsoft Press books.
	
	5. Delete Ro2kindx.exe.
	
	6. Open the index file using Wordpad or Microsoft Word.
	
	Additional query words: 1-57231-936-4 1-57231-945-3 1-57231-958-5 eubook eurun off2000
