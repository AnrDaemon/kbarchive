## Q241273: XCLN: Copy Folder Design Loses Properties

{% raw %}

	Article: Q241273
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:2000; winnt:5.5,5.5 SP1,5.5 SP2
	Operating System(s): 
	Keyword(s): exc55 exc55sp1 exc55sp2
	Last Modified: 08-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 5.5, 5.5 SP1, 5.5 SP2 
	- Microsoft Outlook 2000 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	If you copy the design of a folder, the values of the following properties are
	not properly copied to the destination folder:
	
	- The "Automatically generate Exchange views" property.
	
	- The "When Posting to this folder, use" property.
	
	- The "Only show views created for this folder" property.
	
	These properties need to be copied because they are part of the folder's design.
	
	RESOLUTION
	==========
	
	To resolve this problem, obtain Microsoft Office 2000 Service Release 1/1a
	(SR-1/SR-1a).
	
	To obtain SR-1/SR-1a, click the article number below to view the article in the
	Microsoft Knowledge Base:
	
	  Q245025 OFF2000: How to Obtain and Install Microsoft Office 2000 Service
	  Release 1/1a (SR-1/SR-1a)
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.This problem was corrected in Microsoft
	Office 2000 SR-1/SR-1a.
