## Q241101: Streets and Trips: Description of the Pushpin Conversion Tool

{% raw %}

	Article: Q241101
	Product(s): Microsoft Automap
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbtool kbimu
	Last Modified: 27-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Expedia Streets and Trips 2000 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to use the free Pushpin Conversion Tool to convert
	pushpin sets created in any of the following programs
	
	- Microsoft Expedia Streets 98
	- Microsoft Expedia Streets 98 Deluxe
	- Microsoft Expedia Trip Planner 98
	- Microsoft AutoRoute Express Great Britain 98
	- Microsoft AutoRoute Express Europe 98
	
	to a format that is recognized by the following programs:
	
	- Microsoft Expedia Streets and Trips 2000
	- Microsoft AutoRoute Express Great Britain 2000
	- Microsoft AutoRoute Express Europe 2000
	- Microsoft MapPoint 2000
	- Microsoft MapPoint 2000 Great Britain
	
	The following file is available for download from the Microsoft Download Center:
	
	  PushpinConversionTool.exe
	  (http://download.microsoft.com/download/expediastreets98/Utility/1.0/W9XNT4/EN-US/PushpinConversionTool.exe)
	
	Release Date: Jun-29-1999
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
	
	Microsoft used the most current virus detection software available on the date of
	posting to scan this file for viruses. Once posted, the file is housed on secure
	servers that prevent any unauthorized changes to the file.
	
	MORE INFORMATION
	================
	
	To install the Pushpin Conversion Tool, download the PushpinConversionTool.exe
	file from the Microsoft Download Center and save it in a folder on your hard
	disk.
	
	To use the Pushpin Conversion Tool, double-click the PushpinConversionTool.exe
	file. If multiple programs that use pushpins are detected, all existing pushpin
	sets are converted simultaneously. Also, if both Streets and Trips 2000 and
	MapPoint 2000 are installed on your computer, you can convert the earlier
	pushpin sets into both of the later pushpin formats.
	
	After you successfully convert the pushpins, the converted pushpin sets are
	available in the Pushpin Explorer in Streets and Trips 2000, and the converted
	pushpin template is available when you click New on the File menu in MapPoint
	2000.
	
	NOTE: The Pushpin Converson Tool is provided in English, French, and German
	versions. However, all dialog boxes and messages in the tool are in English.
