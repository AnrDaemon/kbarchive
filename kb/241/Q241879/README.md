## Q241879: Encarta: &quot;Go To Article&quot; Command Links to Incorrect Article

{% raw %}

	Article: Q241879
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:; :
	Operating System(s): 
	Keyword(s): kbui kbimu
	Last Modified: 25-JUN-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Encyclopedia 99 
	- Microsoft Encarta Encyclopedia 2000 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you click inside a map and then click Go To Article, the article that appears
	may contain information about an incorrect location.
	
	CAUSE
	=====
	
	This behavior can occur if you click inside a map of the following cities:
	
	- Toronto, Ontario, Canada
	- Vancouver, British Columbia, Canada
	- Vatican City
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
