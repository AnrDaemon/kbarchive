## Q241021: MS Windows NT 4.0 Security, Audit, and Control Corrections

	Article: Q241021
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 10-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Microsoft Windows NT 4.0 Security, Audit,
	and Control, ISBN 1-57231-818-x.
	
	The following topics are covered:
	
	- Page 146: Table 9-3, Correction To Read Special Permissions
	
	MORE INFORMATION
	================
	
	Page 146: Table 9-3, Correction To Read Special Permissions
	-----------------------------------------------------------
	
	On page 146, in Table 9-3 change the entry.
	
	Read (R) special permissions do not allow the user to "Change the Data In and
	Append to the File." Only Write (W) and Full Control (RWXDPO) permissions allow
	the user to do this.
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
