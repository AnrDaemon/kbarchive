## Q179345: WD97: Comments Pane Doesn't Close When Opened Via Hyperlinks

{% raw %}

	Article: Q179345
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): word97 kbwdinternet
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you click a hyperlink to a bookmark that is in a comment, Microsoft Word
	opens the comment pane. When you click the Back button on the Web toolbar to go
	back to the location of the hyperlink, Word does not close the comment pane.
	
	WORKAROUND
	==========
	
	To work around this problem, close the Comment Pane manually.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products listed at
	the beginning of this article.
