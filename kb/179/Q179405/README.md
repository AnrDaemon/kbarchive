## Q179405: FS98/2000: &quot;Go to Airport&quot; Search Filters No Longer Available

	Article: Q179405
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbui fs98 kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator 2000 
	- Microsoft Flight Simulator 2000 Professional Edition 
	- Microsoft Flight Simulator 98 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to search for a particular airport in the Go To Airport dialog
	box in Microsoft Flight Simulator 98 or Microsoft Flight Simulator 2000, there
	may be a limited number of airports, countries, regions, states or provinces, or
	cities available.
	
	CAUSE
	=====
	
	This behavior can occur if you use the Search Places option to search for an
	airport, country, region, state/province, or city, and you do not click New
	Search to clear the previous search results before you attempt a new search.
	
	Flight Simulator does not reset the search results until you click New Search.
	
	RESOLUTION
	==========
	
	To resolve this issue, reset the search results. To do this, use the appropriate
	method for your version of Flight Simulator.
	
	Flight Simulator 2000
	---------------------
	
	1. On the World menu, click Go To Airports.
	
	2. Click Search Places.
	
	3. Click New Search.
	
	Flight Simulator 98
	-------------------
	
	1. On the World menu, point to Go To, and then click Airports.
	
	2. Click Search Places.
	
	3. Click New Search.
	
	MORE INFORMATION
	================
	
	The following table describes the filters available in the Go To Airport dialog
	box:
	+--------------------------------------------------------------------+
	| Filter         | Description                                       | 
	+--------------------------------------------------------------------+
	| Global Region  | Narrows the search to a region of the globe.      | 
	+--------------------------------------------------------------------+
	| Country        | Narrows the search to a single country.           | 
	+--------------------------------------------------------------------+
	| State/Province | Narrows the search to a single state or province. | 
	+--------------------------------------------------------------------+
	| City           | Narrows the search to a single city.              | 
	+--------------------------------------------------------------------+
