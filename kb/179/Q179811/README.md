## Q179811: Bookshelf 98: Minimum System Requirements

	Article: Q179811
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kbmm kbSysSettingskbfaq
	Last Modified: 05-MAY-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf 98 for Windows 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the minimum system requirements to run Microsoft Bookshelf 98
	for Windows.
	
	MORE INFORMATION
	================
	
	The minimum system requirements to run Bookshelf 98 are:
	
	- A multimedia computer with a 486SX/33-MHz or higher processor.
	- Microsoft Windows 95 operating system or Microsoft Windows NT Workstation
	  operating system version 4.0 or later.
	- 8 megabytes (MB) of memory (RAM) for Windows 95; 12 MB for Windows NT
	  Workstation.
	- 10 MB of available hard disk space.
	- A double-speed or faster CD-ROM drive.
	- A Super VGA, 256-color monitor supporting 640 x 480 or higher resolution.
	- A local-bus video adapter with 1 MB or more of video memory.
	- A Microsoft Mouse or compatible pointing device.
	- An audio adapter with speakers or headphones.
	- A modem (required for Web links).
	
	Additional query words: 98 multimedia mmtitles required
