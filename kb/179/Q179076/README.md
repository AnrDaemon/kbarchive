## Q179076: WD97: Visio Object Background Turns Black When Doc Saved as HTML

{% raw %}

	Article: Q179076
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kbwdinternet
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you save a Microsoft Word 97 document containing Visio, version 4, objects
	as HTML, the background of the objects will turn black.
	
	MORE INFORMATION
	================
	
	Visio is a drawing package that lets you assemble illustrations and diagrams by
	dragging objects created from stencils, called SmartShapes. The Visio program
	installs icons on the Word and Excel toolbars so that you can easily insert
	Visio drawings.
	
	For information about how to contact Visio, query in the Knowledge Base for one
	of the following articles:

	  Q60781 Hardware and Software Third-Party Vendor Contact List, L-P
	
	  Q60782 Hardware and Software Third-Party Vendor Contact List, Q-Z
