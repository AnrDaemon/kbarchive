## Q179591: Cinemania: Monthly Update Files Are No Longer Available

{% raw %}

	Article: Q179591
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 
	Operating System(s): 
	Keyword(s): kbpolicy kbtool kbimu
	Last Modified: 25-OCT-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Cinemania for Windows, 1992, 1993, 1994, 1995, 1996, 1997 editions 
	- Microsoft Cinemania for Macintosh, 1994, 1995, 1996, 1997 editions 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The last of the twelve monthly update files available for Cinemania 97 is the
	August, 1997 update.
	
	The last of the twelve monthly update files available for Cinemania 96 is the
	August, 1996 update.
	
	As of December 31st, 1998, the Cinemania update files and the Cinemania Web site
	are no longer available.
	
	
	MORE INFORMATION
	================
	
	For reviews of current movies, please visit MSN Sidewalk at the following Web
	site:
	
	  http://sidewalk.msn.com
