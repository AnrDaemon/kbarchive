## Q179115: XADM: Custom Attributes Not Updated in Column View

{% raw %}

	Article: Q179115
	Product(s): Microsoft Exchange
	Version(s): WINDOWS:4.0,5.0,5.5
	Operating System(s): 
	Keyword(s): kbusage
	Last Modified: 22-MAR-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, versions 4.0, 5.0, 5.5 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you change Custom Attributes a second time using the Exchange Administrator
	program, the column view for the Custom Attributes is not automatically updated.
	
	WORKAROUND
	==========
	
	You must exit and restart the Exchange Administrator program in order to see the
	updates to the Custom Attributes in column view.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Exchange Server,
	versions 4.0, 5.0, and 5.5. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
