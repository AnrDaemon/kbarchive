	SUMMARY
	=======
	
	This article points to two self-extracting files to help you create a toolbar
	that is similar to the toolbar that is used in Internet Explorer 3.0.
	
	Comct33i.exe is a self-extracting compressed file that installs the Microsoft
	Windows Common Control 3 5.0, which contains the Coolbar ActiveX control. You
	can use the Coolbar ActiveX control to create a toolbar that is similar to the
	Internet Explorer toolbar.
	
	CoolBSmp.exe is a self-extracting file that installs a Visual Basic project that
	uses the Coolbar ActiveX control. The sample project contains a user interface
	that is similar to Internet Explorer.
