## Q179417: WD97: Format Font Option in Find and Replace Does Not Work

	Article: Q179417
	Product(s): Word 97 for Windows
	Version(s): WINDOWS:97
	Operating System(s): 
	Keyword(s): kb3rdparty
	Last Modified: 14-NOV-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Word 97 for Windows 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you perform a find or replace operation (on the Edit menu) and you try to
	select Font in the Format drop-down list, nothing happens or Word stops
	responding.
	
	NOTE: You may need to click More to see the Format drop-down list.
	
	
	CAUSE
	=====
	
	This problem may occur if you have TextBridge Pro 98 software installed.
	
	WORKAROUND
	==========
	
	To work around this problem, uninstall the TextBridge Pro 98 software.
	
	For additional information about how to do this, please contact Xerox
	Corporation.
	
	For information about how to contact Xerox Corporation, query in the Knowledge
	Base for one of the following articles:
	
	  Q65416 Hardware and Software Third-Party Vendor Contact List, A-K
	
	  Q60781 Hardware and Software Third-Party Vendor Contact List, L-P
	
	  Q60782 Hardware and Software Third-Party Vendor Contact List, Q-Z

	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
	
	For additional information about TextBridge Pro 98 software, please see the

	  Q164686 WD97: Instant Access OCR, Word Proofreader Commands Unavailable
