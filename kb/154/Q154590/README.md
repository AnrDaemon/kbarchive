## Q154590: Print Option Unavailable in Tabworks

	Article: Q154590
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kb3rdparty kbprint kbPrinting
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to print from Xsoft's Tabworks, the Print option is unavailable,
	even though a printer is installed.
	
	CAUSE
	=====
	
	Tabworks is attempting to get printer information from an incorrect key in the
	registry.
	
	RESOLUTION
	==========
	
	Contact Xsoft, Inc. for information concerning an updated version of Tabworks.
	
	The product discussed here is manufactured by XSoft, a manufacturer independent
	of Microsoft. We make no warranty, implied or otherwise, regarding this
	product's performance or reliability.
