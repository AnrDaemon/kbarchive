## Q154593: SMS: Administrator Utility Does Not Print Attribute Titles

	Article: Q154593
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.1,1.2
	Operating System(s): 
	Keyword(s): kbprint kbPrinting kbsmsAdmin smsadmin
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server versions 1.1, 1.2 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	When you are printing data displayed in the Personal Computer Properties window
	of the Systems Management Server Administrator utility, attribute titles that
	have no value defined are not printed.
	
	This only occurs when you run the Systems Management Server Administrator on a
	system running Windows NT version 4.0.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	versions 1.1 and 1.2. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
	
	Additional query words: prodsms prodnt field printed blank
