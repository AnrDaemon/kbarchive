## Q154912: Novell's NPRINT Command Does Not Work in Windows NT 4.0

{% raw %}

	Article: Q154912
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbnetwork kbprint kbPrinting
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Novell's NPRINT command does not work in Windows NT 4.0.
	
	CAUSE
	=====
	
	This is by design. Novell's NPRINT command is not supported by the Windows NT
	4.0 NetWare redirector.
