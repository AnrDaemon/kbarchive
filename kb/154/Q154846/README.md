## Q154846: AP Error Running Gdsset.exe in Japanese IIS

	Article: Q154846
	Product(s): Microsoft Windows NT
	Version(s): winnt:1.0,3.5,3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation versions 3.5, 3.51 
	- Microsoft Windows NT Server versions 3.5, 3.51 
	- Microsoft Internet Information Server 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run the Microsoft Internet Information Server (Japanese release), and
	you are running Gdsset.exe with the -c option, the following error may appear:
	
	  AP Error
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 3.51 and
	Internet Information Server version 1.0. This problem was corrected in the
	latest Windows NT 3.51 U.S. Service Pack. For information on obtaining the
	Service Pack, query on the following word in the Microsoft Knowledge Base
	(without the spaces):
	
	  S E R V P A C K
