## Q154991: How to Set Up Dr. Surf to Work with MSN

	Article: Q154991
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.3,2.0,2.5,2.51,2.52,2.6
	Operating System(s): 
	Keyword(s): kbmsn
	Last Modified: 08-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.3, 2.0, 2.5, 2.51, 2.52, 2.6 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Dr. Surf is translation software that translates English Web pages into Japanese
	text in the Japanese version of Microsoft Windows 95. This article lists the
	settings required for Dr. Surf to work with MSN, The Microsoft Network, and
	Microsoft Internet Explorer.
	
	MORE INFORMATION
	================
	
	Use the following settings for Dr. Surf:
	
	1. Start Dr. Surf, and then click Browser Settings on the Settings menu.
	
	2. In the Port box, type 7070.
	
	3. Do not enter any text in the Proxy Server Host or other Port boxes.
	
	4. In the Browser box, click Other Browser, and then click Microsoft Internet
	  Explorer.
	
	5. Click OK.
	
	6. In the Mail screen, enable Real Time Translation.
	
	Use the following settings for Microsoft Internet Explorer 3.0:
	
	1. In Control Panel, double-click Internet.
	
	2. On the Connection tab, click the Connect to the Internet as needed check box
	  to select it.
	
	3. In the Use the following Dial-Up Networking connection box, click MSN.
	
	4. Click the Connect Through A Proxy Server check box to select it.
	
	5. Click Settings, type localhost in the HTTP: box, and then type 7070 in the
	  Port: box.
	
	6. Click OK until you return to Control Panel.
	
	Use the following settings for Microsoft Internet Explorer 4.x:
	
	1. In Control Panel, double-click Internet.
	
	2. On the Connection tab, click Connect to the Internet using a modem, and then
	  click Settings.
	
	3. In the Use the following Dial-Up Networking connection box, click MSN, and
	  then click OK.
	
	4. Click the Access the Internet using a proxy server check box to select it.
	
	5. In the Address box, type localhost, and then type 80 in the Port box.
	
	6. Click OK.
	
	Use the following settings for TCP/IP:
	
	1. In Control Panel, double-click Network.
	
	2. Click TCP/IP, and then click Properties.
	
	3. Click the DNS Configuration tab.
	
	4. Click Enable DNS, and then type drsurfthe Host box.
	
	5. In the Domain box, type msn.com.
	
	6. Click OK until you return to Control Panel.
	
	7. Restart the computer.
	
	For further instructions on how to use Dr. Surf, please see the documentation
	included with the program. The third-party products discussed in this article
