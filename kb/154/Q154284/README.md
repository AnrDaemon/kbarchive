## Q154284: SMS: RAWCHECK for SMS 1.2 Does Not Check SMS 1.2 &#42;.RAW Files

	Article: Q154284
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.2
	Operating System(s): 
	Keyword(s): kbnetwork kbsms120 kbsms120sp4fix kbsms120bug kbsmsUtil smsutil
	Last Modified: 06-AUG-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run Rawcheck.exe from the Systems Management Server 1.2 CD on a Systems
	Management Server 1.2 *.raw file, one of the following error message appears and
	Rawcheck does not check the files:
	
	  The Raw file is either corrupted or a unsupported version.
	
	-or-
	
	  Error: Unable to open console.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.2. This problem has been corrected in the latest U.S. service pack for
	Systems Management Server version 1.2. For information on obtaining the service
	pack, query on the following word in the Microsoft Knowledge Base (without the
	spaces):
	
	  S E R V P A C K
	
	MORE INFORMATION
	================
	
	The fix for this problem was originally provided in Systems Management Server
	1.2 Service Pack 2 (SP2), and was also included in the Systems Management Server
	1.2 Resource Kit part 2. Because Systems Management Server service packs are
	cumulative, Service Pack 4 (SP4) for Systems Management Server 1.2 also includes
	this fix.
	
	Additional query words: prodsms inventory
