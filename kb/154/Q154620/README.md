## Q154620: Windows NT 4.0 DNS Server Loses the Forwarders Settings

	Article: Q154620
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbnetworkkbbuglist kbfixlist
	Last Modified: 10-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	A Windows NT 4.0 Domain Name Service (DNS) Server may lose its Server
	Properties\Forwarders settings causing DNS queries that rely on the forwarders
	to fail.
	
	The forwarders settings are removed from the registry and the DNS Manager/Server
	Properties/Forwarders.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 4.0. This
	problem was corrected in the latest Microsoft Windows NT 4.0 U.S. Service Pack.
	For information on obtaining the service pack, query on the following word in
	the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
	
	Additional query words: prodnt ntdns sp2 service pack 2 missing vanish empty blank gone disappear
