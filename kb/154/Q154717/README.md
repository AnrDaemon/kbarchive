## Q154717: Phone Disk Business PowerFinder 96 Does Not Run in Windows NT

{% raw %}

	Article: Q154717
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kb3rdparty appscomp kbAppCompatibility
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are installing the MS-DOS based version of PowerFinder 96, you receive
	an "Invalid Kernel" error message on a blue screen. When you install the Windows
	3.x-based version, you receive the error message "Cannot find Win3x1."
	
	RESOLUTION
	==========
	
	Create a shortcut to the Pdwin.exe file on the CD-ROM and run the program from
	the CD-ROM.
