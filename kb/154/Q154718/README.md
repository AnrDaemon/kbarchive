## Q154718: Incorrect Memory Size Reported in Windows NT 4.0

	Article: Q154718
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kbhw kbHardware
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Windows NT 4.0 may report a different amount of physical memory than is actually
	installed in the computer.
	
	CAUSE
	=====
	
	This behavior occurs if the computer's BIOS reports an incorrect amount of
	memory.
	
	RESOLUTION
	==========
	
	Check with the computer's manufacturer for a possible BIOS upgrade.
