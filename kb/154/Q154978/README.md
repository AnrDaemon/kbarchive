## Q154978: Err Msg: The Product Information File Is Not Available

	Article: Q154978
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.0,1.05,1.2,1.3,2.0,2.5,2.51,2.52,2.6
	Operating System(s): 
	Keyword(s): kbmsn
	Last Modified: 02-SEP-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.0, 1.05, 1.2, 1.3, 2.0, 2.5, 2.51, 2.52, 2.6 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to view the properties for The Microsoft Network icon, you may
	receive the following message:
	
	  The product information file is not available.
	
	CAUSE
	=====
	
	This behavior occurs if you have not run the sign-up program for MSN, The
	Microsoft Network, on the computer.
	
	RESOLUTION
	==========
	
	Run the sign-up program by double-clicking the MSN icon, or by using Internet
	Explorer. To run sign-up via Internet Explorer, follow the appropriate method
	below.
	
	To run sign up for an additional MSN Account using Internet Explorer 4.0 or earlier:
	------------------------------------------------------------------------------------
	
	1. Open Internet Explorer, click View, and then click Internet Options.
	
	2. Click the Connection tab, and then click Setup.
	
	To run sign up for an additional MSN Account using Internet Explorer 5.0: 
	-------------------------------------------------------------------------
	
	1. Open Internet Explorer, click Tools, and then click Internet Options.
	
	2. Click the Connection tab, and then click Setup.
	
	After following these directions the Internet Connection Wizard appears, at which
	point you follow the instructions on the screen to continue the process.
