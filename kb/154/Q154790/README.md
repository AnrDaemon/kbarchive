## Q154790: Weitek PCI Video Fails to Load on PowerPC Secondary PCI Bus

{% raw %}

	Article: Q154790
	Product(s): Microsoft Windows NT
	Version(s): winnt:3.5,3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation versions 3.5, 3.51 
	- Microsoft Windows NT Server versions 3.5, 3.51 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	On IBM PowerPC computers, the Weitek-based PCI adapters may not initialize when
	installed on the Secondary PCI bus.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in in Windows NT versions 3.5 and
	3.51. This problem was corrected in the latest Windows NT 3.51 U.S. Service
	Pack. For information on obtaining the Service Pack, query on the following word
	in the Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K
