## Q154557: Cache Icon Does Not Appear in DNS Manager

	Article: Q154557
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kbnetwork
	Last Modified: 21-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The cache icon does not appear under the name server selected in the servers
	list in Domain Name System (DNS) Manager.
	
	If the name server selected is authoritative for the root domain, the cache icon
	will not appear.
	
	Additional query words: prodnt ntdns
