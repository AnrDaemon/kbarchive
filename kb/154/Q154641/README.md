## Q154641: Express Meter 3.1 Causes General Protection Fault

{% raw %}

	Article: Q154641
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kb3rdparty
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you install Express Meter 3.1, you may receive the following error message
	the next time you log on to Windows NT:
	
	  EM caused a General Protection Fault in module EM.exe
	
	CAUSE
	=====
	
	Express Meter 3.1 is not supported in Windows NT 4.0.
	
	RESOLUTION
	==========
	
	Contact Express Systems to inquire about obtaining an updated version of Express
	Meter.
	
	
	MORE INFORMATION
	================
	
	Express Meter is manufactured by Express Systems, a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding this product's
	performance or reliability.
