## Q321726: Flight Simulator 2002: How to Print an Aircraft Checklist

{% raw %}

	Article: Q321726
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 06-AUG-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator 2002 Professional Edition 
	- Microsoft Flight Simulator 2002 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to print a checklist for a specific aircraft.
	
	MORE INFORMATION
	================
	
	To print the checklist for a specific aircraft, follow these steps:
	
	1. Start Windows Explorer.
	
	2. Locate the following folder
	
	  C:\Program Files\Microsoft Games\FS2002\Aircraft\<Name>
	
	  where <Name> is the aircraft for which you want to print the checklist.
	
	3. In the <Name> folder, click <Name>_check.txt to view the
	  checklist for that aircraft.
	
	4. On the File menu, click Print.
	
	5. When the printing process is complete, close the checklist file.
