## Q321746: Dungeon Siege: How to Create a Backup Copy of a Saved Game

	Article: Q321746
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 06-AUG-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Dungeon Siege, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to create a backup copy of a Microsoft Dungeon Siege
	game.
	
	NOTE: The procedure in this article creates backup copies of games only. The
	procedure does not back up settings or preferences that you may have set in
	Dungeon Siege.
	
	MORE INFORMATION
	================
	
	To create a backup copy of a saved Dungeon Siege game, follow these steps.
	
	NOTE: If you want to use a CD-R (recordable) to store the backup files, see your
	CD-creating software documentation for information on how to transfer data to a
	CD-R.
	
	1. Right-click Start, and then click Explore.
	
	2. Locate and then right-click the following folder
	
	  <path>\My Documents\Dungeon Siege\Save
	
	  where <path> is the path to the My Documents folder.
	
	3. Click Copy.
	
	4. Right-click the backup location, and then click Paste.
	
	How to Use the Backup Copies:
	
	When you want to use the back up games, follow these steps:
	
	1. Right-click Start, and then click Explore.
	
	2. Locate and right-click the location of the back up files, and then click
	  Copy.
	
	3. Locate and then right-click the following folder
	
	  <path>\My Documents\Dungeon Siege
	
	  where <path> is the path to the My Documents folder.
	
	4. Click Paste.
	
	NOTE: If you used a CD-R to back up the files, continue to the next procedure.
	
	If You Used a CD-R to Back Up the Games:
	
	After you copy the backup files from the CD-R to the hard disk, remove the
	read-only attribute. To do this, follow these steps:
	
	1. Locate and then open the following folder
	
	  <path>\My Documents\Dungeon Siege\Save
	
	  where <path> is the path to the My Documents folder.
	
	2. On the Edit menu, click Select All.
	
	3. On the File menu, click Properties.
	
	4. Click to clear the Read-only check box.
	
	5. Click Apply, and then click OK.
	
	6. Close Windows Explorer.
