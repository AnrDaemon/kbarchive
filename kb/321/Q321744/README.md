## Q321744: Dungeon Siege 1.0: Err Msg: Could Not Connect to the ZoneMatch

{% raw %}

	Article: Q321744
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 03-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Dungeon Siege, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to start a multiplayer game by using ZoneMatch, you may receive an
	error message similar to the following:
	
	  Could not connect to the ZoneMatch Server.
	
	CAUSE
	=====
	
	This error may occur if Port 2300 is unavailable (blocked).
	
	You must have Port 2300 available if you want to play Microsoft Dungeon Siege as
	a multiplayer game through a firewall, a proxy server, a router, Network Address
	Translation (NAT), or Internet Connection Sharing (ICS).
	
	RESOLUTION
	==========
	
	To resolve this issue, open port 2300.
	
	To verify that port 2300 is open or to open this port, please contact your
	network administrator or Internet service provider (ISP).
	
	If you are the administrator of the network, please see your networking software
	documentation to determine how to open this port.
	
	MORE INFORMATION
	================
	
	For additional information about the ports that you have to have to play a
	multiplayer game of Dungeon Siege, click the article number below to view the
	article in the Microsoft Knowledge Base:
	
	  Q320020 Dungeon Siege: Network Ports Required for Multiplayer Game
