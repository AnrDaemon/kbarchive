## Q269813: MSB: No Sound When You Run the Program on Terminal Server

	Article: Q269813
	Product(s): Microsoft Home Kids Products
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbenv kbsound mskids kbimu
	Last Modified: 14-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Scholastic's The Magic School Bus in Concert 
	- Scholastic's The Magic School Bus Lands on Mars 
	- Scholastic's Magic School Bus Series: Volcanoes, version 1.0 
	- Scholastic's The Magic School Bus Discovers Flight, version 1.0 
	- Scholastic's The Magic School Bus Whales and Dolphins, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run one of the programs listed at the beginning of this article over a
	Microsoft Windows NT 4.0 Terminal Server connection, sounds in the program are
	not played.
	
	RESOLUTION
	==========
	
	To work around this issue, turn on closed captions in the program. To do this,
	click the CC sign inside the bus.
	
	The CC sign is located to the right of the Exit sign, above the bus door.
