## Q269345: SMS: Systems Management Server Technical Resources

	Article: Q269345
	Product(s): Microsoft Systems Management Server
	Version(s): 2.0
	Operating System(s): 
	Keyword(s): kbtool kbsms200
	Last Modified: 22-JUN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	
	Microsoft provides a wide variety of technical resources to help you plan,
	deploy, and maintain Systems Management Server (SMS) sites. Online resources
	include:
	
	- Systems Management Server home page:
	
	  http://www.microsoft.com/smsmgmt
	
	- SMS Technology Center on TechNet:
	
	  http://www.microsoft.com/technet/sms
	
	- SMS 2.0 Support Online FAQ and highlights:
	
	  http://support.microsoft.com/support/default.asp?PR=SMS200
	
	- SMS 1.2 Support Online FAQ and highlights:
	
	  http://support.microsoft.com/support/default.asp?PR=sms
	
	- Support Online Service Desk:
	
	  http://www.microsoft.com/directaccess/products/sms/
	
	The "More Information" section of this article provides references to white
	papers and other online resources, and these are categorized by typical
	scenarios.
	
	MORE INFORMATION
	================
	
	For paper copies of the Systems Management Server Administrator's Guide, if you
	obtained SMS through the Select or Open program, you can call (800) 248-0655,
	which is Microsoft World Wide Fulfillment. Use part number 271-00617. If you did
	not obtain SMS through the Select or Open program, you can call Microsoft
	Supplemental and Replacement Parts, at (800) 360-7561 and request the Systems
	Management Server 2.0 Administrator's Guide, part number 271-00617. In Canada,
	call (800) 933-4750.
	
	An updated version of the Systems Management Server Administrator's Guide is
	included in online form on all SMS Administrator consoles as of Service Pack 1,
	in the SMS program group.
	
	The Systems Management Server Resource Guide is available at your favorite book
	Web site or bookstore. The ISBN number is 0-7356-0928-4. It is also included in
	the Microsoft BackOffice Resource Kit version 4.5. This ISBN number is
	0-7365-0583-1/ The Microsoft Systems Management Server Software Development Kit
	is available as part of the Microsoft Platform SDK, available through Microsoft
	Developer Network. For more details, visit the following Microsoft Web site:
	
	  http://support.microsoft.com/support/smsmgmt/content/sms20sdk.asp
	
	Additional technical papers are available at the following Microsoft Web site:
	
	  http://www.microsoft.com/smsmgmt
	
	These papers discuss software deployment, SMS integration with related
	technologies (including Microsoft Windows 2000), security, reporting, and other
	subjects. SMS product and evaluation details are also available at this Web
	site.
