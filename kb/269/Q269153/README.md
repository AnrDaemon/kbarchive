## Q269153: Kodak Imaging for Windows Does Not Invert TIFF Images

	Article: Q269153
	Product(s): Microsoft Windows NT
	Version(s): WINDOWS:2000; winnt:4.0
	Operating System(s): 
	Keyword(s): kbtool
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows 2000 Server 
	- Microsoft Windows 2000 Professional 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Kodak Imaging for Windows does not recognize the Photometric value when it
	displays Tag Image File Format (TIFF) images in CCITT Group 1.3 or 4 format. The
	Photometric value controls whether the image has to be inverted for display.
	
	RESOLUTION
	==========
	
	To work around this behavior, save the TIFF Image in a different format.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
