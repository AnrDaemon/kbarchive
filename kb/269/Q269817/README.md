## Q269817: MSB Error Message: Cannot Find Vids:IV50 Decompressor

{% raw %}

	Article: Q269817
	Product(s): Microsoft Home Kids Products
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbenv kberrmsg mskids kbimu
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Scholastic's The Magic School Bus in Concert 
	- Scholastic's The Magic School Bus Lands on Mars 
	- Scholastic's The Magic School Bus Discovers Flight, version 1.0 
	- Scholastic's The Magic School Bus Whales and Dolphins, version 1.0 
	- Scholastic's Magic School Bus Series: Volcanoes, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to play a video clip in one of the programs listed at the
	beginning of this article, you may receive the following error message:
	
	  Cannot find vids:iv50 decompressor.
	
	CAUSE
	=====
	
	This behavior can occur if you run the programs listed at the beginning of this
	article on a Microsoft Windows 95-based computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest Indeo video codecs from
	the following Ligos Web site:
	
	  http://www.ligos.com/indeo/downloads/
