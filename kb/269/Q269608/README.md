## Q269608: XADM: &quot;Data Error (cyclic redundancy check) 0xC0020017&quot; Error

{% raw %}

	Article: Q269608
	Product(s): Microsoft Exchange
	Version(s): 5.5
	Operating System(s): 
	Keyword(s): exc55
	Last Modified: 23-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Exchange Server, version 5.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Microsoft Exchange Server, when you attempt to run Setup, you may receive the
	following error message:
	
	  Data error (cyclic redundancy check)
	  Microsoft Windows NT
	  ID no: c0020017
	
	CAUSE
	=====
	
	This error can occur because the installation CD-ROM is damaged. It may also
	occur if your system has faulty cabling or hardware.
	
	WORKAROUND
	==========
	
	To work around this issue, clean or replace the Microsoft Exchange Server
	CD-ROM.
	
	MORE INFORMATION
	================
	
	The cyclic redundancy check (CRC) is a very powerful but easily implemented
	technique that is used to obtain data reliability.
