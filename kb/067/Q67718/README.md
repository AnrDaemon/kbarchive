## Q67718: Fatal Error When Running DayBook

	Article: Q67718
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	If you experience a fatal error when you run DayBook, it may have been
	caused by a corruption of the DAYBOOK.TBK file's script. If you have
	data in the DAYBOOK.TBK file that you would like to recover, use the
	following procedure:
	
	1. Rename DAYBOOK.TBK to OLDBOOK.TBK.
	
	2. Reinstall ToolBook.
	
	3. From the File menu, choose Merge DayBook Data
	
	4. Type OLDBOOK.TBK
