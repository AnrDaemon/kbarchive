## Q67679: Mac Wkst: 2.00 Voice Form and the Macinstosh LC and IIsi

	Article: Q67679
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:2.0,2.0a,2.0b,3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Mail version 2.00 voice form does not work with the Macintosh LC
	and the Macintosh IIsi. The current voice form is not compatible with the sound
	capabilities of these Macintoshes.
	
	To use the sound capabilities of the Macintosh LC and the Macintosh IIsi, users
	should upgrade to Microsoft Mail version 3.00.
	
	Additional query words: 2.00 2.00a 2.00b 3.00
