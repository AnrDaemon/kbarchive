## Q67221: Changing Decks in Solitaire with the Keyboard

{% raw %}

	Article: Q67221
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	To change the deck for the Solitaire game in Microsoft Windows version
	3.0 using the keyboard, do the following:
	
	1. Press ALT+G to choose the Game menu.
	
	2. Press C to choose the Deck menu item.
	
	3. Use the LEFT or RIGHT ARROW keys to move through the selections and to the
	  deck face to which you would like to change.
	
	4. Press the SPACEBAR once. The card should flash once.
	
	5. Press the ENTER key. You should return to the game window.
