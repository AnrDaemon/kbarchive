## Q67027: Bitstream Fontware 3.0 Installation Kit Description

{% raw %}

	Article: Q67027
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following is a brief description of the Fontware 3.0 Installation Kit.
	
	MORE INFORMATION
	================
	
	- The kit makes fonts for the most popular laser printers from Hewlett-Packard,
	  Apple Computer, and other vendors.
	
	- It makes fonts for dot-matrix printers and displays (screens) for certain
	  applications/environments.
	
	- It makes fonts for devices equipped with the Bitstream Quality Enhancement
	  Module (QEM) (Alternative PostScript PDL printers that provide Bitstream
	  "hints").
	
	- It makes fonts in sizes ranging from 2 points to 144 points, depending on
	  your application.
	
	- It supports most popular character sets, including ASCII, IBM PC, HP Roman 8,
	  and PostScript.
	
	- It allows merging of several Fontware 3.0 Installation Kits. You can share
	  directories and typefaces among the applications supported by the kits. You
	  can also create fonts for multiple applications simultaneously and share
	  laser-printer fonts among them.
	
	- It allows for easy addition and removal of support for individual devices or
	  applications.
	
	- It creates a dated and time-stamped summary log file each time you make
	  fonts.
	
	- It creates a dated and time-stamped error log file each time an error or
	  warning occurs when you make fonts, so that you may troubleshoot more
	  effectively.
	
	- It Provides extensive online help.
	
	A Bitstream Fontware Installation Kit is available through a fulfillment coupon
	included with the retail version of Microsoft PowerPoint for Windows.
