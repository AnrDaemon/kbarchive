## Q67190: ASD: Moving a Group of Static Scenery Items

	Article: Q67190
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	In Aircraft & Scenery Designer version 4.00, you cannot move a group of
	scenery items (for example, an entire city) all at once.
	
	You can move static scenery items individually in the Scenery Editor.
