## Q67486: International Versions of MS-DOS for End Users

{% raw %}

	Article: Q67486
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:3.x,4.x,5.0
	Operating System(s): 
	Keyword(s): msdos
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 3.1, 3.2, 3.21, 3.3, 3.3a, 4.0, 4.01, 5.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Foreign language versions of MS-DOS are generally only available from the
	respective Microsoft foreign subsidiary, and generally only for licensing to
	OEMs, not end users. However, there are two exceptions to this rule: Arabic
	MS-DOS and Hebrew MS-DOS.
	
	Arabic MS-DOS and Hebrew MS-DOS customers can obtain technical support from the
	following companies:
