## Q67219: GTCO Digitizer Tablets and Windows 3.0

{% raw %}

	Article: Q67219
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	GTCO has a Microsoft Windows version 3.0 compatible device driver available for
	the following digitizer pads:
	
	- GTCO Type 5A Digi-Pad tablets
	
	- GTCO Micro Digi-Pad tablets
	
	- GTCO Sketch Master tablets
	
	MORE INFORMATION
	================
	
	The driver is offered free to all GTCO customers with these digitizer pads.
	Contact GTCO to order the driver.

	we make no warranty, implied or otherwise, regarding these products' performance
	or reliability.
