## Q62826: Zenith Computers and SWAPFILE.EXE with Windows 3.0

	Article: Q62826
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft Windows version 3.0 SWAPFILE.EXE does not work correctly on Zenith
	computers.
	
	If you try to create a swapfile using Microsoft's version of the utility, you
	receive the following error:
	
	  Swapfile could not find any drives suitable for creating a swapfile
	
	If you receive this error message, you must obtain Zenith's version of swapfile.
	
	MORE INFORMATION
	================
	
	Zenith has manufactured an OEM version of Windows version 3.0 for its computers.
	Zenith has also modified SWAPFILE.EXE to allow it to work properly on its
	computers.
	
	The SWAPFILE.EXE file is available in application note form from Microsoft
	Product Support Services by calling (425) 454-2030.
	
	For more information on the Zenith version of Microsoft Windows, contact your
	Zenith dealer.
	
	The Zenith products included here are manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding these products'
	performance or reliability.
