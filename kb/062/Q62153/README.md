## Q62153: Schweizer 2-32 Sailplane Performance Specifications

{% raw %}

	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	The following is a summary of the performance specifications for the Schweizer
	2-32 Sailplane in Flight Simulator version 4.00:
	
	  Feature                Specification
	  -------                -------------
	
	  Length                 26.75 feet
	  Wing span              57 feet
	  Height                 9 feet
	  Wing area              180 square feet
	  Aspect ratio           18.05
	  Gross weight           1340 pounds
	  Empty weight           831 pounds
	  Maximum L/D            34 at 45 knots
	  Minimum sink (solo)    2 feet per second at 40 knots
