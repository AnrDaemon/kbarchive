## Q62748: PPT: Copying Slides Between Presentations

	Article: Q62748
	Product(s): Microsoft PowerPoint for Windows
	Version(s): 2.0,2.0e,3.0,4.0,7.0
	Operating System(s): 
	Keyword(s): kbinterop
	Last Modified: 20-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft PowerPoint for Windows 95, version 7.0 
	- Microsoft PowerPoint for Windows, versions 2.0, 2.0e, 3.0, 4.0 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	You can copy slides from one presentation to another by using the Cut, Copy and
	Paste commands.
	
	MORE INFORMATION
	================
	
	To copy slides between presentations, follow these steps:
	
	1. Open your new presentation and the presentation containing the slides that
	  you want to copy. From the Window menu, click the presentation that has the
	  slides you want to copy.
	
	2. In Slide Sorter view, select the slides you want to copy, and then click Copy
	  or Cut on the Edit menu.
	
	3. From the Window menu, click the presentation that you want to copy slides to,
	  and then click Slide Sorter on the View menu.
	
	4. On the Edit menu, click Paste.
	
	The slides are copied to your new presentation. The copied slides assume the
	format of the presentation they're copied to. If the formats are different, you
	may have to move objects on the copied slides to fit their new format.
