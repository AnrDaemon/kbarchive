## Q303231: Running MSN Explorer In &quot;Compatibility Mode&quot; In Windows XP May C

	Article: Q303231
	Product(s): The Microsoft Network
	Version(s): 6.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network version 6.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Using Windows XP, you right-click the MSN Explorer icon on your desktop. You
	select the Run in compatibility mode box, and then select either Windows 95 or
	Windows 98. You start MSN Explorer, sign in, and then perform a few tasks
	(viewing Web sites, sending or receiving e-mail). When you quit MSN Explorer, a
	message may appear stating that "there was a problem running MSN Explorer".
	
	CAUSE
	=====
	
	This is a known issue.
	
	RESOLUTION
	==========
	
	Right-click the MSN Explorer icon on the desktop and clear the Run in
	compatibility mode box.

	================
