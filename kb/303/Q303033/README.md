## Q303033: Asherons Call: You Cannot Communicate with Other Players

	Article: Q303033
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 16-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Asheron's Call, version 1.0 
	- Microsoft Asheron's Call: Dark Majesty, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you are playing Microsoft Asheron's Call, you may be unable to communicate
	with other players, view combat messages, or receive other chat dialog.
	
	CAUSE
	=====
	
	This behavior occurs if the global filter is turned on.
	
	RESOLUTION
	==========
	
	To resolve this issue, start Asheron's Call, and then type "@unfilter -all"
	(without the quotation marks) in the Chat window.
	
	MORE INFORMATION
	================
	
	For additional information about chat commands, click the article number below
	to view the article in the Microsoft Knowledge Base:
	
	  Q243622 Asheron's Call: List of Chat Commands
