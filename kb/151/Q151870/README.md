## Q151870: BUG: Mastering Exchange: Help Option Doesn't Work During Setup

{% raw %}

	Article: Q151870
	Product(s): Microsoft Mastering Series
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbmm kbsetup kbnokeyword kbBug kbISS
	Last Modified: 27-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Mastering Microsoft Exchange Development ISBN 1-57231-337-4, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Help option does not function in the Setup error dialog you receive when
	your hard drive is too full.
	
	RESOLUTION
	==========
	
	There is no resolution to this problem.
	
	MORE INFORMATION
	================
	
	When you install Mastering Exchange Development to a hard drive that does not
	have enough space available, you get the following message:
	
	  There does not appear to be sufficient disk space available on drive c to
	  install Mastering Microsoft Exchange Development.
	
	with the option to click OK or Cancel. If you click OK, the program will try to
	continue with the installation and you will receive the following error
	message:
	
	  Cannot install files from <filename>.
	
	This dialog gives you the option to click Continue Operation (the default),
	Cancel Operation, Help, or More. If you click Help. nothing happens.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the products listed above. We
	are researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
