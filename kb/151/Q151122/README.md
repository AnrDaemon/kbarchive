## Q151122: Error Message: The Printer Command Is Invalid

{% raw %}

	Article: Q151122
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you change a local printer's properties to print directly to the printer,
	you may receive the following error message:
	
	  The following error occurred while trying to save your changes. The printer
	  command is invalid.
	
	CAUSE
	=====
	
	The print queue for that printer has been paused.
	
	RESOLUTION
	==========
	
	To resume the print queue, follow these steps:
	
	1. Click the Start button, point to Settings, and then click Printers.
	
	2. Double-click the printer.
	
	3. On the Printer menu, click Pause Printing to remove the check mark.
	
	4. On the Printer menu, click Close.
