## Q151386: Fury3 AVI Play Black When Only 8MB RAM and Only 10 MB Swapfile

	Article: Q151386
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Fury 3 for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If Microsoft Fury3 version 1.0 is run on a machine that only has 8 megabytes of
	RAM, and swapfile that is 10 MB or smaller, then AVI files will play with a
	completely black screen, and the introduction AVI files will play black, and
	without sound as well.
	
	RESOLUTION
	==========
	
	Add memory, or increase the swapfile size.
	
	MORE INFORMATION
	================
	
	To add memory to a computer, consult the owners manual.
	
	To increase the swapfile size, perform the following steps:
	
	Windows 95/98
	-------------
	
	1. Click Start, and point to Settings.
	
	2. Click Control Panel, and double-click the System icon.
	
	3. Click the Performance tab at the top of the window.
	
	4. Click Virtual Memory.
	
	5. Click the "Let Windows manage my virtual memory (recommended)" option.
	
	6. Click OK. Click OK again.
	
	7. Close the Control Panel.
	
	Windows 3.1
	-----------
	
	1. Double-click the Control Panel icon (usually found in the Main group).
	
	2. Double-click the 386 Enhanced icon.
	
	3. Click Virtual Memory, and click Change.
	
	4. Increase the swapfile size to larger than 10 megabytes.
	
	5. Close the virtual memory window.
	
	6. Close the Enhanced window.
	
	7. Close the Control Panel, and click OK when asked if you want to restart
	  Windows.
