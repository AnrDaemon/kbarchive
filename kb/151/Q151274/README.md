## Q151274: Remote Control GP Faults in IDIS_LM.DLL

{% raw %}

	Article: Q151274
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.1
	Operating System(s): 
	Keyword(s): kbtshoot smsremtshoot kbRemoteProg
	Last Modified: 31-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	
	Remote control may return a general protection (GP) fault in IDIS_LM.DLL when
	controlling an application.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem Systems Management Server version
	1.1. This problem was corrected in Systems Management Server version 1.2.
