## Q151035: Encarta 96-97: Minimize Pinpointer Preference Not Functioning

	Article: Q151035
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:; :1996 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 24-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 96 Encyclopedia for Windows 
	- Microsoft Encarta 96 Encyclopedia for Macintosh 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If the option Minimize Pinpointer After I Select An Article in the Options
	dialog (Preferences dialog on the Macintosh version) is selected, the Pinpointer
	closes instead of minimizing.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem Microsoft Encarta 1996. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
	
	WORKAROUND
	==========
	
	To retain your ability to minimize the Pinpointer window, use the instructions
	below for your version of Encarta.
	
	Encarta for Windows
	-------------------
	
	1. On the Information menu, click Options.
	
	2. Remove the check from Minimize Pinpointer After I Select An Article option.
	
	3. Click OK.
	
	Encarta for the Macintosh
	-------------------------
	
	1. On the File menu, click Preferences.
	
	2. Remove the check from Minimize Pinpointer After I Select An Article option.
	
	3. Click OK.
	
	Now when you've selected an article from the Pinpointer window, you can minimize
	the Pinpointer by clicking the Minimize button on the Pinpointer title bar
	(located next to the Close button). The minimized Pinpointer appears as an icon
	above the text of the article.
