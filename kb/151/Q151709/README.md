## Q151709: NT Server Manager Can Lock LAN Manager Server

	Article: Q151709
	Product(s): Microsoft Windows NT
	Version(s): 2.2b,3.5,3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server versions 3.5, 3.51 
	- Microsoft LAN Manager, version 2.2b 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you try to view a LAN Manager server through Server Manager from two Windows
	NT Servers simultaneously, the LAN Manager server stops responding. This may
	also happen with the versions of Server Manager included with the client
	versions of the NT Server tools.
	
	WORKAROUND
	==========
	
	View LAN Manager servers from only one instance of Server Manager at a time.
