## Q151008: Sony 7000 and DEC TZ09 4mm DAT Not Supported Under 3.51

	Article: Q151008
	Product(s): Microsoft Windows NT
	Version(s): winnt:3.5,3.51
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server versions 3.5, 3.51 
	- Microsoft Windows NT Workstation versions 3.5, 3.51 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	A computer running Windows NT 3.51 fails to work with the Sony 7000 and the DEC
	TZ09 4mm digital audio tape (DAT) devices.
	
	CAUSE
	=====
	
	The devices fail due to the incompatibility of the firmware string with the
	Sony4mm DAT driver.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT version 3.51. This
	problem was corrected in the latest Windows NT 3.51 U.S. Service Pack. For
	information on obtaining the Service Pack, query on the following word in the
	Microsoft Knowledge Base (without the spaces):
	
	  S E R V P A C K

	of Microsoft; we make no warranty, implied or otherwise, regarding these
	products' performance or reliability.
