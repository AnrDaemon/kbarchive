## Q151415: Fury3 Power Ups Symbol Not Anchored On The Instrument Panel

{% raw %}

	Article: Q151415
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Fury 3 for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	While in panoramic format with full cockpit view, the purple Power Ups symbols
	will appear to be floating in the main window, rather than on the instrument
	panel.
	
	RESOLUTION
	==========
	
	Microsoft has confirmed this to be a problem in Microsoft Fury3 for Windows
	version 1.00.
	
	Additional query words: 1.00 fury display
