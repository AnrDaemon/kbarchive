## Q151420: Fury3: AVI Music Turns Off When Resizing Window

{% raw %}

	Article: Q151420
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Fury 3 for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Fury3 AVI sounds may stop playing if you perform a Restore Down or Maximize
	Window and have assigned sound to those events.
	
	RESOLUTION
	==========
	
	Disable the sounds for the Restore Down and Maximize Window events.
	
	MORE INFORMATION
	================
	
	To Disable Sound For The Restore Down And Maximize Window Events
	----------------------------------------------------------------
	
	Windows 95/98
	-------------
	
	1. Click Start and point to Settings.
	
	2. Click Control Panel.
	
	3. Double-click Sounds.
	
	4. In the Events list, click Maximize, and in the Name list, click None.
	
	5. In the Events list, click Restore Down, and in the Name list, click None.
	
	6. Click OK.
	
	Windows 3.1
	-----------
	
	1. Double-click the Control Panel (usually in the Main group).
	
	2. Double-click Sound.
	
	3. In the Events list, click Maximize, and in the Name list, click None.
	
	4. In the Events list, click Restore Down, and in the Name list, click None.
	
	5. Click OK, and close the Control Panel.
	
	Microsoft has confirmed this to be a problem in Fury3 version 1.0.
