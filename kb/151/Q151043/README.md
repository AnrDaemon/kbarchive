## Q151043: PPT7: How to Customize Right Mouse Button Behavior in PowerPoint

	Article: Q151043
	Product(s): Microsoft PowerPoint for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-APR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft PowerPoint for Windows 95, version 7.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you run a slide show in PowerPoint for Windows 95, clicking the right mouse
	button displays a list of slide show options. This behavior is different from
	PowerPoint 4.0, in which clicking the right mouse button displays the previous
	slide.
	
	MORE INFORMATION
	================
	
	To customize the behavior of the right mouse button in a PowerPoint 97 slide
	show, follow these steps:
	
	1. On the Tools menu, click Options.
	
	2. Click the View tab.
	
	3. Clear the "Pop-up Menu on Right Mouse Click" check box.
	
	4. Click OK.
	
	Clicking the right mouse button now returns the presentation to the previous
	slide.
