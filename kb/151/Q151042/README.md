## Q151042: FP: FrontPage Personal Web Server Supports Standard CGI Only

{% raw %}

	Article: Q151042
	Product(s): Word Front Page
	Version(s): windows:1.0,1.1,97
	Operating System(s): 
	Keyword(s): kbusage kbdta
	Last Modified: 04-OCT-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft FrontPage 97 for Windows with Bonus Pack 
	- Microsoft FrontPage for Windows, versions 1.0, 1.0a, 1.1 
	-------------------------------------------------------------------------------
	
	For a Microsoft FrontPage 98 version of this article, see Q194225.
	
	SUMMARY
	=======
	
	FrontPage Personal Web Server supports only the standard Common Gateway
	Interface (CGI) stdin/stdout method. FrontPage Personal Web Server does not
	support the DLL-, WIN-, or DOS-based CGI.
	
	MORE INFORMATION
	================
	
	For additional information about standard CGI and NCSA HTTPD, please see the
	NCSA HTTPd Development Team World Wide Web site:
	
	  http://hoohoo.ncsa.uiuc.edu
