## Q151340: Cannot Use &quot;@msn.com&quot; in Personal Address Book

	Article: Q151340
	Product(s): The Microsoft Network
	Version(s): WINDOWS:1.2,1.3
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 1.2, 1.3 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Mail that you send to another member of MSN, The Microsoft Network, may be
	returned as undeliverable when you address the mail using your personal address
	book.
	
	CAUSE
	=====
	
	This problem occurs if you create a personal address book entry for an MSN
	member and include the domain (@msn.com) in the Member ID box.
	
	RESOLUTION
	==========
	
	When you create a personal address book entry for an MSN member, do not include
	the domain (@msn.com) in the Member ID box.
