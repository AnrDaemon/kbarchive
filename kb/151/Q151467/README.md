## Q151467: Mastering Visual Basic: Setup Does Not Complete

	Article: Q151467
	Product(s): Microsoft Mastering Series
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbmm kbsetup
	Last Modified: 22-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Mastering Microsoft Visual Basic 4.0 ISBN 1-55615-913-7, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Mastering Microsoft Visual Basic 4.0 setup does not complete.
	
	CAUSE
	=====
	
	Microsoft Visual Basic is running during the Mastering Setup procedure.
	
	RESOLUTION
	==========
	
	Quit Microsoft Visual Basic and then run setup again.
	
	MORE INFORMATION
	================
	
	If you run setup while Microsoft Visual Basic is running, Setup does not
	complete successfully. However, there is no message requesting you to close the
	running application.
