## Q151085: Mastering Visual C++: Help/Print Closes Unexpectedly

{% raw %}

	Article: Q151085
	Product(s): Microsoft Mastering Series
	Version(s): 1.0,1.0a; WINDOWS:95
	Operating System(s): 
	Keyword(s): kbmm
	Last Modified: 09-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Mastering Microsoft Visual C++ ISBN 1-55615-914-5, versions 1.0, 1.0a 
	- the operating system: Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to change your printer properties while you try to print a Help
	topic from Microsoft Mastering Visual C++, the Help window and the Print dialog
	box close without warning. Changes made to the printer properties are not
	preserved.
	
	This occurs when you open and then close the Print Properties dialog box.
	
	WORKAROUND
	==========
	
	To change printer properties, do the following:
	
	1. Click the Start button, point to Settings, and then click Printers.
	
	2. With your right mouse button, click the printer you want to use, and then
	  click Properties.
	
	3. Change the printer properties.
	
	MORE INFORMATION
	================
	
	This behavior does not occur on systems running Microsoft Windows NT.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Mastering Microsoft Visual C++,
	version 1.0. We are researching this problem and will post new information here
	in the Microsoft Knowledge Base as it becomes available.
