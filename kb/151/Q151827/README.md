## Q151827: BUG: Mastering VC++: Print Topic Doesn't Print Topic Title

{% raw %}

	Article: Q151827
	Product(s): Microsoft Mastering Series
	Version(s): 1.0,1.0a
	Operating System(s): 
	Keyword(s): kbmm kbnokeywordkbbuglist
	Last Modified: 10-MAR-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Mastering Microsoft Visual C++ ISBN 1-55615-914-5, versions 1.0, 1.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you print a topic from Mastering Visual C++, the topic's title is not
	printed.
	
	RESOLUTION
	==========
	
	There is no resolution to this problem at this time.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the products listed above. We
	are researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
