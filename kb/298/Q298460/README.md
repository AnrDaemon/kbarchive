## Q298460: Windows NT 4.0 Workstation Deployment Resources

	Article: Q298460
	Product(s): Microsoft Windows NT
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): kbtool
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article is designed to help you find the resources and tools you may need
	to deploy Windows NT Workstation version 4.0 in your organization. This article
	includes information about:
	
	- Windows NT Workstation Deployment Resources
	- Microsoft Windows NT Workstation Resource Kit
	- Updated Sysdiff Tool
	- Microsoft Windows NT Workstation Reviewer's Guide
	- Microsoft Internet Explorer Administrator's Kit Corporate Deployment Guide
	- Implementing Profiles and Policies for Windows NT 4.0
	
	
	MORE INFORMATION
	================
	
	Windows NT Workstation Deployment Resources
	-------------------------------------------
	
	  http://www.microsoft.com/ntworkstation/technicalresources/deployment/default.asp
	  (http://www.microsoft.com/ntworkstation/technicalresources/deployment/default.asp)
	
	This Web pages provides links to several tools that are designed to help you plan
	for and deploy Windows NT Workstation 4.0.
	
	Microsoft Windows NT Workstation Resource Kit
	---------------------------------------------
	
	  http://www.microsoft.com/ntworkstation/technicalresources/ResourceKit/default.asp
	  (http://www.microsoft.com/ntworkstation/technicalresources/ResourceKit/default.asp)
	
	This Web page provides information about how to plan for, install, configure, and
	support Windows NT Workstation 4.0 in your organization.
	
	Updated Sysdiff Tool
	--------------------
	
	  ftp://ftp.microsoft.com/bussys/winnt/winnt-public/fixes/usa/nt40/utilities/sysdiff-fix
	  (ftp://ftp.microsoft.com/bussys/winnt/winnt-public/fixes/usa/nt40/utilities/sysdiff-fix)
	
	You can use this link to download the latest version of the Sysdiff tool and read
	Microsoft Knowledge Base articles that address some common Sysdiff issues.
	
	Microsoft Windows NT Workstation Reviewer's Guide
	-------------------------------------------------
	
	  http://www.microsoft.com/TechNet/winnt/ntwrkstn/prodfact/revguide/ntw4revg.asp
	  (http://www.microsoft.com/TechNet/winnt/ntwrkstn/prodfact/revguide/ntw4revg.asp)
	
	You can use this link to download a comprehensive source of information about
	Windows NT Workstation 4.0 features.
	
	Microsoft Internet Explorer Administrator's Kit Corporate Deployment Guide
	--------------------------------------------------------------------------
	
	  http://www.microsoft.com/windows/ieak/en/deploy/Corp/default.ASP?LNK=1
	  (http://www.microsoft.com/windows/ieak/en/deploy/Corp/default.ASP?LNK=1)
	
	This Web page provides information about how to customize, deploy, and administer
	Internet Explorer across multiple platforms.
	
	Implementing Profiles and Policies for Windows NT 4.0
	-----------------------------------------------------
	
	  http://www.microsoft.com/ntserver/management/deployment/planguide/prof_policies.asp
	  (http://www.microsoft.com/ntserver/management/deployment/planguide/prof_policies.asp)
	
	The purpose of this Web page is to provide those who are planning Windows NT
	4.0-based deployments with information about how to design their Windows NT
	4.0-based networks in anticipation of Microsoft Windows 2000 Server-based design
	considerations.
