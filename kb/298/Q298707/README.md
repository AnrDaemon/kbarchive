## Q298707: MSN Explorer (MSN Version 6.1) Sign-in And Sign-up Error Message

	Article: Q298707
	Product(s): The Microsoft Network
	Version(s): 6.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 23-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network version 6.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	For any of the specific sign-in and sign-up error messages below, use the
	corresponding solution. In this KB article, the sign-in and sign-up error
	messages are categorized by type, and are listed alphabetically under the type
	heading.
	
	MORE INFORMATION
	================
	
	Error Message: Please select a country.
	---------------------------------------
	
	This error occurs when the user has not selected anything in the Country box.
	Select a country from the list, and then click Continue.
	
	Error Message: Please select a country before continuing.
	---------------------------------------------------------
	
	A user gets this message if they have not selected a country while going through
	Setup. Select a country from the list, and then click Continue.
	
	Error Message: Please select a postal code.
	-------------------------------------------
	
	This error occurs when the user has left the postal code field blank. Enter a
	postal code, and then click Continue.
	
	Error Message: Please select a region.
	--------------------------------------
	
	This error occurs when the user has not selected anything in the Region (or
	State) box. Select a country from the list, and then click Continue.
	
	Error Message: Please enter a city.
	-----------------------------------
	
	The error occurs when the user leaves the City box blank. Enter the city, and
	then click Continue.
	
	Error Message: Please select a state.
	-------------------------------------
	
	The error occurs when the user leaves the State box blank. Enter the state, and
	then click Continue.
	
	Error Message: Please enter a shorter street name.
	--------------------------------------------------
	
	This error occurs when a user has entered a street that is longer than the
	maximum number of characters allowed. Shorten their street name (possibly by
	abbreviating), and then click Continue.
	
	Error Message: Please enter a street.
	-------------------------------------
	
	The error occurs when the user leaves the Street box blank. Enter the street
	address, and then click Continue.
	
	Error Message: Please enter a Zip code
	--------------------------------------
	
	The error occurs when the user leaves the Zip code box blank. Enter the zip code,
	and then click Continue.
	
	Error Message: Postal code must be greater than # digits.
	---------------------------------------------------------
	
	This error occurs when the user enters a postal code that lacks enough digits to
	be a valid postal code. The number of digits will vary depending on the country
	and region selected. Enter a valid postal code, and then click Continue.
	
	Error Message: Sorry, we are unable to upgrade you because the MSN Internet Access service is only available in the United States. \n\n If you are in the U.S., please check that your Country information in your personal profile is set correctly and try again.
	-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
	
	This issue can occur if the country set in the user's personal profile is set to
	a country other than the United States. Have the user sign in to the account
	with MSN Explorer (this may require a sign-in as a "guest"), and then follow the
	steps below.
	
	To resolve this issue
	
	1. Click Help&Settings, and then click Accounts.
	
	2. Click Change Your Personal Profile Information.
	
	3. In the Country / Region box, select United States.
	
	If the problem persists, contact Technical Support.
	
	Error Message: The name of this city is too long, please enter an abbreviated version.
	--------------------------------------------------------------------------------------
	
	This error occurs when a user has entered a city that is longer than the maximum
	number of characters allowed. Shorten the city name (possibly by abbreviating),
	and then click Continue.
	
	Error Message: The Zip code that you have entered is too long, please check your entry and try again.
	-----------------------------------------------------------------------------------------------------
	
	This error occurs when a user has entered a zip code that is longer than the
	maximum number of characters allowed. Enter the correct zip code, and then click
	Continue.
	
	Error Message: The Zip code you entered is not valid for the state you entered. Please verify that you entered the correct zip code and state and try again.
	------------------------------------------------------------------------------------------------------------------------------------------------------------
	
	The user has entered a zip code that does not match the specified state. Have the
	user check the state and zip code to make sure these are correct, and try
	again.
