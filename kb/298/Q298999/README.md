## Q298999: Baseball2001: Cannot Find the Roster Update

	Article: Q298999
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 29-JAN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Baseball 2001 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you search the Microsoft Baseball 2001 Web site, you do not find a roster
	update.
	
	CAUSE
	=====
	
	There is no updated roster available, and there are currently no plans for one.
