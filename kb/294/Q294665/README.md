## Q294665: Unsolicited E-mail Reduction--Workaround Through Hotmail

	Article: Q294665
	Product(s): The Microsoft Network
	Version(s): 1.0,2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Companion, versions 1.0, 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	To limit unsolicited e-mail or "spam" in your MSN Companion inbox
	
	1. In the search the web box, type www.hotmail.com.
	
	2. Type your Sign-in name and Password.
	
	3. Click the Options button on the horizontal navigation bar.
	  The Options page appears.
	
	4. Under "Mail Handling", click the Inbox Protector link.
	  The Inbox Protector page appears.
	
	5. Select the Typical or Custom box to enable the protector.
	  If you chose Custom, select the boxes as appropriate to direct incoming mail
	  to your Inbox from the types of senders listed under "Custom".
	
	6. Under Discard Folder, do one of the following:
	   - Select the box next to "Bulk Mail" to have Bulk Mail messages
	     automatically deleted 14 days after receipt.
	
	   - Select the box next to "Trash Can" to have Bulk Mail messages
	     automatically deleted several times a week.
	
	7. Do one of the following:
	   - Click OK to apply the filters.
	
	   - Click Cancel to return to the Options page without saving the filters.
