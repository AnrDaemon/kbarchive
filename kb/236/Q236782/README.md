## Q236782: Mapi32.dll File Is Not Included in Service Pack 4 or Later

{% raw %}

	Article: Q236782
	Product(s): Microsoft Windows NT
	Version(s): winnt:4.0
	Operating System(s): 
	Keyword(s): kbtool
	Last Modified: 10-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Server version 4.0 
	- Microsoft Windows NT Workstation version 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Mapi32.dll file is not included in Windows NT 4.0 service packs beginning
	with Service Pack 4.
	
	MORE INFORMATION
	================
	
	The Mapi32.dll file is not included to prevent version conflicts with MAPI-aware
	programs that automatically install an enhanced version of the file.
