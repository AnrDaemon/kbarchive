## Q236462: Asheron's Call: Characters Receive Damage Through Doors

{% raw %}

	Article: Q236462
	Product(s): Microsoft Home Games
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu msgamekbfaq
	Last Modified: 16-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Asheron's Call, version 1.0 
	- Microsoft Asheron's Call: Dark Majesty, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you play Microsoft Asheron's Call, you may receive damage from an enemy
	through a closed door, and you may be able to damage an enemy through a closed
	door.
	
	CAUSE
	=====
	
	This behavior can occur if you stand too close to the door.
	
	RESOLUTION
	==========
	
	To prevent this issue from occurring, do not stand in close proximity to doors.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Asheron's Call,
	version 1.0 and Microsoft Asheron's Call: Dark Majesty.
