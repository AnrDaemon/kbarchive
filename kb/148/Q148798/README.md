## Q148798: Windows 3.1 &#91;MCI&#93; CD Audio Driver Available

{% raw %}

	Article: Q148798
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:1.0,3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-DEC-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Scholastic's Magic School Bus series: Explores the Ocean for Windows, version 1.0 
	- Microsoft Windows 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft Download Center contains the MCI CD Audio driver for Windows 3.1.
	The file is named MCICDA.EXE. Magic School Bus Explores the Ocean requires the
	MCI Audio driver to play sounds correctly. In addition, this driver allows your
	computer to play music from compact discs that are in the Compact Disk-Digital
	Audio (CD-DA; Redbook) format.
	
	For more information about problems MCICDA.EXE corrects, see the following
	article in the Microsoft Knowledge Base:
	
	  Q139659 MSB Ocean: ErrMsg: CD Audio MCI Driver Not Installed
	
	MORE INFORMATION
	================
	
	The following file is available for download from the Microsoft Download
	Center:
	
	  Mcicda.exe
	  (http://download.microsoft.com/download/oceans/Utility/1/W31/EN-US/Mcicda.exe)
	
	For additional information about how to download Microsoft Support files, click
	the article number below to view the article in the Microsoft Knowledge Base:
	
	  Q119591 How to Obtain Microsoft Support Files from Online Services
	
	Microsoft used the most current virus detection software available on the date of
	posting to scan this file for viruses. Once posted, the file is housed on secure
	servers that prevent any unauthorized changes to the file.
