## Q148341: Do-it-Yourself Guide: Minimum System Requirements

	Article: Q148341
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1.00
	Operating System(s): 
	Keyword(s): kbfaq
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Reader's Digest Complete Do-It-Yourself Guide for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following article contains a copy of the minimum system requirements for
	Microsoft Reader's Digest Complete Do-It-Yourself Guide, as listed on the
	product box.
	
	MORE INFORMATION
	================
	
	The minimum system requirements to run Complete Do-It-Yourself Guide are:
	
	- A Multimedia PC or compatible with a 486/33 or higher microprocessor
	
	- Microsoft Windows 95 operating system, or Windows NT Workstation operating
	  system 3.51 or later (will not run on Windows 3.1 or earlier)
	
	- Double-speed CD-ROM drive
	
	- 8 MB of memory (RAM) for Windows 95 or 16 MB for Windows NT Workstation
	
	- 10 MB available hard disk space
	
	- Super VGA display with 256 colors or better
	
	- Windows-compatible sound board required for audio
	
	- Speakers or headphones required for audio
	
	- Microsoft Mouse or compatible pointing device
