	- Microsoft Mail for PC Networks, versions 3.0, 3.2, 3.2a, 3.5

	- Microsoft Mail Multitasking MTA 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article is to assist you in determining the version of executable(s) being
	used when you transmit mail from one Mail postoffice to another Mail postoffice,
	and when you perform Directory Synchronization (Dir-Sync) between Mail
	postoffices.
	
	If the file dates you have are different than those listed below, you probably
	have an updated file applied to your system. If you have questions about a
	specific version not listed, please contact Microsoft Product Support Services.
