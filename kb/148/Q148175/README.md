## Q148175: Scenes 2.0: Minimum System Requirements

{% raw %}

	Article: Q148175
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:2.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Scenes (all collections), version 2.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following article list the minimum system requirements to run Microsoft
	Scenes 2.0 (all collections).
	
	MORE INFORMATION
	================
	
	The minimum system requirements to run Scenes 2.0:
	
	- Personal computer with a 386 or higher processor.
	
	- MS-DOS version 3.1 or later and Microsoft Windows version 3.1 or later.
	
	- 2 MB of available memory (RAM).
	
	- 3 MB of available hard-disk space.
	
	- VGA or higher-resolution monitor (256-color Super VGA monitor recommended).
	
	- Microsoft Mouse or compatible pointing device (recommended).
	
	- Microsoft Scenes supports the most popular graphics file formats including:
	  .BMP, .DIB, .GIF, .JPG, .PCD, .PCX, .TIF, and .TGA.
