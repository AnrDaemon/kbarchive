## Q148794: Gardening: Unable to Enlarge Pictures Using Keyboard Keys

	Article: Q148794
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Complete Gardening for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you explore pictures in Complete Gardening, you are cannot use the keyboard
	TAB key to select and enlarge a picture.
	
	RESOLUTION
	==========
	
	To resolve this problem, move your mouse cursor over a picture and click your
	mouse button. This enlarges the picture.
	
	MORE INFORMATION
	================
	
	You can select many items in Microsoft Complete Gardening with the keyboard
	keys. You can select some items using the TAB key.
