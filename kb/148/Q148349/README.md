## Q148349: Basketball 95-96: Minimum System Requirements

	Article: Q148349
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1995-1996 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Complete NBA Basketball for Windows 1995-1996 edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following article contains a copy of the minimum system requirements for the
	1995-1996 edition of Microsoft Complete NBA Basketball, as listed on the product
	box.
	
	MORE INFORMATION
	================
	
	The minimum system requirements to run Complete NBA Basketball are:
	
	- A Multimedia PC or compatible with a 386SX (486 required for Windows 95) or
	  higher microprocessor
	
	- MS-DOS operating system 3.1 or later with Windows 3.1 or later, OR Windows
	  95, OR Windows NT Workstation 3.51 or later
	
	- CD-ROM drive
	
	- 4 MB of RAM (8 MB recommended)
	
	- 3 MB available hard disk space
	
	- Super VGA or higher-resolution display (for 256 color support)
	
	- Audio board
	
	- Speakers or headphones
	
	- Microsoft Mouse or compatible pointing device
