## Q148236: Bookshelf 96-97: Newsgroup Logon Asks You to Disconnect

	Article: Q148236
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf 1996-97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to access a Newsgroup link from within Bookshelf '96-'97 for
	Windows, the MSN logon screen appears. If you click the Connect option, the
	modem dials out, and after logging in prompts you to disconnect from MSN, The
	Microsoft Network.
	
	RESOLUTION
	==========
	
	To resolve this problem, click No, or wait a few seconds. The disconnect prompt
	disappears and the newsgroup downloads.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem with the '96-'97 edition of
	Bookshelf for Windows. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
