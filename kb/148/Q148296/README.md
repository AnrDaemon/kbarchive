## Q148296: PPT7: Handout Page Numbers Always Start at One

	Article: Q148296
	Product(s): Microsoft PowerPoint for Windows
	Version(s): WINDOWS:7.0
	Operating System(s): 
	Keyword(s): kbprint kbusage kbtemplate kbPrinting
	Last Modified: 07-SEP-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft PowerPoint for Windows 95, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you print handout pages in Microsoft PowerPoint, and you select a range of
	slides to print (for example, slides 11-20), PowerPoint starts numbering the
	handout pages from 1 instead of starting at the first handout page in the slide
	range.
	
	CAUSE
	=====
	
	Page numbering is a built-in feature in PowerPoint that cannot be changed.
	
	Additional query words: 7.00 incorrect wrong page printing 2 3 6 9 per format numbering
