## Q148211: Seven Habits Tools Goals Not Appear When Recurring

	Article: Q148211
	Product(s): Microsoft Schedule+ for Windows
	Version(s): WINDOWS:7.0; Win95:7.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 21-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Schedule+ for Windows, version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Goals that were initially defined in Seven Habits Tools that have been made
	recurring will not appear in the Seven Habits Tools, Goals tab. The goal should
	still appear in the Goals tab and the stone graphic should still be displayed.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in version 7.0 of Microsoft
	Schedule+ for Windows. We are researching this problem and will post new
	information here in the Microsoft Knowledge Base as it becomes available.
	
	
	Additional query words: schedule plus 7.00
