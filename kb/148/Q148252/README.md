	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following article contains a copy of the minimum system requirements for
	Microsoft Bookshelf for Windows, 1994 edition, as listed on the product box.
	
	MORE INFORMATION
	================
	
	The minimum system requirements to run Bookshelf are:
	
	- A Multimedia PC or compatible with a 386SX/25 or higher microprocessor
	
	- MS-DOS operating system version 3.1 or later
	
	- Microsoft Windows operating system version 3.1 or later
	
	- CD-ROM drive
	
	- 4 MB of RAM
	
	- 2 MB available hard disk space
	
	- VGA display (VGA+ with 256 color display recommended)
	
	- Audio board
	
	- Speakers or headphones
	
	- Microsoft Mouse or compatible pointing device
