## Q148258: Bookshelf 1993: Minimum System Requirements

	Article: Q148258
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1993 edition
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf for Windows 1993 edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following article contains a copy of the minimum system requirements for
	Microsoft Bookshelf for Windows, 1993 edition, as listed on the product box.
	
	MORE INFORMATION
	================
	
	The minimum system requirements to run Bookshelf '93 are:
	
	- A Multimedia PC or compatible with a 386SX or higher microprocessor.
	
	- MS-DOS operating system version 3.1 or later
	
	- Microsoft Windows operating system version 3.1 or later
	
	- CD-ROM drive
	
	- 2 MB of RAM
	
	- 2 MB available hard disk space
	
	- VGA display (display capable of 256 colors recommended)
	
	- Audio board
	
	- Speakers or headphones
	
	- Microsoft Mouse or compatible pointing device
