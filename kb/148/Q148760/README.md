## Q148760: Return of Arcade: Setup Does Not Offer Custom Installation

	Article: Q148760
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:1.0
	Operating System(s): 
	Keyword(s): kbsetup kbusagekbfaq
	Last Modified: 08-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Return of Arcade for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The setup program for Return of Arcade does not offer a Custom setup. All four
	games are installed. It is not possible to install a single game.
	
	MORE INFORMATION
	================
	
	The Uninstall option for Return of Arcade removes all four games.
