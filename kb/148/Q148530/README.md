## Q148530: Bookshelf 96-97: Changes to Internet Updates Not Saved

{% raw %}

	Article: Q148530
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf 1996-97 for Windows 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to make a change to the Internet Updates dialog box, the
	changes are not saved.
	
	CAUSE
	=====
	
	Internet Updates are not designed for modification.
	
	Microsoft Windows allows you to change the Internet Updates text. However,
	Bookshelf discards any changes made when you click OK.
	
	WORKAROUND
	==========
	
	To add your own specific notes to any Bookshelf article, press and hold down the
	keyboard CTRL key while you press the N key (CTRL+N). This feature allows you to
	add notes to the article.
	
	MORE INFORMATION
	================
	
	Internet Updates' notes are changed by downloading a new Updates file from the
	Bookshelf World Wide Web (WWW) page.
