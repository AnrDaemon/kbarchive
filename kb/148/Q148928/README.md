## Q148928: Microsoft Plus! Inbox Icon Replaced with Standard Inbox Icon

	Article: Q148928
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 95
	Operating System(s): 
	Keyword(s): win95
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 95 
	- Microsoft Plus! for Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Microsoft Plus! Inbox icon may be replaced with the standard Inbox icon
	after you install MSN, The Microsoft Network.
	
	CAUSE
	=====
	
	This behavior can occur if you install MSN after you install Microsoft Plus!.
	
	RESOLUTION
	==========
	
	Reinstall Microsoft Plus!.
