	- Microsoft The Ultimate Frank Lloyd Wright for Windows, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following article contains a copy of the minimum system requirements for
	Microsoft The Ultimate Frank Lloyd Wright for Windows, as listed on the product
	box.
	
	MORE INFORMATION
	================
	
	The minimum system requirements to run Frank Lloyd Wright are:
	
	- A Multimedia PC or compatible with a 486/33 or higher microprocessor
	
	- MS-DOS operating system 3.1 or later
	
	- Microsoft Windows operating system version 3.1 or later
	
	- Double-speed CD-ROM drive
	
	- 4 MB of RAM (8 MB recommended)
	
	- 500k of available hard disk space
	
	- Super VGA display
	
	- MPC compatible sound card
	
	- Microsoft Mouse or compatible pointing device
