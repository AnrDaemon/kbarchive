## Q148340: Bookshelf 95: Minimum System Requirements

	Article: Q148340
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Bookshelf '95 for Windows 95 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article lists the minimum system requirements for Microsoft Bookshelf '95
	for Windows 95, as listed on the product box.
	
	MORE INFORMATION
	================
	
	The minimum system requirements to run Bookshelf are:
	
	- A Multimedia PC or compatible with a 386SX/25 or higher microprocessor.
	
	- EITHER MS-DOS operating system version 3.1 or later with Microsoft Windows
	  operating system 3.1 or later, OR Windows 95
	
	- CD-ROM drive
	
	- 4 MB of memory
	
	- 4.5 MB of available hard disk space
	
	- Super VGA display 256-color monitor
	
	- Audio board and headphones or speakers (recommended)
	
	- Microsoft Mouse or compatible pointing device
