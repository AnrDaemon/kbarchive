## Q148275: Baseball 95: Minimum System Requirements

	Article: Q148275
	Product(s): Microsoft Home Multimedia Titles
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 16-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Complete Baseball Guide for Windows, 1995 edition 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following article contains a copy of the minimum system requirements for the
	Microsoft Complete Baseball Guide for Windows, 1995 edition, as listed on the
	product box.
	
	MORE INFORMATION
	================
	
	The minimum system requirements to run Complete Baseball are:
	
	- A Multimedia PC or compatible with a 386SX or higher microprocessor
	
	- MS-DOS operating system 3.1 or later
	
	- Microsoft Windows operating system version 3.1 or later
	
	- CD-ROM drive
	
	- 4 MB of RAM
	
	- 3 MB available hard disk space
	
	- Super VGA (640 x 480, 256 color) or higher-resolution display.
	
	- Audio board
	
	- Speakers or headphones
	
	- Microsoft Mouse or compatible pointing device
	
	- Baseball Daily requires a 2400-baud or higher Hayes-compatible modem and an
	  additional 500 KB of hard disk space.
