## Q250582: MSB Bugs: Program Runs Slowly on Macintosh Computer

	Article: Q250582
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS|MACINTOS:1.0
	Operating System(s): 
	Keyword(s): kbenv mskids kbimu
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Scholastic's Magic School Bus series: Explores Bugs, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you run Scholastic's Magic School Bus series: Explores Bugs on a Macintosh
	computer, the program may run slowly and you may experience other performance
	issues.
	
	CAUSE
	=====
	
	This behavior can occur if insufficient system resources are available to run
	the program.
	
	NOTE: This issue has been observed on a Macintosh computer running Mac OS System
	9 with 32 megabytes (MB) of RAM.
	
	RESOLUTION
	==========
	
	To resolve this issue, install more memory in your computer.
	
	MORE INFORMATION
	================
	
	You may be able to work around this issue by disabling resource intensive
	extensions that are loaded when you start the computer.
	
	For more information about how to disable extensions, please refer to the
	following articles in the Apple Tech Info Library:
	
	  http://til.info.apple.com/techinfo.nsf/artnum/n30929
	
	  http://til.info.apple.com/techinfo.nsf/artnum/n6822
