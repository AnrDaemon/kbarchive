## Q250363: Works 2000 Icons Change After Upgrade to Windows 2000

	Article: Q250363
	Product(s): Windows for Workgroups and Windows NT Networking Issues
	Version(s): WINDOWS:2000
	Operating System(s): 
	Keyword(s): kbgraphic kbinterop kbtool kbui kbOSWin2000 kbGrpDSW2KISVkbfaq
	Last Modified: 17-FEB-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows 2000 Professional 
	- Microsoft Works Suite 2000 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you upgrade your computer to Windows 2000, the Works Suite 2000 menu icons
	may be changed to Works word processor icons. Also, the Works Suite 2000 address
	book icon may not be available.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
