## Q250299: Age of Empires II: Game Quits After You Start a New Game

	Article: Q250299
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:2.0
	Operating System(s): 
	Keyword(s): kbenv aoe kbimu msgame
	Last Modified: 12-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Age of Empires II: The Age of Kings, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you start a new game in Microsoft Age of Empires II, the game may quit
	unexpectedly.
	
	CAUSE
	=====
	
	This behavior can occur if your system clock is set to the year 2039 or later.
	
	RESOLUTION
	==========
	
	To resolve this issue, set your system clock to the current date and time. To do
	this:
	
	1. Click Start, point to Settings, and then click Control Panel.
	
	2. Double-click Date/Time.
	
	3. Under Date, click the current month, year, and day.
	
	4. Under Time, click the current time of day.
	
	5. Click OK, and then close Control Panel.
	
	Additional query words: 2.00 aoeii desktop crash close shutdown
