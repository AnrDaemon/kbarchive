## Q250973: FS2000 Professional: Stops Responding When You Start Program

	Article: Q250973
	Product(s): Microsoft Home Games
	Version(s): WINDOWS:
	Operating System(s): 
	Keyword(s): kb3rdparty kbdisplay kbimu msgame
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator 2000 Professional Edition 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to start Microsoft Flight Simulator 2000 Professional Edition,
	the program may stop responding (hang).
	
	CAUSE
	=====
	
	This behavior can occur if an EONtronics Michelangelo video adapter is installed
	in your computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, disable hardware acceleration in Flight Simulator 2000
	Professional Edition:
	
	1. Start Flight Simulator 2000.
	
	2. On the Options menu, point to Settings, and then click Display.
	
	3. Click to clear the "Enable hardware acceleration" check box, and then click
	  OK.
