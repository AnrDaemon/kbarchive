## Q250571: Date Changes in Date/Time Tool Take Effect Dynamically

	Article: Q250571
	Product(s): Windows for Workgroups and Windows NT Networking Issues
	Version(s): WINDOWS:; winnt:4.0,4.0 SP1,4.0 SP2,4.0 SP3,4.0 SP4
	Operating System(s): 
	Keyword(s): kbtool kbui win95 win98 win98se
	Last Modified: 08-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation versions 4.0, 4.0 SP1, 4.0 SP2, 4.0 SP3, 4.0 SP4 
	- Microsoft Windows 98 Second Edition 
	- Microsoft Windows 98 
	- Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you change the date in the Date/Time tool in Control Panel, the computer
	date is changed dynamically, and you do not need to click OK or Apply for this
	change to take place. However, if you click Cancel, no permanent changes are
	made.
	
	STATUS
	======
	
	This behavior is by design.
