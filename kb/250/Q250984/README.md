## Q250984: MSB Bugs: Error Message When You Click &quot;Run Demo&quot;

	Article: Q250984
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS|MACINTOS:1.0
	Operating System(s): 
	Keyword(s): kbenv kberrmsg mskids kbimu
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Scholastic's Magic School Bus series: Explores Bugs, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you click Run Demo on the Scholastic's Magic School Bus series: Explores Bugs
	splash screen, you may receive the following error message:
	
	  Cannot find vids:iv50 decompressor.
	
	CAUSE
	=====
	
	This behavior can occur if you run Magic School Bus series: Explores Bugs on a
	Microsoft Windows 95-based computer.
	
	RESOLUTION
	==========
	
	To resolve this issue, download and install the latest Indeo video codecs from
	the following Intel Web site:
	
	  http://developer.intel.com/ial/indeo/video/details.htm
	
	For additional information about how to reinstall or troubleshoot video
	compression, please click the article numbers below to view the articles in the
	Microsoft Knowledge Base:
	
	  Q177198 How to Reinstall Windows 95 Audio and Video Compression
	
	  Q138261 Error Message: Video Not Available, Cannot Find Decompressor
