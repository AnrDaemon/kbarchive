## Q48884: Mac Wkst: Saving an Undeliverable Mail Message

	Article: Q48884
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:2.0,2.0a,2.0b,3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If Save As is chosen from the Summary window for an undeliverable mail message
	in Microsoft Mail, only the message title, destination server, and date fields
	are saved. The actual message or the enclosure is not saved.
	
	To save the message or the enclosure, use the Retry button to forward the message
	to yourself. Once the message is received, it can be saved normally.
	
	Additional query words: 2.00 2.00a 2.00b 3.00
