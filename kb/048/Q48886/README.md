## Q48886: Mac Wkst: Chime Sounds When Volume Set to Zero

	Article: Q48886
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:2.0,2.0a,2.0b,3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In Microsoft Mail, setting the Macintosh sound volume level to zero (0) will not
	prevent the Mail sound chime from ringing when new mail arrives.
	
	To deactivate the Mail chime, choose Preferences from the Mail menu and select
	Never under Sound Chime.
	
	Additional query words: 2.00 2.00a 2.00b 3.00
