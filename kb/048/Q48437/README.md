## Q48437: View Panning in Flight Simulator Version 4.00

{% raw %}

	Article: Q48437
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	With Flight Simulator Version 4.00, press SHIFT+ENTER to pan downward and
	SHIFT+BACKSPACE to pan upward. Press the "view control" key (SCROLL LOCK on an
	IBM PC with function keys on left, keypad * for IBM PC with function keys on
	top, BREAK on Tandy) to return to level view.
	
	Sideways smooth panning is available on the Tandy 1000 with the LEFT ARROW and
	RIGHT ARROW keys. Press the LEFT ARROW key to pan left and the RIGHT ARROW key
	to pan right.
