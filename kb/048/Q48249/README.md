## Q48249: Mac Form: Shadowed Boxes Are Not Available for Custom Forms

	Article: Q48249
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:2.0,2.0a,2.0b,3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Shadowed boxes are not available for custom forms created with the Microsoft
	Mail Form Designer. When the custom form is compiled, the error message "Can't
	create shadowed boxes" appears. This is a limitation of the program.
