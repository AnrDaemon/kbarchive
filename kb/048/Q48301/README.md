## Q48301: FltSim: Dynamic Scenery Doesn't Stop When Recording a Demo

{% raw %}

	Article: Q48301
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	When recording a Demo using Dynamic Scenery, the scenery continues to move and
	does not relay back to the original location shown when the Demo was recorded.
	
	The Dynamic Scenery is a separate library and runs in the background of a Demo.
