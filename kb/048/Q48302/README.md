## Q48302: Demo or Replay Viewed in Current View in Flight Simulator 4.00

	Article: Q48302
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	The Demo or Replay is viewed in the current view rather than the view in which
	it was recorded. This is a flexible feature that allows you to record a demo of
	landing, for example, then to replay it from various views.
