## Q48759: Paine Field Runway Labled Incorrectly in Flight Simulator 4.00

	Article: Q48759
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	The Paine Field (Snohomish County, WA) runway is labeled incorrectly. The
	correct runway label should be 16-32.
	
	Microsoft is researching this problem and will post new information here as it
	becomes available.
