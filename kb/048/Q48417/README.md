## Q48417: FltSim: Mouse Shadow Remains in Edit Box

	Article: Q48417
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	In Flight Simulator Version 4.00, an image of the mouse pointer remains on the
	edit field when clicking the mouse to activate the selected field.
	
	Microsoft is researching this problem and will post new information here as it
	becomes available.
