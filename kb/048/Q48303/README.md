## Q48303: FltSim: Using the Mouse During a Demo

	Article: Q48303
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	In Flight Simulator Version 4.00, the mouse does not select menus during a Demo
	until the Demo is canceled by pressing the ESC key. This is by design.
