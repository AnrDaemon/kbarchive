## Q48882: Mac Wkst: The Mail Notifier and Cricket Graph

	Article: Q48882
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:2.0,2.0a,2.0b,3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft Mail will not display the Notifier window when any of the tools are
	selected in Cricket Graph. When no tools are selected, the Notifier window will
	be displayed. This problem occurs only with Cricket Graph version 1.10. Cricket
	Graph version 1.30 corrects this problem.
	
	NOTE: If the Display Notifier Window option is not checked in the Mail
	Preferences window, the Notifier window will not be displayed when a new message
	arrives.
