## Q48760: Flight Simulator 4.00: Version 2.00 Parameters Lost

	Article: Q48760
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	Several parameters (altitude, air speed, elevators, flaps, heading, throttle)
	that are saved in Flight Simulator Version 2.00 modes are not loaded accurately
	into Flight Simulator Version 4.00.
	
	Microsoft is researching this problem and will post new information here as it
	becomes available.
