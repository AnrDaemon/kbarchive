## Q19973: Running Flight Simulator on an IBM PC AT with an EGA

	Article: Q19973
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	Flight Simulator Versions 2.12, 2.13, 3.00, and 4.00 can run on an IBM PC AT
	with an installed EGA.
	
	In Versions 2.12 and 2.13, the "Microsoft Flight Simulator Information Manual and
	Flight Handbook" states that you must start the program from Drive A as "FS E";
	for Flight Simulator Versions 3.00 and 4.00, you specify the kind of graphics
	card you have on the start-up menu.
