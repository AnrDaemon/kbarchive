## Q125512: PC Adm: Changing Administrator Name Changes Keep Updates Field

	Article: Q125512
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.2
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for PC Networks, version 3.2 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In version 3.2.x of the Administrator program (ADMIN.EXE), when you change the
	Administrator mailbox name to greater than eight characters, it causes the Keep
	Updates (Days): number to change. Also, if you change the Keep Updates (Days):
	to some number, the administrator line will go blank.
	
	RESOLUTION
	==========
	
	Use an Administrator mailbox name that is eight characters or less.
	
	MORE INFORMATION
	================
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in ADMIN.EXE version 3.2.3, 3.2.9,
	or 3.2.12. We are researching this problem and will post new information here in
	the Microsoft Knowledge Base as it becomes available .
