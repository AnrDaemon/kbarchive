## Q125601: Err Msg: &quot;Could Not Open the Application Database&quot; as PGC Starts

	Article: Q125601
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.1
	Operating System(s): 
	Keyword(s): kbnetwork kbPGC smspgc
	Last Modified: 30-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.1 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	Program Group Control Fails to Open Application Database
	
	When Program Group Control checks your client computer's configuration, the
	following error message appears:
	
	  Could not open the application database.
	  Your configuration will not be updated this time.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.1. This problem has been corrected in Systems Management Server
	version 1.2.
