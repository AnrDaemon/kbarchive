## Q125764: Writer: Edited Stickers Are Not Saved

	Article: Q125764
	Product(s): Microsoft Home Kids Products
	Version(s): WINDOWS:1.0,1.1,1.1a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 29-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Creative Writer for Windows, versions 1.0, 1.1, 1.1a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you click a sticker to edit it on the Crayon Easel, and you then click OK
	to save the changes, the changes you just made are not saved.
	
	CAUSE
	=====
	
	This behavior is most commonly related to the video driver.
	
	RESOLUTION
	==========
	
	To resolve this problem, change the video driver to a standard Windows 640x480
	driver with 256 colors that supports the video card installed in you computer.
	
	For more information about how to perform this task in Windows, see your Windows
	printed documentation or online Help.
