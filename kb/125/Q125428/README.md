## Q125428: Writer/Artist: How to Make Backup Disks

{% raw %}

	Article: Q125428
	Product(s): Microsoft Home Kids Products
	Version(s): MACINTOSH:1.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 29-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Creative Writer for the Macintosh, version 1.0 
	- Microsoft Fine Artist for the Macintosh, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	To make backup copies of your original Microsoft Fine Artist or Creative Writer
	for the Macintosh disks, you must use a third-party disk copying utility.
	
	If you try to make backup copies of the original disks by copying the files from
	the original disks to blank disks, you will not be able to run Setup from the
	backup disks.
	
	
	MORE INFORMATION
	================
	
	A utility called DiskCopy, available from your Apple dealer, can make backup
	copies of the Fine Artist and Creative Writer disks. Please contact your Apple
	dealer for more information about the DiskCopy utility.
