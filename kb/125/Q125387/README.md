## Q125387: SMS Inventory on Serial Mouse Hardware Takes a Long Time

{% raw %}

	Article: Q125387
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:1.1
	Operating System(s): 
	Keyword(s): kbnetwork kbInventory smsinv
	Last Modified: 25-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 1.1 
	-------------------------------------------------------------------------------
	
	
	SYMPTOMS
	========
	
	On computers running Windows 95 and using a serial Microsoft Mouse, inventorying
	the mouse hardware takes over two minutes (instead of a few seconds) and does
	not produce a failed hardware check as it should.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version 1.1. This problem has been corrected in Systems Management Server
	version 1.2.
