## Q125860: Incorrect Range for RegistrySizeLimit in Resource Kit

{% raw %}

	Article: Q125860
	Product(s): Windows for Workgroups and Windows NT Networking Issues
	Version(s): 3.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Windows NT Resource Kit, version 3.5 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Windows NT Registry Entries Help File, REGENTRY.HLP, included with the
	Resource Kit for Windows NT 3.5 incorrectly specifies the RegistrySizeLimit
	parameter range as from "4 Bytes to Unlimited." The correct range is "4
	Megabytes to Unlimited."
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows NT Resource Kit for
	Windows NT 3.5. This problem should be fixed in a future release.
