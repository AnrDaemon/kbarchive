## Q125171: PROFS: How to Print a PROFS Address List

{% raw %}

	Article: Q125171
	Product(s): Microsoft Mail For PC Networks
	Version(s): 3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail Gateway to IBM PROFS and OfficeVision, version 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Mail administrators can generate a list of their PROFS users that can also
	contain the users' assigned user ID numbers. To generate this list, use the
	IMPORT program with the following syntax:
	
	  import admin -ppassword -dm -f<filename to be generated> -x -gprofs
	
	Files generated by IMPORT are text files that can be viewed or printed.
