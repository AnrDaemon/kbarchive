## Q125257: PC Win: Draft Messages Not Sent from Outbox Folder

{% raw %}

	Article: Q125257
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0,3.0b,3.2,3.2a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 04-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, versions 3.0, 3.0b, 3.2, 3.2a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In versions 3.0, 3.0b, 3.2, and 3.2a of Microsoft Mail for Windows, a message
	opened from the Outbox folder may not be sent. If you then close the message, it
	will not be sent until you open it again and choose the Send button. You can
	tell a message will not be sent from the Outbox folder because its icon will
	change to signify a draft message (gray, open envelope).
