## Q125034: Artist: How to Remove or Add Custom Stickers

{% raw %}

	Article: Q125034
	Product(s): Microsoft Home Kids Products
	Version(s): MACINTOSH:1.0,1.1,1.1a; WINDOWS:1.0,1.1,1.1a
	Operating System(s): 
	Keyword(s): kbfaq
	Last Modified: 29-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Fine Artist for Windows, versions 1.0, 1.1, 1.1a 
	- Microsoft Fine Artist for the Macintosh, versions 1.0, 1.1, 1.1a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In Fine Artist, you can create your own stickers. If you want to remove a
	sticker that you have created, do the following:
	
	NOTE: Steps 1 through 3 allow you to save some of your stickers. If you don't
	want to save any of your stickers, you can skip steps 1 through 3.
	
	1. Run Fine Artist and create a new document.
	
	2. Paste any custom stickers you want to keep in the document.
	
	3. Save this document and quit Fine Artist.
	
	4. Locate the ~STICKERS.ART file in your Users folder or directory and select
	  it.
	
	5. To delete the stickers file, follow the procedure appropriate for the kind of
	  computer you are using:
	   - If you are using a computer that runs Windows, press the Delete key, and
	     click OK.
	
	   - If you are using a Macintosh computer, drag the ~STICKERS.ART file to the
	     Trash and choose Empty Trash from the Special menu.
	
	For more information about how to perform this task in Windows, see your Windows
	printed documentation or online Help.
	
	MORE INFORMATION
	================
	
	To create your own stickers in Fine Artist or to restore the stickers you pasted
	into the document in step 3 above, use the following steps:
	
	1. Open a Fine Artist document and add an object, or open the document created
	  in steps 1-3 above.
	
	2. Click the Transformer Tool, then click the Bank Switcher.
	
	3. Click the Sticker Adder Tool.
	
	4. Click the picture or sticker you want to add.
