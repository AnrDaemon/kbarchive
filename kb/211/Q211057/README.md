## Q211057: Computer Stops Responding When Your Plane Stalls

{% raw %}

	Article: Q211057
	Product(s): Microsoft Home Games
	Version(s): MS-DOS:5.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator for MS-DOS, version 5.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When your airplane stalls, your computer may seem to stop responding (hang) for
	several minutes before it continues to respond.
	
	CAUSE
	=====
	
	This problem can occur because this version of Flight Simulator for MS-DOS does
	not work correctly due to it's high timer rate when generating digitized
	sounds.
	
	
	RESOLUTION
	==========
	
	To work around this problem, disable all sound options in Flight Simulator 5.0
	for MS-DOS.
	
	To resolve this problem, upgrade to a version of Microsoft Flight Simulator for
	Windows.
