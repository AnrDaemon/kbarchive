## Q226809: SMS: Incorrect Values Reported for Swap File Size and Swap Used

	Article: Q226809
	Product(s): Microsoft Systems Management Server
	Version(s): winnt:2.0
	Operating System(s): 
	Keyword(s): kbsms200 kbsms200bug
	Last Modified: 22-OCT-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Systems Management Server version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	During a Remote Tools session, incorrect values may be reported for the Swap
	File Size and Swap Used categories on Microsoft Systems Management Server 2.0
	with Microsoft Windows 95 or Windows 98 clients when you use the Windows Memory
	tool.
	
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Systems Management Server
	version SMS 2.0.
	
	MORE INFORMATION
	================
	
	Microsoft is researching this problem and will post more information here in the
	Microsoft Knowledge Base when it becomes available.
