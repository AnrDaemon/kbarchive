## Q35916: Second ATIS Frequency for Oakland International Airport

	Article: Q35916
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	There is a second ATIS frequency for Oakland International Airport. This ATIS is
	for the south field, which is runway 11. The frequency is 128.50 (126.00 is the
	ATIS frequency for the north field).
