## Q92426: LANtastic Allows Hidden and System Files to be Deleted

	Article: Q92426
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:3.x,4.x,5.x,6.0,6.2,6.21,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 17-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 3.1, 3.2, 3.21, 3.3, 3.3a, 4.0, 4.01, 5.0, 5.0a, 6.0, 6.2, 6.21, 6.22 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	LANtastic version 4.1 allows you to delete hidden and system files on LANtastic
	servers. This functionality is specific to LANtastic, and cannot be changed by
	MS-DOS.
	
	For more information on deleting hidden and system file on a LANtastic server,
	please consult your LANtastic documentation.
