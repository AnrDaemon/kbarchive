## Q92643: Some Fonts Are Not Displayed in WordPerfect with Windows 3.1

	Article: Q92643
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 26-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When you run version 5.1 of WordPerfect with Windows 3.1, the following fonts
	are not displayed in font lists:
	
	- Light, Demibold, and Heavy Fonts
	
	- WingDings and Symbol Fonts
	
	WordPerfect Corporation has corrected this problem and has an update available.
	WordPerfect customers can call WordPerfect for information about obtaining the
	update.
	
	MORE INFORMATION
	================
	
	Light, Demibold, and Heavy Fonts
	--------------------------------
	
	WordPerfect does not display the following TrueType fonts:
	
	  Lucida Bright Demibold
	  Lucida Bright Demibold Italic
	  Lucida Calligraphy
	  Lucida Fax Demibold
	  Lucida Fax Demibold Italic
	  Lucida Handwriting
	  Lucida Sans Demibold
	  Lucida Sans Demibold Italic
	  Lucida Sans Typewriter
	  Lucida Sans Typewriter Italic
	
	WordPerfect recognizes only regular and bold versions of a font. If other
	variations, such as light, demibold, or heavy, are available, WordPerfect for
	Windows has difficulty using these fonts.
	
	WingDings and Symbol Fonts
	--------------------------
	
	If you run WordPerfect with Windows 3.1 and use a Windows 3.1 printer driver,
	WordPerfect does not display Symbol or WingDings TrueType fonts in its font
	selection dialog box.
