## Q92778: Setup Hangs, Reboots, or Generates Errors with Racal NI6510

{% raw %}

	Article: Q92778
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you install Windows for Workgroups over a network using a Racal NI6510
	adapter, you may experience various problems. Your machine may stop responding
	(hang), reboot, or produce file copy errors.
	
	CAUSE
	=====
	
	These problems may be caused by an interrupt conflict.
	
	WORKAROUND
	==========
	
	To successfully install Windows for Workgroups, first try changing the interrupt
	on the Racal adapter. If this does not correct the problem, install Windows for
	Workgroups from the floppy disks.
