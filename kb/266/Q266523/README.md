## Q266523: Encarta 2001: Cannot Move Away from Web Center Screen

	Article: Q266523
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 
	Operating System(s): 
	Keyword(s): kbhw kbui kbimu kbHardware
	Last Modified: 25-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Africana, Third Edition 
	- Microsoft Encarta Encyclopedia Deluxe 2001 for Windows 
	- Microsoft Encarta Reference Suite 2001 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use one of the programs listed at the beginning of this article, you
	may be unable to move away from the Web Center screen.
	
	CAUSE
	=====
	
	This behavior can occur if you press and hold down the SHIFT key while you use
	the wheel on your pointing device to move between articles and the Web Center.
	
	RESOLUTION
	==========
	
	Tp resolve this issue, use the Back or Forward navigation buttons on the toolbar
	to move away from the Web Center screen.
	
	When the Web Center screen is no longer displayed, you can press and hold down
	the SHIFT key while you use the wheel on your pointing device to move between
	screens in the products listed at the beginning of this article.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
