## Q266599: Encarta Researcher: Cannot Copy Text from the Web

	Article: Q266599
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 
	Operating System(s): 
	Keyword(s): kbenv kbnetwork kbimu
	Last Modified: 26-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Africana, Third Edition 
	- Microsoft Encarta Encyclopedia Deluxe 2001 for Windows 
	- Microsoft Encarta Encyclopedia Standard 2001 for Windows 
	- Microsoft Encarta Interactive World Atlas 2001 
	- Microsoft Encarta Reference Suite 2001 
	- Microsoft Encarta Researcher 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to copy text from a Web page to the Microsoft Encarta
	Researcher, you may receive the following error message:
	
	  To copy information from the Web:
	  1. Select the text or picture you want.
	  2. Click Add to Researcher.
	  Encarta Researcher collects citation information for you.
	
	CAUSE
	=====
	
	This behavior can occur if either of the following conditions is true:
	
	- Microsoft Internet Explorer security settings may be set too high.
	
	- The "Allow paste operations via script" security setting may be configured to
	  Disable or Prompt.
	
	RESOLUTION
	==========
	
	To resolve this issue, use the following methods in the order in which they are
	presented:
	
	Restore the Default Security Settings in Internet Explorer
	----------------------------------------------------------
	
	To restore the default security settings in Internet Explorer:
	
	1. In Internet Explorer, click Internet Options on the Tools menu.
	
	2. On the Security tab, click Internet.
	
	3. Click Default Level, and then click OK.
	
	  NOTE: You may need to refresh the screen to be able to view the article after
	  you change the security settings in Internet Explorer.
	
	Enable the Allow Paste Operations Via Script Feature
	----------------------------------------------------
	
	To enable the "Allow paste operations via script" feature:
	
	1. In Internet Explorer, click Internet Options on the Tools menu.
	
	2. On the Security tab, click Internet, and then click Custom Level.
	
	3. Under "Allow paste operations via script", click Enable.
	
	4. Click OK, and then click Yes.
	
	5. Click OK.
