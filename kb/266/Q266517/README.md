## Q266517: Encarta Africana: Dictionary Feature Is Not Included

{% raw %}

	Article: Q266517
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 
	Operating System(s): 
	Keyword(s): kbui kbimu
	Last Modified: 26-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Africana, Third Edition 
	- Microsoft Encarta Reference Suite 2001 
	- Microsoft Encarta World English Dictionary 2001 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Microsoft Encarta Africana version 3 does not include a dictionary feature.
	
	MORE INFORMATION
	================
	
	To look up a word, use Microsoft Encarta World English Dictionary 2001.
	
	NOTE: You can install Encarta World English Dictionary 2001 as a stand-alone
	program or as a component of one of the Encarta 2001 programs listed at the
	beginning of this article.
	
	To use Encarta World English Dictionary 2001 to look up a word in Encarta
	Africana, click the QuickShelf icon in the status area of the taskbar, and then
	search for the word that you want to define in the QuickDefine window.
	
	If you copy Encarta World English Dictionary 2001 to the hard disk, you can use
	the following steps to look up the definition of a word:
	
	1. Click the word that you want to look up in Encarta Africana.
	
	2. Click the QuickShelf icon in the status area of the taskbar. The QuickDefine
	  window opens and the definition of the word that you selected in Encarta
	  Africana is displayed.
