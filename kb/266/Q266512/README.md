## Q266512: Encarta Researcher: Keyboard Shortcuts Launch IE Commands

	Article: Q266512
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbtool kbimu
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Africana, Third Edition 
	- Microsoft Encarta Encyclopedia Deluxe 2001 for Windows 
	- Microsoft Encarta Interactive World Atlas 2001 
	- Microsoft Encarta Reference Suite 2001 
	- Microsoft Encarta Researcher, version 1.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you use a keyboard shortcut in one of programs listed at the beginning of
	this article, a Microsoft Internet Explorer command is run instead of an Encarta
	Researcher command.
	
	For example, when you press CTRL+N, you open a new Internet Explorer window
	instead of opening a new Encarta Researcher project.
	
	CAUSE
	=====
	
	When the same keyboard shortcut is used by both Internet Explorer and Encarta
	Researcher, the Internet Explorer command overrides the Encarta Researcher
	command.
	
	RESOLUTION
	==========
	
	To resolve this issue, use the menu commands in Encarta Researcher to perform
	actions instead of using keyboard shortcuts.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in the Microsoft products that are
	listed at the beginning of this article.
