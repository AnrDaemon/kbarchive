## Q81527: International Versions of Windows and Windows for Workgroups

	Article: Q81527
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): 3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): win31
	Last Modified: 04-FEB-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1 
	- Microsoft Windows for Workgroups version 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The following is a list of the international versions of Microsoft Windows that
	are currently available. Additional localized versions are being considered for
	the future.
	
	MORE INFORMATION
	================
	
	Most of these versions are OEM and hardware specific and must be run on the
	hardware suggested by the manufacturer.
