## Q81531: PC Win: Color OLE Object from Excel Displays as Black &amp; White

	Article: Q81531
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you insert a color Object Linking and Embedding (OLE) object from Microsoft
	Excel version 3.0 into version 3.0 of Microsoft Mail for Windows, the object is
	displayed as black and white.
	
	RESOLUTION
	==========
	
	If you double-click the object, then choose Update from the Excel File menu, the
	object appears in color.
