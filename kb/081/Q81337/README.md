## Q81337: PrncDlg.exe - Common Dialogs and Printing Functions

	- Microsoft Windows Software Development Kit (SDK) 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	PrncDlg.exe is a file in the Microsoft Download Center that demonstrates using
	the new printing functions available in Windows 3.1. The sample demonstrates the
	SetAbortProc(), StartDoc(), EndDoc(), StartPage(), and EndPage() functions.
	
	The sample also demonstrates using the ChooseFont() and PrintDlg() common dialog
	boxes provided by the common dialog box library. The sample allows the user to
	select a font and display a line of text to the screen using that font. Also,
	the user can print the same line of text to a printer selected through the
	PrintDlg() common dialog box.
