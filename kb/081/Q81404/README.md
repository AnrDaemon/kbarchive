## Q81404: Fujitsu Color Printer Drivers Available from Fujitsu

{% raw %}

	Article: Q81404
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 07-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Fujitsu has confirmed that the original printer driver for their DL 3400 24-pin
	dot-matrix printer did not support color printing. Fujitsu has reported that new
	drivers available from them have corrected this problem.
	
	MORE INFORMATION
	================
	
	This driver is part of a new drivers package for use in Windows version 3.0 and
	other applications (for example, Word for MS-DOS version 5.0, Works for MS-DOS
	2.x, and Windows 2.x). New drivers for their 24-pin printer line as well as
	their laser printers are included.
	
	The drivers can be obtained from Fujitsu.
	
	The product included here is manufactured by a vendor independent of Microsoft;
	we make no warranty, implied or otherwise, regarding these product's performance
	or reliability.
