## Q81709: Windows: Description of NEC P3200 Printer Driver, Emulation

	Article: Q81709
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	NEC technical support currently has a Windows 3.0 driver available for the NEC
	P3200 Pinwriter printer. This driver can be obtained from NEC technical support
	or from the NEC bulletin board service.
	
	The NEC P3200 printer emulates the NEC P5 series (5200 and 5300) and the Epson LQ
	series of printers.
	
	The NEC products included here are manufactured by a vendor independent of
	Microsoft; we make no warranty, implied or otherwise, regarding these products'
	performance or reliability.
	
	
	Additional query words: 3.00 3.00a pin writer p 3200
