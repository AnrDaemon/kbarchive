## Q81537: Control Panel Shows Only 24-Point in True Type &amp; Plotter

	Article: Q81537
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Control Panel in Microsoft Windows version 3.1 displays True Type and
	plotter fonts in 24-point format only.
	
	MORE INFORMATION
	================
	
	The Fonts dialog box in the Control Panel displays a sample of each point size
	for a bitmapped font with multiple point sizes. It displays True Type fonts and
	plotter fonts, which are scalable, in 24-point only.
