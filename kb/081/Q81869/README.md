## Q81869: Using mciSendCommand() to Control CD-ROM Drive

	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows Software Development Kit (SDK) versions 3.0, 3.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	PLYCDNO.exe is a file in the Microsoft Download Center that demonstrates using
	Media Control Interface (MCI) functions provided by the Windows API to control a
	CD-ROM (compact disk read-only memory) drive.
	
	Functions in PLYCDNO play the CD, pause, move forward and backward by tracks,
	determine the position within the CD, and eject the CD from the drive.
