## Q81711: Windows: IMSI Three-Button Serial Mouse as Mouse Systems

	Article: Q81711
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	To operate the IMSI three-button serial Optimouse with Microsoft Windows, use
	the Mouse Systems drivers included with Windows.
	
	To install the Mouse Systems mouse driver, do the following:
	
	1. Run Setup from Windows or from the Windows directory in MS-DOS.
	
	2. Select Mouse Systems Mouse Connected To COM1: (or COM2:, as appropriate) from
	  Setup.
	
	The IMSI three-button serial Optimouse is manufactured by Mouse Systems and is
	physically identical to the Mouse Systems PC Mouse.
	
	MORE INFORMATION
	================
	
	According to Koon-Ying Enterprises (manufacturer) and IMSI (distributor), the
	Genius Mouse also emulates the Mouse Systems mouse, and Windows should be
	installed as described above for operation with the Genius Mouse.
