## Q81532: PC Win: Row/Column Numbers Missing in Excel OLE Object

	Article: Q81532
	Product(s): Microsoft Mail For PC Networks
	Version(s): WINDOWS:3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for Windows, version 3.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you insert a spreadsheet Object Linking and Embedding (OLE) object from
	Microsoft Excel version 3.0 for Windows into version 3.0 of Microsoft Mail for
	Windows, the object will be displayed with no column or row numbers.
	
	RESOLUTION
	==========
	
	If you double-click the spreadsheet object, then choose Update from the Excel
	File menu, the spreadsheet object then contains the column and row numbers.
