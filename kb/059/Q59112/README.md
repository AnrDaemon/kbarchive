## Q59112: Mac Form: Form Designer Adds Scroll Bar to Rectangular Fields

	Article: Q59112
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:2.0,2.0a,2.0b,3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	When creating a Microsoft Mail custom form, the compiler in the Form Designer
	adds a scroll bar to all unlocked rectangular fields that contain more than 1
	line. If the field is locked, the compiler does not add a scroll bar. If the
	field contains only a single line, the compiler does not add a scroll bar
	whether or not the field is locked.
