## Q59664: Two Errors Reading Empty Drive A in Paintbrush

{% raw %}

	Article: Q59664
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	In Microsoft Windows 3.0 Paintbrush, if you select File, Open, and then select
	Drive A in the dialog box without a disk in Drive A, you receive a "Cannot read
	from drive A:" system error. You also receive a "Cannot find file A:*.BMP" from
	Paintbrush.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in version 3.0. We are researching
	this problem and will post new information here as it becomes available.
