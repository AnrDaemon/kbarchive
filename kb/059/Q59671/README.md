## Q59671: Incorrect Pages in HPPCL Duplex Mode

{% raw %}

	Article: Q59671
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When printing multiple copies of a document to an HP LaserJet IID using the
	HPPCL driver, if Duplex mode is used and a document with an odd number of pages
	is printed, page 1 of the second copy is printed on the back of the last page of
	the first copy.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows version 3.0. We are
	researching this problem and will post new information here as it becomes
	available.
