## Q59639: Must Select Font in List Box to Install

	Article: Q59639
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 02-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	The Control Panel online Help incorrectly states that to add fonts,
	"Enter the font filename and choose OK."
	
	This is not correct. Entering the font filename will not install the
	font. To install a font, select the font name from the list box and
	then choose the OK button.
	
	Additional query words: 3.00 helpfile documentation error
