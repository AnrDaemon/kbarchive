## Q59692: Macro Recorder Won't Record CTRL as First Key Press

	Article: Q59692
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-1999
	
	3.00 3.00a
	WINDOWS
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The Microsoft Windows version 3.0 macro recorder does not record a CTRL+key
	combination as the very first keystroke in a macro recording session.
	
	CAUSE
	=====
	
	To work around this problem, perform some action before recording the CTRL+key
	combination.
	
	STATUS
	======
	
	Microsoft is researching this problem and will post new information here in the
	Microsoft Knowledge Base as it becomes available.
