## Q59683: No Default Saved for Disk Format Options

{% raw %}

	Article: Q59683
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	The Microsoft Windows 3.0 File Manager defaults for disk formatting are
	High Density=yes and Make System Disk=no. If you change these settings,
	they are not saved between sessions.
	
	This limitation is under review and will be considered for modification in
	a future release.
