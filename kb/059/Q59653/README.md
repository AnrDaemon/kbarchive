## Q59653: Inappropriate File Save Error Message in Windows 3.0

	Article: Q59653
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The following inappropriate error message may be displayed in Microsoft Windows
	version 3.0 if you save a file from any application to a floppy disk drive that
	has a write-protected disk or no disk at all:
	
	  Unable to save file A:\FILENAME.EXT; close one or more files in another
	  Windows application and try again.
	
	WORKAROUND
	==========
	
	Closing files in another Windows application has no effect. The solution is to
	have a valid disk in Drive A.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows version 3.0. This
	problem was corrected in later versions of Windows.
