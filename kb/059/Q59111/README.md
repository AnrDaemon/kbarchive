## Q59111: Mac Form: Description of Custom Form Template

	Article: Q59111
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:2.0,2.0a,2.0b,3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Form Template stack on the Microsoft Mail Utilities disk provides template
	cards for each of the components of a custom form (Send screen, Read screen,
	icons, printed form). The template cards contain the objects you manipulate to
	create a custom form. This file is a HyperCard stack.
	
	When creating a custom form, the Form Designer script automatically opens this
	template.
	
	MORE INFORMATION
	================
	
	For more information on creating forms, see the Appendix "Using Custom Forms in
	Microsoft Mail" in the "Microsoft Mail Network Manager's Guide."
