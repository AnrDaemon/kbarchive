## Q59674: Must Restart for Desktop Tile/Center Change

	Article: Q59674
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 15-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you change the Tile/Center option for the desktop bitmap by using the Windows
	Control Panel, the change does not take effect until you quit and restart
	Windows. However, if you change the bitmap as well as the Tile/Center option,
	you see the change immediately.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in versions 3.0 and 3.0a. This
	problem does not occur in later versions of Windows.
