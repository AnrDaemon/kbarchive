## Q59708: Windows 3.0: IBM Proprinter Margin Limitations

	Article: Q59708
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	In Portrait mode, the Microsoft Windows version 3.0 IBM Proprinter driver
	has a top- and bottom-margin limitation of .5 inch, and a left- and right-
	margin limitation of .25 inch. The printer itself is not physically limited
	to these margins.
