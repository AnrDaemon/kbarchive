## Q59663: Program Manager Group Items Arranged Vertically

{% raw %}

	Article: Q59663
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you add program items to a minimized group in Program Manager, the icons are
	arranged vertically below the last item in the group.
	
	This also occurs when Windows Setup installs applications to a minimized group.
	
	WORKAROUND
	==========
	
	To rearrange the icons, choose Arrange Icons from the Window menu.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows version 3.0. This
	problem was corrected in later versions of Windows.
