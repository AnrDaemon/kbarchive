## Q59695: Can't Have Groups Within Groups in Program Manager

	Article: Q59695
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	The Microsoft Windows 3.0 Program Manager does not support groups
	within other groups. This feature is under review and will be
	considered for inclusion in a future release.
	
	Additional query words: 3.00 win30 3.00a
