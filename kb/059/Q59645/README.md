## Q59645: Empty Dialog Box on Novell in Windows 3.0

	Article: Q59645
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You may receive an empty error dialog box if your Novell NetWare server goes
	down while executing an Attach from the NetWare network dialog box in the
	Microsoft Windows version 3.0 Control Panel.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows version 3.0. This
	problem does not occur in later versions of Windows.
