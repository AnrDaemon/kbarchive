## Q59696: Write Not Printing Graphics in Draft Mode

{% raw %}

	Article: Q59696
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	Write is designed to not print graphics when the Draft Mode box is
	checked in the File Print dialog box. Draft Mode is intended to be a
	text-only output mode.
	
	Additional query words: 3.00 win30 3.00a win31 3.10 3.11
