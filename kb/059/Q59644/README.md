## Q59644: HPPCL Err Msg: Please Specify the Complete Path

{% raw %}

	Article: Q59644
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	If you receive the error 
	
	  Please specify the complete path
	
	when
	installing printer fonts via the HPPCL Font Installer, this means that
	there is not enough disk space to create the \PCLFONTS directory.
	
	Additional query words: 3.00 win30 3.00a
