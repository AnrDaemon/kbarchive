## Q59649: Using CTTY Under Windows 3.0

	Article: Q59649
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	Use of the MS-DOS CTTY command is not recommended under Microsoft Windows
	3.0. It does not work properly in real or standard modes, although it may
	work in enhanced mode. (Use in enhanced mode has not been tested.)
	
	Additional query words: 3.00 win30 3.00a
