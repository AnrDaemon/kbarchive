## Q59662: Can't Arrow Between Items in Program Manager Group

	Article: Q59662
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you have only one group in Microsoft Windows 3.0 Program Manager, you cannot
	use the arrow keys to move around between items in that group.
	
	WORKAROUND
	==========
	
	The workaround is to create a second group.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Windows version 3.0. We are
	researching this problem and will post new information here as it becomes
	available.
