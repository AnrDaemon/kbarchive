## Q59680: File Manager &quot;System Error&quot; When SUBST'd Drive Present

	Article: Q59680
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 08-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	Microsoft Windows version 3.0 does not support the use of the MS-DOS SUBST
	command. If you have a SUBST'd drive present, when you try to run Windows
	3.0 File Manager, the following error message is displayed:
	
	System Error: Network Error on drive ?:
	
	Additional query words: 3.00 3.0 3.0a 3.00a win30
