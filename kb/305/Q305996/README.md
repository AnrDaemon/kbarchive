## Q305996: MSN Broadband DSL:  How To Determine The Version Of Windows

	Article: Q305996
	Product(s): The Microsoft Network
	Version(s): 4.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN DSL, version 2.0, used with:
	   - the operating system: Microsoft Windows XP 
	   - the operating system: Microsoft Windows 2000 
	   - the operating system: Microsoft Windows Millennium Edition 
	   - the operating system: Microsoft Windows 98 Second Edition 
	   - the operating system: Microsoft Windows 98 
	   - the operating system: Microsoft Windows 95 
	   - the operating system: Microsoft Windows NT 4.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article will help you determine which operating system you have, so you can
	determine the level of support available for MSN Broadband DSL.
	
	MORE INFORMATION
	================
	
	Windows 95, Windows 98, Windows 98 Second Edition, Windows Millennium Edition,
	and Windows NT
	
	1. Click Start, point to Settings, and then click Task Bar and Start Menu.
	
	2. Make sure that the Show small icons in Start menu box is clear.
	  If this box is selected, clear the box, and then click Apply.
	
	3. Click the Start button.
	  The version will appear on the the left side of the Start menu.
	
	Windows 2000 Professional, Windows 2000 Server, or Windows XP Professional
	
	1. Press CTRL+ALT+DEL.
	
	2. The splash screen will show the version.
	
	Depending on the operating system you have, please note the level of support offered:
	-------------------------------------------------------------------------------------
	
	- Windows 95: This is not a supported operating system. You must upgrade to
	  receive support.
	
	- Windows 98: This is not a supported operating system. You must upgrade to
	  receive support.
	
	- Windows 98 Second Edition: Full support.
	
	- Windows Millennium Edition: Full Support.
	
	- Windows NT: This is not a supported operating system. You must upgrade to
	  receive support.
	
	- Windows 2000 Professional: Full support.
	
	- Windows 2000 Server: This is not a supported operating system.
	
	- Windows XP: Full support.
