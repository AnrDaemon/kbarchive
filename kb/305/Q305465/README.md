## Q305465: Microsoft Excel Version 2002 Inside Out Comments and Corrections

{% raw %}

	Article: Q305465
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 13-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Microsoft Excel 2002 Inside Out ISBN 0-7356-1281-1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Microsoft Excel Version 2002 Inside Out,
	ISBN 0-7356-1281-1.
	
	The following topics are covered:
	
	- Page 908: Change "Locals0" to "Locales"
	
	- Page 919: Insert 1/2 Symbol In Table
	
	- Page 974: Incorrect Function Name In Table Entries
	
	MORE INFORMATION
	================
	
	Page 908: Change "Locals0" to "Locales"
	---------------------------------------
	
	On page 908 change "Locals0" to "Locales" and insert a closing parenthesis
	directly after. The sentence should read as follows:
	
	"If the Text Services icon is not available in your Windows Control Panel,
	double-click the Keyboard icon, click the Languages (or Input Locales) tab,
	where you can change or add a language."
	
	
	Page 919: Insert 1/2 Symbol In Table
	------------------------------------
	
	On page 919, in Table B-2, change the last entry in the "To enter" column from:
	"-"
	
	To:
	"1/2"
	
	
	Page 974: Incorrect Function Name In Table Entries
	--------------------------------------------------
	
	On page 974, in the "Description" column of the table, the following changes
	should be made:
	
	In the 2nd entry, 2nd line, change "=REPLACE" to "=REPLACEB"
	
	In the 5th entry, 2nd line, change "=RIGHT" to "=RIGHTB"
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
	
	Additional query words: OFFICE 0-7356-1281-1 STINSON DODGE
