## Q305821: ENC2002: Minimum System Requirements for Encarta

	Article: Q305821
	Product(s): Microsoft Home Multimedia Titles
	Version(s): 1.0
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 12-NOV-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta Encyclopedia Deluxe 2002, version 1.0 
	- Microsoft Encarta Encyclopedia Standard 2002, version 1.0 
	- Microsoft Encarta Reference Library 2002, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article summarizes the minimum system requirements needed to run any of the
	Microsoft Encarta programs listed at the beginning of this article.
	
	MORE INFORMATION
	================
	
	To run any Microsoft Encarta 2002 reference product, your computer must meet the
	following minimum system requirements:
	
	- A Microsoft Windows 98, Microsoft Windows Millennium Edition (Me), Microsoft
	  Windows 2000 Professional, or Microsoft Windows XP operating system.
	  Microsoft Windows 95 and Microsoft Windows NT are not supported.
	
	- A Pentium 200-or-higher processor on a multimedia computer. A Pentium
	  300-or-higher processor is recommended.
	
	- A minimum of 32 or 64 megabytes (MB) of random access memory (RAM). A minimum
	  of 64 MB is recommended in all cases, although 32 MB is supported for Windows
	  98 and Windows Me only. For 3D Virtual Tours (Deluxe and Library), 64 MB of
	  RAM is required.
	
	- Hard drive space requirements:
	
	  Reference Library - 330 MB
	  Encyclopedia Deluxe - 265 MB
	  Encyclopedia Standard - 170 MB
	
	- A DVD drive (for the DVD edition), or a 4X-or-faster CD-ROM drive.
	
	- A 16-bit sound card with speakers or headphones.
	
	- A monitor with a resolution of at least 800 x 600 pixels and 16-bit color.
	
	- A Microsoft Mouse-or-compatible pointing device.
	
	- Local bus video with 1 MB or more of video memory.
	
	Some features require an Internet connection:
	
	- Microsoft Internet Explorer 5.5 (which is included) or later:
	
	  - Requires up to 45 MB of additional hard-disk space; 25 MB of additional
	  disk space is needed during installation.
	
	- Update Encarta, Web links, Live Media, FactFinder, and Online Deluxe:
	
	  - Requires a 14.4 Kbps-or-faster modem, or another connection to the
	  Internet.
	
	- Club Encarta (register over the Internet):
	
	  - Requires up to 2 MB of additional hard-disk space for each monthly
	  installment.
	
	- Online Deluxe (not included with Encyclopedia Standard):
	
	  - Requires a free Microsoft Passport account.
	
	The Copy to Hard Drive feature requires the following amount of hard-disk space:
	
	   - Encyclopedia Standard - 500 MB
	
	- Encyclopedia Deluxe - 1.6 gigabytes (GB)
	
	- Reference Library - 2.2 GB
