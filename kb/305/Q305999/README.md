## Q305999: MSN Broadband DSL: Lost DSL Connection During New User Setup

	Article: Q305999
	Product(s): The Microsoft Network
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-AUG-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN DSL, version 2.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You lose your connection.
	
	RESOLUTION
	==========
	
	The correct resolution depends on which screen you had displayed when you the
	disconnection occurred:
	
	Page 1:  The screen where you enter your user name and password.
	----------------------------------------------------------------
	
	- If connection is lost here, start the installation again.
	
	Page 2:  This screen displays, "Please wait while we configure your modem"
	--------------------------------------------------------------------------
	
	- If connection is lost here, start the installation again.
	
	- If a second attempt does not resolve the issue, unplug the Customer Premises
	  Equipment (CPE) or Digital Subscriber Line (DSL) modem, wait 20 seconds, plug
	  the modem back in, and then try installation from the CD again.
	
	- If the installation does not start again automatically after reconnecting the
	  modem, double-click the MSN Explorer icon. This starts the new user Setup
	  process from the beginning.
	
	Page 3:  This screen displays, "Almost Finished--Click Continue"
	----------------------------------------------------------------
	
	- Unplug the modem, wait 20 seconds, plug the modem back in, and then try
	  installation from the CD again.
	
	- If the installation does not start again automatically, double-click the MSN
	  Explorer Icon on the desktop. This starts the new user Setup process from the
	  beginning.
	
	Page 4:  Error Page.
	--------------------
	
	Note: This may come up at any time during the process.
	
	- If connection is lost here, start the installation again.
	
	- If a second attempt does not resolve the issue, unplug the Customer Premises
	  Equipment (CPE) or Digital Subscriber Line (DSL) modem, wait 20 seconds, plug
	  the modem back in, and then try installation from the CD again.
	
	- If the installation does not start again automatically, double-click the MSN
	  Explorer Icon on their desktop. This starts the new user Setup process from
	  the beginning.
