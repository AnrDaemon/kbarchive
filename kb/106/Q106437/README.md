## Q106437: The F1 Key in Chat and NetWatcher May Not Open Help

	Article: Q106437
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 04-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	In most Windows-based applications, you can use the F1 key to open Help;
	however, this is not always the case with Chat and NetWatcher.
	
	
	MORE INFORMATION
	================
	
	In Windows for Workgroups 3.1, the F1 key does not open Help for Chat or
	NetWatcher.
	
	In Windows for Workgroups 3.11, the F1 key does not open Help for NetWatcher. It
	does, however, open Help for Chat.
	
	The only workaround is to open Help using the Help menu within the application.
