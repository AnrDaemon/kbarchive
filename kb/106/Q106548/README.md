## Q106548: Encarta: GP Fault with Diamond Speedstar Pro Video Card

{% raw %}

	Article: Q106548
	Product(s): Microsoft Home Kids Products
	Version(s): 1994 edition; WINDOWS:
	Operating System(s): 
	Keyword(s): 
	Last Modified: 13-JUN-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 1994 The Complete Multimedia Encyclopedia 
	- the operating system: Microsoft Windows 95 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Using the Diamond Speedstar Pro video card and driver in 1024 x 768 256-color
	Large Fonts mode may cause a general protection fault (GPF) when Encarta draws
	its contents screen. Also, some screens may be partially cut off, or text may be
	illegible. This problem may also occur when using the Diamond Speedstar Pro
	video card and driver with 800 x 600 256-color Large Fonts.
	
	All other video modes work correctly, including the 1024 x 768 256-color Small
	Fonts mode.
	
	RESOLUTION
	==========
	
	If You Are Using 1024 x 768 256-Color Large Fonts
	-------------------------------------------------
	
	Switch to a video mode other than the 1024 x 768 256-color Large Fonts mode.
	
	If You Are Using 800 x 600 256-Color Large Fonts
	------------------------------------------------
	
	Switch to the 640 x 480 256-color Small Font video mode.
