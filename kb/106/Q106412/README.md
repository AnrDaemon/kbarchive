## Q106412: MSAV: Disk Error Prevents Further Virus Checking

	Article: Q106412
	Product(s): Microsoft Disk Operating System
	Version(s): MS-DOS:6.2,6.22
	Operating System(s): 
	Keyword(s): 
	Last Modified: 20-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft MS-DOS operating system versions 6.2, 6.22 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If Microsoft Anti-Virus or Microsoft Anti-Virus for Windows encounters a disk
	error while scanning for viruses, it forces you to exit and does not continue
	scanning.
	
	
	RESOLUTION
	==========
	
	To work around this problem, run ScanDisk and then run Microsoft Anti-Virus
	again.
	
	Additional query words: MWAV 6.20
