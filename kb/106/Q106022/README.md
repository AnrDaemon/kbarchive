## Q106022: Network Dialog Err Msgs Truncated with Small Fonts

	Article: Q106022
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 03-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	If you install the XGA small fonts video driver, only part of error message text
	may be visible in several of the network dialog boxes. For example, you may see
	the following error messages:
	
	  Access has been
	
	  The selected server cannot be found on the
	
	  The specified share name cannot be
	
	  Invalid
	
	  instead of
	
	  Invalid password
	
	  The computer name specified in the network path cannot be
	
	WORKAROUND
	==========
	
	To work around this problem, run Windows Setup from the Main group and install
	the XGA large fonts video driver.
	
	STATUS
	======
	
	Microsoft has confirmed this to be a problem in Microsoft Windows for Workgroups
	version 3.11. We are researching this problem and will post new information here
	as it becomes available.
