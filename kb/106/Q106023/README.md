## Q106023: WFWG Setup Leaves Two Sets of Mail and Schedule+ Icons

	Article: Q106023
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you upgrade from Windows for Workgroups version 3.1 to Windows for
	Workgroups version 3.11, you are left with two sets of Microsoft Mail and
	Microsoft Schedule+ icons: one set in the Main group and one set in the Network
	group.
	
	RESOLUTION
	==========
	
	Both sets of icons point to the same programs. You can delete either set of
	icons or keep both.
	
	NOTE: Deleting a program icon does not delete the program files.
