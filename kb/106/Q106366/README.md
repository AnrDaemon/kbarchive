## Q106366: WFWG Hangs at Startup with Future Domain 32-Bit Disk Access

{% raw %}

	Article: Q106366
	Product(s): Microsoft Windows 3.x Retail Product
	Version(s): WINDOWS:3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 27-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows for Workgroups version 3.11 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	Versions earlier than 1.02 of Future Domain's 32-bit disk access drivers may
	cause Windows for Workgroups to stop responding (hang) at startup.
	
	RESOLUTION
	==========
	
	Disable Future Domain's 32-bit disk drivers until you obtain an updated
	driver(s).
	
	MORE INFORMATION
	================
	
	For more information on obtaining updated 32-bit disk access drivers for Future
	Domain controllers, contact Future Domain at one of the following numbers:
	
	  Software Upgrade Order Line: (800) 879-7599
