## Q106686: Encarta GP Fault: Printing with Universal and Thirdparty Video

{% raw %}

	Article: Q106686
	Product(s): Microsoft Home Kids Products
	Version(s): 1994 edition
	Operating System(s): 
	Keyword(s): kbbuglist
	Last Modified: 26-MAY-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Encarta 1994 The Complete Multimedia Encyclopedia 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	You may receive a general protection (GP) fault when printing from MindMaze and
	using the universal printer driver.
	
	This problem has been reported with the following drivers:
	
	- version 1.5 of the ATI Mach 32 video driver. This problem occurs if you are
	  in 62,000-color mode.
	
	- Vision 864 1.2b8 display driver set at 800x600x64k colors.
	
	Microsoft has confirmed this to be a problem in Encarta 1994 edition. We are
	researching this problem and will post new information here in the Microsoft
	Knowledge Base as it becomes available.
	
	RESOLUTION
	==========
	
	ATI Driver
	----------
	
	To correct this problem with the ATI driver, do one of the following:
	
	- Use version 2.0 of the ATI Mach 32 video driver.
	
	- Switch to 256-color mode.
	
	- Use a printer driver other than the universal driver.
	
	To obtain the updated Mach 32 driver, contact the ATI Technologies Inc. technical
	support department at (905) 882-2626.
	
	Other Drivers
	-------------
	
	To correct this problem, do one of the following:
	
	- Switch to 256-color mode.
	
	- Use a printer driver other than the universal driver.
	
	The third-party products discussed here are manufactured by vendors independent
	of Microsoft; we make no warranty, implied or otherwise, regarding these
	products' performance or reliability.
