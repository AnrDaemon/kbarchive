## Q51239: Flt Sim: Nose Pitches Up When Executing a Turn with Joystick

	Article: Q51239
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	In Flight Simulator version 4.00 or 4.00a, if you set the elevator trim control
	and then execute a level turn with the joystick, the nose of the Cessna or
	Learjet will pitch up.
	
	Microsoft has confirmed this to be a problem in Flight Simulator versions 4.00
	and 4.00a. This problem was corrected in Flight Simulator version 4.00b.
