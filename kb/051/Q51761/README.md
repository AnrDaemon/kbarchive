## Q51761: Flight Simulator 4.00: Heading Drifts to the Left

	Article: Q51761
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	Flight Simulator version 4.00 has a tendency to drift to the left approximately
	2 degrees per minute when there is no autopilot input.
	
	Microsoft has confirmed this to be a problem in Flight Simulator versions 4.00
	and 4.00a. This problem was corrected in Flight Simulator version 4.00b.
