## Q51280: Mac Form: Minimum System Requirement for Running Form Designer

{% raw %}

	Article: Q51280
	Product(s): Microsoft Mail For Appletalk Networks
	Version(s): WINDOWS:2.0,2.0a,2.0b,3.0
	Operating System(s): 
	Keyword(s): 
	Last Modified: 09-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Following are the minimum system requirements for running the Microsoft Mail
	Form Designer:
	
	1. Macintosh Plus or newer computer with at least 1 megabyte (MB) of available
	  memory
	
	2. One 800K disk drive
	
	3. System version 6.0.2 or later
	
	4. An AppleTalk or compatible local area network
	
	5. HyperCard version 1.2.2 or later
	
	6. Network Manager log-in to Microsoft Mail Server
	
	Additional query words: 2.00 2.00a 2.00b 3.00
