## Q51897: Using Flight Simulator Version 3.00 Logbook with Version 4.00

{% raw %}

	Article: Q51897
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	It is possible to use the logbooks created using Microsoft Flight Simulator
	Version 3.00 with Microsoft Flight Simulator Version 4.00. The logbook is a text
	file and can be copied from its current location into the Flight Simulator 4.00
	directory or floppy disk.
