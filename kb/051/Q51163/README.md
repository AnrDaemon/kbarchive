## Q51163: Tandy 1000 Beeps When Flight Simulator 4.00 Is Started

	Article: Q51163
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	With some Tandy 1000 series computers running Flight Simulator version 4.00, a
	continuous beep can be heard when the program loads. Even after exiting the
	program, the beeping continues. The only way to stop the beeping is to reboot
	the computer.
	
	The problem is associated with ROM BIOS versions 1.00 and 1.01 in the computer.
	
	Additional query words: sound tone noise loud
