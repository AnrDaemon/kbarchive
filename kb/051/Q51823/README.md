## Q51823: Flt Sim: Can't Connect with Other User in Multiplayer Mode

	Article: Q51823
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	Flight Simulator version 4.00 experiences problems connecting with other users
	when you use a modem in the multiplayer mode.
	
	Microsoft has confirmed this to be a problem in Flight Simulator versions 4.00
	and 4.00a. This problem was corrected in Flight Simulator version 4.00b.
