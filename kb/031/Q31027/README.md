## Q31027: Windows 3.0 Video Setup for Compaq Deskpro 286

	Article: Q31027
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:1.x,2.x,3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 1.1, 2.03, 2.1, 2.11, 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you use a Compaq Deskpro 286 Portable that does not have a plasma display,
	but has a built-in green screen display, install Microsoft Windows with the CGA
	graphics adapter.

	================
