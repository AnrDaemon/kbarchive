## Q31381: Everex RAM 3000 Card

	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Everex RAM 3000 is an extended/expanded memory card with a maximum capacity
	of 3 megabytes. It runs at a maximum bus speed of 10 megahertz with one wait
	state. Everex has a LIM (Lotus-Intel-Microsoft) EMS 4.0 specification driver
	available, which Everex claims is compatible with Microsoft Windows.
	
	MORE INFORMATION
	================
	
	The RAM 3000 is technically a LIM 3.2 card with a LIM 4.0 driver. Backfilling
	motherboard memory is possible with the RAM 3000, but the EMS page-frame size
	remains limited to a maximum of 64K.
	
	Further inquiries on the RAM 3000 should be directed to Everex Technical
	Support.
