## Q31987: Maximum Number of Object Modules LINK Supports

	Article: Q31987
	Product(s): Microsoft Programming Utilities
	Version(s): MS-DOS:3.x,5.01.20,5.01.21,5.03,5.05,5.1,5.13,5.15,5.3,5.31.009,5.5,5.6; winnt:1.0,2.5
	Operating System(s): 
	Keyword(s): 
	Last Modified: 05-NOV-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft LINK for MS-DOS, versions 3.x, 5.01.20, 5.01.21, 5.03, 5.05, 5.1, 5.13, 5.15, 5.3, 5.31.009, 5.5, 5.6 
	- Microsoft LINK for Windows NT, versions 1.0, 2.5 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	The Microsoft LINK utility does not impose any limit on the number of object
	modules specified on the linker command line. However, LINK is constrained by
	the amount of available system resources.
