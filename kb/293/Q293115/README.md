## Q293115: Viewing .ema Or .email Attachments In MSN Explorer

	Article: Q293115
	Product(s): The Microsoft Network
	Version(s): 6.0,6.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 20-JUL-2001
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network versions 6.0, 6.1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	If you receive an e-mail attachment with the extension of .ema or .email in MSN
	Explorer, you may receive an error message when you try to open the attachment.
	
	To view .ema or .email attachments
	
	1. Open the e-mail message that includes the attachment.
	
	2. Click the attachment link in the e-mail message.
	  A dialog box appears.
	
	3. Select Save to Disk, and then click OK.
	
	4. In the Save In box, select Desktop, and then click the Save button.
	  If MSN Explorer is maximized (covering the entire desktop), click the Restore
	  button to display your entire desktop.
	  An icon for the attachment that you saved appears on the desktop.
	
	5. Do one of the following:
	   - Click the attachment icon, and then drag the mouse pointer to the middle
	     of the MSN Explorer window.
	     You can now view the attachment in the MSN Explorer window.
	
	   - Double-click the attachment icon on the desktop.
	     A box appears asking you what program you want to open the attachment with.
	     Scroll through the list until you see Notepad, select Notepad, and then
	     click OK.
