## Q39807: Error Code: 0006 in Flight Simulator Version 3.00 and 4.00

	Article: Q39807
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	If the following message appears, check to make sure your program disk is not
	write protected:
	
	  Error Code 0006: Error occurred while closing file;
	
	                   filename AUTOEXEC.FS3 for Version 3.00
	                   filename AUTOEXEC.FS4 for Version 4.00
	                   filename A.G3D for Version 4.00
	
	If the message still appears, delete the file AUTOEXEC.FS3 for Version 3.00, or
	AUTOEXEC.FS4 for Version 4.00. This file saves the selections you made the last
	time you chose "Go through startup menus" from menu option 1. Reentering the
	Startup menu option responses will re-create the contents of the AUTOEXEC.FS3 or
	AUTOEXEC.FS4 file. The file A.G3D may be corrupt and need to be deleted and
	copied again from the distribution diskette.
	
	MORE INFORMATION
	================
	
	If you get Error 0006 with Microsoft Flight Simulator 5.0 query on the following
	keywords "FS5 and ERROR 0006".
