## Q39321: Okidata 182 Printer Under Windows

	Article: Q39321
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:2.x,3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 18-DEC-2000
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 2.03, 2.1, 2.11, 3.0, 3.0a, 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The Okidata 182 printer is not a supported device under Microsoft Windows.
	
	Okidata Technical Support offers an upgrade ROM kit that allows the Okidata 182
	to emulate the IBM Graphics printer. The part number for this kit is 310-324-01.
	To obtain the kit, contact Okidata.
	
	The products included here are manufactured by vendors independent of Microsoft;
	we make no warranty, implied or otherwise, regarding these products' performance
	or reliability.
