## Q318335: FS2002: Err Msg: Cannot Detect Windows Media Player

	Article: Q318335
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): kberrmsg kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator 2002 Professional Edition 
	- Microsoft Flight Simulator 2002 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you try to start Flight Simulator, you may receive an error message similar
	to the following:
	
	  Flight Simulator cannot detect Windows Media Player on your system.
	
	CAUSE
	=====
	
	This issue can occur because Microsoft Windows Media Player version 7.1 is not
	installed when you install Flight Simulator on your computer.
	
	RESOLUTION
	==========
	
	To solve this problem, install Media Player from the Flight Simulator 2002
	CD-ROM. To do this, follow these steps:
	
	1. Insert Flight Simulator 2002 CD 1 into the CD-ROM drive or DVD-ROM drive.
	
	  Setup may start automatically. If Setup starts, click Close to quit Setup.
	
	2. Double-click the My Computer icon on your Windows desktop, right-click
	  FS_DISK_1, and then click Install Windows Media Player.
	
	  Setup starts the installation process automatically.
	
	3. When the installation process is complete, restart your computer.
