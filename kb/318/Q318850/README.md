## Q318850: FS2002: Error Message When You Try to Read a Library/Help File

{% raw %}

	Article: Q318850
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): kbimu
	Last Modified: 07-MAY-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Flight Simulator 2002 Professional Edition 
	- Microsoft Flight Simulator 2002 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	When you attempt to open any of the Microsoft Flight Simulator 2002 handbooks in
	Library/Help, you receive an error message similar to one of the following:
	
	  You must have Adobe Acrobat Reader installed to read this handbook. Please
	  install Acrobat Reader and restart Flight Simulator 2002.
	
	  -or-
	
	  Paint cannot read the file, this is not a valid bitmap file or its format is
	  not currently supported.
	
	CAUSE
	=====
	
	This problem occurs when one of the following conditions is true:
	
	- The Adobe Acrobat Reader program was not installed correctly when Flight
	  Simulator was installed.
	
	  -or-
	
	- The Adobe Acrobat Reader program on your computer is corrupted or damaged.
	
	RESOLUTION
	==========
	
	To resolve this problem, reinstall Adobe Acrobat Reader. To do this, follow
	these steps:
	
	1. Insert Flight Simulator 2002 Disc 1 into your CD-ROM drive or DVD-ROM drive
	  and hold down the SHIFT key to prevent Autorun from starting.
	
	2. Double-click My Computer on the desktop.
	
	3. Right-click FS_DISK_1, and then click Install Adobe Acrobat Reader 4.05.
	
	4. Follow the on-screen instructions to install Acrobat Reader.
