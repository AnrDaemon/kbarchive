## Q318972: How To Add A POP3 MSN E-mail Account To Microsoft Entourage

	Article: Q318972
	Product(s): The Microsoft Network
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-JUN-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Internet Access for Macintosh, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to add a POP3 MSN e-mail account to Microsoft
	Entourage.
	The information in this article applies to:  - Microsoft Entourage
	
	MORE INFORMATION
	================
	
	To add a POP3 MSN e-mail account to Microsoft Entourage
	
	1. Start Microsoft Entourage.
	
	2. Click Tools, and then click Accounts.
	
	3. In the Accounts window, click New.
	
	4. Select POP as the Account Type, and then click OK.
	
	5. On the Edit Account window, configure the account with the following
	  settings:
	   - Account name: Type a user-friendly name to describe the account.
	     Example: MSN News
	
	   - Name: Type a "Display Name". This name will display to readers of your
	     e-mail messages.
	     Example: John Doe
	
	   - E-mail address: Type your @msn.com e-mail address.
	     Example: msnuser@msn.com
	
	   - Account ID: Type your MSN member ID. Type all characters before the "@"
	     symbol in your e-mail address.
	     Example: msnuser
	
	   - POP Server: pop3.email.msn.com
	
	   - Save password: Type your password in this box, and then select the check
	     box if you do not want to be prompted each time you check your e-mail.
	
	   - SMTP server: smtp.email.msn.com
	
	6. Click OK to close the Edit Account window.
