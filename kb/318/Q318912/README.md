## Q318912: Cannot Connect to Network After Upgrade to Windows NT 4.0 SP6a

	Article: Q318912
	Product(s): Microsoft Windows NT
	Version(s): 4.0 SP6a
	Operating System(s): 
	Keyword(s): kbenv kbhw kbnetwork kbtool
	Last Modified: 17-APR-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows NT Workstation version 4.0 SP6a 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	After you apply Windows NT 4.0 Service Pack 6a (SP6a) to your computer that has
	a 3COM 90x network adapter, your network adapter may lose network connectivity.
	
	CAUSE
	=====
	
	This issue may be caused if you are using an network adapter driver that is not
	completely compatible with Windows NT 4.0 SP6a.
	
	RESOLUTION
	==========
	
	To resolve this issue, contact 3COM to inquire about the availability of an
	updated network adapter driver for your 3COM 90x network adapter. For
	information about how to do so, please view the following 3COM Web site:
	
	  http://www.3com.com

	you find the technical support you need. This contact information is subject to
	change without notice. Microsoft in no way guarantees the accuracy of this
	third-party contact information.
