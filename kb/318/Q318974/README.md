## Q318974: How To Add A Web-based Hotmail E-mail Account To Microsoft Entou

{% raw %}

	Product(s): The Microsoft Network
	Version(s): 
	Operating System(s): 
	Keyword(s):

	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSN Internet Access for Macintosh, version 1.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article describes how to add a Web-based Hotmail e-mail account to
	Microsoft Entourage.
	
	The information in this article applies to:  - Microsoft Entourage
	
	MORE INFORMATION
	================
	
	To add a POP3 MSN e-mail account to Microsoft Entourage
	
	1. Start Microsoft Entourage.
	
	2. Click Tools, and then click Accounts.
	
	3. In the Accounts window, click New.
	
	4. Select Hotmail as the Account Type, and then click OK.
	
	5. On the Edit Account window, configure the account with the following
	  settings:
	   - Account name: Type a name that will help you easily identify the account.
	     Example: My Hotmail e-mail
	
	   - Name: Type a "Display Name". This name will display to readers of your
	     e-mail messages.
	     Example: John Doe
	
	   - E-mail address: Type your @msn.com e-mail address.
	     Example: msnuser@msn.com
	
	   - Account ID: This is your Hotmail member ID. Type all characters before the
	     "@" symbol in your email address.
	     Example: msnuser
	
	   - Save password: Type your Hotmail password in this box, and verify that the
	     check box is selected if you do not want to be prompted each time you
	     check your e-mail.
	
	6. Click OK to close the Edit Account window.
