## Q65171: COMPAQ Keyboard Utilities May Disable Mouse in Windows 3.0

	Article: Q65171
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 12-OCT-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	You experience loss of mouse control when you use the PS/2-style mouse port on a
	COMPAQ computer with Microsoft Windows version 3.0. This occurs if you are using
	the COMPAQ utility KEYBP.COM or KEYBOARD.SYS version 6.01 and 7.00.
	
	There are two possible solutions to this problem:
	
	- Remove KEYBP.COM or KEYBOARD.SYS from the CONFIG.SYS file.
	
	  -or-
	
	- Upgrade your COMPAQ DOS to COMPAQ DOS 3.31 revision G or COMPAQ DOS 4.01
	  revision D or later.
	
	MORE INFORMATION
	================
	
	The KEYBP.COM and KEYBOARD.SYS utilities allow you to use extended keyboard
	features. These utilities do NOT affect mouse functioning in Windows versions
	2.x. These utilities DO affect Logitech Mouse driver version 4.01.
