## Q65989: Flight Sim: Plane Stalls or Nose Dives Unexpectedly

	Article: Q65989
	Product(s): Microsoft Home Games
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 30-JUL-2001
	
	SUMMARY
	=======
	
	When the plane stalls or nose dives unexpectedly, check the plane's
	specifications to make sure that you have not exceeded the plane's limitations.
	To view the aircraft specifications menu, do the following:
	
	1. Press 1 to access the Mode Menu.
	
	2. Press E to access the Aircraft Library Menu.
	
	3. Press 7 to view the Performace Spec Report.
	
	MORE INFORMATION
	================
	
	In the Performance Spec Report, pay close attention to the "Service Ceiling,"
	which is the maximum height that the plane can fly. Also check the "Maximum Rate
	of Climb at Sea Level" and "Stall Speeds" in the report.
