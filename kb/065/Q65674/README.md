## Q65674: Unrecoverable Application Error on COMPAQ LTE Model 20

{% raw %}

	Article: Q65674
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a
	Operating System(s): 
	Keyword(s): 
	Last Modified: 14-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a 
	-------------------------------------------------------------------------------
	
	The COMPAQ LTE Model 20 may get random Unrecoverable Application
	Error (UAE) messages when Word for Windows or Excel are installed to
	operate under the Microsoft Windows version 3.0. This problem appears to be
	resolved by upgrading the BIOS to ROM revision B.3. The problem is most
	prevalent in machines having a BIOS revision of B.1.
	
	For more information, query on the following words:
	
	  ROM and Revisions and COMPAQ and LTE
	
	Additional query words: 3.00 3.0 3.0a 3.00a KBHW
