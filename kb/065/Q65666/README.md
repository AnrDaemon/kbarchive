## Q65666: Emulation Modes for for Xerox 4045 Printer

	Article: Q65666
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 10-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	The Xerox 4045 laser printer is supported under Microsoft Windows by a Xerox
	driver. Contact Xerox at (800) 445 5554; this printer can also be made to
	emulate a Hewlett-Packard (HP) LaserJet+ with the Xerox "HP Personality
	Cartridge" (actually two cartridges, reference order #9R- 88425).
	
	MORE INFORMATION
	================
