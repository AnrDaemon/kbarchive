## Q65457: Print Manager Print Queue File Limit

{% raw %}

	Article: Q65457
	Product(s): Microsoft Windows 95.x Retail Product
	Version(s): WINDOWS:3.0,3.0a,3.1,3.11
	Operating System(s): 
	Keyword(s): 
	Last Modified: 06-DEC-1999
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
	- Microsoft Windows for Workgroups versions 3.1, 3.11 
	-------------------------------------------------------------------------------
	
	
	SUMMARY
	=======
	
	In Microsoft Windows version 3.0 Print Manager, each print queue will spool a
	total of 20 jobs. After 20 files have been sent to the queue, another file
	cannot be printed from an application until one of the spooled files has
	finished being sent to the printer.
	
	The print queue limit for Print Manager under the Windows version 3.1 is 100
	print jobs.
