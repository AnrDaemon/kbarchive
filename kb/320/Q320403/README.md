## Q320403: Building Enterprise Active Directory Services: Notes from the Fi

	Article: Q320403
	Product(s): Microsoft Press
	Version(s): 
	Operating System(s): 
	Keyword(s): kbdocfix kbdocerr
	Last Modified: 27-MAR-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- MSPRESS Building Enterprise Active Directory Services: Notes from the Field ISBN 0-7356-0860-1 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	This article contains comments, corrections, and information about known errors
	relating to the Microsoft Press book Building Enterprise Active Directory
	Services: Notes from the Field, ISBN 0-7356-0860-1.
	
	The following topics are covered:
	
	- Page 319: Incorrect Reference To Resource Kit Tool
	
	MORE INFORMATION
	================
	
	Page 319: Incorrect Reference To Resource Kit Tool
	--------------------------------------------------
	
	On page 319, in the last sentence of the second paragraph from the bottom,
	change:
	
	"DHCP does not support changes to the Domain Suffix Search list, so the team used
	a resource kit tool (SetTCPIPParams.exe by Georg Zanzen) from a login script."
	
	To:
	"DHCP does not support changes to the Domain Suffix Search list, so the team used
	a Microsoft in-house tool (SetTCPIPParams.exe by Georg Zanzen) from a login
	script."
	
	
	Microsoft Press is committed to providing informative and accurate books. All
	comments and corrections listed above are ready for inclusion in future
	printings of this book. If you have a later printing of this book, it may
	already contain most or all of the above corrections.
