## Q320572: How to Locate Your Version of MSN Explorer

	Article: Q320572
	Product(s): The Microsoft Network
	Version(s): 
	Operating System(s): 
	Keyword(s): 
	Last Modified: 29-MAR-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network Version 7.0 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The correct number of email messages may not appear on the MSN Explorer E-mail
	button when you have the E-mail window displayed.
	
	RESOLUTION
	==========
	
	To resolve this issue
	
	1. On the MSN Explorer navigation bar, click More Choices and then click About
	  MSN Explorer.
	  You'll see a box that lists the version number of the MSN Explorer you have
	  installed.
	
	2. On the box, click More Info.
	  A new page opens.
	
	3. Click Update MSN Explorer now.
	  The issue should be resolved.
