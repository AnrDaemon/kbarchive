## Q320316: MSN Explorer Sign In Screen Activates Unexpectedly

	Article: Q320316
	Product(s): The Microsoft Network
	Version(s): 6.0,6.1
	Operating System(s): 
	Keyword(s): 
	Last Modified: 26-MAR-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- The Microsoft Network Version 7.0 
	- The Microsoft Network versions 6.0, 6.1 
	-------------------------------------------------------------------------------
	
	SYMPTOMS
	========
	
	The MSN Explorer sign-in screen continues to activate while while you may be
	doing other tasks that do not require MSN Explorer to connect.
	
	CAUSE
	=====
	
	You have MSN Explorer connection settings set to Always dial my default
	connection.
	
	RESOLUTION
	==========
	
	Change connection settings to Dial whenever a network connection is not
	present.
	
	To change connection settings to Dial whenever a network connection is not
	present
	
	1. Click Start, click Settings, and then click Control Panel.
	  Note: In Microsoft Windows XP click Start, and then click Control Panel.
	
	2. Double-click Internet Options, and then click the Connections tab.
	
	3. Select Dial whenever a network connection is not present, click Apply, and
	  then click OK.
