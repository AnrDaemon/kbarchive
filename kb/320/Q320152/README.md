## Q320152: INFO: Resources for InstallShield Express for VFP

	Article: Q320152
	Product(s): Microsoft FoxPro
	Version(s): 3.0,5.0,6.0,7.0
	Operating System(s): 
	Keyword(s): kbHOWTOmaster
	Last Modified: 12-AUG-2002
	
	-------------------------------------------------------------------------------
	The information in this article applies to:
	
	- Microsoft Visual FoxPro for Windows, versions 3.0, 5.0, 6.0, 7.0 
	-------------------------------------------------------------------------------
	
	SUMMARY
	=======
	
	Visual FoxPro 3.0, Visual FoxPro 5.0 and Visual FoxPro 6.0 use the Setup Wizard
	to create distribution solutions. Visual FoxPro 7.0 is included with
	InstallShield Express - Visual FoxPro Limited Edition to create distribution
	media for Visual FoxPro distributable applications. This article provides links
	to useful information, including online documentation, about how to use
	InstallShield Express - Visual FoxPro Limited Edition version 3.x.
	
	NOTE: Some of these resources offer information about other products that are
	offered by InstallShield.
	
	MORE INFORMATION
	================
	
	Vfpdeploy.doc is included with Visual FoxPro 7.0, and is located in the
	Technical Articles folder on the CD-ROM. This document offers an overview about
	using InstallShield Express with Visual FoxPro 7.0.
	
	Several Web sites provide valuable information about using InstallShield Express.
	The InstallShield Web site offers an Adobe Acrobat (.pdf) file that describes
	using InstallShield Express. To obtain this file, visit the following
	InstallShield Web site:
	
	  InstallShield PDF File
	  (http://www.installshield.com/downloads/isx/productguides/expressgettingstarted.pdf)
	
	Articles:
	
	The Microsoft Developer Network contains a link to an article by Whil Hentzen

	information, visit the following Microsoft Developer Network (MSDN) Web site:
