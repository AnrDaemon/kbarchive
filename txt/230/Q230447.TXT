TITLE   :Zone: ZoneFriends Status Bar Displays a Status of "Logging On.."
PRODUCT :Microsoft Home Games
PROD/VER::
OPER/SYS:
KEYWORDS:kbenv igz kbimu msgame kbFAQ

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Zone.com 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to connect to ZoneFriends, the status bar may continue to
display a status of "Logging On" (an increasing number of periods appear after
the status as time passes) and "Disconnected".

CAUSE
=====

This behavior can occur for any of the following reasons:

 - You attempt to connect to ZoneFriends through a firewall, ICS, or NAT.
 - You are using America Online's (AOL's) Parental Controls feature.
 - You set security settings to high in Internet Explorer.

RESOLUTION
==========

To resolve this issue, use the appropriate method.

Configure the Network
---------------------

Configure your network for use with ZoneFriends.

For additional information about how to configure your network for use with the
ZoneFriends, click the article number below to view the article in the Microsoft
Knowledge Base:

   Q159031 Zone: Network Ports Required to Play

Adjust the Parental Controls Settings
-------------------------------------

To resolve this issue, configure the AOL Parental Controls Web Control component
to allow access to Zone.com. To do this:

1. On the My AOL button, click Parental Controls.

2. Click Set Parental Controls.

3. In the Edit controls for box, click the account you want to change.

4. Click Web Control.

5. Click Access all Web Sites (recommended for adults only).

6. Click Save, and then click OK.

7. Close the Parental Controls windows.

To change security settings Internet Explorer, follow these steps:

Microsoft Internet Explorer 5 or Later

1. On the Tools menu, click Internet Options.

2. On the Security tab, click Internet.

3. Click Default Level, and then click OK.

MORE INFORMATION
================

If the proxy server is not configured properly for use with the Zone, your
computer does not receive the data necessary to establish a connection to the
Zone.
