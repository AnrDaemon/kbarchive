DOCUMENT:Q230211  11-DEC-2001  [winnt]
TITLE   :Tdpipe.sys Symbols Are Missing in Terminal Server Service Pack 4
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0,4.0 SP4
OPER/SYS:
KEYWORDS:kbProgramming

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server versions 4.0, 4.0 SP4, Terminal Server Edition 
-------------------------------------------------------------------------------

SUMMARY
=======

The Service Pack 4 version of the following drivers are missing from the symbols
tree of Microsoft Windows NT Server 4.0, Terminal Edition Service Pack 4:

 - Tdasync.dbg

 - Tdipx.dbg

 - Tdnetb.dbg

 - Tdpipe.dbg

 - Tdspx.dbg

 - Tdtcp.dbg

MORE INFORMATION
================

These symbols will be included in Windows NT Server 4.0, Terminal Server Edition
Service Pack 5.
