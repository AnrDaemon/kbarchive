TITLE   :Flight Simulator: Saving Startup Modes to Disk
PRODUCT :Microsoft Home Games
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
SUMMARY
=======

Below are instructions for saving the startup mode to disk for Flight Simulator
versions 3.00 and 4.00:

Flight Simulator Version 4.00
-----------------------------

1. Select the MODE menu (1) from the menu bar.

2. Select the MODE LIBRARY (option F).

3. Select the mode to be saved as the startup mode (options 1-6).

4. Select "Save Startup mode to disk" (option E).

Flight Simulator Version 3.00
-----------------------------

1. Select the MODE menu (1) from the menu bar.

2. Select the MODE LIBRARY (option F).

3. Select the mode to be saved as the startup mode (options 1-6).

4. Select "Save Startup mode to disk" (option D).

The next time Flight Simulator is started, it will start in the new mode.
