TITLE   :Bitstream Font Versions Compatible with Windows 3.0
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------


SUMMARY
=======

Bitstream Fontware versions 2.00 and later are compatible with Microsoft Windows
version 3.0. Although Bitstream has released a 3.00 version of Fontware for
Windows, Bitstream has confirmed that Fontware versions 2.00 and later are
compatible with Windows 3.0.

MORE INFORMATION
================

For additional information, contact Bitstream.
