DOCUMENT:Q160791  09-AUG-2001  [winnt]
TITLE   :Excel Charts Lose Color When Pasted into Word
PRODUCT :Microsoft Windows NT
PROD/VER:WINDOWS:; winnt:4.0
OPER/SYS:
KEYWORDS:kbdisplay kbgraphic

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 4.0 
 - Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When inserting a Microsoft Excel chart into a Microsoft Word document, the chart
loses its color when the focus is set to the document.

CAUSE
=====

The code in the Cirrus.dll file was updated to correct a problem with clipping.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 4.0. This
problem has been corrected in the latest U.S. Service Pack for Windows NT
version 4.0. For information on obtaining the Service Pack, query on the
following word in the Microsoft Knowledge Base (without the spaces):

   S E R V P A C K


Additional query words: prodnt word excel winnt display chart color
