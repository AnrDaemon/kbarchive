The information in this article applies to:

 - Microsoft Encarta Encyclopedia 97 Deluxe for Windows 
 - Microsoft Encarta 98 Encyclopedia for Macintosh 
 - Microsoft Encarta 98 Encyclopedia for Windows 
-------------------------------------------------------------------------------

SUMMARY
=======

The versions of Encarta Encyclopedia edition listed above include two compact
discs. The first disc is used to install the program. The second disk contains
the special media features included only with the Deluxe edition of Encarta. As
you are using the program, if you click any of the special features, you are
prompted to insert the second disc. You can start Encarta with either the first
or the second disc in your CD-ROM drive.

NOTE: During the Encarta Setup, you are not prompted to insert the second disc.
You will be prompted to insert the second disc when you are running the program
and the item that you wish to retrieve is located on the second disc.
