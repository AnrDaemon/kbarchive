DOCUMENT:Q160386  09-AUG-2001  [winnt]
TITLE   :Incorrect MediaType Parameter on IBM PCMCIA Token Ring Card
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:kbnetworkkbbuglist kbfixlist

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 4.0 
 - Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When installing the IBM PCMCIA Token Ring card in a computer running Windows NT
4.0, the MediaType parameter of the card may be incorrect or not present.

The result is a possible Account Unknown or Account Deleted error when creating a
network-disabled hardware profile.

For additional information, please see the following article in the Microsoft
Knowledge Base:

   ARTICLE-ID: Q156608
   TITLE : Err Msg: Account Unknown--Account Deleted

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 4.0. This
problem was corrected in the latest Microsoft Windows NT 4.0 U.S. Service Pack.
For information on obtaining the service pack, query on the following word in
the Microsoft Knowledge Base (without the spaces):

   S E R V P A C K


Additional query words: prodnt
