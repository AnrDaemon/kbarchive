DOCUMENT:Q160472  07-SEP-2001  [winnt]
TITLE   :Print Screen in Windows NT 4.0 Prints Only First Four Lines
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:kbprint

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 4.0 
 - Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

After you set up properties for an MS-DOS-based program to enable the print
screen function in Windows NT 4.0 and you attempt to print the screen, only the
first 4 lines of the screen are printed.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 4.0. This
problem was corrected in the latest Microsoft Windows NT 4.0 U.S. Service Pack.
For information on obtaining the service pack, query on the following word in
the Microsoft Knowledge Base (without the spaces):

   S E R V P A C K


Additional query words: prodnt ntvdm wow
