======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta Encyclopedia 97 Deluxe for Windows 
 - the operating system: Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

The following article contains a brief summary of the features that are not
available when you run Encarta Encyclopedia on the Windows 3.x operating system.

MORE INFORMATION
================

The Collages and 360 Degree Views features are not available if you run Encarta
Encyclopedia on the Windows 3.x operating system.

If you are having problems accessing the download site for the Yearbook Updates
through Encarta, or you are having problems accessing the Encarta Home page
directly, we recommend that you upgrade your browser to the latest available
version.
