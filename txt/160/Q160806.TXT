DOCUMENT:Q160806  31-JAN-2001  [win95x]
TITLE   :Problems Running Iomega Jaz Tools
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER::2,2.1
OPER/SYS:
KEYWORDS:osr2 win95

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 OEM Service Release, versions 2, 2.1 
-------------------------------------------------------------------------------

SYMPTOMS
========

When Iomega Tools for Windows 95 is installed on your computer, your computer
may stop responding (hang), or you may receive an error message referencing the
Iomega.vxd file, when you attempt to use the Iomega Jaz tools.

CAUSE
=====

You may be using a version of Iomega Jaz tools earlier than Iomega Tools for
Windows 95 version 5.0. Earlier versions are not compatible with Windows 95 OEM
Service Release 2 (OSR2).

RESOLUTION
==========

 - Uninstall the Iomega Tools for Windows 95.

 - Contact Iomega to inquire about obtaining version 5.0 or later of Iomega
   Tools. Version 5.0 of Iomega Tools is FAT32 File System-aware.
