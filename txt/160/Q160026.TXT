TITLE   :Double-Clicking MSN Icon Displays AT&T Worldnet Sign In Screen
PRODUCT :The Microsoft Network
PROD/VER:WINDOWS:1.2,1.3,2.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - The Microsoft Network versions 1.2, 1.3, 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you install AT&T Worldnet software, the AT&T Worldnet sign in
screen may appear when you double-click the icon for MSN, The Microsoft Network.

CAUSE
=====

The AT&T Worldnet software changes the default connection for Microsoft
Internet Explorer to the AT&T Worldnet service.

RESOLUTION
==========

To change the default connection back to MSN, follow these steps:

1. Use the right mouse button to click the MSN icon, and then click Properties
   on the menu that appears.

2. Click the Connection tab.

3. In the "Use the following Dial-Up Networking connection" box, click The
   Microsoft Network.

4. Click OK.

Additional query words: signin
