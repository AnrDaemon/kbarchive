DOCUMENT:Q259564  11-JAN-2001  [vbwin]
TITLE   :INFO: Windowless Controls in the Mswless.ocx File Not Supported
PRODUCT :Microsoft Visual Basic for Windows
PROD/VER:WINDOWS:6.0
OPER/SYS:
KEYWORDS:kbCtrl kbVBp600 kbGrpDSVB kbDSupport

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Visual Basic Professional Edition for Windows, version 6.0 
 - Microsoft Visual Basic Enterprise Edition for Windows, version 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Microsoft Windowless or Lightweight controls in the ActiveX control file
Mswless.ocx that shipped in Visual Basic 6.0 (in the Common\Tools\VB\winless
folder on the CD) are not installed by the Setup for the product.

They are provided as is and are not supported by Microsoft Product Support
Services (PSS). Using these controls is at your own risk.
