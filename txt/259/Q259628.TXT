TITLE   :XADM: Restoring Exchange Databases from Alpha to Intel Servers
PRODUCT :Microsoft Exchange
PROD/VER:winnt:5.5 SP3
OPER/SYS:
KEYWORDS:kbenv exc55

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.5 SP3 
-------------------------------------------------------------------------------

SUMMARY
=======

You can move existing Exchange Server database(s) from an Alpha-based platform
to an Intel-based platform. This is most easily accomplished by performing a
full disaster recovery of the Exchange Server databases.

MORE INFORMATION
================

To perform a full disaster recovery of the Exchange Server databases, refer to
the following articles in the Microsoft Knowledge Base:

   Q155216 XADM: How to Move Exchange Server to a New Computer with the Same
   Name

   Q185078 Recommendations for Successful Disaster Recovery


Additional query words: XMRP
