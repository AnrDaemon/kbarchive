TITLE   :Allegiance: Ship Stops Responding to Keyboard Controls
PRODUCT :Microsoft Home Games
PROD/VER::1.0
OPER/SYS:
KEYWORDS:kbui igz kbimu msgame

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Allegiance, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use the keyboard to pilot a ship in Microsoft Allegiance, the ship may
stop responding to the controls.

CAUSE
=====

This behavior can occur if you press more than one key at a time.

RESOLUTION
==========

To resolve this issue, press the ESC key.

STATUS
======

Microsoft has confirmed that this is a problem in Microsoft Allegiance, version
1.0.

Additional query words: 1.00 msgame igz msngz locked lost control autopilot
