TITLE   :Error Messages When Canon LBP-4 Set to Emulate Diablo
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

According to Canon technical support, the Canon LBP-4 laser printer may not
print reliably in Microsoft Windows when set to emulate a Diablo printer.

By default, the Canon LBP-4 is set to emulate a Diablo printer. If you are using
the Windows Canon LBP-4 printer driver and the printer is emulating the Diablo,
the following error may be displayed on the printers LCD screen while printing:

   20 Page Full

RESOLUTION
==========

According to Canon technical support, the Canon LBP-4 must be in ISO mode to be
compatible with the Windows printer driver. They provide the following steps to
set the printer to ISO emulation mode.

Steps to Set the Canon LBP-4 to ISO Mode
----------------------------------------

Perform these steps on the printers control panel:

1. Take the printer offline.

2. Press the MENU key until the display reads "COMMAND INITIAL."

3. Press ENTER twice.

4. Press MENU until the display reads "ISO."

5. Press ENTER.

6. Press ERROR SKIP.

7. Press MENU until the display reads "SAVE NVRAM LOAD."

8. Press ENTER.

9. Press ERROR SKIP.

10. Hold down ENTER until the display reads "03 RESET."

MORE INFORMATION
================

For more information on the above steps, or the Canon LBP-4 printer, please
contact Canon technical support.
