TITLE   :Sound Finder Converts All Files in Convert Dialog Box
PRODUCT :Miscellaneous Windows Products
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows Sound System, versions 1.0, 1.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

Windows Sound System Sound Finder may convert all files in the Convert dialog
box when only one file has been selected. This problem does not occur on all
machines.

STATUS
======

Microsoft has confirmed this to be a problem in Windows Sound System Sound
Finder versions 1.0 and 1.0a. This problem was corrected in Windows Sound System
version 2.0.

MORE INFORMATION
================

To ensure that all the files in the Convert dialog box are not converted, select
one file at a time in the main file list when you are converting. This way, only
that one file appears in the Convert dialog box. Select this file to convert.
