TITLE   :Group File Error After Launching WordPerfect
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

The following error message may appear after you start version 1.0 of
WordPerfect for Windows:

   Group File Error: Insufficient memory to perform this operation. Quit one or
   more Windows applications and try again

The error also occurs if you attempt to move icons or open Cardfile.

RESOLUTION
==========

WordPerfect Corporation has confirmed this to be a problem and has made
available a WordPerfect maintenance release from WordPerfect technical support.

MORE INFORMATION
================

For more information, contact WordPerfect Corporation.
