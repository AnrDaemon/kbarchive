TITLE   :14 Default TrueType Fonts in Windows 3.1

OPER/SYS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 3.1 
-------------------------------------------------------------------------------

SUMMARY
=======

Listed below are the 14 TrueType fonts included with Microsoft Windows.

File listing from the MS-DOS command prompt:

   ARIAL    TTF     65692 03-10-92   3:10a
   ARIALBD  TTF     66080 03-10-92   3:10a
   ARIALBI  TTF     71880 03-10-92   3:10a
   ARIALI   TTF     61656 03-10-92   3:10a
   COUR     TTF     72356 03-10-92   3:10a
   COURBD   TTF     78564 03-10-92   3:10a
   COURBI   TTF     84436 03-10-92   3:10a
   COURI    TTF     80588 03-10-92   3:10a
   TIMES    TTF     83260 03-10-92   3:10a
   TIMESBD  TTF     79804 03-10-92   3:10a
   TIMESBI  TTF     76452 03-10-92   3:10a
   TIMESI   TTF     78172 03-10-92   3:10a
   SYMBOL   TTF     64516 03-10-92   3:10a
   WINGDING TTF     71052 03-10-92   3:10a

   Fonts                     Font Types
   -----                     ----------

   Arial                     Regular, Bold, Italic, Bold Italic
   Courier New               Regular, Bold, Italic, Bold Italic
   Times New Roman           Regular, Bold, Italic, Bold Italic
   Symbol
   Wingdings

For more information, query on the following words in the Microsoft Knowledge
Base:

   44 truetype offered

Additional query words: 3.10 dir .ttf list lists listing
