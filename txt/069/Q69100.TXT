TITLE   :Windows Err Msg: PS Header Has Not Been Downloaded...
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a,3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

You may receive the following printed error message when trying to print from
Windows to a PostScript printer:

   Windows PS header has not been downloaded.
   Please download header.

RESOLUTION
==========

Windows 3.1 or 3.11
-------------------

To resolve this problem in Windows 3.1 or 3.11, do the following:

1. From the Control Panel, choose Printers.

2. Verify the PostScript Printer is the selected active printer.

3. Select Setup.

4. Select Options.

5. Select Send Header with each job.

6. Choose OK, and return to the Control Panel.

Windows 3.0 or 3.0a
-------------------

To resolve this problem in Windows 3.0 or 3.0a, do the following:

1. From the Control Panel, choose Printers.

2. Verify the PostScript Printer is the selected active printer.

3. Select Configure.

4. Select Setup.

5. Select the correct PostScript printer.

6. Select Options.

7. From the header section, select Download Each Job.

8. Choose OK, and return to the Control Panel.

The header is now downloaded and the message should not appear.
