TITLE   :Mac Srv: Err Msg: ...Mail Server Could Not Be Loaded...
PRODUCT :Microsoft Mail For Appletalk Networks

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for AppleTalk Networks, versions 3.0, 3.1a, 3.1c, 3.1d 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you restart the Macintosh with a locked file called Microsoft Mail Data in
the root directory of the disk containing the Microsoft Mail Server, and the
Microsoft Mail Data file is not in the System Folder, or if the Microsoft Mail
Data file in the System Folder is locked, you will get the following error
message:

   The Microsoft Mail Server could not be loaded because
   an unexpected error occurred.

RESOLUTION
==========

To unlock the Microsoft Mail Data file, do the following:

1. Select the Microsoft Mail Data file.

2. From the File menu, choose Get Info.

3. Clear the Locked check box.

Additional query words: 3.00 3.10a 3.10c 3.10d
