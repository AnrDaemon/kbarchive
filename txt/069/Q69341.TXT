DOCUMENT:Q69341  06-DEC-1999  [win95x]
TITLE   :Bottom Part of Screen Blanks Out on Sharp Portable PC 8000
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run Microsoft Windows version 3.0 on a Sharp Portable 386 with an LCD
display, and you switch or exit from an MS-DOS application back to Windows, the
bottom part of the screen turns black. This problem only occurs when you run
Windows in enhanced mode.

CAUSE
=====

This problem is caused by the video card and the way the video memory is mapped.
This problem does not occur if you use an external monitor or you run Windows in
real or standard mode.

STATUS
======

There is no workaround. Sharp is aware of the problem and Microsoft will post
new information here as it becomes available.

MORE INFORMATION
================

The Sharp Portable is a 386 20 MHz computer with a 14-inch LCD display. The
screen is an illuminated double supertwist 640 x 480 VGA color display. The
video card is manufactured by Yamaha and uses an Award VGA BIOS.
