TITLE   :XADM: Cannot Create an Exchange Cluster Resource
PRODUCT :Microsoft Exchange
PROD/VER::
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange 2000 Server 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you use Microsoft Exchange 2000 Server to create an Exchange cluster
resource, you are unable to create the resource if the resource type name is
different from the name of the Physical Disk resource. Third-party storage
devices are blocked and cannot be deployed with the Exchange 2000 cluster.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange 2000 Server.


Additional query words: cs cluster services third party clustering
