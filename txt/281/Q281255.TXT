TITLE   :MechWarrior 4: Err: STOP: Invalid Allocation, 0 Bytes with...
PRODUCT :Microsoft Home Games
PROD/VER::
OPER/SYS:
KEYWORDS:kberrmsg

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft MechWarrior 4: Vengeance 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start or play Microsoft MechWarrior 4: Vengeance, you may receive the
following error message:

   STOP:Invalid Allocation, 0 bytes

CAUSE
=====

This behavior can occur if your joystick has 32 buttons, or if you have one of
the following input devices connected to your computer:

 - StickWorks F22 Digital Pro controller

 - Wacom Intuos graphics tablet

RESOLUTION
==========

If you have StickWorks F22 Digital Pro controller connected, obtain the latest
update from the joystick manufacturer. For more information about StickWorks
Digital Upgrades, visit the following StickWorks Web site:

   http://www.stickworks.com

If you are using Wacom Intuos graphics tablet, you must remove the device and
uninstall the software that comes with it. Please see the removal instructions
that are included with your hardware documentation.
