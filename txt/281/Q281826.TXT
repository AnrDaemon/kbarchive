TITLE   :MapPoint: How to Submit Map Feedback

PROD/VER::1.0
OPER/SYS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft MapPoint 2002 
 - Microsoft MapPoint 2001 
 - Microsoft MapPoint 2000 
 - Microsoft Streets & Trips 2002, version 1.0 
 - Microsoft Streets and Trips 2001 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to submit feedback to Microsoft about the map data in
the programs listed at the beginning of this article.

MORE INFORMATION
================

If you find any errors in the map or map data that is included with the programs
listed at the beginning of this article, use the following steps to submit
feedback to Microsoft:

1. On the File menu, click Tools.

2. Click Send Map Feedback.

3. Follow the instructions in the Feedback Wizard to submit the data to
   Microsoft.

If you have a suggestion for a feature that you would like to see in future
editions of MapPoint, or if you have a comment about the current version, please
send them to

   Microsoft MapPoint Program Manager
   Microsoft Corporation
   One Microsoft Way
   Redmond, WA 98052-6399

or visit the following Microsoft Web site:

   http://www.microsoft.com/mswish

If you have a suggestion for a feature that you would like to see in future
editions of Streets and Trips, or if you have a comment about the current
version, please send them to

   Microsoft Streets & Trips Program Manager
   Microsoft Corporation
   One Microsoft Way
   Redmond, WA 98052-6399

or visit the following Microsoft Web site:

   http://www.microsoft.com/mswish
