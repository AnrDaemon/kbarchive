TITLE   :RAS Support for Security Dynamics ACE/Server
PRODUCT :Microsoft Windows NT
PROD/VER:3.51 4.0
OPER/SYS:
KEYWORDS:kb3rdparty

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation versions 3.51, 4.0 
 - Microsoft Windows NT Server versions 3.51, 4.0 
-------------------------------------------------------------------------------

Security Dynamics ACE/Server provides remote access security for Windows
NT environments. In order to access the ACE/Server, you must install the
ACE/Client for Windows NT Remote Access Services (RAS), which requires
Windows NT version 3.51. The ACE/Client for Windows NT RAS handles dial-in
authentication based on credentials identified on the ACE/Server. For more
information, contact Security Dynamics, Inc., at (617) 547-7820.

The third-party product discussed here is manufactured by Security
Dynamics, Inc., a vendor independent of Microsoft; we make no warranty,
implied or otherwise, regarding this product's performance or reliability.
