TITLE   :IBM Thinkpad Supports Only 640x480 16-Color Video Resolution
PRODUCT :Microsoft Windows NT
PROD/VER:3.10 3.50
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 3.1 
 - Microsoft Windows NT Workstation version 3.1 
 - Microsoft Windows NT Advanced Server, version 3.1 
 - Microsoft Windows NT Workstation version 3.5 
 - Microsoft Windows NT Server version 3.5 
-------------------------------------------------------------------------------

The IBM Thinkpad series computer supports only 640x480 resolution with 16
colors in Windows NT. The BIOS of the IBM Thinkpad series computer emulates
the Western Digital WD90c24 video chipset. Due to limited BIOS space, only
the 640x480 resolution with 16 colors is supported.
