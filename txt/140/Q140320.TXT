-------------------------------------------------------------------------------

SUMMARY
=======

This article demonstrates how to view the latest version numbers of all the
files in a particular project by using the graphical user interface (GUI).

MORE INFORMATION
================

1. Select the project.

2. Perform a history of the project.

3. Select the most recent history item (of the project). This works only for
   items directly in project, not in subprojects.

4. Click View.
