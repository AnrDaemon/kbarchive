TITLE   :Xircom IIps Credit Card Adapter Fails in Windows NT 3.51
PRODUCT :Microsoft Windows NT
PROD/VER:3.51
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.51 
 - Microsoft Windows NT Server version 3.51 
-------------------------------------------------------------------------------

SUMMARY
=======

Installing the Xircom IIps Credit card adapter using the file provided in the
DRVLIB on the 3.51 Server/Workstation CD-ROM may not work with newer Xircom IIps
cards.

MORE INFORMATION
================

The driver in the DRVLIB\NETCARD\X86\XCE10 is version 1.17. Xircom has updated
this driver to version 2.25 which can be found on CompuServe, Xircom's BBS, or
their FTP Site. This driver supports Windows for Workgroups, and Windows NT
versions 3.1, 3.5, and 3.51.

The file name to download is CE2DRV.EXE with a date of 4/8/96.

File location:

   CompuServe - GO Xircom
   BBS - 805-376-9130
   FTP Site - FTP.Xircom.com

Xircom Technical Support:

   United States............. TEL: 805-376-9200

The IIps credit card adapter is manufactured by Xircom, a vendor independent of
Microsoft; we make no warranty, implied or otherwise, regarding this product's
performance or reliability.
