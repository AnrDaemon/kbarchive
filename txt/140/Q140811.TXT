TITLE   :User Interface Takes 1 Minute to Display Job Details
PRODUCT :Microsoft Systems Management Server
PROD/VER:winnt:1.1
OPER/SYS:
KEYWORDS:kbnetwork kbui kbsmsAdmin smsadmin smsui

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Systems Management Server version 1.1 
-------------------------------------------------------------------------------


SYMPTOMS
========

A delay occurs every time you access the Systems Management Server details
screen. This symptom even occurs for active jobs where the values cannot be
changed.

STATUS
======

Microsoft has confirmed this to be a problem in Systems Management Server
version 1.1. This problem has been corrected in Systems Management Server
version 1.2.
