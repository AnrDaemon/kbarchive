======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for AppleTalk Networks, version 3.1x 
-------------------------------------------------------------------------------

SUMMARY
=======

You can determine when your Macintosh Mail server last communicated with another
Macintosh Mail server by reading the Last Seen field on the server report.

MORE INFORMATION
================

To generate a server report, run the Mail Network Administrator utility, and
from the Global menu, choose Server Report. The report will show the following
information:

Postoffice Name
---------------

   Serial number :(The mail server's serial number.)
   Server version :(Mail for Macintosh Networks version number.)
   Site name :(Site name; Local is default.)
   Last seen :(Date\time of most recent server to server communication.)
   Zone name :(AppleTalk zone name.)
