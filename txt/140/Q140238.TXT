DOCUMENT:Q140238  09-MAR-2000  [mastering]
TITLE   :PRB: Can't Run Mastering Visual FoxPro with ATI Video Drivers
PRODUCT :Microsoft Mastering Series
PROD/VER::1.0
OPER/SYS:
KEYWORDS:kb3rdparty kbmm

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - MSPRESS Mastering Microsoft Visual FoxPro 3.0, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

Mastering Visual FoxPro demos and video clips exhibit erratic behavior, or demos
or expert point of view video clips do not run.

CAUSE
=====

Mastering Visual FoxPro is incompatible with some features of video cards such
as the Mach32 chip set running the ATI drivers.

RESOLUTION
==========

For Mastering Visual FoxPro to run on the ATI driver, set the 256 color palette
control in the Advanced features of the ATI Control Panel to ON. If you still
encounter some problems, try installing the Microsoft Super VGA driver.

STATUS
======

Microsoft is researching this problem and will post new information here in the
Microsoft Knowledge Base as it becomes available.

REFERENCES
==========

See the documentation for your video card for more information on the ATI
driver. See your Windows documentation for more information on the Microsoft
Super VGA driver.
