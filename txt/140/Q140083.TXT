TITLE   :Compuserve Service Pack 2 Documentation Error
PRODUCT :Microsoft Windows NT
PROD/VER::3.51 SP2
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 3.51 SP2 
-------------------------------------------------------------------------------


The Readme First messages for Windows NT 3.51 U.S. Service Pack 2 Symbol
files for the Intel, MIPS, and PPC computer platforms located on the
Compuserve WINNTDL forum incorrectly state the following:

   These are the updated symbols for Service Pack 1 for Windows NT 3.51.

However, the Readme First messages should state:

   These are the updated symbols for Service Pack 2 for Windows NT 3.51.

Steps to Reproduce Problem
--------------------------

1. On Compuserve go to the WINNTDL forum.

2. Choose U.S. Service Pack 2 for Windows NT 3.51.

3. Choose the Service Pack 2 Symbol Files.

4. Choose the Intel, MIPS, or PPC computer platform.

5. Choose the Readme First file.
