TITLE   :Installing Windows NT 3.51 Service Pack 2 on Compaq Computers
PRODUCT :Microsoft Windows NT
PROD/VER:3.51
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.51 
 - Microsoft Windows NT Server version 3.51 
-------------------------------------------------------------------------------

After you install a Windows NT 3.51 Service Pack, you must choose Restart
Computer to start the updated version of Windows NT 3.51. If you have a
Compaq computer with Compaq Support Software Diskette (SSD) version 1.14 or
greater, the following steps are recommended:

1. Before you choose Restart Computer, press ALT+TAB to Program Manager.

2. Run the Compaq SSD Setup program again. Make sure you look at NTREADME.HLP on
   the Compaq SSD CD-ROM disc.

3. Press ALT+TAB to Windows NT Setup (Service Pack Setup) and choose Restart
   Computer.

The SSD is manufactured by Compaq, a vendor independent of Microsoft; we
make no warranty, implied or otherwise, regarding this product's
performance or reliability.
