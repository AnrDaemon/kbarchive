DOCUMENT:Q114577  08-AUG-2001  [winnt]
TITLE   :Watermark Software HP 1300T Removable Disk Stuck or Corrupted
PRODUCT :Microsoft Windows NT
PROD/VER:3.5
OPER/SYS:
KEYWORDS:kbhw kbHardware

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.5 
 - Microsoft Windows NT Server version 3.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

Your Watermark Software - HP 1300T removable SCSI hard disk becomes stuck or
files on the disk are corrupted when you eject the disk shortly (20 seconds or
less) after Windows NT has written to it.

There is no problem when reading from the disk and then ejecting it.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT Workstation and
Windows NT Server version 3.5. This problem was corrected in the latest U.S.
Service Pack for Windows NT version 3.5. For information on obtaining the
Service Pack, query on the following word in the Microsoft Knowledge Base
(without the spaces):

   S E R V P A C K

The HP 1300T is manufactured by Watermark Software, Inc., a vendor independent of
Microsoft; we make no warranty, implied or otherwise, regarding this product's
performance or reliability.
