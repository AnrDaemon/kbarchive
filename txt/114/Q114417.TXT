TITLE   :Net Help Start <Service> Works only for Windows NT 3.1 Services
PRODUCT :Microsoft Windows NT
PROD/VER:3.5 4.0
OPER/SYS:
KEYWORDS:kbnetwork

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation versions 3.5, 3.51, 4.0 
 - Microsoft Windows NT Server versions 3.5, 3.51, 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Typing "Net Help Start <service>" from the command line in Windows NT 3.5
provides help only for Windows NT 3.1 services and for Windows NT 3.5 services
that were part of Windows NT 3.1. The command line help refers you to NTCMDS.HLP
for network service information.

MORE INFORMATION
================

Net Help Services lists all services available under Windows NT 3.5.
