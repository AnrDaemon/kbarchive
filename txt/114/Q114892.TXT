DOCUMENT:Q114892  08-AUG-2001  [winnt]
TITLE   :Err Msg: "Out of Memory" Browsing NetWare Svr as Administrator
PRODUCT :Microsoft Windows NT
PROD/VER:3.5
OPER/SYS:
KEYWORDS:kbtool

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.5 
 - Microsoft Windows NT Server version 3.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

File Manager reports an "Out of Memory" or "Not Enough Storage" error when you
browse NetWare server network files after logging on to a Windows NT Workstation
and Windows NT Server version 3.5 as Administrator.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT Workstation and
Windows NT Server version 3.5. This problem was corrected in the latest U.S.
Service Pack for Windows NT version 3.5. For information on obtaining the
Service Pack, query on the following word in the Microsoft Knowledge Base
(without the spaces):

   S E R V P A C K

NetWare is manufactured by Novell, Inc., a vendor independent of Microsoft; we
make no warranty, implied or otherwise, regarding this product's performance or
reliability.
