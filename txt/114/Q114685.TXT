TITLE   :STOP - The Parameter is Incorrect
PRODUCT :Microsoft Windows NT
PROD/VER:3.1
OPER/SYS:
KEYWORDS:kbother

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - 
    - Microsoft Windows NT Server version 3.1 
    - Microsoft Windows NT Workstation version 3.1 
 - Microsoft Windows NT Advanced Server, version 3.1 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use Server Manager to examine or modify shares on a LAN Manager server
when the LAN Manager server is sharing a large number of resources, the
following error message appears:

   STOP - The parameter is incorrect.

Server Manager should be able to handle up to the maximum number of shares (500
in LAN Manager 2.2).

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT and Windows NT
Advanced Server version 3.1. This problem has been corrected in the latest U.S.
Service Pack for Windows NT and Windows NT Advanced Server version 3.1. For
information on obtaining the Service Pack, query on the following word in the
Microsoft Knowledge Base (without the spaces):

   S E R V P A C K

Additional query words: prodnt
