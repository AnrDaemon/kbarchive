TITLE   :PC Adm: Using Import to Modify Home Postoffice Name
PRODUCT :Microsoft Mail For PC Networks
PROD/VER:WINDOWS:3.2
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for PC Networks, version 3.2 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use version 3.2 of the Microsoft Mail for PC Networks Import program
(IMPORT.EXE) to modify the home postoffice mailbox name, one of the following
error messages may be displayed:

   Notice 34 - Could not access log information file

   -or-

   Mail could not connect to Mail Server

RESOLUTION
==========

To correct this problem, run the Mail Administrator program (ADMIN.EXE) and
update the account information by selecting Local-Admin, Modify.

STATUS
======

Microsoft has confirmed this to be a problem in version 3.2 of Microsoft Mail
for PC Networks. We are researching this problem and will post new information
here in the Microsoft Knowledge Base as it becomes available.
