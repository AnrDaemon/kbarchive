TITLE   :MS Reader: Message Prompting You to Uninstall Earlier Version
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER::1.5
OPER/SYS:
KEYWORDS:kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Reader, version 1.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When installing Microsoft Reader version 1.5, you may see a message asking you
to uninstall an earlier version of Microsoft Reader.

CAUSE
=====

This behavior can occur if Microsoft Reader did not uninstall successfully or
corruption occurred during uninstallation.

RESOLUTION
==========

To resolve this problem, download the following fix. After you download the
file, double-click it. This program removes a registry key that causes this
problem. After the registry key is removed, you can reinstall Microsoft Reader.

The following file is available for download from the Microsoft Download Center:

   Downloadhttp://download.microsoft.com/download/msreader/Patch/1/W98/EN-US/msread.exe
   (http://download.microsoft.com/download/msreader/Patch/1/W98/EN-US/msread.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

   Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.
