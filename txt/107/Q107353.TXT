TITLE   :Real-Time Pass Through Not Possible with the WSS Card
PRODUCT :Miscellaneous Windows Products
PROD/VER:1.00 1.00a 2.00
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows Sound System, versions 1.0, 1.0a, 2.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Real-time pass through or real-time playback is not possible with the Windows
Sound System card and microphone.

MORE INFORMATION
================

It is not possible to plug the microphone into the Windows Sound System card,
talk into the microphone, and receive amplification and output to the speakers
at the same time. This is not a feature of Windows Sound System, and there is no
supported way of doing this.

Products such as "PC Karaoke" will not work with the Windows Sound System card.
