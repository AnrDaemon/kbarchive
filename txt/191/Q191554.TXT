TITLE   :Err Msg: UA Caused an Invalid Page Fault in Module Unknown
PRODUCT :Microsoft Home Games
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kb3rdparty kbdisplay kberrmsg kburl kbimu msgamekbfaq

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Urban Assault, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you press ESC to skip through the introductory movie when you start Microsoft
Urban Assault, you may receive the following error message:

   UA caused an invalid page fault in module Unknown.

CAUSE
=====

This behavior can occur if a Hercules Dynamite 128 Series display adapter is
installed in your computer.

RESOLUTION
==========

To resolve this issue, download and install the latest Tseng Labs ET6000 Windows
95 video driver from the following Web site:

   http://www.tseng.com

NOTE: The file name is W95et6.zip.
