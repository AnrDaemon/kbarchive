SUMMARY
=======

This articles explains how you can display Web content in the Component
Gallery.

The Component Gallery provides a flexible architecture for organizing and
displaying a wide range of components. These can include various Visual FoxPro
components such as classes, forms, and programs. In addition, the Component
Gallery also has the ability to display your favorite Web sites.

MORE INFORMATION
================

The following steps outline the process of adding Web sites to the Component
Gallery.

1. Start the Component Gallery.

2. Open the Component Gallery Options dialog box, click the Standard tab and
   make sure that Advanced editing enabled is selected.

3. Create a new folder called Web Favorites.

4. Create another new folder under the Web Favorites folder called "MS VFP Web
   Site".

5. Right click the "MS VFP Web Site" folder and choose properties.

6. Select the Node tab.

7. Enter the following URL into the Dynamic folder text box:

   http://www.microsoft.com/vfoxpro/

When you select the MS VFP Web Site folder, the Visual FoxPro Web site displays
in the right-hand pane of the Component Gallery. You can repeat steps 4-7 above
to add as many Web sites as you like.
