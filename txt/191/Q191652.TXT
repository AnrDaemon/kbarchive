TITLE   :MCM: Text Missing from Default Button and Random Button
PRODUCT :Microsoft Home Games
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kbui kbimu msgamekbfaq

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Motocross Madness, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you play a single-player event in Microsoft Motocross Madness and select
the Baja track, the text describing the function of the Default and Random
buttons on the Event Options screen is missing.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Motocross Madness,
version 1.0.

MORE INFORMATION
================

The Default and Random buttons determine the layout and number of checkpoint
gates on the Baja track.

NOTE: You can select the number of checkpoint gates only if you click the Random
button.
