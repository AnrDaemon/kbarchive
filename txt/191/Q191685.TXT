 - Microsoft Visual FoxPro for Windows, version 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Vfp6int.exe contains the Visual FoxPro 6.0 ODBC Driver resource file FoxPro.int.
This file does not ship with the latest version of the ODBC driver, but it is
required by the driver. This file should be copied to your Windows\System or
WinNT\System32 directory.

MORE INFORMATION
================

The following file is available for download from the Microsoft Download
Center:

   Vfp6int.exe
   (http://download.microsoft.com/download/vfox60/sample10/1/W9X/EN-US/Vfp6int.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

   Q119591 How to Obtain Microsoft Support Files from Online Services

posting to scan this file for viruses. Once posted, the file is housed on secure
