DOCUMENT:Q173460  14-NOV-2000  [word97]
TITLE   :WD97: Word Hangs If Assistant Options Change When Editing Object
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:word97

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you open and close either the Microsoft Office Assistant options or the
Choose Assistant dialog boxes while saving an object, the Office Assistant
disappears and Microsoft Word stops responding.

This problem may occur when you edit and then close any of the following
objects:

 - Bitmap Image
 - Media Clip
 - Microsoft Equation
 - Microsoft Graph 97 Chart
 - Microsoft Excel Worksheet
 - Microsoft Excel Chart
 - PowerPoint Presentation
 - Microsoft Map
 - Paintbrush Picture
 - Video Clip
 - WordPad Document

NOTE: To insert these objects, use the Object command on the Insert menu.

RESOLUTION
==========

To work around this problem, press ESC to continue working on the document.

If you want to change the options of the Office Assistant, first save the
document and then proceed with the changes to the Office Assistant.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.

This problem has been corrected in Word 2000.

MORE INFORMATION
================

For additional information about how to change the Office Assistant, please see
the following article in the Microsoft Knowledge Base:

   Q162763 How to Modify or Remove Microsoft Office Assistant Help
