TITLE   :How to Sign Up for an MSN Account

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - The Microsoft Network versions 2.5, 2.51, 2.52, 2.6 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to sign up for an MSN, The Microsoft Network, account
if you are not prompted to do so while you are installing MSN.

MORE INFORMATION
================

The installation program for MSN will normally take you through the steps
necessary for setting up an MSN account, if it should fail then follow the
procedures below.

To sign up for an MSN account, use the following steps:

Windows 95
----------

1. Click Start, and then click Run.

2. In the Open box, type inetwiz, and then click OK.

3. When the the Internet Connection Wizard screen appears, select either the
   Connect using my phone line or Connect using my local area network (LAN) as
   appropiate, and then click Next.

4. Follow the instructions on the screen.

Windows 98
----------

1. Click Start, point to Programs, point to Online Services, and then click The
   Microsoft Network.

2. Follow the instructions on the screen to set up The Microsoft Network.

3. If this fails follow the Windows 95 instructions.

You can also sign up for an MSN account over the telephone by calling MSN Member
Services at (800) 386-5550.
