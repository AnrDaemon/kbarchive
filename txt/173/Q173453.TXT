DOCUMENT:Q173453  07-JAN-2000  [exchange]
TITLE   :XADM: Preview Beta Expires in 120 Days After Installation
PRODUCT :Microsoft Exchange
PROD/VER:winnt:5.5
OPER/SYS:
KEYWORDS:kbpolicy

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SUMMARY
=======

There is a 120 day time bomb included in the Microsoft Exchange Server, version
5.5 Preview Beta Code. 120 days after installation, the Exchange Server will
stop functioning.

MORE INFORMATION
================

This preview release is not intended to be used in production and has been
modified so that it cannot be installed into a Microsoft Exchange Site with
other Exchange Servers. This preview release is non-upgradeable to the Release
Candidate (RC) version and the Released To Manufacturing (RTM) version. It is
also not possible to upgrade from this release to any future release.
