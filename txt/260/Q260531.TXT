======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for PC Networks, version 3.5 
 - Microsoft Mail Remote for Windows, version 3.2 
 - the operating system: Microsoft Windows 98 
-------------------------------------------------------------------------------

SYMPTOMS
========

You may receive the following error message when you attempt to open Microsoft
Excel, Microsoft PowerPoint, or Microsoft Word e-mail attachments:

   MSMAIL caused a general protection fault in module VFORMS.DLL at
   000c:00000d49.

This issue occurs when you install Microsoft Office 2000 on a computer running
Microsoft Windows 98 and the Microsoft Mail for Windows client version 3.5 or
the Microsoft Mail Remote for Windows client version 3.2.

NOTE: In order to restore the workstation to its original state prior to the
fault, it may be necessary to shut down and restart the computer.

WORKAROUND
==========

To work around this issue, upgrade to Microsoft Outlook 2000.




STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Mail for PC Networks
version 3.5.
