TITLE   :SMS: Collection Properties Comment Box Allows 130 Characters
PRODUCT :Microsoft Systems Management Server
PROD/VER:winnt:2.0
OPER/SYS:
KEYWORDS:kbsms200 kbsms200bug

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Systems Management Server version 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Comment box on the General tab in the Collection Properties dialog box does
not activate the scroll bar after you fill the comment box. The comment box
currently stop accepting additional input after 130 characters.

WORKAROUND
==========

To work around this issue, design your comment to contain no more than 130
characters.

STATUS
======

Microsoft has confirmed this to be a problem in Systems Management Server
version 2.0.
