DOCUMENT:Q185264  14-NOV-2000  [word97]
TITLE   :WD97: Crop Button on Picture Toolbar Unavailable in a Template
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:kbdta kbtemplate word97

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

The Crop button on the Picture toolbar is not available (appears dimmed).

CAUSE
=====

This problem occurs if the current document is a template.

WORKAROUND
==========

To crop the picture, use the Crop From commands in the Format Picture dialog
box. To do this, follow these steps:

1. Click the picture to select it.

2. On the Format menu, click Picture.

3. On the Picture tab, crop the picture using the Crop From settings. Click OK.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article.

This problem has been corrected in Word 2000.

MORE INFORMATION
================

This problem does not occur when the picture is located in a document rather
than a template.
