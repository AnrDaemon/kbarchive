TITLE   :XCLN: Displaying This Message Correctly Requires a Custom E- For
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:4.0,5.0; :8.0,8.01,8.02,8.03
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Windows 95/98 client, versions 4.0, 5.0 
 - Microsoft Exchange Windows NT client, versions 4.0, 5.0 
 - Microsoft Outlook 97, versions 8.0, 8.01, 8.02, 8.03 
 - Microsoft Outlook 98 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you try to open a message that is located in an MS Mail Shared Folder and
has been created using Microsoft Electronic Forms Designer version 1.0, the
following error message will be displayed in a standard note form:

   Displaying this message correctly requires a custom E-Form that is not
   available or is incorrectly installed. Please contact your network
   administrator for more information.

This problem occurs if the message is being opened with an Exchange or Outlook
client.

CAUSE
=====

The Exchange and Outlook clients are unable to view an E-Form stored in an MS
Mail shared folder.

WORKAROUND
==========

To work around this problem, drag the message into your Inbox folder.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article. We are researching this problem and will post new
information here in the Microsoft Knowledge Base as it becomes available.
