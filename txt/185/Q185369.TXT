The information in this article applies to:

 - Microsoft CART Precision Racing, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft has released an update to Microsoft CART Precision Racing.

MORE INFORMATION
================

The following file is available for download from the Microsoft Download
Center:

   Cart101.exe
   (http://download.microsoft.com/download/cart/Patch/1/WIN98/EN-US/Cart101.exe)

Release Date: May-07-1998

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

   Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

This update offers the following improvements:

 - Improved Artificial Intelligence (AI)
 - Increased frame rate
 - Game controller null-zone
 - Gamma control for object brightness
 - Rain tires cheat fixed
 - Friction levels on grass and sand
 - More precise control over graphics settings
 - Steering ratio maximum increased in garage
 - Default setups improved for beginners
 - Wing-type error fixed for short ovals
 - Sound on replays fixed
 - Accelerated Graphics Port support
 - AI cars more competitive at intermediate and professional level
 - New Frequently Asked Questions list

Additional query words: 1.00 msgame indy car race patch
