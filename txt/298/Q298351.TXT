TITLE   :How to Change User Passwords on Windows NT Server by Web Browser
PRODUCT :Microsoft Windows NT
PROD/VER::4.0
OPER/SYS:
KEYWORDS:kbtool

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to use Web Administrator version 2.0 to change user
passwords on a server computer that runs Windows NT Server version 4.0. Web
Administrator 2.0 is included in the Windows NT Server 4.0 Resource Kit;
however, it is not officially supported by Microsoft.

MORE INFORMATION
================

To use Web Administrator 2.0 to change a user password:

1. Start your Web browser, type
   "http://<your_server_name>/ntadmin/default.asp" (without the quotation
   marks) in the address section of the browser, and then press ENTER.

2. Click Accounts, and then click Users.

3. Click the appropriate user account, and then click Password.

4. In the New Password box, type the new password.

5. In the Confirm Password box, type the new password again.

6. Click Update.

For more information about Web Administrator 2.0, refer to the following Web
sites:

   http://www.microsoft.com/ntserver/nts/downloads/management/NTSWebAdmin/default.asp
   (http://www.microsoft.com/ntserver/nts/downloads/management/NTSWebAdmin/default.asp)

   http://www.microsoft.com/ntserver/management/exec/feature/WebAdmin.asp
   (http://www.microsoft.com/ntserver/management/exec/feature/WebAdmin.asp)

The first Web site contains information about downloading the tool if you do not
have the Windows NT Server 4.0 Resource Kit. It also contains links to
frequently asked questions (FAQs) and to information about security and
support.

The second Web site lists the features of Web Administrator 2.0.
