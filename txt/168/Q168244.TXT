DOCUMENT:Q168244  14-MAY-1999  [exchange]
TITLE   :XCLN: "Locate Link Browser" Displayed When You Click URLs
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:4.0,5.0
OPER/SYS:
KEYWORDS:kbusage

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Windows 3.x client, versions 4.0, 5.0 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you attempt to access an embedded URL from within the Microsoft Exchange
Windows 3.x client, the Locate Link Browser dialog box is displayed and the URL
is not activated.

This only occurs if Internet Explorer is already running. If Internet Explorer is
not running, the URL launches properly.

NOTE: This problem does not occur when you attempt to open embedded .htm files
from within Exchange Client.

WORKAROUND
==========

 - Close Internet Explorer before accessing a URL from within the Exchange
   client.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server
versions 4.0 and 5.0. We are researching this problem and will post new
information here in the Microsoft Knowledge Base as it becomes available.
