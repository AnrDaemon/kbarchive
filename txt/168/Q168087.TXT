TITLE   :NTMMTA Locks Serial Port When Run as Service on Windows NT 4.0
PRODUCT :Microsoft Mail For PC Networks
PROD/VER:WINDOWS:3.5
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for PC Networks, version 3.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When the Mail Multitasking MTA for Windows NT (NTMMTA) is configured to run as a
service on Windows NT 4.0 and is configured to initialize an async modem, the
modem/com port will be unavailable even after the NTMMTA service is stopped.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Mail for PC Networks,
version 3.5. Microsoft is researching this problem and will post new information
here in the Microsoft Knowledge Base as it becomes available.
