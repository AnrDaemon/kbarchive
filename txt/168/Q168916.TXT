TITLE   :Volume Control Icon No Longer Visible on the Taskbar
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:4.0 95
OPER/SYS:
KEYWORDS:kbsound kbusage win95

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
 - Microsoft Windows NT Server version 4.0 
 - Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Volume Control icon may no longer be visible on the taskbar.

CAUSE
=====

Another user may have disabled the Volume Control icon.

RESOLUTION
==========

To resolve this issue, follow these steps:

1. In Control Panel, double-click Multimedia.

2. On the Audio tab, click the "Show volume control on the taskbar" check box to
   select it.

3. Click OK.

Additional query words: systray.exe system tray winnt
