TITLE   :Terminal Server Client Hangs Up After Repeated Logon and Logoffs
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0 SP4
OPER/SYS:
KEYWORDS:kbenv

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0 SP4, Terminal Server Edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Terminal Server client may hang up after repeated logons and logoffs from a
Remote Desktop Protocol (RDP) client to Terminal Server with Service Pack 4
installed.

When this occurs, the Logon dialog box appears, but you are unable to log on.

STATUS
======

To resolve this problem, obtain the latest service pack for Windows NT 4.0,
Terminal Server Edition. For additional information, please see the following
article in the Microsoft Knowledge Base:

   Q152734 How to Obtain the Latest Windows NT 4.0 Service Pack

Additional query words: connection available
