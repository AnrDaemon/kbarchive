SUMMARY
=======

This article contains comments, corrections, and information about known errors
relating to the Microsoft Press book Mastering Microsoft Visual Basic 6.0
Fundamentals, ISBN 0-7356-0898-9.

The following topics are covered:

 - Correction To Running Headers

 - CD-ROM: Typo In Default.htm

 - CD-ROM: Typo In Steps To Register Msvbvm50.dll

 - Page xvii: Typographical Error

 - Page 375: Missing Word In Answer C

MORE INFORMATION
================

Correction To Running Headers
-----------------------------

Change the running headers from:
"Microsoft Visual Basic 6 Fundamentals"

To:
"Microsoft Visual Basic 6.0 Fundamentals"


CD-ROM: Typo In Default.htm
---------------------------

First sentence under the Welcome subheading:

Change:
"This CD-ROM contains contains lab files..."

TO:
"This CD-ROM contains lab files..."


CD-ROM: Typo In Steps To Register Msvbvm50.dll
----------------------------------------------

Default.htm, Installing Course Files link, Step 2 under To Register
Msvbvm50.dll:

Change:
"Using Explorer, register Msvbvm50.dll by dragging the file over Regsver32.exe in
the same directory."

To:
"Using Explorer, register Msvbvm50.dll by dragging the file over Regsvr32.exe in
the same directory."


Page xvii: Typographical Error
------------------------------

There is a spelling error on the first line of page xvii.

Change:
"Chapter 5: Controllwing Program Execution"

To:
"Chapter 5: Controlling Program Execution"


Page 375: Missing Word In Answer C
----------------------------------

On page 375 under answer C there is a word missing.

Change:
"A property is a named attribute of an object that is used to a characteristic of
an object, such as name, size, enabled, and visible."

To:
"A property is a named attribute of an object that is used to describe a
characteristic of an object, such as name, size, enabled, and visible."


Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of this book, it may
already contain most or all of the above corrections.
