 - Microsoft Mail for AppleTalk Networks, version 3.1 
-------------------------------------------------------------------------------

SUMMARY
=======

In version 3.1 of Microsoft Mail for the Macintosh, a local username data file
is created on the user's workstation if there has been at least one online
session with the server and the Offline preference has been enabled. This file
is stored in the MS Local Mail folder located inside the Preferences folder.
This file contains all information regarding local folders that have been
created by the user.
