-------------------------------------------------------------------------------

SUMMARY
=======

The text below summarizes the requirements to run and capabilities of Microsoft
FoxBASE version 2.17 for MS-DOS.

FoxBASE requires 256K of memory to run. It supports the following data types:

   Character
   Numeric
   Logical
   Undefined

Although FoxBASE has no date data type, it supports several date functions, for
example, CTOD and DTOC.

System Specifications
---------------------

   Maximum # of files open:                              16
   Maximum # of records per database:                65,535
   Maximum # of characters per record:                1,000
   Maximum # of fields per record:                       48
   Maximum # of characters per field:                   255
   Digits of precision in numeric computations:          14
   Maximum # of characters per character string:        254
   Maximum # of characters per command line:            254
   Maximum # of characters per report header:           254
   Maximum # of characters per index key:               100
   Maximum # of memory variables:                       128
   MS-DOS version for single user:                      2.x
   MS-DOS version for multiuser:                        3.1 or higher

Requires SHARE to be loaded unless running on Novell. MFOX.EXE and FOXPCOMP.EXE
should be marked as read-only.

   IBM/SCO XENIX version:                             3.0/5.0

Additional query words: 2.17 Fox Base maximum minimum
