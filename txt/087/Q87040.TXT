-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail Gateway to SMTP, version 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Microsoft Mail Gateway to SMTP enables encapsulation by default; no
additional configuration is required for the gateway to support encapsulation.
To disable this feature, specify the -RE parameter on the command line when
starting the gateway.

REFERENCES
==========

Microsoft Mail Gateway to SMTP Administrator's Guide
