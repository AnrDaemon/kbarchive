TITLE   :PROFS: Versions of MS Mail Compatible with PROFS Gateway
PRODUCT :Microsoft Mail For PC Networks
PROD/VER::3.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail Gateway to IBM PROFS and OfficeVision, version 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

For version 3.0 of the Microsoft Mail Gateway to IBM PROFS and OfficeVision, you
must have version 2.1 or later of the Microsoft Mail Server Version installed
before you can install the gateway.

If you are currently running version 2.0x of The Network Courier, you must
upgrade to Microsoft Mail before installing or upgrading to version 3.0 of the
Microsoft Mail Gateway to IBM PROFS and OfficeVision.

REFERENCES
==========

"Microsoft Mail Gateway to IBM PROFS and OfficeVision LAN Administrator's
Guide"
