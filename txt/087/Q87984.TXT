TITLE   :Novell Server Disconnect Hangs Enhanced Mode Windows
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a,3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

If Microsoft Windows is running on a Novell workstation in 386 enhanced mode
while connected to a Novell server, and the server stops functioning and/or
disconnects the workstation, then the workstation may hang and display the error
message:

   System Error cannot read from device network.

This occurs even if Windows is installed in a standalone configuration. Novell is
aware of the problem and is working to correct this.

WORKAROUND
==========

To work around this problem run Windows in standard mode.

The NetWare product included here is manufactured by a vendor independent of
Microsoft; we make no warranty, implied or otherwise, regarding this product's
performance or reliability.
