TITLE   :Hellbender: Connection Failed Error in Network Game
PRODUCT :Microsoft Home Games
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kbfaq

======================================================================
1.0
WINDOWS
kbusage kbnetwork kberrmsg kbfaq

-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Hellbender for Windows, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you try to host a multiplayer game after leaving a previous multiplayer game,
you may receive the following error message:

   Connection Failed

RESOLUTION
==========

To host a new multiplayer game after leaving a previous game, you must exit the
multiplayer dialog box completely and start a new multiplayer game.


Additional query words: 1.00 Hellbender modem connection failed error
