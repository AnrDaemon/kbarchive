TITLE   :Matrox Video Drivers Limited to 640 x 480 Resolution
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:kbdisplay kbhw kbHardware

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0 
 - Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you are using a Matrox Millennium display adapter and an Intergraph
Intervue 21-inch monitor, the display may become distorted after you set the
Desktop Area setting in Display properties to any resolution higher than 640 x
480 and then restart the computer.

RESOLUTION
==========

To resolve this issue, restart your computer using the Last Known Good option.
Contact Matrox Technical Support for an updated driver.
