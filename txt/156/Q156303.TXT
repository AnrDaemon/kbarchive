SUMMARY
=======

The following article lists the minimum system requirements for running the 1997
edition of Microsoft Music Central.

MORE INFORMATION
================

The minimum system requirements needed to run Music Central 97 are the
following:

 - A Multimedia PC with a 486/33 MHz or higher processor

 - Microsoft Windows 95 operating system or Windows NT Workstation version 3.51
   or higher.

 - 8 MB of memory (RAM) when running on Windows 95 and 16 MB when running on
   Windows NT

 - Double-speed or faster CD-ROM drive

 - Super VGA 256-color monitor

 - Required hard-disk space:

   No hard-disk space required to run from CD

   11 MB of hard-disk space recommended for best performance

   8 MB of hard-disk space needed for monthly updates

 - Audio board and speakers required for sound

 - Microsoft Mouse or compatible pointing device

 - Windows-compatible modem and online service required for online monthly
   updates
