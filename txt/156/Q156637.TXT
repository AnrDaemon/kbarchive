TITLE   :HP DeskJet 660C and 855C Drivers Do Not Print Color at 600 DPI
PRODUCT :Microsoft Windows NT
PROD/VER:4.0
OPER/SYS:
KEYWORDS:kbprint kbPrinting

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0 
 - Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Hewlett-Packard (HP) DeskJet 660c and 855C printer drivers included with
Windows NT version 4.0 provides only monochrome printing at 600 dots per inch
(dpi), although the printer is capable of color printing at 600 dpi x 300 dpi.


MORE INFORMATION
================

Hewlett-Packard provides updated printer drivers that use Hewlett- Packard's
Color Resolution Enhancement Technology (CRET) to support color printing at 600
dpi x 300 dpi. To obtain these updated printer drivers, please visit
Hewlett-Packard's Web site.


For additional information about these updated drivers and their capabilities,
please contact Hewlett-Packard Technical Support.
