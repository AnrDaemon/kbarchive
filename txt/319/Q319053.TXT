SUMMARY
=======

This article describes how to modify Program Settings in MSN Internet Access for
Macintosh version 1.0.

MORE INFORMATION
================

The minimum requirements for connecting to MSN Internet Access are as follows:

 - Macintosh OS 9.x

 - A paid MSN Internet Access account

 - A modem with the minimum baud rate of 28.8 Kbps

 - Internet Explorer 5.x

 - Outlook Express 5.0.4

To modify Program Settings in MSN Internet Access version 1.0 for Macintosh

1. Start MSN Internet Access.

2. On the MSN Internet Access window, click Change Settings.
    - To open MSN Internet Access window each time you turn on your Macintosh,
      select Start MSN Internet Access when I start my computer.

    - To sign in to MSN Internet Access each time the MSN Internet Access window
      starts, select Sign in when MSN Internet Access starts.

    - To open Microsoft's Internet software (Outlook Express, MSN Messenger, or
      Internet Explorer) immediately upon signing in, make the following
      selections:
       - Under Internet Explorer, select Browse the Web.

       - Under Outlook Express, select Read my e-mail.

       - Under MSN Messenger, select Chat with friends.

3. Click OK to close the Change Settings window.
