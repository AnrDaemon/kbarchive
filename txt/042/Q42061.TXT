-------------------------------------------------------------------------------

SUMMARY
=======

DumpDesc.exe is a sample that demonstrates how to read the description line from
an .exe file. During the processing of the WM_PAINT message, the program reads
its own description line and displays the contents in its client area.

The most relevant portion of the code is in the DumpDescPaint() function.
