TITLE   :Mapi32.dll File Is Not Included in Service Pack 4 or Later
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:kbtool

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0 
 - Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Mapi32.dll file is not included in Windows NT 4.0 service packs beginning
with Service Pack 4.

MORE INFORMATION
================

The Mapi32.dll file is not included to prevent version conflicts with MAPI-aware
programs that automatically install an enhanced version of the file.
