TITLE   :PPT: Automatic Timings Don't Work in Playlist Slide Show
PRODUCT :Microsoft PowerPoint for Windows
PROD/VER:WINDOWS:4.0,4.0a,4.0c,7.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft PowerPoint for Windows 95, version 7.0 
 - Microsoft PowerPoint for Windows, versions 4.0, 4.0a, 4.0c 
-------------------------------------------------------------------------------

SYMPTOMS
========

Automatic timings may not work when you use a playlist to view multiple
PowerPoint presentations in the PowerPoint Viewer.

STATUS
======

Microsoft is researching this problem and will post additional information here
in the Knowledge Base as it becomes available.

RESOLUTION
==========

If you selected the Use Automatic Timings check box in the PowerPoint Viewer and
your slides do not advance automatically, then perform the following steps:

PowerPoint 7.0 (Using 32-bit Viewer)
------------------------------------

1. Open a presentation from your playlist in PowerPoint.

2. On the View menu, click Slide Sorter.

3. On the Edit menu, click Select All.

4. On the Tools menu, click Transition.

5. Select Advance Automatically After <x> Seconds and enter a number
   between 0 and 86399.

   NOTE: If you want different slides to appear on screen for different lengths
   of time, you can select each slide individually (instead of using Select All)
   and give each slide its own slide timing.

6. Click OK.

7. On the View menu, click Slide Show.

8. Under Advance, select Use Slide Timings.

9. Click OK to run the slide show with automatic timings.

10. On the File menu, click Save.

11. On the File menu, click Close.

12. Repeat steps 1 through 11 for each presentation in your playlist.

13. Open the PowerPoint Viewer.

14. Select your playlist file.

15. Viewer follows the settings saved in the presentation. The default setting
   in PowerPoint 7.0 is Manual Advance.

16. Click Show.

Automatic timings now work using a playlist in the PowerPoint Viewer.

PowerPoint 4.0 and PowerPoint 95
--------------------------------

1. Open a presentation from your playlist in PowerPoint.

2. On the View menu, click Slide Sorter.

3. On the Edit menu, click Select All.

4. On the Tools menu, click Transition.

5. Select Advance Automatically After <x> Seconds and enter a number
   between 0 and 86399.

   NOTE: If you want different slides to appear on screen for different lengths
   of time, you can select each slide individually (instead of using Select All)
   and give each slide its own slide timing.

6. Click OK.

7. On the View menu, click Slide Show.

8. Under Advance, select Use Slide Timings.

9. Click OK to run the slide show with automatic timings.

10. On the File menu, click Save.

11. On the File menu, click Close.

12. Repeat steps 1 through 11 for each presentation in your playlist.

13. Open the PowerPoint Viewer.

14. Select your playlist file.

15. Select the Use Automatic Timings check box.

16. Click Show.

Automatic timings now work using a playlist in the PowerPoint Viewer.
