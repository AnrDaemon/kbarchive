TITLE   :Windows NT Setup Displayed Twice in Task List
PRODUCT :Microsoft Windows NT
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 3.1 
 - Microsoft Windows NT Workstation version 3.1 
 - Microsoft Windows NT Advanced Server, version 3.1 
 - Microsoft Windows NT Workstation version 3.5 
 - Microsoft Windows NT Server version 3.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run the Windows NT Setup Help program (SETUPNT.HLP) and the Windows NT
Setup program (SETUP.EXE) concurrently, two versions of Windows NT Setup are
listed in the Task Manager task list.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT versions 3.1 and 3.5.
We are researching this problem and will post new information here in the
Microsoft Knowledge Base as it becomes available.
