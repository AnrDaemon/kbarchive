TITLE   :PC Win: Search All Folders Does Not Search Deleted Mail Folder
PRODUCT :Microsoft Mail For PC Networks
PROD/VER:WINDOWS:3.2,3.2a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for PC Networks, versions 3.2, 3.2a 
-------------------------------------------------------------------------------

SUMMARY
=======

In Microsoft Mail for PC Networks windows client, you can use the Message Finder
command to search for messages. You can choose to search a single folder or all
folders. If you select Look In All Folders, the messages in the Deleted Mail
folder will not be searched.


Additional query words: 3.20 3.20a
