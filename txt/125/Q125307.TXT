TITLE   :EPP and ECP Bi-Directional Port Support
PRODUCT :Microsoft Windows NT
PROD/VER:3.1 3.5 3.51 4.0
OPER/SYS:
KEYWORDS:kbother

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 3.1 
 - Microsoft Windows NT Workstation version 3.1 
 - Microsoft Windows NT Advanced Server, version 3.1 
 - Microsoft Windows NT Workstation versions 3.5, 3.51, 4.0 
 - Microsoft Windows NT Server versions 3.5, 3.51, 4.0 
-------------------------------------------------------------------------------

Windows NT does not support EPP (Enhanced Parallel Port) or ECP
(Microsoft Enhanced Communication Port) bi-directional communication
ports.


Additional query words: prodnt Remote Access Service (RAS) server bidirectional
