DOCUMENT:Q125597  27-JUL-2001  [sms]
TITLE   :SMS Help File (SM32.HLP) Not Installed on Windows 95 Client
PRODUCT :Microsoft Systems Management Server
PROD/VER:winnt:1.1
OPER/SYS:
KEYWORDS:kbnetwork kbsetup smssetup

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Systems Management Server version 1.1 
-------------------------------------------------------------------------------


SYMPTOMS
========

The Systems Management Server Client help file (SM32.HLP) is not installed on
the Windows 95 client during Systems Management Server client installation.

STATUS
======

Microsoft has confirmed this to be a problem in Systems Management Server
version 1.1. This problem has been corrected in Systems Management Server
version 1.2.
