TITLE   :LaserJet IIIsi Support Under Windows
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a,3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------


SUMMARY
=======

The Hewlett-Packard (HP) LaserJet IIIsi laser printer is not supported by the
HPPCL5A printer driver version 3.61 from Microsoft.

To fully support the LaserJet IIIsi, Microsoft Windows version 3.0 or 3.0a
requires the HPPCL5A version 30.3.86 driver.

NOTE: Microsoft Windows version 3.1 updates the HPPCL5A driver to version 30.3.86
during Windows setup.

MORE INFORMATION
================

The LaserJet IIIsi printer is manufactured by a vendor independent of Microsoft;
we make no warranty, implied or otherwise, regarding this product's performance
or reliability.
