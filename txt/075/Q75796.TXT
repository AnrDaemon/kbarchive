TITLE   :UAE When Running X-Vision 4.0
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SYMPTOMS
========

You may receive an Unrecoverable Application Error (UAE) when using X-Vision 4.0
in Microsoft Windows version 3.0.

CAUSE
=====

The XHOST.TXT file contains blank lines.

WORKAROUND
==========

You may be able to work around the problem by deleting the XHOST.TXT file or
filling the blank lines with text.

MORE INFORMATION
================
