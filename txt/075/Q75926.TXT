TITLE   :Installing Microsoft Entertainment Pack for Windows
PRODUCT :Microsoft Home Games
PROD/VER::1.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Best of Windows Entertainment Pack, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

To install the Microsoft Entertainment Pack for Windows:

1. Run Microsoft Windows.

2. Insert Disk 1 from the Microsoft Entertainment Pack for Windows into the
   floppy disk drive.

3. In Windows 3.x, choose Run from the File menu in Program Manager.

   In Windows 95, click the Start button on the Taskbar and click Run.

4. In Windows 3.x, type the following in the Command Line box

   " <drive>:setup " (without the quotation marks)

   where <drive> is the drive letter of your floppy disk drive. For
   example, type the following if your drive letter is drive A:

   " a:setup " (without the quotation marks)

   In Windows 95, type the following in the Open box

   " <drive>:setup " (without the quotation marks)

   where <drive> is the drive letter of your floppy disk drive. For
   example, type the following if your drive letter is drive A:

   " a:setup " (without the quotation marks)

5. Press the ENTER key or use the mouse to choose the OK button.

6. Follow the instructions displayed on the screen.

The Setup routine installs the Entertainment Pack files onto the hard disk.
