SUMMARY
=======

This article is part 1 of a series of two Microsoft Knowledge Base articles that
present a systematic approach to eliminating printing problems in Microsoft Word
97 for Windows.

The following topic is covered in part 1 of this article:

   
   What Are Some Specific Error Messages I May Get?

The following topics are covered in part 2 of this article:

 - How to Eliminate Document-Specific Printing Problems

 - What Are Some Other Things I Can Try?

 - Additional Troubleshooting Resources

 - Microsoft Support Options

For additional information and troubleshooting steps, click the article number
below to view the article in the Microsoft Knowledge Base:

   Q254233 WD97: Part 2: How to Troubleshoot Printing Problems from Word

   WINWORD caused an invalid page fault in module MSO97.dll.

      Word flashes the Printing dialog box.

      printer. Cancel printing and wait until the other software program

      -or- 

      Windows cannot print due to a problem with the current printer setup.
      Try one or more of the following:

      -or- 

      WINWORD caused an invalid page fault in UNKNOWN at <address>.

   Word cannot print due to a problem with the current printer setup.

   Word cannot change printers. No printers are installed.

Specific Error Messages When Printing
-------------------------------------

The following are some error messages you may receive when you attempt to print
from Microsoft Word. Please see the referenced articles to resolve printing
problems that are related to these specific error messages and situations.

 - This problem can be caused by one of the following:

    - When you print or view a document in print preview, page borders in your
      document may not be printed or may not be shown, or you may receive the
      following error message:

   One or more margins are set outside the printable area of the page. Choose
   the Fix button to increase the appropriate margins.

      For additional information, click the article number below to view the
      article in the Microsoft Knowledge Base:

   Q167874 WD97: Page Borders May Be Displayed or Printed Incorrectly

      -or-

    - If you set your document margins closer to the edge of the paper than your
      printer is capable of printing, Microsoft Word for Windows returns the
      following warning message for each copy that you send to the printer:

   One or more margins are set outside the printable area of the page. Choose
   the Fix button to increase the appropriate margins.

For additional information, click the article number below to view the article in
the Microsoft Knowledge Base:

   Q84647 WD97: "Margins Set Outside Printable Area of Page"

 - An error message caused by one of the following:

    - When you create a document in Microsoft Word 97 that contains page numbers
      and a merged table cell that spans several pages, you may get the
      following error message when you try to print a page other than page 1:

   WINWORD caused an invalid page fault in module WINWORD.EXE.

      For additional information, click the article number below to view the
      article in the Microsoft Knowledge Base:

   Q170875 WD97: IPF Printing Document with Large Merged Cells

      -or-

    - When you print the current page of your document, you may receive the
      following error message:

   WINWORD caused an invalid page fault in module WINWORD.EXE at
   0137:<address>.

For additional information, click the article number below to view the article in
the Microsoft Knowledge Base:

   Q164338 WD97: Invalid Page Fault Printing Current Page

 - When you click Print on the File menu, the following error message may
   appear:

   This program has performed an illegal operation and will be shut down.

   If the problem persists, contact the program vendor.

   When you click Details, the following message appears:

   WINWORD caused an invalid page fault in MSO97.dll at <address>.

   NOTE: The actual memory address may vary.

   For additional information, click the article number below to view the article
   in the Microsoft Knowledge Base:

   Q162411 OFF97: Page Fault in MSO97 When Printing


 - When you cancel printing to a Hewlett-Packard (HP) DeskJet 820CXi printer in
   Microsoft Word and then try to print again, you may experience the following
   symptoms or error messages:

   Word flashes the Printing dialog box.

   -or-

   Printer Driver Busy

   Another software application is using the printer.

   Cancel printing and wait until the other software program has finished. Try
   printing again.

   -or-

   Windows cannot print due to a problem with the current printer setup. Try one
   or more of the following:

   * Check the printer by printing a test page from Windows.
   * Make sure the printer is turned on and online.
   * Reinstall the printer driver.

   -or-

   This program has performed an illegal operation and will be shut down.

   If the problem persists, contact the program vendor.

   When you click Details (on Microsoft Windows Millennium Edition, press ALT+D),
   you receive the following message:

   WINWORD caused an invalid page fault in UNKNOWN at 014f:30080f6c.

   NOTE: The actual memory address may vary.

   For additional information, click the article number below to view the article
   in the Microsoft Knowledge Base:

   Q167369 WD97: Cannot Print After Canceling Using HPDJ820CXi Driver

 - When you try to print, the following error message may appear:

   Word cannot print due to a problem with the current printer setup.

   For additional information, click the article number below to view the article
   in the Microsoft Knowledge Base:

   Q140754 OFF: "Cannot change printers" and Similar Error Messages


 - When you click Print on the File menu, the following error message may
   appear:

   Word cannot change printers. No printers are installed.

   For additional information, click the article numbers below to view the
   articles in the Microsoft Knowledge Base:

   Q140754 OFF: "Cannot change printers" and Similar Error Messages

   Q162411 OFF97: Page Fault in MSO97 When Printing
