======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.5 
 - the operating system: Microsoft Windows NT, versions 4.0 SP5, 4.0 SP6, 4.0 SP6a 
 - Microsoft Commercial Internet System versions 2.0, 2.5 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the article numbers for new Collaboration Data Objects for
Microsoft Windows NT Server (CDONTS) bugs that have been fixed. For information
about how to obtain the fixes listed in this article, click the article number
next to the title of the article about that issue to view the article in the
Microsoft Knowledge Base.

NOTE: Exchange Server fixes for a particular component are cumulative and contain
all of the previous fixes for that component. Fixes with a particular version
number contain all of the fixes that have an earlier version number.

MORE INFORMATION
================

CDONTS fixes include the following files:

+------------------------------+
| File name  | Current version | 
+------------------------------+
| Cdonts.dll | 5.5.1877.28     | 
+------------------------------+

Fixes Released on February 29, 2000
-----------------------------------

The following files are modified:

 - All files incremented to version 5.5.1877.28

The following fixes are included:

   Q181697 PRB: Problems Sending Many Messages Using Multiple Threads
   Q214685 XFOR: CDONTS Not Thread-Safe, Crashes Under Stress
