TITLE   :WordArt Preview Is Blank After Installing Service Pack 5
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:3.51
OPER/SYS:
KEYWORDS:kbtool

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 3.51 
 - Microsoft Windows NT Workstation version 3.51 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you insert a WordArt object into a document, the WordArt preview may appear
blank.

CAUSE
=====

This behavior occurs if you have installed Service Pack 5 in Windows NT 3.51.
