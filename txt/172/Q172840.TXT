The information in this article applies to:

 - Microsoft Visual Basic Control Creation Edition for Windows, version 5.0 
 - Microsoft Visual Basic Professional Edition for Windows, version 5.0 
 - Microsoft Visual Basic Enterprise Edition for Windows, version 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The file vb5data.exe is a self-extracting Word document file that outlines the
features of Visual Basic and the differences between the Control Creation,
Professional and the Enterprise Editions. The document contains the following:

 - Product Highlights

 - Features of each of these editions of Visual Basic

 - Web Information

 - Product Requirements

The following file is available for download from the Microsoft Download Center:

   Vb5data.exe
   (http://download.microsoft.com/download/vb50ent/Sample28/1/WIN98/EN-US/vb5data.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

   Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.
