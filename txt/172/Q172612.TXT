TITLE   :SMSINST: SMS Installer Does Not Create Mscreate.dir File
PRODUCT :Microsoft Systems Management Server
PROD/VER::1.0,97
OPER/SYS:
KEYWORDS:kbusage smsinst

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Systems Management Server Installer version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

Systems Management Server Installer does not create the Mscreate.dir file, a
zero byte, hidden file used to create empty subdirectories during Microsoft
Office installations.

STATUS
======

Microsoft has confirmed this to be a problem in Systems Management Server
Installer version 1.0. We are researching this problem and will post new
information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION
================

The Mscreate.dir file is required for Office installations if you repackage or
customize Office installations.
