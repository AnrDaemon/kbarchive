TITLE   :WD97: How to Change Automatic Page Number Font or Size
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:kbdta word97

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to change the format (such as font or font size) of
automatic page numbers.

NOTE: Automatic page numbers do not include numbers applied in a header or
footer.

MORE INFORMATION
================

Follow these steps:

1. On the Format menu, click Style.

2. In the List box, click All Styles.

3. Select Page Number from the Styles box and click Modify.

4. Click the Format button and select Font.

5. Select the Font tab and select the desired font, style and size.

6. Click OK.

If the formatting for the Page Number style is to be used as the default for
newly created documents, then click to select the Add To Template check box in
the lower-left corner of the Modify Style dialog box and then click OK.

7. Click the Close button in the Style dialog box.
