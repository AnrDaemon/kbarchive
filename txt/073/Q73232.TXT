TITLE   :Cannot Delete Files in MS-DOS Executive with DELETE Key
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a,3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

When using the MS-DOS Executive, you cannot delete files with the DELETE
key. The files can only be deleted by using the Delete option in the File
Menu.

Additional query words: 3.0 3.00 3.0a 3.00a 3.1 3.10 3.11 win31
