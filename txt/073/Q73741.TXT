TITLE   :Task Swapper Problems on Tandy 1000 Computers
PRODUCT :Microsoft Disk Operating System
PROD/VER:MS-DOS:5.x,6.0
OPER/SYS:
KEYWORDS:msdos

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft MS-DOS operating system versions 5.0, 5.0a, 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Use of MS-DOS Shell Task Swapper on the Tandy 1000 computer may cause the system
to hang. A machine may lock up completely or return an "Invalid COMMAND.COM...
system halted" message. Use of MS-DOS Shell with Task Swapper disabled prevents
the machine from hanging.



Additional query words: 5.00 5.00a 6.00
