TITLE   :How to Obtain Updated Printer Drivers for Windows NT
PRODUCT :Microsoft Windows NT
PROD/VER:
OPER/SYS:
KEYWORDS:kbprint

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation versions 3.51, 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to obtain printer drivers for Windows NT 3.51 and
4.0.

MORE INFORMATION
================

There have been several printer driver releases for Windows NT. Information
about these drivers is available in the following article in the Microsoft
Knowledge Base:

   ARTICLE-ID: Q142643
   TITLE : Windows NT 4.0 Driver Library

You can query the Microsoft Knowledge Base on the World Wide Web by connecting to
the following site:

   http://www.microsoft.com

Click the Support button, and then click Knowledge Base.

You may also be able to obtain a driver for your printer from the printer's
manufacturer. Contact your printer's manufacturer for more information.

Each of these files needs to be extracted in its own directory with the -d
command.
