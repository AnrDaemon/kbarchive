TITLE   :PRB: "Error Reading Drive" with Windows NT 3.5 as Server
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:3.5
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 3.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When trying to access a remote file on a Windows NT Server or a Windows NT
Workstation acting as a server, users may receive one of the following error
messages:

   Error reading drive <drive name>

- or -

   System error: Network error on drive <drive name>

STATUS
======

Microsoft is researching this behavior and will post new information here in the
Microsoft Knowledge Base as it becomes available.
