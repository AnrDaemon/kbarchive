TITLE   :Flight Simulator Not Supported with A/UX Operating System
PRODUCT :Microsoft Home Games
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
SUMMARY
=======

Microsoft Flight Simulator for the Macintosh versions 1.0, 1.1, and 4.0 are not
supported with A/UX (Apple's version of the UNIX operating system). Flight
Simulator may appear to run temporarily, but input from the keyboard and mouse
have no effect. The program usually fails within a few minutes.

Additional query words: 1.00 1.10 4.00
