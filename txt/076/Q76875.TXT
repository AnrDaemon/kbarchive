TITLE   :Audio Option and Windows with Multimedia Extensions Setup
PRODUCT :Microsoft Windows 3.x Retail Product
PROD/VER::1.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows with Multimedia Extensions, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Windows with Multimedia Extensions Setup program does not detect your Audio
card.

NOTE: This problem affects Microsoft Windows with Multimedia Extensions version
1.0 running with Windows version 3.0 or 3.0a; it does not affect later versions
of Windows.

CAUSE
=====

This problem occurs because of conflicts with certain hardware.

WORKAROUND
==========

You must manually select the appropriate option to have sound capability.

In the Setup program, a screen listing all available hardware is shown. In the
Audio option, it says "Audio board not selected".

To change this option:

1. Press the UP ARROW key to highlight the Audio option.

2. Press the ENTER key. A list of available options will drop down.

3. Press the ARROW keys to select one of the options.

4. Press the ENTER key. Now, proceed with the rest of setup.
