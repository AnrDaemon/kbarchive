The information in this article applies to:

 - Microsoft 3D Movie Maker for Windows, version 1.0 
 - Microsoft Creative Writer for Windows, versions 1.0, 2.0

 - Microsoft My Personal Tutor: Alphabet Playhouse, version 1.0

-------------------------------------------------------------------------------

SUMMARY
=======

This article discusses the availability of the Microsoft Kids Products
troubleshooter.

The Microsoft Kids Products troubleshooter provides step-by-step instructions to
help you resolve common issues with Microsoft Kids products.

MORE INFORMATION
================

The Microsoft Kids Products troubleshooter is available at the following
Microsoft Web site:

   http://support.microsoft.com/support/tshoot/kids.asp
