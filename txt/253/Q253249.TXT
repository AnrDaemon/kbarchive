TITLE   :XADM: Exchange Server Versions Supported on Windows 2000 Server
PRODUCT :Microsoft Exchange
PROD/VER:winnt:4.0,5.0,5.5
OPER/SYS:
KEYWORDS:exc4 exc5 exc55

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, versions 4.0, 5.0, 5.5 
-------------------------------------------------------------------------------

SUMMARY
=======

If you want to install Exchange Server on a computer that is running Microsoft
Windows 2000 Server, the supported versions of Exchange Server are Exchange
Server version 5.5 Service Pack 3, or any later versions. Exchange Server
versions 4.0 and 5.0 are not designed to work on Windows 2000 Server, and are
not supported.

MORE INFORMATION
================

When you run Exchange Server 5.5 Setup, Windows 2000 Server generates the
following message:

   This program does not run correctly on this Version of Windows. To continue,
   click Run Program. To exit, click Cancel. For more information, click
   Details.

Click Run Program to finish the installation, and then complete the Exchange
Server 5.5 Setup. When Setup is finished, install Exchange Server 5.5 Service
Pack 3.
