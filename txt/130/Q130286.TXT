SUMMARY
=======

This article describes the TRAFFIC.EXE program that can be used to gather and
analyze message volume data for external Microsoft Mail for PC Networks
postoffices.

TRAFFIC.EXE is a Microsoft Windows-based application that allows a Microsoft Mail
Administrator to gather information about the number of messages sent to
external Microsoft Mail postoffices. TRAFFIC.EXE is available as part of the
Microsoft Mail for PC Networks Resource Kit.

MORE INFORMATION
================

TRAFFIC.EXE allows a Microsoft Mail Administrator to view the month to date and
year to date totals for the number of messages sent to each of the external
Microsoft Mail postoffices defined on a Microsoft Mail postoffice.

The program allows the Administrator to export this information to a
spreadsheet-compatible text file, which can then be imported into a spreadsheet,
like Microsoft Excel, for further analysis. The program also allows the user to
view the information in one of many chart formats.
