TITLE   :PC WSPlus: Offline Additions May Be Gone After Working Online
PRODUCT :Microsoft Schedule+ for Windows
PROD/VER:WINDOWS:1.0,1.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Schedule+ for Windows, versions 1.0, 1.0a 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you work in Microsoft Schedule+ for Windows and choose to work with
Schedule+ online, a project may be removed after a new task has been added to
the project while you were working offline.

If a project is deleted while you work online, any concurrent offline changes to
that project will be deleted when the user chooses to work online.

MORE INFORMATION
================


RESOLUTION
==========

This is by design.
