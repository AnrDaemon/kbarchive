The information in this article applies to:

 - Microsoft Visual FoxPro for Windows, version 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Below is a list of the files installed for a Laptop Setup of Visual FoxPro. If
Win32s is not installed on a Windows for Workgroups system, Win32s will also be
installed. The laptop setup will install all of the files for Win32s.

For a full description of files that are installed with Win32s from the Visual
FoxPro setup, please see the following article in the Microsoft Knowledge Base:

   Q130384 Critical System Files Installed With Visual FoxPro Setup
