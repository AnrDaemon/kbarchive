DOCUMENT:Q130403  17-DEC-2000  [win95x]
TITLE   :Cannot Install Unlisted Driver During Setup
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:95
OPER/SYS:
KEYWORDS:win95

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

You cannot install an unlisted or updated driver when you are installing Windows
95.

CAUSE
=====

The Windows 95 Setup program does not have a button or command to allow you to
install an unlisted or updated driver.

RESOLUTION
==========

Install the unlisted or updated driver after Setup is finished.
