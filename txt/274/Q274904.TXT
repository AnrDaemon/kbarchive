TITLE   :CFS2: Stops Responding or Quits Unexpectedly
PRODUCT :Microsoft Home Games
PROD/VER::
OPER/SYS:
KEYWORDS:kb3rdparty kbimu msgame

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Combat Flight Simulator 2: WWII Pacific Theater 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start or play Microsoft Combat Flight Simulator 2, the program may stop
responding (hang) or quit unexpectedly.

CAUSE
=====

This behavior can occur if a monitoring program for your motherboard is running
on the computer.

RESOLUTION
==========

To resolve this issue, quit the motherboard monitoring program before you start
Combat Flight Simulator 2.

For instructions about how to prevent the motherboard monitoring program from
starting automatically when you start your computer, please see the printed
documentation provided with the program.

Additional query words: 1.00 msgame combatfs cfs2 mother board monitor
