TITLE   :CFS2: Invalid Page Fault or Fatal Exception in Terrain.dll
PRODUCT :Microsoft Home Games
PROD/VER::1.0
OPER/SYS:
KEYWORDS:kberrmsg kbimu msgame

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

If you receive an invalid page fault or a fatal exception in module Terrain.dll
while you fly in Microsoft Combat Flight Simulator 2, you must restart the
computer before you continue to use the program.

NOTE: In certain cases, you may need to uninstall and reinstall Combat Flight
Simulator 2 before you can successfully start or run the program again.

MORE INFORMATION
================

Combat Flight Simulator 2 stores its scenery information in a series of files
and databases. In rare cases, these files and databases can become damaged. To
repair a damaged scenery file or database, you must restart the computer or
uninstall and reinstall the program.

For additional information about uninstalling and reinstalling the program, click
the article number below to view the article in the Microsoft Knowledge Base:

   Q303469 Combat Flight Simulator 2: Black Screen Appears When You Start the
   Game

Additional query words: 1.00 msgame combatfs cfs2 abnormal termination gpf ipf
