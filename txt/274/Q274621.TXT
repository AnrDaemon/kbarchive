TITLE   :MSN Explorer Closed Caption Services Not Available
PRODUCT :The Microsoft Network
PROD/VER::6.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - The Microsoft Network version 6.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

Closed Captioning does not work in MSN Explorer embedded Media Player.

CAUSE
=====

The feature has not been implemented

RESOLUTION
==========

This feature is unavailable in MSN Explorer Version 6.0. This functionality is
scheduled to be added to Version 6.1.

MORE INFORMATION
================
