TITLE   :DirectX: Files That Are Updated When You Install DirectX 6.1
PRODUCT :Microsoft Home Games
PROD/VER::
OPER/SYS:
KEYWORDS:kbref kbimu msgame

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - the operating system: Microsoft Windows 95 
 - the operating system: Microsoft Windows 98 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the DirectX files that are updated when you install DirectX
6.1. This article also lists the files that are not updated as well as new files
that are installed when you install DirectX 6.1.

MORE INFORMATION
================

The following files are updated when you install DirectX 6.1:

 - D3drm.dll
 - D3dxof.dll
 - D3dpmesh.dll
 - Ddraw.dll
 - Ddraw16.dll
 - Ddrawex.dll
 - Ddraw.vxd
 - Ddhelp.exe
 - Dplaysvr.exe
 - Dplayx.dll
 - Dpmodemx.dll
 - Dpsockx.dll
 - Dsetup.dll
 - Dsetup16.dll
 - Dsetup32.dll
 - Dxapi.sys
 - Dxsetup.exe

NOTE: The version of these files is 4.06.02.0436.

The following files are not updated when you install DirectX 6.1:

 - Dsound.dll
 - Dsound.vxd
 - Dinput.dll
 - Gcdef.dll
 - Gchand.dll
 - Joy.cpl
 - Msanalog.vxd
 - Vjoy.vxd

NOTE: The version of these files remains 4.05.01.1998.

DirectMusic is a new component that is installed by DirectX 6.1. The following
files are installed by DirectMusic:

 - Dmband.dll
 - Dmcompos.dll
 - Dmime.dll
 - Dmloader.dll
 - Dmstyle.dll
 - Dmsynth.dll
 - Dmusic.dll

NOTE: The version of these files is 4.06.02.0436.

The following file is a new Direct3D file installed by DirectX 6.1:

 - D3dramp.dll

NOTE: The version of this file is 4.06.02.0436.

For more information about DirectX 6.1, please see the following articles in the
Microsoft Knowledge Base:

   Q216078 DirectX: List of New Features Included in Microsoft DirectX 6.1

   Q216355 DirectX: List of Files Installed by the DirectX 6.1 Update

For additional information about how to use the DirectX Diagnostic Tool, click
the article number below to view the article in the Microsoft Knowledge Base:

   Q190900 DirectX: Description of the DirectX Diagnostic Tool

Additional query words: direct-x dx61 dx6.1 updates directory contents
