TITLE   :Flight Simulator 95/98: Game Controller Does Not Appear in List
PRODUCT :Microsoft Home Games
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kb3rdparty kbui fs98 kbimu msgamekbfaq

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Flight Simulator for Windows 95, version 1.0 
 - Microsoft Flight Simulator 98 
-------------------------------------------------------------------------------

SYMPTOMS
========

Your game controller may not appear in the Control Devices list on the Customize
Controls screen in Microsoft Flight Simulator.

Where your game controller should appear in the list, you see an empty line or a
string of illegible characters.

CAUSE
=====

This behavior can occur if the name of your game controller exceeds 47
characters in length. Flight Simulator cannot properly display the name of a
game controller exceeding 47 characters in length.

RESOLUTION
==========

To resolve this issue, contact the manufacturer of your game controller for
instructions about how to change the name of the game controller.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.
