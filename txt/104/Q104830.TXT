TITLE   :Cinemania: Demo Won't Run with Dell S3 Video Driver
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER::1994 edition
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Cinemania for Windows 1994 edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you are using a Dell S3 video driver, you are not able to run the Cinemania
Demo. When you try to start the demonstration, the following error message
appears:

   Display Problems - This program cannot continue. Your display driver does not
   properly support the following functions or capabilities: The GetPixel()
   function. One solution to this problem may be to use a display card or driver
   which offers more complete graphics support. Contact the manufacturer of your
   display card.

This problem occurs with Dell S3 chipset drivers including versions 1.4 and
2.02.

The products described here are manufactured by vendors independent of Microsoft;
we make no warranty, implied or otherwise, regarding these products' performance
or reliability.

RESOLUTION
==========

To avoid this problem, use one of the following two video drivers:

 - The Windows VGA driver that is included with Windows

   -or-

 - The Microsoft 640 x 480 Dell video driver that is shipped with Microsoft
   Video for Windows

If you use either of these drivers, the Cinemania Demo should display correctly.

For more information about how to perform this task in Windows, see your Windows
printed documentation or online Help.

For more information about Dell video cards and drivers, contact Dell Computer
Corp. technical support at (800) 624-9896.
