TITLE   :Oak VGA Prostar Excelerator May Cause Distortion in Shell

-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
 - Microsoft MS-DOS operating system versions 5.0, 5.0a, 6.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

The video display for the MS-DOS Shell is distorted after you run Windows 3.1.

CAUSE
=====

This problem can occur when you use an Oak VGA ProStar Excelerator video card
with video BIOS version 1.01 and Oak Technologies display drivers for Microsoft
Windows 3.1. This problem occurs only if you run an MS-DOS-based application
under Windows.

WORKAROUND
==========

To work around this problem, install the Microsoft VGA driver that ships with
Microsoft Windows version 3.1 (VGA.DRV).
