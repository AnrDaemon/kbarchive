TITLE   :PC Interface Version 4.1.3 Is Required with MS-DOS 6.0
PRODUCT :Microsoft Disk Operating System
PROD/VER:MS-DOS:6.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft MS-DOS operating system version 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Locus Technical Support has confirmed that version 4.1.3 or higher of PC
Interface is required for use with MS-DOS 6.0. Versions earlier than 4.1.3 may
return the following error message when the PC Interface software is loaded:

   Invalid drive in search path
   Current Drive is no longer valid
   Extended error 15

MORE INFORMATION
================

PC Interface is a network software that allows personal computers to connect to
UNIX servers. When the command PCIINIT.COM is invoked in either the AUTOEXEC.BAT
file or from the MS-DOS command line, the above error message appears.

For information about upgrading to version 4.1.3 of PC Interface, call Locus
Technical Support.
