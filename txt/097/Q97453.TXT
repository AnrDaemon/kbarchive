TITLE   :Backup for Windows Shows Incorrect Date on Directory Printout
PRODUCT :Microsoft Disk Operating System
PROD/VER:MS-DOS:6.0,6.2,6.21,6.22
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft MS-DOS operating system versions 6.0, 6.2, 6.21, 6.22 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you print a directory listing from Microsoft Backup for Windows, the date
displayed at the bottom of the page is incremented by one month. For example, if
the date set on your computer is 04/07/93, when you print the directory listing,
the date 05/07/93 appears as the footer on each page.

WORKAROUND
==========

At this time, no workaround for this problem is available.

Additional query words: 6.22 6.00 6.20 mwbackup msbackup back-up tree
