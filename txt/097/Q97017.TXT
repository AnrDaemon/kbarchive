TITLE   :Tandon Pacs: DoubleSpace Requires Updated DPACCESS.SYS
PRODUCT :Microsoft Disk Operating System
PROD/VER:MS-DOS:6.0,6.2,6.21,6.22
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft MS-DOS operating system versions 6.0, 6.2, 6.21, 6.22 
-------------------------------------------------------------------------------

SYMPTOMS
========

After using DoubleSpace to compress drives on a Tandon Pac computer, you cannot
see your compressed drives.

CAUSE
=====

This problem is caused by an older version of the DPACCESS.SYS device driver
from Tandon.

RESOLUTION
==========

To correct this problem, obtain an updated DPACCESS.SYS file from Tandon. For
more information, call Tandon Technical Support.
