TITLE   :PC WRmt: Installing Remote and LAN Drivers on One Computer

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail Remote for Windows, version 3.2 
-------------------------------------------------------------------------------

SUMMARY
=======

If you want to be able to switch between version 3.0 or 3.2 of Microsoft Mail
for PC Networks and version 3.2 of Microsoft Mail Remote for Windows clients on
the same computer, you must have installed Mail for PC Networks first.

Microsoft Mail Remote for Windows will run as a standalone application if it is
installed before Microsoft Mail for PC Networks is installed.
