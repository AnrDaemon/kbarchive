TITLE   :Zone Error Message: Browser Not Supported
PRODUCT :Microsoft Home Games
PROD/VER::
OPER/SYS:
KEYWORDS:kberrmsg kbimu msgame

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Zone.com 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to enter a lobby or a game room on Zone.com, you may receive
the following error message:

   Browser Not Supported.

CAUSE
=====

This behavior can occur if one of the following conditions is true:

 - Your computer is not running Microsoft Windows 95 or a later operating
   system.

 - Your computer is a Macintosh.

 - Microsoft Internet Explorer 4.0 or later is not installed on your computer.

 - Your Web browser is not properly installed.

RESOLUTION
==========

To play games on the Zone, Microsoft Windows 95, Microsoft Windows 98, Microsoft
Windows Millennium Edition (Me), Microsoft Windows NT 4.0 (with Service Pack 3
or later), Microsoft Windows 2000, or Microsoft Windows XP must be installed on
your computer.

For additional information about Zone.com system requirements, click the article
number below to view the article in the Microsoft Knowledge Base:

   Q163887 Zone: System Requirements

To obtain and install the latest version of Microsoft Internet Explorer, visit
the following Microsoft Web site:

   http://www.microsoft.com/windows/ie/
