TITLE   :XADM: Prompted to Overwrite Fradm32 Upgrading to Exchange 2000
PRODUCT :Microsoft Exchange
PROD/VER::5.5
OPER/SYS:
KEYWORDS:exc55 kbExchange550sp4Fix

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you upgrade from Exchange 5.5 Server to Exchange 2000 Server, you may
receive a message that states that the Fradm32.dll file that you are installing
is an older version than the original file. You are then prompted to keep or
replace this file.

WORKAROUND
==========

To work around this problem, click Yes when you are asked if you want to replace
the file.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server
version 5.5. This problem was first corrected in Exchange Server 5.5 Service
Pack 4.
