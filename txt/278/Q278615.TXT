SUMMARY
=======

MSN Companion Features

MORE INFORMATION
================

Communication
-------------

MSN Instant Messaging.

E-mail.

 - Mail waiting light.

 - One-button e-mail.

 - Hotmail based with a custom, easy-to-use user interface.

 - Automatic account creation and logon.

Internet Browsing
-----------------

Uses the latest Internet standards including:

 - HTML 4.0.

 - DHTML.

 - CSS.

 - JavaScript.

 - Secure connections using HTTPS and SSL.

 - Streaming and local multimedia support using Windows Media Technologies.

 - Favorites.

Commerce
--------

 - MSN Shopping.

 - Passport.

 - Ad rotator.

Personal
--------

 - Downloadable themes to customize the user interface to the user's
   preferences.

 - Framed picture. Displays a user-selected picture when the unit is not in use.

Ease of use
-----------

 - Start page.

 - One button access to e-mail, search, favorites, communities, and shopping.

 - Instant On / Off.

 - Very easy on-line setup and connection.

 - Tutorials / tours to help new users learn about the Web and MSN Companion.

 - Color USB printing of Web pages and e-mail.

 - One full-screen window.

 - Automatic upgrades.
