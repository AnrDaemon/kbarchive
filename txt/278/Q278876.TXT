TITLE   :Motocross Madness 2: Game Stops Responding on Startup Screen
PRODUCT :Microsoft Home Games
PROD/VER::2.0
OPER/SYS:
KEYWORDS:kb3rdparty kbimu msgame

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Motocross Madness 2, version 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start Microsoft Motocross Madness 2, the game may stop responding on
the startup screen.

CAUSE
=====

This behavior can occur if you are using one of the following pointing devices:

 - Logitech pointing device

 - Memorex Scrolling Mouse

RESOLUTION
==========

To resolve this issue, download and install the latest driver for your pointing
device from the appropriate Web site:

   http://www.logitech.com/cf/support/mousefiles_pc.cfm

   http://www.memorex.com/tech/input/updates.html
