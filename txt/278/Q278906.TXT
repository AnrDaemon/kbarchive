TITLE   :MSN Companion Supported Printers

OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - MSN Companion, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The MSN Companion supports the following printers.

MORE INFORMATION
================

 - Epson Stylus Color 670 (USB)

 - Epson Stylus Color 740 (USB)

 - Epson Stylus Color 777 (USB)

 - Epson Stylus Color 860 (USB)

 - Epson Stylus Photo 870 (USB)

 - Epson Stylus Color 900 (USB)

You can attach these printers via the USB port in the back of the MSN Companion
device to the device via the USB port in the back.

Note: these printers are supported because they all use the same driver, which is
included in the MSN Companion system image.
