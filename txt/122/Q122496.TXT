TITLE   :Head Coach Football Err Msg: SU Installed Properly...But...
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you install TSoft Head Coach Football for Windows under Windows 95, the
following error message is displayed:

   SU installed correctly...but could not add group to Program Manager

However, the game runs properly. The game successfully creates a folder and
corresponding icon on the desktop.

CAUSE
=====

Head Coach Football version 2.0 for Windows attempts to create a group and icons
in Program Manager but fails and returns the above error message.

STATUS
======

This situation is most likely caused by design changes in Windows 95. Microsoft
has confirmed that it is not caused by a problem in Windows 95. For more
information about resolving this issue, please contact TSoft.
