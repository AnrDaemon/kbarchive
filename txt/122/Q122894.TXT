TITLE   :Help Button Not Functional in Network Section of Control Panel
PRODUCT :Microsoft Windows NT
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
3.50

WINDOWS

kbui kbbug3.50

-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.5 
 - Microsoft Windows NT Server version 3.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When installing or configuring the Intel Token Express 16/4 driver from the
Network section of Control Panel, the Help button is not functional.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT Workstation and
Windows NT Server version 3.5. We are researching this problem and will post new
information here in the Microsoft Knowledge Base as it becomes available.
