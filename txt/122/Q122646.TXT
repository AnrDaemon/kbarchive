TITLE   :Windows NT 3.5 Setup on AST Bravo LP466D Fails

kb3rdparty

-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.5 
 - Microsoft Windows NT Server version 3.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you create the three Setup floppy disks and reboot your computer when you
are installing Windows NT version 3.5 on an AST Bravo LP466D, the following
error message may appear:

   Windows NT could not start due to a computer disk hardware configuration
   problem.

CAUSE
=====

This problem occurs if the Flash BIOS on the computer is version 1.00.

RESOLUTION
==========

To correct this problem, upgrade the Flash BIOS to version 1.06. You can
download the file called 516038.EXE, which is a self-extracting file, from AST
BBS at (714) 727-4132. For more information, call AST Technical Support at (800)
727-1278.
