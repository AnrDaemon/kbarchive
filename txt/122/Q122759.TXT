TITLE   :PC Win: Unexpected Cursor Movement When Using Arrow Keys
PRODUCT :Microsoft Mail For PC Networks
PROD/VER:WINDOWS:3.2
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for Windows, version 3.2 
-------------------------------------------------------------------------------

SYMPTOMS
========

Using the keyboard arrow keys in conjunction with the page down key to scroll
through a message in version 3.2 of Microsoft Mail for Windows causes the cursor
to be placed at the last line of the message. This is not the expected results.

STATUS
======

Microsoft is researching this problem and will post new information here in the
Microsoft Knowledge Base as it becomes available. We are researching this
problem and will post new information here in the Microsoft Knowledge Base as it
becomes available.

MORE INFORMATION
================

Steps to Reproduce Problem
--------------------------

1. Start to scroll by using the down arrow key.

2. Place the cursor one line past the end of the screen.

3. Press the page down key, then the up arrow key.

Instead of performing the desired cursor movement, the cursor will be placed at
the last line of the message.
