DOCUMENT:Q228772  10-AUG-2001  [winnt]
TITLE   :Cannot Customize Microsoft Year 2000 Product Analyzer Database
PRODUCT :Microsoft Windows NT
PROD/VER:WINDOWS:95; winnt:3.51,4.0
OPER/SYS:
KEYWORDS:kbtool kbui kbdocerr win95 win98

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation versions 3.51, 4.0 
 - Microsoft Windows 98 
 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you try to locate more information about how to customize the Microsoft
Year 2000 Product Analyzer tool compliance database on the following Microsoft
Web site, you may be unable find this information:

   http://www.microsoft.com/technet/year2k/pca/pca.htm

CAUSE
=====

This problem can occur because of documentation errors in the Microsoft Year
2000 Product Analyzer tool that specify you have the ability to edit the
database to add information about third-party products. This information is
incorrect.

RESOLUTION
==========

To work around this problem, do not try to edit the database to add information
about third-party products.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.
