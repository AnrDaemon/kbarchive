TITLE   :Baseball 2000: Game Controller Commands Are Not Recognized Corre
PRODUCT :Microsoft Home Games
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kbhw kbimu msgame kbHardware

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Baseball 2000, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you play Microsoft Baseball 2000, the commands you send using your game
controller may result in unexpected game actions and player movements.

CAUSE
=====

This behavior can occur if you switch game controllers while you play a game.

RESOLUTION
==========

To prevent this issue from occurring, select the game controller you want to use
before you start Baseball 2000, or save your game and then quit Baseball 2000
before you switch game controllers.

Additional query words: 1.00 bball2000 bball2k reaction wrong
