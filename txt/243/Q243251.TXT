TITLE   :Missing Password-Protected Screen Saver Leaves Computer Unlocked
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0 
 - Microsoft Windows NT Server version 4.0, Terminal Server Edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

If the specified screen saver is set to use password protection, but the screen
saver does not exist on the computer, the computer is not locked when the screen
saver time-out occurs. Users can still start programs.

At this point, pressing CTRL+ALT+DELETE does not display the Windows Security
dialog box. Instead, the workstation is locked and displays the Unlock Windows
dialog box.
