DOCUMENT:Q171042  07-SEP-2001  [word97]
TITLE   :WD97: Red Box Prints Behind WordArt Object on Canon BJC 4550
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:kbprint word97

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you print a Microsoft Word 97 document that contains a particular style of
WordArt to a Canon BubbleJet BJC-4550 printer, a red box is printed behind the
WordArt object.

CAUSE
=====

This problem occurs when you use the WordArt style in the third row and third
column of the WordArt Gallery dialog box, and print with version 3.70 of the
printer driver.

RESOLUTION
==========

Upgrade the printer driver to version 3.90.

For information about how to contact Canon International Systems, query in the
Knowledge Base for one of the following articles:

   Q65416 Hardware and Software Third-Party Vendor Contact List, A-K

   Q60781 Hardware and Software Third-Party Vendor Contact List, L-P

   Q60782 Hardware and Software Third-Party Vendor Contact List, Q-Z


STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows.

MORE INFORMATION
================

The Canon BubbleJet BJC-4550 printer is manufactured by Canon International
Systems, a vendor independent of Microsoft; we make no warranty, implied or
otherwise, regarding this product's performance or reliability.
