TITLE   :WD97: Borders Print Poorly with Hewlett-Packard 820Cse
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:kb3rdparty kbprint winword word97

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

Some paragraph borders print with very poor quality, extra lines, or red dots
across the border when printing from Microsoft Word 97 to a Hewlett- Packard
(HP) 820Cse printer.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows.

MORE INFORMATION
================

Hewlett-Packard has released only two versions of drivers for the HP 820Cse
printer; the problem occurs with both versions of the printer driver. (The
latest version is 8.2.) Unlike other HP DeskJet printers, the HP 820Cse cannot
use any other printer driver. Hewlett Packard has confirmed this information.

For information about how to contact Hewlett-Packard, query in the Knowledge Base
for one of the following articles:

   Q65416 Hardware and Software Third-Party Vendor Contact List, A-K

   Q60781 Hardware and Software Third-Party Vendor Contact List, L-P

   Q60782 Hardware and Software Third-Party Vendor Contact List, Q-Z


Additional query words: 8.0 hp820 hp820cse
