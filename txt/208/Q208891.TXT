TITLE   :MSN Connection Manager Disconnects Modem.
PRODUCT :The Microsoft Network
PROD/VER:WINDOWS:2.5,2.51,2.52,2.6,95,98,98 Second Edition
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - The Microsoft Network versions 2.5, 2.51, 2.52, 2.6 
 - the operating system: Microsoft Windows 95 
 - the operating system: Microsoft Windows 98 
 - the operating system: Microsoft Windows 98 Second Edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use the connection manager to connect to MSN, the connection manager
hangs up the modem not giving it a chance to connect to the Internet. This does
not apply if connecting using Dial Up Networking or the hyperterminal.

CAUSE
=====

This behavior occurs if an improper modem driver or an outdated modem driver is
being used to establish your connection.

RESOLUTION
==========

To resolve this issue, you need to use the proper modem driver or you need to
update your existing modem driver.
