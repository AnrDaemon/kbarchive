TITLE   :Irwin Magnetics EasyTape Backup with MS-DOS 5.0

KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft MS-DOS operating system versions 5.0, 5.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

Irwin Magnetics has confirmed that the EasyTape Backup System 2000 Series
requires version 2.22 of the EasyTape software to be compatible with Microsoft
MS-DOS version 5.0.

MORE INFORMATION
================

Using EasyTape Software versions earlier that 2.22 can cause various errors
including the following:

   Critical Error. Sector Not Found Reading Drive C

The product included here, Irwin Magnetic EasyTape Backup System, is manufactured
by a vendor independent of Microsoft; we make no warranty, implied or otherwise,
regarding this product's performance or reliability.
