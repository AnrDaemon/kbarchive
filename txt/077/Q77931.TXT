The information in this article applies to:

 - Microsoft Windows Software Development Kit (SDK) 3.1 
-------------------------------------------------------------------------------

SUMMARY
=======

In Windows 3.1, there are a number of macros available that can be used by the
writer of help systems to enhance an application's help files. MACROHLP was
written to demonstrate each of the macros that can be used by the person writing
help files. The RTF source files are also included in MACROHLP as examples for
the person authoring help files to follow.

MACROHLP is one of the Advanced Samples provided with the Windows Software
Development Kit (SDK), version 3.1.

The following file is available for download from the Microsoft Software
Library:

   Macrohlp.exe
   (http://support.microsoft.com/download/support/mslfiles/Macrohlp.exe)

For more information about downloading files from the Microsoft Software Library,
please see the following article in the Microsoft Knowledge Base:

   Q119591 How to Obtain Microsoft Support Files from Online Services



Additional query words: softlib MACROHLP.EXE
