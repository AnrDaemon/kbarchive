TITLE   :Enhanced Mode Windows Does Not Reset COM Ports
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SYMPTOMS
========

Your communications or modem program runs correctly before starting Microsoft
Windows, but does not run or initialize after enhanced mode Windows has been run
and exited.

CAUSE
=====

Serial ports used by a program in enhanced mode of Microsoft Windows 3.0 may not
be reset upon exiting Windows.

RESOLUTION
==========

If this occurs, there are two ways to correct the problem.

 - Reboot the computer. This fully resets the serial ports.

   -or-

 - Run a program specifically written to reset serial ports.

NOTE: There are several shareware and freeware programs written that perform a
COM port reset.
