TITLE   :DOCERR: User Must Execute "CMD" in PAUSE Statement Under OS/2
PRODUCT :Microsoft Fortran Compiler
PROD/VER::5.1
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft FORTRAN compiler for OS/2, version 5.1 
-------------------------------------------------------------------------------

Page 210 of the Microsoft FORTRAN reference manual describes how executing
the word "COMMAND" from the PAUSE statement within a FORTRAN program will
allow the user to execute a series of commands. However, under OS/2, the
user must type the word "CMD".

Additional query words: 5.10
