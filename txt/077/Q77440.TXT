TITLE   :DEC PathWorks VT320 Fonts Must Be Installed
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

Digital Equipment Corporation (DEC) Pathworks' VT320 emulation software requires
the following display fonts to be installed into Microsoft Windows from the
Windows Control Panel.

    Fonts               Description
    -----               -----------

    DECTERM.FON         VGA resolution
    DECTERME.FON        EGA resolution
    DECTERMC.FON        CGA resolution

MORE INFORMATION
================

Follow these steps to install these fonts:

1. Run Control Panel and select the Fonts icon.

2. Choose the Add button, and then enter font filename and location.
