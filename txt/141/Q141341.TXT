TITLE   :XCLN: No Print Option When Using EFD on Windows NT
PRODUCT :Microsoft Exchange
PROD/VER::4.0
OPER/SYS:
KEYWORDS:kbprint kbPrinting

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft Exchange Electronic Forms Designer version 4.0 does not have an option
for Print Setup when you run on Microsoft Windows NT. When you print from a
Forms Designer Electronic Form, the form will use the Windows NT system printing
defaults.
