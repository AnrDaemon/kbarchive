TITLE   :"The Application Cannot Start" When Trying to Start Chat
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:95
OPER/SYS:
KEYWORDS:win95

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you try to run Chat, you may receive the following error message:

   The application cannot start

CAUSE
=====

This error can occur if you are trying to run the version of Chat (Winchat.exe)
included with Microsoft Windows for Workgroups.

RESOLUTION
==========

Install and use the version of Chat included on the Windows 95 CD-ROM.

For information about how to install the Windows 95 version of Chat, please see
the following article in the Microsoft Knowledge Base:

   Q141827 How to Install Chat in Windows 95
