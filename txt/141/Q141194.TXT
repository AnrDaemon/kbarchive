TITLE   :Cannot Reply By E-mail To Messages With Brackets In Address
PRODUCT :The Microsoft Network
PROD/VER::1.0,1.05,1.2,1.3,2.0,2.5,2.6
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - The Microsoft Network versions 1.0, 1.05, 1.2, 1.3, 2.0, 2.5, 2.6 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you reply to a bulletin board (BBS) or newsgroup message sent from an
address that includes angle brackets (for example, Mike Smith
<mike@example.com>), the message is returned to you as undeliverable.

CAUSE
=====

The Microsoft Network e-mail does not support this address format.

NOTE: This is not a common problem with Microsoft Outlook or Microsoft Outlook
Express but does occur with certain addressses.

RESOLUTION
==========

Remove the angle brackets and correctly format the address before you send the
reply.

================
