TITLE   :Mac Encarta 96: Program Fails to Run from the Compact Disc
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER::1996 edition
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta 96 Encyclopedia for Macintosh 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you attempt to start Encarta 1996 from the compact disc, you receive the
following error message:

   Error creating annotation file

This message also occurs whenever you try to create a Notemark.

Or, if you attempt to use the Yearbook Builder, Encarta 1996 may freeze or
unexpectedly quit.

CAUSE
=====

Encarta 1996 was not designed to run from the compact disc.

RESOLUTION
==========

Install Encarta on your computer using the installation process and instructions
included with your compact disc.
