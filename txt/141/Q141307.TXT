TITLE   :Cannot See All Files on LAN Manager for UNIX Server
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:95
OPER/SYS:
KEYWORDS:win95

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you are viewing files on a LAN Manager for UNIX version 2.x server, you may
not be able to see all the files in a directory. However, all the files are
accessible.


RESOLUTION
==========

To work around this problem, use File Manager to view the files. To do so,
follow these steps:

1. Click the Start button, and then click Run.

2. Type the following line in the Open box, and then click OK:

   " winfile.exe " (without the quotation marks)

3. View the files with File Manager.

You may also want to contact your LAN Manager for UNIX vendor to inquire about an
updated version that resolves this problem.
