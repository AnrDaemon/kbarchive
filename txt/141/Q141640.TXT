 - Microsoft Encarta 96 Encyclopedia for Macintosh 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the steps to manually install Encarta 1996 for the Macintosh.

MORE INFORMATION
================

To perform a manual installation of Encarta 1996 for the Macintosh, do the
following:

1. Insert the Encarta 96 compact disc into the CD-ROM drive.

2. Create a new folder on your hard drive where you want to install Encarta.
   Name the folder Encarta 96 Encyclopedia.

3. Double-click the Encarta 96 Encyclopedia CD-ROM icon.

4. Locate and open the Installer Files folder.

5. Drag the following items to the Encarta 1996 Encyclopedia folder on your hard
   drive:

   Animations Launcher
   Encarta '96
   Encarta Projector
   Animation
   Microsoft System Information

6. Locate and open the System Files folder on the CD-ROM.

7. Drag the files in the System Files folder on the disc to the System Folder
   icon on your hard drive. When you are prompted to place items into the
   appropriate locations, click OK.

8. Open the Fonts folder in the System Files folder on the compact disc.

9. Drag the files in the Fonts folder on the disc to the System Folder icon on
   your hard drive. When you are prompted to place items into the appropriate
   locations, click OK.

10. Close the System Files folder.

11. Close the Installer Files folder.

12. Open the Encarta 96 Encyclopedia folder on the hard drive.

13. Create two new folders in the Encarta 96 Encyclopedia folder. Name the
   folders:

   Data Files
   Yearbook Files

14. Locate and open the Encarta Files folder on the compact disc.

15. Copy the following file to the Encarta '96 Encyclopedia folder you created
   on your hard drive:

   Yearbook State

16. Copy the following files to the Data Files folder you created on your hard
   drive:

   Encarta Data
   See Also Index
   String2 Table
   Time1 Data
   Time2 Data

17. Copy the following file to the Yearbook Files folder you created on your
   hard drive:

   YB895A.YBK

18. If you have Microsoft Works 4.0x installed on this machine, create a folder
   named Encarta 1994 Preferences. Place this folder in the Preferences folder
   located in your System Folder.

19. Close all the windows and restart your Macintosh.

After restarting, the manual installation is complete.
