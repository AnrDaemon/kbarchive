TITLE   :WD97: Formatting Toolbar Shows Incorrect Text Direction in Table
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:kbualink97 kbui kbusage

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Formatting toolbar buttons may show an incorrect text direction for the
selected table cell. This problem occurs when the following are true:

 - You have a multiple column table.

   -and-

 - You have rotated text in the first column of the table.

   -and-

 - You select a different table cell on the same row containing the rotated
   text.

NOTE: The toolbar buttons that display an incorrect direction are Alignment,
Bullets, Numbering, and Indent.

WORKAROUND
==========

Place the insertion point in the text or select the text of the table cell.

NOTE: Do not select the entire table cell.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article. We are researching this problem and will post new
information here in the Microsoft Knowledge Base as it becomes available.
