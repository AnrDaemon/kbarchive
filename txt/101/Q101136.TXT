TITLE   :PC Win: Err Msg: The Old Message File Could Not Be Deleted
PRODUCT :Microsoft Mail For PC Networks
PROD/VER:WINDOWS:3.0,3.2
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for Windows, versions 3.0, 3.2 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you try to move an .MMF file from local storage to the postoffice, you are
unable to move the file and receive the following error message:

   The Old Message File Could Not Be Deleted

CAUSE
=====

This problem occurs if you do not have Erase or Delete rights to the MMF
subdirectory of the Mail database on the server.

RESOLUTION
==========

Obtain Erase or Delete rights to the MMF subdirectory of the Mail database.

Additional query words: 3.00 3.20
