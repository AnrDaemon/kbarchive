TITLE   :Scenes 1.0: How to Uninstall Scenes
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Scenes (all collections), version 1.0 
-------------------------------------------------------------------------------

To remove Microsoft Scenes from your computer, follow these steps:

1. Delete the Microsoft Scenes folder (usually, C:\Scenes).

2. Delete the following files from the Windows folder (usually, C:\Windows):

   BMPBDR.DLL
   DVJPG.DLL
   FX.DLL
   MSSBMP.DLL
   MSSCTRL.DLL
   MSSFX.DLL
   MSSHOOK.DLL
   MSSJPG.DLL
   MSSLOCAL.DLL
   MSSPIC.DLL
   MSSRENU.DLL
   MSSAD.DLL
   MSSCENES.BMP
   MSSPAPER.BMP
   SCENES.INI
   SSMSSCNS.SCR

3. If you added Microsoft Scenes to your current After Dark 2.0 setup, delete
   any After Dark modules added by Microsoft Scenes from the After Dark folder.
   All or some of the following files may exist in the After Dark folder
   (usually, C:\Afterdrk):

   MSS00001.AD
   MSS00002.AD
   MSS00003.AD

4. From each folder where Custom Picture Sets were set up, delete the following
   files

   ENU.INI
   PICSET.INI
   MSS_<number>.CAT

   where <number> is any number between zero (0) and 3.

5. Delete any Microsoft Scenes icons from the Microsoft Scenes or Startup group.

For more information about how to remove Microsoft Scenes from your
computer in Windows, see your Windows printed documentation or online Help.
