TITLE   :Mac Form: Installing a Custom Form on a Mail Server
PRODUCT :Microsoft Mail For Appletalk Networks
PROD/VER:WINDOWS:2.0,2.0a,2.0b,3.0

KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

To install a custom form in Microsoft Mail, do the following:

1. Sign in to Microsoft Mail as Network Manager for the server on which you want
   to install the form.

2. Open the stack of the form you want to install. Compile all components of the
   form (Send Screen, Read Screen, icons, and Printed Form).

3. Click the Form Mover icon to display the Form Mover screen.

4. Select the installation option you want for the form by doing one of the
   following:

   a. To install the form so that your users can send messages with it, select
      the Send/Read option.

   b. To install the form so that your users can only display messages in the
      Read form, select the Read Only option.

5. Click the Install button. A dialog box will be displayed that enables you to
   choose the position of the form in the list of forms (Summary screen). Once a
   position is chosen, click Install to install the form in the server and
   return to the Form Mover screen.

6. Click Done to return to the form's title screen.

MORE INFORMATION
================

For more information on installing a custom form, see the appendix "Using Custom
Forms in Microsoft Mail" in the "Microsoft Mail Network Manager's Guide."
