-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Flight Simulator for MS-DOS, version 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

You may receive the following error message if you try to create a logbook in
Microsoft Flight Simulator 5.0:

   File cannot be created because: Two line acknowledge second line

MORE INFORMATION
================

This error message occurs when no filename is entered in the Filename box in the
Create Logbook dialog box.

To correct the problem, type a valid MS-DOS filename in the Filename box.

Additional query words: 5.00 flightsim fltsim dos FS5
