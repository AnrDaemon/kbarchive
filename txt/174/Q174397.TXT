TITLE   :Hewlett Packard NetRAID Card May Require Updated Firmware
PRODUCT :Microsoft Windows NT
PROD/VER:WinNT:4.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server, Enterprise Edition version 4.0 
 - Microsoft Cluster Server 
-------------------------------------------------------------------------------

SUMMARY
=======

If you use a Hewlett Packard (HP) NetRAID adapter card with Microsoft Cluster
Server, the card may need a later firmware revision for it to operate
correctly.


MORE INFORMATION
================

HP clusters using the HP NetRAID card will not correctly run Microsoft Cluster
Server unless the NetRAID driver and firmware have been upgraded (version
A.03.16 or higher).


The Microsoft Windows NT Server, Enterprise Edition 4.0 Hardware Partners can be
found at the following URL:

http://www.microsoft.com/ntserver/ntserverenterprise/exec/vendors/hardwarepartners.asp

The Microsoft Windows Hardware Compatibility List (HCL) may be found with the
URL:

www.microsoft.com/hwtest/hcl
