The information in this article applies to:

 - The Microsoft Network versions 1.3, 2.0, 2.5, 2.6, 5.0, 5.1, 5.2, 5.3 
-------------------------------------------------------------------------------

SUMMARY
=======

The following Integrated Services Digital Network (ISDN) adapters have been
tested and are compatible with the MSN, The Microsoft Network, ISDN network.
Additional information about the installation of each adapter is available in
the Microsoft Knowledge Base.

Dual-Channel ISDN Adapters
--------------------------

 - 3Com Impact IQ

 - Ascend Pipeline 25 PX

 - Motorola BitSurfr Pro

 - USR Sportster 128k

Single-Channel ISDN Adapters
----------------------------

 - 3Com Impact IQ

 - Motorola BitSurfr

 - Motorola Bitsurfr Pro

 - USR Courier I-Modem

Other MSN-Compatible Adapters
-----------------------------

 - Diamond Net Commander

 - Digi Datafire

 - Digi PCIMAC

 - Eicon Diva

NOTE: The adapters listed in the "Other MSN-Compatible Adapters" section require
the Windows 95 Dial-Up Networking 1.2 Upgrade. To obtain the Dial- Up Networking
1.2 Upgrade, visit the following Microsoft Web site:

   http://www.microsoft.com/windows95/info/dialup.htm
