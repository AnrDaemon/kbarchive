TITLE   :XWEB: Outlook Express Does Not Notify on Copy Failure
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:5.5
OPER/SYS:
KEYWORDS:kbusage

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Outlook Web Access, version 5.5 
 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you are using Outlook Express and you drag a message to a public folder
with that client, you will not receive notification of success or failure. You
may think there is a server problem when in fact the client is at fault.

CAUSE
=====

Outlook Express doesn't pass along the NO response to you, the user.

WORKAROUND
==========

There is no workaround at this time.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server
version 5.5. Microsoft is researching this problem and will post new information
here in the Microsoft Knowledge Base as it becomes available.
