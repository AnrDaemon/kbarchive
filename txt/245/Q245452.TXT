TITLE   :Streets & Trips 2000: Unable to Update Construction Information
PRODUCT :Microsoft Automap
PROD/VER:WINDOWS:
OPER/SYS:
KEYWORDS:kberrmsg kbfile kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Expedia Streets and Trips 2000 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to update the construction information in Microsoft Expedia
Streets and Trips 2000, you may receive one of the following error messages:

    - The server name or address could not be resolved.

    - Could not download construction information.

RESOLUTION
==========

To resolve this issue, download and install the St2kup.exe file from the
Microsoft Download Center.

The following file is available for download from the Microsoft Download Center:

   St2kup.exe

Release Date: Apr-21-2000

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

   Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

MORE INFORMATION
================

The St2kup.exe file is not supported by Microsoft Windows NT 4.0 with Service
Pack 4.
