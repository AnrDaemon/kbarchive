DOCUMENT:Q271403  14-NOV-2000  [word97]
TITLE   :WD97: Word Document Opens Read-Only on FTP Server
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:kbdta word8 word97 kbwdinternet

======================================================================

-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you open a Word document that is on a File Transfer Protocol (FTP) server,
the document is opened read-only.

CAUSE
=====

This problem occurs when the document file name contains a space.

WORKAROUND
==========

To work around this problem, save the Word document without a space in the file
name to the FTP server.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.

MORE INFORMATION
================

For more information about opening a document on an FTP site, click "Contents
and Index" on the Help menu, click the Index tab in Microsoft Word Help, type
the following text

   opening documents, Internet

and then double-click the selected text to go to the "Open a document" topic. If
you are unable to find the information you need, ask the Office Assistant.

For more information about saving a document to an FTP site, click "Contents and
Index" on the Help menu, click the Index tab in Microsoft Word Help, type the
following text

   saving documents

and then double-click the selected text to go to the "Save a document to an FTP
site" topic. If you are unable to find the information you need, ask the Office
Assistant.
