TITLE   :Cannot Open Research Organizer Files in Encarta Researcher
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER::
OPER/SYS:
KEYWORDS:kbtool kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta Africana, Third Edition 
 - Microsoft Encarta Encyclopedia Deluxe 2001 for Windows 
 - Microsoft Encarta Encyclopedia Standard 2001 for Windows 
 - Microsoft Encarta Interactive World Atlas 2001 
 - Microsoft Encarta Reference Suite 2001 
 - Microsoft Encarta Researcher 
-------------------------------------------------------------------------------

SUMMARY
=======

You cannot open a research file that you created using Microsoft Encarta
Research Organizer in Microsoft Encarta Researcher (the updated version of
Encarta Research Organizer that is included with the products listed at the
beginning of this article).

MORE INFORMATION
================

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.

Additional query words: multi multi-media media mm unable ero
