TITLE   :Microsoft Reader: Cannot Download E-Books with Internet Explorer
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER::1.5
OPER/SYS:
KEYWORDS:kbenv kberrmsg kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Reader, version 1.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use Microsoft Reader to download an e-Book, you may receive any of the
following error messages:

    - Opening http://<downloadurl>/scripts/ebds.dll. Some files can contain
   viruses or otherwise be harmful to your computer. It is important to be
   certain that this file is from a trustworthy source. What would you like to
   do with this file?

    - An error has occurred in the script on this page. The script operation has
   been canceled. Microsoft JScript runtime error [Line: 30]
   'MSReaderWebShop_GetMSReaderActivationID' is undefined.

    - An error has occurred in the script on this page. The script operation has
   been canceled. Microsoft JScript runtime Error [Line: 35]
   'MSREaderWebShop_IsMSReaderInstalled' is undefined.

CAUSE
=====

This behavior can occur if Microsoft Internet Explorer 3.0 or earlier is
installed on your computer.

RESOLUTION
==========

To resolve this issue, download and install Microsoft Internet Explorer 4.01
Service Pack 1 or later.

To download the latest version of Microsoft Internet Explorer free of charge,
visit the following Microsoft Web site:

   http://www.microsoft.com/windows/ie/download/default.htm
   (http://www.microsoft.com/windows/ie/download/default.htm)
