TITLE   :WD97: Word Hangs When Started on Compaq Armada M300 or M700
PRODUCT :Word 97 for Windows
PROD/VER::
OPER/SYS:
KEYWORDS:kbdta

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows, used with:
    - the operating system: Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start Word, either from a shortcut or by double-clicking a document
name in Windows Explorer, Word stops responding (hangs). You must restart the
computer.

CAUSE
=====

This problem occurs on Compaq Armada M300 and M700 notebook computers that are
running Windows 95 OSR-2.1 with the ATI Rage Mobility Driver.


WORKAROUND
==========

To work around this problem, browse to the following Compaq Web site for an
updated driver:

   http://www5.compaq.com/support/files/notebooks/us/locate/3_1009.html#Display
