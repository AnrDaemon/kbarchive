TITLE   :FS98/2000: Delete Button Unavailable in Select Flight Dialog Box
PRODUCT :Microsoft Home Games
PROD/VER:WINDOWS:
OPER/SYS:
KEYWORDS:kbui fs98 kbimu msgame

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Flight Simulator 2000 
 - Microsoft Flight Simulator 2000 Professional Edition 
 - Microsoft Flight Simulator 98 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you open the Select Flight dialog box in Microsoft Flight Simulator 98 or
Microsoft Flight Simulator 2000 and then click a flight, the Delete button may
be unavailable.

CAUSE
=====

This behavior can occur if you select a built-in Flight Simulator flight or a
custom flight that is currently in use.

RESOLUTION
==========

To delete a custom flight in Flight Simulator 98 or Flight Simulator 2000:

1. If you are currently using the custom flight you want to delete, quit the
   custom flight, and then select a built-in Flight Simulator flight.

2. On the Flights menu, click Select Flight.

3. In the Available Flights box, click the custom flight you want to delete.

4. Click Delete, click Yes, and then click OK.
