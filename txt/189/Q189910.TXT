TITLE   :INFO: Samples File Does Not Install with Typical MSDN Install
PRODUCT :Microsoft FoxPro
PROD/VER:WINDOWS:6.0
OPER/SYS:
KEYWORDS:kbvfp600 kbDSupport kbHTMLHelp130

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Visual FoxPro for Windows, version 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

When doing a 'Typical' installation of MSDN 6.0 Visual Studio Edition, the
samples file will not be installed.

MORE INFORMATION
================

Once the Typical MSDN 6.0 installation is completed, try to Run/Open any of the
samples from the Visual FoxPro Samples help page will generate the following
error:

   File not found
   Please make sure the following file is installed
   C:\Microsoft Visual Studio\MSDN98........<sample filename>

To Run the samples file, you can either do the Custom installation at the
beginning of the MSDN 6.0 installation, or the Maintenance Setup to install the
samples file.
