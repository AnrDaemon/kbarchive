TITLE   :MCM2: Cannot Switch Game Controller During Game Play
PRODUCT :Microsoft Home Games
PROD/VER::2.0
OPER/SYS:
KEYWORDS:kbhw kbimu kbHardware

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Motocross Madness 2, version 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you play Microsoft Motocross Madness 2, you may experience one of the
following symptoms if you switch game controllers during game play:

 - The game stops responding (hangs).

 - The new game controller is not detected correctly.

CAUSE
=====

This behavior can occur because Motocross Madness 2 requires you to install the
primary game controller before you start the game.

RESOLUTION
==========

To resolve this issue, quit Motocross Madness 2, install the game controller
that you want to use, and then restart Motocross Madness 2 again.

Additional query words: 2.00 mcm msgame joystick usb unplug replug gamepad lock crash freeze disconnect
