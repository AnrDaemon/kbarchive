DOCUMENT:Q249307  12-NOV-2001  [homegame]
TITLE   :Mac Arcade Error Message: The Finder Needs Your Attention
PRODUCT :Microsoft Home Games
PROD/VER:MACINTOSH:1.0
OPER/SYS:
KEYWORDS:kbenv kberrmsg kbsetup kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Arcade for the Macintosh, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to install Microsoft Arcade for the Macintosh, you may receive
the following message:

   The Finder needs your attention. Please select the Finder from the
   Application menu.

When you click Finder on the Application menu, you receive the following error
message:

   The application "Microsoft Arcade Setup" has unexpectedly quit, possibly
   because the server that contains it has been disconnected.

CAUSE
=====

This behavior can occur if you attempt to install Microsoft Arcade for the
Macintosh on a computer running Mac OS System 9.

RESOLUTION
==========

To work around this issue, do not click Finder when the first message appears.
If you dismiss this message, you do not receive the second message and are able
to complete the installation of Arcade.

MORE INFORMATION
================

This issue does not occur if you install Arcade on a computer running Mac OS
System 8.6 or earlier.
