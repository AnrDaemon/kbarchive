DOCUMENT:Q249111  11-NOV-2000  [exchange]
TITLE   :XFOR: Exchange Connector for Lotus cc:Mail Not Supporting Korean
PRODUCT :Microsoft Exchange
PROD/VER:winnt:5.5
OPER/SYS:
KEYWORDS:exc55 kbExchange550preSP4fix kbExchange550sp4Fix

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------


SYMPTOMS
========

If you use the following scenario, when you send e-mail messages in either
direction, the messages are garbled:

   Korean Microsoft Outlook and Korean operating system --- U.S. Exchange Server
   and U.S. Microsoft Windows NT Server (Connector for cc:Mail) --- Korean
   cc:Mail post office --- U.S. cc:Mail client and Korean operating system

CAUSE
=====

This issue can occur because the Microsoft Exchange Connector for Lotus cc:Mail
does not support the Korean language.

RESOLUTION
==========

To resolve this problem, obtain the latest service pack for Exchange Server 5.5.
For additional information, please see the following article in the Microsoft
Knowledge Base:

   Q191014 XGEN: How to Obtain the latest Exchange Server 5.5 Service Pack


STATUS
======

Microsoft has confirmed this to be a problem in Exchange Server version 5.5.
This problem was first corrected in Exchange Server 5.5 Service Pack 4.
