TITLE   :Future Domain Disk Maestro Version 6.0 with MS-DOS

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft MS-DOS operating system versions 5.0, 5.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

According to Future Domain, to install MS-DOS 5.0 on partitions created by the
Future Domain Disk Maestro disk partitioning software, version 6.0 of Disk
Maestro must be used. Disk Maestro is available through local distributors and
retailers.

The name of the driver for Disk Maestro is FDDEVICE.SYS.

MORE INFORMATION
================

Future Domain recommends the following procedure to upgrade to MS-DOS 5.0:

1. Install the MS-DOS 5.0 upgrade onto the computer containing the current
   version of Disk Maestro.

2. Back up all information contained on the hard disk.

3. Reformat and repartition the hard disk per the Disk Maestro 6.0 software
   documentation.

4. Restore all data to the hard disk.
