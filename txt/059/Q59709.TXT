DOCUMENT:Q59709  14-DEC-1999  [win95x]
TITLE   :Cutting Large Image in Real Mode Windows 3.0 Paintbrush
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you run Paintbrush under Microsoft Windows in real mode and attempt to Cut an
image that is larger than the available memory, an error message displays
explaining that you don't have enough memory. When you choose the OK button, the
area that you had selected to be cut is blank.

WORKAROUND
==========

Press ALT+TAB to switch from Paintbrush and then return to Paintbrush. The image
is restored.

STATUS
======

Microsoft has confirmed this to be a problem in version 3.0. This problem does
not occur in later versions of Windows.
