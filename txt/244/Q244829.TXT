DOCUMENT:Q244829  07-MAY-2002  [homegame]
TITLE   :FS2000: Adventures and Lessons Are Not Available
PRODUCT :Microsoft Home Games
PROD/VER:WINDOWS:
OPER/SYS:
KEYWORDS:kbsetup kbimu msgame

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Flight Simulator 2000 
 - Microsoft Flight Simulator 2000 Professional Edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start Microsoft Flight Simulator 2000, Lessons and Adventures may not
be available.

CAUSE
=====

This behavior can occur if you select the Compact installation option or if you
select the Custom installation option and then click to clear the Install
Adventures check box when you install the program.

RESOLUTION
==========

To resolve this issue, remove and reinstall Flight Simulator 2000 using the
Typical installation option.

NOTE: Although you can reinstall Flight Simulator 2000 using the Custom
installation method, the Install Adventures check box must remain selected.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.

MORE INFORMATION
================

You cannot add converted Lessons and Adventures or Microsoft Flight Simulator 98
Lessons and Adventures to a Compact installation of Flight Simulator 2000 or to
a Custom installation of Flight Simulator 2000 if the Install Adventures check
box is cleared.
