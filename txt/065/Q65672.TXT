TITLE   :Random Colored Lines in Output from HP PaintJet XL
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------


SYMPTOMS
========

Output created from a Hewlett-Packard (HP PaintJet XL version 1.0) in Microsoft
Windows version 3.0 may have random colored lines running across the page.

RESOLUTION
==========

To resolve this problem, verify that the "Compression" and "High-Speed LPT"
options are disabled in the printer options of Windows as follows:

1. From the Main group of Program Manager, select the Control Panel icon.

2. From the Control Panel, select the Printer icon.

3. Select the HP PaintJet Series as the printer.

4. Select Configure.

5. Select Setup.

6. Select Options.

7. Under Output Options, remove the Xs from the boxes in front of "Compression"
   and "High-Speed Direct LPT".

8. Choose OK for each window until you are returned to the Control Panel.

There is a new version of this driver provided by Hewlett Packard which may
require slightly different steps than are outlined above. The new version number
as of 3/10/92 is 1.2. If you require assistance with this version of the driver,
please contact Hewlett Packard technical support.
