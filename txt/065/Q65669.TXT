TITLE   :Microsoft Word's Use of ALT+SPACEBAR Under Windows
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a,3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

The ALT+SPACEBAR key sequence is used to format characters for normal
text in Microsoft Word for MS-DOS version 5.0. When you run Word 5.0
under Microsoft Windows in 386 enhanced mode, the ALT+SPACEBAR key
sequence puts Word in a window.

To enable Word's use of ALT+SPACEBAR, do the following:

1. Highlight the range of text to be changed.

2. Press ALT+X, then release both the ALT key and the X key.

3. The message "Enter second character of key code..." will be displayed.

4. Press the SPACEBAR key.
