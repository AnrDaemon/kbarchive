TITLE   :Mac Gty: Err Msg: Account "##" Already Contains "Name"
PRODUCT :Microsoft Mail For Appletalk Networks
PROD/VER:WINDOWS:2.0,2.0a,2.0b,3.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

When the Network Manager adds a Microsoft Mail version 2.0 or 3.0 user to a
gateway account who is already a user on another gateway account, an error
message will appear that states "Account '<Account name>' already contains
'<User name>'."

A Mail user can have access to only one gateway account per gateway at a time.

Additional query words: 2.00 2.00a 2.00b 3.00
