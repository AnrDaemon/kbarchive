TITLE   :How to Obtain 128-Bit Encryption for Windows NT Service Packs
PRODUCT :Microsoft Windows NT
PROD/VER::4.0,95
OPER/SYS:
KEYWORDS:kburl win95 win98

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 4.0 
 - Microsoft Windows NT Server version 4.0 
 - Microsoft Windows 98 
 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to obtain 128-bit encryption for Windows NT service
packs and networking clients.

MORE INFORMATION
================

You can download 128-bit versions of service packs or networking updates from
the following Microsoft Web site:

   http://www.microsoft.com/ntserver/

After you read the license agreement that is displayed on this page, click I
Agree to accept and acknowledge that you have read and understand the agreement.
You can then choose to download a service pack or an update package.

To download the 128-bit version of the latest service pack, click the appropriate
link at the top of the page. To download 128-bit hotfixes and updates, click any
of the following items in the list, and then click Download Now:

 - Windows NT 4.0 Post-SP3 Hotfix: Updated SChannel.dll -x86

 - Windows NT 4.0 Post-SP3 Hotfix: Updated SChannel.dll -Alpha

 - PPTP Performance and Security Upgrade for Windows NT 4.0 -x86

 - PPTP Performance and Security Upgrade for Windows NT 4.0 -Alpha

 - PPTP Performance and Security Upgrade Symbols for Windows NT 4.0 -x86

 - PPTP Performance and Security Upgrade Symbols for Windows NT 4.0 -Alpha

 - 128-bit version of the Windows 98 Dial-Up Networking Security Upgrade

 - Windows 95 Dial-Up Networking 1.3 Utility to enable 128-bit encryption

NOTE: The last two items in the list are not included in Windows NT 4.0 Service
Pack 4 (SP4) because they are updates to Microsoft Windows 95 and Microsoft
Windows 98. For all other items in the list, you should install SP4 or later to
correct the problems.
It is also important to note that the latest service packs include additional
fixes and service-related updates, including Microsoft Internet Information
Server (IIS) 4.0 components and Microsoft Internet Explorer components. Please
refer to the Readme.txt file of the latest service pack to obtain a more
complete list of available fixes.
