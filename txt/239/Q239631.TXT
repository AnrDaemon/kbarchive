TITLE   :EWA: Web Page Is Not Opened When You Click Web Link
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:WINDOWS:; :
OPER/SYS:
KEYWORDS:kb3rdparty kberrmsg kburl kbimukbfaq

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta Interactive World Atlas 2000 
 - Microsoft Encarta Interactive World Atlas 2001 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you click a Web link in Microsoft Encarta World Atlas 2000 or Microsoft
Interactive World Atlas 2001, you may experience one or both of the following
symptoms:

 - The Web site is not opened, and the Encarta World Atlas 2000 page that
   contains the Web link remains displayed on the screen.

 - You receive one of the following error messages:

       - Cannot find server.

       - There is no browser installed on your computer.

CAUSE
=====

This behavior can occur if both of the following conditions are true:

 - You run Encarta World Atlas 2000 on a Microsoft Windows 95 or Microsoft NT
   4.0-based computer.

 - The default browser on your computer is any version of Netscape Navigator.

RESOLUTION
==========

To resolve this issue in Encart Interactive World Atlas 2001, click Options,
click Settings, and then click to clear the "Browse WebLinks inside Encarta
Interactive World Atlas" check box.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.
