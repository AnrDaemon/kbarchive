TITLE   :Links 2001: "No 3D Acceleration" Message When Starting Links
PRODUCT :Microsoft Home Games
PROD/VER::
OPER/SYS:
KEYWORDS:kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Links 2001 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to start Microsoft Links 2001, a black Links 2001 window
appears, you receive the following message

   No 3D Acceleration

and then Links 2001 stops responding (hangs).

RESOLUTION
==========

An update is available to resolve this issue. Browse to the following Microsoft
Web site to download the Links 2001 Update v1.21 (11.3MB):

   http://www.microsoft.com/games/home/games.asp?g=127#support

MORE INFORMATION
================

For additional information about the "No 3D Acceleration" message, click the
article number below to view the article in the Microsoft Knowledge Base:

   Q279768 Links 2001: No 3D Acceleration Message Is Displayed on the Screen

Additional query words: msgame
