SUMMARY
=======

The Passport Manager Administration utility is a graphical-interface alternative
to editing the registry to change Passport Manager object settings.

MORE INFORMATION
================

The most common use of the Passport Manager Administration utility is to change
the defaults that Passport Manager methods use if certain optional parameters
are omitted when the method is called. For details about how defaults can be set
and affect the behavior of Passport Manager methods, see the following Passport
SDK Web site:

   Setting Passport Manager Defaults
   http://www.passport.com/sdkdocuments/sdk14/reference/operations/Passport_Admin_init.htm
