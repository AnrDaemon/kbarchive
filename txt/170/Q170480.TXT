TITLE   :XWEB: Delete Check Box Not Lined Up with Field Header Using OWA
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:5.5
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Outlook Web Access, version 5.5 
 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------


SYMPTOMS
========

When running Microsoft Exchange Server version 5.5 and using the Outlook Web
Access component, the Delete check boxes for messages do not line up exactly
with the header for the Delete check box.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server
version 5.5 for Windows NT.
