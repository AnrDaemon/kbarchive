TITLE   :Windows NT 4.0 RAS Support for 3Com Impact ISDN Modems
PRODUCT :Microsoft Windows NT
PROD/VER:WinNT:4.0
OPER/SYS:
KEYWORDS:kb3rdparty kbinterop kbnetwork

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0 
 - Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

3Com manufacturers the Impact ISDN modem, which is not listed in the current
Windows NT Hardware Compatibility List (HCL). However, 3Com provides software
that allows you to use the ISDN modem with Windows NT 4.0 Remote Access Service
(RAS).

For information about non-HCL hardware support, see the following article in the
Microsoft Knowledge Base:

   Q142865 Microsoft Support Policy on Hardware Not On Windows NT HCL

MORE INFORMATION
================

You can use the following 3Com ISDN products with Windows NT 4.0:

 - 3C871

 - 3C872

 - 3C882

To use these products with RAS, follow the directions in the manual shipped with
the 3Com Impact modem, or obtain the directions from 3Com's Web site:

   http://infodeli.3com.com/infodeli/tools/isdn/spt-tips/install/winnt40.htm

Additional support tips for the External 3Com Impact ISDN Modem are available on
3Com's Web site at the following address:

   http://infodeli.3com.com/infodeli/tools/isdn/spt-tips/impct-x.htm
