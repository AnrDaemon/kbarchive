The information in this article applies to:

 - Microsoft Visual FoxPro for Windows, versions 5.0, 5.0a, 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

An update to FOXISAPI has been posted to the Product Updates page on the
Microsoft Visual FoxPro Web site.

To access this file (named "FOXISAPI.EXE") directly, navigate to the following
URL in your Web browser:

   http://msdn.Microsoft.com/vfoxpro/downloads/foxisapi.exe

MORE INFORMATION
================

This update includes version 6.0.8451.0 of the FOXISAPI.DLL and the C++ source
code used to build it. The update contains the features added to the Visual
Studio Service Pack 3 version, as well as a fix for a binary data transfer
issue.

NOTE: Visual FoxPro 7.0 ships with version 4.0.993.0 of FOXISAPI.DLL. Despite the
version number, this is a later version than the one posted on the Web. (The
creation date is June 14, 2001.) (c) Microsoft Corporation 1999, All Rights
