TITLE   :Asheron's Call Err Msg: Failed to Execute
PRODUCT :Microsoft Home Games
PROD/VER::1.0
OPER/SYS:
KEYWORDS:kberrmsg kbimu msgame

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Asheron's Call, version 1.0 
 - Microsoft Asheron's Call: Dark Majesty, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you enter an Asheron's Call game room on Zone.com, you may receive the
following error message:

   Failed to execute.

After this error message is displayed, the Zone downloads and installs the
ACD3DSetup.exe file.

CAUSE
=====

This behavior can occur if Microsoft DirectDraw is not properly installed on the
computer.

RESOLUTION
==========

To resolve this issue, reinstall Microsoft DirectX. The latest version of
DirectX is available from the following Microsoft Web site:

   http://www.microsoft.com/directx/homeuser/downloads/default.asp

For more information about how to obtain the latest version of DirectX, please
see the following article in the Microsoft Knowledge Base:

   Q179113 How to Download and Install DirectX
