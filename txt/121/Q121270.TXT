TITLE   :WGAO Error 2116: The Device or Directory Does Not Exist
PRODUCT :Microsoft Access Distribution Kit
PROD/VER:MS-DOS:3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Workgroup Add-On for MS-DOS, version 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you install Workgroup Add-On for MS-DOS version 3.11 and then try to share
a directory, you receive the following error message:

   ERROR 2116: The Device or Directory Does Not Exist.

CAUSE
=====

This problem occurs when the directory you are trying to share is hidden.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Workgroup Add-On for
MS-DOS, version 3.11. We will post new information here in the Microsoft
Knowledge Base as it becomes available.

Additional query words: wfwg kbbuglist subdirectory attrib attribute NET SHARE
