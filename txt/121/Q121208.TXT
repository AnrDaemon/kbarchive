SUMMARY
=======

SNDPLAY.EXE is a self-extracting file that contains all files necessary to build
the SNDPLAY program that demonstrates a simple application that plays a sound
file specified on the command line. It displays an error message if no sound
file was specified in the command line or the specified sound could not be
played, and immediately exits whether the sound is played or not.
