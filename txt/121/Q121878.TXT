TITLE   :RAS Doesn't Support NetBIOS Chain Sends Greater than 64KB - 1
PRODUCT :Microsoft Windows NT
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
3.5

WINDOWS

kbnetwork

-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.5 
 - Microsoft Windows NT Server version 3.5 
-------------------------------------------------------------------------------

Windows NT 3.5 Remote Access Services (RAS) does not support NetBIOS chain
sends of greater than 65534 bytes (64KB - 1). Very few applications require
larger chain sends.
