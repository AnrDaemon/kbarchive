-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to remove entries from the Find list in Microsoft
Streets and Trips 2001.

MORE INFORMATION
================

To remove entries from the Find list, download and install the MruCln2.exe file
from the Microsoft Download Center.

NOTE: You must quit the program before you install the MruCln2.exe file.

The following file is available for download from the Microsoft Download Center:

   MruCln2.exe
   (http://download.microsoft.com/download/streetstrips2000/Utility/1.0/W9X2K/EN-US/MruCln2.exe)

Release Date: Jul-19-2000

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

   Q119591 How to Obtain Microsoft Support Files from Online Services
