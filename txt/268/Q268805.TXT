TITLE   :AoE II Expansion: Computer Hangs During Multiplayer Game
PRODUCT :Microsoft Home Games
PROD/VER::1.0
OPER/SYS:
KEYWORDS:kbenv aoe kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Age of Empires II Expansion: The Conquerors, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you play a multiplayer game of Microsoft Age of Empires II Expansion: The
Conquerors on Zone.com or a local area network (LAN), one or more computers may
temporarily stop responding (hang) and then resume normal game play.

CAUSE
=====

This behavior can occur if Microsoft Systems Management Server (SMS) is running
on any one of the computers that are connected to the game.

RESOLUTION
==========

To resolve the issue, shut down SMS services before you start a multiplayer game
of Microsoft Age of Empires II Expansion: The Conquerors.

MORE INFORMATION
================

For additional information about Microsoft Systems Management Server (SMS),
visit the following Microsoft Web site:

   http://www.microsoft.com/smsmgmt/default.asp
