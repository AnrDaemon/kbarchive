TITLE   :Plus! Game Pack: Error Message When You Start Pandora's Box
PRODUCT :Microsoft Home Games
PROD/VER::
OPER/SYS:
KEYWORDS:kbenv kberrmsg kbimu msgame

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Plus! Game Pack: Cards and Puzzles 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start the trial version of Microsoft Pandora's Box that is included in
the Microsoft Plus! Game Pack: Cards & Puzzles, you may receive the
following error message:

   ComCtl32.dll:InitComControlsEx

CAUSE
=====

This behavior occurs if you install the trial version of Pandora's Box on a
computer that is running the retail version of Microsoft Windows 95.

RESOLUTION
==========

To resolve this issue, install Microsoft Windows 98 on the computer or the
latest version of Microsoft Internet Explorer.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.
