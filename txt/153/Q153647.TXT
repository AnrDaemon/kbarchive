TITLE   :SMS: Shared Applications Fail Using Specific Drive Letter
PRODUCT :Microsoft Systems Management Server
PROD/VER:winnt:1.1
OPER/SYS:
KEYWORDS:kbnetwork smsgeneral kbArtTypeINF

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Systems Management Server version 1.1 
-------------------------------------------------------------------------------

SYMPTOMS
========


If a shared application is set to "Require specific drive letter" and that drive
letter or other drive letters toward the end of the alphabet are not available
or already in use, then the application fails to start and a network error
occurs.


STATUS
======

Microsoft has confirmed this to be a problem in Systems Management Server
version 1.1. This problem has been corrected in Systems Management Server
version 1.2.

Additional query words: prodsms drive letter shared applications
