TITLE   :V-Chat Folders Created in Transferred Files Folder
PRODUCT :The Microsoft Network
PROD/VER:WINDOWS:1.3,2.0,2.5,95
OPER/SYS:
KEYWORDS:kbmsn

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - The Microsoft Network versions 1.3, 2.0, 2.5 
-------------------------------------------------------------------------------

SUMMARY
=======

After you run Setup for V-Chat, the Program Files\The Microsoft
Network\Transferred Files folder may contain folders with the names files such
as the following: Vchat.000, Vchat.001, Vchat.002 and so on.

MORE INFORMATION
================

These folders were needed during V-Chat Setup, but were not deleted by the Setup
program. These folders can now be safely deleted.

Additional query words: msn 1.30 2.00 2.50
