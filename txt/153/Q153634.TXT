DOCUMENT:Q153634  31-JUL-2001  [sms]
TITLE   :Too Many NAKs" Error When Remote Controlling a NetWare Client
PRODUCT :Microsoft Systems Management Server
PROD/VER:winnt:1.0,1.1
OPER/SYS:
KEYWORDS:kbnetwork kbtshoot smsremtshoot kbRemoteProg

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Systems Management Server versions 1.0, 1.1 
-------------------------------------------------------------------------------

SYMPTOMS
========


When you remote control a NetWare client computer, the following error message
appears on the Remote Control agent:

   Too many NAKs - terminating

When you click OK on the client computer, the Systems Management Server's Quick
Viewer window appears white in color with the following error message:

   The remote computer has entered a video mode that is only supported by the
   Remote Control Agent. The Remote Control Agent is not currently running on
   the Remote Computer.

STATUS
======

Microsoft has confirmed this to be a problem in Systems Management Server
versions 1.0 and 1.1. This problem has been corrected in Systems Management
Server version 1.2

Additional query words: prodsms naks remote control
