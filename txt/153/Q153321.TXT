TITLE   :Network Monitor Err Msg: Parser DLL Is Not Valid Message
PRODUCT :Microsoft Systems Management Server
PROD/VER:winnt:1.0,1.1
OPER/SYS:
KEYWORDS:kbtshoot smsremtshoot kbRemoteProgkbfixlist

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Systems Management Server versions 1.0, 1.1 
-------------------------------------------------------------------------------

SYMPTOMS
========


When stopping a capture in Network Monitor while a Remote Control session is
going, the Network Monitor gives the following error message:

   Parser DLL is not valid message


STATUS
======

Microsoft has confirmed this to be a problem in Systems Management Server
version 1.1. This problem has been corrected in Systems Management Server
version 1.2.

Additional query words: netmon prodsms
