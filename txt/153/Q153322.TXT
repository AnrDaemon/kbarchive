TITLE   :Remote Control GPF Using Adobe ATM Fonts
PRODUCT :Microsoft Systems Management Server
PROD/VER:winnt:1.0,1.1
OPER/SYS:
KEYWORDS:kbtshoot smsremtshoot kbRemoteProgkbfixlist

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Systems Management Server versions 1.0, 1.1 
-------------------------------------------------------------------------------

SYMPTOMS
========


When you use the Adobe Type Manager 3.01 to change the screen fonts, Remote
Control causes a general protection fault (GPF).


STATUS
======

The Adobe Type Manager 3.01 is manufactured by Adobe Systems, Inc., a vendor
independent of Microsoft; we make no warranty, implied or otherwise, regarding
this product's performance.

Microsoft has confirmed this to be a problem in Systems Management Server version
1.1. This problem has been corrected in Systems Management Server version 1.2.
