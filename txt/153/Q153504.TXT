DOCUMENT:Q153504  09-AUG-2001  [winnt]
TITLE   :PCL5EMS Does Not Support All Memory Available on HP LJ 5Si MX
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:3.5,3.51
OPER/SYS:
KEYWORDS:kbnetwork

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation versions 3.5, 3.51 
 - Microsoft Windows NT Server versions 3.5, 3.51 
-------------------------------------------------------------------------------


SYMPTOMS
========

The PCL5EMS.DLL printer driver does not support all of the memory that is
installable for the Hewlett-Packard (HP) LaserJet 5SI MX printer. The dialog box
of the printer driver limits configurable memory to 34,816 bytes.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT versions 3.5 and
3.51. This problem was corrected in the latest Windows NT 3.51 U.S. Service
Pack. For information on obtaining the Service Pack, query on the following word
in the Microsoft Knowledge Base (without the spaces):

   S E R V P A C K


Additional query words: prodnt
