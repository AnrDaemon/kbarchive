TITLE   :Disconnecting from MSN Does Not Reset Your Modem to Answer Mode
PRODUCT :The Microsoft Network
PROD/VER:WINDOWS:2.5,2.51,2.52,2.6,95
OPER/SYS:
KEYWORDS:kbtool win95 kbmsn

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - The Microsoft Network versions 2.5, 2.51, 2.52, 2.6 
 - the operating system: Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you disconnect from MSN, The Microsoft Network, your modem may not answer
incoming calls although you configured it to do so.

CAUSE
=====

This behavior can occur if you are running Microsoft Windows 95 retail release
(version 4.00.950), Windows 95 OEM Service Release 1 (OSR1) or Windows 95
Service Pack 1 (version 4.00.950A), or Windows 95 OEM Service Release 2 (version
4.00.950B) and your modem is not reset correctly when you disconnect from MSN.

RESOLUTION
==========

To resolve this behavior, install Dial-Up Networking 1.3. To do so, follow these
steps:

1. Double-click the Internet icon on your desktop.

2. In the Address box, type the following Internet address, and then press
   ENTER: http://www.microsoft.com/windows/getisdn/dload.htm

3. Follow the instructions on the screen to download and install Dial-Up
   Networking 1.3.

4. Quit Internet Explorer.

For more information about the Dial-Up Networking 1.3 upgrade, please see the
following article in the Microsoft Knowledge Base:

   ARTICLE-ID: <LINK TYPE="ARTICLE" VALUE="Q191494">Q191494</LINK>

   TITLE     : Dial-Up Networking 1.3 Upgrade Available

Additional query words: msnet msnetwork microsoft-net m.s.n. dun 2.50 2.60
