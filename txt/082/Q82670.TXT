TITLE   :Error Creating an Invalid Directory from File Manager
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

Under the Microsoft Windows operating system version 3.1, the wrong error
message is displayed when you attempt to create a directory with an invalid name
from File Manager. File Manager will incorrectly return the following error
message if the directory name contains an illegal character:

   Cannot find path.
   Please verify that the correct path is given.

For a list of valid file and directory names, please see pages 69-70 of the
"MS-DOS User's Guide and Reference" for version 5.0.

STATUS
======

Microsoft has confirmed this to be a problem in Windows versions 3.1 and 3.11.
We are researching this problem and will post new information here as it becomes
available.
