TITLE   :Linked Excel Charts Lose Patterns When Printed in Write
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

Printing Microsoft Write documents with OLE-linked Microsoft Excel charts causes
patterns and gray shades to be lost on the printout.

STATUS
======

Microsoft has confirmed this to be a problem in the Write program included with
Microsoft Windows 3.1. We are researching this problem and will post new
information here as it becomes available.

Additional query words: 3.10 3.11
