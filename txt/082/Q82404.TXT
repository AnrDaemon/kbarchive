TITLE   :Rumba 3.1.03 FastPath and Windows Screen Saver
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

There is an option in Rumba 3.1.03 FastPath called Configure and Connect. If
selected, the program moves the cursor around the terminal screen to pull up
configuration options. If the Microsoft Windows version 3.1 screen saver is
active, this action causes the screen saver to activate.

WORKAROUND
==========

The workaround is to turn off the screen saver by selecting the Desktop icon in
the Control Panel.
