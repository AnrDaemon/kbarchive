TITLE   :Desktop Can't Print from Directory with an Extension
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a,3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

When using the Norton Desktop for Windows as the shell under the Microsoft
Windows, problems printing files may occur. If you print a file from a directory
with an extension, an error message will display stating that the file cannot be
found.

product's performance or reliability.
