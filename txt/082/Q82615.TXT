TITLE   :Version Conflict Message When Installing Windows 3.1
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

When upgrading over the Microsoft Windows operating system versions 3.x, you may
receive a "version conflict" message for the following two files:

   OLECLI.DLL
   OLESVR.DLL

You should choose to install the versions found on the Microsoft Windows 3.1
disks.
