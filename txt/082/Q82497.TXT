TITLE   :Epson Clone Printers Print Unreadable Characters
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------


SYMPTOMS
========

Some Epson clone printers may print unreadable characters or garbled text. For
example, when printing with the Epson Homewriter 10 under the Microsoft Windows
with the model type set to an Epson FX 86e, unreadable characters are printed at
the start of each line.

WORKAROUND
==========

If this problem is occurring, in addition to checking the printer DIP- switch
settings, set the printer model type to an Epson FX 80 or Epson FX 85.

STATUS
======

Microsoft is researching this problem and will post new information here as it
becomes available.
