TITLE   :WingZ Help During Install Does Not Respond
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

While installing Informix's WingZ into Microsoft Windows operating system
version 3.1, clicking the Help button in any of the install dialog boxes
generates Windows Help with the message "cannot open help file."

There is no solution to this problem. In Windows 3.0, this brings up the Windows
Help with the appropriate information.
