TITLE   :Finale GP Faults with Right Mouse Button Click
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

When Finale version 2.0 is running, if you select an empty tool and click the
right mouse button, Microsoft Windows operating system version 3.1 generates a
general protection (GP) fault message.

The same behavior also occurs with Windows version 3.0.

Finale is manufactured by a vendor independent of Microsoft; we make no warranty,
implied or otherwise, regarding this product's performance or reliability.

Additional query words: 3.10 3.11 3.00 3.00a 3.0 3.0a 2.00 2.0
