DOCUMENT:Q184710  03-JUN-1999  [exchange]
TITLE   :XADM: Exchange Admin Program Will Not Run On Windows 95/98
PRODUCT :Microsoft Exchange
PROD/VER:winnt:4.0,5.0,5.5
OPER/SYS:
KEYWORDS:exc4 exc5 exc55

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, versions 4.0, 5.0, 5.5 
-------------------------------------------------------------------------------

SUMMARY
=======

The Microsoft Exchange Server Administrator program will not install or run on
Windows 95/98.

MORE INFORMATION
================

The Exchange Server Administrator program requires Windows NT workstation or
Windows NT Server.

Additional query words: Win95 OSR2
