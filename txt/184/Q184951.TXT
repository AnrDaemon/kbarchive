TITLE   :XCLN: Multiple Lines for Installation Description Truncated
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:8.5
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Outlook 98 Deployment Kit 
 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

Stage 3 of the Outlook Deployment Kit Wizard allows you to name your
installation and type multiple lines of descriptive text. When you run the
installation, the installation description will only show the first line of text
that was typed. For example:

   "Descriptive line of text

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Outlook 98 Deployment
Kit.


Additional query words: missing gone close
