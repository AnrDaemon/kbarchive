TITLE   :NT40 Enterprise Training: W95brows.cap and Ntwbrows.cap Missing
PRODUCT :Microsoft Press
PROD/VER::
OPER/SYS:
KEYWORDS:kbfile kbdocerr

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - MSPRESS Microsoft Windows NT Server 4.0 Enterprise Technologies Training ISBN 1-57231-710-8 
 - MSPRESS Microsoft Windows NT Server 4.0 Enterprise Technologies Training Kit, Deluxe Multimedia Edition ISBN 1-57231-829-5 
 - MSPRESS Microsoft Certified Systems Engineer Core Requirements Training Kit ISBN 1-57231-905-4 
-------------------------------------------------------------------------------

SYMPTOMS
========

W95brows.cap, first mentioned on page 399, and Ntwbrows.cap, first mentioned on
page 403, are missing from the companion disc in the first printing of the books
listed above.

WORKAROUND
==========

Microsoft Press has created a patch file that includes these two files. The
following file is available for download from the Microsoft Download Center:

   Ntetkfix.exe
   (http://download.microsoft.com/download/winntent/MSPRESS/2.0/NT4ALPHA/EN-US/Ntetkfix.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

   Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.


STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article.
