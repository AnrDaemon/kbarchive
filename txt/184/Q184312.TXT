TITLE   :Microsoft Management Console Fails to Function Correctly
PRODUCT :Internet Information Server
PROD/VER:WINNT:4.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Internet Information Server 4.0 
 - Microsoft Management Console 
-------------------------------------------------------------------------------

SUMMARY
=======

 

SYMPTOMS
========

When using the Microsoft Management Console (MMC) included with Microsoft
Internet Information Server 4.0 and the Windows NT Option Pack, you may
experience problems using the console if your available disk space is less than
2 MB on the system partition.

WORKAROUND
==========

To restore normal functionality of the MMC, you must free up additional disk
space on the system partition where Windows NT is installed.

Additional query words: IIS NTOP 4.0 MMC
