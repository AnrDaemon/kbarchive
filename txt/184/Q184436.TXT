DOCUMENT:Q184436  13-OCT-1999  [msnetwork]
TITLE   :Removal of Connection Manager Is Not Confirmed
PRODUCT :The Microsoft Network
PROD/VER:WINDOWS:2.5
OPER/SYS:
KEYWORDS:kbsetup kbmsn

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - The Microsoft Network version 2.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you remove Connection Manager using the Add/Remove Programs tool in Control
Panel, the removal of Connection Manager is not confirmed.

STATUS
======

Microsoft has confirmed this to be a problem in MSN, The Microsoft Network,
version 2.5.

Additional query words: msnet msnetwork microsoft-net m.s.n.
