TITLE   :Taskbar Disappears When Changing Screen Resolution on Alpha
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0, Terminal Server Edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

The taskbar disappears when you change screen resolution on Alpha platforms
running Windows NT Server version 4.0, Terminal Server Edition.


RESOLUTION
==========

To work around the problem:

1. Press CTRL+ESC and choose Settings, Taskbar.

2. Click Auto Hide.

3. Click Apply.

4. Click Auto Hide.

5. Click OK.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 4.0, Terminal
Server Edition running on Alpha platform.
