TITLE   :XWEB: Comic Sans Font in Meeting Requests
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:5.5
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Outlook Web Access, version 5.5 
 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SYMPTOMS
========


When you use the Microsoft Outlook Web Access client, it may be difficult to tell
which tab is active in a meeting request if the browser font is set to Comic
Sans.

WORKAROUND
==========

To work around this problem, use a different font other than Comic Sans.
