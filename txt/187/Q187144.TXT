TITLE   :Options Set in Outlook Are Not Used from the Send To Menu
PRODUCT :Microsoft Windows NT
PROD/VER:WinNT:4.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0, Terminal Server Edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you create a mail message using the Mail Recipient option from the Send To
menu, Outlook does not use the options that you may have selected within
Outlook's option menu.

STATUS
======

This behavior is by design.
