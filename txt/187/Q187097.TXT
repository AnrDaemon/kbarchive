TITLE   :Chat and Clipbook Viewer Produce NetDDE Errors
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0, Terminal Server Edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use Chat or Clipbook Viewer to establish a Network Dynamic Data
Exchange (NetDDE) connection between a Terminal Server and a Windows NT 4.0
Service Pack 3 computer, you may receive the following error messages:

   The other computer did not respond.

   There is a problem with the connection to the other computer. Try reconnecting
   to the computer that contains the Clipbook page. (0x4000)

STATUS
======

This behavior is by design.
