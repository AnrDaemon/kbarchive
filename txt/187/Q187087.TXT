TITLE   :Hardware Encryption Devices Incompatible with Terminal Server
PRODUCT :Microsoft Windows NT
PROD/VER:WINDOWS:4.0
OPER/SYS:
KEYWORDS:kb3rdparty kberrmsg kbhw kbHardware

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0, Terminal Server Edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to use a hardware-based encryption device to provide licensing
or security features to enhance a third-party program on Windows NT Server 4.0,
Terminal Server Edition, you receive the following error message:

   invalid key = missing module

MORE INFORMATION
================

Hardware devices providing encryption or security services to manage and control
software do not work correctly on Terminal Server and are not supported.
