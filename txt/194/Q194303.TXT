TITLE   :New Features Available in Windows NT 4.0 Service Pack 4
PRODUCT :Microsoft Windows NT
PROD/VER:WinNT:4.0
OPER/SYS:
KEYWORDS:kbFEA kbWinNT400sp4fea

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 4.0 SP4 
 - Microsoft Windows NT Server version 4.0 SP4 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists new features available in Windows NT 4.0 Service Pack 4. For
more information on these features, please see the Readme.txt file and the
Microsoft KB article listed. For information on Windows NT service packs and how
to obtain them, please see the Windows NT Service Packs web site:

   http://support.microsoft.com/Support/NTServer/Content/ServicePacks/Default.asp

MORE INFORMATION
================

Windows NT 4.0 Service Pack 4 includes the following new features that are
installed by Update.exe:

 - Routing Information Protocol (RIP) Listener

   For more information, please see the following article in the Microsoft
   Knowledge Base:

   Q172514 Silent RIP for IP Available for Windows NT Workstation

 - Telephony Application Programming Interface (TAPI) 2.1

   For more information, please see the following article in the Microsoft
   Knowledge Base:

   Q186358 How to Enable TAPI 2.1 in Windows NT 4.0 with Service Pack 4

 - Microsoft File and Print Service for NetWare (FPNW) Support for Client32

 - Active Accessibility Support

 - DCOM/HTTP Tunneling

 - Visual Studio-MICS

 - Remote Procedure Calls (RPC) Enhancements for Visual Basic (VB)

 - Remote Winsock (DNS/Port 53)

 - IP Helper API (IPHLPAPI)

   For more information, please see the following article in the Microsoft
   Knowledge Base:

   Q193059 INFO: IP Helper APIs Add Net Config and Stat Info to Win32 Apps

 - InternetGroup Management Protocol (IGMP) version 2

   For more information, please see the following article in the Microsoft
   Knowledge Base:

   Q193907 Windows NT 4.0 SP4 Supports IGMP Version 2

 - Euro Key Patch

   For more information, please see the following article in the Microsoft
   Knowledge Base:

   Q182005 Euro Currency Not Available in Windows NT Character Sets

 - Year 2000 (Y2K) Issues

   For more information, please see the following article in the Microsoft
   Knowledge Base:

   Q175093 User Manager Does Not Recognize February 2000 As a Leap Year

   Q180122 After Changing the Time, Windows NT May Skip a Day

   Q183123 Find Files Displays Garbled Date if Year is 2000 or Greater

   Q183125 Shell Doc Property Dialog Custom Date Incorrect after Year 2000

 - CryptoAPI and Authenticode

 - Profile Quotas (Proquota.exe)

   For more information, please see the following article in the Microsoft
   Knowledge Base:

   Q185561 How to Enable Profile Quotas in Windows NT 4.0

 - Microsoft Routing and Remote Access Service (RRAS)

   For more information, please see the following article in the Microsoft
   Knowledge Base:

   Q189594 RRAS Upgrade for WinNT Server 4.0 Hotfix Pack 3.0 Release Notes

 - PPTP Performance and Security Update

   For more information, please see the following article in the Microsoft
   Knowledge Base:

   Q189595 PPTP Performance & Security Upgrade for WinNT 4.0 Release Notes

 - CHKDSK Enhancements

   For more information, please see the following article in the Microsoft
   Knowledge Base:

   Q191603 Modifying the Autochk.exe Time-out Value

   Q187941 An Explanation of CHKDSK and the New /C and /I Switches

 - New functionality with NBTSTAT.EXE

   For more information, please see the following article in the Microsoft
   Knowledge Base:

   Q188598 Renewing Wins Client Registrations with NBTSTAT -RR

The CD-ROM version of Windows NT 4.0 Service Pack 4 includes the following new
features:

 - Compaq Fibre Storage Driver

 - Web-Based Enterprise Management (WBEM) For more information, please see the
   Service Pack 4 Readme.txt file and the Readme.txt file in the \Wbem folder.

 - Message Queue (MSMQ) for Windows 95 Client

 - Security Configuration Manager (SCM)

   For more information, please see the Service Pack 4 Readme.txt file and the
   Readme.txt file in the \Mssce folder.

 - Windows NT Server NetShow Services

   For more information, please see the Service Pack 4 Readme.txt file and the
   Ntsp4-ns.htm and Ns-readme.htm files in the \NetShow folder.

 - Windows Media Player

   For more information, please see the Service Pack 4 Readme.txt file and the
   Readme.txt file in the \Mplayer2 folder.

 - Site Server Express 3.0 (Upgrade Only)

 - Microsoft Internet Explorer 4.01 Service Pack 1

 - Microsoft Data Access Components 2.0 Service Pack 1

For additional features added to a system with Windows NT 4.0 Service Pack 4,
query in the Knowledge Base on the key word NT4SP4FEA. For bug fixes, query on
the key word NT4SP4FIX.

Additional query words: 4.00 sp4
