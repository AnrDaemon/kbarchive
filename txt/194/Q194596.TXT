======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, versions 4.0, 5.0, 5.5 
 - Microsoft Exchange Windows 95/98 client, version 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Mailbox Converter for Lotus cc:Mail Archives and Microsoft Exchange Server
is a program used to convert Lotus cc:Mail archive files containing e-mail
messages to Exchange Server folders directly from the users' desktops.

The Mailbox Converter administrator runs the Setup program. The administrator may
then perform the archive file conversions, or may provide end-users with the
Mailbox Converter program and documentation so that they can perform the
conversions.

MORE INFORMATION
================

Hardware Requirements
---------------------

The required minimum system is an 80386 25 MHz with 16 MB RAM. A Pentium PC with
32 MB RAM is recommended.

Software Requirements
---------------------

The Setup program can be run on any Windows NT 3.51 or later workstation. After
successfully running the Setup program, the MailBox Converter can be distributed
to end-users running Windows NT 3.51 or later, or Windows 95.

The 32-bit Exchange Client program must be installed on the client workstation
before running the MailBox Converter. The cc:Mail archive files (typically .cca
or .imp) must be accessible from the client workstation.

Storage Requirements
--------------------

To store the MailBox Converter program, you need a minimum of 1 MB of disk space
plus additional space for mail folders and messages created in Exchange Server
by the conversion process.
