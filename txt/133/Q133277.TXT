SUMMARY
=======

This article describes how to copy files in Windows 95/98.

To copy files, use one of the following methods:

Method 1
--------

1. Locate the file you want to copy.

2. Use the right mouse button to click the file and click Copy.

3. Locate the folder (called the destination folder) into which you would like
   to copy the file.

4. Use the right mouse button to click the destination folder and click Paste.
   Depending on the size of the file, the time it takes to copy the file may
   vary.

Method 2
--------

1. Locate the file you want to copy.

2. Press CTRL+C (this copies the file).

3. Locate the folder (called the destination folder) into which you want to copy
   the file.

4. Press CTRL+V (this places a copy of the file into the new location).

After completing the above steps, the file is copied to the destination folder
while the original file remains in the source folder.
