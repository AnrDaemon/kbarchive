DOCUMENT:Q133306  08-AUG-2001  [winnt]
TITLE   :Maximum Disk Space of 1.99 GB Displayed for NetWare Volume
PRODUCT :Microsoft Windows NT
PROD/VER:3.51
OPER/SYS:
KEYWORDS:kbnetwork

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.51 
 - Microsoft Windows NT Server version 3.51 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you connect to a NetWare 3.11 server through Gateway Service for NetWare
(GSNW) or Client Service for NetWare (CSNW), and the server has a partition
larger than two gigabytes (GB), the maximum available or total disk space
displayed in Windows NT 3.51 File Manager is 1.99 GB.


RESOLUTION
==========

To correct this problem, install the latest U.S. Service Pack for Windows NT
version 3.51.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 3.51. This
problem has been corrected in the latest U.S. Service Pack for Windows NT
version 3.51. For information on obtaining the Service Pack, query on the
following word in the Microsoft Knowledge Base (without the spaces):

   S E R V P A C K

Additional query words: prodnt winfile fpnw
