TITLE   :Crystal Reports Cannot Find DLL Files in Windows NT
PRODUCT :Microsoft Systems Management Server
PROD/VER:winnt:3.5,3.51
OPER/SYS:
KEYWORDS:kb3rdparty smsgeneral kbArtTypeINF

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation versions 3.5, 3.51 
 - Microsoft Windows NT Server versions 3.5, 3.51 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run Crystal Reports under Windows NT 3.5 and 3.51, error messages
appear stating that PDBSODBC.DLL and other .DLL files for the application cannot
be found.

NOTE: Crystal Reports version 3.0.1.29 ships with Microsoft Systems Management
Server.

RESOLUTION
==========

To correct this problem, obtain a fix from Crystal Technical Support at (604)
669-8379.

The third-party product discussed here is manufactured by a vendor independent of
Microsoft; we make no warranty, implied or otherwise, regarding this product's
performance or reliability.
