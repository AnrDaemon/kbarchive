TITLE   :Password Reset For MSN Explorer "msn.com" E-mail Accounts

SUMMARY
=======

This article includes instructions for helping users of MSN Explorer Version 1.0
to change the passwords on "msn.com" e-mail accounts. The article applies to the
following types of users:

 - Users have a secret question that they can use to retrieve a password.

 - Users who do not have a secret question that they can use to retrieve a
   password.

MORE INFORMATION
================

 - To retrieve or change a password for MSN Explorer "msn.com" e-mail if the
   user has a secret question

1. Click the MSN Explorer icon.

2. Click the "Forgot Password" option in the upper right corner of the screen.

3. Type the e-mail name, state, and zip code, and then click "Continue".

4. On the next screen, type the answer to the secret question.

5. Type the new password in the appropriate cell.

6. Type the new password again, in the "re-enter password" cell to confirm it.

7. Click "Continue" to update the password information

From the sign on screen, select the user and sign in with the new password.

 - To retrieve or change a password for MSN Explorer "msn.com" e-mail if the
   user does not have a secret question

1. Send an e-mail message requesting help with resetting a password to:
   ResetPword@msn.com.

2. In the "Subject" box, type: PW VERIFICATION INFO ENCLOSED

3. Provide the following information.
    - Name (first and last)

    - MSN Member name

    - Birthdate

    - Country/region

    - State

    - Zip or postal code

    - Approximate date of the last time the MSN password was updates, or the
      approximate date of the account registration.

   If the user provides all of the requested information and the information is
   correct, MSN will reset the password and send the password by e-mail to the
   user.

   Note: Instruct users to change their account password as soon as they get
   access to the Internet again.
