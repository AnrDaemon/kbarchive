TITLE   :XWEB: Small File Attachments Show Incorrect Size in OWA
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:5.5
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Outlook Web Access, version 5.5 
 - Microsoft Outlook Web Access, version 5.5 Service Packs 1, 2, 3 
 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you attach a file to a message in the Outlook Web Access (OWA)client, the
an incorrect file size will be displayed if the file size is under 2 KB. All
attachments under 2 KB will be displayed as 2 KB.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.
