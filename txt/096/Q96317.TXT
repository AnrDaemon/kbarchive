SUMMARY
=======

Microsoft TrueType Master Set for the Macintosh is a collection of 102 TrueType
fonts. These scalable fonts look nice on the screen and provide a high-quality
printout from the output device you print them to. These fonts are compatible
with PostScript printers as well as (QuickDraw-based) Macintosh printers.

The TrueType Master Set matches the fonts currently available for Microsoft
Windows operating system version 3.1, these are:

 - 14 core fonts supplied with Windows version 3.1

 - 44 fonts supplied with Microsoft TrueType FontPack 1 for Windows

 - 44 fonts supplied with Microsoft TrueType FontPack 2 for Windows

For more information on the availability of Master Set, query on the following
words in the Microsoft Knowledge Base:

   " truetype and master and set and macintosh and font " (without the quotation
   marks)


MORE INFORMATION
================

Master Set facilitates better cross-platform conversions and provides a source
of high-quality scalable fonts for the Apple Macintosh. The font metrics are the
same as the corresponding Windows font, which makes it easier to move documents
between the two platforms.

System Requirements for TrueType Master Set
-------------------------------------------

   System 7.0 or later
   2 megabytes (MB) of random access memory (RAM)
   One disk drive and a hard disk
   LaserWriter printer driver version 7.1.2 or later recommended

Supported Printers
------------------

   Any Macintosh-compatible printer

Documentation Included
----------------------

   User's Guide
   Quick Reference Card
