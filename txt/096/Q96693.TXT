TITLE   :PC Win: Name Finder Doesn't Find Correct Names
PRODUCT :Microsoft Mail For PC Networks
PROD/VER:WINDOWS:3.0b
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for Windows, version 3.0b 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use the Mail for Windows Name Finder to search the Global Address List
(GAL), the Name Finder does not always find the correct names.

CAUSE
=====

This problem occurs because of the way the GAL index is built.

STATUS
======

Microsoft has confirmed this to be a problem in version 3.0b of Microsoft Mail
for Windows. This problem was corrected in version 3.2.
