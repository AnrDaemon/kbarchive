TITLE   :Microsoft SoundBits Browser Does Not Run Minimized
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER::1.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft SoundBits version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft SoundBits Browser does not run minimized if you select the Run
Minimized check box in the Program Item Properties dialog box, if you press
SHIFT+ENTER, or if you press SHIFT and double-click the mouse on the Browser
icon.

MORE INFORMATION
================

To minimize the SoundBits Browser, run the program first, then minimize it using
either the Control menu (-) or the Minimize button.

Microsoft has confirmed this to be a problem in SoundBits. We are researching
this problem and will post new information here in the Microsoft Knowledge Base
as it becomes available.
