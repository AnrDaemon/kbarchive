TITLE   :PC Win: Err Msg: Are You Sure You Want to Remove...
PRODUCT :Microsoft Mail For PC Networks
PROD/VER:WINDOWS:3.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for Windows, version 3.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When deleting users from the Personal Address Book in version 3.0 of Microsoft
Mail for Windows, the following error message will be displayed:

   Are you sure you want to remove the selected users? Note that removing users
   from your address book will also remove them from all personal groups.

This error message will be displayed even when the user selected does not belong
to any personal groups.

Selected users that are contained in personal groups will be removed from the
groups.

RESOLUTION
==========

If all the users in a personal group have been removed, the personal group will
remain defined with no one in the group. To delete the personal group, choose
Personal Groups from the Mail menu. Select the group from the list and choose
Remove.

Additional query words: 3.00 PAB user(s) errmsg alert
