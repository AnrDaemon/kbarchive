TITLE   :PC WSPlus: Importing WordPerfect Files
PRODUCT :Microsoft Schedule+ for Windows
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Schedule+ for Windows, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you try to open an encrypted WordPerfect file in Microsoft Schedule+ for
Windows, the following message is displayed:

   The file you're trying to import is not in a valid import format

RESOLUTION
==========

To work around this problem, export the file from WordPerfect Office with
encryption turned off before you try to import the file again.
