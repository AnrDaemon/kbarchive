TITLE   :Updated ATI CRYSTAL Fonts Driver Required for Windows 3.1
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:win31

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

To use ATI's CRYSTAL Fonts display technology with Microsoft Windows operating
system version 3.1 at a resolution of 1024 x 768, you must obtain version 1.2 of
the ATI CRYSTAL Fonts driver.

MORE INFORMATION
================

When running Microsoft Windows operating system version 3.1 with all models of
the ATI Graphics Accelerator line of video cards (ATI Graphics Ultra, ATI
Graphics Vantage, or ATI 8514 Ultra adapter), you must use version 1.2 of the
ATI CRYSTAL Fonts video driver for 1024 x 768 resolution. Versions earlier than
1.2 do not support TrueType.

The update driver is available from ATI technical support, or by downloading the
file named WIN31XTL.ZIP.

To obtain the phone numbers for ATI, query on the following words in the
Microsoft Knowledge Base:

   ATI and vendor and hardware and software

This updated driver is also available on CompuServe in the IBMHW forum as
WIN31X.ZIP.


CRYSTAL Fonts is a font display technology developed by ATI designed to give an
increased WYSIWYG appearance. In the near future, ATI plans to release an
updated ATI CRYSTAL Fonts Converter Kit that will work with Windows 3.1.

The CRYSTAL Fonts products included here are manufactured by a vendor independent
of Microsoft; we make no warranty, implied or otherwise, regarding these
products' performance or reliability.
