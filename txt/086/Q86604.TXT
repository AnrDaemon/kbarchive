TITLE   :PC WSPlus: Err Msg: Could Not Open the File Named...
PRODUCT :Microsoft Schedule+ for Windows
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Schedule+ for Windows, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

During Microsoft Schedule+ for Windows setup, the following error message is
displayed:

   Could not open the file named:
   MAILSPL.EXE.
   It is in use by another application

CAUSE
=====

This problem occurs if a Microsoft Mail-enabled program (for example, Microsoft
Excel version 4.0 for Windows) is already running.

RESOLUTION
==========

To run the Setup program without these conflicts, exit from all running
applications and remove all icons from the Program Manager Start Up group and
restart Windows.
