TITLE   :3COM: Hardware Required by the 3Com Gateway
PRODUCT :Microsoft Mail For PC Networks
PROD/VER::3.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail Gateway to 3Com 3+Mail, version 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Microsoft Mail Gateway to 3Com 3+Mail requires a dedicated network
workstation, which can be a PC, XT, or AT, with MS-DOS version 3.1 or later.
This machine must have a hard drive so it can store temporary files.

The gateway must be able to access the Microsoft Mail data files, and must be
able to log into the 3+Mail service.

REFERENCES
==========

"Microsoft Mail Gateway to 3Com 3+Mail Administrator's Guide"
