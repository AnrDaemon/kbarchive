TITLE   :PC OS2: PM Setup Does Not Check for Disk Space
PRODUCT :Microsoft Mail For PC Networks
PROD/VER:WINDOWS:3.0,3.0b,3.2
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for PC Networks, versions 3.0, 3.0b, 3.2, on platform(s):
    - the operating system: OS/2 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Setup program for Microsoft Mail for Presentation Manager does not check for
sufficient disk space before installing. If it runs out of disk space, it does
not remove the files it has already installed.

STATUS
======

Microsoft has confirmed this to be a problem in versions 3.0, 3.0b, and 3.2 of
Microsoft Mail for Presentation Manager. We are researching this problem and
will post new information here in the Microsoft Knowledge Base as it becomes
available.

Additional query words: 3.00 3.00b 3.20
