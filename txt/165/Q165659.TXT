DOCUMENT:Q165659  13-JUN-2001  [sna]
TITLE   :3270 Applet for Windows NT Does Not Use Ctl3d32
PRODUCT :Microsoft SNA Server
PROD/VER:WINDOWS:3.0
OPER/SYS:
KEYWORDS:kbnetwork

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft SNA Server, version 3.0, on platform(s):
    - the operating system: Microsoft Windows NT 
-------------------------------------------------------------------------------


SYMPTOMS
========

The 3270 Applet for Windows NT does not use Ctl3d32. This may result in some
problems when the system colors are changed.

RESOLUTION
==========

The 3270 Applet for the Windows NT client has been updated to correct this
problem. The new version of win3270.exe is included in the most recent service
pack. See the Status section to follow for more information on how to obtain the
most recent service pack.

STATUS
======

Microsoft has confirmed this to be a problem in SNA Server version 3.0. This
problem was corrected in the latest Microsoft SNA Server 3.0 U.S. Service Pack.
For information on obtaining the service pack, query on the following word in
the Microsoft Knowledge Base (without the spaces):

   S E R V P A C K
