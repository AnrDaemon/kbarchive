The information in this article applies to:

 - Microsoft PowerPoint for Windows 95, version 7.0 
 - Microsoft PowerPoint for Macintosh, version 4.0 
 - Microsoft PowerPoint for Windows, version 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the versions of the Internet Assistant for Microsoft
PowerPoint that are currently available.

The Internet Assistant for PowerPoint is an add-in that allows you to convert a
PowerPoint presentation to a series of HTML documents which you can view in an
Internet browser, such as Microsoft Internet Explorer.

MORE INFORMATION
================

Microsoft PowerPoint 7.0 for Windows 95
---------------------------------------

The Internet Assistant for PowerPoint 7.0 is available as a PowerPoint add-in.
There are versions available for the following languages:

 - English (also works with any language that uses ANSI characters.)

 - Japanese

 - Korean

 - Traditional Chinese

For more information about obtaining this add-in, please see the following
article in the Microsoft Knowledge Base:

   Q129386 Internet Assistant for PowerPoint Available on Online Services

For information about which files the PowerPoint Internet Assistant includes,
please see the following article in the Microsoft Knowledge Base:

   Q147893 PPT7: Files Included with PowerPoint Internet Assistant

Microsoft PowerPoint 4.0 for Macintosh and Windows
--------------------------------------------------

There is currently no Internet Assistant available for these versions of
PowerPoint.

NOTE: The Internet Assiatant features are built into PowerPoint 97 for Windows
and PowerPoint 98 Macintosh Edition.
