DOCUMENT:Q224964  10-AUG-2001  [winnt]
TITLE   :SP4 Server Hangs with St. Bernard OFM and McAfee NetShield
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0,4.0 SP4
OPER/SYS:
KEYWORDS:kb3rdparty

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server versions 4.0, 4.0 SP4 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use Windows NT 4.0 Service Pack 4 with St. Bernard Open File Manager
(OFM) and McAfee NetShield, your computer may stop responding (hang).

CAUSE
=====

This behavior occurs because there are compatibility issues between St. Bernard
OFM System Component and McAfee NetShield 4.0.3.

RESOLUTION
==========

To work around this issue, use the appropriate method:

 - Uninstall McAfee NetShield and St. Bernard OFM.

 - Refer to the St. Bernard Web site for updated information about this issue
   at:

   http://stbernard.com/techbulletin.html
