DOCUMENT:Q224193  27-MAY-2001  [winnt]
TITLE   :Errors Starting Microsoft Exchange Administrator Tool
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0,4.0a,4.5,5.0,5.5
OPER/SYS:
KEYWORDS:kbenv kberrmsg kbtool kbui

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft BackOffice Small Business Server versions 4.0, 4.0a, 4.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you try to start the Microsoft Exchange Administrator tool, you may receive
the following error message:

   The application failed to initialize

Also, the message may state this is because of MAPI 1.0 errors.

CAUSE
=====

This issue can occur if certain essential program files are missing or have been
replaced with incorrect versions.

RESOLUTION
==========

To resolve this issue, uninstall the Exchange Administrator tool, and then
reinstall it.
