TITLE   :Cannot Install Reference Suite DVD on Windows 95-Based Computer
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:WINDOWS:
OPER/SYS:
KEYWORDS:kbenv kberrmsg kbsetup kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta Reference Suite 99 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to install Microsoft Encarta Reference DVD-ROM Suite 99 on a
Microsoft Windows 95-based computer, you receive one of the following error
messages:

   Reference Suite Error
   Autorun was unable to run a required application.

   F:\Setup.exe
   F:\Setup.exe is not a valid Win32 application.

RESOLUTION
==========

There is no known resolution to this issue at this time.

If you want to return Encarta Reference DVD-ROM Suite 99, please contact the
software vendor where you purchased the product or telephone Customer Service at
(800) 426-9400. Encarta Reference DVD-ROM Suite 99 is covered by a 30 day money
back guarantee.

If you are outside the United States, contact the Microsoft subsidiary for your
area. To locate your subsidiary, visit the Microsoft World Wide Offices Web site
at the following address:

   http://www.microsoft.com/worldwide/default.htm

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Encarta Reference
DVD-ROM Suite 99.

MORE INFORMATION
================

Windows 95 requires that two file systems be written to a DVD-ROM disc, ISO 9660
and UDF. The Encarta Reference Suite 99 DVD-ROM was not created with both file
systems written to the disc.

NOTE: This information does not apply to the Microsoft Encarta Encyclopedia 99
DVD-ROM, as it was created with both the ISO 9660 and the UDF file systems.
