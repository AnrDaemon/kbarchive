TITLE   :ELL: Virtual Challenge Stops Responding When You Press Reset
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER::
OPER/SYS:
KEYWORDS:kbdisplay kbui kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta Language Learning French 
 - Microsoft Encarta Language Learning French Deluxe 
 - Microsoft Encarta Language Learning Spanish 
 - Microsoft Encarta Language Learning Spanish Deluxe 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run the Virtual Challenge in one of the Microsoft Encarta Language
Learning programs listed at the beginning of this article, the Virtual Challenge
stops responding when you click the Reset button.

CAUSE
=====

This behavior occurs because the Reset button is still available, even though
the button is dimmed.

RESOLUTION
==========

To work around this issue, wait for the Reset button to be outlined in white
before you click the button.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.
