TITLE   :ELL: Program Hangs When You Record in Virtual Challenge
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER::
OPER/SYS:
KEYWORDS:kbenv kbsound kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta Language Learning French 
 - Microsoft Encarta Language Learning French Deluxe 
 - Microsoft Encarta Language Learning Spanish 
 - Microsoft Encarta Language Learning Spanish Deluxe 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you record in the Virtual Challenge feature included in any of the
Microsoft Encarta Language Learning programs listed at the beginning of this
article, the program may stop responding (hang).

CAUSE
=====

This behavior can occur if you do not use proper pronunciation or the correct
words in the Virtual Challenge.

RESOLUTION
==========

To resolve this issue, restart the Encarta Language Learning program.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.

Additional query words: multi multi-media media mm ellf ells crash lock freeze
