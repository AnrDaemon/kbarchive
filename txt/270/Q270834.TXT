TITLE   :ELL: Input Boxes Do Not Appear in Dictation Exercises
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER::
OPER/SYS:
KEYWORDS:kbdisplay kbenv kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta Language Learning French 
 - Microsoft Encarta Language Learning French Deluxe 
 - Microsoft Encarta Language Learning Spanish 
 - Microsoft Encarta Language Learning Spanish Deluxe 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run the dictation exercises in the Microsoft Encarta Language Learning
programs listed at the beginning of this article, the blank boxes into which you
type words may not appear on the screen.

CAUSE
=====

This behavior can occur if the retail version of Microsoft Windows 95 is
installed on your computer.

RESOLUTION
==========

To resolve this issue, install Microsoft Windows 98 or later on your computer.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.
