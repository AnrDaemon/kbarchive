TITLE   :ELL: Virtual Keyboard Menu Disables Pointing Device
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER::
OPER/SYS:
KEYWORDS:kbenv kbhw kbimu kbHardware

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta Language Learning French 
 - Microsoft Encarta Language Learning French Deluxe 
 - Microsoft Encarta Language Learning Spanish 
 - Microsoft Encarta Language Learning Spanish Deluxe 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you type a letter that can have accents applied to it in one of the Encarta
Language Learning programs listed at the beginning of this article, your
pointing device may stop responding (hang) while the virtual keyboard menu is
displayed on the screen.

RESOLUTION
==========

To resolve this issue, use your keyboard to select one of the options on the
virtual keyboard menu before you attempt to use the pointing device.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.
