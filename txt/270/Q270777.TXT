TITLE   :ELL FR & SP: Menu Not Active in Virtual Challenge
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER::
OPER/SYS:
KEYWORDS:kberrmsg kbui kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta Language Learning French 
 - Microsoft Encarta Language Learning French Deluxe 
 - Microsoft Encarta Language Learning Spanish 
 - Microsoft Encarta Language Learning Spanish Deluxe 
-------------------------------------------------------------------------------

SUMMARY
=======

When you run the Virtual Challenge in one of the Microsoft Encarta Language
Learning programs listed at the beginning of this article, the Topics menu is
not active when the following message is displayed on the screen:

   Task <number> Incomplete
   You have not successfully completed this task.

   Click anywhere to continue...

where <number> is the number of a task in the Virtual Challenge.

MORE INFORMATION
================

To activate the Topics menu, you must click on this message to dismiss the
message.

Microsoft has confirmed that this is a problem in the Microsoft products that are
listed at the beginning of this article.
