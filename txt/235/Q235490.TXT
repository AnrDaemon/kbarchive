DOCUMENT:Q235490  05-SEP-1999  [exchange]
TITLE   :XFOR: Contacts Are Not Supported By GroupWise
PRODUCT :Microsoft Exchange
PROD/VER:winnt:5.5
OPER/SYS:
KEYWORDS:exc55

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------


SYMPTOMS
========

When GroupWise users receive a contact as part of an e-mail message from an
Exchange Server user, the resulting contact information is converted to a Rich
Text Format (RTF) attachment with only the following information:

   From
   To
   Cc
   Subject

CAUSE
=====

This problem occurs because of a limitation in messaging features between
Microsoft Exchange Server and Novell GroupWise server. The GroupWise system does
not support contacts as an attachment type.
