TITLE   :SMS: DOS and Windows 3.1 Clients Fail Site Assignment
PRODUCT :Microsoft Systems Management Server
PROD/VER:winnt:2.0
OPER/SYS:
KEYWORDS:kbClient kbsms200 kbsms200bug kbsms200fix kbDiscovery

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Systems Management Server version 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

IP addresses are not retrieved from DOS or Windows 3.1 computers using TCP/IP
from Microsoft Network Client 3.0.

This causes the client to fail assignment to the site.


STATUS
======

Microsoft has confirmed this to be a problem in Systems Management Server
version 2.0. This problem has been corrected in the latest U.S. service pack for
Systems Management Server version 2.0. For information on obtaining the service
pack, query on the following word in the Microsoft Knowledge Base (without the
spaces):

   S E R V P A C K

Additional query words: prodsms
