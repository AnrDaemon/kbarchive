TITLE   :SMS: Y2K Software Compliance Query Displays Incorrect File Names
PRODUCT :Microsoft Systems Management Server
PROD/VER:winnt:2.0
OPER/SYS:
KEYWORDS:kbsms200 kbsms200bug kbsms120 kbsms120bug

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Systems Management Server version 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When running the Year 2000 (Y2K) queries in Systems Management Server 2.0, the
results show products that are installed on specific System Management Server
2.0 clients. In some cases, these queries may mistakenly show that the same
product is also installed on other clients when, in fact, it is not installed.

STATUS
======

Microsoft has confirmed this to be a problem in Systems Management Server
version 2.0. This problem has been corrected in the latest U.S. service pack for
Systems Management Server version 2.0. For information on obtaining the service
pack, query on the following word in the Microsoft Knowledge Base (without the
spaces):

   S E R V P A C K
