TITLE   :PRINT SCREEN to Clipboard Appears as Black-and-White Image
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you are running Microsoft Windows 3.0 in real mode and you press PRINT SCREEN
or ALT+PRINT SCREEN, the image may be placed in the Clipboard as a
black-and-white image.

CAUSE
=====

On some computers, if Windows is in real mode the system may not have enough
system resources or available memory to display the image in color. The image
may also be distorted and unreadable.

RESOLUTION
==========

Run Windows in standard or enhanced mode if possible.

Additional query words: 3.00 3.00a PRT SCREEN ALT black and white win30
