TITLE   :Applications Run Minimized When SHIFT Key Held Down
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a,3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

When you start an application from Microsoft Windows using any of the following
methods, and you hold the SHIFT key down, the application starts minimized.

This information applies to both Windows-based and MS-DOS-based applications.

MORE INFORMATION
================

Program Manager
---------------

To start the application minimized, hold down the SHIFT key and do one of the
following:

 - Double-click the program's icon.

 - Choose the icon. From the File menu, choose Open to start the application.

 - Choose the application's icon and press ENTER.

File Manager
------------

To start the application minimized, hold down the SHIFT key and do one of the
following:

 - Start the application by double-clicking the program's filename.

 - Select the file. From the File menu, choose Open to start the application.

 - Select the application's filename and press ENTER.
