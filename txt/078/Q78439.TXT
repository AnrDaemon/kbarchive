TITLE   :Fulbak Backup Utilities 2.0 or Later with MS-DOS 5.0
PRODUCT :Microsoft Disk Operating System
PROD/VER:MS-DOS:5.x,6.0,6.2,6.21,6.22
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft MS-DOS operating system versions 5.0, 5.0a, 6.0, 6.2, 6.21, 6.22 
-------------------------------------------------------------------------------

SUMMARY
=======

Westlake Data Corporation has confirmed that version 2.0 of Fulbak is required
for compatibility with MS-DOS 5.0 and later.

Fulbak is an application used for backing up and restoring files.

MORE INFORMATION
================

The following are symptoms of using versions of Fulbak earlier than version 2.0
with MS-DOS 5.0:

 - File compare may not complete successfully when using the Fulbak backup
   utility with the compare files utility enabled.

 - When using the Fulbak backup utility without the the compare files utility
   enabled, an error will only occur if there is a problem with the drive
   itself.

 - The following error messages may be displayed when using Fulbak restore:

   Invalid File
   Not a Valid Backup Disk
   Directory Does not Exist
   Cannot Uncompress
