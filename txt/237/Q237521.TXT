TITLE   :EWED: Can't Start Reference Suite Program by Clicking Suite Link
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:WINDOWS:; :
OPER/SYS:
KEYWORDS:kberrmsg kbinterop kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta World English Dictionary 
 - Microsoft Encarta World English Dictionary 2001 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you click a suite link in Microsoft Encarta World English Dictionary 2000,
you may experience either of the following symptoms:

 - If you click a suite link to Microsoft Encarta Encyclopedia 2000, you may be
   prompted to insert the Microsoft Encarta Reference Suite 2000 DVD-ROM.

 - If you click a suite link to Microsoft Encarta World Atlas 2000, you may
   receive the following error message:

   File Error: Encarta Interactive World Atlas is unable to continue running on
   your computer. Your network based installation is unavailable at this time.

   NOTE: You receive this error message even if you do not run Encarta World
   English Dictionary 2000 or Encarta World Atlas 2000 from a network server.

   When you click OK, you return to Encarta World English Dictionary 2000.

CAUSE
=====

This behavior can occur if all of the following conditions are true:

 - You install Encarta Encyclopedia 2000, Encarta World Atlas 2000, and Encarta
   World English Dictionary 2000 from the Encarta Reference Suite 2000 DVD-ROM.

 - You install Encarta Dictionary HD to your hard disk (you install the entire
   contents of Encarta World English Dictionary 2000 to your hard disk).

 - The Encarta Reference Suite 2000 DVD-ROM is not inserted in your DVD-ROM
   drive.

RESOLUTION
==========

To resolve this issue, insert the Encarta Reference Suite 2000 DVD-ROM into your
DVD-ROM drive.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Encarta World English
Dictionary 2000.
