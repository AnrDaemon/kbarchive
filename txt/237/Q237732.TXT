TITLE   :NFL 2000: Quits Unexpectedly If Main Screen Remains Displayed
PRODUCT :Microsoft Home Games
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kbinterop kbimu msgame

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft NFL Fever 2000, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If the Main screen in Microsoft NFL Fever 2000 remains displayed on your monitor
for more than ten minutes, the program may quit unexpectedly when you attempt to
start a game.

CAUSE
=====

This behavior can occur if the Microsoft SideWinder Game Controller software is
installed on your computer.

RESOLUTION
==========

To work around this issue, quit and then restart NFL Fever 2000.

To prevent this issue from occurring, do not leave the Main screen of NFL Fever
2000 displayed on your monitor for more than ten minutes.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft NFL Fever 2000,
version 1.0.
