DOCUMENT:Q237712  27-AUG-1999  [homegame]
TITLE   :NBA 2000: Can Start Trial Version but Not Retail Version
PRODUCT :Microsoft Home Games
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kbsetup kbimu msgame

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft NBA Inside Drive 2000, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to play the retail version of Microsoft NBA Inside Drive 2000,
you may be unable to start the game even though you are able to start the trial
version of NBA Inside Drive 2000.

CAUSE
=====

This behavior can occur if you install the trial version of NBA Inside Drive
2000 into the same folder as the retail version of the game. By default, the
retail version of NBA Inside Drive 2000 is installed in the following folder:

   C:\Program Files\Microsoft Games\NBA Inside Drive 2000

RESOLUTION
==========

To resolve this issue, reinstall the retail version of NBA Inside Drive 2000.

Additional query words: 1.00 nba2k nba2000 basketball setup msgame crash hang freeze fouls
