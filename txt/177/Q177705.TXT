TITLE   :SMS: List Of Supported Video Adapters for SP3's IDISNTKM
PRODUCT :Microsoft Systems Management Server
PROD/VER:winnt:1.2
OPER/SYS:
KEYWORDS:kbtshoot smsremtshoot kbRemoteProg

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Systems Management Server version 1.2 
-------------------------------------------------------------------------------

SUMMARY
=======

The following video cards have been tested and found to work with the Service
Pack 3 (SP3) Idisntkm driver. Systems Management Server Site properties will be
updated after installing SP3 with the new entries. These entries will be
available for both Windows NT 3.51 and Windows NT 4.0 clients.

 - ATI 3D Xpression

 - ATI Mach 64 Wincharger

 - Canopus total 3D

 - Cirrus Logic GD 5430

 - Creative Labs 3D Blaster

 - Diamond Edge 3d 2120

 - Diamond Monster 3D

 - Diamond Stealth 64 2001

 - Diamond Stealth 64 VRAM

 - Genoa Video Blitz 9200VL (This uses the Weitek 9000 chipset)

 - Hercules dynamite 128/Video

 - Hercules Terminator 3D

 - Hercules Terminator 64

 - Matrox Millennium

 - Matrox Mystique

 - Number Nine Motion 531

 - Number Nine Motion 771

 - Number Nine Reality 332

 - Oak Spitfire

 - Orchid Righteous 3D

 - Paradise Pipeline 64

 - S3 Trio

 - S3 Virge

 - Sigma Real Magic

 - Sigma Winstorm

 - STB Lightspeed 128

 - STB Powergraph 64V

 - Trident

 - Videologic Apocalypse 3D

 - Videologic Graphix Star 600

Additional query words: prodsms smsremtshoot accelerated
