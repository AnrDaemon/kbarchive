DOCUMENT:Q177455  14-NOV-2000  [word97]
TITLE   :WD97: Problems with Document Based on Contemporary Memo Template
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:kbdta kbtemplate word97

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you are working on a Microsoft Word document based on the Contemporary Memo
template, the following can occur:

 - The floating picture within the template disappears.

   -or-

 - When you run the Undo Update Fields command, Microsoft Word stops responding.

CAUSE
=====

This problem occurs when the Automatically update document styles option has
been turned on in the Templates and Add-ins dialog box.

WORKAROUND
==========

To work around this problem, do not turn on Automatically update document styles
for this document. You can find this setting by clicking Templates and Add-Ins
on the Tools menu.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.

MORE INFORMATION
================

For additional information about updating document styles, please click the
article number below to view the article in the Microsoft Knowledge Base:

   Q166174 WD: Styles Aren't Automatically Updated Attaching to Normal.dot
