TITLE   :HP 350C and 750C Plotter Output Contains a Blue Background
PRODUCT :Microsoft Windows NT
PROD/VER:WINNT:4.0
OPER/SYS:
KEYWORDS:kbenv kbprint

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 4.0 
 - Microsoft Windows NT Server version 4.0 
 - Microsoft Windows NT Server, Enterprise Edition version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you print to a Hewlett-Packard (HP) 350C or 750C plotter, the output may
contain a blue background.

RESOLUTION
==========

To work around this issue, change the settings on your printer device.

DesignJet 750C-Series Plotters
------------------------------

Change the Palette option under Pen Settings to Software on the plotter.

For information about changing the printing options, contact HP, view the plotter
documentation included with your plotter, or see the HP Web site.

DesignJet 3xx-Series Plotters
-----------------------------

Change the Pen Settings option to Use Settings From Software on the plotter.

For information about changing the printing options, contact HP, view the plotter
documentation included with your plotter, or see the HP Web site.

STATUS
======

Microsoft is researching this issue and will post new information here in the
Microsoft Knowledge Base as it becomes available.
