DOCUMENT:Q177410  10-AUG-2001  [winnt]
TITLE   :No IGMP Packets Received When Using Starlight Software
PRODUCT :Microsoft Windows NT
PROD/VER:WinNT:3.5,3.51
OPER/SYS:
KEYWORDS:kb3rdparty kbnetwork

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation versions 3.5, 3.51 
 - Microsoft Windows NT Server versions 3.5, 3.51 
-------------------------------------------------------------------------------

SYMPTOMS
========

You are using Starlight Networks Software to provide multimedia and video
solutions to your intranets. While your Windows 95 clients receive Internet
Group Management Protocol (IGMP) datagrams successfully, your Windows NT clients
do not.

For more information regarding IGMP, see the following article in the Microsoft
Knowledge base:

ARTICLE-ID: Q135679
TITLE : IGMP Group members Do Not Respond to IGMP Group Query Frames

RESOLUTION
==========

To resolve this problem, please contact Starlight Network Technical Support.

For more information, see the following Web site:

www.starlight.com

NOTE: Because web sites are constantly updated, the site address may change
without notice. If this occurs, please contact the manufacturer directly.

For information about how to contact Starlight Networks, query in the Knowledge
Base for one of the following articles:
