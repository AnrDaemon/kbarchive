TITLE   :Computer Hangs Starting Encarta Using iNFRA1800 CD-ROM Drive
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:WINDOWS:
OPER/SYS:
KEYWORDS:kb3rdparty kbhw kbHardwarekbfaq

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta Encyclopedia 97 Deluxe for Windows 
 - Microsoft Encarta 97 Encyclopedia for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to start Microsoft Encarta 97 Encyclopedia for Windows on a
computer using a Creative Labs iNFRA1800 CD-ROM drive, the computer may stop
responding (hang).

CAUSE
=====

This behavior can occur if an outdated version of the Creative Labs iNFRA1800
CD-ROM driver is installed on the computer.

RESOLUTION
==========

To resolve this issue, obtain an updated driver for the Creative Labs iNFRA1800
CD-ROM drive from the following Creative Labs Web site:

   http://www.soundblaster.com/wwwnew/tech/tech.html

For more information about how to contact Creative Labs, query in the Microsoft
Knowledge Base for the following article:
