TITLE   :QuickTime Not Listed in Add/Remove Programs Under Windows NT 4.0
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:WINDOWS:; WINNT:4.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Bookshelf 98 for Windows 
 - the operating system: Microsoft Windows NT 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you are using a Microsoft Windows NT 4.0-based computer, the Add/Remove
Programs tool in Control Panel may not list Apple QuickTime for Windows if
Microsoft Bookshelf 98 for Windows is installed.

CAUSE
=====

The installation program for the version of Apple QuickTime for Windows included
on the Bookshelf 98 CD-ROM adds an Uninstall key to the Microsoft Windows 95
registry, but does not do so to the Windows NT 4.0 registry.

RESOLUTION
==========

To resolve this issue, use the Uninstall QuickTime 32-Bit program on the Start
menu to remove QuickTime for Windows. To do so, follow these steps:

1. Click Start, and then point to Programs.

2. Point to QuickTime for Windows, and then click Uninstall QuickTime 32-Bit.

3. Follow the instructions on the screen to remove QuickTime for Windows.

================
