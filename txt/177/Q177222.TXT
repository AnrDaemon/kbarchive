DOCUMENT:Q177222  25-MAR-2002  [win95x]
TITLE   :NetWare Printers or Volumes Unavailable in Windows 95
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER::2.0
OPER/SYS:
KEYWORDS:kbnetwork osr2 win95

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
 - Microsoft Windows 95 OEM Service Release, version 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

Persistent connections to print queues or volumes on two or more NetWare servers
may be inadvertently disconnected. This may occur when you log on, or during
normal usage.

RESOLUTION
==========


STATUS
======

This problem no longer occurs in Windows 98. To resolve this problem, install
the current version of Windows. For information about the current version of
Windows, visit http://www.microsoft.com/windows.
