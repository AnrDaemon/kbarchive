TITLE   :XADM: Error 0x8004010f Occurred In Function EcConfigMsgToSend
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:5.5
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------


SYMPTOMS
========

When a newsfeed is configured on a computer running Exchange Server 5.5, the
following error may occur repeatedly in the event log:

   Event ID:  11710
   Source:  MSExchange NNTP
   Type:  Error
   Category:  Replication

   Error 0x8004010f occurred in function EcConfigMsgToSend during NNTP
   outbound.

This is a bogus error and can be ignored.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server
version 5.5. We are researching this problem and will post new information here
in the Microsoft Knowledge Base as it becomes available.
