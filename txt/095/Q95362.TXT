TITLE   :AST Cupid 386 or 486 May Need BIOS Upgrade
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you install Windows version 3.1 on an AST Cupid 386 or 486 model computer,
the Windows Setup program may stop (hang) when moving from character-mode setup
to Windows-mode setup. According to AST, obtaining a BIOS upgrade corrects this
problem.

MORE INFORMATION
================

AST has released the following upgraded BIOS versions that may be required to
run Windows 3.1.

   Computer Type                    BIOS Version     Part Number
   -------------------------------------------------------------

   Integrated Video and Mouse:      1.20             500775-002

   Non-Integrated Video and Mouse:  1.40             500772-001

The integrated video and mouse model can be identified by the presence of a PS/2
type mouse port.

For more information, contact AST Technical Support.
