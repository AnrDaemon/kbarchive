TITLE   :Err Msg: DDHELP Has Caused Error In GDI.EXE
PRODUCT :The Microsoft Network
PROD/VER::6.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - The Microsoft Network version 6.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When downloading MSN Explorer an error message prompts with

   DDHELP has caused error in GDI.EXE

and will not allow you to continue.

CAUSE
=====

This is an interaction between early versions of MSN Explorer Preview 2 and
earlier versions of Macromedia Flash.

RESOLUTION
==========

1. Install the current version of Macromedia Flash. You can download the latest
version from:

   http://sdc.shockwave.com/shockwave/download/frameset.fhtml

2. Reinstall the latest version of MSN Explorer. To ensure you have the latest
version, download it from:

   http://preview.msn.com
