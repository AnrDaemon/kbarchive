TITLE   :Flight Simulator: Cannot Use Numeric Keypad to Control Aircraft
PRODUCT :Microsoft Home Games
PROD/VER::1.0
OPER/SYS:
KEYWORDS:kbui kbimu msgame

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Combat Flight Simulator: WWII Europe Series, version 1.0 
 - Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
 - Microsoft Flight Simulator 98 
 - Microsoft Flight Simulator 2000 
 - Microsoft Flight Simulator 2000 Professional Edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to control your aircraft by using the numeric keypad in one of
the programs listed at the beginning of this article, the view from the aircraft
may change when you press a key.

NOTE: In addition, pressing the numeric keypad no longer displays a Map view in
Microsoft Combat Flight Simulator 2.

CAUSE
=====

This behavior can occur if the state of the NUM LOCK key is off.

RESOLUTION
==========

To resolve this issue, press the NUM LOCK key to toggle the state of the key and
activate the numeric keypad.

Additional query words: msgame flightsim fsim combatfs cfs numbers key pad
