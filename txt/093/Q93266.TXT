The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
 - Microsoft MS-DOS operating system versions 3.1, 3.2, 3.21, 3.3, 3.3a, 4.0, 4.01, 5.0, 5.0a, 6.0, 6.2, 6.21, 6.22 
 - Microsoft Windows for Workgroups versions 3.1, 3.11

SUMMARY
=======

The following is a list of FCC ID (first three digits) numbers and the mice that
use those FCC ID numbers. This can help in determining which Windows or MS-DOS
mouse driver to use.

The FCC ID number can usually be found on the bottom of the mouse.

MORE INFORMATION
================

FCC ID #               Mouse
-------------------------------

ANO                    IBM
BCG                    Apple
BKL                    Kraft
B94                    Hewlett-Packard
C3K                    Microsoft
CWE                    Mouse Systems
DZL                    Logitech
E5Y                    Diamond Flower
E6Q                    Znix
EMJ                    Cutie
EW4                    Mitsumi
F4Z                    AMBRA
FSU                    Genius
FUU                    Microspeed
GJK                    Honeywell
GV3                    Expert by Kensington
GYU                    Packard Bell
H8G                    A4TECH
HQA                    Compuadd
HQX                    Emmerson
HQX                    Marstek
IAP                    Leaguer Int. Corp.
IFS                    Laser
IOW                    Addison Technologies
IOW                    Leading Edge
IQ7                    JETTA
KAM                    ATI
KH2                    Packard Bell
X31                    Z-NIX Inc.

Additional query words: mouses manufacturer 3.0 3.0a 3.1
