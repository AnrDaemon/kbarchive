TITLE   :Windows Err Msg: The Directory Path for Shared Code Invalid
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
 - Microsoft Windows for Workgroups version 3.1 
-------------------------------------------------------------------------------

SYMPTOMS
========

It is not possible to run WordPerfect for Windows from a network server or from
a shared network drive unless it has been specifically installed to do so. If a
local installation of WordPerfect is executed remotely, the following error
message occurs:

   Error: The directory path for the shared code is invalid.

RESOLUTION
==========

To resolve this problem, the network administrator must use the INSTALL.EXE
utility from WordPerfect for Windows to create a shared installation for use in
a network environment.

For more information on this process, consult the WordPerfect documentation, or
contact WordPerfect technical support.
