TITLE   :MapPoint 2001: Finish Button Is Unvailable When You Import Data
PRODUCT :Microsoft Automap
PROD/VER::
OPER/SYS:
KEYWORDS:kbtool kbui kbimukbfaq

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft MapPoint 2002 
 - Microsoft MapPoint 2001 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use the Data Import Wizard to import data into Microsoft MapPoint 2001,
the Finish button may be unavailable.

CAUSE
=====

This behavior can occur if the data that you imported contains duplicate column
headings.

RESOLUTION
==========

To prevent this issue from occurring, select a unique heading for each column of
data before you import the data into MapPoint 2001.

STATUS
======

Microsoft has confirmed that this is a problem in Microsoft MapPoint 2001.

Additional query words: mp2001 map point importing dimmed grayed greyed
