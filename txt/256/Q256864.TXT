TITLE   :SMS: Windows 2000 Professional May Appear as Windows NT Server
PRODUCT :Microsoft Systems Management Server
PROD/VER:winnt:2.0
OPER/SYS:
KEYWORDS:kbsms200 kbsms200bug

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Systems Management Server version 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you accidentally target a Microsoft Windows 2000 Professional-based computer
to play a Site role (CAP, DP, etc.) in the Systems Management Server
Administrator console, the computer appears as a Microsoft Windows NT
Server-based computer when you view the Site Systems node in the Systems
Management Server (SMS) console.

WORKAROUND
==========

Do not target Windows 2000 Professional-based computers for Site Server roles.

STATUS
======

Microsoft has confirmed this to be a problem in Systems Management Server
version 2.0.
