TITLE   :MapPoint 2001 Err Msg: Could Not Find the Microsoft MapPoint...
PRODUCT :Microsoft Automap
PROD/VER::
OPER/SYS:
KEYWORDS:kberrmsg kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft MapPoint 2002 
 - Microsoft MapPoint 2001 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to start Microsoft MapPoint 2001, you may receive the following
error message:

   Could not find the Microsoft MapPoint Run disc on this machine.

CAUSE
=====

This behavior can occur if the MapPoint 2001 Run CD-ROM is not inserted in the
CD-ROM drive.

RESOLUTION
==========

To resolve this issue, insert the MapPoint 2001 Run CD-ROM into the CD-ROM
drive.

Additional query words: mp2001 map point compact disk
