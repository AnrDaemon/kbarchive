TITLE   :Games: Games Do Not Run As Expected on Virtual PC
PRODUCT :Microsoft Home Games
PROD/VER::1.0,2.0
OPER/SYS:
KEYWORDS:kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Midtown Madness, version 1.0 
 - Microsoft MechWarrior 4: Vengeance 
 - Microsoft MechCommander 2.0, version 1.0 
 - Microsoft Monster Truck Madness 2, version 2.0 
 - Microsoft Midtown Madness 2, version 2.0 
 - Microsoft Metal Gear Solid, version 1.0 
 - Microsoft Motocross Madness, version 1.0 
 - Microsoft Motocross Madness 2, version 2.0 
 - Microsoft Asheron's Call, version 1.0 
 - Microsoft Asheron's Call: Dark Majesty, version 1.0 
 - Microsoft Baseball 2001 
 - Microsoft Baseball 2000 
 - Microsoft Baseball 3D, version 1.0 
 - Microsoft Hellbender for Windows, version 1.0 
 - Microsoft Outwars, version 1.0 
 - Microsoft Golf 1999 Edition, version 1.0 
 - Microsoft NFL Fever 2000, version 1.0 
 - Microsoft Flight Simulator 2000 
 - Microsoft Flight Simulator 2000 Professional Edition 
 - Microsoft Soccer, version 1.0 
 - Microsoft StarLancer, version 1.0 
 - Microsoft Crimson Skies 
 - Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
 - Microsoft Train Simulator, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to run any of the products listed at the beginning of this
article on a Macintosh computer that has Virtual PC installed, the games may not
run, or may not run as expected.

CAUSE
=====

Virtual PC does not meet the minimum requirements to run the game. For example,
Virtual PC does not support hardware accelerated video.
