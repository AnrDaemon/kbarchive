TITLE   :Shortcut Inserted as Object Loses Command-Line Options
PRODUCT :The Microsoft Network
PROD/VER:WINDOWS:1.2,1.3,2.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - The Microsoft Network versions 1.2, 1.3, 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you click Insert Object and insert a shortcut with command-line options into
a message, the command-line options are lost.

RESOLUTION
==========

Use the Insert File command or drag the shortcut into the message to insert the
shortcut into the message.

Additional query words: msn short cut inbox exchange
