TITLE   :Windows NT 3.5 Setup and the Pentium Pro Processor
PRODUCT :Microsoft Windows NT
PROD/VER:WinNT:3.5
OPER/SYS:
KEYWORDS:kb3rdparty kbhw kbsetup kbHardware

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.5 
 - Microsoft Windows NT Server version 3.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you perform a new installation of Windows NT version 3.5 on a system
containing an Intel Pentium Pro (P6) processor, you receive the following error
message at the beginning of the graphical portion of Windows NT setup:

   Setup Message:

   STOP: Setup cannot install on the current processor.  Cannot proceed
   with install.

The user must click OK. The user is then prompted to restart the system and abort
the setup attempt.

RESOLUTION
==========

Use the full (non-upgrade) version of Windows NT version 3.51 or later to
perform the installation.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 3.5. We are
researching this problem and will post new information here in the Microsoft
Knowledge Base as it becomes available.
