DOCUMENT:Q151671  09-AUG-2001  [winnt]
TITLE   :Incorrect Event ID 6 in the Event Viewer
PRODUCT :Microsoft Windows NT
PROD/VER:3.51
OPER/SYS:
KEYWORDS:kbsetup

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.51 
 - Microsoft Windows NT Server version 3.51 
-------------------------------------------------------------------------------


SUMMARY
=======

An event message may be incorrectly generated when you have a PC Card
installed.

   Event ID: 6
   Description: The IRQ of <n> was selected for a network PCMCIA PC Card.
                This IRQ is not supported for PC Cards on this system.
                For network service, please change the configuration.

MORE INFORMATION
================

You may find the PC Card is still operating regardless of the error. The error
can be ignored because it does not indicate any problem with the device.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 3.51. This
problem was corrected in Windows NT version 4.00.
