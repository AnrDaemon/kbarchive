TITLE   :Testnet.exe Only Available with BackOffice SMS
PRODUCT :Microsoft Systems Management Server

OPER/SYS:
KEYWORDS:kbnetwork kbsmsUtil smsutil

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Systems Management Server version 1.1 
-------------------------------------------------------------------------------

SUMMARY
=======

Testnet.exe is an executable file found in PSSTools on the BackOffice version of
SMS 1.1. It will not be found on the stand-alone SKU of SMS 1.1.

MORE INFORMATION
================

Testnet.exe is a command-line tool which enumerates users and user groups using
the same enumeration API calls made by SMS Site Configuration Manager. It is
useful for troubleshooting SCMan enumeration issues.
