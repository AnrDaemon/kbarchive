TITLE   :No Helpdesk Template under SMS Security Manager
PRODUCT :Microsoft Systems Management Server
PROD/VER:winnt:1.1
OPER/SYS:
KEYWORDS:kbnetwork kbdocerr smsdocerr

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Systems Management Server version 1.1 
-------------------------------------------------------------------------------

SYMPTOMS
========


The SMS Security Manager Online help and the SMS printed documentation state that
there is a "helpdesk" template. But when "templates" are selected, there is no
"helpdesk" template available.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Systems Management
Server Version 1.1 Documentation. We are researching this problem and will post
new information here in the Microsoft Knowledge Base as it becomes available.

Additional query words: prodsms
