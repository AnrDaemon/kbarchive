DOCUMENT:Q151792  09-FEB-2000  [exchange]
TITLE   :XFOR: One-off Addressed Mail from ccMail User Not Tracked
PRODUCT :Microsoft Exchange
PROD/VER:winnt:5.0
OPER/SYS:
KEYWORDS:kbusage exc5

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.0 
-------------------------------------------------------------------------------


SYMPTOMS
========

One-off recipients (for example, Internet addresses) from Lotus cc:Mail are not
written to the tracking log in the Gateway Message Transfer In (GMTI) Event. If
the one-off recipient is the only recipient, there is no tracking log entry. If
there are other Microsoft Exchange Server recipients, only the Exchange Server
recipients are written to the log.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server
version 5.0. This problem was corrected in the latest Microsoft Exchange Server
5.0 U.S. service pack. For information on obtaining the service pack, query on
the following word in the Microsoft Knowledge Base (without the spaces):

   S E R V P A C K
