TITLE   :MSB Earth: How to Start the Bus Stop Utility
PRODUCT :Microsoft Home Kids Products
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kbui kbimukbfaq

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Scholastic's Magic School Bus series: Explores Inside the Earth for Windows, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

When you install Magic School Bus Explores Inside the Earth (MSB Earth) for
Windows, no icon for the Bus Stop Utility is installed. However, you can gain
access to the Bus Stop Utility for MSB Earth from inside the Help file.

MORE INFORMATION
================

To gain access to the Bus Stop Utility for MSB Earth, use the appropriate method
for your operating system.

Microsoft Windows 95/98
-----------------------

To start the MSB Earth Bus Stop Utility, follow these steps:

1. Quit MSB Earth.

2. Click Start, point to Programs, point to Microsoft Kids, and then click Bus
   Help - Earth.

3. Click Troubleshooting, and then click Open.

4. Click Bus Stop Utility, and then click Display.

5. Click "Click here to start the Bus Stop Utility."

You can also start the MSB Earth Bus Stop Utility in Windows 95/98 by following
these steps:

1. Click Start, and then click Run.

2. In the Open box, type the following line (including the quotation marks), and
   then click OK:

   "C:\Program Files\Microsoft Kids\MSB Earth\Msbearth.exe" #70

   where C is the letter of the hard disk that contains the MSB Earth folder.

Microsoft Windows 3.x
---------------------

1. Quit MSB Earth.

2. In Program Manager, double-click the Microsoft Kids program group.

3. Double-click Bus Help - Earth.

4. Click Troubleshooting.

5. Click Bus Stop Utility.

6. Click the "Click here to start the Bus Stop Utility" button.

You can also start the MSB Earth Bus Stop Utility in Windows 3.x by following
these steps:

1. On the File menu in Program Manager, click Run.

2. In the Command Line box, type the following line, and then click OK:

   C:\Mskids\Msbearth\Msbearth.exe #70

   where C is the letter of the hard disk that contains the Msbearth folder.
