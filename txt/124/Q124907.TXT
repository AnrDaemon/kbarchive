TITLE   :How to Identify the CD Sampler Version Number
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER::Fall 1993 edition,Fall 1994 edition,Spring 1994
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Home CD Sampler, versions Fall 1993 edition, Fall 1994 edition, Spring 1994 
-------------------------------------------------------------------------------

SUMMARY
=======

The Microsoft Home CD Sampler version numbers are not printed on the individual
Sampler CDs. Determining the version number depends on the SETUP.EXE file date.

MORE INFORMATION
================

The table below lists the SETUP.EXE file dates.

No.   Version       Printed          Icon           EXE File   Setup
                     on CD                             Date     Engine
----------------------------------------------------------------------

1.0   Fall '93         N/A       Rainbow-colored CD   10/21/93   Setup
2.0   Spring '94   42-Product    House on Keyboard      5/6/94    ACME
3.0   Fall '94     55-Programs   House on Keyboard     2/20/94    ACME
