======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Complete NBA Basketball for Windows 1994-1995 edition 
 - Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia for Macintosh 
 - Microsoft Encarta 96 Encyclopedia for Windows 
 - Microsoft Windows Sound System, versions 2.0, 2.0a 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you are using the Sound Blaster Pro card with the Microsoft Windows Sound
System versions 2.0 or 2.0a, you may be unable to play certain sounds while
using Basketball or Encarta.

If you are using the Microsoft Windows 3.x operating system, you may get the
following general protection fault:

   Application Error - <filename> caused a General Protection Fault in
   module WSSSBPRO.DRV

If you are using Windows 95, the Play button may be unavailable (greyed out)
beneath affected sound files.

STATUS
======

Microsoft has confirmed this to be a problem in the Windows Sound System
versions 2.0 and 2.0a. We are researching this problem and will post new
information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION
================

To hear sounds correctly in the above applications, uninstall the Windows Sound
System drivers, and install the Sound Blaster Pro drivers provided by Creative
Labs.
