TITLE   :SMS Sender Properties Help Incorrectly References HERMSRHx
PRODUCT :Microsoft Systems Management Server
PROD/VER:winnt:1.0,1.1
OPER/SYS:
KEYWORDS:kbdocerr smsdocerr

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Systems Management Server versions 1.0, 1.1 
-------------------------------------------------------------------------------

SUMMARY
=======

The Microsoft Systems Management Server Sender Properties dialog box context
sensitive help says you should create HERMSRHx if it does not exist. The file
name is incorrect; you should create SMS_SHRx if it does not exist.

Additional query words: sms docerr doc err prodsms
