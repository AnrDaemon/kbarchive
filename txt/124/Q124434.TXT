The information in this article applies to:

 - Microsoft MS-DOS operating system version 6.22 
-------------------------------------------------------------------------------

SUMMARY
=======

The STEP<XXX>.EXE files (where <XXX> indicates the specific foreign
language version [for example, STEPFRN.EXE is the French version of Step-Up])
contain downloadable versions of Microsoft MS-DOS 6.22 Step-Up. To use MS-DOS
6.22 Step-Up, your computer must be running MS-DOS 6.0, 6.2, or 6.21 in the same
language as this version of the STEP<XXX>.EXE file.

MORE INFORMATION
================

The following files are available for download from the Microsoft Download
Center:

   DownloadDownload Stepfrn.exe now

   DownloadDownload Stepger.exe now

   DownloadDownload Steppor.exe now

   DownloadDownload Stepspa.exe now

NOTE: These files are: Stepfrn.exe (French), Stepger.exe (German), Steppor.exe
(Portuguese), Stepspa.exe (Spanish).

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

   Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

STEP<XXX>.EXE is a self-extracting executable file that contains the
following files:

   SETUP.BAT (This filename differs depending on the language)
   README.NOW (This filename differs depending on the language)
   1MSDOS62.EXE
   2MSDOS62.EXE
   3MSDOS62.EXE
