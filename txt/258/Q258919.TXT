The information in this article applies to:

 - Microsoft Pandora's Box, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes the cheat code that is available in Microsoft Pandora's
Box.

You can use this cheat code to receive extra hints and wildcards, or to move to
another level in the game.

MORE INFORMATION
================

While you are playing Pandora's Box, press and hold down the ALT key, and then
type "CHISLER" (without the quotation marks).

The Pandora Cheat dialog box appears in the upper-left corner of of the screen.
This dialog box contains the following spin boxes:

 - Level

 - WildCards

 - Hints

You can increase the value in each spin box to increase the number of available
hints and wildcards, or to move to another level in the game. Click OK to apply
the changes.

NOTE: To move to another level in the game after you increase the value in the
Level spin box, you must quit and then restart Pandora's Box.
