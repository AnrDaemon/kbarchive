SUMMARY
=======

This article discusses the purpose, usage, and known support issues for the
SuperDocStatistics macro supplied with Microsoft Word in the Macros8.dot file.
This macro can be used to troubleshoot a document by displaying several document
attributes in one dialog box at one time.

MORE INFORMATION
================

Purpose and Features of Macro
-----------------------------

 - Shows information about the following categories: General document properties
   and grammar statistics, font used, styles in use, sections and section
   format, hyperlinks, table information, and fields.

 - Can generate a report on each of the topics above.

 - Allows you to go to document items such as fields, tables, sections, and so
   on.

Known Issues
------------

The Dialog tab name disappears after you click on a different menu tab. When you
click in the general area of the tab, the name reappears. This does not affect
the macro's functionality.
