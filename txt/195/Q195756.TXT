TITLE   :Err Msg: Your System Is Missing the Required Setup Engine
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:WINDOWS:95
OPER/SYS:
KEYWORDS:kbenv kberrmsg kbtool kbimukbfaq

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta Encyclopedia 99 
 - Microsoft Encarta Reference Suite 99 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to install the Microsoft Reference Troubleshooters from one of
the products listed earlier in this article, you may receive the following error
message:

   Your system is missing the required Setup engine.

CAUSE
=====

This behavior can occur if you do not have a Web browser installed on your
computer.

NOTE: To install the Microsoft Reference Troubleshooters, you must have Microsoft
Internet Explorer 3.02 or later installed on your computer.

RESOLUTION
==========

To resolve this issue, install Internet Explorer 4.0 or the ActiveX distribution
files from the Installation and Resources CD-ROM for the program, and then
install the Microsoft Reference Troubleshooters. To do this, use the appropriate
method.

Installing Internet Explorer 4.0
--------------------------------

1. Insert the Installation and Resources CD-ROM for your program into your
   CD-ROM drive. Press and hold down the SHIFT key when you insert the CD-ROM to
   prevent the program from starting automatically.

2. Double-click the My Computer icon on the desktop.

3. Right-click the CD-ROM drive containing the Installation and Resources
   CD-ROM, and then click Explore.

4. Double-click the Ie4 folder.

5. Double-click the Ie4setup.exe file, and then follow the instructions on the
   screen to install Internet Explorer.

   NOTE: If you are prompted to restart the computer, do so.

6. Install the Microsoft Reference Troubleshooters.

Installing the ActiveX Distribution Files
-----------------------------------------

To install the ActiveX distribution files from the Installation and Resources
CD-ROM, follow these steps:

1. Insert the Installation and Resources CD-ROM for your program into your
   CD-ROM drive. Press and hold down the SHIFT key when you insert the CD-ROM to
   prevent the program from starting automatically.

2. Click Start, and then click Run.

3. In the Open box, type the following line, and then click OK:

   <cdrom>:\aref\system\axdist.exe

   where <cdrom> is the letter of your CD-ROM drive.

4. Follow the instructions on the screen to install the ActiveX distribution
   files.

5. Install the Microsoft Reference Troubleshooters.
