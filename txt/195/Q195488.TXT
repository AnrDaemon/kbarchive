The information in this article applies to:

 - Microsoft Visual Basic Enterprise Edition for Windows, versions 5.0, 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article shows where to find and how to install Visual Modeler for Visual
Basic 5.0 and 6.0.

MORE INFORMATION
================

Visual Modeler for Visual Basic 5.0 is only available from the Web site for
Enterprise Edition owners.

To install Visual Modeler for Visual Basic 5.0:

1. Register at the Microsoft Web site for Visual Basic:

   http://msdn.microsoft.com/vstudio/register/vb6.htm

2. Click the appropriate registration wizard, such as Visual Basic 5.0
   Registration Wizard.

3. Fill out the appropriate information in the registration form.

4. When asked for the PID (Product ID), you may enter the Serial number located
   in the About box for your product.

5. After completing the registration form, you will be redirected to the
   Microsoft Home page. Navigate back to the registration page in step 1.

6. Navigate to the download site for Visual Modeler for Visual Basic 5 at:

   http://premium.microsoft.com/vbasic/updates.htm

   Download and install.

   Information on Visual Modeler can also be obtained at this site.

   NOTE: Visual Modeler 2.0 may be installed from the Installation CD for Visual
   Studio 6.0 under <Enterprise Tools>.
