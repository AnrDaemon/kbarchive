DOCUMENT:Q195086  24-NOV-2000  [vbwin]
TITLE   :INFO: File REGMSVM.BAT is Missing in Visual Modeler 1.1 Release
PRODUCT :Microsoft Visual Basic for Windows
PROD/VER::5.0
OPER/SYS:
KEYWORDS:kbnokeyword kbVBp500 kbVMod kbGrpDSVBDB kbVMod110

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Visual Basic Enterprise Edition for Windows, version 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

In Visual Modeler 1.1, the documentation indicates that file regmsvm.bat is
located at the Visual Modeler installation folder and to use this file to
register appropriate VM DLLs. However, regmsvm.bat is not included in Visual
Modeler 1.1.

The regmsvm.bat file is available from the Visual Basic Owner's area. You must
register your Visual Basic Enterprise edition before being allowed to enter the
Owner's area and download the regmsvm.bat file.

The location of the Visual Basic Owner's area is:

   http://premium.microsoft.com/vbasic/updates.htm

Click on the Visual Modeler link and go to the section titled Regmsvm.bat to
download the file.

MORE INFORMATION
================

In order to install the Visual Modeler 1.1 product, you must have a licensed
copy of Visual Basic 5.0 Enterprise edition.

Additional query words: Visual Modeler REGMSVM.BAT REGMSVM
