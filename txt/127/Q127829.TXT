TITLE   :Automap Road Atlas Err Msg: Cannot Find an Existing Version...
PRODUCT :Microsoft Automap
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Automap Road Atlas for Windows, versions 1.06, 1.1 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run the Update installation program for Automap Road Atlas version 3.04
or 3.05 for Windows, you may receive the error message:

   Cannot find an existing version of Automap Road Atlas

The error occurs when you attempt to install the Update for Automap Road Atlas
version 3.04 or 3.05 for Windows to an existing copy of Automap Road Atlas
version 1.06 and 1.1 for Windows.

RESOLUTION
==========

Install Automap Road Atlas version 3.0, 3.02, 3.03, or 3.04 for Windows.

MORE INFORMATION
================

When the Update for Automap Road Atlas version 3.04 or 3.05 for Windows is
installed, the program attempts to locate the current version of Automap Road
Atlas for Windows version 3.0 or later. To update the program, version 3.0,
3.02, or 3.04 must exist on the computer.
