TITLE   :Unrecognizable Characters Displayed Using Remote Control
PRODUCT :Microsoft Systems Management Server
PROD/VER:winnt:1.0,1.1,1.2
OPER/SYS:
KEYWORDS:kb3rdparty smsgeneral kbArtTypeINF

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Systems Management Server versions 1.0, 1.1, 1.2 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you take remote control of a client computer from Help Desk in Microsoft
Systems Management Server, unrecognizable characters are displayed when you
start a third-party 3270 terminal emulation program for Windows.

CAUSE
=====

This problem occurs when a font used on the client computer is not available in
the Windows NT system.

RESOLUTION
==========

To correct this problem, use a different font that is available on the Windows
NT system or install the same font in Windows NT.

Microsoft; we make no warranty, implied or otherwise, regarding this product's
performance or reliability.
