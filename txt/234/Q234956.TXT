-------------------------------------------------------------------------------
The information in this article applies to:

 - MSPRESS Desktop Applications for Microsoft Visual Basic 6.0 MCSD Training Kit ISBN 0-7356-0620-X 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to test the application created in Exercise 1, you will receive
the following error message:
"Can not find the \Labs\Lab12\helpfile.hlp file. Do you want to try to find this
file yourself?"

CAUSE
=====

The helpfile.hlp file is missing from the CD-ROM that accompanies the training
kit.

RESOLUTION
==========

Microsoft Press has prepared a self-extracting executable file called
VBMCSD1.EXE that contains the missing helpfile.hlp file.

The following file is available for download from the Microsoft Download Center:

   VBMCSD1.exe
   (http://download.microsoft.com/download/vba6/MSPRESS/1.0/WIN98Me/EN-US/VBMCSD1.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

   Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

STATUS
======

Microsoft has confirmed this to be a problem in Desktop Applications with
Microsoft Visual Basic 6.0 MCSD Training Kit for Exam 70-176.
