DOCUMENT:Q234405  09-FEB-2000  [exchange]
TITLE   :XFOR: GroupWise API Gateway Expands All DLs to the To Box
PRODUCT :Microsoft Exchange
PROD/VER:winnt:5.5
OPER/SYS:
KEYWORDS:exc55

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------


SYMPTOMS
========

When a Microsoft Outlook user sends a message to Novell GroupWise and includes a
GroupWise distribution list (DL) in the Cc or Bcc box, the members of the
distribution list are expanded to the To box; they are not preserved in the box
where the Outlook user originally included the distribution list.

CAUSE
=====

The Novell GroupWise Application Programming Interface (API) Gateway expands the
distribution list into the To box.

RESOLUTION
==========

This behavior is by GroupWise product design. For additional information, refer
to the following Novell Web site for additional information:

   http://support.novell.com
