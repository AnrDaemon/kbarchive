TITLE   :Cannot Use Defragment Now Button in Drive Properties
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:kbtool

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0 
 - Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Defragment Now button on the Tools tab in the properties for a drive may be
unavailable.

CAUSE
=====

This behavior occurs if there is no disk defragmenter tool installed. Windows NT
4.0 does not include a disk defragmenter tool.

RESOLUTION
==========

To use the Defragment Now button, you must first install a third-party disk
defragmenter tool.
