TITLE   :WD97: Column Paragraph Bordering May Print Unevenly
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:kbprint kbPrinting

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you print a document created in Microsoft Word 6.0/7.0 that contains
paragraph bordering within columns, the bottom paragraph borders between the
columns may print uneven.

STATUS
======

Microsoft has confirmed this to be a problem in the products listed at the
beginning of this article. We are researching this problem and will post new
information here in the Microsoft Knowledge Base as it becomes available.


WORKAROUND
==========

Manually insert a column break in the text where the new column begins. Use the
following procedure to insert the column break:

1. Column one should be correct and need no modification.

2. Place the insertion point at the start of the next column.

3. On the Insert menu, click Break.

4. Select Column Break and click OK.

5. Repeat steps 1 through 4 for each additional column.

Additional query words: 8.0 word97 misaligned
