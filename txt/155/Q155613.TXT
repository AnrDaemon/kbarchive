TITLE   :AMD Nx586 Processor Incompatible with Windows NT 4.0
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:kbhw kbsetup kbHardware

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0 
 - Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

When you are installing Windows NT 4.0 on a computer using the Advanced Micro
Devices (AMD) Nx586 processor, the following message is displayed:

   Windows NT requires a 486 or later processor

MORE INFORMATION
================

The AMD Nx586 is based on the 80386 processor instruction set. This processor
does not support the minimum 486 instruction set which is required to use
Windows NT 4.0.
