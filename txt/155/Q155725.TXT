DOCUMENT:Q155725  09-AUG-2001  [winnt]
TITLE   :Description Window in Devices Tool Is Too Small
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:kbtool kbui

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0 
 - Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

In the Devices tool in Control Panel, the window for the description of devices
is too small for some descriptions to be read fully, and there is no horizontal
scroll bar to let you see the rest of the description.

STATUS
======

This is a known problem with Windows NT 4.0.
