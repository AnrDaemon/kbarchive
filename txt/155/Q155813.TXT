DOCUMENT:Q155813  04-APR-1999  [exchange]
TITLE   :XADM: Simplified Chinese MS Mail Connector Corrupts Messages
PRODUCT :Microsoft Exchange
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:kbusage

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

In the Simplified Chinese environment, the Simplified Chinese version of the
Microsoft Exchange Microsoft Mail Connector is used to connect Microsoft
Exchange to the Simplified Chinese version of Microsoft Mail 3.x. When you
exchange messages with Chinese characters in the header and body between a
Simplified Chinese Microsoft Exchange Windows 95 client and a Simplified Chinese
Microsoft Mail Windows client, the Chinese characters will be corrupt at the
destination.


STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server
version 4.0. This problem was corrected in the latest Microsoft Exchange Service
Pack. For information on obtaining the Service Pack, query on the following word
in the Microsoft Knowledge Base (without the spaces):

   S E R V P A C K


Additional query words: DBCS CHS
