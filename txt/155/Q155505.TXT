SUMMARY
=======

The following article lists the minimum system requirements to run the 1997
edition of Microsoft Cinemania.

MORE INFORMATION
================

The minimum system requirements needed to run Cinemania 97 are:

 - A Multimedia PC with a 486/33 MHz or higher processor

 - Microsoft Windows 95 operating system or Windows NT Workstation operating
   system version 3.51 or later

 - 8 MB of RAM when running on Windows 95 (16 MB for Windows NT)

 - 256-color or higher-resolution monitor

 - Hard Disk space required:
    - 500k for best performance

    - 6 MB needed for monthly updates


 - Double-speed or faster CD-ROM drive

 - Audio board and speakers required for sound

 - Microsoft Mouse or compatible pointing device

 - Windows-compatible modem and online service required for online monthly
   update service
