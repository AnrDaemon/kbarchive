TITLE   :Canvas 95 Does Not Run in Windows NT
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:kb3rdparty

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0 
 - Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you install Canvas 95 in Windows NT 4.0, you cannot start Canvas 95. You
do not receive any error message.

CAUSE
=====

Canvas 95 is designed to work with Windows 95 explicitly. The Windows 95 version
does not work with Windows NT 3.51 or 4.0. A separate version of Canvas that
runs with Windows NT is available.

RESOLUTION
==========

Contact Deneba Software regarding Canvas and Windows NT.
