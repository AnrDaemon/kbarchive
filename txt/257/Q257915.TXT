DOCUMENT:Q257915  25-JUN-2000  [homemm]
TITLE   :Encarta 99 RO: Text in Field 2 on Notecard Appears as Asterisks
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:WINDOWS:
OPER/SYS:
KEYWORDS:kbdisplay kbenv kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta 99 Research Organizer 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you type in Field 2 on a notecard in Microsoft Encarta Research Organizer
99, the text in the field may appear as asterisks.

CAUSE
=====

This behavior can occur if a newer version of the Riched32.dll file is installed
on your computer.

RESOLUTION
==========

To resolve this issue, upgrade to Encarta Encyclopedia 2000.

To work around this issue, extract a copy of the Riched32.dll file from the
Microsoft Windows CD-ROM or installation disks to the Windows\System folder on
your hard disk.

NOTE: When you install an older version of the Riched32.dll file, you may cause
problems with other programs that use the updated version of this file. If this
is the case, restore the newer version of the Riched32.dll file, or reinstall
the program that requires the newer version of the file.

The following table lists the location and the version number of the Riched32.dll
file for versions and installation medai of Microsoft Windows 95 and Microsoft
Windows 98.

   Version                  Media             Location       File version
   ----------------------------------------------------------------------

   Windows 95               DMF disks         Win95_04.cab   4.0
                                              (Disk 4)

   Windows 95               Non-DMF disks     Win95_06.cab   4.0
                                              (Disk 6)

   Windows 95 (retail)      CD-ROM            Win95_04.cab   4.0

   Windows 95 OEM Service   CD-ROM            Win95_05.cab   4.00.834.839
   Release 2 (OSR2)

   Windows 98               CD-ROM            Win98_37.cab   4.00.834.839

   Windows 98 Second        CD-ROM            Win98_40.cab   4.00.834.839
   Edition(SE)
NOTE: If you are prompted to overwrite an existing file, do so.

For additional information about how to extract files, click the article number
below to view the article in the Microsoft Knowledge Base:

   Q129605 How to Extract Original Compressed Windows Files

STATUS
======

Microsoft has confirmed that this is a problem in the Microsoft products that
are listed at the beginning of this article.
