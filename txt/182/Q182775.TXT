 - Microsoft Encarta Virtual Globe 1998 edition 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the minimum system requirements needed to run Microsoft
Encarta Virtual Globe, 1998 edition.

MORE INFORMATION
================

To run Microsoft Encarta Virtual Globe, 1998 edition, you need:

 - A multimedia PC with a 486DX/50 MHz or higher processor.

 - Microsoft Windows 95 or Microsoft Windows NT workstation, version 4.0 or
   later.

 - 8 MB of memory (RAM) for Windows 95 (16 MB recommended); 16 MB for Windows NT
   workstation (24 MB recommended).

 - 29 MB of available hard disk space (you may need additional hard disk space
   during installation - 50 MB of available hard disk recommended).

 - A double-speed or faster CD-ROM drive.

 - A Super VGA, 256 color monitor that supports 640 x 480 or higher resolution.

 - A Microsoft Mouse, Microsoft IntelliMouse, or compatible pointing device.

 - Microsoft Internet Explorer 3.02 (included on the Encarta Virtual Globe
   CD-ROM) or later (after you install Internet Explorer, you can use another
   browser as your default browser; Internet Explorer 3.02 is compatible with
   Netscape Navigator version 2.0 or later).

To use Microsoft Encarta Web Links, you need:

 - A 9600-baud or faster modem.

 - An Internet service provider (ISP) and access to the Internet.
