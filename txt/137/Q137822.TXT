TITLE   :Printing to a File Initiates the Fax Wizard
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you print to a file using Notepad or WordPad, the Send Fax Wizard may
appear, and Print To Fax may be the only available action.

CAUSE
=====

This behavior can occur if the Microsoft Fax printer driver has been set as the
default printer driver.

RESOLUTION
==========

Change the default printer driver to another printer driver. To do so, follow
these steps:

1. Click the Start button, point to Settings, and then click Printers.

2. Use the right mouse button to click the printer you want to use as the
   default printer, and then click Set As Default on the menu that appears.
