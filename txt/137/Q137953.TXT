TITLE   :DOCERR: Visual FoxPro Help File Gives Wrong Index Key Limit
PRODUCT :Microsoft FoxPro
PROD/VER:WINDOWS:3.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Visual FoxPro for Windows, version 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Visual FoxPro Help file under the section "Error Messages A-Z, Invalid Key
Length" incorrectly states that the Index Key limit is 254 characters . This
value should be 240 characters.

MORE INFORMATION
================

The Index Key limit has not changed from previous versions of FoxPro. This
documentation error occurs only in this topic in the Help file. The correct
value is stated in the chart on page 168 of the Developer's Guide.
