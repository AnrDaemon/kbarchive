DOCUMENT:Q137185  08-NOV-2001  [homemm]
TITLE   :Julia Child: Tip Popup Text Prints Small
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kbdisplay kbmm kbui

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Julia Child: Home Cooking with Master Chefs for Windows, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When a Tip popup is printed, the text is printed in a smaller font than other
popup text, such as Recipes, Notes, ShopList, Ingredient, Cookware, and Tour.

STATUS
======

Microsoft has confirmed this to be a problem in Julia Child: Home Cooking with
Master Chefs version 1.0 for Windows.

WORKAROUND
==========

The Tips text displays directly from the application at an approximate 6-point
size.

To print the tip in another font, use the steps below for your version of
Windows. For more information about copying and pasting text, see your Windows
printed documentation or online help.

Windows 95
----------

Copy, then paste the text into Microsoft Windows 95 WordPad, and then print from
WordPad.

NOTE: This issue does not occur in Windows 98.

Windows 3.x
-----------

Copy, then paste the text into Write or Notepad and print from that application.
