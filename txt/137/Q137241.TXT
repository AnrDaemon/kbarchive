TITLE   :CPU Type Information Different in Windows NT Diagnostics
PRODUCT :Microsoft Windows NT
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation versions 3.5, 3.51 
 - Microsoft Windows NT Server versions 3.5, 3.51 
-------------------------------------------------------------------------------

The CPU Type information in Windows NT Diagnostics (WINMSD.EXE) Hardware
is different between Windows NT 3.5 and 3.51:

   Processor      CPU Type (Windows NT 3.5)     CPU Type (Windows NT 3.51)
   ---------      -------------------------     --------------------------

   Intel 486          Intel 486                     Intel x86
   Intel Pentium      Intel Pentium                 Intel x86
