SUMMARY
=======

The following information is a copy of the information in the Readme.txt file in
the Drivers\Storage\Iomega folder on the Windows 95 CD-ROM. Setup does not copy
this file to your hard disk.

MORE INFORMATION
================

The Zip drivers included with Windows 95 provide basic access to the Zip drive
for reading and writing. For full Zip functionality in Windows 95, contact
Iomega at 1-800-456-5522 or 1-801-778-3000 to obtain the Zip Tools for Windows
95. Zip Tools provide utilities for formatting, read/write protection, disk
cataloging, disk copying and other useful functions.
