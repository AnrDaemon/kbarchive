TITLE   :Cannot Back Up Mapped Banyan Drives
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:95
OPER/SYS:
KEYWORDS:3rdpartynet win95 kb3rdPartyNetClient

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

You cannot backup mapped Banyan drives using Microsoft Backup. You can see all
shared volumes on the Banyan Vines server, but you cannot see the contents of
the share.

CAUSE
=====

Backup does not support backing up to or from drives mapped to a Banyan Vines
server.

MORE INFORMATION
================

The Backup tool included with Windows 95 is designed to provide basic backup
functionality, and has limitations when used as a server backup tool.
