TITLE   :Add/Remove Programs Tool Displays Incorrect Available Space
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:95
OPER/SYS:
KEYWORDS:win95

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use the Add/Remove Programs tool in Control Panel, the Space Required
and the Space Available On Disk values may be incorrect.

In addition, selecting different components to add or remove may not cause these
values to be updated.


RESOLUTION
==========

Proceed with the addition or removal of the selected Windows 95 components. The
incorrect values do not prevent the Add/Remove Programs tool from adding or
removing components.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Windows 95. We are
researching this problem and will post new information here in the Microsoft
Knowledge Base as it becomes available.

MORE INFORMATION
================

The Add/Remove Programs tool may indicate that some disk space is required, even
if you have not selected any components to add. The value that is displayed
varies from one computer to another.
