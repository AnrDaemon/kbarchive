TITLE   :Windows 3.1 File Manager Does Not Refresh After Sizing Windows
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 3.1 
-------------------------------------------------------------------------------

SYMPTOMS
========

Random characters may be left in the Microsoft Windows version 3.1 File Manager
window after resizing the File Manager windows.

RESOLUTION
==========

To remove any random characters, press the F5 key to refresh the File Manager
screen.

STATUS
======

Microsoft has confirmed this to be a problem in Windows operating system version
3.1 File Manager. We are researching this problem and will post new information
here as it becomes available.

Additional query words: 3.1 3.10 repaint refresh resize winfile
