TITLE   :Write: Save As Leaves the Find Dialog Box on the Screen
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
 - Microsoft Windows for Workgroups versions 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use Windows Write version 3.1, the Find dialog box may remain on the
screen after you save a file. If you save a file by choosing Save As from the
File menu before you close the Find dialog box, the Find dialog box is not
cleared from the screen.

STATUS
======

Microsoft has confirmed this to be a problem in Windows and Windows for
Workgroups. We are researching this problem and will post new information here
as it becomes available.

Additional query words: 3.1 3.11 3.10
