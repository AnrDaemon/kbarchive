The information in this article applies to:

 - Microsoft Windows 3.1 
-------------------------------------------------------------------------------

SUMMARY
=======

The Interaction Systems Inc. (ISI) touch screen works in conjunction with a
mouse. Interaction Systems driver version 2.2 is required for use with Microsoft
Windows operating system version 3.1.

MORE INFORMATION
================

If Windows 3.1 is installed on a system using earlier versions of Interactions
Systems drivers, the keyboard may be unusable. Interaction Systems makes a touch
screen that works in conjunction with a conventional mouse. If you set up
Windows version 3.1 using the ISI touch screen and mouse driver version 2.0, you
may not be able to use the keyboard. The disk from Interaction Systems is
labeled "Installation Disk for Windows 3.0 driver version 2.0. Touchscreen
driver version 1.0."

An updated driver is available from Interaction Systems.

The ISI product included here is manufactured by a vendor independent of
Microsoft; we make no warranty, implied or otherwise, regarding this product's
performance or reliability.
