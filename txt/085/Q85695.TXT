TITLE   :PC Win: Two OLE Objects on One Line Print on Two Lines
PRODUCT :Microsoft Mail For PC Networks
PROD/VER:WINDOWS:3.0,3.0b,3.2
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for Windows, versions 3.0, 3.0b, 3.2 
-------------------------------------------------------------------------------

SYMPTOMS
========

Multiple object linking and embedding (OLE) objects placed on the same line in
version 3.0, 3.0b, or 3.2 of Microsoft Mail for Windows are printed on separate
lines.

WORKAROUND
==========

Compose the message in Microsoft Word for Windows with the embedded objects,
then send the message as a Word attachment. The objects then print from Word
correctly.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Mail for Windows
versions 3.0, 3.0b, and 3.2. We are researching this problem and will post new
information here in the Microsoft Knowledge Base as it becomes available.
