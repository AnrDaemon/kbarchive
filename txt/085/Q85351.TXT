TITLE   :FAX: MS-DOS Version Requirement for the Fax Gateway
PRODUCT :Microsoft Mail For PC Networks
PROD/VER:MS-DOS:3.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail Gateway to Fax, version 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Microsoft Mail Gateway to Fax will run using MS-DOS versions 3.3 and later,
but using MS-DOS 5.0 to load network drivers into high memory increases the
stability of the gateway.

REFERENCES
==========

"Microsoft Mail Gateway to Fax Administrator's Guide"
