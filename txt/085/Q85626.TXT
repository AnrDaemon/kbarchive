TITLE   :Canon BJ-130e Driver Causes Continuous Form Feeds
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you use the Canon BJ-130e printer driver version 1.0 included with
Microsoft Windows operating system version 3.1, continuous form feeds result
when the paper source is set to "tractor".

WORKAROUND
==========

To work around this problem, use the Canon BJ-330 driver supplied with Windows
3.1.

According to Canon technical support, the BJ-330 driver supports all the features
of the BJ-130e printer.

MORE INFORMATION
================

Steps to Reproduce this Problem
-------------------------------

1. Run the Control Panel and install the Canon BJ-130e driver.

2. Choose Setup and set Paper Source to Tractor.

3. Exit the Control Panel and run Windows Write.

4. Print from Write.

5. Take the printer offline and press the Form Feed button. The paper will form
   feed continuously until the printer is turned off.

Additional query words: 3.10 form feed bj 3.11
