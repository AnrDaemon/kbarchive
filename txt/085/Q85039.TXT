SUMMARY
=======

The File Format Application Programming Interface (FFAPI) is a set of executable
programs which, when provided with a correctly formatted file, inserts mail into
the Microsoft Mail database, and extracts mail from a specified user's mailbag.

FFAPI is provided in three formats: Gateway FFAPI, Application FFAPI, and Remote
FFAPI.
