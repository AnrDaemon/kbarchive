-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail Gateway to Fax, version 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Here is a list of files included on the Microsoft Mail Gateway to Fax floppy
disks.

Gateway Component
-----------------

   Filename         Description
   --------         -----------

   DCX2TIF.EXE      DCX to TIFF conversion program
   FAXATT.TPL
   FAXCOV.TPL       Template for fax logo sheet
   FAXGTW.EXE       Fax gateway program
   FAXLOGO.PCX      Fax logo sheet
   INSTALL.EXE      Installation program
   MAKEDCX.EXE      Creates DCX file from multiple PCX files
   PROFILE.EXE      Profile editor
   PROFILE.GLB      Profile configuration file
   RRS.FAX          Status file
   TIF2DCX.EXE      TIFF to DCX conversion program

Access Component
----------------

   Filename         Description
   --------         -----------

   FAX.HLP          Help file
   FAX.TPL          Template for entering fax addresses
   FAXSTD.TPL       Template for entering fax addresses
   INSTALL.EXE      Installation program

Fax Viewer
----------

   Filename         Description
   --------         -----------

   FAXVIEW.EXE      Fax view program
