TITLE   :Cannot Change Paper Sizes with Canon Bubble-Jet 10e
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------


SYMPTOMS
========

You are unable to change the paper size for a Canon Bubble-Jet 10e. The Paper
Size option is unavailable (dimmed) for the Canon Bubble-Jet 10e in any
application that uses the Windows 3.1 common dialog boxes (COMMDLG.DLL), such as
Windows Write.

WORKAROUND
==========

To work around this problem, change the paper size using the Printers icon in
Control Panel before printing the document.


STATUS
======

Microsoft has confirmed this to be a problem in CANON10E.DRV version 1.0. We are
researching this problem and will post new information here as it becomes
available.


Additional query words: 3.10 3.11 greyed grayed cannon
