TITLE   :Directly Accessing Control Panel Options
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a,3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

You can access part of the Control Panel directly without first selecting the
Control Panel icon.

MORE INFORMATION
================

To access Control Panel options without opening Control Panel, do the
following:

1. From the Program Manager File menu, choose Run.

2. In the Command Line box, enter

   control <options>

   where <options> is one of the Control Panel selections:

   Color
   Fonts
   Ports
   Mouse
   Desktop
   Keyboard
   Printers
   International
   Date/Time
   Network
   Drivers
   Sound
   386 Enhanced

3. Choose the OK button.

NOTE: Windows 3.0 Control Panel does not contain the Drivers and Sound icons.
