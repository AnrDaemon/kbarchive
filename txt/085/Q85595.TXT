TITLE   :Toshiba PageLaser 12 May Require Updated Version of ROM
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------


SUMMARY
=======

If you are having problems printing TrueType fonts from Microsoft Windows 3.1 to
a Toshiba PageLaser 12, Toshiba Technical Support recommends obtaining the
printer's latest ROM revision, version 3.8b. This problem may occur only in
landscape orientation.

To obtain the version 3.8b ROM, contact Toshiba Technical Support.

MORE INFORMATION
================

To print TrueType fonts with earlier ROM versions, try one of the following
workarounds:

 - Print TrueType fonts as graphics:

   a. Run Control Panel.

   b. Choose the Printers icon.

   c. Select the Toshiba PageLaser 12.

   d. Choose the Setup button, then choose the Options button.

   e. Select the Print TrueType Fonts As Graphics check box.

-or-

 - Install and use the Hewlett-Packard (HP) LaserJet 500 driver. Note that using
   this driver disables some of the Toshiba's paper-handling features.


Additional query words: 3.10 paper handling 3.11
