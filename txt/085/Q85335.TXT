TITLE   :FAX: Description of Fax Gateway Component
PRODUCT :Microsoft Mail For PC Networks
PROD/VER:MS-DOS:3.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail Gateway to Fax, version 3.0 
-------------------------------------------------------------------------------

The gateway component of Microsoft Mail Gateway to Fax allows the
administrator of the gateway postoffice to administer the Fax Gateway,
enter fax addresses, and set up audit information.

The gateway component must be installed on the gateway postoffice
before the access component is installed.

The gateway component also contains the gateway executable program,
FAXGTW.EXE, which is run on the dedicated gateway PC.

The "A Brief Introduction" chapter of the "Microsoft Mail Gateway to Fax
Administrator's Guide"
