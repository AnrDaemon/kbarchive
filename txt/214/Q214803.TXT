TITLE   :INFO: System Requirements for Windows CE Toolkit for VB 6.0

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows CE Toolkit for Visual Basic 6.0, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

In order to run the Windows CE Toolkit for Visual Basic 6.0 on a device, or in
emulation, you need the following items:

 - A computer with a Pentium class processor; Pentium 90-MHz or higher processor
   recommended.

 - Microsoft Windows 95 or later operating system or Microsoft Windows NT®
   operating system version 4.0 with Service Pack 3; Windows NT Workstation 4.0
   is required for emulation.

 - Microsoft Visual Basic, Professional or Enterprise Edition, version 6.0
   (Visual Studio 6.0 Service Pack 1 or later recommended).

 - 16 MB of RAM for Windows 95 or later (32 MB recommended); 24 MB for Windows
   NT 4.0 (32 MB recommended).

 - CD-ROM drive compatible with multimedia PC specification.

 - VGA or higher-resolution monitor required; Super VGA monitor recommended.

 - Microsoft Mouse or compatible pointing device.

Hard Disk Space Requirements
----------------------------

 - Minimum installation: 60 MB

 - Complete installation: 145 MB (300 MB required if installing online
   documentation.)

NOTE: You need Windows NT to run emulation because Windows CE is Unicode only and
Windows NT provides the necessary Unicode environment.

A Serial port is required for debugging support.
