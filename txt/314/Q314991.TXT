TITLE   :INFO: VB 6.0 Does Not Support Applications for Multiple Monitors
PRODUCT :Microsoft Visual Basic for Windows
PROD/VER::6.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Visual Basic Enterprise Edition for Windows, version 6.0 
 - Microsoft Visual Basic Professional Edition for Windows, version 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft does not support the use of Visual Basic 6.0 to design applications
that are intended for environments with multiple monitors.

MORE INFORMATION
================

You can use the Visual Basic 6.0 Integrated Development Environment (IDE) in an
environment that has multiple monitors.

Additional query words: multi-monitor environment
