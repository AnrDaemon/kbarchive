======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Bookshelf 1996-97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

In Bookshelf, when you click the Tools menu Copy To PowerPoint option, you
receive the following error message:

   Microsoft PowerPoint!
   Run time error '440':
   OLE Automation error

The error occurs if you attempt to copy a pronunciation to PowerPoint when Power
Point is not running.

RESOLUTION
==========

Start PowerPoint before you use the Copy To PowerPoint option.
