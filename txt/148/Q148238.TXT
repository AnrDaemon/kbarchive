TITLE   :Bookshelf 96-97: Remove All Deletes Internet Directory Updates
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:WINDOWS:
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Bookshelf 1996-97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you choose the Remove All option in Bookshelf '96-'97, the Setup routine
deletes the Internet Directory updates.

RESOLUTION
==========

NOTE: These updates are optional.

Reinstall Bookshelf and copy the Internet Directory update file from your Web
browser folder to the Bookshelf folder. The following is the typical locations
for the file. Use the folder path for your version of Windows:

Microsoft Windows 95:

   C:\Program Files\Microsoft Reference\Bookshelf 96

Microsoft Windows 3.x:

   C:\Books96

For more information about how to perform this task in Windows, see your Windows
printed documentation or online Help.

The file is named Bs96ingn.lst. If the file has been deleted from your computer's
Web browser folder, you need to download the file from the Internet.

The World Wide Web (www) location is:

   http://www.microsoft.com/bookshelf/updates.htm

MORE INFORMATION
================

Bookshelf '96-'97 allows you to download Internet Directory updates from the
World Wide Web. If you run Setup and click Remove All. this updated file is
removed.

To download the Internet Directory Updates, do the following:

1. Start Bookshelf.

2. On the Books menu, click Internet Directory 96.

3. Click the Update Your Internet Directory button.

   In Windows 95, your Web browser starts automatically and link to the Bookshelf
   Web site.

   In Windows 3.x, start your Web browser manually and enter the following link:

   http://www.microsoft.com/bookshelf/updates.htm

4. After your browser connects to the link, follow the on-screen instructions.
