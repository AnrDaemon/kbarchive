TITLE   :Windows 95 Service Pack 1 Files Not Found After Setup
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:95
OPER/SYS:
KEYWORDS:win95

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you install Microsoft Windows 95 Service Pack 1, some or all of the files
may not be found or may not work correctly.

CAUSE
=====

This behavior can occur if you cancel Windows 95 Service Pack 1 Setup while it
is copying files. If you do so, you receive the following message:

   Windows 95 Service Pack 1 Update completed successfully.

   These changes will not take place until you shutdown and restart.
   Please restart now.

RESOLUTION
==========

Run Windows 95 Service Pack 1 Setup again. Allow Setup to finish normally.

STATUS
======

Microsoft is researching this problem and will post new information here in the
Microsoft Knowledge Base as it becomes available.
