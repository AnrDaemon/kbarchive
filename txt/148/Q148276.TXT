SUMMARY
=======

The following article contains a copy of the minimum system requirements for
Microsoft Isaac Asimov's The Ultimate Robot for Windows, as listed on the
product box.

MORE INFORMATION
================

The minimum system requirements to run The Ultimate Robot are:

 - A Multimedia PC or compatible with a 386SX or higher microprocessor

 - MS-DOS operating system 3.1 or later

 - Microsoft Windows operating system version 3.1 or later

 - CD-ROM drive

 - 4 MB of RAM (8 MB recommended)

 - 1-3 MB of available hard disk space

 - VGA+ graphics (640 X 480 X 256 colors)

 - MPC-1 compatible sound card

 - Microsoft Mouse or compatible pointing device

 - Headphones or Speakers
