SUMMARY
=======

Ole_samp.exe is a sample extracting file that provides an introduction to the
concepts of OLE Automation using Visual FoxPro as the OLE client and Microsoft
Word for Windows version 7.0, Microsoft PowerPoint for Windows 95 version 7.0,
and Microsoft Excel for Windows version 7 as the OLE servers.

MORE INFORMATION
================

The following files are available for download from the Microsoft Download
Center:

   OLE_samp.exe
   (http://download.microsoft.com/download/vfox50/info/1/WIN98/EN-US/OLE_samp.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

   Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

This document introduces the concepts of OLE Automation using Visual FoxPro as
the OLE client. Its purpose is to provide you with a basic understanding of OLE
Automation, the flexibility and power of using OLE Automation, and three
practical applications of these concepts. The accompanying Word document
contains an overview of how to use the Visual FoxPro form and step- by-step
procedures that guide you through the process of using and viewing this example
of OLE Automation.
