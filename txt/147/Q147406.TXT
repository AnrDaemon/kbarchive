SUMMARY
=======

The following article contains a copy of the minimum system requirements listed
on the Microsoft Multimedia Stravinsky for Windows program package.

NOTE: Some of these requirements have changed since package printing. For a list
of requirement changes, please see the "System Requirement Changes" section at
the end of this article.

MORE INFORMATION
================

Printed System Requirements
---------------------------

The minimum system requirements for running Stravinsky are:

 - A Multimedia PC or compatible with a 386SX or higher microprocessor.

 - MS-DOS operating system 3.1 or later.

 - Microsoft Windows operating system version 3.1 or later.

 - Double-speed (2x) or better CD-ROM drive.

 - 2 MB of RAM.

 - 30-MB hard disk.

 - VGA display.

 - Windows-compatible sound card.

 - Speakers or headphones.

 - Microsoft Mouse or compatible pointing device.

System Requirement Changes
--------------------------

Multimedia Stravinsky only requires 3 MB of hard disk space.
