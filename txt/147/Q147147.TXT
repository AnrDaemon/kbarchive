TITLE   :XCLN: Cannot Select Multiple Files to Attach to a New Message
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:4.0,5.0
OPER/SYS:
KEYWORDS:kbusage

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Windows 95/98 client, versions 4.0, 5.0 
 - Microsoft Exchange Windows 3.x client, versions 4.0, 5.0 
 - Microsoft Exchange Windows NT client, versions 4.0, 5.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attach files to a new message, only one file can be selected at a time.
You cannot select multiple files to be attached at one time.

CAUSE
=====

This behavior is by design.

WORKAROUND
==========

You can select multiple attachments in the Windows 95 Explorer or Windows File
Manager and use drag-and-drop editing to attach the multiple files to an open
Compose Note window.


Additional query words: attachments
