The information in this article applies to:

 - Scholastic's Magic School Bus series: Explores the Human Body for the Macintosh, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

If you want to run one of the programs listed above from the hard drive rather
than from the compact disc to increase performance or to make use of the product
more convenient, you can follow the steps listed below.

MORE INFORMATION
================

IMPORTANT: The information below is given for information purposes ONLY. The
products are designed to run from the CD-ROM drive; therefore, the
configurations discussed below are NOT recommended or supported by Microsoft.

Magic School Bus Explores the Human Body
----------------------------------------

Version 1.0
Approximate disk space needed: 320.2 MB

1. Copy the following items from compact disc onto the Macintosh Hard Drive
   icon:

   Setup
   ReadMe
   Mband.ini
   Icon
   Bod-R

2. Rename Macintosh Hard Drive to "Magic School Bus" (without the quotation
   marks).

   NOTE: To rename your hard drive, Sharing must be turned off.

3. Run Magic School Bus Setup from the hard drive.

4. Drag the following files into the trash can:

   Icon
   Mband.ini
   Setup

5. Run Magic School Bus Explores the Human Body.

You should now be able to run Magic School Bus Explores the Human Body from the
hard drive without inserting the compact disc.

IMPORTANT: If you need to run Setup from the compact disc at a later time, make
sure that your hard drive is named something other than "Magic School Bus".
Otherwise, Setup may fail.
