TITLE   :Message Size on BBS Differs in Message List and Property Sheet
PRODUCT :The Microsoft Network
PROD/VER:WINDOWS:1.2,1.3
OPER/SYS:
KEYWORDS:kbmsn

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - The Microsoft Network versions 1.2, 1.3 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you open the Properties dialog box of a bulletin board message, the size
shown may be different from the size shown in the message list.

CAUSE
=====

The size shown in the message's Properties dialog box is the decompressed size
of the message's body only. The size shown in the message list is the size of
the header plus the compressed body.

MORE INFORMATION
================

To display the Properties dialog box of a message, open the message and then
click Properties on the File menu.
