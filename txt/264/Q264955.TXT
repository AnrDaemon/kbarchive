TITLE   :XADM: Installing Exchange Administrator Program on Windows 2000
PRODUCT :Microsoft Exchange
PROD/VER:winnt:5.5
OPER/SYS:
KEYWORDS:exc55

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SUMMARY
=======

The Microsoft Exchange Server Administrator program can be installed on a
computer that is running either Microsoft Windows NT 4.0 Workstation or
Microsoft Windows 2000 Professional.

MORE INFORMATION
================

To Install Exchange Server Administrator
----------------------------------------

1. Log on to the workstation with an account that has permission to install
   applications and modify the registry (such as the local administrator
   account).

2. Start Setup for Exchange 5.5, choose "Setup Server and Components", and then
   click Agree to the license agreement.

3. Select either the complete or the custom installation.

4. Click to clear everything except Microsoft Exchange Administrator, and then
   continue with Setup.

After you install the Exchange Server Administrator program, you should apply the
most recent Exchange Server 5.5 service pack to the workstation.

For additional information about how to obtain the latest Exchange Server 5.5
service pack, click the article number below to view the article in the
Microsoft Knowledge Base:

   Q191014 How to Obtain the Latest Exchange Server 5.5 Service Pack
