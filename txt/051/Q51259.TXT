TITLE   :Mac Wkst: Invisible Files Sent as File Enclosures
PRODUCT :Microsoft Mail For Appletalk Networks
PROD/VER:WINDOWS:2.0,2.0a,2.0b,3.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Invisible files can be sent as enclosures in Microsoft Mail version 2.00. The
invisible (for example, Desktop) files appear in the Open dialog box once the
Enclosure button has been clicked.

Microsoft Mail version 3.0 flags these files to remain invisible when the
Enclosure button is clicked.

Additional query words: 2.00 2.00a 2.00b 3.00
