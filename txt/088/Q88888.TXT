TITLE   :SET SCOREBOARD and SET STATUS Commands in FoxBASE+
PRODUCT :Microsoft Fox Miscellaneous Products
PROD/VER:MS-DOS:1.21,2.1
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft FoxBASE+ for MS-DOS, versions 1.21, 2.1 
-------------------------------------------------------------------------------

SUMMARY
=======

The SET SCOREBOARD command is used in conjunction with the SET STATUS command to
determine where information messages are displayed on the screen.

MORE INFORMATION
================

If SCOREBOARD is ON and STATUS is OFF, information will display on line 0. If
STATUS is ON, information will display on line 22 regardless of the SCOREBOARD
setting. In FoxBASE+, SET SCOREBOARD, SET STATUS OFF, and SET STATUS ON commands
clear the screen.
