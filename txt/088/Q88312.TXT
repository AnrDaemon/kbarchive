TITLE   :Windows Hangs on Dell 310 with Genius 420 Monitor
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run Microsoft Windows 3.1 on a Dell 310 computer with a Genius 420
monitor and Genius 420 mouse driver, the system stops responding (hangs) when
you start Windows 3.1 in standard or 386 enhanced mode.

CAUSE
=====

Ran-Ger Technologies Incorporated confirmed this as a compatibility problem
between the Genius 420 monitor and Dell 310 computer. Ran-Ger Technologies is
working with Dell to resolve this problem.

MORE INFORMATION
================

The Ran-Ger 420 monitor and the Dell 310 computer included here are manufactured
by vendors independent of Microsoft; we make no warranty, implied or otherwise,
regarding these products' performance or reliability.
