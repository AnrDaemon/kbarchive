TITLE   :Insufficient memory error executing System icon in Control Panel
PRODUCT :Microsoft Windows NT
PROD/VER:3.51
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.51 
 - Microsoft Windows NT Server version 3.51 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you click Control Panel System, the following error message appears:

   Insufficient Memory for this operation;
   Close one or more Windows applications to increase available memory.

If you click OK, the Control Panel System screen appears.

CAUSE
=====

This problem occurs if Windows NT configures 7 or more logical drive
partitions.


STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 3.51. We are
researching this problem and will post new information here in the Microsoft
Knowledge Base as it becomes available.
