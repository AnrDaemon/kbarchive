DOCUMENT:Q142257  19-FEB-2002  [exchange]
TITLE   :XCON: MTA Database XFER-IN Thread Error
PRODUCT :Microsoft Exchange
PROD/VER::4.0
OPER/SYS:
KEYWORDS:kbother

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

You might see Windows NT Event Log non-fatal errors from the Microsoft Exchange
Message Transfer Agent (MTA) regarding an MTA Database error in the XFER-IN
thread. You might also find that a message in an MTA queue is marked as Send in
Progress, but it never gets sent.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server
version 4.0. This problem was corrected in Microsoft Exchange Server 4.0 U.S.
Service Pack 2. For information on obtaining the service pack, query on the
following word in the Microsoft Knowledge Base (without the spaces):

   S E R V P A C K
