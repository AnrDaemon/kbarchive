TITLE   :WINS Proxy Button is Unavailable
PRODUCT :Microsoft Windows NT
PROD/VER:3.50 3.51
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server versions 3.5, 3.51 
-------------------------------------------------------------------------------

SYMPTOMS
========

It is not possible to make a Windows NT Server act as a WINS Proxy agent. The
WINS Proxy Agent button is grayed out.

WORKAROUND
==========

Remove the WINS Server service from the Windows NT Server.
