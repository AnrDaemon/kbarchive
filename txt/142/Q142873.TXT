TITLE   :STOP 0x0000000A Using Banyan Vines Driver VNS.SYS
PRODUCT :Microsoft Windows NT
PROD/VER:3.50 3.51
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation versions 3.5, 3.51 
 - Microsoft Windows NT Server versions 3.5, 3.51 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you install the Banyan Vines protocol driver (VNS.SYS) and restart Windows
NT, the following STOP error message appears:

   STOP: 0x0000000A - NTOSKRNL.EXE


RESOLUTION
==========

For more information, contact Banyan Technical Support at (800) 222-6926.
