TITLE   :Remember Password Option Is Unavailable
PRODUCT :The Microsoft Network
PROD/VER::2.6,5.0,5.1,5.2,95,98,98 Second Edition
OPER/SYS:
KEYWORDS:kbmsn

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - The Microsoft Network versions 2.6, 5.0, 5.1, 5.2 
 - the operating system: Microsoft Windows 95 
 - the operating system: Microsoft Windows 98 
 - the operating system: Microsoft Windows 98 Second Edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to connect to MSN, The Microsoft Network, the "Remember
Password" option on the Sign In screen is unavailable.

CAUSE
=====

The "Remember Password" option is available only if you have Client for
Microsoft Networks installed or if you have a password set up for Microsoft
Windows 95 or Windows 98.

RESOLUTION
==========

To install Client for Microsoft networks 
----------------------------------------

1. Click Start, go to Settings, and then click Control Panel.

2. In Control Panel, double-click Network.

3. Click Add, click Client, and then click Add.

4. Under Manufacturer click Microsoft and under Network Clients click Client for
   Microsoft networks.

5. Click Ok and restart your computer.

6. When Windows prompts for a username and password, in the Username box type
   your MSN username, leave the Password box empty, and then click Ok.

7. When Windows prompts to confirm the entry, leave the Password box empty again
   and click Ok.

To create a blank password for
Windows:
---------------------------------------

1. Click Start, go to Settings, and then click Control Panel.

2. In Control Panel, double-click Passwords.

3. On the Change Passwords tab, click Change Windows Password, and then click
   OK.

4. Leave all the boxes blank, and then click OK.

5. Close Control Panel, click OK until you return to Windows, and then restart
   your computer.
