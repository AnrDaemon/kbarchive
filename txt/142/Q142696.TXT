TITLE   :WinNT 3.51 SP3 GARAM4.TTF Font Not Displayed Properly
PRODUCT :Microsoft Windows NT
PROD/VER::3.51 SP3,4.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation versions 3.51 SP3, 4.0 
 - Microsoft Windows NT Server versions 3.51 SP3, 4.0 
-------------------------------------------------------------------------------


SYMPTOMS
========

After you install Windows NT 3.51 Service Pack 3, the GARAM4.TTF font is not
displayed properly.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 3.51. This
problem was corrected in the latest Windows NT 3.51 U.S. Service Pack. For
information on obtaining the Service Pack, query on the following word in the
Microsoft Knowledge Base (without the spaces):

   S E R V P A C K


Additional query words: prodnt 3.51
