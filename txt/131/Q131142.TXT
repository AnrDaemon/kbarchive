TITLE   :Display Driver Setup Fails for Intergraph G91 Video Adapter
PRODUCT :Microsoft Windows NT
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.51 
 - Microsoft Windows NT Server version 3.51 
-------------------------------------------------------------------------------

SYMPTOMS
========

The installation of the Weitek Compatible Display Driver included in Windows NT
version 3.51 fails for an Intergraph G91 video adapter on Intergraph TD-5
computers.

WORKAROUND
==========

To work around this problem:

 - Use the video adapter driver shipped with the Intergraph computer or obtain
   the driver from Intergraph at (214) 669-9680.

   -OR-

 - Use the Weitek Compatible Display Driver included on the Windows NT
   Workstation 3.5 or Windows NT Server 3.5 compact disc. Run Control Panel,
   choose Display, and specify the I386 directory on the compact disc.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 3.51. We are
researching this problem and will post new information here in the Microsoft
Knowledge Base as it becomes available.
