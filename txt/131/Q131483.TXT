TITLE   :Error Message: Quick Recorder - A Memory Error Has Occurred
PRODUCT :Miscellaneous Windows Products
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows Sound System, version 2.0 
 - Microsoft Excel for Windows, version 5.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

Embedding a sound object in Microsoft Excel version 5.0 for Windows activates
Quick Recorder 2.0, causing the following error message to be displayed:

   QUICK RECORDER
   A memory error has occurred.

STATUS
======

Microsoft is researching this problem and will post new information here in the
Microsoft Knowledge Base as it becomes available.

MORE INFORMATION
================

Clicking OK allows Quick Recorder to function normally. Clicking the sound
object in Microsoft Excel after the first error message does not produce any
further error messages, and the sound object plays normally.

Media Player can also be used to play embedded sound objects.
