TITLE   :Windows Sound System 2.0 Modified for WaveJammer Sound Card
PRODUCT :Miscellaneous Windows Products
PROD/VER:2.00
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows Sound System, version 2.0 
-------------------------------------------------------------------------------

SUMMARY
=======

NewMedia's WaveJammer sound card may be bundled with Windows Sound System
software. This software has been altered to work with the WaveJammer sound card,
but the disk labels, file sizes, and dates are the same as other versions of
Windows Sound System.

MORE INFORMATION
================

This version of the software can be identified only when you are installing the
software. In the modified version of the software, there is a NewMedia
WaveJammer option in the hardware selection screen. The usual list of supported
hardware is not available in this screen.

You can install this software only on a computer with a WaveJammer sound card
installed. For support of this type of installation, please contact NewMedia.
