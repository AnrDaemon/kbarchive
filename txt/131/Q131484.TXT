TITLE   :XCLN: Delete Button in Address Book Displayed As Active
PRODUCT :Microsoft Exchange
PROD/VER:4.0 5.0
OPER/SYS:
KEYWORDS:kbusage

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Windows 3.x client, versions 4.0, 5.0 
 - Microsoft Exchange Windows NT client, versions 4.0, 5.0 
 - Microsoft Exchange Windows 95/98 client, versions 4.0, 5.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

On the Address Book toolbar, the Delete button is active when you select the
Type Name Or Select From List box; however, the button does not perform any
function unless you type text in the box and select that text.

CAUSE
=====

This is by product design.
