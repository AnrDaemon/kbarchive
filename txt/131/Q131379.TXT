TITLE   :XCLN: Cannot Drag Directories into Messages
PRODUCT :Microsoft Exchange
PROD/VER:4.0 5.0
OPER/SYS:
KEYWORDS:kbusage

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Windows 3.x client, versions 4.0, 5.0 
 - Microsoft Exchange Windows NT client, versions 4.0, 5.0 
 - Microsoft Exchange Windows 95/98 client, versions 4.0, 5.0 
-------------------------------------------------------------------------------


SYMPTOMS
========

The drag-and-drop feature does not support the insertion of directories.

CAUSE
=====

File Manager and Windows Explorer allow you to use the drag-and-drop feature;
however, you cannot insert directories or folders in this manner.


STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft Exchange version
4.0 Windows 3.x, Windows 95, and Windows NT clients. We are researching this
problem and will post new information here in the Microsoft Knowledge Base as it
becomes available.
