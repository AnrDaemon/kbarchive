TITLE   :Mac Encarta 95: Encarta Fails to Start and Stopwatch Appears
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER::1995 edition
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia for Macintosh 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you try to start Encarta, a stopwatch appears and Encarta fails to start.

CAUSE
=====

You may experience problems starting Encarta if the QuickDraw GX extension is
installed.

RESOLUTION
==========

Turn off or remove QuickDraw to resolve your problem.

Turning off the QuickDraw GX Extension
--------------------------------------

1. From the Apple menu, choose Control Panels.

2. Click Extensions Manager.

3. From the list of extensions, locate QuickDraw GX. Make sure the check box to
   the left if QuickDraw GX is cleared.

4. Close Extensions Manager.

5. From the Special menu, choose Restart.

Removing QuickDraw GX
---------------------

1. Open the System Folder.

2. Open the Extensions folder.

3. Locate QuickDraw GX. Drag the extension to the desktop or the Trash.

4. From the Special menu, choose Restart.

MORE INFORMATION
================

The Encarta ReadMe file, Section 9: Known Problems of Encarta 1995, discusses
problems with the QuickDraw extension. Please refer to the ReadMe file for more
information.
