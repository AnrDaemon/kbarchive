TITLE   :Recurring Task Shown Once in Weekly Schedule View
PRODUCT :Microsoft Schedule+ for Windows
PROD/VER:WINDOWS:7.0; Win95:7.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Schedule+ for Windows, version 7.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

Only one instance of a recurring task appears in the Weekly Schedule view, even
though the filter (on the View menu) is set to Active. Multiple instances of the
same reoccurring task do show correctly in the To Do view.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Schedule+ version 7.0.
We are researching this problem and will post new information here in the
Microsoft Knowledge Base as it becomes available.

Additional query words: schedule plus 7.00 task list view
