TITLE   :Adaptec AHA-2940 SCSI and Viper PCI Video Hang Windows NT Setup
PRODUCT :Microsoft Windows NT
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 3.1 
 - Microsoft Windows NT Workstation version 3.1 
 - Microsoft Windows NT Advanced Server, version 3.1 
 - Microsoft Windows NT Workstation versions 3.5, 3.51 
 - Microsoft Windows NT Server versions 3.5, 3.51 
-------------------------------------------------------------------------------

SYMPTOMS
========

Your computer with an Adaptec AHA-2940 SCSI PCI controller and a Viper PCI video
card stops responding (hangs) during the text mode of Windows NT Setup.

WORKAROUND
==========

There is no workaround at this time. You must replace one of the cards.


STATUS
======

Adaptec is aware on this problem and is working on it.

Additional query words: prodnt 3.10
