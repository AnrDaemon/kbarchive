-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Visual SourceSafe for Windows, versions 4.0, 5.0 
 - Microsoft SourceSafe for Windows, versions 3.04, 3.1 
 - Microsoft SourceSafe for Windows NT, versions 3.04, 3.1 
 - Microsoft SourceSafe for MS-DOS, versions 3.04, 3.1 
 - Microsoft SourceSafe for Macintosh, versions 3.04, 3.1 
-------------------------------------------------------------------------------

SUMMARY
=======

The following is a list of the file names and descriptions for those files
located in the \DATA directory for SourceSafe version 3.x:

AAAAAAAA.cnt   Last Filename used by SourceSafe
MACNAME.dat    Long filename information
RIGHTS.dat     User rights information
STATUS.dat     Date/Checkout information for files
UM.dat        User information: User Names, Passwords, Serial Numbers
VERSION.dat    Database version information

The following lists the same information for Visual SourceSafe version 4.0.

AAAAAAAA.cnt   Last Filename used by SourceSafe
NAMES.dat      Long (greater than 8.3 format) filename information
RIGHTS.dat     User rights information
STATUS.dat    Cached Date/Checkout information for files
UM.dat         User information: User Names, Passwords, Serial Numbers
VERSION.dat   Database version information

XXXXXXXX       Logfile for files and projects
XXXXXXXX.A     File and project contents
XXXXXXXX.B

Additional query words: 4.00 5.00
