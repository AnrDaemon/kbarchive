DOCUMENT:Q131482  08-AUG-2001  [winnt]
TITLE   :Netstat Does Not Display Listening TCP Ports
PRODUCT :Microsoft Windows NT
PROD/VER::3.5,3.51,4.0,4.0 SP1,4.0 SP2
OPER/SYS:
KEYWORDS:kbnetwork

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation versions 3.5, 3.51, 4.0, 4.0 SP1, 4.0 SP2 
 - Microsoft Windows NT Server versions 3.5, 3.51, 4.0, 4.0 SP1, 4.0 SP2 
-------------------------------------------------------------------------------

SYMPTOMS
========

Netstat does not display TCP ports that are in the listening state.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT Workstation and
Server versions 3.5, 3.51, and 4.0 thru Service Pack 2. The problem was
corrected by Service Pack 3 for NT 4.0.

Additional query words: prodnt tcpip tcp/ip
