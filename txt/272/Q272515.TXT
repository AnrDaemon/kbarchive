TITLE   :How to change themes on the MSN Companion user interface
PRODUCT :The Microsoft Network
PROD/VER::1.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - MSN Companion, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

MSN Companion themes let the customer change the colors and graphics of the user
interface, making it feel more personal and comfortable.

MORE INFORMATION
================

To change MSN Companion Themes

1. On the Web Companion Start page, click Settings.

2. Click themes.

3. From the list of available themes, select your choice, and then click use
   this theme to save settings.

4. When the selected theme is loaded, you may select either yes, keep theme,
   which returns you to the Start page, or no, choose another, which returns you
   to step three.

NOTE: After selecting a theme, it will download and install the new theme. This
could take several minutes. Remember, you are using your phone line!
