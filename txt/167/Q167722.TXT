TITLE   :WD97: How to Format a Hyperlinked Object in an HTML Document

PROD/VER:WINDOWS:97

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SUMMARY
=======

When you attempt to format a hyperlinked picture or object inserted into a
document, there is no viewable option to choose to format the picture or object.
This article describes the steps you can take to format a hyperlinked picture or
object.

MORE INFORMATION
================

To format a hyperlinked picture or object, use one of the following methods:

Method 1: Use Select Hyperlink
------------------------------

1. Position the insertion point over the object.

2. Right-click the object to display the shortcut menu.

3. On the shortcut menu, click Select Hyperlink.

4. On the Format menu, click Picture (or click Object).

Method 2: Use the CTRL Key
--------------------------

1. Hold the CTRL key and then click to select the picture or object.

2. Do one of the following:
    - On the Format menu, click Picture (or Object).

      -or-

    - Right-click the selected picture or object. From the shortcut menu, click
      Format Picture or Format Object.
