TITLE   :WD97: ODMA: File Open Dialog May Display Behind Application
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97; winnt:3.51,4.0
OPER/SYS:
KEYWORDS:kb3rdparty kbusage word97

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
 - the operating system: Microsoft Windows NT, versions 3.51, 4.0 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you close your document management system dialog box to return to the Word
Open dialog box, the Open dialog box may appear behind the active applications.

WORKAROUND
==========

Microsoft Windows NT 4.0
------------------------

Click the Word application button on the Windows NT 4.0 task bar to bring the
Open dialog box to the foreground.

Microsoft Windows NT 3.51
-------------------------

Press CTRL+ESC to bring up the Windows task list, and then close the task list.
This brings the Open dialog box to the foreground.

STATUS
======

This problem occurs when you use Microsoft Word 97 for Windows and PC DOCS on
the Microsoft Windows NT platform.
