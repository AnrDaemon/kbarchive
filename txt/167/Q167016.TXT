TITLE   :Lexmark Optra C Series Printer Incorrectly Presents Duplex Optio
PRODUCT :Microsoft Windows NT
PROD/VER:4.0
OPER/SYS:
KEYWORDS:kbprintkbbuglist kbfixlist

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 4.0 
 - Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you set up the paper options for the Lexmark Optra C Series printers under
Windows NT version 4.0, you are presented with a Duplex option. The Optra C
Series printer does not have duplex capability.

STATUS
======

Support for duplex has been removed for the C series Optra printers that use
Optra.dll driver is included in the latest Windows NT version 4.0 U.S. Service
Pack. For information on obtaining the Service Pack, query on the following word
in the Microsoft Knowledge Base (without the spaces):

   S E R V P A C K

Additional query words: 4.00 prodnt nt4sp3fea
