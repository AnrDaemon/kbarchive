TITLE   :Lexmark Optra Ep Shows Duplex Option and Memory Available
PRODUCT :Microsoft Windows NT
PROD/VER::4.0
OPER/SYS:
KEYWORDS:kbprintkbbuglist kbfixlist

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 4.0 
 - Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you set up the paper options for the Lexmark Optra Ep Series printers under
Windows NT version 4.0, you are presented with a Duplex option. The Optra Ep
Series printer does not have duplex capability.

also:

In Printer Properties for the Ep Series printer, it shows 5 MB of available RAM
when the printer has the capacity for 6 MB.

STATUS
======

Support for duplex has been removed and it now correctly shows 6 MB maximum
available RAM for the Ep series Optra printers that use the Optra.dll driver is
included in the latest Windows NT version 4.0 U.S. Service Pack. For information
on obtaining the Service Pack, query on the following word in the Microsoft
Knowledge Base (without the spaces):

   S E R V P A C K

Additional query words: 4.00 prodnt nt4sp3fea
