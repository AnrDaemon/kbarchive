TITLE   :STREETS: Err Msg: The Main Application Cannot Be Launched
PRODUCT :Microsoft Automap
PROD/VER::
OPER/SYS:
KEYWORDS:kberrmsg kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Expedia Streets and Trips 2000 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start Microsoft Expedia Streets and Trips 2000, you may receive one of
the following error messages:

   The main application cannot be launched. Rerun Setup.

   -or-

   Streets & Trips 2000 cannot launch main application. Rerun Setup.

CAUSE
=====

This behavior can occur if your computer is infected with the SirCam virus.

RESOLUTION
==========

To resolve this issue, contact an antivirus software manufacturer to obtain an
up-to-date version of an antivirus program that can detect and disinfect the
SirCam virus from your computer.

Microsoft does not provide software that can detect or remove computer viruses.
If you suspect or confirm that your computer is infected with a virus, obtain
current antivirus software. For a list of antivirus software manufacturers, view
the following article in the Microsoft Knowledge Base:

   Q49500 List of Antivirus Software Vendors

For more information about the SirCam virus, browse to the following McAfee and
Symantec Web sites:

   http://vil.mcafee.com/dispVirus.asp?virus_k=99141&

   http://www.sarc.com/avcenter/venc/data/w32.sircam.worm@mm.html

   http://www.symantec.com/avcenter/venc/data/w32.sircam.worm@mm.removal.tool.html

MORE INFORMATION
================

The Sircam32 virus is normally spread through e-mail attachments. The title of
the e-mail message that contains this virus may vary, but the text of the
message may be similar to any of the following:

 - Hi! How are you?

   I send you this file in order to have your advice.

   -or-

 - I hope you can help me with this file that I send.

   -or-

 - I hope you like the file that I send you.

   -or-

 - This is the file with the information that you ask for.

   See you later.

   Thanks.

The same message may also be received in Spanish.

For additional information about the SirCam virus, click the article number below
to view the article in the Microsoft Knowledge Base:

   Q306913 Error Message Caused by Sircam32 Virus When You Start a Program
