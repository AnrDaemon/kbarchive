TITLE   :Sanyo 3-CD-ROM Changer Not Recognized by Windows NT 4.0
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:kb3rdparty kbhw kbHardware

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0 
 - Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start a computer with a Sanyo 3-CD-ROM changer, the device may not be
recognized by Windows NT 4.0.

CAUSE
=====

This behavior can occur if the device has an older firmware revision that does
not work with Windows NT 4.0 reliably.

RESOLUTION
==========

Upgrade the firmware on the Sanyo 3-CD-ROM changer to version 2.13 or later.


MORE INFORMATION
================

This CD-ROM Changer is on the Windows NT 4.0 Hardware Compatibility List under
the name "Torisan (Sanyo) CDR-C3G Changer."

Additional query words: quad speed 4x
