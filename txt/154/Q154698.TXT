DOCUMENT:Q154698  08-AUG-2001  [winnt]
TITLE   :WinNT & Microsoft Exchange 4.0 Takes Long Time to Shut Down
PRODUCT :Microsoft Windows NT
PROD/VER:4.0
OPER/SYS:
KEYWORDS:kbinterop

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

A computer running Microsoft Windows NT Server 4.0 and Microsoft Exchange Server
4.0 no longer shuts down in a reasonable period of time.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Windows NT Server
version 4.0 and Microsoft Exchange Server version 4.0. This problem has been
corrected in the latest U.S. Service Pack for Microsoft Exchange 4.0. For
information on obtaining the Service Pack, query on the following word in the
Microsoft Knowledge Base (without the spaces):

   S E R V P A C K

Additional query words: prodexchange prodnt
