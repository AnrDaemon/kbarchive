TITLE   :Problem Connecting to Xylogics Annex-3 Terminal Server
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:3.5,3.51
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation versions 3.5, 3.51 
 - Microsoft Windows NT Server versions 3.5, 3.51 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you are using Microsoft Windows NT Workstation or Server, you may not be
able to connect using Remote Access Service (RAS) to a Xylogics Annex-3 terminal
server.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 3.51. This
problem was corrected in the latest Windows NT 3.51 U.S. Service Pack. For
information on obtaining the Service Pack, query on the following word in the
Microsoft Knowledge Base (without the spaces):

   S E R V P A C K


MORE INFORMATION
================

The Annex-3 terminal server is manufactured by Xylogics, Inc., a vendor
independent of Microsoft; we make no warranty, implied or otherwise, regarding
this product's performance or reliability.
