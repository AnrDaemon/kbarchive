DOCUMENT:Q154951  13-FEB-2002  [winnt]
TITLE   :Blue Screen w/ S3 Video Driver Upgrade from 3.51 to SP3 or SP4
PRODUCT :Microsoft Windows NT
PROD/VER::3.5,3.51
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation versions 3.5, 3.51 
 - Microsoft Windows NT Server versions 3.5, 3.51 
-------------------------------------------------------------------------------


SYMPTOMS
========

After you upgrade to Service Pack 3 or 4 from Windows NT 3.51 (with or without
SP1 or SP2), if your system has an S3 video card you will see the following
message upon restart:

   The required system file S3 is bad or missing.

RESOLUTION
==========

Obtain the fix referenced below, which contains an updated S3 driver.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 3.51.

MORE INFORMATION
================

A fix to this problem is in development, but has not been regression-tested and
may be destabilizing in production environments. Microsoft does not recommend
implementing this fix at this time. Contact Microsoft Product Support Services
for more information on the availability of this fix.
