SUMMARY
=======

Codewiz.exe is a file that allows you to convert Xbase code written in Clipper,
FoxPro, FoxBASE, or dBASE into a Visual FoxPro form. It converts the following
Xbase commands into their equivalent objects in Visual FoxPro:

      @ ... SAY
      @ ... GET
      USE
