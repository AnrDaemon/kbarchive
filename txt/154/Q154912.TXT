TITLE   :Novell's NPRINT Command Does Not Work in Windows NT 4.0
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:kbnetwork kbprint kbPrinting

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0 
 - Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

Novell's NPRINT command does not work in Windows NT 4.0.

CAUSE
=====

This is by design. Novell's NPRINT command is not supported by the Windows NT
4.0 NetWare redirector.
