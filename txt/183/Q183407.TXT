TITLE   :XADM: Err Msg: There Are No Mailboxes in Container Specified
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:4.0
OPER/SYS:
KEYWORDS:kbusage

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run the Microsoft Exchange Simport.exe program, the following message
appears:

   There are no mailboxes in the container specified.

WORKAROUND
==========

To work around this problem, verify that the site name entered is correct.

MORE INFORMATION
================

Simport.exe is a program that automatically generates temporary security tokens
in bulk. These security tokens are then used to enable users with Advanced
Security.
