TITLE   :SNA Windows NT Client Contains Corrupt Help File
PRODUCT :Microsoft SNA Server
PROD/VER:WINDOWS:3.0,3.0 SP1,3.0 SP2
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft SNA Server, versions 3.0, 3.0 SP1, 3.0 SP2 
-------------------------------------------------------------------------------

SYMPTOMS
========

Some of the help files installed with the SNA Windows NT Client version 3.0 are
corrupt.

WORKAROUND
==========

To work around this problem, select the page for "VTAM Sample Definitions for
3270" and then use the back arrows to get to the page for Sample Host
Definitions. The forward and back arrows can now be used to move through the
appendices.

STATUS
======

Microsoft has confirmed this to be a problem in SNA Server versions 3.0, 3.0
SP1, and 3.0 SP2. We are researching this problem and will post new information
here in the Microsoft Knowledge Base as it becomes available.
