TITLE   :PBA: Incorrect Country/Region Names Listed in PBAdmin
PRODUCT :Internet Information Server
PROD/VER::1.0,4.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Internet Information Server version 4.0 
 - Microsoft Phone Book Administrator, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to add a country or region to a phone book entry, you may
receive a message stating that one of the countries/regions listed below is
incorrectly named. The correct names are shown in the right column:

   Country/Region                Correct Name
   --------------                ------------

   Taiwan republic of China      Taiwan
   Hong Kong                     Hong Kong SAR, PRC
   Nevis and St. Kitts           St. Kitts and Nevis
   Cape Verde Islands            Cape Verde
   Zaire                         Democratic Republic of the Congo(Note
                                 that this is different from Congo)
   Seychelle Islands             Seychelles
   Slovak Republic               Slovakia
   Fuji Island                   Fuji
   Western Samoa                 Samoa
   Kiribati Republic             Kiribati
   Kyrgyz Republic               Kyrgyzstan


STATUS
======

Microsoft has confirmed this to be a problem in Phone Book Administrator version
1.0.


A supported fix is now available, but has not been fully regression- tested and
should be applied only to systems experiencing this specific problem. Unless you
are severely impacted by this specific problem, Microsoft recommends that you
wait for the next Service Pack that contains this fix. Contact Microsoft
Technical Support for more information.
