TITLE   :WD97: Shortcut Key Doesn't Bring Word to the Front with Focus
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:kbinterop word97

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you create a shortcut key to start Word 97, the shortcut key does not bring
Word to the forefront when Word is already running and another program has the
focus.

For example, Word is initially started using the Shortcut key. Another program is
then started and has the focus. When you click the shortcut key again, Word does
not return to the forefront.

WORKAROUND
==========

Click on the Word application from the Windows taskbar.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article.
