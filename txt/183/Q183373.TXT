TITLE   :IEAK Included with ResKit Incompatible with Non-English Systems
PRODUCT :Microsoft Press
PROD/VER::
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - MSPRESS Microsoft Internet Explorer Resource Kit ISBN 1-57231-842-2 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you are using a non-English version of Windows 95 or Windows NT, when you
attempt to load the Internet Explorer Administration Kit, you will receive an
error message.

CAUSE
=====

The Internet Explorer Administration Kit included on the CD-ROM is localized for
English systems only. Windows systems that use a different language must
download a new copy of the Internet Explorer Administration Kit.

RESOLUTION
==========

Download the copy of the Internet Explorer Administration Kit appropriate for
your language from the following Web site:

   http://ieak.microsoft.com/

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft Internet Explorer
Resource Kit.
