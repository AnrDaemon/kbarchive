DOCUMENT:Q222650  07-SEP-2001  [winnt]
TITLE   :Incorrect Fax Printer Properties in Windows 95 or Windows 98
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.5
OPER/SYS:
KEYWORDS:kbprint

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft BackOffice Small Business Server version 4.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you view or attempt to modify the properties (such as paper size,
orientation, billing code, or e-mail address) for the fax printer on a client
computer running Microsoft Windows 95 or Microsoft Windows 98, incorrect
properties are displayed.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.

MORE INFORMATION
================

When you view the fax printer properties on a client computer running Microsoft
Windows NT, the correct properties are displayed. When you view the properties
for the same printer on a client computer running Windows 95 or Windows 98, the
standard printer properties are displayed.
