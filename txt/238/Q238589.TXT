SUMMARY
=======

A white paper is now available on the Microsoft FTP Web site that is designed to
assist you with domain sizing and capacity planning issues using Microsoft
Windows NT Server 4.0.

MORE INFORMATION
================

The "Domain Sizing and Capacity Planning with Windows NT Server 4.0" white paper
describes some of the key characteristics and capabilities of Windows NT Server
4.0 in supporting very large single or multiple master domains in the corporate
environment.

The following topics are included in the document:

 - Identification of key hardware components and configurations that affect the
   design and characteristics of a domain.

 - How to use Performance Monitor and Network Monitor for planning and
   production.

 - A description of user's service levels, event log monitoring, domain size
   (SAM) limitations, and general recommendations.

The white paper is also available for download at the following Microsoft Web
site:

   ftp://ftp.microsoft.com/bussys/winnt/winnt-docs/papers/DomainSizingWPv2.doc
