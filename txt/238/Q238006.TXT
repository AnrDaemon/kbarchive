DOCUMENT:Q238006  11-DEC-2001  [winnt]
TITLE   :Problem Viewing Main Client Printer Check Box in User Manager
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:kbtool kbui

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0, Terminal Server Edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you select multiple users in User Manager, click Properties, and then click
Config, the "Default to main client printer" check box should have 3 possible
states:

 - Checked if all users have this setting enabled.

 - Unchecked if all users do not have this setting enabled.

 - Unavailable if some users have this setting but others do not.

However, your computer may only display the following 2 possible states:

 - Checked if all users have this setting enabled.

 - Unchecked if all users do not have this setting enabled.

RESOLUTION
==========

To resolve this problem, apply Windows NT Server 4.0, Terminal Server Edition
Service Pack 4.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.
