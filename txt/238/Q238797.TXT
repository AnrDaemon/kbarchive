======================================================================
SUMMARY
=======

A document that describes how to install and configure a Microsoft DNS server is
available for download from the Microsoft Windows NT FTP Web site.

MORE INFORMATION
================

This document contains graphic images that show the user interface and describe
the options available during Setup. Other topics described in this document
include:

 - Installing a DNS server

 - Configuring a DNS server

 - Setting up zones for name resolution

The document is available as Dnsinstall.exe at the following Microsoft Web site:

   ftp://ftp.microsoft.com/bussys/winnt/winnt-docs/papers/
