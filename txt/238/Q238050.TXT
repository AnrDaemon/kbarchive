-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta World English Dictionary 
-------------------------------------------------------------------------------

SYMPTOMS
========

You may be unable to scroll through the Pinpointer in Microsoft Encarta World
English Dictionary 2000 using the Microsoft IntelliMouse wheel.

CAUSE
=====

This behavior can occur if you use the version of the Microsoft IntelliPoint
software that is included in Microsoft Windows 98 or Microsoft Windows 2000.

RESOLUTION
==========

To resolve this issue, download and install the latest version of the Microsoft
IntelliPoint software from the following Microsoft Web site:

   http://www.microsoft.com/products/hardware/mouse/intellimouse/default.htm

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.
