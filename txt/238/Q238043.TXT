TITLE   :Encarta 2000: Setup Unexpectedly Removes Virtual Globe 99
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:WINDOWS:
OPER/SYS:
KEYWORDS:kbsetup kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta Reference Suite 2000 
 - Microsoft Encarta Interactive World Atlas 2000 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you install Microsoft Encarta Reference Suite 2000 or Microsoft Encarta
World Atlas 2000, Setup unexpectedly removes Microsoft Encarta Virtual Globe 99.

CAUSE
=====

Encarta World Atlas 2000 Setup detects Encarta Virtual Globe 99 as a previous
version of Encarta World Atlas 2000.

RESOLUTION
==========

To work around this issue, reinstall Encarta Virtual Globe 99 after you install
Encarta Reference Suite 2000 or Encarta World Atlas 2000.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.
