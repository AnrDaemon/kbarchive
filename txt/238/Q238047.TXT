TITLE   :"Insufficient Hard Disk Space to Install Natural Language Query"
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER::
OPER/SYS:
KEYWORDS:kbenv kberrmsg kbsetup kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta Encyclopedia 2000 
 - Microsoft Encarta Encyclopedia Deluxe 2001 for Windows 
 - Microsoft Encarta Encyclopedia Standard 2001 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to install one or more of the following components

 - Microsoft Natural Language Search
 - Microsoft Text-to-Speech Engine
 - Microsoft Speech Recognition Engine

during Microsoft Encarta Encyclopedia 2000 Setup, you may receive an error
message similar to the following, even if sufficient hard disk space is
available to install these components:

   There is insufficient hard disk space to install Microsoft Natural Language
   Query (NLQ).

NOTE: This issue occurs only in Encarta Encyclopedia Deluxe 2000.

CAUSE
=====

This behavior can occur if less than 65 megabytes (MB) of hard disk space is
available on the hard disk on which Microsoft Windows is installed.

RESOLUTION
==========

To prevent this issue from occurring, make sure that at least 65 MB of space is
available on the hard disk on which Windows is installed before you install
Encarta Encyclopdia 2000 Deluxe.
