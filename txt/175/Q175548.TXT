DOCUMENT:Q175548  14-JUN-2001  [msnetwork]
TITLE   :About MSN Displays Incorrect Version of Internet Explorer
PRODUCT :The Microsoft Network
PROD/VER:WINDOWS:2.51,4.0,95
OPER/SYS:
KEYWORDS:kbui kbmsn

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - The Microsoft Network version 2.51 
 - Microsoft Internet Explorer version 4.0 for Windows 95 
 - the operating system: Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you view the About MSN dialog box in MSN, The Microsoft Network, the
version of Internet Explorer is shown as 4.0 (4.71.1712.5).

The correct build number of Internet Explorer is 4.71.1712.6.

CAUSE
=====

MSN obtains the build number of Internet Explorer from the Shdocvw.dll file. The
Shdocvw.dll file was not updated in the 4.71.1712.6 build of Internet Explorer.

STATUS
======

Microsoft has confirmed this to be a problem in The Microsoft Network version
2.51. Microsoft is researching this problem and will post new information here
as it becomes available.

MORE INFORMATION
================

To display the correct build number of Internet Explorer, click About Internet
Explorer on the Help menu in Internet Explorer.
