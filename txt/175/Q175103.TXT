DOCUMENT:Q175103  18-MAR-2000  [exchange]
TITLE   :XFOR: Migration Wizard Contains Inaccurate Help File Option
PRODUCT :Microsoft Exchange
PROD/VER:winnt:5.5
OPER/SYS:
KEYWORDS:kbusage

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

In the Migration Wizard, if you set the focus on the Help button and then press
F1, the following note is displayed:

   Click this to display an overview of this dialog box.
   For Help on an item, click ? at the top of the dialog box, and then
   click the item.

The Migration Wizard dialog boxes do not have the Help (?) button in the
upper-right corner of the dialog box.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server
version 5.5.
