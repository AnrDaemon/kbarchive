TITLE   :XWEB: Changing View in OWA Does Not Update Immediately
PRODUCT :Microsoft Exchange
PROD/VER:WinNT:5.5
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Outlook Web Access, version 5.5 
 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you change a view in the Outlook Web Access client using the Windows 16-bit
version of Netscape 3.01, the view does not immediately change.

NOTE: Refreshing the browser window will cause the changes to take effect.

STATUS
======

This problem occurs when you use Microsoft Outlook Web Access version 5.5 and
Netscape 3.01. Microsoft is researching this problem and will post new
information here in the Microsoft Knowledge Base as it becomes available.
