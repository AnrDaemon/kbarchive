DOCUMENT:Q175291  09-AUG-2001  [winnt]
TITLE   :Long Local or Global Group Names Do Not Display Properly
PRODUCT :Microsoft Windows NT
PROD/VER:WinNT:4.0
OPER/SYS:
KEYWORDS:kbsetup kbui kbusage

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 4.0 
 - Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you view local or global group membership names in User Manager, some long
names cannot be fully viewed from the Member Of windowpane.

CAUSE
=====

The Member Of windowpane cannot be resized to display some long group names. The
number of characters that can be viewed in the names are dependent on the point
size of the font being used for this window.


STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 4.0. We are
researching this problem and will post new information here in the Microsoft
Knowledge Base as it becomes available.

MORE INFORMATION
================

This issue does not cause any problems with the functionality of User Manager or
Windows NT.
