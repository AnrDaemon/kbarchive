TITLE   :Plus! For Kids: All Bitmap (.bmp) Files Open With Paint It!
PRODUCT :Microsoft Home Kids Products
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kbgraphic kbinterop

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Plus! for Kids, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

After you install Microsoft Plus! For Kids, version 1.0, all bitmap (.bmp)
graphic files open with the Plus! For Kids Paint It! program.

MORE INFORMATION
================

When you install Plus! For Kids, ALL .bmp files become associated with Paint
It!

To assign a new file type to .bmp graphic files, use the following steps:

1. Delete the .bmp file type.
   a. Double-click My Computer.

   b. On the View menu, click Options.

   c. Click the File Types tab.

   d. In the Registered File Types box, click Bitmap Image.

   e. Click Remove.

2. Make a new file type for .bmp graphic files.
   a. Double-click My Computer.

   b. On the View menu, click Options.

   c. Click the File Types tab.

   d. Click New.

   e. In the Description of Type box, type the following:

   "Bitmap Image" (without the quotation marks)

   f. In the Associated Extension box, type the following:

   "bmp pcx" (without the quotation marks)

   g. Click New.

   h. In the Action box, type the following:

   "open" (without the quotation marks)

   i. In the Application Used To Perform Action box, type the full path to the
      program you wish to use to open bitmap images. For example, if you wish to
      use Microsoft Paint, you would type the following:

   "C:\Program Files\Accessories\MSPAINT.EXE" (without the quotation marks)

      or click Browse to find the correct path.

   j. Click OK twice. The .bmp files will now be associated with the new
      program.

After the new file type has been created, you be able to launch Paint It! without
replacing the newly created file type.

Additional query words: plus kids for bmp pcx bitmap image association file take over
