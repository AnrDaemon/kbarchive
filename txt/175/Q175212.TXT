SUMMARY
=======

The MSChart control sample project demonstrates how you can use the MSChart
control to graphically display numeric information. This control allows you to
display several different 2D and 3D charts in a variety of ways by controlling
the appropriate properties. This sample also includes a supplemental Help file
that contains additional information about the MSChart control.

The sample is available for download from the Visual Basic Owner's site at:

   http://premium.microsoft.com/download/vbasic/ChrtSamp.exe (181Kb).

Note that you have to register to enter this site.

When you run the self-extracting executable, the following files are expanded
into the VBWebSamples directory of your Visual Basic installation:

 - ChrtSamp.vbp 2Kb

 - frmchart.frm 13Kb

 - frmchart.frx 11Kb

 - gas.xls 17Kb

 - modchart.bas 8Kb

 - mschart.cnt 1Kb

 - mschart.hlp 296Kb

 - Readme.txt 2Kb
