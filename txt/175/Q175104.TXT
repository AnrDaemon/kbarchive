TITLE   :XFOR: AppleDouble Attachments Lost When Message Converts
PRODUCT :Microsoft Exchange
PROD/VER:WinNT:5.5
OPER/SYS:
KEYWORDS:kb3rdparty kbusage

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

If a Microsoft Exchange Server computer receives an Internet message with
AppleDouble attachments (such as those generated by the Netscape mail client for
the Macintosh), the attachments are lost when the message is converted to an
Internet message for a POP3 client.

WORKAROUND
==========

This behavior does not occur if you access the attachments through the Microsoft
Exchange Client or Microsoft Outlook Client.
