TITLE   :Quickeys Err Msg: Cannot Find Requested Book
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER::2.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Quickeys, version 2.0 
-------------------------------------------------------------------------------

SUMMARY
=======

If Quickeys is running and you use the combination for a book you may get the
error message:

   Cannot find requested book

This may indicate a corrupt VIEWER.INI file.

MORE INFORMATION
================

To correct the problem:

1. Close the Quickeys.

2. Delete the file VIEWER.INI located in the C:\WINDOWS directory.

3. Restart any VIEWER application so Windows can rebuild the VIEWER.INI.

4. Exit the VIEWER application

5. Restart Quickeys.

6. Reset the key combinations in Quickeys.

Additional query words: 2.00 quick keys fail
