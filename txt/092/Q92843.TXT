-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Bookshelf for Windows, 1991, 1992, 1993, 1994, 1995 editions 
 - Microsoft Bookshelf for MS-DOS 
-------------------------------------------------------------------------------

SUMMARY
=======

This article discusses the availability of network versions of Microsoft
Bookshelf.

MORE INFORMATION
================

Bookshelf 1991, 1992, and 1993
------------------------------

Microsoft does not produce a network version of Bookshelf 1991, 1992, or 1993.
However, network versions are available from Online Computer Systems, Inc. for
both MS-DOS and Windows versions of Bookshelf. The MS-DOS version of Bookshelf
is the 1991 edition.

For more information about these network versions of Bookshelf, contact Online
Computer Systems, Inc. at (800) 922-9204.

Address
-------

Online Computer Systems, Inc.
20251 Century Blvd.
Germantown, MD 20874


Bookshelf 1994 for Windows
--------------------------

Microsoft provides a Multi-License Pak for both the academic and retail markets
of Bookshelf 1994 edition for Windows. For more information about this
Multi-License Pak, contact Microsoft Sales Information Center at (800)
426-9400.
