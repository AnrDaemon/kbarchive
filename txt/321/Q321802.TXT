TITLE   :SMS: Chinese Client Cannot Report Inventory to Japanese Server
PRODUCT :Microsoft Systems Management Server
PROD/VER::2.0
OPER/SYS:
KEYWORDS:kbsms200

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Systems Management Server version 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

Chinese-language Systems Management Server (SMS) clients may not be able to
report hardware or software inventory information to a Japanese SMS site server.
This issue occurs if the SQL server is not correctly configured for
international use.

MORE INFORMATION
================

Microsoft provides support for SMS 2.0 English-language client running on a
non-English operating system with some requirements. If the SQL server does not
meet these requirements, the issue that is described in this article may occur.
For additional information, click the article number below to view the article
in the Microsoft Knowledge Base:

   Q307531 SMS: Support for the SMS 2.0 English-Language Client Running on a
   Non-English Operating System
