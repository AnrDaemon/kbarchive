TITLE   :Mac Bookshelf 96-97: Bookshelf Guide Could Not Be Found
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:MACINTOSH:
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Bookshelf 1996-97 for Macintosh 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you try to access the Bookshelf Guide you may receive the message:

   Help for this window is not available because 'Bookshelf Guide' could not be
   found.

CAUSE
=====

This message may occur if certain files are missing or have been moved from the
Bookshelf 1996-97 folder.

RESOLUTION
==========

To prevent this error, the following files must be located in your Bookshelf
1996-97 folder on the hard drive.

 - Microsoft Bookshelf

 - Bookshelf Guide

If the files have been moved, try one of the following:

 - Try locating that file and dragging it back to this folder.

   -or-

 - Run the Installer program located on the Bookshelf compact disc.

MORE INFORMATION
================

If it is necessary to have the Microsoft Bookshelf application file located in a
different folder or on your desktop, an alias of the file should be created and
moved instead. To make an alias, do the following:

1. On your hard drive, open the Bookshelf 1996-97 folder.

2. Select (highlight) the Microsoft Bookshelf icon.

3. On the File menu, click Make Alias.

An alias icon appears in the folder. You can move this icon to any location on
your computer and use the icon to launch Bookshelf.
