SUMMARY
=======

The following file is available for download from the Microsoft Download
Center:

   DownloadDownload Cdextras.exe now
   (http://download.microsoft.com/download/powerpoint95/sample/1/WIN98/EN-US/cdextras.exe)

This file is a self-extracting executable file containing the following
additional PowerPoint templates.

Airmail.Pot   Brknbar.Pot   Black.Pot      Blstrip.Pot   Banner.Pot
Bluebox.Pot   Cheers.Pot    Bubbles.Pot    Celebrat.Pot  Brownbag.Pot
Checks.Pot    Dropstar.Pot  Elegant.Pot    Fiesta.Pot    Coins.Pot
Content.Txt   Diamond.Pot   Confetti.Pot   Corkbord.Pot  Flag.Pot
Currency.Pot  Fireball.Pot  Forest.Pot     Gridwrks.Pot  Locknkey.Pot
Filmstar.Pot  Jigsaw.Pot    Marble.Pot     Medical.Pot   Music.Pot
Island.Pot    Multbox.Pot   Neonlit.Pot    Numbers.Pot   Pharmacy.Pot
Movnglin.Pot  Patchwk.Pot   Redlines.Pot   Risk.Pot      Seashore.Pot
Pastel.Pot    Scroll.Pot    Shadebar.Pot   Southwst.Pot  Stratgic.Pot
Rules.Pot     Splats.Pot    Sunidays.Pot   Sunrise.Pot   Teched.Pot
Speed.Pot     Tablet.Pot    Theatre.Pot    Topline.Pot   World.Pot
Sunset.Pot    Travel.Pot    Triumph.Pot    Watercol.Pot

NOTE: The .pot extension indicates a PowerPoint template file format.

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

   Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.

MORE INFORMATION
================

The following sample files are located in the Valupack\PPSamples folder on the
PowerPoint 95 compact:

 - Calendar.ppt

 - Flowchrt.ppt

 - Printme.ppt

 - Tables.ppt

 - Timeline.ppt

For additional information about applying templates, please see the following
article in the Microsoft Knowledge Base:

   Q113373 PPT: Removing or Restoring Default Template
