TITLE   :Macintosh Multimedia: Cannot Access Program Guide
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER::
OPER/SYS:
KEYWORDS:kberrmsg kbmm kbimukbfaq

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Bookshelf 98 for Macintosh 
 - Microsoft Bookshelf 1996-97 for Macintosh 
 - Microsoft Encarta 98 Encyclopedia for Macintosh 
 - Microsoft Encarta 97 Encyclopedia for Macintosh 
 - Microsoft Cinemania for Macintosh, 1994, 1995, 1996, 1997 editions 
 - Microsoft Music Central for the Macintosh, 1996, 1997 editions 
-------------------------------------------------------------------------------

SYMPTOMS
========

On the Help menu, the guide does not start. Nothing happens, or you receive the
an error message. Possible error messages include the following:

   Apple Guide Extension was not found!

   One or more files required for Apple Guide are missing. To correct this
   problem, run the Installer on the Bookshelf 1996-97 CD.

CAUSE
=====

The problems occur because at least one of the following files is missing from
the active extensions folder:

 - Apple Guide
 - AppleScriptLib
 - AppleScript

RESOLUTION
==========

To resolve this issue, follow these steps:

1. Quit the program if it is running.

2. On the program compact disc, double-click Installer.

3. At the Easy Install window, click Customize.

4. Select the Apple Guide option appropriate for your computer and version of
   system software.

5. Click Install.

When the installation has completed, restart your computer and then start
Bookshelf.

MORE INFORMATION
================

The Help menu is the question mark in the upper-right corner of the menu bar.
