TITLE   :BUG: NMAKE Utility May Compile Project Files More Than Once
PRODUCT :Microsoft Programming Utilities
PROD/VER::4.0
OPER/SYS:
KEYWORDS:kbFortranPS kbLangFortrankbbuglist

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Program Maintenance Utility (NMAKE), included with:
    - Microsoft Fortran PowerStation for Windows 95 and Windows NT, version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

The NMAKE utility that is used to compile your project file may cause some of
the project's source files to be compiled more than once.

STATUS
======

Microsoft has confirmed this to be a bug in the Microsoft products listed at the
beginning of this article. We are researching this problem and will post new
information here in the Microsoft Knowledge Base as it becomes available.
