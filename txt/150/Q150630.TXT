The information in this article applies to:

 - Microsoft Schedule+ for Windows, version 7.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use the Seven Habits Wizard, you may get the following error message:

   Bad File Name.

RESOLUTION
==========

NOTE: DO NOT INSTALL THIS UPGRADE IF YOU HAVE PURCHASED OR INSTALLED THE SEVEN
HABITS TOOLS ADD-ON. All changes included in this upgrade are also included in
the SEVEN HABITS TOOLS ADD-ON.

Call Franklin Covey at 1-800-304-9799 or 1-800-330-8175 and request the file
7hbtools.zip. Franklin Covey will send this update through email.

7hbtools.zip includes fixes to a number of known bugs, as well as some refinement
of the features.
