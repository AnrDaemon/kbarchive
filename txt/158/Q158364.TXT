TITLE   :Flight Simulator Flight Shop & Flight Simulator for Windows 95
PRODUCT :Microsoft Home Games
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kb3rdpartykbfaq

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Flight Simulator for Windows 95, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

The BAO program Flight Simulator Flight Shop will not run in Flight Simulator
for Windows 95. The planes created in Flight Simulator Flight Shop will not run
in Flight Simulator for Windows 95. Adventures created in Flight Simulator
Flight Shop will not run in Flight Simulator for Windows 95.

RESOLUTION
==========

Microsoft has created a converter that translates Flight Simulator Flight Shop
planes and adventures for use with Flight Simulator for Windows 95. This
converter is available online from the following Microsoft World Wide Web site:

   http://www.microsoft.com/games/products.asp?filter=fsim

This converter is not supported by Microsoft Technical Support.

Additional query words: 1.00 flightsim fltsim95 FSFS converter planes adventures fs6 fs95
