TITLE   :GEX: Cannot Run GEX on Windows NT 3.51
PRODUCT :Microsoft Home Games
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kbfaq

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft GEX, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

GEX does not run on Windows NT 3.51.

RESOLUTION
==========

GEX requires Windows NT version 4.0 or Windows 95. For more information about
system requirements to run GEX, see the following article in the Microsoft
Knowledge Base:

   Q158021 GEX: System Requirements

Additional query words: 1.00 hang crash freeze up stops stop pauses pause doesn't start
