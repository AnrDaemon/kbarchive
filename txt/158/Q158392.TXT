TITLE   :Hellbender and MTM: Screen Goes Black with Hercules Dynamite
PRODUCT :Microsoft Home Games
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Hellbender for Windows, version 1.0 
 - Microsoft Monster Truck Madness, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

Your screen may turn black or your computer may stop responding (hang) when you
run games in 640 x 480 resolution using some Hercules video cards.

RESOLUTION
==========

The following troubleshooting steps may temporarily resolve this problem until
updated drivers are available from Hercules. Try one of the following:

 - Change the game resolution.
 - Restore the Windows 95 drivers (Windows 95 only).
 - Run the game in a window rather than full screen.

See below for more information about how to perform these troubleshooting steps.

Change the Game Resolution
--------------------------

Change the game resolution as follows:

1. Start the game.

2. In Hellbender, click Settings on the Options menu. In the Graphics tab,
   change the Game Resolution to 320 x 400.

   In Monster Truck Madness, click Graphics on the Options menu. Change the
   Simulation Resolution to 320 x 400.

Restore the Windows 95 Drivers (Windows 95 only)
------------------------------------------------

Restore the Windows 95 drivers as follows:

1. Run Dxsetup.exe, which is located in the DirectX folder on the game CD.

2. Click the Restore Windows 95 Drivers button and follow the instructions on
   the screen.

Run the Game in a Window Rather than Full Screen
------------------------------------------------

Run the game in a window rather than full screen. To do this, do the following:

1. In Hellbender, clear the Full Screen box in the starting dialog box.

   In Monster Truck Madness, click Graphics on the Options menu.

2. Clear the Automatic Full Screen box.
