TITLE   :Mac Encarta 97: Anthems Do Not Play with Low Memory
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER::1997 edition
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta 97 Encyclopedia for Macintosh 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you try to play an anthem in Encarta and your computer is running in a low
memory situation one of the following may occur:

 - the anthem may not play.

   -or-

 - You receive the following error message:

   There is insufficient free system memory for Encarta to complete this task.
   See the Encarta 97 Read Me for more assistance.

CAUSE
=====

Encarta is running with less than the suggested memory (listed in the Get Info
box for the Encarta 97 Encyclopedia icon). The suggested size is a number that
is set by the developers of the program. This amount cannot be adjusted.

The suggested size is the amount of memory that the program developers decided
Encarta needed to function optimally. You should always try to set the Preferred
size to a number at least as large as the suggested size.

RESOLUTION
==========

Optimize your computer's memory configuration and System memory usage so that
Encarta 97 is able to run with the preferred amount of memory.

For more information about determining the memory settings for a Macintosh
program, please see the following article in the Microsoft Knowledge Base:

   Q141682 How to Change Memory Allocations for Macintosh Programs


MORE INFORMATION
================

When a program runs with less than the recommended memory, some of the features
may not function properly.
