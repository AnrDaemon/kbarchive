TITLE   :Files with Long Extensions Bypass the Recycle Bin When Deleted
PRODUCT :Microsoft Windows NT
PROD/VER::4.0
OPER/SYS:
KEYWORDS:kbui

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 4.0 
 - Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

Files with extensions longer than three characters are not displayed in the
Windows NT 4.0 Recycle Bin when they are deleted from My Computer or the Windows
NT Explorer.


STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 4.0. We are
researching this problem and will post new information here in the Microsoft
Knowledge Base as it becomes available.


NOTE: This problem was rectifed in succeeding Service Packs. To resolve this
issue, update with the latest SP.

Additional query words: prodnt recycle bin delete trash can
