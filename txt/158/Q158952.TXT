TITLE   :RAS Detects Motorola BitSurfr Pro Incorrectly
PRODUCT :Microsoft Windows NT
PROD/VER:4.0
OPER/SYS:
KEYWORDS:kbhw kbHardware

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0 
 - Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

Remote Access Server detects a Motorola BitSurfr Pro as one of several models,
none of which is correct.

CAUSE
=====

The BitSurfr Pro is an unsupported ISDN adapter.

RESOLUTION
==========

Contact Motorola for an updated information file for the BitSurfr Pro.
