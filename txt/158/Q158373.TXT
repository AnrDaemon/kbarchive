TITLE   :Monster Truck Madness: Viewing Options Vary Depending on View
PRODUCT :Microsoft Home Games
PROD/VER::1.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Monster Truck Madness, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article discusses some of the driving view options available during game
play in Monster Truck Madness.

VIEW WITH DASHBOARD
-------------------

     Cockpit   Chase(Near)   Chase(Far)   Autorace Cam   Goodyear Blimp
-----------------------------------------------------------------------

LEFT    Yes       Yes           Yes            No                Yes
RIGHT   Yes       Yes           Yes            No                Yes
UP      No        Yes           Yes            No                Yes
DOWN  Rearview    Yes           Yes            No                Yes

VIEW WITHOUT DASHBOARD
----------------------

     Cockpit   Chase(Near)   Chase(Far)   Autorace Cam   Goodyear Blimp
-----------------------------------------------------------------------

LEFT    Yes      Yes            Yes            No                Yes
RIGHT   Yes      Yes            Yes            No                Yes
UP      Yes      Yes            Yes            No                Yes
DOWN    Yes      Yes            Yes            No                Yes

If the individual views are available, the following table describes what keys
are used to access them.

VIEW CONTROLS
-------------

View               Keys

Left               INSERT or NUMERIC KEYPAD 0
Right              DELETE or NUMERIC KEYPAD period (.)
Up                 NUMERIC KEYPAD MINUS SIGN (-)
Down               NUMERIC KEYPAD PLUS SIGN (+)

Additional query words: 1.00 outside belly under side top bottom see view truck
