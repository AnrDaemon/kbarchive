TITLE   :Not Reminded at Correct Time in Full Screen Command Prompt
PRODUCT :Microsoft Schedule+ for Windows
PROD/VER:WINDOWS:7.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Schedule+ for Windows, version 7.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you are working in a full-screen command prompt session, and you have set
reminders in Schedule+, the reminders will not be displayed at the correct time.

STATUS
======

Microsoft has confirmed this to be a problem with Microsoft Schedule+ version
7.0. We are researching this problem and will post new information here in the
Microsoft Knowledge Base as it becomes available.


Additional query words: 7.00 schedule plus ms-dos
