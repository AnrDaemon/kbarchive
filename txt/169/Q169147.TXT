TITLE   :Briefcase Error "DLLCanUnloadNow"
PRODUCT :Windows for Workgroups and Windows NT Networking Issues
PROD/VER:WINDOWS:95; winnt:4.0
OPER/SYS:
KEYWORDS:kbinterop win95 win98 win98se

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 4.0 
 - Microsoft Windows NT Server version 4.0 
 - Microsoft Windows 98 
 - Microsoft Windows 98 Second Edition 
 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you copy an .xls file into My Briefcase you may receive the following
error:

   DllCanUnloadNow
   Beginning of function

Your only option is to click the OK button. Once you click OK then the file is
copied into My Briefcase.


MORE INFORMATION
================

At this time we have not experienced any problems or file corruption in relation
to this pop-up message. It is safe to ignore the message.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 4.0. We are
researching this problem and will post new information here in the Microsoft
Knowledge Base as it becomes available.
