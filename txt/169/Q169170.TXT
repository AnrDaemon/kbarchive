TITLE   :RAS Error 692 Using USR Sportster ISDN 128k
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:kbnetwork

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 4.0 
 - Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you try to use Dial-Up Networking with a US Robotics Sportster ISDN 128k,
you may see the following error:

   Error 692: Hardware failure in port or attached device.


RESOLUTION
==========

To resolve this issue, install the latest driver from 3Com for your hardware
device.

For information about how to obtain and install the latest hardware drivers,
please see the following 3Com Web site:

   http://support.3com.com/
