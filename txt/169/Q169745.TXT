TITLE   :Golf 2.0: PlayerNet Patch Available, Includes Modem Play
PRODUCT :Microsoft Home Games
PROD/VER:WINDOWS:2.0
OPER/SYS:
KEYWORDS:kbfile kbusage

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Golf for Windows, version 2.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Pnupdate.exe, a self-extracting, self-installing patch file, allows you to
play a Multiplayer game of Golf 2.0 via a modem.

NOTE: For the patch to work, you must have the full retail version of Golf 2.0
with PlayerNet installed before installing the patch.

MORE INFORMATION
================

The following file is available for download from the Microsoft Download
Center:

   Pnupdate.exe
   (http://download.microsoft.com/download/golf20win/Patch/1/WIN98/EN-US/Pnupdate.exe)

Release Date: Sep-04-1997

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

   Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.
