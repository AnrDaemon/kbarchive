DOCUMENT:Q169561  09-AUG-2001  [winnt]
TITLE   :Unable to View Subdirectory if Parent Directory has NO ACCESS
PRODUCT :Microsoft Windows NT
PROD/VER:WinNT:4.0
OPER/SYS:
KEYWORDS:kbnetwork

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 4.0 
 - Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

Windows NT Explorer will not show subdirectories if the parent directory has NO
ACCESS permissions even though the user has full control at the subdirectory
level. The subdirectory is viewable in File Manager and at the command prompt.



STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 4.0. We are
researching this problem and will post new information here in the Microsoft
Knowledge Base as it becomes available.
