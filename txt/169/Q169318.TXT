TITLE   :Golf 3.0: Theme Pack Available for Download

PROD/VER:WINDOWS:3.0
OPER/SYS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Golf for Windows, version 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Golf version 3.0 Theme Pack is now available from the Microsoft Download
Center. The theme pack includes golf-related sounds, desktop backgrounds, and
animated cursors for Windows users who have Microsoft Plus! installed.

Glftheme.exe, a self-extracting, self-installing file downloads to your Themes
directory on your hard disk by default. The file is approximately 515 KB in
size.

MORE INFORMATION
================

The following file is available for download from the Microsoft Download
Center:

   Glftheme.exe
   (http://download.microsoft.com/download/golf30win95/theme/1/W95/EN-US/Glftheme.exe)

Release Date: Sep-04-1997

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

   Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.
