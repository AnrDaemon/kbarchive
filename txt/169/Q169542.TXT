TITLE   :WD97: Messages When Attempting to Fax from Word
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:95,97
OPER/SYS:
KEYWORDS:kberrmsg kbinterop kbusage winword word97

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to fax from Microsoft Word, you may receive the error message

 - No transport provider was available for delivery to this recipient

   -or-

 - You may receive "Undeliverable" mail messages for each document you send.

CAUSE
=====

This problem occurs for either of the following reasons:

 - Microsoft Fax or another fax service transport is not installed.

   -or-

 - You have not added the fax service to your user profile.

RESOLUTION
==========

To work around this problem, install Microsoft Fax or a different fax service
transport to your computer and configure your user profile to include the fax
service.

MORE INFORMATION
================

For additional information, please see the following article in the Microsoft
Knowledge Base:

   Q152196 How to Install and Use Microsoft Fax

   Q141347 Err Msg: No Transport Provider Was Available for Delivery

   Windows 98 Users

   Q187549 Microsoft Fax Does Not Appear in Add/Remove Programs
