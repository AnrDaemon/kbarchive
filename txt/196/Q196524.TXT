TITLE   :How to Generate a List of Keyboard Commands for Word

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SUMMARY
=======

This article describes how to obtain a list of keyboard assignments in Microsoft
Word for Windows.

MORE INFORMATION
================

1. On the Tools menu, point to Macro and then click Macros.

2. From the "Macros in" list, choose Word Commands.

3. Select ListCommands in the Macro Name list.

4. Click Run.

5. In the List Commands dialog box, choose either "Current menu and keyboard
   settings" or "All Word commands".

6. Click OK.

Word creates a new document that lists the commands and their associated shortcut
key assignments.

Additional query words: keystrokes listcommands list commands shortcut shortcuts keyboard assignments combination 8.0 8.00
