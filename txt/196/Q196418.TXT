DOCUMENT:Q196418  14-MAY-1999  [exchange]
TITLE   :XADM: Move Server Pauses if not Enough Space in Application Log
PRODUCT :Microsoft Exchange
PROD/VER:WINNT:5.5
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------


SUMMARY
=======

Prior to executing the Move Server operation, you need to make sure that the
application event log has enough room for the Move Server Wizard events. If the
application log is full, the move will be paused until you change settings in
the event log.
