DOCUMENT:Q196311  08-APR-2000  [exchange]
TITLE   :XFOR: Chat Performance Monitor Counter Does Not Reset After Paus
PRODUCT :Microsoft Exchange
PROD/VER:winnt:5.5
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SYMPTOMS
========


After you pause and then restart the Microsoft Exchange Chat Server service, the
Chat Service Total Client Connects counter in Performance Monitor does not reset
to "0".

STATUS
======

To resolve this problem, obtain the latest service pack for Exchange Server 5.5.
For additional information, please see the following article in the Microsoft
Knowledge Base:

   Q191014 XGEN: How to Obtain the Latest Exchange Server 5.5 Service Pack
