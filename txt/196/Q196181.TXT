TITLE   :XADM: Moveserv.htm Contents and Index are not Accessible
PRODUCT :Microsoft Exchange
PROD/VER:WINNT:5.5
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

If Moveserv.htm is opened from Internet Explorer 3.0, with the Security level
set to High, the Contents and Index are not accessible.

Moveserv.htm comes with the Move Server Wizard.

WORKAROUND
==========

Set the Security level to Medium on the browser to view the Contents and Index.
