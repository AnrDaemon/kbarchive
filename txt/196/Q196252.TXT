TITLE   :XFOR: 119 Character Limit for "Administrator info" in Web Admin
PRODUCT :Microsoft Exchange
PROD/VER:winnt:5.5
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SYMPTOMS
========


When you attempt to create an "Administration Information" message with 119
characters using the Microsoft Exchange Chat Service Web Administrator, the
message will not be created.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Chat Server
version 5.5. This problem was corrected in the latest Microsoft Exchange 5.5
U.S. Service Pack. For information on obtaining the service pack, query on the
following word in the Microsoft Knowledge Base:

   " SERVPACK " (without the quotation marks)
