SUMMARY
=======

The Inf Generator tool (Ig.exe) can help you customize the automatic
installation of Windows 95 in a corporate networking environment.

The following features are available in Inf Generator:

 - You can specify separate settings for the company, department, and user when
   Windows 95 is being installed. This is especially useful when static TCP/IP
   addresses are assigned.

 - You can configure a batch Setup that prevents icons for The Microsoft Network
   and the Inbox from being created on the desktop.

MORE INFORMATION
================

To install the Inf Generator tool, follow these steps:

1. Download the Ig.exe file from one of the services listed below to an empty
   folder.

2. Double-click the Ig.exe file to extract the files it contains.

3. Double-click the Readme.txt file for installation instructions.

You can find Ig.exe,
(http://support.microsoft.com/download/support/mslfiles/Ig.exe,) a
self-extracting file, on the following services:

 - Internet (anonymous FTP):

   ftp ftp.microsoft.com
   Change to the PEROPSYS/WIN_NEWS/FREESOFTWARE directory
   Get Ig.exe

 - Internet (World Wide Web):

   http://www.microsoft.com/windows/software/servpak1/isprof.htm

Note that the Inf Generator tool is not the same tool as the Inf Installer tool
included on the Windows 95 CD-ROM.
