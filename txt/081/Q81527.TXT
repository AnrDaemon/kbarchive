The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a, 3.1 
 - Microsoft Windows for Workgroups version 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

The following is a list of the international versions of Microsoft Windows that
are currently available. Additional localized versions are being considered for
the future.

MORE INFORMATION
================

Most of these versions are OEM and hardware specific and must be run on the
hardware suggested by the manufacturer.
