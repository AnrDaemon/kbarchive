TITLE   :Encarta Interactive World Atlas 2001: Home Screen Is Blank
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER::
OPER/SYS:
KEYWORDS:kbenv kbgraphic kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta Encyclopedia Deluxe 2001 for Windows 
 - Microsoft Encarta Interactive World Atlas 2001 
 - Microsoft Encarta Reference Suite 2001 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start Microsoft Encarta Interactive World Atlas 2001, the home screen
may be blank.

Additionally, most of the commands on the Features menu may be blank.

CAUSE
=====

This behavior can occur if the Microsoft Internet Explorer Security settings are
configured to High.

RESOLUTION
==========

To resolve this issue, lower the Security settings in Internet Explorer:

1. Click Start, point to Settings, and then click Control Panel.

2. Double-click Internet Options.

3. On the Security tab, click Internet, and then click Default Level.

4. Click OK.

Additional query words: multi multi-media media mm black blue green home page
