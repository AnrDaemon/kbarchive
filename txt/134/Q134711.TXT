TITLE   :X-Tree Gold for Windows 4.0 Produces GP Fault
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:95
OPER/SYS:
KEYWORDS:win95 appscomp kbAppCompatibility

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you are scrolling down through a file using the DOWN ARROW key in X-Tree
Gold for Windows 4.0, you may receive a general protection (GP) fault error
message.

CAUSE
=====

The GP fault is caused by an uninitialized variable in X-Tree Gold for Windows
4.0. This uninitialized variable has no apparent effect in earlier versions of
Windows.

RESOLUTION
==========

To work around this problem, use the mouse to scroll down through files instead
of using the DOWN ARROW key. For additional information about resolving this
problem, please contact Symantec.
