DOCUMENT:Q134292  13-OCT-1999  [msnetwork]
TITLE   :Cannot Paste Windows Explorer or Desktop Object into BBS
PRODUCT :The Microsoft Network
PROD/VER:WINDOWS:1.2,1.3
OPER/SYS:
KEYWORDS:kbmsn

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - The Microsoft Network versions 1.2, 1.3 
-------------------------------------------------------------------------------

SYMPTOMS
========

You are unable to paste an object you copied from Windows Explorer or the
desktop into a bulletin board (BBS) message.

RESOLUTION
==========

Use the mouse to drag the object into the message.

Additional query words: msn bbs
