TITLE   :Cannot Edit Microsoft Excel Worksheet in WordPad
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:95
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you insert a Microsoft Excel worksheet into WordPad and create a chart
from the worksheet data, it may not be possible to enter data in the Microsoft
Excel worksheet.

CAUSE
=====

This problem can occur if you create an invalid chart in Microsoft Excel and
ignore alerts from Microsoft Excel that something is wrong.

RESOLUTION
==========

To resolve this problem, deactivate and then reactivate the object. After you do
so, re-create the chart.
