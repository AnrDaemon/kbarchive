TITLE   :Windows 95 Cannot Detect EtherExpress 16/16TP on Pentium PC
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

Windows 95 Setup may fail to detect the Intel EtherExpress 16/16TP network
adapter in your computer.

CAUSE
=====

This problem can occur if you are installing Windows 95 on a Pentium-based
computer. There is a known incompatibility between the Intel EtherExpress
16/16TP network adapter and Pentium-based computers with a Phoenix 4.04 BIOS.

RESOLUTION
==========

Please contact Intel for information about resolving this problem.

MORE INFORMATION
================

Windows 95 cannot detect or set up an EtherExpress 16/16TP network adapter on
Pentium-based computers with a Phoenix 4.04 BIOS. If you run the Softset tool
from Intel, the network adapter diagnostics test fails.
