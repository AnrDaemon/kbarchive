-------------------------------------------------------------------------------
The information in this article applies to:

 - MSPRESS Mastering Microsoft Visual Basic 4.0 ISBN 1-55615-913-7, version 1.0 
 - MSPRESS Mastering Microsoft Visual FoxPro 3.0, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When accessing an Audio Video Interleave (.avi) file in Microsoft Mastering
series products, you receive the error message:

   AVI needs VIDS.CVID

RESOLUTION
==========

If you are using Windows 3.x, run Setup from the Vfw folder located on the
compact disc.

If you are using Windows 95, add Video Compression from the Windows 95 Control
Panel Add/Remove Programs.

MORE INFORMATION
================

Microsoft Mastering Series products require Video for Windows to play video
files. Video for Windows is not automatically installed during the normal setup
process.
