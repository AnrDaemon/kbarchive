TITLE   :PC Subito Corso - Multimediale di Inglese Fatal Error Err Msg
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:95
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run the PC Subito Corso - Multimediale di Inglese program, an English
learning program for Italian speaking people from Eudida Software, you receive
the following error message:

   Fatal error Creating First SoundRec Instance

CAUSE
=====

The program depends upon the Windows 3.1 version of Sound Recorder
(Soundrec.exe) to run. Windows 95 replaces Sound Recorder with an updated
version (Sndrec32.exe).

RESOLUTION
==========

Contact Eudida Software support for an updated version of this program.

MORE INFORMATION
================

This program works correctly in Windows 3.1. If you cannot contact Eudida, you
can attempt to locate the Windows 3.1 version of Sound Recorder (Soundrec.exe)
from the Windows 3.1 installation disks and copy it to the Windows 95 Windows
folder.
