TITLE   :Some NE2000 Compatable Drivers Not Included in Windows NT 3.51
PRODUCT :Microsoft Windows NT
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.51 
 - Microsoft Windows NT Server version 3.51 
-------------------------------------------------------------------------------

Although the Kingston PCMCIA EtherRX network interface card (NIC) driver is
listed in the Windows NT 3.51 Hardware Compatibility List (HCL), it is not
included with Windows NT 3.51.

To use the Kingston PCMCIA EtherRX NIC in Windows NT 3.51, install the
NE2000 NIC driver.


The following NICs also use the NE2000 driver:

 - IBM PCMCIA Ethernet Adapter II

 - Socket Communications Socket EA PCMCIA
