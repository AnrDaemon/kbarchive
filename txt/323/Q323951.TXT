======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Dungeon Siege, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to transfer a multiplayer character from the trial
version of Microsoft Dungeon Siege 1.0 to the retail version of Dungeon Siege.

Single-player games that were saved in the trial version of Dungeon Siege are not
compatible with the retail version of Dungeon Siege.

MORE INFORMATION
================

Transfer a Multiplayer Character
--------------------------------

To transfer a multiplayer character from the trial version to the retail version
of Dungeon Siege, follow these steps:

1. Start Windows Explorer, and then locate and open the following folder

   <path>\My Documents\Dungeon Siege Demo\Save

   where <path> is the path to the My Documents folder.

2. Right-click the file of the character that you want to transfer, and then
   click Copy.

   NOTE: The saved game file names are in the form of <CharacterName
   (X-XX-XX)>.dsparty.

3. Locate and then right-click the following folder:

   <path>\My Documents\Dungeon Siege\Save

4. On the shortcut menu, click Paste.
