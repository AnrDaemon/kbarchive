-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows for Workgroups version 3.11 
-------------------------------------------------------------------------------

The following problems can occur when you use Delrina WinFax Pro 3.0 for
Networks with Windows for Workgroups 3.11 configured for network use:

 - Conflicts occur between the Microsoft At Work PC Fax drivers and the WinFax
   for Windows terminate-and-stay-resident (TSR) programs.

 - You receive device conflict error messages.

 - You receive the

   Netbios not present

   error message when loading the WinFax Pro 3.0 drivers.

 - You receive the

   Image contains invalid data

   error message.

Delrina Technology offers an application note (document number 2018) that
suggests ways to correct these problems. This application note also
recommends document number 9003 for additional information.

For more information, contact Delrina Technical Support.
