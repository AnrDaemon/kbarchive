DOCUMENT:Q198615  21-MAY-1999  [homegame]
TITLE   :MCM: Program Quits and You Are Returned to the Windows Desktop
PRODUCT :Microsoft Home Games
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kbimu msgame

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Motocross Madness, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to start Microsoft Motocross Madness, the program may quit
without warning and you may be returned to the Windows desktop.

CAUSE
=====

This behavior can occur if the Microsoft SideWinder Game Controller software
CD-ROM is in your CD-ROM drive.

RESOLUTION
==========

To resolve this issue, remove the SideWinder Game Controller software CD-ROM
from the CD-ROM drive, insert the Motocross Madness CD-ROM into the CD-ROM
drive, and then start Motocross Madness.

MORE INFORMATION
================

This issue does not occur with the SideWinder Game Controller software CD-ROM
that is included in the product box with the Microsoft SideWinder FreeStyle Pro
game pad, as this disc also contains the Motocross Madness program files.
