DOCUMENT:Q198324  14-NOV-2000  [word97]
TITLE   :WD97: Cannot Assign Shortcut Keys for Insert Bookmark
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:word97

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you try to assign the Insert Bookmark command to a shortcut keys assignment
(on the Tools menu, click Customize, and then click Keyboard), the Insert
Bookmark command is missing.

CAUSE
=====

The command name is EditBookmark.

WORKAROUND
==========

To assign EditBookmark to a new shortcut keys assignment, follow these steps:

1. On the Tools menu, click Customize.

2. In the Customize dialog box, click Keyboard.

3. In the Customize Keyboard dialog box, under Categories, click Edit.

4. Under Commands, click EditBookmark.

5. Press the new shortcut keys assignment for EditBookmark.

6. Click Assign, and then click Close to return to the Customize dialog box.

7. Click Close.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article.
