TITLE   :WD97: Outline Number "Number Format" Wrong or Changes to Square
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:
OPER/SYS:
KEYWORDS:word97 kbnumbering

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you customize an Outline Numbered style, one or both of the following may
occur:

Case 1
------

When you select a level and then click to select a previous level number, the
previous level number is added to the right of the number format.

Case 2
------

When you move the insertion point in the Number Format box to the left of the
number format and then click to select a previous level number, the numbering in
the Number Format box changes to a square.

WORKAROUND
==========

In an Outline Numbered style, when you want to include the number from a
previous level as part of the numbering scheme, follow these steps:

1. On the Format menu, click Bullets And Numbering.

2. Click to select the Outline Numbered tab.

3. Click to select the numbering style you want to change, and then click
   Customize.

4. Under Level, click to select the level you want to change. For example, click
   to select Level 2.

5. Click the Number Format box, and then move the insertion point to the left of
   the number format (the left-most position in the Number Format box).

6. In the previous level number box, click to select the previous level. For
   example, click to select Level 1.

7. Make any other changes you want to the Outline Numbered List, and then click
   OK.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article.

MORE INFORMATION
================

To reset a numbering style to its default, follow these steps:

1. On the Format menu, click Bullets And Numbering.

2. On either the Numbered or Outline Numbered tab, click to select the numbering
   style you want to reset, and then click Reset.

   When the following message is shown

   Do you want to reset this gallery position to the default setting?

   click Yes.

For more information about how to customize outline numbering, click the Office
Assistant, type "outline numbered", click Search, and then click to view "Modify
outline numbering formats".

NOTE: If the Assistant is hidden, click the Office Assistant button on the
Standard toolbar. If Microsoft Help is not installed on your computer, please
see the following article in the Microsoft Knowledge Base:

   Q120802 Office: How to Add/Remove a Single Office Program or Component
