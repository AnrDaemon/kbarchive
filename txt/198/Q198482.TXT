TITLE   :XWEB: Next/Previous Button Confused in Message with PF Link
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:5.0,5.5
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Active Server Components, version 5.0 
 - Microsoft Outlook Web Access, version 5.5 
 - Microsoft Outlook Web Access, version 5.5 Service Pack 1 
-------------------------------------------------------------------------------

SYMPTOMS
========


When you use Outlook Web Access (OWA) to open a public folder link in a mail
message, if the link window is minimized, and you click either the Next or
Previous button, the next item opened is from the public folder, not the Inbox.

WORKAROUND
==========

Instead of minimizing the public folder link, close the public folder link, and
then click either the Previous or Next button.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article.
