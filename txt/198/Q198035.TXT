DOCUMENT:Q198035  20-MAY-2002  [win95x]
TITLE   :User Profiles Not Copied to Server with Updated Mprserv.dll File
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER::
OPER/SYS:
KEYWORDS:win95kbfixlist

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

On a computer running Windows 95 and configured for user profiles on a Microsoft
or Novell NetWare network, user profiles may not be copied to the server as
expected when you log off the network.

CAUSE
=====

This problem can occur if the Mprserv.dll file version 4.00.959 or earlier is
installed on the computer running Windows 95.

For additional information on updated versions of Mprserv.dll that can cause this
problem, see the following article in the Microsoft Knowledge Base:

   Q184838 OLE Problems Using MSNDS Are Not Resolved By NWNP32 Update

RESOLUTION
==========


STATUS
======

This problem no longer occurs in Windows 98. To resolve this problem, install
the current version of Windows. For information about the current version of
Windows, visit http://www.microsoft.com/windows.
