DOCUMENT:Q198411  07-MAY-2002  [homegame]
TITLE   :Combat FS: Color Palette of Mission Briefing Maps Is Distorted
PRODUCT :Microsoft Home Games
PROD/VER::1.0
OPER/SYS:
KEYWORDS:kbdisplay kbenv kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Combat Flight Simulator: WWII Europe Series, version 1.0 
 - Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you play a mission in Microsoft Combat Flight Simulator, the color palette
of the map on the Mission Briefing screen may be distorted.

CAUSE
=====

This behavior can occur if an outdated version of the Comctl32.ocx component is
installed on your computer.

RESOLUTION
==========

To resolve this issue, install the current version of the Comctl32.ocx component
on your computer.

To do this, you must install a program that includes the latest version of the
Comctl32.ocx component. An example of such a program is the latest version of
Microsoft Internet Explorer.

You can download and install the latest version of Internet Explorer free of
charge from the following Microsoft Web site:

   http://www.microsoft.com/windows/ie/download/default.htm

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Combat Flight
Simulator: WWII Europe Series, version 1.0.
