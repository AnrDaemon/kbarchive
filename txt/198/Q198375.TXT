TITLE   :WD97: Registered Trademark Symbol Conversion Problems
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:word97

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you save a Word document containing a registered trademark symbol (ASCII
0174) in a text file format (such as text only or text with line breaks), the
registered trademark symbol may not be retained in the new format.

WORKAROUND
==========

To ensure that a registered trademark symbol is retained when you save a file in
text format, see the table in the "More Information" section of this article and
use one of the text file formats that indicates "OK" under Conversion Results.

MORE INFORMATION
================

The following table presents the results of what happens to the registered
trademark symbol in a Word document when you save the document in a certain text
format. In the table, "OK" means that the registered trademark symbol is
retained.

Text Type Format                    Conversion Results
------------------------------------------------------

Text Only                           (r)

Text only with Line Breaks          OK

MS-DOS Text                         (r)

MS-DOS Text with Line Breaks        r

MS-DOS Text with Layout (Windows)   r

Text With Layout (Macintosh)        N/A

For additional information, please see the following article in the Microsoft
Knowledge Base:

   Q58917 WD: Using the ASCII and ANSI Characters in Word for Windows
