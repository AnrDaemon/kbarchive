DOCUMENT:Q198313  14-NOV-2000  [word97]
TITLE   :WD97: Can't Cancel Options of "Cannot Find Data Source" Error
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:word97 kbmerge

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you open a mail merge main document whose underlying data source is
missing, the following error message appears:

   <Document name> is a mail merge main document. Word cannot find its
   data source, <Data source document name>.

On the error message dialog box, there are a Find Data Source button and an
Options button.

When you click the Options button, there is no Cancel button on the Options
dialog box to allow you to return to the prior dialog box if you decide to
choose the Find Data Source button instead.

WORKAROUND
==========

To return to the previous dialog box and display the choices of Find Data Source
or Options, press ESC on your keyboard.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.
