DOCUMENT:Q136180  19-NOV-1999  [powerpt]
TITLE   :Sorry, This Presentation Contains Open Embedded Objects...
PRODUCT :Microsoft PowerPoint for Windows
PROD/VER:WINDOWS:7.0
OPER/SYS:
KEYWORDS:kberrmsg kbinterop kbole

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft PowerPoint for Windows 95, version 7.0 
-------------------------------------------------------------------------------


SYMPTOMS
========

If you create a graph and click away from it, and then immediately click the
Start button, and then click Shut Down, you may receive the following error
message:

   Sorry, this presentation contains open embedded objects which cannot be
   closed automatically. Close the objects and try again.

At this point, you will receive the same error message if you try to close the
presentation.

RESOLUTION
==========

Change to a different slide, or click Outline on the View menu. This will
release the graph and allow PowerPoint to quit properly.

NOTE: You may receive similar symptoms any time you create an OLE object and
click away from it.

Additional query words: 7.00 powerpnt msgraph 5.0 winppt macppt graph4 ms graph chart lock hang shut off
