TITLE   :Performance Problems in Windows NT 3.51 With Intergraph PC-NFS
PRODUCT :Microsoft Windows NT
PROD/VER::3.51
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.51 
 - Microsoft Windows NT Server version 3.51 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run Intergraph PC-NFS version 1.01.05 under Windows NT 3.51, you may
encounter performance problems. In addition, if you use the MS-DOS Editor
(EDIT.COM) concurrently with Intergraph PC-NFS, the editor stops responding
(hangs).

RESOLUTION
==========

To correct these problems, obtain version 2.01.06 of Intergraph PC-NFS for
Windows NT 3.51. For additional information:

 - please see the following article in the Microsoft Knowledge Base:

   ARTICLE-ID: Q134695
   TITLE : BW-Connect NFS 3.5 for Windows NT Hangs Windows NT 3.51 VDMs

 - Contact Intergraph Technical Support at (800) 633-7248.
