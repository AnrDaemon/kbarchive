======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Julia Child Home Cooking with Master Chefs version 1.0 
 - the operating system: Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

Microsoft Julia Child does not include Windows 95 Setup instructions.

RESOLUTION
==========

This article provides instructions on how to install Microsoft Julia Child on
computers running Microsoft Windows 95.

NOTE: The following instructions assume your CD-ROM drive is drive D.

To setup Julia Child in Windows 95, do the following:

1. Insert the Julia Child compact disc into your CD-ROM drive.

2. Click the Start button, and then click Run.

3. In the Open dialog box, type the following and click OK.

   "d:\setup" (without the quotation marks)

4. Follow the on-screen installation instructions.

MORE INFORMATION
================

Windows 95 applications use an Autorun.inf file to automatically install
multimedia applications. Julia Child does not contain an Autorun.inf file.

The compact disc case insert states Microsoft Julia Child Home Cooking with
Master Chefs will install to Microsoft Windows operating system versions 3.1 or
later, including Windows 95. However, the installation program does not
automatically detect and install Julia Child on systems running Microsoft
Windows 95.
