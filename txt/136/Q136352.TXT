TITLE   :Fury3: Screen Turns Blank with Actix Picaso
PRODUCT :Microsoft Home Games
PROD/VER::1.0
OPER/SYS:
KEYWORDS:

======================================================================
1.00
WINDOWS
kb3rdparty kbdisplay

-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Fury 3 for Windows, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you are using a Picaso Actix video card and you choose N when you are asked
to retry a mission again, the screen turns black.

RESOLUTION
==========

Exit the game and restart.

This problem only occurs when you run Fury3 in Windows 3.x.


Microsoft has confirmed this to be a problem in Fury3 version 1.0 for Windows.
