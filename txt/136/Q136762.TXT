TITLE   :BETA: Windows NT Shell Technology Preview Beta Available
PRODUCT :Microsoft Windows NT
PROD/VER::3.51
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.51 
 - Microsoft Windows NT Server version 3.51 
-------------------------------------------------------------------------------

SUMMARY
=======

A beta version of the Windows NT Shell Technology Preview Update is now
available. This beta update gives Windows NT version 3.51 a user interface like
that offered on Windows 95.

It is available on the following sources:

 - MSDN Level 2 CD-ROM

 - ftp.microsoft.com/bussys/winnt/winnt-unsup-ed/newshell

NOTE: The Shell Technology Preview has not been fully tested and should not be
deployed in production environments. There is currently no official Microsoft
Product Support Services support available for using the Shell Technology
Preview.

More information on this topic may be available from Microsoft Information
Services. Microsoft Information Services are available on the World Wide Web by
connecting to http://www.microsoft.com. The FTP site is located at
ftp.microsoft.com.
