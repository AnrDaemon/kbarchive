DOCUMENT:Q119567  13-JUN-2001  [accother]
TITLE   :Cannot Distribute Access 2.0 ODBC Driver with ADT Apps
PRODUCT :Microsoft Access Distribution Kit
PROD/VER:WINDOWS:2.0,97
OPER/SYS:
KEYWORDS:kbinterop kbdta

======================================================================

-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Access 97 
 - Microsoft Access Developer's Toolkit, version 2.0 
-------------------------------------------------------------------------------

Moderate: Requires basic macro, coding, and interoperability skills.

SYMPTOMS
========

The Microsoft Access version 2.0 ODBC is not licensed for distribution with
applications created with the Microsoft Access Developer's Toolkit (ADT).

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.

MORE INFORMATION
================

Version 01.00.15.21 of the ODBC driver is included with Microsoft Office
Professional version 4.3 and Microsoft Access version 2.0. You cannot distribute
this driver as part of an application even though this file is not listed as one
of the files prohibited from distribution in the ADT documentation.
