TITLE   :Kyocera FS-3500A PCL Driver Will Not Print from Lower Paper Tray
PRODUCT :Microsoft Windows NT
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
3.50

WINDOWS

kbprint kbbug3.50 kbfix3.50.sp2

-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.5 
 - Microsoft Windows NT Server version 3.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Windows NT PCL driver for the Kyocera FS-3500A laser printer does not allow
you print from the lower paper tray. The printer pulls paper from the upper
instead.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT Workstation and
Windows NT Server version 3.5. This problem was corrected in the latest U.S.
Service Pack for Windows NT version 3.5. For information on obtaining the
Service Pack, query on the following word in the Microsoft Knowledge Base
(without the spaces):

   S E R V P A C K

The FS-3500A laser printer is manufactured by Kyocera, a vendor independent of
Microsoft; we make no warranty, implied or otherwise, regarding this product's
performance or reliability.
