======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for AppleTalk Networks, version 3.1d 
-------------------------------------------------------------------------------

SYMPTOMS
========

When signing into version 3.1d of Microsoft Mail for AppleTalk Networks, it is
possible to receive the following error message:

   The Microsoft Mail Server is not the correct version for use with this desk
   accessory. Please see your Network Manager.

RESOLUTION
==========

This error is generated when attempting to sign into a version 3.0 Microsoft
Mail Server using version 3.1 of the Mail workstation.

While it is possible to use both 3.0 and 3.1 versions of the workstation software
to connect to a Mail 3.1 server, only the 3.0 version of the workstation can be
used with a Mail 3.0 server.

Additional query words: 3.10 3.00 incorrect
