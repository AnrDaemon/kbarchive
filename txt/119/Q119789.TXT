TITLE   :Word and Excel for Windows NT Require Windows NT Version 3.5

-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 3.1 
 - Microsoft Windows NT Workstation version 3.1 
 - Microsoft Windows NT Advanced Server, version 3.1 
 - Microsoft Windows NT Workstation version 3.5 
 - Microsoft Windows NT Server version 3.5 
-------------------------------------------------------------------------------

SUMMARY
=======

Word and Excel for Windows NT require Windows NT version 3.5 or later.

MORE INFORMATION
================

This is because Windows NT 3.5 now has OLE2 compatibility which is required for
Word for Windows NT and Excel for Windows NT.

Microsoft Word for Windows NT version 6.0 and Excel for Windows NT version 5.0 do
not run on Windows NT version 3.1 or Windows NT Advanced Server version 3.1.
