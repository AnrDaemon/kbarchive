DOCUMENT:Q118309  01-DEC-1999  [macmail]
TITLE   :CONN: Mail Conn 3.2 Installer Incompatible with PowerTalk
PRODUCT :Microsoft Mail For Appletalk Networks
PROD/VER::3.1c,3.1d
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail Gateway for Appletalk Networks, versions 3.1c, 3.1d 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you try to install the Microsoft Mail Connection 3.2 Gateway on a Macintosh
running the PowerTalk Extension, the Installer launches but you cannot get past
the initial screen.

CAUSE
=====

There is an INIT conflict between the Installer program and the PowerTalk
Extension.

RESOLUTION
==========

Move the PowerTalk Extension from the Extensions Folder and onto the Desktop,
then restart the Macintosh. After installing the Gateway, you can move the
PowerTalk Extension back to the Extensions folder.

STATUS
======

Microsoft is researching this problem and will post new information here in the
Microsoft Knowledge Base as it becomes available.
