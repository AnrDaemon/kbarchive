TITLE   :Undocumented SYS(2033) Returns Microsoft & System Folders
PRODUCT :Microsoft FoxPro
PROD/VER:MACINTOSH:2.6a,3.0b
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft FoxPro for Macintosh, version 2.6a 
 - Microsoft Visual FoxPro for Macintosh, version 3.0b 
-------------------------------------------------------------------------------

SUMMARY
=======

The system function, SYS(2033) returns the path of the System, Preferences,
Extensions, or Microsoft folders.

MORE INFORMATION
================

Syntax: SYS(2033, <expN>)

The following values for <expN> return the paths listed below:

   0    Path to System folder
   1    Path to Extensions folder
   2    Path to Preferences folder
   3    Path to Microsoft folder

Additional query words: VFoxMac FoxMac directory subdirectory pathname path name powermac ppc 68k volume local hard drive akz
