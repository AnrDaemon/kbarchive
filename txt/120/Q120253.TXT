SUMMARY
=======

The Multimedia Registration Packet (MDRK.EXE) is a collection of updated
documents that developers can use for help in writing multimedia applications
for the Windows environment,

MORE INFORMATION
================

The following files are available for download from the Microsoft Download
Center:

Mdrk.exe

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

   Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.


The packet includes Microsoft Word and ASCII text versions of the following
files:

MMREG Multimedia Registration Kit
---------------------------------

This kit enables a developer to register new formats and product IDs with
Microsoft for the purpose of providing support for the format by other
developers.

MSFTMM Multimedia Document Overview
-----------------------------------

This document provides a list of the current documents available from the
Multimedia Technologies group at Microsoft.

RIFFNEW Multimedia Data Types and Techniques
--------------------------------------------

This document provides a comprehensive description of available multimedia data
types including Resource Interchange File Format (RIFF) and wave types.

PROD_IDS New Multimedia IDs
---------------------------

This is a complete list of registered manufacturer's and their registered
products.

MMREG.H
-------

This header file contains multimedia data types and product IDs that you can
include in the source code of a multimedia C program.
