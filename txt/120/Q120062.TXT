TITLE   :Corel Draw 5.0 Running in Second VDM Loses Data
PRODUCT :Microsoft Windows NT
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
3.50

WINDOWS

kb3rdparty bug3.50

-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.5 
 - Microsoft Windows NT Server version 3.5 
-------------------------------------------------------------------------------

SUMMARY
=======

If you run separate versions of Corel Draw 5.0 in separate VDMs, Corel's temp
file algorithm may cause data loss. Corel Draw is currently working on a fix (as
of 8/30/94).
