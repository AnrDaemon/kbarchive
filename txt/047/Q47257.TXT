SUMMARY
=======

Random real-time weather variations add realism to Flight Simulator's Version
4.00 flying environment. Select Menu 3-Enviro, Option D (Weather Generator), to
display the Weather Generator menu. This menu contains the following options:

   Option           Description
   ------           -----------

   Cloud Buildup    Makes clouds build up and dissipate as you
                    fly.

   Front Drift      Creates weather fronts that can drift through
                    the flight area. You can identify these by
                    their distinctive line of overcast clouds.

   Wind Changes     Varies the winds at different altitudes.

   Turbulence       Creates turbulence layers.
   Layers
