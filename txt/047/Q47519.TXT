The information in this article applies to:

 - Microsoft Mail for AppleTalk Networks, versions 3.0, 3.0a, 3.0b, 3.1, 3.1a, 3.1c 
-------------------------------------------------------------------------------

SUMMARY
=======

The following are keyboard shortcuts for the Microsoft Mail Send Note, Phone,
and Image windows:

Description                         Command Notes
---------------------------------------------------------------------

Address Mail                        COMMAND+A
Copy Selected Text or Image         COMMAND+C
Enclose File                        COMMAND+E
Return Receipt                      COMMAND+R          2.0, 2.0a, 2.0b only
Send Message                        COMMAND+S
Urgent                              COMMAND+U
Paste Text or Image                 COMMAND+V
Close Window                        COMMAND+W
Cut Selected Text or Image          COMMAND+X
Get Help                            COMMAND+?
Options                             COMMAND+O            3.0 and later only
Add recipients to Address Book      COMMAND+O, COMMAND+A 3.0 and later only
Retain copy in Sent mail folder     COMMAND+O, COMMAND+S 3.0 and later only
Return Receipt                      COMMAND+O, COMMAND+R 3.0 and later only
Urgent                              COMMAND+O, COMMAND+U 3.0 and later only

For more information, see Appendix A of the Microsoft Mail "User's Guide."

Additional query words: 3.00 3.00a 3.00b 3.10 3.10a 3.10c
