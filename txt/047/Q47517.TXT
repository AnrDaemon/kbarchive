The information in this article applies to:

 - Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The following are keyboard shortcuts for the Microsoft Mail Send Inquiry
window:

   Description                         Command
   -----------                         -------

   Copy Selected Text                  COMMAND+C
   System Error                        COMMAND+E
   System Hang                         COMMAND+H
   Help!                               COMMAND+L
   No                                  COMMAND+N
   Other                               COMMAND+O
   Send Message                        COMMAND+S
   Paste Text                          COMMAND+V
   Close Window                        COMMAND+W
   Cut Selected Text                   COMMAND+X
   Yes                                 COMMAND+Y
   Get Help                            COMMAND+?

MORE INFORMATION
================

For more information, see Appendix A of the "Microsoft Mail User's Guide."
