TITLE   :XCLN: Running 16-Bit Exchange Client on NT Is Not Supported
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:4.0,5.0
OPER/SYS:
KEYWORDS:kbinterop

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Windows 3.x client, versions 4.0, 5.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run the 16-bit Windows client Setup on Windows NT, you may receive the
following error:

   This Setup is not intended for your version of Windows.

CAUSE
=====

This is by product design; running the 16-bit Windows client on a machine
running Microsoft Windows NT is not supported.
