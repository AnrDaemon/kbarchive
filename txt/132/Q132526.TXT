SUMMARY
=======

In July and August 1995, Microsoft released two versions of Microsoft Exposition
software. Exposition is a multimedia catalog that includes demos of many
Microsoft products.

To run the catalog, do the following:

1. On the File menu in Program Manager, click Run.

2. In the Command Line box, type "<drive>:\Expo" (without the quotation
   marks), where <drive> is the drive letter of your CD-ROM drive.

NOTE: These instructions, in addition to information on the necessary system
requirements for this product, are printed on each CD-ROM.

For more information about how to perform this task in Windows, see your Windows
printed documentation or online Help.

If you need further assistance, call the Setup Support Line at (425) 635- 7135.
