TITLE   :Tutor Err Msg: Skyspace Caused an Invalid Page Fault...
PRODUCT :Microsoft Home Kids Products
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kb3rdparty kberrmsg kbhw kbsound mskids kbimu kbHardware

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft My Personal Tutor: Sam's Hide and Seek Adventure, version 1.0 
 - Microsoft My Personal Tutor: Sky's Space Station Voyage, version 1.0 
 - Microsoft My Personal Tutor: Turru's Daring Sea Quest, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to start one of the My Personal Tutor titles listed in the
applies to section of this article, you may receive one of the following error
messages:

 - Skyspace caused an invalid page fault in module <unknown>.

 - Skyspace caused an invalid page fault in module Kernel32.dll.

 - Cannot find picture resource file \Skyspace\Skyspace.mmp.

CAUSE
=====

These behavior can occur if you are using an IBM Aptiva computer.

RESOLUTION
==========

To resolve this issue, download and install the "Aptiva 2140/2142 Internet
Explorer 4.0 updates for Microsoft DirectX 5.0 and Crystal Audio drivers" from
the following IBM Web site:

   http://www3.pc.ibm.com/techinfo/5492.html
