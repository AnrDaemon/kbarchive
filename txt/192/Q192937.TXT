TITLE   :Motocross Madness: White Lines Displayed in Background
PRODUCT :Microsoft Home Games
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kb3rdparty kbdisplay kbhw kburl kbimu msgame kbHardwarekbfaq

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Motocross Madness, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start a race or a practice game in Motocross Madness, you may see a
white horiziontal line displayed in the background.

CAUSE
=====

This behavior can occur if you use a Dell Optiplex GX1 computer with an ATI 3D
Rage Pro AGP video adapter installed in the computer.

RESOLUTION
==========

To resolve this issue, download and install the latest video drivers for the ATI
3D Rage Pro AGP display adapter from the following Dell Computer Web site:

   http://support.dell.com/filelib

If the issue continues to occur, download and install the latest version of
DirectX from the following Microsoft Web site:

   http://www.microsoft.com/directx/homeuser/downloads/default.asp

MORE INFORMATION
================

Older ATI 3D Rage Pro AGP video drivers for the Dell Optiplex GX1 are not fully
compatible with DirectX. Motocross Madness uses DirectX to display graphics in
the game, including the background and other objects.
