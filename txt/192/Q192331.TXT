TITLE   :AbirNet SessionWall-3 with Windows NT 4.0 Displays Blue Screen
PRODUCT :Microsoft Windows NT
PROD/VER:WinNT:4.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0 
 - Microsoft Windows NT Server, Enterprise Edition version 4.0 
 - Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you apply AbirNet SessionWall-3 Release 2.1 to Windows NT 4.0 Service Pack
3 or later, the system may display blue screen STOP messages such as the
following:

 - On trying to save a new rule:

   STOP 0x0000000A

 - On exiting from the program:

   STOP 0x0000001E

 - On starting the program:

   STOP 0x0000000A

CAUSE
=====

This problem is caused by running AbirNet SessionWall-3 Release 2.1 with Windows
NT 4.0 Service Pack 3 or later.

RESOLUTION
==========

To resolve this problem, get the "High Performance Enhanced Windows NT Driver
Patch" for Release 2.1 or Release 3.0 or later of SessionWall-3 from the AbirNet
Web site.
