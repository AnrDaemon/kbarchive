SUMMARY
=======

Dbf2rs.exe is a self-extracting executable that contains a Visual FoxPro program
named Dbf2rs.prg, which shows how to convert a Visual FoxPro cursor to an
ActiveX Data Objects (ADO) recordset. The executable file also contains the
necessary ADO include files. Running the program in Visual FoxPro creates an ADO
recordset from a Visual FoxPro cursor.
