The information in this article applies to:

 - Microsoft Train Simulator, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft Train Simulator now has a Tools Update available for download. This
article discusses the location of this update, and how to install it.

MORE INFORMATION
================

The Microsoft Train Simulator Tools Update contains updated versions of
MakeAce.exe and Conv3ds.exe, plus updated documentation for these tools. These
tools allow you to create new objects, including locomotives, for use in
Microsoft Train Simulator.

To download and install this update, follow these steps:

1. See the following Microsoft Web site:

   http://www.microsoft.com/games/trainsim
   (http://www.microsoft.com/games/trainsim)

2. Click "DOWNLOAD THE TOOLS UPDATE NOW".

3. In the File Download dialog box, click "Save this program to disk", and then
   click OK.

4. In the Save As dialog box, click Desktop in the "Save in" list box, and then
   click Save.

5. When the download is complete, click Close.

6. Disconnect your computer from the Internet.

7. On the desktop, double-click MSTStoolsupdate.exe.

8. Click OK.

9. Confirm the path the program is being extracted to, and then click Unzip.

   NOTE: The default path is c:\Program Files\Microsoft Games\Train
   Simulator\Utils. If you have changed the default installation path of Train
   Simulator, make sure that this path matches the path the program is installed
   to.

10. Click OK.

   NOTE: The ToolsUpdateReadme file opens automatically.

11. Quit Notepad when you are finished reading the ToolsUpdateReadme file.

   The update is now complete.
