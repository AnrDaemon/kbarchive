TITLE   :FS2002: How to Import Adventures from Flight Simulator 2000
PRODUCT :Microsoft Home Games
PROD/VER::
OPER/SYS:
KEYWORDS:kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Flight Simulator 2002 Professional Edition 
 - Microsoft Flight Simulator 2002 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes how to import your Microsoft Flight Simulator 2000
Adventures in order to make them available in Microsoft Flight Simulator 2002.

MORE INFORMATION
================

Microsoft Flight Simulator 2000 Adventures are stored in the following default
location:

   C:\Program Files\Microsoft Games\FS2000\ADV

If you have Adventures designed for Flight Simulator 2000 that you want to use in
Flight Simulator 2002, place the files in the fs2002\flights\fs2kadv folder on
your hard disk. To do this, follow these steps:

1. Start Windows Explorer, and then navigate to the following location:

   C:\Program Files\Microsoft Games\FS2000\ADV

2. Under ADV, expand the folder that contains the Adventures that you want. For
   example, expand ADVANCED.

3. On the Edit menu, click Select All, and then click Copy.

4. Navigate to the following folder:

   C:\Program Files\Microsoft Games\FS2002\flights\fs2kadv

5. On the Edit menu, click Paste.

6. Quit Windows Explorer.

To play the Flight Simulator 2000 Adventure in Flight Simulator 2002, follow
these steps:

1. Start Microsoft Flight Simulator 2002, and then click SELECT A FLIGHT.

2. In the "Choose a category" list, click "Flight Simulator 2000 Adventures"

3. In the "Choose a flight" list, select the flight that you want, and then
   click the FLY NOW! tab.
