SUMMARY
=======

The Hewlett-Packard (HP) PaintJet Series Printer Driver (1.01) for Microsoft
Windows version 3.0 is available for the PaintJet and PaintJet XL (for a nominal
fee) and offers several features that enhance the printers ability to produce
color documents.

MORE INFORMATION
================

New Features Include
--------------------

 - Screen-to-printer color-appearance matching (not available for transparency
   printing).

 - PANTONE** certified colors (not available for transparency printing).

 - Installable HP MasterType library printer font outlines.

 - Mixed text and graphics in both portrait and landscape modes.

Installing the Driver
---------------------

NOTE: At least 3 megabytes (MB) of available hard disk space is needed to install
the driver.

1. From the Control Panel, select Printers.

2. Choose Add Printers. The Printers dialog box expands todisplay the List of
   Printers and the Install button.

3. Scroll to the end of the list of printers and double click Unlisted Printer.
   The Add Unlisted Printer dialog box appears.

4. Insert the HP PaintJet Series Driver (1.01) for Windows 3.0 Installation Disk
   into the disk drive.

5. In the Directories list box, double click the drive designator for the drive
   that contains the installation disk.

6. Choose HPPJ1.DRV in the Driver Files list box.

7. If your printer is connected to LPT1 and the driver is assigned to LPT1 and
   Active, choose OK to close the Printers dialog box.

NOTE: The driver does not function correctly unless the CG Times font is
installed from the Installation disk. You must install this font, even if you do
not plan to use it.

To order the driver, contact HP Distribution.
