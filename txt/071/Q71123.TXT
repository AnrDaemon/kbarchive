TITLE   :Display Driver Version 1.06 for NEC Graphics Engine (DGIS)
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

To ensure proper operation of Microsoft Windows 3.0 and Windows applications,
verify that you are using version 1.06 or later of the NEC DGIS driver. To
verify the version number, select Setup from within the Windows Program Manager.
If you experience problems with a driver version earlier than 1.06, you should
obtain the latest version of the DGIS driver from NEC.

MORE INFORMATION
================

The NEC DGIS driver for Windows 3.0 is a high-resolution display driver
supporting resolutions up to 1024 x 768. The latest version of the driver is
available from the NEC bulletin board in a single self-extracting file called
MGEWN30.EXE. According to the README.1ST file with version 1.06 (dated 2/25/91),
the following corrections have been made:

 - Missing text in Excel 3.0 spreadsheet cells has been corrected.

 - Reversal of dark and light gray colors has been corrected.

The MGEWN30.EXE self-extracting file creates the following files (their
respective sizes, dates, and times as well):
