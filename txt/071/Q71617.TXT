TITLE   :Taipei Is Unreadable in 1024 x 768 Resolution
PRODUCT :Microsoft Home Games
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
SUMMARY
=======

The Taipei game packaged with the Windows Entertainment Pack (WEP1) may be
unreadable when Windows is run in 1024 x 768 resolution.

The WEP only supports those drivers shipped with Windows 3.00. Third-party
drivers that allow Windows to be run at 1024 x 768 are manufactured by vendors
independent of Microsoft, and we make no warranty, implied or otherwise,
regarding these drivers performance or reliability.

Solution
--------

Configure Windows for standard VGA, running at 640 x 480 resolution.
