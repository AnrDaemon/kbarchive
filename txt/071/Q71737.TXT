TITLE   :Aldus PageMaker version 4.0 and Bitstream Facelift
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

Bitstream Technical Support has confirmed that if you use Bitstream Facelift
versions earlier than 1.2 with Aldus PageMaker version 4.0, an Unrecoverable
Application Error (UAE) will be generated.

Registered users of Bitstream Facelift versions earlier than 1.2 should call
Bitstream technical support to obtain an update to version 1.2.

MORE INFORMATION
================

For more information, contact Bitstream Technical Support.
