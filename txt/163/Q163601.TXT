TITLE   :WD97: More Cool Stuff Template Refers to Personal Web Server
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SUMMARY
=======

The More Cool Stuff template which ships with Word 97 states that the Personal
Web Server (PWebSrv) is located on the Word 97 or Office 97 CD in the ValuPack
folder. The Personal Web Server did not ship with Word 97 or Office 97.

MORE INFORMATION
================

The Microsoft Personal Web Server for Windows 95 turns any Windows 95 computer
into a Web server and enables easy publication of personal Web pages.

The Personal Web Server can be downloaded from the Download page of the Internet
Explorer home page at

   http://www.microsoft.com/ie/download/
