TITLE   :WD97: "Network or File Permission Error" Saving Document
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:kb3rdparty kbnetwork kbdta kbconversion WordCon word97 kbconvert

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you save an existing file on a Novell server to a path that has been mapped
using the MAP ROOT command, you may receive the following error message:

   Network or file permission error

RESOLUTION
==========

To correct this problem, obtain Microsoft Word 97 Service Release 2 (SR-2).

STATUS
======

Microsoft has confirmed this to be a problem in Word 97. This problem was
corrected in Microsoft Office 97 Service Release 2 (SR-2).

For additional information about SR-2, please see the following article in the
Microsoft Knowledge Base:

   Q151261 OFF97: How to Obtain and Install MS Office 97 SR-2
