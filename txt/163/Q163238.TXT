TITLE   :Team Manager 97 not present on Step by Step CD-ROM
PRODUCT :Microsoft Press
PROD/VER::
OPER/SYS:
KEYWORDS:kbdocerr

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - MSPRESS Microsoft Team Manager 97 Step by Step 
-------------------------------------------------------------------------------

SUMMARY
=======

The last page of the book states that "The enclosed CD-ROM contains Microsoft
Team Manager 97 as well as timesaving, ready-to-use practice files that
complement the lessons in this book." Microsoft Team Manager 97 is not included
with this book.

MORE INFORMATION
================

Microsoft Team Manager 97 is available as a separate software product from your
local software retailer. For further information, please check the following
World Wide Web site:

   http://www.microsoft.com/teammanager/

If you purchased this book based on the indication that Microsoft Team Manager
was included on the companion CD-ROM, you can return this book (postage prepaid)
with your receipt for a full refund. The book must be accompanied by the packing
slip or bookstore receipt, either original or photocopied.

In the United States, please send returns (including sales receipt) to:

   Microsoft Press
   Customer Service Returns
   3106 Longhorn Blvd.
   Austin, TX 78758
   USA

If you have any questions regarding Microsoft Press product returns, please call
Microsoft Press Sales at (800) MS-PRESS.

If you are outside the United States, contact your local Microsoft Press
distributor for further information on returning this book. A list of Microsoft
Press distributors worldwide is available at the following World Wide Web site:

   http://mspress.microsoft.com/worldwide/

Correction:

Page 306, paragraph 1:
Replace first paragraph with:

   The enclosed CD-ROM contains timesaving, ready-to-use practice files that
   complement the lessons in this book. To use the CD, you'll need Microsoft
   Team Manager 97 and either the Windows 95 operating system or version 3.51
   Service Pack 5 or later of the Windows NT operating system.

Microsoft Press is committed to providing informative and accurate books. All
comments and corrections listed above are ready for inclusion in future
printings of this book. If you have a later printing of this book, it may
already contain the above corrections.
