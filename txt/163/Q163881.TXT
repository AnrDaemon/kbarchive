DOCUMENT:Q163881  09-AUG-2001  [winnt]
TITLE   :Windows NT Does not Display Some Fonts
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:kb3rdparty kbdisplay

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 4.0 
 - Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------


SYMPTOMS
========

Some characters in the fonts of third-party applications are not displayed
correctly on Windows NT; they are displayed properly on Windows 3.x and Windows
95.

This has been observed in the custom Polish fonts that are shipped with CorelDRAW
5 (Corel Corporation).

RESOLUTION
==========

To resolve this problem, obtain the hotfix mentioned below. To use the fix, you
must have Windows NT 4.0 Service Pack 2 installed on the system.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 4.0. This
problem was corrected in the latest Microsoft Windows NT 4.0 U.S. Service Pack.
For information on obtaining the service pack, query on the following word in
the Microsoft Knowledge Base (without the spaces):

   S E R V P A C K
