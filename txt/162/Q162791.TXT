TITLE   :"Access Denied" Error Accessing IBM LanServer 4.0 Share
PRODUCT :Microsoft Windows NT
PROD/VER:WINNT:4.0
OPER/SYS:
KEYWORDS:kb3rdparty

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 4.0 
 - Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

Trying to run a program from a share set to Execute Only on a computer running
IBM LanServer 4.0 gives an Access Denied error message. This problem only occurs
on Windows NT; a Windows 95 or an OS/2 client can run the program correctly with
no error.

RESOLUTION
==========

To resolve this problem, grant Read and Execute permissions on the share.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 4.0. We are
researching this problem and will post new information here in the Microsoft
Knowledge Base as it becomes available.
