DOCUMENT:Q162485  19-DEC-1999  [exchange]
TITLE   :XWEB: ASC Doesn't Show Number of Unread Messages
PRODUCT :Microsoft Exchange
PROD/VER:winnt:5.0
OPER/SYS:
KEYWORDS:kbusage

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use the Microsoft Exchange Active Server Components to view messages in
your inbox, the total number of messages in the inbox is displayed but the
number of unread message is not.

CAUSE
=====

This behavior is by design.
