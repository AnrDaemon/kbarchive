TITLE   :Remote Access Client Forgets Saved Password
PRODUCT :Microsoft Windows NT
PROD/VER:WINNT:4.0
OPER/SYS:
KEYWORDS:kbnetwork

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 4.0 
 - Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

Occasionally, the RAS client forgets a saved password. The saved password must
be reentered.

CAUSE
=====

In certain situations, the RAS client clears the saved password. A fix for this
problem is available in Service Pack 3 for Windows NT 4.0.

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT Workstation and
Server version 4.0. This problem was corrected in the latest US Service Pack for
Windows NT and Windows NT Advanced Server. For information on obtaining this
update, query on the following word in the Microsoft Knowledge Base (without the
spaces):

   S E R V P A C K
