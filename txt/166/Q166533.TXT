TITLE   :XFOR: Message Body Corruption Due to Ambiguous Codepage
PRODUCT :Microsoft Exchange
PROD/VER:5.0
OPER/SYS:
KEYWORDS:kbusage

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use Microsoft Exchange Server, version 5.0 and the Internet Mail
Service (IMS), messages sent with ambiguous codepages may be delivered with
corrupt message bodies. This typically happens when the message is sent from the
Microsoft Exchange Client and starts with a font that has an unknown or
ambiguous codepage.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server
version 5.0. This problem was corrected in the latest Microsoft Exchange Server
5.0 U.S. Service Pack. For information on obtaining the service pack, query on
the following word in the Microsoft Knowledge Base (without the spaces):

   S E R V P A C K
