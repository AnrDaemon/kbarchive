TITLE   :WD97: ODMA File Open Not Supported in Mail Merge
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:kb3rdparty

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you attempt to create a data source from the Mail Merge Helper, the Word
Open dialog box is displayed instead of the Document Management System (DMS)
custom Open dialog box.

WORKAROUND
==========


STATUS
======

Microsoft is researching this problem and will post new information here in the
Microsoft Knowledge Base as it becomes available.
