DOCUMENT:Q166614  10-FEB-2000  [exchange]
TITLE   :XFOR: Attachments Migrated from ccMail Are Not Viewable
PRODUCT :Microsoft Exchange
PROD/VER:winnt:5.0
OPER/SYS:
KEYWORDS:kbusage exc5

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.0 
-------------------------------------------------------------------------------


SYMPTOMS
========

Fax attachments migrated from cc:Mail may not be viewable from the client
mailbox. Fax attachments in user messages do not migrate properly from cc:Mail
DB version DB6. Some of the attachments are missing extensions and none are in
the correct PCX format.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server
version 5.0. This problem was corrected in the latest Microsoft Exchange Server
5.0 U.S. Service Pack. For information on obtaining the service pack, query on
the following word in the Microsoft Knowledge Base (without the spaces):

   S E R V P A C K

For additional information on similar behavior in Exchange Server version 5.5,
click the article number below to view the article in the Microsoft Knowledge
Base:

   Q184053 Fax from cc:fax to Exchange Recipient Is Not Readable
