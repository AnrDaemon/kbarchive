DOCUMENT:Q166696  10-AUG-2001  [winnt]
TITLE   :NT 4 Err Msg: "The INF OEMNADDI is missing the referenced file"
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:kberrmsg kbnetworkkbbuglist

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 4.0 
 - Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When performing an unattended installation of Windows NT 4.0, an error may occur
and you will receive the following error messages:

   The INF OEMNADDI is missing the referenced file description section files.

   The error prevents the operation from continuing

   Software not correctly installed

   Want to exit?

If the message window is moved, the following window can be observed in the
background:

   Upgrading DIGI PCIMAC ISDN Adapter Family Driver

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 4.0. This
problem was corrected in the latest Microsoft Windows NT 4.0 U.S. Service Pack.
For information on obtaining the service pack, query on the following word in
the Microsoft Knowledge Base (without the spaces):

   S E R V P A C K


Additional query words: prodnt ntras
