DOCUMENT:Q166069  07-SEP-2001  [word97]
TITLE   :WV: Document Properties Blank When Printed from Word Viewer
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:
OPER/SYS:
KEYWORDS:kbprint kbdta kbviewer word97

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word Viewer 97-2000 
-------------------------------------------------------------------------------

SYMPTOMS
========

In Microsoft Word Viewer 97-2000, if you select the option to include the
document properties when you print the document, the document properties print
out blank when you print the document.

WORKAROUND
==========

To work around this problem, print a screen shot of the Document Properties
dialog box by following these steps:

1. On the File menu, click Summary Info.

2. Click the Statistics button.

3. Press ALT+PRINT SCRN on your keyboard to copy the picture of the dialog box
   to the Clipboard.

4. Start Microsoft Paint.

5. On the Edit menu in Microsoft Paint, click Paste.

6. Print the picture from Paint.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.

REFERENCES
==========

Knowledge Base article:
