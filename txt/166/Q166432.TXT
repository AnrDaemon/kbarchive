TITLE   :Chinese Code Converter Saves Incorrect Code Page Format
PRODUCT :Microsoft Windows NT
PROD/VER:4.0
OPER/SYS:
KEYWORDS:kbnetwork

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation, Pan Chinese Edition, version 4 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Chinese Code Converter in Pan Chinese Windows NT Workstation version 4.0
saves a converted file in the wrong code page if you view another code page and
click Save As. This is true even if you choose the original code page.

RESOLUTION
==========

After converting a file using Convert Now, click Save As button to save the
converted content before viewing another file.

For additional information, please see the following article in the Microsoft
Knowledge Base:

   ARTICLE-ID: Q166430
   TITLE : PCNTW: Error Viewing Text File Using Chinese Code Converter.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article. We are researching this problem and will post new
information here in the Microsoft Knowledge Base as it becomes available.
