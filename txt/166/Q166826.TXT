DOCUMENT:Q166826  25-MAR-2002  [win95x]
TITLE   :Data Read from ROM PC Card Device May Be Damaged
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER::2,2.1
OPER/SYS:
KEYWORDS:kbhw osr2 win95 kbHardwarekbfixlist

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 OEM Service Release, versions 2, 2.1 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you read data or copy files from a ROM PC Card device with Windows 95 OEM
Service Release 2 or 2.1 (OSR2), the read data or copied file may be damaged.

CAUSE
=====

The PC Card driver in OSR2 may use an incorrect address for the memory window
for reading data from ROM PC Card devices.

RESOLUTION
==========


STATUS
======

This problem no longer occurs in Windows 98. To resolve this problem, install
the current version of Windows. For information about the current version of
Windows, visit http://www.microsoft.com/windows.

MORE INFORMATION
================


Additional query words: 95 pcmcia
