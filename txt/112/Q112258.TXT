TITLE   :PC Win: Err Msg: There Was An Error Opening the Specified File
PRODUCT :Microsoft Mail For PC Networks
PROD/VER:WINDOWS:3.0b
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for Windows, version 3.0b 
-------------------------------------------------------------------------------

SUMMARY
=======

When you try to export Mail folders to your local hard disk drive, after you
type your password, the following error message is displayed:

   There was an error opening the specified file.

This problem occurs if you are not using SHARE.EXE. To avoid this problem, either
load the SHARE.EXE file or export your folder to a file server.

Additional query words: 3.00b
