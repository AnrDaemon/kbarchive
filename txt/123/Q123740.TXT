TITLE   :Unexpected Error Accessing MS Mail Postoffice Through GSNW
PRODUCT :Microsoft Windows NT
PROD/VER:3.10 3.50
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 3.1 
 - Microsoft Windows NT Workstation version 3.1 
 - Microsoft Windows NT Advanced Server, version 3.1 
 - Microsoft Windows NT Workstation version 3.5 
 - Microsoft Windows NT Server version 3.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run Windows for Workgroups and attempt to access a Microsoft Mail
version 3.2 postoffice on a Novell NetWare 3.12 server, an Unexpected Error
appears. This problem occurs when you attempt to access the NetWare server
through a Windows NT computer running Gateway Services for NetWare (GSNW).

STATUS
======

Microsoft has confirmed this to be a problem in Windows NT Workstation and
Windows NT Server version 3.5. This problem was corrected in the latest U.S.
Service Pack for Windows NT version 3.5. For information on obtaining the
Service Pack, query on the following word in the Microsoft Knowledge Base
(without the spaces):

   S E R V P A C K

Additional query words: prodnt post office
