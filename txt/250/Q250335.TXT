The information in this article applies to:

 - Microsoft Visual C++, 32-bit Learning Edition, version 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

VCSPInfo.exe is an unsupported sample that provides information on the Visual
C++ 6.0 Service Pack.

One of the methods to find out which Visual C++ 6.0 Service Pack you have
installed is to examine the file versions of the DLLs installed on the system.
The following Microsoft Knowledge Base article lists file versions of DLLs
installed by all the Visual C++ 6.0 Service Packs:

   Q194295 HOWTO: Tell That Visual Studio 6.0 Service Packs Are Installed

The VCSPInfo.exe application finds the file versions of the Visual C++ 6.0 DLLs
listed in the referenced article, performs a comparison with the file version of
the same DLL installed on your computer, and displays the results.

MORE INFORMATION
================

The following file is available for download from the Microsoft Download
Center:

   VCSPInfo.exe
   (http://download.microsoft.com/download/vc60std/info/6/NT4/EN-US/VCSPInfo.exe)

The self-extracting VCSPInfo.exe file contains the following files:

+-------------------------+
| File Name       | Size  | 
+-------------------------+
| VCSPInfoSRC.exe | 44KB  | 
+-------------------------+
| VCSPInfo.exe    | 132KB | 
+-------------------------+
| VCSPInfo.hlp    | 9KB   | 
+-------------------------+

VCSPInfoSRC.exe contains the source code for the application, VCSPInfo.exe is the
application, and VCSPInfo.hlp is the help file. All the files are unsupported.

For additional information about how to download Microsoft Support files, click

   Q119591 How to Obtain Microsoft Support Files from Online Services
