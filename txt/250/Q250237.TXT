SUMMARY
=======

This article discusses use of the ScriptIt.exe utility to automate the setup of
partitions and printers with Microsoft Windows NT Server 4.0.

MORE INFORMATION
================

The ScriptIt.exe utility is a tool for automating interactive software
installations and system configuration tasks that are not part of the current
Unattend or Deployment process provided with Windows NT 4.0. ScriptIt can be
used to start a process, monitor the window titles of active processes, and when
appropriate send a set of keystrokes to the correct window.

You can download the ScriptIt.exe utility from the following Microsoft Web site:

   http://www.microsoft.com/technet/winnt/
