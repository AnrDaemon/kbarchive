TITLE   :SMS: IME May Not Be Viewable in a Remote Control Session
PRODUCT :Microsoft Systems Management Server
PROD/VER::2.0
OPER/SYS:
KEYWORDS:kbenv kbtool kbui kbsms200 kbsms200bug

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Systems Management Server version 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use Systems Management Server (SMS) to remotely control Microsoft
Windows XP-based computers, you may not be able to locate the Input Method
Editor (IME), or the Language bar, in the SMS Remote Control session.

WORKAROUND
==========

To work around this problem, attach the IME bar to the task bar before you start
the SMS Remote Control session.

STATUS
======

Microsoft has confirmed this to be a problem in Systems Management Server
version 2.0.
