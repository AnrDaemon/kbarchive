DOCUMENT:Q178857  10-SEP-1999  [exchange]
TITLE   :XADM: Setup.exe Not Found On Premier Select CD-ROM
PRODUCT :Microsoft Exchange
PROD/VER:winnt:5.5
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Setup.exe file is not found on the Premier Select CD-ROM.

CAUSE
=====

Select CD-ROMs include both the Standard and Enterprise versions of Exchange
Server.

WORKAROUND
==========

To avoid including two Setup.exe files, you must run the version specific
executables.

To install or upgrade to Exchange Server 5.5 Standard Edition, use the Srvmin.exe
file. To install or upgrade to Exchange Server 5.5 Enterprise Edition, use the
Srvmax.exe file.
