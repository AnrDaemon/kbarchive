DOCUMENT:Q178093  20-MAR-1999  [exchange]
TITLE   :XADM: Attachments with Long Names Crash SNADS or PROFS Connector
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:5.5
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you send a message from an Exchange Server computer to a SNADS or PROFS
e-mail system that has an attachment with a name consisting of more than eight
characters, the SNADS or PROFS connector on the Exchange Server computer may
crash.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server
version 5.5. This problem has been corrected in the latest U.S. Service Pack for
Microsoft Exchange Server version 5.5. For information on obtaining the Service
Pack, query on the following word in the Microsoft Knowledge Base (without the
spaces):

   S E R V P A C K
