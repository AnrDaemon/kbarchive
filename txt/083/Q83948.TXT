TITLE   :Lexis 2000 Version 1.76 Is Required for Windows 3.1
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

If you run Lexis 2000 for Windows, a legal-research program, with the Microsoft
Windows operating system version 3.1, you may receive a general protection (GP)
fault message. Running version 1.75 or an earlier of this program causes this
error. Lexis 2000 for Windows version 1.76 corrects this problem.

MORE INFORMATION
================

According to MeadData Central, versions prior to 1.76 are not compatible with
the Windows 3.1. To obtain an upgrade, contact MeadData Central at (800)
346-9759.

Lexis 2000 for Windows is manufactured by MeadData Central, vendor independent of
Microsoft; we make no warranty, implied or otherwise, regarding this product's
performance or reliability.
