DOCUMENT:Q266747  11-JAN-2001  [vbwin]
TITLE   :PRB: Toolbox or Menu Items Are Missing in the Visual Basic IDE
PRODUCT :Microsoft Visual Basic for Windows
PROD/VER:WINDOWS:5.0,6.0
OPER/SYS:
KEYWORDS:kbide kbMenu kbVBp kbVBp500 kbVBp600 kbGrpDSVB kbDSupport

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Visual Basic Professional Edition for Windows, versions 5.0, 6.0 
 - Microsoft Visual Basic Enterprise Edition for Windows, versions 5.0, 6.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you open a Visual Basic Project, either you are not able to see the toolbox
or several items are missing from the menu options.

CAUSE
=====

One possible cause of this problem is a Registry corruption.

RESOLUTION
==========

To reset the options to their default settings, follow these steps:

1. Right-click the main menu bar and select Customize. The Customize dialog box
   opens.

2. Select the Toolbars tab and click the Reset button.

3. Keep the Customize dialog box open, right-click every menu option you want to
   reset on the main menu bar (such as File, Edit, and so forth), and then
   select the Reset option.

4. Close the Customize dialog box.
