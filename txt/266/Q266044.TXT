The information in this article applies to:

 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SUMMARY
=======

You may want to make changes to the Exchange Server 5.5 environment to increase
speed and reduce the amount of time required for directory and public folder
replication. A white paper is available that describes what each of the registry
parameters does to improve replication time and performance.

MORE INFORMATION
================

The "Optimization for Directory and Public Folder Replication" white paper
presents changes that you can make to the Exchange Server 5.5 environment to
improve speed and reduce the amount of time required for directory and public
folder replication. To view the white paper, see the following Microsoft Web
site:

   http://support.microsoft.com/support/exchange/content/whitepapers/foldrepl.doc
   (http://support.microsoft.com/support/exchange/content/whitepapers/foldrepl.doc)
