TITLE   :ASD: Add a Point After Feature
PRODUCT :Microsoft Home Games

OPER/SYS:
KEYWORDS:

======================================================================
SUMMARY
=======

The Add A Point After feature in the Static Scenery Designer will insert a new
point after the currently selected point.

To insert a new point between existing points, do the following:

1. Press TAB to move to the point after which you want to insert a new point.

2. Press P to select that point.

3. Press INS to add a point after the selected point.

4. Use the arrow keys to move the new point (press Keypad 5 to stop it's
   movement).

5. Press INS to set the new point.

6. Press TAB to break from adding new insertion points.
