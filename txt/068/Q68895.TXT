TITLE   :Adobe Type Manager 1.0 Does Not Print in Color
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

Adobe Type Manager 1.0 for Windows 3.0 does not print in color to the
Hewlett-Packard (HP) PaintJet family of printers. When this utility is
installed, text that normally would print in color prints in black, regardless
of the text color.
