TITLE   :Unrecoverable Application Error with Z-Soft Soft Type Fonts
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a,3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

You may receive Unrecoverable Application Error (UAE) or General Protection (GP)
fault error messages when you use large font sizes with Z-Soft's Soft Type Fonts
running under Microsoft Windows.

These error messages appear in Windows applications such as Word for Windows,
Write, and Excel.

CAUSE
=====

This problem can occur if you are using Soft Type version 1.00 fonts.

WORKAROUND
==========

Updating to Z-Soft's Soft Type Fonts version 1.6 may correct the problem.

MORE INFORMATION
================

For more information, contact Z-Soft product support.

Soft Type Fonts is manufactured by Z-Soft, a vendor independent of Microsoft; we
make no warranty, implied or otherwise, regarding this product's performance or
reliability.
