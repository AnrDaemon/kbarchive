TITLE   :IOLINE Plotter: No Windows 3.0 Compatible Driver
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

There are no Microsoft Windows version 3.0 printer drivers available for the
IOLINE 3700 or 4000 Plotter. IOLINE suggests using the HP-7585B driver. IOLINE
supposedly is working on a Windows 3.0 compatible driver for these plotters.

MORE INFORMATION
================

For more information, contact IOLINE Corporation technical support.

Additional query words: 3.00 win30 3.00a
