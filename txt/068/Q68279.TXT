TITLE   :Expanding the File Manager Directory Tree Using the SHIFT Key
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

To expand the Directory Tree in File Manager with the SHIFT key, do one of the
following:

 - Press SHIFT as you select the drive that you want to expand.

   -or-

 - Press SHIFT+ALT and the letter of the drive that you want to expand.

NOTE: Pressing CTRL+ASTERISK(*) also expands all branches in a Directory Tree.

To cancel the Expand Tree function, press the ESC key while File Manager is
expanding the Directory Tree.

MORE INFORMATION
================

For more information on expanding Directory Levels in File Manager, refer to
pages 109-110 of the "Microsoft Windows User's Guide."
