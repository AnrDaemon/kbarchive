DOCUMENT:Q68139  08-OCT-1999  [win95x]
TITLE   :Cannot Create Macro with Macro Recorder to End Windows Session
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a,3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

The Microsoft Windows Recorder program cannot be used to create a macro
that can completely end a Windows session. A macro cannot be recorded to
continue past the "This will end your Windows session" dialog box message.
Once this message is reached, you must to manually choose OK or press ENTER
to fully exit Windows.

Additional query words: 3.00 3.0 3.1 3.10 3.11
