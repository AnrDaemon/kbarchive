TITLE   :Unrecoverable Application Error in Aldus Pagemaker 3.01
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SYMPTOMS
========

You may receive an "Unrecoverable Application Error" (without the quotation
marks) (UAE) in Aldus Pagemaker 3.01 when you open a document that was created
with an earlier version of Pagemaker.

WORKAROUND
==========

According to Aldus Corporation, the problem can be corrected as follows:

1. Start Microsoft Windows version 3.0 in real mode.

2. Load PageMaker.

3. Open the document.

4. Recompose the document and save it.

If this does not solve the problem, contact Aldus Technical Support.
