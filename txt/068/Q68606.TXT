TITLE   :Mac Wkst: System Error Occurs When Closing Address Book Window
PRODUCT :Microsoft Mail For Appletalk Networks
PROD/VER:WINDOWS:3.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for AppleTalk Networks, version 3.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

In Microsoft Mail, a System error might occur when you close the Address Book
window if the WindowShade INIT is located in the System Folder. Deinstalling the
WindowShade INIT will prevent this problem from occurring.

RESOLUTION
==========

To deinstall WindowShade, do the following:

1. Remove WindowShade from the System Folder.

2. Restart the Apple Macintosh.

STATUS
======

Microsoft has confirmed this to be a problem in version 3.0 of AppleTalk Mail.
We are researching this problem and will post new information here in the
Microsoft Knowledge Base as it becomes available.
