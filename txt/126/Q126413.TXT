TITLE   :Empty Program Groups Created on 16-bit Windows Clients
PRODUCT :Microsoft Systems Management Server
PROD/VER:winnt:1.1
OPER/SYS:
KEYWORDS:kbnetwork kbPGC smspgc

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Systems Management Server version 1.1 
-------------------------------------------------------------------------------


SYMPTOMS
========

Empty program groups for 32-bit distributed shared packages are created on
16-bit Windows client. Program groups are generated on all clients, even if the
application does not run on that client operating system.

STATUS
======

Microsoft has confirmed this to be a problem in Systems Management Server
version 1.1. This problem has been corrected in Systems Management Server
version 1.2.
