TITLE   :Media Player Command-Line Switches Not Documented

-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
 - Microsoft Windows for Workgroups 
 - Microsoft Windows NT Workstation version 3.5 
 - Microsoft Windows NT Server version 3.5 
-------------------------------------------------------------------------------

The Windows, Windows for Workgroups, and Windows NT documentation does not
document the following command-line switches Media Player (both for the 16-
bit [mplayer.exe] and 32-bit [mplay32.exe] versions):

   /play        play Media Player file immediately
   /close       close after playing Media Player file (used with /play)
   /embedding   run as an OLE server
   <filename>   file or device to open
