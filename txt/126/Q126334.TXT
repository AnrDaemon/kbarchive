TITLE   :Turbo Tax Fails to Print Under Windows NT
PRODUCT :Microsoft Windows NT
PROD/VER:3.5 4.0
OPER/SYS:
KEYWORDS:kb3rdparty kbprint kbPrinting

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation versions 3.5, 4.0 
 - Microsoft Windows NT Server versions 3.5, 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you try to print from Turbo Tax for Windows under Windows NT, either the
following error message appears

   The exception privileged Instruction (0xc0000096] occurred in the application
   at location [0x01fd6756).

or a DR. Watson log error message is created from NTVDM.EXE. The numbers in the
error message may vary.

This problem occurs with all printer drivers, and occurs when you print over a
network, locally, or to a file.

CAUSE
=====

According to Intuit Technical support, they do not support Turbo Tax on Windows
NT.
