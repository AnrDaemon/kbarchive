TITLE   :PC WSPlus: Schedule+ Does Not Remember Print Margin Units
PRODUCT :Microsoft Schedule+ for Windows
PROD/VER:WINDOWS:1.0,1.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Schedule+ for Windows, versions 1.0, 1.0a 
-------------------------------------------------------------------------------

SYMPTOMS
========

In Microsoft Schedule+ for Windows, the units of measure are not retained when
the paper format is set to Junior.

STATUS
======

Microsoft has confirmed this to be a problem with Microsoft Schedule+ versions
1.0 and 1.0a. We are researching this problem and will post new information here
in the Microsoft Knowledge Base as it becomes available.


Additional query words: schedule plus 1.00 schedplus print margin Windows NT
