TITLE   :PC Win: Err Msg: This Name Has Already Been Assigned...
PRODUCT :Microsoft Mail For PC Networks
PROD/VER:WINDOWS:3.2
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for Windows, version 3.2 
-------------------------------------------------------------------------------

SYMPTOMS
========

When changing the name of a shared folder in version 3.2 of Microsoft Mail for
Windows to the name of an existing shared folder, you may receive the following
message:

   This name has already been assigned to a folder. Type a unique name.

If you then try to enter a valid unique name, you may receive the following
message:

   The folder properties could not be changed.

If you try to rename the folder again, you may receive the following message:

   There is not enough memory or system resources to complete this action. Try
   closing some windows and performing the action again.

STATUS
======

Microsoft has confirmed this to be a problem in version 3.2 of Microsoft Mail
for Windows. We are researching this problem and will post new information here
in the Microsoft Knowledge Base as it becomes available.

================
