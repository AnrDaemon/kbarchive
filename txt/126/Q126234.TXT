TITLE   :Cannot Save or Print Buildings in Modeling Wright
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft The Ultimate Frank Lloyd Wright for Windows, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Modeling Wright module in The Ultimate Frank Lloyd Wright allows you to use
your own imagination to design a building. The options available are Conceive,
Articulate, and Visualize. The Ultimate Frank Lloyd Wright program does not
provide any functions for saving or printing the building.

RESOLUTION
==========

Copy the image to the Microsoft Windows Clipboard by pressing ALT+PRINT SCREEN.

NOTE: The Microsoft Clipboard is a temporary holding location for the image. For
more information about working with the Microsoft Windows Clipboard, see your
Windows printed documentation or online Help.

MORE INFORMATION
================

The Clipboard image can be pasted into a program that allows you to modify,
save, or print the image. Microsoft Windows Paintbrush can be used to import and
print the image. For more information about using Paintbrush, see your Windows
printed documentation or online Help.
