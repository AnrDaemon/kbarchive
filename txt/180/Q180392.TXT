DOCUMENT:Q180392  02-APR-2000  [sna]
TITLE   :Uninstall Button Is Not Displayed On Windows NT 3.51
PRODUCT :Microsoft SNA Server
PROD/VER:WINDOWS:3.0,3.0 SP1,3.0 SP2
OPER/SYS:
KEYWORDS:kbnetwork

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft SNA Server, versions 3.0, 3.0 SP1, 3.0 SP2 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Uninstall button is not displayed during SNA Setup on Windows NT version
3.51. When you click the screen where the button should be, Uninstall continues.

CAUSE
=====

The file Uninstal.exe is not working correctly.

STATUS
======

Microsoft has confirmed this to be a problem in SNA Server versions 3.0. 3.0
Service Pack 1 (SP1), and 3.0 SP2.

This problem was corrected in the latest SNA Server version 3.0 U.S. Service
Pack. For information on obtaining this Service Pack, query on the following
word in the Microsoft Knowledge Base (without the spaces):

   S E R V P A C K
