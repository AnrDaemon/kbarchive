TITLE   :Windows 3.X Client Environment Variable BackupRemote Is Invalid
PRODUCT :Microsoft SNA Server
PROD/VER:WINDOWS:3.0,3.0 SP1,3.0 SP2,4.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft SNA Server, versions 3.0, 3.0 SP1, 3.0 SP2, 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The SNA Server help file inadvertently lists BackupSponsors as one of the SNA
Server Windows 3.X client environment variables. The correct name for the
variable that provides this functionality is BackupRemote.

MORE INFORMATION
================

For the SNA Server Windows 3.X client, use the BackupRemote variable to specify
a backup subdomain or a backup sponsor server.
