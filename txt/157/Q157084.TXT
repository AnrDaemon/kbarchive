SUMMARY
=======

This article lists which text converters are installed during a typical setup of
Word 97 or Office 97, which converters are not installed, and which converters
are native (built-in) to Word 97.

MORE INFORMATION
================

Text Converters Installed During a Typical Setup
------------------------------------------------

The following text converters are installed during a typical setup:

 - Microsoft Excel 2.x-95 and Microsoft Excel 97 (import)

 - Microsoft Word 6.0/95 for Windows/Macintosh (import)

 - Microsoft Word 6.0/95 for Windows/Macintosh (export)

 - Word 97 for Windows/Macintosh (import)

 - WordPerfect 6.x for MS-DOS and Windows (import)

 - WordPerfect 5.x for MS-DOS and Windows (import and export)

 - File Recovery (import)

 - Text with Layout (import/export)

Text Converters Not Installed During Typical Setup
--------------------------------------------------

The following converters are not installed during a typical setup. To install
these converters, rerun the setup program, click Add/Remove, and then select the
converters you want.

 - Hypertext Markup Language (HTML) (import/export) - Windows & Macintosh

 - Lotus 1-2-3, versions 2.x, 3.x, 4.0 (import) - Windows

 - Microsoft Word 4.0-5.1 for the Macintosh (export) - Windows

 - Microsoft Word 2.0 for Windows (export) - Windows & Macintosh

 - Microsoft Works 3.0 for Windows (import/export) - Windows

 - Microsoft Works 4.0 for Windows (import/export) - Windows

 - Converters for use with Lotus Notes (allows you to read Microsoft Word and
   Excel files in Lotus Notes 3.x and 4.x) - Windows

 - Microsoft FoxPro 2.x/Borland dBase III, III+, and IV - Macintosh

Native (Built-in) Text Converters
---------------------------------

Word 97 can directly import the following text formats:

 - MS-DOS Text
 - MS-DOS Text with Line Breaks
 - Rich Text Format (RTF)
 - Text Only
 - Text with Line Breaks
 - Unicode Text
 - Word 4.x-5.1 for the Macintosh
 - Word 1.0 for Windows
 - Word 2.0 for Windows
 - Word 6.0-95 for Windows

For additional information, click the article number below to view the article in
the Microsoft Knowledge Base:

   Q212265 WD: Additional Text Converters and Image Filters Available in
   Microsoft Office Converter Pack
