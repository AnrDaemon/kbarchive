TITLE   :WD97: Browse For Folder Dialog Box Behind Conversion Wizard
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:kbtool kbualink97

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

In the Conversion Wizard, when you click Conversion Wizard in the Browse For
Folder dialog box, the Browse For Folder dialog box moves behind the Conversion
Wizard.

If you click Browse in the Conversion Wizard, nothing happens.

If you click Cancel in the Conversion Wizard, the wizard is closed, but the
Browse For Folder dialog box remains open.


WORKAROUND
==========

To work around this problem and reactivate the Browse For Folder dialog box,
drag the Conversion Wizard out of the way, and then click the Browse For Folder
dialog box.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows.
