TITLE   :WD97: Headings Inside a Table Do Not Appear in the Document Map
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:kbualink97 kbtable

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

Text formatted in a heading style or multilevel outline entry that occurs within
a table is not displayed in the Document Map.

MORE INFORMATION
================

The Document Map is a vertical pane that displays an outline of the document's
headings. You can use the document map to quickly move through a long document
or keep track of your location within a document. To activate this option, click
Document Map on the View menu.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows.
