TITLE   :WD97: Excel 97 Chart Object Becomes Picture Converting to Word 7
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:kbinterop kbole kbdta word97

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

In Word 7.0, when you open a Word 97 document that contains a Microsoft Excel
chart object, and you double-click the Excel chart object, the Word Picture
Editor is opened instead of Excel.

WORKAROUND
==========

To open a Word 97 document containing an Excel chart object in Word 7.0, and to
be able to double-click the Excel chart object to open Excel, install the
Microsoft Word 97-2000 Import Converter. For additional information, please see
the following article in the Microsoft Knowledge Base:

   

   Q162214 WD: How to Obtain the Word 97-2000 Import Converter

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.
