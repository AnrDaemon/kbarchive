The information in this article applies to:

 - Microsoft Mail for AppleTalk Networks, versions 3.0, 3.1 
-------------------------------------------------------------------------------

SUMMARY
=======

The AppleLink gateway for Microsoft Mail is a free gateway supported by
Microsoft Product Support Services. It can be found on AppleLink in the
following location:

  Third Parties:Microsoft:Mail:Gateways:AppleLink GW for MS Mail 3.0

The AppleLink GW package can be expanded using the AppleLink application and
contains the following files:

   AppleLink GW Documentation
   AppleLink GW Readme
   MS Mail GW
   AppleLink Comm.
   AppleLink Gateway
   Gateway Installer
   GWS Monitor
   USA Modem.EF3.CCL

The gateway can be found only at this location and is not useful without an
AppleLink account. To obtain an AppleLink startup package, contact the Apple
Programmer's and Developer's Association (APDA) at (800) 282-2732.

The AppleLink Gateway is compatible with Microsoft Mail version 3.1.
