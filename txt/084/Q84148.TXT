TITLE   :USER.EXE in System Directory Has Date of Installation
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
3.10 3.11

WINDOWS

kbsetup

-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

If you are troubleshooting and searching for files in the Windows and Windows
SYSTEM subdirectory using file dates, you may notice that USER.EXE does not have
a date of 3/10/92. USER.EXE has the date of the installation (setup) because the
customer name or company information is recorded in this file. The file is not
corrupted. The size is always 264016. The name input cannot exceed 30
characters.

Additional query words: 3.10 3.11
