TITLE   :WaveEdit Version 1.0b Required Under Windows 3.1
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

WaveEdit version 1.0 supplied with the Multimedia Development Kit (MDK) requires
an update, to version 1.0b, for proper functionality under Windows 3.1.

MORE INFORMATION
================

When using WaveEdit 1.0, you may receive the following error message

   Cannot open <filename> for playing and recording

when loading a .WAV file in WaveEdit. This problem has been corrected in WaveEdit
1.0b.

To obtain WaveEdit 1.0b (part no. 191-050-050), contact Microsoft End User Sales
at (800) 426-9400.
