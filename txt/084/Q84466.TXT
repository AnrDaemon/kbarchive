TITLE   :Express Publisher Not Printing Graphics to EPS File
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you print to an encapsulated PostScript (EPS) file from PowerUp Software's
Express Publisher for Windows under Microsoft Windows version 3.1, some graphics
may not print. Mixed text and graphics should print correctly, but graphics on a
page without any text may print as a blank page.


MORE INFORMATION
================

Express Publisher is manufactured by PowerUp Software, a vendor independent of
Microsoft; we make no warranty, implied or otherwise, regarding this product's
performance or reliability.
