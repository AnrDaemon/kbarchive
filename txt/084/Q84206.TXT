TITLE   :CAPS LOCK Does Not Function Correctly on Toshiba T2000 SXE
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

When you run the Microsoft Windows operating system version 3.1 on a Toshiba
T2000 SXE, the CAPS LOCK key works but does not light up when pressed.

MORE INFORMATION
================

Toshiba has confirmed this to be an incompatibility problem between the keyboard
BIOS and Windows 3.1.

For more information, contact Toshiba laptop support.

The Toshiba product included here is manufactured by a vendor independent of
Microsoft; we make no warranty, implied or otherwise, regarding this product's
performance or reliability.
