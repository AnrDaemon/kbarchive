TITLE   :Failed Installation When Using Full Page Genius Video Monitor
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

When you install Microsoft Windows operating system version 3.1 on a system with
a Genius Plus Model 420 Full Page Color video display adapter and monitor, it
may be necessary to obtain updated video drivers from Ran-Ger Technologies.

If you receive the following error messages when installing Windows 3.1 for EGA
or VGA resolution, contact Ran-Ger for a set of updated, Windows 3.1 compatible
video drivers:

   Standard Mode: Fault in MS-DOS Extender

   -or-

   Error Initializing VGA Device

For additional information, please contact Ran-Ger Technologies Incorporated, as
they now support the Genius video display card and monitor.

For more information, query on the following words in the Microsoft Knowledge
Base:

   genius and ran-ger
