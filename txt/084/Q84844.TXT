DOCUMENT:Q84844  12-DEC-1999  [win16sdk]
TITLE   :PRB: Borland Turbo Debugger Cannot Load Windebug.dll
PRODUCT :Microsoft Windows Software Development Kit
PROD/VER:WINDOWS:3.1
OPER/SYS:
KEYWORDS:kbfile

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows Software Development Kit (SDK) 3.1 
-------------------------------------------------------------------------------

SYMPTOMS
========

An attempt to run the Borland Turbo Debugger under Windows 3.1 generates a
message stating that the Windebug.dll file is not found.

CAUSE
=====

The Borland Turbo Debugger requires the Windebug.dll file to work with the
Windows environment.

RESOLUTION
==========

WINDBG is a file in the Microsoft Download Center that contains the
Windebug.dll. WINDBG is for registered owners of Borland language products that
require the file to run the Borland Turbo Debugger under Windows 3.1. WINDBG may
not be redistributed in any way. The following file is available for download
from the Microsoft Download Center:

   Windbg.exe
   (http://download.microsoft.com/download/vb30/Sample43/1/W9XNT4/EN-US/Windbg.exe)

For additional information about how to download Microsoft Support files, click
the article number below to view the article in the Microsoft Knowledge Base:

   Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft used the most current virus detection software available on the date of
posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.
