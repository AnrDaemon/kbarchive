TITLE   :Gerber Software Requires Upgrade for Windows 3.1
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

When using Gerber Scientific's Composer version 3.0 with Microsoft Windows
operating system version 3.1, you may receive the following general protection
(GP) fault:

   GPF in COMPOSER.EXE at 003A:09BC

You may also be missing the toolbox display.

Gerber has an upgrade to correct this problem. For more information on this
upgrade, contact Gerber at (800) 336-1698 and ask for the Graphic Advantage
group.

MORE INFORMATION
================

Gerber Scientific sells dedicated computer systems with Windows 3.0 for
scientific plotting tasks.
