======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.1, 3.11 
 - Microsoft Windows for Workgroups versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

There is a documentation error in the user's guides included with Windows for
Workgroups 3.1 and with Windows 3.1. The error occurs in the Windows 3.1 "User's
Guide" with the part number 52207 and the Windows for Workgroups 3.1 "User's
Guide" with the part number 32404. The part number is found on the outside back
cover of these manuals.

NOTE: This does not apply to the Windows 3.1 "User's Guide" with the part number
21669.

The error is on page 94 of the Windows for Workgroups manual and page 133 of the
Windows 3.1 manual. These pages reference an online help subject that does not
exist. "Using keystrokes to create selected characters" is included in Help.

Additional query words: 3.00 3.0 3.10
