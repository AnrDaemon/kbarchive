TITLE   :DOC: Commands Missing From the Index List in Help
PRODUCT :Microsoft FoxPro
PROD/VER:WINDOWS:6.0
OPER/SYS:
KEYWORDS:kbdocerr

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Visual FoxPro for Windows, version 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The following is a list of commands that are missing from the Visual FoxPro 6.0
Help index:

   DEFINE MENU
   DEFINE PAD
   DEFINE POPUP
   DEFINE WINDOW

MORE INFORMATION
================

The commands listed above are missing in the Index list when the commands are
typed into the "Type in the keyword to find:" textbox in the Index tab. However,
the commands listed above can still be found in the Help either by looking at
the A to Z Language Reference or performing a search on above commands in the
Search tab.

Microsoft is researching this behavior and will post new information here in the
Microsoft Knowledge Base as it becomes available.
