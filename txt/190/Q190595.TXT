TITLE   :INFO: Visual Modeler 2.0: Where Is the ORDERSYS Sample?
PRODUCT :Microsoft Visual Basic for Windows
PROD/VER:WINDOWS:6.0; winnt:6.0
OPER/SYS:
KEYWORDS:kbVBp600 kbVC600 kbVS600 kbGrpDSVBDB

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Visual Basic Enterprise Edition for Windows, version 6.0 
 - Microsoft Visual C++, 32-bit Enterprise Edition, version 6.0 
 - Microsoft Visual C++, 32-bit Professional Edition, version 6.0 
 - Microsoft Visual C++, 32-bit Learning Edition, version 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Visual Modeler 2.0 Order System sample, referenced in Online Help, is not
installed from the MSDN setup. However, you can navigate in MSDN to the Visual
Modeler section (type "ORDERSYS" as search keyword), and select the links
associated with the sample. This will copy the sample to the hard drive.

You can also manually copy the ORDERSYS sample from the MSDN CD-ROM. It is
located under the folder:

   \Samples\VM\VMOrder\

In addition, the following sample models are provided:

   \Samples\VM\VMAdo\
   \Samples\VM\VMEnroll\
   \Samples\VM\VMMTS\
   \Samples\VM\VMRdo\
