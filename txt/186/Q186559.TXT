TITLE   :Common Question: Will Terminal Server Run on Windows NT 4.0?
PRODUCT :Microsoft Windows NT
PROD/VER:WinNT:4.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0, Terminal Server Edition 
-------------------------------------------------------------------------------

SUMMARY
=======

Windows Terminal Server 4.0 is a fully self-contained, stand-alone operating
system based on Microsoft Windows NT Server 4.0 with Service Pack 3. It is not a
service and cannot be added to existing servers running Windows NT Server.

Do not apply system updates such as Service Packs or Hotfixes written by
Microsoft for Windows NT Server 4.0 to a computer running Terminal Server. Apply
only updates written specifically for Terminal Server.
