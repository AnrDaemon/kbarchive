TITLE   :Mac Bookshelf 96-97 Err Msg: Unable to Start Because of an Error
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:MACINTOSH:
OPER/SYS:
KEYWORDS:kbenv kberrmsg kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Bookshelf 1996-97 for Macintosh 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to start Bookshelf 96-97 for Macintosh, you may receive the
following error message:

   Unable to start because of an error.

CAUSE
=====

This behavior can occur if a memory compression extension is running on the
computer.

RESOLUTION
==========

To resolve this issue, disable any memory compression extensions that are
running. To do so, follow these steps:

1. On the Apple menu, point to Control Panels, and then click Extensions
   Manager.

2. Click the check box for the memory compression extension to clear it.

3. Quit Extensions Manager.

4. Restart the computer.
