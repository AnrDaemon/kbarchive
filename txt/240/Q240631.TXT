TITLE   :World Atlas Err Msg: You Must Have NT SUR 4.0 GOLD to Install...
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:WINDOWS:
OPER/SYS:
KEYWORDS:kbenv kberrmsg kbsetup kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta Interactive World Atlas 2000 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you install Microsoft Encarta World Atlas 2000, you may receive the
following error message:

   You must have NT SUR 4.0 GOLD to install. Please click OK to cancel
   installation.

When you click OK, Encarta World Atlas 2000 Setup continues to run.

CAUSE
=====

This behavior can occur if you install Encarta World Atlas 2000 on a Microsoft
Windows 2000-based computer.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Encarta World Atlas
2000.

MORE INFORMATION
================

When you complete Encarta World Atlas 2000 Setup, you are able to run the
program without any further problems.
