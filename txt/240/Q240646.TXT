TITLE   :World Atlas: Error Message When You Quit Program and Browser
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:WINDOWS:
OPER/SYS:
KEYWORDS:kberrmsg kbinterop kbui kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta Interactive World Atlas 2000 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you click a Web link in Microsoft Encarta World Atlas 2000 and then quit both
Encarta World Atlas 2000 and your Web browser, you may receive the following
error message:

   Encarta Interactive World Atlas is unable to continue running. The problem
   may be the result of low system resources such as memory or disk space. Try
   closing other running applications or freeing up disk space before restarting
   Encarta Interactive World Atlas.

CAUSE
=====

This behavior can occur if both of the following conditions are true:

 - The "Browse Web Links within Encarta Interactive World Atlas" check box
   (located in the Settings dialog box in Encarta World Atlas 2000) is cleared.

 - You quit both Encarta World Atlas 2000 and your Web browser while the Web
   site is still being opened.

RESOLUTION
==========

To prevent this issue from occurring, use both of the following methods:

 - Click Settings on the Options menu in Encarta World Atlas 2000, and then
   click to select the "Browse Web Links within Encarta Interactive World Atlas"
   check box.

 - Do not quit Encarta World Atlas 2000 and your Web browser while a Web page is
   being opened.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Encarta World Atlas
2000.
