TITLE   :Flight Simulator: Computer Hangs when Using Virtual Pilot Pro
PRODUCT :Microsoft Home Games
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kb3rdparty kbhw kbHardware

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Flight Simulator for Windows 95, version 1.0 
 - Microsoft Flight Simulator 98 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use Microsoft Flight Simulator, your computer may seem to stop
responding (hang) or slow down substantially when you use the CH Products
Virtual Pilot Pro gaming device.

CAUSE
=====

You may be using an outdated or incompatible version of the Virtual Pilot Pro
driver for Windows 95.

RESOLUTION
==========

To resolve this issue, download and install the most current version of the
Virtual Pilot Pro driver for Windows 95. You can obtain this driver from the
following CH Products Web site:

   http://www.chproducts.com/FAQ4WEB/CHF00051.html
