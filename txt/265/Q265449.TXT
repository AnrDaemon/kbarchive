TITLE   :WD97: Word Document Statistics Different in Windows Explorer
PRODUCT :Word 97 for Windows
PROD/VER:WINDOWS:97
OPER/SYS:
KEYWORDS:kbdta word8 word97

======================================================================

-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you view your document statistics from Microsoft Windows Explorer and then
view the statistics for the same document in Word, the statistics are
different.


STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.

MORE INFORMATION
================

In Microsoft Word:

To view your document statistics in Microsoft Word, follow these steps:

1. Open your Word document.

2. On the File menu, click Properties.

3. Click the Statistics tab.

In Microsoft Windows Explorer:

To view your document statistics in Microsoft Windows Explorer, follow these
steps:

NOTE: Make sure your document is closed in Microsoft Word.

Microsoft Windows 95, Microsoft Windows 98, and Microsoft Windows NT 4.0:

1. Right-click the Windows Start button.

2. On the shortcut menu that appears, click Explore.

3. In Windows Explorer, change to the directory that contains your Word
   document.

4. Right-click your Word document, and then click Properties on the shortcut
   menu that appears.

5. Click the Statistics tab.

Microsoft Windows 2000:

1. Right-click the Windows Start button.

2. On the shortcut menu that appears, click Explore.

3. In Windows Explorer, change to the directory that contains your Word
   document.

4. Right-click your Word document, and then click Properties on the shortcut
   menu that appears.

5. Click the Summary tab.
