TITLE   :Err Msg: Your Computer Is Not Properly Configured...
PRODUCT :Microsoft Automap
PROD/VER::1.0
OPER/SYS:
KEYWORDS:kberrmsg kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft MapPoint 2001 
 - Microsoft MapPoint 2002 
 - Microsoft Streets and Trips 2001 
 - Microsoft Streets & Trips 2002, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use Microsoft MapPoint or Microsoft Streets and Trips, you may receive
one of the following error messages:

    - Your computer is not properly configured to connect to the Internet. You must
   have an Internet connection in order to download construction updates.

   -or-

    - The server name or page could not be found.

CAUSE
=====

This issue can occur if America Online (AOL) is your Internet service provider
(ISP) and you attempt to perform one of the following actions:

 - Connect to the Internet.

 - Update the construction information.

This symptom has also been known to occur if your ISP provider is Adelphia.com
(cable modem connection).

RESOLUTION
==========

To resolve this issue, download and install the most current Construction Update
file from the Microsoft Download Center.

The following file is available for download from the Microsoft Download Center:

   Construction#######.dat
   (http://www.microsoft.com/downloads/release.asp?ReleaseID=17725)

where <#######> is a series of numbers.

Release Date: Jan-26-2000 (Subject to change)

For additional information about how to download Microsoft Support files, click
the following article number to view the article in the Microsoft Knowledge
Base:

   Q119591 How to Obtain Microsoft Support Files from Online Services

Microsoft scanned this file for viruses. Microsoft used the most current
virus-detection software that was available on the date that the file was
posted. The file is stored on secure servers that prevent any unauthorized
changes to the file.

If you are using MapPoint 2001, save the Construction Update file to the
following folder

   <drive>:\Program Files\Microsoft MapPoint\Data

where <drive> is the drive letter of the hard disk on which MapPoint 2001
is installed.

If you are using Streets and Trips 2001, save the Construction Update file to the
following folder:

   <drive>:\Program Files\Microsoft Streets & Trips\Data

where <drive> is the drive letter of the hard disk on which Streets and
Trips 2001 is installed.

If you are using either MapPoint 2002 or Streets & Trips 2002, download the
Construction file from the following site:

   http://download.microsoft.com/download/mappoint2002/update/2002/w98nt42kme/en-us/ConstructionA1031413.dat

If you are using MapPoint 2002, save the construction file to the following
folder (assuming the default location):

   <drive>:\Program Files\Microsoft MapPoint\Data

where <drive> is the drive letter of the hard drive on which MapPoint 2002
is installed. If you are using Streets & Trips 2002, save the construction
file to the following folder (assuming the default installation):

   <drive>:\Program Files\Microsoft Streets & Trips\Data

where <drive> is the drive letter of the hard drive on which Streets &
Trips 2002 is installed.
