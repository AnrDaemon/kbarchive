TITLE   :Cinemania/Music Central 97 Err Msg: Win32s - Error
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER::1997 edition; WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Cinemania for Windows 1997 edition 
 - Microsoft Music Central for Windows 1997 edition 
 - the operating system: Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you try to run Microsoft Cinemania or Microsoft Music Central, you receive
the following error message:

   Win32s - Error.

   The procedure entry point "VerLanguageNameA" could not be located in the
   Dynamic Link Library "VERSION.dll".

After you click OK, you receive the message:

   Unexpected error: 21.

CAUSE
=====

You are attempting to run Cinemania 97 or Music Central 97 under Microsoft
Windows or Windows for Workgroups version 3.1x.

The programs listed at the top of this article are not designed to run on
Microsoft Windows version 3.1x.

MORE INFORMATION
================

The system requirements listed on the product boxes specify that the programs
listed at the top of this article need Microsoft Windows 95 or later operating
system or Windows NT Workstation operating system version 3.51 or later.
