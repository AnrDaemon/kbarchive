TITLE   :Encarta 97 Fails to Start or Quits with Encarta 96 Running
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:WINDOWS:3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta 97 Encyclopedia for Windows 
 - Microsoft Encarta Encyclopedia 97 Deluxe for Windows 
 - the operating system: Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start Encarta Encyclopedia in Windows 3.x, one of the following may
occur:

 - Encarta 1997 displays the opening screens and then closes.

   -or-

 - Encarta 1997 starts, but when you attempt to use the program with the mouse
   or keyboard, Encarta closes.

CAUSE
=====

Encarta Encyclopedia for Windows, 1996 edition, is also running on your
computer.

RESOLUTION
==========

Close Encarta Encyclopedia for Windows 1996 before you start Encarta
Encyclopedia for Windows 1997.

MORE INFORMATION
================

Microsoft has confirmed this to be a problem in the 1997 edition of Encarta
Encyclopedia for Windows. We are researching this problem and will post new
information here in the Microsoft Knowledge Base as it becomes available.
