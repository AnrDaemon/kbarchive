TITLE   :NDS Printer Comments Not Available with MS Service for NDS
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER::2,2.1
OPER/SYS:
KEYWORDS:osr2 win95

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
 - Microsoft Windows 95 OEM Service Release, versions 2, 2.1 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use the Microsoft Service for NetWare Directory Services (MSNDS),
printer comments may not be displayed for NetWare Directory Services (NDS)
printers on a Novell NetWare version 4.1 server.

CAUSE
=====

This functionality is not included in MSNDS.

RESOLUTION
==========

Contact Novell for information regarding using printer comments a NetWare
version 4.1 server.
