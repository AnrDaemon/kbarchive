TITLE   :FP: Error, Broken Image Inserting WordPerfect Graphic File
PRODUCT :Word Front Page
PROD/VER::; WINDOWS:1.1
OPER/SYS:
KEYWORDS:kb3rdparty kberrmsg kbgraphic kbdta

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft FrontPage 97 for Windows with Bonus Pack 
 - Microsoft FrontPage for Windows 1.1 
-------------------------------------------------------------------------------

For a Microsoft FrontPage 98 version of this article, see Q194250.

SYMPTOMS
========

In FrontPage Editor, version 1.1, when you insert a WordPerfect graphic file
(.wpg) you receive the following error message:

   Could not load <path to the file>.

In FrontPage Editor 97, when you insert a WordPerfect graphic file, the image
appears as a broken icon.

CAUSE
=====

The .wpg graphics converter used by FrontPage is unable to convert all .wpg
files.

WORKAROUND
==========

To work around this problem, use an image editing software program (such as
Microsoft Imager) to convert the .wpg file to GIF or JPEG format.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.
