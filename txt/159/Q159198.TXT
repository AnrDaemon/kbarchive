DOCUMENT:Q159198  10-APR-1999  [exchange]
TITLE   :XFOR: Event ID 246 Attempt to Read Beyond End of Object
PRODUCT :Microsoft Exchange
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:kbusage

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When sending messages by means of the X.400 connector with "Disclosure of
Recipients" enabled, messages are trapped in the MTADATA, and are not sent. No
nondelivery report (NDR) is sent to the originator.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server
version 4.0. This problem was corrected in the latest Microsoft Exchange 4.0
U.S. Service Pack. For information on obtaining the service pack, query on the
following word in the Microsoft Knowledge Base (without the spaces):

   S E R V P A C K
