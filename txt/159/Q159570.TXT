TITLE   :INF: Access ODBC Driver Is Double-Byte Enabled
PRODUCT :Open Database Connectivity (ODBC)
PROD/VER:WINDOWS:2.5,3.4
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Open Database Connectivity, version 2.5 
 - Microsoft ODBC Driver for Access, version 3.4 
-------------------------------------------------------------------------------

The Microsoft Access ODBC Driver, starting with version 3.40, is enabled to
support double-byte character strings (DBCS). However, the Access ODBC
Driver version 3.40 does not yet support Unicode.

Additional query words: 3.40 2.50
