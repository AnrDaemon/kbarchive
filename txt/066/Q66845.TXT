TITLE   :ASD: Install Error Opening FOREIGN.FS4
PRODUCT :Microsoft Home Games
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
SUMMARY
=======

When installing Aircraft & Scenery Designer (ASD), the following error
message may appear:

   Error opening C:\FLTSIM4\FOREIGN.FS4, File not found

The ASD Install program is attempting to open the FOREIGN.FS4 file, but cannot do
so. Verify that Flight Simulator 4.00, including the file FOREIGN.FS4, is
installed in the directory specified with the Install command.

MORE INFORMATION
================

The Install program must have the Flight Simulator directory explicitly
specified, and the Install program must be run from the current drive.

For more information on installing ASD, query in this Knowledge Base on the
following words:

   ASD and proper and install
