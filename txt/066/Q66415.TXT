SUMMARY
=======

This article covers installation and compatibility of the Pacific Data 25inOne!
Font Cartridge printer driver.

MORE INFORMATION
================

Features and Compatibility
--------------------------

The Pacific Data software includes specific instructions on the following:

 - Windows 3.0 driver installation

 - Installing the Windows driver

 - Previously installed soft font files

 - Selecting fonts in Excel

 - Selecting fonts in Microsoft Windows Write

 - Selecting fonts in Samna Ami and Ami Professional

 - Selecting fonts in Word for Windows

 - Extended characters

The 25inOne! fonts cartridge should work on the following printers:

   HP LaserJet Plus
   HP LaserJet series(II,IID,IIP)
   HP LaserJet 500+
   Canon LPB-8II
   Olivletti PG-108/PG-208

Installation
------------

The following are the steps to install the Pacific Data 25inOne! Font Cartridge
printer driver in Microsoft Windows version 3.0 or 3.0a:

1. Run Control Panel and choose the Printers icon.

2. Select the correct printer from the list of installed printers and choose the
   Configure button.

3. Choose the Setup button.

4. In the PCL/HP LaserJet on <port> dialog box, verify that the correct
   printer is selected.

5. Choose the Fonts button.

6. Choose the Add Fonts button.

7. Put the Pacific Data driver disk in drive A and type "a:\windows" (without
   the quotation marks) for the path for font files.

8. Choose the OK button.

9. Select 25inOne!(172) or PDFONTS in the right hand list box.

10. Choose the Add button.

11. Type the path to the directory to which you want to add the fonts and choose
   the OK button. This moves the cartridge name to the left box.

12. Choose the Exit button.

13. In the PCL/HP LaserJet on <port> dialog box, highlight 25inOne!(172)
   or PDFONTS in the cartridge list, and choose the OK button.

14. In the Printers Configure dialog box, choose the OK button.

15. In the Printers dialog box, choose the OK button.
