TITLE   :Setup Directory Greater than Eight Characters in Games
PRODUCT :Microsoft Home Games
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
SUMMARY
=======

If the directory in which you are trying to install the Windows Entertainment
Pack is longer than eight characters, you will receive the following error
message:

   An error occurred while copying files to your hard disk. Please correct the
   problem and run setup again

WORKAROUND
==========

The only workaround is to limit the name of the directory to eight characters.
