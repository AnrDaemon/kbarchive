TITLE   :Olivetti DOS 4.01 Revision 1.06 or Later Needed to Run Windows
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

According to Olivetti Corporation, Olivetti DOS version 4.01 with a revision
number earlier than 1.06 does not run Microsoft Windows version 3.0 correctly.

If you have a version of Olivetti DOS 4.01 with a revision date earlier than
1.06, contact Olivetti for Olivetti DOS 4.01 Rev 6.


MORE INFORMATION
================

A possible symptom might be an

   Unsupported DOS

error from Windows when trying to run Windows 3.0 in any mode (real, standard, or
enhanced) on an Olivetti computer running Olivetti DOS.

Additional query words: 3.00 3.0 3.0a 3.00a win30
