TITLE   :Location of Windows NT Server Tools for Windows 95
PRODUCT :Microsoft Windows NT
PROD/VER:WINDOWS:95; winnt:3.5,3.51; :
OPER/SYS:
KEYWORDS:kbinterop

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server versions 3.5, 3.51 
 - MSPRESS Microsoft Windows NT Resource Kit 
 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SUMMARY
=======

The Windows NT Server Tools for Windows 95 is available in the following
locations:

 - Windows NT Resource Kit version 3.51. Go to the Win95Admin directory and
   follow the instructions in SRVTOOLS.TXT.

 - Microsoft World Wide Web site. Go to:

   http://www.microsoft.com/ithome
