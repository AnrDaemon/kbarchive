-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the minimum system requirements to run Microsoft Ancient
Lands version 1.0 for the Macintosh.

MORE INFORMATION
================

The minimum system requirements to run Ancient Lands for the Macintosh are:

1. Any Apple Macintosh computer supporting a 256-color, 640x480 resolution
   (13-inch), or higher resolution monitor.

2. System 7 or later.

3. Minimum 4 MB RAM memory.

4. Minimum 1 MB available hard-disk space.

5. Double-speed CD-ROM: The CD-ROM drive should be able to transfer data from
   the CD-ROM at 300 KB per second while using less than 40% of the CPU
   bandwidth and has an average seek time of less than 400 milliseconds.
