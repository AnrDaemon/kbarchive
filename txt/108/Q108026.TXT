TITLE   :PC Win: Global Address List Name Search Returns All Names
PRODUCT :Microsoft Mail For PC Networks
PROD/VER:WINDOWS:3.0,3.0b,3.2
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail for Windows, versions 3.0, 3.0b, 3.2 
-------------------------------------------------------------------------------

SYMPTOMS
========

In versions 3.0, 3.0b, or 3.2 of Microsoft Mail for Windows, when you perform a
name search on the Global Address List (GAL), if the selection criteria are the
surname and first initial of the given name, and these letters are the same (for
example, Smith, Steve), all names that match the surname will be returned.

STATUS
======

Microsoft has confirmed this to be a problem in versions 3.0, 3.0b, and 3.2 of
Microsoft Mail for Windows. We are researching this problem and will post new
information here in the Knowledge Base as it becomes available.

Additional query words: 3.00 3.00b 3.20
