DOCUMENT:Q94965  17-OCT-1999  [win95x]
TITLE   :HiJaak for Windows 1.0 Doesn't Install If SHARE.EXE Is Loaded
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a,3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

The install program for Inset Systems HiJaak for Windows, version 1.0 fails on
Disk 2 and the following error message is displayed if you have SHARE.EXE
loaded:

   SYSTEM ERROR
   Cannot Read From Drive X:

WORKAROUND
==========

 - Temporarily disable SHARE.EXE before you install HiJaak.

   -or-

 - Copy the contents of Disk 1 and Disk 2 to a temporary directory on your hard
   disk drive, and run the INSTALL.EXE program from Windows.

STATUS
======

Inset Systems is aware of this problem and is currently researching it. For more
information, contact Inset Systems Technical Support.

To obtain the phone numbers for Inset Systems, query on the following words in
the Microsoft Knowledge Base:

   Inset and phone list and pss
