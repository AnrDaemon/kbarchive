TITLE   :Maximum Number of Characters on WIN.INI Load Line Is 127
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a,3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
 - Microsoft Windows for Workgroups versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

The maximum number of characters on the Load= or Run= lines in the WIN.INI file
is 127 characters. In Windows 3.1 and Windows for Workgroups, you can work
around this limit by setting up a program item for an application in the StartUp
group.

For more information about the StartUp group and starting an application when you
start Windows, see page 87 of the "Microsoft Windows User's Guide" for Windows
version 3.1 or page 35 of the "Microsoft Windows for Workgroups User's Guide"
for Windows for Workgroups 3.1.

Additional query words: 3.00 3.00a 3.10 3.11 automatic loading starting
