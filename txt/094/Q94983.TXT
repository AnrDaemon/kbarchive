The following is a list of the filenames and S numbers for the source code from
the March 1993 issue of the "Microsoft Systems Journal" (Vol. 8, No. 3):

 - S13983 DIAG.ZIP

   "MFC Diagnostic Facilities Part II: More-Sophisticated Debugging Techniques"
   by Brian Meyers, page 55.

   This issue, Brian Meyers explains advanced debugging techniques for the
   Microsoft Foundation Classes, demonstrating memory leak detection, hooking
   memory allocations, run-time type checking, dynamic type checking, and object
   validity testing.

   This file can be found in the Software/Data Library by searching on the word
   DIAG, the Q number of this article, or S13983. DIAG.ZIP was archived using
   the PKware file-compression utility.

 - S13982 MEMMAN.ZIP

   "An Introduction to WIn32 Heap and Virtual Memory Management Routines" by
   Jeffrey Richter, page 15.

   Jeffrey Richter explains the various heap and virtual memory management
   techniques available to Win32-based applications. He illustrates these
   techniques with sample programs that work with virtual address spaces and
   report statistics on virtual memory.

   This file can be found in the Software/Data Library by searching on the word
   MEMMAN, the Q number of this article, or S13982. MEMMAN.ZIP was archived
   using the PKware file-compression utility.
