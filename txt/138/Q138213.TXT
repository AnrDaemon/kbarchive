TITLE   :Uninstall Incorrectly Removes Dynamic Link Library Files
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kbbuglist

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Dogs for Windows, version 1.0 
 - Microsoft Oceans for Windows, version 1.0 
 - Microsoft World of Flight for Windows, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you try to uninstall one of the programs listed at top of this article, the
uninstall process removes (deletes) a Dynamic Link Library file required by
other programs.

Microsoft Dogs Removes:

 - Ir32_32.dll
 - Msrle32.dll
 - Wing32.dll

Microsoft Oceans Removes:

 - Otoon32.dll
 - Wing32.dll

World of Flight removes:

 - Otoon32.dll
 - Wing32.dll

RESOLUTION
==========

Reinstall the program that is experiencing problems.


STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.
