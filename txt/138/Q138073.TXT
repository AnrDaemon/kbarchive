TITLE   :Saved Fury3 Game Does Not Save the Window Size
PRODUCT :Microsoft Home Games
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:

======================================================================
1.00
WINDOWS
kbui kbusage

-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Fury 3 for Windows, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Fury3 does not save the window size when you save a game. Fury3 also does not
save the window size when you exit Fury3. When you resume a saved game or start
a new game, Fury3 starts in a maximized window.

Additional query words: kbhowto fury 3 fury-3 1.00 displayed displaying sized sizing maximize screen multi media multi-media multimedia
