TITLE   :SNA Server Client for Windows 95 Is in the Latest Service Pack
PRODUCT :Microsoft SNA Server
PROD/VER:WINDOWS:2.0,2.1,2.11,3.0
OPER/SYS:
KEYWORDS:kbnetwork

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft SNA Server, versions 2.0, 2.1, 2.11, 3.0, on platform(s):
    - the operating system: Microsoft Windows NT 
-------------------------------------------------------------------------------

SUMMARY
=======

The SNA Server Client for Windows 95, previously available only through the
Internet on the Microsoft World Wide Web and the Microsoft FTP server sites, is
now included in the latest SNA Server Service Pack. For 3.0, the SNA Server
Windows 95 client is included.
