TITLE   :No Sound with MediaVision Pro Audio PAS-16 CD Player
PRODUCT :Microsoft Windows NT
PROD/VER:3.5
OPER/SYS:
KEYWORDS:kbtool

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.5 
 - Microsoft Windows NT Server version 3.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you have the MediaVision Pro Audio PAS-16 driver for Windows NT installed and
you try to play a CD-ROM with CDPLAYER.EXE, the CD Player application appears to
be working properly, but there is no sound.

WORKAROUND
==========

1. Open the Volume Control applet.

2. Select Expanded View.

3. In the Device column, select each one of the check boxes.


The MediaVision product discussed here is manufactured by a vendor independent of
Microsoft; we make no warranty, implied or otherwise, regarding this product's
performance or reliability.
