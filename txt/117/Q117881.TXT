TITLE   :Windows 95 Err Msg: This Application's Attempt to Get Access...
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:95
OPER/SYS:
KEYWORDS:win95 appscomp kbAppCompatibility

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start Windows 95, the following error message is displayed

   Windows

   This application's attempt to get access to COM<x> has failed because
   the port is in use by a virtual device.

   Press a key to continue.

where <x> specifies a particular communications port.

CAUSE
=====

If you install the pen driver that ships with Fractal Design Painter version
1.2, you receive the above error message.

RESOLUTION
==========

According to Fractal Design Corporation, upgrading to Fractal Design Painter
version 2.0 and choosing to use the existing pen drivers during Setup corrects
this error.

MORE INFORMATION
================

independent of Microsoft; we make no warranty, implied or otherwise, regarding
this product's performance or reliability.
