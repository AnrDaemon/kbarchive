SUMMARY
=======

ISCDROM: Demonstrates Testing if Drive is CD-ROM.

ISCDROM is an assembly code fragment which demonstrates how to determine if a
drive is a CD-ROM drive.

ISCDROM is a companion source code fragment to the technical article, "CD-ROM
Design and Optimization." Please reference this article for more information.
