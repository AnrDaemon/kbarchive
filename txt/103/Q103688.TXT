TITLE   :PRB: "Cannot Write VER.DLL" Installing FoxPro with Windows 3.0
PRODUCT :Microsoft FoxPro
PROD/VER:WINDOWS:2.5,2.5a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft FoxPro for Windows, versions 2.5, 2.5a 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you are installing FoxPro for Windows under Windows version 3.0, the error
message "Cannot write VER.DLL" might appear.

RESOLUTION
==========

To work around this problem, rename the VER.DLL file in the Windows SYSTEM
subdirectory.

Additional query words: FoxWin 2.50 WINDOWS 3.0 VER.DLL INSTALL
