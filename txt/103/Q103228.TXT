SUMMARY
=======

VBSource and VBDest: Demonstrate DDE Communication Over a Network

The VBSource and VBDest sample applications demonstrate how applications can
communicate over the network using Network Dynamic Data Exchange (Network DDE).
These sample applications accompany the "Reach Out and Exchange: A Network DDE
Overview," "Browsing for Network Servers in C and Visual Basic," and "Learning
to Share: Network DDE Shares and Visual Basic" technical articles on the
Microsoft(R) Developer Network CD.

VBSource and VBDest were built and tested under Microsoft Windows(TM) version 3.1
and Microsoft Visual Basic(TM) version 2.0.
