TITLE   :Windows Sound System Does Not Update Lotus
PRODUCT :Miscellaneous Windows Products
PROD/VER:
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows Sound System, versions 1.0, 1.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

Page 30 of the "Microsoft Windows Sound System User's Guide" states that Windows
Sound System automatically installs the Proof menu in Microsoft Excel or Lotus
1-2-3 for Windows.

Windows Sound System does not automatically add the Proof Menu to Lotus 1-2-3 for
Windows version 4.0.

Lotus 1-2-3 for Windows version 1.10.01 is the version of Lotus 1-2-3 for Windows
that will work with the Windows Sound System.

For more information about Lotus combatibility, query on the following words in
the Microsoft Knowledge Base:

   lotus and compatibility
