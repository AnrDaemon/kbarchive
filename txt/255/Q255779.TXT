DOCUMENT:Q255779  08-MAY-2002  [winnt]
TITLE   :Can View System Drive Even Though View Is Disabled by a Policy
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0,4.0 SP4,4.0 SP5
OPER/SYS:
KEYWORDS:kbWinNT400PreSP7Fix

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server versions 4.0 SP4, 4.0 SP5, Terminal Server Edition 
 - Microsoft Windows NT Server version 4.0 
 - Microsoft Windows NT Server, Enterprise Edition version 4.0 
 - Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you use System Policy Editor to enable the "Hide drives in My Computer" shell
restriction option, after you install Service Pack 4 or Service Pack 5 and you
restart the computer, you can still view the system drive.

==========

--------------

correct the problem described in this article and should be applied only to
systems experiencing this specific problem.
