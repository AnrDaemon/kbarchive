TITLE   :Age of Empires II: Game Does Not Start as Expected
PRODUCT :Microsoft Home Games
PROD/VER::
OPER/SYS:
KEYWORDS:kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Age of Empires II Expansion: The Conquerors 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to start Microsoft Age of Empires II: The Conquerors, the game
may not start as expected. Additionally, you may receive a "stop" error message.

CAUSE
=====

This behavior can occur if either of the following conditions is true:

 - You install Age of Empires II: Age of Kings, and then install The Conquerors
   expansion without first running Age of Empires II.

   -or-

 - You install the version 1.0b patch for The Conquerors on a Microsoft Windows
   2000-based computer.

RESOLUTION
==========

To resolve this issue, use either of the following methods, as appropriate to
your circumstances.

If You Installed Both Products
------------------------------

1. Start Age of Kings, and then accept the End User License Agreement.

2. Quit Age of Kings.

3. Insert The Conqueror's CD-ROM into your computer's CD-ROM or DVD-ROM drive.

4. Start The Conquerors.

If You Installed The Conquerors Update v1.0b on a Windows 2000-Based Computer
-----------------------------------------------------------------------------

NOTE: These steps assume that you have WinZip installed on your computer. To
obtain WinZip, browse to the following WinZip Computing Inc. Web site:

   http://www.winzip.com

1. Right-click the 1.0b patch file, and then click "Open with Winzip".

2. Click SECDRV.SYS, and then click Extract.

3. In the "Folders/drives" box, select Winnt\System32\Drivers, and then click
   Extract.

4. Quit WinZip.

5. Restart the computer.
