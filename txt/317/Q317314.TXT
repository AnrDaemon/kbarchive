TITLE   :Midtown Madness 2: Game Is Excessively Slow to Start
PRODUCT :Microsoft Home Games
PROD/VER::1.0,2.0
OPER/SYS:
KEYWORDS:kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Midtown Madness, version 1.0, used with:
    - the operating system: Microsoft Windows XP 
 - Microsoft Midtown Madness 2, version 2.0, used with:
    - the operating system: Microsoft Windows XP 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to start Midtown Madness 2 on a Microsoft Windows XP-based
computer, the game may be excessively slow to start. The program may appear to
stop responding (hang), although it will likely start if you wait approximately
10 minutes.

CAUSE
=====

This issue has been seen on computers with APIC (Advanced Programmable Interrupt
Controller) based Motherboards.

RESOLUTION
==========

To resolve this issue, use either of the following methods.

Method 1: Install Update
------------------------

An update has been created to resolve this issue. To download and install the
update, browse to the following Microsoft Web site:

Midtown Madness 1

   http://www.microsoft.com/games/midtown/downloads.htm

Midtown Madness 2

   http://www.microsoft.com/games/midtown2


Method 2 (Workaround): Start Game from CD-ROM
---------------------------------------------

1. Insert the Midtown Madness 2 CD-ROM into your computer's CD-ROM or DVD-ROM
   drive, and then click Play.

2. Press CTRL+ESC to display the Start menu.

3. If Midtown Madness 2 does not start immediately, point to All Programs, point
   to Microsoft Games, point to Midtown Madness 2, and then click Midtown
   Madness 2.
