DOCUMENT:Q310573  06-AUG-2002  [homegame]
TITLE   :Train Simulator: You Cannot Open the Train Simulator Manual
PRODUCT :Microsoft Home Games
PROD/VER::1.0
OPER/SYS:
KEYWORDS:kberrmsg kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Train Simulator, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to open the Train Simulator Manual, one of the following
behaviors may occur:

 - You receive an Open With dialog box, and then you are prompted to select the
   program that you want to use to open the Trainsim.pdf file.

   -or-

 - You receive the following error message:

   Failed to open package

CAUSE
=====

This behavior can occur if the Adobe Acrobat Reader program is damaged or
missing (not installed).

The Acrobat Reader program is required to open the Train Simulator Manual.

RESOLUTION
==========

To resolve this issue, install Acrobat Reader. To do this, follow these steps:

1. Start Windows Explorer.

2. Navigate to the following location, where <drive> is the drive letter
   of your computer's CD-ROM or DVD-ROM drive, which contains your Microsoft
   Train Simulator Disc 1 compact disc:

   <drive>:\Extras

3. Double-click Ar405eng.

   Acrobat Reader Setup starts.

4. Click Next, and then follow the steps to install Acrobat Reader.

MORE INFORMATION
================

To download the latest version of Acrobat Reader, browse to the following Adobe
Web site:

   http://www.adobe.com/products/acrobat/readermain.html
