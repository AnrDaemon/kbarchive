TITLE   :Games: Video Problems with Evans &amp; Sutherland Video Cards
PRODUCT :Microsoft Home Games
PROD/VER::1.0,2.0
OPER/SYS:
KEYWORDS:kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Age of Empires, version 1.0 
 - Microsoft Age of Empires Expansion: The Rise of Rome, version 1.0 
 - Microsoft Age of Empires II Expansion: The Conquerors 
 - Microsoft Age of Empires II: The Age of Kings, version 2.0 
 - Microsoft Age of Empires, Gold Edition 
 - Microsoft Allegiance, version 1.0 
 - Microsoft Asheron's Call, version 1.0 
 - Microsoft Combat Flight Simulator 2: WWII Pacific Theater, version 1.0 
 - Microsoft Combat Flight Simulator: WWII Europe Series, version 1.0 
 - Microsoft Crimson Skies 
 - Microsoft Flight Simulator 2000 
 - Microsoft StarLancer, version 1.0 
 - Microsoft Motocross Madness 2, version 2.0 
 - Microsoft Midtown Madness 2, version 2.0 
 - Microsoft MechWarrior 4: Vengeance 
 - Microsoft Links LS 2000 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start one of the programs listed at the beginning of this article you
may experience one or more of the following symptoms:

 - The computer stops responding (hangs).

 - Graphics in the game appear distorted or fuzzy.

 - The screen turns black.

 - Horizontal lines appear across the screen.

 - The game quits unexpectedly immediately after you start it.

 - An error message about DirectX or DirectDraw appears.

 - An "invalid page fault in module <unknown>" error appears when you
   start the game.

CAUSE
=====

This behavior can occur if you have an Evans & Sutherland video adapter
installed in your computer. The Evans & Sutherland video adapters do not
fully support DirectX. Some cards provide limited support for DirectX 5.0 and
earlier. For more information, contact Evans & Sutherland support.


RESOLUTION
==========

You may be able to resolve this issue by downloading and installing the latest
version of the video driver. For more information, visit the Evans &
Sutherland Web site:

   http://www.es.com (http://www.es.com)
