TITLE   :Extra Files That Can Be Manually Installed from the SP4 CD
PRODUCT :Microsoft Windows NT
PROD/VER::4.0 SP4
OPER/SYS:
KEYWORDS:kbsetup

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0 SP4 
 - Microsoft Windows NT Server version 4.0 SP4, Terminal Server Edition 
 - Microsoft Windows NT Workstation version 4.0 SP4 
-------------------------------------------------------------------------------

SUMMARY
=======

The Service Pack 4 (SP4) CD contains files in the Alpha and i386 folders that
are not installed automatically. These files are included only in the CD version
of SP4. They are not included in the downloadable version of SP4.

The files are of the following types:

 - Files for Remote Access Service (RAS), used to install versions of Windows CE
   Services.

 - Sound Blaster Plug and Play files.

 - Updated files for debugging.

 - Multimedia Setup files.

 - Updated printer files and drivers.

MORE INFORMATION
================

The list below has been broken down by component for easier identification and
mapping of file to driver.

Additional Remote Access Service Components
-------------------------------------------

Service Pack 4 allows Windows CE Services to install RAS directly from the SP4
CD. These files contain no additional fixes from the retail version of Windows
NT 4.0 and are not installed by Update.exe.

NOTE: This applies only to the installation of versions of Windows CE Services
earlier than version 3.0.
