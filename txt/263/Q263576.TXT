DOCUMENT:Q263576  12-JUN-2001  [homegame]
TITLE   :Motocross Madness 2: Program Quits During CD-ROM Verification
PRODUCT :Microsoft Home Games
PROD/VER::2.0
OPER/SYS:
KEYWORDS:kbimu msgame

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Motocross Madness 2, version 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to start Microsoft Motocross Madness 2, the program may quit
unexpectedly during the CD-ROM verification process.

CAUSE
=====

This behavior can occur if you are running Microsoft NetMeeting when you attempt
to start Motocross Madness 2.

RESOLUTION
==========

To prevent this issue from occurring, quit NetMeeting before you attempt to
start Motocross Madness 2.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Motocross Madness 2,
version 2.0.

Additional query words: 2.00 msgame mcm2 motorcross crash exits stops
