DOCUMENT:Q242090  10-AUG-2001  [winnt]
TITLE   :Event 3113 using Novell intraNetWare Client Version 4.11a.
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

On a computer running Microsoft Windows NT 4.0 Workstation Service Pack 3 and
Novell's intraNetWare client version 4.11a, the Workstation service does not
start, and Event 3113 is recorded in Event Viewer.

CAUSE
=====

This issue is known to occur when you use intraNetWare client version 4.11a.

RESOLUTION
==========

To resolve this issue, contact Novell for information about obtaining and
installing the latest version of the intraNetWare client.

This issue was first resolved in Novell's intraNetWare client version 4.3.
