TITLE   :Cannot Minimize PowerPoint 4.0 Viewer
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:95
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you try to minimize the Microsoft PowerPoint version 4.0 viewer in Windows
95, the viewer is closed instead of minimized.

CAUSE
=====

In Windows 95, the Close button in PowerPoint changes to the Minimize button,
but still closes the viewer.

RESOLUTION
==========

To minimize the viewer, use the right mouse button to click the viewer button on
the taskbar, and then click Minimize on the menu that appears.

STATUS
======

This problem no longer occurs in Microsoft PowerPoint version 7.0 for Windows
95.
