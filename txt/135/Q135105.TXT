TITLE   :MSN Err Msg: You Cannot Select Any Names Now...
PRODUCT :The Microsoft Network
PROD/VER:WINDOWS:1.2,1.3,2.0,2.5,95
OPER/SYS:
KEYWORDS:kberrmsg kbmsn

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - The Microsoft Network versions 1.2, 1.3, 2.0, 2.5 
 - the operating system: Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

While connected to The Microsoft Network, you open the address book in Microsoft
Exchange. When you select Microsoft Network in the Show Names From The box, you
receive the following error message:

   You cannot select any names now. Cannot connect to The Microsoft Network.
   Please try again later. The Microsoft Network Address Book - 974782474 -
   [80040119]

CAUSE
=====

The address book server to which you are connected is not functioning properly.

RESOLUTION
==========

To fix the problem, quit and restart Microsoft Exchange. If the problem
persists, all the address book servers are not functioning properly.
