The information in this article applies to:

 - Microsoft PowerPoint for Windows 95, version 7.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes steps to change the default working directory in
PowerPoint for Windows 95, version 7.0. By default, PowerPoint will open
presentations from and save presentations to the My Documents folder.

MORE INFORMATION
================

To change the working directory:

1. Open a presentation in PowerPoint.

2. Click Options on the Tools menu.

3. Click the Advanced tab.

4. In the Default File Location box, type the name of the folder in which you
   would like to open presentations from and save them to by default. You must
   enter the entire path including the drive letter. For example, type:

   "c:\msoffice\powerpnt\files\save" (without the quotation marks)

5. Click OK.
