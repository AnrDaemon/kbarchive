TITLE   :PRB: Save As Command Is Unavailable from Visual Class Designer
PRODUCT :Microsoft FoxPro
PROD/VER:WINDOWS:3.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Visual FoxPro for Windows, version 3.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When working in the Visual Class Designer of Microsoft Visual FoxPro, you find
that the Save As command under the File Menu and the Save As command button on
the Toolbar are unavailable (dimmed or greyed out).

WORKAROUND
==========

If you want to save a Visual Class object to a different Class Library, use the
Visual Class Browser instead.

STATUS
======

This behavior is by design.
