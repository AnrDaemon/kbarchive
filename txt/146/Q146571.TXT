TITLE   :XCLN: Microsoft Word May Not Post to Public Folder
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:4.0
OPER/SYS:
KEYWORDS:kbinterop

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Windows 3.x client, version 4.0 
 - Microsoft Exchange Windows NT client, version 4.0 
 - Microsoft Exchange Windows 95/98 client, version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you click Post to Exchange Folder on the File menu in Microsoft Word
version 7.0 for Windows 95, you may receive the following error message

   Cannot find the folder called 'Public Shortcuts Folder.'
   Please connect to a Microsoft Exchange Server to create the folder.

followed by:

   Word was unable to post your document.

RESOLUTION
==========

Upgrade to Microsoft Word for Windows 95 version 7.0a.

To obtain the Microsoft Word for Windows version 7.0a upgrade, call the Microsoft
Order Desk at (800) 360-7561 and request part number 059-00138. If you are
outside the United States, contact the Microsoft subsidiary for your area. To
locate your subsidiary, see the Microsoft World Wide Offices Web site at:

   http://www.microsoft.com/worldwide/default.htm.
