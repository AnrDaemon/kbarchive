TITLE   :Large Width Scrollbars Causes Wisdom Lit. to Word Wrap
PRODUCT :Microsoft Schedule+ for Windows
PROD/VER:WINDOWS:7.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Schedule+ for Windows, version 7.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

Large width scrollbars in Microsoft Windows 95 cause Wisdom Literature to word
wrap in inappropriate places, misalign buttons to text fields.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Schedule+ version 7.0.
We are researching this problem and will post new information here in the
Microsoft Knowledge Base as it becomes available.


Additional query words: schedule plus 7.00
