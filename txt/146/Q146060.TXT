TITLE   :No Left-Hand Support for BallPoint Mouse in Windows 95
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:95
OPER/SYS:
KEYWORDS:win95

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

There is no left-hand support for a BallPoint-style mouse when you use the
Standard PS/2 Port Mouse or the Serial BallPoint driver in Windows 95.

RESOLUTION
==========

Left-hand support for a BallPoint-style mouse is included with the Microsoft
IntelliPoint version 1.1 software. For information about upgrading to this
software, please call the Microsoft Sales and Information Center at (800)
426-9400.

Note that the IntelliPoint 1.1 software supports only two buttons on the
Ballpoint mouse instead of all four buttons.
