DOCUMENT:Q146780  19-MAR-1999  [exchange]
TITLE   :XCLN: Setup Is Unable to Create Item In Program Manager Group
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:4.0
OPER/SYS:
KEYWORDS:kbsetup

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Windows NT client, version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When the Microsoft Exchange Client Setup program is creating the Program Manager
group, it might display the following error message:

   Setup was unable to create an item in the Program Manager group: 'Microsoft
   Exchange', item: 'Microsoft Exchange Setup,,0,0,0,,0,0,0'.

NOTE: If the user changes the default name of the Program Manager group for
Microsoft Exchange during the setup process, that name will appear in the error
message above.

STATUS
======

Microsoft has confirmed this to be a problem in version 4.0 of Microsoft
Exchange. We are researching this problem and will post new information here in
the Microsoft Knowledge Base as it becomes available.
