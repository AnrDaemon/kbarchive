SUMMARY
=======

The following article lists the minimum system requirements necessary to
successfully run Microsoft Pregnancy and Child Care.

MORE INFORMATION
================

The minimum system requirements to run Microsoft Pregnancy and Child Care are:

 - Personal computer with a 386DX processor.

 - Microsoft Windows 95 operating system.

 - 8 MB of RAM memory.

 - 5 MB of additional hard disk space (beyond the normal Windows 95
   requirements).

 - Super VGA, 256-color monitor.

 - Modem (9600 baud minimum, 14,400 baud strongly recommended).

 - Microsoft Mouse or compatible pointing device.

 - Account with MSN, The Microsoft Network.

Optional:

 - Sound card and speakers/headphones
