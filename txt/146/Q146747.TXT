DOCUMENT:Q146747  19-MAR-1999  [exchange]
TITLE   :XCLN: Subject Not Changed When Forwarding Message
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:4.0,5.0
OPER/SYS:
KEYWORDS:kbenv

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Windows 3.x client, versions 4.0, 5.0 
 - Microsoft Exchange Windows NT client, versions 4.0, 5.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you open a message, change the Subject line in General Properties and
forward the message, the Subject line has not changed.

WORKAROUND
==========

To work around this problem, manually change the subject line when you forward
the message or do the following:

1. Open the message, and select Properties from the File menu.

2. Change the Subject line in General Properties and click Apply. Click OK.

3. From the File menu, select Save and close the message.

4. Re-open the message and forward the message.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft Exchange version
4.0 Windows and Windows NT clients. We are researching this problem and will
post new information here in the Microsoft Knowledge Base as it becomes
available.
