TITLE   :PIF Files Created by Windows 3.0 Setup from M to N
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

The following are the default parameters for program information files (PIFs)
created by the Microsoft Windows version 3.0 Setup program, in alphabetical
order from M to N. These defaults may be modified by loading the PIF file into
the Windows 3.0 PIF Editor, PIFEDIT.EXE.

MORE INFORMATION
================
