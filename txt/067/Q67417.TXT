TITLE   :PIF Files Created by Windows 3.0 Setup from P to R

PROD/VER:WINDOWS:3.0,3.0a
OPER/SYS:

-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

The following are the default parameters for program information files (PIFs)
created by the Microsoft Windows version 3.0 Setup program, in alphabetical
order from P to R. These defaults may be modified by loading the PIF file into
the Windows 3.0 PIF Editor, PIFEDIT.EXE.

MORE INFORMATION
================

Name of Executable: PARADOX2.EXE
Name of PIF File Created: PARADOX2.PIF
Optional Parameters: None
Minimum Memory Required: 128
Uses Graphics: No
Uses COM Ports: No
Allow Fast Paste: Yes
Runs in the Background: No
