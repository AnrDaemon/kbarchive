TITLE   :Fontware Installation Kit Not Enough Disk Space Available
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

If you get the following message after you press F10 to make fonts

   Time and Space Estimates. Not enough disk space available

delete some font requests and check the estimates again. The Bitstream Fontware
Installation Kit (FIK) has checked your available disk space against its space
estimates and determined that there is not enough room on your disk.

MORE INFORMATION
================

A Bitstream Fontware Installation Kit is available through a fulfillment coupon
included with the retail version of Microsoft PowerPoint for Windows. This
starter kit will install fonts for Windows applications only; however, it can be
merged with other starter kits to provide support for multiple applications as
described above.
