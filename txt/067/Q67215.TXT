DOCUMENT:Q67215  24-OCT-1999  [win95x]
TITLE   :Terminal's Font Point Sizes Change When Running MS-DOS
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a,3.1,3.11
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
 - Microsoft Windows for Workgroups versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

The available point sizes for the Terminal program will change in the Terminal
Font list (accessible by choosing Terminal Preferences from the Settings menu)
if you are running a non-Windows-based application.

The default point sizes available for the Terminal font in Windows 3.0 are:

   12, 24, 36, 48, 60

The default point size available for the Terminal font in Windows 3.1x and
Windows for Workgroups 3.1x is:

   12

If you are running a non-Windows-based application, the available point sizes for
the Terminal font in Windows 3.0 will change to the following:

   8, 12, 16, 24, 32, 36, 40, 48, 60

If you are running a non-Windows-based application, the available point sizes for
the Terminal font in Windows 3.1x and Windows for Workgroups 3.1x will change to
the following:

   8, 12, 16, 18

Additional query words: 3.0 3.00 3.0a 3.00a 3.10 3.1 3.11
