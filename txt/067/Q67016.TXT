TITLE   :Fontware Installation Kit Requires Different Character Set
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:3.0,3.0a
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

If you request point sizes for a Symbol typeface and you get the following
message

   This typeface requires a different character set. Go to the Control Panel and
   select a different character set

go to the Fontware Control Panel and select one of the symbol character sets
available to make fonts for your device(s). Symbol typefaces require special
symbol character sets.

MORE INFORMATION
================

A Bitstream Fontware Installation Kit is available through a fulfillment coupon
included with the retail version of Microsoft PowerPoint for Windows. This
starter kit will install fonts for Windows applications only; however, the kit
can be merged with other starter kits to provide support for multiple
applications as described above.
