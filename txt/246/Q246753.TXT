TITLE   :Configuring Peer Web Services May Conflict with Norton Utilities
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you have installed Peer Web Services (PWS) on a computer running Microsoft
Windows NT Workstation 4.0 with Norton Utilities installed, you may find that
you cannot remove or configure the Peer Web Services settings.

CAUSE
=====

This behavior may occur because of a conflict between Norton Utilities and Peer
Web Services.

RESOLUTION
==========

To resolve this issue, remove Norton Utilities using the Add/Remove Programs
tool in Control Panel.
