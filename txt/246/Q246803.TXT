TITLE   :STOP 0xA Error After Heavy Usage of NetFlex-3 with Compaq Driver
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Server version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use the Compaq driver for the Intel NetFlex-3 Ethernet card
(N100nt.sys) to handle heavy network traffic on a computer running Microsoft
Windows NT Server 4.0, you may receive a STOP 0xA error message. This may occur
when the driver is from Compaq Software Support Diskette (SSD) 2.12a or earlier.

RESOLUTION
==========

To resolve this behavior, download an updated driver from the Compaq Corp. Web
site at:

   http://search.service.digital.com/

The correct file is named PW3271A_E01 and dated July 20, 1999.
