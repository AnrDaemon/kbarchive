SUMMARY
=======

This article describes how to import pushpins from Microsoft Streets & Trips
2000 into Microsoft MapPoint.

MORE INFORMATION
================

To import Streets & Trips 2000 pushpins into MapPoint:

1. On the Tools menu in Streets & Trips 2000, click Pushpin Explorer.

2. In the Sets box, click to select the pushpin set that you want to use in
   MapPoint, and then click Export.

3. In the "Save as type" box, click All Files (*.*).

4. In the File name box, type "<filename>.csv" (without the quotation
   marks), where <filename> is the name of the pushpin set.

5. Quit Streets & Trips 2000, and then start MapPoint.

6. On the Tools menu, click Pushpin Explorer.

7. Click Import.

8. In the "File of type" box, click Text Files (*.txt, *.csv, *.tab, *.asc).

9. In the list of files, click the file that you created in step four, and then
   click Open.

10. Click Pushpin Map, and then click Next.

11. In the Pushpin Set Symbol box, click the symbol that you want to use for all
   pushpins in the pushpin set.

12. In the Set Name box, type a name for the pushpin set, and then click Finish.
