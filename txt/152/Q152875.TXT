DOCUMENT:Q152875  28-MAR-1999  [exchange]
TITLE   :XCON: Failure to Send to Embedded DL with Delivery Restrictions
PRODUCT :Microsoft Exchange
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:kbusage

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you are a member of one Microsoft Exchange Distribution List (DL), this DL is
a member of a parent DL, and you have permissions to send to both DLs, when you
send a message to the parent DL, the message will be returned with a
Non-Delivery Receipt (NDR) and the following error message:

   The originator does not have permission to submit message.


STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server
version 4.0. This problem was corrected in Microsoft Exchange Server 4.0 U.S.
Service Pack 2. For information on obtaining the service pack, query on the
following word in the Microsoft Knowledge Base (without the spaces):

   S E R V P A C K

Additional query words: NDR non delivery report DL nest nested
