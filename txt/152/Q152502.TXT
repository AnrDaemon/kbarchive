TITLE   :Restoring Schedule File Doesn't Reset Disallow Overlapping
PRODUCT :Microsoft Schedule+ for Windows
PROD/VER:WINDOWS:7.0
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Schedule+ for Windows, version 7.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you restore a backup of Schedule+ 7.0, the restore unconditionally clears
the Disallow Overlapping Appointments option (click Set Access Permissions on
the Tools menu. Select the Global tab).



STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Schedule+ version 7.0.
We are researching this problem and will post new information here in the
Microsoft Knowledge Base as it becomes available.
