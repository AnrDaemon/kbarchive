DOCUMENT:Q152884  28-MAR-1999  [exchange]
TITLE   :XADM: Err Msg: IS Generates EcBufferTooSmall
PRODUCT :Microsoft Exchange
PROD/VER:winnt:4.0
OPER/SYS:
KEYWORDS:kbusage

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Server, version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

If there are a lot of folders in one Microsoft Exchange Server Site, any attempt
to open the Instances tab on the properties page of any Public Information Store
in the Microsoft Exchange Administrator program will fail with the following
error message:

   EcBufferTooSmall

This problem has been seen with 18,000 folders in one Site.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server
version 4.0. This problem was corrected in Microsoft Exchange Server 4.0 U.S.
Service Pack 2. For information on obtaining the service pack, query on the
following word in the Microsoft Knowledge Base (without the spaces):

   S E R V P A C K

Additional query words: STORE
