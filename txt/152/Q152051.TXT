TITLE   :IBM Gothic Box Font Appears Garbled
PRODUCT :Microsoft Windows NT
PROD/VER:winnt:3.51
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.51 
 - Microsoft Windows NT Server version 3.51 
-------------------------------------------------------------------------------


SYMPTOMS
========

After you apply Windows NT 3.51 Service Pack 3 (SP3) or Service Pack 4 (SP4,)
the IBM Gothic Box True Type font does not display the correct character set
mapping; for example, the keystroke 'A' displays the ampersand (&) on the
screen.


STATUS
======

Microsoft has confirmed this to be a problem in Windows NT version 3.51 Service
Pack 3 and Service Pack 4. This problem was corrected in the latest Windows NT
3.51 U.S. Service Pack. For information on obtaining the Service Pack, query on
the following word in the Microsoft Knowledge Base (without the spaces):

   S E R V P A C K


Additional query words: 3.51 prodnt
