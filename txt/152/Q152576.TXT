TITLE   :XCLN: Components Installed During Exchange 4.0 Client Setup
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:4.0
OPER/SYS:
KEYWORDS:kbsetup

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Windows 95/98 client, version 4.0 
 - Microsoft Exchange Windows 3.x client, version 4.0 
 - Microsoft Exchange Windows NT client, version 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The following is a list of the component differences between a Typical and a
Laptop setup for the Microsoft Exchange clients for Windows. Typical and Custom
installations will install the same components on the client. The only
difference is that Custom allows the user to modify the selected components.

WIN95 & WIN NT CLIENT                   TYPICAL   LAPTOP
=====================                   =======   ======
Microsoft Exchange                      X         X
   Information Services                 X         X
      Personal Address Book             X         X
      Personal Folders                  X         X
      Microsoft Exchange Server         X         X
      Microsoft Mail                    -         -
   Spelling                             X         -
   Help                                 X         -

Microsoft Schedule+ Program Files       X         X
   Print Layouts and Paper Formats      X         X
   Help                                 X         -
   Additional Importers and Exporters   X         -
   Seven Habits Tools                   X         -

WINDOWS 3.x CLIENT                      TYPICAL   LAPTOP
==================                      =======   ======
Microsoft Exchange                      X         X
   Information Services                 X         X
      Personal Address Book             X         X
      Personal Folders                  X         X
      Microsoft Exchange Server         X         X
      Microsoft Mail                    -         -
   Spelling                             X         -
   Help                                 X         -
   Shiva                                X         X

Microsoft Schedule+ Program Files       X         X
   Print Layouts and Paper Formats      X         X
   Help                                 X         -
   Additional Importers and Exporters   X         -
   Seven Habits Tools                   X         -
