DOCUMENT:Q152069  29-MAR-1999  [exchange]
TITLE   :XCLN: Can't Save Multiple Attachments From Exchange
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:4.0
OPER/SYS:
KEYWORDS:kbusage

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Windows 95/98 client, version 4.0 
 - Microsoft Exchange Windows NT client, version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to save multiple attachments to a message in a Microsoft
Exchange client, you will see the following incorrect error message if the files
already exist.

   The file exists with read only attributes.
   Do you want to skip this file and continue?

WORKAROUND
==========

If the files are saved individually, the option to overwrite the existing file
is available and the files can be successfully saved.


STATUS
======

Microsoft has confirmed this to be a problem in version 4.0 of Microsoft
Exchange. We are researching this problem and will post new information here in
the Microsoft Knowledge Base as it becomes available.
