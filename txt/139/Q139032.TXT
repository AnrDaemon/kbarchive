TITLE   :PC NTMMTA: Dispatch Instance Name Is Not Displayed
PRODUCT :Microsoft Mail For PC Networks
PROD/VER::3.5
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Mail Multitasking MTA for Windows NT, version 3.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you monitor a Dispatch instance using the Microsoft Windows NT Multitasking
MTA (NT MMTA), the Instance name is not displayed in the monitor window.

NOTE: With multiple instances of Dispatch running on one machine, it is difficult
to determine which Monitor Window pertains to which Instance.

STATUS
======

Microsoft has confirmed this to be a problem in version 3.5 of Microsoft Mail
Multitasking MTA for Windows NT. We are researching the problem and will post
new information here in the Knowledge Base when it becomes available.
