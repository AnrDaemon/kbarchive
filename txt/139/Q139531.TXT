TITLE   :Multimedia Fonts Not Reinstalled in Maintenance Mode
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:WINDOWS:; :1995 edition
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Bookshelf '95 for Windows 95 
 - Microsoft Bookshelf 1996-97 for Windows 
 - Microsoft Bookshelf for Windows, 1995 Intro Edition 
 - Microsoft Bookshelf for Windows 1995 edition 
 - Microsoft Encarta 96 Encyclopedia for Windows 
 - Microsoft Encarta 96 World Atlas for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you reinstall the applications listed at the top of this article using
Maintenance Mode setup, the fonts are not reinstalled.

RESOLUTION
==========

If the fonts are not reinstalled, remove the program and reinstall as follows:

Removing the Program
--------------------

1. Click the Start button and then click Run.

2. Type "<CD-ROM drive letter>:\setup.exe" (without the quotation marks)
   for example, if your CD-ROM drive is drive D, type "d:\setup.exe" (without
   the quotation marks).

3. From the Maintenance Mode Setup choices, click Remove All.

Installing the Program
----------------------

1. Click the Start button and then Click Run.

2. Type "<CD-ROM drive letter>:\setup.exe" (without the quotation marks)
