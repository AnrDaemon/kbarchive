TITLE   :PPT7: No Support for Natural Keyboard Application Key
PRODUCT :Microsoft PowerPoint for Windows
PROD/VER:WINDOWS:7.0
OPER/SYS:
KEYWORDS:kbhw kbinterop kbusage kbdta kbHardware

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft PowerPoint for Windows 95, version 7.0 
-------------------------------------------------------------------------------

SUMMARY
=======

In PowerPoint for Windows 95, version 7.0 pressing the Application key on the
Microsoft Natural Keyboard has no effect. PowerPoint 7.0 does not support this
key.

MORE INFORMATION
================

The Microsoft Natural Keyboard has several special keys that make using Windows
95 easier. One of these keys is the Application key. This key is to the right of
the Windows key (on the right side of the keyboard). This key provides quick and
easy access to pop-up menus and help topics.

The Application key is supported in the following Microsoft products:

 - Microsoft Word for Windows 95, version 7.0

 - Microsoft Excel for Windows 95, version 7.0


 - Microsoft Publisher for Windows 95, version 7.0

 - Microsoft PowerPoint 97 for Windows
