TITLE   :Maximum for Compression Agent UltraPack Setting is 999 Days
PRODUCT :Microsoft Windows 95.x Retail Product
PROD/VER:WINDOWS:95
OPER/SYS:
KEYWORDS:win95

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows 95 
 - Microsoft Plus! for Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you enter a value of 1000 or more in the "UltraPack only files not used
within the last <xxx> days" field in Compression Agent, the value defaults
to 999.

CAUSE
=====

Although you can enter a 4-digit number in the field, the field has a maximum
value of 999 days.

RESOLUTION
==========

Do not enter a value larger than 999 in the field.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Plus! version 1.0 for
Windows 95. We are researching this problem and will post new information here
in the Microsoft Knowledge Base as it becomes available.
