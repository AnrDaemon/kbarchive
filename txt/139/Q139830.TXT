TITLE   :Multimedia, Kids: Windows 95 Icons Moved in Full Screen Mode
PRODUCT :Microsoft Home Kids Products
PROD/VER::1.0; WINDOWS:1.0,95
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft 3D Movie Maker, version 1.0 
 - Microsoft Dogs for Windows, version 1.0 
 - Microsoft Oceans for Windows, version 1.0 
 - Microsoft My Personal Tutor: Sam's Hide and Seek Adventure, version 1.0 
 - Microsoft My Personal Tutor: Sky's Space Station Voyage, version 1.0 
 - Microsoft My Personal Tutor: Turru's Daring Sea Quest, version 1.0 
 - Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run one of the products listed at the top of the article in full-
screen mode and your desktop area is set to a resolution greater than 640 x 480,
you see random icon positioning.

STATUS
======

Microsoft has confirmed this to be a problem in the above named products. We are
researching this problem and will post new information here in the Microsoft
Knowledge Base as it becomes available.

MORE INFORMATION
================

When you run the program in full-screen mode, the video resolution is 640 x 480.
When the program releases control of the desktop, any icons outside of the 640 x
480 area move inside the 640 x 480 viewing area.
