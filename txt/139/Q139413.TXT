TITLE   :Installing SCSI2GO PCMCIA Card in a Compaq Elite Laptop
PRODUCT :Microsoft Windows NT
PROD/VER:3.51
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Windows NT Workstation version 3.51 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you install a Future Domain SCSI2GO PCMCIA Card in a Compaq Elite Laptop
the following error message appears after you reboot the computer:

   Event ID 7026: The following boot start or system start driver or drivers
   failed Fd16_700.

WORKAROUND
==========

To work around this problem, change the Device bootup properties in Control
Panel Devices from Boot to System.
