TITLE   :EVG99: Labels Do Not Appear When You Browse the Map
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kbdisplay kbui kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta Virtual Globe 99, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you browse the map in Microsoft Encarta Virtual Globe 99, the labels for
certain locations are not displayed. However, if you type the name of a location
in the Find box, and then click the name of the location in the Places list,
Encarta Virtual Globe moves the map view to the selected location and displays
the label for that location.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Encarta Virtual Globe
99, version 1.0.

MORE INFORMATION
================

This issue has been reported with the following locations in Encarta Virtual
Globe 99:

 - Cushetunk, New Jersey, United States
 - Tsurugi-dake (peak), Japan
