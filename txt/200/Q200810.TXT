TITLE   :XWEB: Korean Language Outlook Web Access Empty Mail Dialog Box S
PRODUCT :Microsoft Exchange
PROD/VER::5.0,5.5
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Active Server Components, version 5.0 
 - Microsoft Outlook Web Access, version 5.5 
 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you use the Korean language version of Outlook Web Access (OWA) to access
your Inbox and then click "Empty Deleted Items Folder," the subsequent message
is not displayed correctly. The dialog box with text that should read "Are you
sure you want to permanently delete all the items and subfolders in the Deleted
Items Folder?" appears with vertical lines where the text should be.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article.
