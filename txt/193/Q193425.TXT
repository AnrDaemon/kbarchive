TITLE   :Comprehensive Map Style Is Displayed When You Exit Game
PRODUCT :Microsoft Home Multimedia Titles
PROD/VER:WINDOWS:1.0
OPER/SYS:
KEYWORDS:kbdisplay kbtool kbui kbimu

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Encarta Virtual Globe 99, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you exit the Name That Place game in Encarta Virtual Globe 99, the
Comprehensive map style is always displayed on the screen.

STATUS
======

This behavior is by design.

MORE INFORMATION
================

When you exit the Name That Place game in Encarta Virtual Globe 99, the location
and zoom level of the correct answer to the last question asked in the game is
displayed on the screen in the Comprehensive map style.

To restore the location, zoom level, and map style displayed prior to starting
the Name That Place game, click the Back button on the Encarta Virtual Globe
menu bar until you return to the screen displayed before you started the Name
That Place game.
