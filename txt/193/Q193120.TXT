TITLE   :XCLN: JScript Error Using "Open Link in New Window" in OWA
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:5.0,5.5
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Exchange Active Server Components, version 5.0 
 - Microsoft Outlook Web Access, version 5.5 
 - Microsoft Outlook Web Access, version 5.5 Service Pack 1 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you use the Outlook Web Access (OWA) client and try to open a link in a new
window, a JScript error message similar to the following appears:

   JScript Error
   An error has occurred in the script on this page.
   Object doesn't support this property or method.
   Do you want to continue running scripts on this page?

WORKAROUND
==========

When you are using Internet Explorer with OWA, on the File menu, point to New,
and then point to Window to open the Inbox in a new window. If you are using
Netscape Navigator, there is no workaround other than to close and reopen the
program.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article.
