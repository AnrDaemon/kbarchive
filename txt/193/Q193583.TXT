TITLE   :INFO: SSInt.exe Is Added to Visual SourceSafe 6.0
PRODUCT :Microsoft SourceSafe
PROD/VER:
OPER/SYS:
KEYWORDS:_IK kbSSafe600fix

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Visual SourceSafe for Windows, version 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

SSInt.exe is a utility that enables you to integrate Visual SourceSafe with
other Microsoft Products. SSInt.exe is included with Visual SourceSafe 6.0.

MORE INFORMATION
================

The SSInt.exe utility originally was included with SourceSafe 4.0, but it was
not included with version 5.0. It has been added back in SourceSafe 6.0. By
running this utility, you might solve some integration issues with other
applications. SSInt.exe is located in the Win32 directory of your Server or
Workstation Visual SourceSafe installation.

REFERENCES
==========

For additional information, please see the following article in the Microsoft
Knowledge Base:

   Q150642 HOWTO: Use SSInt.exe in Visual SourceSafe
