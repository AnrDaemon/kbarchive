The information in this article applies to:

 - Microsoft Visual Basic Learning Edition for Windows, version 6.0 
 - Microsoft Visual Basic Professional Edition for Windows, version 6.0 
 - Microsoft Visual Basic Enterprise Edition for Windows, version 6.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article provides information for ordering the printed version of the Visual
Basic 6.0 product documentation. The printed version of the Visual Basic 6.0
product documentation is published by Microsoft Press and is available directly
from Microsoft Press or a retail bookseller.

MORE INFORMATION
================

For more information about ordering the product documentation from Microsoft
Press, go to:

http://mspress.microsoft.com/

The following are the printed versions of the Visual Basic 6.0 product
documentation published by Microsoft Press:

 - Microsoft Visual Basic 6.0 Programmer's Guide
   ISBN# 1-57231-863-5
   For more information, see:
   http://mspress.microsoft.com/prod/books/1605.htm

 - Microsoft Visual Basic 6.0 Reference Library
   ISBN# 1-57231-864-3
   For more information, see:
   http://mspress.microsoft.com/prod/books/1613.htm
