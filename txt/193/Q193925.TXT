TITLE   :XWEB: OWA: Cannot Log On without Using NT Credentials
PRODUCT :Microsoft Exchange
PROD/VER:WINDOWS:5.5
OPER/SYS:
KEYWORDS:

======================================================================
-------------------------------------------------------------------------------
The information in this article applies to:

 - Microsoft Outlook Web Access, version 5.5 Service Pack 1 
 - Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------


SUMMARY
=======

When using Microsoft Outlook Web Access to log onto a mailbox which has
permissions set to Everyone, the browser will open an authentication dialog box,
and require the user to enter proper Windows NT Credentials.

MORE INFORMATION
================

Even though the permissions on the mailbox are set to Everyone, a valid Windows
NT domain account is required to access an Exchange Server computer.
