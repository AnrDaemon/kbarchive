Q173545: XADM: Exclude Newsgroup Button Is Disabled

Article: Q173545
Product(s): Microsoft Exchange
Version(s): winnt:5.5
Operating System(s): 
Keyword(s): kbusage
Last Modified: 07-JAN-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you are configuring an inbound newsfeed, if you include a newsgroup and hit
Apply, the Exclude button is disabled (unavailable).

WORKAROUND
==========

To exclude the newsgroup, click on a different newsgroup, and then click back on
the original newsgroup.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Exchange Server
version 5.5. We are researching this problem and will post new information here
in the Microsoft Knowledge Base as it becomes available.
