Q186701: XWEB: Time and Date of Delivery Missing from Delivery Receipt

Article: Q186701
Product(s): Microsoft Exchange
Version(s): WINDOWS:5.5
Operating System(s): 
Keyword(s): 
Last Modified: 22-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Outlook Web Access, version 5.5 
- Microsoft Outlook Web Access Service Pack 1 
- Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SYMPTOMS
========


When a message is sent with a delivery receipt from a Microsoft Outlook Web
Access client, a delivery receipt notification is returned to the user. When
viewed by a Microsoft Outlook client, the date and time that the message was
delivered may not appear in the delivery receipt.

WORKAROUND
==========

View the delivery receipt from the Outlook Web Access client.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article.
