Q157548: WD97: Objects in Converted Document Don’t Display in Normal View

Article: Q157548
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): 
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you open a Word 97 document containing objects, into Word 6.x or 7.x, while
you are using normal view, you do not see any objects. This is different from
previous versions of Word, in which OLE objects displayed in normal view.

CAUSE
=====

You are in Normal view. In Word 97, objects are created inside text boxes. Text
boxes are created on the drawing layer and do not display in Normal view.

RESOLUTION
==========

After you open the document in Word 6.x or 7.x, switch to page layout view to
display the objects.
