Q157513: Adaptec EZ-SCSI 4 Power Management Does Not Work

Article: Q157513
Product(s): Microsoft Windows NT
Version(s): 4.0
Operating System(s): 
Keyword(s): kb3rdparty kbhw kbHardware
Last Modified: 08-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Server version 4.0 
- Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

On a computer with an Adaptec 2940 PCI SCSI controller and the Adaptec EZ-SCSI
software version 4, the disk power management features do not work.

The power settings seem to operate normally when you are using the EZ-SCSI
software configuration tool, but the hard disks never power down.

CAUSE
=====

Windows NT 4.0 does not support Power Management features.
