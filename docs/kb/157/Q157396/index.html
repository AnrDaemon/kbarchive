Q157396: WD97: WordPerfect Help Topic Incomplete, Truncated

Article: Q157396
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): 
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

The last line in the WordPerfect Help topic is truncated. This problem appears
when you access the topic by pressing the F4 key when the Help for WordPerfect
Users check box is selected. (To locate this option, click Options on the Tools
menu and then click the General tab.)

Text is missing from the last paragraph in the WordPerfect Help topic for the F4
key (CTRL+T and CTRL+M). This problem is seen when you view the topic from the
Help for WordPerfect Users dialog box. (To locate this topic, double-click WPH
in the status bar. In the Command Keys list, click the Indent-> >>
topic.)

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows.


WORKAROUND
==========

To work around this problem and view information about indentation, click the
Index tab in Microsoft Word 97 Help, and type the following text:

  "indentation" (without the quotation marks)

Double-click the selected text to display a list of topics.
