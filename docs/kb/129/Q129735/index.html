Q129735: When You Paste to FIND, 500 Nations Jumps to Northeast Nations

Article: Q129735
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): 
Last Modified: 19-OCT-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft 500 Nations, version 1.0, on platform(s):
   - the operating system: Microsoft Windows NT, on platform(s):
      - the operating system: Microsoft Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you paste text into the FIND field, if pasted text is not a valid item from
the default list, the Northeast Nations screen will appear when you choose OK.

RESOLUTION
==========

Do not paste text into the FIND window unless the word(s) are part of the
default search list.

In the Contents screen is the A-Z Index option. To search, type the word(s) you
wish to search and select OK.

Additional query words: 1.00 mmtitles search engine pasting searching
