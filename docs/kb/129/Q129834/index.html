Q129834: Garbled Screen After Installing Extras for Pan Chinese WinNT

Article: Q129834
Product(s): Microsoft Windows NT
Version(s): 
Operating System(s): 
Keyword(s): kbprint kbPrinting
Last Modified: 06-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation, Pan Chinese Edition, version 4 
-------------------------------------------------------------------------------

SYMPTOMS
========

Garbled text may appear on your display after you install the Taiwan locale
Extra for Pan Chinese Windows NT Workstation 4.0.

CAUSE
=====

This problem occurs because the screen fonts are being mapped to an incorrect
font.


RESOLUTION
==========

To resolve this problem, download the latest version of the Taiwan locale Extras
set from the following Internet location:

http://www.microsoft.com/hk/pcntw/html/extras.htm

Additional query words: 4.00
