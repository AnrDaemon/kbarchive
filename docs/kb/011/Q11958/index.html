Q11958: Tab.exe Demonstrates Modal and Modeless Dialog Boxes

Article: Q11958
Product(s): Microsoft Windows Software Development Kit
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kbfile kbsample kbDlg kbOSWin2000 kbSDKWin32 kbGrpDSUser kbOSWin310 kbWndw kbWndwMsg kb
Last Modified: 11-JUL-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows Software Development Kit (SDK) 
-------------------------------------------------------------------------------

SUMMARY
=======

Tab.exe is a sample application that demonstrates the differences between modal
and modeless dialog boxes. It also demonstrates using a dialog box within a
dialog box, changing the text in a push button, tabbing between push buttons,
and disabling a push button. TAB also demonstrates how an application processes
the messages received when the user selects a push button.
