Q239627: Encarta Interactive World Atlas Is Unable to Continue Running…

Article: Q239627
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kberrmsg kbui kbimu
Last Modified: 25-JUN-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Interactive World Atlas 2000 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to add a favorite in Microsoft Encarta World Atlas 2000, a
Multimedia Map may open, and then Media Viewer may open in front of the map.

If you click a hyperlink in the Media Viewer article or on the map behind Media
Viewer, or click the close button to close Media Viewer, you receive an error
message similar to the following:

  Encarta Interactive World Atlas is unable to continue running. The problem
  may be the result of low system resources such as memory or disk space. Try
  closing other running applications or freeing up disk space before restarting
  Encarta Interactive World Atlas.

When you click OK, Encarta World Atlas 2000 quits unexpectedly.

CAUSE
=====

This behavior can occur if the following conditions are true when you click OK
in the Add A Favorite dialog box:

- A Multimedia Map is open.

- Media Viewer is open.

RESOLUTION
==========

To work around this issue, restart Encarta World Atlas 2000. When you want to
add a favorite, make sure that Media Viewer is closed.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Encarta World Atlas
2000.

Additional query words: multi multi-media media mm ewa2k crash
