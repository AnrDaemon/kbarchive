Q132845: DynaComm Elite 3.51 GP Faults Connecting to Novell SAA Gateway

Article: Q132845
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you configure Dynacomm Elite version 3.51 to connect to a 3270 host via a
Novell SAA Gateway, you may receive the following error message during the host
session:

  Trap 13 (0DH) - General Protection Fault 0000

RESOLUTION
==========

Contact Netsoft to inquire about obtaining an updated version of the Tixwx.dll
file.

Additional query words: gpf
