Q132954: Mac Srv: How to Get Mail Version Information

Article: Q132954
Product(s): Microsoft Mail For Appletalk Networks
Version(s): WINDOWS:3.1
Operating System(s): 
Keyword(s): 
Last Modified: 09-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail for AppleTalk Networks, version 3.1 
-------------------------------------------------------------------------------

SUMMARY
=======

To troubleshoot Microsoft Mail for AppleTalk Networks, you can run the
Command+Option+? key combination with the AppleTalk Mail client in the
foreground. Once you press the key combination, a dialog box will give the
following information:

- The desk accessory version number.

- The mail extensions version number.

- The mail extensions serial number.

- The mail servers version number.

- The mail servers serial number.

- The mail servers name as it appears in the Chooser.

MORE INFORMATION
================

The Command+? key combination or choosing Help from the Mail menu will run the
Help dialog. If you add the option key, the version information is shown.

The most current Mail Desk Accessory and Extension version number displayed is
31004. The most current Mail Server version displayed is 3104.

Additional query words: 3.10
