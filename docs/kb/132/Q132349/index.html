Q132349: STOP Message 0xFACEFEED on Alpha Computers

Article: Q132349
Product(s): Microsoft Windows NT
Version(s): 
Operating System(s): 
Keyword(s): kberrmsg
Last Modified: 08-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Server version 3.1 
- Microsoft Windows NT Workstation version 3.1 
- Microsoft Windows NT Advanced Server, version 3.1 
- Microsoft Windows NT Server version 3.5,3.51 
- Microsoft Windows NT Workstation version 3.5,3.51 
- Microsoft Windows NT Server versions 3.5, 3.51 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run Windows NT on an Alpha-based computer, your computer may stop
responding (hang) and you may receive the following error message:

  STOP: 0xFACEFEED

CAUSE
=====

The Alpha Hardware Abstraction Layer (HAL) generates this error when it
encounters a hardware failure.


RESOLUTION
==========

Contact your hardware vendor to inquire about a possible update to address this
issue.

Additional query words: 3.10 DEC
