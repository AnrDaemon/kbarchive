Q263708: SMS: Event ID 7016 with Remotely Possible or ControlIT Software

Article: Q263708
Product(s): Microsoft Systems Management Server
Version(s): winnt:2.0
Operating System(s): 
Keyword(s): kbsms200
Last Modified: 20-OCT-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Systems Management Server version 2.0 
-------------------------------------------------------------------------------

SUMMARY
=======

When you perform a hardware inventory on a system that has Remotely Possible or
ControlIT software installed, the system event log may log the following
message:

  Event ID: 7016
  Source: Service Control Manager
  Type: Error
  Category: None
  Description:
  The ControlIT service has reported an invalid current state 0.

MORE INFORMATION
================

The event ID does not affect the functionality of Systems Management Server
2.0.

Remotely Possible has been renamed ControlIT. Information about ControlIT and
Computer Associates can be found at the Computer Associates Web site:

  http://www.cai.com

Computer Associates International, Inc. (CA) acquired Avalan Technology, Inc.,
and its product, the Remotely Possible family of remote control software, in
November, 1997.
