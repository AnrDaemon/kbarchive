Q28226: DOCERR: Division Rather than Multiplication

Article: Q28226
Product(s): Microsoft Macro Assembler
Version(s): 5.0,5.1,5.1a
Operating System(s): 
Keyword(s): 
Last Modified: 06-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Macro Assembler (MASM), versions 5.0, 5.1, 5.1a 
-------------------------------------------------------------------------------

SUMMARY
=======

On Page 315 of the "Microsoft Macro Assembler 5.0 Programmer's Guide" and on
page 315 of the "Microsoft Macro Assembler 5.1 Programmer's Guide" the last
sentence of the first paragraph incorrectly states "After the multiplication the
result..."

The sentence should read "After the division the result..."

Additional query words: 5.00 5.10 5.10a
