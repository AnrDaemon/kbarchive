Q131656: MAC ANCIENT LANDS: Installs QuickTime Musical Instruments

Article: Q131656
Product(s): Microsoft Home Multimedia Titles
Version(s): MACINTOSH:1.0
Operating System(s): 
Keyword(s): 
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Ancient Lands for the Macintosh, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Ancient Lands Installer installs the QuickTime Musical Instruments
extension.

MORE INFORMATION
================

The QuickTime Musical Instruments extension is not required for Ancient Lands to
operate normally. The extension can be removed from your computer.
