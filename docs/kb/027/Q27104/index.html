Q27104: DOCERR: LODS Adjusts the SI Register, Not the DI Register

Article: Q27104
Product(s): Microsoft Macro Assembler
Version(s): 5.0,5.1,5.1a
Operating System(s): 
Keyword(s): 
Last Modified: 06-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Macro Assembler (MASM), versions 5.0, 5.1, 5.1a 
-------------------------------------------------------------------------------

SUMMARY
=======

Page 374 in the "Microsoft Macro Assembler 5.0 Programmer's Guide" and page 374
in the "Microsoft Macro Assembler 5.1 Programmer's Guide" incorrectly state
"LODS does adjust the DI register...".

The sentence should read "LODS does adjust the SI register..."

Additional query words: 5.00 5.10 5.10a
