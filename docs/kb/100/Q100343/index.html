Q100343: MS-DOS Editor Does Not Print to Laser Printer

Article: Q100343
Product(s): Microsoft Disk Operating System
Version(s): MS-DOS:5.0,5.0a,6.0,6.2,6.21,6.22
Operating System(s): 
Keyword(s): 
Last Modified: 26-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MS-DOS operating system versions 5.0, 5.0a, 6.0, 6.2, 6.21, 6.22 
-------------------------------------------------------------------------------

SYMPTOMS
========

MS-DOS Editor (and other MS-DOS-based applications) may not print correctly to
laser printers. This problem occurs when MS-DOS Editor doesn't force the last
page to print. As a result, if you print a one-page print job, it appears that
nothing prints.

WORKAROUND
==========

To print the last page, do one of the following:

- Press the page-feed button on the printer.

- Press CTRL+ALT+PRINTSCREEN.

- The printer receives another print job.
