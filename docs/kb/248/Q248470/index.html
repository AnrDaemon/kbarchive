Q248470: Fighter Ace II: Minimum System Requirements

Article: Q248470
Product(s): Microsoft Home Games
Version(s): 
Operating System(s): 
Keyword(s): igz kbimu msgame
Last Modified: 09-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Fighter Ace II 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the minimum system requirements needed to run Microsoft
Fighter Ace II.

MORE INFORMATION
================

To run Fighter Ace II, you need:

- A Multimedia PC with a Pentium 200 MHz or faster microprocessor (Pentium II
  300 MHz or faster microprocessor recommended).

- Microsoft Windows 95 or Microsoft Windows 98 operating system (Microsoft
  Windows 98 operating system recommended).

- 32 megabytes (MB) of RAM (64 MB recommended).

- 100 MB of available hard disk space.

- Microsoft DirectX 6.1 or later.

- A Super VGA video adapter and a Super VGA monitor capable of supporting
  16-bit color at a screen resolution of 800 x 600 pixels (Super VGA monitor
  capable of supporting 16-bit color at a resolution of 1024 X 768 pixels
  recommended).

- A DirectX 6.1 compatible sound card with speakers or headphones for sound
  playback is recommended.

- A Microsoft mouse or compatible pointing device.

- Force Feedback peripherals are compatible with DirectX 6.1.

- 28,800 bits-per-second (bps) or faster modem or a local area network (LAN)
  with TCP/IP or IPX protocol.

- Microsoft Internet Explorer 4.x, America Online (AOL) 3.0 or later, Zone.com
  2.0 or later, or Netscape Communicator 4.x or later.
