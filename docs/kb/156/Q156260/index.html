Q156260: Task Manager Performance Tab Not Displayed Correctly

Article: Q156260
Product(s): Microsoft Windows NT
Version(s): 4.0
Operating System(s): 
Keyword(s): kbtool kbui
Last Modified: 08-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Server version 4.0 
- Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run Task Manager in a normal window, data in the Physical Memory and
Kernel Memory areas may be truncated. This data is displayed correctly if you
maximize Task Manager.

CAUSE
=====

This behavior can occur if you select the Large Fonts option in the Display tool
in Control Panel.

RESOLUTION
==========

To resolve this problem, use one of the following methods:

- Select the Small Fonts option in the Display tool in Control Panel.

- Maximize Task Manager when you view the Performance tab.
