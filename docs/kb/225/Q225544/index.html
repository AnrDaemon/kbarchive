Q225544: FTP Messages Cannot Contain the Plus Symbol (+) Using HTMLA

Article: Q225544
Product(s): Internet Information Server
Version(s): winnt:5.0
Operating System(s): 
Keyword(s): 
Last Modified: 22-OCT-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Internet Information Services version 5.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use the HTMLA Web Administration forms to administer an Internet
Information Services (IIS) 5.0 FTP Server, in the "Welcome," "Exit," and
"Maximum Connection" messages, a plus symbol (+) may be saved as a space.

WORKAROUND
==========

You can use the Internet Services Manager's MMC interface to set these messages
if a plus symbol (+) is necessary for you FTP site.

STATUS
======

Microsoft has confirmed this to be a problem in Internet Information Services
5.0.
