Q59685: CTRL+ESC Not Working with Hotkeys in Windows 3.0

Article: Q59685
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a
Operating System(s): 
Keyword(s): 
Last Modified: 14-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SYMPTOMS
========

In Microsoft Windows 3.0, sometimes CTRL+ESC does not work if used in an MS- DOS
application that has been assigned a hotkey.

WORKAROUND
==========

Switch to another application and use the CTRL+ESC key sequence.

STATUS
======

Microsoft has confirmed this to be a problem in Windows version 3.0. We are
researching this problem and will post new information here as it becomes
available.

Additional query words: 3.00 win30 3.00a
