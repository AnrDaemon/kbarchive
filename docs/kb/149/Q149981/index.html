Q149981: Baseball Daily No Longer Offered

Article: Q149981
Product(s): Microsoft Home Multimedia Titles
Version(s): 1994 edition; WINDOWS:
Operating System(s): 
Keyword(s): 
Last Modified: 16-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Complete Baseball for Windows, version 1994 edition 
- Microsoft Complete Baseball Guide for Windows, 1995 edition 
-------------------------------------------------------------------------------

SUMMARY
=======

Baseball Daily has been discontinued. The service is not offered for the 1996
season.

MORE INFORMATION
================

Baseball Daily was a feature offered with the 1994 and 1995 editions of Complete
Baseball. It was an extra-cost option that was valid for one year for each
product.

Because of the wealth of baseball information available over the Internet and
other online networks, such as MSN, The Microsoft Network, Baseball Daily is no
longer offered as an addition to Complete Baseball.
