Q149191: XADM: Maximum Number of Recipients in a Distribution List

Article: Q149191
Product(s): Microsoft Exchange
Version(s): winnt:4.0
Operating System(s): 
Keyword(s): kbusage
Last Modified: 22-MAR-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, version 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The number of recipients in a Distribution List (DL) is not limited within
Microsoft Exchange. Microsoft has confirmed that as many as 6000 recipients can
be contained within a DL. The expansion time of such a DL can be long, however.
DLs can also be nested within other DLs.
