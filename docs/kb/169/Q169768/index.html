Q169768: XFOR: Exchange Connector for AppleTalk Mail Version 5.0

Article: Q169768
Product(s): Microsoft Exchange
Version(s): WinNT:5.0
Operating System(s): 
Keyword(s): 
Last Modified: 13-MAY-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, version 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft Exchange Server 5.0 does not ship with the Microsoft Exchange
Connector Series - Microsoft Mail Connector Setup for AppleTalk and Compatible
Networks disk.

MORE INFORMATION
================

The AppleTalk files can be down loaded from the following location:

  ftp://ftp.microsoft.com/bussys/exchange/exchange-public/tools/macconn.hqx

When the file has been downloaded you can use any SIT expander (such as Stuffit
Expander) to expand the file.

Additional query words: ATMTA, MAC, Appletalk, Connector
