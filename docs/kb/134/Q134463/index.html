Q134463: PPT7: System Requirements for PowerPoint for Windows 95

Article: Q134463
Product(s): Microsoft PowerPoint for Windows
Version(s): WINDOWS:7.0
Operating System(s): 
Keyword(s): kbsetup
Last Modified: 21-MAR-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft PowerPoint for Windows 95, version 7.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The following are the system requirements for PowerPoint for Windows 95, version
7.0:

- 386DX processor or higher (486 recommended)

- Windows 95 or Windows NT 3.51

- 6 MB RAM for Windows 95; 12 MB RAM for Windows NT

- CD-ROM or 3.5-inch HD disk drive

- VGA or higher (256 or more colors recommended)

- Microsoft Mouse or compatible pointing device

- Network with TCP/IP protocol for Presentation Conferencing

- Audio board with headphones or speakers required for sound functionality

MORE INFORMATION
================

The amount of hard disk space required to install PowerPoint for Windows 95
depends on the type of installation you choose. Hard disk space requirements are
as follows:

     Workstation installation             3 MB*
     Compact (laptop) installation       14 MB
     Typical installation                24 MB
     Custom (maximum) installation       35 MB

*NOTE: In a Workstation installation, PowerPoint is run from a server location on
a network. Only certain necessary files are written to your local hard drive.
