SUMMARY
=======

The following is the directory listing of the version 3.5 Microsoft Mail for PC
Networks, Macintosh Workstation 3.5-inch [1.44MB] Disk.

<H3>Disk 1 - Apple Macintosh Workstation Version</H3>
<BR/><BR/>
Name                         Size      Last Modified
------------------------------------------------------------------

Microsoft Mail Application    420K       Thu, May 25, 1995, 1:31 PM
Microsoft Mail Notifier       41K       Fri, Mar 17, 1995, 3:42 PM

NOTE: This is a Macintosh version of the Microsoft Mail client. This will only
operate on Apple Macintosh hardware and is designed for Macintosh client
workstations that are connected to a Microsoft Mail for PC Networks postoffice.
