Q124581: Explorapedia: Minimum System Requirements

Article: Q124581
Product(s): Microsoft Home Kids Products
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbfaq
Last Modified: 29-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Explorapedia series: World of Nature for Windows, version 1.0 
- Microsoft Explorapedia series: World of People for Windows, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the minimum system requirements for the Microsoft
Explorapedia series.

MORE INFORMATION
================

The minimum system requirements to run Explorapedia are:

- 486SX or higher microprocessor.

- 4 MB of RAM (8 MB recommended).

- 8 MB of available hard disk space.

- Double-speed (or higher) CD-ROM drive.

- Super VGA video adapter with a 256-color monitor.

- Audio card.

- Microsoft Windows version 3.1 or later.

- MS-DOS operating system version 3.1 or later.

- Microsoft Mouse or other compatible pointing device.

- Speakers or headphones.
