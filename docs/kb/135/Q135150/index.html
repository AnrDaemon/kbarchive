Q135150: GP Fault in Beginning Reading When Clicking Jack

Article: Q135150
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:95
Operating System(s): 
Keyword(s): 
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you click Jack several times in the Story Reading module of the Beginning
Reading program by Sierra, the program may generate a general protection (GP)
fault error message.

CAUSE
=====

The Beginning Reading program is not completely compatible with Windows 95.

RESOLUTION
==========

Do not click Jack several times in quick succession.
