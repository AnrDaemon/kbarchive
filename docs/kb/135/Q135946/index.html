Q135946: BITFAT Blue Screen Error with Compressed Floppy Disk

Article: Q135946
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 18-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
- Microsoft Plus! for Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use a floppy disk that was compressed with DriveSpace 3, you may
receive the following error message:

  Drvspace Error: Drive A: (F:\DRVSPACE.000). An error occurred while reading
  or writing the bitfat on this drive. To fix the problem, run ScanDisk and
  perform a thorough test of drive A and its host drive F.

CAUSE
=====

This error message may occur because of a misalignment of the heads on the
floppy drive or because of a problem with the floppy disk.


RESOLUTION
==========

The error message may be safely ignored. If you continue to receive the error
message, use one of the following resolutions:

- Reformat the floppy disk.

  -or-

- Have the floppy drive checked by your hardware manufacturer.
