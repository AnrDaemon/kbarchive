Q135921: Windows NT 3.51 Hangs On Dell 4XXDL, DM, or DMT Computer

Article: Q135921
Product(s): Microsoft Windows NT
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 08-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation version 3.51 
- Microsoft Windows NT Server version 3.51 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you upgrade from Windows NT 3.5 to Windows NT 3.51 on a Dell 4XXDL, DM, or
DMT computer (with the Dell Neon system board), one or more of the following
symptoms occur:

- Your computer stops responding (hangs).

- Program Manager appears without program groups.

- The screen displays black, horizontal stripes.

CAUSE
=====

Your computer has a Cirrus 542x video adapter possibly causing a system cache
conflict.

RESOLUTION
==========

To work around this problem, install the Windows NT VGA Compatible Display
Adapter driver.

To correct this problem, contact Dell Computers Technical Support at (800)
624-9896.
