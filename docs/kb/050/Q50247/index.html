Q50247: Mac Srv: Sleep Mode on Macintosh Portable Turns Server Off

Article: Q50247
Product(s): Microsoft Mail For Appletalk Networks
Version(s): WINDOWS:2.0,2.0a,2.0b,3.0
Operating System(s): 
Keyword(s): 
Last Modified: 09-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

If you are using a Macintosh Portable as a server for Microsoft Mail and the
machine goes into sleep mode, the server's ID on the network disappears.

This problem causes users logged in to see the Give Up box and forces them to
quit Mail by clicking the Give Up button.

To correct this problem, turn off sleep mode when using the Portable as a server
on the network, or use a different Macintosh as the server.

Additional query words: 2.00 2.00a 2.00b 3.00
