Q84124: Panasonic KX-P1180 Driver with WordPerfect for Windows

Article: Q84124
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 01-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

The Panasonic KX-P1180 driver for Windows 3.1 does not properly generate lines
and other graphics when used with WordPerfect for Windows. Although lines and
graphics display correctly using the Print Preview command, output to the
printer is not correct.

The KX-P1180 printer driver generates correct graphics when used with other
applications, such as Paintbrush.

a vendor independent of Microsoft; we make no warranty, implied or otherwise,
regarding this product's performance or reliability.

For additional information about WordPerfect, please contact WordPerfect
Corporation technical support at (800) 228-1029.
