Q84118: TrueType Fonts Not Printing Correctly on Panasonic KX-P4450i

Article: Q84118
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 08-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------


SUMMARY
=======

To print TrueType fonts on a Panasonic KX-P4450i from Microsoft Windows, you
must first select the Print TrueType As Graphics check box in the Windows
Control Panel, as described below.

NOTE: The information in this article applies to the Panasonic KX-P4450i, not the
KX-P4450.

MORE INFORMATION
================

To set the Print TrueType As Graphics option:

1. Run Control Panel.

2. Choose the Printers icon.

3. Choose the Setup button.

4. Choose the Options button.

5. Select the Print TrueType As Graphics check box.

This setting is required due to a firmware problem in the Panasonic KX-P4450i
version 1.3 through 2.0.

Use the following printer drivers for the KX-P4450 series printers:

- KX-P4450i: Use the HP LaserJet Series II driver

- KX-P4450: Use the HP LaserJet Plus driver

The KX-P4450 and KX-P4450i are manufactured by Panasonic, a vendor independent of
Microsoft; we make no warranty, implied or otherwise, regarding these products'
reliability or performance.
