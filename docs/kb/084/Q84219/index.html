Q84219: SYSTEM.INI [386enh] Entries Not Removed When Driver Is Removed

Article: Q84219
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 11-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you use the Control Panel Drivers option to remove a driver, the driver is
removed, but any entries in the [386enh] section of the SYSTEM.INI file are not
removed.

RESOLUTION
==========

To remove these drivers, use SysEdit, Notepad, or any other text editor to
manually delete the entries.

Additional query words: 3.10 3.11
