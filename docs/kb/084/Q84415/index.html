Q84415: DrawPerfect 1.0 Mouse Problems in Window Mode

Article: Q84415
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 13-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

DrawPerfect version 5.1 performs its own internal mouse acceleration.
If DrawPerfect is run in a window under Microsoft Windows operating
system version 3.1, the mouse pointer in the DrawPerfect window may
not match the mouse pointer in Windows.

Additional query words: 3.10 3.11 3rdparty
