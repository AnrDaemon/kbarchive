Q158056: WD97: Quick View Doesn’t Convert Word 97 Documents

Article: Q158056
Product(s): Word 97 for Windows
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kbinterop word97
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to use Quick View to read a Microsoft Word 97 document, the
title bar reflects the name of the document, but the Quick View window does not
show the contents of the document.

CAUSE
=====

Quick View uses a text converter that cannot read Word 97 documents.

WORKAROUND
==========

To work around this problem in Quick View, on the File menu, click Open File for
Editing. The file should now open in Microsoft Word 97 for Windows.


STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Windows 95. This
problem was corrected in Microsoft Windows 98.


MORE INFORMATION
================

Quick View, which is installed by Microsoft Windows 95, allows you to view
documents that are created by Windows-based programs without starting the
program. If Quick View is not available on the shortcut menu, either the program
that created your file does not support Quick View, or Quick View is not
installed on your computer. Please see your Windows 95 documentation for
information about how to install Quick View on your computer.

For additional information, please see the following articles in the Microsoft
Knowledge Base:

  Q154907 How to Enable or Disable Quick View


  Q158748 OFF97: Unable to Preview File Using Quick View


  Q186014 WD97: Setting Up QuickView to Preview Word Files w/ Word Viewer
