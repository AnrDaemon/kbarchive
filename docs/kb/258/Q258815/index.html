Q258815: Wave Audio Volume Is Unexpectedly Raised

Article: Q258815
Product(s): Microsoft Home Multimedia Titles
Version(s): 
Operating System(s): 
Keyword(s): kbsound kbimu
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Encyclopedia Deluxe 2001 for Windows 
- Microsoft Encarta Encyclopedia Standard 2001 for Windows 
- Microsoft Encarta Encyclopedia 2000 
- the operating system: Microsoft Windows 98 
- Microsoft Bookshelf 2000 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you open or close Encarta Explorer in Microsoft Encarta Encyclopedia 2000,
the volume level of Wave Audio and Wave/Direct Sc sounds is unexpectedly
raised.


CAUSE
=====

This behavior can occur if a Creative Labs SoundBlaster Live! sound card is
installed in your computer.

RESOLUTION
==========

To work around this issue, reduce the volume level:

1. Double-click the Volume Control icon on the taskbar.

2. Under Wave Balance or Wave/Direct Sc Balance, move the Volume slider down to
  reduce the volume level.

3. Close the Volume Control dialog box.
