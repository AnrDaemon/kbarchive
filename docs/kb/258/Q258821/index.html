Q258821: MapPoint 2001: Map Is Not Printed Properly

Article: Q258821
Product(s): Microsoft Automap
Version(s): 1.0
Operating System(s): 
Keyword(s): kbprint kbimu
Last Modified: 12-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MapPoint 2002 
- Microsoft MapPoint 2001 
- Microsoft Streets & Trips 2002, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you print a map from Microsoft MapPoint 2001, you may encounter any of the
following symptoms:

- The printed map may be incomplete.

- Unexpected characters may be printed on the map.

- The printed map may not spool to the printer properly.

CAUSE
=====

This behavior can occur if 1 megabyte (MB) or less of memory is installed in the
printer.


STATUS
======

Microsoft has confirmed that this is a problem in Microsoft MapPoint 2001.

Additional query words: mp2001 map point printout random symbols missing
