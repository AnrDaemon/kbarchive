Q255969: Allegiance: Some Graphics Are Not Displayed Correctly

Article: Q255969
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kbdisplay igz kbimu msgame
Last Modified: 12-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Allegiance, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you play Microsoft Allegiance, some of the graphics on the screen may not
be displayed correctly.

CAUSE
=====

This behavior can occur if a video adapter based on the 3DLabs Permedia2 chip
set is installed in your computer.

The 3DLabs Permedia2 chip set does not support all of the graphics options used
by Allegiance.

RESOLUTION
==========

To resolve this issue, disable the graphics options that are not supported by
your video adapter:

1. On the Main Menu screen, click Options, and then point to Graphics Options.

2. Click Environment On to change it to Environment Off.

3. Click Posters On to change it to Posters Off.

4. Click anywhere outside of the menu to close it.
