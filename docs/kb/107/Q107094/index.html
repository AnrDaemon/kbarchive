Q107094: PROFS: Transferring Reader Files with CMS Transfer

Article: Q107094
Product(s): Microsoft Mail For PC Networks
Version(s): 3.2,3.3,3.4
Operating System(s): 
Keyword(s): 
Last Modified: 12-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail Gateway to IBM PROFS and OfficeVision, versions 3.2, 3.3, 3.4 
-------------------------------------------------------------------------------

SUMMARY
=======

Here is the command line format for using CMS Transfer to move a file from a
virtual machine ID's reader to RSCS:

  TRANSFER RDR rdr_file_# TO RSCS


Additional query words: 3.20 3.2 3.30 3.3 3.40 3.4
