Q76800: Description of a Multimedia PC

Article: Q76800
Product(s): Miscellaneous Windows Products
Version(s): 1.0; WINDOWS:3.1
Operating System(s): 
Keyword(s): 
Last Modified: 31-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows with Multimedia Extensions, version 1.0 
- Microsoft Windows 3.1 
-------------------------------------------------------------------------------

SUMMARY
=======

Multimedia PCs are a specific class of PCs that meet or exceed the Multimedia PC
specification 1.0. The "MPC" trademark indicates that the hardware complies with
this specification.

MORE INFORMATION
================

The Multimedia PC combines high quality graphics, stereo audio and animation
capabilities with the massive storage capacity of CD-ROM.

The minimum hardware requirements are listed below:

- 80386sx microprocessor

- 2 megabytes (MB) or more of Random Access Memory (RAM)

- VGA (16-color) or VGA (256-color) display

- 30 MB hard drive or larger

- CD-ROM drive able to output digital audio

- Audio hardware able to process music and voice

Multimedia PCs fully support Windows with Multimedia Extensions version 1.0 and
Windows version 3.1.
