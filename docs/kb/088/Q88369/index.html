Q88369: PC Win: Starting Mail for Windows 3.0 Minimized

Article: Q88369
Product(s): Microsoft Mail For PC Networks
Version(s): WINDOWS:3.0
Operating System(s): 
Keyword(s): 
Last Modified: 16-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail for Windows, version 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

It is possible to start Microsoft Mail for Windows minimized, with no user input
required.

The following procedure assumes you are using Microsoft Windows 3.1:

1. Place the icon for Mail for Windows in the Startup group.

2. To edit the File Properties, hold down the ALT key and double-click the Mail
  icon.

3. Select the Run Minimized option box.

4. At the end of the command line, add one space and then the following

  " /p<password>" (without the quotation marks)

  (where <password> represents your unique password for Mail access).

5. Choose the OK button.

Additional query words: 3.00
