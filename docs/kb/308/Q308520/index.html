Q308520: Games: Computer Appears to Hang Running Setup

Article: Q308520
Product(s): Microsoft Home Games
Version(s): 1.0,2.0
Operating System(s): 
Keyword(s): kbimu
Last Modified: 17-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Age of Empires II: The Age of Kings, version 2.0 
- Microsoft Crimson Skies 
- Microsoft Flight Simulator 2002 
- Microsoft Flight Simulator 2002 Professional Edition 
- Microsoft Train Simulator, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to install any of the products listed above, your computer
appears to stop responding (hang). You receive an error message similar to the
following:

  Setup engine is initializing.

CAUSE
=====

This behavior can occur because your computer takes a long time to start the
Setup program.

MORE INFORMATION
================

On some computers, the Setup program is slow. In some cases, Setup can take
twenty minutes to an hour. Please allow at least an hour at this stage of
installation.

Additional query words: msgame
