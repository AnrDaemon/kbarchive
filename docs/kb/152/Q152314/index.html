Q152314: Mac Bookshelf 96-97: About Box Shows Odd Hard Disk Space Info

Article: Q152314
Product(s): Microsoft Home Multimedia Titles
Version(s): MACINTOSH:
Operating System(s): 
Keyword(s): 
Last Modified: 16-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Bookshelf 1996-97 for Macintosh 
-------------------------------------------------------------------------------

SUMMARY
=======

When you click About Microsoft Bookshelf, located on the Apple menu, the
information about Disk Space is not what you expect.

MORE INFORMATION
================

The Disk Space information varies according to where you started Bookshelf from.
If you started Bookshelf from the icon on your hard drive, the value shown for
Disk Space value represents the free disk space available on your hard drive.

If you started Bookshelf from the compact disc itself, the Disk Space value is
approximately 90K.

If you launched Bookshelf from a server, the Disk Space value represents whatever
space is available on the server's hard drive.
