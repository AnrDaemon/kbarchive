Q161531: Wine Guide 2.0: Blank Box on Printout When Printing Contents

Article: Q161531
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:2.0
Operating System(s): 
Keyword(s): 
Last Modified: 17-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Wine Guide for Windows, version 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you print the Wine Guide Contents screen using a HP Laser Jet 4+ printer, a
blank box prints in the lower left corner of the page.

CAUSE
=====

This occurs if you are using one of the following printer drivers:

- HP LaserJet 4 Plus/4M Plus Printing System, version: 31.V1.30

- HPPCL5E Printer Driver For Windows 3.1, version: 31.V1.30

- HP Status Window, version: 2.10


RESOLUTION
==========

Obtain an updated printer driver from the hardware manufacturer.

The third-party products discussed here are manufactured by vendors independent
of Microsoft; we make no warranty, implied or otherwise, regarding these
products' performance or reliability.
