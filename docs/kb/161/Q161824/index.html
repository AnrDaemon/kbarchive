Q161824: WD97: Can’t Make First Letter of Sentence a Capital Letter

Article: Q161824
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbualink97 kbusage word97
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

Word may not automatically capitalize the first sentence of a paragraph, even
when the "Capitalize first letter of sentences" option is selected (click
AutoCorrect on the Tools menu, and then select the AutoCorrect tab).

CAUSE
=====

This problem may occur when the first sentence of a paragraph follows any of the
following:

- A form field.

  -or-

- A hyperlink

  -or-

- A number (or letter) followed by punctuation (for example, a period).

WORKAROUND
==========

To work around this problem, manually capitalize the first letter of the
sentence with the SHIFT key.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article.
