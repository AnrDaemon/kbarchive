Q126722: Using Floptical Disk Drives with Windows 95

Article: Q126722
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 13-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

SUMMARY
=======

If you are using a floptical disk drive with Windows 95, you should note the
following items:

- In Windows 95, floptical disk drives are supported only by real-mode ASPI
  drivers.

- Floptical disk drives controlled by the BIOS are not supported.

- You must format floptical disks from an MS-DOS command prompt using a
  vendor-supplied formatting tool (such as C Format or EZSCSI). Windows 95 will
  recognize only partitioned media.
