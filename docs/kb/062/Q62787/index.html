Q62787: PageMaker Versions Compatible with Windows 3.0 and 3.1

Article: Q62787
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a,3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 13-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

The Microsoft Windows version 3.0 Applications Reference List states that Aldus
PageMaker version 3.0 is compatible with Windows version 3.0. You must have
PageMaker version 3.01 to run with either standard mode or 386 enhanced mode
Windows version 3.0.

PageMaker 3.01 or later is required to run with Windows versions 3.1 and 3.11.

MORE INFORMATION
================

The most current release of Aldus PageMaker is 5.0. For more information about
PageMaker, call Aldus technical support.

PageMaker is manufactured by Aldus Corporation, a vendor independent of
Microsoft; we make no warranty, implied or otherwise, regarding this product's
performance or reliability.
