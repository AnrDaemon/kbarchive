Q62152: Gates Learjet 25G Performance Specifications

Article: Q62152
Product(s): Microsoft Home Games
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 30-JUL-2001

SUMMARY
=======

The following is a summary of the performance specifications for the Gates
Learjet 25G in Flight Simulator version 4.00:

  Feature                        Specification
  -------                        -------------

  Type designation               LR-25G
  Engines                        General Electric CJ610-8A,
                                    2950 pounds each
  Seats                          10
  Length                         47 feet 7 inches
  Height                         12 feet 3 inches
  Wing span                      35 feet 7 inches
  Wing area                      247 square feet
  Wing aspect ratio              5.5
  Maximum ramp weight            16,200 pounds
  Maximum takeoff weight         16,200 pounds
  Standard empty weight          8250 pounds
  Empty weight as tested         8616 pounds
  Maximum useful load            8550 pounds
  Zero-fuel weight               11,400 pounds
  Maximum landing weight         13,700 pounds
  Wing loading                   66 pounds per square foot
  Maximum altitude               51,000 feet
  Power loading                  12.8 pounds per pound
  Maximum usable fuel            6638 pounds
  Maximum pressurization
     differential                9.4 pounds per square inch
  8000-foot cabin altitude at    51,000 feet
  Normal cruise at 45,000 feet   445 knots
  High speed cruise at
     41,000 feet                 464 knots
  Fuel flow at normal cruise
     (as tested)                 1370 pounds per hour
  Stall speed clean              118 knots (14,500 pounds)
  Stall speed full flaps         102 knots (14,500 pounds)
  Stall speed flaps/gear down    99 knots
