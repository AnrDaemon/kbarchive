Q138247: List of Documentation Errors in Windows 95 Readme.txt File

Article: Q138247
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists known documentation errors in the Microsoft Windows 95
Readme.txt file.

The following known documentation errors are described in this article:

- If you purchased Windows 95 on floppy disks

MORE INFORMATION
================

If You Purchased Windows 95 on Floppy Disks
-------------------------------------------

Step 1 of the "If you purchased Windows 95 on floppy disks" section of the
Windows 95 Readme.txt file states: "Insert Disk 1 into drive A."

Step 1 should state: "Insert Disk 2 into drive A."
