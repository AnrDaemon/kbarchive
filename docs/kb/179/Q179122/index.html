Q179122: FS95/98/2000: How to Start and Shut Down Jet Engines

Article: Q179122
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbui kbusage fs98 fs6 kbimu msgame
Last Modified: 07-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Flight Simulator 2000 
- Microsoft Flight Simulator 98 
- Microsoft Flight Simulator for Windows 95, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article discusses how to start and shut down jet engines in Microsoft
Flight Simulator.

MORE INFORMATION
================

To start and shut down jet engines in Flight Simulator, use the following
methods:

Start Jet Engines
-----------------

To start jet engines in Flight Simulator:

1. Press M, and then press the PLUS SIGN (+) key at the top of the keyboard to
  start the magnetos.

  NOTE: Do not press the PLUS SIGN (+) key on the numeric keypad.

2. Press CTRL+SHIFT+F4 to start the fuel flow.

3. Press J, and then press the PLUS SIGN (+) key at the top of the keyboard to
  start the engines.

  NOTE: Do not press the PLUS SIGN (+) key on the numeric keypad.

NOTE: You may be able to start the engines even if you use these steps in a
different order. Depending on conditions in the simulation, you may not need to
complete all of these steps to start the engines.

Shut Down Jet Engines
---------------------

To shut down jet engines in Flight Simulator, press CTRL+SHIFT+F1. After a short
pause, the engines shut down.

NOTE: These keyboard commands work for the most common keyboard configurations.
If these commands do not work using your keyboard, consult your keyboard
documentation to find the appropriate commands for your keyboard.
