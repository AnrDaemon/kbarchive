Q179917: Save and Save As Options Dimmed on File Menu of 5250 Applet

Article: Q179917
Product(s): Microsoft SNA Server
Version(s): WINDOWS:3.0,3.0 SP1,3.0 SP2,4.0,4.0 SP1
Operating System(s): 
Keyword(s): 
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft SNA Server, versions 3.0, 3.0 SP1, 3.0 SP2, 4.0, 4.0 SP1 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use the 5250 applet provided with SNA Server, the Save and Save As
options on the File menu are dimmed when a successful session is established.
They remain dimmed as long as the applet is running, even if the session is
disconnected.

WORKAROUND
==========

To restore the options, the applet must be shut down and restarted.

STATUS
======

Microsoft has confirmed this to be a problem in SNA Server versions 3.0, 3.0
SP1, 3.0 SP2, 4.0, and 4.0 SP1. We are researching this problem and will post
new information here in the Microsoft Knowledge Base as it becomes available.
