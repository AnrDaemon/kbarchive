Q208879: Urban Assault: Graphic Elements Are Not Displayed Correctly

Article: Q208879
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kb3rdparty kbdisplay kbimu msgame
Last Modified: 02-FEB-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Urban Assault, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you play Microsoft Urban Assault, graphic elements such as smoke, weapons
fire, buildings, and vehicles may not be displayed correctly on the screen.

CAUSE
=====

This behavior can occur if an ATI Xpert@Work video adapter is installed in your
computer.

RESOLUTION
==========

To resolve this issue, download and install the latest video driver for the ATI
Xpert@Work video adapter from the following ATI Web site:

  http://www.atitech.com/

MORE INFORMATION
================

If the issue continues to occur, you may be able to work around the issue by
disabling the 16-Bit Textures and DrawPrimitive features in Urban Assault. To do
this, follow these steps:

1. Start Urban Assualt.

2. On the startup screen, click Options.

3. Click to clear the 16-Bit Textures check box.

4. Click to clear the DrawPrimitive check box.

5. Click OK.
