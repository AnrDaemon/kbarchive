Q180454: Fonts Installed by Encarta 98 Encyclopedia for Windows

Article: Q180454
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:98
Operating System(s): 
Keyword(s): kbenv kbmm kbsetup
Last Modified: 12-AUG-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 98 Encyclopedia for Windows 
-------------------------------------------------------------------------------

SUMMARY
=======

This article lists the fonts that are installed and used by Microsoft Encarta 98
Encyclopedia for Windows.

MORE INFORMATION
================

Encarta 98 Encyclopedia installs and uses the following fonts:

  Font Name        Filename       Font Type
  -----------------------------------------

  Arial            Arial.ttf      TrueType
  MS Reference 1   Msref1.ttf     TrueType
  MS Reference 2   Msref2.ttf     TrueType
  Phonetic         Phonetic.fon   Raster
  Symbol           Symbol.ttf     TrueType

Encarta 98 Encyclopedia uses the following font, but does not install it:

  Font Name         Filename    Font Type
  ---------------------------------------

  Times New Roman   Times.ttf   TrueType

Additional query words: 98 multi multi-media media mm
