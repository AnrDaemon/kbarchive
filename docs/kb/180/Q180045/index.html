Q180045: Virtual Globe: Colored Text Is Not Printed on HP LaserJet 5L

Article: Q180045
Product(s): Microsoft Automap
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kbprint
Last Modified: 07-SEP-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta Virtual Globe 1998 edition 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you print an article from Encarta Virtual Globe to a Hewlett-Packard
LaserJet 5L printer, colored text in the article may not be printed.

RESOLUTION
==========

To resolve this issue, use one of the following methods:

Set Print Quality to Primarily Text Printing
--------------------------------------------

To set print quality to primarily text printing, follow these steps:

1. Open an article in Encarta Virtual Globe.

2. On the Options menu, click Print.

3. Click Properties.

4. On the Print Quality tab, click Primarily Text Printing, and then click OK.

5. Click OK.

Copy Unformatted Text to Another Program
----------------------------------------

To copy unformatted text from Encarta Virtual Globe to another program, follow
these steps:

1. Open an article in Encarta Virtual Globe.

2. On the Options menu, click Copy.

3. Start a word processing program, such as Microsoft WordPad.

4. On the Edit menu, click Paste Special.

5. Click Unformatted Text, and then click OK.

6. On the File menu, click Print.
