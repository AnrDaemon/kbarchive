Q51266: Air Traffic Control Communications Using Transponder

Article: Q51266
Product(s): Microsoft Home Games
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 30-JUL-2001

SUMMARY
=======

Option 7, Air Traffic Control Communications, on the NAV/COM menu allows you to
communicate with the control tower. When this option is turned on, you are
instructed to tune the transponder to a specific frequency by the air traffic
control (ATC) when requesting take-off clearance. You can also request landing
clearance.

The ENTER key is your microphone for communicating with the tower. To request
take-off clearance, press "ENTER-1-INSERT". For landing clearance, press
"ENTER-2-INSERT".
