Q51720: FltSim Hangs on CompuAdd 386/16 with Award BIOS 3.04

Article: Q51720
Product(s): Microsoft Home Games
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 30-JUL-2001

SUMMARY
=======

Using Flight Simulator Version 4.00 on a CompuAdd 386/16 equipped with an Award
BIOS 3.04 results in the program hanging when it is started.

CompuAdd Computers has confirmed this to be a problem with the Award BIOS 3.04.
CompuAdd is currently offering the Award BIOS 3.03 as a replacement. For further
information and a replacement Award BIOS, contact CompuAdd Computers at (800)
999-9901 or (512) 250-1489.
