Q260732: StarLancer: Textures in the Cockpit Flash or Flicker

Article: Q260732
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kb3rdparty kbdisplay kbimu msgame
Last Modified: 12-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft StarLancer, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you fly a mission in Microsoft StarLancer, textures in the cockpit may
flash or flicker when you pull back on the joystick.

CAUSE
=====

This behavior can occur if a Number Nine Revolution IV video adapter is
installed in your computer.

RESOLUTION
==========

To resolve this issue, download and install the latest version of the video
driver for your video adapter from the following Number Nine Web site:

  http://www.nine.com
