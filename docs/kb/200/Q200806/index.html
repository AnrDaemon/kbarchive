Q200806: SMS: Column widths in SMSMON16 cannot be resized.

Article: Q200806
Product(s): Microsoft Systems Management Server
Version(s): winnt:2.0
Operating System(s): 
Keyword(s): kbsms200 kbSoftwareDist
Last Modified: 02-MAR-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Systems Management Server version 2.0 
-------------------------------------------------------------------------------

SUMMARY
=======

The Available Programs Monitor for 16-bit clients, such as Windows 3.1, has an
icon in Control Panel that you can use to monitor whether there are any
available programs waiting to be installed.

The ability to resize a column width in this program is not available. These
columns are of a fixed length. You can, however, resize column widths in the
32-bit version of this program.

Additional query words: prodsms swdist16
