Q64685: Flight Simulator Does Not Have Serial Numbers

Article: Q64685
Product(s): Microsoft Home Games
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 30-JUL-2001

SUMMARY
=======

Serial numbers are not used in the Flight Simulator product.

The earlier registration cards requested the serial number. However, these are
generic cards, which are used with different Microsoft products, and the serial
number is needed only for owners of earlier versions of Microsoft Word.
