Q99665: PC WSPlus: Displaying the Time in a Military Time Format

Article: Q99665
Product(s): Microsoft Schedule+ for Windows
Version(s): WINDOWS:1.0,1.0a
Operating System(s): 
Keyword(s): 
Last Modified: 18-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Schedule+ for Windows, versions 1.0, 1.0a 
-------------------------------------------------------------------------------

To view version 1.0 of Microsoft Schedule+ for Windows in a military time
(24-hour) format, you must first change the Windows time format to a
military time format. Schedule+ inherits the Windows environment variables,
so Windows must first be set to display time in the desired format.

To change the time to display in a military format
--------------------------------------------------

1. From the Control Panel, choose the International icon.

2. Select the Time Format option, and choose Change

3. Select 24 Hour, and choose OK.

This will cause all Windows applications to use a 24-hour time format.
