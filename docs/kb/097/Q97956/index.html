Q97956: CONN: CNU Needs MultiFinder

Article: Q97956
Product(s): Microsoft Mail For Appletalk Networks
Version(s): 1.0,1.0b; WINDOWS:3.0,3.0a,3.0b,3.1,3.1a,3.1c,3.1d
Operating System(s): 
Keyword(s): 
Last Modified: 09-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail Connection Gateway, versions 1.0, 1.0b 
- Microsoft Mail for AppleTalk Networks, versions 3.0, 3.0a, 3.0b, 3.1, 3.1a, 3.1c, 3.1d 
-------------------------------------------------------------------------------

SYMPTOMS
========

The Connection Name Utility (CNU) of the Microsoft Mail Connection for PC and
AppleTalk Networks version 1.0 or 1.0b needs to run on a Mail server using
System 6 or System 7 with MultiFinder enabled. With System 6, operating with
Finder only may cause errors.

STATUS
======

Microsoft has confirmed this to be a problem with Microsoft Mail for AppleTalk
Networks, versions 3.0, 3.0a, 3.0b, and 3.1, 3.1a, 3.1c, and 3.1d. We are
researching this problem and will post new information here in the Microsoft
Knowledge Base as it becomes available.
