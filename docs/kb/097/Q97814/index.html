Q97814: Interfaces Allowing Direct LAN Man Connection to HP

Article: Q97814
Product(s): Microsoft LAN Manager
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 30-JUL-2001

SUMMARY
=======

For Hewlett-Packard (HP) LaserJet II, IID, III, and IIID printers that work with
LAN Man, the following are interfaces that allow direct connection:

Number         Network Type
------         ------------

C2071C         802.3 Thin EtherNet
C2071D         802.3 10Base-T
C2071F         802.5 (4 and 16 Mbps)

Install the interface in the optional I/O slot on the printer's back panel.

Reference(s):

For additional HP product information, call 208-344-4809 (in the U.S. only).

Additional query words: 2.00 2.10 2.10a 2.20
