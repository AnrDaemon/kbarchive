Q160047: 3DMM: Cannot Load Studio.chk File

Article: Q160047
Product(s): Microsoft Home Kids Products
Version(s): 1.0
Operating System(s): 
Keyword(s): 
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft 3D Movie Maker, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you start 3D Movie Maker, you may receive the following error message:

  Cannot Load Studio.chk...

3D Movie Maker installs correctly, but after the program starts this error
message appears.

RESOLUTION
==========

To troubleshoot this problem, use the following steps:

1. Clean the compact disc.

2. Ensure that the computer is running protect-mode CD-ROM drivers.

3. Remove 3D Movie Maker by following these steps:
  a. Click the Start button, point to Settings, and then click Control Panel.

  b. Double-click Add/Remove Programs.

  c. Click the Install/Uninstall tab.

  d. Select 3D Movie Maker.

4. Re-install 3D Movie Maker.

5. Run the Windows Setup program to repair damaged or missing components.

Additional query words: kids mskids artist maggie max writer cw2 cw2.0 3dmm moviemaker 3dmoviemaker 3-d three d movie-maker cd
