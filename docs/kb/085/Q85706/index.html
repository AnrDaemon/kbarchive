Q85706: Calendar Displays a Maximum of 14 Lines in Daily Schedule

Article: Q85706
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a,3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 14-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

The Calendar program included with Microsoft Windows does not display more than
14 lines in the daily schedule.

With the default interval of 60 minutes set, this equates to 14 hours of the day.
If a shorter interval is selected, still only the first 14 intervals are
displayed.

MORE INFORMATION
================

Maximizing Calendar will not increase the number of lines available in the
window. If the window is resized to a smaller dimension, the number of lines
displayed will be reduced. Use the scroll bars to view additional intervals in
the schedule.

Additional query words: 3.0 3.0a 3.1 3.11 3.10 3.00 3.00a
