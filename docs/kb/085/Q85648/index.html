Q85648: Mac DOS: Err Msg: Your Server Doesn’t Seem to Be Able to…

Article: Q85648
Product(s): Microsoft Mail For Appletalk Networks
Version(s): 2.0,3.0
Operating System(s): 
Keyword(s): 
Last Modified: 09-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail for AppleTalk Networks, MS-DOS workstation, versions 2.0, 3.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When logging into the MS-DOS client of Microsoft Mail for AppleTalk networks,
you may receive the following error message:

  Your server doesn't seem able to handle that request, OK?

CAUSE
=====

This error occurs when the 2.0 version of the MS-DOS client is trying to log in
to a 3.0 version of the AppleTalk Mail server.

RESOLUTION
==========

The workstation software, like the Macintosh workstations, must be upgraded to
version 3.0. It is not possible to combine versions of the Workstation and
Server for Microsoft Mail.

Additional query words: 2.00 3.00 Daystar login logon
