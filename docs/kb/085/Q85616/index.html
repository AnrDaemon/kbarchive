Q85616: FAX: Major Limitations of Microsoft Mail Gateway to Fax

Article: Q85616
Product(s): Microsoft Mail For PC Networks
Version(s): MS-DOS:3.0
Operating System(s): 
Keyword(s): 
Last Modified: 05-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail Gateway to Fax, version 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Here is a list of the major concerns customers have expressed about the
Microsoft Mail Gateway to Fax:

1. Inability to handle more than one fax board

2. Inability to deliver to individual mailboxes

3. Inability to view faxes from Windows mail client
