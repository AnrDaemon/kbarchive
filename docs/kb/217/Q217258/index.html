Q217258: WD97: Graphic Colors Display Incorrectly in Word and IE 5

Article: Q217258
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbdta kbwdinternet wd2000
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you use Microsoft Internet Explorer 5 to view a document or Web page
created in Word, and the document or Web page contains a graphic, the graphic's
colors may not be displayed correctly.

CAUSE
=====

This problem can occur if you are running Microsoft Internet Explorer 5 and your
screen is set to display 256 colors.



RESOLUTION
==========

To resolve this problem, change your screen settings to a higher color setting,
for example, True Color (24 bit).

To change the number of colors your monitor displays:

1. On the Windows Start menu, point to Settings, and then click Control Panel.

2. Double-click Display.

3. On the Settings tab, in Colors, click the number of colors that you want your
  monitor to display.

NOTE: Your monitor and display adapter determine the maximum number of colors
that can appear on your monitor.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.
