Q96362: PRB: "Chosen Printer Not Supported" When Page Size Is Chosen

Article: Q96362
Product(s): Microsoft Fox Miscellaneous Products
Version(s): MACINTOSH:2.01
Operating System(s): 
Keyword(s): 
Last Modified: 23-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft FoxBASE+ for Macintosh, version 2.01 
-------------------------------------------------------------------------------

SYMPTOMS
========

Choosing Page Size from the File menu in FoxBASE+/Mac produces the error message
"Chosen Printer Not Supported" when the Apple LaserWriter printer driver is
chosen.

CAUSE
=====

The LaserWriter driver does not include a Page Size dialog box.

RESOLUTION
==========

To view the available page size options for the Apple LaserWriter, choose Page
Setup from the File menu.

Additional query words: 2.01 FoxBASE+/Mac laser writer errmsg err msg
