Q298712: MSN Explorer (MSN Version 6.1) Sign-in And Sign-up Error Message

Article: Q298712
Product(s): The Microsoft Network
Version(s): 6.1
Operating System(s): 
Keyword(s): 
Last Modified: 23-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- The Microsoft Network version 6.1 
-------------------------------------------------------------------------------

SUMMARY
=======

For any of the specific sign-in and sign-up error messages below, use the
corresponding solution. In this KB article, the sign-in and sign-up error
messages are categorized by type, and are listed alphabetically.

MORE INFORMATION
================

Error Message: Sorry, we were unable to finish setting up your e-mail service.  We will try again the next time you sign in.   \r\n\r\n  Note: You cannot use e-mail until it is completely set up.
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

When a user receives this error, the user can sign in and use all other services,
but cannot be able to access e-mail through MSN Explorer. E-mail activation will
be attempted automatically the next time the user signs in. Sign out and sign
back in until this error message goes away.

Error Message: Sorry, this is not a valid MSN Internet Access e-mail name.  Please check that you entered the correct e-mail name and try again.
------------------------------------------------------------------------------------------------------------------------------------------------

To resolve this issue

1. Retype the user name and password.

  This error is usually caused by a typo. Make sure the user is not typing
  @msn.com in the username field.
