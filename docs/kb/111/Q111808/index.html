Q111808: COMMAND+<Number> Combinations in FoxPro for Macintosh

Article: Q111808
Product(s): Microsoft FoxPro
Version(s): MACINTOSH:2.5b,3.0b
Operating System(s): 
Keyword(s): 
Last Modified: 08-FEB-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Visual FoxPro for Macintosh, version 3.0b 
- Microsoft FoxPro for Macintosh, version 2.5b 
-------------------------------------------------------------------------------

SUMMARY
=======

COMMAND key+<number> assignments are reserved to mimic function key
operations on the Macintosh.

MORE INFORMATION
================

Many Macintosh computers do not have keyboards containing function keys (F1
-F15). In order to give these machines the same functionality, the
COMMAND+<number> key combination is typically reserved.

This means that in FoxPro for Macintosh, these key assignments, if used as
shortcut keys in menus, will yield a key label and key text of F<#>.
