Q111063: Install Add-On Runs Setup for Other Programs

Article: Q111063
Product(s): Microsoft Home Games
Version(s): MS-DOS:5.0
Operating System(s): 
Keyword(s): 
Last Modified: 07-MAY-2002

kbsetup

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Flight Simulator for MS-DOS, version 5.0 
-------------------------------------------------------------------------------

SUMMARY
=======

When you run the Setup program for Flight Simulator 5.0, selecting the Install
Add-On option may start the Setup program for other applications.

MORE INFORMATION
================

The Install Add-On option is designed to run SETUP.EXE on the designated drive.
If it cannot run SETUP.EXE on the designated drive for any reason, it will
search the active path for SETUP.EXE. Many programs, including Microsoft
Windows, use a setup utility with the filename SETUP.EXE.
