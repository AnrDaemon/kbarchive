Q61768: Hewlett-Packard Global Text Font Cartridge Description

Article: Q61768
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:2.x,3.0,3.0a,3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 18-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 2.03, 2.1, 2.11, 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

Below is a description of the capabilities of the Hewlett-Packard (HP) Global
Text Font Cartridge.

The font cartridge supports a print resolution of 300 x 300 dots per inch (dpi)
and has an average lifetime of 500 insertions. Separate fonts are required for
landscape and portrait. Some fonts may not be available in both orientations on
a specific cartridge.

For more information about this font cartridge, please contact Hewlett-Packard.

MORE INFORMATION
================

The following explains Global Text Cartridge capabilities:

--------               ----    -------   -----        -----  ----------

CG Century Schoolbook  Medium  P and L   Proportional  8      *

CG Century Schoolbook  Medium  P and L   Proportional  10     *

CG Century Schoolbook   Bold   P and L   Proportional  10     *

CG Century Schoolbook   Italic P and L   Proportional  10     *

CG Triumvirate          Medium P and L   Proportional  10     *

CG Triumvirate          Bold   P and L   Proportional  14     *

* Desktop, Roman-8, PC-8, PC-8 D/N and PC-850 Symbol Sets
