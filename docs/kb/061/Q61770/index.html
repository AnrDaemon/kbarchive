Q61770: Hewlett-Packard Great Start Font Cartridge Description

Article: Q61770
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:2.x,3.0,3.0a,3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 18-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 2.03, 2.1, 2.11, 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

Below is a description of the capabilities of the Hewlett-Packard (HP) Great
Start Font Cartridge.

The font cartridge supports a print resolution of 300 x 300 dots per inch (dpi)
and has an average lifetime of 500 insertions. Separate fonts are required for
landscape and portrait. Some fonts may not be available in both orientations on
a specific cartridge.

For more information about this font cartridge, please contact Hewlett-Packard.

MORE INFORMATION
================

The following explains Great Start Cartridge capabilities:

Typeface        Type    Orient.   Pitch           Point  Symbol Set
--------        ----    -------   -----           -----  ----------

Letter Gothic   Medium  Portrait  27.27           3.6     *

Letter Gothic   Medium  Portrait  12              12      *

Letter Gothic   Bold    Portrait  12              12      *

Letter Gothic   Italic  Portrait  12              12      *

CG Times        Medium  Portrait  Proportional    11      *

CG Times        Bold    Portrait  Proportional    11      *

CG Times        Italic  Portrait  Proportional    11      *

* Desktop, Roman-8, ECMA-94 Latin 1, PC-8, PC-8 D/N and PC-850 Symbol Sets
