Q72775: Mac Wkst: Urgent Mail Is Not Delivered Faster

Article: Q72775
Product(s): Microsoft Mail For Appletalk Networks
Version(s): WINDOWS:2.0,2.0a,2.0b,3.0
Operating System(s): 
Keyword(s): 
Last Modified: 09-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail for AppleTalk Networks, versions 2.0, 2.0a, 2.0b, 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

In Microsoft Mail, sending a mail message with the Urgent option selected does
not give the mail message a greater priority in the Mail system and does not
decrease the time it takes for the mail message to be delivered.

If the Urgent option is selected for a mail message and the recipient of the mail
message has selected Display Urgent Messages Immediately in the Preferences
dialog box (on the Mail menu), the mail message will be automatically opened and
displayed on the recipient's workstation.
