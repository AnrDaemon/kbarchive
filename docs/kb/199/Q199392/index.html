Q199392: XCLN: Schedule+ View (Username.vue) Settings Overwritten after U

Article: Q199392
Product(s): Microsoft Exchange
Version(s): WINDOWS:7.5
Operating System(s): 
Keyword(s): 
Last Modified: 22-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Schedule+, version 7.5 
- Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you upgrade Schedule+ from version 7.0 to version 7.5, some view settings
are not retained.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft Schedule+ version
7.5.

MORE INFORMATION
================

View Second Time Zone settings are retained; however, Number of Days will
default to 1 and Tab Settings defined in Tab Gallery will revert back to the
default tabs.
