Q198376: WD97: Scroll Bars Reappear After Print Preview Full Screen

Article: Q198376
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbdta word97
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

Vertical and horizontal scroll bars that have been turned off reappear when you
return to normal or page layout view from print preview full screen view.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products that are
listed at the beginning of this article.

MORE INFORMATION
================

To turn the scroll bars off, follow these steps:

1. On the Tools menu, click Options.

2. Click the View tab.

3. Click to clear the Horizontal Scroll Bar or Vertical Scroll Bar check box.


Additional query words: scrollbar scroll bar scrollbars bars come back
