Q140146: Changing "(SMS)" Text in Shared Group Name

Article: Q140146
Product(s): Microsoft Systems Management Server
Version(s): winnt:1.1,1.2
Operating System(s): 
Keyword(s): kbnetwork smsappman kbSMSAppMan
Last Modified: 31-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Systems Management Server versions 1.1, 1.2 
-------------------------------------------------------------------------------

SUMMARY
=======

When a shared application group is created in Systems Management Server, "(SMS)"
is placed at the end of the title. This can be changed to any text you choose.

MORE INFORMATION
================

To change the group title text use the following steps:

1. Open the Program Groups window in SMS Administrator.

2. Create a new program group.

3. Give it a title with the following format:

  <title> (<text>)

The text you place in the parentheses replaces the "(SMS)" that is placed at the
end of the group title by default.

Additional query words: prodsms sms
