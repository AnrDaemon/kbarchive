Q167219: WD97: Word Hangs When You Open Version of Protected Document

Article: Q167219
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbusage word97
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you are using the versioning feature in Word 97 with a protected document
containing form fields, Word 97 may hang when you open an earlier version of the
document.


STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article. We are researching this problem and will post new
information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION
================

Versioning allows you to maintain a working history of a document. With this
feature, you can see exactly who did what to a document and when. You can also
enter descriptive text about the changes made in each version. All the
versioning information is stored within the document, so you don't need to
maintain and track multiple copies of the document

REFERENCES
==========

For more information on using the versioning feature in Word 97, please see the
following article in the Microsoft Knowledge Base:

  Q157775 WD97: Can't Save Version for Read-Only Document

  Q162989 WD97: Can't Save Versions of Word Document from Office Binder

  Q165739 WD97: How Word Determines the Name of a Saved Version

  Q159730 WD97: Word 97 Doesn't Save Version When Document Is Closed

  Q157477 WD97: Features Not Available During Web Authoring

For more information about "versioning," click the Office Assistant, type
"versioning," click Search, and then click one of the following topics:

- Review version information for a file.

- Delete one or more versions of a document.

For more information about "form fields," click the Office Assistant, type "form
fields," click Search, and then click one of the topics.

NOTE: If the Assistant is hidden, click the Office Assistant button on the
Standard toolbar. If Word Help is not installed on your computer, please see the
following article in the Microsoft Knowledge Base:

  Q120802 Office: How to Add/Remove a Single Office Program or Component
