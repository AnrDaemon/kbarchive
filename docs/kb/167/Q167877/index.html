Q167877: WD97: Voice Dictation System Appears to Run Slowly with Word

Article: Q167877
Product(s): Word 97 for Windows
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kb3rdparty word97kbfaq
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you use a voice dictation system with Word, the time spent generating and
sending keystrokes to Word appears to be slow.

CAUSE
=====

The following Word background options are turned on, which causes the apparent
slowness:

- All AutoCorrect options

- AutoComplete (AutoText option)

- All "AutoFormat As You Type" options

- Check Spelling As You Type

- Check Grammar As You Type

WORKAROUND
==========

To increase the speed of the voice dictation system in transferring keystrokes
to Word, turn off the background options listed in the "Cause" section of this
article.

Additional query words: 8.0 viavoice voicetype via voice type
