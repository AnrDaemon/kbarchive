Q70744: Running MS-DOS Shell in 43-Line Mode w/Renaissance EGA Board

Article: Q70744
Product(s): Microsoft Disk Operating System
Version(s): MS-DOS:5.x
Operating System(s): 
Keyword(s): 
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft MS-DOS operating system versions 5.0, 5.0a 
-------------------------------------------------------------------------------

SUMMARY
=======

If you run Microsoft MS-DOS version 5.0 Shell with a Renaissance GRX Legacy II
EGA board, you should not use 43-line mode. Because of a problem in the BIOS of
this EGA board, running Shell in 43-line mode will fail, causing the screen to
go completely blank.

Additional query words: renaissance 5.00 5.00a dosshell
