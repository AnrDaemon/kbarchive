Q195057: PA: Display Settings Incorrect or Cannot Close Dialog Box

Article: Q195057
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kb3rdparty kbdisplay kbenv kbimu msgamekbfaq
Last Modified: 10-AUG-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Pinball Arcade, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you play Pinball Arcade on a Microsoft Windows NT 4.0-based computer, you
may experience any of the following symptoms:

- When you quit Pinball Arcade, your default screen resolution and color
  palette may not be restored.

- When you start Pinball Arcade, and then click Options on the startup screen
  before you start a game, you may be unable to close the Options dialog box.

- When you quit a game in Pinball Arcade, return to the startup screen, and
  then click High Scores, you may be unable to close the High Scores dialog
  box.

CAUSE
=====

This behavior can occur if both of the following conditions are true:

- A Matrox Millennium display adapter is installed in your computer.

- You are using an outdated video driver for the display adapter.

RESOLUTION
==========

To resolve this issue, download and install the latest Windows NT 4.0 video
driver for the Matrox Millennium display adapter from the following Matrox Web
site:

  http://www.matrox.com
