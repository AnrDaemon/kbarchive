Q66840: ASD: Spot Plane View Is Too Close to 747

Article: Q66840
Product(s): Microsoft Home Games
Version(s): 
Operating System(s): 
Keyword(s): 
Last Modified: 30-JUL-2001

SUMMARY
=======

When viewing the 747 from spot view, it may be desirable to see more of the
plane and scenery.

To alter the spot view, from the Views menu, select option D to change the
distance, altitude, and position of the spot plane.

MORE INFORMATION
================

From the Views menu, option 9, Full screen external view, will double the amount
of screen space dedicated to spot view.
