Q183243: SMS: Access Violation in Wuser32.exe

Article: Q183243
Product(s): Microsoft Systems Management Server
Version(s): 1.2 SP3
Operating System(s): 
Keyword(s): 
Last Modified: 12-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Systems Management Server version 1.2 SP3 
-------------------------------------------------------------------------------


SYMPTOMS
========

When opening a remote control session with a client computer that is running
Microsoft Windows NT, Wuser32.exe may fail with an access violation C0000005.

This problem should only occur when the computers are using French regional
settings.


STATUS
======

Microsoft has confirmed this to be a problem in Systems Management Server
version 1.2. This problem has been corrected in the latest U.S. service pack for
Systems Management Server version 1.2. For information on obtaining the service
pack, query on the following word in the Microsoft Knowledge Base (without the
spaces):

  S E R V P A C K


Additional query words: prodsms gpf drwtsn32 av c0000005 wuser32
