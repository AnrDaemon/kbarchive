Q174850: Age of Empires: Minimum System Requirements

Article: Q174850
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbenv kbsetup aoe kbimu msgamekbfaq
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Age of Empires, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

This article describes the minimum system requirements to run Microsoft Age of
Empires.

MORE INFORMATION
================

The minimum system requirements to run Age of Empires are:

- A multimedia PC with a Pentium 90 or faster processor.

- Microsoft Windows 95 operating system or later, or Windows NT Workstation 4.0
  with Service Pack 3 or later.

- 16 megabytes (MB) of random access memory (RAM) for Windows 95, or 24 MB of
  RAM for Windows NT (or for an 8-player multiplayer game in Windows 95/98).

- 80 MB of available hard disk space.

- 50 MB of available disk space for swap file.

- A local Bus SVGA video adapter with 1 MB of Video RAM and a color SVGA
  monitor.

- A keyboard and Microsoft Mouse or compatible pointing device.

- A double-speed (2X) or faster CD-ROM drive for gameplay, or a quad-speed
  CD-ROM drive for cinematics.

- A Windows-compatible sound card with speakers or headphones to hear sound
  playback.

- A 28.8 kilobytes per second (KBPS) or faster modem for head-to-head or
  Internet play.
