Q174324: WD97: Can Add Bookmark to Hyperlink When No Bookmark Exists

Article: Q174324
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): word97
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you click Browse under "Named location in file (optional)" in the Edit
Hyperlink dialog box, you can type in a bookmark name, even though there are no
bookmarks in the file that the hyperlink points to.

Word does not show any errors, and if you click the hyperlink, Word opens and
shows the first page of the correct document in the document window.

STATUS
======

Microsoft has confirmed this to be a problem in the Microsoft products listed at
the beginning of this article.

MORE INFORMATION
================

There are no apparent negative effects from this behavior.
