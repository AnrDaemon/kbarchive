Q82593: Legacy 1.1 Hangs when Installed from File Manager

Article: Q82593
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 08-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

When you install the MBI Legacy word processor version 1.1 from Windows File
Manager by choosing Run from the File menu and then pressing the ENTER key or
choosing the OK button, the following error messages appears:

  Cannot Open Setup.inf

  Cannot find section in the setup file

If the error messages are ignored, then the installation hangs the computer. This
behavior does NOT occur if you double-click the SETUP.EXE file or choose Run
from Program Manager's File menu.

This problem was corrected in Legacy version 2.0.
