Q92435: ASMClock.exe Uses MASM 6.0 to Program in Windows

Article: Q92435
Product(s): Windows for Workgroups and Windows NT Networking Issues
Version(s): 3.11
Operating System(s): 
Keyword(s): kbfile kbsample kbgraphxlinkcritical
Last Modified: 13-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows Software Development Kit (SDK) for Windows for Workgroups, version 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

ASMClock.exe is a sample program that demonstrates how to perform programming
for Microsoft Windows with Microsoft Macro Assembler (MASM) version 6.0.

NOTE: You must have the Microsoft Windows version 3.1 Software Development Kit
and MASM version 6.0 to build ASMClock.exe.

MORE INFORMATION
================

The following files are available for download from the Microsoft Download
Center:

  DownloadDownload Asmclock.exe now
  (http://download.microsoft.com/download/platformsdk/sample/4/W31/EN-US/Asmclock.exe)

the article number below to view the article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services

posting to scan this file for viruses. Once posted, the file is housed on secure
servers that prevent any unauthorized changes to the file.
