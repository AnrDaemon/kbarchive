Q176165: XADM: Setup Error Message: This version of setup is older…

Article: Q176165
Product(s): Microsoft Exchange
Version(s): WinNT:5.5
Operating System(s): 
Keyword(s): kbusage
Last Modified: 14-MAR-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Exchange Server, version 5.5 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run Setup to add or remove components from an existing Microsoft
Exchange 5.5 Server computer, you may receive the following error message:

  This version of Exchange Setup is older than the one you previously
  used. Use a newer version of the Server Setup program.

CAUSE
=====

You are running the Setup program from Microsoft Exchange Server version 5.0.

RESOLUTION
==========

Use the Microsoft Exchange Server version 5.5 Setup program located on the
Microsoft Exchange Server version 5.5 CD-ROM.

Additional query words: install remove all reinstall
