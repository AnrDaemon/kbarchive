Q176846: Encarta Encyclopedia Does Not Recognize the Second CD-ROM

Article: Q176846
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:
Operating System(s): 
Keyword(s): kbenv kbhw kbmm kbHardwarekbfaq
Last Modified: 25-JUL-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 97 Encyclopedia for Windows 
- Microsoft Encarta Encyclopedia 97 Deluxe for Windows 
- Microsoft Encarta 98 Encyclopedia for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

While you are using Microsoft Encarta Encyclopedia for Windows, Encarta
Encyclopedia may prompt you to insert the second CD-ROM. When you insert the
second CD-ROM and click OK, Encarta Encyclopedia may continue to prompt you to
insert the second CD-ROM.

CAUSE
=====

This behavior can occur if you are using a real-mode CD-ROM driver. Many
real-mode CD-ROM drivers are unable to automatically detect when you insert a
different CD-ROM.

RESOLUTION
==========

To resolve this issue, contact your hardware manufacturer to inquire about
obtaining a 32-bit (protected-mode) driver for your CD-ROM drive.
