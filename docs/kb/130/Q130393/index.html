Q130393: Writer/Artist: Colored Squares Appear Around Some Objects

Article: Q130393
Product(s): Microsoft Home Kids Products
Version(s): WINDOWS:1.0,1.1,1.1a
Operating System(s): 
Keyword(s): 
Last Modified: 29-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Creative Writer for Windows, versions 1.0, 1.1, 1.1a 
- Microsoft Fine Artist for Windows, versions 1.0, 1.1, 1.1a 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you view graphics in Microsoft PowerPoint while you run either Creative
Writer or Fine Artist, blocks of color may appear around many of the objects in
Creative Writer and Fine Artist.

This problem occurs with drawing and painting tools in Fine Artist and is most
likely to occur when running a PowerPoint Slide Show.

WORKAROUND
==========

Use the following steps to work around the problem:

If you are using Fine Artist:

1. Quit Fine Artist.

2. Do NOT save the changes you have made to your work.

3. Exit PowerPoint.

4. Run Fine Artist.

5. If after following these steps the problem persists, restart Windows. Start
  Fine Artist again.

If you are using Creative Writer:

1. Quit Creative Writer. Save any changes you have made to your work.

2. Exit PowerPoint.

3. Start Creative Writer.

4. If the problem persists, restart Windows. Restart Creative Writer.

After following these steps, you should be able to run Creative Writer and Fine
Artist successfully.

MORE INFORMATION
================

Do not save your work in Fine Artist, because the colors may save incorrectly
and the original work may be lost.
