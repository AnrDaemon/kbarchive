Q172678: WD97: Changes in Word SR-2 Spelling and Thesaurus

Article: Q172678
Product(s): Word 97 for Windows
Version(s): WINDOWS:97
Operating System(s): 
Keyword(s): kbproof winword word97
Last Modified: 14-NOV-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Word 97 for Windows 
-------------------------------------------------------------------------------

SUMMARY
=======

The U.S. proofing tools that ship with Microsoft Word 97, Service Release 2,
contain a few changes from the U.S. proofing tools that shipped with Microsoft
Word 97.

MORE INFORMATION
================

The U.S. proofing tools that ship with Microsoft Word 97, Service Release 2,
contain the following changes:

- The word millenum is now flagged as misspelled. "Millennium" is suggested.

- Many common, socially unacceptable thesaurus suggestions have been removed.

- The Msth_am.lex and Mssp2_en.lex files were updated to remove spelling and
  thesaurus suggestions of obscene or politically incorrect words. Obscene
  words will not be suggested for misspellings that closely resemble them.

REFERENCES
==========

For more information about how to obtain the SR-2 update, please see the
following article in the Microsoft Knowledge Base:

  Q151261 OFF97: How to Obtain and Install MS Office 97 SR-2
