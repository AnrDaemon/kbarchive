Q94674: HPPCL5E: Memory Setting Goes Up to 130 MB

Article: Q94674
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 08-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------


SUMMARY
=======

The memory setting in the Hewlett-Packard (HP) LaserJet 4/4M driver, HPPCL5E,
can be set up to 130 MB (megabytes). This is by design and is not an error in
the driver.

The HP LaserJet 4/4M has the capability to use 130 MB of internal memory through
the use of 16 and 32 MB single in-line memory modules (SIMMs). The SIMMs are not
available at this time, but may be on the market in the future.
