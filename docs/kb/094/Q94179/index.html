Q94179: PC WFW: Description of Extensions for Windows for Workgroups

Article: Q94179
Product(s): Microsoft Mail For PC Networks
Version(s): 3.0
Operating System(s): 
Keyword(s): 
Last Modified: 15-NOV-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Mail and Schedule+ Extensions for Windows for Workgroups, version 3.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft Mail and Schedule+ Extensions for Windows for Workgroups provides
enhanced connectivity and advanced administrative features for a Windows for
Workgroups postoffice. It allows you to extend the mail and scheduling
functionality in three ways:

- It allows multiple postoffices to be connected together.

- It connects to other mail and scheduling postoffices.

- It provides remote dial-in support for Mail.

Extensions upgrades the Windows for Workgroups postoffice to version 3.0 of
Microsoft Mail for PC Networks, allowing Workgroups Mail to do all that the
retail Mail for Windows package can accomplish. The Extensions for Windows for
Workgroups product number is 068-499V300 and can be ordered through Microsoft
Customer Service at (800) 426-9400.
