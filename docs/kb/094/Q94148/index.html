Q94148: HPPCL5E: Printer Intellifont Fonts Are Not Listed

Article: Q94148
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a
Operating System(s): 
Keyword(s): 
Last Modified: 08-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------


SYMPTOMS
========

When using the printer driver for the Hewlett-Packard (HP) LaserJet III family
(HPPCL5A.DRV), version 30.3.86, text which is rotated 270 degrees and positioned
on the right margin may not print.

RESOLUTION
==========

This problem was corrected in the HPPCL5A.DRV driver version 31.3.89 released
with Microsoft Windows 3.1 and with HPPCL5MS.DRV version 2.0 released with
Microsoft Windows for Workgroups and available from Hewlett-Packard.
