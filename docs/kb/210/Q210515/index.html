Q210515: Golf 99: Distance Displayed on Screen Does Not Match Scorecard

Article: Q210515
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbui kbimu msgame
Last Modified: 09-FEB-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Golf 1999 Edition, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you play the twelfth hole in the Medalist Golf Club course in Microsoft
Golf 1999 Edition, the distance to the hole displayed on the screen may not
match the distance to the hole listed on the scorecard or stated in the flyby
commentary script for the hole.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Golf 1999 edition,
version 1.0.

MORE INFORMATION
================

If you edit a player to use the black tee, a distance of 189 yards to the
twelfth hole is displayed on the screen. However, the scorecard for the black
tee and the flyby commentary script both indicate that the distance to the hole
is 206 yards.
