Q210497: Golf 99: Incorrect Distance Stated in Flyby Commentary Script

Article: Q210497
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kbui kbimu msgame
Last Modified: 09-FEB-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Golf 1999 Edition, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you play the thirteenth hole in the Medalist Golf Club course in Microsoft
Golf 1999 Edition, the distance to the hole displayed on the screen and listed
on the scorecard may not match the distance to the hole stated in the flyby
commentary script for the hole.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Golf 1999 edition,
version 1.0.

MORE INFORMATION
================

If you edit a player to use the black tee, a distance of 663 yards to the
thirteenth hole is displayed on the screen, matching the distance to the hole
listed on the scorecard for the black tee. However, the flyby commentary script
states that the distance is only 614 yards.
