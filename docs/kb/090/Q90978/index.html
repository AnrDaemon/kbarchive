Q90978: Setup and Control Panel: Different Recommended Swap File Size

Article: Q90978
Product(s): Microsoft Windows 3.x Retail Product
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 01-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows for Workgroups versions 3.1, 3.11 
-------------------------------------------------------------------------------

You may notice that the recommended permanent swap file size is
different in Windows for Workgroups Setup than it is in Control Panel.
This is because Setup is running Windows for Workgroups in standard
mode, while Control Panel is probably being run from enhanced mode
Windows. The algorithm for calculating the recommended size of the
permanent swap file differs in the two modes of Windows.

Additional query words: 3.10 3.11 swapfile
