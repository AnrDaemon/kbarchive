Q148896: Return of Arcade: List of Supported Sound Cards

Article: Q148896
Product(s): Microsoft Home Games
Version(s): WINDOWS:1.0
Operating System(s): 
Keyword(s): kb3rdparty kbhw kbsound kbHardware
Last Modified: 08-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Return of Arcade for Windows, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Microsoft Return of Arcade uses DirectSound. DirectSound supports the following
sound cards:

- Sound Blaster 8-bit cards, 16-bit cards and the AWE32.

- ESS 488, 688, 1488, 1688

- Media Vision PAS 16, PAS Studio, PAS Plus, Pro 3D and Blue Lite Special

- Aztech Nova 16, Washington 16, Rocky 2

- Microsoft Windows Sound System

If your sound card is not listed above, you might experience problems ranging
from poor performance to sound delays and page faults when you play the games.

Additional query words: 1.00 roa sounds supported supports
