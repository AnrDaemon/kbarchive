Q147013: Past Reminders for Reoccuring Tasks Show After Event

Article: Q147013
Product(s): Microsoft Schedule+ for Windows
Version(s): WINDOWS:7.0
Operating System(s): 
Keyword(s): 
Last Modified: 16-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Schedule+ for Windows, version 7.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you set a reoccurring task with a past start date, the notification may
appear up to ten days late.


CAUSE
=====

Because send reminders are sent for up to 10 days past due, this has been
implemented as per design.

Additional query words: sched+ reminders past schedule plus
