Q68210: Using Coda’s Finale’ Music Software with Windows 3.0

Article: Q68210
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a
Operating System(s): 
Keyword(s): 
Last Modified: 13-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a 
-------------------------------------------------------------------------------

SYMPTOMS
========

If you use Coda Music Software program Finale' version 1.1 or 1.1.1, and your
computer uses a MIDI interface as an input source, you can only run Microsoft
Windows in real or standard mode.

CAUSE
=====

This problem is caused by the Coda MIDI driver. Coda is working to correct this
limitation.

MORE INFORMATION
================

If you have Finale' version 1.1, you may experience some problems when printing
to a PostScript printer. In this case, contact Coda for an upgrade to version
1.1.1 of Finale'. This upgrade is available to all registered customers for no
charge.

Both versions of Finale' have version 1.1 printed on the disk label. The version
number "1.1.1" can only be seen upon starting the Finale' program.

we make no warranty, implied or otherwise, regarding this product's performance
or reliability.
