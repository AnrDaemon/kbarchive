Q281447: USB Keyboard does not work properly with Compaq IA-2

Article: Q281447
Product(s): The Microsoft Network
Version(s): 1.0
Operating System(s): 
Keyword(s): 
Last Modified: 12-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- MSN Companion, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

The customer wants to use a universal serial bus (USB) keyboard with a Compaq
IA-2 device; the keyboard does not work.

CAUSE
=====

The customer has a USB keyboard that does not work, has stopped working, or
works sporadically

RESOLUTION
==========

Unplug the USB keyboard, and then plug the keyboard back in. The device should
work properly now.

MORE INFORMATION
================
