Q146404: Only Text Can Be Pasted into a Meeting Request

Article: Q146404
Product(s): Microsoft Schedule+ for Windows
Version(s): WINDOWS:7.0
Operating System(s): 
Keyword(s): 
Last Modified: 06-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Schedule+ for Windows, version 7.0, on platform(s):
   - the operating system: Microsoft Windows 95 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you send a meeting request, only the text can be pasted into the body of
the message.

CAUSE
=====

This is by design. The format of the clipboard contents is not checked until
after you choose to paste; a check is made if there is something to paste.
