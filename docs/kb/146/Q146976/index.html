Q146976: XCLN: Deleted Appointments are Downloaded to Intellilink

Article: Q146976
Product(s): Microsoft Schedule+ for Windows
Version(s): WINDOWS:7.0; Win95:7.0
Operating System(s): 
Keyword(s): 
Last Modified: 21-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Schedule+ for Windows, version 7.0 
-------------------------------------------------------------------------------


SYMPTOMS
========

When you download a recurring appointment to Intellilink, instances of the
appointment that have been deleted in Microsoft Schedule+ will still appear in
Intellilink.

MORE INFORMATION
================

The latest importers can be obtained through Microsoft Sales by ordering the
Microsoft Schedule+ 7.0 Import Converters, part number 061-00001.

These converters can also be downloaded with no charge at
http://www.microsoft.com/msscheduleplus/fs_sch.htm in the free software section
of the Microsoft Schedule+ page. Instructions for downloading are on the page.
Import converters include Lotus Organizer, ACT, and ECCO Pro.

STATUS
======

Microsoft has confirmed this to be a problem in version 7.0 of Microsoft
Schedule+. We are researching this problem and will post new information here in
the Microsoft Knowledge Base as it becomes available.
