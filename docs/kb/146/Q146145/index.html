Q146145: Schedule+ 1.0 Does Not Adjust for Time Zone Differences

Article: Q146145
Product(s): Microsoft Schedule+ for Windows
Version(s): WINDOWS:1.0,7.0
Operating System(s): 
Keyword(s): 
Last Modified: 16-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Schedule+ for Windows, versions 1.0, 7.0 
-------------------------------------------------------------------------------

SUMMARY
=======

If you use the Meeting Organizer Wizard in Schedule+ version 7.0 to set a
meeting for 6 P.M. Pacific Standard Time (PST), attendees using Schedule+
version 1.0 who are in other time zones will see the meeting time as 6 P.M. PST
because Schedule+ version 1.0 does not adjust for time zone differences.

MORE INFORMATION
================

Schedule+ version 7.0 is able to adjust for time zone differences. It will
adjust to the time zone of the Meeting Organizer.
