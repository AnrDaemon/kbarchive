Q146432: Bookshelf 95: How to Register Online After Setup Completes

Article: Q146432
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:95
Operating System(s): 
Keyword(s): 
Last Modified: 17-DEC-2000

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Bookshelf '95 for Windows 95 
- Microsoft Windows 95 
-------------------------------------------------------------------------------

SUMMARY
=======

This article tells how to register Bookshelf '95 for Windows 95 using the online
registration feature.

MORE INFORMATION
================

Bookshelf Setup provides an online registration option. During Setup, you can
use your modem to register the product online. You also can click OK to end the
installation without registering. If you choose this option, you can register
online at a later time.

How to Register Bookshelf Online
--------------------------------

1. Run Bookshelf Setup. Setup will start in Maintenance mode if Bookshelf is
  already installed.

2. Click the Online Registration button. Follow the onscreen instructions.
