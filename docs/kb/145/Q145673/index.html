Q145673: How Much Does Pregnancy and Child Care Cost?

Article: Q145673
Product(s): Microsoft Home Multimedia Titles
Version(s): WINDOWS:95
Operating System(s): 
Keyword(s): 
Last Modified: 01-JUN-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Pregnancy and Child Care, Preview on MSN 
-------------------------------------------------------------------------------

SUMMARY
=======

The cost for using this special Preview version of Microsoft Pregnancy and Child
Care is included in the standard membership fees for MSN, The Microsoft Network
online service.

MORE INFORMATION
================

For more information about this topic and other related issues, see the
Microsoft Knowledge Base (KB). The KB contains thousands of articles that
include answers to common questions about using Microsoft programs. To access
the Knowledge Base, return to the container where you found this document and
click the Microsoft Knowledge Base icon.
