Q115064: Encarta: Screen Corruption with Diamond Viper Using VGA Driver

Article: Q115064
Product(s): Microsoft Home Multimedia Titles
Version(s): 1994 editions,1995 editions; WINDOWS:
Operating System(s): 
Keyword(s): 
Last Modified: 26-MAY-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Encarta 95 The Complete Interactive Multimedia Encyclopedia 
- Microsoft Encarta 1994 The Complete Multimedia Encyclopedia 
- Microsoft Encarta 96 Encyclopedia for Windows 
-------------------------------------------------------------------------------

SYMPTOMS
========

Using the VGA video driver that came with Microsoft Windows (VGA.DRV) on a
system with a Diamond Viper video card can cause screen corruption in the
Windows Shell and/or in Encarta.

RESOLUTION
==========

Use the most current Diamond Viper video driver with 256 or more colors.

For more information about Diamond Viper video cards and which Viper drivers are
most current, contact Diamond Computer Systems technical support at:

- Technical support: (408) 325-7100

- BBS support 2400 only: (408) 325-7080

- BBS support 9600 only: (408) 325-7175

MORE INFORMATION
================


Viper video cards are manufactured by Diamond Computer Systems, a vendor
independent of Microsoft; we make no warranty, implied or otherwise, regarding
this product's performance or reliability.
