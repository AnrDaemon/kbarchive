Q163938: Inserting Multifunction PC Card May Hang Computer

Article: Q163938
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 2,2.1
Operating System(s): 
Keyword(s): kbhw osr2 win95 kbHardware
Last Modified: 25-MAR-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 OEM Service Release, versions 2, 2.1 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you insert certain multifunction PC Cards in a PC Card slot in some
computers, the computer may stop responding (hang). When this occurs, you do not
hear an audible insertion tone.

This behavior has been observed when a Xircom CreditCard Ethernet+Modem (CEM28)
PC Card is inserted in one of the following computers:

- Hewlett-Packard Omnibook 4000CT

- Hewlett-Packard Omnibook 5500CT

- Texas Instruments Extensia 570CDT


RESOLUTION
==========


STATUS
======

This problem no longer occurs in Windows 98. To resolve this problem, install
the current version of Windows. For information about the current version of
Windows, visit http://www.microsoft.com/windows.

MORE INFORMATION
================
