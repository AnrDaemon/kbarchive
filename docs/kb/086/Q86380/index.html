Q86380: PC WSPlus: Problem Sending Mail Does Not Update Appointment

Article: Q86380
Product(s): Microsoft Schedule+ for Windows
Version(s): WINDOWS:1.0,1.0a
Operating System(s): 
Keyword(s): 
Last Modified: 19-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Schedule+ for Windows, versions 1.0, 1.0a 
-------------------------------------------------------------------------------

SYMPTOMS
========

After you send a meeting notice with Microsoft Schedule+ for Windows, the
Appointment Book window will display an envelope icon next to the name of each
attendee the message was sent to. If there is a problem sending the message to
one or more of those attendees and the message is returned, Schedule+ will not
update the icons to reflect this.

STATUS
======

Microsoft is researching this problem and will post new information here in the
Microsoft Knowledge Base as it becomes available.

Additional query words: 1.00 1.00a schedule plus
