Q86273: Panasonic KX-P 1695 Missing Paper Size

Article: Q86273
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 08-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------


SUMMARY
=======

The Microsoft Windows version 3.1 printer driver for the Panasonic KX-P 1695
version 1.1 is missing the US Standard Fanfold 14 7/8-by-11-inch paper size.

WORKAROUND
==========

1. Set the Panasonic KX-P 1695 to Epson emulation mode. Consult your manual for
  instructions.

2. Install and use the Epson FX-286e driver, which has the additional paper
  size.

MORE INFORMATION
================

Microsoft has confirmed this to be a problem in Windows version 3.1. We are
researching this problem and will post new information here as it becomes
available.


Additional query words: 3.10 fan fold KX-P1695 KXP1695 KXP 1695 3.11
