Q69598: Epson Emulation Modes for ALPS Printers

Article: Q69598
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.0,3.0a,3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 07-DEC-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.0, 3.0a, 3.1, 3.11 
-------------------------------------------------------------------------------


SUMMARY
=======

There are no Microsoft Windows compatible printer drivers available for the ALPS
printer series. ALPS printers emulate a variety of Epson printers. Install the
appropriate Epson printer driver(s) that your ALPS emulates.

MORE INFORMATION
================

The following is the list of the proper emulation modes for each of the ALPS
printers.

  Printer         Emulation
  ---------------------------------------------
  ASP1600         Epson FX-85
  Allegro 500     Epson LQ-2550
  Allegro 500XT   Epson LQ-2550
  ALQ224/324      24-pin Epson LQ1500
  ALQ224e/324e    24-pin Epson LQ 2500,
                  9-pin Epson FX 185 (Color JX80)
  P2000+          Epson FX-100
  P2100           Epson FX-100
  LPX600          HP LaserJet+, IBM Proprinter,
                  IBM Graphics Printer, Epson FX-80
