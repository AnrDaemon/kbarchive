Q193138: Windows NT 4.0 Does Not Support Unimodem V

Article: Q193138
Product(s): Microsoft Windows NT
Version(s): WINNT:4.0
Operating System(s): 
Keyword(s): 
Last Modified: 09-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Server, Enterprise Edition version 4.0 
- Microsoft Windows NT Server version 4.0 
- Microsoft Windows NT Workstation version 4.0 
-------------------------------------------------------------------------------

SUMMARY
=======

Windows NT version 4.0 does not support Microsoft Unimodem V.

MORE INFORMATION
================

Windows NT Telephony Service Provider (TSP) supports the initial implementation
of the Unimodem driver. Microsoft Telephony Application Programming Interface
(TAPI) 2.0 is a Win32 service that calls program functions, which TSP translates
to modem AT commands.

Unimodem V is available for Microsoft Windows 95 and is included with Microsoft
Windows 98.

For additional information about Unimodem V, please see the following article in
the Microsoft Knowledge Base:

  Q140323 Microsoft Unimodem V Readme.txt File

REFERENCES
==========

For more information about the Unimodem V specification, refer to the following
Microsoft sites on the Internet:

  http://msdnisv.microsoft.com/communication/

  ftp://ftp.microsoft.com/developr/TAPI/
