Q127042: Automap: Streets Install Cannot Find Existing Automap

Article: Q127042
Product(s): Microsoft Automap
Version(s): 1.01,1.06,1.1,4.0
Operating System(s): 
Keyword(s): 
Last Modified: 02-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Automap Road Atlas for Windows, versions 1.06, 1.1 
- Microsoft Automap Streets for Windows, versions 1.01, 4.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you run the installation program for Automap Streets and select the Install
To Work With Existing Automap Road Atlas option, the following message may
appear:

  Streets installation cannot find an existing version of Automap Road Atlas

This error occurs when you try to install Automap Streets to work with an
existing copy of Automap Road Atlas version 1.06 and 1.1.

WORKAROUNDS
-----------

Workaround 1
------------

Select the Install Automap Streets And 4 Downtown Maps option and allow the
installation to install Streets as a separate program.

Workaround 2
------------

Install Automap Road Atlas versions 3.0, 3.02, or 3.04 for Windows.

MORE INFORMATION
================

When Automap Streets is installed, the program attempts to update the current
version of Automap Road Atlas to 3.04. To update the program, Automap Road Atlas
version 3.0, 3.02, or 3.04 must exist on your computer.
