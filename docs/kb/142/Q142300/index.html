Q142300: Creative Labs Sound Blaster Plug and Play Does Not Work

Article: Q142300
Product(s): Microsoft Windows NT
Version(s): 3.50 3.51
Operating System(s): 
Keyword(s): 
Last Modified: 08-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation versions 3.5, 3.51 
- Microsoft Windows NT Server versions 3.5, 3.51 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you install the Windows NT Sound Blaster 16 driver or the Sound Blaster 16
driver provided by Creative Labs for the Creative Labs Sound Blaster Plug and
Play (PnP) sound cards, the following error message appears:

  The sound card cannot be found with this port number. Check your settings and
  try again.

CAUSE
=====

This problem occurs on most of the Creative Labs Sound Blaster PnP sound cards,
such as the following:

  - Creative Labs Sound Blaster 32 PnP
  - Creative Labs Sound Blaster AWE 32 PnP

Windows NT does not provide plug and play support for the sound cards above.

RESOLUTION
==========

For more information, contact Creative Labs Technical Support at (405)
742-6655.
