Q83841: Novell Search Paths to Windows 3.0 Can Cause Problems for 3.1

Article: Q83841
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 10-OCT-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

If you are unable to start Microsoft Windows operating system version 3.1 on a
Novell network after performing a network Setup (SETUP/N), old Windows 3.0 files
may still be in the Novell Search or Map paths.

If Windows 3.0 files are in the Map or Search path, you may experience system
hangs, network hangs, or receive the following error messages when starting
Windows 3.1 in enhanced mode:

  Invalid VxD Dynamic Link Call to Device # 0021, Service 8006
  Invalid VxD Dynamic Link Call to Device # 0010, Service 8001

Additional query words: 3.10
