Q262247: Asheron’s Call: How to Restore Accidentally Deleted Characters

Article: Q262247
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kbimu msgame
Last Modified: 16-MAY-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Asheron's Call, version 1.0 
- Microsoft Asheron's Call: Dark Majesty, version 1.0 
-------------------------------------------------------------------------------

SUMMARY
=======

If you accidentally delete a character in Microsoft Asheron's Call, you may
restore the lost character within one hour of the deletion.

MORE INFORMATION
================

When you delete a character in Asheron's Call, the name of the character remains
on the character list. However, the name appears red and a line is drawn through
it.

To restore the deleted character, click the name of the character to select the
character. When you do so, the Delete Character button changes to a Restore
Character button. Press the Restore Character button to restore the deleted
character.

NOTE: If you do not restore a character within one hour after you delete the
character, the character is permanently lost.
