Q261998: Allegiance: Actions You Assign to Game Controller Do Not Occur

Article: Q261998
Product(s): Microsoft Home Games
Version(s): 1.0
Operating System(s): 
Keyword(s): kbhw kbimu msgame kbHardware
Last Modified: 12-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Allegiance, version 1.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you assign an action to an axis or a button on your game controller in
Microsoft Allegiance, the assigned action may not occur when you move the axis
or click the button.

RESOLUTION
==========

To work around this issue, assign all of the actions that you want to the axes
and buttons on your game controller, and then follow these steps:

1. Start a game or a training mission in Allegiance.

2. Press the ESC key to display the Options menu.

3. Point to Game Options.

4. Click "Map Keys and Controls".

5. Click Accept.

When you return to the game or the training mission in Allegiance, all of the
assigned actions occur as expected.

Additional query words: 1.00 msgame igz msngz gaming device joystick command
