Q261084: MCM2: Cannot Join Multiplayer Game on Zone.com

Article: Q261084
Product(s): Microsoft Home Games
Version(s): 2.0
Operating System(s): 
Keyword(s): kbenv kbnetwork kbimu msgame
Last Modified: 09-NOV-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Motocross Madness 2, version 2.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

When you attempt to play Microsoft Motocross Madness 2 with other players on
Zone.com or a Local Area Network (LAN), you may be unable to join a multiplayer
game.

CAUSE
=====

This behavior can occur if you attempt to connect to the multiplayer game by
using Microsoft Windows 98 Internet Connection Sharing (ICS).

RESOLUTION
==========

To resolve this issue, connect directly to the Internet to play Motocross
Madness 2 with other players.

STATUS
======

Microsoft has confirmed this to be a problem in Microsoft Motocross Madness 2,
version 2.0.

MORE INFORMATION
================

Motocross Madness 2 is not compatible with Windows 98 ICS or certain other forms
of Network Address Translation (NAT).
