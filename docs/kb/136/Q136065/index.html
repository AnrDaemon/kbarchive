Q136065: Windows 95 Help File Available

Article: Q136065
Product(s): Microsoft Windows 95.x Retail Product
Version(s): 
Operating System(s): 
Keyword(s): kbfile
Last Modified: 13-FEB-2002

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows 95 
-------------------------------------------------------------------------------

SUMMARY
=======

The Microsoft PSS Knowledge Base group regularly extracts the most current
articles from the Windows database and creates Windows Help files that are then
posted to the appropriate CIS forum and the Microsoft Software Library on CIS.
These files are also updated as new information becomes available.

The file(s) listed below are available to download from the Microsoft Software
Library (MSL). For information about downloading these files, please see the
"Instructions for Downloading" section at the end of this article.

FILE(S) AVAILABLE FROM ONLINE SERVICES
--------------------------------------

The following file(s) are available:

Item ID      Filename           Description
--------------------------------------------------------------
S15541       Win95.exe          Basic questions for Windows 95


INSTRUCTIONS FOR DOWNLOADING
----------------------------

The following file is available for download from the Microsoft Software
Library:

  ~ Win95.exe

For more information about downloading files from the Microsoft Software Library,
please see the following article in the Microsoft Knowledge Base:

  Q119591 How to Obtain Microsoft Support Files from Online Services
