Q136038: Diskeeper 1.0 Compatibility In Windows NT 3.51

Article: Q136038
Product(s): Microsoft Windows NT
Version(s): 3.51
Operating System(s): 
Keyword(s): 
Last Modified: 08-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Workstation version 3.51 
- Microsoft Windows NT Server version 3.51 
-------------------------------------------------------------------------------

Diskeeper 1.0, a third-party on-line disk Defragmentor for Windows NT,
does not work under Windows NT 3.51. It does not have the capability to
support the newer features of the Windows NT file system (NTFS) in 3.51
nor does it support the new file and directory compression feature.

A Windows NT 3.51 compatible version of Diskeeper, version 1.01 is
available on Executive Software's on-line services. The incremental
release (DK101I.EXE) is available free to all registered users. Executive
Software's on-line services:

CompuServe:  Windows Utility Vendor A Forum (WINUTA), Library 13

FTP Site:    ftp.netcom.com
            /pub/ex/execsw

A newer release of the Diskeeper upgrade for Windows NT 3.51 has been
released. You can obtain this release on the following WEB site on the
Internet:

  http://www.execsoft.com/download/

If you cannot download the incremental release, you can order the software
on disk directly from Executive Software for $9.95, the cost of
shipping and handling.

  Executive Software
  701 North Brand Blvd., 6th Floor
  Glendale, California, USA 91203-1242

  Technical Support: (818) 547-2050
  FAX: (818) 545-8801

This upgrade is not backwards compatible with Windows NT 3.5.

Diskeeper is manufactured by Executive Software, a vendor independent of
Microsoft; we make no warranty, implied or otherwise, regarding this
product's performance or reliability.
