Q87084: VGA Chip Sets and Revisions Used on Video Cards

Article: Q87084
Product(s): Microsoft Windows 95.x Retail Product
Version(s): WINDOWS:3.1,3.11
Operating System(s): 
Keyword(s): 
Last Modified: 27-SEP-1999

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows versions 3.1, 3.11 
-------------------------------------------------------------------------------

SUMMARY
=======

Below is a summary of the video chipsets used by many video cards and a
description of the firmware/ROM revisions required to use Windows 3.1 video
drivers.

MORE INFORMATION
================

Display Type               Models
------------               -------

ATI Ultra                  ATI 8514 Ultra
                          ATI Graphics Ultra
                          ATI Graphics Vantage
                          All other ATI Ultra based
                          adapters, including OEM versions

  NOTE: ATI Ultra BIOS version 1.3 or later is required. Contact ATI
  or your video manufacturer for BIOS updates. Also, ATI Ultra
  adapters work well with the Windows 3.1 8514/a driver. ATI's
  drivers provide extra performance.

DGIS 3.0                   GSS 1000 series
                          Hewlett-Packard (HP) IGC-10, IGC-20
                          Spectragraphics Squeegee
                          NEC MultiSync Graphics Engine (MGE)
                          Zenith Data Systems Z-649
                          COMPAQ AG1024
                          All other DGIS 3.0 compatible adapters

  NOTE: DGIS adapters require firmware updates for Windows 3.1
  (DGIS.EXE, DGIS.OUT). These files can be obtained from your display
  adapter manufacturer. Also, the Spectragraphics/GSS BBS contains
  the latest DGIS files. This BBS can be accessed by modem at (503)
  646-1743.

S3 Based Adapters          Actix Quantum
                          Diamond Stealth
                          Orchid Fahrenheit 1280
                          STB Wind/X
                          VideoMagic PCI-v805 (Phoenix)
                          All other S3 compatible adapters

Tseng ET4000               Everex Viewpoint, Vision
                          Diamond SpeedStar
                          Orchid Prodesigner II, IIs
                          Cardinal VGA
                          Sigma Designs VGA Legend
                          Sota Technology Lightning VGA
                          STB PowerGraph VGA, Ergo
                          All other Tseng ET4000 compatible adapters

Additional query words: 3.10 fixlist getwdl 3.11 chip set chipset
