Q104926: Peachtree for Windows May Not be Able to Open a Company

Article: Q104926
Product(s): Microsoft Windows NT
Version(s): 3.1
Operating System(s): 
Keyword(s): kb3rdparty
Last Modified: 08-AUG-2001

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Windows NT Server version 3.1 
- Microsoft Windows NT Workstation version 3.1 
- Microsoft Windows NT Advanced Server, version 3.1 
-------------------------------------------------------------------------------

SYMPTOMS
========

Under Windows NT, Peachtree Accounting for Windows version 2.0 (Batch 02) may
not be able to use some of the company functions.


Peachtree will run through the correct steps in setting up companies, but if you
choose Open Company from the File menu, no dialog box appears so that you can
select the company of choice to open. The same thing may happen if you choose
Delete Company from the File menu; you may not be given a list of companies to
delete.

RESOLUTION
==========

According to Peachtree for Windows Technical Support, upgrading to Batch 05
should correct this problem.

For additional information, contact Peachtree Technical Support at (404)
279-2099.


Peachtree Accounting for Windows is manufactured by Peachtree Software, a vendor
independent of Microsoft; we make no warranty, implied or otherwise, regarding
this products' performance or reliability.
