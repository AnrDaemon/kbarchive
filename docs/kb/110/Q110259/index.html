Q110259: Flight Simulator 5.0: Map View Is Black

Article: Q110259
Product(s): Microsoft Home Games
Version(s): 5.0
Operating System(s): 
Keyword(s): 
Last Modified: 07-MAY-2002

kbusage

-------------------------------------------------------------------------------
The information in this article applies to:

- Microsoft Flight Simulator for MS-DOS, version 5.0 
-------------------------------------------------------------------------------

SYMPTOMS
========

The map view in Flight Simulator 5.0 may be filled black when you zoom out to
the global view. To correct this problem, try one of the following resolutions.

RESOLUTION
==========

RESOLUTION 1
------------

This problem occurs when Textured Ground and Aircraft Texture are disabled.

Select Textured Ground or Aircraft Texture from the Display Preferences dialog
box. As long as at least one of these options is selected, the map view should
display correctly.

RESOLUTION 2
------------

This problem also occurs when there is less than 80K of expanded (EMS) memory
configured on the system.

Expanded memory is required to take advantage of most scenery options in Flight
Simulator 5.0. The memory must be configured for at least 80K of expanded memory
or some options may not be available.

MORE INFORMATION
================

For more information about how to configure the RAM memory as expanded memory,
consult the inside back cover of the Microsoft Flight Simulator Pilot's Handbook
or your MS-DOS or Windows manual.
